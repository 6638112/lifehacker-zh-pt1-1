# 如何在 5 分钟内将 Adobe Flash 从 Windows 10 中移除

> 原文：<https://lifehacker.com/how-to-remove-adobe-flash-from-windows-10-in-5-minutes-1845509875>

实在没有理由再让 Adobe Flash 无所事事了。没有人使用它，现代网络浏览器将在明年年初摆脱它，多年来它一直是设备安全方面的一个永久的刺——尽管，是的，它让我们在过去的上有很多乐趣。



该说再见了。你很幸运，因为微软有一个全新的 Windows 更新——[kb 4577586](https://support.microsoft.com/en-us/help/4577586/update-for-removal-of-adobe-flash-player)，你可以然后通过微软的更新目录 [这里](http://www.catalog.update.microsoft.com/search.aspx?q=4577586)——将 Adobe flash 从 Windows 8.1 和 Windows 10 中剥离出来。

事实上，你需要做的最困难的事情是记住你正在使用的 Windows 版本。你可以点击你的开始按钮，输入**“系统信息”，**加载应用程序，看一下“系统类型”行:

之后，再次打开你的开始菜单，输入**“winver”，**，点击**进入**。您应该会看到类似这样的内容:

因此，在我的情况下，我知道我需要 Windows 版本 20H2 的 x64 (64 位)版本的更新。然而，具体的更新并不存在。在更新的第二页——是的，有两页，你可以使用右上角的“下一页”链接翻到第二页——只有 Windows 10 version 2004 的更新，也就是 Windows 10 20H1。

这就是我在 Windows 10 上用来抛弃 Flash 的东西。这应该没问题，因为这是 Windows 10 最新版本的更新。否则，你应该确保尽可能选择与你的 Windows 10 版本一致的工具。

点击正确的文件下载，你会得到一个弹出窗口，看起来像这样。

点击*链接*开始下载，大约需要一秒钟。然后你需要拉起。MSU 文件，你在文件浏览器下载并运行它。

这也需要大约十秒钟才能完成。你不需要重启你的电脑。Adobe Flash 将会消失。这真的再简单不过了，除了将此更新作为标准 Windows 更新过程的一部分提供，这肯定会很快发生。这只是移除工具的早期版本。生活在边缘。

正如 [电脑](https://www.bleepingcomputer.com/news/microsoft/microsoft-releases-update-to-remove-adobe-flash-from-windows) 所说，微软的更新不会*而不是*移除你自己安装的任何版本的 Flash。抓住 Adobe 的卸载工具 来抛弃它们。
