# 新数据显示，怀孕期间不接种疫苗是更危险的选择

> 原文：<https://lifehacker.com/new-data-shows-not-getting-vaccinated-in-pregnancy-is-t-1847475082>

COVID 疫苗一直是任何怀孕或考虑怀孕的人以及母乳喂养者的一种选择。但是起初，数据缺乏，和 [怀孕的人可以自己决定](https://lifehacker.com/yes-you-can-get-the-covid-vaccine-if-youre-pregnant-1846151312) 是否注射。现在，多项研究表明，怀孕期间接种疫苗的益处大于风险。



疾病预防控制中心现在已经发布了一组关于在怀孕前或怀孕期间接种疫苗的人，或正在哺乳的人的数据，这些数据来自所有这些都指向同一个方向:疫苗是安全的，而*而不是*接种 COVID 疫苗是风险更大的选择。

## COVID 疫苗导致流产风险增加

根据新发布的研究 之一的 [，流产在接种疫苗的人群中并不比在普通人群中更常见。研究人员跟踪了 2500 名在怀孕前或前 20 周接受了辉瑞或 Moderna 疫苗的人。13%的人流产过；疾病预防控制中心报告称，通常情况下， 11-16%的妊娠以流产告终。](https://www.researchsquare.com/article/rs-798175/v1)

## 妊娠数据监测无问题

疾病预防控制中心跟踪从 [多个数据来源](https://www.cdc.gov/coronavirus/2019-ncov/vaccines/safety/monitoring-pregnant-people.html) 的妊娠结果，他们指出，到目前为止，他们“没有发现任何对怀孕后期接种疫苗的孕妇或其婴儿的安全担忧。”(前面提到的关于流产的研究，观察了怀孕早期接种疫苗的人。)

[那项研究](https://www.nejm.org/doi/full/10.1056/nejmoa2104983) 包括超过 35，000 人，发现妊娠丢失、早产和出生体重过轻的比率与 COVID 之前的研究相当。换句话说，这项研究或流产研究没有直接的对照组(为了研究而拒绝接种疫苗将被认为是不道德的)，但数据支持这样的观点，即接种疫苗的人与未接种疫苗的人一样有可能健康怀孕。

## 接种疫苗可能会保护你的宝宝，甚至在出生后

另一项重要的研究是 [这一项](https://www.ajog.org/article/S0002-9378(21)00187-3/fulltext) ，它考察了疫苗在怀孕人群中的有效性。(记住，如果人们怀孕了，他们会被排除在最初的疫苗试验之外。)正如你所料，疫苗不仅有效，婴儿出生后还保留了一些针对 COVID 病毒的抗体。

这种类型的免疫在其他疫苗中是已知的，这就是为什么每次怀孕 期间都推荐接种百日咳疫苗(TDaP)[的原因，即使你作为孕妇已经接种了疫苗——y 我们的宝宝在几个月内会保留其中一些抗体。](https://www.cdc.gov/pertussis/pregnant/mom/get-vaccinated.html)

虽然还没有太多关于疫苗和母乳喂养的信息，但有证据表明母乳中可能存在抗 COVID 的抗体。该疫苗被认为对哺乳期的人及其婴儿是安全的。

## COVID 比注射更危险

怀孕期间想要避免任何潜在的危险是合理的。这就是为什么人们经常拒绝可以等到分娩后的药物和程序。但是在 COVID 疫苗的情况下，选择不是在疫苗和什么都没有之间；这是疫苗和潜在感染 COVID 之间的问题。

怀孕或最近怀孕的人如果感染 COVID，患严重疾病(包括住院和死亡)的风险会增加。与未感染该病毒的人相比，感染 COVID 的人早产或生下健康状况更差的婴儿的风险也更大。

基于这些原因，CDC 现在同意 [美国妇产科医师学会](https://www.acog.org/clinical/clinical-guidance/practice-advisory/articles/2020/12/covid-19-vaccination-considerations-for-obstetric-gynecologic-care) 和 [母胎医学学会](https://s3.amazonaws.com/cdn.smfm.org/media/3040/COVID_vaccine__Patients_JULY_29_2021_final.pdf) 在强烈敦促 怀孕患者接种疫苗。
