# 你能为混乱的脸书数据泄露做些什么

> 原文：<https://lifehacker.com/what-you-can-do-about-that-messy-facebook-data-breach-1846619772>

旧的数据漏洞仍然是一个数据漏洞，当它涉及到大多数人在某个时候使用过的网站脸书时，你可能仍然需要注意它。你可能已经听说过，在一名黑客发布了大量从被黑账户中窃取的个人信息后，过去影响 5 亿脸书用户的数据泄露事件又开始流传了。即使真正的黑客攻击发生在两年前，你仍然可以采取一些预防措施，以确保最近的事件不会对你产生太大影响。

Watch

首先，花时间检查你的数据，包括你的电子邮件地址、电话号码、姓名和其他识别特征，是否出现在上述漏洞中。尝试以下选项之一来搜索可能已被破坏的不同识别信息:

*   [我被 pwn 了](https://haveibeenpwned.com/) (邮件)
*   [我被 Facebooked 了吗？](https://haveibeenfacebooked.com/) (电话号码)
*   [每天的新闻](https://www.thenewseachday.com/facebook-phone-numbers-us) (电话号码)
*   [脸书审核人](https://facebookleak.netlify.app/) (电话号码)
*   [原始数据转储](https://www.dropbox.com/s/1u9xhru6uwh97hk/phone-numbers.csv) (一切)

如果您的电话号码或其他详细信息没有被泄露，那太好了！你很棒。如果是的话，你也无能为力，因为信息已经公开了。从好的方面来说，你不必担心你的密码被盗，但这些数据*可能*被用于其他地方的网络钓鱼企图，甚至可能重置你的密码或强行进入你的账户，这取决于服务的密码恢复/重置机制有多糟糕。

也就是说，由于这一数据已经浮动了一段时间，如果你还没有*受到*影响，那么你很可能*没有*受到影响。如果有什么不同的话，这整集显示了注册账户时尽可能使用虚假信息的重要性——尤其是在社交媒体上。混淆完全在你的控制之内。不要觉得你必须放弃你生活中合理的细节。

对脸书来说，这可能包括:

*   如果没有必要的话，不要提供任何可选信息(你的教育经历，你生活过的地方，你的兴趣，你的工作经历等等。)
*   被询问时提供虚假信息，比如假的生日或假名。(在 [你最喜欢的密码管理器](https://lifehacker.com/the-five-best-password-managers-5529133) 的注释部分记录下这些信息，以防你因为伪造信息而被质疑，以便重新访问你的帐户或重置你的密码。)
*   使用一个假的电子邮件地址(即使是稍微修改过的，比如 youremail+facebook@gmail.com 而不是 youremail@gmail.com)和一个电话号码 [，而不是你的真实电话号码](https://lifehacker.com/install-the-google-voice-app-if-you-use-that-number-for-1846453445) 来注册一个账户。像以前一样，将这些凭证保存在您的密码管理器中，以防万一。
*   使用一个不同的“真实的”名字或稍微修改过的名字。不要在所有不同的服务中使用相同的“网名”或账户名(毫无疑问，这是我们都会犯的错误)。

为什么这很重要？这使得攻击者很难利用从一个数据漏洞收集到的信息在其他地方影响您。如果你总是在你使用的各种服务中尽可能地使用不同的信息，攻击者就更难通过社交工程进入你的帐户——他们只是不太了解你的每项服务。

展望未来，请确保您始终处于任何潜在网络钓鱼企图的顶端。如果你突然收到一条短信或电子邮件，试图通过向你提供一些关于你自己的信息(现在很容易获得)来说服你相信它的合法性，不要仅仅根据这条信息就对某项服务采取行动。不要向任何人提供任何其他信息作为回应。如果你不确定链接来自哪里，不要点击链接。取而代之的是，打开你的浏览器，直接访问这个服务，看看是否真的有什么需要你去处理的。
