# 你现在可以将重要的应用程序固定在 Windows 的屏幕上

> 原文：<https://lifehacker.com/you-can-now-pin-important-apps-to-the-screen-in-windows-1848505336>

如果你曾经希望在记下 Zoom 讲座或 YouTube 视频的笔记时，可以将 Microsoft Word 固定在屏幕上，那么你很幸运。多亏了微软的 PowerToys for Windows——具体来说，就是 [总是在最上面](https://aka.ms/PowerToysOverview_AlwaysOnTop) 实用程序——你现在可以将你最重要的窗口钉在最上面，这样它们就可以保持固定(暂时)在所有和其他之上。



[PowerToys](https://github.com/microsoft/PowerToys/) 是一组为 Windows 增加有用功能的工具，而 Always On Top 是最新加入该套件的实用程序。要安装它，进入 PowerToys 的 [GitHub 页面，下载安装文件。](https://github.com/microsoft/PowerToys/releases/tag/v0.55.1)

一旦你安装了 PowerToys，点击左窗格中的**总是在顶部**并检查右窗格中的**启用总是在顶部**是否被启用。将任何窗口固定到屏幕上的默认快捷键是 **Windows 键+ Control + T** 。(如果您愿意，可以在 PowerToys 应用程序的“总是在顶部”部分更改此设置。请记住，自定义键盘快捷键必须以 Shift、Control、Alt 或 Windows 键开头。)

若要将任何应用程序锁定在屏幕上，请确定它是活跃窗口，然后使用键盘快捷键。您将看到窗口周围有一个蓝色边框，并听到一声蜂鸣，这表明该应用程序现在已被固定在显示屏上。它会一直呆在那里，直到你把它拔掉。要从屏幕上取消固定应用程序，请再次使用相同的键盘快捷键(Windows + Control + T)。

你可以通过 PowerToys 应用程序获得一些额外的定制选项，值得一试。您可以更改锁定窗口周围边框的颜色或粗细，或者完全删除它。PowerToys 还允许您在激活该功能时禁用通知声音。

还有一个**排除应用**部分，让你决定哪些应用不应该被钉在屏幕上。不幸的是，你不能从列表中选择这些应用程序，你必须键入应用程序进程的名称。例如，如果你想排除 Firefox，你必须输入 firefox.exe 的**。**
