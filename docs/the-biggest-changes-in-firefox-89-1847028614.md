# 火狐 89 最大的变化

> 原文：<https://lifehacker.com/the-biggest-changes-in-firefox-89-1847028614>

最近的 [Firefox 89 更新](https://blog.mozilla.org/en/mozilla/news/modern-clean-new-firefox-clears-the-way-to-all-you-need-online/) 彻底改革了浏览器的桌面界面，并引入了几项受欢迎的生活质量调整和扩展的数据隐私保护。Mozilla 基于用户数据和反馈更新了界面，所以希望大多数用户会对这些变化感到满意。也就是说，即使你喜欢新的外观，事情已经被重新安排和删除——并增加了一些新的设置——所以可能需要一些时间来适应。

为了帮助你完成这个转变，我们将回顾一下你需要知道的 Firefox 89 的所有更新。要亲自尝试这些新功能，请通过前往**设置>常规>火狐更新>检查桌面上的更新**来安装最新的火狐更新，或者您可以从火狐官方下载页面下载最新的安装程序。

## **探索火狐 89 的新标签栏**

Firefox 89 最大的用户界面变化是标签如何出现在浏览器中，以及网站如何在后台显示 T2。

打开的标签现在“漂浮”在地址栏上，每个标签都有自己的矩形。虽然标签看起来有所不同，但它们的行为还是一样的:你可以点击并拖动打开的标签来重新排列它们的顺序，或者将它们拖出标签栏，在新的 Firefox 窗口中打开一个网站。Mozilla】增加了新的图标来指示标签何时播放音频，如果你在新的背景标签中打开一个链接，浏览器现在会自动暂停自动播放媒体。(是的，甚至讨厌的自动播放视频广告。)

您现在可以在**设置>常规>选项卡**和**设置>主页>新窗口和选项卡**下更改某些选项卡和窗口功能——例如默认情况下总是切换到新打开的选项卡。在新标签页上还有一个**“个性化”**按钮，可以让你自定义打开一个新标签页时看到的内容。

## **火狐 89 的其他 UI 变化**

Mozilla 还简化了 Firefox 的主工具栏。导航按钮已经被削减为后退、前进和页面刷新，并且“肉丸”三点设置按钮已经被删除，在应用程序的右上角留下三行“汉堡”按钮作为打开设置的主要方式。

类似地，Firefox 现在将浏览器通知整合到一个弹出“面板”中每次 Zoom 网站要求访问您的麦克风、网络摄像头和其他权限时，它们将出现在一个单独的框中，而不是得到单独的弹出窗口。

## **Firefox 89 在隐私浏览模式下的总 Cookie 保护**

虽然界面的变化是突出的更新，浏览器也收到了一个小的，但受欢迎的隐私增强:总 Cookie 保护设置 [现在在私人浏览模式](https://blog.mozilla.org/security/2021/06/01/total-cookie-protection-in-private-browsing/) 下工作，防止你在私人标签中打开的网站跟踪你，即使你同时在浏览其他页面。只要在**设置>隐私&安全>增强跟踪保护下启用了总 Cookie 保护，默认情况下非现场保护将在私有窗口中工作。**
