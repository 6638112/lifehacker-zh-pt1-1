# 如何玩 Zoom 的新内置游戏而不是工作

> 原文：<https://lifehacker.com/how-to-play-zooms-new-built-in-games-instead-of-working-1847457051>

在我们向远程工作生活过渡的早期，我们都跳上了 [集体视频游戏列车](https://lifehacker.com/our-13-favorite-free-games-to-play-over-video-chat-1844648370)——虽然 Jackbox 派对是与我们的同事(以及家人和朋友)进行“社交”的一个很差的替代品，但它们至少比几个小时的无休止的专注于工作的变焦会议要好一点。



令人惊讶的是，Zoom 花了一年多的时间在全球范围内进行虚拟工作，才将游戏直接集成到平台上，但最近推出的[Zoom Apps market place](https://marketplace.zoom.us/)有一些内置选项——例如，平视、扑克和琐事——你可以在会议或快乐时光安装并玩。

## 如何从 Zoom 应用程序安装游戏

使用 Zoom 应用程序的一个警告:如果你不是帐户所有者(例如，你的帐户是组织的一部分，而不是个人的)，你可能需要请求管理员的预先批准才能安装和使用应用程序。

1.  确保您已安装并打开了最新版本的 Zoom 桌面应用程序。如果您有个人帐户，您应该会在顶部任务栏中看到应用程序图标。
2.  切换以发现并选择您想玩的游戏。(请注意，这里还没有对应用进行分类或搜索。)点击**添加**按钮，这将把你重定向到一个浏览器页面以授权应用程序。
3.  安装游戏后，你将返回到桌面应用程序。所有已启用的应用程序将出现在我的应用程序选项卡下。
4.  要启动游戏，点击**打开>开始会议**或**复制邀请**分享链接。或者，你可以加入一个常规会议，切换到会议屏幕底部的应用程序选项卡开始游戏。

如果您在 Zoom 中看不到“应用程序”标签，您可能没有最新版本或正确的权限。

1.  进入[Zoom App market place](https://marketplace.zoom.us/)，从应用类别列表中选择游戏。
2.  选择您想要安装的游戏。如果您尚未登录 Zoom，请单击蓝色的**登录以安装**按钮。从那里，您将看到**访问现场安装**或**请求预先批准**的选项。
3.  如果您能够在没有管理员批准的情况下安装，您将被重定向到下一个屏幕以授权应用程序并启动桌面缩放，在那里您将继续上面列出的步骤。
