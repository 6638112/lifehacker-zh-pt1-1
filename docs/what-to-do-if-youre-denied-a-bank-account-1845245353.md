# 如果你被拒绝银行账户该怎么办

> 原文：<https://lifehacker.com/what-to-do-if-youre-denied-a-bank-account-1845245353>

银行账户比信用卡更容易获得批准，但如果你有不良信用记录，银行账户仍有可能被拒绝。但是不要让拒绝阻止你——还有其他方法来获得一个支票或储蓄账户。

### **知道你被拒绝的原因**

如果你被拒绝了，很可能是因为你的信用记录不好。ChexSystems 或早期预警服务消费者报告等服务寻找“负面行为”，包括:

*   空头支票
*   未付费用
*   信用查询
*   你订购了多少张支票
*   支票欺诈和身份盗窃/欺诈的实例
*   未付负余额
*   过度提款

一旦你知道是什么样的消极行为导致了你的拒绝，你就可以确定它们是合法的，然后开始纠正它们。

### **要求银行重新考虑**

虽然银行家利用《消费者报告》对申请人进行预筛选，但他们可以自由批准或拒绝他们认为合适的新账户。你也许能让银行重新考虑它的决定，允许你开一个账户。还要记住，联邦法律要求这些机构披露*你被拒的原因*，所以也要询问这些信息。

### **获得一份报告副本**

如果你不能让银行重新考虑，要求一份你的 [支票系统](https://www.chexsystems.com/web/chexsystems/consumerdebit/page/home/!ut/p/z1/04_Sj9CPykssy0xPLMnMz0vMAfIjo8ziDRxdHA1Ngg183AP83QwcXX39LIJDfYwM3M30wwkpiAJJ4wCOBkD9URAlMBP8PUKMgCa4-rgbG3kbugeaoCtAs8LAHKYAtyUFuREGmZ6OigAWLRKn/dz/d5/L2dBISEvZ0FBIS9nQSEh) 或 [预警服务](https://www.earlywarning.com/consumer-information) 报告。你只需要你现在的地址，社会安全号码，和你过去五年的住址就可以了。在您填写申请表后，报告将在五个工作日内邮寄给您。

### **检查你报告中的错误，并对它们进行争论**

接下来，检查你的报告是否有欺诈行为或差异，因为你可能只是错误甚至欺诈的受害者。如果您查看您的报告并发现任何不正确或过时的信息，您还可以向 [ChexSystems](https://www.chexsystems.com/web/chexsystems/consumerdebit/page/dispute/!ut/p/z1/04_Sj9CPykssy0xPLMnMz0vMAfIjo8ziDRxdHA1Ngg183AP83QwcXX39LIJDfYwMnM30w1EV-Du7mwMVuFg4O_o5GRkYGOhHAaXNAiw83Y0sDHwMPJxA-oPMnM28_YwMjA3Q9aNZYGBOnH48CqJIcj-mArB-HMARZH8UqhVYQgCvG0BeJGRJQW5oaGiEQaano6IiAF-w9-c!/dz/d5/L2dBISEvZ0FBIS9nQSEh) 或 [预警服务](https://consumerservices.earlywarning.com) 以及您的金融机构提交争议。

然而，如果你的报告是准确的，并且你确实欠了钱，你需要先还清这些债务，然后才能从报告中删除。否则，负条目可能会在您的报告中保留长达五年。

### **调查“第二次机会”账户**

一些银行为那些信用和银行历史不好的人提供第二次机会账户 。对于这些账户，银行在对你进行资格审查时不会看你的消费报告或负面活动，因此更容易获得批准。然而，结果是费用往往会更高。另一个选择是向信用合作社或小银行申请银行账户，因为其中一些没有订阅 ChexSystems。

### **暂时使用预付借记卡**

如果你还在纠结要不要开一个银行账户，那就申请一张 [预付借记卡](https://www.nerdwallet.com/article/banking/best-prepaid-debit-cards) ，里面一定要预装一定金额的现金。这种卡提供普通银行账户的一些功能，例如支付账单和转账。随着时间的推移，你可以还清债务，或者等到你的“负面行为”到期，这样它们就不再出现在你的消费者报告中。
