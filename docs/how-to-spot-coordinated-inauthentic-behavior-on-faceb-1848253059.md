# 据Snopes称，如何发现脸书的“协同不真实行为”

“协同不真实行为”是脸书创造的一个短语，指的是出于政治或经济目的，使用多个社交媒体账户或页面来误导或影响人们。作为欺骗的一部分，这些网页隐藏了身份，掩盖了他们背后的人的动机。

比如这里曝光的 [网络页面](https://www.snopes.com/news/2020/04/10/blue-lives-matter-fake-fb-pages/) 围绕推送 c orona 病毒造谣、“蓝色生命要紧”宣传等亲警亲军内容。脸书的网页和群组已经获得了200多万美国追随者，但主要是由科索沃人管理的。他们通过承诺免费食品杂货和免费移动房屋的赠品来收集追随者和喜欢者，但试图进入虚假的竞赛最终需要用户向塞浦路斯以外的网站提供信用卡信息。

这是个大问题。 [根据脸书大学前数据科学家杰夫·艾伦撰写的](https://arstechnica.com/tech-policy/2021/09/facebook-forced-troll-farm-content-on-over-40-of-all-americans-each-month/) 报告，2019年，来自巨魔农场的内容每周到达约 1亿美国用户。

除了发誓永远远离社交媒体之外，你可能无法保护自己免受这种事情的影响。但是你可以学会认清它是什么。下面是一些关于如何识别这种特殊品牌假新闻的小技巧，据T5【Snopes.com】T6报道。

## 查看脸书的“页面透明度”部分

脸书的每个页面都有一个名为“页面透明”的部分，可以让你看到页面管理者发布的内容来自哪个国家，以及页面经历的任何名称变化。页面透明度部分位于桌面脸书页面的右侧；当你在移动视图中滚动到页面帖子时，你会发现它。

## **检查验证**

已验证的页面在群组或简档名称旁边有蓝色徽章。如果你看到其中之一，这意味着该网页或个人资料可能代表了它所说的人或事。如果它不见了，这可能表明有什么不对劲。例如，一个未经证实的代表洛伦佐·拉马斯的网页可能实际上并不是洛伦佐·拉马斯。为了得到验证，页面管理员必须将文档发送到脸书以验证其真实性。

## “喜欢和分享”的帖子可能表明有问题

如果一个页面充满了鼓励读者喜欢和/或分享它们的图片和迷因，这可能表明协调的不真实行为。虽然没有明确指出不正当的行为，但是Snopes的工作人员指出，他们经常在不真实的页面上看到这种行为。

## 检查“页面创建”日期

从一个页面的创建时间可以看出很多东西。Snopes警告说，要寻找创建日期非常近的高度政治化的页面。如果一个页面是关于美国热点政治问题的，但是是两天前创建的，并且是由来自另一个国家的人运行的，要有所怀疑。要查找创建日期，请单击页面上的“页面透明度”链接或分组中的“关于”页面。

## 查看脸书群的管理员和版主

脸书群组(而不是页面)列出了他们的管理员、版主和成员，所以点击群组上的“成员”可以看到是谁在运行它，这样你就可以确定管理员是否看起来诚实。

## 这些真的有用吗？

很难说提供这些信息实际上有多有效。脸书表示，它承诺“对这些外国和国内的不真实团体采取积极的执法行动”，但其他人，如前脸书雇员艾伦 [坚持认为，脸书可以做得更多](https://arstechnica.com/tech-policy/2021/09/facebook-forced-troll-farm-content-on-over-40-of-all-americans-each-month/) 。

“即使只是添加一些简单的功能，如Graph Authority，并从纯粹的基于参与的功能中抽身，也可能会在诚信空间和...很可能也订婚了，”艾伦写道。
