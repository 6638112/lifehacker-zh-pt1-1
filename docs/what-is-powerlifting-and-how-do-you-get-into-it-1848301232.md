# 什么是力量举重，你是如何进入其中的？

> 原文：<https://lifehacker.com/what-is-powerlifting-and-how-do-you-get-into-it-1848301232>

竞技运动通常需要特殊的场地或设备，大多数力量运动都是如此。举重 (奥运会中的那种)需要有弹性的缓冲板和一个不会介意你把它们从头顶上扔下去的健身房。大力士训练使用各种不同寻常的设备，这些设备在普通的商业健身房里是找不到的。但是如果你想用基本的设备——仅仅是杠铃——为力量竞赛进行训练，有一项运动适合你，它叫做力量举重。



## 力量举重和普通举重有什么区别...举？

举重是一项竞技运动，包括杠铃深蹲、卧推和硬拉。你可以做这些练习，而不需要考虑你自己是一个举重运动员。它们因任何目的增强力量和肌肉而受欢迎。足球运动员做他们。Instagram 模特做的。数以百万计的人也是如此，他们只是出于健康和健身的原因想变得强壮。

如果你刚刚开始力量训练，你不必担心举重和力量举重的区别。因为深蹲、卧推和硬拉都是很好的力量练习，所以它们值得学习。

如果你一直在举重，并且对变得更有竞争力感到好奇，那该怎么办？好了，接下来是时候了解比赛中什么是允许的，什么是不允许的细节了。(比如你的深蹲要打到一定深度不然不算。)还有一些你可能想学习的技巧，这样你就可以提高你的竞争力，比如在卧推中更好地使用腿部驱动。至少在某些时候，你会想按照比赛标准来做你的体操托举。

最后，你可能最终会想改变你的训练计划，从策略上接近托举。作为一名竞争者，你将弥补你的弱点，利用你的优势，并适当安排你的艰难和轻松训练的时间，这样你就能在强大但充分休息的情况下参加比赛。

## 有哪些升降机？

如果你想成为一名举重运动员，你必须练习三种举重。以下是对每一项的快速概述，按照它们在比赛中的顺序，以及它们与你已经做的有什么不同:

### 深蹲

在深蹲中，会有一个杠铃放在一个架子上，略低于肩高。你走向它，把你的肩膀放在它下面，然后把它抬起来，(通常)向后走几步，到你可以开始下蹲的地方。

下蹲时，你同时弯曲膝盖和臀部，直到你的臀部折痕在膝盖骨以下(如果你想具体了解下蹲的深度，我们有一整篇关于下蹲深度的文章 )。然后你站起来。

我们有一个关于蹲坐 的所有细节的帖子供进一步阅读。如果你只是一个人蹲着，你需要注意的是:

*   力量举重运动员通常会深蹲*到法定深度，而不会更深，即使他们能够更深。*
*   许多(但肯定不是所有)力量举重运动员将使用“低杠”姿势，杠铃放在后三角肌上。“高杠”应该是把杠抬高几英寸，放在脖子根部的斜方肌上。
*   腰带可以帮助你在重压下保持核心力量，让你举起更多的重量，所以几乎所有的力量举重运动员都会使用腰带。

### 卧推

在这个动作中，你仰面躺在长凳上，杠铃放在你脸上方的架子上，保持一臂的距离。你要么把杠铃从架子上提起来，要么让别人帮你解开，然后举起开始，杠铃在你的手臂上锁定在你的上方。

你弯曲你的手肘和肩膀把杠铃放下来接触你的胸部，然后你把杠铃压回到起点。 [下面是我们关于卧推](https://lifehacker.com/everything-you-need-to-know-to-master-the-bench-press-s-1769031371) 的深度帖子。

关于力量举重- 式卧推你应该知道的一些事情:

*   你不能把杠铃从胸前弹开。法官会等着看你胸前的杠铃静止不动，然后大喊“按！”在你能再次提高门槛之前。当你一个人训练时，你会想把胸前的横杠暂停大约一秒钟。
*   可以，也应该， [拱背](https://lifehacker.com/is-it-cheating-to-arch-your-back-when-you-bench-press-1847827740) 。这使你的肩部和胸部肌肉处于一个更好的位置来移动更多的重量，如果你非常灵活，这可以减少杠铃移动的距离。有点拱起也只是肩部收紧的副作用，这同样有助于你保持稳定和移动更多的重量。
*   你会用你的腿。当你有一个好的拱形设置时，你的腿可以将力量从地板转移到你的躯干。在举重过程中，你不会弯曲或屈曲你的腿，但你的腿提供了一个重要的支撑作用，有助于稳定你身体的其余部分，再次让你安全地举起更多的重量。
*   对你的身体姿势有要求。这些有所不同，但一般来说，在整个托举过程中，你的屁股必须在长凳上，你的脚必须在地板上。一些组织要求你的头放在长凳上，你的脚后跟，不仅仅是脚趾，要放在地板上。

### 硬拉

硬拉是最简单的解释:地板上有一根横杆，你用手抓住它，然后站起来。 [下面是我们的硬拉指南](https://lifehacker.com/why-deadlifts-are-worth-the-hype-and-how-to-do-them-sa-1756526319) 。

如果你的腿位于手臂外侧，这被称为“相扑”姿势硬拉，如果你的腿在手臂内侧，这被认为是“常规” [两者都是合法竞争](https://lifehacker.com/are-sumo-deadlifts-cheating-1847388851) 。

关于力量举重中的硬举，你可能想知道一些事情:

*   比赛中不允许使用皮带。如果你抓不住杠，你就不能得到托举的荣誉。所以 [用一个混合或钩握](https://lifehacker.com/how-to-hold-onto-a-deadlift-bar-1839783159) ，现在就开始练习你的握力。
*   这里也经常使用腰带，原因和深蹲一样。

## 比赛是什么样的？

我在这里 写了关于 [我的第一次(也是迄今为止唯一一次)举重比赛。在你参加比赛之前，你会想要阅读你的联合会的规则。](https://lifehacker.com/the-best-advice-i-learned-from-my-first-powerlifting-me-1840067277)[【USAPL】](https://www.usapowerlifting.com)和 [USPA](https://www.uspa.net) 是其中两个大的，他们的网站除了规则手册和其他有用的信息外，还包括即将举行的比赛的列表。

比赛的基本理念是每个人都有三次机会去做最好的深蹲、最好的卧推和最好的硬拉。这些电梯(你最好的每一个)加在一起，得到你的总数。在一个重量级别内的最高总数赢得该重量级别。

重量级？是的，举重运动员是按体重分开的，所以矮小的人不会和高大的人直接竞争。比赛当天早上或前一天会有一次称重。

你会带着一些想法去参加比赛，这些想法是关于你第一次、第二次和第三次尝试的重量。你的第一次尝试应该是一个安全的数字，一些你 100%知道你可以做到的。你的第二次托举通常会达到或接近你最近在健身房做的最好的托举，如果你今天过得很好，你可能会在第三次托举中创下新的个人记录。如果你一次尝试失败了，你不能减肥，但是你可以选择在下一次尝试中保持同样的体重。如果你成功了，显然你会在下一次尝试中要求更多。

在典型的流程中，棒线以任何人要求的最轻的开盘重量开始。那个人试了试，然后装卸工为下一个人增加重量。当每个人都有机会时，杠铃会变得更重，然后一旦小组中的每个人都完成了第一次尝试，负重者就会减轻重量，这样就可以重新开始每个人的第二次尝试。

所有的托举完成后，每个人的总成绩被加起来，每个重量级别都有奖励。通常还有相对于他们体重而言总分最高的男女举重运动员的奖项。

## 我如何开始？

第一步是学习如何深蹲、仰卧起坐和硬拉。如果你现在所在的健身房没有杠铃、深蹲架或卧推台，你就不能练习举重。没有这种设备，你仍然可以变得强壮，所以如果你现在停留在行星健身，你用哑铃和机器锻炼的任何肌肉都将在以后对你有益。

作为一名初学者，任何有大量深蹲、长凳和硬拉的训练计划都会让你为比赛做好准备。如果你更有经验，你会受益于寻找专门的举重训练计划，把你的努力集中在最能让你有竞争力的事情上。因此，找一个举重教练会很有帮助。你可以在当地的举重馆找到一个，或者在网上搜索找到一个好的。

和喜欢举重的人交朋友，或者至少和举重爱好者在一起，也真的很有帮助。(专业提示:加入力量举重馆，而不仅仅是碰巧有杠铃的商业健身房。)这样做，你可以看到其他人是如何训练的，你会有一群人喜欢谈论一些事情，比如即将到来的比赛，你应该买哪件汗衫，以及你们都变得有多强壮。
