# 用疾病预防控制中心的旅行地图计划一个低风险的假期

> 原文：<https://lifehacker.com/plan-a-low-ish-risk-vacation-with-the-cdcs-travel-map-1847458812>

如果你正在考虑近期的国际旅行，你的第一站应该是疾病预防控制中心的 [旅游推荐目的地](https://www.cdc.gov/coronavirus/2019-ncov/travelers/map-and-travel-notices.html) 地图。在那里，您可以看到您的目的地国家在 COVID 案例方面的排名。



在我写这篇文章的时候，总的来说，事情并不像 T2 那样好。有四个类别:低、中、高和非常高的风险。“非常高”名单是最长的，大约有 75 个国家在上面，包括美国。“高”列表是下一个最长的。

谁风险低？目前，这份名单包括中国、澳大利亚、一些东欧国家(包括波兰和罗马尼亚)以及从马里到苏丹的一系列非洲国家。

这张地图给出了当前状况的快照，所以它不能保证在你预定的假期到来时，一个国家仍然是低风险的。但它可以给你一个想法，你可以在约会临近时继续查看。

以下是疾病预防控制中心针对不同风险等级推荐的预防措施:

*   **极高风险:**“避免前往这些目的地。如果你必须前往这些目的地，请确保在旅行前充分接种疫苗。”
*   **高风险:**“在前往这些目的地之前，确保你已经完全接种了疫苗。未接种疫苗的旅行者应该避免不必要的旅行。”
*   **中度风险:**“在前往这些目的地之前，确保你已经完全接种了疫苗。来自新冠肺炎的在 [的未接种疫苗的旅客患严重疾病的风险增加](https://www.cdc.gov/coronavirus/2019-ncov/need-extra-precautions/index.html) 的旅客应避免前往这些目的地的不必要旅行。”
*   **低风险:**“在前往这些目的地之前，确保你已经完全接种了疫苗。”

没有可用数据的国家被视为高风险国家，并发出“避免旅行”的警告。是的，疾病预防控制中心建议你在旅行前接种疫苗。在 [上阅读更多关于国际旅行的页面](https://www.cdc.gov/coronavirus/2019-ncov/travelers/international-travel-during-covid19.html) 。

如果你在国内旅行，顺便说一下，这里有一张按当前 COVID 状态排列的美国各县 的 [地图，以及给国内旅行者的](https://lifehacker.com/a-lot-of-us-should-start-wearing-masks-again-according-1847372587) [建议](https://www.cdc.gov/coronavirus/2019-ncov/travelers/travel-during-covid19.html) 。
