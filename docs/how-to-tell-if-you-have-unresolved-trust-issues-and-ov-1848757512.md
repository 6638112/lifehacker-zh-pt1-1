# 如何辨别你是否有未解决的信任问题(并克服它们)

> 原文：<https://lifehacker.com/how-to-tell-if-you-have-unresolved-trust-issues-and-ov-1848757512>

牢固的关系建立在信任的基础上，但信任也是脆弱的——如果你以前被欺骗过，你就会知道重新获得信任有多难。即使你认为你已经从过去背叛你的人中走出来，你可能仍然会以不太明显的方式与信任问题作斗争。



无法信任会阻止你完全投入到一段关系中；没有的信任，你就无法感受到与他人的脆弱、亲密和安全。但是你能做些什么呢？我采访了惠特尼·古德曼，她是一位有执照的精神治疗师，也是《毒性阳性反应》一书的作者。以下是你识别和克服信任问题的指南，这样你就可以让你们的关系充分发挥潜力。

## **为什么我们会产生信任问题**

信任问题是某人的信任被侵犯后的自然结果，特别是当某人认为他们应该能够信任时。 Goodman 举了一个常见的例子，一个在长大的人发现他们的父母或照顾者不做他们说要做的事情，这个人在成年后可能会产生信任他人的问题。另一个例子是人们在他们的关系中经历了不忠或欺骗。

## **信任问题的征兆**

难以信任他人以不同的方式在中体现。古德曼列出了以下迹象，表明某人不信任他人:

*   不断质疑他人
*   感觉人们在对你撒谎
*   不断要求他人的认可或确认
*   使用控制行为，比如需要查看某人的电话或随时知道他们在哪里

以下是信任问题的一些其他迹象，起初可能不太明显:

*   在你们的关系中保持距离，拒绝从一开始就建立信任
*   总是把别人往坏处想，在他们表现出任何不值得信任的逻辑迹象之前
*   破坏你自己，例如，表现出来或者破坏关系(在对方有机会背叛你的信任之前)

如果你在自己或你爱的人身上发现了这些行为，以下是你如何克服这些信任问题的方法。

## **如何克服你的信任问题**

信任是慢慢建立起来的，开始一场诚实、开放的对话是建立信任的唯一途径。古德曼建议，要开始一场关于克服你的信任问题的对话，一开始可以不经意地提出来，集中讨论为什么它对你如此重要。她提供了一个开始讨论的样本:*“由于我所经历的事情，我很难信任别人。我想让你明白我为什么做这些事情。我们能谈谈吗？”*

古德曼接着说:“当一个人和另一个人在一起，这个人按照他们说的去做，跟进，并允许建立信任，这个人就能痊愈。也可以学会相信自己，练习相信自己。”信任是安全关系的基础，治愈信任问题的第一步是明确沟通，将建立信任作为首要任务。

## **信任需要实践**

如果你很难相信别人，你可能需要努力相信自己。古德曼说，建立这种自信的唯一方法是实践:“验证自己，审视事实，给自己时间建立信任”都是开始信任自己的重要方法。

信任不是一蹴而就的，而是一砖一瓦建立起来的。你必须在小的时候展现自己，向自己承诺你能做到。只有这样你才有基础，才能重新开始信任别人。
