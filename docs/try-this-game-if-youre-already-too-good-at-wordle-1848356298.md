# 如果你已经非常擅长单词游戏，试试这个游戏

> 原文：<https://lifehacker.com/try-this-game-if-youre-already-too-good-at-wordle-1848356298>

你太擅长沃尔多了吗？你当然是。你知道 [所有的策略](https://lifehacker.com/how-to-master-or-cheat-at-wordle-the-internets-favor-1848312354) :先猜哪个单词，一开始就测试哪个元音。你可能厌倦了用你的绿色方块让推特上的每个人尴尬。如果这些天 Wordle 觉得太简单了，你可能想试试它扭曲的表亲、、T6】荒谬的、T8】。

由 qntm 开发，游戏遵循大部分与 Wordle 相同的规则。你猜一个五个字母的单词；如果你在一个字母上看到一个绿色方块，那么你在正确的位置上有正确的字母；如果你看到一个黄色方块，你有一个正确的字母在一个不正确的空间；如果你看到一个灰色的方块，这个字母根本不在单词里。

在 Wordle 中，你有六次机会把单词拼对。然而，荒谬给你无限的尝试。然而，在你嗤之以鼻之前，想想这两者之间的另一个主要区别:每次你猜错了，谜题的答案就会改变。

但这还不是全部；荒谬不仅仅是挑选一个单词，看看你是否匹配，而是挑选尽可能多的与你的单词相似的单词，试图尽可能延长你的荒谬游戏。如果你最初猜测是 SPEAK，Absurdle 会搜索数据库中不包含这些字母的每个单词，这个单词库就是它的答案。因此，第一次尝试就猜出这个单词是不可能的。第二个也是猜不出来的。根据 qntm 的说法，最短的荒谬游戏是四回合。

你真正想做的是把游戏逼入绝境；一旦不可能有任何独特的单词不包含你所猜测的字母，你将开始看到黄色和绿色方块出现在你的猜测中。Absurdle 仍然会切换可能的单词，但随着你继续猜测，这个池会变得越来越小。最终，根据游戏进行的方式，将只剩下一个可能的单词，并且，本质上，它将变回常规单词。

关于荒谬要记住的一个关键点是，它完全是决定性的。不像 Wordle，它有一套每天都在变化的答案，在 Absurdle 中使用相同的猜测总是会导致相同的结果。游戏就是这样设计的，因为从一开始就没有“答案”。

如果这些规则听起来有点混乱，不要担心；qntm 有一个扩展的解释器 来解释整个事情是如何工作的。然而，理解荒谬的最好方法是去演奏它。
