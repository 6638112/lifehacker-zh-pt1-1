# 举重和举重的区别

> 原文:[https://life hacker . com/the-difference-of-weighting-and-weight-lifting-1846723548](https://lifehacker.com/the-difference-between-weightlifting-and-weight-lifting-1846723548)

这将是一篇非常学究气的文章，也是我非常抱歉不得不写的一篇。我喜欢举重(两个词)已经很久了，但是在过去的一两年里，我喜欢举重(一个词)。事实证明这是两码事。

举重，一个词，是奥运会上的竞赛项目，穿着看起来像老式泳衣的人们拿起装有幼儿园颜色重物的杠铃。在其中一个项目抓举中，杠铃以一个快速的动作从地面举过头顶。在另一个项目中，挺举，杠铃举到肩膀处，举重运动员停顿一下，让T2屏住呼吸，也许在T4把杠铃举得很高之前做个鬼脸。(第二种方式可以举起更多的重量，这就是为什么它们是分开的事件。每个举重运动员的最佳抓举和最佳挺举成绩加在一起，看谁赢。)

如果你从未听说过这个,或者当你说或听到“举重”这个词时从未想到过它，请在这里耐心听我说。

有许多不同的力量运动。一个是力量举重，人们在深蹲、卧推和硬拉中竞争。有大力士，人们(不仅仅是男人)举起各种各样的工具，如石头、小桶和原木形状的杠铃，根据发起人的奇思妙想来选择。还有健美，一年到头都在健身房举重，然后参赛者尽可能多的消耗身体脂肪，然后登上舞台，在类似选美的活动中展示肌肉。

当然，你也可以举重。这不是举重；这是“举重”或“举重”或“力量训练”如果必须的话，你可以称之为“举重”

我讨厌自己被迫变得如此迂腐。举重是一个可怕的、糟糕的、不好的、非常糟糕的名字，是人们举重的众多运动之一。顺便说一句，力量举重的名字几乎一样糟糕；奥运升降机展示力量，“力量升降机”展示力量。所以像我这样的人只能抗议说我们是举重运动员，而不是力量举重运动员或健美运动员，普通人在健身房里举哑铃也不知道为什么我们如此在意“重量”和“举”之间是否有间隔。

最终的问题是，没有人为他们在奥运会中的运动想出一个更好的名字。有些人会称之为“奥运举重”，当你告诉你的朋友你做了这件事，但又说你不会为此去参加奥运会时，这会引起混乱。

交叉健身者已经找到了一个解决办法，他们随意地提到了“oly lifting”，我在理论上支持这一说法，但是举重运动员还没有接受这个术语。我们参加举重比赛，一边模仿T2抓举，一边说“你知道，举重”来阐明我们的意思。对不起这是我们目前最好的。
