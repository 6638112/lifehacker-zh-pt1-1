# 如何让你的标签对读者更友好

> 原文:[https://life hacker . com/how-to-make-your-hash tags-more-reader-friendly-1846397479](https://lifehacker.com/how-to-make-your-hashtags-more-reader-friendly-1846397479)

用标签语言交流是社交媒体一个恼人的普遍现象。但如果你在Instagram帖子、推文或脸书状态更新上使用标签，你应该确保它们对于有视觉和认知障碍的人来说是可读的。

### 将每个单词的第一个字母大写

标签的设计并不容易阅读。它们基本上是一个在互联网上编目内容、营销或制造低级笑话的工具。但是当你创建一个巨大的标签，把没有空格或标点符号的单词挤在一起，你就在创建一个无法解读的东西。

如果你真的觉得有必要创建一个长标签，把每个单词的第一个字母大写。像#comeforlivemusicstayforbeer这样的标签更容易阅读。正如互联网无障碍局 指出的那样，需要 [屏幕阅读器](https://www.afb.org/blindness-and-low-vision/using-technology/assistive-technology-products/screen-readers)——帮助视障用户阅读屏幕文本的软件程序——的人以及那些有阅读障碍的人将从这种简单的解决方案中受益。

如果你的标签像一些长篇大论的组合词一样拼凑在一起，它可能会混淆屏幕阅读程序，使它像一个单词一样读出你的标签——换句话说，就是胡言乱语。对于有阅读障碍的人也是如此。将每个单词都大写可以让你的标签更容易被看到你的帖子的人阅读。

### 如果使用多个标签，请使用空格

我不敢相信我不得不写这个，但是即使在2021年，仍然有一些straphangers倾向于将多个没有空格的标签混在一起。例如，克制所有可能的冲动，不要发布类似这样的灾难:

> # insta fun # insta good # regram # Sunday funday #早午餐#besties。

相反，如果你一定要像上面的例子一样用空洞的标签塞满社交媒体，就像这样写出来:

> # InstaFun # InstaGood # ReGram #周日周日#早午餐#闺蜜

如果你没有使用空格，你的标签将会变成一个巨大的标签小说，实际上不会导向任何地方。点击它将引导用户进入404页面。你真的不想发生这种事，是吗？

### 不要过度

在Instagram的初期(2012年或2013年，有人吗？)让用户对他们的标签感到有点兴奋。在这一点上，它们是一种新颖有趣的方式来展示对数字术语的理解。然而，现在它们基本上相当于你社交媒体体验中的墙纸——它们并不引人注目。

所以不要在你的文章中充斥着一个又一个标签。在社交媒体上，最好是有品味的，有影响力的人和品牌使用社交媒体的方式没有其他人那么令人讨厌，他们一贯表现出对标签的克制。如果你使用标签来推广一个品牌或其他计划，也许在你的文章中只使用两三个标签。如果你想让你的帖子在Instagram上可以被各种标签访问，你可以随时将它们添加到帖子的评论中，这样主文本看起来就不会那么杂乱。
