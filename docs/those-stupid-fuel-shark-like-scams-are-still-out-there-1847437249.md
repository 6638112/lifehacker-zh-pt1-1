> 原文：<https://jalopnik.com/those-stupid-fuel-shark-like-scams-are-still-out-there-1847437249>

几年前，我很开心地揭穿了那些把它放在你的点烟器里以节省汽油的把戏之一，[当他们要求道歉的时候](https://jalopnik.com/fuelshark-wants-us-to-apologize-because-they-sell-bulls-1527588505) 又做了一次。这些东西绝对是骗局，我还以为它们现在已经消失了。我错了。 [一个读者给我发了一个链接，同样的东西，现在叫 fuel save](https://lulaleaves.com/products/fuelsave%E2%84%A2?fbclid=IwAR277KYUXmtD4FGERop6DshTPUyiPQq2SXQ4eyHnuNwK-WbsARl0E30H2uY_aem_AYvVKzPNgMnQ6YYjjOvHsVFawo-2CAJZYWIBVLSd3PeiwBbv6g58v-78ObYNKcheez6UgB0BM_bf-Onv-tU237xa6_snhgb8IlKYQFx1rPJeZWBz6fhaVjVknMfAJjAQemc)，也是一样的扯淡。也许更多，因为这个人声称他们用这种“技术”驾驶一辆旧甲壳虫用一箱汽油行驶了 2096 英里。

在那个截图里，那不是 1969 年的甲壳虫。好像是 62 年的。我也很确定从匹兹堡到大西洋城的路线看起来不是这样的，虽然我承认，自从我开车以来已经有一段时间了。也许现在是一片贫瘠的荒地。但即使是这样，这两个城市之间的距离也只有大约 365 英里。

不只是胡扯，是邋遢，懒惰的胡扯。

一位名叫彼得的读者发给我的网站是如此荒谬可笑的糟糕，很难想象有人会被它欺骗。但是，可悲的是，人们是。人们相信各种各样愚蠢的事情，这个愚蠢的，欺诈性的网站，有许多明显的拼写错误，完全缺乏任何验证，很可能说服一些可怜的，容易上当受骗的奶奶们花血汗钱买一个从你仪表板的洞里伸出来的 LED。

这个狗屁解释基本上和老燃料沙克说的一模一样:

再说一遍，绝对是扯淡。

正如你所看到的，FuelSave 与 FuelShark 是完全一样的东西——它可能是由同一个无名的中国工厂制造的，正如你从描述中看到的，它有一个电容器。我打开了一个 FuelShark，想看看里面有多大的电容器，虽然 FuelSave 可能没有完全相同大小的电容器，但没有足够的空间来容纳任何实质上不同的东西。

以下是 FuelShark 中的内容，根据这些图片，应该与 FuelSave 中的内容几乎相同:

当我问一位电气工程师，加州理工学院的 Joseph Shepherd 教授，航空学和机械工程学教授，当连接到汽车的 12V 电气系统时，这样的电容器会做什么，这是我被告知的:

> “电容器是无源的。瓶盖开始放电，你打开发动机，瓶盖充电 12V，并保持充电状态。它需要电能来充电，它只是停留在 12v——不会回到系统或任何东西。你切断汽车电源，它就会放电。就是这样。”

简而言之，用一个专业术语来说，确实如此。没什么。它会点亮一个 LED。就是这样。

再说一次，这些都是我们以前讨论过的，但既然似乎仍有地方试图出售这种废话，随着油价再次攀升，人们将会不顾一切地省钱，并可能被这种掠夺性的废话所欺骗，我不会接受这种情况。

这篇文章不是写给我们的普通读者的，他们在一英里之外就能发现这种东西，而是写给你的普通读者，写给你对汽车不太感兴趣，对汽车更无知的朋友或亲戚，他们可能会被诱惑:不要买这个。

事实上，不要买任何插在 12V 插座上并声称能帮你省油的东西，因为那根本不是这些东西的工作原理。

看看那些宣称——更大的马力！更清洁的排放！每月节省 40%的油耗！还有那些评论！中间人的车现在"*散发着清澈干净的烟雾*！"

谎言。所有的一切。胡说，胡说，胡说。
