# 没有一种食物对你是“坏”的

> 原文：<https://lifehacker.com/no-one-food-item-is-bad-for-you-1846388498>

作为一个社会，我们被某一种食物对我们是“好”还是“坏”的问题所困扰。但是除了有毒的蘑菇(T2，我不会把它归类为“食物”)之外，没有食物对你有害。



最近一次，我看到一个标题承诺对一种特定的食物做出裁决，它是关于奶酪的，但你知道类型。咖啡对你有没有坏处；乳制品对你是好是坏；鸡蛋，黄油，大豆，果汁，什么都行。B 当你问一种特定的食物是否对你有害时，你已经问错了问题。

食物本身不可能是健康的或不健康的；影响你健康的是你饮食的全貌。健康饮食的基本要素 很容易查找，而且很可能你已经知道了。吃营养丰富的食物，尽可能少吃加工食品，摄入合理数量的卡路里，限制糖和饱和脂肪(理想情况下每种少于 10%的卡路里)。

你到底想知道什么？你喜欢奶酪，并希望毫无愧疚地享用它吗？你可以直接吃奶酪。你担心你吃太多奶酪吗？*嗯*，*加起来当大卡*。

我想知道我们是否喜欢听到食物是“好”还是“坏”，这样我们就可以对购买或食用它们产生即时的情绪反应。你可能会选择看一部恐怖电影而不是喜剧，仅仅是为了激情；类似地，你可能喜欢吃巧克力，同时想到“这对我有好处，所以可以享受它，”或者从“这对我很糟糕，我现在很糟糕”中获得某种兴奋也许一边吃着一块巧克力，一边想着“嗯，只是另一种食物”并没有那么有趣。

### 科学是怎么说的

每次关于某一种食物的研究出来，总是范围有限，方法间接。有时，研究人员将食物，或者更常见的是食物中的一种分离的化学成分，喂给动物吃(或者有时甚至喂给人吃)，并测量它们生物学的一些特定结果。其他时候，大群人被要求填写 [食物频率问卷](https://www.ucdenver.edu/docs/librariesprovider28/ctrc/nutrition/food_frequency_questionnaires.pdf?sfvrsn=eb8865b9_2) ，并从这些人的健康结果中得出结论，如他们的体重或寿命或心脏病发病率。

但是在这两种情况下，我们实际上并没有测试食物的某些特定的东西。在问卷调查的情况下，调查人员问了一个类似这样的问题:吃很多奶酪的人有什么共同的健康结果？

那个问题隐藏了很多变数。例如，吃很多披萨的人，要么是因为太忙没时间做饭，要么是因为太穷买不起更好的外卖，在吃奶酪的人群中占主导地位吗？这些研究不像药物试验，在药物试验中，你可以将人们随机分配到奶酪组或无奶酪组。我们都吃不同的饮食，一项研究能做的最好的事情就是对吃不同饮食的不同人进行归纳。

当我们看结果的时候，他们经常因研究而异。一项研究可能发现，吃大量某种食物的人比不吃的人稍微长寿一些；另一个人可能会发现他们更容易超重。说第一项研究表明这种食物“对我们有益”，而另一种“有害”，真的公平吗？我不这么认为。“好”和“坏”是对食物对我们健康影响的总结和基本判断。它不可能同时是“好”和“坏”，即使这两项研究都做得很好，而且他们的结论或多或少是准确的。

最后，我们唯一能真正判断的是我们总体上是否吃得好，有很多方法可以做到这一点。没有一种食物有任何神奇的特性可以取代你饮食中的其他食物。所以，让我们停止判断食物，好像它们本身就可以是“好”或“坏”的。
