# 跑步时穿合适的袜子

> 原文:[https://life hacker . com/wear-the-right-socks-when-you-run-1845927062](https://lifehacker.com/wear-the-right-socks-when-you-run-1845927062)

你大概在 [选择跑鞋](https://vitals.lifehacker.com/how-i-found-the-perfect-running-shoes-1784386202) 上花了不少心思，但是你的袜子呢？它们和更显眼的鞋底(ha)伙伴走过同样多的里程和脚步，它们可以让你在跑步中感到舒适，也可以让你感到不适。

有些人从来没有想过他们的跑步袜，而有些人可能想太多了。这里有一些注意事项，可以帮助你找到适合自己的袜子。

### 如果你现在穿的袜子有效果，那就坚持穿下去

每次跑步时和跑步后，你的脚感觉好吗？如果是这样，不要改变什么，即使跑步商店的那个家伙坚持说你错过了最前沿的袜子技术。

只是要意识到你的需求可能会改变。如果你买的是六只装的廉价棉袜，而且天气好的时候穿着你最喜欢的鞋子短途旅行也没问题，那就太好了——就目前而言。但是也许有一天你会开始马拉松训练，在长跑中磨出水泡。当这种情况发生时，重新评估。

### 跑步袜通常更合脚

普通的袜子就像一个袋子，你可以把脚伸进去。相比之下，跑步袜的制作非常适合你的脚。有时它们成对出现，一只适合你的左脚，另一只适合你的右脚。

它们通常比普通的袜子更薄，可以包住你的脚，所以当你跑步时，它们不太可能挤成一团或错位。

### 跑步袜减少皮肤发炎

跑步袜的紧密贴合性与较薄的材料和平缝相结合，有助于避免水泡。这种面料通常比标准的棉质袜子更滑，这意味着袜子和鞋子之间的摩擦比袜子和脚之间的摩擦更容易发生。这种织物通常有助于排汗，进一步减少起泡的机会。

### 冬季跑步袜既保暖又显瘦

当天气变冷时，羊毛或绝缘袜子可以帮助你的脚保暖。你在雪地靴里穿的那种厚厚的保暖袜子通常不会适合你的跑鞋，但一只为冬天制作的跑步袜子可以让你保暖，同时又足够薄，适合你。

这很重要，因为厚袜子会加剧我们讨论过的所有与袜子相关的问题。冬天穿的跑步袜可以薄也可以厚，但是它们总是很贴身，而且比同等标准的袜子更不容易起泡。

一旦你找到你的完美的跑步袜，买几双。如果你喜欢羊毛袜，它的好处在于可以穿几次 [而不会让](https://vitals.lifehacker.com/wool-socks-and-workout-clothes-aren-t-antibacterial-the-1788963270) 发臭。
