# 你们把这叫什么？

> 原文：<https://jezebel.com/what-do-yall-call-this-1847033182>

虽然看起来 Jezebel 的员工只是一群聚集在地下室的纽约老妪，用我们画的爪子不停地吸烟和点击键盘，但事实是我们来自不同的地方。正因为如此，这些地域和文化差异会导致沟通失误，比如今天发生在一场关于冰淇淋车 的严肃的 [新闻讨论中的失误。像往常一样，当我的同事，才华横溢的黑兹尔·席尔斯使用了一个我有生以来从未听过的短语时，这场谈话就分支成了一个更大更深入的关于美味的夏季点心的讨论](https://jezebel.com/lets-hit-the-open-road-for-ice-cream-cones-1843542486) [。](https://jezebel.com/lets-fight-about-which-ice-cream-treat-is-the-best-1837101692)

黑兹尔说，夏季最好的食物之一是“水冰”现在，我认为自己是一个对其他文化有见识和敏感的世俗女人，但是水冰到底是什么？水和冰不是一回事吗？

我能想到的唯一可能输给水冰的是 piragua，这是纽约市夏季的主要刨冰，上面浇有调味糖浆——大概覆盖了水和冰的底部。但是在我们开始讨论水冰的质地之后，在我走了一条懒散的路线，只是谷歌了一下水冰之后，我有了一个惊人的发现。

水冰就是意大利冰，也就是 coquito ( [冰不是饮料](https://www.youtube.com/watch?v=qh8nIa-uDdU) )。他们是同一个人！但是有点不同！但是一样吗？

不幸的是，我们从未达成共识，因为另一个地区性难题在我和朱丽安·埃斯科韦多·谢泼德(Julianne Escobedo Shepherd)之间爆发了，我每天都在这个网站上与她争夺拉丁霸权，当她问及另一种基于冰的美食 Mamitas 时。

“Mamita 就像一个柔软的人吗？”我问朱丽安，我们两种截然不同的传统又一次发生了冲突。“他们可能是，”她回应道，一如既往地寻找共同点的外交官。

对于那些没有在酒店附近或波多黎各家庭中长大的人来说， [Mamitas](https://mamitasices.com/) 是调味冰棒，有塔玛琳多、番石榴或巴塔塔等口味。另一方面， [limber](https://www.youtube.com/watch?v=GH69Z56dltA) 是一种更大、更甜的冰淇淋，我从来没有在商店里找到过，但我记得在波多黎各的一个堂兄的冰箱里吃过。

虽然我和朱丽安不能确定玛米塔斯和林波儿是远房表亲还是疏远的姐妹，但有一点是肯定的。我们都打算在今年夏天的某个时候尝试生产，如果结果不太好，那么我们将为拉米却肯纳帕莱塔斯 T2 T3 下一个大订单。每个人都叫他们帕莱塔斯，对吗？
