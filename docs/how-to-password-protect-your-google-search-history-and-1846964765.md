# 如何用密码保护你的谷歌搜索历史，隐藏你的秘密

> 原文：<https://lifehacker.com/how-to-password-protect-your-google-search-history-and-1846964765>

如果你不想让你的伴侣、配偶或室友窥探你的谷歌搜索或 YouTube 观看历史，但你又不想求助于匿名模式，好消息是:你现在可以用 [将两者隐藏在密码验证](https://www.theverge.com/2021/5/24/22452122/google-my-activity-page-password-privacy-verification-web-and-app-history) 之后。



你在谷歌的所有活动，包括网页和地图搜索、YouTube 历史、和谷歌助手查询，都会被编译，并且默认情况下，在 activity.google.com 可见。这意味着任何能够访问您登录的设备的人都可以看到您搜索、观看、和询问的内容。

一旦你启用了验证，你(或任何其他人)将不得不输入你的谷歌账户密码来显示你的活动历史。

要打开此功能，请前往。如果你还没有登录，c 点击**“管理我的活动验证”**链接。

在弹出窗口中，打开**“需要额外验证”**按钮，按**保存**。

您将被要求在谷歌主登录屏幕上输入您的密码进行确认。一旦启用，每次你想显示你的活动历史时，你必须点击**验证按钮**并输入你的密码。

实际上，这种验证并不是超级安全的——它只是要求你输入你的谷歌账户密码。如果你想躲避的人也知道你的密码(可能因为你们共用一个账户)，他们就能轻易进入。

如果你*真的*不想让任何人看到你的历史，你可以手动或者 T3】自动删除你的谷歌活动 。谷歌将自动删除超过 3 个月、18 个月、或 36 个月的任何内容(从您的“我的谷歌活动”页面，转到 Web &应用活动>自动删除，然后选择您的偏好)。您也可以手动从您的 Web &活动页面中删除项目或关闭保存活动功能。同样的过程也适用于你的 YouTube 和位置历史。
