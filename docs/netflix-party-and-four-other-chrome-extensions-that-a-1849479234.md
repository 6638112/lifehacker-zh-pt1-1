# “网飞党”和其他四个实际上是恶意软件的Chrome扩展

> 原文:[https://life hacker . com/网飞-party-and-four-other-chrome-extensions-that-a-1849479234](https://lifehacker.com/netflix-party-and-four-other-chrome-extensions-that-a-1849479234)

[Chrome扩展](https://lifehacker.com/13-of-the-best-chrome-extensions-of-2021-according-to-1848244043) 被认为可以让你的浏览器更好——但并不是所有的扩展都可以信任。尽管谷歌很努力，但Chrome网上商店并不是一个无辜公用事业的市场。虽然大多数都经过了适当的审查，但有些人还是漏网了，就像他们在Play Store 甚至苹果应用商店上做的那样。现在，我们知道了另一组伪装成合法扩展的恶意软件，包括两个与该公司无关的“网飞”选项。

[迈克菲的安全分析师](https://www.bleepingcomputer.com/news/security/chrome-extensions-with-14-million-installs-steal-browsing-data/) 发现了这最新一批五个恶意的Chrome扩展。他们总共有超过140万次的下载。这些扩展互不相同，但在设计上有着相同的最终目标。一旦你在浏览器上安装了这些扩展，它就会等待你访问一个电子商务网站，比如百思买或亚马逊。然后，扩展将你的浏览数据发送到他们自己的域，如果你的网站与他们的一个分支机构相匹配，他们要么将他们的分支机构链接注入页面的URL，要么用他们自己的链接劫持你的cookies。无论哪种方式，目标都是从你购买的商品中赚取佣金。

这样看来，这些扩展和典型的骗局是不一样的。他们不是在追踪你的个人信息，也不是试图侵入你的账户。相反，他们把你和其他140万人当作中间人，以获取他们自己的佣金。然而，这并不意味着它们比其他恶意软件更好，无论如何都应该被阻止。

更糟糕的是，每一个恶意的扩展都像宣传的那样惊人地工作。FlipShope实际上是一个价格追踪器，全页截图确实可以捕捉到你的浏览器截图。通常，包含恶意软件的应用程序和实用程序只不过是欺骗您安装它们的幌子。然而，这些扩展背后的开发者不希望你怀疑任何事情，因为你在Chrome中安装这些工具的时间越长，他们赚的钱就越多。

以下是McAfee分析师发现的恶意应用程序，按下载次数排序:

*   **网飞聚会** : 80万次下载
*   **网飞派对2** : 30万次下载
*   **整页截图–截图** : 20万次下载
*   **flips hope——价格跟踪扩展** : 80，000次下载
*   **AutoBuy Flash销售** : 20，000次下载

截至本文发表时， **AutoBuy Flash Sales** 和**flips hope——价格跟踪扩展**仍在Chrome网上商店出售，但谷歌已经删除了其他三个。然而，即使这五个都从网络商店中删除，谷歌也不能从你的浏览器中删除它们。如果你在Chrome中安装了这些扩展，现在就删除它们。

## 如何保护自己免受恶意Chrome扩展的攻击

无论如何，这都不是一个新问题。事实上，我们过去已经讨论过如何避免恶意扩展。简而言之，这个建议类似于保护自己免受App Store上的恶意应用程序的攻击:

*   确保你正在下载你正在寻找的正确的扩展。恶意扩展利用与另一个流行扩展相似的名称来迷惑用户。
*   密切关注扩展的web商店列表。一切拼写正确吗？图片与广告中的特征相符吗？
*   看看扩展的评论。审稿人看起来合法吗？他们正在审查正确的扩展吗？
*   在下载之前对扩展做一些外部研究。当你搜索它的时候，谷歌会拾取什么？

谷歌无法阻止每一个恶意的扩展，但是你可以运用良好的判断力来避免将它们安装到你的浏览器中。
