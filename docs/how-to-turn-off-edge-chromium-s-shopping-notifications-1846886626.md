# 如何关闭 Edge Chromium 的购物通知

> 原文：<https://lifehacker.com/how-to-turn-off-edge-chromium-s-shopping-notifications-1846886626>

你可能不会嫉妒 Edge Chromium 浏览器试图帮你省钱，但当一个又一个关于返现计划和优惠券节省的提示弹出时，这可能会很烦人。有时交易很棒；其他时候，它们更烦人，而不是有用。



我喜欢认为自己是一个非常聪明的购物者，我发现像这样的提示——以及类似的浏览器扩展，如 [蜂蜜](https://twocents.lifehacker.com/should-you-trust-online-shopping-apps-like-honey-1840069187)——通常不会增强我的购物体验。要么这些网站很少有疯狂的促销代码(亚马逊)，要么我在那里策略性地购物，因为我*知道*那里有值得利用的交易(谢谢， [Slickdeals](https://slickdeals.net/) )。

如果你像我一样，你可以在 Edge Chromium 中关闭这些提示。你可能需要，因为默认情况下，你将在第一次安装浏览器后得到它们。

## 如何关闭 Edge Chromium 的额外服务

首先，点击右上角的三点图标，调出浏览器的设置，然后点击**设置**。然后你会想要点击**隐私、搜索和服务**，并开始向下滚动，直到你看到**服务**部分。在其中，查找**“在 Microsoft Edge 购物节省时间和金钱”**选项。禁用它。

当你在那里时，如果你经常使用浏览器的收藏功能，但不太需要 Pinterest，你可能需要确保“Pinterest”的连接也被禁用。此外，“地址栏和搜索”选项是当你在地址栏中输入内容时，你可以更改浏览器使用的默认搜索引擎。如果你不是粉丝，你还可以完全禁用浏览器的建议。

然而，如果你打算保留购物建议，要知道你不必等待浏览器提示你网站上的交易。如果你不耐烦了，或者这个过程因为某种原因搞砸了，只需在浏览器地址栏的最右边寻找小小的价格标签图标。点击它，无论你在哪里购物，你都可以看到微软发现的任何交易、促销和其他产品。
