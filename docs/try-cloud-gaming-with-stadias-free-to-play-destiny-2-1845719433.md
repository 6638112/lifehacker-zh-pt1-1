# 尝试Stadia的免费游戏“命运2”的云游戏

> 原文:[https://life hacker . com/try-cloud-gaming-with-stadias-free-to-play-destiny-2-1845719433](https://lifehacker.com/try-cloud-gaming-with-stadias-free-to-play-destiny-2-1845719433)

在推出整整一年后，谷歌Stadia终于推出了第一款公开的免费游戏，这是一款不错的游戏: *《命运2》*。

这与自2019年6月以来在其他平台上提供的*命运2* 的免费播放版本相同——包括付费订户的Stadia——但现在你可以 [在Stadia上播放它而不用花一毛钱。](https://community.stadia.com/t5/Stadia-Community-Blog/This-Week-on-Stadia-Play-PAC-MAN-instantly-Immortals-Fenyx/ba-p/39455) 这可是件大事；在此之前，你必须要么付费进入 Stadia的订阅图书馆，要么通过这项服务购买全价游戏。早期采用者还必须购买一个支持视距的设备和一个控制器 [，这相当昂贵](https://lifehacker.com/do-you-really-need-the-stadia-founder-s-pack-on-launch-1839135086) 。幸运的是，Stadia支持已经扩展到 Chromecast Ultra、PC和许多 [Android手机和chrome book](https://support.google.com/stadia/answer/9578631?hl=en)*，并且它可以与几个第三方控制器一起工作。甚至还有一种针对移动用户的 [触控变通方法](https://lifehacker.com/how-to-enable-touch-controls-for-stadia-games-on-androi-1841911767) 。*

*免费游戏版本的 *Destin* y *2* 只包括原版*Destin 2*的基本内容，而没有DLC资料片。如果你想要那些，你必须买*命运2:收藏*和最近的*超越光*资料片。还有stadia版本不支持跨平台播放，免费用户被限制在1080p模式。每月10美元的会员可以在4K模式下玩游戏，尽管你的实际游戏分辨率将取决于你的网络质量。* 

*值得一提的是，微软的云服务和 [Nvidia的GeForce Now](https://lifehacker.com/how-to-sneak-fortnite-on-your-iphone-1845598969) 也可以让你在旅途中玩 *Destiny 2* ，但微软的应用程序需要15美元的每月游戏通行证会员资格，GeForce每月5美元(或者你可以免费玩，但有1小时的会话限制)。然而，在Stadia上玩游戏是免费的，没有时间限制。* 

*如果你对*命运2* 感到好奇，并且没有足够强大的游戏硬件来本地玩它，那么通过Stadia免费播放它是一个尝试游戏的好方法。对于任何对Stadia或云流媒体持观望态度的人来说，这也是一个在零财务承诺的情况下尝试这项服务的绝佳机会。只需前往 [Stadia的主页](https://stadia.google.com/home) 并创建一个新账户即可开始。*
