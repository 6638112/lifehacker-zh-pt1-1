# 如何阻止Windows 11自动更新

> 原文:[https://life hacker . com/how-to-stop-windows-11-from-updating-automatically-1848877844](https://lifehacker.com/how-to-stop-windows-11-from-updating-automatically-1848877844)

Windows 10在更新方面名声不佳——互联网上充斥着令人沮丧的轶事关于Windows在完全错误的时间重启以安装更新，比如当你要开始一场演示时，或者当你正在赶时间时。这是微软在Windows 11中改进的地方之一。

Watch

系统在安装更新时更加透明，在安装之前会给你通知和警告。如果你告诉它延迟更新，它真的会。即便如此，一旦延迟的时间框架结束，就会强迫你安装更新。

*正式*，没有办法阻止一个自动更新。但是我们有一些变通办法。

## 尝试延迟更新

通常，我们建议您保持计算机最新。但是，如果你今天或未来几天不能安装更新，最好推迟(这也是确保你不会被一个有问题的软件版本卡住的好方法)。

Windows 11允许你延迟更新长达五周。你可以通过**设置** > **Windows更新**来做到这一点。在这里，单击“暂停更新”部分旁边的下拉菜单，选择一到五个周。现在更新将根据您的时间表推迟；y 您可以随时返回此处，单击“**恢复更新**”按钮手动更新您的电脑。

## 设置计量连接以避免大量更新

这是一个在Windows 11中仍然有效的老把戏。Windows具有计量连接功能，可防止高数据使用率。当此功能启用时，它不会下载更新。

您可以通过**设置** > **网络&互联网** > **Wi-Fi** 进行设置。在这里，选择您的Wi-Fi网络，然后单击“**计量连接**”功能旁边的开关。

## 使用服务管理器禁用Windows更新

停止Windows更新的一种方法是停止执行更新的服务管理器。在“设置”中没有开关，但你可以进入“服务”应用禁用它。

点击**赢** + **R** 打开**运行**对话框，输入“ **services.msc** ”，按**回车**键。向下滚动并双击“ **Windows Update** ”选项。

在**启动类型**选项中，选择**禁用。**。点击**确定，**，重启电脑。

## 或者，使用Windows更新拦截器应用程序

每次您想要停止和恢复更新时，都要求助于服务管理器，这是一项繁重的工作。有一款名为 [的Windows更新拦截器](https://www.sordum.org/9470/windows-update-blocker-v1-7/) 的应用可以为你做同样的事情，只需一键工具。

安装app，打开，切换到“**禁用更新**功能，点击**立即应用。**
