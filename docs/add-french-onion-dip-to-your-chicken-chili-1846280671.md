# 在你的鸡肉辣椒中加入法式洋葱酱

> 原文：<https://lifehacker.com/add-french-onion-dip-to-your-chicken-chili-1846280671>

辣椒是一种完美的寒冷天气食物，然而距离上一期的长柄锅辣椒烹饪比赛已经过去了整整两个月，该比赛的特色是混乱--令人惊讶- 美味 [披萨辣椒](https://lifehacker.com/pizza-chili-is-an-agent-of-chaos-1845861870) 。当然，我有借口——我在搬家，圣诞节就要到了——但重要的是我们带着更多的辣椒回来了，这次是洋葱蘸酱。

Watch

如果我从比萨饼辣椒事件中学到了什么，那就是有些人在谈到辣椒时没有幽默感。一个人甚至说，比萨饼辣椒(或任何偏离“con carne”格式的辣椒)“盗用了德克萨斯文化”，这是我读过的最德克萨斯的东西之一。

那个家伙会讨厌这个，但是那个家伙不是这里的高级美食编辑，所以他没有权利选择辣椒——我有权利选择辣椒，而且，抛开一切不谈，我真的很喜欢一个好的鸡肉辣椒。 [这个](https://lifehacker.com/1845374934) 引起我的注意是因为它的“秘密成分”，也就是迪安的法式洋葱，这是我最喜欢在商店买的蘸酱之一。

如果你了解我，你就会知道我喜欢蘸酱，即使是那种装在室温玻璃罐和冷藏塑料桶里的蘸酱，而且我已经是一个在土豆泥 中使用商店购买的洋葱蘸酱 [的热情倡导者。正如我之前解释的，“乳化油产品有助于将亲水性和疏水性结合起来，创造出一碗天鹅绒般柔滑的马铃薯，具有令人惊讶的微妙洋葱风味和足够的鲜味，”我不明白为什么它不能在鸡肉辣椒中做类似的事情。我的效果甚至比我预期的还要好，但是在我们开始之前，让我们花点时间来看看这个食谱，它是由 threedayante 写的:](https://skillet.lifehacker.com/just-add-store-bought-french-onion-dip-to-mashed-potato-1839621708)
