# 现在在欧洲，你可以去哪里旅游？

> 原文:[https://life hacker . com/where-can-you-travel-as-a-tourist-in-Europe-right-now-1847047920](https://lifehacker.com/where-can-you-travel-as-a-tourist-in-europe-right-now-1847047920)

欧盟 [已经同意在今年夏天晚些时候为接种疫苗的旅行者](https://www.wsj.com/articles/eu-set-to-allow-vaccinated-tourists-11621426176) 重新开放边境，但是令人困惑的是，他们的指导对于成员国来说并不是强制性的。结果，一些国家已经开放了边境，而另一些国家仍然关闭。为了帮助你了解情况，我们来看看哪些国家现在对美国游客开放，以及他们可能会有什么要求。

## **美国游客在欧洲可以去哪里旅游？**

虽然大多数欧洲国家都遵循欧盟的指导方针，不对“非必要”旅行(即旅游)开放，但欧盟预计将在今年夏天晚些时候向完全接种疫苗的游客开放旅行(计划正在制定中，预计将于下周宣布)。然而，以下依赖游客的国家已经提前向游客开放:

*   **波斯尼亚和黑塞哥维纳**对美国游客 [开放](https://ba.usembassy.gov/covid-19-information/) ，尽管COVID的阴性PCR测试必须在抵达前48小时内进行。该国建议游客遵循社交距离指南，在无法进行社交距离时戴上口罩。
*   **克罗埃西亚**已经 [向美国公民](https://hr.usembassy.gov/covid-19-information-2/) 重新开放边境，只要他们能够证明他们要么已经完全接种疫苗，要么在抵达前48小时进行了阴性PCR检测，要么在过去180天内从新冠肺炎康复。两岁以下的儿童免于测试。
*   **丹麦**从技术上讲不对美国游客开放，但*有 [【甜心宣言](https://dk.usembassy.gov/u-s-citizen-services/security-and-travel-information/covid-19-information/#:~:text=Travel%20to%20Denmark%20Under%20Danish,in%20Denmark%20to%20enter%20Denmark.)允许“甜心”、未婚妻和其他亲密关系的人去丹麦旅游，尽管应该是从始发国直飞。*
*   随着许多旅游目的地在6月份开放(埃菲尔铁塔将于7月16日开放)，法国正从漫长的封锁中走出来。边境将于6月9日对美国游客开放，前提是他们在抵达后72小时内出示COVID的 [阴性PCR测试](https://thepointsguy.com/news/france-entry-requirements-june/) 证明(或者，在抵达前48小时出示阴性抗原测试证明)。

*   希腊在4月份开放了边境，T2的旅行者如果在抵达前至少14天接种了疫苗，就不需要隔离。没有接种疫苗的T4旅行者必须在抵达希腊的72小时内提供阴性新冠肺炎PCR检测。五岁及以下的儿童免票。所有的旅行者可能会接受随机但强制性的COVID测试，你必须在抵达前[填写一份](https://www.afar.com/magazine/greece-has-reopened-to-travelers-heres-everything-you-need-to-know) [乘客定位表，根据旅游网站Afar](https://travel.gov.gr/#/user/login) 。
*   冰岛是第一个欢迎接种疫苗的游客的国家，只要他们能证明接种过疫苗或者他们最近从COVID康复。还有强制性的PCR检测，你抵达后需要在你的住处隔离一天[。](https://www.icelandair.com/blog/iceland-is-open-faq/)
*   爱尔兰将从7月19日开始接受接种疫苗的美国人，他们需要疫苗接种证明以避免14天的隔离。旅客还需要在抵达前填写一份 [旅客定位表](https://cvd19plf-prod1.powerappsportals.com/en-us/) 。
*   意大利有一个强制性的10天隔离期，但是游客可以通过乘坐从美国到意大利的[【Covid测试航班】](https://www.reuters.com/article/us-health-coronavirus-airlines-italy/u-s-airlines-add-covid-tested-flights-to-italy-as-country-reopens-to-tourists-idUSKCN2CY2OC) 绕过这个要求。作为该计划的一部分，前往意大利的游客必须在登机前48小时内接受新冠肺炎测试，并在抵达时再次接受测试。
*   科索沃仍然处于半封锁状态，晚上10:30实行宵禁，但他们接受美国游客，条件是他们 [在进入该国前72小时](https://xk.usembassy.gov/coronavirus/) 提供PCR检测呈阴性的证明。
*   **塞尔维亚**欢迎能在抵达48小时内提供阴性PCR或快速抗原检测 [的旅客。没有宵禁，但社交距离仍然有效。](https://rs.usembassy.gov/serbia-covid-19-information/)
*   从今天起，西班牙开始接受来自欧盟以外的的游客接种疫苗。被接受的疫苗仅限于辉瑞、Moderna、阿斯利康和强生&公司。
*   **土耳其**是 [对起飞前72小时PCR检测呈阴性的美国旅客](https://tr.usembassy.gov/covid-19-information-2/) 开放。晚上10点也有宵禁，但游客不受此限制。
*   英国使用一种颜色编码系统，在该系统中，一些国家面临更大的限制，其中包括美国，它在“琥珀名单”上不幸的是，这意味着美国人必须填写一份 [乘客定位表](https://onemileatatime.com/uk-passenger-locator-form/) ，在抵达前72小时接受PCR检测，并承诺10天的隔离期——即使他们已经接种了疫苗。许多观察家 [预计限制将很快解除](https://thepointsguy.com/news/uk-3-june-traffic-light-changes/) ，尽管还不清楚确切的时间。
