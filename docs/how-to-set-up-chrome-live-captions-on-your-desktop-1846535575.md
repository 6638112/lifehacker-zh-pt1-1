# 如何在你的桌面上设置Chrome直播字幕

> 原文:[https://life hacker . com/how-to-set-up-chrome-live-captions-on-your-desktop-1846535575](https://lifehacker.com/how-to-set-up-chrome-live-captions-on-your-desktop-1846535575)

谷歌最近在桌面版Chrome上添加了实时字幕。启用实时字幕后，Chrome会自动转录浏览器中播放的任何音频，即使声音被静音。

该功能类似于YouTube的自动转录功能，但适用于通过Chrome播放的所有音频，包括视频、播客和语音通话。它也适用于Chrome渐进式网络应用程序(PWAs) [，如YouTube和YouTube Music](https://lifehacker.com/how-to-install-youtube-as-a-progressive-web-app-on-wind-1846137659) — ，该功能也可用于部分Android设备。

对于那些有听力障碍的人，或者任何想要跟随视频和其他语音媒体但不能打开他们的音频的用户来说，直播字幕是一个有用的辅助功能。也就是说，在这一点上，的语音识别还不完善。我注意到Chrome很难识别口音重或音质差的人——而且的直播字幕在发布时只提供英语音频。然而，尽管有这些限制，实时字幕是一个受欢迎的功能，随着谷歌继续更新其精确度并增加对其他语言的支持，它有望变得更好。

### 如何在Chrome桌面上启用实时字幕

Chrome的实时字幕设置默认是关闭的。打开字幕很容易，但是在你真正使用这个功能之前还有几个步骤要完成。

首先，确保Chrome是最新的。直播字幕仅在Chrome 89(或更高版本)上可用。要更新，打开Chrome，进入**帮助>关于谷歌Chrome** 或者**设置>关于Chrome** 。如果可用，单击安装更新。

安装Chrome 89后，你可以在浏览器的设置中打开实时字幕:

1.  在Chrome中，进入**设置>高级>辅助功能。**
2.  启用**“实时字幕”** Chrome会下载必要的语音识别文件，只需要几秒钟。当该功能准备就绪时，您会在实时字幕下看到“语音文件已下载”。
3.  只要浏览器检测到音频，直播字幕就会自动开始转录。如果没有，请单击媒体图标打开Chrome的播放控件(这是位于Chrome的浏览器扩展工具栏和您的Google帐户档案图标之间右上角的按钮)。确保**“实时字幕”**已打开。

实时字幕出现在浏览器标签底部附近的窗口中，该窗口正在播放音频。您可以在**设置>高级>辅助功能>字幕首选项**下调整字幕窗口的大小和外观。

如果你决定在时刻不想打开直播字幕，点击**“X”**关闭字幕窗口并关闭当前媒体，或者关闭Chrome媒体控制面板中的**“直播字幕”**选项以完全禁用它们。
