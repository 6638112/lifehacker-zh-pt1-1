# 此修复使Windows 10上的多显示器设置更好

> 原文:[https://life hacker . com/this-fix-makes-multi-display-setups-better-on-windows-1-1846791725](https://lifehacker.com/this-fix-makes-multi-display-setups-better-on-windows-1-1846791725)

当您在玩最喜爱的在线游戏时，如果您正在进行流媒体播放、编辑或试图用背景中的一个节目来分散注意力，多显示器可以极大地提高您的工作效率。太糟糕了，Windows 10让它们成为了一个使用的累赘。

Watch

出于某种原因，当你在使用两个或更多显示器的情况下将Windows电脑置于睡眠状态时，当系统唤醒时，打开的窗口会跳转到主显示器，迫使你每次从睡眠模式启动电脑时都要重新组织布局。

令人欣慰的是，微软正在努力解决这个问题——微软称之为 [快速热插拔检测(快速HPD)](https://devblogs.microsoft.com/directx/avoid-unexpected-app-rearrangement/)——让你打开的应用程序和文件夹保持在你让电脑进入睡眠模式*之前的位置。*微软保证修复“不限于显示器制造商、硬件规格、连接器类型或电缆品牌”，并适用于所有配有多台显示器的Windows 10 PCs和连接到外部显示器的笔记本电脑，无论用户的系统设置如何。

快速HPD修复程序将于今年晚些时候作为2021年10月更新的一部分发布，但那些无法忍受每次PC唤醒时重新调整应用程序的人可以加入Windows Insider计划，立即获得更新。

请注意，Windows Insider计划并不适合所有人。虽然您很早就获得了新功能，但测试功能往往尚未完成，系统错误更常见，您可能需要更频繁地更新您的PC。如果你不介意这些警告，你可以 [点击](https://insider.windows.com/en-us/) 加入Windows Insider计划。

Windows Insider build 2187(目前在Dev频道上可以找到)应该会在您注册Insider计划后不久到达您的PC，但您可以在**设置>更新&安全> Windows更新**中检查可用的系统更新，然后单击**“检查更新”**

你也可以在**设置>系统>关于**中查看你的Windows 10版本。操作系统内部版本号列在**“Windows规格”下**

Insider Build 2187(或今年晚些时候的2021年10月更新)是修复快速HPD问题的唯一要求。安装后，在使用多台显示器时，您应该不会再遇到桌面重新洗牌的问题。
