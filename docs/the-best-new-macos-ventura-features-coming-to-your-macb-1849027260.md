# 您的 MacBook 将拥有最好的新 macOS Ventura 功能

> 原文：<https://lifehacker.com/the-best-new-macos-ventura-features-coming-to-your-macb-1849027260>

Watch

在 WWDC 2022 上，苹果终于让我们先睹为快了 macOS 的更新:macOS Ventura 将于 2022 年秋季上市，但它已经可以作为开发者测试版使用(如果你喜欢冒险的话)。

这不是一个革命性的更新，但它确实带来了一些很好的补充，绝对可以让你的生活更轻松。以下是今年晚些时候将会出现在你的 Mac 上的一些最好的功能。

## 舞台管理器将帮助您整理凌乱的窗口

如果你在任何时候都有几十个窗口打开，并且发现很难保持在它们之上，macOS Ventura 可能会有一个适合你的解决方案:它被称为 Stage Manager，它对多任务处理有独特的见解。

激活后，所有打开的窗口都会根据应用程序和群组排列在左侧。你当前的窗口是你唯一能看到的东西。但是你可以在群组中添加新的窗口，重新排列它们，或者从左边的舞台管理器中快速切换一组新的窗口。

## 你的 iPhone 可以作为你的 Mac 网络摄像头，不需要设置

这个功能是晚了两年，但它仍然会受到 Mac 和 iPhone 用户的赞赏: Y 你终于可以使用你 iPhone 的后置摄像头作为你 Mac 的网络摄像头，而无需任何设置。只需将你的 iPhone 与 Mac 对接(你甚至不需要解锁)，它就会作为一个摄像头出现在流行的视频会议应用中。

因为这是一个 iPhone 相机，你可以访问人像模式和工作室灯等功能，这些功能模糊了背景并增强了脸部周围的照明。苹果在这里也尝试了一些古怪的东西:我的新桌面视图功能使用 iPhone 上的超宽传感器提供了一个自上而下的视图(再次强调，根本不需要移动 iPhone )。

## 万能密码想要让密码成为过去

科技世界的三大巨头——苹果、和微软正致力于一个 [无密码的未来](https://fidoalliance.org/) 。你将拥有一个密钥，而不是密码，这个密钥是 一个连接并存储在设备上的安全密钥，使用触控 ID 或 Face ID 进行认证。苹果公司称他们的版本为密码。

如果网站或应用程序支持，你可以在所有设备上只使用 Face ID 或触控 ID 进行注册和登录。没有需要记住的密码，没有可以被黑的密码。别担心:如果你用的是 Windows 电脑，你可以通过 iPhone 用二维码登录。

## Safari 中的共享标签页

协作是 Ventura 的关键词，也是 Safari 的关键词。您将能够与使用“信息”的人共享整个标签组。一旦有人加入标签组，他们就可以添加更多的标签，您可以实时跟踪他们的活动。

## 邮件应用获得了一些急需的可用性更新

在被抛弃多年后，Mac 上的邮件应用程序终于得到了它所需要的关注。Mac 应用中有许多新功能，首先是一个新的撤销发送功能，可以让你在电子邮件发出之前快速停止发送。您还可以安排稍后发送电子邮件。

苹果还改进了搜索功能。最近的搜索被保存，自动完成的结果更准确。

## 最后，为您的家人提供一个共享的照片库

这不是严格意义上的 Mac 功能，但在你的 Mac 上使用会很棒。照片应用程序获得了一个单独的 iCloud 共享照片库，可以与家人共享功能配合使用。

家庭成员可以选择自动将照片添加到库中，也可以手动移动照片和视频。iCloud 共享照片图库为每个人提供了平等的权限，因此您可以轻松地从 Mac 上整理和编辑您所有的家庭度假照片。

## macOS Ventura 的其他有趣变化

macOS Ventura 有许多较小的变化，例如:

*   时钟和天气的专用应用程序。
*   Spotlight search 现在有了更丰富的设计，允许你查看更多关于电影、电视节目和名人的信息。另外，你现在可以从搜索结果中预览文件。文本搜索将显示照片的搜索结果。
*   重新设计的 Home 应用(与 iOS 16 和 iPadOS 16 中相同)。
*   苹果新闻中专门的“我的运动”部分，提供更新和实时比分。
*   视频的实时文本支持，带有暂停帧。
*   现场音频的实时字幕。
*   在 iPhone、iPad 和 Mac 之间即时切换 FaceTime 通话。

## 支持 2017 年的 MAC 电脑

macOS Ventura 可以追溯到 2017 年，因此即使是六岁的英特尔 MacBook 也将获得新功能。这是支持的设备的完整列表。

*   **iMac—**2017 年及以后
*   **iMac Pro**
*   **MacBook Air—**2018 年及以后
*   **MacBook Pro–**2017 年及以后
*   **Mac Pro–**2019 年及以后
*   **Mac Mini—**2018 年及以后
*   **MacBook**–2017 年及以后
