# 使用此扩展显示 Priceline 隐藏的酒店名称

> 原文：<https://lifehacker.com/reveal-pricelines-hidden-hotel-names-with-this-extensio-1849173602>

当你通过第三方供应商预订酒店时，你实际上是在以较低的价格换取灵活性。例如，Priceline 提供他们所谓的[Priceline Express Deal](https://www.priceline.com/elmo/pl/hotel/express-deals/list)，在这里你可以以低于广告价格 60%的价格预订酒店房间，但问题是他们不会在预订时告诉你你将住在哪个酒店。相反，他们只给你 [一张酒店所在位置的总图](https://www.priceline.com/relax/at/express/3000021082/DF97E78834788C855D682CD430C0F4DFB9B71248A0562A7C883F1D1D2263CCA50B29561839BC983FDB73CBF5D9EB3714959F4AAA9150E16B4ED78A7CC5982E90A2A6E554E21AD41A62103BF50D5E02F2BBF886A779D995E37063A6D2B2276E3D5D6C644E1A7F76D2B1C4BB0F5CAC5121/from/20220720/to/20220722/rooms/1?cur=USD&qdp=92&vrid=4a29e48cc28a204ab780c6c0f39dfd0c) ，并保证酒店是公司“值得信赖的合作伙伴”；直到你实际预订房间并支付房费后，你才知道你住的是哪家酒店。

## 如何揭露 Priceline 隐藏的酒店名称

不过，多亏了 redditor[u/current pangolin](https://old.reddit.com/user/CurrentPangolin)创建的谷歌 Chrome 扩展，你可以找到提供的酒店。通过使用扩展功能 [TravelArrow、](https://chrome.google.com/webstore/detail/travelarrow-hidden-hotel/coplmfnphahpcknbchcehdikbdieognn?hl=en&authuser=0) ，您可以在预订之前，发现 Priceline 通过其快捷交易*为您提供的酒店名称。TravelArrow 使用其算法，通过交叉引用所列地区的酒店和 Priceline 公告上列出的便利设施来预测隐藏的酒店。*

Reddit 用户 u/emotionalchungus positively[描述了他们使用该应用程序的体验](https://old.reddit.com/r/LifeProTips/comments/vw8yx5/lpt_find_out_the_name_of_the_priceline_express/) ，解释道:“我在旧金山的旅行中找到了 Priceline express 交易的名称，并为我的住宿预订了一家五星级酒店。我在两家“隐名”五星级酒店之间做选择，如果我事先没有找到名字，我会选择最差的一家。”

这一扩展应该有助于让您免除通过第三方供应商预订的焦虑。当然，你可以查看酒店评论，也可以将供应商网站上列出的价格与酒店网站上列出的价格进行交叉对比，看看你是否真的买到了划算的东西。不管广告中的上市是否站得住脚，你可能会感谢自己将不确定性从等式中剔除。
