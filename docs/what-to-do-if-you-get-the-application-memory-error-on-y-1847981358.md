# 如果您的Mac出现“应用程序内存”错误，该怎么办

> 原文:[https://life hacker . com/what-to-do-if-you-get-the-application-memory-error-on-y-1847981358](https://lifehacker.com/what-to-do-if-you-get-the-application-memory-error-on-y-1847981358)

“您的系统应用程序内存不足”不是一个友好的警告。感觉又冷又暗，好像你的Mac在告诉你“末日即将来临”幸运的是，末日*并未临近*，但是在你的电脑上收到这个警报仍然令人担忧。根据您的机器，它可能只是一个bug，也可能是您应该注意的一个持续存在的问题。让我们探讨一下如何区分这些问题。

## 你Mac的内存是这个警告的关键

让我们从一个基本的解释开始:你的Mac有固定数量的RAM(随机存取存储器)。这种内存允许您的电脑同时运行多个应用程序和任务。简而言之，你的RAM越多，你的计算机一次能做的就越多；内存越少，你的电脑能做的就越少。很简单。

苹果的macOS通常在内存管理方面做得很好，所以如果你一次运行太多任务，你不需要经常担心。然而，偶尔，系统会被推到极限之外。如果运行的应用程序多于可用内存，macOS将无法继续运行，从而触发此应用程序内存警报。

问题是，这个应用程序内存警报影响了拥有大量内存的MAC电脑。你不会想到内存为16、32或64 GB的Mac在运行Mail、Photoshop甚至Final Cut Pro等应用程序时会遇到问题。然而，各种类型的MAC电脑在执行任务时都会看到这个警告，而它们的电脑应该能够顺利完成这些任务。受影响的机器从全新的M1 Pro和M1 Max MacBook Pro，到基于英特尔的旧MAC。

## 这个问题可能是许多MAC电脑的软件缺陷

这些应用程序也没有使用典型的系统内存。有些app在啃GB内存，完全不正常。 [据9to5Mac](https://9to5mac.com/2021/11/02/application-memory-error-affecting-many-mac-apps-intel-as-well-as-m1/) 报道，一位用户报告称，苹果的Pages和Keynote应用程序使用了80至90 GB的内存或更多。这是一个巨大的危险信号，表明出了问题，而且，鉴于有大量类似占用内存的报道，这表明问题源于软件缺陷。

虽然有报道称该问题会影响旧版本的macOS，但大多数情况是在运行macOS Monterey的macOS中发现的。那么，苹果很有可能会很快发布这个漏洞的补丁。

## 如何辨别警报是bug还是合法的

但是，需要注意的是，这个警报不仅仅是一个bug。它的存在是有目的的；你的机器*可能会*耗尽应用程序内存，在这种情况下，你需要终止应用程序来让事情再次运行。但是，如果您看到此警报，可能很难知道它是合法的，还是bug的一部分。

要注意的最大线索是一个应用程序使用了多少内存。通常情况下，大多数应用程序将使用兆字节的内存，可能是个位数的千兆字节的内存。如果你看到任何应用程序无缘无故地使用两位数的内存，那就是你的Mac受到了这种内存漏洞的影响。然而，如果所有的数字看起来合理，您可能已经用完了应用程序内存。当然，也有例外，因为专业和重型应用程序会使用大量内存。但是，由于这些警报在正常使用过程中会影响用户，这就是我们在这里看到的。

在合法的情况下，此警报往往会影响RAM容量较低的MAC。例如，如果你正在摇一台4 GB内存的MacBook Air，你可能已经熟悉这些警告了。如今，大多数软件和程序都喜欢至少8 GB的内存，如果不是更多的话，所以内存越少，越有可能将系统推向极限，尤其是如果你使用像谷歌Chrome这样占用内存的应用程序。

## 如果可以，给Mac增加更多内存

在某些情况下，您不会被现有的内存所困扰。如果你的Mac有资格升级内存，你应该考虑一下。给系统更多的内存将有助于避免这些令人沮丧的情况，并允许您一次运行更多的任务。

要确定你是否能升级你的Mac电脑的内存，唯一的方法就是查看你的电脑。苹果在这里有一份 [兼容IMAC的清单](https://support.apple.com/en-us/HT201191) 、 [MacBook Pros在这里](https://support.apple.com/en-us/HT201165) ，还有[Mac mini在这里](https://support.apple.com/en-us/HT205041) ，此外还有如何给机器安装更多内存的说明。如果你没有看到你的Mac被列在这些特定的页面上，这意味着内存不能升级，你只能用你现有的。然而，如果你有一台Mac mini，苹果会明确说明哪些型号不能升级，而不是将这些型号排除在外。

这是近年来苹果设计理念的不幸副作用。他们没有把RAM之类的组件做成用户可接触和可拆卸的，而是直接把它们焊接到计算机的主板上，使得升级变得不可能。如果你的Mac有足够的内存，这没问题，但对于那些努力跟上的老Mac来说，没有办法给它们额外的提升。

除了买一台新机器，你能为你的低内存Mac做的最好的事情就是留意你一次打开了多少应用。尽量减少浏览器标签页(尤其是Chrome ),如果你经常使用应用程序，尽量只打开它们。这很痛苦，但总比一天多次遇到应用程序内存警报要好。
