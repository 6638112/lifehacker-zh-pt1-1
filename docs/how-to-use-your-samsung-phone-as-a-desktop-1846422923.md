# 如何将您的三星手机用作桌面

> 原文：<https://gizmodo.com/how-to-use-your-samsung-phone-as-a-desktop-1846422923>

许多人不知道你可以使用三星 DeX 将你的三星 Galaxy 手机或平板电脑作为桌面体验。如果你对 DeX 的想法完全陌生，那么“桌面体验”可以从你的 Galaxy 设备上运行，让你使用带有显示器、鼠标、和键盘的手机或平板电脑。三星的 OneUI Android 软件适应成为更类似于 Windows 或 macOS 的东西，所以你可以在更大的屏幕上访问你的文件和应用程序。



自 2017 年推出以来，三星已经为 DeX 添加了一系列功能。下面是您现在可以使用 Samsung DeX 做的一切，以及如何开始使用。

### **设置 DeX 连接**

当 DeX 与 [三星 Galaxy S8](https://gizmodo.com/samsung-galaxy-s8-review-the-prettiest-phone-wins-1794526975) 手机一起推出时，它需要一个特殊的 [DeX Pad](https://www.amazon.com/Samsung-Desktop-Experience-Charger-Warranty/dp/B079TL2TGY?asc_campaign=InlineText&asc_refurl=https://gizmodo.com/how-to-use-your-samsung-phone-as-a-desktop-1846422923&asc_source=&tag=kinjagizmodolink-20) 或 [Dex Station](https://www.amazon.com/Samsung-Station-Desktop-Experience-Warranty/dp/B06XR9M1ZP?asc_campaign=InlineText&asc_refurl=https://gizmodo.com/how-to-use-your-samsung-phone-as-a-desktop-1846422923&asc_source=&tag=kinjagizmodolink-20) 来处理手机或平板电脑与显示器或电视之间的连接。虽然不再需要这些配件，但它们确实使连接所有设备变得更容易，并且还让你插入有线鼠标或键盘，而不是试图通过蓝牙连接来使用它们。

你还可以为 Note 8、Tab S4、和更新的设备选择一条 [DeX 线缆](https://www.amazon.com/Original-Official-Samsung-Type-C-Packaging/dp/B07GBGJ2FD?asc_campaign=InlineText&asc_refurl=https://gizmodo.com/how-to-use-your-samsung-phone-as-a-desktop-1846422923&asc_source=&tag=kinjagizmodolink-20) ，将其插入设备的 USB-C 端口，以连接电视或显示器上的 HDMI 端口。(这条线其实没什么特别的，任何 USB-C 转 HDMI 线都可以。)一旦你插上显示器，你的三星设备应该会检测到它，并开始在第二个屏幕上显示 DeX。

2019 年，三星针对 PC (以及 Mac)推出了 [DeX。这为您的笔记本电脑或台式机带来了 DeX 体验。在你下载并安装了软件之后，你用任何标准的 USB 线将你的手机或平板电脑连接到 Windows 或 macOS。自 2018 年以来发布的任何旗舰三星 Galaxy 手机都可以兼容，从 Galaxy S9 系列和](https://www.samsung.com/us/explore/dex/) [Galaxy Note 9](https://gizmodo.com/samsungs-galaxy-note-9-is-the-best-big-phone-1828348138) 开始。

DeX 桌面软件增加了一些额外功能，如将文件从移动设备快速传输到计算机的能力，以及从 Windows 或 macOS 向联系人发送短信的实用程序。你马上就有了内置的键盘和鼠标(或触控板)控制，因为它们是电脑自带的。请注意，一些最新的三星设备现在可以通过本地 wifi 网络使用 DeX for PC [而无需线缆](https://www.androidpolice.com/2021/02/25/samsung-dex-now-works-wirelessly-on-pcs-if-you-have-a-galaxy-s21/) 。

还有几个其他的 DeX 模式可以讨论。最近的三星 Galaxy 平板电脑也可以在设备本身上运行 DeX 模式:只需点击快速设置面板上的 **DeX** 按钮(用两个手指从显示屏顶部向下滑动即可看到)，即可切换到类似桌面的窗口体验。如果你已经购买了三星官方键盘盖，它也有自己专用的 DeX 模式按钮。

然后是电视的无线镜像，这可以通过 Note 20 和 Galaxy S21 设备上的 Miracast 协议实现。用两个手指从设备屏幕顶部向下滑动，打开快速设置，然后在电视或显示器上点击 **DeX** 和**DeX**:如果有支持 Miracast 的电视或流媒体棒(如亚马逊 Fire TV)，它会出现在**无线连接**标题下。

### **使用 DeX 体验**

一旦你建立了 DeX 连接，你就可以开始使用这个软件了。如果你在手机或平板电脑上看到一个通知，询问你是否要启动 DeX 模式，确认你要启动。你可能还会在屏幕上看到一个提醒，让你为手机或平板电脑启用触摸板模式，这意味着你可以通过在手机或平板电脑的显示屏上滑动来控制显示器或电视大屏幕上的光标。

DeX 设置面板是一个很好的起点。你会在桌面上看到设置的快捷方式，或者你可以点击通知托盘(右下角)和 cog 图标到达相同的位置。从列表中选择**三星 DeX** ，你可以配置一系列不同的选项，从软件使用的桌面壁纸和字体大小，到任务栏在不使用时是否自动隐藏。

设置列表中的**三星 DeX 实验室**条目也值得一游。在此之前，你可以配置更多的实验性功能，包括启动 DeX 时自动打开你正在使用的最后一个应用程序的选项，以及如何处理应用程序大小调整的控件。同时，如果你在 PC 上使用 DeX，那么你可以通过点击右上角的 cog 图标进入桌面软件程序的设置客户端。

点击左下角的小应用网格按钮，调出你的应用列表，并启动尚未打开的应用。应用程序可以使用窗口标题栏移动，使用窗口的边缘调整大小，使用角落的按钮最大化和最小化——如果你熟悉 windows 和 macOS 的操作方式，这一切都非常简单。你甚至可以将窗口贴在屏幕的一侧(只需将它们拖到左边或右边)，然后右击将应用程序固定在任务栏上。

通过点击 **Recents** 按钮，你可以看到每个打开的应用程序的概览，这个按钮在左下角显示三条水平线。请记住，不是所有你使用的安卓应用都能完美运行，尽管大多数应该能够很好地适应更大的界面(特别是像微软 Office 应用这样的生产力应用，已经可以在平板电脑上无缝运行)。

你会意识到，你本质上只是在操作你的手机，但通过不同的界面和不同的输入方法(如果你设置了鼠标和键盘)。你仍将登录到所有应用，你的历史和书签仍将出现在移动浏览器中，等等。三星本身警告说，运行 DeX 时，你的手机可能会变热，这可能会有点消耗电池寿命，但如果你只是需要快速完成一些工作，将手机变成桌面的选择是有用的。
