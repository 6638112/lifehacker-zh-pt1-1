# 是时候再次使用您的旅行积分卡了吗？

> 原文:[https://life hacker . com/is-it-time-to-start-using-your-travel-rewards-cards-aga-1846551019](https://lifehacker.com/is-it-time-to-start-using-your-travel-rewards-cards-aga-1846551019)

随着疫情旅游限制的结束似乎指日可待，你可能想知道这是否又是一个注册旅游奖励信用卡的好时机。一些公司正在提供比以往更大的欢迎奖金，但在2021年底之前，安全旅行可能不是一个现实的选择。在那之前，你能凑合着享受现有的福利吗？或者，在有更多确定性之前，是否值得推迟？

### **立即获得旅行奖励卡的理由**

由于信用卡可能是一个有风险的 [债务陷阱](https://www.thebalance.com/how-to-avoid-the-credit-card-trap-2386313) ，真的没有什么*需要*去办一张旅行卡——也就是说，除非你确定每个月都能还清余额，否则旅行福利是值得的，获得这些福利符合你预期的消费习惯。通常，信用卡购物者会寻找:

*   高现金价值的可实现的欢迎优惠或奖金。
*   收益的价值超过了年费的成本，年费可能高达数百美元。
*   提供可转让、易于兑换的奖励积分的卡。

旅行卡非常适合经常旅行的人，因为它们提供基于旅行的积分，以及其他旅行专属的额外待遇，如旅行保险、优先登机、休息室使用权和免费座位升级。当然，如果没有旅行，这些额外津贴大部分都是无用的，这就是为什么信用卡公司在疫情期间转向餐饮、食品杂货和网上购物的奖励奖金。

在许多情况下，这些额外津贴是慷慨的，但它们也会影响你如何使用你的卡——尤其是如果你已经有了用于食品杂货和网上购物的专用卡。另一方面，旅游*正在*复苏，现在旅游奖励卡上有一些非常好的交易，所以至少考虑再次购物卡是有意义的。

### 旅行是个好主意吗？

现在旅行安全吗？简短的回答是“不会”，因为疾病预防控制中心只建议在这个时候进行必要的旅行(查看这个关于这个话题的 [Lifehacker帖子](https://vitals.lifehacker.com/is-it-safe-to-travel-yet-1846488590) )。然而，随着疫苗接种的步伐加快，有理由认为安全旅行可能在2021年下半年实现。

因此，要知道现在是否是获得旅行奖励卡的正确时机，你必须现实地考虑你实际可能会旅行多少次，你是否会实际使用该卡的非旅行津贴，以及这些津贴是否值得该卡的年费。
