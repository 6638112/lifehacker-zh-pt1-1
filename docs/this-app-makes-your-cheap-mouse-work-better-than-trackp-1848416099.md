# 这个应用程序让你的廉价鼠标比触控板手势更好地工作

> 原文：<https://lifehacker.com/this-app-makes-your-cheap-mouse-work-better-than-trackp-1848416099>

一方面，在 Mac 上使用廉价的蓝牙鼠标比用手指移动光标要好。然而，另一方面，你错过了一些 macOS 的触控板手势，包括三指拖动以切换到不同的工作空间。但有一款名为 Mac Mouse Fix 的免费应用程序，可以为你廉价的蓝牙鼠标添加触控板手势和平滑滚动功能，你应该试试。

Watch

一旦你下载了 [Mac 鼠标修复 2](https://mousefix.org) ，它会立即捕捉你鼠标的所有按键(除了左键和右键)，让你决定当你按下它们时会发生什么。如果你使用廉价的蓝牙鼠标，它应该只有三个按钮，Mac Mouse Fix 会捕捉中间的按钮供你使用。该应用程序允许您配置当您单击、按住、双击或单击并拖动鼠标中键时会发生什么。

如果你热衷于将“高效的”应用和“分散注意力的”应用分开，你可以为每一类应用使用单独的工作空间，并通过鼠标轻松访问它们。例如，Messages 和 Telegram 可以占用一个工作区，Slack 和 Safari 可以占用另一个工作区，您可以设置一个点击拖动操作来更改工作区。或者，您可以设置中击移动到下一个工作区，双击返回到上一个工作区。

由于应用程序中的平滑滚动选项，中间的滚轮按钮变得更好，如果这还不够，你还可以让你的鼠标复制一些键盘快捷键。例如，你可以使用默认实用程序的**命令+ Shift + 3** 或**命令+ Shift 4** 键盘快捷键或 Mac 上的 [其他截图应用](https://lifehacker.com/how-to-replace-your-mac-s-screenshot-tool-with-somethin-1848370944) 进行快速截图。使用 Mac Mouse Fix，您可以将这些快捷键与单击或双击中间按钮配对。

我的蓝牙鼠标侧面有几个额外的按钮，所以我将截图操作与这些按钮配对，这使我的工作流程快了很多。如果你点击 Mac Mouse Fix 偏好设置窗口顶部的**更多**按钮，你会在**应用特定设置**下找到一些更有用的调整，比如启用或禁用单个应用的设置。

当你随心所欲地设置了 Mac Mouse Fix 后，勾选**启用 Mac Mouse Fix** 并关闭应用，你的廉价蓝牙鼠标就不会再那么差了。
