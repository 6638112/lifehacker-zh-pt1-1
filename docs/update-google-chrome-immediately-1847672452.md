# 立即更新谷歌浏览器

> 原文：<https://lifehacker.com/update-google-chrome-immediately-1847672452>

安全漏洞是数字生活中不可避免的一部分。谷歌最近在 Chrome 上发现了 11 个漏洞——其中两个处于活跃状态——并迅速发布了修复程序。因此，您应该尽快更新。

## 这些谷歌 Chrome 漏洞是怎么回事？

9 月 13 日星期一，谷歌 [发布了 Chrome 浏览器的安全更新](https://chromereleases.googleblog.com/2021/09/stable-channel-update-for-desktop.html) ，修补了 11 个已知问题，包括两个该公司确定为积极利用零日漏洞的问题。这意味着坏人在谷歌或其他相关实体不知情的情况下利用了其中两个安全漏洞。

谷歌表示，“匿名研究人员”负责向公司报告这两个关键错误。9 月 8 日发布的报告详细说明了这些漏洞——名为 **CVE-2021-30632** 和**CVE-2021-30633**——是通过 V8 中的越界写入和索引数据库 API 中的免费使用来利用的。修补这些问题的新更新是版本 93.0.4577.82。

## 如何更新 Chrome 来修补这些漏洞？

Chrome 有可能在后台自动为你更新，所以这些补丁可能已经安装了。否则，您可能会在浏览器上看到一个更新按钮，颜色为以下三种颜色之一:

*   绿色:不到两天前发布了更新。
*   橙色:大约 4 天前发布了一个更新。
*   红色:至少一周前发布了一个更新。

在本文发表时，更新足够新，您应该只能看到一个绿色按钮。拥有这条路，你可能最终会看到另外两种颜色中的一种。

要更新，点击菜单栏中的**更多**按钮(三个垂直点)。如果有可用的更新，点击**更新谷歌浏览器**如果你没有看到这个选项，谷歌说你的浏览器是完全最新的。但还有另一种方法可以进行双重检查，同时确认你运行的是哪个版本的 Chrome:

## 如何检查我运行的是哪个版本的 Chrome？

头回到**更多**按钮，点击**帮助**，然后选择**关于谷歌 Chrome** 当应用程序搜索任何可用的更新时，你会看到你当前版本的 Chrome。如果有可用的，您可以按照屏幕上的说明从这里安装它。记住，新版本是 93.0.4577.82。
