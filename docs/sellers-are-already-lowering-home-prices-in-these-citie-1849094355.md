# 卖家已经在降低这些城市的房价

> 原文：<https://lifehacker.com/sellers-are-already-lowering-home-prices-in-these-citie-1849094355>

随着通货膨胀和利率同步上升，美国某些城市的房价开始下跌——或者至少趋于平稳。 [据 Redfin 房地产新闻](https://www.redfin.com/news/price-drops-increase-may-2022/) 报道，越来越多的房屋卖家本月开始降低要价，标志着最近火热的房地产市场降温。Redfin 将这一下降归因于迅速上升的抵押贷款利率削减了潜在买家的预算，由于对衰退的担忧、股市波动以及经济不确定性的普遍影响，导致购房情绪下降。

据 Redfin 称，价格下跌最多的地区正是疫情开始时从飙升的估值中获益最多的地区。Redfin 解释说，“价格下跌在西部的中型大都市尤其常见，尤其是在犹他州，其中许多城市在疫情期间价格上涨过快，因为它们是从美国其他地区迁入的人们的热点地区。”在美国，超过 10%的房屋卖家本月在所有大都市地区降价。

《财富》杂志将这种现象称为“大减速”。最近上市的人把 h omes 放到市场上，却发现他们预期的需求并不存在，迫使他们从最初的要价下调。given[美国房价在过去两年中平均上涨了 34.4%](https://fred.stlouisfed.org/series/CSUSHPINSA)，超过了 2008 年抵押贷款危机前夕公布的房价两年来的最大涨幅(21.2%)，任何放缓都是买家欢迎的，但卖家似乎仍在调整他们的预期。

不可能知道买房的确切“完美”时间，但看起来激烈的竞标大战正在结束——特别是在美国的这 10 个城市，其中在过去的一个月里房价下降幅度最大:
