# 使用一副便宜的护目镜，而不是“洋葱护目镜”

> 原文:[https://life hacker . com/use-a-pair-of-chief-safety-gogs-to-onion-Gog-1846279635](https://lifehacker.com/use-a-pair-of-cheap-safety-goggles-instead-of-onion-gog-1846279635)

洋葱几乎被用在每一种烹饪中，这使得它们在厨房里几乎是不可避免的。我从来不想避免吃洋葱，但是，众所周知，同时切洋葱会导致刺痛、流泪，如果你碰巧涂了睫毛膏“只是为了感觉漂亮”，你的脸上会出现黑色条纹和污迹。

幸运的是，有几种方法可以减轻顺式丙硫乙胺氧化物的影响，这是一种挥发性的、引起疼痛的化学物质，当你破坏洋葱的细胞壁时，它会释放出来。你可以在风扇旁或通风橱下工作(从身体上去除气体)，将它们放入冰箱半小时(减少气体的挥发性)，或者冲洗或浸泡在水中(将化合物冲洗到下水道)。

所有这些方法都有效——尽管最后一种方法会使洋葱变得滑溜溜的，味道有点太淡——但有时我的炉子在用，或者我忘记把它们放在冰箱里，等不及多等30分钟。这时，我伸手去拿一副旧的护目镜，它从物理上阻挡了合成丙硫乙胺。

如果你不得不参加任何种类的化学实验，你可能会看到这些。它们是塑料的，四四方方的，侧面有小通风口，可以根据需要打开或关闭。在家得宝 (或任何当地的五金店)，它们的价格不到4美元 [，它们可以套在眼镜上。专门的“](https://www.homedepot.com/p/HDX-Chemical-Splash-Goggle-SG-604/309028890) [”洋葱护目镜](https://www.bedbathandbeyond.com/store/product/onion-goggles/112667) ”也能阻挡合成丙硫乙氧基化物，但它们价格更高(12-20美元)，不适合你现有的眼镜，而且看起来也不酷。(我真的很喜欢矮胖泳镜的样子，不知道为什么！)

只是把它们戴在头上，拉紧松紧带使之紧密贴合，然后切掉，不要用风扇、冷冻或冲洗。密封越紧，你的眼睛就越安全，但即使是不完美的配合也比没有好。我的旧实验护目镜的带子已经失去了很多弹性，很容易滑落，但我仍然能够毫不费力地穿过三个完整的洋葱。
