# 做土豆泥时，有两件事你绝对不应该做

> 原文：<https://lifehacker.com/two-things-you-should-never-do-while-making-mashed-pota-1847984337>

很多人喜欢把秋天和冬天称为“喝汤季节”，但我一直认为这些寒冷的月份是“土豆泥季节”无论是作为(不可协商的)感恩节大餐，还是作为一周中的一餐，很少有东西能像一碗奶油土豆泥一样让人感到舒适和满足。

这是 [一个足够简单的概念](https://lifehacker.com/how-to-make-perfect-mashed-potatoes-in-the-microwave-1846629900) :煮一些土豆，然后将它们与脂肪一起捣成糊状。用盐调味，然后吃。这并不难，但是有两种主要的方式可以破坏土豆泥的食用体验，我对这两种方式都感到内疚。(很久以前的事了。从那以后我成长了很多。)

## 大错误#1:用刀片捣碎

几乎每个马铃薯捣碎机新手都会犯这个错误。食品加工机、搅拌机等看起来是捣碎土豆的有效方法，但问题是你根本没有捣碎——你在消灭它们。。你最终得到的不是松软的奶油土豆，而是一团黏糊糊、黏糊糊的东西。

捣碎是一个比搅拌或其他发生在你的食品加工机内的事情温和得多的动作。碾米机、马铃薯捣碎机和木勺打碎煮熟的马铃薯块，同时或多或少保持淀粉分子完整。快速移动，超- 锋利的刀片撕裂这些分子。这种释放出来的淀粉然后和你的醪液混合，形成一种令人倒胃口的糊状物。

如果你是一个吃土豆的人，非常讨厌土豆泥里的块状物，买一个米饭。大米加工者通过迫使土豆穿过小孔，将土豆打碎成微小的(类似大米的)碎片。这些挤出来的— 但仍然蓬松的— 马铃薯颗粒然后涂上乳制品和脂肪，形成一种蓬松的马铃薯乳液，而不是粘稠的混合物。

## 大错误#2:把果皮扔进垃圾处理器

把你的土豆皮扔到垃圾处理器里不会破坏土豆泥的味道或质地，但是会让整个体验变酸。有一次，我在父母家试图用这种方法清理土豆皮，差点毁了平安夜。(以前我曾把许多蔬菜皮扔进垃圾箱，但我沮丧地发现土豆是个例外。)

又一次，讨厌的淀粉和锋利的刀片是罪魁祸首。果皮又滑又薄，这使得它们很容易滑入管道，缠绕在你要处理的刀片上；试图用这些刀片打碎它们会释放出淀粉，形成粘稠的糊状物。

这是一个糟糕的场景，而不是你想在假日期间创造的场景，因为那时水管工已经被超额预定了。最简单的解决方案是首先不要制造问题，但是如果一些土豆新手用果皮堵塞了你的处理，一切都没有失去。小苏打和醋 可以帮助分解堵塞物，尽管我必须承认我没有尝试过这种特殊的方法，因为自从那次灾难性的平安夜之行后，我再也没有用土豆皮堵塞过厕所。(一盎司的预防确实抵得上一磅的治疗，或者在这种情况下，抵得上小苏打。)
