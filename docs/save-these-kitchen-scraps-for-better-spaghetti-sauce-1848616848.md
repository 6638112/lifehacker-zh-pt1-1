# 把这些厨余留着做更好的意大利面酱

> 原文:[https://life hacker . com/save-these-kitchen-scrap-for-better-spaghetti-sauce-1848616848](https://lifehacker.com/save-these-kitchen-scraps-for-better-spaghetti-sauce-1848616848)

很多人谈论把蔬菜残渣 [存起来](https://lifehacker.com/how-to-make-seafood-stock-out-of-scraps-1823958899) (就像他们应该做的那样)，但是太多的人睡在 *酱*残渣袋里。功能上，过程是相似的:收集你的蔬菜皮、烟蒂、和其他各种各样的蜉蝣，把它们放在一个冷冻袋里；把袋子放进冰箱里；下次你做高汤的时候把它取出来，然后用这些残渣给一锅煨汤调味。启动酱料废料袋的过程与相似，但的区别在于焦点。

Watch

库存废料袋可以包含任何东西——周围没有不属于的蔬菜废料——但是酱料废料袋稍微更有管理性。如果你正在做的蔬菜、香草或贝类在意大利面酱中味道很好，抓住它们相应的皮、茎或壳，扔进冷冻袋，然后把袋子放在冰箱里，直到下次你做酱的时候。

## 意粉酱废料袋里要加什么

你放在你酱料包里的是个人选择，我也不敢对它的内容指手画脚。T4:但是我确实有一些建议。(上周 [之后](https://newsletters.theatlantic.com/humans-being/62237275ebb7370020a1d48d/why-im-on-strike-at-my-other-job/) ，我早该给你建议了。)

*   洋葱蒂和皮。储存前冲洗掉根部的污垢，并确保去掉带有少许洋葱果肉的果皮。煨上你的调味汁，增加蒜味。
*   **胡萝卜蒂。**这些宝贝添加了暗示的甜味；它们放在肉酱面里特别好吃。
*   [](https://lifehacker.com/make-a-flavor-packed-stock-from-shrimp-shells-1801187937)**。要不要鲜味？因为这些男孩带来了鲜味。把它们想象成凤尾鱼或者 [虾酱](https://lifehacker.com/use-a-spoonful-of-shrimp-paste-when-you-cant-find-ancho-1843732484)——略带腥味，带着浓浓的味道。(只是不要像吃凤尾鱼一样吃它们；那会令人不愉快。)**
*   **[](https://lifehacker.com/use-papery-garlic-skins-to-give-broth-big-flavor-1797726954)**。去掉外层纸质层，把注意力集中在你直接从丁香上撕下的粘性果皮上。它们放在简单的番茄酱里特别好吃，尤其是如果你只是想要一点补充的大蒜味道。****
*   ******药草茎。罗勒和欧芹的茎几乎和它们的叶子一样有味道。用厨房细绳把它们捆起来扔进去。******
*   ****帕尔马干酪皮。你最喜欢的意大利面条装饰的硬蜡皮，当它融化到你的番茄酱或周日肉汁中时，含有盐、奶油和大量谷氨酸盐。****

****不像高汤或汤，番茄酱不能过滤掉你的残渣。不要浪费时间从你的酱汁中捞出壳、茎和皮，在浸入西红柿、橄榄油和葡萄酒的混合物之前，用厨房用的麻绳将所有东西固定在一起，或者用一些奶酪布将它们绑起来。(奶酪布会变得很粘，但我认为这是一个合理的牺牲。)****
