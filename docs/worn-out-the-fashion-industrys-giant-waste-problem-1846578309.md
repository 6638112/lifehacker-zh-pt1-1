# 破旧:时尚产业的巨大浪费问题

> 原文：<https://gizmodo.com/worn-out-the-fashion-industrys-giant-waste-problem-1846578309>

春天来了，我对我的衣柜想得更多了——我对它有多厌恶。即使我最近清理了我的衣柜，我仍然数出了至少 25 件我不再经常穿的衣服。但是我该怎么处理我所有的毛衣和裙子呢？

我知道我不应该扔掉那些东西。如果我这样做了，它很可能会被扔进焚化炉或垃圾填埋场，这两者都会对环境造成毁灭性的破坏。[两种方法](https://gizmodo.com/waste-disposal-in-us-landfills-has-been-underestimated-1732517793) 向大气中释放有毒的、使地球变暖的污染物，而 [两种](https://earther.gizmodo.com/trash-burning-power-plants-wind-up-in-our-most-marginal-1834928924) [种](https://qz.com/939612/race-is-the-biggest-indicator-in-the-us-of-whether-you-live-near-toxic-waste/) 的设施往往建在有色人种的贫困社区，造成了环境不公平的公共卫生负担。

我想把它们捐给当地的慈善机构会是一个更好的选择，在某种程度上也是这样。但是根据记者亚当·明特的说法，他写了一本书 [*二手货:在新的全球车库拍卖中旅行*](https://bookshop.org/books/secondhand-travels-in-the-new-global-garage-sale/9781635570106)*旧货店只卖出大约三分之一的商品。*

*其余三分之二不一定会被丢弃，而是进入了全球贸易的迷宫。一些多余的衣服——尽管没有确切数量的数据——最终进入了巨大的全球再利用市场。来自世界各地的商人收集衣服，对它们的状况进行分级，然后将它们送去回收，再次穿着，或者变成抹布和床垫填料。*



*不过，有一个问题。目前，时尚产业正在生产如此多的东西，以至于这个全球网络 [变得不堪重负。](https://www.bloomberg.com/opinion/articles/2018-01-15/no-one-wants-your-used-clothes-anymore) 服装产量从 2000 年起已经翻了一番，时尚产业现在每年生产[1500 亿件](https://www.vogue.com/article/fast-fashion-environmental-impact-sustainability-parsons-zady) 。社会通过购买更多的衣服来跟上所有新衣服的生产:今天平均每个人购买的衣服比 20 年前多 60%。*

*更多的新衣服也意味着旧衣服更快地被扔在一边。自 2002 年以来，一件衣服在被丢弃之前被穿的平均次数下降了三分之一以上，这意味着我们正在制造更多的废物。明特说，解决这个问题的一个方法是迫使公司生产耐穿的服装，而不是像现在这样快速时尚。*

*“毫无疑问，衣服是故意做得不太耐用的，”他说。*

*为了解决这个问题，政府可以对服装实施耐久性标准，禁止有计划的淘汰——有点像法国对电子产品 所做的[——以确保它们经久耐用。](https://gizmodo.com/the-right-to-repair-busted-appliances-is-getting-a-big-1838660740)*

*但是丢弃的衣服并不是这个行业唯一的浪费问题。该行业还在服装生产过程中丢弃大量未使用的布料。这个问题启发了像 Fabscrap 这样的非营利组织从马克·雅可布、奥斯卡·德拉伦塔和艾琳·费希尔这样的品牌那里收集未使用的材料。然后，该组织将布料分类，转售或捐赠给艺术家和新兴品牌。*

*但这不应该是非营利组织的责任——政府应该介入，帮助处理所有这些材料并监管公司，例如通过限制使用难以重复使用或回收的织物，如 [聚酯混纺](https://www.bbc.com/future/article/20200710-why-clothes-are-so-hard-to-recycle) 。*

*如果你正在寻找减少衣物浪费的方法，你可以考虑修补破损的衣服，而不是更换它们，这也是一个省钱的好方法。如果你和我一样，有一堆你讨厌穿的衣服，考虑和你的兄弟们交换一下衣服。我们不能在个人层面上解决问题，但是减少个人衣物浪费从来都不是一个坏主意。*
