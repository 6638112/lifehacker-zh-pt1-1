# please dont use ellipses in your text messages 1846102805

> 原文：<https://lifehacker.com/please-dont-use-ellipses-in-your-text-messages-1846102805>

我...有...一份供词...我认为，当你在文本中插入省略号时，你无意中剥夺了你的信息的线性思路。

长期以来，书面单词一直受制于不断演变的风格和惯例，但如果我可以成为一个坏脾气的人，历史上从未有过英语词汇如此惊人地充斥着点点点点。尽管听起来像年龄歧视，但事实仍然是，这是我们中最年长的人天生的怪癖——这一代人在没有无所不包的互联网的便利和诅咒的情况下长大成人。但是，与其贬低婴儿潮一代——或任何人——还不如让他们的交流适应数字时代，我希望这能有所帮助。



这就是为什么你的文本——以及所有阅读它们的人——应该得到比多个省略号更好的结果。

### 他们让事情看起来好像不对劲

不要让你的读者分析你短信中的微妙之处。对于大多数千禧一代和 Z 世代成员(甚至一些 X 世代)来说，发短信是一种主要的沟通方式，即使它可能真的很乏味和没有人情味。回复“好的……”会让文本读者觉得你好像有什么话没说。例如，让你的句子乞求终结的三个挥之不去的点推断出什么？

语言学家、《因为互联网:理解语言的新规则》一书的作者格雷琴·麦卡洛克最近向《今日美国》[阐述了这个话题。](https://www.usatoday.com/story/life/2020/09/28/texting-etiquette-what-exclamation-point-period-ellipses-mean-to-different-generations/3524169001/)



简而言之，它归结为空间的使用，当在任何数字设备上通话时，空间是非常珍贵的。她告诉该报，省略号在手写信件中远没有那么尴尬，因为思想和句法的正常转换通常更容易破译。

> 如果你给某人写了一封信或明信片，你知道使用一个小的标点符号是从一个想法进入下一个想法的有效方式。

但是当涉及到数字媒体时，省略号正被逐渐淘汰，取而代之的是换行符，这是一种从一个想法过渡到下一个想法的更直接的方式。她总结道，“计算机空间很便宜，换行符占用的代码和点-点-点一样多。”



### 这是一个非常特殊的语法惯例

省略号不是句号。是的，从技术上讲，这是三个句号，每个句号之间都有一个空格。然而，它的实际功能与老式的句号有明显的不同，因为它随书写媒介的不同而不同。

没有省略号是真正平等的。它的用法因格式而异，从小说和新闻写作到信件和更口语化的媒介，比如说短信。

正如语法爱好者在 [你的字典里解释的](https://grammar.yourdictionary.com/punctuation/when-to-use-ellipses.html) 一样，你可能会发现散布在你收件箱里的三个点被普遍误解了:

> 你经常在小说和新闻故事中看到的省略号是英语中最容易被误解的标点符号之一。它在短信、即时消息和电子邮件中被不加区别地使用，社交网站和博客也没有帮助遏制这一趋势。



例如，在新闻写作中(这是我们在 Lifehacker 上做的)，省略是为了通过省略不需要说的内容，在原本不清楚的引用中灌输更清晰的感觉。当引用的一段话可以被浓缩以使读者更加清晰时，就用这个词。

用你字典里的例子，这个引用:

> “我们从所有可获得的数据中得出结论，理解了最近一连串纵火事件的影响，已经肯定地、绝对地、毫无疑问地确定，这场火灾是意外事故”

变成了这句话:



> "我们已经确定…这场火灾是意外事故。"

省略号的用法还有更多细微差别。当在小说或其他书中使用时，它们意味着传达对话或故事情节的暂停。也就是说，这条规则引出了第一点:当使用省略号时，看起来好像还有更多要说的，你可能隐瞒了你的真实意图。

### 太多省略号简直难以阅读

如果...你们...通讯...喜欢...这...你是...不...很...容易的...到...明白。这也许是从文本交流中去除省略号的最基本原因。

无论你是试图反映你的思想慢慢形成单词，还是你不知何故忘记了键盘上有空格键，请记住，看到这种笨拙的语法就足以引发癫痫发作。省略号是一种过于具体的语法规则，不能模糊地使用。另外，当你使用常规换行符时，文本看起来更好。
