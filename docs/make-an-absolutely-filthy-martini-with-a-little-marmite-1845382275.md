# 用一点点马麦酱做一杯绝对肮脏的马提尼

> 原文：<https://lifehacker.com/make-an-absolutely-filthy-martini-with-a-little-marmite-1845382275>

马麦酱是一种迫使人克制的调味品，人们因此讨厌她。这是一种浓稠的、酵母糊，尝起来像浓缩的、轻微烧焦的酱油，即使是很少的一点点也会淹没一个人的味觉，毁掉一片完美的烤面包。但是一旦你学会利用它的咸味力量，你就会拥有一个强大的(纯素食)深层、黑暗鲜味的来源。



Marmite 适合做汤、意大利面、肉汁和任何烘烤的食物，而且——因为它是你能买到的最美味的调味品之一——它绝对适合做美味的鸡尾酒。它在 [血腥玛丽](https://lifehacker.com/tag/bloody-mary) 中很棒，它可以用来做素食者，有点像假凯撒或米歇尔达(这两种都需要蛤蜊汁)，通过提供一点没有任何肉(或蛤蜊)的“肉味”。八分之一 茶匙每血饮、凯撒或红啤酒应该足够了，但你可以随时添加更多，所以宁可太少。(太多的马麦酱几乎不可能再来。)

这也是一种令人惊讶的美味，绝对肮脏的马提尼，如果你愿意，也可以称之为 Marmtini。还是那句话，你不想要很多。只需将一根筷子浸在马麦酱中，然后在你的饮料中搅拌，让马麦酱溶解，这将提供一个漂亮的琥珀色和令人震惊的微妙的黑色，美味的味道。

为了确保马麦酱真的进入你的饮料，在你加冰之前，先把它加到室温的酒里。冰会使浆糊卡住，凝固，粘在你的筷子或量勺上。一旦马麦酱溶解到你的饮料中，加入冰块，然后像往常一样摇晃或搅拌。但是，一定要用筛子过滤；马提尼酒里不是你想要的一小块马麦酱，不管你喜欢它有多脏。
