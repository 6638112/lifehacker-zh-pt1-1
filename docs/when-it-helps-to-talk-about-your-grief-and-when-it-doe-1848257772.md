# 当谈论你的悲伤有帮助的时候(或者没有帮助的时候)

> 原文：<https://lifehacker.com/when-it-helps-to-talk-about-your-grief-and-when-it-doe-1848257772>

悲伤是应对损失的正常部分。无论是所爱的人去世，应对重大疾病，还是结束一段关系，悲伤对每个人来说看起来和感觉都不一样。虽然许多人发现谈论他们的悲伤可以帮助他们理解他们的损失，但也有一些时候不会有帮助——或者当它可能甚至让你感觉更糟。

主持播客 [丧偶妈妈](https://www.coachingwithkrista.com/podcast/) 的悲伤专家克里斯塔·圣日耳曼 说:“根据你在悲伤中所处的位置、损失的性质和你的支持网络，谈论悲伤真的很有用。“在早期的急性悲伤中，谈论它有助于让它变得真实，并帮助你到达接受的地方。”

但是圣日耳曼也指出，在两个主要的时候，谈话可能不会帮助你减轻悲伤。

## **当你和不懂的人说话时**

当然，这听起来很明显:与不理解你悲伤的人交谈可能不会有帮助。但是提前预测通常是很难的，甚至那些和我们最亲近的 T2 人也可能不理解我们正在经历的 T4。

我们也可能经历 [被剥夺权利的悲痛](https://lifehacker.com/how-to-cope-with-disenfranchised-grief-1847499750)当损失被最小化，被误解，甚至不被他人承认。例如，你可能会为失去一段浪漫的关系或亲密的友谊而悲伤，在外人看来，这似乎是一个微不足道的损失。但是对于经历它的人来说，它会带来毁灭性的打击。

## 当它阻碍你前进的时候

第二种情况是可能不会对你的悲伤有所帮助，当谈论它的时候实际上会阻止你继续你的生活。

“有时发生的事情是，我们开始用失败作为坚持下去的理由，作为限制我们可能性的理由，”圣日耳曼说。

虽然悲伤可能永远不会完全消失，特别是如果是因为失去了亲近的人，重要的是能够记住并尊重这一损失，同时仍然努力过上充实、丰富的生活。

“谈论我的悲伤是让我朝着我想要的生活前进，还是让我离我想要的生活越来越远？”圣日耳曼说。“如果我和帮助我处理它的人交谈，它会让我朝着我想要的生活前进。”

对每个人来说，这看起来都不一样，就像悲伤本身一样，谈话的有用性会起起落落。在某些情况下，谈论悲伤将有助于处理损失，在某种程度上可以帮助一个人在享受完整生活的同时尊重损失。在其他情况下，谈话可能会适得其反，只会让一个人停留在过去。

“悲伤不会结束，”圣日耳曼说。“你的想法和感觉可能会随着时间的推移而改变，但我们并没有试图达到一个终点——我们试图接受这种损失，并将其调整到我们的生活体验中。”
