# 如何判断一款游戏已经针对M1 MAC电脑进行了优化

> 原文:[https://life hacker . com/how-to-tell-when-a-game-has-optimized-for-m1-Macs-1847695479](https://lifehacker.com/how-to-tell-when-a-game-has-been-optimized-for-m1-macs-1847695479)

苹果公司的M1苹果电脑速度很快。然而，不仅仅是硬件决定了这种性能——当应用已经针对苹果芯片进行了适当的优化时，这种速度的部分就来了。如果你想在你的M1 iMac、MacBook Air、MacBook Pro或Mac mini上获得最好的游戏体验，你需要知道哪些游戏是为了充分利用技术而设计的。

我们已经介绍了 [如何检查你Mac上的应用程序是否是M1优化的](https://lifehacker.com/the-difference-between-universal-and-intel-apps-and-wh-1847613701) 。我们还介绍了 [如何检查热门应用是否是为苹果芯片](https://lifehacker.com/use-this-website-to-see-what-software-runs-on-m1-macs-1845750628) 设计的。这一次，重点是真正的游戏。这并不是Mac真正出名的地方，但如果苹果电脑是你唯一的硬件，你可能会想知道如何最好地在上面玩游戏。

## 如何找出哪些游戏是M1本土的

输入 [AppleGamingWiki](https://www.applegamingwiki.com/wiki/M1_compatible_games_master_list) 。这个方便的wiki汇编了所有能在你的Mac上运行的游戏。它列出了从*模拟人生3* 到*巫师3* 以及从*古墓丽影*到 *Terraria* 的标题。在本文撰写之时，列出了792款游戏，其中678款至少可以在你的M1 Mac电脑上玩。

但是除了“可玩”，AppleGamingWiki还分析了每款游戏在M1的表现。类别包括“完美”、“可播放”、“运行”、“菜单”(只有菜单可以运行)和“不可播放”“完美”是最理想的，显然是，而“可玩”应该给你足够的性能来享受游戏。

这里列出的不仅仅是M1的游戏，因为只有32款游戏是M1本土的。如果这听起来不是很多，那么，它不是。正如我们所说，Mac真的不是一个游戏平台。随着苹果继续开发苹果芯片，希望这种情况会有所改变，但是，现在，M1本土的头衔仍然是稀有动物。

在此期间，AppleGamingWiki的列表分解了为英特尔芯片设计的游戏的性能，这些游戏需要通过Mac的Rosetta 2仿真来运行。您还会看到这里列出的可以通过macOS Big Sur玩的iOS和iPadOS游戏，以及通过CrossOver或Parallels软件运行的任何Windows游戏。你应该能找到理论上可以在你的M1 Mac上运行的任何游戏，事实上，以及关于它的可玩性的信息。

默认情况下，网站会向你展示M1上运行的游戏列表，但你可以根据自己的喜好过滤这个列表。在“所有列表”下，从提供的类别中选择来查看这些游戏。例如，你可以点击“苹果街机游戏列表”来查看所有可以在你的M1机器上运行的*六个*苹果街机游戏。
