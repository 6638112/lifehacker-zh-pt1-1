# 为什么你需要尽快更新谷歌浏览器...又

> 原文：<https://lifehacker.com/why-you-need-to-update-google-chrome-as-soon-as-you-can-1848798753>

如果你认为读完这个标题后你有似曾相识的感觉，你没有:谷歌报告了另一个影响 Chrome 的零日漏洞，并延伸到其所有用户。幸运的是，现在有一个补丁:谷歌 [于 4 月 14 日周四发布了一份安全报告](https://chromereleases.googleblog.com/2022/04/stable-channel-update-for-desktop_14.html) ，称该公司已将 Chrome 更新至新版本 100.0.4896.127，以解决这一新发现的缺陷。



## 谷歌 Chrome 浏览器的最新安全漏洞是什么？

该漏洞被识别为 CVE-2022-1364，是 V8 JavaScript 引擎中的一个类型混淆漏洞。当一段代码在使用对象之前没有检查对象的类型时，就会出现这个问题。通常，这种类型的混淆只会使浏览器崩溃，但是一旦发现，坏人就可以利用这个漏洞。4 月 13 日星期三，谷歌威胁分析小组的克莱门特·勒辛纳报告了此事，这意味着谷歌在 24 小时内修补了该问题。

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1514719656619114506&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1514719656619114506&autosize=1) 

不幸的是，对于整个 Chrome 社区来说，谷歌证实了 CVE-2022-1364 的这种漏洞存在于野外。这意味着在某个地方，有人知道这个漏洞，并想出了如何利用它来对付其他人。当零日漏洞存在可用漏洞时，开发人员必须尽快修补。

## Chrome 为什么还没打补丁？

虽然补丁已经完成，但谷歌目前还没有为所有 Chrome 用户推出。据该公司称，将在未来几天或几周内推出，这意味着你可能在一段时间内看不到它。但是，由于情况的严重性，我们建议经常检查更新，直到它在您的浏览器上可用。

要进行检查，点击浏览器窗口右上角的三个点，选择“**帮助**，然后选择“**关于谷歌浏览器**”给 Chrome 一点时间寻找新的更新。如果有，你可以在这里看到。一旦安装了更新，Chrome 将重新启动，防止 CVE-2022-1364。
