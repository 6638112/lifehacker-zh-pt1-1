# 你需要这些几乎不可能被杀死的蔓生植物

> 原文：<https://lifehacker.com/you-need-these-trailing-plants-that-are-almost-impossib-1848520024>

如果你曾经浏览过一个专注于植物的 Instagram 账户，并想知道这个人的家怎么可能看起来如此绿色和茂盛，可能有几种不同的解释。愤世嫉俗的人有财力在他们的空间里种上无数的植物——可能还会付钱给其他人来照顾它们——或者树叶实际上是假的。



但是有另一种方法可以达到这种几乎杂草丛生的效果——用植物卷须从书架两侧垂下:用蔓生植物。幸运的是，有许多种类很容易照顾，几乎不可能被杀死。这里有一些关于蔓生植物的知识，包括对你自己家的一些建议。

## 什么是蔓生植物？

即使你不熟悉这个术语，你也能想象出蔓生植物的样子。它们有多个茎，可以长到令人印象深刻的长度，并且 [不会沿着茎的节点](https://www.provenwinners.com/learn/dictionary-gardening-terms) 生根。你会经常看到蔓生植物生长在吊篮里，它们的茎从旁边流过。

另一个受欢迎的选择是将它们种植在花盆中，并将其放置在一件高大的家具上——茎在生长时可以垂下——或放置在一个长而平的水平面上(如地幔)。

## 易于护理的蔓生植物

这里有几个室内蔓生植物的例子，它们很容易存活:

*   **银色或缎面的坑洞**:不仅拖尾，还有可爱的心形叶子。它 [喜欢明亮、间接的光线，](https://www.house-plant-hobbyist.com/blog/2019/3/20/top-10-easiest-trailing-plants) 却能忍受中等的光线。
*   英国常春藤:它不仅仅可以覆盖东海岸大学校园里的旧砖房。室内生长时，常春藤需要 [明亮、间接的阳光](https://gardenerspath.com/plants/houseplants/english-ivy-indoors/#How-to-Grow) 。
*   心叶喜林芋:和其他植物一样，它在明亮、间接的阳光下生长得最好。但与其他不同的是，它也能在光线较弱的房间中生存。
*   Hoya curtisii :长着银色铲形叶子的茎干真正开始长成长长的藤蔓可能需要比预期更长的时间，但这是值得的。这个需要湿度和一个每天至少有几个小时明亮光线的地方。

*   **串龟**:又名，这种半肉质植物喜欢高湿度，只需要中等光照。
