# 这是在 Mac 上合并 pdf 的最快方式

> 原文：<https://lifehacker.com/this-is-the-fastest-way-to-merge-pdfs-on-a-mac-1848620856>

曾经想要将几个不同的 PDF 文件合并成一个文档吗？你可能已经谷歌了第三方应用程序或者在网上找到了这样做的网站，但是你 r Mac 实际上有一个内置工具允许你即时合并 PDF 。这是内置在 Finder 中的，，它已经隐藏在众目睽睽之下有一段时间了。



## 如何在 Mac 上合并 pdf

第一步是将这些 pdf 文件收集到 Mac 上的一个文件夹中。一旦你这样做了，最重要的一步就是**按照正确的顺序选择你的文件**。您首先选择的文件将出现在合并后的 PDF 的第一页上；类似地，您在此之后选择的每个文件都将在您将要创建的单个 PDF 文档中的相应位置结束。

您可以按住 Command 键并按照您希望在最终文档中看到的顺序选择所有 PDF 文件。当你选择完你的文件后，右击任何一个文件并选择**快速动作>创建 PDF** 。几秒钟后，一个合并的 PDF 文件将出现在同一文件夹中。想改名就改名。

一旦创建了合并的 PDF，请确保打开它并检查所有页面的顺序是否正确。如果是，那么你做的一切都是对的。如果没有，您可以随时返回 PDF 文件夹，在重复过程之前，按正确的顺序选择文件。

## 不过，你不能合并受密码保护的 PDF 文档

值得注意的是，这个过程只适用于未加密的 PDF 文档。如果您有一堆受密码保护的 PDF 文件，您必须先删除密码，然后保存这些文档的副本。完成后，你可以使用这个技巧来合并你的 PDF 文件。

如果您想要减少点击次数，您需要使用在 Finder 中合并 pdf，您可以切换到分栏视图。打开 Finder，按下 **Command + 3** 切换到分栏显示。您现在可以选择所有要合并的 PDF 文件，并在 Finder 的右窗格中点击**创建 PDF 按钮**。
