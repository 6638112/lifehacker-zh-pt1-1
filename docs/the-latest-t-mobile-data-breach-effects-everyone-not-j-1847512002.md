# 最新的T-Mobile数据泄露事件影响到所有人，而不仅仅是客户

> 原文:[https://life hacker . com/the-latest-t-mobile-data-breach-effects-every one-not-j-1847512002](https://lifehacker.com/the-latest-t-mobile-data-breach-effects-everyone-not-j-1847512002)

T-Mobile [证实了一个安全漏洞](https://www.t-mobile.com/news/network/additional-information-regarding-2021-cyberattack-investigation) 泄露了超过4800万当前、以前和未来用户的信息。T-Mobile表示，它关闭了黑客用来访问文件的服务器漏洞，但泄露的信息现在在网上出售。

Watch

据出售数据库的黑客称，该数据库包括780多万后付费用户、85万预付费客户和4000多万“过去或潜在”客户的姓名、出生日期、社会保险号和驾照号，以及780万后付费用户中一些人的帐户pin。

从“好的一面”来看，T-Mobile表示没有财务或账单信息被泄露。然而，所有泄露的信息都是高度敏感的，如果您的信息被盗，很容易帮助他人窃取您的身份。

T-Mobile将联系受影响的人员，指导他们更改您帐户的PIN并更新您的安全设置。受影响的用户还将获得两年的McAfee ID盗窃保护服务。

然而，即使你没有T-Mobile的消息，你的信息仍然有可能被泄露。卖家声称，数据库中有超过1亿名当前和以前客户的记录——包括那些只是申请了T-Mobile服务但从未开设账户的人。这意味着比T-Mobile已经确认的4800万用户更多的用户处于危险之中。

所有活跃的T-Mobile客户应该立即更改他们的密码和帐户pin，任何过去使用过该运营商，甚至申请过潜在服务的人都应该调整他们的一般数据安全。以下是一些建议:

*   更新与您的T-Mobile帐户使用相同密码的任何其他帐户，并请[*停止重复使用密码*](https://lifehacker.com/why-even-your-bullshit-accounts-deserve-strong-password-1826307919)*。一个密码管理器可以帮助你记录你所有的登录信息。*
*   *[注册双因素登录](https://lifehacker.com/no-one-knows-about-two-factor-authentication-and-privat-1838913065) ，在你使用的每个账户上登录提醒。*
*   *从您的T-Mobile个人资料和其他帐户中删除尽可能多的个人身份信息。*
*   *监控您的银行账户和其他金融服务以及PayPal或Venmo等支付方式，以发现可疑活动。报告任何你不认识的人。*
*   *当心网络钓鱼攻击、恶意软件木马和 [其他诈骗](https://lifehacker.com/the-complete-guide-to-avoiding-online-scams-for-your-l-5420356) 。黑客和骗子会向被窃取的电子邮件地址、电话号码和社交媒体账户发送含有恶意链接或虚假优惠的误导性信息。*
