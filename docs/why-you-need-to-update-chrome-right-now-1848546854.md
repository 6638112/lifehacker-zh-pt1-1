# 为什么你现在需要更新Chrome

> 原文:[https://life hacker . com/why-you-need-to-update-chrome-right-now-1848546854](https://lifehacker.com/why-you-need-to-update-chrome-right-now-1848546854)

安全更新很重要，你不应该忽视它们。然而，尽管Chrome等应用程序和浏览器会提醒你该更新了，但你也可能幸福地意识不到甚至有一个可用的应用程序和浏览器，这让你面临安全风险。这恰好是其中的一次。

Watch

2月14日星期一， [谷歌](https://chromereleases.googleblog.com/2022/02/stable-channel-update-for-desktop_14.html) 宣布了Chrome的新更新版本98.0.4758.102，该版本解决了之前版本浏览器的11个安全风险。虽然堵塞安全漏洞总是好建议，但这次更新之所以重要还有一个额外的原因: O 在这11个安全漏洞中，谷歌承认有一个漏洞“存在于野外”从本质上讲，谷歌以外的人知道这个安全缺陷，如果他们还没有这样做，就可以利用它。

这个特别的漏洞是[$NA][ [1296150](https://crbug.com/1296150) ]高CVE-2022-0609:在动画中免费后使用。 [一个释放后使用的漏洞](https://encyclopedia.kaspersky.com/glossary/use-after-free/) ，或者UAF，是一个关于程序动态内存的漏洞。当一个程序清除了一个内存位置，但没有清除指向该内存的指针时，就会发生这种情况，从而允许一个坏的参与者攻击该特定的程序。

谷歌没有透露这一漏洞是否被利用，或者被谁利用。然而，它的存在足以让你立即更新Mac或PC上的Chrome浏览器。

要更新，点击Chrome窗口右上角的三个点，然后选择**帮助>关于谷歌Chrome** 。允许Chrome检查更新。如果98.0.4758.102可用，您将在此处看到它的报告。只需点击“重新启动”，让更新生效。

Chrome并不是最近唯一存在严重安全缺陷的浏览器。苹果最近发布了两次Safari的更新； [最新更新](https://lifehacker.com/you-already-need-to-update-safari-again-1848522230) 修复了一个允许黑客运行恶意代码的WebKit安全漏洞， [而之前的更新](https://lifehacker.com/update-safari-immediately-1848426551) 修补了一个严重的JavaScript API漏洞，允许不良行为者从浏览器中其他打开的标签中提取个人信息。
