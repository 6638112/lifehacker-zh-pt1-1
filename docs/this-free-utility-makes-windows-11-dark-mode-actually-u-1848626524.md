# 这个免费的工具使Windows 11黑暗模式实际可用

> 原文:[https://life hacker . com/this-free-utility-makes-windows-11-dark-mode-actually-u-1848626524](https://lifehacker.com/this-free-utility-makes-windows-11-dark-mode-actually-u-1848626524)

Windows 11附带了一个“黑暗模式，”但不是一个好的。它更像是一个黑暗的*主题*，缺少你从其他平台期待的功能。黑暗模式应该在日落时自动启动，或者在你选择的小时启动。它应该很容易打开和关闭，最好从一个快速设置面板。在Windows 11 中，这些选项中的都不存在，这就是一个叫做自动黑暗模式的免费实用程序出现的地方。

自动黑暗模式使Windows 11的黑暗模式实际上起作用，甚至包括一个全局热键功能，使启用黑暗模式只是一个 [键盘快捷键远离](https://lifehacker.com/the-new-windows-11-keyboard-shortcuts-you-should-actual-1848620565?rev=1646758800932) 。这是一个可定制的实用程序，非常值得下载。你可以在 [微软商店、](https://apps.microsoft.com/store/detail/auto-dark-mode/XP8JK4HZBVF435) 或者从开发者的 [GitHub页面](https://github.com/AutoDarkMode/Windows-Auto-Night-Mode) 找到。在安装完，之后，开始你的定制——你可以设置它自动启用，切换你的桌面壁纸，等等。

## 如何使用键盘快捷键启动Windows 11黑暗模式

热键功能是自动黑暗模式应用程序中最好的功能之一。它让你设置一个系统范围的键盘快捷键，在按下键时启用亮模式或暗模式。

转到“**切换模式**部分，确保“**启用系统范围热键**功能被禁用。(您只能在禁用该功能时编辑或创建热键。) C 点击“**强制光照模式热键**”和“**强制黑暗模式热键**”选项下方的方框，将您的全局键盘快捷键设置为您喜欢的。最后，点击“**启用全系统热键**功能旁边的切换按钮。

## 如何从任务栏启动Windows 11黑暗模式

不喜欢键盘快捷键？你可以从任务栏快速启用黑暗模式。进入系统托盘，右键点击自动黑暗模式图标，选择**强制黑暗主题**选项切换到黑暗模式，强制光明模式回到光明面。
