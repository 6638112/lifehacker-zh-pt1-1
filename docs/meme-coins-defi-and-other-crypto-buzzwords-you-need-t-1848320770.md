#### **区块链**

虽然不新，但如果你想有机会理解 crypto，你需要知道它的意思。区块链是一个分散的加密分类账，记录了全球所有的加密货币交易。更具体地说，它是一个共享数据库，其中的数据块(如账户余额和交易)按时间顺序链接在一起。在区块链上，数字信息被永久记录且不可编辑。(哎呀。)



#### **亚铁元素**

如果说 [比特币](https://www.nerdwallet.com/article/investing/cryptocurrency-7-things-to-know) 是 crypto 的祖师爷，那么 altcoin 就是它所有的“替代币”或者替代版本。最突出的包括:以太坊、索拉纳和 Dogecoin。

#### **Stablecoins**

stablecoins 是 altcoin 的一个子集，它提供了更稳定的加密优势。(加密价格的波动是出了名的——众所周知，比特币的股价会在一夜之间下跌 9000 美元。)旨在拥有相对稳定的价格，稳定硬币(如系绳、美元硬币和币安)将其价值与现实世界的资产挂钩，如黄金或其他货币等商品，如美元或欧元。

#### **梅角**

如果十年前有人告诉你，受模因和网络笑话(如“doge”是“dog”的错误拼写，最初出现在一部动画网络系列中，后来在 Reddit 上流行开来)启发的新货币形式可以在股票市场上交易，你会相信吗？是啊，我也是。然而， [我们到了](https://www.bloomberg.com/news/articles/2022-01-06/meme-coins-return-to-earth-as-gloom-overtakes-crypto-fanatics) 。

#### **Dogecoin**

(DOHJ-coin):这是上述迷因币最突出的例子。这种加密货币于 2013 年首次铸造，作为一个笑话来讽刺 crypto 的广泛流行和(当时)似乎毫无根据。在被埃隆·马斯克吹捧之后，它获得了广泛的吸引力。



#### **Web 3.0/Web3**

本质上是互联网的第三次迭代。如果说 Web 1.0 见证了最初的静态网站、无广告和拨号接入的“万维网”的诞生，Web 2.0 引领了科技巨头(亚马逊、谷歌)和用户生成内容(博客、播客和社交媒体)的崛起，那么 Web 3.0 则设想 Web 将演变为一个建立在点对点连接 上的 [去中心化网络，理论上，个人将保留其数字财产的更多所有权。(Web3 的主要功能包括 3D 图形、人工智能、虚拟现实和语义元数据。)](https://lifehacker.com/what-is-web3-and-why-should-you-care-1848133134)

#### **加密钱包**

存放您的私人“密钥”或密码的地方，这些密钥或密码可让您访问区块链。它们可以是“热钱包”，意思是虚拟的(如移动应用程序 [、比特币基地钱包](https://www.coinbase.com/wallet) )，或者是“冷钱包”，意思是有形的硬件，如。无论哪种方式，如果你想交易密码，这是一个必须具备的。

#### **DeFi**

有时被称为“金融的狂野西部”，分散金融是一个基本上不受监管的自动化金融工具行业，它允许加密用户通过衍生品进行借贷、投机，并直接通过区块链进行加密交易，而不需要经纪人、银行或其他中间人。

#### **NFTs**

不可替代的代币允许买家拥有可收藏的数字艺术、音乐、或游戏，以及其他比美国银行股票有趣得多的资产。把它想象成时髦的密码；这不仅仅是为了金融科技极客，也是为了那些可能喜欢花 20.8 万美元在史诗般的勒布朗詹姆斯扣篮——上的投资者。
