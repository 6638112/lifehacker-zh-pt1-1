# 如何正确(几乎)每次使用复合词

> 原文：<https://lifehacker.com/how-to-use-compound-words-correctly-almost-every-time-1846738469>

英语中到处都是复合词——小词的无限组合形成新的更大的词。然而，它们在书面语中是如何组合在一起的会有所不同，你可能会停下来想知道:这是一个长期的问题吗？这是因为有三种不同类型的复合词，一个复合词应该写成一个词、两个词还是用连字符连接并不总是显而易见的。



### 什么是复合物？

每当我们处理语法问题时，我认为最好从基础开始——以防你现在忘记了小学的英语课。所以，当你把两个或两个以上的单词连在一起组成一个有自己意思的新单词时，你就形成了一个复合词*。床+房间=卧室。后+生=来世。*

*然而，并不是所有的复合词都融合成一个词。复合词有三类:*

*   ***开复合词:**这些复合词保留词与词之间的空格。(冰淇淋，餐厅，咖啡杯，校车，甜食。)*
*   *闭合复合词:这些复合词组合成一个单词，没有一个空格。(舞厅，亲子，棒球，一生，水母。)*
*   *用连字符连接的复合词:单词用连字符连接。(岳父，知名，一流，旋转木马，双止。)*

### *永久'对''临时'化合物*

*已经广泛使用多年的普通复合词被认为是永久复合词，经常可以在字典中找到。例如，关于“卧室”应该是一个词还是两个词的争论很少。但是当新的、临时的复合词开始使用时，人们可能会混淆开放、封闭还是连字符是首选的样式。这通常会随着时间的推移而得到解决；同时，只要读者能够理解你的意思，任何使用都是公平的。*

> *对于许多术语来说，在开放的、带连字符的和封闭的选项中自由选择通常是完全可以接受的，即使这个术语已经在英语中使用了很长一段时间(例如，*生活方式*、*生活方式*或*生活方式*)。虽然复合词的最终风格可能仅仅由编辑偏好决定，但有一种模式经常适用于新复合词在英语中变得根深蒂固的情况。*

*在这些情况下，如果你有一个强烈的偏好，不符合一个更广泛使用的变化，这不是一个大问题。*

### *“规则”的时间到了*

*我将“规则”放在引号中，因为我们在谈论英语，我们都明白英语语法规则更像是松散的指南。但是一个宽松的指导方针总比没有好，所以如果你对使用哪种类型的复合词感到困惑(你不想去查，或者你的研究没有结论， [，你通常可以假设](https://www.merriam-webster.com/help/faq-compound-words) :*

*   *复合名词是一个单词*
*   *复合动词是两个单词*
*   *复合词和目标词用连字符连接*

*同样，这些所谓的规则也有各种各样的例外。但是其中最明显的是——作为名词，，“客厅”是两个单词，例如——不太可能是你无论如何都要去困惑的。在这些情况下，只需使用最常用的变体即可。*
