## 白炽灯和LED灯泡的区别

在我们进行成本比较之前，让我们来看看LED灯泡和白炽灯泡的区别。

自19世纪 以来，白炽灯泡就已经出现在 [附近，直到最近，它们还是镇上唯一的展示品。你知道那种类型:当](https://www.energy.gov/articles/history-light-bulb#:~:text=Incandescent%20Bulbs%20Light%20the%20Way,possible%20with%20the%20arc%20lamp.) [电流通过玻璃灯泡内的钨丝](https://www.bulbs.com/learning/incandescent.aspx) 时，加热到足以发光。

LED(发光二极管)是固态照明——这意味着用于产生光的材料被包裹在固体材料中。因此，发光二极管不会像白炽灯泡那样发热。虽然早期版本的LED灯泡产生寒冷、刺目的光，使你的客厅像超市的牛奶盒，但这些年来，它们变得更柔和、更温暖(在颜色上)。

## **白炽灯泡与LED灯泡:成本比较**

每个灯泡大约5到7美元，LED灯最初比白炽灯泡(每个灯泡大约2到3美元)更像是投资 。但是，当你考虑到能源成本时，从长远来看，它们是更好的价值。根据美国能源部的数据，家用LED灯泡比白炽灯至少节能75%,寿命延长25倍。

那么，随着时间的推移，成本是如何分解的呢？赫顿电力与照明 的团队为我们做了计算。这些数字来自2019年，并使用弗吉尼亚州当年的平均电力成本(每千瓦时0.11美元)进行计算，但你会得到大致的想法。

比较是使用两个亮度相似的灯泡进行的:一个60W白炽灯和一个12W LED:

*   60W **白炽灯泡**每1000小时消耗60千瓦时(kWh)的电能。按照每千瓦时0.11美元的价格计算，白炽灯泡工作1000小时将花费**6.60美元**。

*   12W **LED灯泡**每1000小时使用12千瓦时的电能。按照每千瓦时0.11美元的价格计算，LED灯泡工作1000小时将花费**1.32美元**。
