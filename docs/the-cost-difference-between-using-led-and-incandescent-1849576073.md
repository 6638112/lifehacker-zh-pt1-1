# 使用 LED 和白炽灯泡的成本差异

> 原文：<https://lifehacker.com/the-cost-difference-between-using-led-and-incandescent-1849576073>

灯泡是那种你不会真正注意到的东西，直到有一个坏了，你不得不去更换它。那么，当那个时刻到来时，你会用白炽灯泡，还是更新的 LED 灯泡来代替它呢？下面介绍一下这两者在成本上的区别。

## 白炽灯和 LED 灯泡的区别

在我们进行成本比较之前，让我们来看看 LED 灯泡和白炽灯泡的区别。

自 19 世纪 以来，白炽灯泡就已经出现在 [附近，直到最近，它们还是镇上唯一的展示品。你知道那种类型:当](https://www.energy.gov/articles/history-light-bulb#:~:text=Incandescent%20Bulbs%20Light%20the%20Way,possible%20with%20the%20arc%20lamp.) [电流通过玻璃灯泡内的钨丝](https://www.bulbs.com/learning/incandescent.aspx) 时，加热到足以发光。

LED(发光二极管)是固态照明——这意味着用于产生光的材料被包裹在固体材料中。因此，发光二极管不会像白炽灯泡那样发热。虽然早期版本的 LED 灯泡产生寒冷、刺目的光，使你的客厅像超市的牛奶盒，但这些年来，它们变得更柔和、更温暖(在颜色上)。

## **白炽灯泡与 LED 灯泡:成本比较**

每个灯泡大约 5 到 7 美元，LED 灯最初比白炽灯泡(每个灯泡大约 2 到 3 美元)更像是投资 。但是，当你考虑到能源成本时，从长远来看，它们是更好的价值。根据美国能源部的数据，家用 LED 灯泡比白炽灯至少节能 75%,寿命延长 25 倍。

那么，随着时间的推移，成本是如何分解的呢？赫顿电力与照明 的团队为我们做了计算。这些数字来自 2019 年，并使用弗吉尼亚州当年的平均电力成本(每千瓦时 0.11 美元)进行计算，但你会得到大致的想法。

比较是使用两个亮度相似的灯泡进行的:一个 60W 白炽灯和一个 12W LED:

*   60W **白炽灯泡**每 1000 小时消耗 60 千瓦时(kWh)的电能。按照每千瓦时 0.11 美元的价格计算，白炽灯泡工作 1000 小时将花费**6.60 美元**。

*   12W **LED 灯泡**每 1000 小时使用 12 千瓦时的电能。按照每千瓦时 0.11 美元的价格计算，LED 灯泡工作 1000 小时将花费**1.32 美元**。
