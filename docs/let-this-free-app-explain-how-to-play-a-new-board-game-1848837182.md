# 让这个免费的应用程序来解释如何玩一个新的棋盘游戏(所以你不必)

> 原文：<https://lifehacker.com/let-this-free-app-explain-how-to-play-a-new-board-game-1848837182>

学习或教授一个新的棋盘游戏既有挑战性又让人抓狂。要么你是一个试图向冷漠的人群解释规则的人，要么你正在被“教导”，并被期望从单调的讲师那里学到每一分钟的细节。如果你厌倦了每个游戏之夜都以同样的方式开始，那么对于桌游老师和学生来说，有一个更好的选择。

Watch

[Dized](https://dized.com/) 是一个免费的应用程序，旨在通过为每个标题专门创建的迷人教程来引导玩家完成新的棋盘游戏。(程序 [背后的开发人员启动了一个 Kickstarter](https://dized.com/) 来实现他们的愿景，尽管我第一次看到它要感谢 gamesbymondo 的 [。)在与支持者取得一些成功后，这种体验在](https://vm.tiktok.com/TTPd4WYtAB/) [iOS](https://itunes.apple.com/us/app/dized/id1041701520?ls=1&mt=8) 、 [Android](https://dized.com/android/) 和[web](https://dized.com/)上直播，尽管容量有限。我们去看看。

当您第一次选择教程时，您会看到一个初始的摘要屏幕，其中包括一个游戏的预计持续时间、可以同时玩的玩家数量以及相应的年龄分级。你还会看到游戏的概要，并可以选择查看教程或规则。

让我们把重点放在教程上，因为这是这款应用的真正卖点。下载教程后，你会看到一个带有游戏标志和插图的闪屏，并伴有音乐。一个引人入胜的画外音介绍了你将要玩的游戏，并询问有多少玩家参与，因此它可以根据你的特定设置定制演练。

从这里开始，你的棋盘游戏和它的所有组成部分(卡片、棋盘、棋子、纸张等)。)出现在屏幕上，带有相应的 voiceover 描述和说明。它的节奏很好:在一个人点击屏幕继续前进之前，指令会暂停，让玩家有时间接受并获取他们需要的任何材料。

教程制作得很好，当然比听一个人读规则书而每个人都半专注更有吸引力。在没有真正玩过一个新游戏之前，我往往很难学会它，这些动画演示有助于填补这一空白。我可以肯定地看到自己使用 Dized 更快地掌握复杂游戏的规则。

例如，点击*小型史诗海盗*的教程，会发现它确实是一款*非常*复杂的游戏。如果我只根据规则书来玩这个游戏，我会一塌糊涂，很可能会毁了游戏之夜。
