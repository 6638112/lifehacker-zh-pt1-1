# 如何利用白日梦来消除无聊

> 原文：<https://lifehacker.com/how-to-use-daydreams-to-ward-off-boredom-1846422782>

还记得小时候在教堂或学校集会上，让你的思绪游走，然后突然间，无聊的活动结束了吗？白日做梦把我们带离了我们实际所在的地方，基本上把我们带到了其他任何地方，并使时间过得(或至少感觉上)更快。

但作为成年人，当我们的大脑得到休息时(甚至没有休息时)，我们的想法往往会变得消极——担心财务、社会不公、家庭问题、经历全球疫情——只要你能想到的。正如佛罗里达大学心理学教授艾琳·韦斯特盖特 博士解释的那样，白日做梦“是我们尚未开发的认知工具包的一部分，这有点可悲。”以下是如何重新学习如何做白日梦作为一种抵御无聊的策略。

### 为什么做白日梦这么难？

白日做梦——或者为了快乐和/或逃避现实而刻意思考——是一项远比大多数人意识到的要求更高的技能。“你必须是精神表演的演员、导演、编剧和观众，”韦斯特盖特 [在一份关于她最近在佛罗里达大学的研究的声明](https://news.ufl.edu/2021/03/daydreaming/) 中说。"即使看起来你什么都没做，这也是认知上的负担."

另外，我们并不直觉地知道如何思考愉快的想法。“我们相当无知，”她补充道。“我们似乎不知道该考虑些什么来获得积极的体验。”

### 如何更好地做白日梦

在最近发表在《情感 杂志上的研究中(与弗吉尼亚大学的蒂莫西·威尔逊、尼古拉斯·巴特里克和雷米·弗勒以及哈佛大学的丹尼尔·吉尔伯特合著)，韦斯特盖特和她的同事们发现，白日做梦是“无聊的解药”，但人们需要一点帮助才能实现这一点。

下面是如何磨练 [技能，据](https://news.ufl.edu/2021/03/daydreaming/)研究者:

> 相信如果你的大脑充满了你会觉得愉快的话题，就有可能有一个好的体验。**“**一旦有了概念，这是我们所有人都能做到的事情。我们给 4 岁和 5 岁的孩子这些指示，对他们来说是有意义的。”*   That is, "It's hard for everyone.
