# 是时候用另一个安全补丁更新你的谷歌Pixel了(2021年5月版)

> 原文:[https://life hacker . com/time-to-update-your-Google-pixel-with-other-security-1846821885](https://lifehacker.com/time-to-update-your-google-pixel-with-another-security-1846821885)

2021年5月的像素安全更新已经准备好了，你不应该浪费时间在你的设备上安装它。

Watch

本月的补丁修复了Android 8.1、9、10和11版本中发现的42个漏洞，这些漏洞影响了几家制造商的通用组件和Android软件的各个部分，包括Android内核(控制手机的操作系统软件的核心部分)。

许多被评为“高级”漏洞，黑客可能会窃取或暴露您的信息，控制某些设备功能，或远程执行恶意代码。完整的Android安全公告注释 [在这里](https://source.android.com/security/bulletin/2021-05-01) 可以找到。

至于 [Pixel特有的问题](https://source.android.com/security/bulletin/pixel/2021-05-01) ，补丁修复了影响Pixel设备中高通组件和Android OS内核的七个中等严重程度的漏洞。

谷歌没有提供关于已披露漏洞的更多细节，也没有提供补丁如何修复这些漏洞，但鼓励所有用户在有更新时安装更新，以防止恶意行为者利用安全漏洞。

该更新现适用于Pixel 3、3a和4手机(包括XL型号)、Pixel 4a和4a 5G以及Pixel 5。你可以在Android settings应用程序中查看5月份的安全补丁是否可用，或者它已经安装在你的设备上。

1.  进入**设置>系统>高级>系统更新。**
2.  向下滚动并检查**“安全补丁级别”下的日期如果是2021年5月5日或之后，那么你是最新的。**
3.  如果安全补丁级别早于2021-05-05，进入**设置>安全。**点击**“检查安全更新”**如果有，请按照屏幕上的步骤下载。

未来几周，非Pixel Android手机也将作为OTA补丁接收Android安全更新。确切的更新时间取决于您的设备制造商和手机服务运营商。
