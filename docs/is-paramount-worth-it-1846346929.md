# 派拉蒙+值得吗？

> 原文:[https://lifehacker.com/is-paramount-worth-it-1846346929](https://lifehacker.com/is-paramount-worth-it-1846346929)

为了不被迪士尼和华纳兄弟超越， [维亚康姆最近宣布了自己的流媒体服务](https://www.cbssports.com/general/news/viacomcbs-announces-paramount-plus-rebranding-live-sports-news-and-entertainment-coming-in-march-2021/) ，创造性地命名为派拉蒙+，将于3月4日推出。与Hulu和网飞这样的老牌公司相比，数量不断增加的新流媒体——迪士尼+、探索+、苹果电视+和你能想到的所有plus-es——正在变得势不可挡。你可能想知道这是否值得。以下是派拉蒙+所提供的内容，以帮助您决定加入派拉蒙+是否适合您。

### 这是什么？

派拉蒙+并不像看起来那么新。它将有效取代现有的CBS-All Access流媒体平台，该平台于2014年推出。在2019年哥伦比亚广播公司和派拉蒙母公司维亚康姆 合并之后，派拉蒙+旨在为观众提供更广泛的内容。重新推出的服务将覆盖更多的维亚康姆品牌，包括BET，Comedy Central，Smithsonian Channel和Nickelodeon，以及派拉蒙图片库的各种电影。现场体育和新闻也在其中。

这似乎是一个相对聪明的流媒体服务更名，该服务以而闻名(除非你是《星际迷航》的粉丝)。新的外观和内容的提升可能会让新用户值得一试。

### 派拉蒙+如何工作

如果你已经拥有CBS All Access，你的账户会在3月4日自动转移到派拉蒙+(与从HBO/HBO Go/HBO Now到HBO Max的灾难性的过渡相去甚远。)当前CBS的所有访问订户将使用他们现有的登录来访问新品牌的平台，无需额外设置。

如果你是该平台的新用户，你可以 [注册](https://www.cbs.com/cbs-all-access/plan/) 获得CBS All Access now为期一周的免费试用。此后的费用是5.99美元/月的有限广告计划，或9.99美元/月的无广告观看。订户还可以提前购买59.99美元的一年服务(99.99美元的无商业广告服务)，根据您选择的套餐，可以节省大约10-20美元。

目前，派拉蒙+(通过即将成为前CBS All Access)正在为从现在起到更名服务推出期间的注册提供50%的折扣。在3月3日晚上11:59加入CBS All Access，并使用折扣代码PARAMOUNTPLUS预付一整年的费用，您将享受一年的服务，每月仅需2.50美元。但是一年的CBS全频道真的值得吗？
