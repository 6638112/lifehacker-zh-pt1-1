# 如何避免对斋月一无所知

> 原文：<https://lifehacker.com/how-to-not-be-totally-clueless-about-ramadan-1795654296>

斋月从昨天开始，所以本周你可能开始注意到你的穆斯林同事和朋友不吃午饭了。斋戒月是为了纪念真主向穆罕默德揭示古兰经，信徒们利用这段时间练习反省和自我控制。所以，如果你不是穆斯林，你怎么能更体谅那些穆斯林呢？这里有一些来自斋月的人们的建议，关于他们希望别人知道的事情。



首先， [斋月](https://en.wikipedia.org/wiki/Ramadan) 的基本规则:在整个农历月期间，许多虔诚的穆斯林在日出和日落之间不吃不喝。这是一项宗教活动，在此期间，人们的目标是 [更接近上帝和他们的社区](http://www.islamforchristians.com/purpose-ramadan/) 。其中一部分包括白天禁食，晚上吃一顿庆祝餐。由于我们的生活如此关注食物，事情会变得尴尬。那么你应该怎么做呢？

“我不介意你在我面前吃东西。真的，写的是 [奈拉·克拉尼在厨房](http://www.thekitchn.com/8-things-i-wish-non-muslims-knew-about-ramadan-231626) 。如果你能避免，那很好:如果可以的话，把你的常规午餐会议移到一天中不同的时间。但不需要踮着脚。凯拉尼写道，“热情洋溢的道歉和夸张的食物隐藏是不必要的。”

**“对你的问题深思熟虑”，** [副](https://www.vice.com/en_us/article/how-to-talk-to-muslims-during-ramadan) 的莎拉·哈吉写道。好奇是很自然的，但是“连水都没有？”你可以谷歌一下，你的穆斯林朋友可能已经厌倦了解释。问别人为什么*不*禁食也是不合适的，因为那是个人隐私。例如，怀孕或生病的人可以免于遵守这一规定，这与你无关。也就是说，Hagi 很乐意和别人谈论斋月和她的信仰。这不是一个禁忌话题。

但是真的，连水都没有。这意味着中午的锻炼，甚至是大热天的户外散步，都会变得相当糟糕。禁食一天快结束时感到疲倦也是正常的——所以如果你正在制定计划，请记住这一点。“有一年夏天，我最好的朋友放了一条冷毛巾在她的冰箱里让我降温，而不是给我一杯水，”Hagi 写道。"这是别人为我做过的最善良、最体贴的事情之一。"

**“你不需要和我们一起斋戒，”**凯拉尼写道。这不是一场受益于团结的政治运动；这也不是一个饮食潮流。也就是说，你可能会被邀请和你的穆斯林朋友一起开斋。一天结束时的聚餐被称为，通常是一个社区庆祝活动。(亲提示:带一份素食开胃菜。)并随意祝大家一个 [斋月穆巴拉克](http://www.express.co.uk/news/world/809994/ramadan-2017-mubarak-kareem-quotes-happy-wishes-greeting-muslim-date-explained) 。
