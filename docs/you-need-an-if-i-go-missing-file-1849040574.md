# 你需要一份“如果我失踪了”的文件

> 原文：<https://lifehacker.com/you-need-an-if-i-go-missing-file-1849040574>

Statista 报道[2021 年有 521，705](https://www.statista.com/statistics/240401/number-of-missing-person-files-in-the-us-since-1990/#:~:text=In%202021%2C%20the%20number%20of,in%20the%20U.S.%20since%201990.) 失踪人口案件，这是美国自 1990 年以来的最低数字。对于一个人来说,失踪肯定是不寻常的，但是当它发生的时候，它肯定会给他们的朋友和家人带来创伤。

如果你失踪了，你可能会希望寻找你的人掌握尽可能多的关于你的信息，以帮助寻找。这是一个“如果我失踪了”的文件或文件夹开始发挥作用的时候——其中的信息将在更多的情况下有用，而不仅仅是你被绑架的可能性。



## 为什么你需要一份“如果我失踪了”的文件？

“如果我失踪了”文件是一个包含关键个人信息的文件，在紧急情况下可以用来帮助找到你(T1)，但它在其他情况下也很有用(T2)。这个概念是通过*犯罪迷*播客 推广开来的，主持人是两个痴迷于犯罪的女人，阿什利·弗劳尔斯和布里特。每周播客描述围绕真实犯罪故事的事实，并使用传统的讲故事形式来吸引听众。

《犯罪瘾君子》的主持人是 T2 强烈提倡在危机中通过帮助别人来帮助自己。换句话说，他们建议向你所爱的人提供有用的信息，以帮助他们在最坏的情况发生时找到或识别你。

但这并不是唯一有用的时候。假设你的配偶去世了，他们是这段关系中掌管你所有财务的人。在这种情况下，你需要简单的 [访问他们的用户名和密码](https://www.bdslawinc.com/blog/2019/06/spouses-need-to-access-usernames-and-passwords/) 来进入他们的银行账户。如果你手头有这些信息，这将比通过向银行提供死亡证明来获得访问权限要快得多。为你所爱的人提供这些宝贵的数据，会让他们在你去世后生活得更轻松。

## 如何创建“如果我失踪了”文件

鉴于不断变化的网络安全形势，我们不建议您创建包含您的敏感个人信息的数字文件。你最不希望的事情就是有人侵入你的设备，窃取你所有的密码和个人数据。

取而代之的是，创建一个物理文件夹并存储在你家的一个安全的地方，或者 [在银行的一个保险箱](https://www.investopedia.com/safe-deposit-box-what-to-store-and-not-store-in-yours-4589854#toc-what-to-store-in-a-safe-deposit-box) 里，在那里只有几个可信的人能够访问它。棘手的部分是在你更改密码和开设新账户时保持你的文件最新——你需要养成定期重访文件夹的习惯，也许是每年或每隔几年，以确保其中的所有内容仍然适用。

## “如果我失踪了”档案里应该写些什么

尽管你所包含的内容可能会因个人情况而异，但以下是一些需要考虑的要点:

*   你自己的近照，以及你的驾照、和护照的复印件
*   车辆信息，如牌照、品牌和型号、年份和颜色
*   健康信息，如血型、药物、和健康状况
*   重要账户的用户名和密码， [，包括使用双因素认证](https://www.techtarget.com/searchsecurity/definition/two-factor-authentication)【2FA】代码的访问方式
*   身体数据，如你的身高、年龄、头发颜色、眼睛颜色和出生日期
*   社交媒体档案信息
*   银行账户详细信息
*   你每日日程的细目表
*   你经常去的地方的列表
*   家人、朋友、同事、和其他重要人物的姓名及其联系信息
*   笔迹样本和指纹
*   允许某人在紧急情况下使用您的文件或文件夹来帮助找到您的书面许可

本质上，你包含的关于自己的信息越多，对在危机中试图帮助你的人就越有帮助。
