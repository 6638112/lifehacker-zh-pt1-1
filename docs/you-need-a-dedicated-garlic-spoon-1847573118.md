# 你需要一个专用的大蒜勺子

> 原文：<https://lifehacker.com/you-need-a-dedicated-garlic-spoon-1847573118>

除了我的鱼锅铲之外，几乎所有我喜欢的勺子和旋转器都是木制的。木制炊具是我最喜欢的纪念品，我去过的每个国家都收集一件。



毫无疑问，我最喜欢的勺子是我在法国南部捡到的这个小巧的橄榄木美人 。我经常使用它，这种几乎持续的使用给了勺子一种明显的香味。坦率地说:它闻起来有大蒜的味道，而且会一直有大蒜的味道，因为木头是多孔的。

这既不意外，也不是非常不受欢迎。香味是微弱的，但它是存在的，而且——很像木制砧板——一个芳香的勺子可以调味(或至少嗅出)更精致、不那么可口的食物，在那里你可能不想要大蒜。幸运的是，解决方法很简单:我只用这个勺子盛那些含有大蒜的菜(大多数都含有大蒜)和稍微有点大蒜味的菜。

这个想法不完全是我的。在看了梅丽莎·克拉克的厨房之旅 后，我开始在我的砧板上使用类似的策略。梅丽莎有专门的大蒜切菜板，很像我的勺子轮廓，起初看起来过于珍贵，但事实上，非常聪明:

> 当你在品尝新鲜水果的时候品尝甚至闻到大蒜的味道并不好，至少有一个砧板专门用来处理更难闻的东西可以保护你和你的家人免受这样的经历。

有一个专用的大蒜勺子意味着我再也不用担心把黄油和糖搅在一起，却发现它有昨晚意大利面晚餐的味道。(是的，使用非木制工具会消除这种区分的必要性，但木制勺子和 turners 非常好用，我不认为我会放弃它们。)
