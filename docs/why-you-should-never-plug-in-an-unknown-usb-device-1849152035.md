# 为什么永远不要插入未知的 USB 设备

> 原文：<https://lifehacker.com/why-you-should-never-plug-in-an-unknown-usb-device-1849152035>

让我们来设定一下场景:你在外面的世界里，做你正在做的事情，这时你偶然发现了一个废弃的 u 盘。里面会有什么？也许它只是某人工作中的电子表格——也许还有一些识别所有者的信息，允许你归还它。但也可能是政府机密。找出答案的唯一方法就是把它插到你的电脑上进行调查。不过，问题是:不要那样做。

当然，你找到的 USB 设备可能是完全无辜的，不知不觉地被和你走同一条路的人掉了。然而，它也可能是一个陷阱，旨在利用你的好奇心，当你决定将它插入你的个人电脑时，你会发现它是恶意软件。

## 受恶意软件感染的 USB 设备是一个真正的问题

虽然这听起来像电影里的东西，但人们真的会用恶意软件感染 USB 设备，并让不知情的受害者发现它们。目标有大有小，最引人注目的黑客攻击可能是 2010 年针对伊朗的: [一次这样的尝试用超级恶意软件](https://www.malwarebytes.com/stuxnet) 感染了伊朗的核设施，尽管整个系统与任何互联网通信都断开了。

在风险较低的情况下，这可能看起来像是一种相当迂回和随机的攻击手段。毕竟，钓鱼邮件和文本可以直接发送给马克，而 USB 设备必须先拿起来，然后插上电源才能工作。

事实证明，有人插入陌生 USB 的可能性非常高。 [一项研究](https://www.thesslstore.com/blog/usb-flash-drive-malware-how-it-works-how-to-protect-against-it/) 通过一个“大型大学校园”丢下近 300 个 USB 设备，发现 98%的设备被学生和工作人员捡起来，近一半的人决定将 USB 设备插入他们的电脑——第一次连接发生在研究启动后六分钟。总而言之，在这种情况下，黑客的投资可能会有回报。

这不是一个新问题。2008 年，美国计算机应急小组(US-CERT)[发布了一份关于感染了恶意软件的 USB 设备](https://www.cnet.com/news/privacy/usb-devices-spreading-viruses/) 的警告。在那之前，软盘也是以类似的方式使用。虽然我们可能已经远离物理存储而转向云，但 USB 设备仍然无处不在，足以构成这种威胁。

很难说这种威胁到底有多普遍，但是随着网络攻击的增加，安全总是比后悔好。避免将一个陌生的 USB 设备连接到你的个人电脑只是一个网络安全的最佳实践，就像不重复使用同一个密码有助于保护你的账户安全一样。

也就是说，如果你不能击退你的好奇心，你也不是完全没有选择(尽管你可能会踏入不道德的领域)。在 Reddit 上一个主题为 的帖子中，一位用户描述了他们如何将找到的每一个 USB 设备拿到百思买商店的电脑上进行测试。我不会担保这种方法，因为我不能容忍拿商店的财产冒险，但总的想法是——在不拿你的个人设备和信息或其他任何人的信息冒险的情况下检查 USB 是合理的。这很好，因为，让我们现实一点:你肯定要把那个 USB 设备插到*里。*
