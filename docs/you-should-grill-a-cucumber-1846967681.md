# 你应该烤一根黄瓜

> 原文：<https://lifehacker.com/you-should-grill-a-cucumber-1846967681>

我们倾向于认为黄瓜又凉又脆，最好捣碎放在沙拉里，或者用盐水腌制。但是煮熟的黄瓜也有美，尤其是烤黄瓜。



“煮熟的”南瓜令人惊讶地香甜，有一种类似于西葫芦但又不完全相同的令人陶醉的香味。由于其极高的水分含量，它们在高温下保持坚硬，给你一个内部酥脆多汁，外部味道浓郁的鱼叉。我想说的是，我比我预想的更喜欢烤黄瓜。

高温是烧烤这些宝贝的关键——你不想烤你的黄瓜，你只是想给它们一点颜色和香味。你的部分越大，它们在里面就越凉爽和坚固。我烤过黄瓜半片和黄瓜四分之一片，都很好吃，尽管四分之一片会显示出额外的褐变面积(褐变等于味道)。就皮肤而言，你可以把它们留在皮肤上，也可以把它们剥掉。我喜欢它们添加的淡淡苦味，它们以一种令人愉快的方式起泡。

除了一点点刀(和/或削皮器)的工作，烤羊肉几乎不需要准备。用足够的油裹住它们——中性的油更好，但我用了一些便宜的橄榄油，结果很棒——用大量的盐裹住它们。用高火烧烤，直到它们变成棕色，起泡，稍微变软。让它们冷却到室温，然后淋上或浇上一层咸的、酸性的调料或酱，或者把它们切碎，拌上沙拉。我还认为他们会做一个杀手茶三明治。你知道那种——奶油奶酪和白面包总是可以从一点点维度中受益，烤黄瓜至少有五个维度。
