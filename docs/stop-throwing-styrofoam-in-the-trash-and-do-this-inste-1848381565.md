# 不要再往垃圾桶里扔泡沫塑料了(而是去做这件事)

> 原文：<https://lifehacker.com/stop-throwing-styrofoam-in-the-trash-and-do-this-inste-1848381565>

聚苯乙烯泡沫塑料仍然是外卖容器、一次性咖啡杯和运输过程中易碎物品保护缓冲的普遍材料。这种被称为膨胀聚苯乙烯或 EPS 的材料可以保持食物和饮料的热度，保证包裹的安全，但不幸的是，这对环境非常有害。它占据了垃圾填埋场的大量空间，据信需要几个世纪才能分解(这意味着它基本上不能生物降解)。



EPS 也不容易回收——尽管我们知道即使回收 [也可能不像我们被引导相信的那样环保](https://lifehacker.com/should-you-even-bother-recycling-your-plastics-1839640349)——这意味着我们经常最终把它扔进垃圾桶。然而，有一些方法可以减少剩余的聚苯乙烯泡沫塑料的垃圾填埋场。下面是如何处理它的。

## 如何回收聚苯乙烯泡沫塑料

回收聚苯乙烯是有可能的，尽管通常不是和你剩下的塑料和纸板一起回收。想象一下碎包装花生粘在几乎所有东西上，你就会明白为什么了。然而， [Earth911 维护着一个各种材料回收者的数据库](https://search.earth911.com/?utm_source=earth911-header)——只需输入“EPS”和你的邮政编码，就能找到那些回收特种塑料的回收者，或者只需在谷歌上快速搜索一下。确保你提前打电话确认他们接受什么。

## 如何销售泡沫聚苯乙烯

向回收塑料公司出售 EPS 有点像为瓶子和罐子收集硬币——你不会得到太多，但至少它会被重复使用或回收。[Plasticsmarkets.org](https://www.plasticsmarkets.org/search/eps)可以帮你找到收购废塑料的公司。诚然，没有太多的选择，有些不接受消费者的材料，但你可以打电话问他们推荐什么。

## 哪里有泡沫聚苯乙烯

包装花生在商业运输中并不常见，但如果你碰巧有一个盒子，你当地的 UPS、联邦快递、或家族拥有的邮件商店可能会接受捐赠，尤其是在繁忙的运输季节。或者，学校和社区中心可以将干净的聚苯乙烯泡沫塑料重新用于艺术或花园项目。

## 如何再利用聚苯乙烯泡沫塑料

各种形式的聚苯乙烯泡沫塑料都可以有多种用途，无论是用于其预期用途还是更具创造性(但实用)的形式。因为它的隔热性能非常好，你可以用它来保温或保冷——例如，包装花生可以在必要时代替冰块。将一堆食物容器混合在一起，自己动手制作包装花生或庭院枕头的填充物。在运输用的聚苯乙烯泡沫塑料片上剪几个洞，做一个漂浮的饮料架。你也可以在干净的 EPS 杯中种植室内植物，或者在往上面添加土壤之前，用它作为花盆的底层。
