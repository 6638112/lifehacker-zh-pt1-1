# 这是抖音黑客剥鸡蛋的最好方法吗？

> 原文:[https://life hacker . com/is-this-tiktok-hack-the-best-way-to-peel a-egg-1846189855](https://lifehacker.com/is-this-tiktok-hack-the-best-way-to-peel-an-egg-1846189855)

我喜欢煮鸡蛋。我把它们放在沙拉里，做调味蛋(可能比我应该做的还要多)，把它们作为健康的零食吃。因此，当一个Tik Tok视频展示了一种快速剥鸡蛋的干净方法时，我被激起了兴趣。这种方法非常适合做快速硬煮鸡蛋点心，尤其是当去皮的时间比煮鸡蛋的时间长的时候。(我个人的问题是要有一个干净的果皮，因为有时候我会花太多的时间去挑选小块的外壳，或者大块的蛋白在去皮的时候脱落。)

这种破解包括在一边敲开并剥开一个小洞，然后敲开另一端，直到它失去T2，而不是剥开T4。一旦裂缝形成，你拿着鸡蛋用小孔端对着你的嘴，吹。看这里的[TikTok](https://twitter.com/_JASSSYY/status/1341101187039449088)了解并设想当你对着鸡蛋吹气时会发生什么。

这种手法不一定是日常剥鸡蛋的方式，但是看起来很有趣，非常搞笑。看一下上面的视频，看看这个过程实际上是如何运作的- 以及大便和笑声。
