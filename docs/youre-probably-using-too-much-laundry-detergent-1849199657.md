# 你可能用了太多的洗衣粉

> 原文:[https://life hacker . com/you-probably-use-more-doing-洗衣液-1849199657](https://lifehacker.com/youre-probably-using-too-much-laundry-detergent-1849199657)

当你把一堆特别脏、有汗或有臭味的衣物放进洗衣机时，你可能会禁不住诱惑，加入比推荐量多一点(或很多)的洗涤剂。理论上，这是有道理的:比平常更脏的衣物可能需要额外的洗涤剂来促进洗涤，对吗？

不，不完全是。不仅如此，你可能在日常衣物中使用了过多的洗涤剂。以下是需要了解的内容。

## 当你使用过多的洗衣液时会发生什么

信不信由你，使用过多的洗衣液实际上会在你的衣服或亚麻制品上留下一层额外的粘性物质。这是因为更多的肥皂意味着更多的泡沫——这不是你的洗衣机的漂洗程序所能处理的。

此外，额外的洗涤剂会让你的衣物特别滑，从而防止衣物相互摩擦，这是帮助去除表面污垢的一部分。

结果呢？洗涤剂 [残留，危害你的衣物](https://www.nytimes.com/wirecutter/blog/stop-using-so-much-laundry-detergent/) 和你的洗衣机。

使用过多的洗涤剂会让你的洗衣机消耗更多的水——如果你花钱买水，这是你应该避免的，同时也是出于环境原因。

随着时间的推移，留在衣服上的残留物也是洗衣机的一个问题——特别是在所有的角落、角落和缝隙，这些地方为细菌和霉菌 的生长提供了 [的理想条件，并且](https://www.tomsguide.com/news/this-big-laundry-mistake-is-ruining-your-clothes) [最终会导致恶臭的“干净”衣物](https://www.rd.com/article/using-too-much-laundry-detergent/) 。

## 如何知道自己是否使用了过多的洗涤剂？

一些 [迹象表明你在一堆衣物中使用了过多的洗涤剂](http:// https://www.rd.com/article/using-too-much-laundry-detergent/) 包括:

*   当衣物从洗衣机中取出时，感觉 [黏糊糊的](https://www.rd.com/article/using-too-much-laundry-detergent/)
*   “干净的”和烘干的衣物感觉起来 [松脆或者](https://www.nytimes.com/wirecutter/blog/stop-using-so-much-laundry-detergent/) 粗糙，穿起来很不舒服
*   有色衣服或亚麻布暗淡或褪色
*   白色的衣服或亚麻布变成灰白色或灰色
*   皮肤敏感的人可能会对穿衣服有反应
*   “干净”衣物上令人不快的、可能发霉的气味

## 我应该使用多少洗涤剂？

洗涤衣物所需的洗涤剂用量取决于多种因素，包括衣物的大小、您使用的洗涤剂类型以及您是否有高效洗衣机。然而，作为一般规则， [洗衣专家通常建议](https://www.tomsguide.com/news/this-big-laundry-mistake-is-ruining-your-clothes) 使用一汤匙液体洗涤剂或一杯粉状洗涤剂在传统洗衣机中装入衣物。

而且，正如莎拉·博德根在《断线钳 的视频中指出的那样，对洗涤剂标签上的建议持保留态度:“一些洗涤剂制造商将在标签上撒谎。但请记住，他们在销售一种产品，他们希望你多买一些。”
