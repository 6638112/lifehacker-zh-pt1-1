# 这些超级碗广告在一大堆加密广告中脱颖而出

> 原文：<https://thetakeout.com/super-bowl-ad-lays-pringles-seth-rogen-paul-rudd-1848533402>

每年超级碗期间，我的家人都会打赌开球后的第一个广告会是什么。在我年轻的时候，这纯粹是传闻。“百威啤酒*一定是高端产品，”我妈妈会说。“这是*一定是*百事可乐，”我爸爸会自信地说。通常我们的猜测至少会在第一季度出现。*

Watch

如今，猜测变得更加精确了。相反，我们会讨论我们在社交媒体上看到的被取笑的广告，先露面的名人。经典超级碗广告的日子似乎已经一去不复返了——没有什么比 LVI 超级碗期间加密货币的一连串商业广告更能说明这一点了，每一个广告都在我的超级碗聚会上引发了公众的抱怨。但有两个品牌坚持了我年轻时的超级碗商业传统，体现了超级碗广告的柏拉图式理想，并庆祝了地球上最伟大的食物之一:不起眼的薯片。

## **雷氏完美依靠明星力量**

 [https://thetakeout.com/embed/inset/iframe?id=youtube-video-0KjatG0hqwE&start=0](https://thetakeout.com/embed/inset/iframe?id=youtube-video-0KjatG0hqwE&start=0) 

超级碗广告中的名人过饱和意味着需要一些特殊的东西才能脱颖而出。在这位作家看来，凯罗·路德和塞斯·罗根一起演唱仙妮娅·唐恩的《非诚勿扰》的确很特别。这种名人选角并不依赖于争议或震惊——事实上，没有什么比这些老朋友一起吃薯条更让人感到安慰和正常的了。(作为可信的背书加分；这些家伙绝对喜欢薯片。)

这并不意味着 Lay's 不会全力以赴。这个广告实际上是一个提前与一些 [同样令人愉快的预告片](https://www.youtube.com/watch?v=Ylw1kiFjDfE) 戏弄的广告，现场的概念允许一些全面的好莱坞级别的制作，引导惊悚片，动作片和恐怖片，同时仍然是两个当今最受欢迎的演员工作的伙伴喜剧。

## 品客薯片让它保持甜蜜、简单、易记

 [https://thetakeout.com/embed/inset/iframe?id=youtube-video-rjZ_O-c3QRw&start=0](https://thetakeout.com/embed/inset/iframe?id=youtube-video-rjZ_O-c3QRw&start=0) 

我们当中有谁没有把脏爪子卡在品客薯片罐里？这个广告的主人公不是一个名人，而是一个真正喜欢品客薯片的人，在这个 90 秒的广告结束时，我们(和那罐薯片)跟随了他的一生。这是一个简单但相关的视觉效果，很适合咯咯地笑，并奇怪地拨动心弦。

在一个充斥着喧嚣浮华的广告的夜晚，这个广告以其简单性，视觉上和概念上脱颖而出。尽管如此，如果品客知道它在做什么，它会把这个广告带到抖音——我已经可以想象出品客罐头手挑战赛(即将在你的 FYP 推出)的视频滚动。
