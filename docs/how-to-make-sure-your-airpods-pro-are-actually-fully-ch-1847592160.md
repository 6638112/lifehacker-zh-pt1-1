# 如何确保您的 AirPods Pro 实际上充满电

> 原文：<https://lifehacker.com/how-to-make-sure-your-airpods-pro-are-actually-fully-ch-1847592160>

有没有想过为什么你的 [AirPods Pro 的电池电量](https://lifehacker.com/how-to-check-all-your-apple-devices-battery-levels-in-o-1847556015) 几乎总是徘徊在 80%左右？这是因为苹果自动启用了一项减少电池老化的功能——因此，一旦 AirPods Pro 的电池充满 80%，则会显著降低充电速度。如果你急着给 AirPods Pro 充满电，有一个简单的方法可以绕过这个功能。



## 为什么苹果会减慢 AirPods Pro 的充电速度？

苹果使用一种叫做优化电池充电的功能来改善你的 AirPods Pro 的电池健康状况。该功能增加了您的 AirPods Pro 电池的寿命，这意味着您在购买该产品的一年内不会以无法使用的电池寿命而告终。那很好。但是，当你长时间离开充电器时，它也会阻止电池以你希望的速度充电。没那么好。

这个功能可能会让你想起最近关于苹果电池节流做法 的诉讼，但这并不是苹果邪恶的一个例子。AirPods Pro 使用锂离子电池技术(就像几乎所有其他消费电子产品一样)，这种技术的局限性是电池容量会随着时间的推移而减少。没有办法阻止这些电池老化，但是有种方法来减缓这个过程。苹果的解决方案是在 AirPods Pro 上默认启用优化电池充电。

## 如何更快地给你的 AirPods Pro 充满电

如果你需要将 AirPods Pro 的快速充满电，尝试禁用优化电池充电。请注意，我们绝对不建议永久禁用此功能 — 如果你关闭了它，最好在你需要快速充电的时候重新启用，这样你就不会不必要地缩短你的 AirPods Pro 的电池寿命。

然而，如果你在紧要关头绝对*需要尽快给你的 AirPods Pro 充电，以下是你需要做的:*

1.  将 AirPods Pro 连接到 iPhone 或 iPad。(对大多数人来说，这涉及到把你的 AirPods Pro 从盒子里拿出来，塞在耳朵里；它们通常会自动连接到 iPhone 或 iPad。)
2.  在设备上，进入**设置>蓝牙**。
3.  轻按 **AirPods Pro** 旁边的 **i** 按钮。
4.  向下滚动并禁用**优化电池充电**。
5.  当你这样做的时候，你会在你的 iPhone 上看到一个弹出窗口，要求你确认这个操作。你可以选择**关机到明天**来暂时禁用它，这可能是一个好主意，原因如上所述。或者你可以过危险的生活，选择**关闭**永久禁用优化的电池充电按钮(直到你手动重新打开)。

一旦您的 AirPods Pro 将充电到 100%，您可以按照相同的步骤重新启用优化的电池充电(或者等到它自动重新打开)。

注:该功能需要 iOS 14.2 或 iPadOS 14.2 版本或更新版本。您可以通过进入**设置>常规>关于**并检查**软件版本**旁边的数字来检查您是否拥有所需的版本。如果您有旧版本的 iOS 或 iPadOS，请转到**设置>常规>软件更新**并更新到新版本的 iOS。
