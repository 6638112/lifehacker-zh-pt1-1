# 蒸汽清洁器有网上宣称的那么有用吗？

> 原文:[https://life hacker . com/is-a-steam-cleaner-as-used-as-the-internet-claims-1845824477](https://lifehacker.com/is-a-steam-cleaner-as-useful-as-the-internet-claims-1845824477)

有一天，我们受人尊敬的主编让我放松了一下，说她的许多朋友发誓说蒸汽清洁器是最好的清洁工具。他们声称这种蒸锅可以清洁基本上所有的东西——地板、墙壁、电器、床垫、厕所、窗户——这份清单还在继续。

自然地，我觉得有必要——不，有义务——验证这些说法。因此，我买了一个价格合理的型号(仅30.99美元)，阅读了一些安全信息，并着手确定蒸汽清洁器是否是我们一直祈祷的多功能清洁救星。

在开始之前，我学会了不要在任何未密封的表面上使用蒸锅，如木地板或多孔的石头。我的公寓里没有这些表面，但这是租来的，我想要回我的全额保证金，所以我犯了一个谨慎的错误。此外，对墙壁进行消毒的说法似乎也是可疑的；我的研究表明，蒸锅对去除旧墙纸很有帮助，但也能去除墙上的油漆——所以我也避免了这个测试。

我用蒸汽清洁器测试了几个表面:我的床垫、烤箱、后挡板、踢脚板、马桶、水泥浆和一件衬衫。我本以为只要一股快速的蒸汽就能立刻照亮所有的表面，但事实并非如此。一些表面据称被蒸汽“消毒”,但我看不出T2有任何清洁剂。在某些表面上，的效果看起来和用湿布擦拭差不多。但是一些测试— 特别是后溅和灌浆— 是真正革命性的:仅仅经过几秒钟的蒸汽处理，这些表面焕然一新，没有使用化学品。
