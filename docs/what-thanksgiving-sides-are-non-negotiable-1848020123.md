# 感恩节有哪些方面是不可协商的？

> 原文：<https://lifehacker.com/what-thanksgiving-sides-are-non-negotiable-1848020123>

我喜欢认为自己是平静、正常、几乎放松的感恩节观察者，但事实上，我有点像暴君。我通过自愿主持和烹饪大部分食物来隐藏我的暴虐倾向，所以我可以确保火鸡是湿润的，蔓越莓酱足够甜，以及 [Jiffy 玉米砂锅](https://lifehacker.com/an-ode-to-jiffy-corn-casserole-1830231857?rev=1636419545276) 在桌子上。

Watch

我在这个背后的哲学是双重的:你不能提出要求而不提供相关的劳动(或者其他种类的劳动，作为一种交易，无论是摆桌子还是搬家具)，你应该总是尊重主人。如果你提议带上“主人需要的任何一道菜”，那就带上他们要的东西。如果他们做不出你最喜欢的，你可以自己带。这保持了文明。

人们对自己最喜欢的菜肴有很多情感。火鸡是最明显的一种，如果你试图用火鸡换另一种蛋白质，爸爸们最有可能不高兴，但许多人也对肉有*感觉*。我无法想象没有玉米面包砂锅的感恩节，所以我总是主动提出做它，除非有人想做。(我的朋友 Bec 现在通常会做，在我把它带到她家的那年尝过之后。是一道很厉害的配菜。)

每个人都有他们的必上桌，不容商量的菜，我想知道你的。有点出乎意料的是，我变成了一个土耳其人(进一步证明了我姐姐的观点，即我是跨性别者，成为了我们的父亲)。我过去对火鸡一点也不在乎，但现在每年向我的朋友和家人赠送一只多汁美味的火鸡是一种骄傲。我还需要土豆泥、某种红薯菜(这里是 [今年精选](https://lifehacker.com/these-3-ingredient-scalloped-sweet-potatoes-are-incredi-1847794486) ) ，还有一个山核桃派。

在评论中分享你的想法，这样你就能找到你的人。你是团队山核桃派，团队苹果派，还是 [团队红薯派](https://lifehacker.com/forget-pumpkin-and-make-a-sweet-potato-pie-this-thanksg-1820083902) ？土耳其对你来说很重要吗？最重要的淀粉是什么？让我知道，然后我会收集你的令人愉快的评论与世界分享。(我迫不及待地将结果与莎拉在感恩节餐桌上列出的[dI shes tsshould be banned 进行交叉对比。我打赌会有相当多的重叠。)](https://lifehacker.com/what-foods-should-be-forever-banned-from-the-thanksgivi-1847981158)
