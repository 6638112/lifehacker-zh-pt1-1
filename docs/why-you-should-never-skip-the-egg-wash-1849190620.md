# 为什么你永远不应该错过洗鸡蛋

> 原文：<https://lifehacker.com/why-you-should-never-skip-the-egg-wash-1849190620>

洗蛋是烹饪的后备歌手——当它出现的时候你不会注意到它，但你肯定会注意到它的缺席。然而，总会有这样的时刻，当你提前阅读食谱时，咕哝着，“啊，我需要 T2 洗鸡蛋吗？”是的，你确实有点。洗蛋是一些食谱中最重要的步骤之一，无论如何，它们都提高了最终产品的质量。

蛋液的常见用途是作为面漆。你可以打赌 doublee-硬皮派需要它，一些饼干食谱需要它，像奶油蛋卷这样的强化面包也需要它。对于特别甜的东西，粘性表面是撒糖的好机会；颗粒会抓住洗涤物而不是弹开。在你的糕点表面覆盖一层薄薄的鸡蛋也会创造出一种不可抗拒的金棕色，散发着迷人的光泽。这是由于我们的烹饪 MVP，即 [美拉德反应](https://en.wikipedia.org/wiki/Maillard_reaction) ，导致 [鸡蛋中的蛋白质很容易变成棕色](https://www.incredibleegg.org/professionals/manufacturers/real-egg-functionality/browning-color#:~:text=The%20proteins%20within%20eggs%20can,dark%20color%20of%20roasted%20coffee.) ，产生一种深度烘烤和有光泽的涂层，这种涂层远比一种干燥、几乎不呈棕色、未经鸡蛋清洗的馅饼更诱人。它在这种应用中也是无味的，因为涂层刷得很薄，没有留下鸡蛋味来分散甜点的注意力。这种用法纯粹是美学上的，从技术上来说，如果你没有鸡蛋，你可以跳过它，但是如果你有鸡蛋，你就不敢离开它。这就相当于烤了一个 Funfetti 蛋糕，但是因为你不想打开包装而漏掉了面包屑。

洗蛋可能纯粹是为了美观，但它也可以用来保持一些甜点的结构完整性，而不增加味道。在馅饼和果馅饼中，用一茶匙水搅拌一个鸡蛋的简单应用可能是面包店完美糕点和从你的烤箱中冒出的辛辣烟雾之间的差异。

对于双层馅饼来说，蛋液应该加在顶部边缘。一旦馅料被填满，顶部的面皮将接触到鸡蛋清洗过的边缘，形成一个最重要的密封。顶部和底部的外壳由鸡蛋粘合在一起，由于 [鸡蛋中的蛋白质凝固](https://www.loc.gov/everyday-mysteries/physics/item/is-it-possible-to-fry-an-egg-on-the-sidewalk-if-its-hot-enough/#:~:text=An%20egg%20needs%20a%20temperature,start%20and%20maintain%20the%20process.) 并在 158 华氏度下烹饪，它们将在馅饼馅料开始膨胀并在 212 华氏度下起泡(水沸腾并蒸发)之前密封馅饼的接缝。这使得所有的馅饼馅都在它应该在的地方，而不是从边缘爆出来，烧到烤箱的地板上。

对于烘烤前有非烘烤馅料或特别薄的液态馅料的蛋挞和馅饼，如柠檬凝乳蛋挞或南瓜派，一些食谱建议在底部外皮上涂上蛋液以防止渗漏。这个洗蛋应用程序是不可见的，所以使用 [右洗蛋](https://lifehacker.com/how-to-know-which-egg-wash-to-use-1829507257) 可以有所不同。在这种情况下，炒鸡蛋白就可以了，因为在馅料下面不会欣赏深金棕色。要做到这一点，根据食谱的需要，盲目烘烤馅饼或馅饼皮。一旦面包皮从烤箱中拿出来并且 [馅饼的重量](https://lifehacker.com/weigh-down-your-pie-crusts-with-sugar-1830228677) 被移走，当还很热的时候，沿着里面底部和内部边缘*刷蛋白洗液。这就用鸡蛋填满了所有的毛孔和细小的裂缝，而外壳的余热仍然在 158°F 左右或以上(但还不足以引发美拉德反应的褐变)，在你刷过后煮鸡蛋来做密封。倒入蛋羹或一层巴诺菲馅料，由于有了蛋液，你的面包皮不会有保罗好莱坞级别的湿底。*

这并不需要太多——一层薄涂层是密封和抛光应用所需要的全部(如果你做得太多，鸡蛋将会搅成一个小炒鸡蛋)。一个鸡蛋可以用很长时间，所以提前检查食谱，看看你是否有足够的鸡蛋备用。当你的烤箱展现出一种魅力- 值得一试的糕点时，你会感谢过去的自己如此能干。
