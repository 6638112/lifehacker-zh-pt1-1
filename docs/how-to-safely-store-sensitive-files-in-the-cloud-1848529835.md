# 如何在云中安全地存储敏感文件

> 原文:[https://life hacker . com/how-to-safely-store-sensitive-files-in-the-cloud-1848529835](https://lifehacker.com/how-to-safely-store-sensitive-files-in-the-cloud-1848529835)

理想情况下，你不会在云中存储任何敏感的个人信息。你的在线账户总有被黑的风险，所以理论上你最好离线存储你所有的文件和数据。但这不再是这个世界真正的运作方式，保持轻松访问财务记录、身份证和医疗文件的便利性(或必要性)通常意味着承担一定的风险。

但许多人冒了不必要的风险，最终在Google Drive、iCloud或OneDrive等云存储服务上存储了大量个人信息。如果你也这样做，考虑用密码加密来更好地保护数据。我们将向您展示一些在您将敏感文件上传到网上之前加密它们的方法。

## 上传前用密码保护你的文件

对于大多数人来说，给他们的文件添加一个密码就足够了。有各种简单的方法来用密码保护你的文件，但最方便的是使用应用程序，如 [7Zip](https://www.7-zip.org/download.html) 或 [Keka](https://www.keka.io/en/) ，它们可以让你将多个文件压缩到一个单一的存档文件中，你可以很容易地用密码保护。您的受密码保护的文件可以在任何设备上访问；你只需要密码就可以查看它的内容。如果您有许多想要快速加密的文件，这种方法最有效。

如果您专门处理只有一种文件类型的文档，您可能不需要使用不同的应用程序来加密它们。例如，如果您正在使用Microsoft Word创建敏感的doc文件，请转到**文件>信息>保护文档>用密码**加密。您可以在首选的PDF应用程序中使用类似的方法来加密这些类型的文件。不过要小心不要丢失这些密码——这也是 [考虑使用密码管理器](https://lifehacker.com/7-of-the-best-password-managers-to-choose-from-before-l-1848165088) 的另一个好理由。

## 将文件直接存储在密码管理器中

密码管理器，如 [BitWarden](https://bitwarden.com/) 或 [1Password](https://1password.com/) 在其高级计划中提供高达1GB的加密文件存储。如果您已经为密码管理器支付了订阅费，那么作为存储敏感文档的一种方式，值得探索这个选项。这样，您不必记住或存储额外的密码来访问它们；您只需要密码管理器应用程序的主密码。Bitwarden允许您在每年10美元起的计划中使用该功能，而在1Password上使用该功能需要每年30美元的计划。

## 在您的云存储服务中创建加密保管库

[Dropbox Vault](https://help.dropbox.com/files-folders/sort-preview/dropbox-vault) 和[one drive Personal Vault](https://www.microsoft.com/en-us/microsoft-365/onedrive/personal-vault)让你为自己最重要的文件创建一个加密空间。您必须使用额外的密码或数字PIN才能访问存储在这些文件夹中的信息。OneDrive个人保管库免费提供，但您只能上传三个文件。为了避开这个限制，创建一个. zip归档文件，其中包含您需要存储的尽可能多的文件。带有密码的zip文件)。OneDrive的付费层取消了此限制。Dropbox Vault仅适用于该服务的付费层级。

## 在您的计算机上创建加密文件夹

如果你使用的是没有加密的云存储服务，你可以自己免费创建一个加密文件夹。在Windows上，使用7Zip可以快速完成工作。右键单击任何文件或文件夹，并选择 **7-Zip > Add to archive** 。在**加密**部分，您可以输入并确认密码，然后点击**确定**创建档案。

在你的Mac上，内置的磁盘工具应用程序也可以让你免费这样做。使用Spotlight Search ( **Command +空格键**)来搜索并启动“磁盘工具”。然后从文件夹中导航到**文件>新图像>图像。然后你可以选择你电脑上的任何文件夹，然后从**加密**旁边的下拉菜单中选择 **256位AES加密**。“磁盘工具”会要求您键入并确认密码。完成后，您还应该从 **Where** 旁边的下拉菜单中选择想要存储文件夹的位置。之后，在**图像格式**下，选择**读/写**，点击**创建**。**

这将创建一个磁盘映像(。dmg)文件，可以在任何Mac上轻松打开(只要你有密码)。dmg文件是原始文件夹的受密码保护的副本，因此请记住删除原始文件夹，该文件夹无需密码即可访问。

## 使用第三方加密应用

这种方法可能是最不方便也是最安全的。第三方应用程序如[cryptomor](https://cryptomator.org/)和 [Boxcryptor](https://www.boxcryptor.com/en/) 允许你在电脑上创建一个安全的存储空间，并允许你在任何设备上访问这些文件。这两项服务都兼容流行的云存储服务，如Google Drive、OneDrive、Dropbox和iCloud Drive。

在你的电脑上安装Cryptomator或Boxcryptor的应用程序，并使用它来快速加密你的云存储服务中的任何文件夹。例如，如果你在Google Drive上有一个包含所有身份证扫描副本的文件夹，你可以将其添加到应用程序中，它会对该文件夹进行密码保护。

如果你想在任何其他设备上访问此文件夹，你必须在该设备(包括智能手机)上安装该应用程序，并输入密码才能访问它。

尽管从安全的角度来看，这是最好的解决方案，但只有在准备解决小问题时，才应该使用这种方法。我们在使用苹果的文件应用程序时遇到了问题，有时我们无法访问iPhone上的加密文件夹，而Windows的运行更加流畅。
