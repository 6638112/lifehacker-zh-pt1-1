# 抓到 COVID 怎么投票

> 原文：<https://lifehacker.com/how-to-vote-if-you-catch-covid-1845424962>

人们已经在 [通过邮件投票创下了今年](https://apnews.com/article/record-early-votes-2020-transforms-efd6eefbd3d140bdc8909360de0bff62) 的记录，但是如果你计划亲自投票，但在选举日之前生病了，会发生什么呢？也许是新冠肺炎，也许是别的什么。不管是什么，你都要确保不会让任何投票工作人员或其他选民生病(另外，你可能不应该做像亲自投票这样费力的事情)。在这种情况下，你可能需要一张紧急选票——这种选票在大多数州都有。如果这种情况发生在你身上，你可以这么做。



### 首先，检查你所在州邮寄选票的截止日期

当谈到邮寄选票时，需要记住几个不同的截止日期——，是的，每个州都有自己的时间表。在某些情况下，要求缺席投票有截止日期，以及其他截止日期，说明选票上可能的最新邮戳和/或地方选举委员会必须收到的日期。此时，在游戏中请求缺席投票已经很晚了，但一如既往地， [检查您所在州的选举指南](https://lifehacker.com/check-on-your-states-absentee-voting-rules-right-now-1842378031) 以了解一切何时到期。

### **如何获得紧急选票**

如果你生病超过了可以申请缺席投票的时间，大多数州会给你申请某种形式的紧急投票的选择。再次， [与你所在州或当地的选举办公室](https://www.usa.gov/election-office) 核实一下，看看这个过程涉及到什么，但是可以肯定的是你需要找到并填写某种形式的申请(你所在州的网站上应该也有)。

根据 [全国州议会会议](https://www.ncsl.org/research/elections-and-campaigns/absentee-voting-in-case-of-a-personal-emergency.aspx)【NCSL】各州以不同方式处理紧急选票:

#### 38 个州允许在医疗紧急情况下进行紧急缺席投票

阿拉巴马州、亚利桑那州、阿肯色州、加利福尼亚州、科罗拉多州、康涅狄格州、佛罗里达州、佐治亚州、伊利诺伊州、印第安纳州、爱荷华州、堪萨斯州、肯塔基州、路易斯安那州、缅因州、马萨诸塞州、密歇根州、明尼苏达州、蒙大拿州、内华达州、新罕布什尔州、新墨西哥州、纽约州、北卡罗来纳州、北达科他州、俄亥俄州、俄克拉荷马州、宾夕法尼亚州、罗德岛州、南卡罗来纳州、南达科他州、田纳西州、德克萨斯州、犹他州、佛蒙特州、弗吉尼亚州、西弗吉尼亚州和威斯康星州。

#### 6 个州的选举官员会给你一张紧急选票，如果你因为医疗紧急情况不能去投票站的话

亚利桑那州、加利福尼亚州、佐治亚州、明尼苏达州、佛蒙特州和西弗吉尼亚州。

#### 17 个州的住院选民可以指定某人请求/递送/提交他们的紧急选票

阿肯色州、科罗拉多州*、康涅狄格州、佛罗里达州、伊利诺伊州、马里兰州、内布拉斯加州、内华达州、新墨西哥州、北卡罗来纳州、北达科他州、俄亥俄州、宾夕法尼亚州、南卡罗来纳州、南达科他州、田纳西州和犹他州。

*科罗拉多州向所有合格选民邮寄选票，但如果在邮寄选票的截止日期后出现紧急情况或自然灾害，并且选民无法亲自获得替换选票，他们可以指定一名授权代表代为获得替换选票。

关于你们州法规的全部细节，请参见 NCSL 网站 。
