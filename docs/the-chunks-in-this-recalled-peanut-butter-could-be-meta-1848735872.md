# 这次召回的花生酱中的“大块”可能是金属碎片

> 原文：<https://lifehacker.com/the-chunks-in-this-recalled-peanut-butter-could-be-meta-1848735872>

如果花生酱是你厨房和餐具室的主要食品，检查一下它的品牌和标签，看看它是否是 Skippy Foods，LLC 发布的自愿召回的一部分。根据公司发布的消息——该消息也发布在 [美国美国食品药品监督管理局(FDA)网站](https://www.fda.gov/safety/recalls-market-withdrawals-safety-alerts/voluntary-class-ii-recall-announced-limited-number-jars-skippyr-reduced-fat-creamy-peanut-butter) 上——产品被召回是因为它们可能已经被小金属片污染。(厚花生酱不应该是这样工作的。)下面是要知道的。

## 为什么会有花生酱召回？

总共 9，353 箱(或 161，692 磅。)的 Skippy 花生酱产品被召回，因为一件制造设备上的小块不锈钢碎片可能已经进入了有限数量的罐子中。当制造工厂的内部检测系统发现问题时，问题就暴露出来了。

目前，还没有消费者受伤或其他与此次召回相关的投诉，所有收到受影响花生酱的零售商都已得到通知。

## 哪些类型的花生酱被召回？

确定你的花生酱是否在召回之列的最简单的方法是寻找罐子顶部的“最佳食用日期”。召回指定日期的以下品种:

**SKIPPY 减脂奶油花生酱，40 盎司**

*   最好由 MAY0423 使用
*   最好由 MAY0523 使用

**SKIPPY 减脂奶油花生酱——俱乐部，2/40 盎司**

*   最好由 MAY0523 使用

**SKIPPY 减脂矮胖花生酱，16.3 盎司**

*   最好由 MAY0723 使用
*   最好由 MAY0623 使用

**SKIPPY 混合植物蛋白的奶油花生酱，14oz**

*   最好在 2023 年 5 月前使用

只有这些产品是召回的一部分:根据 FDA 的规定，SKIPPY 品牌花生酱或花生酱涂抹酱 [不包括其他尺寸、品种或其他包装配置。](https://www.fda.gov/safety/recalls-market-withdrawals-safety-alerts/voluntary-class-ii-recall-announced-limited-number-jars-skippyr-reduced-fat-creamy-peanut-butter)

## 如果你已经购买了被召回的 Skippy 花生酱，该怎么办

如果你已经购买了上面列出的任何花生酱产品，不要吃它。取而代之的是，将它退还给购买它的零售商进行更换，或者拨打 Skippy Foods 消费者参与热线 1-866-475-4779(美国东部时间周一至周五上午 9 点到下午 5 点)或 [访问他们的网站](https://www.peanutbutter.com/recall) 获取说明和更多信息。
