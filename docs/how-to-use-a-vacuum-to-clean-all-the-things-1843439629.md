# 如何使用吸尘器清洁所有的东西

> 原文:[https://life hacker . com/how-to-use-a-vacuum-to-clean-the-things-1843439629](https://lifehacker.com/how-to-use-a-vacuum-to-clean-all-the-things-1843439629)

真空吸尘器的伟大之处在于它的多功能性。想清除地毯上的日常生活碎屑吗？很明显，你的吸尘器很适合这个。它还可以清除沙发缝隙中的碎屑，吸走百叶窗上的灰尘，快速清理厨房地板上的咖啡渣。

一台好的吸尘器不仅仅是地毯清洁器。它还可以接管你的扫帚和抹布的职责。有了合适的附件，你的吸尘器可以成为你打扫房间的主要工具——但前提是你知道如何正确使用它。

## 知道如何调整你的吸尘器的高度

让我们从吸尘最基本的方面开始:在地毯上运行。我们都做过(或者看着别人做过，你们这些懒惰的家伙)，但是并不是所有人都在打开吸尘器前额外检查了它的高度。

你看，地毯和小地毯有所谓的“绒毛”，从技术上讲是指制成它们的纤维和织物。如果你愿意，你可以把它想象成“堆在地毯背衬上的所有东西”。

重要的是要知道有些地毯的绒毛比其他的高。你已经体验过步入毛绒绒的柔软地毯和走过廉价地毯的感觉，对吗？有时候你的地毯堆得太高了，你几乎可以在里面扭动你的脚趾——在这种情况下，你需要调整你的吸尘器的绒头高度，以适应地毯纤维的长度。

我不能告诉你如何调整吸尘器的高度，因为我不知道你用的是哪种吸尘器。寻找一个旋钮或开关，使你的吸尘器前端——你在地毯上运行的部分——上升或下降。(会很微妙，所以仔细看。)

从本质上讲，如果你在低绒地毯上将吸尘器设置为“高绒头高度”,那么吸力机制就不足以吸附嵌在纤维底部的灰尘。另一方面，如果你试图将一个“低绒毛高度”的吸尘器推过一个高绒毛地毯，那就要做很多额外的工作。这会损坏地毯，而且肯定会让你讨厌吸尘。

没有人应该讨厌吸尘！

## 直线吸尘(尽可能)

我不想在这上面花太多时间，但是如果你想彻底地用真空吸尘器清扫地毯，用你的吸尘器在地毯上来回直线移动，一个接一个，直到你覆盖了地毯的每一寸。如果你只是随意地前后推它，你更有可能错过一个点。

我也知道，我们大多数人并不真的这样做，因为家具会碍事，我们不想每次吸尘时都把它们全搬走。尽可能直线吸尘就好，尽可能容纳家具。用吸尘器把沙发和椅子下面的灰尘吸出来。

## 了解您的真空附件

一个典型的吸尘器至少有四个基本附件:一个缝隙工具(旨在进入狭窄的地方)，一个室内装潢工具(旨在清洁你的沙发和椅子)，一个除尘刷(打赌你猜不到那是干什么的)和一个延伸杆(有点像缝隙工具，但旨在触及吊扇和窗户顶部)。

这些附件可以帮助你不费吹灰之力就把东西弄干净。在你的沙发垫上运行室内装潢工具，然后使用缝隙工具收集掉在它们之间的所有面包屑和狗毛，然后用延长棒吸走藏在沙发下面的任何灰尘。用除尘刷掸掉咖啡桌腿上的细小缝隙。(如果你愿意，可以用除尘刷打扫你的整个家。)

不要把你的吸尘器附件扔在壁橱里，忘记它们在那里。使用它们。去了解他们。你会很快学会爱他们。

## 一些吸尘器可以处理没有铺地毯的地板

我不知道你是否在一个只允许你用吸尘器清扫地毯，不得不用扫帚清扫没有地毯的地板的房子里长大，但是让我告诉你，事情已经发生了变化。如果你有一个带有硬木地板附件的吸尘器，你可以用它来清洁几乎所有没有铺地毯的地板(木头、油毡、瓷砖等等)。).

顺便说一下，你不能在硬木上使用标准真空吸尘器的原因是，它用旋转的小纤维从地毯上吸走灰尘。随着时间的推移，这些纤维会损坏你没有铺地毯的地板——所以除非你的吸尘器有一个特殊的附件，可以连接其他类型的地板，或者可以完全关闭滚刷，否则你可能会想继续用传统的方式扫地和扫地。

## 机器人吸尘器是为了补充，而不是接管

这里是最后一条建议:我知道我们都喜欢我们可爱的小机器人真空吸尘器——例如，它们在控制宠物毛发方面非常出色——但是机器人真空吸尘器只能让你到此为止。你的Roomba不够敏捷，无法从游戏椅的缝隙中吸出多力多滋面包屑。它还不能爬上你的百叶窗，也不能清除你吊扇上的污垢。

所以让你的机器人做它的工作，事后感谢它，然后使用真正的真空来做只有人类(用真正的真空)才能做的工作。
