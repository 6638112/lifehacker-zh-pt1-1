# 对于跑步者来说，Stride 是一款令人上瘾的抢占地盘游戏

> 原文：<https://lifehacker.com/stride-is-an-addictive-territory-grabbing-game-for-runn-1847073736>

大多数早上我都跑步，通常每次都让 T1 在 T2 附近做同样无聊的循环。但是今天，我发现自己在附近的一个公园里，仔细查看地图，为我的团队策划最好的方式来获取最多的瓷砖。我当时正在玩一款名为 [Stride](http://www.runanempire.com/stride/) 的游戏，我可以看到它变得令人上瘾。



该应用在 [iOS](https://apps.apple.com/us/app/stride-walk-run-capture/id1526274172) 和 [Android](https://play.google.com/store/apps/details?id=com.lgl.stride&hl=en_US&gl=US) 上是免费的(用于基本功能)，游戏的机制是这样的:T 他的世界被分成六边形的瓷砖，w 无论你在哪里跑步(或走路，或坐在轮椅上旅行)，你都可以要求一个瓷砖。如果您的路线带您进入一个封闭的循环，您也可以在该循环的中心获得瓷砖。每块瓷砖都归曾经访问过它最多的人所有。

如果你喜欢在 Strava 上争夺 KOM(山之王)的吹牛权，你可能会欣赏 Stride 的不同风格的竞争。不同的是，步幅瓷砖是通过访问量来赢得的，而不是速度——y 你可能是跑得最慢的人，但如果你每天都走同一条公园小径，它就可能是你的。

## 玩 Stride 是什么感觉

这个游戏是新的，至少对我的领域是如此。打开地图，我可以看到大部分的六边形瓦片无人认领，除了这里和那里的集群:县公园，一些随机的街区。每个人都在红队，和我一样(你可以随时改变队伍，但应用程序说红队需要成员，所以我来了。)

我的第一个想法是在我通常的晨跑中使用这个应用程序，但那是在我家开始和结束的。与允许您记录公共或私人活动的 Strava 不同，Stride 完全是关于公共地图的。所以我去了公园。

公园里我最喜欢的一个跑步点无人认领，所以我去了那里。当你跑步时，你会看到你声称的瓷砖，所以我没有像往常一样自动驾驶和绕圈行驶，而是注意我何时到达瓷砖的边缘。有时我可以通过走一条小路或者选择一条更长的小路而不是捷径来获得额外的魔力。经过两英里的短途旅行，我自豪地拥有了 10 块以前无人认领的瓷砖。

不满意，我回到我的车，把我的目光放在一个名叫曼格勒的队友拥有的一大块瓷砖上。我把车停在无人认领的地方，向他拥有的瓷砖慢跑过去。我捡了几个，然后站在路边，结束了我的跑步，这样我就可以看到结果了。我偷了五块曼格勒的瓷砖，还抓了五块我自己的新瓷砖。被偷的牌现在各有一个双人排行榜:曼格勒和我都去过一次，但我是赢家，因为我最近去过。(Mangler 可能已经收到了通知，所以我们会看看他们是否会尽快前往把他们带回去。)

决心保护我的新领地，我重启了应用程序。我之前已经注意到，在一条来回的路径上，即使我穿过了两次，我也只能得到一次访问每块瓷砖的积分。因此，在我回到我的车之前，我开始了一次新的运行，结果，记录了对一些瓷砖的两次访问。访问量越大，我的排名就越有保障。

最后，即使 Mangler 夺回了他们的领地，至少我知道我帮助了我的团队，为我们获得了额外的瓷砖。

## 玩 Stride 的技术方面

如果你有一个 Garmin 设备，你很幸运:Stride 应用程序可以导入你用 Garmin 跟踪的跑步记录。它不与任何其他应用程序或设备同步(如 Apple Watch) 据我所知，当我出去跑步时，我不得不打开 Stride 应用程序，让它跟踪我。

一开始地图很难辨认——我在红队，但是所有的彩色瓷砖都是蓝色或绿色的。原来有一个团队地图视图(红色团队区域为红色)和另一个个人地图视图(我的地图块为绿色，其他人的地图块为蓝色)。

你可以在不购买订阅的情况下玩游戏，但详细的排行榜和团队统计数据只对会员开放(成为会员将花费你$ 8.99/月或$ 44.99/年)。如果这个游戏在你所在的地区变得流行，与队友合作占领领地可能是一种有趣的方式来计划你的跑步并与其他跑步者保持联系。
