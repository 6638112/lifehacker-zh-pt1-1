# 感恩节仍然可以很特别

> 原文：<https://lifehacker.com/thanksgiving-can-still-be-special-1845747052>

感恩节对我来说有点奇怪。在我成长的过程中，以及整个大学期间，我每年都会在祖父母家吃同样的饭。但是大学毕业后，当我为了在一家报社工作而搬到全国各地时，事情变得不稳定了。第一年，我在感恩节工作，负责给的老年居民送当地的饭菜，然后为其他一些新记者举办了一次家常便饭。在接下来的几年里，我当时的男朋友(现在的丈夫)和我与朋友一起享受了几次感恩节，然后他的家人开始旅行加入我们的假期。几年前，当我们搬回东海岸时，我们开始开车去俄亥俄州和家人一起度假。今年，火鸡日第一次只有我们三个人——我、我丈夫和我们的儿子。



我认识的人都与( [斗争过或者可能还在与](https://vitals.lifehacker.com/its-time-to-cancel-your-thanksgiving-travel-plans-1845659175) 斗争过？)决定今年感恩节是否去旅行看望亲人。这可能让人感觉是在传统的田园诗般的大家庭聚会和彻底的假期之间做出选择。但是，在我自己的感恩节餐桌旁经历了这么多年的变脸之后，我想指出，无论今年你们的“聚会”多么小，它仍然可以是特别的。

忍耐一下，因为我知道这个*很烂*。我非常想去俄亥俄州旅行。在整个疫情期间，我没有见过我的父母，也没有见过我年迈的祖父，也没有见过我的兄弟，我的嫂子，或者我的三个侄女——除了一次尽可能远离社交距离的快速旅行，那是为了六月份我祖母的葬礼。很糟糕的，但也很好，因为该死的，我仍然可以让感恩节变得特别。你也可以。

### 让孩子们多参与进来

如果有一年可以让孩子们在感恩节晚餐准备期间在厨房里狂欢，那就是这一年。为什么？因为食物上桌晚了也没关系。如果土豆泥有点太稀或者你烧焦了馅料也没关系。你不需要打动任何人，但是你可以让你的孩子体验帮助烹饪和“招待”他们的第一个感恩节，这是我们大多数人在成年之前都不会尝试的事情。

今年，我第一次让我的儿子帮忙做我祖母的苹果酱。这将有助于他感受到与我们俄亥俄州家庭的联系，这将是对她的尊重(即使这不可能像她过去那样好)，这将有助于他感受到自己是这一天努力的一部分。让你的孩子轮流给火鸡涂黄油——或者，如果你有一个十几岁的孩子，让他们第一次尝试切火鸡，因为*为什么不呢。*让这顿饭成为*他们的*饭；如果这部分(或全部)是一场灾难，它将在未来几年成为一个更好的故事。

### 无论如何，使用“好东西”

拿出闪闪发光的餐垫、你拥有的最精美的盘子、手绘玻璃杯和餐巾布。当然，只有你，你的伴侣，一个三岁，和一个六岁，但是谁在乎呢？ G o 全部出动。即使你没有做火鸡和所有的配菜——即使你只是点了几个披萨或者买了一些外卖——当地方布置看起来尽可能的别致时，食物和饮料的味道会更好。

这是给你的大脑和你孩子的大脑的一个信号，这顿饭*是*特别的。这可能是他们第 47 次在疫情的 Bruno 披萨店吃意大利辣香肠片，但是*这*片是感恩节的特殊切片。

### 营造气氛

也许你的家庭是那种点燃一束蜡烛，调暗灯光，播放一些轻松音乐的家庭。或者，播放音乐，举行餐前舞会更适合你。无论你想营造什么样的情绪，无论是感恩反思还是全力以赴的派对模式，都全力以赴。当 [*国家讽刺剧《圣诞假期》*](https://www.imdb.com/title/tt0097958/) 在背景中循环播放时，和穿着你今年花钱买的那些配套的假日睡衣的每个人一起吃晚餐。

这是与众不同的一年。抛开传统，做任何你想做的事情，让它比任何一个随机的星期四都更特别。
