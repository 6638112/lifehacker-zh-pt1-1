# 如何阻止游戏随意关闭你的电脑

> 原文：<https://lifehacker.com/how-to-stop-games-from-randomly-shutting-down-your-pc-1849583721>

很少有什么事情比你的电脑在游戏中突然关机更令人烦恼(或担忧)的了。如果你已经排除了这个问题的其他解决方案——比如升级你的 GPU 驱动程序或主板的 BIOS，或者修复安装不正确的硬件——最有可能的罪魁祸首是你的 PC 的电源单元(PSU)。它要么太弱，无法为您的硬件供电，要么没有正确连接到像您的显卡这样的耗电组件。幸运的是，有几种方法可以防止这些与 PSU 相关的突然崩溃。



## 升级您的 PSU

有几个潜在的原因可能导致你的 GPU 超过你的 PSU，但最有可能的只是你的 PSU 不够强大。例如，虽然 750 瓦的 PSU 在技术上足以满足大多数 GPU，包括 RTX 3080 和 3090 这样的高端显卡，但一些游戏可能会导致超过 PSU 极限的功率峰值，并使你的电脑崩溃。

正如 Reddit 主题 中 [的回复所示，许多用户确认从 750 瓦的电源设备切换到 1000 瓦的 PSU(或更高)解决了他们的随机关机问题。对于一些用户来说，简单地换成质量更好的 750 瓦 PSU 就足够了——尽管增加瓦数更有可能解决问题。](https://www.reddit.com/r/buildapc/comments/xo1sam/certain_games_cause_pc_to_turn_itself_off/)

然而，像任何电脑部件升级一样，将你的 PSU 换成 1000 瓦的型号将会花费你很多钱。在购买和安装之前，您还需要确保新的 PSU 与您的电脑外壳和其他组件*兼容。在一些罕见的情况下，如果您购买了一台带有定制外壳的预制游戏电脑，需要一个专有的电源单元，例如某些带有外部 PSU 的外星人“控制台”电脑，您甚至可能无法切换到新型号。*

无论如何，也有可能一个更大的 PSU 是不必要的。当然，升级 PSU 几乎肯定能解决这个问题(除非另一个组件出现一些不可预见的问题)，但上面的 Reddit 帖子有许多其他解决方案，你可以在花钱升级之前尝试一下。

## 确保您的电脑组件正确连接到 PSU

另一个可能的原因是你的 GPU 的电源线没有正确连接到 PSU。您的 GPU 需要为其插入的每个可用插头配备专用线缆。关闭电脑电源，拔下插头，然后打开机箱，确保所有设备都连接好。

## 什么是欠载？

您还可以尝试降低 GPU 的功率限制，也称为“欠载”这只能在 Nvidia 和 AMD 的最新显卡上实现，但这是一个安全且相对简单的过程，可以降低 GPU 的功耗和热量输出，并且对性能的影响很小(如果有的话)。这对预超频的 GPU 特别有帮助。Undervolting 是通过你的 GPU 制造商的软件来完成的，比如微星的 afterburner 或者 AMD 的 Adrenaline 镭龙应用。确切的过程将根据您的具体硬件而有所不同，但是来自 PC World 的这个指南是一个很好的概述。

## 减少游戏中的图形设置

最后，如果上述解决方案对你不起作用(而且你买不起更好的 CPU)，尝试降低你玩的所有游戏的游戏内质量。最简单的方法是选择一个较低的图形预设，比如从“高”降到“中”或“低”，看看这是否能解决问题。限制帧速率，或者将 FPS 上限设置为低于当前的值。不要在启用“无上限”帧速率的情况下玩游戏。如果你想保持尽可能多的图形保真度，你也可以自定义游戏中的其他个人设置，尽管这可能需要大量的尝试和错误。
