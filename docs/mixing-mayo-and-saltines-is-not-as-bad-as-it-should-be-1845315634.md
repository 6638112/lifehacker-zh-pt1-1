# 混合蛋黄酱和盐并没有想象中的那么糟糕

> 原文：<https://lifehacker.com/mixing-mayo-and-saltines-is-not-as-bad-as-it-should-be-1845315634>

不可否认，我很白。看看我——我的脸看起来像未煮过的面包卷。还有看看我写了多少 [关于蛋黄酱](https://lifehacker.com/tag/mayonnaise) 。证据确凿。这就是为什么，当育儿编辑 Meghan Walbert 将下面的“零食”建议搁置时，我知道我遇到了某种麻烦。那种暴露我真实本性的麻烦。

蛋黄酱和沙拉酱是一种稍微令人不快的组合，肯定会引起任何关注或听说过它的人的本能反应。每种成分都是相当无害的，但是我们习惯于看到它们*和*其他东西在一起。T4 的问题更多的是蛋黄酱而不是盐。蛋黄酱不应该是主料——她是一个合唱团的女孩！——看到在同样苍白的饼干上分离出摇晃、苍白的调味品的想法令人不安。

反正我现在已经尝过了加蛋黄酱的 saltines(俄罗斯鹌鹑蛋蛋黄酱和 Kewpie 都有)，我恨自己当初没恨到什么程度。从视觉上看，它仍然让我感到害怕。它看起来未完成，贫血，绝望。它尖叫道“大萧条。“But——很像那个时代的花生酱和泡菜三明治——一旦放进嘴里就没那么难受了。

你可以在上面的视频中观看戏剧的展开，但我要告诉你:这不是我吃过的最糟糕的悲伤小吃。咸饼干松脆而咸，但足够温和，突出了蛋黄酱奶油般的浓郁品质。我不会把它给任何人。我甚至不会在任何人面前吃它。但是这些天没什么人在我身边——除了我的男朋友，他已经知道我是什么了。
