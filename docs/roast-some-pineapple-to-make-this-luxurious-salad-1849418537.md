# 烤一些菠萝来做这道美味的沙拉

> 原文：<https://lifehacker.com/roast-some-pineapple-to-make-this-luxurious-salad-1849418537>

有水果沙拉，还有作为沙拉主要成分的水果。前者变成了一团湿漉漉、被氧化的垃圾，浸在邻居的汁液中。把它留给 12 岁的足球训练后的人群吧。更成熟的做法是，用大块的烤菠萝做一道精致的沙拉，给你的晚餐约会对象留下深刻的印象，你可能会，实际上，真的会喜欢这一次。

如果你喜欢 [烤菠萝](https://lifehacker.com/grilled-pineapple-wrapped-in-prosciutto-is-your-new-go-1796140335) ，你一定会爱上*烤*菠萝。它有一种更彻底的焦糖味，弥漫在整个矛中，还有一种更强烈的甜味。虽然烤菠萝在这道菜里会很棒，但我喜欢你从烘烤中得到的强烈味道。它在烤箱中花费的额外时间导致更多的水分蒸发和天然糖的浓缩。这种烤菠萝沙拉对于那些没有烤架的人，或者在冬天偶然发现菠萝的人来说是一个极好的选择。最后，在烤箱下烤五分钟，你就基本完成了。

在盘子里放上两片热菠萝(我喜欢把它们交叉放在一起)，在烤好的水果上面放上三四大块菠萝。你甚至可以在这里参加一个完整的布拉塔舞会，来一次完整的奶酪展示，我会为你鼓掌。撕一些新鲜的罗勒或薄荷，撒在上面，然后放入少许盐核桃。最后在整个盘子上淋上 [白香醋](https://lifehacker.com/use-more-white-balsamic-vinegar-1849406097) ，撒上盐和辣椒粉。必要时你可以用黑醋，但是白醋味道更浓郁，我认为是更好的搭配。温暖的菠萝点缀着奶油般凉爽的布拉塔，菠萝的酸甜，核桃的咸苦，辣椒片的温热，创造了一道诱人的成品菜，既令人放松又完美平衡。

这道菜是一个很好的素食选择，但喜欢吃肉的人可以很容易地添加一堆火腿丝或一点脆培根。虽然下面的食谱给出了你的尺寸，但这份沙拉的最大特点是你可以根据自己的喜好调整水果、奶酪、核桃和香料的数量。

## 烤菠萝沙拉

**配料:**

*   4 根 - 英寸菠萝矛
*   1 球布拉塔
*   杯盐烤核桃(片或半)
*   3 或 4 片新鲜薄荷和罗勒叶
*   茶匙辣椒粉(我用的是 [墨西哥胡椒辣椒粉](https://www.wholespice.com/products/calabrian-peperoncino-chili-flakes) )
*   茶匙盐
*   涂饰用白色香醋

将烤箱预热到 400 华氏度。在纸盘中铺上羊皮纸。

将菠萝矛放在铺有羊皮纸的托盘上，烘烤 25-30 分钟。翻转一次。最后烤一面 5 分钟。(如果你非常着急，你可以把菠萝的每一面烤 5 分钟。你会怀念那种浓浓的甜味，但还是会很棒。)

每盘交叉两个矛。在每个盘子里放半个小球。切几块奶酪，把装饰性地放在菠萝中间。将一半核桃撒在每个盘子里。粗略地撕碎薄荷叶和罗勒叶，把它们放在上面。最后淋上一点白色香醋，在每份食物上分别放一茶匙辣椒粉和盐。立即享受(在你的约会对象离开后，偷偷思考你的第二次布拉塔舞会的计划)。
