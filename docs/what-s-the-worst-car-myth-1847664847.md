# 最烂的汽车神话是什么？

> 原文:[https://jalopnik.com/what-s-the-worst-car-myth-1847664847](https://jalopnik.com/what-s-the-worst-car-myth-1847664847)

汽车被疯狂的神话所包围。关于它们是如何工作的，它们应该如何被照顾，它们应该如何被修改，都有一些神话。有些汽车神话是真的，或者说很久以前是真的，而另一些则没有现实基础。有些只会浪费你的钱，而另一些 [会对你的车造成严重伤害](https://jalopnik.com/why-you-really-shouldnt-run-jet-fuel-in-your-car-5938552) 。

3000英里换一次油是一个即使在今天看起来也很有说服力的神话。

过去，发动机制造公差宽松，用恐龙汁润滑，恐龙汁能相对快速地分解发动机。在那些日子里，你需要每隔几千英里换一次油。甚至也不会阻止你在未来需要一个引擎改造。

但是那些日子已经过去了。现代发动机组装得更好，我们可以获得更好的石油。查阅现代汽车的用户手册，你会发现制造商建议每7000英里、10000英里甚至更长时间更换一次机油。但是，在网上留言板和脸书小组中筛选，你仍然可以发现有人告诉别人每3000英里就要换一次机油。

埃德蒙兹 [说](https://www.edmunds.com/car-maintenance/stop-changing-your-oil.html) 这个问题源于服务业:

> 然而，这种浪费的循环仍在继续，很大程度上是因为汽车服务行业在充分意识到技术进步的同时，继续宣扬3000英里的福音，以此作为保持服务站繁忙的一种方式。

开一辆现代汽车去你当地的换油处，然后检查他们贴在你挡风玻璃上的标签。大概会说只开了3000英里就把车还回去。

遵循用户手册中的建议，你会没事的。没有必要把钱浪费在不必要的服务上。如果你认为你的手册是错误的，然后提交你的油进行分析。当然， [如果你的坐骑又老又邋遢](https://jalopnik.com/i-found-out-what-happens-when-you-dont-change-your-oil-1762314484) ，那就一定要坚持3000英里的变化。

所以换油间隔是一个，但是哪个神话让你气得发抖？最糟糕的汽车神话是什么？
