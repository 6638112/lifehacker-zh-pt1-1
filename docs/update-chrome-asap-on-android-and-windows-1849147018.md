# 尽快在Android和Windows上更新Chrome

> 原文:[https://life hacker . com/update-chrome-asap-on-Android-and-windows-1849147018](https://lifehacker.com/update-chrome-asap-on-android-and-windows-1849147018)

如果在Windows或Android上使用谷歌Chrome，需要尽快到更新。有一个新的浏览器更新为的每个平台包括补丁为新发现的安全漏洞。坏消息是:这些安全漏洞中有一个已知的漏洞，这意味着你的浏览器及其数据处于危险之中，除非你现在更新。

谷歌在其Chrome发布博客上证实了这些针对Windows 和安卓 的更新。Windows用户将需要更新到103.0.5060.114版本，而Android用户将看到103.0.5060.71版本。该公司表示，这些更新修补了Chrome for Windows 的四个安全漏洞和Chrome for Android的三个漏洞。然而，奇怪的是，谷歌从其列表中遗漏了一个漏洞:

*   [$ TBD][[1341043](https://crbug.com/1341043)]高CVE-2022-2294:WebRTC中堆缓冲区溢出。来自Avast威胁情报小组的Jan Vojtesek于2022-07-01
    报道
*   [$ 7500][[1336869](https://crbug.com/1336869)]高CVE-2022-2295:V8中的类型混乱。由avaue和Buff3tts在S.S.L .于2022-06-16报道
*   [$ 3000][[1327087](https://crbug.com/1327087)]高CVE-2022-2296:在Chrome OS外壳中免费后使用。哈利勒·扎尼于2022年5月19日报道(仅桌面版)

虽然保护自己免受所有可能的安全漏洞很重要，但这三个问题中的第一个才是最令人担忧的，因为谷歌在其更新日志中证实，CVE-2022-2294在野外存在已知的漏洞。当一个安全漏洞被利用时，这意味着至少有人知道如何利用这个漏洞，如果他们还没有这样做的话。这是危险的，因为在漏洞被修补和他们的设备被更新之前，Windows和Android上可能的十亿Chrome用户将处于危险之中。

CVE-2022-2294是一个缓冲区溢出漏洞。当一个程序试图向一个内存位置写入超过该位置所能接受的数据时，就会发生这种情况，导致它溢出到另一个空间。这些漏洞并不罕见，但当它们被发现时，恶意用户就可以利用它们。如果Chrome没有在你的电脑或安卓设备上完全更新，你就有危险了。

虽然谷歌目前还没有公开意识到对其他漏洞的利用，但这种利用很可能最终会被发现。您等待更新的时间越长，需要应对的漏洞就越多。

出于某种原因，这些安全缺陷似乎不会影响Mac或iOS用户。因此，你不会在这些平台上看到新的Chrome更新。

## 如何在Wiwindows和一个 droid上更新谷歌浏览器

为了保护您的浏览器和数据，请立即更新Chrome。要在窗口上完成，点击浏览器窗口右上角的三个点，然后选择**帮助>关于谷歌浏览器**。允许Chrome寻找新的更新。如果有可用的更新，选择“重新启动”将更新安装到您的浏览器。

要更新Android上的应用程序，请前往谷歌Play商店。搜索谷歌浏览器，然后选择应用程序旁边的“更新”。如果你启用了自动更新，应用程序可能会自动更新:如果你只看到一个“打开”选项，你就一切就绪了。
