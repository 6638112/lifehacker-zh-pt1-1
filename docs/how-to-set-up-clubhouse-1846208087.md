# 如何建立俱乐部会所

> 原文:[https://lifehacker.com/how-to-set-up-clubhouse-1846208087](https://lifehacker.com/how-to-set-up-clubhouse-1846208087)

我们正在接近这样一个点，club house——这个让人们围坐在一起聊天几个小时的专属应用程序，但实际上是与行业内相当知名的人聊天——开始感觉不那么“酷”社交服务正在蓬勃发展，但是让每个人都能使用麦克风也带来了一系列问题。

尽管如此，如果你想在获得某个已经在使用该服务的人的邀请后查看Clubhouse，有一些方法可以让你使用起来不那么烦人。

在我讲到 [的提示](#tips) 之前，先说一下那些邀请。截至本文撰写之时，你仍然只能通过被邀请才能让*进入*俱乐部。作为一个俱乐部会员，让我告诉你，秘密的诱惑比现实更有趣。

Clubhouse很整洁，但它感觉像是当今的时尚——许多想要成为影响者的人、想教你成为影响者的人，或者已经非常知名的人，这款应用程序只是一种简单而“精英”的方式，无需制作YouTube或Twitch流。

一旦闸门打开，每个人都想在Clubhouse上建立自己的品牌，我认为这将真正削弱这项服务的可搜索性——或者它只会变得*像*一样抽搐，在任何给定的类别中，前20名似乎是唯一真正重要的人，90%的人如果不能让5个人听他们一个小时的聊天，就会感到无聊。

所以，是的。如果你还没有被邀请，不要担心。*请不要购买其他人的邀请*或者，更糟糕的是，被骗去放弃你的个人信息或帐户安全，而对方承诺会邀请你加入俱乐部。

### 我们把俱乐部变成酷孩子俱乐部的最佳建议

当你最终下载应用程序并加入Clubhouse时，你会被要求将你的联系人连接到Clubhouse，这样你就可以在Clubhouse上自动找到并关注你现实生活中的朋友。那很好。当你进入“选择你的兴趣”部分时，要知道你现在不必选择任何东西。事实上，我鼓励你暂时跳过这个过程，以免你发现你的主要俱乐部会所订阅的内容充满了你不关心的垃圾聊天室。

你可能最感兴趣的是听到你的朋友和你在俱乐部会所找到的其他低调的名人，而不一定是800种不同的关于“网络”的潜在讨论你可以随时通过应用程序的主**设置屏幕>兴趣**来调整你的兴趣。

你还会得到一份俱乐部会所推荐用户的名单。这是你想要像躲避瘟疫一样躲避的部分。不要因为应用程序希望你关注，就随意关注一些人；当人们向你证明他们创造了你想听的有趣的对话和经历时，就跟随他们。否则，你的俱乐部饲料将得到真正的垃圾，真正的快。

之后，你会被要求允许来自应用程序的通知。现在说是，但要知道你很快就会改变这个设置，以免当你的一个朋友开始聊天时你会收到一个ping(这可能会很烦人)。

那你就在app里了！在你跳进房间听别人说什么之前，你应该做的第一件事是点击Clubhouse的设置(在右上角你的个人资料图标下)。

即使你一开始不想调整你的通知频率，也要知道你有一个五级选项，以防你被应用程序困扰(或者想要一个俱乐部更新的消防水管)。我关掉了“包括趋势房间”，因为这是我俱乐部里的大部分垃圾邮件。

最后，在这里你可以调整你的兴趣——如前所述——连接或断开你的Twitter和Instagram账户，这取决于你希望Clubhouse能够推荐多少新的潜在朋友。请记住，在我写这篇文章的时候，联系要么会让他们进入你的公共俱乐部档案，而且没有办法隐藏这些社会联系。

要特别注意保持你的兴趣，或专注于你真正关心的几个主题，因为Clubhouse将使用这些来填充你的“即将为你”提要，这是一个预定的俱乐部聊天列表，可通过Clubhouse主屏幕上的日历图标访问。
