# 如何在 Xbox 上玩区域锁定游戏

> 原文：<https://lifehacker.com/how-to-play-region-locked-games-on-your-xbox-1847574208>

像任天堂 Switch 和索尼的 PlayStation 4 和 5 游戏机一样，Xbox Series X| S 和它的老兄弟 Xbox One 是不受地区限制的设备。这意味着你可以购买和玩在其他国家发行的 Xbox 游戏，并在你的主机上玩。



但是当 [任天堂](https://lifehacker.com/how-to-play-japanese-and-european-eshop-games-on-your-r-1847556998) 和 [索尼要求](https://lifehacker.com/how-to-play-playstation-games-that-arent-available-in-t-1847567325?rev=1630013630442) 地区账户从另一个国家的店面购买和下载数字游戏，甚至在线玩某些游戏时，微软就宽松多了。

您可以轻松访问另一个国家的 Microsoft Store 来购买新游戏、访问地理锁定的内容，甚至从访问 Microsoft Store 的任何设备下载您的库中的游戏和应用程序的地区版本。而且你不需要创建新的 Xbox 帐户或将你的帐户迁移到一个新的地区，就可以做到这一点，就像你在旧的 Xbox 游戏机上必须做的那样。您只需临时更改 Microsoft Store 的语言和位置设置。

## 如何更改 Microsoft 商店的位置和语言

有几种方法可以改变微软商店的地区:从你的 Xbox 主机的设置；通过 Windows PC 的设置在桌面上；或者在您首选的浏览器中的 Microsoft Store 网站上。

**在 Xbox 系列 X|S 和 Xbox One 上:**

1.  按下控制器上的 **Xbox** 按钮，调出 Xbox Series X| S 或 Xbox One 上的指南菜单
2.  进入**配置文件&系统>设置>系统>语言&位置。**
3.  从**“位置”**列表中选择您想要的位置。
4.  选择**“立即重启”**以应用更改并重启控制台。
5.  在控制台重新启动后，打开 Microsoft Store 页面，，它将根据您的控制台的新位置加载商店。

**在微软商店网站:**

1.  打开 [微软商店网站](https://www.microsoft.com/en-us?icid=CXSMC_storehelp) 。
2.  向下滚动到页面底部，然后单击地球图标旁边的语言。
3.  从列表中选择一个位置。
4.  网站将重新加载并启用新的设置。

**在您的 Windows 电脑上:**

1.  打开 Windows 开始菜单。
2.  前往**设置>时间&语言>地区。**
3.  从**“国家或地区”**列表中选择一个新位置。
4.  关闭设置菜单。
5.  打开 Microsoft Store，它应该会根据您的区域设置打开到该商店。

请注意，如果您选择使用不同语言的国家/地区，这可能会更改您的设备和 Microsoft 商店中的语言。谷歌翻译可以帮助你浏览这些菜单，但在大多数情况下，所有地区的布局和选项都是一样的。

## 从其他国家购买、下载和玩游戏

一旦你加载到微软商店，你就可以浏览、购买和下载游戏。但是，您需要一个有效的支付选项来完成交易。一些银行和信用卡可以在国际上使用，但大多数不能。如果他们这样做，你可能会被收取转换费。相反，您可以购买所需货币的 Microsoft Store 礼品卡代码，并以这种方式结账。这些 [很容易从亚洲](https://www.play-asia.com/search/xbox+gift+card) 和其他网上零售商那里买到。

下载您的新游戏后，您可以像平常一样使用您的主帐户玩游戏，并且只要您拥有有效的 Xbox Live 或 Game Pass 订阅，您就可以访问在线模式，即使您是用 U. S. 美元购买的。

至少，那是*通常*的情况。在极少数情况下，一些游戏和应用程序可能会锁定您，即使您的主机设置被设置为正确的区域。如果发生这种情况， [你家路由器](https://www.linksys.com/us/r/what-is-a-wifi-access-point/vpn-router/) 的 VPN 可能会帮助你看起来好像你真的是从要求的国家连接。
