# 什么是“感觉记忆”(你怎么知道你有感觉记忆)？

> 原文：<https://lifehacker.com/what-is-a-feeling-memory-and-how-do-you-know-youre-hav-1849349258>

惊恐发作是一种创伤反应，典型的是对某种触发因素的反应，比如在创伤时刻看到出现的人或物。不过，有时候，恐慌发作或情绪的负面变化可能没有明显的解释。这是一种令人沮丧的，潜在的可怕经历，但并非闻所未闻——你可能只是在经历一种完全潜意识触发的“感觉记忆”或“创伤记忆”。

## **什么是感觉记忆？**

正如创新技术研究所医学虚拟现实主任、南加州大学研究教授斯基普·里佐博士告诉 Lifehacker 的那样，感觉记忆是“一种感知配置，即使它对你所处的世界没有威胁，也会唤起那种触发反应。”

本质上，你的大脑可能会将某些环境因素与过去的创伤经历联系起来。里索提供了一个老兵的例子，他可能会在一条布满垃圾的路上开车时有一种感觉记忆，这让下意识地提醒他们在部署时开车，害怕埋在垃圾袋等普通物品中的爆炸装置。这个假想场景中的老兵可能根本没有意识到路边的袋子是引发他们的原因。

在 [中，一位*今日心理*](https://www.psychologytoday.com/us/blog/making-the-whole-beautiful/202207/youre-not-crazy-youre-having-feeling-memory) 的解释者，婚姻和家庭治疗师安妮·莱特给出了另一个例子:一个女人可能会感到恶心，当她脾气平和的丈夫在晚上脱下他的皮带时，没有意识到这是因为她将那个动作与她小时候目睹的父亲实施的身体虐待联系在一起。

里佐说，当感觉记忆出现时，反应可能与患有创伤后应激障碍的人经历的相似，尽管有感觉记忆并不一定意味着一个人可以被诊断为创伤后应激障碍。

## 为什么感觉记忆的触发点很难确定？

如果你经常经历强烈的情绪不适，但不知道是什么触发了它，你应该开始详细记录每件事，包括事情发生时你在哪里，你在做什么。考虑一下你在每一个事件中看到、闻到、尝到、感受到和听到了什么。

“某些刺激模式唤起了深层的记忆，有时人们——当他们受到创伤时——会分离，所以他们失去了对事件的简单认知回忆，”里佐解释说。

在她的文章中，赖特指出，当威胁发生时，无论是真实的还是感知的，，大脑的额叶可以关闭“作为一种自我保护措施，，“而脑干和边缘系统保持正常运行，因此创伤体验存储在大脑中，“作为一套缺乏连贯叙事的感觉和身体反应。”本质上，你的身体会通过感官细节记住创伤事件，而你的意识大脑可能不会识别。

## **如果你正在经历情感记忆，该怎么办**

好消息是有治疗选择。作为一名临床心理学家，里佐自 1995 年以来一直在为临床应用构建虚拟现实系统。这些虚拟现实系统可以成为长期暴露疗法的工具，这是一种有证据支持的治疗方法，可以帮助患者在安全的环境中面对和再处理困难的情感记忆。

Rizzo 警告说，暴露疗法是一个漫长而艰巨的过程，而且退出率很高，但却是一种安全有效的治疗手段。临床医生控制着虚拟现实体验，所以他们开始时，“就其挑衅性而言非常小，然后逐渐升级”到更具挑衅性，他解释道。

与恐惧症 一样，暴露疗法在这里是一个很好的选择，但也有其他的选择。里佐指出了认知处理疗法，这种疗法“不是关于高强度暴露，而是帮助一个人回顾它。”患者写下或描述他们创伤的故事，和治疗师然后和他们一起回顾故事帮助他们认识和面对功能失调的思维。例如，一个因所发生的事情而责备自己的攻击受害者正在经历功能失调的认知，这可能是习惯性情绪反应(如感觉记忆)的一个促成因素。利用 CPT 的治疗师将帮助那个人重新构建他们的认知，希望防止随之而来的负面情绪反应。
