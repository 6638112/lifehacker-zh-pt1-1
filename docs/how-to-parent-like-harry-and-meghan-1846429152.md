# 如何像哈里和梅根一样为人父母

> 原文：<https://lifehacker.com/how-to-parent-like-harry-and-meghan-1846429152>

即使那些高度期待奥普拉采访哈里王子和梅根汗·马克尔的人可能也会发现自己被昨晚两小时的特别节目 中披露的各种信息震惊了。随便挑几个你可能已经看过的头条新闻——从梅根的自杀想法到至少一名皇室成员对他们孩子肤色可能有多黑的担忧——今天都有很多可以谈论的。但作为一名育儿作家，我很自然地通过这个特殊的镜头过滤了他们的许多启示。

采访中充满了关于好父母如何优先考虑整个家庭的健康、安全和幸福的提醒，即使正确的事情也是困难的事情。从哈里和梅根所做的决定中，我们可以总结出一些为人父母的经验。

### 你可以做出艰难的选择

我从未成为皇室成员——也没有嫁入皇室——但我不得不想象，将自己和家人从一个强大的、存在了几个世纪的体系中解脱出来是很困难的，这个体系实际上就是你的整个世界。尤其是当你的每一个举动都被一群小报媒体报道和评判时，他们似乎铁了心要把你描绘成最负面的一面，我认为与其试图为自己铺路而招致更多的批评，不如试着顺从他们。

但是，当面临继续成为真正的英国皇室成员(不管这会给你带来多少压迫)和挣脱束缚，让你和你的孩子过上更正常的生活之间的选择时，他们做出了可以说是更艰难的选择。通过这样做，他们为自己的孩子(儿子阿奇和梅根现在怀上的女儿)树立了一个早期的榜样，那就是做对你来说是正确的事情，即使是具有挑战性的事情，也比做别人期望你做的事情要好。

### 我们是孩子最好的拥护者

在每个父母的旅程中，都有这样的时刻，为你的孩子辩护是必要的——即使这很困难或尴尬。家长可能需要在学校倡导他们孩子的特殊学习需求，尽管对学校想要提供的服务有所抵触。或者，我们可能需要为他们争取他们需要的心理健康服务。在梅根和哈里的案例中，他们说当他们的儿子被拒绝授予皇家头衔(王子)时，他们需要为他的人身安全辩护，而这个头衔本可以让他获得 T2 的个人安全。

倡导你的孩子接受他们自己的安全细节不是我们大多数人必须做的事情，但我们也不必向我们的祖母行屈膝礼。然而，最终，他们被拒绝了，这至少是他们抱起孩子向城堡大门走去的部分原因。

### 我们需要优先考虑我们自己的心理健康

当你刚为人父母，正在与任何焦虑、抑郁或其他心理健康问题作斗争时，寻求帮助可能会很难。毕竟那应该是你一生中最快乐的时光！这应该是关于*宝宝*和*宝宝*的需求——而不是你的。即使在最“正常”的情况下，承认你在挣扎并寻求帮助也需要很大的勇气。

但是当梅根开始有自杀的想法时——她参加了一个皇家活动，而不是独自呆在家里，因为害怕她可能会做什么——她确实寻求了帮助。她告诉丈夫她的感受，并说她向一位资深王室成员寻求帮助(被告知她不可能寻求住院治疗)。

即使是现在，以如此公开和脆弱的方式谈论她的经历，也向所有初为人父母的人传达了一个重要的信息——还有她的孩子。如果你在挣扎，寻求帮助；因为每个人都应该得到他们需要的帮助，而不必为自己需要而感到羞耻。

### 勇敢面对恶霸是很重要的

维护自己是大多数父母希望他们的孩子能够做到的事情。但是当你是皇室成员时，你通常被期望为了更大的(家庭)利益保持沉默。然而，当你经历的不仅仅是来自公众和媒体的种族歧视，还有来自你自己家庭的种族歧视时，那就是扯淡。

哈利和梅根详细描述了他们感受到的被虐待和被压制的方式，为他们的孩子树立了榜样，他们的声音很重要——甚至(或者尤其)当你面对一个比你强大得多的实体时。

### 拯救动物是一件好事

在采访中最甜蜜的时刻，奥普拉拜访了这对夫妇在加州蒙特西托的家，他们和他们的儿子以及两只宠物狗(其中至少有一只是梅根说她从肯塔基州的动物收容所救出来的)住在那里。狗显然是很好的宠物，但母鸡也是，这家人最近还从一个工厂化农场救出了母鸡。他们甚至建造了一个鸡舍，上面有一个标志，标明“阿奇的小鸡客栈:成立于 2021 年”，这是各种有益健康和甜蜜的。
