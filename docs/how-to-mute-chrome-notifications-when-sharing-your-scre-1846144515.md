# 共享屏幕时如何将 Chrome 通知静音

> 原文：<https://lifehacker.com/how-to-mute-chrome-notifications-when-sharing-your-scre-1846144515>

就像视频聊天一样，屏幕共享现在是任何在家工作或社交的人的必备工具。不幸的是，如果你不采取预防措施防止意外的*过度分享*，分享你的设备屏幕可能会有隐私风险——或者只是尴尬。为了帮助减轻这种风险，谷歌正在测试一项新功能，在你共享 Chrome 窗口时静音 Chrome 通知，这样你的浏览器就不会在没有警告的情况下播放你的脏衣服。



虽然 Google Meet 已经自动关闭了 Chrome 通知，但新的 Chrome 功能可以与 Messenger、Discord、Microsoft Teams 和 Zoom 等屏幕共享应用一起使用。打开它可以防止 Chrome 与您的观众共享通知细节，并有助于保持您的演示文稿专业且不受干扰。

这不是一直停用通知，而是仅在您共享屏幕时将它们静音，它们不会被忽略或丢弃。当你完成分享后，你将能够从 Gmail、Drive 和任何通过 Chrome 向你发送通知的第三方网站查看你错过的任何内容。

最终，通知静音将通过稳定的 Chrome 频道对所有谷歌账户开放，包括普通用户、谷歌工作区和 G Suite 企业用户，但你现在可以在 Chrome Canary 中尝试早期版本。Chrome Canary 用于测试未完成的功能和设置，因此容易出现稳定性错误和崩溃，但如果你想在演示过程中静音 Chrome 通知，你必须使用这个版本。下面就来试试:

1.  [为你的设备下载 Chrome 金丝雀](https://www.google.com/chrome/canary/) 。
2.  打开浏览器，然后进入**chrome://flags/# mute-notifications-due-screen-share。**
3.  从下拉菜单中选择**“启用”**。
4.  重启 Chrome Canary 以应用更改。

启用该设置后，Chrome 会在屏幕共享时隐藏所有接收通知的细节。通知仍然会出现在屏幕上，带有 Chrome 图标，但它只会简单地说“在共享您的屏幕时，内容是隐藏的。”选择**“关闭”**隐藏弹出窗口。如果您想显示所有通知信息，还可以选择**【显示全部】**。否则，隐藏信息只有在停止屏幕共享后才可见。

如果你不想下载 Chrome Canary，但仍然需要一种方法来静音通知——或者你根本不想看到任何 Chrome 提醒——你可以在 Chrome 的设置中禁用它们，方法是进入**设置>隐私&安全>站点设置>权限>通知。**
