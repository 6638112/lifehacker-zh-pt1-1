# 用这个“庆祝公式”庆祝父亲节

> 原文：<https://lifehacker.com/use-this-celebration-formula-for-father-s-day-1848890743>

父亲节给我丈夫买什么礼物总是让我心里充满了一点点惊喜。他不挑剔，所以如果他没有得到某样特别的东西，他不会不高兴。(事实上，*没有什么*是他真正想要的，除了他“能用的东西，”这些东西帮不上忙。)他已经收到了所有装裱好的运动用品，超级马里奥主题的“超级爸爸”杯，以及一个男人可能想要的烧烤工具。(我看着你，三年后依然未开封的不锈钢“野兽注射器”)。



给爸爸买东西可能很难。通常，他们已经拥有或不会使用我们赠送给他们的许多装备——除了一件“行走的父亲”僵尸 t 恤，这款 t 恤售价 16.99 美元，似乎还不够。但是抖音的创造者@ kelsewhathelse[最近分享了她创造一个伟大的父亲节的获奖“庆祝公式”](https://www.tiktok.com/@kelsewhatelse/video/7109498353095855403?is_copy_url=1&is_from_webapp=v1) 。

## 三个—“庆祝公式”步骤

事情是这样的。为了创造一个难忘而愉快的父亲节，问你生活中的父亲三个简单的问题:

1.你想干嘛?

2.想吃什么呢？

3.你想收到什么？

这样，如果他回答，“去博物馆，吃泰国菜，并收到一个棋盘游戏”(在@ kelsewhatelse 的视频中给出的例子)，这就给送礼者提供了一个机会，可以准确地表达爸爸想要的东西，同时还保留了一丝惊喜。有了这些信息，你可以根据他们的兴趣仔细挑选一个博物馆，一个他们喜欢吃的餐馆，一个他们可能会期待的游戏——但具体是什么他们要到开业后才能知道。

这些问题给你生活中的父亲(或者你正在庆祝的任何人，这个公式不是特定于父亲的)留下了足够的空间，让他们按照自己的意愿变得模糊或具体。他们可以让它敞开着，说他们想“放松”或“做些冒险的事情”，或者他们可以指出一项活动，例如:“我想看一场棒球比赛。”

无论他们做出什么决定，它都为每个人的成功做好了准备，同时最大限度地减少猜测，并仍然允许深思熟虑。通过提前制定一个请求，收件人将得到三件肯定会让他们高兴的事情，让他们感到被倾听和被关心。送礼物的人也不必花几个小时仔细研究那些不常见的物品，试图找到完美的定制卷尺、俗气的爸爸笑话沙发枕头，或者他们的父亲或丈夫永远不会用的烧烤用具。
