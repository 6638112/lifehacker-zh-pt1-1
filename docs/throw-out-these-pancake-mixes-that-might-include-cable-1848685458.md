# FDA 说，扔掉这些可能含有电缆碎片的煎饼混合物

> 原文：<https://lifehacker.com/throw-out-these-pancake-mixes-that-might-include-cable-1848685458>

如果你的周末(或工作日)早餐通常包括混合制作的华夫饼或薄煎饼(这里不做评判)，你可能要重新考虑你的菜单了。据 报道，这是因为在克罗格和沃尔玛全国各地销售的某些酪乳煎饼和华夫饼“由于潜在的异物污染”而被召回。以下是需要了解的内容。



## 为什么煎饼和华夫饼被召回

所有被召回的产品都是由大陆面粉厂生产的，有问题的酪乳煎饼和华夫饼干在克罗格和沃尔玛商店品牌下销售(分别为 [克罗格](https://www.fda.gov/safety/recalls-market-withdrawals-safety-alerts/continental-mills-recalls-kroger-buttermilk-pancake-waffle-mix-due-possible-foreign-material#recall-announcement) 和 [超值](https://www.fda.gov/safety/recalls-market-withdrawals-safety-alerts/continental-mills-recalls-walmart-great-value-buttermilk-pancake-waffle-mix-due-possible-foreign) )。

在一些煎饼和华夫饼干混合物中发现了用于清理生产线的电缆 的碎片后，制造商发布了召回令。到目前为止，没有消费者报告在产品中发现电缆碎片，也没有相关疾病或伤害的迹象。

## 如果您购买了召回的产品，该怎么办

如果你最近从任何一家零售商那里购买了上面提到的煎饼和华夫饼，不要吃它们。 [FDA 建议扔掉产品](https://www.fda.gov/safety/recalls-market-withdrawals-safety-alerts/continental-mills-recalls-kroger-buttermilk-pancake-waffle-mix-due-possible-foreign-material#recall-announcement) ，然后拨打召回电话热线 1-800-578-7832(周一至周五，上午 10 点。美国东部时间下午 7 点)退款，或了解更多信息。超值组合还可以通过 [返还给沃尔玛零售商](https://www.fda.gov/safety/recalls-market-withdrawals-safety-alerts/continental-mills-recalls-walmart-great-value-buttermilk-pancake-waffle-mix-due-possible-foreign) 进行退款或更换。
