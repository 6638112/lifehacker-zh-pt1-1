# 你应该试试这些复古汽水混合物吗？

> 原文：<https://lifehacker.com/should-you-try-these-retro-soda-pop-concoctions-1846089449>

很不幸，我是那种喝全脂牛奶的变态。我喜欢它，尤其是当它与花生酱和果冻三明治、【to】香蕉或任何一种巧克力甜点一起食用时。但我也因独自喝一杯冰镇牛奶作为宵夜而出名，因为我还是一个真正的怪人。

但即使是我也有我的极限，至少在饮料方面，这就是为什么我从来没有尝试过我们今天供应的复古乳制品饮料。我听说过百事可乐牛奶(因为拉维恩&雪莉)，但是从来没有给自己弄过。不过最近，在我们的 EIC·爱丽丝·布拉德利(她也是百事可乐的忠实粉丝)的敦促下，我决定最终尝试一下，还有七喜牛奶(我在推特上看到的)，和辣味胡椒博士(这是 1999 年艾丽西亚·希尔维斯通车上的特色，*过去的爆炸*)。

这是一段不寻常的旅程。每一口都是惊喜，我的预期都被颠覆了。我喜欢我认为我会讨厌的东西，只容忍我认为我会喜欢的东西。有史以来第一次，乔尔和我有了分歧。(后来我们和解了。)

如果没有别的，这些怪异的复古汽水混合物告诉我，有些东西真的需要品尝才能得到正确的评价。尽管你可能认为你了解自己的味觉、心脏和头脑，但总有一些东西让你质疑你认为对这三者都了解的一切。这就是在这次味觉测试中发生在我身上的事，我永远地改变了。(不幸的是，乔尔过得并不愉快，但他并不是一个喝牛奶的怪人。)
