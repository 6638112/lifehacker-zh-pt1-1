# 这款浏览器可以同时使用Chrome和Firefox扩展

> 原文:[https://life hacker . com/this-browser-can-use-chrome-and-Firefox-extensions-at-t-1849038809](https://lifehacker.com/this-browser-can-use-chrome-and-firefox-extensions-at-t-1849038809)

如果你有一台苹果电脑、 iPhone或者 iPad ，并且你正在从你的网络浏览器中寻找新的东西，你应该看看猎户座。新浏览器允许你运行谷歌Chrome扩展和火狐插件，同时声称不会牺牲你的隐私。Orion基于WebKit，这是Mac上的Safari和iPhone上所有浏览器使用的浏览器引擎。

Watch

由于Orion最近才进入公测，它仍然有一些粗糙的边缘，但总的来说，它似乎是一个有前途的替代品，你现在可以尝试。不是每个扩展都能在猎户座上工作，但是最受欢迎的，像 uBlock Origin，Dark Reader，和 SponsorBlock，都能完美工作。

要安装浏览器，可以去 [猎户的网站](https://browser.kagi.com/) 。(注意，iOS和iPadOS版本的Orion需要通过苹果的 [TestFlight](https://apps.apple.com/app/testflight/id899247664) 应用安装，因为它仍处于测试阶段。根据我们的测试，与iOS版本相比，Mac版本的Orion似乎要稳定得多。

## 如何在Orion上安装扩展

Orion不要求您创建帐户来安装扩展。你可以访问官方页面的 [Chrome扩展](https://chrome.google.com/webstore/category/extensions) 或 [Firefox插件](https://addons.mozilla.org/en-US/firefox/) ，找到任何扩展，点击 **添加到猎户座**即可安装。这些步骤与移动版本相同。

一些扩展，如 1Password的浏览器扩展，不能与Orion一起使用，因为1Password尚未授权。还是，你可以使用 [1Password 8的通用自动填充功能](https://lifehacker.com/why-1password-is-now-the-best-password-manager-for-mac-1848947731) 让它工作。你也可以在Orion上使用 [uBlock Origin](https://lifehacker.com/change-these-ublock-origin-settings-for-even-more-priva-1848995058) ，但是开发者建议你首先禁用Orion自己的内容拦截器，因为同时运行这两个会导致冲突并破坏网站。

## 猎户座有多快？

同时使用太多的扩展似乎会减慢猎户座的速度。如果你像我一样使用较旧的MacBook，如果你安装了太多的扩展并打开了大约10-20个标签，你会开始看到浏览器(和你的整个电脑)爬行。当然，那个问题适用于所有浏览器，这一点要记住。

## 其他有用的猎户座功能

安装扩展很棒，但是Orion还有一些其他有趣的特性。一个是能够将打开的标签列表移动到左侧栏作为垂直视图，这样更容易找到你要找的标签。这项功能在iPhone和Mac上都可以使用。如果你正在阅读一篇文章，并在一个新标签页中打开一个链接，标签页边栏会添加一个嵌套标签页。如果你是一个收藏者，这是一个很好的分组标签的方法。

当你打开浏览器时，Orion会要求你选择一个默认的搜索引擎，它支持几乎所有你想使用的搜索引擎。它的开发者也在开发自己的搜索引擎Kagi，但默认情况下，浏览器不会强迫你使用任何特定的搜索引擎。

如果你在MacBook上使用Orion，你可以使用它的低功耗模式来减少电池消耗，Orion会暂停浏览器中不活跃的标签。Mac版还允许你覆盖一些网站实现的复制粘贴功能，尤其是银行网站。

## 猎户座的商业模式是什么？

每当我看到一个新的应用程序，我想到的第一个问题是它是否有利于隐私。Orion和所有应用一样，声称是安全和隐私的。为了支持他们的声明，猎户座 [称](https://browser.kagi.com/faq.html#privacy) 它不包括遥测，并且配备了一个内容拦截器，除了广告和跟踪之外，还可以阻止浏览器指纹识别。

目前，该公司称其收入依赖于社区。那些希望支持该浏览器的人可以每月花5美元订阅Orion+,这将为他们带来一个新的图标，提前获得即将推出的浏览器版本，并与开发者交流。

您还应该知道，Orion会使用iCloud sync跨设备同步您的标签、历史记录和设置。如果你需要，你可以禁用它，但是它默认是启用的。
