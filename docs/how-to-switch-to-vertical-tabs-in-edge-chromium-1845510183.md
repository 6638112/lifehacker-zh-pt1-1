# 如何切换到边缘铬垂直标签

> 原文：<https://lifehacker.com/how-to-switch-to-vertical-tabs-in-edge-chromium-1845510183>

Edge 的实验性侧边栏从 8 月份开始在 Edge Canary 中提供，但这个备受欢迎的功能现在已经在浏览器的开发者版本中推出。这使它更接近我们大多数人使用的稳定版本。在那之前，你可以先看看 Edge Chromium 的开发和金丝雀版本中的特性。 (W e 推荐前者，日常使用会比后者稳定很多。)

Watch

1.  下载 [Edge Dev 或者 Edge Canary](https://www.microsoftedgeinsider.com/en-us/download)T3，打开浏览器。
2.  单击窗口左上角的选项卡图标，使用垂直选项卡菜单。再次单击图标切换回水平标签栏。

所有普通的标签功能都适用于垂直标签。单击并拖动选项卡以更改它们在菜单中的顺序，单击 **"+"** 图标打开一个新的选项卡，或单击 **"X"** 关闭它们，右键单击选项卡以打开选项卡管理菜单，您可以在其中锁定选项卡、静音、刷新等等。

还有一些特定于垂直选项卡的控件:

*   单击并拖动垂直选项卡栏的边，以调整其大小
*   单击选项卡列表顶部的**“<”**图标，最小化该栏。

在体验了 Edge 垂直标签的早期版本一段时间后，我明白了为什么它如此受欢迎。管理和查看你打开的标签页要容易得多，隐藏标签页列表给 Edge 一种非常“开放”的感觉布局，尤其是如果你已经隐藏了浏览器的其他工具栏。也就是说，我在使用 Dev build 时经历了几次意外的崩溃，尽管这不一定是垂直标签菜单的错。

Edge Canary 和 Edge Dev 是 Edge 内部版本中最具实验性的，为开发者提供了各种各样的调整和功能，但它们也更不稳定——如果你使用其中任何一个，都会出现崩溃和错误。值得庆幸的是，Edge Dev 和 Canary 作为独立的应用程序安装，不会覆盖你的普通 Edge 浏览器，所以如果你遇到问题，至少很容易切换回来。

### 另一个选择是:在 Edge 中使用 Chrome 的垂直标签浏览器插件

我们不知道 Edge 的稳定版本何时会有新的标签菜单，但你可以安装 [垂直标签 Chrome 扩展](https://chrome.google.com/webstore/detail/vertical-tabs/pddljdmihkpdfpkgmbhdomeeifpklgnm) 作为短期解决方案，不需要不稳定版本的浏览器— [或其他](https://lifehacker.com/how-to-try-out-edge-chromiums-latest-features-months-be-1842562597) ，如果你不是那个附加组件的特别粉丝的话。

垂直标签插件*做的事情*和 Edge 的大部分是一样的，除了一个主要的例外:正常的水平标签菜单仍然显示在浏览器的顶部，而不仅仅是垂直标签。我在 Edge 的设置和垂直标签的附加配置中寻找隐藏普通标签栏的方法，但你唯一真正的选择是全屏模式浏览(F11)。

我还注意到 Edge 中启用垂直标签时出现了一些奇怪的错误，比如点击出现在页面的错误部分。当在谷歌文档中点击移动光标时，这一点特别明显(也很烦人)。

根据我使用 Edge 的垂直标签功能和垂直标签插件的时间，我建议下载 Dev 或 Canary 版本，如果你热衷于早期尝试 Edge 的垂直标签选项，请使用该版本。
