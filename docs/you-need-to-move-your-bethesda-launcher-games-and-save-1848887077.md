# 您需要移动您的Bethesda启动器游戏，并将数据保存到Steam

> 原文:[https://life hacker . com/you-need-to-move-your-Bethesda-launcher-games-and-save-1848887077](https://lifehacker.com/you-need-to-move-your-bethesda-launcher-games-and-save-1848887077)

贝塞斯达工作室，许多流行系列如《上古卷轴》、*《毁灭战士》*和*《辐射尘》*的发行商，将于5月11日关闭其PC发布平台。这意味着你将不再能够通过该应用程序购买、下载或玩Bethesda游戏。然而，该公司正在让玩家将他们购买的Bethesda发布的游戏、DLC、保存文件和某些特定账户数据(如游戏中的货币)完全免费地移植到Steam上。

虽然Bethesda Launcher将于5月11日关闭，但该公司很清楚 [这不是一个转移截止日期](https://bethesda.net/en/article/2RXxG1y000NWupPalzLblG/sunsetting-the-bethesda-net-launcher-and-migrating-to-steam) ，即使在Launcher应用程序关闭后，你也可以转移你的游戏和数据。

然而，由于您的Bethesda游戏和信息将无法访问，直到您将您的信息传输到Steam，因此最好尽快完成该过程。你需要做的是:

## 第一步:准备你的Steam账户

首先，确保你有一个Steam账户，可以把你所有的Bethesda信息转移到这个账户。如果你玩电脑游戏，你可能已经有一个，但是如果你需要，你可以通过Steam网站 [创建一个新的](https://store.steampowered.com/join/) ，并从这里 [下载Steam应用](https://store.steampowered.com/about/) 。

是的，蒸汽是你唯一的转移选择。你不能迁移到Epic Games Store或GOG等其他平台，也不能迁移到PlayStation或Xbox等非PC平台，即使Bethesda的游戏在那里可用。

## 第二步:将你的Bethesda游戏迁移到Steam

如果您已经安装了Steam并准备就绪，现在可以执行数据传输了:

1.  进入 [Bethesda的在线转账](https://bethesda.net/en/dashboard?cogs_modal=login&redirect_to=account%2Ftransfer-library) 工具，使用您的Bethesda.net账户登录。
2.  登录后，点击右上角的您的帐户名称并选择**“传送库”**
3.  按照屏幕上的指示，在出现提示时点击**“开始蒸汽传输”**。
4.  等待传输完成。
5.  如果您在这些步骤中遇到任何问题，请咨询 [Bethesda的支持常见问题解答](https://bethesda.net/en/article/2RXxG1y000NWupPalzLblG/sunsetting-the-bethesda-net-launcher-and-migrating-to-steam) 寻求帮助。

一旦传输完成，您的信息将被转移到Steam，任何购买的游戏和DLC现在都将被添加到您的Steam库中。然而:D还没有玩游戏！如果您想从停止的地方继续玩，我们需要再完成一步来传输您保存的数据。

## 步骤3: 将保存的数据导入Steam

从技术上来说，这一步是可选的，但如果你已经在《辐射4》这样的游戏中投入了数百个小时，并希望保持这种进展，或者你深陷于《T2》的《毁灭战士永恒》的单人战役中，不想重新开始，那么这一步是值得完成的(注意，像《t 4》《上古卷轴Online》这样的游戏的在线数据，或者《毁灭战士》*或《雷神之锤》中多人模式的基于账户的统计数据会自动转移)。*

*在Bethesda的游戏中，手动导入保存数据的过程各不相同，但该公司有 [一个有用的指南](https://help.bethesda.net/#en/answer/55036) ，为每个可以导入保存数据的游戏提供一步一步的指导。*

*一旦以上所有步骤完成，您就可以开始在Steam上玩您的Bethesda游戏，并从您的PC上安全卸载Bethesda Launcher应用程序。*
