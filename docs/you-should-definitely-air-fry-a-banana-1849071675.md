# 你绝对应该空气油炸香蕉

人们总是想把香蕉做成冰淇淋，这又能怪谁呢。冰激凌真的很难吃。她甜美而富有，冰冷却令人欣慰，是节食开始后最先要做的事情之一。这些近似值并不坏——几乎崩溃的Pinterest的冷冻香蕉""无可否认很好吃，来自抖音用户 [Jen Jones](https://www.tiktok.com/@thejenjones) 的空气油炸香蕉船因某种原因而走红。香蕉味道不错，但不是冰淇淋。

话虽如此，空气油炸香蕉也有美。琼斯将她的蛋糕切片，用天然花生酱、核桃、巧克力填充，并撒上肉桂，然后在400度的空气油炸锅中烹制“五到六分钟”她最后喝了酸奶，称之为“健康版的香蕉船”

虽然不涉及冰淇淋，但最终产品很诱人，就像几年前在Pinterest上非常受欢迎的空气- 油炸版 [营火香蕉船](https://www.foodnetwork.com/recipes/ree-drummond/campfire-banana-boats-recipe-2119815) 。(Pinterest和香蕉有什么关系？)我(显然)不得不自己做空气油炸香蕉片，但在尝试琼斯的食谱之前，我想把它拨回来，让香蕉自己说话。

## 你只需要一勺糖

正如《今日美国》的希瑟·马丁在她为琼斯的食谱 撰写的文章 [中指出的，热香蕉可能需要一段时间来适应。“煮熟的香蕉乍一看可能听起来很奇怪，但也许你在](https://www.today.com/food/trends/air-fried-banana-split-rcna33247) 之前已经吃过 [香蕉，而](https://www.today.com/recipes/bananas-foster-recipe-t248006) [bananacue](https://panlasangpinoy.com/bananacue/) (香蕉和烧烤的组合)几十年来一直是菲律宾街头食品烹饪的主要内容。”像香蕉福斯特这样的甜点所带来的焦糖化正是空气油炸香蕉吸引我的地方，所以我把它切成两半，撒上糖，直到它闪闪发光，然后在400度的空气油炸锅中炸10分钟。

我用肉桂完成了这道菜，还加了一点鲜奶油。太棒了。我把整个都吃了。空气油炸的香蕉比生的更软更甜，上面的糖确实焦糖化了。虽然它不像烤焦的布勒里那样脆脆的，但有足够的纹理对比使它看起来不像“糊状的、温暖的香蕉”我会再吃一次，特别是作为一个真正的香蕉船(配冰淇淋)的基础。

## 没有必要给香蕉百合镀金

当我第一次做油炸香蕉船时，我严格遵循了琼斯的食谱。我放了一勺花生酱，然后撒了一些巧克力片和山核桃，最后放了一点肉桂。然后我在400度的空气油炸锅里煮了7分钟。

当我把它拿出来的时候，我注意到部分山核桃被烧焦了，一些巧克力碎片看起来也被烧焦了(咬了一口证实了这一点)。下面的香蕉也不像只含糖的香蕉那么软也不那么甜，这是有道理的，因为香蕉已经完全分成两半，更多的果肉暴露在加热元件下。

由于对烧焦的坚果和巧克力不满意，我又做了一个，这次把馅料往香蕉缝里塞得更深，并在上面覆盖一层红糖来保护坚果。

燃烧减少了，但正如你从这句话正上方的照片中看到的，仍然有一些燃烧。(这是箔纸包裹的营火香蕉的一个小优势——箔纸保护馅料，这样当香蕉完全变软时，它们可以融化和融合而不会燃烧。)香蕉底部仍然有点硬，尝起来有点“生”，我想这很好，尤其是如果你一开始就不喜欢热香蕉的话。我没有酸奶，所以我把我的那份和鲜奶油一起吃了。

这很好，甚至令人愉快，但我渴望第一个，只有糖的香蕉。我确实认为冷酸奶会有助于给它冰淇淋的感觉，但加入巧克力、花生酱和坚果会掩盖下面美丽的香蕉，并使它无法充分发挥空气油炸的潜力。馅料阻挡了大部分肉的加热元件，只留下边缘暴露在外，所以只有边缘焦化。

如果你想尝试空气油炸香蕉，我建议简单地开始。将香蕉切成两半，在上面撒些糖，让糖在400度的空气油炸锅里变成褐色、起泡和焦糖状。如果可能不像“分裂”那样华而不实，但你总是可以在它的基础上建立一个真正的分裂。然而，我鼓励你至少尝试一次不加冰淇淋的。我想你会对香蕉的美妙感到惊讶，尤其是当你让它保持原样的时候。
