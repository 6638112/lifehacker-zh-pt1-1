# 是的，如果你坚持使用杀虫剂，你需要和你的邻居谈谈

> 原文：<https://lifehacker.com/yes-you-need-to-talk-to-your-neighbors-if-you-insist-o-1847435793>

如果你足够幸运，住在有漂亮的草坪、景观或花园的地方，你可能有自己的方法来维护它们——对某些人来说，这包括使用杀虫剂。如果你和你的邻居不同意喷洒，这可能会导致一些紧张局势。但是考虑到一个支持杀虫剂的人正在向附近地区和环境释放化学物质，体贴的做法是让住在隔壁的人知道他们在喷洒什么、在哪里、何时喷洒。



在 [一篇关于 BobVila.com](https://www.bobvila.com/articles/pesticides-and-property-lines-how-to-talk-to-neighbors/)的文章中，马克·沃尔夫分享了一些与你的邻居进行关于农药使用、以及如何处理分歧的(文明)对话的技巧和策略。以下是需要了解的内容。

## 你邻居使用杀虫剂会如何影响你和你的财产

在我们进入细节之前，有一点需要注意:这里的重点不是讨论杀虫剂在住宅或商业环境中的使用——那是另一天的主题。在这里，我们将重点关注与你的邻居就他们的喷洒习惯进行有效沟通，从他们使用杀虫剂会如何影响你和你的财产开始。

[根据沃尔夫](https://www.bobvila.com/articles/pesticides-and-property-lines-how-to-talk-to-neighbors/) 的说法，一个令人担忧的问题是过度喷涂。也称为喷雾漂移，它通常发生在有风的日子里，当液体或灰尘处理应用于草坪或植被区时，允许其到达人的财产线以外的区域。如果你注意到你的院子和邻居的院子之间的边界上或附近有一片区域，那里的植物被损坏或死亡，这是喷洒过量的迹象，需要向你的邻居提出来。

## 如何和你的邻居谈论杀虫剂的使用

正如沃尔夫指出的那样，对杀虫剂采取不同态度的邻居之间可能会产生紧张，但就此进行一次深思熟虑的对话有助于缓解这种局面。以下是一些主要的话题:

### 喷的是什么

如果你是喷药的人(或者雇佣其他人来做)，让你的邻居知道你在使用哪种化学药品。如果你在等式的另一边，请随意询问你的邻居(或他们雇佣的专业人员)他们使用的是哪种杀虫剂、除草剂和/或杀虫剂。

然后， Wolfe 建议查找正在使用的物质，看看它们是否对人 或环境构成任何风险。你也应该考虑你是否有任何宠物在户外度过。如果有什么让你担心的事情，做一些研究，并把它告诉你的邻居，因为他们可能没有意识到它的影响。(但如果你打算这么做，不要消极攻击或对此耿耿于怀。理想情况下，你会在喷什么的问题上达成某种妥协。

### 什么时候喷洒杀虫剂

处理某些情况的另一种方法是询问你的邻居(或他们雇佣的人)他们计划什么时候进行喷洒。沃尔夫解释说，这样，你会得到提前警告，可以计划当天外出，或者呆在家里以避开化学物质和任何潜在的气味。

不管你站在哪一边， Wolfe 说这有助于向你的邻居解释你的立场。人们决定使用或不使用杀虫剂有各种各样的原因，听听他们的想法是个好主意。

### 保持沟通

十有八九，这不会是一劳永逸的聊天。在持续的(合理的)基础上与你的邻居保持联系，可以让你将任何新问题扼杀在萌芽状态，并讨论什么可行，什么不可行。正如沃尔夫解释的那样，有时你不得不同意不同意见。在某些情况下，可能需要采取法律行动，但是 Wolfe 建议把它作为最后的手段。
