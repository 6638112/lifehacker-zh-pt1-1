# 如何在 iOS 上拍摄可滚动的整页网站截图

> 原文：<https://lifehacker.com/how-to-take-scrollable-full-page-screenshots-of-websit-1846248905>

大约一年一次，我花一些时间试图找到无缝截取整个网站的最佳方式，因为我并不总是喜欢 CTRL + S 保存网站数据的方法。我只希望我能早点知道在 iOS 上这么做是多么的简单。

我已经尝试了许多不同程度的浏览器扩展(直到 [我发现了为什么](https://lifehacker.com/how-to-fix-full-page-browser-screenshots-that-render-bl-1831429544) 它们中的大多数通常会让我搞砸)。 C 在我的 iPhone 上捕捉一个网站的全貌现在打开了许多新的可能性，包括特别是 T9】方便，如果我想给某人发送一个网站在我查看的确切时刻的快照，它或想向某人展示一个错误是如何搅乱我的体验的。

不管是什么原因，在 iOS 上抓取一个网站的整页截图都很简单。首先，在确定您使用的是最新版本的 iOS 之后，在设备的 Safari 浏览器中启动该网站。像平常一样给一个网站截图——在我的例子中是**侧边按钮+增大音量**。您现在会看到这个屏幕:

如果你没有看到顶部的两个选项——屏幕**和 T2【全页】可能是 iOS 太挑剔了。关闭 Safari，重新打开它，并尝试拍摄另一个屏幕截图。这些选项至关重要，因为点击**整页**是将你带到这个屏幕的神奇调味汁:**

右边的小侧边栏，你可以上下滚动，向你展示你的手机捕捉到的整个页面。然而，当你去保存它时，你不会把它放到你的相册里，因为从技术上来说，你的设备并不是在创建一张巨幅照片。相反，你的 iPhone 会将这张超大截图转换成 PDF 格式，所以你需要将它发送到你设备上的其他文件夹:
