# 变性顾客如何在信用卡上使用他们选择的名字

> 原文：<https://lifehacker.com/how-transgender-customers-can-use-their-chosen-name-on-1845436821>

花旗银行现在提供 [实名信用卡和借记卡](https://banking.citi.com/cbol/updatemyname/default.htm?BTData=DI7.B.gAB4f.B.Byy.SDrL.eDA.lN*.w8*.Bj.SX.0f.E&ProspectID=BC417443184A4992B85576BD06A6067D) ，这是为变性人和非婚生人士创建的，允许他们使用自己选择的名字，而不是他们的法定姓名。随着第二家大银行支持实名计划，更多的客户现在可以使用这一功能。

### **工作原理**

跨性别者和非双性恋者在使用不能反映其真实身份的卡时可能会面临审查。根据 [国家跨性别平等中心](https://www.transequality.org/sites/default/files/docs/USTS-Full-Report-FINAL.PDF) 的数据，近三分之一的受访者在出示身份证时，其姓名或性别与其性别不符，并遭到口头骚扰、拒绝服务、要求离开或攻击。

还有惩罚性的财务方面。超过三分之一的跨性别者没有合法地更改他们的名字,因为他们负担不起，根据调查，这样做要花费 100 到 250 美元，尽管有时要 1000 美元或更多。

万事达卡于 2019 年 6 月推出了实名制，让跨性别者和非双性恋者更容易支付服务费用，而不会受到骚扰。开户时，客户会被询问他们的借记卡或信用卡上的名字应该是什么——即使他们还没有合法地更改他们的名字。(顾客不会被要求表明自己是跨性别者或非双性恋者，以获得真实的名片)。

一旦您注册，一张新的卡将在接下来的 4-7 个工作日内与一起到达。更改姓名后，无论客户何时通过网络或电话联系银行，都将使用该姓名。如果商家对与客户 ID 不符的信用卡提出异议，他们可以拨打卡背面的号码来验证您的身份。
