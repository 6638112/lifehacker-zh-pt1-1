# 如何开始一个室内菜园

> 原文：<https://lifehacker.com/how-to-start-an-indoor-vegetable-garden-1847548438>

在室内种植蔬菜需要一些技巧，但完全可行。如果你愿意做一些研究(也许先在一些室内植物上练习)，这对初学者来说也是可行的；如果你是一个中级园丁，那么，你还在等什么？是时候接受种植室内花园的挑战了。



## 打造室内花园需要什么

供应清单比一般的花园更广泛，因为你需要补偿户外自然发生的情况，包括合适的土壤、排水、光线和气流。你需要的东西包括:

*   种植灯
*   盆栽土
*   繁殖种子的容器
*   带排水孔的花盆(适用于任何商店购买的蔬菜植物)
*   种植灯(可选，但强烈推荐)
*   种植棒和镀锌铁丝(用于植物培训)
*   植物标签
*   和加湿器(可选)

## 在你的室内花园里种植什么植物

你可能想挑选最简单的蔬菜，尤其是如果这是你第一次尝试在室内种植蔬菜。初学者的室内花园不是看你是否能种出像甜瓜那样的东西的时候。取而代之的是，关注那些不太需要维护的植物，比如喜欢温暖天气的番茄、辣椒和矮柑橘，以及耐寒的蔬菜，比如芥菜、花椰菜、甜菜和羽衣甘蓝。当然，像迷迭香、百里香、鼠尾草和牛至等草本植物作为室内植物茁壮成长(这里是我们的 [快速种植药草园指南](https://lifehacker.com/why-you-need-an-herb-garden-1843999388) ，因为你值得拥有一个)。

## 如何开始你的室内菜园

如果你从种子开始，你将从小容器和盆栽土开始，按照种子包装上的指示种植种子。一旦你的种子长出足够的叶子，相应地重新种植。

但是一旦显而易见的部分被排除，你将需要更仔细地考虑把它们放在哪里——你现在发芽的绿色植物将需要根据它们需要的气候和光线在你的房子周围进行组织(T2)。大多数蔬菜需要大约 4-6 小时的光照，所以无论你是使用自然光还是种植灯，都要记住这一点。

此外，要考虑到你的植物需要的气候。像羽衣甘蓝和花椰菜这样的冷气候植物不需要阳光直射和高温，所以把它们放在凉爽的地方。对于喜欢温暖的植物来说，冬季月份可能是具有挑战性的，和是当一个 加湿器可能有用的时候。一个简单的振荡风扇也有帮助。

当然，记住推荐的浇水方法，根据你种植的蔬菜种类在生长月份施肥。总的来说， [佛罗里达大学园艺解决方案](https://gardeningsolutions.ifas.ufl.edu/care/fertilizer/applying-fertilizer-to-vegetables.html) 说“在整个生长季节，每三到四周施用一次平衡肥料。当果实出现时，不要停止施肥——根据需要继续施肥，以确保持续生产。”一旦你的植物长出蔬菜，它们就可以收获了，你可以拍拍自己的背。你做到了。接下来试试甜瓜。
