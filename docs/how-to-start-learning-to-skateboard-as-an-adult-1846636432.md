滑板从来没有遵循过规则手册，这意味着在某种程度上，一篇关于如何学习滑板的文章可能违背了它的反文化根源。尽管滑冰促进了全球社区的发展，但它在 T2 是一项孤独的运动，在 T4 是一项孤独的运动，这可能就是为什么在疫情，滑冰成为成年人新的消遣方式。

作为一个从小滑冰长大的人，我被最近的几个故事逗乐了，这些故事是由一些人写的，他们在 30 多岁时第一次去滑板公园，作为逃避疫情式孤独的一种方式。观看成年人学习如何踢转身，奥利，和在坡道上下降是一个有趣的镜头，反映了 COVID 期间社会的集体不安，但我向任何跳上滑板的人致敬，因为他们正确地认为滑板将是一种冒险的出路。



这是你成年后学习滑冰的方法。当你走向滑板公园，被比我们更努力的孩子包围时，不要感到惊讶。

### 买一块真正的板

一个完整的滑板由多个部分组成:一个有防滑带的滑板甲板，卡车，轮子，轴承，可能还有提升器来缓冲你的金属卡车撞击滑板的冲击力。还有八个螺栓和锁紧螺母将你的卡车固定在木制甲板上。

不要吝啬买一块真正的木板——你真的不想要来自 [【沃尔玛】](https://www.skateboardershq.com/should-you-buy-walmart-skateboard/) (或另一家大盒子零售商)的由劣质木材制成的东西，以及由弱金属制成的卡车。与学习如何 [打鼓](https://lifehacker.com/so-you-want-to-learn-to-play-the-drums-1846112277) 或其他乐器不同，当你开始时，你不是在寻找低端设备，因为便宜的东西会让你更容易受伤。

我建议你戴上面具去当地的滑板店，问问在那里工作的人你应该买什么。但在现实中，只要你从合法的制造商那里购买主板和配件，你的情况就很好。当你购买你的主板时，有许多伟大的——甚至是标志性的——传统公司可供选择:Flip、Baker、Element、Girl、Toy Machine、和 Enjoi(以及其他许多公司)都是很好的选择。



卡车和车轮也是如此。像 Independent、Venture 和 Tensor(以及其他公司)这样的公司是卡车的好选择，而 Spitfire、Pig、Bones、和其他许多公司生产的车轮应该可以让你开动起来。当谈到生产滑板和其他产品的公司时，你会发现一些交叉——有时生产滑板的公司也生产轮子，但通常生产卡车的公司只专注于这一点。

我想说的总的一点是相信名牌。生产合法滑板和产品的公司几乎总是会赞助职业滑冰运动员，制作滑冰视频，并生产自己的服装。一旦你熟悉了真正的滑板产品，你就可以很快分辨出真假滑板。

### 找一个容易滑冰的地方，然后从那里继续前进

不要期望像动物一样从山上冲下来(但是如果你想从这样做的人那里获得灵感，就不要再找了 )。你应该做的是找到一个好的，宽容的环境来收集你的第一次瘀伤。我很幸运，我在一个死胡同里长大，这让我可以四处游荡而不用担心被车撞倒，但你可能没有这种奢侈。

找一个开放的网球场或篮球场，学习如何推、踢、转、和奥利。你需要一个容易推动的光滑表面。熟悉你脚下木板的感觉，在学习新技巧时相信你的直觉。一旦你对自己的平衡有了更多的自信，你可能会开始培养自己对技巧和障碍的品味。



作家史蒂夫·卢梭 在疫情期间开始玩滑板，他建议降低期望值，原谅自己这项运动带来的陡峭的学习曲线。

“按你自己的速度走就行了，”他说。“每当你学到新东西时都要庆祝，不管你认为它与‘真正的’滑板运动员相比是多么微不足道。”

### 获得灵感

这项运动的创造性动力很大程度上来自于吸取其他溜冰者的能量。滑冰视频展示了大量可供识别的风格，也展示了这项运动演变的伟大历史教训。谈到收集灵感，卢梭是这样说的:

> 看很多滑板视频就好了！滑冰有这么多不同的风格、氛围和方式——找一个和你有共鸣的吧。不要把滑冰看作是你想学的一系列技巧，而更像是你想模仿的一种风格。



综上所述，学习如何下注确实没有“正确”的方法。只要明白你会经常摔倒，但这是过程的一部分，即使是对最有经验的溜冰者来说。底线是，如果你有一块板，把它放在人行道上，然后从那里开始。
