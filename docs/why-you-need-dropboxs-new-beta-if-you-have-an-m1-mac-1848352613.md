# 如果你有一台M1 Mac，为什么还需要Dropbox的新测试版

> 原文:[https://life hacker . com/why-you-need-Dropbox s-new-beta-if-you-an-m1-MAC-1848352613](https://lifehacker.com/why-you-need-dropboxs-new-beta-if-you-have-an-m1-mac-1848352613)

去年，Dropbox声明他们将不会开发针对苹果硅(M1)MAC电脑 [优化的Dropbox客户端，直到他们看到对它的需求](https://twitter.com/mitchellh/status/1453394500848537605?s=20)——谢天谢地 [退得很快](https://www.theverge.com/2021/10/28/22751178/dropbox-native-apple-silicon-app-m1-2022) 的话，并承诺在2022年上半年推出新的Dropbox客户端。随着我们今年的推出，M1应用程序正式成为测试版，任何人都可以下载。

一个应用程序被更新到一个新的平台有什么大不了的，在这个平台上，即使是旧的应用程序也能正常工作。问题在于Dropbox如何在这些新的MAC电脑上运行。或者更确切地说，它在新M1 MAC电脑上的表现有多糟糕。

有太多关于性能问题和电池寿命问题的广泛报道，我们的官方建议是 [从Dropbox切换到在Mac上原生运行得更好的东西](https://lifehacker.com/the-best-alternative-dropbox-clients-for-mac-if-you-ve-1848046501) 。

尽管我们仍在袖手旁观，但不可能让每个人都轻易跳槽。如果你对Dropbox有一个长期计划，或者是Dropbox商业计划的一部分，你就必须用这个应用来凑合。谢天谢地，这款应用现在至少已经针对苹果芯片进行了优化。

## 如何使用测试版改进M1 MAC电脑上的Dropbox

要下载Dropbox测试版，请前往 [Dropbox的论坛页面](https://www.dropboxforum.com/t5/Dropbox-desktop-client-builds/Beta-Build-140-3-1903/td-p/570154) ，点击 **Mac OS X** 部分的“**标准安装程序**”链接( [这里是直接下载链接](https://www.dropbox.com/downloading?build=140.3.1903&plat=mac&type=full&arch=arm64) )。这将下载一个900KB的dmg文件。一旦安装完毕，你就可以下载并安装最新的Dropbox客户端了。如果您已经在运行Dropbox，此安装程序将替换现有版本。

测试版应该小心安装，gut早期的报告似乎很有希望。根据 [MacStories](https://www.macstories.net/news/dropbox-releases-apple-silicon-version-of-its-app-to-beta-testers/) 所做的初步测试，新的M1优化应用程序将内存使用量从巨大的4GB降至更好的645MB。

安装新的Dropbox应用程序后，你可以自行确认它是苹果硅版本。点击菜单栏中的**苹果**按钮，进入**关于这台Mac** 。在这里，点击“**系统报告**，进入侧边栏的“**应用**”部分。找到“ **Dropbox** ”应用程序，它应该会在旁边显示“ **Apple Silicon** ”。
