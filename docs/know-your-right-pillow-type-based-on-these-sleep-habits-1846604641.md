# 根据这些睡眠习惯，知道你合适的枕头类型

> 原文：<https://lifehacker.com/know-your-right-pillow-type-based-on-these-sleep-habits-1846604641>

睡眠让你的身体 [充电，恢复，并抵御疾病](https://www.sleepfoundation.org/how-sleep-works/why-do-we-need-sleep) 如心脏病和糖尿病。想要睡个好觉可能很难(无论是疫情之前还是现在)，对我们许多人来说，最大的困难之一就是找到合适的枕头。我花了无数的钱在枕头上，以为我找到了最完美的一个，也许你也试过了，从记忆泡沫和冷却凝胶，到硬枕头，软枕头，侧睡枕头，等等。



坏消息是:世上并不只有一个完美的枕头。所有的枕头都有不同的功能，没有放之四海而皆准的尺寸。不过，好消息是，有些事情你可以考虑，以帮助找到适合你的枕头。

### 评估你的健康状况

我们的睡眠方式影响着我们许多人面临的无数情况， [从鼻窦问题到眩晕再到胃灼热](https://www.health.harvard.edu/pain/is-your-pillow-hurting-your-health) 。如果你患有睡眠呼吸暂停并使用 CPAP 机器，你使用的枕头也会影响你的面罩并阻止气流输送——特别是侧睡者，他们经常需要 [特制 CPAP 枕头](https://www.cpap.com/productpage/SleePAP-cpap-pillow-with-pillowcase) 。了解你的健康状况是找到最适合你的枕头的第一步。

### 考虑你的睡眠习惯

像仰卧、侧卧和俯卧这样的习惯需要不同类型的枕头支撑。斯波尔丁康复医院的理疗师马修·奥鲁克(Matthew O'Rourke)告诉哈佛健康，“尽量让脊柱处于相对笔直的位置。但每个人的食谱都会有一点不同。”脊柱健康有一些有用的说明，告诉你哪些枕头能为你的背部提供适当的支撑，但这里有一些你应该知道的总结:

#### **如果你侧睡**

侧睡者需要一个结实的枕头，支撑颈部，使其与肩膀保持水平。也可以考虑在你的双腿之间放一个小枕头，这样可以在你睡觉的时候让你的脊柱和脖子保持一致。

#### 如果你仰卧而睡

仰卧者倾向于需要一个较低或较软的枕头。硬枕头会导致头部前倾，给颈部和呼吸道带来压力。罗彻斯特健康中心 建议平躺睡觉时在膝盖下放一个枕头，以保持脊柱的正确位置。

#### 如果你趴着睡觉

胃睡者的情况最糟糕，因为这种睡姿对身体的压力最大。建议使用平枕，或者根本不用枕头，以避免颈部和背部的压力。在骨盆下面放一个枕头可能有助于减轻颈部和背部的压力。

### 底线

你选择的枕头类型是个人喜好，但尽量让它们与你的睡眠习惯相匹配。虽然我喜欢蓬松一点的枕头，但我知道像 [枕立方](https://www.pillowcube.com/?gclid=CjwKCAjwgZuDBhBTEiwAXNofRFB6lFKAEHbd79WRJwR3P_ha6UTVRx7eixVtjHUhiA0Ybtn-imjOOxoCYywQAvD_BwE) 这样比较结实的枕头更适合侧卧。我会把羽绒枕头的快乐留给你们这些睡懒觉的人。
