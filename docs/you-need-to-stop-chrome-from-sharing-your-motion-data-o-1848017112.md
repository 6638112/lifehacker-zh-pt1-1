# 你需要阻止 Chrome 在 Android 上分享你的运动数据

> 原文：<https://lifehacker.com/you-need-to-stop-chrome-from-sharing-your-motion-data-o-1848017112>

在这一点上，“谷歌对你的数据感到毛骨悚然”的说法并不令人震惊。我们知道谷歌和类似的公司每天都在以商业和利润的名义损害我们的隐私和安全；我们只是，你知道，处理它？然而，最近 Chrome 出现了一些额外的令人毛骨悚然的情况——特别是如果你在 Android 上使用浏览器的话。

Mysk，应用开发者和安全研究人员的阿朵， [最近在推特](https://twitter.com/mysk_co/status/1454016614471553024) 上曝光了 Chrome 的可疑之处。在这条推文中，Mysk 指出，默认情况下，Chrome 会与你访问的网站共享你手机的运动数据。这一点都不酷。

## 为什么你不希望第三方访问你的运动数据

首先，正如我所指出的，这是令人毛骨悚然的房颤。数据来自手机的加速度计，该传感器负责跟踪设备的方向和位置。该传感器可以从纵向模式切换到横向模式，并跟踪你和你的手机的运动。例如，只要你带着手机，它就能让健身应用程序知道你走了多少步。

因为我们大多数人都把手机放在口袋里或随身携带，所以一整天都会有大量的运动数据在手机上生成。谷歌 Chrome，*设计的*，允许你点击任何网站请求动作数据，并兴致勃勃地把它交给用户。 [研究人员发现](https://www.forbes.com/sites/zakdoffman/2021/11/06/stop-using-google-chrome-on-android-samsung-apple-iphone-and-windows-10-devices/?sh=4f79e3977bf8) 这些网站使用加速度计数据来监控广告互动，检查广告印象，并跟踪你的设备(嗯，咄)。然而，前两个非常粗略；网站不仅想知道*你是否会点击某个广告，他们还想知道你如何与这些弹出窗口互动。嘿，为什么停在那里？为什么不打开我的相机，看看我穿的是什么颜色的衬衫？***

## *如何阻止 Chrome 与网站共享运动数据*

*从手机中删除该应用程序。*开玩笑*。我知道绝大多数安卓用户不会想从 Chrome 切换到另一个手机浏览器。也就是说，注重隐私的用户可能想跳槽到更有声望的网站——比如 Firefox——如果是这样，对你有好处。*

*但坚持使用 Chrome 有很多好处，尤其是在 Android 上(考虑到该平台也是由谷歌设计和运营的)。如果你不想采取最激烈的措施，你可以简单地切换设置，阻止谷歌分享这些数据。正如 Mysk 在他们的推文中指出的，你可以从 Chrome 的设置中禁用运动数据共享。*

 *[https://lifehacker.com/embed/inset/iframe?id=twitter-1454016614471553024&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1454016614471553024&autosize=1)* 

*方法如下:打开应用程序，点击右上角的三个点，然后选择“**设置**”接下来，向下滚动，点击“**站点设置**，然后点击“**运动传感器**关闭此处的切换，以确保从现在开始没有第三方网站可以要求您的运动数据。*
