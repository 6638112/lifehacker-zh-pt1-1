# 如何使用Chrome、Safari和Firefox中的隐藏搜索标签功能

> 原文:[https://life hacker . com/how-to-use-the-hidden-search-tabs-feature-in-chrome-sa-1847392502](https://lifehacker.com/how-to-use-the-hidden-search-tabs-feature-in-chrome-sa-1847392502)

人们很容易迷失在多个浏览器窗口之间打开的标签海洋中——有时甚至是在多个标签中的同一个网站的。相反，使用浏览器内置的搜索标签功能 [让事情井井有条](https://lifehacker.com/organize-your-browsing-with-this-chrome-dashboard-exten-1832817515) 。你会在Chrome、Safari和Firefox等所有主流浏览器中找到这一功能。虽然新的微软Edge运行在Chrome引擎上，但它还没有Chrome的搜索标签功能——但你可以使用一个 [扩展](https://lifehacker.com/our-12-favorite-chrome-extensions-1844729409) 来复制它。

## 如何在Chrome中搜索打开的标签页

Chrome版和更高版本在标签栏中增加了一个新的搜索标签功能——你会在的末尾找到一个向下的三角形图标。点击它可以看到所有打开的标签列表，包括最近关闭的。

点击“**搜索标签**文本框，在所有窗口中所有打开的标签之间搜索，s 表示选择一个页面切换到它。

## 如何在Firefox中搜索打开的标签页

Firefox中的搜索标签功能有点隐藏:它直接集成在地址栏中，并且没有专门的按钮。

打开Firefox浏览器，点击地址栏。在这里，输入 **%** 键，然后点击**空格键**。现在，您将看到所有打开的选项卡的列表。您可以继续键入以搜索和过滤选项卡。使用箭头键或鼠标导航到您想要切换到的选项卡，并按下 **Enter** 键。

## 如何在Safari中搜索打开的标签页

Safari在其标签页中有一个内置的搜索标签功能，但问题是它只能让你在当前窗口中搜索标签。

打开Safari浏览器，点击屏幕右上角的**标签**按钮(看起来像两个正方形，一个在另一个上面)。然后，点击右上角的“**搜索标签**文本框，开始搜索。一旦你找到你想要的网站，只需点击预览就可以切换到它。

## 如何在Microsoft Edge中搜索打开的选项卡

微软Edge仍然没有Chrome的搜索标签功能。在添加之前，您可以使用第三方扩展来复制该功能。

首先，安装并激活 [搜索所有标签页的](https://microsoftedge.microsoft.com/addons/detail/search-all-tabs/ggebacfnckaalgfmddgfcokohadkbnmm) 扩展。然后，单击工具栏中的扩展图标(它看起来像一个放大镜)。键入页面标题，或您要切换到的网站。

该扩展将搜索所有窗口中所有打开的标签。找到页面后，按 **Enter** 键打开第一个搜索结果。您可以使用箭头键选择其他结果。
