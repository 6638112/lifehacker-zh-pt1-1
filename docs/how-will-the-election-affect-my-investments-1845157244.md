# 选举将如何影响我的投资？

> 原文:[https://life hacker . com/how-will-the-election-affect-my-investments-1845157244](https://lifehacker.com/how-will-the-election-affect-my-investments-1845157244)

一个没有疫苗的疫情，长期的经济衰退，现在大量的邮寄选票威胁着将选举日变成选举月。在这一片混乱中，个人投资者该如何处理自己的投资组合:调整美国股票配置，还是坚持到底？

“虽然与选举有关的噪音可能令人担忧，但我们不认为任何潜在的选举结果都应该成为违背市场基本面因素并大幅改变投资组合配置的理由，”Principal Global Investors[在最近的一份分析师报告](https://www.principalglobal.com/knowledge/insights/positioning-2020-us-presidential-election) 中建议。

投资者可能会因为过于重视选举结果而陷入困境，从波动性的角度来看，选举结果在历史上会产生短期影响。赌一个结果的问题是它涉及太多需要考虑的情景:参议院、众议院和白宫都是可以争夺的，任何一方都可能获胜，这些情景的任何组合都会对立法议程产生不同的影响。这些情景都不会影响美联储的行动。

当然，确实存在风险，特别是如果投票违规和邮件延误导致以微弱优势赢得旷日持久的法律斗争，就像我们在2000年看到的布什-戈尔那样。话说回来，胜利的幅度可能大到足以让我们摆脱这种痛苦，即使这种情景听起来不太像2020年。我们只是不知道。选举是一个风险事件，结果有好有坏。

分析师倾向于认为拜登获胜对公用事业、绿色能源和医疗保健有利，特朗普的放松管制议程有利于能源行业和金融科技，但不能保证这些都会发生。

Principal Global Investors警告称:“我们的研究表明，为了确定谁最有可能胜出而调整投资组合配置，是一种危险的策略。”。换句话说，如果你是一个忠诚的长期投资者，少做就是多做。

### **重新评估你的风险承受能力**

你可能不想挑选赢家和输家，但你应该降低对美国股票的总体敞口吗？那要看你的风险承受能力了。根据你的时间框架和对风险的承受能力，找到股票、债券和现金的正确平衡。问自己这些问题:

*   自从疫情开始以来，你的财务状况发生了多大的变化？
*   想想你最重要的财务生活目标，估计你什么时候会从你的账户里取钱。你需要多少？
*   你打算近期撤回投资吗？

*   你能处理长期的投资波动吗？

如果你计划的提款日期是很多年以后，你可能会接受一个承担更高风险的投资组合来换取潜在的更高升值。然而，在较短的时间内，你可能希望减少对美国股票的投资，投资更多的债券或现金。
