# 关于今天开始实施的新 COVID 旅行规则，您应该知道些什么

> 原文：<https://lifehacker.com/what-you-should-know-about-the-new-covid-travel-rules-t-1848166165>

如果你正在计划一次国际旅行(或者如果你现在正在旅行)，是时候温习一下与 COVID 相关的法规了，因为测试要求改变了，而且 [它们今天](https://www.cdc.gov/media/releases/2021/p1203-covid-testing-tightens-intl.html) 生效。此外，还记得拜登政府如何在 11 月初决定基于国家的旅行禁令是一个坏主意吗？他们回来了。

## 你现在需要在出发前一天进行测试

当你去美国旅行(或返回美国)时，你需要进行阴性病毒测试。直到最近，接种疫苗的旅行者可以在出发前三天接受这项测试。这对于计划来说很方便，也给了你一些额外的时间，以防处理你的测试的地方需要时间来得到结果。但是现在，无论是否接种疫苗，每个人都回到了一天的要求。

这是基于日历日的，所以不管你周二的航班是什么时间，你都可以在周一的任何时间参加考试。仍然建议(但不要求)你在回国后三到五天进行病毒测试，并隔离七天。

例外情况:2 岁以下的儿童不需要这项测试，在过去 90 天内从 COVID 康复的人如果有记录他们疾病的医疗记录，可以跳过这项测试。

这些变化意味着你需要做更多的计划。如果你要去另一个国家旅行，在你登机回家之前，你知道在哪里可以进行测试吗？那个测试网站要多久才能给你结果？ [《今日美国》推荐](https://www.usatoday.com/story/travel/news/2021/12/03/new-requirements-us-international/8853407002/) 不要依赖于标准的 PCR 检测，你可以查找疾控中心批准的快速检测方法并使用其中之一。一些自我测试是允许的，尽管只有那些在与医疗专业人员进行视频通话时进行测试的测试才符合这一规定。

## 基于国家的旅行禁令又回来了

在放弃基于国家的旅行禁令的整个概念仅仅几周后，COVID [旅行禁令又回到了](https://www.cdc.gov/coronavirus/2019-ncov/travelers/from-other-countries.html) 。现在名单上有八个非洲国家。如果你是美国公民，你仍然可以从这些国家前往美国；如果你不是，那么如果你在出发前的 14 天内曾在这些国家呆过，你就不能去美国旅行。这些国家是博茨瓦纳、斯威士兰、莱索托、马拉维、莫桑比克、纳米比亚、南非和津巴布韦。

旅行禁令 [于 11 月 26 日](https://www.whitehouse.gov/briefing-room/presidential-actions/2021/11/26/a-proclamation-on-suspension-of-entry-as-immigrants-and-nonimmigrants-of-certain-additional-persons-who-pose-a-risk-of-transmitting-coronavirus-disease-2019/) 生效，旨在减缓 [奥米克隆](https://lifehacker.com/what-you-need-to-know-about-the-omicron-variant-1848158084) 变种的传播，尽管众所周知奥米克隆在世界许多地方传播， [包括美国 17 个州](https://www.washingtonpost.com/nation/2021/12/06/covid-omicron-variant-live-updates/) 。

如果这一切都变得令人困惑， [疾病预防控制中心在这里有一个流程图](https://www.cdc.gov/coronavirus/2019-ncov/travelers/pdf/requirements-travel-to-us-by-air-quick-reference-p.pdf) ，所以你可以找出哪些旅行规定适用于你。还有一个 [竞猜](https://www.cdc.gov/coronavirus/2019-ncov/travelers/international-travel/travel-assessment/index.html) 版本。
