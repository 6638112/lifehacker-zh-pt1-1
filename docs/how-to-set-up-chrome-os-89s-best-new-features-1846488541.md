# 如何设置 Chrome OS 89 的最佳新功能

> 原文：<https://lifehacker.com/how-to-set-up-chrome-os-89s-best-new-features-1846488541>

Chrome OS 89 现在正在推出，为运行谷歌操作系统的 Chromebooks 和平板电脑增加了大量功能。你可能会首先注意到漂亮的新应用图标，但在表面之下隐藏着更多的新东西。

为了帮助你充分利用更新的操作系统，这里列出了谷歌在 Chrome 89 中添加的每一个新功能或应用程序，以及如何使用它们。

### 电话集线器

Chrome OS 89 的第一个重大更新是新的 **Phone Hub** 应用程序，它允许 Android 用户通过蓝牙将他们的手机和 Chrome OS 设备配对。Phone Hub 从 Chrome OS 桌面监控和控制您手机的许多功能，允许您:

*   阅读和回复邮件
*   查看通知
*   远程静音你的手机
*   查找您手机的位置
*   同步最近的 Chrome 浏览器标签
*   通过 Wi-Fi Sync 在您的设备之间保存和共享 wifi 密码
*   查看手机的网络信号和剩余电池寿命。

Phone Hub 适用于所有运行最新版本 Google Play 服务的 Android 手机(适用于 [Google Play](https://play.google.com/store/apps/details?id=com.google.android.gms&hl=en&gl=us) )，以及所有支持蓝牙的 Chromebooks。

**使用电话集线器:**

1.  在 Android 手机和 Chromebook 上打开蓝牙。
2.  在你的 Chromebook 上，点击右下角的电话图标，然后点击**“开始”**
3.  按照说明操作。
4.  设备配对后，再次单击电话图标打开电话中枢。

### 更新的虚拟办公桌

Chrome OS 89 为虚拟桌面功能带来了几项变化，**桌面**，包括:

*   一次最多支持 8 张桌子。
*   桌面现在即使在你重启 Chromebook 后仍然保持活动状态。
*   右键单击应用程序窗口，并将其发送到特定桌面。
*   用四个手指在鼠标垫上滑动，可以在不同的桌子之间切换。

要使用书桌，请打开总览模式(在屏幕上向上滑动或按下总览键)并选择**“新书桌”**

### 手提包文件管理器

新的 **Tote** 应用程序可以更容易地跟踪 Chrome OS 中的重要文件。每当您下载文件、录制屏幕截图视频或拍摄屏幕截图时，Tote 都会出现在您的应用程序架上。您还可以将文件拖放到 Tote 窗口中，或者**右键单击>pink**一个文档，将其添加到 Tote 的 **Pinned files** 部分。

### 其他更新

除了新的电话集线器、手提包和办公桌功能，Chrome 89 还有许多更小的功能:

*   **注释图片:**你现在可以在 Chrome OS 的股票媒体应用程序中的图片上书写和绘画了。只需打开一个图像，选择新的**“注释”**工具，位于编辑按钮旁边。
*   **剪贴板管理器:** Chrome OS 的新剪贴板管理器存储了最近复制的五个文本字符串或图像。按下**“一切”键+ V** 查看你的剪贴板项目。
*   **更简单的媒体播放控制:**现在可以通过快速访问菜单访问媒体控制，您可以将控制固定到应用程序架上，以便更快地访问。
*   Google Assistant 快速回答:在浏览网页时右击一个单词，弹出一个 Google Assistant 支持的信息卡。卡片显示了所选单词的定义、翻译和其他有用信息。
*   **新的家庭帐户选项:**简化了家庭设置，使向您的设备添加孩子的学校帐户变得更加容易。新的设置会在你安装 89 版后第一次运行 Chrome OS 时出现，或者你可以去**设置>人>用户名**选择**“添加学校账号”**
*   **个性化锁屏:**你现在可以使用谷歌相册作为锁屏背景，或者从**设置>屏保下的“数百张”预装图片中选择。**
*   **扫描应用:**新的扫描应用可从应用启动器获得。您可以使用连接的扫描仪设备上传图像或 pdf 格式的物理文件。
*   **快速访问中的屏幕截图:**Chrome OS 屏幕截图工具现在可以从快速设置菜单中访问。
