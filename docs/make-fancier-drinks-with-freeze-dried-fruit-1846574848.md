# 用冻干水果制作更好的饮料

> 原文:[https://life hacker . com/make-fancier-drinks-with-free-dry-fruit-1846574848](https://lifehacker.com/make-fancier-drinks-with-freeze-dried-fruit-1846574848)

像任何在充斥着《T2》广告的环境中长大的千禧一代一样，我对《T4》中的冻干食品评价很高。Dippin' Dots可能声称是“未来的冰淇淋”，但你不会在史密森尼国家航空航天博物馆找到它。不，那个特别的礼品店的地产被*冻干*冰淇淋占用了——也被称为“宇航员冰淇淋”——残酷的是， [宇航员不能把它带到太空](https://www.cnet.com/news/astronauts-bust-the-myth-of-space-ice-cream/) (太多灰尘)。

从质地上来说，冻干食品很有趣，并且自年代以来已经变得更加主流。乔氏公司是一家特别多产的冻干水果生产商，这是一种浓缩的、充满活力的水果，可以粉碎并用于给各种东西上色和调味，包括和 [奶油奶酪](https://skillet.lifehacker.com/this-3-ingredient-whipped-dip-is-as-pretty-as-it-is-del-1828423544) 。它也是一种非常好的饮料。

制作水果粉非常简单:只要将一把水果粉扔进食品加工机或搅拌机，让它旋转，直到水果变成粉末。完成后，把灰尘撒在起泡的摇过的酸奶上，用它代替(或与)玻璃杯边缘的糖或盐，或者直接混合到鸡尾酒中。(冷冻干燥的水果通常是不加糖的，可能会有点酸，这取决于品牌，所以在把它放入饮料之前先尝一下粉末。)

即使你不喝酒，这些相同的做法也可以适用于不含酒精的饮料。在酸奶上撒些芒果粉，在一杯柠檬水边上撒些草莓粉，或者在摩卡上摇一些粉碎的覆盆子。
