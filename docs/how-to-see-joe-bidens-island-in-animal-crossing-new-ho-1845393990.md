# 如何参观《动物穿越:新视野》中乔·拜登的岛屿

> 原文:[https://life hacker . com/how-to-see-Joe-Bidens-island-in-animal-crossing-new-ho-1845393990](https://lifehacker.com/how-to-see-joe-bidens-island-in-animal-crossing-new-ho-1845393990)

2020年的总统选举注定会是一场怪异的选举，这不仅是因为新冠肺炎让集会和挨家挨户拉票等面对面的活动比以往任何时候都更加令人担忧。谢天谢地，总有动物穿越:新视野。

玩家已经可以使用数字竞选标志向候选人展示他们在游戏中支持的，现在玩家可以使用梦想代码 **DA-7286-5710-7478** *访问乔·拜登的官方*动物穿越*岛。这个迷人的小镇到处都是整洁的摄影场，包括一个村民版的乔·拜登，如果你感兴趣的话，还有志愿者机会的链接。*

《有点好笑的游戏》的编剧加里·维塔和格雷格·米勒是第一批参观乔·拜登竞选官方岛屿的《T2》动物穿越游戏玩家。今天早些时候，他们在Kinda Funny的 [Twitch频道](https://www.twitch.tv/kindafunnygames) 上亮相，作为Whitta受欢迎的“动物对话”网络系列的一部分，他在玩*动物穿越*时与名人、游戏开发者和其他受欢迎的人物聊天。

对于那些不熟悉的人来说，*动物穿越:新视野*是任天堂Switch上一款流行的视频游戏，也是2020年这一黑暗时期为数不多的亮点之一。令人放松的城镇管理模式让玩家创建自己的个性化岛屿，岛上住满了可爱的动物市民。

这款游戏于2020年3月发布，但已经是 [系统](https://www.gamasutra.com/view/news/367741/Animal_Crossing_New_Horizons_has_sold_over_22_million_copies.php) 中第二畅销的游戏，已经售出近2300万份。这也是日本有史以来第二畅销的游戏。它拥有大量来自各种年龄和背景的 [玩家的追随者——包括美国众议员亚历山大·奥卡西奥-科尔特斯。在Whitta访问游戏中的其他玩家而成为头条新闻后，AOC甚至在Whitta的“动物对话”节目中客串——包括Lifehacker自己的高级技术编辑](https://nintendowire.com/news/2020/09/07/animal-crossing-new-horizons-demographic-has-an-even-male-female-split-with-most-players-in-their-20s-and-30s/#:~:text=%E2%80%9CHowever%2C%20looking%20at%20the%20gender,in%20their%2020s%20and%2030s.%E2%80%9D) [David Murphy](https://kotaku.com/alexandria-ocasio-cortez-is-making-animal-crossing-hous-1843327623) 。
