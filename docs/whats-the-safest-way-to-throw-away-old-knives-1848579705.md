# 扔掉旧刀具最安全的方法是什么？

> 原文：<https://lifehacker.com/whats-the-safest-way-to-throw-away-old-knives-1848579705>

好刀如果你打理的话保质期很长:手洗，存放在专用的地方， [保持锋利](https://lifehacker.com/how-to-keep-your-knives-sharp-1846972299) 。即使是钝的刀片也可以回收。

但是如果你想升级到更高质量的刀具，你需要小心处理掉那些你不再需要的刀具。不要在没有任何保护的情况下将它们简单地扔进垃圾桶。下面是最安全的扔老刀方法，据 [鲍勃维拉](https://www.bobvila.com/articles/how-to-dispose-of-knives/) 。

### 把刀子包起来

不管你最终如何处理旧刀具，你都需要小心包装它们。否则，它们可能会划破垃圾袋，或者对处理它们的人造成危险。

首先，用多层报纸或包装纸包裹刀片，并用胶带将纸固定到位。然后切一张至少是刀片两倍长的纸板，将纸板对折在包裹好的刀片上，锋利的一面在折叠处。用胶带缠在纸板上，把刀子固定住。

一旦您的刀具被安全包装，您可以用以下方法之一处理它们。

### 如何出售你的刀具

状况良好的刀具——尤其是像 Wustof 这样的知名品牌的刀具——可能在你当地的分类广告网站或 Facebook Marketplace 上有需求。如果你的刀片完好且锋利，而你只是想升级你自己的刀片，这是最有可能的选择。查看其他帖子，了解你的帖子的价值。

### 如何捐赠你的刀具

如果你的刀在某种程度上仍然可以使用，一个选择是把它们捐赠给慈善机构或非营利组织或接收二手设备的创业厨房。一个朋友或家庭成员正在准备一个家庭厨房，也可以把它们从你手中拿走。

### 如何回收你的刀具

废金属回收中心可能会收旧刀。做一个快速的谷歌搜索，找到你所在地区的废金属公司。当你打电话时，你需要知道你的刀是由什么金属制成的。

### 如何扔掉你的刀

最后的选择，也许是最后的手段，就是扔掉你的旧刀子。如果刀片碎裂或破损到无法安全使用，这可能是最好的选择。在把它们扔进垃圾桶收集之前，用厚胶带把包好的刀封在纸箱里。
