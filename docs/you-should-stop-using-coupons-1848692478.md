# 你应该停止使用优惠券

> 原文:[https://life hacker . com/you-should-stop-use-coupons-1848692478](https://lifehacker.com/you-should-stop-using-coupons-1848692478)

我喜欢编织、缝纫和做各种各样的工艺品。因此，在很长一段时间里，我是邮寄乔安传单的忠实粉丝。总会有至少一个真正有价值的优惠券，给你任何全价商品的40%折扣，我通常会留着买一大块布料。但是如果你选择的商品已经打折了，你就不能使用它，所以我也会查看打折情况。这一策略有着深远的影响。一些优惠券比其他的要好；有些销售比其他的好。最后，我对这家商店的好买卖了如指掌，并会尽可能地去做，即使我心里还没有特定的项目。

从“我怎样才能在我做的这件衣服上省钱”的角度来看，这是一个明智的策略。但是从更大的角度来看，“我怎样才能在手工制作上花更少的钱”这个问题是一个彻底的失败。

你认为为什么公司会首先提供优惠券和销售？这是因为这样做能让他们在大局中获得更多的钱。你有一样东西的优惠券，所以你去商店装满你的购物车。或者你买了你本来不会买的东西。或者你订阅时事通讯来获得10%的折扣，现在你就永远被困在他们的营销圈里了。

但你不必从我这里拿走。以下是营销人员建议公司发放优惠券的一些原因:

> 在顾客买东西的所有原因中，省钱几乎排在首位。这种节省的需求是折扣券成为成功经营电子商务的重要组成部分的原因。研究表明，有优惠券的顾客比没有优惠券的顾客多花25%的钱。 — [塞尔布里特](https://www.sellbrite.com/blog/coupon-marketing-strategy/)

> 消费者喜欢感觉好像收到了一笔“交易”。用一些折扣或奖励来完成交易是一种积极的感觉和安慰(即使对消费者来说没有那么有利可图)。 — [大商业](https://www.bigcommerce.com/blog/coupon-marketing/#digital-coupons-arent-going-anywhere)

> 提供优惠券的好处包括向你的商店或网站介绍新顾客。优惠券还可以引入新的产品线，帮助出售多余或不需要的库存，为新产品腾出空间。优惠券可用于战略性地鼓励客户购买新的、更有利可图的产品，以帮助提高您的利润率。此外，优惠券可以通过将折扣作为奖励来确保现有客户继续从您的商店购买，从而提高他们的忠诚度。—[Investopedia](https://www.investopedia.com/articles/personal-finance/051815/pros-cons-using-coupons-your-business.asp)

喜欢达成交易的另一面是，我讨厌感觉自己像个傻瓜。最终我意识到，我在打折工艺品上花的钱比我偶尔购买全价工艺品的钱还要多。我走向自由的第一步是不再看乔安的传单。如果我自己决定需要某个特定的工艺品供应，我会在出门的时候拿起传单。这些天，我甚至不保存传单。我现在每一两个月去那里购物一次，而不是每周都想去。

同样，我也曾经是好市多的会员，但在类似的情况发生后，我放弃了它。当然，与其他地方的同类商品相比，那里的几乎所有商品都有不错的单位或每磅价格。但是，考虑到我可以在普通杂货店花4美元买到一个普通大小的罐子，当我只想要一个普通大小的罐子时，我真的需要一加仑6美元的沙拉吗？

这里有一种 [星系大脑模因](https://knowyourmeme.com/memes/galaxy-brain) 。首先是在不必要的时候支付全价。接下来是发现优惠券。之后是在优惠券游戏上投入足够的时间和精力，以确保 *你*不是让优惠券变得有利可图的傻瓜之一。最后，你会意识到完全退出优惠券游戏可以节省金钱和时间。有时候唯一的致胜之举就是不玩。
