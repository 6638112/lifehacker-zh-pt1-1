# 在孩子小的时候注册他们的域名

> 原文:[https://life hacker . com/register-your-kid-s-domain-name-when-they-young-1845574134](https://lifehacker.com/register-your-kid-s-domain-name-when-theyre-young-1845574134)

当你的孩子还小的时候，你不可能知道他们将来会成为什么样的人。但是不管他们将来追求什么样的才能、技能或兴趣，他们很有可能会想要一个个性化的网站来支持或推广某些东西。他们可能想在网上收藏自己的绘画作品、业余电影或畅销小说。也许他们会想避开LinkedIn，转而使用自己的在线简历。但如果他们的名字是萨姆·布鲁尔，而sambrewer.com已经名花有主，他们会很失望。

Watch

(我编造了这个名字，但后来 [我变得很好奇，搜索了](https://www.godaddy.com/domainsearch/find?checkAvail=1&domainToCheck=sambrewer.com) 和 [已经有人用了](https://www.sambrewer.com/) ，所以如果你一直在等那个，那就太晚了。)

这就是为什么，在一个感觉父母对我们孩子的未来几乎无法控制的时代，我建议你做这件小事来帮助确保一个更光明的明天——至少因为它与域名满意度有关。注册他们的firstnamelastname.com，并在他们足够大时交出登录信息，以推广他们自己或他们的工作。

我个人已经为我自己、我的丈夫和我们的儿子这样做了。我们使用 [GoDaddy](https://www.godaddy.com/) ，但是还有 [几个好的注册商](https://www.wpbeginner.com/beginners-guide/how-to-choose-the-best-domain-registrar/) ，你可以从任何注册商的网站上搜索可用的域名。注册一个域名的费用大约是每年15美元，这已经足够便宜了，如果你不确定你的孩子长大后是叫查尔斯还是查理，你可能会想注册几个不同的域名。(好奇心再次袭来:[samuelbrewer.com](http://samuelbrewer.com/)也拍了。)

当然了。com域名后缀不是唯一的选择。如果你孩子的名字已经注册了——或者你想听天由命——还有其他域名后缀。酪com是最受欢迎的，因此也是最令人难忘的。

如果他们的父母付不起年费，或者他们只是没想过要这么做，那么这也是一件不错的礼物，可以送给侄女、侄子或家人朋友。
