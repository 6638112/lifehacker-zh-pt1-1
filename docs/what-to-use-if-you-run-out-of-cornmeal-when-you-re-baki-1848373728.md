## 羊皮纸有一些缺点

让我们把它弄清楚:这是羊皮纸路线。从技术上讲，你可以绕过这个额外配料的问题，把你已经醒发成形的梨放到一张羊皮纸上，然后烘烤。你的面包会发酵并烘烤，你可以轻松地剥去羊皮纸(因为它大多是*不粘)。但是你可能会遇到一些问题。最大的问题是羊皮纸*可以*粘在面包上，而且在你的面包完成烘焙之前，没有人知道它是否粘在面包上。*

*其次，大多数瘦肉面团(不含黄油、糖或鸡蛋的面包)是在高温下烘烤的，通常在500华氏度左右或更高。大多数羊皮纸在包装箱上的额定温度高达400°F或450°F。当长时间处于更高的温度下时，羊皮纸会烧焦、变脆并在搬运时开始分解。足以让你焦虑。此外，如果你用荷兰烤箱烤面包，羊皮纸会在两侧留下折痕或深深的皱纹。如果味道好的话，外表并不代表一切，但是在粘在一起的羊皮纸碎片上吃东西就太恶心了。此外，当你可以使用一些可食用的东西时，为什么要浪费羊皮纸？*

## *考虑燕麦、种子或碾碎的玉米片*

*你需要一种能在面团和其他表面之间形成屏障的材料。配料不应该容易被面团吸收，应该有中性或互补的味道，并且在高温下不应该融化，燃烧或释放有害的化学物质。粗磨玉米粉传统上是面包缓冲剂的选择，因为它符合所有这些标准-多沙，中性味道，并且不燃烧。*

*如果你没有玉米面(或者不喜欢玉米面)，这些不寻常的配料完全有用:燕麦片、种子(嗨，一切百吉饼调料)，或者像碎玉米片这样的谷物。如果你决定吃麦片兔子洞，那就吃不含水果或糖霜的普通干麦片。水果会烧焦，冰糖可能会在烤箱里融化，更不用说它们会失去味道。*

*如果你正在做一个杂粮面包，它可以是一个很好的触摸来反映你的面包里面的种子，增加质感和味道。一定要使用完整的种子，比如完整的亚麻籽，而不是磨碎的——种子和坚果中含有大量天然油，当它们被磨碎时，这些榨出的油在烤箱中特别容易烧焦。*

*出于同样的原因，你要避免任何研磨得太细的东西，比如面粉，甚至是研磨得很细的玉米粉(最好是粗的)。精细的粉末很快被吸收到面团的粘性表面，这又变得粘粘的，促使你在下面放更多的面粉来阻止面团粘连，这可能导致结块。粉碎的谷物也会在某些地方燃烧，每吃一口都会有一层厚厚的面粉。*

*不要让那个空玉米罐头阻止你重燃那肮脏的酸面团事件。你可能会发现你有其他的成分，效果很好，甚至可能使你的肉饼比去年的味道更好。*
