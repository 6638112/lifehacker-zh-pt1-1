# 你可以使用没有电池的笔记本电脑

> 原文：<https://lifehacker.com/you-can-use-a-laptop-without-a-battery-1849456604>

很久很久以前，古代的电脑用户对自己说，“这很棒，但我想在床上看 YouTube。”于是，笔记本电脑诞生了:一台你可以带去任何地方的便携式电脑。这种便携性是你的 MacBook、ThinkPad 或其他笔记本电脑的全部意义所在，当然，大电池让它成为可能。不过，最近我在想:笔记本电脑*需要*电池才能工作吗？



这不仅仅是一个任意的思想实验。有时候，笔记本电脑的电池没电了，*需要*移除。有时，旧电池不再充电，而是开始积聚可燃气体，使电池向外膨胀。你可以在 Reddit 的 [r/spicypillows](https://www.reddit.com/r/spicypillows/) 上看到各种设备的这种扩张。虽然膨胀的电池大多与老化的电池有关，但新电池也会受到损坏或制造问题的影响。

无论如何，笔记本电脑中膨胀的电池都是坏消息，它需要被移除。如果您自己不知道如何正确地移除它，请小心地将它带给能够移除它的技术人员。当坏电池*被*移除后，下一步通常是更换它。但是，如果你没有呢？只要你能在没有便携性的情况下生活，你能省下这笔钱，继续使用没有电池的笔记本电脑吗？

简短的回答？是的。视情况而定。

毕竟，你的笔记本电脑可能是一台便携式电脑，但它仍然是一台电脑:你的台式电脑不需要电池来为你计算所需的所有部件供电(CPU、GPU、RAM、存储等)。)，显然，所有这些部件在电池拆除后仍然存在于您的笔记本电脑中(至少它们应该存在)。

当然，台式电脑依赖外部电源运行，但你的笔记本电脑也是如此。你的充电器不仅仅是在电池电量不足时给电池充电:当你给笔记本电脑接上电源时，它实际上是在用电源适配器，而不是正在充电的电池。当你在 macOS 中查看你的电池设置时，你会看到这一点:在旅途中，你会看到“电池”作为电源，但当你插入充电器时，你会看到它切换到“电源适配器”。

## 并非所有笔记本电脑都生而平等

这可能是 Windows 机器的情况，但在 macOS 上，这是另一个故事。尽管当你插上充电器时，macOS 告诉你它是在电源适配器上运行的，但使用没有安装电池的 MacBook 是不可取的。事实上，它甚至可能根本不起作用。

[如果你设法在没有电池的情况下启动一台 MacBook](https://www.youtube.com/watch?v=j9BQnRda5oM)，你会突然觉得你的笔记本电脑老化了五年。这是因为软件识别出没有电池，大大降低了处理器的时钟频率。虽然它可以工作，而且你可以在没有电池的情况下使用你的 MacBook，但这将是一个慢得多的体验，反过来，也不是一个有趣的体验。

因此，首先很可能不值得尝试:如果你需要移除 MacBook 中的电池，你应该支付一个新电池才能正确使用它。话又说回来，这个建议可能适用于所有的笔记本电脑，因为完全移除电池也就移除了机器的一半功能。

但是这篇文章不是关于你是否应该使用没有电池的笔记本电脑:我这是一个你是否可以的问题。你可以！因此，如果你打算尝试使用没有电池的笔记本电脑，无论是为了省钱，还是等待电池更换，或者因为它看起来很有趣，有几件事要记住。

## 如何使用没有电池的笔记本电脑

首先，请确保使用设备附带的电源适配器。没有电池，电源需要按照制造商的意图工作。电量的任何偏差都会给操作带来风险，因为笔记本电脑中没有备用电池来收拾残局。

说到这里，你需要非常小心你的笔记本电脑上的数据，因为你现在只靠一个电源适配器运行。如果停电了，或者你把电源插头从墙上拔掉了，你的笔记本电脑会立即关机，使你的整个系统处于危险之中。节省*所有*的时间。更好的是，使用具有自动保存功能的程序。

也不要忘记在从电源上拔掉插头之前完全关闭机器电源。你可能习惯在离开房间前拔掉笔记本电脑的插头，但由于明显的原因，你再也不能这么做了，但是你需要确保在移动电脑前 100%关机。如果它在关机过程中仍在工作，而你拔掉了它，那就是冒险的事情。
