## **看一看发展最快的职业**

尽管2020年是自大萧条以来就业率最差的一年，但劳工统计局(BLS)预计大多数职业类型的就业增长下降趋势将会逆转。

然而，由于2020年就业下滑如此剧烈，报告中的增长预测偏向受疫情影响最大的工作岗位。例如，增长最快的职业是电影放映员，预计就业率将增长70.5%，受COVID影响最大的其他工作预计也将快速增长，如团体锻炼教练和餐馆工人。

为了避免这一点，该报告提供了一份增长最快的职业清单，*排除了那些就业增长率主要由2020年数字的预期短期复苏驱动的职业。这十年的大赢家将是技术、医疗保健和数据科学，因为远程工作和人口老龄化将推动这些领域的就业增长(包括 24.8 %的火葬场运营商增长——哎呀)。随着越来越多的公司从事数据学习和人工智能工作，预计统计学家、信息安全分析师和数据科学家的工作会更多。*
