# 如果你厌倦了深陷泥潭的伴侣，试试“慢速约会”

> 原文：<https://lifehacker.com/try-slow-dating-if-youre-sick-of-puddle-deep-partners-1847291231>

如果你习惯于在 Tinder、Hinge、Bumble、或任何主要的约会应用程序上浏览海量的个人资料，寻找合适的匹配对象的过程可能会令人疲惫不堪。但是虽然一个应用的即时肯定 可能是鼓励，一旦你真的遇到你喜欢的人，放慢脚步可能会更好。

## 什么是“慢约会？”

这并不是一个新想法，但是在早期求爱阶段慢慢来的策略正在 T2 复兴。慢约会是当两个人让身体亲密消退到次要到关注其他类型的联系。

等待性可能有点清教徒式的敏感，对大多数美国人来说可能不会很有趣，因为许多夫妇 [倾向于在约会的第一个月内发生性关系](https://ifstudies.org/blog/slow-but-sure-does-the-timing-of-sex-during-dating-matter) [某些研究](https://news.cornell.edu/stories/2012/12/fools-rush-sex-early-relationship) 暗示仓促发生性关系会导致最终的不满，尽管。这并不是说，在一段关系的早期，所有身体上的亲密行为都预示着不好的事情即将到来；可以肯定的是，亲密关系的某些方面，如亲吻、拥抱和其他表达爱意的方式，已经被证明能够改善情感和性关系。但是为了达到这一点，尤其是在当代约会充斥着应用程序的环境下，许多专家向推荐慢约会，不仅可以帮助人们找到他们正在寻找的伴侣，还可以回答一些关于他们自己和他们约会意图的问题。

正如心理学家 [凯特·巴勒斯特里告诉男性健康](https://www.menshealth.com/sex-women/a36166213/slow-dating-meaning/) :

> [慢速约会]的核心是专注于同时认识更少的人，这样你就可以和他们在一起——更重要的是，和你自己在一起——并彻底评估联系的质量。

广义地说，慢速约会就是后退一步，从更高的高度评估你的浪漫努力，以更好地了解除了性之外你还需要什么。或者，正如心理学家萨拉·孔拉斯(Sara Konrath)在 2018 年告诉美国全国广播公司新闻频道(NBC News)的那样:“这是基于人们希望放慢脚步，在没有太大压力的情况下相互了解，并专注于高质量的联系和亲密。”

## 你是如何慢慢约会的？

约会应用程序可以接二连三地给你发个人资料，所以限制你的使用可能是个好主意——例如，在 Tinder 或 Bumble 中游泳应该是一项只允许在周二或周四进行的活动。同样，对你的约会对象坦白你在做什么；诚实地推迟性生活没有错——事实上，你的约会对象无疑会欣赏你的坦诚。

另外，你在约会，所以不要害怕广泛撒网。这意味着一次可以随意和多个人约会。这就是约会教练 [杰斯·麦肯告诉问男人](https://www.askmen.com/dating/dating_advice/a-look-at-slow-dating-and-why-it-could-be-a-game-changer-for-your-love-life.html) 的话:

> 我建议在了解一个人的早期阶段，用三四个潜在客户填满你的漏斗...只要你没有和这些潜在客户中的任何一个上床，你就可以和他们慢慢约会。

当然，如果你想慢慢约会的一些人对直接亲密关系的感觉和你不一样，不要惊讶。不过，你可能会得到回报，在发生身体上的亲密行为之前，你会发现 T2 和 T4 有着相同的情感联系。
