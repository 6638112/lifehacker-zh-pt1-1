# 伊维菌素的真相

> 原文：<https://gizmodo.com/the-truth-about-ivermectin-1847570437>

伊维菌素是一种长期用于人类和动物的抗寄生虫治疗药物，最近取代羟氯喹成为“自己做研究”互联网人群的新冠肺炎灵丹妙药。在脸书群体中被广泛讨论， [被黑幕网站兜售，甚至被福克斯新闻频道主持人](https://www.nbcnews.com/tech/tech-news/ivermectin-demand-drives-trump-telemedicine-website-rcna1791) 提及。许多人现在相信伊维菌素既能治疗又能预防新冠肺炎。事实上，尽管这种药物有很有价值的用途，但伊维菌素作为抗病毒治疗或预防药物的证据仍在收集中，目前还很不可靠。公共卫生专家现在不得不警告人们不要购买用于动物的伊维菌素，并将其用作标签外的预防剂，因为它可能导致危险的过量用药，导致人们住院。



那么伊维菌素到底是什么？它真的有希望成为对付新冠肺炎的有用工具吗？

## 伊维菌素是用来做什么的？

虽然伊维菌素最近名声不佳，经常被轻蔑地称为马除虫剂，但这确实是低估了它。几十年来，它一直是动物和人类的无价药物，被世界卫生组织认为是我们拥有的最 [基本药物](https://list.essentialmeds.org/) 之一。其药物类的 [发现者](https://www.nobelprize.org/prizes/medicine/2015/press-release/) 甚至获得了 2015 年的诺奖。伊维菌素如此珍贵，不仅因为它能可靠地治疗多种寄生虫感染，还因为它已经成为一种廉价的仿制药，可以很容易地提供给世界上较贫困的地区。它可以制成药丸、局部凝胶或液体浴，就像一些农民对他们的羊所做的那样。伊维菌素具有科学家所谓的“广谱活性”，这意味着它可以有效地针对许多不同的寄生虫。

## 伊维菌素能治疗或预防新冠肺炎吗？

多年来，科学家们一直想知道这种药物的多功能性是否可以将 T2 应用于其他类型的感染，包括由病毒引起的感染。因此，进入新冠肺炎疫情没多久，一些研究人员又开始研究伊维菌素了。果然，有一些早期的研究表明，它在实验室环境中非常有效地杀死了病毒——当然，这不同于能够帮助感染病菌的人。

当这些早期结果被一些科学家和许多自称疫情专家的人视为伊维菌素对新冠肺炎肯定有效的具体证据时，麻烦就开始了。即使在当时，许多研究人员警告不要过度宣传这些结果，因为伊维菌素不容易被人体吸收。通常情况下，这从来都不是问题，因为当用作抗寄生虫药时，它实际上不需要很高的剂量就能起作用。但是，一些科学家认为，将实验室中用于杀死冠状病毒的剂量转化到现实世界中，可能需要比人类安全使用的剂量高得多的剂量。尽管在其他方面是安全的，但过高剂量的伊维菌素很可能会伤害人类并导致其他方面的 [神经问题](http://www.dar.emory.edu/pi/ivermectin.pdf) ，或者与其他药物产生严重的相互作用。

尽管如此，鉴于当时缺乏任何有效的治疗方法，尝试伊维菌素是绝对值得的。不幸的是，在过去的一年中，大多数声称从治疗或预防新冠肺炎病的药物中获益的研究似乎都非常低质量，而一些研究结果证明是基于 [欺诈性的](https://www.the-scientist.com/news-opinion/surgisphere-sows-confusion-about-another-unproven-covid19-drug-67635) 或 [否则怀疑](https://www.theguardian.com/science/2021/jul/16/huge-study-supporting-ivermectin-as-covid-treatment-withdrawn-over-ethical-concerns) 数据并随后撤回。而当你查看最大的[试验](https://www.cidrap.umn.edu/news-perspective/2021/03/ivermectin-doesnt-speed-recovery-mild-covid-19-study-shows) 或 [总体证据](https://www.cochranelibrary.com/cdsr/doi/10.1002/14651858.CD015017.pub2/full?cookiesEnabled) 时，你会发现伊维菌素几乎没有效果。

尽管有这些发展，这种药物在某些群体中变得非常受欢迎，特别是在社交媒体 上。这些团体，通常是右倾的，对疫情嗤之以鼻，他们会争辩说伊维菌素的真相已经被隐瞒了，并且在大型制药公司的要求下，它现在远离公众。当抗疟药羟氯喹被宣布有可能治愈新冠肺炎时，人们也提出了同样的主张。(羟氯喹作为一种潜在的疗法在世界各地的新冠肺炎患者中进行了研究，但令人遗憾的是 [结果](https://www.nejm.org/doi/full/10.1056/nejmoa2023184) 并不令人满意。)

制药业有 [严重的问题](https://www.npr.org/sections/health-shots/2019/12/31/792617538/a-decade-marked-by-outrage-over-drug-prices) ，它也没有 [回避](https://www.cnbc.com/2021/06/01/big-pharma-launches-campaign-against-biden-over-covid-vaccine-patent-waiver.html) 不时从疫情牟取暴利。然而，在这整个疫情期间使用最广泛的救命药物一直是一种叫做地塞米松的 [廉价类固醇](https://www.nejm.org/doi/full/10.1056/NEJMoa2021436) 。与此同时，伊维菌素的许多主要支持者自己也被 [指控](https://time.com/6092368/americas-frontline-doctors-covid-19-misinformation/) 通过 [推动](https://www.nbcnews.com/tech/tech-news/ivermectin-demand-drives-trump-telemedicine-website-rcna1791) 他们通过自己的生意购买伊维菌素和其他所谓的预防疗法，从轻信和绝望的人身上获利。当无法通过不道德的骗子获得伊维菌素时，这些在线团体会告诉人们从动物用品商店购买并自行服用。至少在密西西比州，这个错误的建议已经导致了一系列来自那些遵循建议并服用药物的人的毒物控制电话。

## 伊维菌素更安全吗？

疾病控制和预防中心最近在 T2 发布了一个警告，警告人们不要在没有处方的情况下使用伊维菌素，引用了两个严重的案例，导致人们出现幻觉和意识模糊而住院。上周，食品药品监督管理局在一条令人难忘的 [推特](https://gizmodo.com/you-are-not-a-horse-you-are-not-a-cow-fda-warns-peo-1847535771) 中做了同样的事情，提醒我们:“你不是一匹马。你不是牛。说真的，你们。别闹了。”

应该强调的是，这些警告并没有发生，因为伊维菌素是一种不安全的药物，当按计划服用时。即使在新冠肺炎的临床试验中，人们在医疗监督下接受了比平时更高的剂量，似乎也没有增加任何重大的健康问题风险。但是人们可能会过量服用伊维菌素，并出现恶心、呕吐和腹泻等症状，而非常高的剂量可能会导致神经并发症，如幻觉、癫痫发作和死亡。和所有药物一样，服用不需要的有副作用的药物也会带来不便。一些伊维菌素使用者可能过于自信，不再担心感染或传播冠状病毒的风险——这种风险实际上可以通过接种疫苗来避免。

即使是现在，伊维菌素也不是不可能作为新冠肺炎药物获得第二次生命。6 月，英国牛津大学 [宣布](https://www.ox.ac.uk/news/2021-06-23-ivermectin-be-investigated-possible-treatment-covid-19-oxford-s-principle-trial) 将该药物加入其正在进行的原理试验，该试验被宣传为世界上最大的针对不需要住院治疗的人的新冠肺炎治疗方法的临床试验。希望伊维菌素可以帮助防止人们的早期症状恶化和/或帮助他们更快地清除感染。

但鉴于目前的证据，这种药物不太可能提供任何超过适度的好处，如果有的话。伊维菌素不仅仅是一种简单的动物除虫剂，但它绝对不是治愈新冠肺炎的灵丹妙药，也不是防止感染的屏障。
