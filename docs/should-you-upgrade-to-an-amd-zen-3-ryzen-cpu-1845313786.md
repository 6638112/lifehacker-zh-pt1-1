# 你应该升级到 AMD Zen 3 锐龙 CPU 吗？

> 原文：<https://lifehacker.com/should-you-upgrade-to-an-amd-zen-3-ryzen-cpu-1845313786>

AMD 今天正式宣布了其全新的 Zen 3 锐龙 CPU，并给了渴望图形的游戏玩家一点点他们可以从本月晚些时候发布的 Big Navi 中期待的东西。我们从 AMD 的事件中提取了尽可能多的细节。如果你想尽快进行系统升级，以下是你需要知道的。



我们将从图形开始，因为没有那么多要谈的。

### 我喜欢你叫我大纳维

对于所有的 PC 制造狂热者来说，我理解你无法得到闪亮的新 Nvidia 显卡的沮丧。AMD 也感受到了你——该公司希望你能在 RX 6000“Big Navi”显卡本月晚些时候正式亮相时给它们一个机会。

该公司忍不住在今天的 CPU 发布会上展示了一点大 Navi。预演的镭龙 RX 6000 卡在三个要求苛刻的游戏中超过了每秒 60 帧——《边疆 3》、《使命召唤:现代战争》和《战争机器 5》——在最苛刻的图形设置下，分辨率为 4K。还不错，但这些都是运行在 AMD 设备上的 AMD 基准测试，所以我们会对它们持保留态度，直到评测人员能够拿到卡片。

这不会花太长时间，因为 AMD 现在忽视渴望显卡的游戏玩家是愚蠢的。我敢打赌，有相当多的人厌倦了试图击败 Nvidia 的 RTX 3000 系列卡经销商机器人，并愿意被 AMD 的产品所左右，即使他们最终没有 Nvidia 的快。(不过，这只是猜测；在 10 月 28 日 AMD 的大 Navi 发布后，我们会知道更多。)

### 让我们谈谈中央处理器:禅 3 锐龙抵达 11 月 5 日

AMD 今天的活动是关于其最新的 Zen 3 锐龙 CPU，包括它所谓的“世界上最好的游戏 CPU”战斗的话，所以让我们变得极客一秒钟。

根据 AMD 的数学计算，从性能功耗比的角度来看，其 Zen 3 锐龙 CPU 的效率是第一代锐龙 CPU 的 2.4 倍，比第二代 Ryzens 的效率高出约 24%。四款 Zen 3 芯片都支持 DDR4-3200，但只有一款自带冷却器——最便宜的锐龙 7 5600X 芯片。BYOC，狂热爱好者。

AMD 抛出了大量其他数据来展示其 Zen 3 处理器的原始性能，声称它们比其最新的发烧级芯片 Zen 2 锐龙 9 3900XT 平均快 25%左右。不过还是那句话，这些都是 AMD 的基准测试。

像以前一样，值得等待一系列不那么偏颇的评论，以在各种游戏基准和现实世界场景中测试 AMD 的芯片。

也就是说，我已经看到 [的报道](https://www.anandtech.com/show/16148/amd-ryzen-5000-and-zen-3-on-nov-5th-19-ipc-claims-best-gaming-cpu/2) 已经暗示 AMD 的数据，如果准确的话，将使其 5900X 和 5950X 芯片成为你能买到的单核性能最快的芯片。我仍然会坚持到评论测试这些芯片，但与顶级 Zen 2 芯片相比，它在单核 [CineBench](https://lifehacker.com/how-to-create-a-portable-version-of-cinebench-r20-1833201374) 测试中近 18%的性能提升不是开玩笑的。

AMD 的 Zen 3 锐龙 CPU 将于 11 月 5 日上市，这里是您可以购买的完整阵容:

*   AMD 锐龙 9 5950X: $799。16/32(内核/线程)，3.4GHz 基本时钟，4.9GHz 加速时钟，72MB 高速缓存
*   **AMD 锐龙 9 5900X:** $549。12/24(内核/线程)，3.7GHz 基本时钟，4.8GHz 提升时钟，70MB 缓存
*   AMD 锐龙 7 5800X: $449。8/16(内核/线程)，3.8GHz 基本时钟，4.7GHz 加速时钟，36MB 高速缓存
*   AMD 锐龙 7 5600X: $299。6/12(内核/线程)，3.7GHz 基本时钟，4.6GHz 增强时钟，35MB 高速缓存。附带幽灵隐形 CPU 冷却器

你不能预购这些芯片。如果你观察入微，你会注意到你为每块芯片支付的费用比同等的 Zen 2 多 50 美元。我怀疑 AMD 的 Zen 3 芯片仍然会以每美元的性能战胜英特尔，但一旦每个人都有机会进行一些测试，我们就会看到这些数字是如何震动的。谁知道明年年初英特尔推出 Rocket Lake 芯片时我们会看到什么。

当你等待 Zen 3 时，请确保你的 AMD 500 系列主板更新到最新的 BIOS，这样你所做的任何升级都将在发布日生效。AMD 400 系列主板*也将*支持锐龙 5000 系列芯片，但测试版 BIOSes 预计要到 1 月份才会上市，稳定版将在明年晚些时候发布，具体时间由各主板制造商决定。
