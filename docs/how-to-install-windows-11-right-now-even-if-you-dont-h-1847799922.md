# 如何立即安装 Windows 11(即使你没有更新)

> 原文：<https://lifehacker.com/how-to-install-windows-11-right-now-even-if-you-dont-h-1847799922>

在所有的新闻、炒作和预览之后，微软的 Windows 11 终于来了。你可能会很兴奋地更新你的 Windows 10 机器，以体验微软在其最新更新中添加的所有新功能和变化。太好了！但是更新可能不像通常那么简单，所以我们会带你在你的机器上安装 Windows 11，即使你还没有看到更新。

## 微软不希望 Windows 11 在所有机器上运行

你可能听说过，微软对 Windows 11 的硬件要求相当严格。为了以 Microsoft 批准的方式在您的计算机上下载并安装最新的更新，您需要:

*   英特尔第八代 Coffee Lake 或 Zen 2 CPUs 及以上
*   至少有两个内核的 64 位 1 GHz 处理器
*   TPM 2.0 支持
*   4 GB 内存
*   至少 64 GB 的可用存储空间

如果你的个人电脑或笔记本电脑不符合这些规格，不要担心；在你的电脑上安装 Windows 11 还是有合法途径的。我们稍后会讨论它们。目前，我们假设你有一台支持 Windows 11 的机器。你从这里去哪里？

## 如何检查你的机器上是否有 Windows 11

不幸的是，微软并没有一次将这个更新推广到所有的电脑上。相反，它是增量式的，一开始就优先考虑更新的硬件。它承诺所有符合条件的机器将被要求在 2022 年年中之前更新，这离现在还有很长时间。

你应该做的第一件事是检查微软是否认为你的机器值得更新邀请。为此，前往**设置>更新&安全> Windows 更新**，查看是否有任何新的更新。如果你的机器是幸运儿之一，你应该会看到一个可用的 Windows 11 更新，就像你会看到任何其他 Windows 更新一样。恭喜你。然后按照屏幕上的说明下载并安装软件。

## 如果没有更新，如何安装 Windows 11

如果你*没有*看到这个更新，那也没关系——由于微软反复无常的，只有有限的几台电脑会在它们的设置应用中发现 Windows 11 更新。F 首先，使用 PC Health Check 应用程序，通过微软检查你的机器是否真的有资格获得官方更新，如果你还没有它，你可以 [在这里](https://aka.ms/GetPCHealthCheckApp) 下载。

如果你的电脑符合微软的硬件要求，你可以手动安装 Windows 11 更新，而无需等待微软将其添加到你的更新页面。为此，前往微软的“ [下载 Windows 11](https://www.microsoft.com/en-us/software-download/windows11?ranMID=24542&ranEAID=nOD/rLJHOac&ranSiteID=nOD_rLJHOac-57mrfNUN94KCYNwXKWq3fQ&epi=nOD_rLJHOac-57mrfNUN94KCYNwXKWq3fQ&irgwc=1&OCID=AID2200057_aff_7593_1243925&tduid=%28ir__np0gvpp19gkf6y1ndwtfs1ms2e2xrcryi0slithf00%29%287593%29%281243925%29%28nOD_rLJHOac-57mrfNUN94KCYNwXKWq3fQ%29%28%29&irclickid=_np0gvpp19gkf6y1ndwtfs1ms2e2xrcryi0slithf00) 页面。从 t 这里开始，你有三个选项可以在你的机器上安装 Windows 11 更新:

*   **Windows 11 安装助手**:微软向那些设备上还没有更新的人推荐这个选项。助手将引导您完成安装过程。点击**立即下载**即可开始。在开始此页面上的之前，您可以通过点击**找到额外的安装说明。**
*   **创建 Windows 11 安装介质** : T 这个选项允许你使用 Windows 11 ISO 在你的机器上执行 Windows 11 的全新安装。您需要使用空白 DVD 或 USB 驱动器在您的电脑上执行安装。如果你以前从未做过，这种类型的安装会很复杂。F 按照微软的指示这样安装 Windows 11，在你开始这个页面上的之前点击**。**
*   **下载 Windows 11 磁盘镜像(ISO)** : T 这个选项可以让你下载 Windows 11 ISO，随心所欲。您可以从 DVD 或 USB 驱动器安装它，如上面的选项，或者直接从 ISO 文件安装。同样，这种类型的安装可能很复杂；在开始这个页面上的之前，你可以通过点击**来跟随微软的安装方向。**

## 如何在不支持的机器上安装 Windows 11

如果你的机器不支持微软对 Windows 11 的硬件要求，你仍然可以在你的电脑上安装该软件。你需要使用上面步骤中的“**创建 Windows 11 安装介质**或“**下载 Windows 11 磁盘映像(ISO** )”选项，因为微软不允许你在不受支持的硬件上使用安装助手。然而，该公司建议你不要在你不支持的机器上安装 ISO，并警告说，那些这样做的人可能不会收到自动更新。这意味着你需要在每个 Windows 11 更新发布时从 ISO 文件中安装它们。

如果这听起来比它的价值更麻烦，还有其他的选择供你探索。我们整理了一个 [演练，让 Windows 10 感觉更像 Windows 11](https://lifehacker.com/how-to-turn-windows-10-into-windows-11-when-you-can-t-a-1847719257) ，以备无法(或不想)更新时使用。这样，你可以获得 Windows 11 的美学优势，而不必经历通过 ISO 一遍又一遍地安装操作系统的麻烦。
