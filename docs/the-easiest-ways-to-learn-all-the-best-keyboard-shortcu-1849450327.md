## Windows: PowerToys 快捷指南

[微软 PowerToys 是一个强大的实用工具](https://lifehacker.com/the-best-microsoft-powertoys-shortcuts-you-ll-want-to-u-1848634350) ，它给你的电脑增加了几个有用的功能。出于我们的目的， [下载 PowerToys](https://github.com/microsoft/PowerToys/releases/tag/v0.61.1) ，打开应用，并点击左侧窗格中的**快捷指南**选项卡。

确保上的【启用快捷键指南】开关设置为**，然后使用【是的，】a 键盘快捷键 ( **Windows 键+ Shift + /** ) 启动实用程序。当你启动快捷键指南时，你会看到一系列对 Windows 最有用的组合键。**

请注意，快捷键通常链接到键盘上的 Windows 键，因此在描述中可能不是指那个键。如果你在快捷指南中看到一个链接到 A 的快捷方式，这意味着你必须按下 **Windows 键+ A** 。每当你想查阅这个快捷键列表时，按下 **Windows + Shift + /** 就可以了。

(当你充分利用这个功能的时候，别忘了查看一下 [这个隐藏的 YouTube 键盘快捷键列表](https://lifehacker.com/youtube-has-a-hidden-list-of-keyboard-shortcuts-1849093707) 。)

## Mac: CheatSheet

要访问所有对你来说最好的快捷方式的便捷指南， Mac， [CheatSheet](https://www.mediaatelier.com/CheatSheet/) 是你需要的应用程序。免费，快速设置，简单易用。

一旦你安装了 CheatSheet，只要按住 **Command** 键几秒钟，你就会看到你正在使用的任何应用程序的所有可用的键盘快捷键。这是一个简单的快捷方式，经常会派上用场，所以苹果为什么没有将其作为 macOS 的默认功能来实现是一个谜——，尤其是它是 iPadOS 的默认功能。

## iPad:命令键

你的 iPad 有一大堆有用的 [键盘快捷键](https://lifehacker.com/every-ipados-15-keyboard-shortcut-worth-knowing-about-1847713494) ，并带有一个内置的学习方法。每当你打开任何应用程序时，按住**命令**按钮来查看你可以使用的所有键盘快捷键。如果你在主屏幕上这样做，你会看到系统默认的键盘快捷键。在任何应用程序中尝试它，你会看到特定于该应用程序的快捷方式。

只有当你在 iPad 上使用物理键盘时，这个功能才有效，因为屏幕键盘快捷键在 iPadOS 上并不存在。只要物理键盘连接到设备上，同样的快捷键也可以在你的 iPhone 上使用。
