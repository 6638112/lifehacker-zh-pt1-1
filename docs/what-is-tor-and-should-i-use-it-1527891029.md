# 什么是Tor，我为什么要使用它？

> 原文:[https://life hacker . com/what-is-tor-and-should-I-use-it-1527891029](https://lifehacker.com/what-is-tor-and-should-i-use-it-1527891029)

Tor是匿名浏览网页最简单的方法之一。但是现在我们已经说了“a”这个词，我们需要在它旁边加上一个大大的星号，因为使用Tor来隐藏你的在线活动会带来一堆警告。让我们来看看Tor做什么，谁使用它，最重要的是，如果你想在网上隐藏，Tor *不会*做什么。

### **Tor如何工作**

Tor是洋葱路由器的缩写(因此是其标志),最初是由美国海军开发的全球服务器网络，使人们能够匿名浏览互联网。现在，它是一个非营利组织，主要目的是研究和开发在线隐私工具。

Tor网络通过加密您的流量并在网络内的不同Tor中继之间传输来伪装您的身份。软件工程师罗伯特·希顿 [有一个很好的总结](https://robertheaton.com/2019/04/06/how-does-tor-work/) 如何让你(理论上)保持匿名:

> *当您使用普通的网络浏览器访问网站时，您的电脑会与网站的服务器建立直接的TCP连接。任何监视你的互联网连接(或服务器连接)的人都可以轻易地检查你的IP数据包报头，发现你和服务器的IP地址，并推断出你们正在相互通信。只要你和服务器使用加密的HTTPS通信，窥探器就无法读取你信息的实际内容。但是——就像X非常了解的那样——有时候，仅仅知道你在和谁交流就足够了。*
> 
> *相比之下，当您使用Tor浏览器访问网站时，您的电脑不会直接与网站的服务器通信。相反，Tor浏览器通过一组随机的3个Tor节点构建了一条曲折的路径，并通过这条线路发送数据。浏览器首先将数据发送到电路中的第一个(或保护)节点。保护节点将您的数据发送到第二个(或中间)节点。中间节点将您的数据发送到第三个(或出口)节点，最后出口节点将您的数据发送到网站的服务器。服务器将其响应发送回出口节点，出口节点负责通过电路的其余部分将响应传播给您。*

你要访问Tor所要做的就是下载 [Tor浏览器](https://www.torproject.org/download/download-easy.html.en) 。启动它，你在浏览器中所做的一切都将通过Tor网络。大多数人不需要调整任何设置；它只是工作。也就是说，由于你的数据要经过许多中继，你在Tor上的体验可能比你正常的互联网浏览更慢。

### **什么Tor好**

Tor对于那些想让自己的互联网活动远离广告商、互联网服务提供商和网站的人来说非常有用。这包括规避本国审查限制的人，希望隐藏ip地址的人，或者任何不希望自己的浏览习惯与自己联系在一起的人。

Tor网络还可以托管其他Tor用户只能访问 的网站 [。换句话说，你现在已经进入了黑暗网络的世界，或者是那些没有被你用来搜索可爱的动物、要购买的东西和琐事答案的常规爬虫索引的网站。你可以在黑暗的网络上](https://medium.com/@jasonrigden/how-to-host-a-site-on-the-dark-web-38edf00996bf) [找到任何东西](https://www.ma-no.org/en/security/the-best-pages-of-the-deep-web-onion-deep-web) 从免费的教科书到药物——甚至更糟——只要你知道带你去这些网站的特殊网址。小心行事。

### **Tor不做的事情**

Tor在理论上听起来很完美——一个免费、简单的系统，你可以用它在网上过着秘密的生活。但远非如此。如果你使用Tor，有很多方法可以放弃你的安全性和匿名性。例如，考虑一下裸体安全公司的 [保罗·小鸭](http://%20https//nakedsecurity.sophos.com/2020/08/13/tor-and-anonymous-browsing-just-how-safe-is-it/) 中的场景:

> 虽然Tor的出口节点无法知道你在哪里，但由于入口守卫和中间中继(经常变化)的匿名效果，它们确实可以看到你最终的解密流量及其最终目的地，因为正是出口节点剥离了Tor的最后一层混合神秘加密。
> 
> *[...]*
> 
> 换句话说，如果你使用Tor浏览一个非HTTPS(未加密)的网页，那么处理你的流量的Tor出口节点不仅可以窥探和修改你发出的web请求，还可以干扰任何返回的回复。

虽然Tor可能有助于隐藏是你的电脑发出了最初的请求，比如说，访问一些粗略的互联网论坛并做各种可怕的事情，但如果你在那个网站上创建帐户，它不会帮你任何忙。如果该帐户与非法活动、支付和/或真实地址有关联，那么您使用什么浏览器或匿名技术访问该网站都无关紧要。 [你终究不会很隐蔽](https://www.wired.com/2013/11/silk-road/) 。

这还不是全部。你使用普通网络浏览器的许多方式也可能在Tor上泄露你的身份——或者，至少，留下足够的面包屑，让一个专门的实体更容易发现你是谁。正如Tor项目 [中描述的](https://support.torproject.org/) :

> Tor Browser会屏蔽Flash、RealPlayer、Quicktime等浏览器插件:它们可以被操纵泄露你的IP地址。同样，我们不建议在Tor浏览器中安装额外的附件或插件，因为这些可能会绕过Tor或损害您的匿名和隐私。

此外:

> 在自动打开由外部应用程序处理的文档之前，Tor Browser会发出警告。不要忽略这个警告。通过Tor下载文档(尤其是DOC和PDF文件，除非您使用Tor浏览器内置的PDF查看器)时，您应该非常小心，因为这些文档可能包含互联网资源，打开它们的应用程序会将这些资源下载到Tor之外。这将显示您的非Tor IP地址。如果您必须处理通过Tor下载的文件，我们强烈建议您使用断开连接的计算机，或者使用dangerzone创建您可以打开的安全PDF文件。然而，在任何情况下，同时使用BitTorrent和Tor都是不安全的。

如果一个实体决心找出你是谁，并愿意为你提供一些恶意软件，你使用Tor的简单事实不会阻止他们。据2013年Motherboard报道，一个以Tor为主题的FBI半身像 展示了这可能是如何工作的:

> 联邦调查局今年夏天的大规模儿童色情搜查也引起了隐私倡导者的一些怀疑，他们怀疑联邦调查局渗透Tor有多容易。 [*美国联邦调查局通过向浏览器注入恶意软件，成功破解了匿名网络*](http://motherboard.vice.com/blog/the-fbi-says-it-busted-the-biggest-child-porn-ring-on-the-deep-web-1) *，以确定其所谓的全球“最大”儿童色情服务商。“在这个过程中，恶意软件泄露了数百名用户的IP地址。*

### **那么，应该使用Tor吗？**

如果你是一个普通用户，在看猫的gif和浏览脸书，你可能不需要担心政府监视你的活动。Tor只会减慢你的连接速度。更有可能的是，当你使用公共wifi 时，你需要*保护*你的互联网，而不是*匿名化*它，比如说。在这种情况下，你会想确保你在所有支持HTTPS的网站上使用它，甚至可能在你出门在外时使用VPN加密你所有的流量。

如果你没有VPN，Tor总比没有好，但我不会用它登录任何服务——尤其是金融服务。你仍然不知道谁控制着你中继中的各个节点，包括那个至关重要的出口节点；比起单一来源的VPN，我更愿意相信我的连接(尽管从理论上讲，你仍然通过另一个实体传递你的数据)。

换句话说，如果不是*真的*需要匿名，就不用担心Tor。

如果你想保持匿名，因为你正在下载大文件，不想让别人看到你正在下载的内容——比如说，在BitTorrent上——[Tor不是一个好的解决方案](https://blog.torproject.org/bittorrent-over-tor-isnt-good-idea) 。不要做那种无缘无故减慢别人交通速度的混蛋。同样重要的是，你 [实际上可能根本不是匿名的](https://www.reddit.com/r/TOR/comments/75v54d/how_does_the_bittorrent_protocol_leak_your_ip/) 。在这种情况下，你会想要 [一个VPN而不是](https://lifehacker.com/how-to-find-a-trustworthy-vpn-1833045522) 。

记住，Tor不仅仅是一个“免费VPN”。这两种方法都可以帮助你在网上匿名，但是方法大相径庭。
