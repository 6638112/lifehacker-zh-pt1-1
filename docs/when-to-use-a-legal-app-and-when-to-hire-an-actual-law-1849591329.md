# 何时使用法律应用程序(以及何时雇佣真正的律师)

> 原文：<https://lifehacker.com/when-to-use-a-legal-app-and-when-to-hire-an-actual-law-1849591329>

我们绝对是生活在未来。智能手机以及人工智能(AI)和机器学习(ML)的进步让普通公民能够获得曾经只为有钱人保留的资源。需要私人厨师？注册让美食套餐送货上门。需要司机吗？有太多的拼车应用程序了。现在，律师们。



拥有一名律师——在某种意义上，拥有一家你可以随时联系的律师事务所——绝对是富人的举动。我们大多数人都负担不起让律师坐在 旁等着我们的电话，公平地说，我们大多数人也不需要律师。这意味着当我们*确实*需要法律建议时，这是一个寻找律师(以及找到聘请律师的钱)的紧张过程。但现在我们有了越来越多的法律应用程序，如 [DoNotPay](https://donotpay.com/) ，一个“机器人流程自动化(RPA)”应用程序，帮助人们做一些事情，如提起诉讼，处理停车罚单，以及重新获得社交媒体账户。每月支付相对较少的费用，你就可以随身携带一名律师。但是这些应用程序真的有用吗？

## 人工智能是如何在法律职业中使用的

需要注意的一点是，人工智能正在被用于法律行业——这是一个大时代。世界各地的律师事务所已经将人工智能和机器学习纳入了他们的流程 ，因为这些技术是定制的，可以处理像发现这样的法律流程生成的大量文本。人工智能工具用于自动审查合同，搜索相关文件，甚至根据法官的记录模拟审判和预测结果。

人工智能对律师如此有效的原因是，人工智能和机器学习擅长快速准确地处理数据，而如此多的法律正在处理大量文件和装满数据的硬盘。当涉及到导航法律代码和法庭时，人工智能可以是有效的，因为法律本质上是世界上最混乱和构建最差的算法。法律程序中的每一步都会触发有限数量的潜在其他步骤，因此人工智能可以加快计算法律行动中接下来的步骤的过程。

## 「口袋律师」到底有多有用、多有效？

虽然人工智能在数据处理方面的惊人效率使其对训练有素、经验丰富的律师极其有用，但这能转化为应用程序“口袋里的律师”的力量吗？答案是褒贬不一。

例如，DoNotPay 的工作原理是将法律分解成自动化的任务，这些任务可以根据需要串联起来。这使得该应用程序比 [信托&遗嘱](https://trustandwill.com/) 更加灵活，后者只能创建(你猜对了)信托和遗嘱，因为它可以随着你的诉讼或案件的进展增加一些法律工作；DoNotPay 声称它提供数百种服务，包括提供虚假的虚拟信用卡来注册免费试用，以及取消公司的订阅，这使得这样做非常困难。

但是，尽管有这种灵活性，所有这些法律应用程序都是有限的，并且您使用它们的成功将取决于您法律需求的复杂性。为一张停车罚单打架？这是一个非常简单的法律算法，步骤非常有限，潜在的复杂性也非常有限，所以像 DoNotPay 这样的东西在这方面相当有效:全国挑战停车罚单的成功率只有 40%，但 DoNotPay 声称它在 70%的情况下成功了。

类似地，当使用应用程序创建遗嘱时，只要你的需求相对简单，你就会做得很好。如果你有一个复杂的财务状况或不寻常的遗产计划，你会很快遇到应用程序算法的硬性限制。这反映在 DoNotPay 的评论中，这些评论并不都很好——以及它在商业改善局 的 [低 F 评级。](https://www.bbb.org/us/ca/san-francisco/profile/legal/donotpay-1116-925387)

另一个考虑因素是你的法律需求需要你走多远，因为法庭诉讼仍然需要人类出庭。你能使用法律应用程序来代表自己吗？实际上，法官禁止你这样做并不是不可能的，因为依赖一个应用程序可能会被认为是你没有能力为自己辩护的证据。

底线呢？智能手机上的法律应用程序可以代替人类律师进行简单、直接的法律诉讼，不需要审判或其他冗长的出庭。但是，如果你的法律需求有一点复杂，你最好支付一个真正的律师——或者，至少，使用一个提供真正律师服务的法律应用程序，如 [火箭律师](https://www.rocketlawyer.com/) 。
