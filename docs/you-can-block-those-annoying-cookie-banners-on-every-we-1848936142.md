# 你可以屏蔽每个网站上那些烦人的Cookie横幅

> 原文:[https://life hacker . com/you-can-block-the-noise-cookie-banner-on-every-we-1848936142](https://lifehacker.com/you-can-block-those-annoying-cookie-banners-on-every-we-1848936142)

有一长串的事情让浏览互联网感觉像是生活在一个反乌托邦的地狱场景中，而令人讨厌的cookie横幅和同意书就在这个列表的最上面。大多数网站在法律上需要显示这些通知，并征得您的同意才能使用cookies。然而，它也屏蔽了你试图阅读的网站的一大块内容。让我们在您的所有设备上解决这个问题。

## 阻止桌面上的cookie横幅

无论您使用什么平台，您都可以轻松安装浏览器扩展来阻止cookie通知。 [超级代理](https://www.super-agent.com/) 是一个免费的扩展，可以在Chrome、Firefox、Safari和Edge上工作。它可以自动清除cookie横幅，并通过进入你访问的几乎所有网站。该扩展的首选项页面允许您定制想要接受的cookiess和想要拒绝的cookie。这是一个关键特性:你不会想使用一个自动接受你去的每个网站上的所有cookies的插件。

如果你不愿意为此安装一个专用的扩展，你可以使用 [uBlock Origin](https://ublockorigin.com/) 作为替代。默认情况下，它会阻止一长串恼人的事情，但你必须手动启用它的cookie- 通知阻止列表。点击子块原点图标，进入到**设置>过滤列表>烦恼**。启用 **EasyList Cookie** 隐藏Cookie横幅。

请注意，自动阻止cookie通知可能会阻止一些网站加载，这就是为什么你可以考虑使用uBlock Origin的元素隐藏功能。当你在任何网站上看到一个cookie通知，你可以从浏览器的工具栏中点击uBlock Origin扩展图标，选择**滴管图标**。

这将进入element- picker模式:H 将鼠标光标放在cookie通知上，直到整个横幅被选中，单击一次，并在弹出窗口中选择**创建过滤器**。您可能需要在多个网站上手动完成这项工作，但对于您访问的每个网站来说，这是一次性的工作，只需点击几次。

## 阻止iPhone和iPad上的cookie横幅

只要你的iPhone和iPad更新到iOS 15，你就可以使用一些 [伟大的Safari扩展](https://lifehacker.com/12-better-ways-to-browse-the-internet-on-your-iphone-1847726577) 来屏蔽网络上的烦恼。然而， [超级代理](https://www.super-agent.com/) 依然是iPhone和iPad最好的cookie- 通知处理程序。它可以加载cookie横幅，并根据您的偏好自动接受或拒绝cookie。这在网站加载后需要一秒钟左右的时间，但是在启用扩展后你不需要做任何事情。

## 阻止Android上的cookie横幅

如果你在Android上使用Firefox，你可以安装浏览器插件uBlock Origin来屏蔽所有网站上的cookie通知。如果你没有，你最好的选择是使用类似于[Android](https://adguard.com/en/adguard-android/overview.html)的广告拦截器，它也可以拦截各种网站上的cookie横幅。不幸的事实是，这些类型的应用程序在Google Play上是不允许的，所以你可能必须将它下载到你的手机上。
