在疫情独自一人或与其他家庭成员一起度过如此长的时间后，人们热情地返回旅行——许多人独自上路。

Airbnb 在其内部数据 中发现了一种趋势:2021 年预订的所有住宿中有 26%，2022 年第一季度预订的长期住宿中超过 50%是单人旅行。作为回应，该公司最近推出了一项新功能，帮助独自旅行者保持安全。以下是需要了解的内容。

## 关于 Airbnb 新的安全功能，你需要了解什么

当独自旅行的人预订私人或共享房间时， [Airbnb 将激活新的应用内功能](https://news.airbnb.com/new-safety-product-for-solo-travelers/) 。除了提供通常的信息——入住/退房时间、租车路线以及联系主人的方式——它还为用户提供独自旅行时保持安全的专家提示。

旅行者还将能够轻松地与朋友、家人或其他信任的人分享他们的 Airbnb 预订，以便在逗留期间发生紧急情况时可以联系到他们。这些信息包括列表地址、预订代码和旅行者的入住和退房日期。

在这一点上，所有 Airbnb 用户——独自旅行或与他人一起旅行——都可以选择在他们的个人资料中添加一个紧急联系人，以防公司不得不代表他们联系某人。这些信息可以添加到页面底部“个人信息”标签的“帐户设置”中，只有 Airbnb 可以看到。

最后，在收到预订确认后，旅行者会看到自动提示，其中有他们可能想问主人的关于房源和周围邻居的问题。

目前，这项新功能仅在单人旅行者预订私人房间或共享空间时提供英语版本，但 Airbnb 计划在未来一年推出其他语言版本，并包括整个房屋清单。
