# 幸存的 COVID 也可能有精神健康的影响

> 原文：<https://lifehacker.com/surviving-covid-can-have-mental-health-implications-to-1846646047>

根据一项新的分析，与从流感或其他呼吸系统疾病中康复的人相比，从新冠肺炎病中康复的人更有可能患有影响其心理健康或大脑和神经系统的疾病。换句话说，如果你在 COVID 之后一直有心理健康问题，你并不孤单。



### 研究发现了什么？

发表在《柳叶刀精神病学》 上的新研究 [研究了数百万份医疗记录，以了解在 COVID 病例后人们得到了什么诊断。研究人员发现了各种各样的“神经和精神”结果，从中风到焦虑和抑郁。许多这些问题发生在患有严重 COVID 的人群中，但在患有轻度和中度疾病的人群中仍然存在大量问题。](https://www.thelancet.com/journals/lanpsy/article/PIIS2215-0366(21)00084-5/fulltext)

大约三分之一的 COVID 患者有其中一种诊断，12.8%的 COVID 患者在冠状病毒发作后有新的*诊断。其余的人包括正在患病的人或过去曾患病并目睹其复发的人。*

很可能并非所有这些都与 COVID 直接相关；例如，如果你过去抑郁，你可能会再次抑郁，无论你在此期间是否生病。但另一方面，COVID 似乎有一些特别之处，这反映在数字中。

我们已经知道 [COVID 增加了中风](https://weillcornell.org/news/what-to-know-about-stroke-and-covid-19) 的风险，而 [在重症监护室度过的时间增加了谵妄](https://www.icudelirium.org/patients-and-families/overview) 的风险，这可能包括混乱和记忆问题。我们也不知道哪些人患有 COVID，但没有去医生或医院接受治疗；他们不包括在这项研究中。

一些从 COVID 感染中康复的人还描述了认知问题， [被称为“脑雾”，](https://healthmatters.nyp.org/what-is-causing-covid-brain-fog/) ，这可能与冠状病毒影响大脑的方式或免疫系统反应有关。

两位精神病学家在 [的一篇附带社论](https://www.thelancet.com/journals/lanpsy/article/PIIS2215-0366(21)00120-6/fulltext) 中写道，COVID 之后的一些精神健康状况可能是“心理社会的”换句话说，你可能会因为你在这个世界上的经历而最终患上抑郁症或焦虑症，这些经历包括去医院就诊、被隔离，或许还有工作或财务问题，而不仅仅是因为冠状病毒对你的大脑造成了影响。

无论确切的原因是什么，这项研究强调 COVID 和心理健康问题是相关的，所以如果你在摆脱感染后没有感觉到自己，你真的不是一个人。如果你想和其他人交流，可以去看看[“long COVID”支持团体](https://www.longcovid.org) ，并考虑是时候 [找一位治疗师](https://lifehacker.com/its-time-to-find-a-therapist-1846009669) 来处理任何挥之不去的(或新的)心理健康问题了。
