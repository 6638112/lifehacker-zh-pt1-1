# 在实际无法更新的情况下，如何把 Windows 10 变成 Windows 11

> 原文：<https://lifehacker.com/how-to-turn-windows-10-into-windows-11-when-you-can-t-a-1847719257>

许多个人电脑不符合 Windows 11 的严格硬件要求 。虽然你可以尝试在不受支持的硬件上安装操作系统，但还有另一个选择:让 Windows 10 更像 Windows 11。只需要几个免费的应用程序，一个自定义的桌面主题，并启用一个增强的安全选项就可以做到这一点。

Watch

当然，它不会是 Windows 11 的完美再现。你的电脑仍将运行 Windows 10，Windows 11 的许多新功能和设置无法重新创建，但你可以非常接近——特别是在匹配 Windows 11 的外观方面。

## 步骤 1:创建还原点

在开始之前，创建一个新的系统还原点。不要跳过这一步！

我们将更改几个重要的设置，因此拥有一个最近的还原点可以确保您在出现问题时可以安全地恢复到当前配置，并且所有文件保持不变。如果您后来不喜欢自定义桌面外观，这也有助于恢复您的更改。

1.  使用 Windows 任务栏搜索**“恢复”**。
2.  从搜索结果中选择**“创建还原点”**，打开系统属性菜单。
3.  在**系统保护**选项卡中，选择您的硬盘并确保保护设置为**开启**如果关闭，点击**“配置”**，然后切换**“开启系统保护”**
4.  打开系统保护后，从列表中突出显示该驱动器，然后选择**“创建”**
5.  为新还原点添加描述，然后单击**“创建”**
6.  等待备份过程完成。

如果您最终需要使用恢复，请使用上述步骤返回系统属性菜单，然后单击**“系统恢复”**，并按照屏幕上的说明进行操作。

## 第二步:让 Windows 10 看起来像 Windows 11

接下来，我们将使 Windows 10 看起来更像 Windows 11——包括 [有争议的新任务栏](https://lifehacker.com/how-to-go-back-to-windows-10-if-you-regret-installing-w-1847283558/amp) 和开始菜单，圆形文件夹和任务栏角，新图标和背景，等等。

为此，你需要安装一个基于 Windows 11 的定制的第三方桌面主题和图标集，这比简单地从微软商店安装主题要多得多。如果你只是想给 Windows 11 一个快速、临时的测试，有一种方法可以让你不用安装任何东西就能直接看到操作系统的新界面。也就是说，如果你想要更完整的体验(并且不能安装测试版),自定义主题是你最好的选择。

然而，在我们应用自定义主题之前，您需要安装主题运行所需的几个第三方应用程序。

要装的东西很多，但不要觉得太不堪重负；每个应用程序或文件的完整安装说明都可以在下面列出的每个下载页面上找到，所有这些应用程序都是开源的和/或经过认证可以安全使用。

这是你需要的:

*   [**SecureUXTheme**](https://github.com/namazso/SecureUxTheme)**:**这款 app 可以让你安装第三方 Windows 10 主题，而不会影响你的系统文件。
*   [**旧浏览器**](https://msfn.org/board/topic/170375-oldnewexplorer-119/) **:** 改变 Windows 资源管理器菜单的外观。
*   [**StartisBack**](https://startisback.com/)T5】($ 4；免费试用版):在 Windows 7 的基础上增加了一个更好的开始菜单。完整版的按时收费 4 美元，但试用版可以无限期使用(尽管功能有限)。你只需要免费试用就可以使用自定义的 Windows 11 主题。
*   [**Stardock Curtain mod**](https://www.stardock.com/products/curtains/)**:**在 Windows 10 上圆化文件夹窗口，就像在 Windows 11 上一样。
*   [**Windows Store 上的 rounded TB mod**](https://www.microsoft.com/en-us/p/roundedtb/9mtftxsj9m7f?activetab=pivot:overviewtab)**:**给任务栏添加圆角边缘。

一旦必备的应用程序安装到你的电脑上，就该让它看起来像运行 Windows 11 了。为此，我们将使用 niivu 的 [Windows 11，用于 Windows 10 主题](https://www.deviantart.com/niivu/art/Windows-11-for-Windows-10-882819383) (它只需要一个免费的 DeviantArt 帐户)。Niivu 的安装说明是 [这里有](https://www.deviantart.com/niivu/art/Windows-11-for-Windows-10-882819383) ，还有 [使用自定义桌面的提示](https://www.deviantart.com/crcustomization/journal/Introduction-to-Desktop-Customization-Themes-800075399) ，不过这里是简化版:

1.  首先，我们需要弄清楚你用的是哪个版本的 indows。按下**视窗键+ R.**
2.  键入**“winver”**然后按回车键。
3.  记下版本号(应该在 1903 或 21H1 之间)，然后关闭窗口。
4.  从上面的链接下载 niivu 的主题。
5.  解压缩下载文件。
6.  在解压缩的文件夹中，打开与您的 Windows 版本匹配的文件夹。
7.  打开另一个 Windows 资源管理器窗口，进入**这台电脑>【驱动器】> Windows >资源>主题**复制下载的文件。
8.  一旦文件在正确的文件夹中，**右击桌面上的**。
9.  选择**“个性化”**
10.  在**“主题”**部分下，从列表中选择 Windows 11 主题。它应该会立即应用到您的桌面上。

至此，你的 Windows 10 桌面应该看起来*非常接近*默认的 Windows 11 主题，但你可以做一些可选的调整，让它更接近:

*   为 Windows 10T3 安装 [niivu 的 Windows 11 自定义图标(需要](https://www.deviantart.com/niivu/art/Windows-11-Icon-Themes-874289797) [7TSP](https://www.deviantart.com/devillnside/art/7TSP-GUI-2019-Edition-804769422) app)。
*   使用 [OpenStartMenu](http://www.chrisnsoft.com/download) 创建一个类似 Windows 11 的开始菜单快捷方式(链接和说明在 [niivu 的 Windows 11 for Windows 10 主题页面](https://www.deviantart.com/niivu/art/Windows-11-for-Windows-10-882819383) 的顶部)。
*   niivu 还建议安装 [xoblite](http://xoblite.net/docs/) 来增强右键菜单功能，但这不是必需的。

同样，这些是可选的 mod，但它们将使 Windows 10 的外观和行为更像 Windows 11。

## 第三步:升级你的 Windows 10 安全选项以匹配 Windows 11 的(某种程度上)

Windows 11 的严格硬件要求的结果是，它是一个更安全的操作系统——至少在理论上是如此。组合 TPM(可信平台模块)和 SecureBoot 要求应该有助于保护 Windows 11 设备免受恶意软件、勒索软件和远程接管的影响。

现在，一些 Windows 10 PCs 已经有了 TPM 并支持 SecureBoot。当然，这意味着它可能可以运行 Windows 11，但仍然值得检查一下，看看你的电脑是否可以使用这些功能。

启用您的 TPM 或 SecureBoot 将在硬件层面为您的电脑增加额外的安全性，但 Windows 10 通常会默认禁用它们。我们已经介绍了如何 [检查并启用您电脑的 TPM](https://lifehacker.com/what-is-a-tpm-and-why-do-you-need-one-to-run-windows-11-1847186073) (如果您有)，所以让我们来谈谈 secure boot——此外，您的电脑更有可能支持该功能。

### 如何启用 SecureBoot

如果代码未经正确验证，SecureBoot 会阻止应用程序安装，并会阻止进程在启动期间接管你的设备，因此如果你可以使用它，它绝对是一个值得的功能。

以下是检查方法:

1.  在任务栏中搜索“系统信息”，然后从搜索结果中选择它。
2.  在系统信息窗口中，滚动并检查 BIOS 模式是否设置为**“UEFI”**以及安全启动是否设置为“开”
3.  如果“安全引导”设定为“关闭”，则它会被停用。如果它说“不支持”，那么，它不可用于您的电脑，*或*您的 BIOS 模式设置为**“BIOS”**

无论哪种情况，都可以从启动 BIOS 菜单中启用 UEFI 和 SecureBoot。这样做将因你的主板及其固件而异，但是快速的谷歌搜索应该会给你指出正确的方向。

然而，一个警告的话:只有在你完全确定你知道你在做什么的情况下，才改变你的电脑的 BIOS 设置。即使你这样做，它总是有可能改变到 UEFI 或启用 SecureBoot 可能会导致重大问题。当然，如果成功了，你就不会遇到任何问题。但如果没有，你就需要采取一些激烈的措施来重置你的主板并重新访问你的电脑。这包括打开你的电脑和暂时移除主板的电池。如果你搞砸了，你最终会得到一台无法使用的电脑。相信我。在研究这本指南时，我经历了艰难的过程。

### 您应该打开的其他安全功能

启用 SecureBoot 和你的 PC 的 TPM(如果你有)将使你最接近 Windows 11 的基本安全功能，但你可以做更多的事情来保护你的 Windows 10 PC 和其上的所有应用程序和文件。这里有一个快速清单:

*   为您的 Microsoft 帐户启用 [无密码登录和二元身份验证](https://lifehacker.com/how-to-log-in-to-windows-without-a-password-1847709647) 。
*   报名参加 [OneDrive 勒索病毒防范](https://lifehacker.com/protect-yourself-from-ransomware-with-microsofts-new-on-1825019644) 。
*   使用 [加密密码管理器](https://lifehacker.com/how-to-get-started-with-a-password-manager-1846890484) 保护您的登录信息。
*   还有 [几个安全增强](https://www.csoonline.com/article/3631387/5-steps-to-improve-security-on-network-pcs-that-cant-run-windows-11.html?utm_content=content&utm_medium=social&utm_source=twitter&utm_campaign=organic) 企业用户和 IT 专业人员如果在网络上使用 Windows 10 就要实施。

正如我已经说过几次的那样，你不可能让 Windows 10 以每一种方式模仿 Windows 11。但是如果你启用了这些额外的安全层，并花时间应用自定义主题和 UI 模块，你会很容易欺骗自己以为这是真的。

对于我们这些还不能升级的人来说，假冒 Windows 11 是一个不错的选择——但不要觉得被排除在外。在 10 月 5 日发布之后的几个月甚至几年里，Windows 11 肯定会经历大量的成长烦恼。希望到用户可以升级的时候，这将是一个更有吸引力的选择。与此同时，微软计划在 2025 年 10 月之前支持 Windows 10，因此它将是几年内的一个可行选择。
