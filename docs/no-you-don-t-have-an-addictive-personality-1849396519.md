# 不，你没有“成瘾人格”

人们很自然地想把我们所有不同的性格特征——好的或坏的————归为一类，或者称之为“性格类型”我对此感到内疚。作为一个总是暴饮暴食的人，不管是吃、喝还是看，我经常形容自己有所谓的“成瘾人格”。(要是这意味着其他人对我上瘾就好了。唉。)

然而，术语“成瘾人格”属于被彻底误用的短语【强迫症】【创伤】的行列。但是与其他被滥用的术语不同，成瘾人格的定义根本不是基于临床心理学的。这里解释了为什么相信“成瘾人格”是一个神话，以及使用它对真正上瘾的人是如何有害的。

## **人们所说的“成瘾人格”是什么意思？**

“成瘾人格”一词被用来描述表现出某些特征的人更容易上瘾，例如滥用药物。具有所谓成瘾人格的人可能会冲动、强迫、缺乏控制、寻求快乐或软弱。把所有这些放在一起，你就有了一个无法自拔的人。据称。

事实上，上瘾是复杂的——至少比人格测验更复杂。家族史、教养、社会经济地位和生活方式都会影响一个人上瘾的风险。

任何人都可能对毒品或赌博等各种物质和行为上瘾。自然地，上瘾的人有共同的特征。然而，试图给一种可靠的“成瘾人格类型”分类，往好里说是不科学的，往坏里说是有害的。

## **成瘾人格的神话**

正如物质滥用学者玛丽安·阿莫多奥 在他的 [2015年的论文中解释的那样，没有可概括的研究来支持“成瘾人格”的观点尽管这个术语在今天的治疗项目和支持团体中频繁出现，但是没有足够的证据表明一种性格类型更容易上瘾。此外，阿莫多奥写道,用来描述成瘾性格的特征并不能预测成瘾；他们*源于*瘾。当谈到预防时，到处说这句话是没有帮助的，更糟糕的是，它可能会对真正的瘾君子造成伤害。](https://www.tandfonline.com/doi/full/10.3109/10826084.2015.1007646)

## **为什么我们应该停止使用术语**

使用成瘾人格这样一个不准确的术语只会导致真正成瘾的病理化，增加耻辱感，以及一种不可避免的感觉。在她关于为什么这个术语应该从治疗领域退出的文章中， [阿莫多奥](https://www.tandfonline.com/doi/full/10.3109/10826084.2015.1007646) 声称，成瘾人格描述伤害了某人对抗成瘾的能力，因为它创造了一种自我实现的预言，即他们不能改变自己的方式。

这个词也使人们认为每个上瘾的人都是一样的，这导致了挣扎中的上瘾者被边缘化。将这个词从我们的词汇中删除有助于宣传这样一个理念:面对药物滥用，没有人是注定要失败的，他们的毒瘾是值得斗争的。

## **底线**

没有一种单一的人格类型可以准确预测一个人可能会如何上瘾并对抗上瘾。尽管如此，对于像上瘾这样痛苦的事情，有一种自然的冲动去尝试和理解它。不幸的是，上瘾是一种复杂的动物。将某一组特质归类为“成瘾人格”弊大于利。好的一面是，没有一部分人注定会上瘾。
