# 列出你已经做对的事情来激励自己

> 原文：<https://lifehacker.com/motivate-yourself-by-listing-the-stuff-youre-already-do-1847907736>

当我们专注于改善我们的健康时，我们经常会在脑海中列出我们没有做的事情:我应该 T2 T3 锻炼 T4 T5，我应该多喝水 T8 T9，我应该 T10 T11 找个时间吃点蔬菜 T12 T13。但是你已经养成的健康习惯呢？大概不止零个。

Watch

这不是一些愚蠢的感觉良好的策略，就像在你的待办事项列表的顶部写下“做一个待办事项列表”然后划掉它。(虽然，老实说？当你开始一项令人生畏的工作时，让自己笑并没有错。我完全赞成愚蠢的感觉良好的东西。)有一个广为接受的心理学理论支持它。我们在建立自我效能。

[自我效能](https://www.simplypsychology.org/self-efficacy.html) 是你相信自己能做好这件事。(不管那东西是什么。)想想当你做你擅长的事情并且你已经做了无数次的时候你是什么感觉:你很自信，你可能只是想想就心情不错，而且你知道你会搞定它。

如果你是一名跑步者，当你在周二早上跑步时，你可能会有这种感觉。但是如果你不把自己当成跑者，一切都是新的，不舒服，不是吗？你不知道你是否能完成一次跑步，更不知道你希望开始的全新习惯。

不过，好消息是:你可以通过多种方式建立自我效能。其中一种方法是回顾你过去做得好的事情。即使你还没有养成跑步习惯，你也可以看看你*已经*养成的习惯。也许你每天都遛狗。也许你已经打包了健康的午餐。也许你至少有时会设法按时上床睡觉。这些都可以列在你的清单上。

看一下这份清单，你会发现你*已经*取得了胜利，即使是小的胜利。为他们给自己加分！然后问问你自己，你如何为自己赢得更多的胜利，并开始获得动力。
