# 使用乔氏甜香番茄酱的三种方法

> 原文:[https://life hacker . com/three-way-to-use-trader-joes-sweet-savory-tomato-SPR-1846432715](https://lifehacker.com/three-ways-to-use-trader-joes-sweet-savory-tomato-spr-1846432715)

如果我的购物车可以算作一个案例研究的话，我想可以肯定地说，乔氏百货公司的大部分收入都来自冲动购物。“是的，我*需要一些新的厨房毛巾吗，”“我想知道这个放在爆米花上会是什么味道，”以及“我应该得到一个 [心情奥德赛](https://www.traderjoes.com/digin/post/mood-odyssey-essential-oil-roller-blends) ！”都是我在他们的过道里漫步时，我的小脑袋里所想的非常标准的例子。我通常会买至少三样我不打算买的东西，这就是为什么我最近买了一罐他们的甜&香喷喷的番茄酱。*

*我希望我能说我对此很高兴。这种酱——基本上是番茄酱——被宣传为“与奶酪和冷盘或烤肉和蔬菜搭配的美味佳肴”，但它一点也不美味。它是甜的，几乎是令人作呕的T2，没有明显的盐，也没有足够的酸。这是果酱，但不是一种好的方式，大量的糖掩盖了你可能希望在番茄制品中找到的任何亮度。*

*这并不意味着它完全没用——当搭配味道浓烈的奶酪或烧焦的超咸肉——时，它还不错，但当搭配大量酸、辛辣的芥末和/或乙醇时，它是最好的。这些成分抑制了糖浆般的甜味，让番茄呼吸并发光。以下是我最喜欢的三种使用TJ番茄酱的方法，尤其是如果你和我一样，不太喜欢它的话。*

### *调制鸡尾酒*

*我对 [果酱鸡尾酒](https://lifehacker.com/use-jam-to-make-any-booze-morning-appropriate-1832060017) 并不陌生，但我通常的2盎司酒、3汤匙果酱和1/2盎司柠檬汁的比例与这种特殊的酱料不太相符。我用这些尺寸调制的鸡尾酒——很像我自己——太浓太甜了。幸运的是，这很容易解决:我减少了果酱，增加了伏特加和柠檬，做出了一种新鲜、甜美、扑鼻的鸡尾酒，尝起来就像来自花园。要做到这一点，你需要:*

*   *两盎司半伏特加*
*   *1 1/2汤匙乔氏甜辣味番茄酱*
*   *3/4盎司新鲜柠檬汁*

*将所有材料加入空的鸡尾酒调酒器中，摇晃以溶解果酱。加入冰块，再次摇匀冷藏。通过筛子双重过滤到鸡尾酒杯。如果有的话，用樱桃番茄装饰。*

### *搅拌调味汁*

*尽管酱料并不像它声称的那样“香甜扑鼻”,但你可以用一堆苹果醋来解决这个问题。这种调味汁令人愉快，充满活力，绝对适合放在脆脆的胡萝卜沙拉上。要制作它，你需要:* 

*   *1/4杯橄榄油*
*   *1/4杯苹果醋*
*   *3汤匙乔氏甜辣番茄酱*
*   *1汤匙第戎芥末*

*将所有东西加入一个 [广口瓶](https://skillet.lifehacker.com/a-mason-jar-is-the-only-tool-you-need-for-perfectly-emu-1796819745) ，盖上盖子，摇动直到调料乳化。倒在一堆未加工的植物碎片上。*

### *给肉涂上釉料*

*将半杯果酱和四分之一杯芥末混合在一起，就成了一种非常棒的烤肉釉料——这是这种酱料的糖果味有益的一个例子。只需将两种配料混合在一起— ( 我在这里会用非常辛辣的芥末)并在烘烤前刷上一些鸡肉、猪肉或羊肉。我特别喜欢它上的 [鸡](https://skillet.lifehacker.com/glaze-roasted-meats-with-mustard-and-jam-1831618342) 。*
