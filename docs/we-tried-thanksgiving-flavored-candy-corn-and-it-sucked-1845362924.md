# 我们尝试了“感恩节风味”玉米糖，感觉糟透了

> 原文：<https://lifehacker.com/we-tried-thanksgiving-flavored-candy-corn-and-it-sucked-1845362924>

我通常不喜欢破坏这些视频的结局，但正是旅程——而不是目的地——让这个特别的味觉测试变得如此“有趣”(对你来说，不是对我)。我品尝- 测试已经有一段时间了，但以前我被认为是“全国领先的奥利奥记者”(至少我是这么看待自己的。)

评论奥利奥是一项愉快的工作。即使是难吃的，比如味道像瑞典鱼的那种，也有一些可取之处(通常是饼干)。品尝- 测试新的和不寻常的玉米糖？没有那么多。玉米糖从来都不是我最喜欢的糖果，尽管搭配蜂蜜烤花生会让它更加美味，甚至令人愉快，所以很高兴向乔尔(一个著名的玉米糖讨厌者)介绍这个 [像黄油手指一样的搭配](https://lifehacker.com/how-to-stop-hating-candy-corn-1838855471) 。在接下来的恐怖表演之前，这为朋友们提供了一个美好的时刻。

我想，对 Brach's 来说，拥有市场上最好的玉米糖还不够。出于我不清楚的原因，他们决定创造一包有六种新口味的玉米糖，每一种都代表感恩节大餐的一个不同方面。有火鸡、填料、青豆、红薯馅饼、姜汁胡萝卜和蔓越莓酱，总体效果是一场糟糕透顶的美国佬烛光集会。(开个玩笑。原因很清楚。因为像我这样的傻瓜会买它“只是为了看看”)

我不会告诉你我最喜欢的(如果它甚至可以被称为)，我也不会告诉你我最讨厌哪个。如果这是你想学的东西，你必须看视频。如果你喜欢看我受苦——谁不是呢？我想你会满意的。毕竟，我这么做是为了让你不必这么做，但我这么做也是为了吸引注意力，对我隐瞒这个是彻头彻尾的残忍。
