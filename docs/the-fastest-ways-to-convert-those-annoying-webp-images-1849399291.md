# 转换那些烦人的网络图片的最快方法

> 原文：<https://lifehacker.com/the-fastest-ways-to-convert-those-annoying-webp-images-1849399291>

WebP 是谷歌版的现代通用图像格式，旨在取代 PNG 和 JPEG 之类的格式。然而，在过去的几年里,所能做的就是,成为一个轻微的烦恼。当网站使用它来显示图像时，这很好——这种格式可以很好地处理压缩——但对于想要下载和使用图像的人来说，这是一个麻烦。如果你曾经下载过一个图片，结果只会让你抓耳挠腮。webp 扩展到最后，你就知道痛苦了。 T 不过这里有一个答案，而且奇怪的是你最好的选择是使用 Chrome 扩展。



## 以 png 格式下载 WebP 图像

最好在源头解决问题，而 [将图像另存为 PNG](https://chrome.google.com/webstore/detail/save-image-as-png/nkokmeaibnajheohncaamjggkanfbphi?hl=en) Chrome 扩展正是这么做的。安装完成后，在 Chrome 或基于 Chrome 的浏览器中右键单击任何 WebP 图像，然后选择“**将图像另存为 PNG** ”选项。扩展将图像转换并保存为 PNG 。这将产生一个大得多的图像文件，但是对于一种你可以在任何地方使用的文件格式来说，这可能是值得的。

## 在 Windows 上转换 WebP 图像

如果你用的是 Windows PC，你就不需要下载第三方应用来转换 WebP 图片。值得信赖的画图应用程序会帮你搞定。在画图 app 中打开 WebP 文件，然后前往**文件** > **另存为**。在这里，您可以根据需要选择“ **PNG 图片**或“ **JPEG 图片**”选项。然后选择目标文件夹就大功告成了。

## 在 Mac 上转换 WebP 图像

同样，内置的预览应用程序将帮助您打开和转换 WebP 图像。在预览中打开图像，进入**文件** > **导出**。在导出菜单中，点击**格式**选项旁边的下拉菜单，选择“ **JPEG** 或“ **PNG** ”选择目标文件夹，点击**保存**转换图像。

如果您使用的是最新版本的 macOS，您可以使用 Finder 中的快速操作功能来快速转换一个或多个 WebP 图像。进入 Finder 选择图片，r 右键点击，进入**快速动作** > **转换图片**。选择格式和大小，然后点击“**转换为 JPEG** ”新文件将保存在同一目录下。

## 在线转换 WebP 图像

如果你想在浏览器中转换 WebP 文件，你会被选项宠坏。你可以使用像 [云转换](https://cloudconvert.com/webp-to-jpg) 这样的网站，它可以让你一次将五张 WebP 图片转换成 JPEG。
