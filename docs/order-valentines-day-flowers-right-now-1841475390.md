# 现在就订购情人节鲜花

> 原文:[https://life hacker . com/order-valentines-day-flowers-right-now-1841475390](https://lifehacker.com/order-valentines-day-flowers-right-now-1841475390)

想在情人节给某人买花吗？你最好快点，因为越接近2月14日，价格只会越贵。

Watch

根据交易搜索网站 [Brad's Deals](https://www.bradsdeals.com/blog/why-are-roses-so-expensive-on-valentines-day) 的一项研究，在盛夏时节，当你从五大送货品牌之一订购玫瑰时，它们的平均价格为32美元，你会得到最低的价格(当然是按打计算)。相比之下，峰值平均价格——情人节前一周49.98美元。根据供需情况，那辆的[可以秒杀到近100美元](https://moneywise.com/a/the-cost-of-love-valentines-day-prices-in-2019)的 ( *之前的*送货费！)在当天作为最后一分钟的订单购买。

虽然这看起来像是一个纯粹为了收费而收费过高的经典案例，但这是一个满足需求的问题。毕竟情人节只有24小时。丽贝卡·莱曼在Brad's Deals博客上写道:“卖花的人不可能打电话给工厂，告诉他们提高产量。”“鲜花需要采摘和运输，通常来自遥远(温暖)的地方，只需在特定的一天内及时送到顾客手中。”

### **趁你还能点的时候点吧**

坏消息是:现在买一打红玫瑰已经太晚了，因为报告称，在1月15日左右订购可以帮你获得当季最低价格。然而，你越早下单，就越不可能支付最后一刻购物者所能承受的最高价格。

全国零售联盟 [预计](https://nrf.com/media-center/press-releases/confident-consumers-and-broader-buying-lead-record-valentines-day) 和另一半一起庆祝情人节的人每年平均消费101.21美元。不要让送花费用超过绝对必要的预算。

### **如何降低成本**

如果你现在订购，许多零售商会提供优惠券代码或其他折扣 。另一个降低成本的策略是，情人节不要玫瑰，换一种不那么“T4”的花。此外，你可以通过在情人节前一两天安排递送这些玫瑰来省钱，因为许多服务在那个特别繁忙的日子会提高递送费用。

这个故事最初发表于2020年，并于2021年2月5日更新了新的信息和更新的上下文。
