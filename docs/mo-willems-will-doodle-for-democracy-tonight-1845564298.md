# 莫·威廉今晚将为民主涂鸦

> 原文：<https://lifehacker.com/mo-willems-will-doodle-for-democracy-tonight-1845564298>

如果说 2020 年教会了我们什么的话，那就是当事情变得艰难时，重要的是找到任何和所有形式的分心 。幸运的是，童书作者兼插画师 [莫·威勒斯](http://www.mowillems.com/) 以前曾与我们一起走过这段压力之旅，提醒我们当事情变得糟糕时，人们可以随时拿起记号笔开始涂鸦。今晚，随着选举结果(也许？)他将主持一场名为“ [民主涂鸦 2020](https://www.kennedy-center.org/mowillems) ”的现场活动



“这是一场非常重要的选举，其他屏幕上的许多人将会说非常重要的事情，”Willems 在一个关于 事件的宣传视频 [中告诉我们。"*我们*要去涂鸦了."](https://www.youtube.com/watch?v=5bVYAYO1pi8&feature=emb_logo)

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-5bVYAYO1pi8&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-5bVYAYO1pi8&start=0) 

作为肯尼迪中心的教育艺术家，Willems 在春天主持了一个类似的系列，名为 [与 Mo Willems](https://offspring.lifehacker.com/doodle-with-childrens-book-author-and-illustrator-mo-wi-1842376859) 共进午餐涂鸦，在这个系列中，他向孩子们(和成年人)展示如何画出他最喜爱的一些角色，包括 [【鸽子](https://twitter.com/The_Pigeon) 。

如果你觉得今晚需要分散一点注意力，拿些纸和记号笔，在美国东部时间晚上 7 点前往肯尼迪中心的网站 ，以民主的名义涂鸦。
