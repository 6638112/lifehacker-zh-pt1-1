# 如何志愿参加科学考察

> 原文：<https://lifehacker.com/how-to-volunteer-for-a-scientific-expedition-1847618946>

经历了 2020 年，以及我们在 2021 年幸存下来的东西，促使许多人反思他们的生活、目标和优先事项。如果你是他们中的一员，并意识到你想(最终)看到更多的世界并开始志愿服务，有一种方法可以同时做到这两者:志愿参加科学研究考察。



但是，除非你碰巧和一位进行实地研究的科学家是朋友，并且碰巧正在寻找额外的帮手，否则你可能不知道从哪里开始。克里斯汀·波普 最近在《国家地理》上发表的一篇文章讲述了一些人的经历，这些人自愿为各个目的地的实地研究项目收集数据和/或标本。它还揭示了如何找到一个寻找志愿公民科学家的项目。以下是需要了解的内容。

## 如何找到一个寻找志愿者的野外研究项目

你需要做的第一件事是确定一个你感兴趣的研究项目，并且需要志愿者。最简单的方法是通过一个非营利组织，在需要志愿者的地方为他们匹配研究项目。

《国家地理》文章中提到的一个叫做，它与 [国家公园管理局](https://www.nps.gov/index.htm)[美国林务局](https://www.fs.usda.gov/) 和 [土地管理局](https://www.blm.gov/) 等机构合作，为各种研究项目提供公民科学家。

寻找志愿者机会的其他地方包括:

*   [志愿赛](https://www.volunteermatch.org/)
*   [地球观察](https://earthwatch.org/)
*   [蓝图地球](https://blueprintearth.org/fieldvolunteers)
*   史密森环境研究中心
*   [史密森公民科学](https://www.si.edu/volunteer/citizenscience)

## 志愿参加研究考察需要了解什么

还有一些其他的事情要记住。首先，不要忘记这是志愿者工作——所以不要指望你的劳动会得到报酬，也不要指望你的差旅费会被报销。也许你会幸运地找到一个提供某种津贴的项目，但这是罕见的例外，而不是规律。

然而，你更有可能找到的是营利性的志愿旅游公司，他们会收取一系列费用，为你提供海外志愿服务的机会。这些不一定是研究旅行(或你想做的事情)，但它们可能会在你搜索的过程中出现。

尽管这个机会是无偿的，但在大多数情况下，你仍然需要通过 [申请和/或](https://www.adventurescientists.org/our-volunteers-2.html) 筛选过程，以确保你的时间和技能与项目相匹配。然后，如果被选中，你将接受某种类型的培训，学习如何收集和提交特定项目所需的数据。

这不会是你典型的旅行经历，但会提供一个以不寻常的方式探索一个新地方的机会。
