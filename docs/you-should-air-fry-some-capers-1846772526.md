# 你应该油炸一些刺山柑

> 原文：<https://lifehacker.com/you-should-air-fry-some-capers-1846772526>

大约两周前，我的空气油炸锅停止工作。幸运的是，即时锅能够寄给我一个新的(谢谢，保修！)，但没有我的 Vortex Mini 的半个月让我意识到我是多么依赖它。

重新加热外卖、烤面包和烹饪完美的小“额外服务”:这些只是我被迫在我的大而低效的“大烤箱”中做的几件事，当我的新桌面对流烤箱到来时，我大大松了一口气。我不仅可以继续享受我的 [脆皮豆腐](https://skillet.lifehacker.com/im-obsessed-with-air-fried-tofu-1846146173) 配以 [油炸葱](https://skillet.lifehacker.com/air-fry-your-scallions-for-the-perfect-garnish-1846184934) 的碗，我终于可以测试我的计划，做一些松脆的小咸肉——也就是油炸马槟榔。

很像 [空气油炸鹰嘴豆](https://skillet.lifehacker.com/toss-a-can-of-chickpeas-in-your-air-fryer-1846527987) ，刺山柑需要一点点油，但远没有你真的需要的那么多， [真正油炸它们](https://skillet.lifehacker.com/capers-fried-in-duck-fat-are-your-new-favorite-topping-1831610350) 。(这可能是我第二次以“减少脂肪”的方式使用空气油炸锅我真的不在乎计算卡路里，但我喜欢省钱，使用更少的配料就能做到这一点。)

油炸马槟榔(无论是真的油炸，还是在涡轮对流烤箱中“假的”油炸)，都是一种松脆、咸辣的快乐。你可以把它们当零食吃——和马丁尼酒一起吃很好——或者你可以把它们撒在沙拉、意大利面或者美味的鱼肉上。我通常会像吃小爆米花一样吃掉它们。

要做油炸山柑，你需要:

*   一些腌马槟榔
*   某种脂肪(橄榄油、熏肉脂肪和鸭肉脂肪)

将刺山柑从盐水中取出，用纸巾擦干。将它们转移到一个碗里，淋上一点油，搅拌直到它们有光泽，但不要浮在脂肪中。如果你有一个篮式空气油炸锅，移开托盘(刺山柑会从孔里掉出来)，把涂油的刺山柑直接放在篮子的底部。在 400℉下油炸 5-7 分钟，直到它们起泡和变脆。短暂冷却后享用。
