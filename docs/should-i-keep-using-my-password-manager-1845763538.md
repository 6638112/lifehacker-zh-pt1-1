# 我应该继续使用我的密码管理器吗？

> 原文：<https://lifehacker.com/should-i-keep-using-my-password-manager-1845763538>

你应该使用*一个*密码管理器，因为这是保护你的数字生活的最好方法之一。有了它，你将能够创建强有力的、难以置信地难以猜到的密码，甚至连你自己都记不住。Y 我们的密码管理器将为您存储这些密码，您将能够使用更易记的密码(或密码短语)和多因素身份验证的组合来验证密码库。



但是你应该使用哪个密码管理器呢？这才是真正的问题。当您选择的密码管理器没有出现在每个人的收藏夹列表中时，这对您的安全意味着什么？Lifehacker 读者 **Jan** 问 me 这个问题在本周的[**Tech 911**](https://lifehacker.com/c/tech-911):

> 我已经订阅 Roboform 很多年了。也许，几十年。我发现它并非没有挑战，而且我注意到它从未出现在“Top X Password Vault”文章中。这是因为新的机器人要好得多，还是因为机器人广告做得不够多？还有别的吗？

### 最好的密码管理器是你经常使用的那个

我承认，简；我从未听说过 Roboform。或者，至少，当我读到你的问题时，我没有立刻想起什么。虽然它的名字出现在网络上各种“最佳密码管理器”的名单上，但我从未见过拿过最高荣誉。我们倾向于生活在一个 [LastPass](https://www.lastpass.com/) 或 [1Password](https://1password.com/) 类型的世界 —reall y，1Password 和 [Bitwarden](https://bitwarden.com/) ，因为后者是一个开源的密码管理器，比 LastPass (每年节省 26 美元)和几乎一样便宜。

回答你的问题，如果 Roboform 不是最好的密码管理器，那也没关系。原因是:只要它做得足够好，并且没有遭受任何可怕的安全失误，它就能保证你的安全。您已经习惯了它的工作方式，您可能已经很好地掌握了它的界面，并且您经常使用它。这使得本身成为机器人最好的密码管理器(给你)。

你的情况让我想起了“升级者的谬误”，或者每当我看到人们蜂拥而至参加 la 测试并为他们的台式电脑购买最棒的设备时，我喜欢思考的问题。举例来说，我喜欢一个品牌-n ew Nvidia R TX 3080 卡，当它出来的时候，我几乎跳进狂野的竞赛中去买一个。我说服自己不要用它，，因为即使它比我现有的更好，那也不会使我现有的设置变坏(RTX2080)。事实上，我已经拥有的对于我的超宽显示器和我玩的游戏(目前:*魔兽世界*)的细微差别来说很棒。我不需要更多的火力，直到有东西引起我的兴趣，那是一个疯狂的好图形和光线跟踪的组合。而且，即使那样，我可能仍然能够用我现有的设置获得相当不错的帧速率。为什么要升级？

这里的“T1”是“T2”比“机器人”更好的“T3”密码管理器，“T4”提供了“机器人”所缺乏的有用功能。例如，mighty 1Password 有一个非常方便的“瞭望塔”功能，当您使用的密码出现数据泄露时，它会提醒您。正如 [Wirecut ter](https://www.nytimes.com/wirecutter/reviews/best-password-managers/) 在其密码管理器综述中指出的，Roboform 还没有通过 [第三方安全审计](https://support.1password.com/security-assessments/) 。

那是拥抱交易吗？如果你是从零开始寻找你的第一个 t 密码管理器，当然可以。如果你已经对 Roboform 很满意，并且每天都用它来管理你的密码，那么你肯定*可以换一个更好的。如果有一种简单的方法可以从你目前使用的服务中获取你将来想尝试的服务的密码，我建议你这么做。如果这会对你的日常工作造成巨大的干扰，那就不值得了——即使你最终一年节省了 20 美元或者获得了一个体面的密码管理器提供的一般安全性之外的一些额外功能。*

我意识到一些读者可能认为的听起来有点奇怪，出自一个技术建议专栏作家之口，他的正常反应是“升级，升级，升级”。T4:烬，并不是每个人都是超级科技通。也许，仅仅是也许，让一个人从开始使用密码管理器是一场足够艰巨的战斗。(我仍在努力说服我那些通情达理又聪明的朋友，他们应该使用一个很棒的密码管理器，而不是他们浏览器更简单的“保存密码”功能。)

不到万不得已，不要捣乱。如果你真的想升级，但要知道如果你正确使用密码管理器，你不会错过很多东西:为你的所有服务提供复杂、独特的密码。如果你想变得超级安全，你可以设置一个谷歌提醒来提醒所有提到机器人的人。这样，万一你选择的密码管理器*发生了什么事情，你就可以切换到别的(更好的)密码管理器。但是，如果你对你所拥有的感到满意，不管它值多少钱，你都很好。*
