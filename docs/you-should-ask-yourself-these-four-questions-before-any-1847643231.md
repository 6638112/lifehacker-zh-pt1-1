# 在购买任何大宗商品之前，你应该问自己这四个问题

> 原文：<https://lifehacker.com/you-should-ask-yourself-these-four-questions-before-any-1847643231>

“去购物”过去指穿上外面的衣服，离开房子，在零售店中寻找你需要的物品。但是现在，购物不再是一件大事，而是你无聊时可以在手机上做的事情。虽然网上购物肯定比逛不同的商店更方便，但这种方便也有不好的一面——看似无限的选择让人们不知所措，包括他们以前可能没有考虑过的功能和产品。



幸运的是，消费者报告最近以 [成为更聪明的购物者指南](https://www.consumerreports.org/shopping/how-to-be-a-smarter-shopper-right-now/) 的形式向提供帮助 。以下是需要了解的内容。

## 购买大件商品前要问的问题

根据消费者报告，在进入虚拟通道之前，购物者应该自己回答这四个问题:

### 1.我想要什么？

有时你已经确切地知道你在找什么，包括品牌、颜色和价格。其他时候，您可能甚至不知道什么功能最适合您的需求。如果你发现自己属于第二类，《消费者报告》交易编辑萨曼莎·戈登建议搜索某类产品的“最佳”品牌/型号。即使你最终没有买清单上的东西，它至少会让你知道现在有什么，以及当你继续购物时你可能想要寻找什么。

### 2.这个产品符合我的需求吗？

如果你已经把范围缩小到了几个产品，那就再深入了解一下公司/商店的退货政策(这样你就能知道在试用之后你是否还会继续使用这个产品)。还可以看看零售商和公司网站上对该产品的评论，以及浏览他们的社交媒体页面，看看是否有不满的消费者留下了投诉或其他信息。(只要记住，不是所有你在网上看到的评论都是合法的 。)

### 3.这要花多少钱？

满足您的需求的一部分是在您的预算范围内。是的，你应该比较不同网站和零售商的价格，但也要对你能负担的东西持现实态度。仅仅因为某样东西带有各种华而不实的功能，并不意味着你需要所有(或任何)这些功能。

### 4.它有多大？

这并不总是一个问题，但是如果你要购买一件适合特定空间的物品，一定要测量尺寸，然后对照产品清单进行核对。照片可能具有欺骗性，所以拿出卷尺不要盯着看。

此外，找出该项目将如何交付。例如，如果它是一个沙发，它是完全组装好的，还是一片片的？如果它是完整的交付，它能通过你的门口吗？这些都是你想在卡车停在你家门口之前弄清楚的事情。
