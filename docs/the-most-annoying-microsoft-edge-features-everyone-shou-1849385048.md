# 每个人都应该禁用的最烦人的微软 Edge 功能

> 原文：<https://lifehacker.com/the-most-annoying-microsoft-edge-features-everyone-shou-1849385048>

现在，微软 Edge 基于 Chromium 引擎(为谷歌 Chrome 提供动力的同一引擎)，它实际上是一个很好的浏览器。你可以把它想象成 Chrome，但更有利于隐私和电池寿命。但它仍然是微软的产品，它有吸收一些臃肿和恼人的功能的亲和力。值得庆幸的是，在设置页面上花几分钟的时间就能解决所有问题。如果您使用 Microsoft Edge 作为 Mac 或 Windows PC 上的默认浏览器，请考虑禁用这些功能以使浏览更加容易。



## 去掉每张图片上的搜索图标

谷歌 Chrome [增加了一个奇妙的功能](https://lifehacker.com/how-to-search-translate-or-ocr-any-image-in-chrome-on-1849006247) ，可以让你使用谷歌镜头(也有 OCR)搜索相似的图片。微软 Edge 也有同样的功能，但是用的是必应而不是谷歌图片。更糟糕的是，当你悬停在任何图像上时，Edge 都会显示一个图像搜索图标(在右上角)。光这一点就够烦人的了。不过，您可以禁用该功能。进入**设置** > **外观** > **视觉搜索**，禁用**图像悬停时显示视觉搜索**如果您愿意，您仍然可以使用右键菜单中的功能。

## 摆脱必应搜索

如果你刚刚开始使用微软 Edge，你可能已经注意到默认的搜索引擎是 Bing。如果那不是你的果酱，去**设置** > **隐私，搜索和服务** > **地址栏，搜索** > **地址栏中使用的搜索引擎**。在这里，你可以切换到“ **Google** 或者“ **DuckDuckGo** ”

## 禁用所有恼人的优惠券弹出窗口

优惠券插件很棒，但微软的内置购物工具会在你每次访问任何店面时弹出。如果您想禁用它们，请进入**设置** > **隐私、搜索和服务**并禁用“**在微软 Edge** 购物节省时间和金钱”功能。

## 删除起始页上的所有垃圾

一旦清除了所有垃圾，Microsoft Edge 中的起始页就非常棒。你可以通过到起始页点击的**设置**图标来开始。在这里，你可以切换到“**聚焦**模式，只保留搜索栏。您还可以 g o 到“**自定义**”部分，保留“**快速链接**”部分，并启用“**当日背景**”功能。确保禁用“**内容**”功能，以消除微软新闻面板。

## 去掉弹出的下载面板

浏览器底部下载栏的时代已经过去，但 Edge 仍然喜欢在每次开始新的下载时弹出下载抽屉来提醒我们。如果你可以找到顶部的下载按钮(直观地显示进度)，禁用弹出- 弹出功能。进入**设置** > **下载**，禁用“**下载开始时显示下载菜单**”功能。

## 阻止 Office 文件在浏览器中打开

因为 Edge 是微软做的，它坚持首先在浏览器中打开微软 Office 文档或文件，而不是专用的应用程序。您可以通过进入**设置** > **下载**来逆转这一点。在这里，禁用“**在浏览器中打开 Office 文件**功能。
