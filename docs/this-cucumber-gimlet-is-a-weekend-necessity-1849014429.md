# 这个黄瓜吉姆特是周末的必需品

> 原文：<https://lifehacker.com/this-cucumber-gimlet-is-a-weekend-necessity-1849014429>

如果需要是发明之母，那么资本主义似乎是过剩之父。我认为，这个世界需要多少护肤产品、播客或手工杜松子酒是主观的、有争议的，但我可以坦然地说，现代世界中发明的主要动力是捏造问题(以销售解决方案)、提供更便宜(或更贵)的替代物来替代现有的东西，或者创造你自己版本的东西，因为为什么不呢？



当你可以装瓶出售的时候，为什么要剥夺全世界你的烤肉酱配方呢？为什么不在 Etsy 上卖钩编的“生活、欢笑、爱”杯垫呢？没有小众太小众不小众！在母亲发明和父亲资本主义的维恩图的中心是创造力，创造力不能受制于道德说教——它是一种混乱的、中性的力量，是存在的基础，能够产生恐惧、敬畏以及介于两者之间的一切，都是同等程度的。因此，在这个思路上，我想和大家谈谈黄瓜脆片。希望能说服你做一个。

你们大多数人可能对杜松子酒很熟悉:杜松子酒和酸橙汁。多少杜松子酒，多少酸橙汁，什么样的杜松子酒，什么样的酸橙汁——这些都是你自己演绎的变量，但最著名的，也是历史上最著名的是，gimlet 指定等量的玫瑰酸橙汁和海军劲杜松子酒(现代营销术语，表示 57%以上的酒精含量)。在某种程度上，海军劲杜松子酒让位于普通杜松子酒，杜松子酒被提高了，新鲜酸橙汁和糖被认为是可接受的(如果不是更好的)罗斯的替代品。

但是在这种美味的结合中，还有比饮料本身更令人耳目一新的东西:一个诞生发明的美丽例子。

你是 18 世纪的一名水手，坏血病(这个问题)是一个真正的麻烦，你需要能够治愈(必要性)。幸运的是，你知道维生素 C 会起作用，但不幸的是，柑橘不会保持你需要的时间(问题)所以你需要找到保存它的方法(必要性)。最终你会发现酸橙是最稳定的，当与一点点朗姆酒混合时会保持更长时间。不幸的是，没有足够的朗姆酒使酸橙汁药物对你的船友可口(问题/必要性，你现在明白了)。看在上帝的份上，他们是水手，坏血病该死——所以你把它和超标的杜松子酒混合，每个人都很高兴，或者至少没有坏血病。最终，一位来自利斯的船商发现用糖保存酸橙汁会更好，他为他的浓缩汁申请了专利，叫做“玫瑰酸橙汁”。就这样，一个吉姆特诞生了。

昨晚我自己做的黄瓜 gimlet 并不像 OG 那样复杂或救命，但它确实帮助我解决了用完冰箱里最后一点黄瓜的问题，以及我喝一杯的必要性。这正是我喜欢的那种鸡尾酒——简单的经典，通过添加一种容易获得的成分而改变。虽然这很棒，但我不想把我自己的装瓶出售，所以我鼓励你给自己在家做一个的快乐

## 黄瓜脆片

*   4 片黄瓜薄片
*   2 盎司杜松子酒
*   1 盎司新鲜酸橙汁
*   盎司简单

将黄瓜片放入罐中搅拌(可选，如果切片足够薄)。将剩余的原料加入到你的摇壶中，加入冰块，用力摇晃 12-16 秒。滤入冰镇的 coupe，用黄瓜片装饰(也是可选的，但非常漂亮)。
