# 您已经需要再次更新Safari

> 原文:[https://life hacker . com/you-already-need-to-update-safari-again-1848522230](https://lifehacker.com/you-already-need-to-update-safari-again-1848522230)

上个月，我们警告过你一个影响苹果Safari浏览器 的 [重大安全漏洞。这个缺陷潜在地允许一个网站上的黑客从其他开放的网站上窃取你的个人信息，你知道，这是不好的。经过几个月的拖延，苹果终于修补了这个安全漏洞，正好赶上另一个漏洞的出现。](https://lifehacker.com/update-safari-immediately-1848426551)

根据苹果 [最新安全笔记](https://support.apple.com/en-us/HT213093) 显示，该公司发现了WebKit的一个安全漏洞，WebKit是iPhone、iPad和Mac上Safari的基础，也是iOS和iPadOS上所有第三方浏览器的基础。这个WebKit bug为不良行为者在不知情的苹果用户的路径上放置恶意代码打开了大门。如果您在您的设备上处理这些恶意代码，黑客可以利用任意代码执行漏洞，这将允许他们在您的系统上运行他们想要的任何代码。

Safari的安全漏洞是一个主要问题，尤其是在移动设备上。当然，在Mac上，你可以暂时切换到另一个浏览器，比如Chrome或Firefox。然而在iOS或iPadOS上，*所有的*浏览器其实都是伪装的Safari，依靠同一个WebKit基础运行。这意味着当WebKit有一个安全缺陷时，所有移动浏览器都有这个缺陷。

苹果表示，有报告称这一安全漏洞可能已经被积极利用，这意味着所有受影响的系统必须尽快更新。如果您有iPhone、iPad或在Mac上使用Safari，您的设备会计入该列表。幸运的是，更新就在这里，以iOS 15.3.1、iPadOS 15.3.1和macOS 12.2.1的形式出现。

## 如何更新和修补这个Safari安全漏洞

要修复这个Safari安全漏洞，你需要做的就是像往常一样更新你的iPhone、iPad或Mac。要在iOS或iPadOS上更新，请前往**设置>常规>软件更新**，然后按照屏幕上的说明下载并安装15.3.1。在Mac上，进入**系统偏好>软件更新**，然后按照屏幕上的说明下载并安装macOS 12.2.1更新。

这里有一个警告:如果你运行的是iOS 15.4、iPadOS 15.4或macOS 12.3的测试版，那就万事俱备了。这些测试版似乎已经修补了安全漏洞，随后，测试版测试人员没有在他们的设备上收到新的安全更新。如果你有兴趣在你的设备上下载并安装这些测试版， [点击](https://lifehacker.com/how-to-install-apples-new-public-betas-for-ios-macos-1848441069) 查看我们的指南。

这些最新的更新不仅让Safari更加安全；在iOS和iPadOS上，15.3.1修补了一个影响盲文显示器的错误。另一方面，macOS 12.2.1修复了一个负责快速彻底耗尽MacBook电池的蓝牙问题。出于所有这些原因，你应该尽快更新你的苹果设备。
