# 如何摆脱“隐藏厨房”的趋势

> 原文：<https://lifehacker.com/how-to-pull-off-the-hidden-kitchen-trend-1849628983>

室内设计的趋势反映了我们生活的时代，近三年来，疫情让在餐馆吃饭变得相当危险，难怪厨房装修在许多人的愿望清单上名列前茅。特别是，所谓的“隐藏”或“隐形”厨房在过去几年里变得非常流行。



隐藏厨房的吸引力很简单:厨房会变得很乱，隐藏这些乱七八糟的东西有时比清理干净更方便。(就像你在举办一个晚宴，希望你的客人认为你没有实际上那么脏。)但即使你是一个洁癖者，把你的厨房和你生活空间的其余部分分开也是很好的——这对那些困于 [开放式概念平面图的人来说是个好消息，他们并没有要求](https://lifehacker.com/how-to-close-off-your-homes-open-concept-now-that-you-h-1834275562) 。

根据《纽约时报》最近的一篇文章 ，隐藏厨房的趋势在过去的两三年里越来越受欢迎，尤其是在拥有慷慨装修预算的富裕房主中。不足为奇的是，杂志和设计博客中的许多“隐藏”或“看不见”厨房的例子都非常昂贵。但是你不需要重新装修你的整个厨房(或者为你的餐具室建造一个新的整体)来扭转这种趋势。这里有一些让你开始的想法。

## 隐藏整个厨房

尝试隐藏厨房的最简单方法是将它藏在房间隔断、滑动门或任何其他你预算内的物理屏障后面，并与你的其他装饰融为一体。这是一个特别好的选择工作室公寓和其他开放的平面图和小厨房的空间；这是一种简单的方法，可以把你的厨房和你的生活空间分开，而不需要进行一项昂贵的大工程。

## 关闭您的存储

有了隐形厨房，你在紧闭的门后放东西的空间越大越好。如果你有任何开放式的架子，考虑用橱柜来代替，或者把它封闭起来。同样的道理也适用于你的厨房岛，如果你有的话。想要营造一种真正无形的氛围，试着让你的橱柜和你生活空间的其他部分保持同样的美感——它们不需要完全匹配，但要力求一致。

## 翻新你的餐具室

如果你想完全实现隐藏厨房的概念，并有足够的餐具室空间，你可以把它变成一个完整的第二厨房。根据你的需求，你可以选择最大化准备空间的设计，或者优先考虑储物空间的设计— 无论你希望你的实际厨房有更多的空间。(有些人全力以赴，给自己的第二个厨房添置了水槽、电磁炉，甚至洗碗机。)无论你选择什么，确保你的隐藏厨房带有可关闭的门——毕竟，最终目标是让杂乱和混乱远离视线。
