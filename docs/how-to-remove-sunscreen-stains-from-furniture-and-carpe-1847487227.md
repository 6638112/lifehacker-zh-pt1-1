# 如何去除家具和地毯上的防晒污渍

> 原文：<https://lifehacker.com/how-to-remove-sunscreen-stains-from-furniture-and-carpe-1847487227>

防晒霜是夏天的必需品，但有时它会出现在你皮肤以外的地方。无论你是否用力挤压瓶子——导致乳液在你最喜欢的地毯上爆炸——或者你家里的某人坐在沙发上从头到脚涂了一层防晒霜，防晒霜污渍都可能发生。



幸运的是，它们也相对容易去除——尤其是如果你行动迅速的话。以下是如何去除家具和地毯上的防晒霜污渍的知识。

## 如何从装有软垫的家具和地毯上去除防晒霜

在大多数情况下，你不需要太多的努力就可以去除装饰家具和地毯上的防晒霜。以下是如何做到这一点的，由澳大利亚 T3 儿童乐园的 [Leah Goulis 提供:](https://www.kidspot.com.au/lifestyle/home/how-to-remove-sunscreen-stains/news-story/155184875cf31a6d1020aded2c44f4a6)

1.  用钝刀或刮刀之类的东西刮掉尽可能多的防晒霜。不要摩擦室内装潢或织物来去除多余的乳液。
2.  倒入适量的小苏打、玉米粉或爽身粉，放置一夜。
3.  第二天，用真空吸尘器吸走粉末。
4.  如果没有成功，在求助于专业人士之前，你可以再试一次。

## 如何治疗皮革上的防晒污渍

防晒霜对皮革并不友好，所以在这种情况下及时处理污渍就更加重要了。下面是去除它的步骤，由亨克尔 T3 的杰西卡·克尔曼提供:

1.  用布或纸巾吸干防晒霜污渍。不要摩擦它。
2.  用两三滴洗碗液配制一杯水的溶液(Kielman 推荐 Dawn)。
3.  使用一块干净的白布，“将混合物轻轻涂抹在污渍上，并以圆周运动的方式工作。”
4.  擦拭受影响的地方，让它变干。
5.  给这件家具涂上皮革护理剂。

无论你处理的是皮革、软垫家具还是地毯，如果污渍仍然存在，你可能需要找专业人士来完成这项工作。
