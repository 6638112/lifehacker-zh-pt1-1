# 使用这个新的键盘快捷键加快您的谷歌搜索速度

> 原文：<https://lifehacker.com/speed-up-your-google-search-with-this-new-keyboard-shor-1846698271>

谷歌刚刚推出了一个很棒的新键盘快捷键，你可以用它来完善你的搜索查询，虽然它不会节省你一周的时间，但它比滚动回搜索栏，点击它，编辑你的查询以启动新的搜索更容易使用。



捷径很简单，也很容易记忆:当你在任何谷歌搜索的结果页面上时，你所要做的就是按下键盘上的**"/**键——一个正斜杠，如果你仍然对键盘上的*和另一个斜杠*感到困惑的话——然后你的光标会跳回到搜索框，并定位在你查询的最末尾。然后，您可以添加或减去单词，添加 [运算符](https://support.google.com/websearch/answer/2466433?hl=en) ，或者快速执行 **CTRL+A 并删除**以重新开始。

这一招在你最初加载[google.com](https://www.google.com/)的时候不管用，因为不一定要；当你加载页面时，你的光标已经在搜索框中准备好了，你可以输入任何你想输入的内容。但是，当你为了挖掘某样东西而进行多次搜索时，这是一种更快的方式来改进你的搜索，直到你找到你想要的东西。

现在，让我们给你一些快捷方式来加快你的搜索。因为你可以使用浏览器的地址栏来启动所有的谷歌查询，而不是访问谷歌的网站，所以当你想在网上找东西时，只需按下 **CTRL + E** 或 **CTRL + K** 即可。如果你不在一个已经为不同目的(比如说，为一篇博客文章添加一个超链接)使用了键盘快捷键的网站上，那么你的光标会立即跳到一个空的地址栏，这样你就可以开始搜索了。

我已经在 Chrome、Edge Chromium 和 Firefox 上测试过了，效果很好。然而，如果你是一个 Safari 迷，你将需要 mash **Command + L** 。macOS 上就是这样。如果你在 Chrome、Edge Chromium 或 Firefox 中混搭 **CTRL+L(或 ALT+D)** ，你将跳转到地址栏并高亮显示那里的所有内容——这与清除所有内容以执行新的搜索略有不同。
