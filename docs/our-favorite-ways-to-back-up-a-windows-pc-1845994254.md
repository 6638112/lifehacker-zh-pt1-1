# 我们最喜欢的 Windows PC 备份方式

> 原文：<https://lifehacker.com/our-favorite-ways-to-back-up-a-windows-pc-1845994254>

我喜欢定期(是的，定期)做的数字任务之一就是清空我的电脑，重新安装 Windows 10。花一两个小时把我的系统恢复到最大的潜在性能是值得的，而且做起来再容易不过了。是的，这意味着你必须花一些时间重新安装或恢复你的应用程序和数据，但如果你事先备份它们，这也很容易。

有多简单？让我们开始吧。这里有几个选项，可以让您的数据更容易地恢复到 Windows 10 的新副本上。

### 生活在云中

这个很简单。如果您将所有重要信息都保存在云中，那么您就不会将任何个人数据从您拥有的任何东西中恢复到全新的 Windows 10 PC 上。你只需要重新建立与你使用的任何服务的连接，或者安装他们提供的任何应用程序，将你的云内容同步到你的硬盘上，你就可以开始了。

这在实践中意味着什么？好吧，例如，你不用使用你的 Windows 10 用户文件夹来存储文档或图片，你只需将所有这些存储在你的[Google Drive/Dropbox/Mega/one Drive](https://lifehacker.com/google-one-is-now-open-for-everyone-but-is-it-a-good-d-1826049257)上。如果你能记住在所有情况下都这样做，包括将文件从实际的 Windows 10 桌面移动到受保护的云文件夹，那么你就万事俱备了。否则，你可以发挥想象力，设置符号链接( [符号链接](https://lifehacker.com/how-to-sync-your-saved-pc-games-between-computers-with-1501078690) )将这一变化硬连接到 Windows 10 本身；您也可以右键单击您的用户文件夹，并通过**属性**更改它们的位置。将它们放在正在同步到云的主文件夹中。

当然，重新安装 Windows 10 后，你必须重新设置这些链接。如果你只是在心里记下将所有照片和文档存储到云中，你将无法在以后访问其他文件夹，如 AppData 文件夹(包含 web 浏览器的设置、历史记录等)。如果你不介意，那太好了！只要确保你知道拷贝不同用户文件夹的内容和拷贝你的整个用户文件夹到在线存储器之间的区别。

### 使用始终在线的备份应用程序

我承认，我让我的 [Backblaze](https://www.backblaze.com/) 账户过期，因为在它将我的电脑备份到云的多年时间里，我从未用它来恢复任何文件。这并不是说它没有用；我只是有一堆不同的备份技术。Backblaze 是其中之一，但它最终被证明是不必要的。

然而，如果你使用这项服务或任何其他服务，在你的系统上安装一个应用程序，自动将所有文件备份到云端，你就成功了。假设你不介意下载——或者你的服务进程恢复——你可以很容易地在新版本的 Windows 10 中将最新的文件恢复到你的系统中。

这种方法最适合您的个人数据；不要指望用它来恢复你之前安装的所有应用和游戏。

### 直接从文件资源管理器中复制数据

在我的 Windows 10 设置中，我所有的个人数据都在我的主 C:\Users 文件夹中。每当我想将系统上的所有重要内容备份到我的一个辅助驱动器上时，我只需将整个用户文件夹拖放到它的备份目的地。它突突前进，我去吃饭；我回来，处理一些关于不重要的小文件或临时文件的提示，这些文件搞砸了，我对我关心的所有东西都有完整的备份。

没有比这更简单的了，只要你有存储空间(你可以通过右键单击你的用户文件夹并选择**属性**来检查，或者通过使用一个很棒的应用程序，如[**wiz tree**](https://lifehacker.com/the-best-disk-space-analyzer-for-windows-5915921)**来查看是什么占用了这么多空间)。将外部驱动器连接到您的系统，或者使用桌面 PC 中的辅助驱动器，复制您的文件，就大功告成了。不需要复杂的程序。**

### **使用第三方应用程序来创建磁盘映像(或克隆)**

**我们早就是 Lifehacker 的[**Macrium Reflect**](https://lifehacker.com/back-up-and-clone-your-hard-drive-with-macrium-reflect-1825289970)的粉丝了。这款免费的应用程序可以让您轻松地将整个硬盘克隆到备用硬盘，或者创建硬盘的完整映像，稍后您可以在全新的操作系统中调出该映像。**

**因为我已经详尽地介绍了这个应用程序，所以我不会在本文中向您介绍成像过程。对大多数人来说这很容易理解。事实上，我不确定你*会不会*出错，只要你在用新版的 Windows 10 清理你原来的硬盘之前检查你做的克隆或者镜像。(即便如此，拥有尽可能多的数据备份总是好的，所以考虑将你的关键数据上传到云中，将其复制到你身边的任何其他存储设备，或者将其临时放到另一个系统上)。**

### ****分别备份您的浏览器标签****

**你应该已经将你的浏览器数据(比如你的书签)同步到云端了，不管你用的是什么浏览器。在你告别你的旧操作系统并重新安装一个新的副本之前，使用一个扩展，如 [**OneTab**](https://lifehacker.com/organize-your-browser-tabs-with-onetab-but-back-them-u-1837740779) 将你所有打开的标签合并成一个标签，然后导出你的列表。把它保存到云端，或者用电子邮件发给你自己，或者把它放在闪存盘上——无论你想保存什么数据都可以。不要忘记这个文件，因为它包含了你所有的标签，如果你和我一样，这是一个相当大的列表。**

### **重装 Windows 10 后如何找回应用和游戏？**

**假设你已经备份了所有数据，深吸一口气， [重装了 Windows 10](https://lifehacker.com/the-ultimate-guide-to-reinstalling-windows-from-scratch-1832897572) ，正盯着一个全新版本的操作系统。我们已经介绍了你想知道的所有 [大步骤和设置](https://lifehacker.com/how-to-set-up-your-windows-laptop-from-scratch-1826146907) ，但是有几个关键步骤你想马上做:**

*   **将你的 Microsoft 帐户连接到 Windows 10，以便你可以激活操作系统并开始同步设置。**
*   **开始将您的数据复制回您的系统。我建议在你的桌面上创建一个“旧 PC”文件，将所有内容复制到那里，并使用*和*作为源文件，计算出你的文件的最终位置。(记住，有太多备份永远不会有坏处。)**
*   **使用类似[](https://ninite.com/)**(或 [**巧克力**](https://lifehacker.com/the-best-way-to-quickly-install-apps-on-a-new-windows-p-1836244140) )的工具开始重新安装你的核心应用。****
*   ****获取你最喜欢的游戏服务——Steam、Epic、Battle.net 等等——并安装它们。但是，不要重新下载你的游戏。如果您保存了它们，请从备份中找到它们，并将它们复制回原始位置。然后，你应该能够使用该应用程序将每个游戏“安装”到该位置，这不会导致它重新下载整个游戏，只需验证现有的文件并根据需要下载较小的更新。(你的游戏服务应用程序甚至可能已经能够扫描现有的游戏文件，如 Battle.net。)这是一个避免为你可能已经拥有的数据增加每月互联网数据上限的好方法。****
*   ****在这一切进行的同时，建立你的在线账户。这些可以包括您需要将内容同步到云存储或从云存储同步内容的任何应用程序、您的主要电子邮件和日历服务，以及需要名称和密码的任何其他东西，包括您的密码管理器。****
