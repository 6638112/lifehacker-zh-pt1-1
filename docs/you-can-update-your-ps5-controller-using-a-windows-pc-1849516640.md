# 您可以使用 Windows PC 更新 PS5 控制器

> 原文：<https://lifehacker.com/you-can-update-your-ps5-controller-using-a-windows-pc-1849516640>

你的 PS5 控制器出问题了吗？一个 固件更新为你的双感控制器可以快速修复许多小问题。如果你有一台 PS5，通过将插入控制台来更新控制器是很容易的。然而，如果你*没有*控制台——也许因为你用控制器玩 PC 游戏——你可以用 Windows 机器来完成这项工作。(对于那些控制器拒绝更新你的 PS5 的罕见情况，这也是一个很好的选择。)



## 如何通过窗口更新 PS5 的双感控制器

首先，使用 USB 电缆将控制器连接到 PC。接下来你要下载索尼的 [DualSense 固件更新器](https://controller.dl.playstation.net/controller/lang/en/fwupdater.html) 。这个链接将带你到一个支持页面，在那里你可以向下滚动并点击蓝色的**下载**按钮。

一旦您在 PC 上下载并安装了 DualSense 固件更新程序，运行该程序并按照屏幕上的说明进行操作。从这里开始，你的双感控制器将在一段时间内自我更新。

固件更新过程中，请勿关闭电脑或拔下 USB 电缆。理想情况下，你应该让控制器在更新时放在一个平坦的表面上——如果你在更新期间拿着它或摆弄它，控制器可能会失去与你的 PC 的连接，特别是如果你用来连接两个设备的 USB 电缆松了，这可能会导致问题。

更新完成后，您的控制器就可以使用了。如果你在游戏机上使用它，你就不需要在 PS5 上单独更新它。享受你(希望)改进的体验，至少在新的固件更新可用之前。
