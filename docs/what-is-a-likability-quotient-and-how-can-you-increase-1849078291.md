# 什么是“讨人喜欢的商数”(以及如何提高你的商数)？

> 原文：<https://lifehacker.com/what-is-a-likability-quotient-and-how-can-you-increase-1849078291>

你听说过智商，或者智商，你可能知道这不是一个超级可靠的方法来确定你有多聪明。但是你知道也有一个“讨人喜欢的商数”，你可以采取措施影响自己的商数，从而让自己更讨人喜欢吗？



## 什么是讨人喜欢的商数？

亨德利·韦辛格博士是情商专家，他在 2015 年的 中写道，讨人喜欢是“生活中所有领域成功的关键预测因素”简而言之，无论你多讨人喜欢，都会直接影响你在友谊、爱情、职业和家庭关系中的成功程度。

这里的商数指的是对你的受欢迎程度的衡量，通常可以通过人们如何对待你和与你互动来衡量。Weisinger 将获得聚会邀请或工作晋升作为衡量你的受欢迎程度及其相关成功的指标，但想想你如何与周围的人相处。如果你的兄弟姐妹定期给你打电话告诉你他们一天的情况，你的朋友信任你并告诉你秘密，或者邀请你参加重要的活动，或者你们有了有意义的爱情纠葛，你可以自信你的可爱指数很高。

## 如何增加你的讨人喜欢的商数？

Weisinger 的文章建议关注你的幽默感和倾听技巧，虽然这两者都会让你对他人有吸引力，但还有一些其他的事情需要考虑。本周，前联邦调查局行为分析师杰克·夏福特博士在他为今日 T4 心理学杂志撰写的文章中详细阐述了威辛格的建议，建议你看看这些所谓的吸引力法则:

*   **相似定律**。有相似想法、态度和兴趣的人倾向于结合在一起。如果你想变得可爱，考虑一下你想吸引的群体——以及他们中的人是否和你有共同的兴趣。会更容易被和你有共同之处的人喜欢。
*   **错误归因法则。在这里，Schafter 指出，当人们玩得开心时，他们倾向于将自己的好心情与当时在他们身边的人联系起来。无论是在健身房还是在办公室的快乐时光，让你周围的人都开心或产生内啡肽，你会和快乐时光联系在一起，增加你的可爱度。**
*   **好奇心法则。**夏福特写道“当你的行为引起另一个人的好奇心时，这大大增加了人们想要与你互动以满足他们好奇心的机会。谈谈你独特的经历，比如你是否在和你生活的地方不同的环境中长大，或者你是否在大学里学了一些与众不同的东西。给人一些好奇的东西。
*   **自我表露法则。**与他人交往时，要脆弱，但不要强求。这可以帮助你发展亲密关系，因为人们自然倾向于敞开心扉，与真诚的人交往。作为回应，他们可能会很脆弱，现在你们有了真正的联系。
*   **幽默法则。**这个显而易见，但是当你搞笑的时候，人们会喜欢你。不要害怕展示你愚蠢的一面。
*   **可用性或稀缺性法则。**不要让自己太容易被人利用，至少不要一下子。这与我们在《好奇心》中讨论的内容有关:当你与人们接触时，你应该让他们有更多的欲望，这样他们就有理由来找你。不要绝望；清楚地表明你是抢手货。

本质上，如果你想增加你的 LQ，就要开放、风趣，不要虚伪或绝望。然后，准备好让成功流淌。
