# 父母应该有多担心 Parents 病毒？

> 原文：<https://lifehacker.com/how-worried-should-parents-be-about-parechovirus-1849200626>

Parechovirus 不是一种新病毒，但由于婴儿病例可能上升，它是最近美国疾病控制和预防中心健康警报 的主题。这种病毒会导致非常年幼的婴儿患上严重疾病。那么父母需要有多担心呢？



## 什么是 parechovirus 病毒？

“帕累托病毒感染在儿童时期很常见，”美国儿科协会 [在](https://publications.aap.org/aapnews/news/20733/CDC-issues-advisory-on-parechovirus-infections-in?autologincheck=redirected) 的一篇新闻文章中指出。大多数孩子在进幼儿园之前就接触到了它。幼儿和学龄前儿童的典型症状是发热、皮疹和呼吸道症状。

这种病毒可以通过呼吸道和胃肠道传播。(或者从孩子们做的事情来说，它可以通过咳嗽和打喷嚏传播，或者通过孩子把带有粪便细菌痕迹的东西放进嘴里传播。)这种病毒在夏季和秋季最为常见。

通常，catch 病毒只是许多神秘病毒中的一种，孩子们会感染并很快痊愈。但是在年幼的婴儿身上，一些病毒会导致非常严重的疾病，副疱疹病毒就是其中之一。CDC 表示，“小于 3 个月的婴儿可能会出现严重疾病，包括败血症样疾病、癫痫发作以及脑膜炎或脑膜脑炎，尤其是小于 1 个月的婴儿。”

## 为什么 **parechovirus** 会出现在的新闻中？

疾病预防控制中心在最近几个月注意到了几起副疱疹病毒病例，所有被检测的病例都属于 PeV-A3 型，这种病毒往往比其他毒株引起更严重的症状。但是这是否意味着这种病毒比平常更常见，或者病例比平常更严重？我们不知道。

“因为目前在美国没有对 PeVs(parecho virus)的系统监测，所以不清楚 2022 年报告的 PeV 病例数量与以前相比如何，”CDC 说。儿童也没有接受常规的帕累托病毒检测，所以我们不知道有多少病例发生在仅轻度患病的儿童或感染了病毒但根本没有患病的儿童中。

## 如何知道我的孩子是否感染了帕累托病毒？

如果你的孩子只有轻微的症状，你可能永远也不会确定这是不是 parecho 病毒。记住，这是一种常见的儿童病毒。正如传染病专家 Claire Bocchini [告诉《今日美国》](https://www.usatoday.com/story/news/health/2022/07/19/parechovirus-rash-symptoms-baby/10090057002/) ，“我 n 个患有轻微疾病的孩子，我们不会对他们做任何测试。这很贵，也不会改变管理。”(管理，如你如何处理病毒。如果结果只是让孩子舒服地等待，那是什么病毒并不重要。)

婴儿的情况就不同了。CDC 的警报提到，在 3 个月以下的儿童中，parecho 病毒是一个特别值得关注的问题，在那里它可能是致命的。但是他们发布的健康警告是针对提供者的，而不是父母。这一信息本质上是让医生知道，当他们考虑一个严重患病的婴儿可能会有什么问题时，他们应该将 parechovirus 病毒包括在他们的可能性列表中。

“在这个年龄段，任何发烧的婴儿都应该接受评估，至少包括 60 天以下的血液检查，”纽约市儿科急诊医生 Heather Giambo 说。他们强调，父母应该为这么小就发烧的孩子寻求医疗护理，不管是什么原因引起的。在生命的最初几个月，美国儿科协会认为[100.4](https://healthychildren.org/English/health-issues/conditions/fever/Pages/Fever-and-Your-Baby.aspx)的体温是一种值得立即就医的发烧。
