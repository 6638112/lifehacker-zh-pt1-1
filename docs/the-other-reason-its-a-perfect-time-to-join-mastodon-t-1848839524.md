# 另一个原因是现在是加入乳齿象的最佳时机，乳齿象是开源的 Twitter 替代品

> 原文：<https://lifehacker.com/the-other-reason-its-a-perfect-time-to-join-mastodon-t-1848839524>

上周，乳齿象终于推出了免费的官方 Android 应用程序 ，使得在 Android 设备上使用开源的 Twitter 替代品比以往任何时候都更容易。乳齿象的时机再好不过了，因为 Twitter 的董事会刚刚批准了埃隆·马斯克(Elon Musk)以 440 亿美元收购 Twitter 。

突然迫近的收购已经分裂了 Twitter 的用户群。一些用户接受马斯克对更“开源”的 Twitter 算法的愿景，以及对编辑推文等功能的承诺。其他人有理由担心马斯克的所有权会破坏这位亿万富翁经常批评的来之不易的温和政策——并担心收购树立的先例。一个人应该能够心血来潮地收购一家大公司，仅仅是为了改变它的经营政策以适应他们的偏好吗？

无论你的观点如何，事实是许多推特用户把收购 T2 视为是时候尝试像乳齿象一样的在线 T4 空间的标志了。

乳齿象很像 Twitter:用户将微博发布到他们的时间轴上，互相关注，分享彼此的帖子。有标签，你可以用“@”喊别人，并查看最近的或热门的帖子。新的 Android 乳齿象应用程序甚至看起来与 Twitter 应用程序非常相似。然而，与其说是一个庞大的全球用户群在互相发推文，不如说是一个分散的平台，拥有众多较小的社区，有点像 subreddits 或公共 Discord 服务器。每一个都有自己独立的“迷你推特”，用户可以在这里登录并联系相似的兴趣爱好。一些社区是地区性的，而另一些则基于特定的主题，如技术、游戏、艺术等。

较小的社区焦点更容易策划你的乳齿象体验——但注意每个社区都有自己的指导方针和规则，你需要遵守。你也应该期待在乳齿象上比在 Twitter 上更安静的体验。虽然乳齿象称其拥有近 450 万用户，但你只能看到来自的消息，即你当时登录的社区内的用户。这可能会使有意义的对话和找到相关内容变得更容易，但你不会培养大量的观众，而且一些群体可能不活跃。

你可以在我们的 [乳齿象](https://lifehacker.com/a-beginner-s-guide-to-mastodon-1828503235) 初学者指南中阅读更多关于乳齿象以及如何使用它的信息，或者从 [【谷歌 Play 商店】](https://play.google.com/store/apps/details?id=org.joinmastodon.android&hl=en_US&gl=US) 下载新的 Android 应用程序，然后直接进入。乳齿象在 iOS[web 浏览器](https://joinmastodon.org/) 上也有 [，或者作为桌面上的](https://apps.apple.com/us/app/mastodon-for-iphone-and-ipad/id1571998974) [渐进式 web app](https://joinmastodon.org/apps) 。
