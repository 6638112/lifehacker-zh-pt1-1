# 那么，应该先玩哪个 Bethesda Game Pass 游戏呢？

> 原文：<https://kotaku.com/so-which-bethesda-game-pass-game-should-you-play-first-1846465954>

从今天开始，你可以在 Game Pass 上玩一堆 Bethesda 发布的游戏，这要感谢微软和 ZeniMax 最近的联姻。有些是以前就有的。许多是新的。无论你如何划分，如果你是微软游戏点播服务的一员，你现在有 20 款非常优秀的游戏唾手可得。



很明显，你不能同时玩所有的游戏。首先，这些游戏并不小。强大的 Xbox 系列 X 可能会提供 1TB 的存储空间，但那个 [会很快填满](https://kotaku.com/how-to-make-the-most-of-your-xbox-series-x-s-internal-s-1845619754) ，用大量共享大量 DNA 的游戏来填满你的图书馆是没有用的。此外，你最宝贵的资源中还有一件小事:时间。

那么，从哪里开始呢？那要看你想玩什么了。如果你在找…

## ...一个射手:玩*末日永恒*。

*末日永恒*只有一个速度:快。大多数射手都很小心:找掩护，排队射击，说一些有趣的军事术语，比如“重新装弹”和“火力掩护”。现代黑客和斜线*厄运*游戏或多或少地说与地狱。因为你通过打击敌人来恢复生命值和弹药，你被迫保持不断的运动。近距离接触是必须的。静静地站一秒钟，你就完了。

尽管 2016 年的*末日*很棒(确实很值得玩)，*末日永恒*将整体疯狂度提升至 11。除了新的敌人类型，你还会得到一个抓钩——一个从未让游戏变得更好的工具——和一把畸形的剑。不用担心赶不上剧情。你真正需要知道的就是:恶魔，坏，杀。

## ...一个身临其境的模拟游戏:玩*猎物*。

Arkane 工作室可能因*不名誉*游戏而闻名(这两个游戏都很棒，都在游戏通行证上)，但工作室真的以*猎物*大获成功。尽管从技术上来说，它与 2006 年的游戏拥有相同的知识产权，但不要担心:这两者之间没有关系。你不需要学习任何知识。你可以完全冷着进去。如果有什么不同的话，新的*猎物*感觉更像*生化奇兵*而不像旧的*猎物*。

现代的*猎物*进入了一个空间站，虚构的塔罗斯 1 号，在另一段历史的不久的将来，JFK 没有被暗杀，这显然导致 60 年代风格的现代装饰一直持续到 21 世纪 30 年代。由于恶意的外星人的存在，塔罗斯一号大部分被洗劫、毁坏和遗弃。这听起来很可怕——它*是*可怕——但是，另一方面，它最终给了你一些非常酷的超能力，包括允许你占据游戏中基本上任何物体的能力。这个故事有点乏味，所以大部分乐趣都在探索空间站，发现新的通道，释放新的力量，慢慢地但肯定地变得足够强大，以实际上对付可怕的怪物。只是不要对一个稳固的结局抱太大希望。

## ...一个角色扮演游戏:玩*辐射 4* 。

你肯定听说过*辐射:新维加斯*是贝塞斯达权限下的重要 RPG。当然，这是一场稳定的比赛，你的决定会产生影响。也有一千年了。那些对“贝塞斯达 RPG”这一广泛类型不熟悉的人应该坦率地从《T2》辐射 4(T3)开始，这是一群人中最新最现代的感觉。

《辐射尘 4》,像其他的《T2 辐射尘》,发生在一个被核弹炸成碎片的美国版本中。这是第一人称射击和第一人称统计混合的真实结合，因为你很可能在一场交火中结束，就像你在一系列分支对话选项中一样。这一切都被包裹在一个巨大的开放世界中，它恳求你去探索它。

但是*辐射尘 4* 有着其他现代*辐射尘*所没有的:它向你展示了“以前的时代”从美国变成放射性尘埃的那一天开始，*辐射尘的第一瞬间就开始了。你被放进一个地下室，出乎意料地被低温冷冻，两个世纪后醒来，你被释放到一个版本的波士顿，看起来就像今天红袜队赢得主场比赛后的波士顿。(旁注:为什么波士顿人在他们的球队*赢了*的时候洗劫他们的城市？从来不明白那个。)最初的样子值得从*辐射 4* 开始。如果你喜欢的话，这个游戏还有一个强大的基础构建元素。*
