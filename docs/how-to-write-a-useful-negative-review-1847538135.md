# 如何写一篇有用的负面评论

> 原文：<https://lifehacker.com/how-to-write-a-useful-negative-review-1847538135>

我们都知道写得不好的网上评论对任何人都没有好处，除了那些愤怒地把它们打出来的人。但时不时地，你会发现自己处于一种需要一些建设性批评的境地。无论你是在评论一笔亚马逊购买的商品、一家餐馆还是一位前雇主，你的基本评论写作策略应该保持不变:让它可读，让它有用。下面是怎么做的。

## 描述发生了什么——以及应该会发生什么

如果你正在考虑写一篇差评，你可能会被至少一次特定的经历所激励。这应该是你的重点。首先，清晰客观地描述发生的坏事，并且紧扣事实——换句话说，不要编造谎言。如果你仍然对你的经历感到激动，那就保密吧。

接下来，描述当时应该发生的事情——而不是你希望发生的反应，或者你认为你欠了什么。这是一个重要的区别，因为这基本上是一个测试，看看你的抱怨是否站得住脚。对于合理的投诉，说出应该发生的事情会非常容易:你的前老板应该把他的总评论留给自己，亚马逊应该就一件商品向你收费一次而不是两次，服务员和厨师应该在知道你过敏后把花生酱从你的盘子里拿走，等等。但是对于归结为你感觉没有得到充分满足的琐碎抱怨，做事情的“正确”方式要么不存在，要么太离谱，你还不如用“请不要认真对待我说的任何事情！我正在以一种不健康的、最终毫无成效的方式解决一些个人问题！”

## 抓住要点

冗长的评论毫无用处，所以你的评论要简明扼要。这将使人们有可能阅读和理解你所说的话，这是写任何评论的全部要点。另外，坚持使用简短的陈述句也会让你无法发表长篇大论，这样你就不会表现得像一个被鄙视的自恋狂，不惜一切代价想要达到自己的目的。(即使在情绪爆发完全是合理反应的情况下，它们也会削弱你的观点。)

底线:如果你真的想让你的评论有所不同，它需要尽可能的有针对性并且易于阅读。讲真，把你的案子，包起来；无论谁最终读了这本书，都会感激他的努力。
