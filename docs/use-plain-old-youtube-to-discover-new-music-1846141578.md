# 使用普通的旧YouTube来发现新音乐

> 原文:[https://life hacker . com/use-plain-old-YouTube-to-discover-new-music-1846141578](https://lifehacker.com/use-plain-old-youtube-to-discover-new-music-1846141578)

围绕音乐流媒体的传统智慧是这样的:Spotify统治一切，在他们选择的平台上品味略有不同的人会选择苹果、或其他完全专注于音乐的服务。大型音乐播放器本身都是技术奇迹，给更广泛的音乐产业带来了革命，给各地的听众带来了无限便利。我在这里告诉你去别的地方听你的音乐。

YouTube是谷歌影响深远的业务的一部分，所以使用这个平台来播放流媒体音乐并不完全是支持一个新贵对美国企业界发起战争，而是YouTube是一个晦涩而精彩的深度切割的宝库，它将帮助你将你的音乐品味延伸到意想不到的领域。我说的也不是YouTube音乐，而是普通的YouTube。

如果你对音乐的探索感到厌倦，以下是你应该回归基础的原因。

### 不是由算法管理的播放列表

别误会，Spotify上有很多稀有宝石。但在大多数情况下，你得到的播放列表是由平台的设计师在幕后策划的，这给了这些安排好的曲目一种人造的氛围。在互联网被用户管理而不是纯粹为了利润的算法的最后几个例子之一中，YouTube充满了播放列表和DJ集，这些播放列表和DJ集是由那些有动机有机分享音乐并将其传播的人管理的。

后者是一个很好的例子，说明了YouTube如何以比一些音乐平台更简单的方式帮助音乐发现。DJ Milos Kaiser的所有曲目都是、*稀有、*用一个术语来说黑胶势利小人的爱。如果不是这家伙从某个地方的板条箱里挖出这些，自己录下播放的视频，并上传到YouTube上，我是不会听到它们的。当然，这里有大量由算法策划的播放列表，但你不必关注它们。

### 它的界面带来了更多的有机发现

我知道Spotify很时尚，其移动优先的设计在移动中无缝运作。但使用YouTube寻找新音乐不会让你感到被轰炸，或者像你必须咨询专门用于发现新艺术家的特殊功能一样。就我个人而言，我从来没有在Spotify上用shuffle听过一位艺术家的歌，但最终却听到了我不喜欢的其他T2艺术家的歌。

YouTube的算法感觉没有那么霸道，因为你会在视频结束时获得新曲目的建议，而不是立即切换到你没有选择的歌曲。当然，这取决于你是否关闭了自动播放功能，但是你仍然可以更好地停留在你选择的音乐领域中。

### YouTube上有真正的音乐社区

与传统的YouTuber页面不同，它有许多现代互联网文化中典型的怪癖和令人头疼的问题，属于唱片公司和各种音乐社区的页面通常带来较少的废话。甚至在互联网成为今天的公司地狱场景之前，DJ团体、DIY音乐出版物、和唱片公司都有自己的YouTube频道。尽管目前你可能不喜欢YouTube上的音乐，但不可否认的是，尽管有其他平台的便利性和受欢迎程度，这些团体中的许多都在蓬勃发展。

例如，以音乐策展团体锅炉房为例，它在YouTube上有260万订户。 在Spotify上，它只是那个强大社区的一个影子，分散的播放列表由锅炉房的听众管理。如果你对这类事情感兴趣——或者DIY音乐领域的任何其他方面——你最好在YouTube上订阅这些频道，每次有有趣的东西上传时，你都会收到电子邮件更新。

音乐发现完全是由直觉来引导的， 所以，无论如何，你应该做你想做的。但是，如果你的日常工作感到无聊，测试不同平台的水域肯定不会有什么坏处，普通的老YouTube可能是一个有用的 返回的地方。
