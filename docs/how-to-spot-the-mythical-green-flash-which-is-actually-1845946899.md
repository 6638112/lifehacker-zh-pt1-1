# 如何发现神秘的绿色闪光(实际上非常真实)

> 原文：<https://lifehacker.com/how-to-spot-the-mythical-green-flash-which-is-actually-1845946899>

虽然这听起来像某种超级英雄——特别是挂在时代广场的那种未经许可的非品牌——“绿色闪光”实际上是一个真实的东西，你可以在天空中亲眼看到。虽然我们知道绿色闪光的存在(有 [大量的照片证据](https://aty.sdsu.edu/pictures.html) ) ，但它已经达到了一个更神秘的状态，因为它不是经常看到的东西。

不像某些流星雨或月亮，你不能在日历上标记绿色闪光:它必须发生。但是，有一些方法可以增加你瞥见难以捉摸的自然光表演的机会。以下是需要了解的内容。

### 绿色闪光到底是什么？

首先要知道的是，虽然有几个自然现象集中在一起被称为“绿色闪光”，但大多数都属于两种形式中的一种，这两种形式都是英国物理学家 [詹姆斯·普雷斯科特·焦耳](https://www.aps.org/publications/apsnews/201506/physicshistory.cfm) 在 1869 年确定的。第一种形式是焦耳所说的“最后一瞥”，是一种蓝绿色的闪光，出现在太阳沉入地平线之前的最后一刻。

第二种形式与模拟海市蜃楼有关，据天文学家、绿色闪光专家、圣地亚哥州立大学 教授安德鲁·t·杨(Andrew T. Young)说，这主要是从高处看到的。(如果你想了解更多关于绿色闪光的信息， [Young 维护着一个网站](https://aty.sdsu.edu/index.html) 里面全是信息和照片链接。)

### 如何捕捉绿色闪光

你不需要任何花哨的设备来看到绿色闪光，但你需要在正确的时间出现在正确的地方吗？就像天空中发生的大多数令人兴奋的事情一样，你看到绿色闪光的最佳机会是在天气晴朗的时候。根据[EarthSky](https://earthsky.org/tonight/how-to-catch-the-elusive-green-flash)上的一篇文章，这也有助于看到非常遥远的地平线，比如海洋。

人们报告看到的大多数绿色闪光都是“最后一瞥”，发生在日落时，即太阳沉入地平线前的最后一秒。是的，绿色闪光 [通常只持续一秒](https://aty.sdsu.edu/index.html)——也许两秒(这不叫“绿色余光”)。也有可能在日出前看到绿色闪光，但时机更难把握。

在日落时捕捉绿色闪光的关键是 [不要看得太早](https://earthsky.org/tonight/how-to-catch-the-elusive-green-flash) 。这一点很重要，因为 a)过早(或任何时候，真的)直视太阳会损害你的眼睛；b)这会让你错过绿灯。相反，看向别处，直到地平线上出现太阳最薄的边缘。
