# 为什么您应该停止设置具有多个电子邮件地址的智能设备

> 原文：<https://lifehacker.com/why-you-should-stop-setting-up-smart-devices-with-multi-1847608716>

当你和某人住在一起时，你分享你的东西。你们都使用智能扬声器在家里听音乐，你们都使用智能灯向朋友炫耀(“看，我可以把它们变成蓝色！”).关键是，你们都使用相同的智能设备，也是时候将这些设备绑定到同一个电子邮件地址了。

## 智能设备需要电子邮件地址才能工作

智能设备——包括智能扬声器、智能灯和智能锁，仅举几例——通常需要连接到一个帐户才能工作。有时，他们会适合一个集体的“家庭”应用程序，但其他时候，他们需要一个独立的帐户。这通常没什么大不了的。你设置好设备，输入电子邮件和密码，然后嘭的一声，你的灯现在由你的声音控制，而你的灯开关再也不会被使用了。

但是，不可避免的是，你需要摆弄那个智能设备。也许你需要用 app 来改变一个设置；也许你需要在系统上执行一个新的更新；或者你需要用一部新手机来控制设备。当你试图在智能设备上完成这个任务时，会发生什么呢？它要求你的电子邮件和密码。

如果你匆忙设置这些东西——我们经常这样做——你可能只是和你的伙伴来回设置系统。也许一个设备在你的帐户下，另一个在他们的帐户下，等等。积累足够多的设备，并试图记住哪个设备在哪个帐户下成为一项沮丧的练习:“Sonos 在你的电子邮件下还是我的电子邮件下？”“我有四封不同的电子邮件；它可能在任何一个下面。”"你有*四封*不同的电子邮件？"

## 为什么应该为智能设备创建共享电子邮件

避免头痛，避免觉得需要创建一个所有设备及其帐户的主列表，避免回答关于您的许多电子邮件地址的问题。J 帮你自己和你的家人一个忙给自己注册一个共享邮箱账户。

想想看:一封邮件下的智能灯，一封邮件下的智能门铃，一封邮件下的电视及其所有应用和服务。不再需要猜测，不再需要等待某人从会议中出来询问智能演讲者的电子邮件是什么— 这一切都在同一个简单的系统下进行。

这条建议基本上适用于你和你的搭档可能一起使用的任何类型的账户。想知道谁的电子邮件与你的流媒体服务如网飞、Hulu 和 HBO Max 绑定了吗？有权访问同一个日历，你们可以一起查看和编辑该日历；注册一个基于云的待办事项应用程序，这样你就可以随时关注购物清单。这只是让联合任务变得容易多了。

## 一个共享的电子邮件地址并不是最重要的

当然，这并不是说单独的电子邮件地址是不可能的。为你自己的信件、独立账户等保留你的个人账户。没有必要把你自己的信息塞满对方的收件箱(已经有足够多的垃圾邮件了)。为你们双方都受益的事情保留联合账户。

共享电子邮件地址也不是唯一的解决方案。您可以使用密码管理器获得类似的结果，因为您可以将所有不同的帐户合并到一个家庭保险箱中，你们都可以使用这个保险箱。然而，最好的密码管理器是付费订阅。邮件是免费的。
