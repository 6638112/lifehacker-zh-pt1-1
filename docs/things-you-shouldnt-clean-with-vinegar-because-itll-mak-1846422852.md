# 什么东西不能用醋清洗

> 原文：<https://lifehacker.com/things-you-shouldnt-clean-with-vinegar-because-itll-mak-1846422852>

众所周知，我们喜欢白色醋，因为它能清洁、光亮、除臭，并在各种家居用品中发挥作用。但是，尽管酸性物质如此强大，它也不能做*所有的事情。*而且，在某些情况下，试图用醋清洗东西只能让事情变得更糟。以下是《每日快报》乔治娜·劳德的一篇文章 。

### 鸡蛋污渍

鸡蛋污渍比看起来更难处理，这包括它们与醋的关系。这是因为当醋中的酸与鸡蛋结合时，它会导致鸡蛋凝结——导致比你开始时更糟糕的硬化混乱。相反，根据赞美:

> 你应该用刀子或勺子把任何固体提起来，把两茶匙洗碗皂和两杯水混合在一起。用海绵、白布或软毛刷吸干——不要摩擦——直到污渍消失。

### 硬木地板

有些人发誓醋会让他们的硬木地板闪闪发光，但其他人发现醋会破坏漆面。这是一个安全比后悔更好的例子，所以不要用醋，使用专门为清洁木材而制造的清洁剂。

### 熨斗

如果你的熨斗变脏或堵塞，醋似乎是显而易见的解决办法，但它会破坏熨斗的内部机制。相反，完全关闭熨斗让其冷却，然后按照制造商的清洁说明进行。

### 石材地砖

事实上，醋可以破坏你家任何类型的天然石材表面——不仅仅是地砖。这是另一种最安全的选择是坚持使用专用石材表面清洁剂的情况。

### 异常顽固的污渍

如果你要处理的是无法去除的草、墨水或酒渍，不要继续倒醋或喷醋，试着在清洗前用去污剂处理该区域。

### 大理石和花岗岩柜台

同样，醋中的酸可能会损坏你的大理石或花岗岩台面，可能会破坏它们光滑的表面并导致变色。这一次你需要的是用温水稀释的温和的洗碗精来安全地完成工作。
