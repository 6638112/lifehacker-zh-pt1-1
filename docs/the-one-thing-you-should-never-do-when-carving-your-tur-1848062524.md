# 切火鸡的时候，你千万不要做的一件事

> 原文：<https://lifehacker.com/the-one-thing-you-should-never-do-when-carving-your-tur-1848062524>

我经常做一个奇怪的噩梦，在这个噩梦中，我切了一块牛排，拍了一张切片的照片，并把照片发布在博客上，却发现我把肉和纹理一起切了。然后每个人——无论是在评论中还是在推特上——都嘲笑我违反了这个基本的肉食规则，我再也不会在这个镇上工作了。

这是一个离奇的焦虑梦，但也许不无道理。你切肉的方式对嫩度有很大的影响。你可以在这里 看一段解释其背后科学的视频 [，但结果是咀嚼长串肌肉纤维比咀嚼由一堆短肌肉纤维横截面组成的切片需要更多的力量。对于牛排和火鸡来说都是如此，这也是为什么你看到的上面这张库存照片——在我看来——是一种烹饪威胁。](https://lifehacker.com/why-the-way-you-slice-meat-can-make-a-huge-difference-1723654074)

直接从鸟身上切下胸脯肉——通过平行于胸骨的切口——是个坏主意。这叫做顺纹切片，你最终会得到有嚼劲的肌肉纤维束，而不是嫩滑的切片。(而火鸡是一种通常需要尽可能多的帮助的肉类。)

幸运的是，在这方面很容易给你的火鸡一个翅膀，并确保它在你的嘴里尽可能的柔软。开始时，用刀沿着两侧的胸骨，切除两瓣胸肉。继续沿着骨头走，直到你能完整地取出每一个耳垂。转移到砧板上，*然后*切*垂直于*纹理。如果你不确定这是什么意思，就把它翻过来，看看肉串运行的方向——你的目标是使它们尽可能短。你切得越薄，火鸡就越容易咀嚼，你(和你的客人)也会越开心。
