# 用尽旧香料的最佳方法

> 原文:[https://life hacker . com/the-best-ways-to-use-up-old-spices-1849423616](https://lifehacker.com/the-best-ways-to-use-up-old-spices-1849423616)

如果你和我一样，在你的成年生活中，你在你的厨房里收集了大量的香料，其中许多是你购买的，然后只使用一次。然后有一天，你 [看着你的调料架](https://lifehacker.com/how-to-konmari-your-spice-rack-and-actually-use-the-sp-1831812987) 注意到那些瓶子上有一层厚厚的灰尘，想知道:调料会变质吗？答案是有*合格*没有*。*

Watch

香料实际上从来不会变质，但随着时间的推移，它们会失去效力，最终变得毫无用处(给你原本灰暗的生活带来味道)。一个简单的方法来判断你的香料是否已经过了最佳时期，那就是让它们通过"[snIFF test](https://lifehacker.com/the-easiest-way-to-test-and-use-up-your-old-spices-1848311310)，"，听起来就像这样。香料应该有强烈的香气；如果你的闻起来很淡(或者一点味道都没有)，那它们太老了。

但是，当你的百里香和月桂叶已经成为他们以前的自我褪色的幽灵，你只是把它们扔掉吗？不一定——香料实际上有很多替代用途。当涉及到破解你的旧香料时，很难区分事实和虚构，这里有一些方法可以完全合法地使用它们。

## 烹调

使用你的旧调料的最简单的方法是*使用你的旧调料*。因为香料从来不会变质，你仍然可以在烹饪中使用。如果它们有点淡，那就比平时多吃一些——你真的不会错得太多，因为你最糟糕的情况是在晚餐时需要大量使用食盐。

一个古老的方法是在烹饪前烘烤你的香料。将一些耗尽的香料放在平底锅或烤架上烤几分钟，暴露在高温下，可以释放那些旧种子和树皮屑中最后的味道和气味，在你将它们加入晚餐之前给它们最后一次提振。

## 防止害虫

昆虫和啮齿动物通常会被香料的气味击退，所以用你的旧香料架来保护你的花园和你的家实际上是可行的——许多园丁会专门种植一些东西来驱赶害虫 。这里的警告是双重的:一方面，你的香料是旧的，所以它们没有那么辣，因此在驱赶害虫时不会像新鲜的香料那样有效。第二，这些技术可能不会像商业产品那样有效。但是！如果你面对的不是一级防御级别的害虫入侵，你架子上的一些香料可以很好地抵御它们。

将老辣椒和一些洗洁精和水混合可以帮助你远离蚜虫和其他饥饿的昆虫。将老月桂叶放在你房子的黑暗角落和缝隙中，可以帮助赶走蟑螂，薄荷袋可以赶走老鼠。你也可以将煮过的迷迭香过滤后倒入喷雾瓶中制成驱蚊剂。好消息是，即使这些技术不起作用，它们也会让你和你的房子闻起来很香。

## 打扫你的房子

有些香料有很强的清洁性能。例如，百里香含有精油麝香草酚，它充当了的相当有效的抗菌剂。百里香酚用于商业清洁产品，你可以用香料架上的旧百里香制作自己的百里香——只需将沸水倒在百里香上，浸泡一夜，过滤到喷雾瓶中，加入一些外用酒精。这将是一个非常有效的室内表面清洁剂。

另一种香料是酒石，这种香料在老化后不再有其他用途，但仍可用于清洁。这种香料可以帮你 [去除厨房甚至浴缸里的污渍、划痕和结块残留物](https://www.hgtv.com/lifestyle/clean-and-organize/3-genius-ways-to-clean-with-spray-lubricant) 。将它与一些小苏打和柠檬汁混合成糊状，可以放在浴缸环和其他顽固污渍上。

## 升级您的衣物

香料闻起来很香，所以不难想象用你的旧物品制作一些简单的香料包，然后把它们扔进烘干机。想让你的衣服闻起来像薰衣草吗？不需要花哨的干燥床单，只需用你的剩菜装一个小袋。你可以使用任何像这样的香料，也可以用任何剩余的布料制作香包。

## 热葡萄酒

在寒冷的夜晚，热葡萄酒是一种令人愉快的享受。你所需要的只是一些红酒、糖和一大堆香料(如果你想变得雄心勃勃，也可以加入白兰地或水果 )。喝一杯香辣的好酒大约需要半个小时，而且你可以使用许多不同的香料来创造不同的体验。肉桂和丁香是最受欢迎的选择，但为什么不拿起一盒便宜的红葡萄酒，花一个周末的时间尝试你的旧香料，看看什么有效？

## 空气清新剂

你的房子有一股气味。这是事实。你没有意识到这一点，因为你住在那里，已经变得对此视而不见(谢天谢地)，但你的客人完全意识到了这一点。如果你想控制那种味道，你可以用一些旧的香料来做到这一点。你所要做的就是把一壶水烧开，加入一些香料，让蒸汽发挥作用。你让它煨的时间越长，香味就会扩散得越远，它会持续令人惊讶的长时间。

## 开始一个香料花园

你的香料又旧又破，几乎不能给你的食物调味——但其中一些可能隐藏着生命。毕竟，许多香料都是种子，你也许能从你架子上的香料中 [长出一些真正的植物](https://balconygardenweb.com/herbs-you-can-grow-from-spice-rack/#:~:text=To%20grow%2C%20fill%20potting%20mix,germinate%20in%208%2D10%20days.) 。芫荽、生姜、小豆蔻和芥菜籽只是你可能会哄进新生活的几种香料。

你在这里的里程会有所不同，因为a)这些香料通常以限制其发芽能力的方式加工，b)它们已经很老了。尽管如此，如果你正在考虑扔掉你的香料，把它们放在一些泥土里，并给它们一些TLC可能会产生一个令人愉快的香料花园——然后你就可以永远收获你自己的极其新鲜的香料了！

## 工艺

最后，如果你是一个狡猾的人，香料可以加入到各种DIY项目中来增加一点活力。自己做蜡烛或者肥皂？加入一些旧香料，味道会更好。几个世纪以来，香料一直被用来制作天然染料，所以无论是你自制的衣服还是染一些复活节彩蛋，你的旧香料都能带来颜色。制作一些简单的香包挂在你的衣柜里，可以给你的衣服和你占据的空间带来可爱的香味。
