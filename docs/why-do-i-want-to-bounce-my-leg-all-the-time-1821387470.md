# 为什么我一直想弹腿？

> 原文：<https://lifehacker.com/why-do-i-want-to-bounce-my-leg-all-the-time-1821387470>

我是那种经常搬家的人。每当我不站起来的时候，我就会有一种无法控制的冲动想移动我的腿。它们连续弹跳几个小时，大多数时候我甚至没有意识到我在做这件事。当我这样做的时候，我有时会试着停下来，以防打扰到我周围的人，但这只会让我感到不舒服。我决定找出是什么导致了这一切令人讨厌的反弹，如果有任何方法来阻止它。

### **是什么导致了弹跳**

很多人出于习惯 弹跳腿 [。这可能是由于](https://www.anxietycommunity.com/index.php?threads/bouncing-your-leg-up-and-down.593/) [的躁动，也可能是由于](https://lifehacker.com/improve-your-focus-by-fidgeting-5980542)的专注的辅助甚至是因为压力。基本上，正如临床医学家谢丽尔·哈桑在 Quora 上解释的那样，这通常是一种自我安慰或应对活动，当人们感到焦虑，他们的大脑忙于做其他事情时就会这样做。因此，如果你只是在极少数情况下才发现自己坐立不安，这可能只是为了让自己感觉更好而排出的一点紧张能量。

然而，如果它一直发生在你身上——正如我所说，这是我几乎每次静止时都会做的事情——它更有可能是一种被称为“”(RLS)的轻度障碍。它也被称为威利斯-埃克博姆病。当 RLS 患者不动时，他们的腿(有时是胳膊)会感到不舒服，尤其是在晚上。坐立不安可以缓解这种不适感，所以约翰霍普金斯湾景医疗中心不宁腿综合征专家理查德·p·艾伦医学博士 [解释道](https://newsinhealth.nih.gov/2012/10/cant-curb-urge-move) ，那些 RLS 患者想不停地移动*:*

> *“患有这种疾病的人觉得他们绝对必须移动他们的腿。他们的腿感到不舒服，甚至疼痛，除非他们移动他们。当情况极端时，患有这种疾病的患者可能会坐着——开会、谈话、看电视——并且他们必须不停地移动他们的腿，这可能会对他们自己和其他人造成很大的干扰。”*

*当我坐着的时候，我的腿在弹跳，就像我在为一个高 BPM 的俱乐部成员打鼓一样；当我躺在床上时，我的腿来回移动，就像在床单的海洋中踩水一样。大多数 RLS 是相当温和的，虽然(像我一样)不会影响人们生活的许多方面，但对于那些极端情况下的人来说，它可能会影响他们获得像样的夜间休息的机会。*

*据估计，7%到 10%的美国人 患有 RLS，尽管女性的发病率是男性的两倍，而且这种疾病主要有两种类型。第一种也是最常见的类型是早发型 RLS，在 45 岁之前开始，随着时间的推移逐渐恶化。迟发性 RLS 在 45 岁后发病，突然出现，但不会随着时间的推移而恶化。*

### ***当你有 RLS 时会发生什么***

*RLS 最大的缺点之一是它会导致疲惫和白天嗜睡，根据国家神经疾病和中风研究所 (NINDS)的说法，这可能会强烈影响情绪，注意力，工作和学校表现，以及个人关系。它还会影响你的注意力、记忆力和完成日常任务的能力。事实上，NINDS 表示，未经治疗的 RLS 会导致工作效率下降约 20%，这也可能导致抑郁和焦虑。*

*NINDS 表明，这种疾病有许多潜在的原因，尽管它通常是遗传的，而且没有已知的治疗方法。例如，我知道我的 RLS 是从我母亲那里遗传的，它通常会因为运动量太少(我注意到当我更经常锻炼时，这不是一个问题)、太多咖啡因(我喝太多咖啡)以及可能偶尔缺铁而恶化。幸运的是，对于不断坐立不安的人来说， [有一些小福利，比如每天燃烧几个额外的卡路里](https://www.livestrong.com/article/320945-the-calories-burned-shaking-a-leg/) 。*

### ***如何停止弹跳***

*如果你认为你患有 RLS，并且它正在损害你的睡眠，或者变得疼痛而不仅仅是不舒服，你应该去看医生。没有治愈的方法，但是医学专家可以做很多事情来减轻这种疾病的影响。但是，如果你的 RLS 只是一个轻微的麻烦，你可以采取一些简单的预防措施和其他技巧来帮助控制它。有一件事你不应该做，与你可能在网上发现的相反，那就是服用奎宁或喝奎宁水(含有微量奎宁)。奎宁是一种抗疟疾药物，今年早些时候作为新冠肺炎 的潜在治疗方法而获得了新的关注，但传统观点——以及 [一些医生](https://www.consumerreports.org/cro/2012/04/quinine-for-night-time-leg-cramps-no-longer-recommended/index.htm)——也认为它可以帮助缓解你抽筋的腿部肌肉。 [原来还没有科学证明](https://www.medicalnewstoday.com/articles/323692) 对 [或者小病](https://www.pennmedicine.org/news/news-releases/2020/september/hydroxychloroquine-no-more-effective-than-placebo-in-preventing-covid19)[长期使用奎宁治疗 RLS 与早死](https://www.sleepreviewmag.com/sleep-disorders/movement-disorders/restless-legs-syndrome/long-term-use-quinine-restless-legs-syndrome-linked-increased-risk-death/) 。

会帮什么*？NINDS 建议一些简单的生活方式改变，比如避免或减少酒精或烟草的使用，保持规律的睡眠模式，每周锻炼几次。你也可以尝试腿部伸展运动、热水浴或冷水浴、按摩腿部、热敷或冷敷腿部，甚至将注意力集中在智力挑战性的任务上，比如纵横字谜。** 

*此外，试着围绕你无法控制的坐立不安、颤抖和弹跳的欲望进行计划。当你的症状不太严重时(通常在一天的早些时候)，安排汽车旅行、飞机旅行和电影，给自己足够的步行和伸展休息时间，在剧院、飞机或火车上选择靠走廊的座位，这样你就可以很容易地起身四处走动。*

*最后，如果你想当场抓住自己，并试图阻止你的腿弹跳， [雷蒙德·米尔滕伯格博士建议](https://www.menshealth.com/fitness/why-am-i-always-bouncing-my-leg-when-im) 在你的口袋里放一些零钱或钥匙。一旦你听到叮当声，交叉你的腿，把你的脚绕在椅子上，或者站起来，让冲动过去。*
