# 如果你小时候被“父母化”了，该如何恢复

> 原文：<https://lifehacker.com/how-to-recover-if-you-were-parentified-as-a-child-1847318087>

经历“父母化”的孩子是那些成长过快的孩子，因为他们被期望承担兄弟姐妹或自己父母的角色。一个孩子成为父母有很多原因——有些是可以避免的，有些是不可以的；但不管怎样，亲子关系的影响是迅速而持久的。



如果父母去世、丧失行为能力或搬出去，孩子可能会觉得他们需要帮助他们剩下的父母。其他时候，父母可能没有必要的资源，无论是财政资源还是其他资源，这迫使他们依靠一个年长的孩子来帮助照顾他们的弟弟妹妹。如果父母精神不稳定，正在处理成瘾问题，或者缺乏扮演适当父母角色的情感能力，也可能发生亲子关系。

## 亲子关系的长期影响

不管是不是有意的，是不是可以避免的，亲子关系对一个孩子来说可能是一个非常紧张的情况，并且它通常会有长期的影响。

“亲子关系往往是一个家庭系统中缺乏界限，”弗吉尼亚州米德洛锡安市[prosper works](https://thriveworks.com/midlothian-counseling/)的婚姻和家庭治疗师克里斯汀·迈尔斯说。

父母化的孩子经常被要求承担他们没有准备好的责任，例如照顾年幼的兄弟姐妹或让父母向他们倾诉个人问题。

迈尔斯说:“人们期望孩子做一些他们在认知上可能没有能力也没有准备好的事情，这可能会导致失败、羞愧或自卑等情绪。”

父母养育的孩子可能会发展出持续到成年的问题，如焦虑、抑郁、自卑，以及无法形成自己的健康关系。对于那些从小就被父母教育的人来说，打破这种循环可能非常困难，因为我们大多数人都是通过观察自己的家庭来了解家庭关系的。换句话说，对于一个在那种环境中长大的孩子来说，亲子关系看起来很正常。

“我们倾向于我们所知道的，”迈尔斯说。

即使一个人认识到这种亲子关系不正常或不健康，他们仍然很难理解一段关系中健康的界限是什么样的，以及如何实施。治疗可以有助于解决他们为应对父母的压力而形成的一些思想和行为，以及建立健康的生活习惯。

## **对治疗有什么期望**

处理成为父母子女的影响通常意味着面对与这种经历相关的功能失调的信念和思维模式。例如，许多父母养育的孩子长大后会强烈地感到对他人负有责任，这是一个很难改掉的习惯。治疗通常旨在解开这些想法和行为，以便处理和解决它们。

他们还需要摒弃一定数量的人际关系习惯，比如极度独立，或者不顾自身需求而关心他人的冲动。

迈尔斯说:“父母养育的孩子，随着年龄的增长，往往会在照顾者的角色中找到自己的身份。

对于一个成年人来说，处理父母化孩子的影响，一些疗法可以帮助包括:

### **基于创伤的认知行为疗法**

这是一种 [类型的治疗](https://www.apa.org/ptsd-guideline/treatments/cognitive-behavioral-therapy) ，帮助患者识别和解决功能失调的思维模式和假设，以帮助他们理解他们的经历，以及他们对自己的理解。

对于从小就被父母教育的成年人来说，这可以包括解决他们的信念，即他们对他人的福祉负有唯一的责任，以及他们对自己没有能力承担与年龄不相称的责任的焦虑。

### **叙事疗法**

叙述疗法帮助患者将自己与他们的问题分开，以便让他们与问题保持足够的距离，从而理解它可能对他们造成的伤害。这可以帮助给病人一个新的视角，一个可以帮助他们改变思维模式和信念的视角。

对于从小就被父母养育的成年人来说，这可以帮助他们理清对他人的强烈责任感，以及过度投资于自己作为看护者的身份，以至于忽略了自己的需求。

### **家庭治疗**

由于亲子关系是由于家庭内部缺乏适当的界限而产生的问题，家庭疗法可以帮助打破家庭单位内的不健康模式，以及建立健康的界限。

### **夫妻治疗**

童年时被父母抚养长大的成年人通常很难建立自己的健康关系。夫妻疗法可以通过帮助在一段关系中建立健康的界限，以及帮助改善沟通来解决这些问题。

亲子关系迫使孩子以一种充满压力和不健康的方式过快成长。尽管长期影响可能很困难，但亲子关系并不需要定义一个人。相反，通过对发生的事情的正确认识，以及正确的治疗，这是一个可以打破的循环。
