# 你可能吸尘太快了

> 原文：<https://lifehacker.com/youre-probably-vacuuming-way-too-fast-1849429518>

一般来说，吸尘有两种:快速扫几下捡起面包屑、洒出来的零食、看得见的脏；在打扫的时候对整个房间进行更彻底的清扫。当然，很大程度上取决于空间中地板的种类，当谈到地毯时，越慢越好。事实上，即使你认为你在慢慢来，你还是可能吸尘太快了。以下是需要了解的内容。



## 什么是“慢速吸尘”？

抖音上不乏清洁技巧和前后揭示，包括帖子 的 [子集，展示了](https://www.tiktok.com/discover/moping-and-vacuuming) [“缓慢吸尘”的结果这正是它听起来的样子:用吸尘器清扫地毯比你通常(可能)会做的要慢得多，然后显示出先前空的装满污垢、灰尘和其他碎片的罐子。](https://www.tiktok.com/@maevers/video/7121078132530089259?is_from_webapp=v1&item_id=7121078132530089259&refer=embed&referer_url=https%3A%2F%2Fdengarden.com%2Fnews%2Fslow-vacuum-trick&referer_url=https%3A%2F%2Fdengarden.com%2Fnews%2Fslow-vacuum-trick&referer_video_id=7121078132530089259)

但是缓慢吸尘并不新鲜；事实上，这就是省力装置的用途。早在 20 世纪初，当第一台电动“吸尘机”出现在消费者面前时，它们被设计用来复制(并取代)拍打地毯。

滚筒刷上的刷毛会扬起灰尘，然后被吸入袋子或罐子里。如果你从拍打地毯的角度来考虑，慢慢吸尘是有道理的。你不会把一块地毯拿到外面(或者把它伸出窗外)只拍打几下，然后就收工——如果你想尽可能多地去除灰尘，你会花更多的时间去拍打它。

## 如何正确清洁地毯

虽然现代吸尘器比早期型号更强大，也更容易操作，但总体概念是相同的——即使是更高端的机器。比如这些是 [戴森](https://www.dyson.com/insights/cleaning/how-to-clean-your-carpet) 对吸尘地毯的部分指令:

> 缓慢吸尘给气流和刷条更多的时间来“搅动”地毯纤维之间的灰尘和污垢，并将其完全清除。这也意味着你更有可能捕捉到藏在地毯堆深处的看不见的过敏原。

那么你应该走多慢呢？真的，这取决于你，以及你有多少时间和想要花在吸尘上。建议的范围很广，从使用地毯清洁器的速度与你使用地毯清洁器的速度相同，到每平方英尺 15-20 秒，到“让你的吸尘器在你的地毯上停留的时间越长越好”
