# 如何确保您在完整的4K流

> 原文:[https://life hacker . com/how-to-assure-you-streaming-in-full-4k-1846940547](https://lifehacker.com/how-to-make-sure-youre-streaming-in-full-4k-1846940547)

还记得高清电视吗？满1080p？那些日子已经过去很久了，宝贝！现在4k是你想看什么都能接受的最低清晰度。不然你用眼睛干嘛？

4K也被称为UHD(“超高清晰度”)或2160p(对你的数学天才来说是1080p的两倍)。这意味着您正在观看的屏幕由3，840 x 2，160像素组成。目前，大多数新的电视机和电脑显示器都支持这种分辨率，但你需要采取一些额外的步骤，以确保你正在观看的内容确实是通过所有4K荣耀。

首先，确保你的流媒体平台确实支持4K。例如，HBO Max最近才开始播放4K的内容，如果你使用FireTV Stick，你需要确保你有一个4K型号。网飞会让你升级你的计划来获得完整的UHD体验。

然后你需要确认你想看的节目来自4K。一些老的节目还没有升级，有时你可以选择高清版或4K版。

最后，确保您的所有电缆、连接和端口都支持4K。有些电视只有4K专用的HDMI端口，所以一定要插入相应的端口。快乐流媒体！
