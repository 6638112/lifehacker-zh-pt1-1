# 用阿拉斯加鸡尾酒淘金

> 原文：<https://lifehacker.com/strike-gold-with-the-alaska-cocktail-1849686889>

在杜松子酒或伏特加、橄榄或柠檬水、盐水或清酒、干的或湿的、味精味的或盐水味的之间，在你自己选择的马提尼酒的冒险中不缺少路径。然而，如果你是马提尼变体的完美主义者，那么让我给你介绍这个鸡尾酒巨头的另一个分支:草本马提尼。我想不出比阿拉斯加鸡尾酒更好的介绍了。



阿拉斯加是一种禁酒令前的饮料，在萨伏伊鸡尾酒书中永垂不朽。它的特色是杜松子酒、黄色黄绿色、和通常是橙色苦味酒。虽然它经历的战争和衰退比你我目前可能知道的任何人都要多——而且还是比它周围的其他国家年轻几十年——但它的巧妙提炼确实让我对围绕它的发明的特殊环境感到好奇。

我能告诉你的是阿拉斯加出生于 1910 年左右；在此之前，至少在美国,“阿拉斯加鸡尾酒”是一壶加冰的冷水(我们今天要做的阿拉斯加当然是冰的，但以一种非常不同的方式解渴)。这两者是否曾经有过联系值得怀疑。虽然阿拉斯加冰深受所有酒保的喜爱(毫无疑问，这是当前精酿鸡尾酒热衷于纯冰的原因)，但不确定这是否是命名的因素，因为这种饮料已经上桌了。普遍的共识是，阿拉斯加之所以被命名为阿拉斯加，是因为这个同名的领土在当时越来越受欢迎，而这种饮料的金色色调让人想起了承诺给成群结队定居在那里的拓荒者的贵金属。很狂野，不是吗？所有这些历史只是为了让我，今天，可以将这位草本球员添加到你的马提尼武器库中。

黄色黄绿色确实在这个数字中闪耀，并且(显然)是草本的驱动力。它比它的绿色兄弟姐妹(有更多的香料和口感)更温和，更甜，它与杜松子酒和橙子苦味完美地和谐在一起，并成为一种令人惊讶的顺滑的饮料，这种包装了这样的酒。我认为它非常适合一年中的这个时候:草本植物的色调给了一种药用的联想，它的高度效力一定是在一路上杀死一些季节性的细菌。虽然它可能不是一罐加了冰的山泉水，但不知何故，它仍然令人神清气爽——就像深深地吸了一口清新的山间空气。

下面的配方是更常见的，现代版本，但对于我的鸡尾酒顽固分子来说，把普通杜松子酒换成稍微更深奥的老汤姆杜松子酒(一种更甜的杜松子酒，介于伦敦干和荷兰杰尼弗之间)，你会更接近于 OG。尽情享受吧！

## 阿拉斯加鸡尾酒

*   2 破折号橙色苦味酒
*   3/4 盎司黄色黄绿色
*   2 又 1/4 盎司干杜松子酒
*   柠檬麻花

将原料倒入冰过的混合杯中，加入碎冰，搅拌大约 25-30 秒，或者直到充分冷却(但不要过度稀释)。滤入冰过的鸡尾酒，在鸡尾酒和装饰上表达柠檬捻。
