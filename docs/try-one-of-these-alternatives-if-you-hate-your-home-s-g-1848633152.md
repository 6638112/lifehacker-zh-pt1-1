# 如果你不喜欢你家的排水沟，试试这些替代品

> 原文：<https://lifehacker.com/try-one-of-these-alternatives-if-you-hate-your-home-s-g-1848633152>

排水沟起着重要的作用——它们保护你家的地基和结构，防止水从屋顶落下。但是任何拥有过房子的人都知道传统的水槽有其缺点:它们会被树叶堵塞，在更冷的天气里，它们会造成冰坝。它们还会开裂和下陷，让水破坏你家的结构。如果你正在寻找一个替代品，有一些不错的。



## 链条

雨链 是一种实用的水槽替代品，在日本已经流行了几个世纪。这种方法通过链条从屋顶取水，有时用一系列的钟或杯子。增加的阻力减缓了水流向地面的速度，防止了飞溅和侵蚀——，还产生了瀑布般的声音。雨水链也可以战略性地放置在雨水桶、容器甚至花园床中收集水，这使它们成为水槽的一个伟大、可持续的替代品，甚至可以减少一点水费。

虽然它们很重，但是如果你有一个更大的屋顶，你将需要很多它们来成功地控制所有的水。

## 分阶段

对于降雨率较低的地区，一个便宜且实用的解决方案是分级。这个过程包括塑造你家周围的地面，让水从地基流走。你可以用石头或土壤来获得适当的坡度，但它应该是 1- 英寸比 1- 英尺的角度，以确保水自由流动。这是一个你可以在相对较短的时间内自己完成的项目；然而，如果你住在山坡上或者离地下水位太近，g 打地基是不可行的，因为它不足以保持你的地基干燥。

常规排水沟的另一种选择是法国排水系统。法国排水管在地面上做着基本相同的事情，而不是在屋顶边缘的排水沟。它由一条以每 8 英尺至少 1 英寸的速度倾斜远离房屋的沟渠、一根多孔管道和一些覆盖它的砾石组成。只有安装得当，这种方法才能保持地基干燥，所以你可能需要雇佣专业人士来确保排水充足。安装可能很昂贵，但是一旦安装到位，Fr ench 排水管几乎不需要维护。

## 雨水分散系统

雨水分散系统是一种将雨水分解成小溪甚至水滴的方法。它没有吸收水分，而是将水分从房子里扩散出去。有时被称为无排水沟系统，如果水的路径被规划好了，它工作很好，但是如果没有规划好，它有时会导致水坑和湿气堆积在你的地基上。一些屋顶类型不适合雨水扩散，但如果你有一个倾斜的屋顶和足够的坡度，它将把水从房子里抽走，并消除瀑布水造成的土壤侵蚀。
