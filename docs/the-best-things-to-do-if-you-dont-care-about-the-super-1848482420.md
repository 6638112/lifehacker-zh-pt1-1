# 如果你不关心超级碗，最好做的事情

> 原文：<https://lifehacker.com/the-best-things-to-do-if-you-dont-care-about-the-super-1848482420>

在过去的七年里， [平均有 1.04 亿人](https://www.statista.com/statistics/216526/super-bowl-us-tv-viewership/) 收看了超级碗——可以肯定地说，当周末公羊队迎战孟加拉队时，*很多人也会这样做。但是，当这个国家的其他人都在狂饮淡啤酒并试图让布法罗鸡翅弹片远离他们的球衣时，一个非足球迷能做些什么呢？你可以拿着一个好的遥控器蜷缩起来，尽情享受你最喜欢的 [网飞新发布的](https://lifehacker.com/whats-new-on-netflix-in-february-2022-1848384998) ，当然。但由于近三分之一的美国人另有所图，现在是去通常拥挤的地方的黄金时间，尤其是如果你在疫情期间因为这个原因一直在避开它们的话。以下是利用一亿美国人看电视的最佳方式。*

## 去斜坡

在超级碗周日比赛上，你不仅会看到更短的提升线和更少的人群，许多滑雪场还会提供大幅折扣让你登上他们的斜坡。根据让你提前购买缆车票的网站 [Liftopia](https://www.liftopia.com/) 的说法，比赛日是一年中最适合雕刻一些新鲜粉末的日子之一。Liftopia 的联合创始人兼首席执行官埃文·里斯(Evan Reece)告诉 [商业内幕](https://www.businessinsider.com/why-you-should-ski-on-super-bowl-sunday-2016-1) ，“在超级碗周日，度假村的观众人数历来是其他旺季周日的一半，尤其是在球队比赛的市场。”在我的区域上快速搜索，比赛日门票的折扣从 13%降至 46%。

## 经营一家大型杂货店

我们一直在疫情中穿梭，但为什么不利用周日晚上的休息时间重新认识一下你的老朋友——真正的杂货店——的四面墙呢？我和其他人一样喜欢有人把食物送到我家门口，但是当我自己在过道里漫步时，想起我忘记存在的东西(嘿，玉米坚果！)，得到灵感做些不一样的晚餐，或者犒劳自己一块去角质肥皂或一盒计划外的寿司。和 Costco 或 Target 没有通常的周末人群？光荣。

## 去那家你永远进不去的餐厅吃饭

你有没有考虑过在那家新开的亚洲融合餐厅预定座位，那家餐厅你很想去，但是*从来没有空桌*？这项对全国 36000 家餐馆 的 [研究发现，比赛日的平均收入比典型的周日下降了 19%。特别是在城市地区，繁忙的、受欢迎的新餐馆经常出现，这可能是最终找到座位的理想时间。(不过，要确保他们不提供外卖披萨；根据同一项研究，超级碗是比萨饼店一年中的第一个星期天。)](https://www.womply.com/blog/super-bowl-sunday-boost-bust-local-restaurants-bars/)

## 去电影院旅行

如果你一直因为害怕和一群陌生人坐在拥挤的房间里而犹豫不决，那么现在是你的机会了。你可能会遇到一些顽固分子，但在大多数情况下，你可以预期电影上座率会比平时低。(而且，现在是周日晚上——一般来说，不是看电影的黄金时间。)有什么好玩的吗？嗯，有给连姆·尼森粉丝的 [*【黑光】*](https://www.imdb.com/title/tt14060094/) ， [*尼罗河上的死亡*](https://www.20thcenturystudios.com/movies/death-on-the-nile) 给神秘谋杀爱好者，还有给想知道欧文·威尔逊去过哪里的人的 [*嫁给我*](https://www.marryme.movie/) 。( [*唱 2*](https://www.singmovie.com/) ，[*Encanto*](https://movies.disney.com/encanto)， [*蜘蛛侠：英雄无归*](https://www.spidermannowayhome.movie/) 在很多地区还在播放，供小朋友和漫威的跟班们看。)大多数大型连锁影院都在网上提供预订座位，所以你在购买之前就会知道影院会有多拥挤。运气好的话，这就像是去电影院看周中的日场电影。

## 去游乐园

大多数孩子痴迷于室内水上公园(不是足球),他们会抓住机会去参观 Kahalari 度假村或大灰狼旅馆，而不是在大人对着电视大喊大叫时被赶走。(给你当地的度假村打电话，询问他们通常在超级碗周日的出席情况。)

取决于你住在哪里，天气可能仍然太冷，公园不能开放。但是，如果你生活在温暖的地方，你当地的六旗可能会有较轻的人群，因为许多人会在赛前或准备举办。(六旗魔山有个很有帮助的 [“打包好了吗？”追踪者](https://www.isitpacked.com/live-crowd-trackers/six-flags-magic-mountain/)T3。)人们有理由认为迪士尼世界的游客数量会有所下降，但实际上这是当地年票持有者的黄金时间，所以跳过这个地球上最快乐的地方。
