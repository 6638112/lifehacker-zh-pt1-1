# 试试这种冷冻苹果酱，而不是昂贵的狗冰淇淋

狗喜欢零食，人类喜欢给狗零食，因为我们喜欢看到我们的狗开心和兴奋。然而，并不是所有的食物都是平等的。有些食物，比如含有大量盐和脂肪的剩菜，对你的狗狗没有好处；还有一些，像那些小狗专用的“冰淇淋”杯，非常贵。冷冻苹果酱不是这两样东西。

冷冻的、不加糖的苹果酱是一种性价比很高的零食，比不冷冻的零食更能让你的狗保持忙碌。大多数狗——也总有例外——不得不慢慢享用冷冻食物,而不是几口就把东西吞下去。再加上天气凉爽清爽，这是夏天的美好对于一个浑身是毛的生物不出汗。

你可以买一个大罐子，把它装进一个 [大岩石冰块托盘](https://www.totalwine.com/accessories-more/accessories/bar-tools/tovolo-king-cube-tray-red/p/176567920) ，或者你可以买一包单个杯子，把它们放进冰盒里，做一个完美的冰淇淋式展示。我喜欢小杯子——我发现容器有助于减缓消耗速度——但要留意你的小狗，确保它们不会太兴奋而吃掉塑料。(如果这是一个问题或担忧，你可以在杯子后面放一些温水，取出冷冻苹果酱，或者做一些小方块，放在他们的食物碗里。)

无论你如何端上，确保你选择的酱料是不加糖的——配料表应该包含苹果、水，可能还有一点抗坏血酸(维生素C)——并且*永远不要*给你的狗吃任何含有人工甜味剂的东西，这会导致危及生命的血糖下降和肾衰竭。还要记住这些是*的零食。*(你的狗可能非常乐意吃一顿100%冷冻苹果酱，但这可能会让它们的胃有点不舒服。)
