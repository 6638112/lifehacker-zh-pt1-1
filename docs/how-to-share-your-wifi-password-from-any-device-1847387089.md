# 如何从任何设备共享您的 Wifi 密码

> 原文：<https://lifehacker.com/how-to-share-your-wifi-password-from-any-device-1847387089>

每次有访客来你家，他们可能会先问你 wifi 密码。你可以走当地咖啡店的路线，把 wifi 名称和密码挂在墙上， 或手动分享这些凭证，直到你记住它们。或者，您可以使用任何设备或平台直接分享您的登录信息，从而大大加快这个过程。我们会告诉你如何做到这一点。

## 如何在 iPhone 和 iPad 上共享 wifi 密码

如果你和拥有其他苹果设备的人分享密码，那就没什么可做的了。只需确保你的 iPhone 或 iPad 更新到最新版本的 iOS 或 iPadOS。另外，请检查您是否已经在设备上登录了 iCloud。(你可以去**设置**做；如果你看到你的名字在顶部，你就可以了。)最后的准备步骤需要你确保对方的 Apple ID 保存到你的通讯录里。

完成后，解锁你的 iPhone 或 iPad，它们应该已经连接到你的本地网络了。A 让你的朋友在他们的苹果设备上选择正确的 wifi 网络名称。你现在会在*你的* iPhone 或 iPad 上看到一个弹出窗口，询问你是否想与你的朋友分享你的 wifi 密码。轻按**共享密码**，一切就绪。

只要您连接到您尝试共享的 wifi 网络，并且您朋友的设备在您的 iPhone 或 iPad 附近，此操作就会顺利进行。

如果你试图与*没有*苹果设备的人分享你的 wifi 密码，你将不得不求助于更多的手动手段。

## 如何在 Mac 上共享 wifi 密码

从你的 Mac 上分享 wifi 密码是非常容易的，只要你分享到其他苹果设备上。准备步骤与 iPhone 和 iPad 上提到的步骤相同——登录 iCloud，确保与你共享信息的人已经将更新到最新版本的 iOS 或 iPadOS ，并将你朋友的 Apple IDs 添加到联系人应用程序中。

一旦这些事情都处理好了，就连上 wifi 网络，让你的朋友们也来试试同样的事情。您的 Mac 将显示一个弹出窗口，询问您是否要共享 wifi 密码。点击**分享**。您朋友的设备将自动连接到 wifi 网络。

一个警告:这个功能只有在你的 Mac 上安装了 macOS High Sierra (10.13)或更新版本的情况下才有效。你可以通过点击 Mac 桌面左上角的苹果标志并选择**关于这台 Mac** 来检查这一点。

对于非 Apple 设备，如果您使用的是 Mac，则必须手动共享保存的密码。要在 Mac 上查找保存的 wifi 密码，您可以打开**钥匙串访问**应用程序(预装在您的 Mac 上)。在左侧窗格中选择 **iCloud** 标签，然后点击右侧的**密码**，就在“钥匙串访问”的下方。您可以使用右上角的搜索框来搜索您的 wifi 网络名称。

找到 wifi 网络后，**右键点击**并选择**复制密码到剪贴板**。然后，您可以快速将它发送给朋友。

## 如何分享安卓系统的 wifi 密码

只要你有一部运行 Android 10 或更高版本的 Android 手机，你就可以生成一个二维码来快速共享你的 wifi 网络密码。只需进入**设置>网络&互联网> Wifi** ，点击你所连接的 Wifi 网络名称旁边的**齿轮图标**。点击**分享**，它也有一个二维码图标，输入你的密码，你会在屏幕上看到一个二维码。

让您的朋友在他们的手机上扫描此代码以连接到您的 wifi 网络。大多数 Android 手机都有谷歌镜头应用程序，可以扫描二维码并快速打开。如果有 iPhone，可以打开相机 app，扫描二维码连接。

## 如何从 Windows 共享 wifi 密码

遗憾的是，Windows 10 没有内置的方式来快速共享 wifi 密码。你最好的办法是使用一个其他设备快速分享你的 wifi 密码。要在 Windows 10 上找到保存的 wifi 密码，进入**设置>网络&互联网>网络和共享中心**。点击**连接:网络名称**和选择**无线属性**。

最后一步是点击**安全**标签，并检查**网络安全密钥**下的**显示字符**框。你可以在电脑上看到你的 wifi 密码，这样你就可以手动与那些想要使用你的互联网连接的人分享密码。
