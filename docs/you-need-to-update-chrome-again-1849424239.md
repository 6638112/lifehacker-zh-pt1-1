# 你需要再次更新Chrome

> 原文:[https://life hacker . com/you-need-to-update-chrome-again-1849424239](https://lifehacker.com/you-need-to-update-chrome-again-1849424239)

如果你最近更新了谷歌Chrome浏览器到第104版 ，你可能会惊讶地发现你的浏览器已经有了另一个更新。毕竟上次更新修补了27个安全漏洞:还剩下什么需要更新的？显然，相当多，包括一个新的安全漏洞，黑客已经知道如何利用。

Watch

谷歌在8月16日星期二发布的博客文章 中宣布了这一更新。这个新的Chrome版本在Mac和Linux上是104.0.5112.101，在Windows上是104.0.5112.102/101，现在可以在所有平台上使用。

该补丁包括对11个安全漏洞的修复，其中一个标记为严重，六个标记为高严重性，三个标记为中等严重性。然而，真实的故事涉及h 高严重性漏洞之一，被标识为CVE-2022-2856:谷歌确认该漏洞存在于野外，使其成为零日漏洞。

零日是危险的。虽然大多数安全漏洞在补丁发布前从未被利用过，但有些却被利用了。当一个人不仅成功地发现了软件中的缺陷，而且发现了如何利用它来对付其他人时，这个漏洞就变成了零日漏洞——CVE-2022-2856就是这样一个漏洞。

该漏洞源于“对意图中不可信输入的验证不充分” [根据哔哔声计算机](https://www.bleepingcomputer.com/news/security/google-fixes-fifth-chrome-zero-day-bug-exploited-this-year/) 的说法，这种类型的漏洞会导致“缓冲区溢出、目录遍历、SQL注入、跨站脚本、空字节注入等”问题这是一个很长的列表，可能会危及你的系统，因为它在野外有一个漏洞，更新Chrome应该是一个优先事项。

然而，不仅仅是这个零日应该说服你更新:其他10个问题仍然很重要，因为它们的身份现在是已知的。黑客仍然可以找到利用这些漏洞的方法，因此全面更新以保护您自己非常重要。

## 如何更新谷歌浏览器

无论你使用的是Mac、Windows还是Linux，你都可以快速更新Chrome，不仅可以修补这个零日漏洞，还可以修补其他10个缺陷。点击浏览器窗口右上角的三个点，然后进入到**帮助>关于谷歌浏览器**。允许Chrome寻找新的更新。如果有可用的，你可以点击“重新启动”来安装它。

如果你启用了自动更新，你可以等待Chrome自动安装更新。然而，这可能需要几周时间——保护你的浏览器的最快方法是自己更新Chrome。
