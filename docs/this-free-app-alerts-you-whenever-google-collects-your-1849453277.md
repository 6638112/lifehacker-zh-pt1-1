# 这个免费的应用程序会在谷歌收集你的浏览数据时提醒你

> 原文：<https://lifehacker.com/this-free-app-alerts-you-whenever-google-collects-your-1849453277>

你猜谷歌多久收集一次你的浏览数据？我们都知道，每当我们使用谷歌的搜索、登录谷歌的应用程序，或者在的 YouTube 上专区时，这个科技巨头都会窃取一些信息，但你可能会惊讶地发现(或者不会，鉴于该公司的声誉)，几乎每次点击和按键都会被发送到谷歌(至少在某些网站上)。如果你使用谷歌的应用程序或在 Chrome 浏览器上浏览，数据共享会更加激进。

Watch

如果这听起来太夸张，甚至对谷歌来说也是如此，你可以使用开发者 Bert Hubert 开发的名为 GoogleTeller 的新免费应用程序来验证一下。

每次你的数据被发送到谷歌的一个公共 ip 地址时，谷歌 Teller 都会发出一声短促的“哔哔”声。目前， [应用只在 Linux 桌面](https://github.com/berthubert/googerteller) 上运行，但是 Hubert 发布了一个演示该软件的视频，所以你可以自己看看结果。

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1561466204602220544&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1561466204602220544&autosize=1) 

你听到的那些小哔哔声不是休伯特的键盘或鼠标；是 GoogleTeller 记录了所有与 Google 服务器的连接。一个不起眼的荷兰政府网站上几乎持续不断的嘟嘟声——这个国家的数据追踪法律比美国 s 更严格——显示了该公司的追踪活动到底有多激进。我想象如果你在 Chrome 上使用谷歌搜索，或者更糟，在 Pixel 手机或 Chromebook 上使用，听起来会是什么样子。

虽然 GoogleTeller 目前的 Linux 排他性意味着 Windows 和 Mac 用户无法直接检查谷歌的跟踪，但主要的好处仍然是有用的:谷歌收集你的数据的很多。一个*批次*。你不一定需要一个应用程序来告诉你，但即使如此，你可能会发现结果令人沮丧。

这是否意味着你的隐私处于危险之中？谷歌主要利用这些数据来诊断漏洞，改善服务，并且很有可能从广告合作伙伴那里赚钱。广告做法是肮脏的，这种积极的跟踪是入侵性的，但真正的问题来自外部威胁，如果没有适当的保护，这些威胁可以拦截和泄露用户数据——这是任何收集用户数据的公司所带来的风险。

好消息是，你基本上可以躲过谷歌的追踪器。 [有浏览器](https://lifehacker.com/6-of-the-best-internet-browsers-for-protecting-your-pri-1847604085) 和 [浏览器扩展](https://lifehacker.com/11-of-the-best-free-extensions-for-google-chrome-1847860706/slides/2) 具有更强的隐私控制，可以阻止谷歌和任何其他公司窥探你的浏览数据。你的谷歌账户中也有用户控件 [限制你与谷歌分享的内容，甚至自动- 删除它存储在你身上的数据](https://lifehacker.com/how-to-automatically-delete-your-google-data-and-why-y-1839614022) 。显然，避免谷歌追踪器的唯一万无一失的方法是删除你的谷歌账户，并完全停止使用该公司的产品。
