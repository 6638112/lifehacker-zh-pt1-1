# 强生疫苗有什么不同？

> 原文:[https://life hacker . com/whats-different-about-the-Johnson-Johnson-vaccine-1846162380](https://lifehacker.com/whats-different-about-the-johnson-johnson-vaccine-1846162380)

强生 [今天发布了来自其](https://www.prnewswire.com/news-releases/johnson--johnson-announces-single-shot-janssen-covid-19-vaccine-candidate-met-primary-endpoints-in-interim-analysis-of-its-phase-3-ensemble-trial-301218035.html) [三期](https://vitals.lifehacker.com/how-do-clinical-trials-work-1842827040) 临床试验(申请紧急使用授权前的最后一步)的数字 ，看起来我们可能很快就会在美国拥有第三种可行的新冠肺炎疫苗。

这种疫苗和其他两种有点不同。来自 Pfizer/BioNTech和Moderna的都是 [mRNA疫苗](https://lifehacker.com/how-mrna-vaccines-work-1845895792) 、需要两剂，需要在冷冻箱温度下保存或在特殊的超冷包装中保存。强生疫苗可以储存在冰箱温度下，就像你在药房或医生办公室得到的大多数其他疫苗一样。而且只需要一剂。

新闻稿称，新的疫苗被发现对预防严重的新冠肺炎病有85%的有效性，在美国，对预防“中度至重度”疾病有72%的有效性。在其他国家，有效性似乎较低，在南非降至56%的有效性，在南非，最常见的COVID 菌株是关于变体 的。目前还不清楚这种差异是由于变异还是其他因素或多种因素的结合。

如果你想知道“中度”和“重度”新冠肺炎的区别，新闻稿中包括了这些定义:

> 在这项研究中，严重新冠肺炎病的定义包括实验室确认的新型冠状病毒和以下一项或多项:与严重全身疾病一致的体征，入住重症监护病房，呼吸衰竭，休克，器官衰竭或死亡，以及其他因素。中度新冠肺炎病被定义为实验室确认的新型冠状病毒和以下一种或多种:肺炎、深静脉血栓形成、呼吸急促或异常血氧饱和度高于93%、异常呼吸率(≥20)的证据；或两种或两种以上提示新冠肺炎的全身症状。

在接种疫苗的人群中，没有人在第一次也是唯一一次接种28天后因COVID而住院或死亡。

即使这种疫苗被证明总体上不如mRNA疫苗有效，但它只需要一剂并且可以在冰箱温度下分发的事实可能意味着它可以覆盖更多的人——并且比那些疫苗更快地覆盖T2和T4。
