# 如何使用你的 iPhone 作为官方 ID

> 原文：<https://lifehacker.com/how-to-use-your-iphone-as-an-official-id-1847597651>

在这个数字时代，*为什么*我们还有实体身份证？能用 Apple Pay 的时候不需要信用卡；有了捷运，我不再需要我的地铁卡了；但是，我仍然需要带着 [和整个钱包](https://lifehacker.com/fuck-wallets-1847565314) 去任何需要驾照的地方。幸运的是，随着 iOS 15 的发布，这种情况似乎正在发生变化，因为苹果公司正在为你的钱包应用程序添加官方 ID。

Watch

新功能与 iOS 15 捆绑在一起，所以还没有上线。但是我们已经对它有了相当多的了解；苹果会让你把驾照或者州 ID 加到钱包里，只要你所在的州支持这个功能。添加后，您可以在通过参与机场的新安检通道时使用它。虽然这是最初的用例，但我们希望看到这种钱包中的 ID 功能在未来的更多形式中被接受，例如在购买酒类或其他年龄限制的商品时。

## 如何将您的 ID 添加到 iPhone 的钱包应用程序中

一旦 iOS 15 出来了，钱包里的 ID 也活了，设置起来就相对简单了。向应用程序添加其他类型的卡的初始过程是相同的；只需轻按 Wallet 主屏幕顶部的(+)按钮，然后按照说明添加许可证或 ID。如果您有 Apple Watch，系统还会提示您将 ID 添加到手表中。

接下来 iOS 会要求扫描你的 ID，要求自拍。根据苹果公司的说法，你的 iPhone 会安全地将自拍发送到你所在的州，以验证你的身份，并且作为一个额外的安全层，会让你记录一系列的面部和头部运动。一旦你所在的州验证了你的身份，你的 ID 将被添加到钱包应用中。

## 如何在钱包中使用您的 ID

虽然 ID 在钱包中肯定有许多潜在的用途，但苹果目前的主要想法是在通过机场安检时向 TSA 出示您的许可证或 ID。你所需要做的就是在身份识别器上点击你的 iPhone。当你这样做时，你会看到一个提示，显示美国运输安全管理局要求的信息。这些信息只有在你向 Face ID 或触控 ID 证明自己的身份后才会透露给美国运输安全管理局。

苹果公司坚称这项功能是安全的；您的所有识别信息都在您的设备上加密，因此第三方无法访问。当您与 TSA 身份读取器共享该信息时，该过程也是加密的，因此数据在传输过程中受到保护。

## 允许你在钱包中使用 ID 的州

此时，钱包中的 ID 尚未激活，因为它是 iOS 15 独占的。该功能何时上线还没有时间表；苹果尚未证实这是 iOS 15 的首日功能，还是需要一些时间。在那之前，苹果确认亚利桑那州和乔治亚州将是美国第一批选择加入钱包 ID 的州。

但是，其他状态也得到确认；他们只是在亚利桑那州和乔治亚州之后的某个时候才会利用这个特性。它们是:

*   康涅狄格州
*   艾奥瓦州
*   肯塔基
*   马里兰州
*   俄克拉何马州
*   犹他州
