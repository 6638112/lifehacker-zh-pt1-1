# 使用 Strava 的热图寻找跑步、骑自行车或游泳的新地方

> 原文：<https://lifehacker.com/use-stravas-heatmap-to-find-new-places-to-run-bike-or-1846789275>

[Strava 的热图](https://www.strava.com/heatmap) 并不新鲜，但直到最近我*才真正*看了一下。这个交互式工具显示了 Strava 用户在哪里跑步、骑自行车、游泳、划独木舟、滑雪和穿雪鞋——当你浏览地图时，你可以将这些活动分开。想寻找秘密的游泳洞吗？查一下哪些高中有对公众开放的赛道？热图会告诉你。



你不必自己使用 Strava 来浏览热图，但你需要一个免费的 Strava 帐户来放大到街道级别。

### 关于隐私的注意事项

热图上的数据来自 Strava 用户追踪的路线。因此，如果你自己使用 Strava，这是对你的提醒，你可能希望 [使用隐私区域功能来排除你的房子](https://vitals.lifehacker.com/how-to-keep-strava-from-telling-everyone-where-you-live-1822516959) 或任何其他你不希望你的跑步被跟踪的区域。

### 寻找安全路线

我住在一个多山的地区，那里的许多道路都很拥挤，缺少人行道。我主要选择熟悉的街区和公园小路，因为我知道在上面跑步或骑车会很安全。但是热图给了我一种方法来找出我可能想尝试的道路，以及哪些道路可能不太安全。

例如，我所在地区的一条主要道路在热图上显示为黑色——这是一个好迹象，表明它没有可供你骑车或慢跑的路肩。(开过这条路，是真的。)但在寻找明亮的区域时，我看到了几组值得一看的道路和小径。例如，有一个墓地，里面有一堆很受跑步者欢迎的小路，还有几个小社区，看起来像是在繁忙的道路之间轻松慢跑的小绿洲。

### 寻找新的路径

许多公园、学校和其他地方都有鲜为人知的小径，如果你没有去过那里，你可能不会知道。不过，通过热图，我可以很容易地看到几条受欢迎的公园小路被延伸到树林中的蜿蜒的线条所环绕。有时，你还会看到你可以在一个地方跑*的路线标志，为提供了一种通过在附近的道路上慢跑来增加几英里到你的赛道锻炼的方法。*

浏览地图找到可能的选项、，然后查看公园网站或张贴在该地区的标志，了解更多关于小径的信息、[以及如何沿着它们走](https://vitals.lifehacker.com/can-you-read-these-trail-blazes-and-avoid-getting-lost-1783282493)、。

### 寻找椭圆形

高中通常有向当地居民开放的跑道；一些中学、社区中心和其他你可能没有意识到的地方也是如此。将热图的过滤器设置为小鞋图标(显示跑步、散步和徒步旅行),并寻找椭圆形。当你找到一个，使用“标签”层找出学校或位置的名称。将图层切换到自行车，您甚至可能会发现一些自行车道。

像以前一样，在这个地方的网站上查看时间和规则，或者寻找该地区张贴的标志。热图给你的是线索，而不是许可。

### 找到游泳池和游泳池

还有一个带水图标的图层，这对于找到人们通常游泳或从事其他水上活动的地方很方便。一个大型公共游泳池展示了几条横过游泳池的水平轨迹，代表了自行车道的位置。(我去过；那正是单圈车道的位置。)

你也可以用这个功能找到最受欢迎的游泳海滩，如果你在内陆，你可以用它找到人们喜欢游泳的小湖或池塘。

请注意，水图标代表游泳*和其他水上运动，如桨板运动和皮划艇运动。一个当地的湖泊在热图上闪闪发光，但我知道那里不允许游泳，我也从未见过有人这样做。不过，T2 非常受皮划艇运动的欢迎，湖上有两个出租船只的地方。所以交叉参考本地业务是个好主意；如果你想找一个地方带上你的皮划艇或桨板，它们的出现将是一个很好的暗示。*

### 寻找滑雪或穿雪鞋的地方

还有一个雪花层，显示了人们喜欢做冬季运动的小径，比如滑雪和穿雪鞋。同样，热图没有区分这两者，所以你自己做研究，看看你是否真的*在*发现了一条秘密的越野滑雪道。(我还注意到一些活动肯定被错误分类了，比如游泳/水层上的一条路线肯定位于道路上。)但是浏览地图可以让你在计划下一次冒险时知道从哪里开始。
