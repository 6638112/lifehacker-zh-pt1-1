# 停止杀死室内植物(转而创造一个自我维持的生态圈)

> 原文：<https://lifehacker.com/stop-killing-houseplants-create-a-self-sustaining-ecos-1848018886>

尽管我会努力让我的室内植物存活，但我不是一个天生的植物妈妈。如果你也同样无能，但仍想在家中保留一小块自然，让我向你展示:光荣的生态圈。



你的科学展的日子可能已经离你很远了，但是正如抖音上的[#生态圈](https://www.tiktok.com/tag/ecosphere) 标签所证明的，这些自我维持的小世界可以成为照顾盆栽植物的一个美丽的替代选择。除了视觉上的吸引力之外，创造你自己的生态圈的最大卖点是它能保持活力。一旦你把它封起来，就不会有任何混乱和维护。扔进去一些微小的生物(读:蜗牛！)喜欢独来独往的人，而你已经提升了你的室内植物游戏，却没有额外的花费。

有很多现成的生态球可以购买，但下面我们将带你通过简单的手工方法使用当地环境的材料。所以，今天就召集一些孩子(最好是你自己的)或者给自己一次单独的科学冒险，为你的家创造一个生态圈。

## 什么是生态圈？

生态圈基本上是一个封闭的生态系统(像地球一样)，但规模要小得多。就我们的目的而言，这是一个自给自足、自我维持的迷你世界，你可以把它放在桌子上的罐子里。

你可以了解 [它们如何在这里工作](https://eco-sphere.com/care/) 的科学，但你需要知道的是，当它们被正确设置时，一个生态圈自身的植物、沉积物和动物提供了环境生存所需的一切。它们不需要喂养、过滤或除了阳光之外的任何东西来茁壮成长，它们可以持续几年。

生态球也很时髦。一整年，我都收到针对封闭生态系统的定向广告 [你可以购买](https://eco-sphere.com/about-the-ecosphere-closed-ecosystem/) 并直接送货上门，但就个人而言，我更感兴趣的是自己动手制作的成就感。但是，如果你在寻找一种有保证的美感，网上就不缺乏生态圈(尤其是在 Etsy 这样的手工艺网站上)。

## 制作自己生态圈的(简要)指南

为了尽可能创造最好的生态圈，我建议求助于视频示例( [，就像这个](https://www.youtube.com/watch?v=KR2Lo0rOF7g) )或者深入的博客帖子( [，就像这个](https://zenaquaria.com/diy-ecosphere/) )来获得逐步的提示。与此同时，这里有一个什么是商店的简要概述。

### **你需要什么**

制作自己的生态圈相对来说既便宜又容易。一旦你确定了你家附近的天然水源，这里是你需要的其他材料。

*   透明、可密封的容器。泥瓦罐、空玻璃瓶，甚至圣诞饰品都是受欢迎的选择。你希望能够观察你的生态圈内的生命，所以选择正确的容器是至关重要的第一步。
*   一个大碗、勺子或长柄勺把水舀进你的透明容器。
*   一个像火鸡涂油器一样的抽吸装置，为你的生态系统收集有机物质。
*   收集蜗牛等小生物的小网(可选，但强烈建议)。

### **你要做什么**

*   参观当地有植被的小溪、溪流或池塘。把你的生态圈想象成一片已经繁荣的环境。你需要收集已经生活在一起的材料，所以从一个地方获得所有的资源。
*   向容器中加入水、沉淀物和土壤。
*   在你的容器中加入一些来自水源的植物。
*   去找一些小蜗牛或小甲壳类动物来补充(它们可能藏在水中的树叶下)。
*   把它封起来，把你的生态圈种在阳光充足的地方。
*   坐下来，看生命在你眼前绽放。

## **最终提醒**

你的生态圈一开始会很模糊，但几天后就会稳定下来。藻类通常会在土壤和岩石中生长，但要记住，大型动物和昆虫无法在这样的封闭生态系统中生存。最后，你的生态圈需要的只是一点阳光——最好是间接的。
