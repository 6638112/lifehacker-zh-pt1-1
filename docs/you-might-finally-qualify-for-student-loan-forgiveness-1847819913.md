# 你可能最终有资格获得学生贷款豁免

多年来，公共服务贷款免除(PSLF)项目一直在努力免除学生贷款债务。B ut昨天，拜登政府 [宣布变革](https://www.ed.gov/news/press-releases/us-department-education-announces-transformational-changes-public-service-loan-forgiveness-program-will-put-over-550000-public-service-workers-closer-loan-forgiveness) 将立即免除大约22，000名借款人的学生贷款债务——和将帮助数十万人比预期更快获得贷款豁免。以下是这些变化可能会如何影响你的学生贷款。

## **放宽对PSLF项目的要求**

2007年，国会创立了联邦学生贷款豁免计划，鼓励人们在联邦和地方政府、公立学校和军队工作。作为约10年全职服务(或120笔学生贷款付款)的交换，该项目免除了联邦贷款的剩余余额——或者至少，这是它应该如何运作的。然而，根据一份 [2019年政府问责局(GAO)报告](https://www.gao.gov/products/gao-19-717t) ，这种情况很少发生，因为它发现只有1%的申请人被认为有资格获得学生贷款豁免。

部分问题是，该计划只适用于直接贷款的借款人，直接贷款是由联邦政府管理的一种特殊类型的学生贷款。一种选择是将现有的学生贷款合并为直接贷款，但这只会进一步阻碍借款人，因为以前的其他贷款付款不计入免除未偿余额所需的120笔付款。

为此，教育部现在提供有限豁免，“允许学生借款人的所有付款计入PSLF，无论贷款计划或付款计划如何。”所有目前拥有FFEL、珀金斯或其他非直接贷款的借款人，如果他们在2022年10月31日之前申请合并到直接贷款计划并提交PSLF表格，就可以获得这一有限豁免的好处。此外，军队成员将能够计算延期和宽容，而现役对贷款的豁免。教育部估计，这将帮助550，000名学生贷款借款人比预期更快地获得贷款豁免。

## **如何申请PSLF豁免**

教育部表示，将联系借款人，鼓励他们在2022年10月31日前合并贷款或申请该项目。要申请豁免，首先要访问StudentAid.gov/PSLFWaiver的。
