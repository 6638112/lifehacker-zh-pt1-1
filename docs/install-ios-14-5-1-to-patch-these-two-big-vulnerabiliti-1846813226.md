# 安装 iOS 14.5.1 来修补这两大漏洞

> 原文：<https://lifehacker.com/install-ios-14-5-1-to-patch-these-two-big-vulnerabiliti-1846813226>

即使你刚刚安装了 iOS 14.5——你*正在*使用新的操作系统，对吗？—是时候回到设置应用程序，从苹果公司获得一个刚刚发布的更新，该更新修补了许多重要的 iOS 漏洞。虽然*所有的*安全更新都很关键，但这个更新修复了两个正在被大肆利用的漏洞。



换句话说，不要等待。你会忍不住这么做，因为它看起来像是一个小更新，而且你现在需要用你的 iPhone 或 iPad 做其他事情，所以*也许以后*。如果是你，确保你已经在**设置>常规>软件更新**中打开了自动更新，但你真的应该花几分钟时间下载并安装更新。

正如苹果在 [其安全说明](https://support.apple.com/en-us/HT212336) 中描述的那样，iOS 14.5.1 修复了 WebKit 的两个问题，如果你访问一个旨在利用这些漏洞的网站，攻击者可能会远程在你的设备上执行代码。至于苹果修复的问题的完整解释:

> 改进的状态管理解决了内存损坏问题。*   The integer overflow problem is solved by improving input verification.

尽管你很有可能*不会遭遇这种攻击，但没有理由等着修补你的 iOS 版本。更新都是 100MB 左右，通过 wifi 下载应该不会花很长时间。(我不会通过手机连接来浪费我的数据。)事实上，在我写这一段的时候，我的 iPhone 已经下载并开始准备更新了——后一个过程大约需要五分钟左右才能在我的 iPhone 12 Pro Max 上完成。当然，你个人的结果可能会有所不同。*

下载并处理后，你需要点击**“立即安装”**来安装更新。你的 iPhone 或 iPad 会自动重启，再过 5-7 分钟你就可以使用了。

总的来说，我认为整个更新过程花了我大约 15 分钟，这包括下载，准备和安装更新。当然，这比我预期的时间要多一点，但是试着在你一天中的某个时间做这件事。当你在开工作会议、上厕所或热晚餐时，让更新持续进行。
