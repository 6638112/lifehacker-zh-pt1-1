# 如何将快速便餐冒充为自制的

> 原文：<https://thetakeout.com/how-to-pass-off-fast-casual-meals-as-homemade-1849034322>

为别人做饭并不总是那么容易。如果你想给他们留下深刻印象，你可能会在细节上绞尽脑汁。但是，如果你突然觉得很懒，或者答应别人做一顿家常菜后时间不够了，你会怎么做？很明显，你从餐馆买食物并撒谎。

## 乱搞温迪的辣椒

温迪的辣椒本身，并不特别令人印象深刻。这是一种由碎汉堡肉、豆类和番茄汤混合而成的稀炖菜，味道并不好。但它确实有一种无害的自制感，就好像你把一些牛肉和一堆罐装狗屎混在一起。我的建议是你排出一些多余的液体，撒上一些孜然和辣椒，让它尝起来像是你做的(谢谢，温迪！)，再浇在一些通心粉面上。在上面放一大堆碎奶酪和一些辣酱，你的用餐伙伴就不会知道了。

## 宣称熊猫快餐宫保鸡丁的所有权

这一个将涉及一些心理策略。去熊猫快递拿一堆宫保鸡丁和白米饭。(如果你得到了橘子鸡，并试图把它当作自己的，祝你好运，因为那是一个错误的决定。宫保鸡丁是你可以在家做的东西。

如果你有一个炒锅，巧妙地在里面放一层薄薄的油，也许还可以加入一些宫保鸡丁，让它看起来像是你刚刚在烹饪。(深而浅的平底锅是你的第二选择。)然后，拿一块菜板和一把菜刀，把它放在你的客人能看到的水槽里。如果你有一个电饭煲，把它插在墙上，放在一个能清楚表明你最近“使用”过它的地方。

恭喜你，你刚刚做了宫保鸡丁！

## 关于自制意大利面撒谎对橄榄园来说很容易

橄榄园的食物尝起来都像是从袋子里、罐子里或微波炉里出来的，所以还有什么比它的意大利面更容易被冒充成你自己的呢？小心选择你的战斗——选择一些非常简单的东西，比如红肉酱意大利面，你自己做是可行的。拿的时候要面和酱分开要。将两者混合在一起，从技术上讲，你*做到了*。

如果你不小心端上了面包棒或他们的招牌沙拉(我在一英里外就能认出那种调料)，游戏就结束了。得了吧，你知道得更清楚。

### 奶酪蛋糕工厂的沙拉是你最好的选择

奶酪蛋糕工厂是一个非常有趣的地方。你知道他们从零开始在内部制作 [所有东西，除了他们的芝士蛋糕](https://thetakeout.com/what-the-cheesecake-factory-keeps-quiet-about-its-menu-1848918579) 吗？买一份他们的科布沙拉或凯撒沙拉，把它扔进一个大碗里，作为一条红鲱鱼，把一瓶你从冰箱门上摘下来的开瓶调料放在它旁边。你看，这是一个智力游戏。你的客人可以看出你显然非常努力地自己切菜，但是通过展示你的藏谷牧场 瓶 [，你也合理地使用了你手头已经有的餐具室配料。](https://thetakeout.com/hidden-valley-ranch-bottled-vs-dry-mix-seasoning-1849015737)

## 吉米·约翰的饼干

作为甜点，为你的客人提供一块吉米·约翰店的饼干。所有的品种都出奇的好，尽管我觉得燕麦葡萄干可能是你能提供的最美味的版本。但这里有一个交易:你不能只是拿出一块饼干，然后告诉你的客人这是你做的。在你朋友来之前去吉米·约翰家，买一打同样的饼干。然后，拿出你最大、最破旧的特百惠容器，你可以用它来储存饼干，在拿出来之前小心地把它们放进去。也许打破一个或两个，你知道，为了现实的缘故。

哦，别忘了把该死的包装纸藏在垃圾桶里。如果你不打算做实际的工作，那么你会想要掩盖你的踪迹。
