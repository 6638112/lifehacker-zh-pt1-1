“秘密”碳危机正在演变。经过多年的低调使用，与不可替代代币(NFT)相关的艺术品和收藏品已经作为下一个大事件爆发到全球话语中。然而，随之而来的是一种生存的紧张。

秘密艺术品购买是建立在目前炒热气候的区块链技术之上的。随着秘密艺术品的潜在泡沫越来越大，它给地球带来的风险也越来越大。每一次交易都是另一次向大气排放大量二氧化碳。批评者警告说，如果不对代币的制造和销售方式进行重大改革，它最终可能会给生物圈，进而给人类带来无尽的恐惧。虽然计划采取措施减轻影响，但如果不尽快采取措施，它可能会让加密艺术延续其有望颠覆的同样破碎的提取和商品交易系统。

收藏家们购买的东西增加了这一奇观。例如，Ja Morant 扣篮在 YouTube 上是免费的。在撰写本文时， [孟菲斯灰熊队的视频](https://www.youtube.com/watch?v=dBj6Xin7MXA) 已经被观看了 28460 次(甚至还有将近两分钟的其他精彩片段)。我可以在 NBA Top Shot 上观看 10 万美元的剪辑，就像我可以在 Apple Music 上看 Grimes 的作品或听 LeonT5】NFT dropT7】的国王一样。因此，购买者购买的不是某个特定剪辑或艺术作品的权利，而这些剪辑或艺术作品在其他地方永远也看不到( [大部分是](https://www.instagram.com/p/CMIkEgSBjdx/) )。相反，他们通常购买的是与特定媒体对象相关的 NFT。可以转而再次出售。惠特克将与 NFT 有关的艺术比作蒙娜丽莎的原作，而张贴在其他地方的剪辑和艺术就像海报、咖啡杯或其他饰有她标志性微笑的小饰品。

NFT 是区块链的一部分，你可以把它想象成一个巨大的交易数字账本，每笔交易都有一个唯一的代码。在一个信息块之后，你添加另一个。如果不添加一个额外的块，就不能改变该链，并且这些块都是公开可见的。对于比特币这样的加密货币，块就是资产。但大多数加密艺术品都在以太坊上运行，这是一种使用以太作为加密货币的区块链。与比特币不同，以太坊用它自己的话说就是，为“成千上万的去中心化应用”如 NFT 和智能合约提供动力。当你买了一个，代币不是艺术品，而是大账本上的交易，本质上证明了所有权的转移。它是“不可替换的”,因为它不能与相同或相似的令牌交换。把它看作是实物所有权的证明，即使所说的实物是某处服务器上的图像。

但是我们在这里不是在讨论非实物的数字艺术品的收藏价值。我们试图评估 NFT 兴趣的爆发及其带来的潜在风险和机遇，这是艺术家们甚至在非传统艺术进入公众意识之前就一直在相互讨论的事情。

“这只是一个完全不受监管的资本主义模式，”艺术家乔安妮·勒默西耶(Joanie Lemercier)说，她最近在计算了 NFT 的碳成本后取消了该项目，并指出该市场“每周都在翻倍”。这就是为什么慢下来思考，或许采取行动是很重要的。”



二月份三大市场的 NFT 交易量超过了去年全年。一笔 [估计为 3.42 亿美元](https://dappradar.com/blog/dapp-industry-report-february-2021) 的交易完成了。许多人都在 NBA 的 Top Shot 上，需要说明的是，它并不在以太坊上运行。但是相对于铸造和交易密码艺术的气候影响，金钱方面是次要的。

以太坊上的交易是由世界各地竞相解决一个难题的计算机支撑的，这个过程被称为工作证明。首先完成任务的计算机获胜，并根据以太坊的加密货币以太的价格为其所有者赚一大笔钱，而其余的人最终无偿工作。这种浪费的能源对气候造成了巨大的损害。

矿业公司倾向于在能源廉价的地方建立工厂，这样他们就可以以尽可能低的成本运行更多的处理器，从而实现利润最大化。便宜通常意味着污染。例如，中国的内蒙古，由于丰富的煤电，是一个矿业热点。该省 [刚刚禁止了加密货币开采](https://earther.gizmodo.com/a-chinese-province-could-ban-bitcoin-mining-to-cut-down-1846397717) 部分原因是与之相关的碳污染问题。(尽管有一些清洁采矿作业的例子，但它们越来越少。)比特币受到了最大的关注，因为它的 [巨大的碳足迹](https://earther.gizmodo.com/mining-bitcoin-can-be-more-of-an-energy-drain-than-actu-1830217426) 与工作证明挂钩，但以太坊也是基于同样的原理运行的。

“它们运行完全相同的算法，这意味着它们有完全相同的浪费，”致力于跟踪比特币和以太坊碳排放的平台 Digiconomist 的创始人亚历克斯·德·弗里斯(Alex de Vries)说。“唯一的区别是以太坊比比特币允许更多的网络活动。因此，你可以比比特币的每笔交易占用更少的空间，因为这个系统的规模比比特币稍大一些。他们都在采矿。不管怎样，过程都是一样的:你让机器把能量浪费在无用的计算上。”



根据 [德弗里斯](https://digiconomist.net/ethereum-energy-consumption) 的计算，以太坊一年的能源使用量大致相当于爱尔兰全年的能源使用量。自去年 12 月以来，随着越来越多的人投资和使用人民币，以及越来越多的矿商参与进来获利，该曲线已大幅向上弯曲，在过去三个月里增加了一倍多。

NFT 的每个方面都会产生碳排放，从铸造代币到开采用于购买代币的硬币，再到转售。一个密码艺术碳计算器于 12 月份在 [cryptoart.wtf](http://cryptoart.wtf/) 上线，允许用户跟踪与任何艺术作品相关的排放量。它的创造者、艺术家和计算机工程师 Memo Akten，从那以后一直在迭代更新除了创作广泛阅读的媒体帖子 [包括方法论](https://memoakten.medium.com/analytics-the-unreasonable-ecological-cost-of-cryptoart-72f9066b90d) 。(地球人向 Akten 提出了关于他的方法论的问题，但在本文发表时尚未得到回复。)但它显示的某些作品的总数高得惊人。根据计算器的计算，一个普通的单版 NFT 消耗的能量相当于一个典型的欧盟家庭一个月的能量。多版本 NFT 更具破坏性。

能源分析师凯坦·乔希(Ketan Joshi)在一封电子邮件中表示，“这种行为太耗费能源，不能被视为中立行为。”。“对此，你会听到很多‘谁让你成为能源警察’的回应，但当然，我们实际上已经有了关于可感知的浪费、贪婪和排放强度的讨论。比如，围绕名人在世界各地乘坐私人飞机，同时敦促更多的气候行动。”

Lemercier 和一群活动家正在使用 Akten 开发的方法对 NiftyGateway 进行审计，NiftyGateway 是 NFT 交易的主要场所之一。这种方法使用了有关加密艺术品网站合同和以太坊整体能源使用的混合数据，来估计单个加密艺术品的碳足迹。结果显示，仅 30 个最大的艺术家在 2 月份的下降就释放了超过 2100 吨的二氧化碳。一个名为“比特币天使”的作品的 [开放版](https://niftygateway.com/marketplace?page=1&collection=0xe9be55ffedb6c2a2f3f8eac31c60d7f122f79958&type=1) 在 Nifty Gateway 上售出了 4157 个版本，导致大约 468 吨碳污染，正如 Akten 广泛使用的计算器所描述的那样，这相当于使用一台笔记本电脑 6000 年。



格里姆斯的高调下降导致了 122 吨碳污染。她承诺将销售收入 的 [部分捐赠给 Carbon180，这是一家专注于二氧化碳去除的非营利组织，一项未经证实但可能是避免气候危机最糟糕情况的必要技术，以及她男友埃隆·马斯克的个人兴趣。人们很容易觉得这是某种逃避，并对少数富有的艺术家为了进一步的利益而破坏地球感到愤怒。但不是每个艺术家都是名人。一些人没有画廊的作品，而非艺术画廊给了他们一个获得报酬的途径。就像所有与气候危机相关的事情一样，非关税壁垒是一个系统问题。](https://www.rollingstone.com/music/music-news/grimes-nft-digital-artwork-1134516/)

“这显然对这个领域的每个人都很重要，包括我自己，”皮普尔在一封电子邮件中说。他可能是当今最著名的秘密艺术家，因为他最近的收入规模和佳士得机构促进了他的作品的销售。“非物质文化遗产在赋予艺术家和创意者权力方面有着惊人的潜力，但它们绝对必须以可持续的方式进行。我可以向你保证，今后我所有的点滴不仅是碳中和的，而且是负碳的。我还将为我过去的下降购买碳信用额，不仅完全抵消这些下降，还使它们成为负碳排放。”

开放地球基金会(Open Earth Foundation)的执行董事马丁·温斯坦(Martin Wainstein)在一封电子邮件中说，他的组织正在帮助 Beeple 使用来自 Clean Coins、Digiconomist 和 Offsetra 的混合数据计算他的工作的碳足迹，并将通过 Creol 提供来自经验证的碳标准注册处的补偿，这些碳标准注册处也可以通过某种方式 [购买](https://opensea.io/collection/creol-verified-carbon-units)NFT。补偿是减少其他地方排放的项目。目前在 NFT 市场 Open Sea 上列出的克里奥尔 NFT 补偿与秘鲁的一个重新造林项目有关，该项目是联合国 REDD 项目的一部分。

单单指责艺术家，并要求他们承担责任，类似于对人们大喊大叫，让他们换灯泡或更好地回收利用。这些解决方案将解决他们自己无法解决的问题的责任推给了个人。热门平台基本上锁定了艺术家，类似于主要社交网络如何捕捉用户，并在如何解决他们负责的碳污染方面有更大的发言权。



与此同时，一些销售 NFT 的平台指责整个以太坊生态系统，并利用艺术家作为挡箭牌。NFT 最受欢迎的网站之一 Super Rare 上周在 Medium 上发布了一篇 [帖子，标题是“不，密码艺术家没有危害地球”，好像人们只对参与的艺术家感到愤怒。在帖子中，Super Rare 表示，它将使用碳抵消作为“减少以太坊碳排放影响”的承诺，但没有明确表示它的足迹将被抵消多少，也没有说明该团队将如何计算。该网站还表示，将捐款升级以太坊的 ETH 2.0 工作，这将彻底改革货币的开采方式，使其更具可持续性——尽管再次强调，它没有说明将捐款多少。(超稀有没有回应我们的置评请求。)](https://medium.com/superrare/no-cryptoartists-arent-harming-the-planet-43182f72fc61)

碳抵消可能有助于缓解围绕 NFT 排放的一些担忧，但它们无论如何都不是万灵药，而且已经被证明在许多方面非常成问题这些问题包括长期固碳失败，以及森林项目(一种常见的抵消方式)经常让居住在那里的土著群体流离失所。这也是一个基于市场的解决气候变化问题的方案，这个方案已经被一个将贪婪的增长放在首位的自由市场烤熟了。最多，补偿可以阻止一些秘密艺术和其他排放行业的气候损害，但它们远不能解决全球的碳排放问题。

“这基本上是百万富翁为了获得数百万的奖励而烧油，”Lemercier 在谈到最近的大规模加密艺术下降时说。“这是我们现在最不需要的。我们需要其他解决方案。解决方案就在这里，如果首席执行官们把它放在优先考虑的清单上，它们随时都可以部署。

“今天，这些平台中的任何一个都可以发表声明，他们将把 Harmony 作为连接这些网站链的桥梁，”他补充说，他指的是一个 [系统](https://www.harmony.one) ，该系统使用利益证明“连接”以太坊，可以帮助降低与 NFTs 相关的排放。“但他们只是太忙于每天赚的钱了。”

最大的秘密艺术公司之一正在选择补偿来处理它的排放，这反而显示了他们的优先权。这也揭示了秘密艺术品市场如何部分依赖于它声称是壁垒的系统的三个难题之一。本应是一个民主化、去中心化的体系，可以让艺术家更多地控制自己的财务命运，但它却依赖于持续的采掘关系，使用采矿系统将污染排放到大气中，并通过抵消来掩盖污染。这就把市场成本转嫁给了社会。惠特克说，“环境的外部性”是加密艺术的最大成本之一，但它与市场如何爆炸的其他问题有联系。



“我认为环境问题非常重要，需要量化，需要为未来做好规划，”她说。"解决方案可能比仅仅解决环境问题更全面。"

“从我的角度来看，允许全球创造性参与 crypto 和减少气候影响的目标是一致的，”阿米尔·卡特(Ameer Carter)在一封电子邮件中说，他化名 Sirsu，在这个空间工作，既出售艺术品，也确保这个空间更加环保和包容。“如果这个领域在很大程度上被全球少数团体所主导，那么只有这些团体才能在 crypto 中塑造我们的集体未来。通过纳入更多的文化群体，他们对创造性工作和可持续性的方法将积极影响我们如何构建数字生态系统向前发展。我们不能指望认为我们是唯一对这个领域有想法的参与者。”

让更多不同的人参与解决这些问题，也可以确保市场本身更加平等，让传统上在画廊艺术界代表性不足的黑人或同性恋艺术家站在塑造在线艺术的前沿。在给 TechCrunch 的 [评论中，卡特提到了 NFT 经济繁荣中已经发生的擦除现象。现在正面解决这一问题，同时解决气候问题，可能会开始弥合日益扩大的鸿沟。](https://techcrunch.com/2021/03/02/explosive-inclusive-potential-nfts/)

ETH 2.0 在某种程度上是以太坊的圣杯，因为它依赖于所谓的股权挖掘证明，而不是当前的工作证明。它自 2014 年以来一直在工作，并将改变工作证明方法，即每台计算机都在比赛解决一个问题，而这实质上是一场抽奖，随机选择一台计算机来解决这个谜题。这减少了大约 99%的能源消耗，并极大地减少了浪费。开发它的小组的目标是在 2022 年推出利益证明算法。这在理论上听起来很棒，可以在一夜之间缓解以太坊的碳排放问题，但有一个小问题。



“自 2014 年以来，他们一直在说我们将进行股权证明，自 2014 年以来，他们一直在说这将需要六个月，”德弗里斯说。“所以对我来说，我有点怀疑这是否会在他们所说的时间表内发生，因为他们在过去几年里一直无法实现他们的最后期限。”

股权证明还要求矿商在该系统中投资，并让拥有更多硬币的矿商有更好的机会获得奖励。因此，虽然这将减轻开采以太坊的一些气候成本，但最终可能会使系统更加不平等，而少数人继续获得巨大收益。

“对我来说，利害关系的证据复制了一些与收入不平等和不稳定的社会结构密切相关的排斥形式，”惠特克说。她提到了美国证券交易委员会(Securities and Exchange Commission)在大萧条(Great Depression)后制定的管理投资非上市公司的规则，她说，“他们用财富来代表成熟。因此，要投资私营公司，你必须是一个合格的投资者或合格的人。它的作用是集中财富，因为你必须富有才能投资高风险的公司，你可能会赔钱。但这些公司会成为下一个脸书、谷歌、Airbnb、比特币基地或其他什么公司。所以对我来说，股权证明的结构完全相同，它要求你有钱才能赚钱，这是有问题的。”

她建议，如果有办法集体拥有采矿股份，或者如果那些拥有资源的人可以拿出股份作为股份开采的证明，然后将收益重新分配给社区，股份开采的证明可能会变得更加平等。其他专家指出，透明度可以借鉴云计算的最佳做法，进一步帮助解决排放问题。



“如果有一种前进的方式，它肯定会创建一个系统，在这个系统中，采矿过程可以以某种方式追溯到一家公司，使其能源消耗公开，例如，我可以选择在谷歌或苹果公司存储我的文件，”乔希说。“我知道他们都购买了 100%的可再生能源购买协议，但我也知道谷歌做了额外的跑腿工作，将服务器的使用与可再生能源的产出相匹配，所以这实际上是更低的排放。”

无论采取哪种方式，国家非物质文化遗产也急需努力，以确保它们有益于艺术家和地球，包括受气候危机影响最大的最贫困人口。这是一个残酷的扭曲，最没有能力出售他们的艺术品或购买作品的人是那些受市场影响最大的人，市场对污染能源的依赖越久。

一些艺术家没有等待平台来解决这个问题，也没有等待 ETH 2.0 成为现实。绿色 NFT 奖和已经被提出，用于减少过程中的碳排放和提高人们的意识。它由 Artnome 创始人杰森·贝利创办，已经 [收到](https://gitcoin.co/grants/2063/green-nft-grant) 捐款总额超过 16000 美元。

“有多种方法可以解决这个问题，要建设一个更美好的未来，不仅仅需要转化非专利技术，”卡特说，他参与了绿色 NFT 的努力，并创建了一个名为 [井](http://ownthewell.com) 的加密工作室。“在此期间，绿色 NFT 奖金的目的是找到所有当前的解决方案，在它们的基础上进行建设，并向空间以外的更广泛的艺术社区展示，我们不是一个对我们如何对待我们居住的世界漠不关心的社区。”



其他人，包括 Lemercier，正在组织一个生态友好型 NFT Discord 服务器。虽然 Lemercier 没有分享细节，但他说他预计一些主要的艺术家可能很快就会离开污染的 NFT 平台，转而使用清洁能源或股权采矿的其他平台。Discord 的成员还编制了一份有用的清单，列出了开发更清洁的 NFTs 的开发者，而其他人则创建了一份详细的清单，列出了最受欢迎的 NFT 网站是如何工作的。

“虽然这些问题显然不仅仅是 NFT 特有的，而是关系到整个区块链，但我在数字艺术领域已经有很多很多年了，我真诚地相信，与我们在更广泛的区块链社区看到的相比，我们将成为积极解决这个问题的领导者，”皮普尔说。他们有自己的工作要做。

隐秘艺术将相对新旧的东西融合在一起，有一种不可否认的诱惑力:区块链和收藏品，数字和画廊艺术，以及更加平等的艺术市场的乐土，让艺术家在转售市场上分享他们的作品。尽管存在准入门槛，但它指向了一个可以纠正过去错误的未来。潜在的新的更好的东西的前景是诱人的。

然而，仅仅这样还不够。这可能会让那些在外面名气不大的艺术家往里看。与绿色新政(Green New Deal)非常相似，密码艺术的下一步将不得不解决与艺术相伴而生的气候问题和社会问题。否则，富有的艺术家和他们富有的资助人可能会在世界其他地方燃烧的时候，在他们密封的家中舒适地反复交易同样的几件作品。
