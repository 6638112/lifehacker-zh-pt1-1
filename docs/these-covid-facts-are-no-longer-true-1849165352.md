# 这些公认的“事实”不再是真的了

> 原文：<https://lifehacker.com/these-covid-facts-are-no-longer-true-1849165352>

这是一个漫长的疫情，COVID 仍然很强大。但是一路走来，我们至少学到了很多东西。就像我们不再沉迷于洗手一样，我们曾经有(或仍然有)很多关于 COVID 的其他想法需要更新。以下是一些大问题。

## 自然感染不再提供完全的保护

曾经有一段时间，如果你得了 COVID，你可以假设在你康复后的至少几个月内不会再得。不幸的是，目前正在传播的菌株似乎并非如此，包括 BA.4 和 BA.5，这两种都是高传染性 Omicron 的变种，它们似乎特别擅长引起感染，即使是在以前从 COVID 康复的人中。

也就是说，形势并不像最近一些头条新闻暗示的那样可怕。一篇文章 [声称，在被](https://web.archive.org/web/20220704035248/https://thetyee.ca/Analysis/2022/07/04/Get-Ready-Forever-Plague/) 纠正之前，再感染赋予“无免疫力”，这是错误的。它还引用了一份在δ感染 后不久发生的 [Omicron 感染的报告，表明这意味着一个人可能会反复感染 Omicron，这不是一回事。](https://www.forbes.com/sites/victoriaforster/2022/04/07/cdc-identifies-covid-19-reinfections-as-little-as-23-days-apart/?sh=707076257458)

我们所知道的是，如果你已经从以前的病毒株中康复，你仍然有可能从不同的病毒株中再次得病。好消息是先前的感染仍然可能至少为 T2 提供一些保护。一份预印本表明，即使你仍然可能生病，你也可能对严重的疾病有很好的保护。更好的消息是，这种疫苗对严重疾病仍然有效。

## “你的面具保护我，我的面具保护你”已经过时了

是的，那是一个可爱的想法，当人们真的戴着他们该死的面具的时候。在疫情诞生之初，这也是我们理解的极限。我们知道口罩很擅长阻挡从我们的嘴和鼻子中流出的飞沫，但我们不知道它们是否能阻挡飞沫，甚至是可能进入 T2 的气溶胶。事实证明确实如此。

这是一个好消息，当你去一个拥挤的地方，那里其他人没有戴口罩。一个合身的口罩，特别是如果它是由 N95、 [这样的优质过滤材料制成的，无论别人在做什么](https://lifehacker.com/is-it-worth-wearing-a-mask-if-youre-the-only-one-doing-1849065951) ，它都会对你的安全起到很大的作用。

## 六英尺没什么特别的

无处不在的关于社交距离的六英尺法则是基于呼吸道飞沫的飞溅区。你的呼吸或言语只会让唾沫飞溅三英尺左右。咳嗽或打喷嚏可以走得更远，所以六英尺似乎应该是一个很好的经验法则。

但后来我们了解到冠状病毒非常善于通过空气传播。如果你和其他人在一个封闭的空间里，空气中可能有病毒，即使你从未接近任何人六英尺以内。保持距离可能会有所帮助，但这不足以保证你的安全。

## 我们可以从疫苗中期待的保护已经改变

这个变了，然后又变回来。当第一批新冠肺炎疫苗获得批准时，FDA 只有数据显示它们在预防严重或有症状的疾病方面有多好，而不是它们是否首先阻止了人们携带病毒。

过了一段时间，看起来疫苗确实能防止人们被感染，甚至是无症状的感染。我们习惯于认为疫苗让我们变得不可战胜；我们被鼓励摘下面具。但是病毒有其他想法。现在，很有可能被感染，甚至生病，即使你已经完全接种了疫苗(甚至使用了推荐的加强剂)。

底线是疫苗不能防止我们感染病毒，但是它能降低我们感染致命疾病的可能性。因此，疫苗及其加强剂是值得的，但我们不应该如此急于回避其他保护措施。
