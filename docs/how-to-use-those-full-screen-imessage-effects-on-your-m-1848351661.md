# 如何在 Mac 上使用那些全屏 iMessage 效果

> 原文：<https://lifehacker.com/how-to-use-those-full-screen-imessage-effects-on-your-m-1848351661>

你可能还没有意识到，苹果电脑上也有 iMessage 的全部效果——即使苹果的桌面信息应用没有让它变得明显。例如，如果你想发送一条带有烟火的 iMessage 消息，你可以在 Mac 上这样做，就像在 iPhone 或 iPad 上一样。以下是如何在 Mac 上使用和发送 iMessage 效果。



## 在 Mac 上发送 iMessage 效果需要什么

要在 Mac 上使用所有有趣的 iMessage 效果，你必须至少升级到 macOS 11 Big Sur。Mac 上的 Messages 应用程序针对这个版本进行了改进，,它获得了发送和接收 iMessage 文本效果的能力。

如果你还没有更新，点击 Mac 屏幕左上角的苹果标志，进入**系统偏好>软件更新**。你也可以 [重新排列系统偏好设置](https://lifehacker.com/how-to-arrange-your-macos-system-preferences-so-you-can-1847583158) ，因为这只需要几秒钟就能让所有选项更容易找到。如果你厌倦了长时间的更新，这里有一个快速的提示来加速 macOS 的更新。

## 如何在 Mac 上给 iMessage 添加效果

当你升级到 macOS Monterey 后，以下是如何在 Mac 上发送带有效果的 iMessage:

1.  在 Mac 上打开“信息”应用程序。
2.  前往任何 iMessage 聊天，或通过点按笔记和铅笔图标开始新的聊天。
3.  键入您的消息，但暂时不要发送。
4.  点击消息左侧的**按钮。**
5.  选择**消息效果**。
6.  您将被带到**发送并生效**屏幕；你可以逐个点击每个效果来预览它。
7.  完成选择后，点击邮件右侧的蓝色向上箭头按钮发送邮件。

效果包括爱、气球、隐形墨水、纸屑、重击、激光、大声、庆祝、回声、聚光灯、烟火和温柔。
