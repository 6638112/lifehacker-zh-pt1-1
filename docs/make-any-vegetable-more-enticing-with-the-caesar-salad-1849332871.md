# 用凯撒沙拉使任何蔬菜更加诱人

> 原文：<https://lifehacker.com/make-any-vegetable-more-enticing-with-the-caesar-salad-1849332871>

凯撒沙拉酱是一种令人难以置信的调味品，充满了各种口味:丰富的蛋黄，鲜味(和盐)来自凤尾鱼和帕尔马，酸来自柠檬，辛辣来自大蒜和新鲜胡椒。为什么，那么，当可以用来使任何蔬菜更诱人的时候，我们却认为它仅仅是生菜的配料？



这正是黑莓农场的这位 [西葫芦凯撒沙拉](https://food52.com/recipes/72755-blackberry-farm-s-zucchini-caesar-salad-with-cheese-crisps) 提出并回答的问题(精选自 [Food52](https://food52.com/blog/20374-genius-zucchini-caesar-salad-blackberry-farm) )。沙拉很简单:薄薄的切片(或曼陀林)西葫芦撒上一点盐和胡椒，然后浇上香草凯撒酱，再浇上弗里科奶酪(因为 [奶酪是油炸面包丁](https://lifehacker.com/cheese-is-a-crouton-1849051473) )。这是一个很棒的模板，没有理由你不能把它应用到任何你喜欢的蔬菜上。这是一个用掉西葫芦意外收获的好方法，但用芦笋、一碗圣女果、一堆青豆，甚至一些起泡的 shishito 辣椒也可以。只要拌上调料，再加点奶酪就行了。

黑莓农场的凯撒调料配方几乎是凯撒和绿色女神的混合体。凤尾鱼被伍斯特郡酱取代(实际上含有凤尾鱼)，柠檬汁被雪利酒醋取代，龙蒿和细香葱使食物更加翠绿。这肯定是一个有趣的凯撒调味汁配方，但是如果你不想扩展，你总是可以用你最喜欢的。

我偏爱一种经典的、凤尾鱼口味的生蛋黄酱，但我也用 [蓝纹奶酪代替小罐头鱼](https://lifehacker.com/make-a-vegetarian-friendly-caesar-dressing-with-blue-ch-1829765284) 做了一个相当不错的素食版本。就奶酪而言，我认为[【frico】](https://lifehacker.com/you-should-fry-stuff-in-your-cheese-crisps-1847712396)是最棒的(它带来了风味*和*质感)，但 [空气油炸哈洛米](https://lifehacker.com/how-to-make-cheese-nuggets-in-your-air-fryer-1849191120) (奶酪块)也很棒，就像商店里买的粗略粉碎的奶酪脆片一样。

最后，不要忽视凯撒烤蔬菜，尤其是当我们度过了这些炎热的夏季。凯撒调味汁作为一种调味烤油，当在高温烤箱中加热时，增加了大量的美味和焦糖化。只需将一碗蔬菜放入足够的调料，轻轻裹上一层，然后在 400 华氏度的烤箱中烘烤，直到蔬菜中心变软，边缘变成褐色。最后加入大量帕尔马干酪和柠檬汁。
