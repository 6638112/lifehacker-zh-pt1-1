# 如何记住发酵粉和小苏打的区别

> 原文:[https://life hacker . com/how-to-memoring-the-difference-powder-an-1846481526](https://lifehacker.com/how-to-remember-the-difference-between-baking-powder-an-1846481526)

小苏打和发酵粉都是化学膨松剂，通过释放二氧化碳的酸碱反应赋予烘焙食品高度，但它们的方式略有不同。为了更好地理解如何使用它们，掌握它们是什么是有帮助的。

小苏打——在英国和一些英联邦国家也被称为“碳酸氢钠”——是纯碳酸氢钠，一种离子盐，化学式为NaHCO <sub>3</sub> 。它是天然的碱性物质，pH值为8，提供酸碱反应的“基础”部分。如果你用小苏打来发酵某些东西，你需要引入一种酸来与之反应，比如醋、柠檬汁、酒石或可可粉(取决于它的加工方式)。

如果你的食谱不含任何自然生成的酸，它可能需要发酵粉，它既提供了小苏打，也提供了一种或多种酸供小苏打反应。将酸和碱储存在容器中似乎不是个好主意，但发酵粉中的酸是以晶体形式存在的，只有当它遇到水时才能与小苏打发生反应。几乎所有你在杂货店买的发酵粉都是“双效”的，这只是意味着混合物中有两种酸——一种在混合成湿面糊时立即反应，另一种稍后被烤箱的热量激活。

现在是记忆术。即使你理解这两种发酵剂背后的理论以及它们是如何工作的，仍然很难记住它们的名字。如果你理解了这个概念，但对名称有困难，只要记住:小苏打仅仅是碳酸氢钠，而发酵粉是完全完整的。而且，如果你只有小苏打，也不要烦恼——你可以 [通过添加少量酒石来制作你自己的](https://lifehacker.com/if-youre-out-of-baking-powder-or-baking-soda-try-these-1843314311) 完全化学膨松剂。
