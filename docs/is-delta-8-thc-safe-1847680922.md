# Delta-8 THC安全吗？

> 原文:[https://lifehacker.com/is-delta-8-thc-safe-1847680922](https://lifehacker.com/is-delta-8-thc-safe-1847680922)

δ-8 THC是一种新流行的获得快感的方式， [，因为它在许多地方是合法的，而普通的大麻不是](https://lifehacker.com/how-you-can-legally-get-high-on-thc-anywhere-in-the-u-s-1846592514) 。但是仅仅因为它钻了法律的空子并不意味着它是绝对安全的，最近食品药品监督管理局和疾病控制和预防中心警告 [消费者](https://www.fda.gov/consumers/consumer-updates/5-things-know-about-delta-8-tetrahydrocannabinol-delta-8-thc) 和 [健康提供者](https://emergency.cdc.gov/han/2021/han00451.asp) 关于一些潜在的负面影响。

## 什么是德尔塔-8？

大麻含有多种自然产生的化学物质，对我们的大脑和身体有影响。最著名的是四氢大麻酚，简称THC。这被认为是大麻的主要来源。从技术上讲，我们关心的大多数四氢大麻酚在化学上被称为 *delta-9* 四氢大麻酚，联邦法规适用于含有一定比例delta-9的大麻植物。

这些天然化学物质中的另一种是大麻二酚(CBD)，自从2018年农业法案 允许THC含量低于0.3%的大麻植物合法销售以来，你可能已经注意到大麻二酚无处不在 [。这意味着如果你从低δ-9-THC大麻植物而不是高δ-9-THC大麻植物开始，你可以制造任何你喜欢的东西并合法出售。这导致了由这些低THC植物制成的CBD产品的繁荣。](https://www.fda.gov/news-events/congressional-testimony/hemp-production-and-2018-farm-bill-07252019)

沿着这条线的某个地方，一些天才发现你可以加工大麻植物来制造四氢大麻酚。联邦法律不禁止这种行为，但它仍然会让你兴奋。在 [我们负责任地享用delta-8](https://lifehacker.com/how-you-can-legally-get-high-on-thc-anywhere-in-the-u-s-1846592514)的指南中有更多关于这方面的信息，但基本上，在大多数州，你可以很容易地在柜台上购买delta-8产品或在网上订购。你会经常在CBD的商店里找到它们，因为它们的可用性也是基于同样的法律漏洞。

## 那有什么问题呢？

FDA和CDC最近的警告没有包括任何关于delta-8本身的可怕的东西，但是我们一会儿将进入细节。首先要知道的重要的是，delta-8仍然是THC，它似乎与正常(delta-9) THC具有相似的风险和收益。

换句话说，它可以让你兴奋，如果你服用太多，你可能会出现“绿化”的症状，包括恶心、嗜睡和不协调。THC还会影响你的心率和血压，这是一个健康问题，尽管过量服用不太可能致命 。

一旦你明白了这一点，有两个主要问题是FDA和CDC所担心的。

### 人们并不总是意识到他们在服用THC

由于delta-8通常与CBD产品一起出售，人们可能很容易忽略这样一个事实，即你正在购买可以让你兴奋的东西。不是每个人都知道delta-8是什么，那些第一次被介绍给它的人可能会听到它被称为一个可爱的昵称，如“减肥杂草”，或者他们可能会听到一个淡化delta-8和普通杂草之间的相似性的推销。

最近的 [CDC报告](https://emergency.cdc.gov/han/2021/han00451.asp) 提到了西弗吉尼亚州毒物控制中心的两个案例，在这两个案例中，成年人服用了他们认为是“CBD样产品”的东西，最终兴奋到给毒物控制中心打电话。

另一对病例，这次来自密歇根州，是儿童吃了父母的delta-8口香糖。这些儿童最终出现呼吸减缓、心率减慢和血压降低。他们被送入重症监护室进行监护和补充氧气。

疾控中心还指出，标签可能并不总是反映一种产品能让你感觉到多高，因为有些情况下，一种杂草产品可能同时含有delta-9和delta-8，但在其标签上只列出其中一种。虽然拥有合法大麻的州通常对质量控制或实验室测试要求有所监督，但delta-8产品却没有这样的框架。

这里的底线是，重要的是要像对待任何其他杂草产品一样对待delta-8产品:让它们远离儿童和宠物，仔细阅读标签，当你尝试任何新产品时，从少量开始。

### Delta-8产品可能被其他化学物质污染

如果delta-8产品只含delta-8，那就完了。但是大麻植物通常不含太多δ-8；这就是为什么我们谈论的是delta-8 *产品*(像口香糖和vapes)而不仅仅是吸食大麻花。要将大麻中的微量大麻素转化成足够的delta-8来制造精神活性口香糖，需要大量的工业加工。

正如《化学与工程新闻》 中的一篇文章所解释的，实验室分析在许多δ-8产品中发现了其他大麻素，如δ-9和δ-10 THC，以及一系列神秘化合物，这些化合物不是天然存在于大麻中的，其效果和潜在风险仍然未知。其中之一是橄榄醇，它可能会降低你的兴奋度，这可能是人们经常感觉delta-8的影响比普通大麻轻的原因——但这只是一个假设。我们也不知道橄榄醇的安全剂量是多少。

接下来的问题是，随着德尔塔-8的生产，产品中会添加什么。通常有一种溶剂，如甲苯或庚烷，和一种强酸。如果你能确定这些化学物质在产品完成前就被清除了，这可能没问题。但是“大多数人实际上并没有花时间蒸馏它或使用色谱法，”一位化学家告诉《化学与工程新闻》。

FDA和CDC还担心，生产delta-8产品的人可能在没有适当设施的情况下生产，他们可能因为颜色或其他原因添加了吸入或摄入可能不安全的成分。如果你还记得几年前的与烟雾有关的肺病(EVALI)，那似乎是由吸入 不安全的杂草烟雾中的 [添加剂引起的。监管不会自动保证安全，但值得注意的是，与EVALI相关的vape产品都不是来自合法药房。](https://lifehacker.com/the-cdc-just-confirmed-that-vitamin-e-acetate-is-their-1839723177)

如果你认为你有与delta-8产品有关的健康问题(或任何药物，就此而言)，FDA有许多方法来报告这些问题，甚至直接与该机构的人交谈，这些方法列在delta-8 概况介绍的底部 [。](https://www.fda.gov/consumers/consumer-updates/5-things-know-about-delta-8-tetrahydrocannabinol-delta-8-thc)
