# 你应该实际使用的新的 Windows 11 键盘快捷键

> 原文：<https://lifehacker.com/the-new-windows-11-keyboard-shortcuts-you-should-actual-1848620565>

Windows 11 不只是涂了一层新漆。这里有 [许多新功能](https://lifehacker.com/the-best-productivity-features-you-should-use-in-window-1848186695) 和重新设计的元素，从 widget 面板的，到快速设置，到专用虚拟桌面。它们都有自己的全新键盘快捷键。

如果你想真正拥抱(并充分利用) Windows 11 ，首先做两件事:找出并记住键盘快捷键，这将加速你的日常任务，然后重新映射其他的来做更好的事情，或者以更好的方式做同样的事情。



让我们从看看新的 Windows 11 键盘快捷键开始。然后，我们将指导您如何使用微软自己的 PowerToys 工具轻松更改这些键盘快捷键。

## 13 Windows 11 键盘快捷键其实值得使用

刚刚启动并运行 Windows 11？值得记住这些键盘快捷键:

*   **Windows + A** :打开快速设置面板，在这里可以控制 wifi、蓝牙、飞行模式、亮度、媒体等。
*   在微软团队中打开聊天功能，如果你喜欢的话。
*   **Windows + N** :打开通知和日历面板。
*   **Windows + W** :新 widgets 面板中的幻灯片。
*   **Windows + Z** :开启新的 Snap Layouts 功能，帮助你以网格格式快速抓取多个窗口。
*   **Windows + Alt +上/下箭头**:一个新的快捷方式，可以快速将选中的窗口贴紧到屏幕的上半部分或下半部分。
*   **Windows + D** :显示或隐藏桌面。(显示桌面会瞬间隐藏所有正在运行的窗口，你只会看到桌面壁纸和你的图标。)
*   **Windows + Tab** :打开新的虚拟桌面界面，显示当前桌面所有打开的窗口。您可以从底部切换到不同的桌面。
*   **Windows + Ctrl + D** :添加新的虚拟桌面。
*   **Windows + Ctrl +左/右箭头**:在虚拟桌面之间切换。
*   **Windows + Ctrl + F4** :关闭当前虚拟桌面。
*   **Windows + H** :启动新的语音打字功能。
*   **Windows + K** :直接跳转到快速设置中的演员表功能

## 如何重新映射 Windows 11 键盘快捷键

既然你已经学会了 Windows 11 中新的键盘快捷键，你可以对其中一些进行修改。例如，你可以重新映射 Windows + W 快捷键来做别的事情，因为你什么时候需要快速打开 widgets 面板？

同样，你可以用让其他键盘快捷键更容易使用。我发现 Windows + Tab v 虚拟桌面键盘快捷键输入起来有点棘手，所以我简单地把它映射到了左边的 Alt 键——上，这是我很少用的。

所有这些都可以使用微软的 PowerToys 工具来完成。它可以在 Windows 10 和 11 上预览，尽管对于日常使用来说足够稳定。你可以从[PowerToys GitHub 页面](https://github.com/microsoft/PowerToys) 的**发布**版块下载最新版本。

一旦你安装好了，g o 到侧边栏的**键盘管理器**部分，点击**重新映射一个键。**”

在**物理键**部分，输入您想要使用的键。(在我的例子中，那就是“ **Alt(左)**”)在映射到部分，输入您希望它成为的键盘快捷键。对我来说，那就是“ **Windows + Tab** ”您可以一次创建多个键盘快捷键。完成后，点击“**确定**按钮保存设置。
