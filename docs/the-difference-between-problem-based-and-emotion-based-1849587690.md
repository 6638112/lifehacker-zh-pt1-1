当你处理不适当的压力时，没有一个正确的方法来应对挑战。不同的情况需要不同的反应，主要取决于手头的问题是否在你的控制范围之内。你怎么知道什么时候你过于情绪化，或者什么时候你唯一能做的就是关注自己的感受？

## 以问题为中心的应对和以情绪为中心的应对有什么区别？

根据 [VeryWell Mind](https://www.verywellmind.com/forty-healthy-coping-skills-4586742) 的说法，基于问题的应对技巧专注于改变情况，而基于情绪的应对技巧则专注于改变你的感受。一种策略并不比另一种更好；不同的挑战需要不同的回应。例如，如果你正在处理一份糟糕的工作表现评估，关注问题而不是情绪可能更有意义。这意味着与你的老板一起制定一个以问题为中心的改善行动计划，而不是简单地关注你自己对这种情况的情绪反应。

另一方面，如果你正在处理悲伤或损失，以情绪为中心的应对技巧应该是最重要的。专注于以问题为中心的技术可能很有诱惑力，这实际上可能是一种否认，因为你对眼前的悲剧几乎无法控制。在这里，照顾好你的情绪是最重要的。

大多数情况下，这两种应对技巧都需要一点。例如，在糟糕的绩效评估回应中，你也可以加入以情绪为中心的策略，比如写日记或锻炼，来应对你的失望。

## 健康应对策略的例子

正如我们提到的，并不总是有一个最好的应对方法。这里有一些应对策略的例子，可以适用于各种各样的挑战:

*   建立和维护界限。
*   [静心](https://lifehacker.com/how-to-benefit-from-meditation-even-if-you-re-bad-at-it-1849196719) 或正念。
*   [记日志](https://lifehacker.com/why-you-should-keep-a-journal-and-how-to-start-yours-1547057185) 。
*   体育活动。
*   结束不健康的关系。
*   在你反应过度或抨击某人之前，给自己留点时间。
*   用爱好或朋友来分散注意力。
*   列任务清单，设定目标。
*   与你的老板、合作伙伴或任何能帮你解决问题的人会面。
*   向朋友或专业人士寻求支持。

理解以问题为中心和以情绪为中心的应对技巧之间的区别是解决你压力来源的有用框架。通常，你会从两种技能中获得。诚实地面对自己，在多大程度上情况是在你的控制之下，这样你就能准确地解决问题，同时也照顾到你的情绪。
