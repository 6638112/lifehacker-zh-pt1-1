# 使用Yelp了解一家餐馆的新冠肺炎安全措施

> 原文:[https://life hacker . com/use-yelp-to-find-out-about-a-restaurants-新冠肺炎-safet-1846074376](https://lifehacker.com/use-yelp-to-find-out-about-a-restaurants-covid-19-safet-1846074376)

无论你是在疫情期间莫名其妙地觉得在室内用餐很舒服的人(当这是一个选项时)，还是你想继续支持你的当地餐馆，但担心他们是否采取了适当的安全预防措施，Yelp希望让整个过程变得更容易。这也符合他们的最佳利益:该公司报告称，对于在2020年9月1日至12月31日期间在Yelp页面上添加新冠肺炎商业更新的企业，消费者的兴趣增加了41% 。

今年夏天，Yelp修改了企业页面上的新冠肺炎部分，让他们可以轻松地添加关于他们更新的服务产品和特定健康与安全预防措施的信息。与此同时，该应用程序使用户能够留下他们在企业中观察到的健康和安全做法的反馈，这是一种提示其他客户任何不太理想的体验的方式。

本周，Yelp推出了额外的功能，帮助客户了解餐厅的新冠肺炎安全预防措施，以及当他们看到员工忽视公共卫生措施时，如戴口罩，可以报告。以下是需要了解的内容。

### Yelp的新新冠肺炎安全功能

1月12日星期二，这些更新被添加到应用程序中。以下是该公司在最近的博客文章 中对它们 [的解释:](https://blog.yelp.com/2021/01/yelp-will-display-user-feedback-on-health-and-safety-practices)

> Yelp将显示用户是否观察到了社交距离的强制执行以及员工是否戴着面具。我们知道许多企业都优先考虑客户的健康和安全。这一新的更新进一步强调了企业如何适应保持客户安全，并旨在培养消费者继续支持当地企业的信心。

> 我们还增加了新的服务产品以及健康和安全实践，包括“室外加热座椅”、“1:1会议可用”和“一次性或非接触式菜单”，以帮助企业交流他们如何继续适应不断变化的健康订单和消费者期望。

除了显示用户观察信息之外，Yelp的新功能还能让顾客更容易对餐厅遵守公共卫生措施的情况留下反馈。

它还允许企业表明他们是否制定了以下 [服务产品和健康与安全实践](https://blog.yelp.com/2021/01/yelp-will-display-user-feedback-on-health-and-safety-practices) :

*   员工检查症状(适用于所有企业)
*   一次性或非接触式菜单(适用于餐厅、酒吧和夜生活)
*   室外加热座椅(适用于餐厅、酒吧和夜生活场所)
*   有盖户外座椅(适用于餐厅、酒吧和夜生活场所)
*   室内餐饮(餐厅、酒吧和夜生活区都有)
*   私人用餐(餐厅提供)
*   DIY餐包(餐厅提供)
*   户外服务(适用于健身和美容企业)
*   提供1:1会议(适用于健身企业)
*   提供有限的小组会议(适用于健身企业)

Yelp的博客文章 上有很多 [更多的信息——包括他们将如何确保用户产生的反馈得到公平的体现。](https://blog.yelp.com/2021/01/yelp-will-display-user-feedback-on-health-and-safety-practices)
