# 如何拍出自己真正喜欢的毕业照

> 原文：<https://lifehacker.com/how-to-take-graduation-photos-youll-actually-like-1846827966>

毕业是一件大事，因为在大多数情况下，你只能被拍到毕业一次。一旦你的毕业生大步走过讲台领取他们的高级文凭，你就没有重来的选择，因为你错过了关键时刻。



不管你爱的人戴着帽子穿着礼服看起来有多尴尬，你都想最大限度地捕捉这个里程碑。这里有一些拍出好毕业照的小技巧。

### 提高你的 ISO，尝试隔离你的拍摄对象

毕业典礼是典型的杂乱无章的事情。使用长焦镜头当然是最好的选择，它主要是为了捕捉远离相机的人的图像。这将帮助你穿过人群，把镜头对准你喜欢的主题，而不是你不认识的其他毕业生。

更重要的是，长焦镜头擅长模糊图像的背景，所以你的主体真的闪闪发光。它们也很适合人像摄影，而这正是你所追求的，尽管有一段距离。正如 [借用镜头注意事项](https://www.borrowlenses.com/blog/what-is-a-telephoto-lens/#:~:text=A%20telephoto%20lens%20will%20allow,in%20front%20of%20the%20camera.) ，这种镜头会特别适合你的拍摄对象。

> 这种效果也能让五官显得更匀称。再加上用长镜头拍摄的美丽散景，你就有了一个很好的人像摄影工具。

如果你使用的是数码单反相机，尤其是如果你不是专家，另一个关键是提高你的 ISO 功能。ISO 指的是你的相机对光线的敏感度。当您使用长镜头或拍摄快速移动的对象(如短跑运动员，甚至是走过舞台的人)时，较大的 ISO 设置将有助于产生更清晰的图像。

如果你在白天拍摄，建议使用较低但仍然较高的 ISO，但它的重要性在光线较暗的环境中会增加，比如说，一个洞穴状的健身房。 [数码摄影学校](https://digital-photography-school.com/reasons-why-shoot-high-iso/) 详细介绍一些最佳实践:

> 当然，当光线明亮时，你可以将 ISO 保持在 100 或 200，最终得到清晰、曝光良好的图像。但是随着光线开始下降，你需要自信地提高你的 ISO。这样，您可以在 300 毫米、400 毫米甚至更远的距离拍摄明亮清晰的照片。

另一个建议，尤其是对那些可能没有漂亮镜头的外行人来说，是把你的相机设置成自动模式。如果你这样做，你也可以尝试使用自动对焦镜头，这样你就可以坐下来享受整个过程。

当然，还要考虑闪光灯(或者不要)。尼康建议只有当你靠近拍摄对象并且在室内时才使用——这两个条件可能不会同时适用于你的毕业典礼。

> 除非你离得比较近，否则不要用闪光灯。闪光灯达不到你的拍摄对象，所以对曝光没有帮助，还可能耗尽你的电池。

### 要捕捉的好图像的指针

优秀的摄影师——即使是业余摄影师——也能捕捉到微妙的瞬间。所以，你必须在当天节日的秘密肖像和舞台照片之间找到一个健康的平衡。

需要强调的一点是，至少从主题意义上来说，这是一个*毕业典礼，*意味着这样一个事件的装备——帽子和长袍、学校、文凭等等。—应该是一个焦点。基本上，你会想尝试一堆不同的东西，都在更广泛的肖像范畴内。给毕业生和家人、朋友以及任何你认为合适的人拍照。尽量融入学校的校园，如推荐的 [油印照片](https://www.mimeophotos.com/blog/college-graduation-photos) ，上面写着:

> 入口通道和门廊是理想的。你不会后悔在学校的标志或吉祥物雕像前拍了那张照片。

就拍摄合法人像而言，专家也建议为他们使用更长的镜头，这对于使用全画幅 D-SLR 拍摄的人来说，意味着“焦距通常在 70 毫米和 200 毫米之间的镜头”，根据 DP Mag 的 [。正如摄影师兼作家杰克·克罗斯比(Jack Crosbie)告诉 Lifehacker 的那样，更长的镜头“让东西看起来更平，这更符合人们的脸型。”另一方面，更宽的镜片有时会让人](https://www.dpmag.com/gear/lenses/picking-the-perfect-lens-for-portraits/#:~:text=Most%20photographers%20will%20tell%20you,%2C%20100mm%2C%20105mm%20and%20135mm.) [看起来更大，与现实](https://lifehacker.com/no-joe-biden-isnt-8-feet-tall-and-other-camera-tricks-1846822104) 不相称。

即使你的摄影技术非常欠缺，也要留意通常决定一天的所有大型庆祝活动，比如扔帽子、走过舞台和与朋友狂欢。
