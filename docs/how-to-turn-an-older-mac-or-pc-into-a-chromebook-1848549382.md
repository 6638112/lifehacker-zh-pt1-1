# 如何将旧的Mac或PC变成Chromebook

> 原文:[https://life hacker . com/how-to-turn-a-older-MAC-or-PC-to-a-chrome book-1848549382](https://lifehacker.com/how-to-turn-an-older-mac-or-pc-into-a-chromebook-1848549382)

在你把运行缓慢的旧电脑带到电子垃圾处理场之前，试着安装一个Chrome操作系统，让它重新发挥作用。谷歌已经打开了 [Chrome OS Flex](https://chromeenterprise.google/os/chromeosflex/) 的下载大门，这是其多功能 OS 的一个新版本，如果你的旧Mac或PC难以接受其原生操作系统的更新，这可能是一个完美的解决方案。

Watch

Flex旨在通过安装一个更精简、以云为中心的操作系统来更新旧电脑。虽然它没有Chrome的所有功能，但谷歌承诺，它基本上会让你的Mac或PC看起来和感觉上都像运行Chrome OS的Chromebook。

谷歌表示，Chrome OS Flex将提供“随时随地的快速现代工作体验”，这将使你的设备启动更快，快速访问VDI(虚拟桌面界面)和网络应用，并防止它随着时间的推移而变慢。另外，它是免费的。

虽然用Chrome操作系统改造你的电脑并不是一个的新黑客 — [CloudReady](https://lifehacker.com/turn-your-old-laptop-into-a-chromebook-with-cloudready-1740556796) 于2015年问世——Flex OS提供了更多的灵活性，通过提供双引导，允许用户通过简单的重启轻松地在操作系统之间切换。

在你安装Chrome OS Flex之前，要知道它处于早期访问模式，所以可能会有bugs不稳定性。此外，我不会做Chrome操作系统所做的一切——一些功能取决于你用来运行该操作系统的硬件，而Google Play和Android应用程序根本不受支持。(如果你需要更多细节，这里有谷歌列出的Chrome和Chrome OS Flex 之间的 [差异。)](https://support.google.com/chromeosflex/answer/11542901?p=flex_chromebook_differences&visit_id=637805415333447097-1267741764&rd=1)

## **如何在笔记本电脑或台式机上安装Google OS Flex**

*   买一个8GB或更大的u盘。
*   [自己注册](https://chromeenterprise.google/os/chromeosflex/#form-section) 。
*   阅读(并遵循) [创建Chrome OS Flex USB启动棒的说明](https://support.google.com/chromeosflex/answer/11552529?p=flex_ent_form_install&_ga=2.256286923.1340841675.1645025661-2041450116.1645025661&visit_id=637806253777018153-2430488289&rd=1) 。
*   插入Chrome OS Flex USB后重启电脑。
*   享受你的Chrome电脑。
