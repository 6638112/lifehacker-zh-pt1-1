# 你应该华夫饼干一些米饭，是的，你应该

> 原文：<https://lifehacker.com/you-should-waffle-some-rice-yes-you-should-1846715704>

我喜欢米饭。我喜欢新鲜烹制的，松软嫩滑的，或者煎得酥脆，再加上一个松软的鸡蛋。这也使得成为一种非常好的、令人满意的松脆华夫饼——一种的华夫饼，因为它变成了，是我继母给我的一些秋葵汤的完美平台。



像炒饭一样，蛋奶饭最好用干米饭制作，因为缺少水分是让它变得真正酥脆的关键。新鲜、鲜嫩的大米太湿了，所有的蒸汽防止了褐变和变脆。你可以在冰箱里过夜晾干你的米饭，或者加速这个过程 [用一个小风扇](https://lifehacker.com/make-perfect-fried-rice-with-help-from-a-small-fan-1759964474) (如果你现在就需要你的米饼*)。*

*一旦你的米饭干透了，倒一点油(芝麻和橄榄都可以)，然后把它舀进设置为中高温的华夫饼机。让华夫饼干机做它自己的事情，直到你有一个脆脆的，有凝聚力的，边缘略带金黄色的米饼。用一些木筷子把它从华夫饼干机里哄出来，放到你的盘子里。然后在上面放些东西。(我怎么推荐秋葵汤都不为过。)*

*就比例而言，半杯煮熟的干米饭加 1/2 茶匙油或其他脂肪是完美的。我用未经过滤的橄榄油做了上面你看到的华夫饼，但是不要睡在黄油(为了更好的褐变)或动物脂肪上——培根脂肪大米华夫饼将是完美的解酒助手。一个警告: T 他的华夫饼干需要时间，或者至少比我习惯的各种华夫饼干要多得多。它花了我整整五(5！)几分钟就搞定了，但这种等待是值得的(对我来说也是一次很好的耐心锻炼)。*

*尝试浇头，我想你会发现无论一天中的什么时候，米饭华夫饼都在你的盘子里。这是一种很好的早餐碳水化合物(尝试用鸡蛋和)但是同样适合午餐和晚餐。我有没有说过它配秋葵汤很好吃？因为确实如此。配秋葵汤真的很好吃。*
