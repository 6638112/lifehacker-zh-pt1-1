# 如何对狗进行心肺复苏

## **如何知道你的狗需要CPR**

根据PetMD的詹妮弗·科茨博士，DVM 的说法，一只狗可能因为许多原因需要CPR，包括外伤、窒息或疾病——但除非万不得已，否则你不应该进行CPR，因为这对一只健康的狗来说可能是潜在的危险甚至是危险的。对狗进行CPR的目的和对人类一样:如果狗的心脏停止跳动或停止呼吸，氧气水平会下降，重要器官会衰竭；CPR可以提供人工呼吸和血液流动，直到心脏可以重新启动并开始自己跳动。

根据红十字会 ，你的第一步应该是检查你的宠物是否有呼吸和心跳。如果他们的胸部不动，你找不到脉搏，开始心肺复苏术。

## **如何对狗进行心肺复苏**

不同大小和类型的狗有不同的手放置方式:

现在检查一下你的狗的体格，并记下在紧急情况下你的手应该放在哪里。最后，正如ASPCA宠物保险T3所指出的，在开始CPR 之前，你应该确保没有任何东西阻塞你的宠物的喉咙，可能会干扰救援呼吸。

*   小狗:如果你的狗很小，握紧拳头，把你的手掌放在它们的心脏上。把另一只手放在拳头上。
*   如果狗的胸部很深，将一只手的脚跟放在胸部最宽的地方，另一只手放在上面。
*   如果你的狗是桶状胸，把狗放在它们的背上，把一只手放在它们胸骨最宽的部分，用你的另一只手盖住它。锁定肘部，确保肩膀位于双手正上方。

红十字会概述了接下来的步骤:

1.  用力快速按压，每分钟100-120次。
2.  确保压缩幅度是动物胸部宽度的三分之一到二分之一。
3.  再次按压前，确保胸部完全恢复。
4.  闭上宠物的嘴，伸长它们的脖子，用你的嘴盖住它们的鼻子，用你的手轻轻按住它们的嘴，呼气，直到你看到狗的胸部上升(对于30磅以下的狗，用你自己的嘴盖住整个鼻子和嘴)。
5.  继续进行CPR，在30次胸外按压和两次人工呼吸之间循环，直到他们开始自主呼吸。
6.  开始心肺复苏后，每两分钟检查一次呼吸和心跳。
7.  继续这个过程，直到有人来帮忙，或者你可以把狗送到动物医院。

## **如何对狗进行海姆利克氏操作法**

如果你检查你的狗的喉咙有没有堵塞，发现有阻塞，你可以自己把它取出来——但是如果你认为你会把它放在呼吸道的更深处，就不要这样做。相反，选择改良的海姆利克氏操作法来清理通道。

1.  把你的手放在狗的胸腔两侧
2.  施加压力。

如果你的狗在堵塞物被清除后没有开始呼吸，开始心肺复苏术，然后去兽医医院。
