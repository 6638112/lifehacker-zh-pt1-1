# 如何在谷歌地图上追踪新冠肺炎趋势

> 原文：<https://lifehacker.com/how-to-track-covid-19-trends-on-google-maps-1845172437>

我真的希望谷歌地图的最新整合能让你看到新冠肺炎病例在一张巨大的地图上以 [*瘟疫公司*](https://www.youtube.com/watch?v=upO8biZqmn8) 的风格出现并扩散。唉，这是不应该的——但你现在至少可以直接在谷歌地图上看到新冠肺炎的趋势。

Watch

(不，这不应该用来帮助你计划去病例较少的地区度假，因为你应该像其他聪明的美国人一样呆在家里保持安全。)

首先，请确保您的谷歌地图应用程序已在您的或 [iOS/iPadOS](https://apps.apple.com/us/app/google-maps-transit-food/id585027354) 设备上更新。启动应用程序，点击屏幕右上角的图层图标(看起来像一个叠一个的方块)，然后在出现的屏幕中选择“新冠肺炎信息”:

不幸的是，你将无法在基于浏览器的谷歌地图版本上找到这些信息(对于那些喜欢在家而不是在旅途中给自己带来焦虑的人来说)。但你仍然有很多其他选择来跟踪你所在地区的新冠肺炎病例，包括约翰霍普金斯大学的、Reddit 的“covid mapping”[mashup](https://covidmap.global)、1Point3Acres [病例地图](https://coronavirus.1point3acres.com/en) ，以及(有趣地命名的) [感染 2020](https://infection2020.com) 地图，等等。

当谷歌的实现为每个人上线时——在撰写本文时它仍在推出——你将能够看到“谷歌地图支持的所有 220 个国家和地区的七天平均情况，以及州或省、县和市级数据，”谷歌写道。COVID trends 将很容易识别出*和*颜色编码，如果你只是需要一个快速的视觉提醒，告诉你你的主场正在变得更好(或更差)，这很有用。

虽然多一点信息不会伤害任何人，但我不会指望这个新功能会对那些拒绝戴口罩的人产生影响。你可以在谷歌地图上把一个大大的“你很笨”放在他们的房子顶上，他们会友好地向你挥挥手，然后和他们的朋友去度一个迷你假期。

保持安全；保持警惕。告诉 [愚蠢的人](https://vitals.lifehacker.com/blame-the-people-in-charge-1845109900) 他们糟糕的决定是如何毫无必要地危及我们所有人的生命。请，*请*不要使用谷歌的数据来计划去一个“更安全”的地方的快速旅行。你应该做得更好。
