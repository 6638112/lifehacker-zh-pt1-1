# 请停止使用这些常见的歧视老年人的短语

> 原文：<https://lifehacker.com/please-stop-using-these-common-ageist-phrases-1847911858>

年龄歧视是“最后可接受的偏见”，至少根据美国退休人员协会 的说法 [。虽然基于年龄的偏见更多地是针对年长的员工，但它确实是双向的“年长和年轻的成年人在工作场所都经常处于不利地位，随着年龄的增长，获得专业培训和教育的机会明显减少，”世界卫生组织表示。特别是在工作场所，](https://www.aarp.org/work/working-at-50-plus/info-2019/age-discrimination-in-america.html)[2019 年的一项研究](https://www.hiscox.com/documents/2019-Hiscox-Ageism-Workplace-Study.pdf) 显示，每五名 40 岁及以上的美国工人中就有一人表示他们面临年龄歧视。

但是年龄歧视并不总是明显的歧视。有时候，这是一个小评论，微妙地延续了“*旧”*是*“坏”的代码的想法*以凯文·哈特的 [为例，现在它被迷化为对唐·钱德尔年龄的反应](https://www.youtube.com/watch?v=0ugn_vsZ9pQ) 。即使是最善意的评论也可能变成年龄歧视的微攻击。以下是在谈论某人的年龄时经常弊大于利的常用短语。

## **“你不要看[此处插入年龄]！”**

荣誉奖: *50 是新的 25！*尽管你可能想把这当成一种赞美，但在大多数情况下，这意味着不管他们实际上是什么年龄，出于某种原因都是“不好的”。在这种情况下，你可能要重新考虑你是否需要评论这个人的年龄。

## “他们内心很年轻。”

来自 [联盟](https://www.alliance1.org/web/news/2019/march/putting-stop-ageist-language.aspx) :

> 虽然可能是为了传达一种积极的内涵，或者是对另一个人的赞美，但这些说法听起来有“年轻是好的，，，“年长是坏的”的含义我们挑战你去思考你到底是什么意思。你指的是高水平的能量吗？敏捷？幽默感还是游戏性？...说出这些话，而不是概括任何一代人的感知特征。

换句话说，试着识别你说的“年轻”是什么意思。这样，你就避免了暗示“老”与他们是对立的。

## **【老练】**

除非你说的是在鸡肉上撒香料，否则“调味”是一种委婉的方式，表示对“老”的人不屑一顾。就像关于“年轻的心”的建议一样，试着说出你真正的意思——无论这个意思是积极的，例如“有经验的”，还是消极的，例如“脱离接触的”

## “好的，布玛儿。”

我承认。我对这句话最大的问题是，在这一点上，它是黑客喜剧。在更严重的层面上，“OK boomer”与将所有千禧一代描述为“有权”有同样的问题。仅仅因为年龄上的成见就解雇某人是懒惰和不公平的。

## “你不能教老狗新把戏。”

这个 [在犬科动物世界](https://vcahospitals.com/know-your-pet/can-old-dogs-learn-new-tricks) 中甚至都不是真的。你可能会随口说这句话，但实际上并没有任何含义。然而，当老年人基本上被视为不可教因而不讨人喜欢时，这种情绪会造成 [真正的经济伤害](https://www.aarp.org/work/working-at-50-plus/info-2019/age-discrimination-in-america.html) 。

## “我能帮你吗，年轻的女士？”

你为什么叫我奶奶小姐？这很奇怪，并假设年轻是一种恭维，而现实肯定是一种侮辱。嘿，也许你根本不需要提及她的年龄。

## **“小伙子”“姑娘”“老姐”**

除非某人真的是个孩子，否则请不要光顾(而且令人毛骨悚然！)条款。

## **“你可能不想使用这个新平台……”/**“**如果你在学习技术方面有困难……”**

即使你试图适应年长的同事或家庭成员，假设他们会拒绝新技术也是一种居高临下的态度。在工作场所，这可能会使他们失去其他团队成员可以获得的机会。如果有人对一个新平台有抵触情绪，那就在个人的基础上解决它，而不是让它成为基于年龄的刻板印象的一部分。

作为一条经验法则，想想你的评论是否强化了这样一个观点，即年龄是一个人如何思考、行动和提供价值的最终决定因素。当有疑问时，你可能根本不需要提及某人的年龄。
