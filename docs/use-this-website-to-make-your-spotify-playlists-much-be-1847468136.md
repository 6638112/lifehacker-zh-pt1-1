# 使用此网站可以让您的 Spotify 播放列表变得更好

> 原文：<https://lifehacker.com/use-this-website-to-make-your-spotify-playlists-much-be-1847468136>

算法推荐以神秘的方式工作，它们并不总是正确的。每个 Spotify 用户都想知道这首歌怎么会出现在*和*播放列表中——或者为什么一首他们两个月前偶然竖起大拇指的歌会一次又一次地出现。



[Songslikex 为 Spotify 不可靠的推荐算法提供了另一种选择](https://songslikex.com/) 。这很容易使用:去 songslikex.com，输入一首你喜欢的歌，然后按回车键。你会得到一个自定义的播放列表，打开你选择的曲目，然后是 50 首随机选择的听起来相似的歌曲。我最近一直在磨损我的“完美歌曲”播放列表，所以我给 Songslikex 输入了我当前最喜欢的歌曲，看看它是否能找到一些新的内容。

这个基于普林斯的“当你是我的”的播放列表肯定会有很多地方——其中一些令人困惑——但我已经准备好了:

总的来说，Songslikex 是有趣和有效的。我很喜欢听这些播放列表，作为奖励，它们帮助我找到了新的艺术家，否则我可能不会遇到。我也很欣赏歌曲选择是随机生成的，，因为它给了你对算法输出的某种控制——如果你不喜欢第一次尝试，你可以继续尝试。每次你都会有新的收获。这比在多个类似主题的 Spotify 电台和播放列表中不断听到同一首歌要好。

不过，有一个缺点:Songslikex *只有*能与 Spotify 合作，这让任何喜欢这种不同服务的人感到失望。作为使用 Pandora Premium 的几十人 之一的 [，我已经习惯了这种事情，但它仍然让我很不爽。我从 Songslikex 获得的播放列表非常棒，我非常想在 Pandora 上使用它们，在那里我支付了每三首歌就不能听到四个 30 秒广告的特权。但如果你使用 Spotify，并想改变你的播放列表，不妨试试 Songslikex。](https://www.youtube.com/watch?v=lKie-vgUGdI&ab_channel=Storage1129)
