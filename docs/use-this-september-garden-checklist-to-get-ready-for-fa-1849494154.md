# 使用这个九月花园清单为秋天做好准备

在《纽约时报》最近的一篇文章中，资深花卉园丁 [珍妮·罗斯·凯里](https://jennyrosecarey.com/) 宣称，谈到花卉，“九月是新的五月。”每年，她都会参观全国各地的秋季花园，看看9月份还有哪些花在盛开，然后将她学到的东西应用到她的家庭花园中。

Watch

凯里的策略包括全年主动维护花园——换句话说，不仅仅是照顾当前当季的花卉和植物，还要提前优化花园的未来生长。

我们其余的人可能没有那么有条不紊，但做一点主动维护可能会受益。如果你不确定这个月到底要在花园里做什么，这里有一份九月花园清单可能会有所帮助。

## 九月花园清单

所有的花园都是不同的，所以这些任务不一定都适用于你的花园。但总的来说，以下是9月份在你的花园里应该做的事情，按类别分类如下:

### 花(一年生和多年生植物)

*   继续**死亡之旅**以鼓励这一季最后几朵花的生长。
*   在月初 分种和/或种植**黄花菜和牡丹** [，如果你还没有这样做的话。](https://www.chicagobotanic.org/plantinfo/checklist/september) 
*   月底开始**种植**[](https://www.groworganic.com/blogs/articles/september-gardening-checklist)**(像水仙花、郁金香、风信子、番红花或鸢尾)春天开花的球茎(或者等到十月)。理想情况下，你要在地面冻结 前六周左右，或者常规夜间温度在40到50度之间时，对 [做这个。很重要的一点是，要在冻僵之前把球茎种到地里，这样根部才有时间扎根。)](https://www.gardendesign.com/checklist/northeast/september.html)**

### **水果、蔬菜和香草**

*   **继续**收获水果和蔬菜** [，因为它们成熟了](https://www.chicagobotanic.org/plantinfo/checklist/september) 。让羽衣甘蓝、羽衣甘蓝和抱子甘蓝单独生长，直到第一次霜冻，然后收获。(这改善了它们的风味。)**
*   **种植**秋季覆盖作物**(如燕麦、冬黑麦、冬小麦、绛三叶草和毛茛) [保护和改善你的土壤](https://extension.umd.edu/resource/cover-crops#:~:text=Popular%20fall%2Dplanted%20cover%20crops,your%20soil%20after%20they%20decompose.) 。**
*   ****可以，发酵，干燥，或冷冻**任何 [剩余产生](https://www.groworganic.com/blogs/articles/september-gardening-checklist) 。**
