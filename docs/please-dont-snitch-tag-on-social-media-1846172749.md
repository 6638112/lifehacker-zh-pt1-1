# 请不要在社交媒体上“打小报告”

> 原文:[https://life hacker . com/please-don-t-snitter-tag-on-social-media-1846172749](https://lifehacker.com/please-dont-snitch-tag-on-social-media-1846172749)

坐在电脑前，在匿名温暖怀抱的保护下，网上分享你的观点有一定的自由。这也意味着公开贬低其他人的影响很小——也就是说，直到你嘲笑的人被一个给他们贴标签的闯入者发现。

Watch

这种现象被称为“打小报告”，它的存在没有其他原因，只是为了轰动一时，就像一个在社交媒体的地狱般场景中漫游的操场上的告密者，唯一的意图是在火上浇油。

如果你曾经是这样的人，知道你可以做得更好。抑制打小报告的冲动是可能的，如果没有别的原因，就让事情平静下来，并意识到在一个话题中加入你自己的声音比懒洋洋地捅马蜂窝更有成效。

### 为什么告密者标记不好？

不管是无意的还是有意的，你都在给一个已经不稳定且重复的网站煽风点火。尤其是Twitter，它在耻辱和冲突中茁壮成长，这些耻辱和冲突扩散到更广泛的媒体生态系统中，甚至渗透到主流政治问题中。 然而，有些用户——无论是作家、知识分子、娱乐界人士，还是卑微的键盘战士——并不想找一场公开的拳击比赛。甚至对每一个温和的指责或批评都给予回应也是没有用的，因为这种回应可能会变成一堆有毒的东西。

无论如何，社交媒体是推进有意义对话的有限媒介。如果你想不同意某人对某个名人的看法，或者甚至让某个公众人物面对铺天盖地的推文或评论无情地抨击他们，有更好的方法可以做到。

### 不要打小报告，写你自己的帖子

社交媒体是一个回音室，这意味着每个人都可以在虚无中释放自己的想法，以追求转瞬即逝的认可和关注。所以好好利用它。如果你看到一条诽谤某人的推文，你有自己的观点，用你自己的声音在你自己的帖子里表达出来。如果你不能满足让某个人知道的强烈愿望，就在你的帖子里给他们贴上标签。

这并不是一个复杂的解决方法，但是在你的日常社交媒体礼仪中使用它可以帮每个人一个小忙，只是通过剥夺它作为日常燃料的一些尖酸刻薄。此外，一些名人和公众人物 [会在Twitter上与他们的粉丝](https://www.someecards.com/entertainment/celebrities/funny-interactions-celebrities-twitter/) 互动，所以你可以通过简单地打破网站的恶性循环来获得一个很酷且令人难忘的互动。
