# 现在更容易获得FHA贷款

> 原文:[https://life hacker . com/its-now-easy-to-get-an-FHA-loan-1846151804](https://lifehacker.com/its-now-easier-to-get-an-fha-loan-1846151804)

政府机构已经宣布了一些变化，允许“梦想家”——拥有 [童年抵美(DACA)身份](https://www.uscis.gov/humanitarian/humanitarian-parole/consideration-of-deferred-action-for-childhood-arrivals-daca) 的美国居民——有资格获得FHA抵押贷款，以及对已经拥有这些贷款并正在努力还款的人延长抵押延期的最后期限。

Watch

美国住房和城市发展部(HUD)上周宣布，大约70万具有DACA身份的人现在有资格申请FHA抵押贷款，前提是他们也可以在美国合法工作之前存在一项条款，要求借款人拥有“合法居留权”——这最初给贷款人造成了混乱。随着这一变化，小时候被非法带到美国的移民现在可以获得这些贷款。

联邦政府支持的FHA贷款传统上由低收入的首次购房者使用，因为这些抵押贷款的信贷和首付要求( [低至3.5%](https://www.investopedia.com/terms/f/fhaloan.asp#:~:text=FHA%20loans%20are%20designed%20for,scores%20than%20many%20conventional%20loans.&text=This%20means%20you'll%20need,at%20least%20580%20to%20qualify.) )比传统的私人贷款更灵活。本月早些时候，联邦住房管理局也宣布，他们将2021年的FHA贷款限额提高到7.4%，以跟上房地产价格的上涨——你可以在这里 看到新的限额。如果你正在考虑申请FHA贷款，你可以在这里 找到更多信息 [。](https://www.hud.gov/buying/loans)

### FHA借款人的额外COVID减免

在与COVID救济相关的另一份公告中，HUD还宣布延长FHA借款人申请和获得延期偿付的最后期限。受到疫情金融影响的房主可以获得长达六个月的延期或减少的抵押贷款付款(尽管利息仍根据贷款条款累计)，前提是他们在2021年3月31日之前提出申请。最初的忍耐期限再延长六个月*也是可能的。*

拥有FHA保险的抵押贷款的借款人寻求更多可用选项的信息，应该访问FHA的 [新冠肺炎房主资源](https://fha.gov/covid-19.html) 网页。
