# how to teach kids about sexism 1846110388

> 原文：<https://lifehacker.com/how-to-teach-kids-about-sexism-1846110388>

看着卡玛拉·哈里斯上周宣誓成为我们国家的第一位女性副总统(更不用说我们的第一位有色人种副总统)，对许多妇女和女孩来说，这是一个胜利的时刻。这也提醒人们，为了取得这样的成就，女性要克服多少性别歧视和厌恶女性的心理。这提醒我们需要教育我们的孩子关于这些问题，从幼年开始，贯穿他们的整个童年。

### 性别陈规定型观念在很小的时候就开始形成

艾琳·帕尔克博士是惠特曼学院 的心理学副教授，她告诉我，年仅两三岁的孩子不仅根据性别给人贴标签和分类，而且他们已经开始形成性别刻板印象。这种情况的发生部分是因为我们每天都在使用明确的标签，比如一位幼儿园老师说，“早上好，男孩女孩们”，或者一位家长说，“你今天在操场上和小男孩玩得开心吗？”这部分是因为注意到颜色、服装、和玩具经常因性别而不同。



Pahlke 说:“孩子们正在获取这些信息，发展模型表明，一旦性别在心理上变得突出，那么一旦孩子们注意到，‘哦，这是一个重要的类别，我们应该注意’——他们就会努力找出女孩和男孩的[特征]。”PAH lke 的研究重点是儿童和青少年如何形成他们对性别和种族的看法。“我们的环境中有很多线索给孩子们提供了关于经典性别刻板印象的信息。”

这是因为即使你，作为一个家长，可能会买中性的衣服和玩具，给他们读充满权力或领导职位的强大女性角色的故事，但当他们打开电视或走出家门时，他们并没有看到这些。在沃尔玛或塔吉特百货的玩具货架上走一圈，你会看到颜色和图片是如何被用来给玩具分类的，是男孩还是女孩。没过多久，孩子们就开始内化这样一种特质，即善良或甜美更经常地适用于女孩，而男孩则更注重身体素质；她们得到的信号是，女性更经常负责打扫房间，而男性则负责割草；他们注意到，在他们的学校、社区、政治中的“领导者”通常是男性。

“他们很早就学会了这些规则，不幸的是，不管你作为父母希望教给孩子什么样的性别知识，这种情况都会发生，”Pahlke 说。“它在我们的文化中无处不在，无处不在，以至于孩子们只要生活在我们的社会中就会形成刻板印象。”

她说，这就是为什么从小就直接和孩子谈论性别歧视很重要。如果你想让他们认识到性别偏见，你需要教他们性别偏见。



“我认为在种族方面有一些相似之处，”Pahlke 说。“如果你想让孩子们认识到种族偏见，你需要和他们谈谈种族问题。就像“色盲”的观点不起作用一样，性别盲的观点在教育孩子性别歧视方面也不起作用。如果你想让他们了解性别歧视，你必须真正谈论性别偏见的问题。”

### 为他们定义“性别歧视”

Pahlke 说，她和她的同事进行了一些研究，他们走进幼儿园教室，明确地向孩子们讲授性别歧视。他们首先给孩子们这个词，然后定义它的意思——不公平地对待某人或因为性别对某人做出判断。

他们会给出这样的例子，比如有人说只有男孩或女孩才能玩某个游戏。或者说一个女孩有一个“男孩发型”或者有些人认为男孩应该当医生，而女孩应该当护士。然后他们会和孩子们一起工作，想出他们在听到这样的陈述时可以使用的回应，比如，“这不公平，这是性别歧视:男孩和女孩可以玩每一个游戏。”



“我认为这需要明确的指示，即这是一些人所想的或这是一些人所说的，这是不对的，如果你听到有人说或做类似的事情，这是我们可以回答的，”Pahlke 说。“然后和他们一起练习。”

### 指出它是不够的

许多孩子在上小学的时候，就知道美国从来没有过女总统。但是，Pahlke 说，如果我们不谈原因——性别歧视——他们很可能会得出自己不正确的结论，比如因为女孩不够聪明不能当总统，或者因为男孩读更多的书，或者因为女人不想让 T2 当总统。

因此，当指出现实世界中突出性别歧视结果的例子时，将这些例子与它们的根源联系起来也很重要。



“如果我只是对我的孩子说，‘哦，我想知道为什么学校董事会里没有更多的女性’，然后我停了下来，我的孩子可能会从那次互动中走出来，心想，‘哦，这是因为女性不是好的领导者，’”PAH lke 说。“所以我认为指出这一点很重要，然后我认为就与歧视相关的问题进行后续对话也很重要。”

随着孩子的成长，我们应该让男孩和女孩都有能力去寻找性别歧视和性别歧视的例子——并且在他们看到时大声说出来。应该鼓励男孩和女孩积极支持妇女平等。比如说，通过参加妇女游行来和你的女儿建立联系是很好的——但是带上她的父亲和哥哥一起去更好。
