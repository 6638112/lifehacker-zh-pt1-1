# 如何通过清除浏览器缓存让你的 iPhone 或 iPad 更快

> 原文：<https://lifehacker.com/how-to-make-your-iphone-or-ipad-faster-by-clearing-its-1847301581>

如果你的 iPhone 或 iPad 运行速度比正常情况下慢，你只需要清空设备的浏览器缓存，就能快速提升性能。每个浏览器应用程序都存储了网站数据的缓存，如 cookies、保存的登录信息、图像等。这有助于您下次访问时页面加载得更快，并且让您即使在关闭浏览器后也能保持登录状态。这对你经常访问的域名很有帮助，但是你缓存中的很多文件都是无用的。这些额外的小数据很快就会累积起来，占据你的 iPhone 或 iPad 相当大的内部存储空间。设备上剩余的空间越少，运行速度越慢。



清除浏览器缓存不会突然使设备的硬件超频，但可以帮助应用程序加载更快，使 iOS(或 iPadOS)运行更流畅。定期这样做将保持您的 iOS 设备平稳运行，尤其是当您使用 web 浏览器时。如果你注意到其他错误，比如突然的浏览器崩溃，这也应该是你首先尝试的。如果你的设备在清空缓存(或者缓存，如果你经常使用多个浏览器)后仍然滞后，是时候采取更严厉的措施了。

清除浏览器缓存的唯一缺点是，它会删除您保存的登录信息以及您为某些页面保存的任何独特设置。每次清除浏览器数据后，您都需要重新登录您使用的网站。这并不是什么大不了的事情，但它可能会很烦人——尤其是对 Safari 用户来说，因为清除缓存会影响所有登录到你 iCloud 帐户的设备*。*

*尽管如此，如果 iOS 设备运行速度比正常情况下慢，清除缓存是提高其性能的一个简单方法。以下是如何在各种浏览器中清除缓存——[以下是如何在安卓](https://lifehacker.com/android-smartphone-running-slow-try-deleting-the-app-c-1833469532) 上做同样的事情。*

## *如何在 Safari 中清除缓存*

1.  *打开 iPhone 或 iPad 的设置应用程序，然后从应用程序列表中选择 Safari。*
2.  *点击**“清除历史记录和网站数据”***
3.  *出现提示时，确认您要删除数据。*

## ***如何在**勇者中清除缓存*

1.  *打开勇者号，点击**“…”**菜单图标。*
2.  *导航至**设置>隐私。***
3.  *向下滚动并点击**“勇敢的盾牌&隐私”***
4.  *向下滚动下一菜单，选择**“立即清除数据”***

## ***如何清除** Chrome 中的缓存*

1.  *打开 Chrome，点击菜单栏上的三点图标，然后进入**设置>隐私>清除浏览数据。***
2.  *确保 **Cookies** 、**站点数据**和**缓存的图像和文件**都被选中，如果你想释放尽可能多的空间，将日期范围设置为**、【所有时间】**。*
3.  *点击**“清除浏览数据”***

## ***如何清除** DuckDuckGo 中的缓存*

*   *打开 DuckDuckGo 应用程序，点击浏览器底部的 fire 图标。*

## ***如何清除**边缘的缓存*

1.  *打开 Edge 并点击**“..”**图标在右下方。*
2.  *进入**设置>隐私>清除浏览数据。***
3.  *点击**“清除浏览数据”**，然后确认您的选择。*

## ***如何清除** Firefox 中的缓存*

1.  *在 Firefox 中，点击三线菜单图标，进入**隐私>设置>数据管理。***
2.  *选择您想要删除的数据类型，然后点击**“清除私人数据”**清空您的整个浏览缓存，或者点击**“网站数据”**并清除从特定网站保存的数据。*
