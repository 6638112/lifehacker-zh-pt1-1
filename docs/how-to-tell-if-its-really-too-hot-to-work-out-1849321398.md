# 如何判断天气是否真的太热而无法健身

天热的时候，你仍然可以经常安全地锻炼——[在这里查看我们的推荐](https://lifehacker.com/the-best-workouts-to-do-when-its-hot-out-1849316676)——但是我们的身体有其极限。确实有一些温度和条件使长时间锻炼变得危险。这里有一些指导方针，可以帮助你确定何时停止户外锻炼。

## 使用湿球温度

除了查看当天的天气预报，了解炎热如何影响你的身体的最好方法是查看一个叫做湿球温度或WBGT的指标。这源于这样一个想法:如果你用湿布包裹一个老式温度计的球，温度计就会给出读数，告诉你空气能让你降温多少。(我在简化；实际过程更加复杂， [涉及三个温度计](https://www.weather.gov/media/safety/heat/2020-WBGT-Handout.pdf) 。)

这很重要，因为我们身体的主要冷却机制之一— 出汗— 在湿度高的时候不能很好地工作。幸运的是，你不需要温度计和湿布:Y 你可以通过在图表上查找温度和湿度来近似它，或者你可以使用像WeatherFX([iPhone](https://apps.apple.com/us/app/weatherfx/id539498084)， [Android](https://play.google.com/store/apps/details) )这样直接报告它的天气app。

## 根据气温调整训练

现在你有了WBGT，你需要决定如何处理它。美国足球有一张的地图，将全国分为三个地区，不同地区取消练习的截止日期不同。(如果你住在德克萨斯州，他们会认为你比住在明尼苏达州能应对更多的高温。)同样，这里有针对公路赛 (如马拉松)组织者的 [指南。如果WBGT超过82度，比赛应该取消。73以上“极度谨慎，节奏较慢”强烈推荐。](https://racemedicine.org/wp-content/uploads/2019/08/WBGTIndex.pdf)

举个例子:82度的WBGT可能发生在75度湿度90%的天气，84度湿度50%的天气，或者100度湿度10%的天气。

对于一般的建议， [以下是美国运动医学学院为不适应高温锻炼的人推荐的](https://journals.lww.com/acsm-msse/Fulltext/2007/03000/Exertional_Heat_Illness_during_Training_and.20.aspx) :

*   如果WBGT温度超过华氏65度，多休息，注意补水。
*   **如果WBGT超过72** ，要额外休息并限制锻炼的持续时间。
*   如果WBGT超过78 ，要额外休息，限制持续时间，也要降低强度(例如，不要试着跑得一样快。)
*   如果WBGT超过82 ，你应该尽可能多的休息，做上面所有的，并密切关注中暑的迹象，因为这里的风险很高。
*   **如果WBGT在86** 以上，就收工回家。

对于适应高温的人来说，这意味着他们已经在高温下安全地锻炼了至少几周，安全指南可以放松一点，因为你的身体已经学会了如何更好地降温。在这些情况下，截止值如下所示:

*   **如果WBGT在72** 以上，你仍然可以正常锻炼，但要格外注意补水。
*   **如果WBGT超过82** ,“谨慎计划激烈或长时间的锻炼”，并留意高风险人群的中暑迹象。
*   **如果WBGT超过86** ，限制剧烈运动，减少在炎热天气下的暴露。注意中暑的迹象。
*   **如果WBGT高于90** ，即使是水土不服的运动员，风险也太大了。

所有这些建议都伴随着通常的预防措施。为你的补水休息带来额外的液体和电解质，并确保找到阴凉的地方降温。(装有一袋冰的儿童泳池是美国足球指南中的建议之一。)

[了解中暑和中暑的迹象和症状](https://lifehacker.com/know-the-signs-of-heat-exhaustion-and-heat-stroke-1827388999) 也很重要，如果你在自己身上，或者你的队友或锻炼伙伴身上观察到这些迹象和症状，该怎么办。中暑是911级别的紧急事件，因此如果需要，请寻求帮助。
