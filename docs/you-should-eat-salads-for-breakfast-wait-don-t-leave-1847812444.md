# 早餐应该吃沙拉(等等，别走)

> 原文：<https://lifehacker.com/you-should-eat-salads-for-breakfast-wait-don-t-leave-1847812444>

首先我要说，我不反对油腻的咸肉鸡蛋和奶酪。然而，作为一个包容大众的人，我也被 [完全说服了早餐吃沙拉的理由](https://lifehacker.com/the-case-for-eating-salad-for-breakfast-1796685555) 。它们很充实，很简单，最重要的是，它们帮助我带着轻微的优越感开始我的一天。声明:我不是营养师。我不是古鲁。我只是一个喜欢在早上吃沙拉的作家，理由如下。



## **以强势开始新的一天**

众所周知，你醒来时的决策会为你一天的剩余时间做好准备。这就是为什么我选择晨跑，也是为什么我*有时候*会读一本书的一章，而不是躺在床上浏览 Twitter。

早上第一件事就是选择健康和令人满意的食物，我会有一种成就感，这种成就感会伴随我一整天。作为参考， [这正是我早上第一件事](https://www.google.com/search?q=lady+laughing+at+salad&source=lnms&tbm=isch&sa=X&ved=2ahUKEwiao66dwLbzAhVKMlkFHYVgCf0Q_AUoAXoECAEQAw&biw=1181&bih=722&dpr=2) 的样子。

## **一般来说多吃蔬菜**

许多经典的“美国”早餐，如华夫饼、百吉饼或谷类食品，与被不公平地归类为“午餐”或“晚餐”的正餐相比，通常营养密度较低。我不喜欢把吃蔬菜看作是一件“不碍事”的事情，但是在一天的第一餐中加入更多的蔬菜是一个很好的策略，可以让你的饮食更加丰富。

## **让您的“业务”运转起来**

有趣的是，早餐沙拉得到了我所有的...消化过程...流动？我不擅长含沙射影。从植物性食物中发现的所有纤维开始我的一天真的 [帮助我走出](https://lifehacker.com/how-to-make-yourself-poop-1846456743) ，好吗？

## **尝试新食谱**

如果你正在想象一个懦弱的花园沙拉，是时候扩展你的视野了。我个人早餐沙拉经历的关键是从一堆绿色蔬菜开始，但是从那里，你可以用你的蛋白质和所有其他种类的配料获得创造性。你可以从[r/eatscheapandhealth](https://www.reddit.com/r/EatCheapAndHealthy/comments/jm5vlm/breakfast_salad_fast_healthy_flexible/)中找到大量的想法。不管你什么时候吃，看看我们的 [秘诀，打造](https://lifehacker.com/11-secrets-to-building-a-better-salad-1846402059) 更好的沙拉。
