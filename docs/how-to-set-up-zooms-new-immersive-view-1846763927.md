# 如何设置 Zoom 的新“沉浸式视图”

> 原文：<https://lifehacker.com/how-to-set-up-zooms-new-immersive-view-1846763927>

不用说我们 [已经厌倦了在缩放](https://lifehacker.com/how-to-overcome-zoom-fatigue-1846347171) 的小框里看到同事的脸——也许我们甚至期待再次在同一个房间里。借助 Zoom 新的沉浸式视图功能，您可以在同一个虚拟空间中加入其他与会者，如会议室、咖啡厅、教室、或(奇怪的是)美术馆。因此，看起来你的同事坐在一个座位上，或者(又一次，奇怪地)挂在你旁边的一个相框里，而不是无聊的方形瓷砖。



默认情况下，免费和专业帐户会启用沉浸式视图，但要实际使用它，您必须使用桌面应用程序的更新版本(5.6.3 或更高版本)。Zoom 有 [指令，用于在其他账户类型中启用](https://support.zoom.us/hc/en-us/articles/360060220511) 功能。

以下是如何在会议中使用它。

### 如何启动沉浸式视图

会议主持人可以控制沉浸式视图。如果您想为您的会议启用它，请单击右上角的“视图”按钮- (您可以在画廊视图和演讲者视图之间切换)。

选择“沉浸式视图”并选择自动选项，该选项包括场景所能容纳的尽可能多的参与者，或者选择手动选项，该选项允许您自己添加和删除参与者。

然后选择您想要使用的场景。Zoom 针对不同的会议规模提供不同的虚拟场景，因此您可以选择适合所有与会者(最多 25 人)的背景。容量显示在每个场景图块上。选择开始启动沉浸式视图。

您将返回到视图按钮来更改场景，或者通过选择另一个视图选项并单击停止来结束沉浸式视图

### 如何定制沉浸式视图

作为主持人，您可以在虚拟场景中移动参与者，或者添加迟到的人。只需拖放，视频就会弹出来。未分配的参与者将出现在场景上方。您还可以通过点击参与者的视频并缩小或扩大蓝色框架来调整参与者的大小。

当然，沉浸式视图有一些限制。如果你有 25 个以上的参与者，只有前 25 个会出现在你的虚拟场景中，其余的会显示在屏幕顶部通常的缩略图条中。沉浸式视图在屏幕共享期间暂停，在临时会议室中不可用。虚拟场景也不会转换为会议录音，而是会显示画廊和/或演讲者视图。

最后，如果你使用的是旧版本的 Zoom 或移动应用程序，即使你的其他同事正在使用它，你也无法看到身临其境的视图。
