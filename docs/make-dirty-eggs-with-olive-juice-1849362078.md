# 用橄榄汁做脏鸡蛋

> 原文：<https://lifehacker.com/make-dirty-eggs-with-olive-juice-1849362078>

鸡蛋本身真的很淡，所以它们在很大程度上得益于添加的调味料，如一点辣椒酱、番茄酱或洋葱粉。我喜欢用许多 [酱汁和](https://lifehacker.com/elevate-the-humble-boiled-egg-with-flavorful-dips-1849348798) 蘸鸡蛋，但最近的一项发现让我在冰箱里囤积了看似空空的橄榄罐。用橄榄汁弄脏你的鸡蛋，让它们变得不可抗拒。

当我说“用橄榄汁弄脏它们”时，我指的是非常脏，就像马提尼。最后你会有第二个特殊的橄榄盐水罐，你只用来装鸡蛋(当然，另一个是用来装 [龙舌兰酒和](https://lifehacker.com/fuck-it-make-your-dirty-martini-with-tequila-1849135066) 马丁尼)。这些盐水是用来将*混合到你的鸡蛋里，然后用勺子将*舀到*鸡蛋上。我理解最初的好奇反应，但鸡蛋橄榄汁非常有意义。鸡蛋有独特的用途。它们温和的风味和蛋白质结构使它们在甜味和咸味应用中都发挥着美妙的功能。这种醇厚的风味让它们可以很容易地与你喜欢的任何酱搭配。如果你是橄榄迷，我暗自怀疑你会喜欢在早餐鸡蛋里加橄榄汁。*

要做一份微妙但美味的咸早餐，在烹饪前将橄榄汁直接加入鸡蛋中。对于炒鸡蛋或煎蛋卷，只需在搅拌时将它倒入鸡蛋中。对于我的炒鸡蛋，我会用两个大鸡蛋和一汤匙盐水。像往常一样搅拌和烹饪。如果你通常加盐、水、牛奶或奶油，不要这样做。反正橄榄汁主要是水、盐和醋。如果你认为你会错过牛奶或重奶油中的脂肪， [添加一点奶酪](https://lifehacker.com/these-are-the-cheeses-you-need-in-your-scrambled-eggs-1829447109) 而不是添加更多的液体来增加丰富度。加入橄榄盐水比简单地加盐给成品鸡蛋带来更复杂的味道。它不会尖叫，“橄榄汁在这里”，但它有一种令人上瘾的美味品质(也称为 [鲜味](https://www.webmd.com/diet/what-is-umami) )，当你意识到你的盘子里没有鸡蛋时，你会泪眼模糊。

如果你喜欢水煮蛋、煎蛋或煮蛋，橄榄汁可以用作调味汁。我发现当有一个结实的杂粮吐司或一个英式松饼时，这是最成功的。要用橄榄盐水做调味汁，只需在吐司上撒一茶匙左右，让它浸透，然后在上面放上你喜欢的鸡蛋。在鸡蛋上加一点盐水，但注意不要过量(通常一汤匙就足够了)，特别是如果你用的是像奶油蛋卷这样更精致的面包，它会变得潮湿并散开。加入一些鳄梨片、薄薄的西红柿片，再放上一片带有橄榄汁的吐司，你就有了一份值得大书特书的早餐三明治。
