# 如何记住“Hone”和“Home in On”的区别

> 原文：<https://lifehacker.com/how-to-remember-the-difference-between-hone-and-home-in-1846778996>

有些事情你想“磨练”*和*有些事情你想“专注”，但这些事情是不一样的。然而，它是那些听起来如此相似并且具有相似含义(如 [更远和更远](https://lifehacker.com/how-to-remember-when-to-use-further-versus-farther-1846649794) 或 [然后比](https://lifehacker.com/how-to-stop-mixing-up-than-and-then-1846581788) )的成对单词和短语中的一种，很容易将它们混淆。但是为了努力磨练我们的语法技能，我们将把重点放在正确的用法上。

### 首先，一些定义

磨砺是指随着时间的推移使某物变得锋利、精炼或完美。你可以真的磨砺(磨砺)一把利刃，或者你可以磨砺一项技能，比如你的卡拉 ok 技能或者你的谈判技能。

动词形式的单词 *home* ，也就是我们在这里使用的形式，意思是“非常准确地向目标或目的地移动或瞄准。”它最常跟随着单词“in on”，例如，狮子正在*追踪它的猎物*。

所以一般来说，如果你在它后面使用“in on”，你可能需要 *home* 而不是 *hone* 。

### 为什么我们首先会感到困惑

如果你总是想在某件事情上磨练自己，你并不孤单。 [韦氏词典说](https://www.merriam-webster.com/words-at-play/home-in-or-hone-in) 这可能是因为“家”作为动词的用法远不如它作为名词的用法为我们所熟悉:

> 就词汇而言，home 这个动词相对来说比较年轻。这个名词可以追溯到古英语，但是我们最早看到这个动词在使用的证据是在 1765 年，当时它被用来表示“去或回家”在接下来的一百年里，这个动词发展出一种动物特有的意义:一只动物回到它的家或出生地就被称为“归巢”通常所说的动物是鸽子，特别是信鸽。

然而，由于动词形式对许多说英语的人来说不太熟悉，而且 *m* 和 *n* 的发音非常相似，人们经常错误地在中使用 *hone，以至于许多 [词典现在将它列为](https://www.merriam-webster.com/dictionary/hone%20in)——尽管有很多人认为这是一个错误。*

当然，如果你想完全避免争论，你可以简单地用[](https://www.merriam-webster.com/dictionary/zero)*中的零来代替你的目标。*
