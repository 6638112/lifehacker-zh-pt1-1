# 为什么你不应该相信重量机上的数字

> 原文：<https://lifehacker.com/why-you-shouldnt-trust-the-numbers-on-weight-machines-1847222119>

为什么你可以在平常的健身房做 250 磅的压腿运动，但如果你去城市另一边的另一个地方只能做 200 磅？你并没有突然变弱，与流行的神话相反，这并不是因为一个健身房在撒谎奉承你。机器只是*不同*。

## 机器设计很重要

我们以压腿为例，因为几乎每个健身房都有。这是一种机器，你有某种座位，和某种脚的平台。你可以用腿推动平台，而座位则保持不动，反之亦然。

制造这种机器有许多不同的方法。有些是垂直的，脚在臀部正上方。有些是水平的。有些人让座位滑上斜坡，一边滑一边拉着一堆重物。每一台都是完全不同的机器，有斜面和砝码组的机器与你的脚放在上面并在上面放置砝码的机器校准不同，这并不奇怪。

回想一下 [小学物理](https://www.livescience.com/49106-simple-machines.html) 一分钟。当一个机器有一个杠杆臂时，比如肩扛机，杠杆臂的长度会改变你在举重时的机械优势(或劣势)。缆索机器中滑轮和缆索的数量也会改变移动重物堆叠所需的力的大小。如果重物在有角度的轨道上移动，角度的陡度将影响重物移动的难易程度。

甚至当你比较两台看似相同的机器时，也会有细微的差别。一个可能比另一个润滑得更好，让它的零件更容易滑动。它们也可能不完全相同；也许制造商在 2018 年和 2019 年之间将滑轮移到了不同的位置。你明白了。

顺便说一下，器械也不等同于自由重量。仅仅因为你可以压腿 X 磅，并不意味着你可以在你的背上负载一个杠铃，然后蹲下同样的量。

## 那么数字是没有意义的吗？

一点也不。那个机器上的数字*很重要。如果堆叠中的一个盘子上写着 50，下一个盘子上写着 55，那么你知道如果你能重复 55，你就比你只能在 50 时重复更强。*

当你去一个新的健身房时，凭感觉去。如果你能在旧健身房的 55 针上做 8 次，找一个你能在新器械上做 8 次的重量。如果你能把它从机器里拿出来放在秤上，那么砝码上的数字或者砝码组的重量都无关紧要。

把它想象成游戏中的货币:你不会浪费时间在谷歌上搜索多少垄断美元等于多少 [【罗布】](https://lifehacker.com/how-to-block-ads-and-in-app-purchases-from-your-kids-ap-1830125754) ，因为这两者没有理由以任何有意义的方式相互匹配。每一种在它自己的世界里都有用，在别的地方都没用。

所以是的，绝对把那些数字记在你的 [训练日志](https://www.theonion.com/guy-at-gym-has-precious-little-diary-to-keep-track-of-a-1819578810) 里。他们会帮助你追踪你的进步。但是不要期望它们在城市另一边的另一台机器上是相同的。
