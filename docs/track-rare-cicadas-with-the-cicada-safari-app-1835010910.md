# 如何追踪今年出现的17年蝉

> 原文:[https://life hacker . com/track-rare-cicadas-with-the-蝉-safari-app-1835010910](https://lifehacker.com/track-rare-cicadas-with-the-cicada-safari-app-1835010910)

如果你住在印第安纳州、俄亥俄州西部、宾夕法尼亚州东部或华盛顿特区附近的任何地方，准备好迎接今年春天一个罕见的奇异现象:X蝉 的出现，它们在过去的17年里一直在地下等待时机。

周期性蝉([magicacid](https://en.wikipedia.org/wiki/Periodical_cicadas)物种)视物种而定，幼虫在地下生活13年或17年。当到了它们生产下一代的时候，它们会在几周内出现、蜕皮、交配并死去。当到了它们的一年，一个地区的树上会爬满巨大的虫子。这整个现象要么很恶心，要么很迷人，这取决于你问谁。(好吧，可能两者都有点吧。)今年的出苗还没有开始，但随着地面变暖，预计将在下个月左右开始。

如果你觉得上次听到17年前的蝉鸣还不到17年，那是因为每一窝蝉都有自己的领地。 [你可以在这里看到它们的地图](https://www.fs.fed.us/foresthealth/docs/CicadaBroodStaticMap.pdf) 。在我居住的宾夕法尼亚州西部，我在2002年和2019年看到了Brood VIII。我看不到今年的幼虫，因为即将年满17岁的幼虫不是生活在这里的。

要跟踪和记录你所在地区的蝉， [从圣约瑟夫山大学下载蝉游猎app](https://scistarter.org/cicada-safari) (安卓和iOS免费)。现在，你可以查看地图寻找目击地点，一旦你当地的蝉开始出现，你就可以给它们拍照，供蝉科学家研究。

即使你不在X种的领地，有时蝉也会弄错时间，提前或推迟几年出现。与此同时， [查看这里](http://magicicada.org/magicicada/broods/) 的育雏地图，看看你是否能期待在另一年里有一些罕见的访客。

*这篇文章最初是在2019年5月为Brood VIII emergence发布的，并在2021年3月19日更新了关于Brood X蝉的更新链接和附加信息。*
