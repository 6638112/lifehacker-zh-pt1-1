# 这种简单的薯条蘸酱组合是一种风味爆发

前Lifehacker执行主编弗吉尼亚·史密斯通过推特向我们提出了这个建议:

> 有人在抖音建议将伍斯特郡酱混合到酸奶油中，作为盐和醋片的完美搭配，昨晚尝试了一下，我的生活将永远不会一样...它基本上创造了一种鱼和薯条的风味体验(就像用醋和少量鞑靼沙司炸出的美味)。

我们现在的副主编乔尔·坎宁安在弗吉尼亚的推特上给我加了标签，提醒我注意潜在的黑客攻击。

看到这条推文大约一个小时后，我从朋友的公寓走到全食超市，拿了酸奶油、伍斯特郡酱和一袋盐醋锅片。然后我偷窥了一下 [原文内容](https://twitter.com/vksmith/status/1549826336507797515) ，将酱汁和酸牛奶混合，直到像视频中建议的那样“呈浅棕色”。我蘸了一片薯片，放进嘴里。

这是一次紧张的经历。

看起来简单、不起眼、听起来很棒的组合*很有冲击力。它味道浓烈，咸得令人难以置信，几乎强烈到无法用“美味”来形容，然而，我无法停止将它铲进嘴里。*

它就像一颗美味的弹头糖果——如此浓郁的味道几乎让人无法抗拒，但它在最后一秒钟退缩了，就像一辆卡车在迎面而来的车流中玩鸡肉游戏。

薯片明显酸咸，蘸酱酸咸，有点甜，有点臭。应该是太多了，但都管用。不过，我还是推荐使用Lea Perrins牌伍斯特郡酱。我去的全食超市没有卖，只有他们愚蠢的商店品牌，没有罗望子和凤尾鱼，这两种成分使伍斯特郡酱如此特别。(尤其是凤尾鱼，它会带来更多的鲜味。)

显然，你可以把这种蘸酱和普通的薯片一起吃，以获得更合理的体验。弗吉尼亚告诉我蘸酱放在沙拉盘里也很好吃。但是，让一个人的味觉一次又一次地经历如此极端的体验，直到你的味觉被酸、盐和松脆的薯片压迫得刺痛，这是一件令人兴奋的事情。
