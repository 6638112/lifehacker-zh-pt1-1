# 当心这些友谊的红旗

> 原文：<https://lifehacker.com/watch-out-for-these-friendship-red-flags-1848676641>

当我们想到有毒的人和关系时，首先想到的往往是浪漫的伴侣、家庭成员和老板。但是朋友也可以归入这一类。无论是你认识多年的人——但你现在才意识到他是个控制欲很强的人——还是你最近才认识的人，我们认为是“朋友”的人实际上可能是我们生活中压力和痛苦的主要来源。

但是因为我们倾向于认为朋友是站在我们这边的，所以当有人在我们的生活中弊大于利时，我们很难意识到。如果我们已经认识他们很长时间了，并且已经习惯了他们的行为，或者认为这是他们性格的一部分，这就变得特别棘手了。为了识别潜在的有害友谊，请注意这些危险信号。

## 友谊红旗的例子

因为每段关系都是不同的，所以在每段友谊中都没有一个标准的危险信号列表。尽管如此，有些场景比其他场景更常见，也更令人担忧。这里有几个例子:

*   你**害怕和那个人**在一起，而 [在你和](https://www.psychologytoday.com/us/blog/lifetime-connections/201505/13-red-flags-potentially-toxic-friendships) 在一起后感觉更糟。
*   **关系是单方面的**，一个人制定所有的计划，和/或 [投入所有的努力](https://www.wellandgood.com/friendship-red-flags/) 来保持联系。
*   他们像他们的治疗师一样对待你——希望你倾听并回应他们所有的问题——但是 [从来没有时间或对你的生活表现出兴趣](https://lifehacker.com/how-to-set-boundaries-with-a-friend-who-treats-you-like-1847575924) 。
*   他们 [只在**需要或者想要从你那里得到什么**的时候伸出手或者表现得像你的朋友](https://www.psychologytoday.com/us/blog/lifetime-connections/201505/13-red-flags-potentially-toxic-friendships) 。
*   他们**不断地批评你**和你和[在生活](https://www.psychologytoday.com/us/blog/lifetime-connections/201505/13-red-flags-potentially-toxic-friendships)和中的选择，以及你不一定能控制的事情，比如你的外表、健康和家庭。
*   他们经常**给你未经请求的建议**并期望你遵循它——即使你已经礼貌地要求他们(甚至多次)不要这样做。

当然，如果一个朋友(或任何其他人)以任何方式进行身体虐待，这是一个普遍的危险信号，也是一个明确的信号，他们不应该出现在你的生活中。
