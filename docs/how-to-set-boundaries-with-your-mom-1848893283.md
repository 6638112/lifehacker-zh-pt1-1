# 如何和妈妈划清界限

> 原文：<https://lifehacker.com/how-to-set-boundaries-with-your-mom-1848893283>

我们和母亲的关系可能很复杂。即使你和你妈妈关系很好，你也可能会有这样的想法，“我希望她不要每天都打电话来”，“我不喜欢她让我为自己的选择感到内疚，”或者“我不喜欢我妈妈干涉我的私生活。”

如果你以前经历过这些感觉，你并不孤单。即使是成年人，在如何与母亲互动的问题上，也很难做到自主和超然。你可能仍然会寻求她的认可，即使同时也很反感。或者你可能很难向她表达你的需求。例如，你可能更喜欢她倾听，而不是给你建议和批评，但不知道如何告诉她你的感受。

这就是创建边界的切入点。

“界限是一项重要的生活技能，对健康的人际关系至关重要。它包括通过口头表达自己来传达你的需求，以及通过行动设定限制的各种行为， [阿米莉亚·凯利](https://angelamelamud-dot-yamm-track.appspot.com/Redirect?ukey=1WKE7ssOmcGTyWI3CmuANqVa9Mm0BqwT60t8JiF3e5PE-0&key=YAMMID-1641495984094&link=http://www.kelley-counseling.com/amelia-kelley-phd-ms-lcmhc-atr-ryt.html) 博士说，她是一位创伤信息治疗师，也是《我希望知道的是什么:在一段虐待关系 后幸存并茁壮成长》一书的合著者。“通过行动设定界限的一个例子是，在事先告知你不能在晚餐时间交谈后，选择不在晚餐时间接电话。”

谈到与母亲建立界限，凯利称之为“一项复杂的努力”，但在试图建立自我意识时“越来越重要”。

她说:“孩子们不断地找出与母亲的界限，但在成年早期，更戏剧性的界限开始建立起来。”“离开家，选择与你母亲不同的生活目标，是发现你是谁的关键。其他需要界限的常见例子包括选择你的生活伴侣，决定你的职业或你想在哪里生活，以及选择你的个人价值观，比如精神信仰或政治倾向。”

凯利说，根据你和你母亲在成长过程中的互动关系，其中一些选择会与你母亲一致，而另一些可能会有很大不同。“如果发生这种情况，界限是必要的，以确保你作为个人受到尊重。”

根据专家的说法，如果你和母亲之间的关系不像你希望的那样平衡，你想开始和她划清界限，以下是你需要知道的。

## 界限的好处

加州心理学家、《从恐惧中获得快乐:让恐惧成为你的朋友》一书的作者卡拉·曼利 博士说，在身体、精神和情感层面上设定界限有很多好处。

“当我们有坚实的界限时，我们能够表达自己的感受，并允许他人表达自己的感受。同样，清晰的界限让我们能够以健康的方式表达我们的个性。”

表达清楚的时候，曼利说健康的情感边界有助于别人知道我们的需求；设定清晰的界限可以缓解困惑，增强个人能力。"良好的界限也为我们提供了物理空间，让我们感到安全和有保障."

## 为什么与母亲保持距离很重要

为了建立健康的关系，与母亲保持明确的界限很重要，曼利说:“当健康的界限不存在时，就会经常发生纠缠(一种抑制个性化的不健康的情感融合)。当母亲和孩子之间没有健康的界限时，就会出现情感上甚至身体上的分离。”

例如，如果母亲不允许孩子在人际关系上有自己的想法，他们长大后可能会模仿母亲的人际关系模式，而不是发展自己的模式。在某些情况下，母亲和孩子在一般信仰、比如政治或宗教问题上没有健康的界限。如果母亲不允许她的孩子探索他们自己的信仰，孩子将被限制为他们母亲的镜像，而不是他们自己的人。

另一个例子，曼利说，如果一个母亲认为她的孩子应该是一个讨人喜欢的人，而不允许他们学习和陈述自己的需求，这个孩子很可能注定会成为一个讨人喜欢的人。此外，一些母亲利用她们的经济手段对一个成年子女施加压力。

“在这些情况下，”曼利说，“母亲可能会侵犯一个成年孩子的界限，并利用金钱作为一种方式来强迫一个成年孩子参与某些行为。”

其他的例子可能包括一位母亲过度参与抚养孩子的实践，变得具有侵犯性，干涉她的女儿抚养自己孩子的方式。

在更严重的情况下，特别是那些通常来自不良童年，与母亲没有健康关系的人，Kelley 说，为了保护自己免受进一步的情感虐待或操纵，更严格的界限是必要的。

“仅仅因为某人是你的母亲并不总是意味着他们对你来说是健康的，”凯利说，“你确实有权要求保持距离，甚至有时练习无线电静默，完全切断关系。”

## 如何与母亲划清界限

根据 Kelley 的说法，虽然与母亲设定界限可能听起来困难且有压力，但有一些方法可以设定界限，这将有助于维持积极的动态和关系。

“首先要开诚布公——她不会知道你没有告诉她的事情，”凯利说。除非你表达你的需求，否则她无法调整自己的行为。对一些人来说，很难直接与一个观点如此重要的人交谈，所以写信或电子邮件可能是开始谈话的一种有益方式。”

她还建议对她的投入、努力以及她过去为你提供的东西表示感谢。

“在母亲和孩子之间创造空间是健康的，但也会导致激烈的情绪，所以表达感激有助于提醒你的母亲你有多在乎她和你的关系，即使你公开反对她或疏远自己。”

当谈到与母亲或你亲密圈子中的任何人建立界限时，曼利建议首先了解你的需求。一旦知道了自己的需求，就可以心平气和，清晰地表达出来。

“一般来说，设定界限的最佳脚本非常简单明了，”她说。“随着新的界限被设定，其他人可能需要一些时间来尊重清晰、健康的界限。”

这里有几个关于边界设置的健康脚本的例子。

1.“请尊重我在这个问题上的界限。你尊重我的需求，这对我和我个人的幸福都很重要。”

2.“我可以理解你不习惯我设定界限，但我知道尊重我的界限对我的幸福很重要。我知道你关心我，所以请尊重我的意愿。”

3.“当我的界限被忽视时，我感到非常不受尊重。你尊重我的需求，这对我个人的幸福和我们关系的健康非常重要。”

同样，添加一句发自内心的“我爱你”通常会让母亲更容易接受界限的设定。

## 为什么与母亲建立界限会如此困难

与同事甚至是好朋友划清界限可能更容易，但当涉及到与我们的母亲划清界限时，就会感到困难。凯利说这是有充分理由的。

“我们和某人越亲近，就越难建立界限，”她说。这是由于关系中的投资。我们投入的越多，关系受损的风险就越大。虽然我们的母亲可能是我们最大的支持者，但她们也可能是我们最大的批评者。通常他们比任何人都了解我们，，正因如此，他们往往对我们的生活方式有更多的看法。”

这就是为什么和你妈妈建立界限更加重要的原因。因为，正如凯利所说，当我们与母亲有健康的界限时，“我们会更放心地与她分享我们的个性，关系变得更加诚实和真实。当界限得不到尊重时，表达自己就很难感到安全，许多人会发现这种关系受到了损害。”
