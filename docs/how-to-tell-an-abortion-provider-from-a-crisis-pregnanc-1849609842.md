# 如何辨别“危机妊娠中心”的堕胎提供者

> 原文：<https://lifehacker.com/how-to-tell-an-abortion-provider-from-a-crisis-pregnanc-1849609842>

假堕胎诊所通常比真堕胎诊所更容易被发现。他们在网上用堕胎相关的搜索词做广告，给自己起的名字和标识看起来和真实的医疗诊所一样令人困惑。但是他们不做堕胎手术，他们不会帮你堕胎，而且他们可能根本没有医务人员。以下是识别它们的方法。



## 什么是危机妊娠中心？

“危机怀孕中心”是这些假诊所的一个术语。它们通常由宗教反堕胎团体运营，他们的全部交易是他们希望你去他们那里，而不是你可以堕胎的地方。他们会浪费你的时间(也许直到去真正的诊所已经太晚了)，并可能对你撒谎。美国医学协会 [称它们为“合法的和不道德的”，](https://journalofethics.ama-assn.org/article/why-crisis-pregnancy-centers-are-legal-unethical/2018-03) 写道:

> 他们努力给人一种印象，即他们是临床中心，提供合法的医疗服务和建议，但他们不受适用于医疗保健机构的监管、许可和资格监督。因为这些中心的所有者和雇员的宗教意识形态优先于在这些中心寻求护理的妇女的健康和福祉，所以妇女无法获得关于所有可用选项的全面、准确、基于证据的临床信息。虽然危机怀孕中心享有第一修正案的权利保护，他们传播错误信息应被视为违反道德，损害妇女的健康。

这些组织的广告和网站通常会让你确认怀孕，并了解你的“选择”当你到达那里时，你可能会得到验孕测试(你可以在商店买到的那种)或“有限的”超声波检查，这两种检查都不需要医学培训。如果你希望转介到堕胎提供者，你会发现他们无法帮助你。

## 要找什么

如果你在寻找一个可以接受实际医疗保健的地方，包括与人谈论堕胎和节育的能力，危机妊娠中心不是那个地方。为了区分，寻找这些危险信号。

### 谷歌搜索结果里有小字吗？

谷歌现在给危机妊娠中心贴上“不提供堕胎”或“可能不提供堕胎”的标签，这些中心在某些与堕胎相关的搜索词下做广告。(像计划生育这样的合法诊所可能会被恰当地贴上“提供堕胎服务”的标签。)但是， [正如彭博报道的](https://www.bloomberg.com/graphics/2022-google-search-abortion-clinic-crisis-pregnancy-center-ads/) ，他们只有在你搜索了某些术语，并且你看到的结果是广告的情况下才会这么做。

我试过了，对于我所做的搜索来说，这绝对是真的。用我所在城市的名字在谷歌上搜索“堕胎”,会出现提供堕胎服务的诊所，并贴上正确的标签。太好了！但当我搜索“怀孕选项”和城市名称时，所有的搜索结果都是假诊所，名字听起来很医学。它们是堕胎搜索标记为不提供堕胎的相同中心，但是在这次搜索中，它们没有被标记。

如果你使用谷歌搜索，考虑在你的搜索词中使用“堕胎”这个词。我们在 Gizmodo [的朋友推荐](https://gizmodo.com/how-to-get-an-abortion-privately-after-roe-v-wade-11-ti-1848884984) 使用 Tor 浏览器，如果你想保持你的搜索隐私的话。

## 他们提供什么服务？

像“计划生育”这样的真正诊所对他们提供的服务是很明确的:堕胎服务和其他医疗保健服务都有明确的说明。

但是在危机妊娠中心的网站上，你会经常发现没有关于堕胎程序甚至节育的可靠信息。相反，典型的服务是怀孕测试、超声波和“咨询”可能会有一些模糊的“节育”的说法，结果证明是关于不包括避孕药或宫内节育器的“自然”方法的咨询。

这些服务往往被证明是医疗角色扮演。超声波可能看起来和感觉起来像医疗程序，但你不需要成为一名有执照的技术员(在大多数州)就可以打开机器，在某人的肚子上摩擦。危机妊娠中心 [被发现在超声波](https://19thnews.org/2021/10/crisis-pregnancy-centers-ultrasounds-accuracy-stakes/) 后给人们关于怀孕的不准确信息，包括给人怀孕太久不符合堕胎条件的印象。

### 一切都是免费的吗？

许多合法诊所提供财政援助，接受保险和医疗补助，或提供其他方式帮助您获得所需的服务，即使您现金短缺。免费服务可能符合这种情况，所以它们本身并不是一个危险信号。但是如果*的一切都是免费的，那么就不会提到付费，他们会提前给每个人提供这个？这绝对值得一试。请记住，如果他们只是试图说服你保持怀孕，这不是一个专业的服务，需要花费大量金钱来提供。*

### *注意操纵语言*

*牙科诊所不会用“害怕？迷茫？想洗牙吗？了解您的选择。”他们只是说，嘿，我们是牙医，你可以在准备好的时候预约清洁或检查。*

*当你找到一个合法的诊所网站时，这就是你应该期待的。直截了当的信息，而不是忧心忡忡的年轻女性的照片，她们向你保证，你可以秘密地“制定计划”，怀孕“不会让你失去希望或未来。”*

*如果你浏览一下危机怀孕中心的网站，你会发现更微妙的东西。比如这个“给男人”的页面， [鼓励你的男朋友告诉“父母和其他人”关于怀孕的事情](https://elwcc.org/be-informed/for-men) ，因为“保守这个秘密只会增加你和她的压力。”全世界的男朋友们，不要这样做。*

*如果你不确定在诊所网站上读到的东西的语气，试着想象它来自牙医。提供的服务列表？当然可以。关于您的伴侣应该如何与您的家人讨论您的护理细节的页面？呃没有。*

### *在阅读他们的网站之前，教育自己关于堕胎的知识*

*危机妊娠中心经常传播关于堕胎的错误信息，或明或暗。他们可能暗示堕胎有风险( [远比怀孕](https://www.scientificamerican.com/article/pregnancy-is-far-more-dangerous-to-women-than-abortion/) )或者人们经常后悔生了一个( [95%不要](https://www.ucsf.edu/news/2020/01/416421/five-years-after-abortion-nearly-all-women-say-it-was-right-decision-study) )。他们可能会提供“堕胎逆转”，这是 [而不是](https://lifehacker.com/abortion-reversal-is-not-a-thing-1822626972) 。*

*真的和假的诊所都可能在他们的网站上有关于堕胎风险的信息。不同的是，危机怀孕中心试图吓唬你；真正的诊所提供关于并发症如何普遍发生的信息，以及当它们发生时如何治疗。 [下面是一个恰当讨论风险](https://abortionclinics.org/risks-and-complications/) 的例子。*

*如果你对堕胎或任何其他生殖健康护理有疑问或担心，去一个合法的来源(即使它不是你附近的提供者)并仔细阅读。计划生育有 [网上有丰富的信息](https://www.plannedparenthood.org/learn/abortion/considering-abortion) ，供新手参考。*

*“计划生育”还推荐了“所有选择对话”( T2)和“所有选择对话”( T4 ),它提供了危机妊娠中心假装提供的那种同伴咨询:无偏见的对话，在对话中你可以获得关于“所有选择”的信息。他们还推荐[Faith loud 神职人员咨询热线](https://www.faithaloud.org/find-clergy-support/clergycounseling/) ，它可以在怀孕期间提供宗教和精神咨询，而不会试图影响你的医疗保健决定，如果那是你正在寻找的东西。*
