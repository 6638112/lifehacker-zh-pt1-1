# 早餐吃西葫芦的最佳方式

> 原文：<https://lifehacker.com/the-best-way-to-eat-zucchini-for-breakfast-1847513297>

我不是女巫，所以我不知道夏天什么时候正式“结束”，但对我来说，这个季节总是在我生日后感觉结束。这个夏天真的过得很快，我“做”的最有趣的事情就是非常喜欢西葫芦。我曾经认为我不喜欢西葫芦。现在我喜欢它。人是会变的。



它始于种植西葫芦。由于相当自我，我更有可能关心我投入时间和精力的事情，所以我终于开始真正感受到 T4 对这些大南瓜、它们的花和为它们授粉的小蜜蜂的喜爱。(我以前很怕蜜蜂！现在我喜欢他们了！人是会变的！)

但是在我从西葫芦容忍者演变成西葫芦斯坦的过程中，最重要的因素是 T2 Smitten Kitchen 的西葫芦黄油意大利面 T4。那份食谱太棒了，它让我意识到我在南瓜方面最大的失误:我只是没有煮够。

制作西葫芦黄油意大利面酱非常简单。你只需拿一束西葫芦丝，用黄油和盐(也许还有一点大蒜)烹饪，直到它们变成一团果酱，轻微焦糖化，可涂抹的糊状物——e 极好的糊状物，除了意大利面还有其他用途。

我最喜欢的时间之一是在吃早餐的时候。我做了玉米粥(有时我会加入樱桃番茄)，涂在吐司上，然后在上面放一个煎蛋。如果那天早上我摘了几朵西葫芦花，我会把它们快速炒一下，然后把它们放在鸡蛋上面。西葫芦糊烤面包是一种美丽的东西，是早餐吃蔬菜的最佳方式。要做到这一点，你需要:

*   一杯碎西葫芦
*   2 汤匙咸黄油
*   1/4 杯樱桃番茄(可选)

*   两大撮海盐
*   1 个鸡蛋(或者 2 个，如果你是炒鸡蛋的话)
*   某种面包
*   2(或更多！)西葫芦开花

那是浆糊——好东西。当面糊正在变糊的时候，继续烤你选择的面包，煮你的鸡蛋。油炸、水煮、炒鸡蛋——都不错，但今天我感觉很颓废，于是点了一份 [焦糖奶油蛋](https://lifehacker.com/fry-your-eggs-in-heavy-cream-1846839933) 。

用叉子或漏勺把面糊从锅里拿出来，注意在锅里留一点油。把面糊涂在烤面包上，把鸡蛋放在面糊上，然后在剩下的面糊油里快速炒西葫芦花，直到枯萎。将花朵覆盖在鸡蛋上，然后立即食用。
