# 利用这种视角的转变，让人们在这个糟糕的市场上购买汽车变得可以忍受

> 原文：<https://jalopnik.com/use-this-perspective-shift-to-make-car-shopping-in-this-1848142627>

经销商库存很低，这种情况看起来不会很快改变。如果你现在不需要买车，等待一年或更长时间可能是明智的，但根据凯利蓝皮书，只有 16%的购物者计划推迟购买。 所以，如果你现在必须买东西，就需要转换视角来避免很多挫败感。

对自己想要的东西过于具体的买家会遇到最多的障碍。我最近有几个客户给我发来了一系列购买新车或二手车的请求，结果我发现整个国家只有不到几辆车符合前两三个参数。

谈到新车，我注意到的另一个模式是，Autotrader、Cars.com 和 CarGurus 等第三方网站上的大量活跃房源都是实际上不出售的车型。虽然这些网站总是有一些已经消失的汽车“死亡列表”，但对于一些品牌来说，我看到超过 60%到 70%的广告汽车是不可用的。这主要是因为这些列表显示的是“在途”汽车，它们不在停车场上，但正在路上，而且经销商基本上已经将这些单元预售给等待数月的客户。

那么购房者应该怎么做呢？我在以前的一篇文章中提到，颜色、装饰和选择方面的灵活性是管理这个困难市场的关键之一，但我将进一步提出“保留我”车的想法。

一辆二手车不会是你的理想选择，而是一辆在一段时间内“足够好”的车，然后你可以重新进入市场。这里的关键是诚实地评估你需要什么，而不是你想要什么。如果你想要一辆混合动力或插电式跨界车，但似乎找不到，或者现有车型价格过高，或许可以转向更具实用性的汽油车型。这些汽车仍然会让家人四处走动，但它们只是满足了你向电动未来靠拢的愿望。

如果你想要一辆带手动变速箱的运动型紧凑型车，但发现选择第三个踏板会大大减少你的选择，也许最好专注于一辆像样的通勤车。这不会很有趣，但它会很可靠，并在接下来的几年里降低你的运营成本。

几年前，我租了一辆马自达 3，当租约到期时，我还在读研究生，正在做一些家庭项目，所以我的预算有限。我更喜欢性能更好的车，但六速马自达 3 是一款不错的通勤车，也足够有趣。由于市场上没有其他我喜欢也买得起的车，我决定买断我的租约，把马自达作为一辆“备用车”用几年，直到我感觉足够舒适可以升级。我的报酬很低，而且我有一辆可靠的车，很好地完成了工作。

尽管花钱买你并不真正喜欢的相对昂贵的东西听起来并不那么吸引人，但这取决于你追求的可能没有更好的选择。我通常不建议听取摇滚乐队的购车建议，但米克·贾格尔确实说得很好——“你不可能总是得到你想要的...但是如果你尝试一下，你会发现…你得到你需要的。”
