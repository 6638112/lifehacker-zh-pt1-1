# 如何用谷歌在Pixel上的“人像照明”美化你的照片

> 原文:[https://life hacker . com/how-to-beautiful-up-your-photos with-Google-portrait-ligh-1845438311](https://lifehacker.com/how-to-pretty-up-your-photos-with-googles-portrait-ligh-1845438311)

谁需要最新的像素呢？谷歌现在正在向旧设备推出其“人像照明”功能——之前在Pixel 4a 5G和Pixel 5发布会上宣布过。我刚刚在我的谷歌Pixel 3 XL上启动了它，发现它是一个伟大的工具，用于人工调整你的肖像图像的光源。

Watch

开始之前，首先确保你运行的是最新版本的安卓系统(**安卓11** )和谷歌照片应用(2020年10月 19日更新)。打开应用程序，然后找到您用设备拍摄的肖像模式照片。你也可以使用你用其他智能手机拍摄的照片，比如三星，甚至是iPhone。

我发现一个奇怪的现象是，不管出于什么原因，谷歌似乎不承认使用iPhone自带的“肖像模式”拍摄的照片实际上是你的脸。另一方面，当我在iOS上进行常规自拍时，以及在我的Pixel 3 XL或三星Galaxy Note 20上进行自拍或“人像”拍照时，我 t没有遇到任何识别我的面部的问题。

另外， 我在这里用词很谨慎，因为你*将不能*只为任何在“人像”模式下拍摄的照片调整灯光。例如，给你最喜欢的植物拍一张可爱的背景模糊的照片，谷歌不会给你提供调整灯光的选项。正如[Android Police](https://www.androidpolice.com/2020/10/20/portrait-light-in-google-photos-is-rolling-out-to-older-pixels)所报道的，人像照明是一个*人脸专用*编辑功能:

> *“重要的是照片中包含了明显的人脸，无论是合适的人像还是(有时)全身照，都包含了一个人或者两个人，人脸直视镜头或者斜向别处，等等。你必须相信谷歌的AI/ML来识别适当的情况，因为如果你没有在特定的图片上获得选项，就没有办法强制它"*

撇开这些奇怪的事情不谈，为你拍摄的照片中的人脸调整光线是一件非常简单(也很有趣)的任务。若要开始，请在Google相册中打开一张人脸照片。点击“编辑”按钮，它看起来像三个滑块，位于屏幕底部的。向右滚动一点，直到你在**“调整”**部分，你应该会看到一个新的选项为**“人像灯”**如果没有，你的形象就不合格。抱歉。

点击它，你现在就可以通过移动照片上叠加的圆圈来选择图像人造光的方向。您还可以使用屏幕底部的滑块来调整人工照明的强度。如果你想让Google相册选出它认为最适合你的肖像照明版本，点击**“自动”**。

现在，你所要做的就是摆弄你的灯光，直到你对结果满意为止，点击**“完成”，**并保存你新编辑的图像的副本——或者在你完成之前做更多的编辑。

如果你觉得懒，我还发现使用“建议”部分下的“肖像”选项是一个很好的方法，可以快速调整你的照明和模糊图像的背景否则拍摄没有散景。这很快 k，很容易，而且可以产生相当不错的结果。
