# 为什么应该在任天堂 Switch 上启用飞行模式

> 原文：<https://lifehacker.com/why-you-should-enable-airplane-mode-on-your-nintendo-sw-1848756672>

几乎我们所有的设备都内置了飞行模式，随时可以在登机时打开。通常当你*不*飞行时，它的效用是有限的(除非你*真的*不想有人给你打电话)。毕竟，你为什么要故意断开自己与互联网的连接呢？好吧，事实上，有几个原因可以解释你的任天堂 Switch:电池寿命和加载速度。

Watch

交换机是一个非常依赖互联网的设备。您需要使用 wifi 来下载新的更新、在线玩游戏和查看您的朋友。也就是说，在你使用它的大部分时间里，你可能不会做那些事情。除非你只在网上玩游戏，否则你更有可能是一个人玩，玩一些你可以在网下轻松玩的游戏。

连接互联网需要消耗电池寿命，但不仅仅是在你积极使用它的时候。访问 eShop、安装更新或连接到朋友的在线比赛会消耗一些电池，这是有道理的，但每当您的系统开机时，还有许多其他可以说是更消耗电池的互联网活动在进行。

你的交换机不断通过 wifi 进行检查，寻找那些新的更新，检查朋友是否登录以及他们正在播放什么，并发送推送通知，以便你购买新的东西。更重要的是，即使在睡眠模式下，它也会继续做所有这些事情，这意味着除非你一直保持对接，否则当你唤醒它时，你的电池续航时间会少得多。当你知道你不需要互联网时，简单地打开飞行模式就可以阻止所有这些过程的发生。

开启飞行模式还有其他好处，可以延长电池寿命。似乎很多游戏——甚至那些不需要网络连接就能玩的游戏——在最终启动前都要检查一下。这可能会减慢加载过程，尤其是当您*不在 wifi 范围内*但 wifi 仍处于启用状态时。

飞行模式取消了这种互联网检查，这可以使你的游戏启动更快。来自这个 Reddit 帖子 的轶事证据也证实了这一点，一位用户报告说*超级马里奥奥德赛*的“玩分离的欢乐游戏”屏幕在启用飞行模式时移动*快得多*。飞行模式的这一优势可能会激发你启用它，即使电池寿命不是问题，例如当你的交换机对接或充电时。

一个警告:如果你是任天堂 Switch 在线的会员，其中包括云节省，稍微快一点的加载时间或多一点的电池寿命的前景可能不值得牺牲这种能力——特别是因为你正在为此付费。

此外，飞行模式破坏了蓝牙通信，这将导致与您分离的 Joy-Cons 的通信失败。如果您需要在分离时使用 Joy-Cons，您需要从**系统设置>** **飞行模式>控制器连接(蓝牙)**手动启用蓝牙。

## 如何在任天堂 Switch 上启用飞行模式

要将开关切换到飞行模式，请按住主屏幕按钮以访问快速设置。从这里，选择**飞行模式**，然后按 A 启用。您也可以从系统设置中激活飞行模式。
