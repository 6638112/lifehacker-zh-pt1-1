# 如何在三星 Galaxy S21 Ultra 上使用 S-Pen

> 原文：<https://lifehacker.com/how-to-use-the-s-pen-with-a-samsung-galaxy-s21-ultra-1846429071>

三星最近宣布，其旗舰手机 Galaxy S21 Ultra 的用户将可以为其添加手写笔。Galaxy 粉丝对以各种方式使用 S-Pen 的想法感到高兴，这些用途不会令人失望。以下是我们最喜欢的两者结合使用的一些方法。

当然，使用手写笔的一个主要原因是为了画画。在实时消息中，您可以在照片上绘制自己的表情符号和消息，并将其发送给朋友。更酷的是 AR Doodle，它可以让你在 3D 空间中绘图。使用智能选择功能，您将能够快速点击并收集图片或视频。

如果你将笔悬停在屏幕上并按下笔的按钮，你可以弹出一个全新的选项和快捷键菜单，所以开始习惯使用它们，直到它们成为第二天性。三星还宣布推出 S-Pen Pro，这款产品仍处于保密状态，但据传有空中手势和遥控快门，因此很快会有更多产品推出。
