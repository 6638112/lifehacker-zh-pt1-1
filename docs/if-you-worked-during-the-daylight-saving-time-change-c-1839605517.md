# 如果你在夏令时改变时工作，检查你的工资单

> 原文:[https://life hacker . com/if-you-worked-in-the-daylight-saving-time-change-c-1839605517](https://lifehacker.com/if-you-worked-during-the-daylight-saving-time-change-c-1839605517)

这个星期天夏令时提前结束。(下面是我们的 [帖](https://lifehacker.com/the-real-history-of-daylight-saving-time-1823611526) 上的时间变化历史。)除了调整你的时钟，还有另一个*原因值得注意这种变化，尤其是如果你是小时工的话。* 

Watch

去年在Twitter上，纽约国会女议员亚历山大·奥卡西奥-科尔特斯为那些在夏令时结束的周六晚上/周日早上工作的人提供了一个有用的提醒。"...如果你上晚班...当时钟倒退时，你必须加班一个小时，确保你查看了本周的工资单，并获得了报酬！”她在 [推特](https://twitter.com/aoc/status/1191026154918072322) 上写道。“计算机有时会漏掉它。确保你得到报酬——不要让你的劳动被偷走！”

这个建议*也*在Reddit上制造了[r](https://www.reddit.com/r/LifeProTips/comments/dqvqzs/lpt_if_youre_working_tonight_while_daylight)[ounds](https://www.reddit.com/r/LifeProTips/comments/dqvqzs/lpt_if_youre_working_tonight_while_daylight)，用户在这里分享他们处理错过一小时工资的经历。“愚蠢的是，一些工资/时钟系统没有考虑到这一小时的损失，而且不是所有的雇主都有足够聪明的工资管理人员来意识到这一点，所以他们只是根据时钟报告的时间签发支票，”u/krazydavid 写道。“作为一个在IT领域工作了很长时间并与这些系统打过很多交道的人，我见过几次这种情况。”

根据公平劳动和标准法案，你绝对应该得到每小时的报酬。劳工部网站 上的 [写道:“FLSA要求雇员必须计入实际工作的所有时间”，这里指的是一名雇员工作“八”小时，但由于时间变化，1点到2点轮班两次的例子。因此，如果员工的工作环境与上面例子中描述的相似，他或她就工作了...夏令时结束当天的九个小时。”当然，这种*可能会随着你特定雇主的政策而变化，但很可能这是不合法的。*](https://webapps.dol.gov/elaws/whd/flsa/hoursworked/screenEr11.asp)

如果这种情况发生在你身上，首先，一定要检查你的工资条和你所有的工作时间。如果你发现这是不准确的，立即告诉你的雇主关于每小时的报酬(并参考上面的网站)。如果你还有问题，你可以直接通过 [DOL](https://www.dol.gov/whd/local/index.htm) 询问你的具体情况，甚至举报你的雇主，特别是如果它影响到几个员工的话；你也可以秘密地报告问题。

当然，如果你在夏威夷或亚利桑那州，那里不实行夏令时，或者如果你是一名受薪员工，这个问题可能不会影响你(尽管，如果你那天晚上作为受薪员工加班，它*可能会*影响你；再次，检查你的雇主的政策)。
