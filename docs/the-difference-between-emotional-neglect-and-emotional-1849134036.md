# 情感忽视和情感无效的区别(以及它们如何影响孩子)

> 原文：<https://lifehacker.com/the-difference-between-emotional-neglect-and-emotional-1849134036>

当照顾者未能满足孩子的情感需求时，就会出现儿童期情感忽视(CEN)。这可能包括公开的言语虐待，以及忽视或忽略孩子需求的被动行为。情感忽视和情感无效都属于 CEN 的大范畴，对孩子有不同的影响。你如何区分这两者，它们对孩子有什么影响？



这里有两个例子说明情感忽视和情感无效的表现方式:

*   想象一下，一个孩子想告诉父母他们在学校和一个朋友打了一架，让他们心烦意乱，而父母对此不予理会，而不是倾听和支持他们。
*   想象一下，另一个孩子和朋友打了一架，想谈谈，而父母的反应是惩罚孩子“太情绪化”。

现在想象一下这种场景每天发生在两个不同的家庭。这些反应看起来和感觉上不同，因此对成年期有不同的影响。前者是*情绪忽视*，这是一种被动的行为，可能很难注意到它的发生。然而，后者是*情感失效*，是一个否定、批评或否决孩子情感的主动过程。

这两种情况都不理想，在情感忽视的环境中长大的孩子可能会经常经历其中一种或两种情况。当父母没有意识到或回应孩子的情绪时，就会发生 CEN。CEN 不一定是关于父母*做什么*，而是关于父母*不做什么*。

## **童年时期情感虐待的影响**

忽视是 [最常见的儿童虐待形式](https://www.ncbi.nlm.nih.gov/books/NBK459146/) 。然而，意识到你可能经历了 CEN 并不总是显而易见的。在忽视情感的家庭中长大的孩子可能会感觉到有什么不对劲，但不知道是什么。与身体虐待不同，儿童身上没有明显的瘀伤或痕迹，所以直到其影响在成年后出现时才被注意到。

“经历过情感忽视的成年人更有可能出现社交退缩、回避亲密关系、人际关系困难、情绪管理困难、自卑、绝望和应对方式受阻的症状，”斯蒂芬妮·沃尔夫博士说，她是儿童心理学家，也是[【Maven Psychology Group](https://www.mavenpsychologygroup.com/)【Maven Psychology Group】的合伙人，“他们有更高的风险出现各种心理健康障碍，，包括抑郁症和社交恐惧症。这些成年人患边缘型或回避型人格障碍的风险也更高。”

CEN 可以有许多形式，像被动忽视和主动情感失效，所以这里有区别。

## **什么是消极的情感忽视？**

这里有一些被动情感忽视的例子，以及孩子从经验学习中可能得到的东西。

1.一个孩子经常心烦意乱。父母不会注意到，也不会理会孩子想要表达他们感受的任何尝试。

孩子得到的教训: T 他们的感受并不重要。

2.一个孩子会犯错误，做出错误的选择。父母完全忽略了糟糕的选择，并认为他们会自己解决。

孩子得到的教训:他们没有机会好好学习。如果没有一个成年人在那里指导他们，他们可能会在成年后变得过于自我批评，并因为犯了错误而攻击自己。

3.家人避免任何不舒服或情绪化的话题。谈话是肤浅的，不惜一切代价避免冲突。

孩子得到的教训:他们从来没有学会如何有效地沟通和表达需求和感受。他们知道最好避免和别人讨论自己的感受。

4.孩子生气了，父母不同意或与孩子分离。

孩子得到的教训:他们认为愤怒在客观上是不好的。他们学会把它藏在心里，否则它会把人赶走。

## **什么是主动情绪失效？**

这里有一些积极情绪失效的例子，以及孩子能从每个场景中学到什么。

1.一个孩子受伤了。他们试图表达自己的感受，而父母的消极反应是称他们为“戏剧女王”或“过度情绪化”

孩子得到的教训是:他们知道为了变得强大，他们不应该有或表达任何情绪。

2.孩子心烦意乱，父母通过表达更大、更强烈的情绪来压制孩子的感受。

孩子学到的教训:他们被教导要对他人的情感负责，他人的情感比他们的更重要。

3.每当孩子表现出负面情绪时，他们就会被送到自己的房间。

孩子得到的教训:他们内化了负面情绪是不可容忍的，应该受到惩罚的信念。

4.一个孩子需要建议，从父母那里寻求情感上的支持或指导，但却遭到拒绝，他们的父母给他们贴上“需要帮助”的标签。

孩子得到的教训是:他们知道他们不应该有任何需求，他们应该为自己的感觉和情绪感到羞耻。

## **这一切在实践中意味着什么**

根据沃尔夫的说法，“如果一个成年人最近意识到他们在童年时经历了情感上的忽视……这种知识对他们能够显著改善生活是非常有力的。”Wolf 继续说道，“我建议他们向在创伤和情感忽视方面有专长的有执照的临床医生寻求治疗。”

无论你经历了消极的情感忽视还是积极的情感无效，这些影响都是真实的。对于那些经历过 CEN 的人来说，容易自我怀疑和自我批评，或者在沟通和处理情绪方面有困难，这并不罕见。被动忽视可能是微妙的，可能很难确定，这只会让自我怀疑更加强烈。如果上面列出的例子你听起来很熟悉，你可能是和 CEN 一起长大的。你也可以参考这个由专门研究 CEN 的心理学家乔尼斯·韦伯博士提供的 [清单](https://drjonicewebb.com/cenquestionnaire/) 。

好消息是，虽然作为一个孩子你没有选择，但作为一个成年人，你有。治愈的方法有很多，其中之一就是治疗。有许多不同的治疗方式已被证明有助于儿童忽视， [其中一些包括](https://psychcentral.com/health/best-types-of-therapy-for-trauma#trauma-focused-cbt) 认知处理疗法(CPT)、内部家庭系统(IFS)和创伤聚焦 CBT (TF-CBT)。找一个列出这些治疗方法的治疗师会很有帮助。

“通过治疗，他们可以学习更健康的亲密模式，以及如何成功满足他们的情感需求，”沃尔夫说。最终，意识到你的心理健康并寻求改变它永远不会太晚。有一条治愈的路，你已经在路上了。
