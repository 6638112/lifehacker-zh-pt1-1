# 使用这个快捷方式来关闭 iPhone 上的 Wifi 和蓝牙

> 原文：<https://lifehacker.com/use-this-one-tap-shortcut-to-actually-turn-off-wifi-and-1849318943>

当你通过控制中心“关闭”iPhone 上的 wifi、蓝牙或蜂窝数据时，它们实际上并没有被禁用——y 我们的 iPhone 只是简单地将自己与当前网络断开，直到你重新打开这些选项。这是苹果有意识的设计选择，因为它希望确保你仍然可以轻松连接到 Apple Watch 和 AirPods 等外设，以及隔空投送和 AirPlay active 等服务。

但是如果你不在乎这些呢？如果你真的*想要*完全关闭 wifi、蓝牙和/或蜂窝数据呢？根据苹果公司的说法，你需要使用设置应用程序来单独禁用每个功能。然而，还有更好的选择。不要每次想禁用连接时都在设置中摸索，你可以使用内置的快捷方式应用程序来创建一个一键式自动化，这将立即完成。

## 如何使用一个 iPhone 快捷方式来关闭 wifi、蓝牙和蜂窝数据

打开 iPhone 上的“**快捷方式**”应用程序。F 在**我的快捷方式**选项卡中，点击**加**按钮创建一个新的快捷方式。点击“**添加操作**按钮，选择“**设置 Wi-Fi** ”操作。然后，切换到“ **Off** ”选项。接下来，您可以类似地添加“**设置蓝牙**”和“**设置蜂窝数据**”功能。再次将两个功能设置为“**关**”

您的快捷方式现在准备好了。在顶部给它起一个名字，点击蓝色的**菜单**图标。点击“**添加到主屏幕**按钮，自定义图标和标签，然后点击“添加”按钮。现在，您将在主屏幕上看到快捷方式。只需点击它即可在后台关闭 wifi、蓝牙和蜂窝数据(无需认证或额外点击)。

或许创造另一条快捷方式来一次性重新启用 T2 无线网络、蓝牙和蜂窝数据也是值得的。步骤如上所述是，但是这一次，在添加动作之后，选择上的**选项。**

由于这是一个快捷方式，它可以以多种方式触发:你可以将一个快捷方式小工具添加到主屏幕，其中有多个快捷方式，或者你可以要求 Siri 来做 —s ay " **嘿 Siri，(快捷方式名称)**来触发快捷方式。

## 但是飞行模式呢？

让你的 iPhone 进入飞行模式，也可以在控制中心找到，不同于点击关闭个人网络选项。当你打开飞行模式时，它会完全断开蜂窝数据连接(蜂窝数据是飞机不喜欢的连接)。如果您在飞行模式下从控制中心禁用 wifi，它也会完全断开连接，并且会在您下次启用飞行模式时自动完全断开连接。

然而，即使在飞行模式下，蓝牙也不会完全关闭。为了一次完全禁用所有三个连接，您需要使用上面的快捷方式。
