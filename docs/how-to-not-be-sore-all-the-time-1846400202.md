# 如何在运动后一直不酸痛

> 原文:[https://life hacker . com/how-to-not-be-sore-all-time-1846400202](https://lifehacker.com/how-to-not-be-sore-all-the-time-1846400202)

延迟性肌肉酸痛——那种让你在蹲了一天后走下楼梯时畏缩的肌肉酸痛——在你第一天的新锻炼后很常见。但是由于一种被称为重复回合效应的现象，第二天之后发生的可能性不大。

Watch

从本质上来说，引发疼痛的锻炼似乎可以保护你免受持续很长时间的额外疼痛。(它会随着时间的推移而消退，但研究表明，某种程度的保护可以持续数月。)如果你曾经跟随一个朋友去健身房，和他们一起锻炼，你就会看到这种情况的发生。你可能会酸痛一个星期，但由于他们一直做同样的锻炼，他们几乎感觉不到什么。

这也解释了为什么在一个锻炼项目开始时，你会比刚开始几个星期时感到更加酸痛。有时人们会怀疑程序是否已经停止工作，但是 [即使你不感到疼痛](https://vitals.lifehacker.com/when-muscle-soreness-helps-muscle-growth-and-when-it-d-1720076931) 你仍然可以变得更强，反之亦然。

### 如何利用重复回合效应为你带来优势

如果你正在开始一项新的锻炼计划，或者在一段时间内第一次进行一种新的锻炼，不要在第一天全力以赴。中等强度的锻炼仍然会触发重复回合效应，允许你在第二天更加努力，一旦你激活了保护。

如果你已经在第一天做了一个杀手，现在你很痛苦，最好的办法就是按时回到健身房。如果你需要，给自己一个额外的休息日是很好的，但是 [请不要认为你必须等到疼痛消失](https://lifehacker.com/how-to-work-out-when-you-re-still-sore-from-the-last-wo-1790803145) 。努力克服它，疼痛会消退。

现在，如果你已经锻炼了一段时间，但是你仍然一直感到酸痛怎么办？很有可能，你没有一直做同样的动作。如果你每周都进行深蹲、仰卧起坐和硬拉锻炼，你就不会因为深蹲、仰卧起坐和硬拉锻炼而酸痛。但是，如果你非常喜欢多样化，以至于每天都在做不同的YouTube视频或不同的CrossFit课程，你就没有给你的身体一个机会来适应任何特定的运动。

(充足的睡眠和充足的饮食也有助于缓解疼痛，所以如果你感到疼痛，并且一直在节食，从来没有按时睡觉，那么也值得看看这些习惯。)

[就像健身中的很多问题](https://lifehacker.com/consistency-is-the-solution-to-most-of-your-fitness-pro-1845417358) 一样，解决酸痛的办法就是坚持。我知道在你第一次锻炼后的第二天躺在床上想着 *唉，我得回去了，这很让人气馁？但是如果你做了，事情会变得更好。我保证。*
