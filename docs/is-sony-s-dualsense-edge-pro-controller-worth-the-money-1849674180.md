# 索尼的 DualSense Edge Pro 手柄物有所值吗？(以及买什么代替)

> 原文：<https://lifehacker.com/is-sony-s-dualsense-edge-pro-controller-worth-the-money-1849674180>

PlayStation 5 的 DualSense 控制器是我最喜欢的索尼最新游戏机。该垫握起来很舒服，感觉制作精良，其自适应触发器和触觉隆隆声为您的游戏体验增加了触觉沉浸感。然而，索尼将在明年年初推出一个名为 [的新版本，即双重感知边缘](https://www.playstation.com/en-gb/accessories/dualsense-edge-wireless-controller/) ，该版本承诺通过更多按钮和精细定制来扩展原有控制器的功能。



DualSense Edge 是一款“专业”控制器，类似于微软的 Xbox Elite 游戏手柄。这些设备拥有面向铁杆玩家和专业玩家的高端功能。例如，DualSense Edge 包括许多增强功能，如:

*   完全可定制的按钮功能
*   能够在不同的按钮配置之间保存和热交换
*   控制器背面的两个可编程控制按钮
*   多个可更换的拇指棒模块和拇指棒帽
*   可调触发按钮长度、操纵杆灵敏度和隆隆声强度
*   充电电缆锁
*   与 PS5 的系统界面集成，可快速调整设置

DualSense Edge 将于 2023 年 1 月 26 日推出，售价 200 美元。是的，两百美元。

客观地说，你可以花 70 美元买一台标准的 DualSense。大多数 PS5 游戏的新价格在 60-70 美元之间。PS5 *本身*的纯数字版本为 400 美元，带光盘驱动器的型号为 500 美元。你可以购买三个 DualSense 控制器或三个新游戏，或者支付 PS5 的一半，几乎与一个 DualSense Edge 一样多。

## 200 美元的 DualSense Edge 值得吗？

简而言之，答案是否定的，但这并不意味着这对每个人来说都是不可能的。200 美元很贵，但与其他专业控制器相比并不奇怪。例如，微软的 Xbox Elite 2 控制器售价约为 155-180 美元，取决于型号和购买地点。

也有一些第三方制造商生产的兼容 PS5 的 pro pads，价格与 DualSense Edge 一样或更高，如 Hexgaming 的竞争对手 Prem ium 的价格为 270 美元 ，Scuf 的可定制 PS5 pro 控制器 [系列的价格在 200 美元至 220 美元之间。虽然有低于 200 美元的第三方选项，但许多都与 PS5 不兼容，或者它们的功能或构建质量不如你从更昂贵的品牌获得的东西令人印象深刻。](https://scufgaming.com/playstation5-scuf-reflex-pro)

与 DualSense Edge 相比，第三方控制器有自己独特的功能，例如更多额外的按钮或可更换的部件，但 DualSense Edge 与这些竞争对手相比的优势在于它无缝集成到 PS5 的界面中，并与其他索尼第一方外设兼容，如官方的 DualSense 充电底座。

尽管如此，尽管 DualSense Edge 可能是 PS5 上高端控制器的最佳选择，但我不建议你购买它，也不建议你购买任何其他 pro 控制器。

## 那么我应该买什么来代替呢？

如果你需要的只是一个新的本地合作游戏控制器或替换你已经有的 pad，只需购买 70 美元的 DualSense，而不是 200 美元的 DualSense Edge。

然而，正如我前面所说，专业控制器对于电子竞技专业人士和一小部分愿意在一个控制器上花那么多钱的铁杆玩家来说是很棒的。就我个人而言，如果我在市场上购买专业控制器，我可能会选择 DualSense Edge。也就是说，DualSense Edge 和其他类似的专业控制器对于绝大多数不会使用其任何特殊功能的玩家来说根本没有必要。

有些人可能觉得可定制控件的想法很吸引人，但是你不需要一个昂贵的控制器来改变你的控制器的按钮布局。显然，DualSense Edge 上的按钮重新映射更快更容易，但标准 DualSense 的按钮映射可以在 PS5 菜单中的**设置>辅助功能>控制器>自定义按钮分配下进行更改。**PS5 的设置中还提供了其他选项，如调节触觉反馈。如果你需要更多帮助，请查看我们关于在任何控制台 上重新映射控制器的深入指南。
