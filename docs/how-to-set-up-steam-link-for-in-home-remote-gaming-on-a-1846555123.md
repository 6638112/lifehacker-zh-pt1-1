# 如何为Mac上的家庭远程游戏设置Steam Link

> 原文:[https://life hacker . com/how-to-set-up-steam-link-for-in-home-remote-gaming-on-a-1846555123](https://lifehacker.com/how-to-set-up-steam-link-for-in-home-remote-gaming-on-a-1846555123)

Steam的家庭远程游戏应用程序 [Steam Link](https://store.steampowered.com/app/353380/Steam_Link/) 终于可以在Mac上使用，为游戏玩家提供了更多控制，让他们可以在哪里以及如何玩Steam游戏。 Steam Link可以让你将游戏从电脑传输到连接到家庭网络的另一台设备上，比如Android手机、iPhones、连接到电视的第二台PC，现在还有Mac或MacBook。

注意，这不是像Stadia一样的云流媒体服务——所以一款游戏的性能依赖于你的电脑硬件和你家庭互联网连接的强度。这也意味着Steam Link只玩你已经在Steam上拥有的游戏，以及与该应用程序兼容的。

大多数兼容Steam Link的游戏都需要一个控制器，尽管一些依赖键盘和鼠标的游戏也可以。然而，该应用程序清楚地标明了哪些游戏可以与Steam Link一起运行，因此在寻找游戏时没有任何猜测。

有了这些注意事项，下面是如何在Mac上设置Steam Link:

1.  从macOS应用商店 下载 [Steam链接。](https://apps.apple.com/us/app/steam-link/id1246969117)
2.  在你的电脑上打开Steam。
3.  确定您的PC和Mac已连接到同一个网络。
4.  打开Mac上的Steam链接，点击**“开始”**
5.  如果您还没有配对遥控器，请按照屏幕上的说明进行配对，或者如果您使用的是不同的输入设备，请单击**“跳过”**。
6.  Steam Link将在您的网络中搜索您的PC。设备名称出现后，从列表中单击它。
7.  主机屏幕上会出现一个PIN。将该数字键入Mac设备的对话框中，以配对设备。
8.  等待设备同步，然后Steam将测试您的连接。如果你的网络强度通过测试，点击**“开始播放”**在Mac上打开你PC的Steam库。
9.  从列表中选择一个游戏开始玩(游戏必须安装在您的主机上)。
