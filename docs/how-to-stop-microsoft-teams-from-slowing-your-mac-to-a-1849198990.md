# 如何阻止微软团队让你的 Mac 慢如蜗牛

> 原文：<https://lifehacker.com/how-to-stop-microsoft-teams-from-slowing-your-mac-to-a-1849198990>

微软团队是否让你的 Mac 慢如蜗牛？你并不孤单。这款应用如此臃肿，以至于即使是拥有 M1 苹果电脑的人也看到他们的机器变得几乎无法使用。退出团队可以解决这个问题，但是如果你需要在工作中使用这个应用，我们为你准备了几个变通办法。

目前，大多数对微软团队表现的抱怨都集中在它的 Mac 应用程序和上。然而，一些人报告了 [及其 Windows 应用](https://news.ycombinator.com/item?id=32087133)的问题，称这对电池寿命和性能产生了重大负面影响。一些人报告说，当他们试图在工作会议上取消静音来回答问题时，该应用程序没有响应。

使用英特尔和 M1 MAC 电脑的人不断抱怨。无论你的机器有多快，优化不好的应用程序都会让它变慢。以下是你可以做些什么来拯救你的 Mac 免受团队性能问题的困扰。

## 安装团队的 Microsoft Edge 应用程序

从团队中保存您的计算机的一种方法是将其安装为 Microsoft Edge 应用程序。这个版本的 Teams 拥有您需要的所有特性，并且没有相关的性能损失。在您的计算机上下载 Microsoft Edgeo nce 安装完毕，打开 [队伍](https://teams.live.com/) ，登录你的账号。C 点击窗口右上角的**三点**图标，导航至**应用>将本网站安装为应用**。点击弹出的蓝色**安装**按钮，选择**允许**。现在将在您的计算机上安装一个更好的 Teams 版本。

## 从 web 浏览器使用 Microsoft 团队

如果你不想将 Teams 作为微软 Edge 应用程序使用，你仍然可以从你选择的网络浏览器直接使用这项服务。只需进入团队网站并登录即可。微软有一份详细的 [兼容性文档](https://docs.microsoft.com/en-us/microsoftteams/limits-specifications-teams#browsers) ，让你知道你的浏览器支持哪些功能。Safari 和 Firefox 等浏览器错过了大多数团队的功能，而 Chrome 和 Edge 支持该应用程序的所有功能。
