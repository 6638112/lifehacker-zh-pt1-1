# 你现在应该在求职面试中问的一个问题是

尽管疫情经久不衰，各公司都在吸引员工回到办公室，即使随着新冠肺炎的 [病例在全国部分地区](https://covid.cdc.gov/covid-data-tracker/) 激增以及接种 [疫苗的努力在很大程度上陷入停滞](https://www.theguardian.com/world/2021/jul/26/us-vaccine-effort-nuance-covid-coronavirus) 。尽管如此，在一年多的远程工作后，回到办公室的整个转变对企业界的任何人来说都是一个大问题，充满了似乎缺乏足够答案的问题。

公司以不同的方式处理这个问题:一些公司提供混合设置，而另一些公司放弃办公室是必需品的想法，允许员工在他们喜欢的地方工作。因此，重要的是你要确切知道未来的雇主打算如何对待这种重返办公室的转变，主要是因为员工在这方面比以往任何时候都更有影响力。

## 求职面试中最重要的问题

这就是为什么，现在问未来雇主最重要的问题是:“你打算如何处理你重返办公室的事情？”这是一个至关重要的问题，它能让你对一份工作产生兴趣，也能让你失去兴趣。来自营销人员 [的一条有用的Twitter帖子艾米丽·科尔曼](https://twitter.com/editoremilye/status/1420467625587482624) 解释了如果公司的重返办公室政策与你的预期不一致，积极的求职面试会突然急转直下。

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1420467625587482624&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1420467625587482624&autosize=1) 

科尔曼说，意识到公司在疫情期间没有采取任何真正的预防措施，再加上无菌的办公环境，最终打击了她对这份工作的兴趣。此外，一家公司重返办公室的计划可能会在更广泛的层面上充分说明它如何重视员工。如果一个工作场所让每个人都遵循一个严格的命令，这不仅让你在工作安排上没有发言权，还让你面临潜在的健康危害，那么可能是时候去寻找其他机会了。

## 求职面试中要问的其他好的、当前的问题

在远程或混合工作环境中，询问公司如何创造其文化是很重要的。如果许多有机的、真实世界的互动不能发生——特别是如果人们被允许住在远离实体办公室的地方——那么问一问如何才能建立友情将是明智的。这是一个要问招聘人员或招聘经理的问题，以获得他们对这些努力是否有效的看法。

你也完全有权利询问一家公司是否已经为其返回的员工制定了疫苗要求。显然，如果有什么“不问，不说”的政策，这可能会损害办公室里每个人的健康。在这方面没有明确的路线，因为有些公司对疫苗接种有不同的政策 。确保你明白你在这个问题上的立场(我们建议你支持强制工人接种疫苗)。

鉴于大量来自工作场所之外的非常现代的问题正在渗入我们的工作生活，你考虑这些事情是很重要的；你会很高兴在下一次面试中提到这些问题。
