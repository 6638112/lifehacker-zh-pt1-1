# 你不必受冬天眼睛的折磨

> 原文：<https://lifehacker.com/you-dont-have-to-suffer-from-winter-eyes-1848520098>

除了我们的 [皮肤](https://lifehacker.com/how-to-deal-with-dry-skin-in-the-winter-1848362017) 和我们的 [头发](https://lifehacker.com/everything-you-shouldn-t-do-to-your-hair-during-the-win-1848405329) 之外，冬天也会对我们的眼睛造成伤害——原因也是一样的。外面冷的时候，我们想在温暖干燥的室内呆更长时间。不幸的是，这些 [温暖干燥的条件](https://www.uabmedicine.org/-/cold-weather-woes-and-dry-eyes) 有使我们的眼睛感到干燥和烦躁的趋势。



但是，如果我们走在户外，外面有风和/或很冷，我们的眼睛会开始流泪——但不知何故会保持干燥。真的，不太理想。这里有一些关于冬季干眼症的知识，包括要寻找的症状，以及如何治疗。

## 什么是冬季干眼症？

“干眼症”不仅描述了你正在经历的事情，它也是这种状况的名称。一年中的任何时候都有可能出现干眼症，因为这是眼泪产量或质量低的结果，但根据阿拉巴马大学伯明翰分校眼科教授 Marissa K. Locy 博士的说法，冬天会变得更糟。

“平均而言，随着天气变冷，冬天的湿度会下降，” [Locy 解释说](https://www.uabmedicine.org/-/cold-weather-woes-and-dry-eyes) 。此外，大多数人打开家中或办公室的暖气来抵御寒冷。因此，你最终得到的是更低的外部湿度，甚至更低的内部湿度，从而形成温暖、干燥的条件，水分可以比正常情况下更快地从眼睛中蒸发。”

## 冬季干眼症有哪些症状？

冬季干眼症的症状往往出现在双眼，戴隐形眼镜的人情况更糟。 [根据](https://www.mayoclinic.org/diseases-conditions/dry-eyes/symptoms-causes/syc-20371863) 梅奥诊所的说法，这些症状可以包括:

*   眼睛有刺痛、灼烧、粗糙或发痒的感觉
*   水汪汪的眼睛
*   眼睛发红
*   眼睛内或周围粘稠的粘液
*   光敏度
*   视力模糊或眼睛疲劳

## 如何治疗冬季干眼症

坏消息是没有治愈干眼症的方法。但好消息是有不同的方法来控制这种情况，包括:

*   在你家使用 [加湿器](https://www.mayoclinic.org/diseases-conditions/dry-eyes/symptoms-causes/syc-20371863)
*   待 [水合](https://www.uabmedicine.org/-/cold-weather-woes-and-dry-eyes)
*   [保护眼睛](https://www.uabmedicine.org/-/cold-weather-woes-and-dry-eyes) 当你在户外时(免受风、冷空气、阳光等)
*   根据需要使用 [人工泪液](https://www.mayoclinic.org/diseases-conditions/dry-eyes/symptoms-causes/syc-20371863) 或其他非处方眼药水

但是，如果你已经尝试了以上所有方法，并且在很长一段时间内持续出现干眼症， [是时候去看医生了](https://www.mayoclinic.org/diseases-conditions/dry-eyes/symptoms-causes/syc-20371863) 。他们可以评估情况，或者推荐其他治疗方案，或者推荐你去找专家。
