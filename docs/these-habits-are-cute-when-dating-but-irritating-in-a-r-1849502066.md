# 据 Lifehacker 的读者称，这些习惯在约会时很可爱，但在恋爱关系中却很烦人

> 原文：<https://lifehacker.com/these-habits-are-cute-when-dating-but-irritating-in-a-r-1849502066>

上周我让你分享一下 你的玫瑰色眼镜最后一次失去玫瑰色的时候。因为随着约会的早期阶段变成一段严肃关系的严酷现实，某些行为会从讨人喜欢变得令人生厌。无害的例子包括持续的嗡嗡声，临床水平的打鼾，或者总是连续打喷嚏一百万次。更严重的冒犯包括像 T4 这样的人，他们总是不停地开玩笑，即使是在严肃的场合。



以下是 Lifehacker 读者报道的最大怪癖，你在第一次坠入爱河时觉得很可爱，但在长期关系中慢慢变成了紧张的来源。

## “取笑”并不总是取笑

很多人通过“笑话”来掩饰被动攻击。读者[**bass beast**](https://kinja.com/therealbassbeast)解释道:“起初这似乎是一种有趣、戏谑的调侃。但过了一段时间，你开始怀疑动机，这就变得很伤人。署名，一个幸福地和这个人结婚，但首先需要许多重要谈话的人。”作为一个经常开一些面无表情、可能伤人的玩笑的人，非常清楚地表明你不是想伤害别人是非常重要的。

另一位评论者[**ThundercatsRidesAgain**](https://kinja.com/thundercatsridesagain)分享了类似的经历，但有一个不同的结局:“我从来不确定他是不是在开玩笑，那太累了。当我以为他是认真的时候，他却在开我的玩笑，感觉他这样做是为了让我失去平衡。我不喜欢这种动态的发展方向，所以在他拒绝接受这一点的几次谈话后，我摆脱了他。”要成为一个交易破坏者，不一定要全力以赴" [否定](https://lifehacker.com/all-the-ways-youre-being-rude-on-dating-apps-without-re-1848188540) "。有时候你们的幽默感完全不匹配，和一个总是让你成为笑柄的人呆在一起没有意义。

## 当“好玩”变成“鲁莽”

当你第一次开始和一个新的人约会时，会很激动人心。也许他们鼓励你从自己的壳里走出来，或者他们通过总是“有趣的人”来逃避现实生活然而，开始时的乐趣可能会变成一个问题。 [**Antifaz**](https://kinja.com/antifaz) 分享道“我在一次聚会上遇到的一个大学时代的旧女友起初看起来自由、棒、狂野、有趣！后来，酒精中毒和大麻成瘾变成了一个小问题……”因此，尽管这种人一开始可能会令人兴奋，但在真实的关系中，这并不总是可持续的或健康的。

## 太放松了

一位读者评论说，“开始的时候，你可以自己制定时间表，选择你想去的地方和想做的事情，这很方便，但在一起一段时间后，当他们无法决定时，这就变得非常令人沮丧。”当你第一次开始和另一个人进行多次约会时，无忧无虑的性格是很有吸引力的。当你和另一个人需要一起做决定时，很明显，当一个人更“冷静”而不是更“乐于助人”

## 如果有什么事情困扰着你，那就说出来

不时对你的伴侣发火是完全正常的。然而，许多读者回答了“你在约会时发现了什么可爱但在一段关系中令人讨厌的事情？”像“我的妻子”这样直率的回答(提示波拉特印象。)如果你发现自己经常对你的另一半和他们做的每一件小事感到恼火，你可能需要缩小视野，找出真正困扰你的是什么。

幸运的是，上面强调的反应是高度可解决的行为。反思一下你现在能发现自己正在做的烦人的事情，或者至少在为时已晚之前，你可以和你的伴侣进行一次富有成效的谈话。
