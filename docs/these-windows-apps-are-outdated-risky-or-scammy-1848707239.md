# 这些 Windows 应用程序已经过时，存在风险，或者是骗人的

> 原文：<https://lifehacker.com/these-windows-apps-are-outdated-risky-or-scammy-1848707239>

随着时间的推移，每台电脑都会产生一些垃圾:你多年前安装但忘记了的小应用，或者当你想释放一些空间时下载的清理工具，但仍在后台运行。另外， [Windows 自带膨胀软件](https://lifehacker.com/you-can-get-rid-of-windows-bloatware-with-one-command-1848707156) 。无论你的 Windows 电脑有多旧(或多新)，你都应该偶尔检查一下你已经安装的应用程序，并决定哪些应用程序可以不用。



这里有一些你应该 尽快删除的 Windows 应用程序——要么因为它们已经过时，它们构成安全风险，要么因为它们只是骗局。

## 清洁器和其他清洁工具

Windows 10 和 11 在自我保护方面做得很好。再也不需要清理 app 了——尤其是 CCleaner。CCleaner 曾经是一款可靠的清理窗户的 app。但在 2017 年，它被 Avast 收购，开始增加不必要的功能和数据- 监控工具。虽然现在的应用程序在这些方面做得更好了，但现在没有必要使用像 CCleaner 这样的清洁工具。

## 冗余技术，如 Flash、Silverlight、Java

Adobe Flash 不复存在。感谢上帝。但它仍然可能是你电脑上的。对于微软 Silverlight、Java 和 Shockwave 等老框架和技术来说，情况可能也是如此。进入你的应用程序列表，如果你在列表中看到任何一个应用程序，就立刻杀死它。

## 像 Lightshot 这样的免费截图工具

Lightshot 是一款免费的截图上传工具。它很有用，但是有安全风险。我们建议您不要使用*任何*第三方截图工具对个人数据进行截图——特别是像 Lightshot 这样连接到云存储的东西，在那里您无法控制数据。

使用 Window 的 Snipping 工具，或者 ShareX，我们为 [推荐的 Windows](https://lifehacker.com/this-is-the-best-screenshot-tool-for-windows-1848690325) 的最佳截图工具(然后只在本地存储截图)。

## 科尔塔纳

Cortana 是微软从未成功的项目之一。在 Windows 10 中就已经隐藏了，在 Windows 11 中也没有了。尽管如此，Cortana 仍然潜伏在后台，占用空间和资源。使用 PowerShell 中的[windows 10 解封器脚本，只需点击即可卸载 Cortana。](https://github.com/Sycnex/Windows10Debloater)

## Bonjour、QuickTime、iTunes 等苹果应用程序

如果你在 2014 年安装了一次 iTunes，你的电脑上仍然安装了至少五个不同的苹果应用程序。在现代，真的不需要 iTunes，如果你确实想备份或同步你的 iPhone 和 PC，你可以使用第三方工具(或只使用 iCloud)。

前往您的应用程序列表并移除与 iTunes 相关的应用程序，如 Bonjour、Apple Software Update、Apple Mobile Devie Support、QuickTime 和 iTunes。

## 制造商膨胀器

如果你从微软之外的任何一家公司购买了一台 Windows PC，很可能它已经预装了你不需要的免费应用程序。转到您的应用程序列表并找到此类应用程序。使用 [应该把它去掉吗？](https://www.shouldiremoveit.com/) 网站，找出哪些应用是膨胀软件，并立即卸载它们。

## 迈克菲膨胀软件

McAfee 是一款本身并不安全的杀毒软件。而且事实证明 [出了名的狡猾，竟然完全将 卸载](https://lifehacker.com/those-apps-you-deleted-on-your-mac-might-still-be-han-1848353773) 在 Mac 上。它大多是预装在电脑上的，并不是一个容易使用的应用程序。不过，在 Windows 上很容易卸载它——你可以从“设置”的应用列表中卸载它。

## 像 uTorrent 和 BitTorrent 这样的 Torrent 应用

uTorrent 曾经是 Windows 最好的 torrent appBitTorrent 紧随其后。但是之后，uTorrent 开始收到广告，T2 和膨胀，BitTorrent 也是如此。 uTorrent 甚至在没有告知用户的情况下，增加了一个密码挖掘软件。哎哟。

卸载这些 app，改用 [qBittorrent](https://www.qbittorrent.org/) ，安全、免费、开源。

## 我赢了

每个人都需要一个解压文件的应用——但不应该是 WinRar。这个应用已经 20 年没有真正的更新了，并且永远提供 40 天的试用。如果你不付钱，它不会把你锁在外面，但是它*会*一直给你显示一个烦人的弹出窗口。切换到免费好用的 [7-Zip](https://www.7-zip.org/) app，。
