# 新的自杀热线988有什么问题？

> 原文:[https://life hacker . com/what-s-the-problem-with-988-the-new-自杀热线-1849196885](https://lifehacker.com/what-s-the-problem-with-988-the-new-suicide-hotline-1849196885)

到现在为止，你可能已经听说过，国家预防自杀生命线[最近有了一个新的三位数电话号码:988](https://lifehacker.com/the-suicide-prevention-lifeline-is-getting-a-new-number-1849154009) 。出于可以理解的原因，这种更容易记住的代码得到了媒体的大力推动，因为倡导者们试图确保任何需要救生索的人都像911一样，将这三个数字放在脑海中。(之前的号码是1-800-273-8255，那个也还能用。)

然而，你可能也注意到了，在媒体的推动下，有一个反运动。许多倡导者大声疾呼——经常使用类似于在社会动荡时期Instagram 上激增的 [信息图表——并鼓励人们*而不是*拨打新号码。但是为什么呢？](https://lifehacker.com/once-you-re-done-reposting-infographics-about-injustice-1848875824)

## 反对988的理由是什么？

一个广为流传的信息图 [这样说这个](https://www.instagram.com/p/CgFZqnRujWj/?igshid=YmMyMTA2M2Y=):“988是*不*友好。在不知道风险的情况下，不要叫，不要贴，不要分享。风险包括警察介入、在急诊室和精神病院进行羞辱性的非自愿治疗、使用医疗暴力来惩罚‘不合作’或痛苦的患者、强迫用药、沉重的医疗债务以及改变生活的创伤。”另一个人说这条线“不是我们需要的热线”，并说它后面的人“不要谈论…你打电话后会发生什么”，尽管它确实说它是“如果你有立即的危险并需要被带到医院以确保安全，这是一个很好的电话号码。”这幅图把美国精神保健系统称为“狗屎秀”，除非危险迫在眉睫，否则应该避免。

看到精神健康倡导者反对将处于危险中的个人与资源联系起来的生命线可能会令人困惑，但他们的批评也反映了最初的生命线所面临的问题。主要问题归结为一种担忧，即执法人员会对心理健康电话做出反应，导致那些寻求热线呼叫中心训练有素的咨询师帮助的人出现问题甚至危险的情况。还有人担心，一个人可能会非自愿住院，甚至药物治疗。最后，人们担心电话可能被追踪，而且尽管这条线路是保密的，但运营这条线路的人确实会利用来电者的信息来找到当地的资源。这些问题中的每一个本身都可能是有害的，但也增加了一个人因拨打该热线而面临社会或职业后果的可能性。

## **生命线对此怎么说？**

当被要求对倡导者提出的问题做出回应时，生命线组织的一名代表指导Lifehacker进入一个常见问题页面 ，该页面处理电话追踪和警方介入的问题。该页内容如下:

> Lifeline目前不具备像911提供商那样直接“追踪”呼叫者、聊天或短信用户的能力。…在非典型情况下，必须联系紧急服务以防止人们严重或致命地伤害自己，并且人们不愿意或不能分享他们的位置信息，生命线顾问必须向911接线员提供他们所拥有的信息——呼叫者/短信用户的电话号码或聊天用户的IP地址——以使他们能够尽一切努力找到该人。

常见问题还说，“生命线建议危机辅导员只有在对自己或他人的伤害风险迫在眉睫或正在发生的情况下，以及在无法与个人合作制定针对呼叫者/发短信者安全的侵入性较小的计划时，才联系紧急服务(911、警察、治安官)寻求帮助。不到2%的生命线电话涉及紧急服务。当涉及紧急服务时，超过一半的紧急调度是在呼叫者同意的情况下发生的。”

背后的组织Vibrant也承认，与紧急服务机构的联系可能是“创伤和危险的”，所以他们尽量避免让这种情况发生。然而，研究人员和倡导者仍然有他们的担忧。正如一位自杀学家 [在推特](https://twitter.com/saltysicky/status/1548782365941477376?s=21&t=KnZTo9vyarA42i1cSlfXIQ) 上写道，尽管988中训练有素的危机小组的人数减少和数量增加是好事，“但这仍然与非自愿的积极救援有关，这意味着他们可以[并将]追踪你的电话，如果他们认为有必要，还会派警察来。”

## 这对你来说意味着什么？

上面提到的自杀学家指出，这里的一个大问题是，在危机时刻需要帮助的人可能不会拨打热线，因为害怕警察联系、非自愿住院或他们的呼叫带来的后果，无论这些后果是以职业、社会或居住方式表现的。

如果你有任何形式的心理健康危机，这条热线会接听你的电话，尽管正如Lifehacker的高级健康编辑贝丝·斯克沃茨基在她的 [新线功能概述](https://lifehacker.com/the-suicide-prevention-lifeline-is-getting-a-new-number-1849154009) 中指出的那样，如果你没有危险，只是想找人聊聊，你可以考虑使用热线。 [在这里找到热线号码列表](https://screening.mhanational.org/content/need-talk-someone-warmlines/) 。

然而，如果你面临迫在眉睫的危险，并决定拨打988，并不保证警察会出动或你会被送往医院。另一端受过训练的危机辅导员没有脚本可循；他们可能会帮助你制定安全计划，鼓励你联系朋友或家人，或者帮助你找到其他资源。他们也可能会安排与您的跟进电话，或甚至派遣一名顾问到您家，如果您同意和选择是可行的。

虽然对警方接触和非自愿干预的担忧是绝对正确的，但你有许多选择，无论你是处于即时危险之中，还是仅仅需要找人倾诉。如果你最终打电话给988，不要害怕提及你担心紧急服务互动，特别是如果这将升级你的情况。危机顾问接受过开放式谈话的培训，他们的最终目标是你的安全，所以要清楚地表明，有些结果——比如警察派遣或任何可能影响你工作的事情——会让你感到不太安全。
