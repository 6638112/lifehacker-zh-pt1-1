# 今天下午最后一次看地球的“迷你月亮”

> 原文:[https://life hacker . com/see-earths-mini-moon-the last-time-this-午-1846171906](https://lifehacker.com/see-earths-mini-moon-for-the-last-time-this-afternoon-1846171906)

随着2020年其他事情的发生，你可能会错过关于2020年9月[坠入地球轨道的神秘飞行物的故事。不确定它到底是什么，它被称为“SO 2020”，并被认为是“迷你月亮”——一个用于描述临时卫星的术语。](https://www.livescience.com/minimoon-human-made.html)

到12月，美国宇航局的研究人员确定它实际上是一块人造太空碎片:[20世纪60年代美国探测器月球任务中使用的火箭助推器](https://lifehacker.com/how-to-see-the-1960s-rocket-booster-that-will-fly-by-ea-1845782045) 的残骸。尽管这是自9月份发现迷你卫星以来对其可能的解释之一，但直到12月1日它最接近地球，天文学家才得以确认其身份。

但是，如果你错过了火箭助推器在12月的亮相，你很幸运:它将在今天下午回来进行最后的告别胜利圈。下面是怎么看的。

### 如何观看迷你月球/火箭助推器的最终轨道

今天，迷你月球/20世纪60年代的空间碎片将距离地球14万英里，或地球和月球之间距离的58%，根据EarthSky 的 [。虽然我们离得还不够近，无法用肉眼看到，但多亏了罗马的](https://earthsky.org/space/2020-so-mini-moon-asteroid-or-space-junk) [虚拟望远镜项目](https://www.virtualtelescope.eu/2021/01/21/farewell-mini-moon-2020-so-very-close-approach-online-observation-01-feb-2021) ，我们还是有机会看到它的最后一次造访。

虚拟望远镜项目的创始人，意大利天体物理学家和天文学家吉安卢卡·马西是这样描述今天的事件的:

> 2020年12月，经过它的近距离飞越后，它再次安全地靠近，这次是告别。我们知道，它是勘测者2号太空任务的助推器，被我们的星球暂时捕获。不久，这个人造迷你月亮将离开我们的邻居，逃到一个新的围绕太阳的轨道上。我们将说再见，现场直播:在您舒适的家中加入我们！

为了观看迷你月亮的最后一次通过， [在这里](https://www.virtualtelescope.eu/webtv/) 调到虚拟望远镜项目的现场直播。根据马西的说法，从T4时间下午5点开始，在意大利中部该组织的机器人望远镜上可以看到它

### 在这之后迷你月球会发生什么？

虽然火箭助推器在今天之后将不再可见，但它将慢慢漂移，最终在2021年3月离开地球轨道。之后，它将开始绕太阳运行。我们祝愿它在未来的努力中一切顺利。
