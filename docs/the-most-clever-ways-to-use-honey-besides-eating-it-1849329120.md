# 最聪明的使用蜂蜜的方法(除了吃蜂蜜)

> 原文：<https://lifehacker.com/the-most-clever-ways-to-use-honey-besides-eating-it-1849329120>

从简单到往茶里加一团糖， [到在你的空气油炸锅里给土豆浇糖浆](https://lifehacker.com/you-should-definitely-make-sticky-honey-tots-in-your-ai-1848573234) ，不缺可以用少量这种天然花蜜装饰的菜肴。但是蜂蜜的用处扩展到了厨房之外:一些研究发现，蜂蜜在[食物保存](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC3758027/) ，甚至 [伤口包扎](https://pubmed.ncbi.nlm.nih.gov/26061489/) 中是一种有用的工具。所以，在你考虑扔掉一个没用过的罐子之前，先看看除了做一些“杀手抱子甘蓝” 之外，你是否还能把它用在别的地方。



**护发。**当涂在头发和头皮上时，蜂蜜能够为头皮皮肤提供皮肤再生功能。 [据《美容皮肤病学杂志》](https://pubmed.ncbi.nlm.nih.gov/24305429/) 报道，“在化妆品配方中，【蜂蜜】发挥润肤、保湿、舒缓和护发作用，保持皮肤年轻，延缓皱纹形成，调节 pH 值，防止病原体感染。”美容作家凯特琳·达根(Caitlin Duggan)建议每周使用这种疗法不超过一次，并与椰子或其他载体油搭配使用。

**制作润唇膏。**一点点蜂蜜会让你的嘴唇受益匪浅。 [据护肤专业人士伊尔迪·佩卡尔介绍，](https://www.womenshealthmag.com/beauty/a19946109/beauty-uses-for-honey/) 蜂蜜具有保湿、舒缓的功效，能给皮肤补水，使其柔嫩有光泽。蜂蜜润唇膏的 DIY [配方是不缺的，如果你发现自己有兴趣尝试一下](https://siouxhoney.com/11-diy-lip-balms-made-with-honey/) 。

**愈合伤口。不要低估你的餐具室甜味剂的威力。 [根据伤口护理学习网](https://www.hmpgloballearningnetwork.com/site/wounds/article/honey-biologic-wound-dressing)2015 年的一项研究，蜂蜜是一种有效的伤口愈合剂。它写道:“蜂蜜的物理特性加快了愈合过程:它的酸性增加了血红蛋白中氧气的释放，从而使伤口环境不太适合破坏性蛋白酶的活动，蜂蜜的高渗透压将液体从伤口床中吸出，产生了淋巴外流，这与负压伤口治疗的情况相同。”** 

在洗澡时使用。把泻盐放到一边，去厨房。 [专业美学家 Jordana Miattoli 告诉 Bustle，蜂蜜由于含有氨基酸和抗氧化剂而具有光滑皮肤的功效。将一杯蜂蜜溶解在热水中，然后加入到浴缸中，你将能够在放松活动后拥有漂亮光滑的皮肤。](https://www.bustle.com/p/6-ingredients-to-add-to-your-bath-to-benefit-your-skin-body-38365) 

**制作蜂蜜酒。**如果你正在寻找一种快感，并且不介意等待，可以考虑酿造蜂蜜酒。 [蜂蜜、水和酵母是蜂蜜酒](https://users.soe.ucsc.edu/~karplus/mead-recipe.html) 的基本成分，这是一种甜酒，发酵时间从三周到六个月不等，取决于你希望它有多烈。如果你对酿造蜂蜜酒感兴趣，但不知道从哪里开始，我们有 [一个指导让你开始](https://lifehacker.com/learn-how-to-brew-your-own-mead-at-home-1789580060)

**末日准备。**所以从技术上来说，这一次确实涉及到吃蜂蜜，但(理想情况下)暂时不会。蜂蜜是任何末日预言者的理想食物，因为它不会过期。 [据科学聚焦](https://www.sciencefocus.com/science/why-doesnt-honey-go-off/) 报道，蜂蜜的低含水量和高酸度是它不变质的两个主要原因。这是因为导致食物变质的细菌在这种条件下无法生长。正因为如此，蜂蜜是你藏在天启地堡里的绝佳选择。如果一枚核导弹击中了你的城市，你要确切地知道你应该做什么，以确保这一切不是徒劳的。
