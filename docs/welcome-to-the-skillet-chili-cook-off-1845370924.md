# 欢迎来到煎锅辣椒烹饪大赛

> 原文：<https://lifehacker.com/welcome-to-the-skillet-chili-cook-off-1845370924>

像比萨饼和烧烤一样，辣椒是具有强烈地区忠诚度的食物之一。我出生在密西西比州，在加利福尼亚州长大，这两个州没有强烈的辣椒特征。从某种程度上来说，这很好——我可以享受所有的辣椒、豆子或没有豆子——但这也意味着我没有最喜欢的食谱，甚至没有最喜欢的风格。

Watch

这就是你进来的地方。从今天开始，我们将发起一场煎锅辣椒烹饪大赛，这是一种竞赛，你可以(在评论中)向我推荐你最喜欢的辣椒食谱，让我来制作、食用和赞美。大概是这样的:

1.  你可以在下面留下你的辣椒食谱的评论。越详细越好，优先给予那些有最清楚指示的人。原创食谱是首选，但经典食谱的修改或调整也是可以接受的。(简单地写“德克萨斯风格”，对我来说是不够的。)
2.  我将挑选八个对我来说听起来最好的辣椒食谱，尽管我会受到公众舆论的影响，所以对任何你认为可能是强有力的竞争者进行评论(除非你提交你自己的)。
3.  在接下来的八个星期里，我会做并尝试每一种辣椒，并记录我对每一种辣椒的体验。
4.  一旦我尝试了所有的辣椒食谱，我会选出赢家，并为你唱赞歌。也许我会为你和你的辣椒献上一杯 [三种成分的欢乐时光](https://lifehacker.com/c/3-ingredient-happy-hour) 饮料。

差不多就是这样！在下面留下你的辣椒评论，支持你的最爱，请不要胡说八道。我知道你们都有强烈的观点，但这里欢迎所有的辣椒风格——甚至是土耳其辣椒——所以让我们保持尊重。毕竟，辣椒是用来分享的。
