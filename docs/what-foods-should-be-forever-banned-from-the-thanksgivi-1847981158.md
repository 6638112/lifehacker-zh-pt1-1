# 哪些食物应该永远禁止出现在感恩节餐桌上？

> 原文:[https://life hacker . com/what-foods-should-be-forever-banted from the-thanksgivi-1847981158](https://lifehacker.com/what-foods-should-be-forever-banned-from-the-thanksgivi-1847981158)

现在十月已经过去了，是时候把注意力转向下一个我们可以尽情享受的节日了:感恩节。虽然万圣节有服装和糖果，圣诞节和光明节有礼物，复活节有一只虚构的小兔子，但感恩节没有特别的角色或惊喜——这里所有的都是食物。(如果你对足球感兴趣的话，我想还有足球。)

如此专注于让远道而来的家人聚在一起吃一顿饭，这顿饭需要非常壮观，没有人真正喜欢的老掉牙的经典，或者可以让整个氛围变得沉闷的实验菜。(我们都经历过；有一年，我哥哥想尝尝火鸭，它不仅花费了*很长时间*，而且，用我姐夫谦卑的评价来说，“尝起来就像是从火鸡屁股里拉出来的鸭子。”但是我跑题了。)

有这么多的菜需要同时上桌，所以很正常——甚至是意料之中——有一些小的火鸡日失败，表现为干燥的填料或松软的土豆泥。我们不是在这里谈论这些。我们正在谈论的是那些在餐桌上根本没有位置的菜肴。那些应该被完全剔除的菜肴——事实上，那些菜肴是对各地味蕾的一种侮辱。像 [果冻酸果蔓酱](https://www.target.com/p/ocean-spray-jellied-cranberry-sauce-14oz/-/A-12936143?ref=tgt_adv_XS000000&AFID=google_pla_df&fndsrc=tgtao&DFA=71700000012510691&CPNG=PLA_Grocery%2BShopping%7CGrocery_Ecomm_Food_Bev&adgroup=SC_Grocery&LID=700000001170770pgs&LNM=PRODUCT_GROUP&network=g&device=c&location=9003944&targetid=pla-277877328643&ds_rl=1246978&ds_rl=1248099&gclid=Cj0KCQjww4OMBhCUARIsAILndv4tKkXD3tp4k94SfVyZ9eWUUKED4plb4JAz8TolkNzy9ZKoAwR-wNgaAmTKEALw_wcB&gclsrc=aw.ds) 这样的东西，其可见的罐标如此巧妙地将它们的成分结合在一起:酸果蔓、高果糖玉米糖浆、水和...普通玉米糖浆。

你认为哪些食物应该被禁止出现在感恩节餐桌上？在一个简短的网上搜索中，我们发现其他人在鼓吹经典的(但潮湿的)绿豆砂锅、美味沙拉、素餐面包卷和汤的消亡？不，汤是美味的。奶油烤胡桃南瓜配烤迷迭香？厨师之吻。

在评论中分享你的想法，我们将在未来的帖子中总结它们，这样所有想成为感恩节晚餐主人的人都不会因为提供自己值得呻吟的菜肴而尴尬。
