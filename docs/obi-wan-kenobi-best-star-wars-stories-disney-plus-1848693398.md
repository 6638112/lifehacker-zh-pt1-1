# 必不可少的欧比万·克诺比《星球大战》故事，在他的展览开始前先看看

> 原文：<https://gizmodo.com/obi-wan-kenobi-best-star-wars-stories-disney-plus-1848693398>

[欧比万·克诺比](https://gizmodo.com/obi-wan-disney-plus-trailer-breakdown-jedi-inquisitors-1848629575) 是*星球大战*中最受喜爱的角色之一，甚至在我们五月追上 伊万·麦格雷戈流亡的绝地大师*之前*的事件之前。如果你想沉浸在重温《西斯的复仇》的氛围中，这里有一些必读(也是必看)的故事，它们将成为欧比万最权威的作品。 

克劳迪娅·格雷的佳能小说深入探究了奎-冈和欧比-万的早期伙伴关系，并提供了一个有趣的故事，讲述了在一个大动荡时期开始这段关系的过程，当时欧比-万正在认真地重新考虑自己是否具备成为一名绝地和像奎-冈·金这样非正统的人的徒弟的条件。在当前的*星球大战*佳能在 T5 前*幽灵的威胁*时代 中难得一见，这是对这两位英雄的精湛(抱歉)检验。

在《克隆人战争》中，欧比-万常常是阿纳金和阿索卡滑稽动作的讽刺陪衬，他是绝地武士团的父亲，当他的前徒弟和新幼徒无视他的时候，他总是推行教义和反对。但这三集——也因对曼达洛文化 的彻底改造而闻名，最初颇有争议——通过介绍曼达洛的和平主义领袖莎廷·克里兹公爵夫人，将欧比-万的个人生活置于引人入胜的重点...也是欧比旺年轻时的前情人。

这部 [五- 发行的迷你剧](https://gizmodo.com/all-of-marvels-star-wars-comics-ranked-1793171041) 背景设定在*《幽灵的威胁》*和*《克隆人的进攻》*之间的十年间，作家 Charles Soule 和艺术家 Marco Chechetto 和 Andres Mossa 将这对绝地二人组派往一个遥远的冰雪世界，调解两个交战派系之间的争端。最终，它成为欧比-万和阿纳金之间不稳定关系的调解，命运和魁刚的意志将他们推入彼此的生活，为他们之间不断发展的关系打开了一扇窗。

《克隆人战争》*的另一个罕见的以欧比万为中心的剧情 [，这实际上是从欧比万似乎被杀死开始的。当然，他还活着——这次暗杀行动成为绝地秘密行动的一部分，目的是让伪装成赏金猎人拉科·哈丁的欧比-万发现一次针对议长的暗杀行动。一个看欧比-万在没有绝地支持或他的原力力量的情况下完成危险任务的机会！](https://gizmodo.com/the-essential-clone-wars-stories-every-star-wars-fan-sh-1842597580)* 

 *在 3DCG 动画系列出现之前，现在不规范的黑马*共和国*系列给了我们 [一个关于克隆人战争的权威会计](https://gizmodo.com/a-guide-to-star-wars-comics-that-can-help-fill-the-clon-1843424882) ，这三个故事——所有这些都可以在该系列的漫威漫画集中找到，*星球大战传奇史诗集:克隆人战争第二卷*——是一个非常好的系列中的一些亮点。

“贾比姆之战”实际上是一个多议题的弧线，奥比万并没有在中出现太多...因为在这场有名无实的可怕战斗中，他很早就失踪了,留下阿纳金加入了一群同样成为孤儿的学徒，他们不得不抵抗强大的敌军。另外两个故事继续透露，当然，欧比万实际上被俘虏了，他必须和克隆人 ARC 突击队阿尔法一起计划逃跑——并与阿萨吉·文崔斯不期而遇。

当归来的达斯·摩尔( [说来话长](https://gizmodo.com/the-star-wars-canon-behind-solos-biggest-surprise-1826387914) )在曼达洛发动政变时，欧比万努力阻止世界陷入混乱...却眼睁睁地看着自己悲惨地败在他最持久的敌人手中。如果你喜欢一个感情受折磨的欧比万·克诺比，那么这是一个很好的 20 分钟左右。

现在，另一个来自旧膨胀宇宙的非经典条目，约翰·杰克逊·米勒对欧比-万在塔图因流放的最初几个月的叙述，肯定会与即将到来的展览形成引人入胜的对比。在这部小说中，欧比-万并不是主角，而是更多地讲述了他的出现如何永远改变了塔图因绿洲商店店主安妮琳·卡尔威尔的生活，以及欧比-万想在世界上做好事的内在愿望与隐藏自己不被人发现之间的矛盾。一个迷人的小赌注和亲密的故事，即将到来的系列将从 中汲取 [一些想法。](https://gizmodo.com/obi-wan-kenobi-disney-plus-eopie-john-jackson-miller-1848638199) 

这部标志性的*反叛者*剧集 [最近迎来了五岁生日](https://gizmodo.com/star-wars-rebels-obi-wan-vs-maul-5-years-later-twin-sun-1848662693) ，它仍然像上映时一样令人震惊和辛酸。欧比-万自己可能并不在其中，但当达斯·摩尔操纵年轻的绝地学徒埃兹拉·布里杰进入一个陷阱，把欧比-万从流放中解救出来时，他的存在被感觉到了...并寻求一劳永逸地结束他对肢解他的绝地的复仇之旅。

这个非常方便的漫威收藏集收集了漫威于 2015 年由杰森·亚伦创作的复兴经典*星球大战*漫画第一卷的多个独立版本。围绕卢克在*《新希望》*事件后不久发现欧比-万的一本日记，这些问题闪回到欧比-万被流放到塔图因的整个时期，让我们一瞥他试图隐藏起来，同时仍然在世界上做好事，从 [保护年轻的卢克](https://gizmodo.com/this-weeks-star-wars-comic-gives-us-a-badass-old-ben-ke-1721183334) 到拥有一个...与小男孩的新监护人欧文·拉尔斯关系不太友好。期待这部剧也能触及这些想法！

*不要和这个列表中的第一个条目混淆，尽管也是克劳迪娅·格雷写的！这个小故事来自*一个新的希望*T5】四十周年纪念文集 *从某个角度来看*，并赶上了欧比-万与魁刚·金的原力幽灵交流的时刻。它实际上写于《师父》*【徒弟】*的几年前，不仅甜蜜地展现了欧比万和魁刚在后者死后的关系演变，而且对我们所知的关于原力幽灵 的 [进行了深刻而迷人的发展。](https://gizmodo.com/so-what-can-a-force-ghost-do-1838080624)*
