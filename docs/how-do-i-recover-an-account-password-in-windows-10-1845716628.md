# 如何在 Windows 10 中恢复帐户密码？

> 原文：<https://lifehacker.com/how-do-i-recover-an-account-password-in-windows-10-1845716628>

我有点健忘，这就是为什么我总是使用密码管理器来记住我所有古怪复杂的密码。即使这样，我偶尔也会输入错误的密码来访问它，尽管我已经使用它很多年了。



然而，我最终得到了正确的密码。当你被提示输入一个你甚至不知道的账户的密码时，情况就不一样了——更糟糕的是，当这个账户是你如何使用操作系统的一个关键部分时。我来让 Lifehacker 读者 **Ken** 为本周的[**Tech 911**](https://lifehacker.com/c/tech-911)Q&A:

> 我有一个新的 Windows 10 桌面，我错误地添加了一个 PIN 码(我知道),开机后必须输入才能进入。
> 
> *一直在设置> Windows 设置>账号>登录选项> Windows Hello PIN >移除>移除>首先，验证你的账号“密码”。*
> 
> 问题是我不知道它想要什么密码，所以我可以删除这个“Hello PIN”号码？？

> 起初我以为这是普通的微软密码。我可以毫无问题地登录 Microsoft 帐户，但这不是它想要的密码！
> 
> 看起来像是某种本地的、Windows 10 电脑独有的密码，但我不知道它是什么！！
> 
> 什么都试过了。我如何清除这台联想台式电脑上的所有密码和个人识别码？？

### Windows 帐户有时会很棘手

这很有趣，肯。我也用 PIN 码登录我的 Windows 10 电脑。当我试图删除它时，我得到了同样的提示。然而，它直接要求我的微软账户的密码，我通常会用这个账户登录 Windows 10。

我的第一个想法可能听起来很明显，但我还是要提出来:你确定你输入的密码是正确的吗？那是假设你的系统要求你的“微软账户”密码，你应该知道。我会确保我非常仔细、缓慢地输入那个，并尝试几次——如果这是 Windows 10 想要的，你已经知道了。

然而，如果它要求另一个帐户的密码，这就是事情变得奇怪的地方。我会首先点击侧边栏上的“你的信息”链接，并确认你可以使用你的 Microsoft 帐户而不是本地帐户登录到你的系统。如果是后者，试试看你是否能设置前者(尽管你可能会被要求输入你的本地账户密码，这没什么用)。如果您不是，则可能切换到您的 Microsoft 帐户登录，然后允许您输入 Microsoft 帐户的凭据来删除您的 PIN。

在我进一步深入之前，我有另一个想法:正如你所说，既然这是一个新的 Windows 10 桌面，为什么不干脆销毁一切，从头开始？在 之前，我已经 [经历过这个过程。基本上，你会想在一个 u 盘上安装](https://lifehacker.com/the-ultimate-guide-to-reinstalling-windows-from-scratch-1832897572) [Windows 10 安装介质](https://www.microsoft.com/en-us/software-download/windows10) ，当你的电脑重新启动时引导到 u 盘上，然后格式化你的系统硬盘。然后，在上面重新安装 Windows 10，你就可以使用一个基本版本的操作系统，不会被任何奇怪的帐户问题搞得一团糟。

我通常不主张这么早就进行故障诊断，但假设你没有真正在你的系统上做任何事情，现在是快速恢复这个问题的最佳时机，并从头开始建立一个干净的 Windows 10 版本。(在此之前，请确保您已经将现有版本的 Windows 10 与 Microsoft 帐户关联，这样您就可以重新登录以激活您的系统。这会帮你减轻很多压力。)

否则，如果你更喜欢坚持使用你现有的 Windows 10，我想我有一个解决方案给你。尝试启动 Run (Windows 键+ R)并键入`lusrmgr.msc`，然后按回车键。您应该会看到如下所示的屏幕:

单击左边的“Users”文件夹，这应该会在中间一栏显示您系统上的所有本地帐户。寻找与您不记得(或从来不知道)其密码的任何帐户的详细信息相匹配的帐户。右键点击并选择“设置密码”忽略这个可怕的警告，选择“继续”，这样你就可以重新设置这个账户的密码了。

如果我没记错的话，你也可以通过管理命令提示符*和*来完成这个任务。在开始菜单中键入“命令提示符”，右键单击并选择“以管理员身份运行”，然后输入以下内容:`net user`

这将为您提供系统上所有用户帐户的列表。同样，找到与您不知道其密码的用户帐户相匹配的帐户，并输入以下内容:`net user [name of that account] [new password]`

显然，您需要用特定于您的设置的信息来替换括号中的部分。也就是说，这些技巧*中的任何一个都应该*适合你的情况。如果没有，还有一个更奇怪的方法来重置账户密码，你可以尝试使用 Windows 10 安装盘(或 u 盘)。但是如果你已经做了一个，并且在命令提示符下忙活，你还不如保存你的数据，重新安装 Windows。享受没有问题的新鲜的新用户体验，而不是通过一个古怪的帐户密码来排除故障。

你可以*也可以*试试像 [Lazesoft 的恢复我的密码](https://www.lazesoft.com/download.html) 这样的第三方应用，看看是否可以为你重置密码，但我倾向于用手动的方式；这只是我。

不过，我还有最后一个诀窍要告诉你。如果你不想弄乱这一切，只想能够登录到您的计算机，而不需要输入 PIN 码，拉起运行(Windows 键+ R)并输入`netplwiz`。确保您用来登录的帐户高亮显示，并取消选中“用户必须输入用户名和密码才能使用这台电脑”框。你必须输入你的帐户密码——希望是你的 Microsoft 帐户，而不是你不认识的这个本地帐户——但这将允许你永久绕过登录屏幕。

现在，很明显，你应该*而不是*在笔记本电脑上使用这个功能，因为那样会非常不安全。我只在我的台式电脑上使用它，它从不离开我的办公室，也从不被我的室友乱动。(我对这个事实非常有信心，所以没有必要就安全性进行说教。)我这么做只是因为我太懒了，每当我启动我的台式机时，我都不会一直输入四位数的 PIN 码。如果你觉得同样安全，试一试。

否则，我认为从零开始重新安装 Windows 10 是你最好的选择。事后，你可能甚至不需要安装任何联想专用的应用程序或驱动程序，但我只是说，一旦 Windows 10 启动并再次运行，你的计算机的某些组件需要一些 TLC。
