# 使用滚碗策略防止脏水滴落

> 原文:[https://life hacker . com/prevent-dirty-drips-with-the-rolling-bowl-manipulate-1845305442](https://lifehacker.com/prevent-messy-drips-with-the-rolling-bowl-maneuver-1845305442)

从视频中可以看出:全在手腕。当你把碗(或混合杯)里的东西倒进锅里，然后把它倒回原来的方向，聚集在碗边和唇边的液体会溢出来，形成一条从碗到柜台的总轨迹。为了防止搅拌好的鸡蛋(或任何其他易滴、脏兮兮的物质)在你倒完之后滴到碗边上，只要继续旋转，沿着你最初倾斜碗的方向完成360度的旋转。残留的液体会滴回你本来要洗的碗中。

我用午餐吃的炒鸡蛋试了一下，效果很好。碗的边上有一滴很小的水滴，但它太小了，以至于在它有机会到达我的工作台面之前就干涸了。滚碗策略(这是它现在的正式名称)防止了一个非常小的混乱，但这是一个非常小的混乱，你真的不需要清理。我认为它对面糊真的很有效，有些面糊(咳咳，布朗尼面糊)是出了名的粘糊糊。
