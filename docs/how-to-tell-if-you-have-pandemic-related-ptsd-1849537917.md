# 如何辨别你是否患有疫情相关的创伤后应激障碍

> 原文：<https://lifehacker.com/how-to-tell-if-you-have-pandemic-related-ptsd-1849537917>

过去的几年对我们所有人来说都压力巨大。从担心健康和安全、到处理朋友和家人的死亡或疾病，应对因失业造成的经济困难，或者不得不不断适应学校关闭和儿童保育中断，的生活一直是充满压力和担忧的过山车和。因此，有些人甚至可能正在应对由所有压力源带来的创伤后应激障碍。

“通常情况下，创伤会让你的身体进入战斗、逃跑、或冻结模式，但对于 COVID 来说，这是一种无形的威胁，”斯蒂芬妮·斯塔萨斯说，她是 prosper works 的注册专业顾问，专门治疗创伤。在过去的几年里，Stathas 发现寻求治疗的人大幅增加，其中许多人正在应对与创伤有关的症状，如焦虑、抑郁、易怒、和睡眠障碍。

## **不停的压力会导致 PTSD 症状**

创伤后应激障碍倾向于在创伤事件 后的几周内出现 [，尽管有时它会在数月或数年后出现。症状包括高度警惕、情绪回避或麻木、闪回、噩梦、易怒、焦虑、抑郁，还可能包括身体症状，如头痛、头晕或胃痛。](https://www.nhs.uk/mental-health/conditions/post-traumatic-stress-disorder-ptsd/symptoms/)

虽然我们通常认为 PTSD 是在特定的创伤事件后发生的，如在车祸或暴力袭击中幸存，但人们也可能在反复暴露于压力或创伤事件后发生这种情况。如果创伤事件正在进行，没有逃脱的机会，这可能导致所谓的复杂的创伤后应激障碍，其症状与创伤后应激障碍相似，但也可能包括内疚、羞耻、或无价值感；调节情绪的能力下降；以及建立和维持健康关系的问题。“它不再仅仅是一个单一的事件，现在你有所有这些事件，所有这些事件结合起来使它变得复杂，”Stathas 说。

复杂的创伤后应激障碍通常发生在那些在虐待环境中长大的人身上，成年后处于虐待关系中，或者经历了另一个类似的长期压力，他们无法逃脱。鉴于疫情的普遍性质，人们正在努力应对的症状往往是持续压力的结果。

然而，正如专家们开始指出的那样， [疫情是一种独特的](https://www.npr.org/2022/04/07/1087195915/covid-pandemic-trauma-mentalhealth)应激源，会有自己的创伤相关症状模式。一些专家已经创造了一个术语【COVID Stress Syndrome】，其中包括对被感染的恐惧、对疫情金融冲击的恐惧；害怕其他可能被感染的人；强迫性检查和寻求安慰；以及其他与疫情相关的压力症状。

正如 Stathas 指出的那样，过去几年的不确定性和不可预测性给我们带来了难以置信的压力。“所有这些变化，所有的时间，只是进入那些无助和对某事无能为力的感觉，这是很困难的，”Stathas 说。“仅仅是对某些事情有一种控制感就能帮助我们感觉更好，但如果两年都没有这种感觉，那就太可怕了。”

## **如果您患有创伤后应激障碍，该怎么办**

如果过去几年的压力已经达到了对你的人际关系、身体健康和幸福、整体情绪状态、产生负面影响的程度，那么尽早寻求帮助就很重要了。

“每当它抓住你，而你不知道为什么的时候，看看你没有解决的问题，没有解决的问题，”Stathas 说。“它会追上来的。我经常看到这种情况。”

一般来说，找一个受过创伤治疗训练的人是个好主意，因为有很多不同的治疗选择。对创伤后应激障碍 的一些更为常见的治疗类型包括认知行为疗法、认知加工疗法、辩证行为疗法，或 eye mmovement deee【EMDR】疗法。

根据您的喜好，一种类型可能比另一种更好。许多治疗师将接受多种类型的培训，并可能根据你的需求调整每种类型的策略。“做治疗没什么丢人的，”Stathas 说。“这与去看医生来照顾你的医疗健康没有什么不同。心理健康和幸福同样重要。”
