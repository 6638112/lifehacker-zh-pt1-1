# 如何选择天气延误几率最低的航班

> 原文：<https://lifehacker.com/how-to-pick-a-flight-with-the-lowest-chance-of-weather-1848179668>

众所周知，假期是坐飞机旅行的困难时期，部分原因是想坐飞机的人太多，部分原因是冬季天气的可能性增加。暴风雪和结冰的跑道会很快使几天的空中旅行脱轨。

但是，虽然你不能为每一个突发事件做好计划，或者在预订航班时知道天气情况，但你可以通过一些智能的调度来帮助将潜在的干扰降至最低。

## 为什么你应该选择早班飞机

一般来说， [专家建议](https://bestlifeonline.com/mistake-booking-winter-flight-news/) 尽早预订航班，因为延误和取消会在一天中累积起来。由于客运航班一般不会在夜间起飞，航空公司有一些停机时间来为未来(或管理现有的)调度混乱做准备。赶上早班飞机可以最大限度地减少陷入延误多米诺骨牌的可能性。如果你的早班航班被取消，你可以选择当天出发，而不是晚上出发。

即使是早上的第一趟航班也要做好晚点的准备，因为飞机停了一夜并积了霜或雪后，通常需要除冰。乘坐最早的航班可能仍然是有意义的，因为大多数在冬季条件下起飞的后来的飞机无论如何都必须除冰，这些操作 [不一定是先到先得](https://webfids.dfwairport.com/cs/groups/webcontent/documents/webasset/p2_626762.pdf) 。

## 尽可能直飞，即使这意味着要开得更远

虽然如果你在较小的城市之间旅行，乘坐直飞航班并不总是可行的，但这可能是最大限度减少干扰的最佳方式。你只需要依赖一架飞机的路径，没有因为延误或在途中丢失托运行李而错过转机的风险。直飞航班肯定会晚点或被取消，但总的来说，与你要赶两架或更多的飞机相比，这种体验可能会花费更少的时间和更少的挫败感。

如果你住的地方离一个主要的交通枢纽只有几个小时的路程，那么开车去一个更大的机场搭乘一趟直飞航班也许是值得的——当然，要权衡总的时间和成本与延误的可能性。

## 仔细选择你的连接

如果不能直达，不要简单默认最快或最便宜的行程。考虑飞更靠南的航线，那里出现冰雪的可能性更小。

当然，这也不是一门精确的科学。虽然北方机场可能更容易受到冬季风暴的袭击，但他们也更好地准备应对影响并保持正常运转。哪怕是一点点的雪 [都能让一个难得一见冬季天气的机场](https://www.usatoday.com/story/travel/flights/todayinthesky/2017/12/07/airlines-waive-change-fees-atlanta-because-snow/933141001/) 瘫痪。

此外，登机前你要去的机场也很重要。即使你正飞过阳光明媚的德克萨斯州，你的飞机也可能来自多雪的明尼苏达州。不幸的是，这不是你所能控制的，需要做大量的研究来提前找到飞行路线。

您可以考虑转机或在较小的机场(或主要城市的二级机场)终止，这些机场的客流量和航班量较小。万一你的第一趟航班延误或不得不除冰，稍微延长转机时间也是值得的。

## 如果天气不好，要注意其他的路线

去之前了解天气，知道有什么替代路线。如果你预计会受到取消航班的影响，你可以提前与你的航空公司重新安排航班(而不是等到所有其他滞留乘客都试图做同样的事情)。当然，要知道天气是不可预测的，航空公司的工作人员在这些时间里是非常忙碌的，而且只有这么多的航班和这么多的座位飞往你的目的地。准备和耐心是关键。
