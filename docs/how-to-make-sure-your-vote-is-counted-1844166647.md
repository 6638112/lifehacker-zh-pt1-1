# 如何确保你的投票被计算在内

> 原文:[https://life hacker . com/how-to-assure-your-vote-is-counted-1844166647](https://lifehacker.com/how-to-make-sure-your-vote-is-counted-1844166647)

正如过去几个月所证明的那样，至少可以说，这个选举周期对选民来说将是一个具有挑战性的周期。

有这么多的不确定性和虚假信息——无论是关于邮寄投票的完整性，投票机技术问题的可能性，社交距离协议的需要，还是选民恐吓的幽灵——你如何确定你将能够投票，你的投票将被正确计算？

在上面的视频中——制作于2020年初选之前——劳尔·马西亚斯，布伦南司法中心的律师，就选举前、选举中和选举后该做些什么提出了建议。

首先，检查你的选民登记。如果你最近搬家或在过去几次选举中没有投票，投票工作人员可能会说你的登记已经过时。

接下来，**查看你所在地区的正确身份识别规则。**确保你在投票时准备好了所需的文件。

马西亚斯指出，几十年来，通过邮件投票一直是一种可靠的投票方式，只是在最近的选举周期中，人们才试图破坏它的可信度。**通过邮件投票**，请确保您按照选票上的说明处理任何邮资、信封或您可能需要的证人。

如果你去投票站看到一条很长的队伍，最重要的事情是 ***排队*** 。只要你在投票结束前排队，你就可以投票。人们可能会告诉你不是这样，但他们只是想让你不投票就离开。

如果出于任何原因，一名工人告诉你不能投票，要求与主管交谈，填写临时选票并获得收据，以便你可以跟踪它是否被计算在内。投票时如有任何问题，请拨打 [选举保护热线866-OUR-VOTE](https://866ourvote.org) 。
