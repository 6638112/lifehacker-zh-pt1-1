# 在墓地跑步可以吗？

> 原文：<https://lifehacker.com/is-it-ok-to-run-in-cemeteries-1848069962>

纽约皇后区的 [髑髅地](https://calvaryandalliedcemeteries.com/calvary-cemetery/)[埋葬了超过三百万具尸体，号称是美国所有墓地中尸体最多的](https://www.atlasobscura.com/places/calvary-cemetery) 。这也是我最喜欢跑步的地方之一。在一座起伏的山顶上，你可以转身眺望一排排墓碑，直到石板的地平线与曼哈顿的天际线相遇——这是一种发自内心的、视觉上的生与死的冲突。令人叹为观止。(我在这篇文章的草稿之间慢跑时拍摄了上面的标题图片)。



然而，每次我去墓地慢跑时，我都会想起*弗莱巴格*T5】中的一个场景，其中弗莱巴格的姐姐告诉她，“在墓地周围慢跑，炫耀你的生活真的不合适。”你或你认识的人可能会因为跑过一个墓地而被骂甚至被吼。我不禁怀疑我是不是错了。在屏住呼吸的人群中屏住呼吸是一种固有的不尊重吗？

我不是唯一一个穿越墓地的人，我也不是唯一一个质疑墓地伦理的人。Reddit 上不乏 [的线程](https://www.reddit.com/r/running/comments/2qpoej/is_it_disrespectful_to_run_in_cemeteries/) 致力于墓地管理礼仪。尤其是在很难找到开阔空间的地方，墓地提供了一个难得的独处和沉思的机会。与此同时，许多人觉得自己像弗莱巴格的妹妹；一想到践踏死者，就有一种不尊重的感觉。我在 Twitter 上四处打听，并从一些在墓地工作的人那里了解到，人们对在你当地的墓地周围跑步的道德观是如何看待的。

## **关于墓地的快速历史课**

墓地和公园有着悠久的、 [交织在一起的历史。事实上，墓地是美国人可以进入的第一批公园。但是在这条路上的某个地方，墓地从公共目的变成了私人名誉。(为了更全面地了解死亡行业的迷人历史，我强烈推荐凯特琳·道蒂的《T4》](https://publichistoryofcemeteries.org/cemeteries-as-parks/)[*烟雾进入你的眼睛&来自火葬场的其他教训*](https://www.goodreads.com/book/show/25189315-smoke-gets-in-your-eyes-other-lessons-from-the-crematory) 】。

现在，虽然听起来很病态，但是墓地已经没有空间了。这意味着他们需要更多的资源来保持开放和合理。马克·特雷斯肯 [在为城市研究所](https://www.urban.org/urban-wire/can-cities-reclaim-cemeteries-public-space) 撰写的文章中写道，我们应该根除我们对墓地固有的保留观念，转而关注它们作为生者空间的作用:

> 在美国，我们通常认为墓地是安静反省和悲伤的私人场所。但是，随着城市寻找新的方式来最大限度地利用现有的基础设施，以造福居民，墓地正在发挥越来越大的作用...通过向公众敞开大门，墓地“证明了它们在公共领域的继续存在和相关性，并寻求额外的资源来保持生存。”

那么，墓地会是给活人的吗？看起来很多跑步的人和不跑步的人似乎都这么认为。

## 反思墓地的作用

类似于特雷斯肯关于墓地适应服务于生活社区的信念，我的适度 Twitter 帖子 的一位回复者就此话题分享了对墓地角色的热情看法。 [本·西尔弗](https://twitter.com/ThisIsBenSilver) 认为“空间是给活着的人的”，每个墓地都应该变成公园。“我理解纪念生命的愿望，”西尔弗说，“但最好的方式不是扣留一些土地作为人质，让活着的人无法享受它。”西尔弗接着说，“在墓地跑步的人(可能是无意的)正在进行一场意义深远的空间开垦……在一场关于将人类生活重新定位为纪念死者的最重要方式的对话中。”(我猜想这是弗莱巴格想告诉她姐姐的话。)

费城劳雷尔山公墓的项目主任艾玛·斯特恩在接受《跑步者的世界》 采访时说，她的墓地“对生者和死者来说都一样”

## 专家一致认为:尊重就好

Twitter 上的另一位回答者要求使用 A. Slinger，他在殡葬行业工作了 10 年，为几个家庭和州法医办公室工作。斯林格 2007 年从太平间学校毕业，在一家殡仪馆上面住了一年，现在仍有业内朋友。

在回应跑步的伦理时，斯林格说他们“从未听说过使用墓地锻炼是禁忌。”他们说，他们不会慢跑经过正在进行的墓地服务，但他们甚至不会走过一个。就慢跑者尊重空间而言，Slinger 想到的唯一问题是人们没有清理他们的狗。

我还打电话给皇后区 Calvary Cemetery 办公室的一名职员(要求匿名)，他告诉我，尽管跑步“在技术上违反了基本上所有墓地的规定”，但普遍的共识是，跑步通常是可以容忍的。从她的角度来看，慢跑者没有什么大的道德禁忌，只要他们尊重。

髑髅地墓地的工作人员也指出大小很重要——因为髑髅地如此之大，人们很容易在奔跑中不经过另一个(活着的)灵魂。有足够的空间来避免打扰任何服务或任何人参观坟墓。

## 如果你在墓地跑步，你应该做什么？

查看当地墓地的规则。正如 Calvary Cemetery 的职员指出的，许多私人拥有的墓地会有一些指示，表明不允许在这片土地上跑步(例如，在布鲁克林标志性的 [格林伍德墓地](https://www.green-wood.com/) )。然而，似乎有一种对那些规则视而不见的规范。许多墓地没有官方立场(至少，他们不会在网上发布)，但这并不意味着你应该认为你有权在那里慢跑，尤其是如果它是私人所有的话。

**实践普通的礼貌。**在《跑步者世界》的采访中，斯特恩还提到“所有的跑步者都待在铺好的路上。他们又不是在墓地里跑来跑去。”只要你考虑到自己所处的位置，墓地之旅并不一定具有冒犯性。这意味着潜在地改变你的路线，远离哀悼者，如果你忍不住要经过一个，那么考虑加入散步休息作为尊重的标志。

我的最后一个建议是，当你被死人包围时，摘下你的耳机，这样你就可以专注于你的跑步，作为活着的证明。
