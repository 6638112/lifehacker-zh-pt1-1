# 25部今天永远拍不出来的电影

> 原文:[https://life hacker . com/25-今天永远无法制作的电影-1847900045](https://lifehacker.com/25-movies-that-could-never-be-made-today-1847900045)

好莱坞制作剧场版电影的方式发生了巨大的变化。COVID以我不确定我们可以预见的方式改变了景观，但也可能只是加速了已经开始的趋势:更短的发行窗口，同步流媒体首映，以及作为观看新电影的*地方的电影院的普遍侵蚀。* 

*但十多年来，市场一直在不断变化——尤其是漫威影业有限责任公司的崛起和全球影响力，巩固了四象限大片作为好莱坞工作室主导模式的地位。如今，超级电影明星越来越少了——至少在特许知识产权领域之外。高质量电视的绝对数量(以及巨大电视不断下降的价格)确实让我对去看电影三思而行——甚至是在疫情之前。*

*从最广泛的意义上来说，这些因素和更多的因素导致了为成人制作的中等预算电影的死亡，至少是在影院。主流趋势是超低价(至少用好莱坞的术语来说)的布伦屋制片公司，或者大预算的超级大片，没有太多的中间选择——电影公司不希望在一个问号上花大钱。电影制作成本很高，人们需要一个去电影院的理由:很难让观众对一部安静的电影产生热情，一两个月后，你就可以在家里用便宜得多的爆米花观看这部电影。* 

*这不是长篇大论，也不是争论电影曾经更好。当我们回首往事时，我们只会想到最好的、最难忘的电影，这扭曲了我们的视角。在电影制作的每个时代都有很多蹩脚的东西被制作出来——今天正在制作的伟大电影在30年前是绝对没有机会的，反之亦然。*

*有些我们不再看的电影感觉像是损失，有些我们不看会更好。其中一些是周期性的:几年前，我会说我们再也不会看到曾经流行的那种大预算、全明星侦探小说了——但那是在《利刃出稍》和《T2在东方快车上的谋杀》之前，这两部电影都有续集(尽管后者是由网飞资助的)。除此之外，出于各种原因，以下25部电影肯定不会在今天拍成。*
