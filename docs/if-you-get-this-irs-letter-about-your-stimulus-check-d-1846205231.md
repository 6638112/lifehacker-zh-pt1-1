# 如果你收到国税局关于你的刺激计划的信，不要惊慌

> 原文：<https://lifehacker.com/if-you-get-this-irs-letter-about-your-stimulus-check-d-1846205231>

如果你收到国税局的“通知 CP21C”，暗示你的刺激付款将用于偿还现有的税收债务，不要惊慌，这封信是错误发送的。国税局不会扣发你的付款，不管你是已经收到了支票，还是计划将退税作为你纳税申报表的一部分。

### **这封信是对过去的一次冲击**

如果你是收到 这封信的 [109，000 人中的一员，你可能已经注意到有些不对劲了，因为它看起来像是美国国税局偶然翻出了 14 年前的一封邮件。它指出:](https://www.taxpayeradvocate.irs.gov/news/nta-blog-economic-impact-payments-the-mystery-of-the-cp21c-letter/)

> 由于新的立法，我们将一笔信贷应用于您的 2007 年税收帐户。在法律允许的情况下，我们使用(抵消)了您的全部或部分经济刺激付款来支付您的联邦税。

这是一个不幸的失误，因为对于国税局是否可以对未支付的税单或拖欠的学生贷款收回或征税刺激支票或纳税申报表中的信贷，一直存在很多困惑。请放心，对绝大多数纳税人来说，情况并非如此:第一笔刺激付款只能用于子女抚养费，第二笔刺激付款根本不能用于任何债务。这些支票也不用缴税。

“这个通知对任何收到的人来说都是不准确的。因为没有付款，所以没有抵消发生。我们对此可能造成的混乱表示歉意。你可以无视这个通知，“美国国税局在其更新的常见问题 中声明， [。这封信还建议打一个普通电话号码来解决这个问题，但是不用麻烦了——电话线路几乎总是被淹没。](https://www.irs.gov/coronavirus/second-eip-faqs)

### 它应该说什么

根据 [纳税人倡导服务](https://www.taxpayeradvocate.irs.gov/news/nta-blog-economic-impact-payments-the-mystery-of-the-cp21c-letter/) 的说法，这封信本应声明国税局不能签发第一张支票，因为国税局尚未处理纳税人的 2019 年申报表，也没有 2018 年 ret urn 作为支付依据。然而，解决方案是在你的 2020 年纳税申报表上申请 [回收退税抵免](https://www.irs.gov/newsroom/recovery-rebate-credit) 。有关申领积分的更多信息， [点击此处](https://www.irs.gov/newsroom/recovery-rebate-credit) 。
