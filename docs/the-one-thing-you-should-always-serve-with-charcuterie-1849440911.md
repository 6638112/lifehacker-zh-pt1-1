# 有一件事你应该永远和熟食拼盘一起吃

> 原文：<https://lifehacker.com/the-one-thing-you-should-always-serve-with-charcuterie-1849440911>

作为派对主持人，你处于游戏的顶端，这意味着你知道如何让大画面变得生动，并且每个细节都被整理出来。无论是食物、衣帽间，还是备用冰块，你的客人都应该能够玩得开心，本能地按照你的计划做每一件事。这就是为什么你应该用一个空碗来盛你的肉和奶酪。

Watch

在丰盛的水果、奶酪、开胃菜和肉片中拿出一个空碗似乎是不必要的(或者完全是错误的)，但这就像在餐馆里多拿一碗辣鸡翅或贻贝一样:你需要一个地方来放不需要的残渣。它清除了你盘子里的贝壳和骨头，所以你可以专注于活跃的可食用食物，这有助于整体的更有组织的体验。

核碗可以用于橄榄核，但它更像是一个普通的弃碗。它可以小到一个调味盘，一个拉面饼，或者一个汤碗，只要比你认为你需要的多一点空间就可以了。聚会时总会有意想不到的碎片出现，比如核果核、核、茎、壳或牙签。如果你不把这个碗和你的肉和奶酪放在一起，你就有可能把所有这些碎屑放在奇怪的地方，比如卷在窗台上、茶几上或吧台上的废弃餐巾里。一些客人可能会觉得不得不经常去厨房倒垃圾，这意味着他们没有那么放松，更重要的是，当你准备其他小吃时，他们会妨碍你。

丢弃的碗可以让你的朋友继续他们的谈话，保持一个看起来新鲜的盘子，并且让你以后清理起来更容易。缺少果核碗最糟糕的结果是客人可能会把茎、皮或果核留在半满的盘子上。帮他们帮你。

做一个真正的英雄，避免混乱，在你的客人到达之前吃一点聚会的食物 并在碗里放一些核或牙签以表明这是做什么的。在聚会期间定期用一个新的空碗把它倒空。
