# 如何在您的桌面上设置 Skype 的主动噪音消除功能

> 原文：<https://lifehacker.com/how-to-set-up-skypes-active-noise-cancellation-on-your-1846496752>

可能已经取代“Skype”成为视频通话的首选简写方式，但微软的聊天服务还没有普及。最新的 Skype 升级为台式机和笔记本电脑带来了主动噪音消除设置(ANC ),可以滤除背景声音，同时保持您的声音清晰可闻。

这项功能建立在微软用于团队通话的基于机器学习的噪音消除技术上。换句话说，人工智能被训练来识别声音模式和环境声音之间的差异；奇迹就是这样发生的。

您可以在 Windows 和 macOS 应用程序的 Skype 音频和视频设置中启用 ANC。然而，ANC 在基于网络的桌面客户端上不可用，而且它跳过了 Skype 在 iOS 和 Android 上的移动应用程序——这些应用程序可能会从 ANC 功能中受益更多。但至少新的降噪技术将有助于让你楼上邻居的脚步声远离你每周在电脑上进行的 Skype 会话。

### 如何在 Skype 中启用主动噪音消除

1.  在 Windows 或 Mac 电脑上打开 Skype 应用程序。
2.  按 **Ctrl +，**打开设置菜单，或者点击联系人栏左上角的三点按钮，点击**“设置”**
3.  选择**音频&视频**
4.  向下滚动到菜单的音频部分，然后选择您希望应用于通话音频的噪声消除级别:**“低”、“高”、“自动”**或**关闭。**自然，低的会过滤掉较少的噪声，高的会过滤掉较多的噪声。Auto 是默认选项，将根据背景声级在低噪声消除和高噪声消除之间动态切换，但如果您注意到 ANC 太不一致，手动切换到低噪声消除或高噪声消除可能会有所帮助。
5.  做出选择后，您可以关闭设置窗口。下次通话时，Skype 会自动将它的噪音消除魔法应用到您的麦克风音频中。
