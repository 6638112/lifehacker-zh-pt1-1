# 如何在任何浏览器中使用谷歌搜索的新黑暗模式

> 原文：<https://lifehacker.com/how-to-use-google-searchs-new-dark-mode-in-any-browser-1847666695>

你能感觉到黑暗在互联网上蔓延吗？这是新的黑暗模式推出的主要谷歌搜索网站的权利。没错，谷歌搜索现在有了自己专用的黑暗模式。



如果这听起来对你来说很熟悉，你可能会想到谷歌搜索移动应用程序中的黑暗模式设置，或者谷歌 Chrome 专用的浏览器级黑暗模式，它已经为移动用户 和桌面用户的搜索页面提供了一个烟雾般的 [。然而，这种新的黑暗模式设置专门适用于谷歌搜索网站，可以在任何浏览器中启用。打开它会显示主登录页面、你的搜索结果和所有菜单，谷歌经常使用炭黑色来显示更暗的用户界面配色方案。](https://lifehacker.com/how-to-enable-chromes-new-dark-mode-on-android-1834339091)

谷歌搜索的新黑暗模式仍在推出，所以一些用户可能还没有它——尽管它最终会对每个人开放。以下是检查设置并打开它的方法:

1.  [在您选择的浏览器中打开谷歌搜索](https://www.google.com/) ，然后执行搜索。
2.  单击搜索结果页面右上角的设置图标。
3.  选择**“搜索设置”**
4.  转到**“外观”**选项卡，，然后从列表中选择您喜欢的主题选项。**【黑暗主题】**将强制谷歌搜索显示黑暗主题，即使你的操作系统或浏览器不显示。**“Light theme”**选项做同样的事情，只是针对默认的白色 UI。如果您在 Windows 或 Mac 设置中启用了黑暗模式，并且希望您的浏览器与您当前的系统级 UI 相匹配，您也可以选择**“设备默认”**。
5.  点击**“保存”**确认您的选择。搜索页面将刷新为黑暗主题。

请注意，如果您删除浏览器数据和/或取消链接您的 Google 帐户，您的 Google 搜索设置将重置为默认设置。使用 [Chrome 的访客模式](https://lifehacker.com/how-to-set-up-guest-mode-on-your-android-1835913254) 时，谷歌搜索页面也会显示默认的光线模式 UI。

谷歌搜索并不是唯一一个有专门黑暗模式的谷歌服务。正如我们上面提到的，少数其他谷歌应用程序有黑暗模式选项可用，如 Chrome、 [【谷歌文档】和 Sheets](https://lifehacker.com/how-to-enable-google-docs-new-dark-mode-theme-on-androi-1844364782) ，加上有几个 [第三方浏览器扩展](https://lifehacker.com/add-a-dark-mode-to-any-website-with-this-chrome-extensi-1826906984) 填补了空白。
