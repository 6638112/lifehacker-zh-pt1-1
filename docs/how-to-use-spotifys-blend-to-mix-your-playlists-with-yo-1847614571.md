# 如何使用 Spotify 的“Blend”将您的播放列表与您的朋友混合

> 原文：<https://lifehacker.com/how-to-use-spotifys-blend-to-mix-your-playlists-with-yo-1847614571>

通过 Spotify 与朋友分享音乐的方式有很多，但该公司正在推出一项新的“混合”功能 ，让用户自动创建一个共享播放列表，根据双方听众的音乐品味每天更新。所有免费和高级用户都可以使用混合，您可以邀请任何其他用户一起创建混合。Spotify 自己处理监管过程，用它认为两个听众都会喜欢的曲目填充播放列表。



然而，Blend 并不仅仅关注共同的爱好。任何一个人喜欢的，添加到他们的曲库的，或者在过去听过的歌曲都可以出现在播放列表中。

当你开始一个新的混合时，你会看到你的“口味匹配分数”，看看你的偏好与你朋友的偏好如何匹配，但即使是口味相似的人也可能会得到意想不到的——或者完全奇怪的——混合播放列表。

为了测试这个特性，我和我的搭档做了一个混搭。根据 Spotify 的数据，我们有 73%的口味匹配分数——这使得，因为她和我都是死忠的重金属(我们真的是在一个酒吧相遇的)。

尽管如此，这轻微的 27%的偏移形成了一个相当折衷的组合，我们最喜欢的死亡金属和硬核朋克乐队与梅根·赛恩分享聚光灯，运行珠宝和视频游戏配乐。

当然，如果你的混音冲突太多，两个用户都可以在不影响另一个用户的播放列表的情况下，从他们自己的混音中自由选择或隐藏歌曲。但怪异播放列表的潜力显然是 Blend mode 吸引力的一部分:看看你的音乐品味如何与你的朋友和家人保持一致，并相互分享新音乐。

## 如何在 Spotify 中与人融合

1.  打开 Spotify，选择**“搜索”**标签。
2.  向下滚动并轻触**“为您定制”**
3.  选择**创建一个混合**
4.  选择一个朋友来创建一个混合。如果您还没有任何 Spotify 好友，请点击**“邀请”**，通过您的消息、电子邮件或社交媒体应用程序与电话联系人共享 Blend 链接。
5.  你的朋友需要接受邀请来创建混合。一旦混合就绪，您将收到应用内提醒和推送通知。您还可以点击**“分享这个故事”**在其他社交媒体应用程序上分享播放列表。
6.  您的混合可以从“库”选项卡中访问。
