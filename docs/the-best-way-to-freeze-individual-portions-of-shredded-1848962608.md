# 冷冻单份鸡丝的最好方法

> 原文：<https://lifehacker.com/the-best-way-to-freeze-individual-portions-of-shredded-1848962608>

让我先说:我晚餐会做四样东西。我有三个不到十岁的孩子，我不会去做实验，吃听起来很美味的咖喱扁豆汤，或者新的有趣的口味。我想尝试的“有趣的食谱”遭到了几次令人心碎的拒绝，比如中东烤虾串和绝对炸弹黄油鸡，这才让我吸取了教训。在大约第一百次“这真难吃”之后，我身体里任何曾经冒险的烹饪骨头都已经离开了这座建筑，它将一直留在那里，直到和我住在一起的小奶酪和喜欢 dino nuggie 的人在面对味道棒极了的新食物时不再呕吐。



也就是说，这四种(非常重复的)食物中有一种是鸡肉卷。每周我都做这些，每周都有鸡丝剩菜。然后我把鸡肉放在特百惠里，让它在我的冰箱里慢慢腐烂，因为它会让我充满罪恶感，不能直接把它扔进垃圾桶。(还因为，尽管有相反的证据，我仍然希望我会把那份鸡丝用到别处。错觉的力量是强大的。)

每次我买鸡丝(或者从烤鸡上剥下鸡肉，虽然更划算、更丰富，但还是让我感到恶心)时，这种情况几乎都会发生。)

所以这一招，来自[life hacker reader u/knowone LSE](https://kinja.com/knowonelse)，正适时走进我的生活。嗯，实际上晚了一天——我昨晚刚刚做了每周一次的把鸡肉扔进垃圾桶/自我鞭笞的例行公事——但不管怎样，这是记住这一课的好时机。

把任何剩下的烤鸡或预先包装好的鸡丝，舀进松饼罐的杯子里，冷冻。 一旦冷冻，将那些小份倒入冷冻袋中备用。这样，鸡肉可以小批量解冻，以防止大量解冻导致进一步浪费和变质。(如果你从好市多 购买大量的手拉烤鸡， [比如这个，这特别有帮助。)将这些解冻后的小份加入鸡肉沙拉、鸡肉馅饼、鸡汤、下周的墨西哥玉米饼——任何需要鸡肉丝或烤鸡的食谱。](https://www.eatwithemily.com/bag-of-hand-pulled-rotisserie-chicken-breast-meat/)

你需要做的只是在你的冰箱里有足够的空间来放松饼罐，并且有足够的精神力量来记住你已经把它冷冻起来了。
