# 是的，你可以在Chromebook上玩游戏

电脑游戏被认为是一项“高端”的电脑任务。虽然许多 [游戏将运行在虚弱或过时的个人电脑和笔记本电脑](https://lifehacker.com/15-of-the-best-steam-games-you-can-play-on-weaker-pcs-1848255566) 上，但最大、最新的 游戏需要强大的硬件和游戏友好的操作系统如Windows或steam OS——这些功能是Chromebooks等小型生产力机器所缺乏的。

然而，尽管ChromeOS的硬件规格不足，游戏支持也乏善可陈，但Chromebooks实际上可以成为可行的游戏设备...如果你有正确的设置。

## 使用云游戏服务

云流媒体是在ChromeOS中玩游戏的关键。Chromebooks不能原生玩 *Halo Infinite* 或 *Elden Ring* ，但它们*可以*通过云流媒体玩这类游戏，这可以将你远程连接到强大的硬件，实际上是在做繁重的工作。你可以通过连接到Chromebook的控制器或键盘来控制动作，游戏通过互联网传输，直接到ChromeOS设备的屏幕上。(这和 [安卓](https://lifehacker.com/this-playstation-remote-play-app-is-better-than-sonys-1846346283) 和 [iOS设备](https://lifehacker.com/how-to-stream-google-stadia-on-ios-ipados-1845223254) 可以用来 [玩最新的PC和主机游戏](https://lifehacker.com/how-to-play-on-xbox-cloud-streaming-and-why-its-such-a-1847197043) 。)

ChromeOS设备上有几项云流媒体服务，可以通过Chrome网络浏览器或专用的ChromeOS应用程序获得。其中包括亚马逊Luna、谷歌Stadia、英伟达的GeForce Now、Shadow、PlayStation Plus Premium(之前的PlayStation Now)和Xbox Game Pass Ultimate。

值得注意的是，所有的云游戏选项都是付费服务，具有不同的功能和游戏库。在某些情况下你必须购买或租用你想玩的游戏(除了每月的订阅费)；在其他情况下，你每个月都有机会接触一系列游戏。一些服务提供两种选择的组合。

下面是每项服务的定价和内容选项的简要概述:

*   亚马逊Luna: 亚马逊Prime会员包含的基本服务(15美元/月或139美元/年)，包括一个旋转游戏库。额外的收费“频道”提供更多游戏，每月额外收费5-18美元。
*   **Google Stadia:**Google Stadia可以让你免费购买游戏和流媒体，或者你可以以每月10美元(30天免费试用)的价格访问Google Stadia Pro游戏集。
*   **PlayStation Plus Premium:**PlayStation的 [即将推出的PlayStation Plus Premium](https://lifehacker.com/should-you-cancel-or-upgrade-your-playstation-plus-su-1848721975)(18美元/月)将把当前的PlayStation Plus(10美元/月)和PlayStation Now(10美元/月)服务合并成一个单一的订阅，其中包括从所有以前的PlayStation家用游戏机和手持设备上对超过740个游戏库的云流媒体访问，以及可以在PS5上玩的免费月度游戏。PlayStation Plus将于今年6月推出。PlayStation Now云流媒体在那之前都是可用的。
*   **Xbox Game Pass Ultimate:**Xbox的最高Game Pass订阅层包括对数百款Xbox和PC游戏的云流媒体访问，这些游戏可以通过旋转收藏或按菜单购买(并非该服务上的所有游戏都支持云流媒体)。新用户第一个月只需支付1美元，之后每月支付15美元。
*   与其他服务不同，**GeForce Now**(10美元/月起)和**Shadow**(30美元/月起)不提供游戏购买或租赁服务。相反，您需要付费远程访问功能强大的游戏PC硬件，在那里您可以玩通过Steam或Epic Games Store等外部市场购买的游戏。
*   同样值得注意的是，如果你家里有一台游戏PC、Xbox主机或PlayStation 4或5，你可以使用Steam、Xbox和索尼的远程播放应用程序在ChromeOS设备上远程播放。

不同服务提供的游戏会有所不同，所以请确保查看每个游戏当前的收藏，看看你最喜欢哪个。最近ChromeOS泄露的消息表明，谷歌将很快让你通过流媒体服务查看哪些游戏可用，其中 [是Chromebooks](https://9to5google.com/2022/03/30/google-aggregate-cloud-streaming-games-chromebooks/) 上的一个新游戏搜索聚合器，但该功能尚未正式宣布。现在，你必须单独浏览每个服务的库，看看有什么可用的。

虽然正确的服务和游戏是Chromebook云游戏设置中最重要的部分，但还有几个最后的元素需要记住。

## 你需要一个良好的互联网连接

由于云流媒体质量取决于你的互联网连接而不是你设备的实际硬件，你需要确保你的速度达到任务。有些服务不会让你在没有至少每秒15Mbs下载速度的情况下订阅，但这只是最低速度。f 或者最好的体验，使用硬有线以太网连接——要么通过Chromebook的内置以太网端口，要么通过USB以太网适配器。如果你在玩无线游戏，确保你靠近你的wifi路由器或者使用5G移动热点。只要记住: I nput延迟、服务中断和图形质量波动总是有可能的，即使是在最好的连接上。

## 考虑你的输入选项

最后，你要确保你能真正*控制*你正在玩的游戏。大多数电脑游戏都需要内置在笔记本电脑中的鼠标和键盘，但连接全尺寸键盘和外部鼠标会更舒服。很多游戏还支持游戏手柄，通常可以通过USB连接到你的ChromeOS设备，或者通过蓝牙无线连接。

有了所有这些组件，你的Chromebook可以成为一个功能齐全的云游戏解决方案，让你能够玩许多最大的新PC游戏(甚至许多主机游戏),而价格只是建造你自己的游戏PC 的一小部分。
