# 眼冒金星的实际颜色，因为不，他们并不都一样

> 原文：<https://lifehacker.com/how-to-see-stars-actual-colors-because-no-theyre-not-a-1845817332>

当我们描绘外层空间时，它往往是小学科学教科书版本:黑暗(通常是黑色)的天空，有明亮的白色星星和一些来自火星和土星等行星的颜色。但在现实生活中，星星也有各种不同的颜色——不借助望远镜或其他设备，我们并不总能看到。好消息是，一年之中，冬季是 [一睹明星们](https://earthsky.org/tonight/can-you-see-the-different-colors-of-the-stars) 真面目的最佳季节。以下是需要了解的内容。

### 等等——星星是不同的颜色？

快速抬头看一眼天空，你可能会认为星星都是亮白色的。或者，如果你成长于 20 世纪 90 年代，你的天花板上贴着那些夜光星星贴纸，你可能会认为它们都是黄绿色的霓虹灯。但是没有:那里有丰富的颜色。根据 [EarthSky](https://earthsky.org/tonight/can-you-see-the-different-colors-of-the-stars) 的说法，星星在天空中爬得越高——在地球大气层的湍流之上——星星的颜色就变得越鲜艳。

### 如何看到星星的颜色

在 [漆黑晴空](https://lifehacker.com/how-to-find-a-truly-dark-location-for-the-best-photos-o-1844881808) 的冬夜，视力相对较好的人应该能看到一些最亮的星星的颜色。如果条件合适，但你仍然什么也看不到， [EarthSky 建议用一副双筒望远镜再试一次。但我们不会在没有更具体指示的情况下就把你送到黑暗的荒野中:这里有一些来自 EarthSky 的提示可以帮助你:](https://earthsky.org/tonight/can-you-see-the-different-colors-of-the-stars)

1.  晚上，在东北方的天空中，寻找一颗明亮的金色星星，叫做“[”](https://earthsky.org/tonightpost/brightest-stars/capella-is-the-stellar-beacon-of-auriga-the-charioteer)，是御夫座的一部分。当这颗星在天空中较低的时候，它往往会闪烁。

2.  在御夫座下方，地平线上应该有一颗明亮的红星。这是阿鲁迪巴，它是金牛座的一部分。

3.  还有 [埃尔纳斯](https://earthsky.org/tonightpost/brightest-stars/elnath-taurus-the-bulls-second-brightest-star) ，一颗蓝白色的星星，正式属于金牛座，但它通常也被认为是御夫座的一部分。
