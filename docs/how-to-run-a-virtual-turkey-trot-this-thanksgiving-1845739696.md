# 如何在这个感恩节跑一场虚拟火鸡赛跑

> 原文:[https://life hacker . com/how-to-run-a-virtual-turkey-trot-this-感恩节-1845739696](https://lifehacker.com/how-to-run-a-virtual-turkey-trot-this-thanksgiving-1845739696)

在感恩节早上或接下来的周末进行一场比赛——通常被称为“火鸡赛跑”——是一种庆祝节日并看到你秋季跑步训练回报的有趣方式。但在《疫情时报》看来，虚拟比赛比亲自跑步更安全，幸运的是，有很多不错的选择。

安全提示:如果你能和其他人保持距离，在户外跑步是相当安全的，即使没有面罩。但是，进行面对面比赛的过程通常包括在畜栏中与其他人站在一起，在穿梭巴士上与其他人分组，如果是寒冷天气的比赛，则在你能找到的任何温暖的建筑物中挤在一起。因此，虽然真人比赛并非不可能安全，但今年是虚拟比赛的大好时机。

要跑一场虚拟比赛，你要像注册一场真正的比赛一样注册，然后自己跑完全程。“虚拟”比赛提供了区分比赛和跑步的一个或两个因素:竞争排名和一个大礼包。

### 奖品袋比赛

这些虚拟比赛收取注册费，然后他们给你邮寄一包好东西。虽然大多数真人比赛只是给你一件t恤，但虚拟比赛组织者明白你可能是为了战利品，所以他们往往会提供更好的东西。毕竟，你的报名费不需要包括任何实际比赛的费用。

[美国的土耳其快步](https://americasturkeytrot.run/anywhereusa/) 售价40美元，提供一件夹克、一顶无檐帽和一枚完成者奖牌。

达拉斯基督教青年会土耳其小跑 售价35美元，提供一件t恤；您还可以在排行榜上记录您的时间，并有可能赢得随机大奖。额外支付5美元就能获得一枚决赛奖牌。

洛杉矶火鸡赛跑 售价37美元，提供一件t恤或背心，以及一枚完成者奖牌。继续执行午夜任务，帮助无家可归的人。

达纳点火鸡小跑 有多种选择，从39美元的t恤和奖牌套餐，到15美元的奖牌套餐和5美元的电子商品门票。

更多选择，请查看来自Active.com的， [这份来自跑步者世界的列表](https://www.runnersworld.com/races-places/a34731078/virtual-turkey-trots/) 或者谷歌一下你的城市或地区+“土耳其小跑”许多传统上亲自举办的比赛今年有了虚拟选项，所以在许多情况下，你仍然可以支持当地的俱乐部和慈善机构，如果你没有这样做，当你的朋友在比赛网站或社交媒体上发布时间时，你可以为他们加油。
