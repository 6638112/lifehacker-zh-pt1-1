# 搬家时打包小而棘手物品的最佳方式

> 原文：<https://lifehacker.com/the-best-ways-to-pack-small-tricky-items-while-moving-1846535279>

我们已经介绍了打包常规移动箱子 的 [最佳技巧，但是当你搬家时，你需要打包*所有的东西*——甚至是那些你很容易丢失的微小、烦人的物品。因此，这里是我们包装这些棘手的、不规则形状的杂烩的最佳方法。](https://lifehacker.com/the-best-way-to-pack-a-moving-box-1838011535)

在厨房里，把你的香料瓶放进锅里，盖上锅盖。它们应该贴合，不会发出很大的嘎嘎声。如果你的刀没有盖子，试着把它们塞在挂钩里。不过，在你打开包装的时候，一定要记得刀子在哪里。银器也可能很棘手(因为叉子和刀子在你打包的地方容易戳洞)，所以把你的银器放在抽屉里，用几层保鲜膜紧紧包好。这使得拆包更加容易。

你也可以使用塑料购物袋作为额外的包装材料。简单地把它们塞在饮料类或任何松散的盒子里作为额外的填充物。为了跟踪家具的螺丝和螺栓，将它们收集在一个小包里，并用胶带粘在相应家具的内部。

最后，为了让搬家变得轻而易举，记得在野餐篮子里装一些零食、一次性盘子、塑料器皿和纸巾。你不会想在紧张的一天变得烦躁不安。
