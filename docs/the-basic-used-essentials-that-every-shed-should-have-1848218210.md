# 每个棚子都应该具备的基本使用要素

> 原文:[https://life hacker . com/the-basic-used-essentials-that-every-shed-should-has-1848218210](https://lifehacker.com/the-basic-used-essentials-that-every-shed-should-have-1848218210)

对于草坪护理和各种户外(和室内)DIY项目，你的棚子通常是你的主要储存容器。不过，所有这些工具和材料可能会很贵——尤其是如果你买的都是新的。幸运的是，你可以买到很多质量仍然很好的二手物品，这样你就不会被价格震惊了。以下是我们建议您购买的二手物品。

## 仓库

如果你买全新的，户外储物会很贵。但是储藏箱，架子，甚至挂钩和挂钉板都是二手的。有时，你甚至可以在社区的“什么也不买”小组上免费获得垃圾箱。在旧货店、当地转售网站和二手商品应用程序上保持你的眼睛和头脑的开放。虽然你可能没有最闪亮的配套架子和箱子，但有时你可以得到一些便宜或免费的储物装备。

另一个购买挂钩、挂钉板和户外储物件的地方是你当地的人类栖息地恢复或再生建筑材料商店。这将节省你的钱，并减少杂乱，因为你建立你的工具集。(如果你的架子或挂钩太脏，不合你的口味，那就刷一层新漆吧。)

## 手工具

手工工具是另一个可以快速增加的成本。在你买这些新东西之前，先看看你是否能在二手应用程序或旧货店找到它们。有时，你甚至可以买到质量更好的二手工具，价格和质量较低的新工具一样或更低。这实际上可以节省你的钱和挫折，因为它们可能比新的替代品更耐用，更容易使用。

## 铲

可以找到各种类型的铲子——如果你自己做景观美化，尖头金属铲、泥铲和平头铲可能是必要的。如果你住在下雪的地方，一两把雪铲肯定会派上用场。所有这些都可以很好地使用——只需检查生锈和松动的把手。

如果铲子上有轻微的锈迹，你可以把它浸泡在白醋和水的混合物中一夜，然后用钢丝刷擦洗，再涂上一些矿物油或蜂蜡。

## 工作台

如果你自己进行保养、维修或家居升级，工作台或结实耐用的桌子是一个很好的选择。根据你正在做的事情，有时候一张旧桌子也可以。如果你需要一个更大的工作空间，你也许可以找到一个二手工作台，或者，如果你有时间和技能，你可以用一张英寸的胶合板，配一个2×4的框架和支腿，来制作你自己的工作台面。如果你幸运的话，你可以在一个回收木材场以一个好价钱找到你需要的材料。

## 台钳

一把好的台钳是几乎每个人的祖父在地下室或车库里都有的东西。原因？它们是一个有用的、多功能的工具，可以帮你省去很多麻烦。这基本上就像有第三只手帮你拿东西，而你胶水，锯，砂，磨，或钻。一个质量好的老虎钳可能很贵，但你可能可以在转售应用程序上找到一个。

## 锯马

如果你打算自己做很多工作，一两匹锯子会很有用。对于绘画、设计更大的项目，当然还有锯切，这些都是我店里工作最辛苦的工具。它们也非常适合搭建临时工作台面。你可以使用锯木机；只是要小心确保它们的高度相同。在不稳定的地面上工作很困难。
