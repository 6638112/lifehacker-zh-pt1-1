# 你可能真的会从这个关于艺术作品的单词中学到一些东西

> 原文:[https://life hacker . com/you-pow-can-this-wordle-for-1849142844](https://lifehacker.com/you-might-actually-learn-something-from-this-wordle-for-1849142844)

[基于单词的单词变体](https://lifehacker.com/the-10-best-wordle-alternatives-ranked-by-difficulty-1848859009) 非常有趣。格式的每一次调整都会给游戏带来新的细微差别。但是有时候，扩展你的视野去猜测那些*不是*字母在网格上的具体位置是值得的。我今天就这么做了，我爱上了 [Artle](https://www.nga.gov/artle.html) 。

Watch

艺术品猜谜游戏 [来自国家美术馆](https://www.washingtonpost.com/arts-entertainment/2022/06/30/artle-national-gallery-wordle/) ，以他们收藏的艺术家为特色。我是艺术史爱好者吗？当然不是。但是我喜欢玩吗？非常喜欢。

在这个游戏中，你会看到一件艺术品，你的任务是猜猜是谁做的。开始键入艺术家的名字，你会得到建议，这样你就可以选择，比如说，“Polidoro da Caravaggio”而不是键入“carav ...”然后纠结于如何拼写那个人的名字。这个特征也有助于区分伦布兰特·皮尔和伦勃朗·范·瑞恩之间的区别。(关于如何玩的另一个提示是:加载页面，如果一幅画没有出现，等待几分钟。在我们的测试中，游戏最初加载非常慢，但一旦第一幅画出现，就很容易玩了。)

对于四次猜测中的每一次，你都会看到不同的作品。所以今天玩游戏的时候，我第一次看到了钢笔画。我大胆猜测了一下，结果猜错了，然后给我看了同一位艺术家的一幅版画。又错过了。然后是另一种更细致的雕刻。我已经不再轻易猜测了，现在我正在努力猜测这个人可能来自哪个世纪。不知道。又失手了。对于我最后的猜测，我得到了一些不同的东西:一幅全彩色的画，风格有点熟悉，但我还是想不起来。

我随便猜了一下，承认失败，然后答案揭晓了:这是一位我从未听说过的艺术家。

在游戏的其他版本中，这将是令人沮丧的。用一个晦涩难懂的单词猜字游戏一点也不好玩。但在这里，这是双赢的:如果你以前不认识这位艺术家，现在你认识了。

或者正如一位Artle玩家 [对《华盛顿邮报》](https://www.washingtonpost.com/arts-entertainment/2022/06/30/artle-national-gallery-wordle/) 说的，“当你意识到你马上就认识的所有艺术家都像19世纪的白人艺术家时，这确实会给你一些自我意识，也许是时候扩展你的艺术欣赏了。”

游戏的创作者表示，他们选择了知名和不太知名的艺术家，以保持游戏的趣味性和教育意义。虽然画廊的收藏绝大多数是死去的白人男性的作品，但入选Artle的作品更加多样化:据《华盛顿邮报》报道，在游戏的前45天，17.8%的艺术家是非白人，相比之下，画廊的整体收藏只有2.3%。

我不会透露今天艺人的身份，但我会说游戏的结局满足了我的好奇心。我不仅被告知这个人的名字，而且这个游戏还把我和这位艺术家的传记联系起来。(我对一个世纪的最佳猜测是100年后。还不错？)它也有标题和链接，提供了四部作品的更多信息。现在我对这位艺术家有了更多的了解，我期待着在11小时48分钟内了解另一位艺术家。
