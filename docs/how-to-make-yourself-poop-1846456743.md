# 如何让自己便便

> 原文:[https://lifehacker.com/how-to-make-yourself-poop-1846456743](https://lifehacker.com/how-to-make-yourself-poop-1846456743)

听着，有时候你没时间等。无论你是想现在便便以确保以后不会，还是你只是需要一些缓解，这里有一些技巧可以帮助你。

### 喝一杯热饮料

咖啡因其对肠道的影响而臭名昭著，但它实际上可能不是咖啡因。任何 [温热的液体](https://lifehacker.com/drink-hot-liquids-to-make-yourself-poop-1687773601) 都可能有类似的效果。所以，如果你不是一个爱喝咖啡的人，那就来一杯茶或热巧克力吧。

### 吃点东西

每次你吃东西，你的肠道都会有连锁反应。你的胃通过把它能倒的东西倒进小肠来为进来的饭腾出空间，小肠把它能倒的东西倒进大肠，以此类推。

如果你曾经训练过一个小孩上厕所或者训练过一只狗，你知道你通常会在饭后30分钟左右排便。你也可以把这些知识用在自己身上。

### 锻炼

锻炼加快了食物通过肠道的速度。消化的食物在你的肠道里停留的时间越长，它就变得越硬越干，所以更快的运动意味着当它准备离开时，它更有可能变软。

任何一种运动都可能有帮助，但像慢跑或骑自行车这样的有氧运动经常被推荐。舞蹈锻炼或高强度锻炼的弹跳动作也可能有所帮助。不幸的是，没有证据表明那种特定的瑜伽姿势，甚至那些被吹捧为有助于“消化”的姿势，能有这种效果。

### 喝水

当你充分水合时，你的肠道工作得最好，所以确保你喝了足够的水。( [)如果你不渴，你可能很好](https://lifehacker.com/do-you-really-need-to-drink-more-water-1734665155) ，但多喝一两杯也无妨。)

### 吃点纤维

在包括全谷物和蔬菜在内的植物性食物中发现的纤维，能让你的消化系统运转得更快。同时，便便会更软，更容易通过。这是因为纤维吸收水分，所以当你增加纤维摄入量时，要确保你喝了足够的水。

纤维不会让你立即便便，但如果你现在增加纤维，你会在一两天内开始看到效果。有一些纤维补充剂，像Metamucil，你可以和水混合，或者你可以只吃含纤维的食物，如[豆类，燕麦片，其他全谷物，以及大多数水果和蔬菜](https://www.mayoclinic.org/healthy-lifestyle/nutrition-and-healthy-eating/in-depth/high-fiber-foods/art-20050948) 都含有丰富的纤维。

纤维被列在营养标签上(它是一种碳水化合物)，所以开始阅读标签，目标是每天25到30克。

### 改变位置

蹲着的姿势是 [有时候](https://lifehacker.com/no-you-dont-have-to-squat-when-you-poop-1785317166) 比坐在马桶上好像坐在餐厅的椅子上更有利于便便。考虑把你的脚放在脚凳上，或者稍微移动一下(向前倾斜，向后倾斜)，找到最适合你的位置。

### 请咨询您的提供商

也有医疗解决方案，包括不同类型的泻药和大便软化剂，都可以在柜台上买到。如果你定期排便有困难——包括总是硬邦邦的，或者一周排便少于三次——你可能需要寻求医疗帮助。
