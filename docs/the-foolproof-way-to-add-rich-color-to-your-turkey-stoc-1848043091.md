# 给你的火鸡高汤添加丰富颜色的简单方法

> 原文：<https://lifehacker.com/the-foolproof-way-to-add-rich-color-to-your-turkey-stoc-1848043091>

火鸡肉汁应该非常浓郁和美味，刚好够厚到覆盖一片火鸡，并且是棕色的。颜色并不总是表明味道，但白肉上的白色肉汁有问题。我喜欢我的肉汁有对比，特别是当作为一顿备受瞩目的饭的一部分时——，没有什么能像洋葱皮一样带来棕色。



洋葱皮富含色素，以至于经常被用作天然染料。当添加到股票，他们赋予了深琥珀色，但没有太多的味道。为了看看*如何把*的颜色带到汤锅里， [库克的《插图》](https://www.cooksillustrated.com/how_tos/8737-making-stock-with-skin-on-onions) 做了不同批次的简单鸡汤:一种是用带皮洋葱做的，一种是用去皮洋葱做的。用带皮洋葱做的高汤颜色要深得多，你可以点击查看 [全文](https://www.cooksillustrated.com/how_tos/8737-making-stock-with-skin-on-onions) 。但在火鸡高汤的情况下，它将成为火鸡肉汁，我喜欢在锅里加入额外的无洋葱皮。(洋葱必须是棕褐色的——红色的洋葱皮不适合我们的目的。)

其实没什么大不了的:每次你剥洋葱的时候，把皮放在一边，然后把它们放在一个冷冻袋里，放在冰箱里，直到你准备好做菜。把它们和去皮的洋葱一起扔进锅里(如果洋葱脏了，就把它的根切掉)，然后像平常一样做高汤。将鱼皮、骨头和蔬菜一起沥干，然后用这种又黑又美的液体做又黑又美的肉汁。
