NFT 已经成为一种投资现象，从 gif 到体育集锦，任何东西都可以卖到数百万美元。但这有多少是炒作——我们应该对罗根·保罗、帕丽斯·希尔顿、林赛·罗韩、罗布·格朗沃斯基和塔可钟 T3】的名人代言感到放心吗？让我们来看看为什么 NFT 市场可能会在短期内崩溃。

### **什么是 NTFs？**

NFT(“不可替代令牌”)使用与比特币等加密货币相同的区块链技术，但有一个关键区别:它们不可互换，而且每一种都是独一无二的。如果你通过加密资产市场购买了一个 NFT，它就属于你了，可以放在你的数字钱包里，也可以在市场上出售。当一台 NFT 被出售时，分散网络上的所有计算机都在一个共享的账本上记录交易，这实际上创建了一个不能被更改或删除的真实性证明。



理论上，NFT 解决了如何通过直接销售或版税来补偿数字内容创作者的问题。当然，你可以很容易地复制这些数码产品，但部分吸引力在于拥有一些基本上由艺术家签名的东西，比如亲笔签名的棒球卡。

NFT 市场包括可收藏的交易卡、体育集锦、艺术、音乐，甚至虚拟资产，如开放世界视频游戏中的财产。最近有很多关于它们的炒作和猜测，售价高达数百万的令人惊讶的故事。例子包括一张 [Nyan Cat meme GIF 卖了近 60 万美元](https://www.businessinsider.com/ethereum-nft-meme-art-nyan-cat-sells-for-300-eth-2021-2) ，Twitter 的 CEO 杰克·多西的 [第一条推文](https://www.cnet.com/news/twitter-ceo-jack-dorsey-is-selling-his-first-tweet-ever-as-an-nft-worth-millions/) 卖了 250 万美元。

### **为什么 NFT 市场可能已经过度膨胀**

对 NFT 的最大打击之一是，这是一个人为稀缺的市场，因为你可以创建多少 NFT 没有限制——几乎任何数据都可以被认证和出售。正如莱特币的创造者、NFT 怀疑论者查理·李在最近的 Twitter 帖子 中解释的那样:

> 不同于 NFT，真实世界的艺术不是零成本的。创作一件作品需要付出努力和时间。这是有效的工作证明。像毕加索这样的著名艺术家，一生只能创作成千上万件艺术品。这种限制造成了稀缺性，有助于保持高价值。
> 
> 另一方面，NFT 制造了人为的稀缺。因为创造另一个 NFT 的成本几乎为零，市场最终将被试图从这股热潮中获利的艺术家们的非艺术作品所淹没。供应将压倒需求，价格最终将暴跌。

此外，Lee 认为，对许多人来说，NFTs 的真正吸引力在于真实性证书，从长远来看，这种证书的价值有限



> 拥有收藏品的大部分价值是通过将所有权从实际收藏品转换到其真品证书而丧失的。NBA TopShot NFTs 基本就是这样。它们是短视频时刻的数字证书，任何人都可以下载。的确，有了 NBA TopShot，你可以指着一个[【nbatopshot.com】](https://t.co/BR5Kv5BY8A?amp=1)的网址自豪地说，看，它说我拥有这一刻。不过这种感觉很快就会消失。

### 其他问题

虽然国家艺术博物馆提供所有权证明，但 NFT 的创作者不需要证明原作的所有权。NFT 市场已经在处理被盗艺术品和侵犯版权的事件。例如，据雅虎 报道，一位名为 Ashtoshi 的艺术家说，未经她的同意，一张自拍照在 Rarible 上被拍卖，价值超过 1051 美元。

正如 [Gizmodo 指出的](https://earther.gizmodo.com/how-to-fix-crypto-art-nfts-carbon-pollution-problem-1846440312) ，NTFs 建立在区块链技术的基础上，这带来了巨大的能源成本。NFT 资产的大多数市场通过以太坊进行销售，以太坊通过一个叫做 [挖掘](https://www.investopedia.com/tech/how-does-bitcoin-mining-work/) 的过程来维护交易安全的分类账，但是这个过程需要大量的能量。2018 年以太坊使用的能源比冰岛还多， [据彭博](https://www.bloomberg.com/news/articles/2021-03-09/gaming-crypto-artists-court-controversy-while-cashing-in-on-nfts) 。随着 NFT 市场的增长，对环境的影响只会变得更糟。

### 底线

在许多方面，NFT 类似于最初的加密货币产品，或 ICOs，这是一种投机性的投资时尚，尽管开局强劲，但在 2018 年失败了。与 NFTs 类似，ICO 的定价最初被大大夸大，产品被明星代言夸大了(DJ Khaled！史蒂文·西格尔！).最初的狂热没有持续多久，市场从 240 亿美元萎缩到 50 亿美元，据华尔街日报 报道。



如果你在考虑购买 NFT，考虑一下这种炒作是否可持续，只投资你能承受损失的金额。毕竟，除非你是一个创造者，否则 NFT 不会提供任何现金流，你赚钱的唯一方式是通过更多的买卖——这意味着如果市场崩溃，你可能只有吹牛的权利。
