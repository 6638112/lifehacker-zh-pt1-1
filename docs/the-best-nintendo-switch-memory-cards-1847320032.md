## 如何更换您交换机的存储卡

更换你的交换机的存储卡既具有欺骗性，又看似简单。首先，确保你的开关完全关闭。(断开坞站时，按住电源按钮三秒钟。然后选择“电源选项”和“关闭”)然后，向上翻转支架。你会在底部看到一个小槽，你可以在那里插入一张新的存储卡。

## 如何将下载的游戏移动到交换机的存储卡中

打开你的系统设置，向下滚动到“数据管理”菜单。点击“在系统/ microSD 卡之间移动数据”然后您会看到两个选项:一个是将数据和软件从存储卡移动到交换机的内部存储器，另一个是将数据和软件从交换机的内部存储器移动到存储卡。选择那个。选择你想移动的游戏，选择“移动数据”，然后*瞧*。

很遗憾，您保存的数据无法移动，但您可以*将数据*备份到云上，将数据[提供给](https://kotaku.com/cloud-saves-will-last-six-months-after-a-switch-online-1829314409)和您已经注册了任天堂 Switch 在线。该选项位于“数据管理”菜单中的“保存数据云”选项下。要上传保存，向下滚动到设置并打开“自动保存-数据备份”您可以在“所有保存数据”字段下看到所有备份的保存数据。一个小小的绿色勾号表示您的存储是“安全的”
