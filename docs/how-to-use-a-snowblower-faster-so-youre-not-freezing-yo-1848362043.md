# 如何更快地使用扫雪机，这样你就不会冻僵了

> 原文：<https://lifehacker.com/how-to-use-a-snowblower-faster-so-youre-not-freezing-yo-1848362043>

有车道的房主住在每年降雪量相当可观的地方，他们需要做出决定:是铲雪、使用扫雪机，还是雇人除雪。那些选择购买扫雪机的人通常会这样做，因为它比铲雪更快更容易。



但是如果我们告诉你，你可以做一些事情来使吹雪更快更有效呢？嗯，有。这里有一些专家建议让你更快完成工作，你可以回到温暖的室内。

## 规划你的路线

在一个完美的世界里(出于某种原因，你仍然必须使用扫雪机)，你可以一次通过你的地产上的所有人行道。据 [消费者报道家居专家埃里克·哈多](https://www.consumerreports.org/cro/news/2014/03/snow-removal-shortcuts-that-save-time-and-energy/index.htm) 称，这是可能的。

如果你的车道两边都有空地，他建议从中间开始，向车道的一边扔雪。“掉头，然后从另一边回来。保持交替， [他告诉消费者报道杂志](https://www.consumerreports.org/cro/news/2014/03/snow-removal-shortcuts-that-save-time-and-energy/index.htm) 。“这样，你就不必经常调整滑槽，任何达不到要求的雪都会在随后的行程中被清除。”

但是如果你的房子紧靠车道，哈多说从最靠近房子的一侧开始，这样你就不会把雪扔到你已经清理过的车道上。

## 一有 6 英寸厚的雪就开始

你可能认为最好等到雪停了，然后*和*拿出扫雪机，但是《家庭杂工》 的 [编辑反对这样做。相反，他们建议在下了六英寸的雪后出门。尽管这确实意味着你可能不得不不止一次地铲雪，“你的机器不需要努力工作，它会把雪扔得更远，”他们注意到。](https://www.familyhandyman.com/list/snow-blowing-tips/)

## 距离很重要

当你把雪从车道上吹下来时，不要只关注你正前方的路面，要注意雪落在哪里。根据 [编辑看家庭杂工](https://www.familyhandyman.com/list/snow-blowing-tips/) ，有四种方法可以获得最大的投掷量:

1.  吃一小口雪
2.  以全速运转鼓风机，但以较慢的行驶速度运转
3.  将滑槽分流器调节至完全升起位置
4.  无法保持一贯的事态进程

当然，不要忘记你正在和一台可能伤害你 的机器 [一起工作，所以一定要把安全放在第一位。](https://www.assh.org/handcare/safety/snow-blower)
