# 这种 3 种成分的纯素食番茄酱是我的新宠

> 原文：<https://lifehacker.com/this-3-ingredient-vegan-marinara-is-my-new-favorite-1847839126>

我不知道这是东海岸意大利的东西，卡拉布里亚的东西，还是什么，但我男朋友的意大利家庭是*不是*到黄油作为烹饪脂肪。他们是橄榄油世家，这是我在为他做意大利面晚餐时了解到的，当时我用了玛塞拉·哈赞的 [三种成分的番茄酱](https://lifehacker.com/the-best-pasta-sauce-has-three-ingredients-1820519444) 和黄油及洋葱。

他不喜欢它，因为他可以“品尝黄油”，对我来说，这是一个不喜欢某样东西的真正疯狂的理由。但是，即使我在哲学层面上不同意他不加黄油的立场，如果我打算喂他吃，我还是会用橄榄油，因为我很好，我相信我爱的人说他们不喜欢某样东西。(如果我试着偷偷把黄油放进去，他真的能尝出来——自从他戒烟后，他的味觉变得相当好了。)

总之。我担心没有黄油会让我变得不丰富，但是你可以用橄榄油达到惊人的奢侈水平——你只需要用足够多的橄榄油。

我以为我用了“大量”的橄榄油，直到周六晚上我用 food 52T3[这个天才配方做了茄子帕尔玛。这个食谱太完美了——我什么也没改。虽然我通常 futz 和“即兴”，我决定相信这个过程，我得到了很大的回报。茄子软而不湿，奶酪中间融化，上面烤得很漂亮，但简单的三种成分的酱料可能会偷走整个节目。它是*所以*好。](https://food52.com/recipes/431-eggplant-parmesan)

酱听起来太简单，不值得注意。只是橄榄油，大蒜，和两罐去皮的西红柿。然而，这是难以置信的。食谱要求“足够的橄榄油来覆盖整个锅”，结果橄榄油的用量惊人。(翻译过来就是我的酱锅里大概半杯。)然后将油注入三瓣蒜片——我多用了两瓣，因为我的蒜片很小——和两罐去皮的“San Marzano style”番茄一起炖。(我买的是便宜的杂货铺牌子。)让这一切减少一半，你就有了一个完美的，没有多余的番茄酱。

大量的脂肪和大量减少的番茄给调味汁带来了丰富的深度和特色。这是我新喜欢的酱。(比起哈赞酱我更喜欢。)香甜扑鼻，*完成了*品鉴——真的不需要再加什么了。(当然可以。我不会在你的厨房里阻止你，但我强烈建议你在发表自己的见解之前，先试一试。)要做到这一点，你需要:

*   橄榄油(大约 1/2 杯)
*   3-5 瓣大蒜，取决于你的瓣的大小
*   2 罐 28 盎司的去皮西红柿
*   大约 3 大撮盐。

把一个平底锅放在中火上，加入足够的橄榄油，完全盖住锅底。加入大蒜，煮到发出咝咝声和香味，但不要变成褐色。加入两罐西红柿和几把盐，然后用木勺或厨房剪刀将西红柿捣碎切碎。将火调至中低，煨煮——不时搅拌和刮底部——直到酱汁减少一半，当出现任何较大的番茄块时，将其捣碎。与意大利面条一起食用，夹在千层面中，或者用调味汁做一道真正美味的茄子干酪。
