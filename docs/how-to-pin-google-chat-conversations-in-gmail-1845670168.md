# 如何在Gmail中启用谷歌聊天对话

> 原文:[https://life hacker . com/how-to-pin-Google-chat-conversations-in-Gmail-1845670168](https://lifehacker.com/how-to-pin-google-chat-conversations-in-gmail-1845670168)

Google Chats正式取代了Gmail中的Hangouts。Gmail的聊天集成去年首次为谷歌工作区和企业谷歌账户推出，但现在在安卓和网络浏览器 上对普通用户 [开放。(抱歉，还不支持iOS。)](https://9to5google.com/2021/04/04/new-gmail-chat-turn-on/)

目前，Gmail的聊天功能在技术上还处于早期访问阶段，需要手动启用。下面是如何做到这一点。

**在安卓系统上:**

1.  从Google Play下载最新的Gmail应用程序更新。
2.  进入**设置>【帐户名】>常规。**
3.  选择**“聊天(提前接入)”**轻触**“尝试一下”**启用。
4.  Gmail应用程序将关闭并重新启动。如果你安装了聊天应用程序，你会得到一个简短的教程，并可以选择在Gmail中打开通知。

**桌面通web:**

1.  前往[mail.google.com/mail/u/0/#settings/chat](http://mail.google.com/mail/u/0/#settings/chat)T3】
2.  在聊天标题下，选择**“谷歌聊天(提前访问)。”**

虽然很容易对着 [翻白眼，但又一个谷歌消息](https://lifehacker.com/how-to-backup-your-allo-conversations-before-google-shu-1830922541) 应用出现了(而 [又一个谷歌应用在这个过程中被砍掉了](https://lifehacker.com/how-to-track-every-google-service-and-app-thats-shut-do-1833553690) ),谷歌聊天将能够做更多的事情，尤其是在Gmail中。例如，您可以通过Chat直接向联系人的Gmail收件箱发送消息，或者在点击URL链接之前使用谷歌的安全浏览设置扫描它们。Chat还与谷歌的其他应用程序集成，如Docs、Meet和Calendar。

### **将Google Chats固定在您的Gmail中，以便轻松联系您最喜欢的人**

另一个方便的功能是对话锁定，它可以在桌面和Gmail移动应用程序中使用，适用于常规的聊天直接消息线程和聊天的“房间”群组对话。一旦Gmail提供了聊天功能，它应该成为所有用户的一个选项。下面是怎么做的:

1.  滚动浏览Gmail的“聊天”或“房间”侧菜单，找到您想要锁定的对话。
2.  选择三点**“更多”**菜单图标，然后选择**“大头针”**您也可以选择**【取消锁定】**来移除锁定的对话。
3.  固定的聊天对话或聊天室组显示图钉图标，并保持在Gmail侧边菜单的顶部。当有未读邮件时，它们还会显示一个红点。

这个小小的黑客技术让重要的对话更容易被追踪——至少当你的Gmail应用程序最终用Google Chat取代了Hangouts。
