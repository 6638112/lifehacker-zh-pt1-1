# 什么是‘远房表亲’？让我们解释一下

> 原文：<https://lifehacker.com/second-cousins-once-removed-and-more-explained-in-1661572056>

想成为下一次家庭聚会上最受欢迎的人吗？做一个能解释“远房表亲”和“远房表亲”之间区别的人，你的亲戚会把你当成一个天才，一个先知，一个他们以血缘关系为荣的扶手椅上的家谱学家。以下是如何计算出你和你所有堂兄弟姐妹的关系有多密切。



上面这张方便的图表，的 Nathan Yau 提供的 [和](https://flowingdata.com/2014/11/05/chart-of-c)的流动数据使得确定这些关系变得非常简单。下面是对图表工作原理的解释:

> *找出两个亲戚之间的共同祖先。然后在最上面一行选择与该祖先的第一个关系，并继续下去，直到与其他家庭成员匹配。结果是第一个和第二个是如何关联的。*

或者，如果你喜欢，这里有一个方便的表亲术语表。

### 堂/表兄弟姐妹

你的表亲是和你有共同祖先的人，你们最近的共同祖先至少是两代人之外的。这将他们与其他人区分开来，比如你的兄弟姐妹，他们是与你有着共同祖先的人，而这些共同的祖先是你的父母。

### 堂兄妹

你的堂兄妹是你叔叔和婶婶的孩子。堂兄妹共有一个祖父母(也就是说，你的祖父母是你和你的堂兄妹的两代人)。

### 第二代表亲

第二代表亲有共同的曾祖父母，并且是第一代表亲的子女。所以你爸爸的第一个堂兄弟的孩子是你的第二个堂兄弟。

### 第三代表亲

第三代表亲有一个曾曾祖父母，并且是第二代表亲的孩子。你爸爸第二个堂兄弟的孩子是你的第三个堂兄弟。你感觉到这里有一种模式吗？第四代堂兄弟共享一个曾曾祖父母等等。

### 一旦移除

如果你是表亲的“远亲”，那意味着你们来自不同的世代。如果你是“远房亲戚”，那意味着你们相隔一代。你爸爸的堂兄妹就是你的堂兄妹。你堂姐的女儿也是你堂姐，隔了一次。

### 两次移除

如果你是表亲的“第二代”，那你就是隔了两代人。你祖母的堂兄妹是你的堂兄妹，隔了两辈。你堂姐的孙女也是你堂姐，隔了两次。
