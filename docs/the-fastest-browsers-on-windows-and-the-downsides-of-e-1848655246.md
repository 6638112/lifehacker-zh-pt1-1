# Windows 上最快的浏览器(以及每种浏览器的缺点)

> 原文：<https://lifehacker.com/the-fastest-browsers-on-windows-and-the-downsides-of-e-1848655246>

简陋的浏览器仍然是许多电脑上最常用的应用程序，它为你节省的每一秒都是无价的。缓慢落后的浏览器的日子已经一去不复返了，大多数 web 浏览器都提供了令人满意的性能，但有些浏览器(显然)比其他浏览器快得多。我们着手研究哪种浏览器对你的 Windows 电脑来说是最快的，结果有点令人惊讶。

## 我们如何测试最快的浏览器

在我们分享结果之前，先来看看我们是如何测试这些浏览器的速度的。我们使用了网络基准测试工具[JetStream](https://browserbench.org/JetStream/)和 [MotionMark](https://browserbench.org/MotionMark1.2/) 来确定哪些浏览器是最快的。速度计测量您的浏览器在运行 web 应用程序时的响应速度，JetStream 是一个专注于 JavaScript 的基准测试，它为能够快速启动和执行代码的浏览器分配较高的分数。MotionMark 是一个图形基准，测试浏览器渲染复杂网页的能力。

我们在最终结果中给了 Speedometer 稍高的权重，因为它模拟了现实世界中与网站的交互，并包括像将项目添加到待办事项列表中所花费的时间这样的测试。这很可能会影响您对浏览器的日常使用。

我们在浏览器的一个新的匿名窗口中运行了所有的基准测试，并在此之前禁用了所有的扩展。在我们运行测试之前，所有的浏览器都更新到了最新的可用版本。我们还在每个浏览器上运行了两次基准测试，以确保结果一致，没有异常高或低的分数。运行两次测试后，所有浏览器都返回了几乎相同的基准分数。

为了进行这些测试，我们使用了一台装有 Windows 10 Pro 的 Zotac 法师 EN 1070 电脑。这款电脑拥有 8GB 内存、英特尔酷睿 i5 6400T 处理器和 120GB 固态硬盘。

浏览器基准测试结果也取决于硬件，因此如果在您的计算机上运行相同的测试，您会看到不同的分数，但结果应该与我们的相似。

## 谷歌浏览器:大多数人最快的浏览器

Chrome 的速度一直很快，在我们的测试中，它以 107.2 的分数高居速度表基准测试的榜首。它在 JetStream 测试中以 99.778 的成绩获得第二名，这也令人印象深刻。它在 MotionMark 测试中以 449.23 的成绩名列第三，这意味着它在渲染复杂网页时落后于其他浏览器。

Chrome 在大多数浏览器测试中得分很高，也让你可以访问一个高度成熟的浏览器扩展生态系统，为 Chrome 添加许多有用的功能。另一方面，Chrome 归谷歌所有，谷歌的商业模式是跟踪用户并为他们提供有针对性的广告。如果你关心你的隐私，你应该看看其他的浏览器，并且立即改变一些 Chrome 的设置，比如 T2 和 T3。

## 勇敢:内置广告拦截的快速浏览器

Brave 在 Windows 最快浏览器的测试中以微弱优势获得第二名。它在速度计测试中以 106 分获得亚军，甚至在 JetStream 测试中以 100.002 分击败 Chrome，在 MotionMark 测试中以 453.29 分击败 Chrome。

虽然 Brave 声称是 Chrome 的隐私友好替代产品，但它也有争议的。这是一个内置广告拦截的快速浏览器，仅此一点就使其成为 Chrome 的良好替代品，但它的争议使其很难毫无疑问地相信其所有的隐私声明。

## Microsoft Edge:最适合繁重的工作负载

Edge 改用 Chromium 让它变得非常快。该浏览器在我们的两次测试中获得第三名，在速度计中获得 99.5 分，在 JetStream 中获得 99.071 分。然而，它在 MotionMark 测试中以 479.30 的优异成绩名列榜首。

测试表明，Edge 是图形密集型网站和较重的网络工作负载的最佳选择。虽然 Edge 越来越好，但它确实有一个非常复杂的上下文菜单(当你在浏览器中右键单击时就会看到)，这使得很难快速找到你需要的选项。如果你首先考虑的是隐私，这也不是你想要的浏览器。

## Opera:不是最快的，但是有额外的特性

Opera 在我们所有的测试中名列第四。它在速度计上的得分为 90.1，在 JetStream 上的得分为 95.914，在 MotionMark 上的得分为 415.26。这些成绩并不差，但与世界上最快的浏览器相比还是有一点差距。

就像目前提到的所有其他浏览器一样，Opera 也是基于 Chromium 的。它还有其他有用的功能，例如与 Facebook Messenger、WhatsApp 以及 Spotify 和 Apple Music 等音乐流媒体服务的集成。这些功能可能使它比其他浏览器更值得使用，但要注意 Opera 包含一些你应该考虑禁用的赞助内容，而且它也不是最隐私友好的浏览器。

## 火狐:慢，但更隐私

不幸的是，Firefox 在我们所有的测试中垫底。它在速度计上的得分仅为 77.48，在 JetStream 上的得分为 62.475，在 MotionMark 上的得分为 222.78。Firefox 是所有这些测试中唯一的非 Chromium 浏览器，这可以解释一些结果。

而火狐，通过 [几个设置调整](https://lifehacker.com/11-default-firefox-settings-you-should-change-if-you-ca-1848626750) ，就可以成为一个安全私密的浏览器。这一点，以及其出色的附加生态系统，使它脱颖而出。

## 关于 Safari 的一句话

由于 Safari 是一款 Mac 专属浏览器，因此将其包含在此次浏览器枪战中没有意义。我们暂时使用了一台 M1 Mac Mini，并在那台机器上的 Safari 上运行了这些基准测试。由于搭载苹果 M1 处理器的 Mac Mini 是一款运行在完全不同的操作系统上的更强大的机器，因此分数无法与我们的测试结果进行比较。

然而，Safari 最终在这些基准测试中得分很高。它在 MotionMark 上的得分为 1658.44，在速度计上的得分为 256，在 JetStream 上的得分为 193.778。如果你有一台 Mac，Safari 在速度、功能和能效之间找到了最佳平衡点。
