# 一些光环无限挑战交换和 XP 提升被品牌交易所挟持

> 原文：<https://kotaku.com/some-halo-infinite-challenge-swaps-and-xp-boosts-are-be-1848150692>

昨晚，我买了一罐品客薯片，主要是因为它可以让我在*光环无限*中提升经验值，却发现我实际上无法兑换代码。你看，我得把我收据的照片上传到公司网站上。这个过程在一定程度上是有意义的(你会想阻止人们在熟食店走动，扫描他们没有付钱的零食的二维码),但拜托，现在谁还留着收据呢？



我最终没有得到我的 XP boost(悲伤的长号),但我确实得到了一个前排座位，参加了 *Halo Infinite* 的奇怪和膨胀的交叉推广品牌活动。就像其他主*光环*游戏的发布一样，你现在可以在一系列小吃、能量饮料和快餐连锁店上看到军士长的不露面。

这些促销大部分都有明确的卖点。《光环无限》的多人游戏部分已经在 Xbox 和 PC 上发布了大约三周，主要是由 100 级战斗通行证驱动的。你每赢得 1，1,000XP，主要是通过击败每日和每周的挑战，你就会提升一级。当你升级时，你为你的*光环*化身解锁各种化妆选项。

随着 [一些关于*【光环无限】*的缓慢进展的声音反弹](https://kotaku.com/halo-infinites-battle-pass-sucks-fans-say-1848065594) ，开发者 343 工业 [调整了系统](https://kotaku.com/halo-infinites-battle-pass-just-got-way-better-1848136540) 所以你仅仅因为看到比赛结束就可以获得经验值。但是最快的进步方式仍然是完成你每周分配的 20 个挑战中的任何一个——比如“在 PvP 中获得五次双杀”或者“赢得三场夺旗比赛”

你可以在 *Halo Infinite* 的微交易商店以 2 美元的价格购买 XP 提升和挑战兑换(两次提升，两次兑换)。你也可以通过战斗通行证在常规的剪辑中获得这两者。虽然战斗通行证在你进行的过程中提供了一些交换，但在大多数情况下，你必须实际上*购买*这些物品，以使你的挑战更容易管理。不过，游戏通行证的持有者确实会得到一点折扣。

如果我保留了收据，那罐品客薯片会送我一个一次性用品，叫做 XP boost。通过激活它，我一个小时赚的经验值会翻倍。与此同时，在快速休闲连锁店 Chipotle 的数字订单上输入促销代码“ [Halo117](https://chipotle.com/halo) ”，理论上我可以获得五次挑战互换。这将让我放弃任何我觉得有点太*有挑战性的每周挑战。(挑战互换对顶点挑战无效，顶点挑战是每周最后一次挑战。)* 

能量饮料 Rockstar 提供了最诱人的(不要与去年的 [活动混淆，赢得一台 Xbox 然后去法国](https://www.monsterenergy.com/promotions/monster-energy-x-halo-infinite) 【与怪物能量合作】)。每一个*光环*品牌的 Rockstar 罐的标签下都有一个代码。兑换该代码可让您获得 Xbox Series X 的日常赠品。您输入的每一个代码都会让您获得一个 XP boost plus，包括挑战交换、武器和车辆的油漆工作以及多人游戏横幅的定制选项。然而，参加比赛不会让 Rockstar 尝起来像辐照酸橙汁以外的任何东西。

整个事情是一个奇怪的练习，管理游戏中来自挑战或不切实际的挑战的烦恼。此外，可供争夺的奖品并不值得大书特书。你赚不到甜美、稀有的化妆品或未加工的 XP。你仅仅是获得了获得更多经验的机会。挑战互换甚至不能保证奖金。如果你背负着一个看起来不可能的任务(“在 PvP 中炸掉三只疣猪”)，你可以使用一个互换，同样很容易得到一个实际上不可能的任务(“在 PvP 中炸掉三只黄蜂”)。只有当一个人处于活跃状态时，你才能真正从 XP 提升中获益。

但这是动物的本性——你会付出几块钱，希望能让你的地位有些许提高。品客薯片的味道？烤墨西哥胡椒。是的，尽管更好的盐和醋品种就在那里。一场有效的促销活动会让我们中的任何一个人都成为傻瓜。
