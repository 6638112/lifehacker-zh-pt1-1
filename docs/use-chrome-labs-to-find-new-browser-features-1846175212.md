# 使用“Chrome 实验室”寻找新的浏览器功能

> 原文：<https://lifehacker.com/use-chrome-labs-to-find-new-browser-features-1846175212>

我喜欢 Chrome 新版本发布的时候，因为这让我有机会在浏览器全面的`chrome://flags/`屏幕上闲逛，挖掘隐藏的功能。但是在这些标志中翻找需要一段时间，因为浏览器不会告诉你什么是新的(或者现在缺少的)。除非你使用的是 Chrome Canary，也就是说，在你的工具栏上有一个隐藏的标志启用了一个新的“Chrome Labs”功能，突出了谷歌测试的一些浏览器功能。



在我进入之前，有一些警告适用。首先，Chrome Labs 功能只存在于 Chrome Canary 上(目前)，这意味着你必须运行谷歌浏览器的测试版才能看到它。如果潜在的不稳定性对你来说不是一个大问题——或者如果你的浏览器变得奇怪，你可以切换到 Edge Chromium 这样的备份——试一试。

第二，Chrome Labs 功能不会突出显示金丝雀包含的每一个潜在功能变化。你还可以选择启用更多的标志来获得预发布功能，但如果你只是想尝试一些高级选项，而不是筛选所有选项，Chrome Labs 就是你的选择。

你可以下载并安装 [**Chrome 金丝雀**](https://www.google.com/chrome/canary/) (它与你的普通 Chrome 浏览器并行运行，而不是取代它)开始使用。启动浏览器，在地址栏中键入`chrome://flags/`，然后点击回车。搜索 Chrome Labs 并启用标志:

现在你会看到这个图标出现在你的工具栏菜单上。单击它，您将看到一个下拉菜单，其中包含一些您可以测试的功能:

启用或禁用任何你想修改的东西，但是注意“默认”并不总是意味着“关闭”例如，标签搜索选项现在是 Chrome 金丝雀的一部分，在工具栏上方显示为一个小箭头:

启用它不会做任何事情，但禁用它将(显然)从你的浏览器中删除图标和功能。

虽然“Chrome Labs”选项菜单并不像我希望的那样全面，但我希望这是一个迹象，表明谷歌 Chrome 的稳定版本最终会有更广泛的版本。我并不反对 flags，但是如果能有一种快速简单的方法来尝试新特性，而不需要每隔几个月就去寻找它们，那就太好了。压倒我们，谷歌。用选项填充下拉菜单，让人们挑选他们想玩的东西。
