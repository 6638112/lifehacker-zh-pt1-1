# 如何写出完美的私人信件

> 原文：<https://lifehacker.com/how-to-write-the-perfect-personal-letter-1848363540>

说写信是一门“失传的艺术”已经是老生常谈了，但是试着想想你最后一次坐下来写私人信件是什么时候。与商业或网络无关——一封写给你生活中另一个人的真实信件，目的是交流信息、消息、和查看。对我们大多数人来说，这是相当长的一段时间。



如果你已经决定，这漫长的冬天是提笔写下你的信件的理想时间，你可能不知道从哪里开始，甚至不知道该说些什么。这里有一些专家的建议，让你走上正轨。

## 私人信件的格式

首先，你必须决定如何写这封信:手写，还是电子方式(打字或使用语音听写)。还有一台打字机，但我们今天将跳过那台。

媒介完全由你决定；只是要意识到手写一封信(或任何东西)比你记得的要难得多。不仅没有办法删除你写的东西(除了涂掉、划掉部分或者干脆把整个东西废弃)，而且你也没有办法编辑或者移动文本而不使其变得混乱或者难以辨认。

就其结构而言，艾米莉·波斯特研究所有一个有用的帖子，将个人信件的分为三个基本部分:开头(地址和日期)、正文和信尾。但是，还是要由你来决定你想让你的信有多正式。

## 如何写私人信件的正文

一旦你写好了问候语(例如“亲爱的人”或你选择的任何开头)，就该处理棘手的部分了:身体。根据艾米莉·波斯特研究所 的说法，私人信件的正文是“分享消息和信息，混合好消息和坏消息，回应前一封信中提出的问题或分享的消息，以及询问收件人”的地方

在 2021 年 8 月对[【霍尔马克贺卡](https://writtenbycourtney.com/about) 的资深作家考特尼·泰勒分享了一些写私人信件正文的小技巧，包括以下三条:

### 脆弱而好奇

带着脆弱和好奇。具体来说， [泰勒说](https://www.npr.org/2021/08/09/1026185212/letter-writing-hallmark-personal-tips) 你应该专注于你要告诉某人的故事，不仅要告诉他们你生活中某个特定事件或情况的事实，还要让读者了解他们给你的感受。然后，问这个人一些开放式的问题，以了解更多关于他们自己的经历，并表达对他们生活的兴趣。

### 包括“普遍性细节”

根据泰勒的说法，在私人信件中包含 [【普遍细节】](https://www.npr.org/2021/08/09/1026185212/letter-writing-hallmark-personal-tips) 意味着写下大多数人都能理解的场景，但提供关于你自己经历的具体细节。例如，如果你最近去某个地方旅行，在路上遇到了问题和烦恼，你可以描述你的遭遇，知道他们可能也经历过类似的事情。

### 要真实

在写求职信或其他类型的与工作相关的交流时，展现最好的自己是有意义的。但是泰勒说写私人信件是展示更多你自己(象征性的)T2 的机会。 [她告诉 NPR](https://www.npr.org/2021/08/09/1026185212/letter-writing-hallmark-personal-tips) “即使是专业的，我想说你还是要对自己和你写的一切保持真实，不管它是什么形式，也不管观众是谁，如果目的是让他们了解你的话。
