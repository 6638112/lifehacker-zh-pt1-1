# 如何停止Zoombombing

> 原文:[https://lifehacker.com/how-to-stop-a-zoombombing-1845700843](https://lifehacker.com/how-to-stop-a-zoombombing-1845700843)

可怕的“”缩放组合对所有缩放会议来说都是一种风险，即使是有密码和 [数据加密](https://lifehacker.com/how-to-turn-on-zooms-new-end-to-end-encryption-1845499757) 保护的私人房间。你所需要做的就是让邀请函落入坏人之手，然后突然有人在你的工作会议上大喊大叫——或者更糟。

Watch

虽然Zoom一直有一些旨在防止会议受到恶意攻击的 [*隐私和报告工具*](https://lifehacker.com/how-to-make-your-video-chats-more-secure-1843242780)，但该公司正在 [增加新的反Zoombombing功能](https://blog.zoom.us/new-ways-to-combat-zoom-meeting-disruptions/) ，让主持人暂停所有会议活动，并移除捣乱的客人。新的安全选项现在面向Mac、PC、Linux、iOS和Android上的免费和付费用户推出。

当你在开会时，你会在安全“盾”图标下找到新选项，但前提是你是主持人或共同主持人。第一个——Zoom相当于核武器——让你立即关闭所有通话活动，包括视频、音频、文本聊天、注释、屏幕共享和通话记录。它还会禁用所有临时会议室。

1.  单击/点击呼叫菜单中的安全图标
2.  选择**“暂停参与者活动”**
3.  主持人将被询问是否要向Zoom的安全团队报告参与者。勾选“报告”旁边的框，如果是，然后点击**“暂停”**
4.  如果您选择报告会议，请遵循屏幕上的说明，然后选择**“提交”**报告的用户将从会议中删除。
5.  当您准备好继续时，单击通话菜单中的安全图标，并重新检查您想要重新打开的每个功能。

主持人也可以随时报告用户，方法是选择通话屏幕中的安全图标、，然后点击 **“报告**，并按照屏幕上的说明进行操作。点击**“提交”**发送报告，并禁止被报告的用户参加会议。

新的暂停和报告工具使处理破坏性参与者变得更容易，但Zoom还有一个锦囊妙计。如果你的会议邀请出现在任何公共社交媒体帖子或其他通常发布这些内容的网站上，该公司也会警告你。

如果您的会议已被共享，自动系统会通过电子邮件通知其主持人，并提供建议的响应方式，包括删除和重新创建会议、将会议切换为私人会议或更改会议密码。

您不需要做任何事情来受益于这些通知-如果您受到影响，它们会自动发生-但您应该在使用Zoom查看任何最后一分钟的警报时监控您的电子邮件收件箱。

还有，确保你 [使用Zoom新的端到端加密设置](https://lifehacker.com/how-to-turn-on-zooms-new-end-to-end-encryption-1845499757) ，这样你聊天的数据是安全的，不会被不请自来的外人访问。
