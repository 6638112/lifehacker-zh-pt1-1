# 硕士学位还值这个费用吗？

> 原文：<https://lifehacker.com/is-a-masters-degree-still-worth-the-cost-1845634996>

在经济低迷时期重返校园攻读硕士学位是有意义的——如果你已经失业了，为什么不提升自己的技能呢？嗯，现在也有不好的一面:远程学习可能无法实现和研究生学位仍然 [昂贵](https://www.cnbc.com/2020/06/12/how-student-debt-became-a-1point6-trillion-crisis.html) 。在你注册 l 之前，你总会想做一下数学计算。

## **获得硕士学位的原因**

当然，追求更多的教育不仅仅是为了赚更多的钱，但这也不意味着你应该忽视潜在的经济利益。一般来说，拥有硕士学位会赚更多的钱，平均周薪为 1，434 美元，相比之下，学士学位的周薪为 1，198 美元，根据美国劳动局 的数据， [。](https://www.bls.gov/careeroutlook/2019/data-on-display/education_pays.htm) 

#### 硕士学位会提高你目前的薪水

一些工作会自动给获得研究生学位的员工加薪(教师就是一个例子)，这使他们成为更安全的投资。但是所有这些都有一个陷阱，因为不是所有的学位都保证有经济回报。劳动力市场对美术硕士学位的需求减少，例如与学士学位相比，平均而言，美术硕士学位只能给 [带来 10，000 美元的中等工资](https://www.bls.gov/careeroutlook/2015/article/should-i-get-a-masters-degree.htm#Other) 。

相比之下，拥有商业或 STEM 相关领域的硕士学位，你会找到更多的就业机会和更高的薪水。举个例子，如果你在金融服务业工作，一个 [MBA 学位可以让你的收入中值增加 8 万美元](https://www.bls.gov/careeroutlook/2015/article/should-i-get-a-masters-degree.htm#Business) ，比学士学位高出 89%的工资。要了解不同工作类型的期望薪资，请点击 [查看下面的 A-Z 平衡表](https://www.thebalancecareers.com/salaries-for-jobs-a-z-list-2063402) 。

#### **你选择的领域需要它**

在许多领域，研究生学位是新的学士学位。一些职业，如教学、数学、和城市规划，可能需要硕士学位才能得到一份入门级的工作(使用本 [职业展望手册](https://www.bls.gov/ooh/occupation-finder.htm) 查看你的领域，该手册根据最低教育要求对职业进行了分类)。当你的职业生涯达到顶峰时，获得研究生学位可能是明智之举，尤其是如果这对晋升到高级职位至关重要的话。

#### **你有能力支付它**

你能避免的债务越多，一个研究生学位就越值得。毕竟，教育是一项投资，但投资仍然有你想最小化的风险。考虑通过你的雇主的学费资助项目，比如 401(k)匹配项目。你还应该申请奖学金和助学金——任何有助于抵消学费的东西。你也可以在保住工作的同时兼职学习。如果可能的话，也可以考虑减少开支，比如在你获得学位的时候和父母住在一起。

## **拿不到硕士学位的理由**

#### **主要是在线**

在疫情，学校通过远程学习提供课程，但费用几乎相同。你错过了在实验室或学生团体中建立关系网和实践学习的机会。事实上，许多学生抱怨没有社交化的教育体验的质量(T2，T3，T4)。

#### **你还没有进入职场**

如果你开始一份新的职业，经验在早期可能更重要。[2018 年的一项调查](https://chronicle-assets.s3.amazonaws.com/5/items/biz/pdf/Employers%20Survey.pdf) 显示，在招聘时，雇主仍然认为实习、大学期间的就业和志愿者经历比 GPA 或相关课程更重要。人们也经常转行，所以一旦你在你的领域获得了一些现实世界的经验，并决定不喜欢它，硕士学位可能就是一种浪费。仔细想想你所在领域的障碍:是经验问题还是教育问题？

#### **你想要的领域不需要更高的学位**

你的职业需要通过研究生学位获得的专业知识吗？新闻业是进入门槛相对较低的职业的一个很好的例子:如果你能在你的笔记本电脑上建立自己的职业生涯，那么哥伦比亚大学 77000 美元的新闻硕士学位对你来说值得吗？

#### **更高的工作安置率和薪水并不能保证**

如果你进入了一个就业前景黯淡的行业，你需要仔细考虑你的学位目标是什么。你所在领域的短期和长期前景是什么？你对现在的就业率有概念吗？这个 [职业就业率排名](https://mooresvilletribune.com/business/investment/personal-finance/college-majors-with-the-highest-and-lowest-unemployment-rates/collection_caab8a7b-980b-5dd6-a62a-df05d4037684.html#1) 是一个很好的起点——让就业率成为你决策的一部分。

## **做数学运算**

每所学校都被要求提供一个估计的“出席费用，”，这应该是你预算的起点。从那时起，你需要知道你自己能负担多少费用，以及你需要从学生贷款中借多少。最后，你将从你的预期收入中减去总的债务偿还额(包括利息),看看是否值得投资。关于如何确定你的学位的投资回报的更多细节，Earnest 网站有一个很好的细分 [这里](https://www.earnest.com/blog/what-is-the-roi-of-a-graduate-degree/) 。

还是那句话，并不是每个人都想通过接受高等教育来赚更多的钱，但是如果你要承担大量的学生债务或者失去你目前的收入，你就需要知道你学位的成本，并且有一个支付它的计划。
