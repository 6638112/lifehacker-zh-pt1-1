# 用老姜黄制作一种亮黄色的织物染料

> 原文:[https://life hacker . com/use-old-姜黄-to-make-a-brilliant-yellow-fabric-dye-1846336290](https://lifehacker.com/use-old-turmeric-to-make-a-brilliant-yellow-fabric-dye-1846336290)

我的姐妹们有我没有的天赋。悉尼(又名“希德”)是一名漫画历史学家(这听起来很假，但显然是一份非常严肃的工作)，梅雷迪思(又名“迪思”)是一名视觉艺术家，他制作令人不安的拟人化 [动物](https://www.instagram.com/p/BzO1ciwDde9/) 版画。她也很狡猾，擅长用我不擅长的方式装饰。例如，我从来没有想过用姜黄作为织物染料，即使我每次做饭都用它来染皮肤。

Watch [https://lifehacker.com/embed/inset/iframe?id=instagram-CKnv2-uBoqc&autosize=1](https://lifehacker.com/embed/inset/iframe?id=instagram-CKnv2-uBoqc&autosize=1) 

当然，这正是狄恩所做的。她拿了一些“肮脏的旧床单”，将大约四分之一罐姜黄与热水混合，让床单在混合物中浸泡一夜，然后在第二天早上清洗并晾干。从照片上可以看出，这很有效。

我不建议用新鲜、芳香的姜黄来做这个，但对于已经失去香味和味道的老姜黄来说，这是一个极好的用途。网上到处都是这样使用姜黄的说明: [Planthunter](https://theplanthunter.com.au/howto/turmeric-dyeing/) 推荐的比例是大约3汤匙姜黄粉对4升水，外加一杯醋(帮助它凝固)，而 [这家布店](https://www.onlinefabricstore.com/makersmill/how-to-dye-fabric-natural-dyeing-with-turmeric-and-cabbage/) 推荐3/4杯姜黄粉加12杯水和几茶匙醋。你能信任谁？)

最后，衡量似乎并不那么重要。狄斯没有测量或寻求醋酸的帮助，结果床单非常棒。(我可能有偏见，但我觉得小姐姐做得很好！)利用你所拥有的，这就是我要说的。如果你有少量无味、无味的姜黄，将一小块织物染成明亮的色调，或者将大块织物染成浅黄色。

如果你曾经和姜黄一起工作过，你会知道它会给它接触到的所有东西上色，所以使用一个你不介意染色的容器，或者是一个可以盛放食物的容器，比如一个大锅。为了容易清理，在浴缸中尝试这种方法可能很诱人，但这样你就会有一个黄色的浴缸(至少在一段时间内)。此外，要知道姜黄不是一种永久的染料。“随着时间的推移，它会逐渐淡化，”迪思解释道。"但它褪色得非常漂亮，你可以随时刷新它."
