## iPhone的新Chrome 103功能

一个令人印象深刻的更新是Chrome的“语言识别模式”:该功能使用机器学习来分析一个网站，不仅可以确定它使用什么语言，还可以确定你是否需要将其翻译成你理解的语言。这种机器学习发生在设备上，这意味着没有任何翻译发生在谷歌或第三方服务器上。这种机器学习的方法更加安全，让你可以更加隐私和安全地翻译网络上的文章。

屏幕角落的三点菜单有了新的外观:你会看到丰富多彩的选项，如密码、下载、最近的标签和设置都在这里。你会在部分展开的菜单中找到其他选项，如重新加载、新标签和新匿名标签，而展开窗口会显示其他选项，如书签、添加到阅读列表和在页面中查找。

Chrome将允许你为某些网站设置摄像头和麦克风权限，而不仅仅是Chrome本身。这将允许您为受信任的站点启用您的摄像头和麦克风，同时阻止其他人使用这些工具。此外，重复的新标签页将被自动删除，现在您可以在Chrome中将iCal文件添加到日历中。

你还可以期待在iOS上引入增强的安全浏览功能:这项功能可以与谷歌安全浏览进行通信，分享你在浏览时遇到的任何可疑网址和下载。然而，这一设置并非对所有用户都可用:当你这样做时，你会在**设置>谷歌服务**中找到它。你还应该看到“最近的标签”在更新时作为一个选项出现，同样也在向安卓发展。

当然，与许多应用程序更新一样，Chrome 103也包括稳定性和性能改进。

## iPhone如何更新到Chrome 103

你可以像在iOS上更新应用程序一样，将iPhone上的Chrome应用程序更新到版本103:进入App Store，点击右上角的个人资料图标，然后下拉页面进行刷新并检查更新。如果你在这里看到Chrome，点击“更新”如果你看到Chrome，但它旁边的按钮显示“打开”，则你的应用程序已经更新。
