# 如果你因为没有被解雇，你还能申请失业救济金吗？

> 原文：<https://lifehacker.com/if-you-get-fired-for-not-being-vaxxed-can-you-still-cl-1847458552>

随着 Delta COVID 变体的传播，越来越多的雇主要求员工接种疫苗。正如我们在一些引人注目的例子中所看到的，不合规导致员工要么 [被解雇](https://www.bbc.com/news/world-us-canada-58112125) 要么 [辞职](https://www.espn.com/nfl/story/_/id/31869652/rick-dennison-minnesota-vikings-assistant-refusing-covid-19-vaccine-sources-say) 以逃避指责。但这是否意味着他们仍然有权享受失业救济？简短的回答是否定的，因为法律在疫苗方面对雇主有利。以下是你需要知道的。



## **拒绝接种疫苗是终止接种的理由**

[据《今日美国》](https://www.usatoday.com/story/news/factcheck/2021/08/09/fact-check-workers-fired-over-vaccination-can-denied-unemployment/5518818001/) 报道，美国司法部 [最近的一份法律意见书](https://www.justice.gov/olc/file/1415446/download) 确认，私人公司拥有授权 COVID 疫苗的合法权利，前提是它们不违反现有的州和联邦法律，你可以在这里查看。这意味着如果你公司的政策现在要求接种疫苗，而你没有接种，你可能会被解雇，并且没有资格领取失业救济金。如果你仅仅因为不想接种疫苗而辞职，你也没有资格。

## 你需要一个好的理由来收集失业数据

一般来说，要领取失业救济金，你必须不是因为自己的过错而失业，比如公司的一轮裁员。如果你因不当行为被解雇，你就没有资格申请失业，这可能包括违反公司新的疫苗政策。(这里值得一提的是 [在某些州](https://www.dol.gov/general/topic/unemployment-insurance) 如果你是因为不太严重的原因被解雇的，比如仅仅是因为工作表现不好，你仍然可以领取失业救济金。 )

唯一的例外是所谓的“正当理由”这种情况下，你基本上是以正当的理由逆向解雇你的雇主，比如不安全的工作条件、缺少报酬或歧视。在许多国家，“正当理由”可以包括基于宗教或残疾的歧视。在这种情况下，如果你的雇主在你辞职或被解雇之前没有提供合理的住宿条件(比如在家工作或其他社交距离措施)，就可以提出索赔。

然而，这些便利条件也不会给雇主造成“过度的困难”。在这种情况下，雇主可能会辩称，拒绝接种疫苗会妨碍你有效地完成工作，你会给其他员工带来健康风险。此外，您需要提供文件证明您因合法的医学或宗教原因不能接种疫苗，并且您的雇主侵犯了您的权利。从法律角度来看，这将是一场艰苦的战斗。

正如就业律师约书亚·范·坎普 [在雅虎！采访](https://www.yahoo.com/now/refuse-vaccinated-could-barred-lawyers-164123490.html) ，不要指望找到愿意为疫苗要求起诉雇主的律师:“我还没有遇到一个有效的宗教反对接种疫苗。我认为，人们越来越惊讶地发现，他们没有太多的法律选择。”

## 结果

如果你违反了公司的强制疫苗接种政策，就不要指望失业救济，因为这将被视为终止雇佣关系的理由。从技术上讲，你可以有资格获得医疗或宗教豁免，但这很难证明。由于这些政策正在成为规范，你最好的选择就是简单地接种疫苗。
