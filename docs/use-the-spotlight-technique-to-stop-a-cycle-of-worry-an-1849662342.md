# 使用“聚光灯技术”来停止担忧和焦虑的循环

> 原文：<https://lifehacker.com/use-the-spotlight-technique-to-stop-a-cycle-of-worry-an-1849662342>

作为人类，当我们不知道事情会如何发展时，我们担心是正常的。有时候，这种担心表现为一种普遍的担忧，你会定期思考这种担忧，同时仍然能够专注于其他事情。

其他时候(对于患有焦虑症的人来说)，这种担心是完全消耗精力的——让你无法专注于任何其他事情，我们不是在谈论紧急的生死场景，而是那些可能会有压力，但不涉及即时危险风险的情况。

为了帮助他的客户处理这种类型的担心和焦虑，一位临床心理学家开发了一种他称之为 [【聚光灯技术】](https://www.psychologytoday.com/us/blog/understanding-the-anxious-mind/202210/how-stop-worrying-the-spotlight-technique) 的方法，当你思绪万千时，这种方法会很有用。以下是需要了解的内容。

## 我们为什么担心？

担忧是害怕不确定和未知的结果。根据陶森大学心理学副教授 [桑德拉·勒拉博士](https://www.towson.edu/news/2020/tu-psychology-covid-worrying.html#:~:text=%E2%80%9CWhile%20many%20of%20us%20are,emotionally%20prepared%20for%20negative%20outcomes.) 的说法，即使大多数人都知道担心某些事情会让他们焦虑不安，我们仍然无法阻止自己走上那条心理道路。

一般来说，这有两个原因， [Llera 说](https://www.towson.edu/news/2020/tu-psychology-covid-worrying.html#:~:text=%E2%80%9CWhile%20many%20of%20us%20are,emotionally%20prepared%20for%20negative%20outcomes.) 。第一是为潜在的负面结果做好心理准备。第二，我们常常把担忧视为解决问题的一种方式，一种控制局面的方式，而不是让局面变得更糟。

## 如何使用“聚光灯技术”

当我们担心时——认为我们正在积极地试图解决我们的问题——我们真正在做的是试图回答关于未知或不确定的问题，迈克尔·斯坦心理学博士，注册临床心理学家和 [焦虑解决方案](https://www.effectivetherapysolutions.com/) 的创始人和所有者，在《今日心理学 最近的一篇文章中写道。

不幸的是，这可能会成为一个恶性循环，因为这些问题都不能肯定地回答。这就是为什么斯坦鼓励他的客户不要回答这些问题，而要把他们的注意力集中在别的事情上——他称之为“聚光灯技术”

[聚光灯技术](https://www.psychologytoday.com/us/blog/understanding-the-anxious-mind/202210/how-stop-worrying-the-spotlight-technique) 很简单:在黑暗的剧院里，想象自己在聚光灯的控制下。舞台是你的思想，聚光灯是你的注意力，最终，它指向哪里取决于你。

舞台上挤满了不同的问题、担忧和想法——所有你担心的事情。但是无论你在那个时刻正在做什么也在上面:完成一个工作项目，在回家的路上听播客，洗碗等等。

根据 Stein 的说法，当你注意到聚光灯对准了他所说的你的“担心的问题”时，把聚光灯转移到你当时正在做的任何事情上，而不要试图回答任何这些问题。

这个想法并不是要分散你对这些问题存在这一事实的注意力(它们不会神奇地消失)，而是要忽略它们。

“他们仍然在那里，但他们在后台的黑暗中，你没有对他们做任何事情，”他 [在《今日心理学》](https://www.psychologytoday.com/us/blog/understanding-the-anxious-mind/202210/how-stop-worrying-the-spotlight-technique) 中写道。“你关注的焦点是你此刻正在做的事情；焦虑的想法被忽视了。”

是的，这很难，需要练习，但斯坦说，最终，这是要给自己继续前进的许可——而不是在你回答完所有担心的问题之前一直停留在精神上。
