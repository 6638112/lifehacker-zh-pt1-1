# 你终于可以将你的安卓系统同步到你的 Windows 电脑上了

> 原文：<https://lifehacker.com/you-can-finally-sync-your-android-to-your-windows-pc-1848805207>

一部 iPhone 可以与苹果电脑无缝兼容——一台苹果电脑上的照片会显示在另一台苹果电脑上，通话和信息也是如此。但是如果你在栅栏的另一边，而使用 Windows 和 Android 呢？



在过去的几年里，微软一直在努力将这种集成引入 Android 智能手机。这项工作似乎得到了回报，新的 Phone Link 应用程序就是证明。

## Phone Link app 做什么？

使用 Phone Link 应用程序，您可以将任何 Android 智能手机与 Windows 10 或 Windows 11 PC 配对。一旦它无线配对(使用一种新的基于二维码的工作流程)，您的 PC 将能够在您的 PC 上获得 Android 通知。而且，就像 Mac 一样，你可以发送信息和访问你的照片库。

## 如何在 Android 和 PC 上设置 Phone Link 应用程序

在使用电话链接之前，需要进行一些设置。在你的安卓智能手机上，安装 [链接到 Windows 应用](https://play.google.com/store/apps/details?id=com.microsoft.appmanager) 。在 Windows 上，Phone Link 应用程序是预装的，但可能不会更新，因此打开微软商店，搜索 Phone Link 应用程序，并安装任何可用的更新。

然后打开 PC 上的 Phone Link 应用程序。点击**开始**按钮。在您的 Android 智能手机上打开 Windows 应用程序的链接，并点击“**链接您的手机和 PC** ”按钮。

O 在你的电脑上，应用程序会询问你是否安装了 Android 应用程序。点击与二维码配对的**按钮。这将在您的电脑上显示一个二维码。在你的 Android 智能手机上，应用程序会问你二维码是否准备好了。点击“**继续**按钮，扫描您在 Windows PC 上看到的二维码。**

授予所需的权限并等待进程完成。在你的电脑上，你会看到一个介绍，介绍你可以使用该应用程序做的所有事情。点击“**开始**按钮，Phone Link app 将准备就绪。在顶部，您可以在“信息”、“照片”、“应用”和“通话”选项卡之间切换。通知将出现在左侧。

还需要最后一步，那就是设置通知中继。在 Phone Link 应用程序中，单击左侧边栏中的通知部分。在这里，您需要获得同步通知的许可。单击“在手机上打开设置”按钮。这将启动设置中的“**设备和应用程序通知**”部分。寻找“**链接到 Windows** 应用程序，并启用它旁边的切换。点击弹出框中的“**允许**”按钮，设置完成。
