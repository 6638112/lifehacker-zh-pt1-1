# 你想试试冰摩卡可乐吗？

> 原文：<https://lifehacker.com/you-want-to-try-this-iced-mocha-cola-1849081053>

夏天的下午很难熬，尤其是当你陷入午餐后的低迷时——你仍然有工作要做，但你的大脑已经检查过一天了。下一次发生这种情况时，可能会弄混一杯 [冰摩卡可乐](https://coffeeguycafe.com/mocha-cola/) ，我发现这是一种完美的含咖啡因饮料。咖啡、巧克力和碳酸饮料的轻松定制组合。



一杯冰摩卡可乐在精神上类似于 [巧克力鸡蛋奶油](https://food52.com/blog/17547-how-to-make-a-proper-chocolate-egg-cream-without-a-recipe) ，这是纽约的一种主食，既不含鸡蛋也不含奶油，而是由全脂牛奶、巧克力糖浆和苏打水制成。这种组合将儿时喜爱的巧克力牛奶提升为一种真正有个性的饮料，而冰摩卡可乐给了它额外的刺激。它需要巧克力牛奶、冷煮咖啡和可乐，并提供巧克力的甜味、咖啡的滑爽和苏打水的气泡。这是很多，但有些日子需要很多。

## 冰摩卡可乐

(制作一份)

*   1/4 杯冷煮咖啡
*   1/3 杯可乐
*   3/4 杯巧克力牛奶
*   可选:鲜奶油或冰淇淋，作为浇头

要制作冰摩卡可乐，混合所有原料，搅拌，然后倒入冰块。如果需要，可以在上面浇上鲜奶油或加入一勺冰淇淋。味道就像一个长大的巧克力蛋奶；咖啡调和了巧克力牛奶的甜味，可乐增加了一种刺激。

## 根据你的喜好定制它

冰摩卡可乐最好的一点是它的适应性。如果你想要一种含咖啡因少一点的饮料，把巧克力牛奶和可乐混合，不要咖啡。如果甜点是你想吃的，在上面放些鲜奶油或冰淇淋，会增加丝滑的甜味。如果你想要更多的成人饮料，你可以用咖啡利口酒代替一些冰镇饮料。如果你不能处理乳制品，你可以使用巧克力替代牛奶。说到可乐，零度可乐永远是一个选择。
