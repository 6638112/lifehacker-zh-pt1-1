# WordAll让你猜所有的单词

> 原文:[https://life hacker . com/word all-makes-you-guess-all-the-words-1848968860](https://lifehacker.com/wordall-makes-you-guess-all-the-words-1848968860)

一些更有趣的Wordle变体来自于不把绿色和黄色方块视为单一解决方案的暗示，而是定义了一个可能的单词池。看起来它好像在你玩的时候改变它的“思维”，但它实际上是在缩小可能答案的范围，直到只剩下一个。《纽约时报》的分析工具 [Wordlebot](https://lifehacker.com/this-bot-can-tell-you-how-good-you-really-are-at-wordle-1848769075) 根据你的猜测将可能性分成不同池的效率，对你的解题表现进行评级。现在， [WordAll](https://wordall.xyz/) 全部拉开帷幕。不是猜测一个单词，而是给你定义一个单词库的线索，你必须找出这个单词库中的所有单词。

Watch

WordAll给了你两个已经填好的猜测，所以不要担心选择一个完美的开始。它会告诉你池中有多少单词。例如，对于上面的游戏板，解决方案变成了纳塔尔，海军，肚脐和塔龙。

只有池中的单词允许作为猜测，所以这有点像在原始Wordle的“硬模式”中玩。(例如，如果一个字母是绿色的，合法的猜测必须在那个地方使用那个字母。)

每当你在池中猜出一个单词，方块的颜色就会更新，提示你剩下的内容。例如，如果你猜了一个以N开头的单词，而那个正方形一半是绿色，一半是黄色，你就知道剩下的单词中至少有一个在那个地方有一个N，至少还有一个在不同的地方有一个N。一旦你猜中了所有以N开头的单词，那个方块现在就会显示为全黄色。

我觉得这是一个有趣的游戏。不是最难变的，也不是最容易变的。你猜对所有单词的时间，以及在此过程中你尝试输入的错误猜测次数将会得到分数。比如我用两个错误的答案在4:20解出了今天的谜题 (不是图中显示的那个)。也许你会有更好的运气？
