# 他汀类药物并不会导致所有被指责的肌肉疼痛

> 原文：<https://lifehacker.com/statins-dont-cause-all-the-muscle-pain-theyre-blamed-fo-1849475989>

他汀类药物是一类降胆固醇药物，通常开在处方上，可能会挽救很多人的生命。他们也因为能够引起肌肉疼痛而臭名昭著，这导致许多人停止他们的他汀类药物，或者一开始就对服用一种药物犹豫不决。但是一项 [的新研究](https://www.thelancet.com/journals/lancet/article/PIIS0140-6736(22)01545-8/fulltext?utm_campaign=tlpr&utm_medium=email&_hsmi=224048558&_hsenc=p2ANqtz-9DZauO_aptPC-lkhErDyWtWPkh1-M9veYxKv4JIuYvPWzjOWCYmUQpOtBIPV7k6Zj5zThflF0rLIfmUYu20zc2BHNNqZo4guQZ6IkfQsL5XPaF5Ts&utm_content=224048558&utm_source=hs_email#seccestitle130) 表明，人们服用他汀类药物引起的大部分肌肉疼痛根本不是来自他汀类药物。

这项研究旨在计算有多少人因服用他汀类药物而导致肌肉疼痛，而不仅仅是服用他汀类药物的 T2。因为，当然，肌肉疼痛可能来自其他来源。在健身房锻炼了一天后，你会感到酸痛，在打扫了一整天的院子后，或因流感而肌肉酸痛。各种医疗条件也可能导致肌肉疼痛，包括纤维肌痛、狼疮和莱姆病。除了他汀类药物之外的其他药物也可能是罪魁祸首。

新的荟萃分析着眼于安慰剂对照的大规模随机他汀类药物研究。一组患者服用了他汀类药物，而另一组没有。)如果他汀类药物通常会导致肌肉疼痛，我们预计他汀类药物组中会出现比相应对照组更多的肌肉疼痛患者。但是研究人员发现，肌肉疼痛、痉挛和相关症状在两组中都很常见，服用他汀类药物的人比服用安慰剂的人报告这些症状的可能性只稍微大一点。他们计算出，在服用他汀类药物并经历肌肉疼痛的人中，如果他们没有服用他汀类药物，至少 90%的人仍然会有肌肉疼痛。

重要的是，他们还发现，因服用他汀类药物而导致肌肉疼痛的人往往会在服药的第一年出现副作用。因此，如果你已经服用他汀类药物五年，只是现在才经历肌肉疼痛，很有可能不是他汀类药物引起的。他们还发现，由他汀类药物引起的肌肉疼痛并不比其他类型的肌肉疼痛更严重。

如果服用他汀类药物的人确实出现肌肉疼痛，目前还没有指导方针；作者认为，该领域需要更好的建议来处理这种情况。一个肌肉疼痛的人可能会考虑停止他汀类药物，但他们也可能受益于调查疼痛的来源，以防是其他需要医疗关注的事情。

最终，这归结为风险与收益。该研究的作者计算出，在每 10000 名开始使用他汀类药物并在五年内看到其低密度脂蛋白胆固醇每升下降 1 毫摩尔的人中，他汀类药物“可能导致 11 次(通常轻微)肌肉疼痛或虚弱发作，但在已有血管疾病的人中预防 50 次主要血管事件(二级预防)，在无血管疾病的人中预防 25 次主要血管事件(一级预防)。”

最终，你是否服用他汀类药物应该由你的医生来决定。但是，如果肌肉疼痛是你犹豫不决的主要原因，新数据可能会帮助你做出更明智的决定。
