# 跑步时如何正确超车

> 原文:[https://life hacker . com/how-to-pass-someone-the-right-way-when-you-running-1847768591](https://lifehacker.com/how-to-pass-someone-the-right-way-when-youre-running-1847768591)

跑步通常是一项令人寒冷的孤独活动，但是如果你是那种过度思考你的社会交往的人，它也会带来一些问题。一个是当你遇到另一个跑步者向你走来时你做的事情(你 [点头或给半波](https://lifehacker.com/the-unwritten-rules-of-running-according-to-reddit-1847540740) )。另一个更棘手的问题是，当你要超过别人时该怎么办。

Watch

有时候那个人是另一个跑者；其他时候，他们只是一个人在公园里散步，也许带着他们的狗或一群朋友。下面是如何礼貌的通过。

## 决定站在哪一边

典型地(在美国 S )。我们尽可能从左边超车——就像你在高速公路上变道超车一样。

但这个要看具体情况。如果你经过的人正靠在路边或栅栏之类的障碍物上，试图挤过去是不明智的。还要考虑是否有安全或礼貌的理由选择一边而不是另一边。如果他们的左边有一个小孩或狗，计划从他们的右边通过。

## 调整你的速度

很有可能，你已经为你的跑步设定了目标配速，并且你会希望平均坚持下去。但这并不意味着你应该花几分钟慢慢地超过那个人，他在10:00的时候你在9:45跑。

当你和那个人保持合理的距离时，准备行动。你会想加快速度，这样你就可以有效地超过他们，确保你的冲刺时间，这样你就不会在另一个分心的时间，比如路过的汽车。

## 礼貌地宣布你自己

超越别人的正确方法是大喊“在你的左边！”如果你走在这个人的左边，但不是每个人都能在你靠近的那一瞬间理解这些话。他们可能只会听到”...左！”并且不确定你是否从左边超过了T1，或者他们是否应该让T2把T3移到他们的左边。(也有相当多的人会花一两秒钟来记起哪一侧*是他们的左边*。)[Reddit的跑步者同意](https://www.reddit.com/r/running/comments/px8eqk/how_do_you_pass_people_while_running/) : A 几乎一半的时间里，人们会意外地踩*进*你的路。

绕过这个困境的一个好办法是把你自己向后退一点(给这个人更多的时间)，然后看他们朝哪个方向走。一声欢快的“早上好！”可以做到这一点，或者类似“对不起！”或者“从你后面过来！”

一些跑步者喜欢故意让他们的鞋子在地上摩擦，或者咳嗽或者清嗓子——任何能让人们意识到你正在接近他们的事情。如果你想成为一个爱开玩笑的人，总会有“蜜蜂！”或者“热汤来了！”
