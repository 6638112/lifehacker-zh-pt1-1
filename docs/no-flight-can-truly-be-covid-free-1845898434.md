# 没有航班可以真正做到“无COVID”

> 原文:[https://life hacker . com/no-flight-can-true-be-covid-free-1845898434](https://lifehacker.com/no-flight-can-truly-be-covid-free-1845898434)

达美航空公司 [现在提供免检飞往欧洲](https://news.delta.com/delta-launch-first-quarantine-free-covid-free-travel-europe-through-new-testing-protocols) 的航班，其中，他们还大胆地称之为“免检”要是保证冠状病毒不在飞机上这么容易就好了。不过，这些航班确实使用了一种测试协议来降低携带COVID 的人成功登机的几率*。*

Watch

到目前为止，该航空公司正在提供两条这样的航线:一条是从亚特兰大到阿姆斯特丹的T1航线，另一条是从T2到T4的T5航线，还有一条是从亚特兰大到罗马的T7航线。意大利和荷兰都要求旅客进行检疫，但同意乘坐这些航班入境的人可以绕过这一限制。

也就是说，这两个国家目前都允许游客旅行，所以这些航班是为有合法工作、上学或健康原因的人准备的。

这些航班将在低于满负荷的情况下运行，中间座位被封锁，强制佩戴口罩。达美描述预飞行测试协议如下:

> 要乘坐达美航空在亚特兰大和罗马之间经COVID检测的航班，客户需要通过以下方式检测新冠肺炎病毒呈阴性:

> *   起飞前72小时进行的COVID聚合酶链反应(PCR)测试
> *   在亚特兰大机场登机前进行的快速测试
> *   抵达罗马时的快速测试-菲乌米奇诺
> *   前往美国前在罗马菲乌米奇诺进行的快速测试
> 
> 顾客还将被要求在进入美国时提供信息，以支持CDC的接触者追踪协议。

达美航空公司报告称，佐治亚州、荷兰政府和罗马政府已经同意该协议，数学模型显示，在这种飞行中感染COVID的风险微乎其微。

也就是说，不可能确保一次飞行是真正无COVID的。例如，当您通过机场时，您可能会接触到COVID，在这种情况下，您很可能在快速检测中检测为阴性。

如果你因为重要的原因需要旅行，尤其是如果你是高风险人群，这个选项仍然有用。达美航空没有说如果你预订了航班，然后在机场检测出阳性，或者如果你坐飞机，然后在着陆时检测出阳性，会发生什么。
