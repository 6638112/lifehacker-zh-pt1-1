# working moms shouldnt have to clean up governments mess 1845497756

> 原文：<https://jezebel.com/working-moms-shouldnt-have-to-clean-up-governments-mess-1845497756>

职业妈妈们以前都听说过:“灵活一点。你会联系你的支持网络吗？你会提前制作和冷冻健康餐吗？你试过用电子表格来保持条理吗？”我们会觉得，每次我们达不到神话般的工作与生活的平衡，都是因为个人的失败。在冠状病毒疫情期间，妈妈们的压力变得更大了。

但是新冠肺炎的教训并不是妈妈们做得不够多。相反，即使在国家紧急状态下，妈妈们也被要求承担更多。冠状病毒给女性带来了比男性更大的经济和家庭负担，我们应该提醒自己，是系统性缺陷而不是个体问题让许多女性难以充分参与我们的劳动力。



在这个疫情时期，女性失去工作的速度更快，被重新雇用的速度也更慢。许多行业都关闭了——比如，酒店、教育和非必要的医疗服务——以女性为主的。现在，由于税收损失和缺乏联邦支持，我们面临着州和地方政府裁员的可能性，女性将再次不成比例地面临裁员。

那些保住了工作的女性——无论是作为基本工作人员还是通过远程工作——都不得不努力应对学校关闭和无法获得托儿服务的问题。女性倾向于承担更多家务和照顾孩子的责任，即使在那些自我报告平等分工的异性恋夫妇中也是如此。因此，在转向远程学习和关闭夏令营的过程中，毫不奇怪，许多女性 [发现自己肩负起](https://www.cnn.com/2020/07/21/economy/imf-covid-inequality-women/index.html) 额外的育儿责任。

疫情冠状病毒并没有使女性的安全网变得脆弱；这让他们四分五裂。最近的一项研究发现，1/4 的女性 因为新冠肺炎而考虑离职或缩减职业承诺。上个月，退出劳动力市场的美国人中有五分之四是女性。

“我做不到”的感觉无疑在今年对妈妈们来说更加明显，但我们都知道 2020 年并不是我们第一次认为杂耍是不可能的。一段时间以来，儿童保育费用一直在攀升 ，工薪家庭在这方面的支出占其收入的比例越来越大。作为国内生产总值的一部分，美国在儿童保育和早期教育上的投资 [不到其他工业化国家平均水平的一半](http://www.oecd.org/els/family/database.htm) ，而且买单的是家庭。



新冠肺炎为有色人种女性设置的障碍甚至更高。甚至在疫情之前，黑人和多种族父母因育儿问题辞职或缩减工作的可能性是非西班牙裔白人父母的两倍。现在，86%有孩子的西班牙裔家庭和 66%有孩子的黑人家庭报告说，由于新冠肺炎，他们遇到了 [严重的财务问题](https://www.npr.org/sections/health-shots/2020/09/30/918158081/juggling-financial-stress-and-caregiving-parents-are-very-not-ok-in-the-pandemic) 。相比之下，51%的白人家庭持相同观点。

尽管面临这些挑战，职业女性还是为我们的经济做出了巨大贡献。我们的劳动力每年为美国经济贡献超过 7.5 万亿美元 。将家庭在儿童保育上的支出限制在收入的 10%可以增加女性的经济参与，足以使 GDP 增长 2100 亿美元。

要点:经济复苏和扩张需要支持女性就业。

我们可以从投资儿童保育开始，而且要快。如果没有帮助，超过五分之四的儿童保育项目 [将撑不过 2021 年夏天的](https://www.naeyc.org/sites/default/files/globally-shared/downloads/PDFs/our-work/public-policy-advocacy/holding_on_until_help_comes.survey_analysis_july_2020.pdf) 。我们的儿童保育系统需要每月近 100 亿美元的资金来避免永久关闭。虽然众议院已经通过立法，提供急需的紧急援助，以支持我们国家的儿童保育提供者，但参议院未能采取行动。与此同时，特朗普总统甚至没有认识到这个问题，更不用说暗示这是一个优先事项。



但不要搞错:虽然加强我们的儿童保育系统是疫情的重要政策，但它也是必要的公共政策。如果我们不采取行动，我们持续的儿童保育危机将比当前的公共卫生危机持续更久。去年，我提出了《儿童和老年人家庭储蓄法》[](https://porter.house.gov/news/documentsingle.aspx?DocumentID=34)*，这将允许家庭在税前留出更多的钱用于儿童保育。根据现行法律，这一限额为 5000 美元，自 20 世纪 80 年代以来没有根据通货膨胀进行调整。乔·拜登(Joe Biden)[有一个计划](https://joebiden.com/caregiving) 来保证 3 至 4 岁儿童的普遍学前教育，并投资于看护者和幼儿教育工作者，我们必须让我们的下一任总统对投资于一个更平等、因此更强大的劳动力队伍负责。* 

*任何职业母亲都会告诉你，在疫情之前，生活已经够艰难了。今天，随着冠状病毒继续攻击美国人民的身体和经济健康，它也不幸地提醒我们，当我们未能支持工作的父母，特别是妇女时，会发生什么样的破坏。* 

*我们一直很灵活，我们通过我们的网络打电话，我们已经做好了准备，我们已经对我们的电子表格进行了颜色编码。这是不够的，也永远不会是——不管是不是疫情——除非我们的领导人不再要求职业妈妈们收拾她们不作为的烂摊子，并开始投资解决方案。* 

*凯蒂·波特代表加州第 45 选区。她是国会中唯一一位有年幼孩子的单身母亲。*
