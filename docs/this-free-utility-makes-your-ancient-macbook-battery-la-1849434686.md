# 这个免费的实用程序可以让你的旧 MacBook 电池持续更长时间

> 原文：<https://lifehacker.com/this-free-utility-makes-your-ancient-macbook-battery-la-1849434686>

如果你在摇一台旧的 MacBook，你可能不指望它能跟上最新最好的——苹果硅机显然会在任何一天胜过那台 5 年前的英特尔 Mac。然而，你的旧笔记本电脑可能还有一些寿命。如果你的手机太热，电池很快就没电了，快速破解可以让它运行得更冷更久。

为了延长英特尔架构 MacBook 的电池寿命，您的第一反应可能是启用低功耗模式。这个设置对于降低你的机器上的电池使用速度肯定是有用的，但是它也带来了一些问题。第一，并非所有的英特尔 MAC 都支持低功耗模式，因为它只与 [macOS Monterey](https://lifehacker.com/31-new-macos-monterey-features-youll-actually-want-to-u-1847900965) 一起推出。如果你的不支持蒙特利，你显然不会有这个选项。

第二，我们实际上不太了解 Mac 上的低功耗模式是如何工作的。我们可以假设该功能会降低性能、后台任务、数据获取和其他资源密集型过程，但苹果没有透露具体是如何降低的。该公司在低功耗模式如何影响 iPhone 和 iPad 方面更加透明；对于 Mac，他们简单地说它“ [管理高能耗的应用程序和处理](https://www.apple.com/macos/monterey/) ”

可以肯定地说，低功耗模式会从多方面改变 Mac 的性能，从而延长电池寿命。这很好，但可能有点过头了。取而代之的是，针对你的英特尔 Mac 电脑过热和电池耗尽的罪魁祸首:睿频加速。

## 睿频加速会耗尽您的英特尔 MacBook 电池

睿频加速是英特尔的一项功能，旨在将更多的处理能力集成到计算机芯片中，供您在需要时使用。对于大多数任务，你的 Mac 电脑的 CPU 在其基本处理速度(比如 2.7GHz)下达到最大值。当电脑感觉到需要更高的处理能力时，它可以启用 Turbo Boost 来暂时提高 CPU 的速度，比如 3.6GHz。理论上，这是一种平衡便携式 CPU 性能和可观电池寿命的聪明方法。然而，尽管 Turbo Boost 只是间歇性地运行，但当它开始运行时，它可能会对 Mac 的老化电池造成损害。它还可以加热旧的英特尔芯片，如果它们变得太热，你的机器的 [性能无论如何都会受到抑制](https://lifehacker.com/what-to-do-when-your-laptop-keeps-overheating-1848317156) ，从而挫败 Turbo Boost 的预期目的。

就我个人而言，比起性能，我更看重更长的电池续航时间，尤其是当我使用电池供电时。如果你同意，试试 [涡轮增压切换器](http://tbswitcher.rugarciap.com/) 。这个免费的工具可以让你随时打开或关闭涡轮增压。这样，无论您是在 Final Cut Pro 中编写电子邮件还是编辑视频，Mac 的 CPU 都不会超过其基本时钟速度。这个简单的改变可以节省几个小时的电池寿命，并保持笔记本电脑冷却和风扇低速运转。为了让你确切地知道它是如何工作的，睿频加速专业版允许你随时监控 CPU 的速度和温度。

涡轮增压切换器声称可以减少 25%的电池使用量，降低 20%的 CPU 温度，尽管它只引用了 2015 年的一篇博客文章作为证据 。也就是说，大部分评论是积极的，我个人也有很好的体验。虽然它可能会也可能不会达到这些崇高的统计数字，但它*将*可能会改善你的整体 MacBook 体验。

这款应用程序在免费状态下完全可用(尽管它经常询问你的 Mac 密码)，但付费的“专业”版提供了一些有趣的额外好处。例如，当电池电量过低时，当您打开某些应用程序时，当您的 CPU 温度和风扇速度达到某个阈值时，或者当您断开电源时，您可以将实用程序设置为自动启用或停用睿频加速。Turbo Boost Switcher Pro 也只要求您输入一次密码，这可能值 10 美元。

无论付费还是免费，主要功能都是一样的:每当你高兴地用潜在的性能提升换取额外的电池寿命时，Turbo Boost Switcher 就会让它发生，而不用担心 macOS 在后台限制其他进程。
