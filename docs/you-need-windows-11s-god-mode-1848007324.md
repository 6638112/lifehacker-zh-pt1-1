# 你需要 Windows 11 的“上帝模式”

> 原文：<https://lifehacker.com/you-need-windows-11s-god-mode-1848007324>

如果你讨厌在 Windows 11 的菜单中寻找隐藏在控制面板或系统设置中的工具或设置，请尝试打开隐藏的主控制面板功能——或者更为人所知的:“上帝模式”



通常情况下，Windows 将许多重要设置隐藏在控制面板和设置菜单的深处，迫使用户仔细查看他们的系统，只为了更改他们电脑的日期和时间，整理硬盘碎片，或创建系统还原点。

然而，在 Windows 11 中打开 God 模式会在你的桌面上添加一个主控制面板快捷方式，其中充满了通常在控制面板和设置菜单中找到的 260 多个快捷方式和设置。从管理 USB 设备，卸载程序，改变你的电脑的电源设置等等，你都可以在桌面上完成。

Windows 11 的上帝模式不仅感觉像一个让你的电脑更容易导航的作弊代码，你甚至可以使用一个秘密代码来打开它，就像你在旧的视频游戏中一样。以下是要做的事情:

1.  右键单击桌面上的空白区域。
2.  选择**新建>文件夹。**
3.  将新文件夹重命名为: **GodMode。{ ed7ba 470-8e 54-465 e-825 c-99712043 e01c }**
4.  打开重命名的文件夹。Windows 会自动将其转换为主控制面板文件夹。

上帝模式现在启用了——明白我说的用作弊代码打开它的意思了吧？这些步骤在 Windows 10 (以及 Windows 7、8 和 8.1)上同样适用 [。](https://lifehacker.com/activate-god-mode-in-windows-10-for-quick-access-to-win-1721853635)

我们应该注意到，上帝模式不会对你的电脑做出任何重大改变，也不需要专家级的理解才能使用。也就是说，你应该只在需要的时候使用文件夹中的工具，如果你不知道它们是做什么的，不要乱用任何高级设置。

各种上帝模式的功能和设置在不同的操作系统之间会有所不同，但是你可以在 Windows 控制面板中找到的所有功能都包括在内。您甚至可以将主控制面板列表中的任何项目拖放到您的桌面上，以创建一个快捷方式来更快地访问，并且有一个搜索栏，您可以使用它来快速找到任何功能。

如果你决定不再需要上帝模式，只需从桌面上删除该文件夹。
