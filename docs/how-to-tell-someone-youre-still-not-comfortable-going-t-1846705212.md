# 如何告诉某人你仍然不愿意在 COVID 期间参加他们的大型活动

> 原文：<https://lifehacker.com/how-to-tell-someone-youre-still-not-comfortable-going-t-1846705212>

虽然很难看到一个全球性的疫情，仅在美国就有超过 566，000 人死亡，并从中找出任何积极的东西，但一些人发现公共卫生措施的一些方面很方便。

Watch

无论是远程工作的能力，还是不必社交的合理借口，可能都有社交距离和不举行大型聚会的要求让事情变得更容易的时候。事实上，他们可能已经帮你摆脱了你一开始就不想参加的事情。

抛开你社交的欲望(或缺乏)不谈，你可能仍然(理所当然地)担心参加面对面的活动，并想知道如何处理已经开始慢慢到来的邀请。你生命中的一些人可能会完全理解。

然而，其他人会反驳，并向你解释为什么他们认为他们的活动实际上是安全的——典型的是“但这将是在户外，每个人都将戴上口罩，超级小心。”很好，但是也不可以。这里有一些方法可以帮助你在拒绝他们的邀请时引导讨论。

### 是的，我们 100%还在疫情

2020 年，人们(在很大程度上)接受了不想传播或感染致命病毒作为你不参加他们的婚礼、毕业典礼、生日、感恩节晚餐等的理由。但是现在，我们在新冠肺炎疫苗推广几个月后，人们似乎认为整个事情已经结束了，我们是清白的。

我们不打算在这里探究为什么不是这样的所有原因( [)甚至全国 25%](https://www.beckershospitalreview.com/public-health/states-ranked-by-percentage-of-population-vaccinated-march-15.html#:~:text=As%20of%206%20a.m.%20EDT,19%2C%20according%20to%20the%20CDC.) 还没有接种疫苗！)，但如果有人给你“但 COVID 基本上已经结束了，大多数与会者至少已经注射了第一剂疫苗”这样的话，不要让他们让你内疚而接受邀请。你没有“走极端”或“小题大做”，也没有抗拒“人们必须过自己的生活”和“世界总有一天会恢复正常”的事实。只是想先解决这个问题。

### 你可能在避免冲突

告诉你的表妹你不能参加她的婚礼在很多层面上都很难，对一些人来说，这包括对冲突情况的焦虑。在 [一篇关于“好+好”](https://www.wellandgood.com/what-say-cant-attend-wedding/) 的文章中，玛丽·格蕾丝·加里斯采访了心理学家 [艾米·达拉穆斯](https://www.audeotherapy.com/) 博士，他解释说，你应对压力的带宽——包括源于潜在冲突的那种——现在可能相当低。

除了不想伤害你和你堂兄的关系之外，你可能不在一个你可以处理任何其他类型风险的地方——例如，传播或感染致命病毒。但是，不要让这为你做决定。“尽管这很难，但尽量不要让政治或你朋友的选择影响你的健康决定——除非你的朋友是病毒或风险管理专家——因为病毒不是受欢迎程度的竞赛，”达拉穆斯告诉 Well+Good。

### 拒绝活动邀请时该说什么

据国家礼仪专家、《让生活更美好的现代礼仪》*——*黛安·戈特斯曼 报道，她也曾为她的文章采访过加里斯——尽快进行强硬的谈话(或发送拒绝邀请的通知)。一定要保持“简短、甜蜜和礼貌”

不知所措？ [Gottsman 提供了这个示例脚本](https://www.wellandgood.com/what-say-cant-attend-wedding/) (根据需要做了一些微小的语言调整)可能会有所帮助:

> “我想亲自联系你，谢谢你的邀请。不幸的是，目前我仍然不喜欢旅行或在任何人群中。我感谢你的理解，并希望你知道，我们祝你好运，疫情一结束，我们就会庆祝。”

理想情况下，这个人会理解——尽管他们可能会失望。但还是那句话:我们仍处于疫情，所以 2020 年前的社会规范和期望已经被抛到了九霄云外(你很可能已经为新鲜空气和通风敞开了大门)。
