# 使用这个网站来预测你下一次飞行中潜在的湍流

当谈到飞行前准备时，经常(甚至偶尔)坐飞机的人往往会分成两种不同的群体。有些人花时间和精力尽可能多地获取关于他们即将到来的航班的信息——从查看出发和到达地点的天气预报，到偷看飞机上的座位图，再到查找飞机的年龄和历史信息。

不管是因为他们害怕飞行，还是对航空着迷，或者仅仅是想知道前方会发生什么，如果你知道在哪里(并且想要)找到这些信息，那么外面有相当多的信息。

此外，还有一些凭感觉飞行的人，在机场展示，希望一切顺利。也许他们选择接收关于航班延误或取消的短信或电子邮件，但除此之外，他们对在机翼上和祈祷中开始他们的旅行很满意。*

如果你喜欢提前了解航班的所有细节，你可能会对一个新网站感兴趣，这个网站可以预测即将到来的航班是否有遭遇颠簸的机会。以下是需要了解的内容。

### 如何使用湍流预测网站Turbli

它被称为 [Turbli](https://turbli.com/) ，我们第一次在旅游网站 [上发现它，一次一英里](https://onemileatatime.com/predict-turbulence-flight/) (它不隶属于Turbli)。使用NOAA/NWS制作的预测——飞行员使用这些预测来计划飞行——tur bli在飞行前36小时为乘客提供湍流预测。

要使用它，请输入您的出发地和目的地，并指明您的航班是今天还是明天。然后Turbli会调出任何符合描述的航班，一旦你选择了你的，它会根据天气和飞机对你的航班进行预测。

### 关于这个工具需要记住什么

对你即将到来的飞行的预测以一份非常详细的报告的形式出现，报告中有图表和大量数字。它不仅会预测你飞行的哪些部分(如果有的话)可能最容易出现湍流，而且还会预测你起飞和着陆的平稳性。

虽然 [湍流](https://turbli.com/) 可能是数据和/或航空爱好者数小时娱乐的来源——或者是那些害怕湍流的人数小时恐惧和焦虑的来源——但重要的是要记住这些是 *预测*，还有许多其他因素导致航班中断。

话虽如此，如果你是一个在颠簸的航班上容易晕机(或焦虑发作)的人，你至少可以在登机时对你可能会经历的事情有所了解。或者你宁愿不知道，在这种情况下，Turbli可能不适合你。
