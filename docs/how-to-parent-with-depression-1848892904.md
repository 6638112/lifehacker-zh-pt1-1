# 如何养育抑郁症患者

> 原文：<https://lifehacker.com/how-to-parent-with-depression-1848892904>

每个人都有糟糕的日子，悲伤的日子，甚至可怕的，可怕的，不好的，非常糟糕的日子，但抑郁症是一种精神疾病，被归类为持续的悲伤，绝望或缺乏兴趣的感觉。疫情看到了这种疾病的上升趋势——国家精神疾病联盟(NAMI)表示[2020 年有 2100 万成年人或 8.4%的美国人口患有抑郁症](https://www.nami.org/About-Mental-Illness/Mental-Health-Conditions/Depression) 。患有抑郁症可能会耗尽精力，很难独自应对，因此在照顾孩子的同时应对抑郁症可能会特别令人生畏。幸运的是，有许多资源、专家和支持系统可以提供帮助。



## 抑郁症的征兆是什么？

留意抑郁的迹象。如果你有超过几周的症状，请联系你的医务人员。“抑郁症是最常见的精神疾病，也是最容易治疗的疾病之一，”罗格斯大学行为健康保健首席心理学家斯蒂芬妮·马塞罗 博士说。"不要隐瞒任何这些迹象或症状."

常见症状包括:

*   睡眠改变
*   食欲改变
*   注意力不集中
*   能损失
*   绝望或内疚
*   失去动力
*   激动
*   疼痛
*   自杀意念

如果你有自残或自杀的想法，请立即拨打类似全国自杀预防热线 的自杀热线 [。](https://suicidepreventionlifeline.org/)

## 如何寻找抑郁症患者的支持

马塞洛说，“患有抑郁症而不寻求支持的人遭受了不必要的痛苦。”寻求支持的第一个地方是你的伴侣、家人或朋友。“我们知道社会孤立会增加一个人患抑郁症的风险，”她说。然而，要小心任何有害的友谊，因为“我们也知道过于关注讨论问题的友谊实际上会增加抑郁的症状。”

养育抑郁症患者的下一步是联系你的医疗服务提供者。他们可能会推荐药物、治疗或两者的结合。“治疗和抗抑郁药物的结合被证明是最有效的，”Marcello 说。“治疗可以帮助一个人明确导致抑郁的生活问题，以及他们可能能够解决或改善这些问题的哪些方面。”

你可以建立自己的社区，加入支持团体，或者找到朋友和家人，他们会在你管理抑郁症时给予你特别好的支持。

## 为休息和恢复留出时间

就像任何疾病一样，抑郁症需要休息和恢复时间。尽管当你为人父母时很难抽出时间给自己，但在这段时间里，你比以往任何时候都更需要充足的睡眠、锻炼、营养和照顾。

请记住，这种恢复可能也有点像恢复到真正的你，恢复到你自己。马塞洛说，“当你感到沮丧时，你经常感觉不像自己。”因此，做你喜欢的活动是很重要的，即使你觉得孤独。

## 留意你的孩子受到影响的迹象

抑郁症可以由多种因素引起，其中之一是遗传。Marcello 建议，“如果父母或看护者知道他们正在应对抑郁症，他们可能希望确保孩子得到筛查和评估。”她说，这对于青少年来说尤其重要，因为与父母患有抑郁症的青少年相比，那些在父母患有抑郁症的情况下成长的青少年更容易在年轻时出现更频繁、更严重的抑郁症她建议，如果你有顾虑，可以联系你的家庭医生、儿科医生或学校。

虽然抑郁症可能更多地发生在青少年身上，但当父母感觉不适时，年幼的孩子也会表现出担忧或悲伤的迹象。儿童心理研究所建议，“以一种清晰的、适合年龄的方式和孩子谈论你的抑郁症，”比如告诉一个年幼的孩子你有一种让你感到疲劳的状况，并在和一个年长的孩子交谈时说出你的抑郁症。他们建议告诉你的孩子你正在接受医生的治疗。

## 这不是你的错

父母兼作家珍妮·劳森说得好:“ [抑郁症在于](https://thebloggess.com/2012/04/30/depression-lies/) 。”抑郁症会告诉你，你的精神疾病是你的错，你的孩子会因此而受苦。并非如此。将父母视为遭受痛苦、获得帮助并努力接受治疗的人，是韧性和诚实的典范。他们更有可能寻求帮助，努力解决自己的问题。通过养育抑郁症患者，你向你的孩子表明你爱并照顾自己，，你也爱并将照顾你的孩子。
