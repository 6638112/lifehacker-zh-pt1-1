对于大多数在苹果产品上遇到麻烦的人来说，获得帮助相当容易:所有苹果支持，描述你的问题，并解决它。然而，对于那些有听力障碍的人来说，这不一定是一个选择。如果那是你，你可能不知道你可以用手语联系苹果寻求帮助:免费的苹果支持服务 sign timesign time 在视频通话中直接与手语翻译联系。

这对听力有困难的人来说是一个福音，因为苹果公司的手语翻译经过培训，能够理解该公司产品特有的技术术语和问题。这意味着你会花更少的时间来解释你的问题，这样翻译可以更快找到问题的根源。

## SignTime 支持哪些语言？

在撰写本文时，SignTime 支持以下语言:

*   美国手语
*   英国手语
*   法国手语(LSF)

## 如何使用 SignTime

当你准备好与苹果的手语翻译交谈时，访问你所在地区的 SignTime 网站。这些网站如下:

*   [SignTime 美国](https://www.signtime.apple/applecare/us-EN/asl)T3】
*   [SignTime 加拿大](https://support.apple.com/en-gb/HT212836)T3】
*   [SignTime 英国](https://www.signtime.apple/applecare/gb-EN/bsl)T3】
*   [法国签字](https://www.signtime.apple/applecare/fr-FR/lsf)

点击蓝色的**连接翻译**按钮开始。该网站将要求访问您的摄像头和麦克风，并自动为您连接翻译。一旦你向你的翻译描述了问题，他们会为你联系苹果支持，并转达你提供的信息。你的 sol utions 也会用手语传递给你。
