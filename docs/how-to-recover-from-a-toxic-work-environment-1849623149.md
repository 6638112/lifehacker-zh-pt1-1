# 如何从有毒的工作环境中恢复

> 原文：<https://lifehacker.com/how-to-recover-from-a-toxic-work-environment-1849623149>

有毒的工作环境会侵蚀你的自信，甚至在你设法逃脱之后，这种影响还会一直持续到下一份工作。无论是过于挑剔的老板还是让你一直担心失业的公司文化，从有毒的工作环境中恢复都不是一件容易的事。



“这种影响往往表现为高度警惕和反应性，”高管教练、《相信自己:停止过度思考并开始疏导情绪以获得工作成功》一书的作者梅洛迪·威尔丁说。这可能表现为害怕和你的新老板讨论想法；很难信任你的新同事；害怕在会议上发言；或者觉得需要过度补偿。

“它粘在周围，”威尔丁说。“在那之后，你可能很难完全展现自己。”不幸的是，把这些习惯带到一家新公司会对你做好工作的能力产生负面影响。威尔丁说:“很多时候，这会让你很难产生受雇于你的影响力。”。从有害的工作环境中恢复需要时间，但是有很多策略可以帮助人们与新工作建立健康的关系。

## 试着结束你上一份工作

“首先，也是最重要的，试着从过去的经历中找到终结，”威尔丁说。考虑到有毒工作的压力和摧毁灵魂的程度，你很容易发现自己一遍又一遍地回忆发生过的事情—想起你没有采取的行动，或者你觉得自己处理得不好的事件。这样反复思考过去会剥夺你继续前进的能力最终会伤害你。

威尔丁经常提出的一个建议是写一封信——要么是向你以前的工作单位告别，要么是给过去的自己写一封信。这件事可以简单到告诉你过去的自己工作中发生的事情不正常或者没关系，在你逃脱之后会变得更好。

“你很可能不会从你的老板或那个真正折磨你的同事那里得到道歉，但你可以对自己表现出欣赏，因为你是如何度过这段经历的，”威尔丁说。“你也可以对自己经历的事情表达同情，因为你很可能没有从其他人那里得到这种同情。”

## **注意潜在的触发因素**

当你从事一份新工作时，可能需要一段时间才能意识到，在以前的工作中被视为正常的东西在新工作中并不是标准。在那之前，总会有某些触发因素会让你想起以前的工作——，无论是与老板的一对一会谈，与同事的社交活动，还是向上司做报告。无论旧工作中让你紧张和不开心的是什么，都有可能让你在新工作中紧张和不开心。

为了解决这个问题，威尔丁建议要格外注意这些潜在的触发因素，想办法挤出一点额外的时间来处理它们——不管是在与新老板会面之前休息十分钟放松一下，花一些额外的时间排练一次演示，还是寻找一些非正式的方式来了解你的新同事。"你现在该如何计划，以减轻你可能感受到的任何情绪反应？"威尔丁说。

## **想办法放下自责**

威尔丁在离开有毒工作环境的人身上看到的一个共同主题是，他们倾向于责怪自己无法在这种情况下生存。“我经常看到处于有毒工作环境中的人痛打自己，责怪自己不够坚强，”威尔丁说。这经常包括说类似*为什么我不能在这里黑？*怀丁说，这种想法的最大风险是“它会让你在不健康的情况下呆得更久”。

无论具体情况如何，唯一重要的是它对你来说是不健康的。即使你的同事能够找到处理有毒物质的方法，或者如果其他人对有毒环境的样子有不同的看法，这也是事实。“你做出的最自尊的决定是离开，找一个能让你得到赏识的地方，而不是强迫自己坚持到底，在那里证明自己，”威尔丁说。
