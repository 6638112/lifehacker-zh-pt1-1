# 《魔兽世界:阴影之地》发布日的最佳插件

> 原文：<https://lifehacker.com/the-best-addons-for-returning-world-of-warcraft-shadow-1845623758>

终于来了:*魔兽世界*的第八个资料片*暗影之地*今天上线。如果你从未玩过，或者你对重新进入广阔的世界持谨慎态度，你仍然有很好的免费到 20 级的体验来看看 *WoW* 是否是你的事情——这是你在*暗影之地*中能够达到的最高级别的三分之一(更不用说你可能在一个周末就能完成的事情了，如果有的话)。

否则，对于经验丰富的玩家来说，我希望你已经完成了最后一分钟的突袭和冰冠农场，因为在*暗影之地*发布之前你已经没有多少时间了。据暴雪称，所有新内容将于太平洋时间下午 **3 点开始出现。**

你不需要在发布的时候疯狂地冲向暴雪的服务器，因为所有新的*暗影之地*内容——分别在你的首都暴风城或奥格瑞玛以 [任务](https://wow.gamepedia.com/Shadowlands:_A_Chilling_Summons) 开始——应该在该出发的时候神奇地出现。(和所有 [其他变化](https://www.wowhead.com/blue-tracker/topic/us/728064) 到任务，奖励，和其他与*暗影之地*发布相一致的事件也将在今天的不同时间点发生。)

无论你是在一个数字城市里等待*暗影之地*开始(像我一样),还是你正计划在感恩节假期及以后慢慢回到 *WoW* 中，我建议你重新认识一些最好的和最基本的插件，你可以用它们来使你的虚拟冒险变得更容易。

哇！插件可以帮助你管理你庞大的库存，告诉你如果你在任务中感觉有点迷失该去哪里，甚至告诉你去哪里*而不是*站，这样你就不会被老板熏死。有如此多的插件可以实现如此多的功能，乍一看很难知道什么是值得安装的。为了帮助你，我收集了一份我认为相当完整的插件列表，这些插件可以让*带你回到游戏中，而不会像一个全新的 20 人团队中的小冒险者那样不知所措。*

### 第一步:安装一个优秀的插件管理器

你需要花一点时间去找一个插件管理器来帮助你找到、安装和更新插件。你当然可以手动安装插件，但使用应用程序要容易一千倍。一个优秀的应用程序也有助于确保你的插件保持最新的功能和修复，尤其是当即将推出的资料片(或任何即将到来的暴雪更新)中的某些东西破坏了你最喜欢的第三方助手时。

有大量的 [惊人的](https://www.wowhead.com/guides/best-addon-managers-wow-shadowlands) 指南和 [数据库](https://ogri-la.github.io/wow-addon-managers/) 你可以用来找到最适合你的插件管理器。如果想要快速回答，我全心全意推荐[**Ajour**](https://www.getajour.com/)。这是一个开源应用程序，作为一个可执行文件下载——无需安装，也无需费力(除非你的浏览器提示这可能是一个恶意文件，我在 Edge Chromium 上注意到了这一点)。

首先，把一个 jour 放在任何你想要的文件夹中，然后运行它——不管在哪里。让应用程序知道你的*魔兽世界* addons 文件夹的位置(类似于*[你安装魔兽世界的地方]\魔兽世界\ _ 零售 _ \界面*，如果你同时玩*魔兽世界*和*魔兽世界经典)。*然后，点击左上角的便捷目录链接购买插件:

你应该安装哪些插件来方便地回到*魔兽世界？*我们开始吧。

### 阿迪巴格斯

无论你是重新开始《魔兽世界》,还是重访一个数月(或数年)未接触过的角色，保持头脑清醒的最好方法之一——在此插入你最棒的《T2 》/搞笑的《T3》——就是让一个插件管理你角色的物品清单。我喜欢 AdiBags 胜过任何其他包插件，包括 Bagnon，因为它有一个关键特性:它会自动将你的东西分类，这让你一眼就能看到你有什么(以及你可以扔掉什么)。

### 自动供应商

在 WoW 中，当你拜访商家时，没有什么比手工出售你库存中的垃圾物品更让你恼火的了。然后，一遍又一遍地点击“修复物品”按钮来修复所有当你不小心碰到一个高级世界老板时损坏的装备。用自动售货机插件解决这个问题，它会自动出售你的垃圾，并在你与商家交谈时帮你解决问题。

### 卒

即使有了《魔兽世界》最近的统计数据挤压——这就是为什么在新的*暗影之地*资料片中，你将战斗到 60 的最高等级，而不是 120+——我仍然发现所有影响你角色整体力量的不同变量有点令人困惑。例如，当一些闪亮的新剑掉落时，你应该优先加速而不是当前剑的掌握属性吗？多才多艺呢？暴击？

用一个类似卒的插件把脑力从升级中去掉，它会告诉你你库存中的某样东西是否比你装备的某件装备更强。如果你在查看任务的奖励选项，但没有看到很大的升级，你至少会知道哪个物品卖得更好，更好。

### 调酒师 4

我并不反对《魔兽世界》的默认用户界面，但我喜欢能够完全控制各种 windows live 的位置。我也喜欢根据自己的目的隐藏和显示 UI 元素，不管是漫无目的的探索还是投入战斗。除了可以使用大量其他插件完成的完全重新皮肤化之外，我认为 Bartender4 提供了一种无需重新学习全新定制 UI 就可以获得额外定制的好方法。

### 致命 Boss Mods

在你冒险的某个时刻，你会想要对付更狡猾的老板——例如，甚至是一次突袭。它们超级有趣，只要你不站在发光的东西上。(如果你碰巧在一个地下城，站在发光的东西上实际上是非常受鼓励的，或者是击败老板的唯一方法，那么考虑一下这个规则。)

除非你真的想享受一个“纯粹”的体验，否则致命 boss Mods 会很好地告诉你任何大型 Boss 战斗的关键细节，包括你接下来可能会遇到什么有趣的伤害能力，以及你何时被所说的 Boss 使用过。你通常会得到一个大喇叭警告，以及在这些时候你应该做什么或不应该做什么的快速提示。这些标注不会让你成为一名优秀的突袭者，但它们肯定会帮助你比你想象的更快地学会更棘手的战斗。此外，只要你注意不可错过的警告，它们会帮助你减少死亡。

### 卡利埃尔的追踪器

说得好听点，暴雪默认的显示你正在进行的任务的窗口有点大。我更喜欢 Kaliel 的追踪器的苗条外观，它也带有一堆有用的自动装置和过滤器，所以你可以在当前区域内专注于任务。如果没有别的，你可以限制你的任务窗口的高度。滚动浏览你要看的内容比盯着屏幕上巨大的侧边栏要好得多。

### BtWQuests(以及它的许多“任务包”)

如果你曾经好奇你在一个特定的领域到底在做什么——无论你是在正确的任务线上努力完成它，还是你需要快速复习你正在做的事情如何融入一个更大的画面——btw quests 就是答案。这个插件向你展示了每一条任务线 *WoW* 提供的内容，给你一棵小树，让你可以回顾你已经做了什么，你接下来会做什么，以及各条任务线如何联系起来形成一个更大的故事。老实说，我只是用它来看看我在比其他人更艰难的区域还有多少事情要做。

### HandyNotes

*魔兽世界的一切都在哪里？不需要去寻找各种各样的物品、宝箱、地下城入口等等，只需要找到 HandyNotes(和它的各种补充包)。所有有趣的，独特的物品散落在周围的地形，你可能会想拿起会出现在你的游戏中的地图，你甚至可以 ALT+右键单击给自己留下任何特定地区的笔记。*

### 冷却

这个很简单。如果你的角色做了任何需要你等待一段时间才能再做的能力或威能——装备也一样——OmniCC 会在你用来做那件事的按钮上方给你一个可爱的倒计时。你会知道，确切地知道，在你再次变成强大的米尔豪斯·马纳斯托之前，还有多长时间。

### TomTom 和 TomCat 的旅行

顾名思义，TomTom 是一个方便的小插件，可以直接在你的 *WoW* UI 上放置一个类似 GPS 的导航箭头。在你的地图上放置一个航路点，或者使用一个插件将你的任务目标直接拉进 TomTom，你将能够看到你应该跑/骑/飞的方向——以及一个方便的小距离测量，告诉你还需要走多远。这不会阻止你在纳兹贾塔尔迷路，但是一点一滴都有帮助。(是的，你可以在屏幕上随意移动箭头。)

至于 TomCat 的旅游，这个单独的插件直接与 TomTom 一起工作，直接在你的地图上转储游戏中所有各种罕见的敌人的列表。鉴于 WoW 已经过渡到了“飞来飞去，杀死重要的东西”的游戏，这些列表对于计算出你需要去哪里是非常有帮助的——也就是说，省去了你的 ALT-TAB 和 [Wowhead](https://www.wowhead.com/) 搜索。

(如果你真的对猎杀一切感兴趣，我倾向于选择 RareScanner，因为每当插件检测到你附近有罕见的敌人或游戏中出现的任何东西时，你都会得到非常有用的标注和恼人的噪音。)

### 稀薄

如果你一直在用头敲打键盘，因为你想要的一个东西在 *WoW* 中拒绝放下——那些讨厌的骑士的缰绳——那么稀有度至少会让你回顾你杀死了特定的生物多少次，失败了多少次。呃，游戏里的 RNG 让你失望了多少次。

### SexyMap

给通常贴在屏幕右上角的小地图增加一点性感。有了 SexyMap，你会得到一个更好看的小地图，你可以用你的活动坐标自定义，调整你想要的图标显示方式，调整你的颜色，直到你的小地图看起来刚刚好。我试图让这听起来令人兴奋，但这个插件主要是为了稍微美化你的界面。考虑到你可能会花很多时间盯着你的小地图，为什么不呢？

### 世界任务追踪

装备你的角色的最好方法之一是在你解锁后，做游戏中的许多日常世界任务。抓取世界任务追踪器，这样你就可以很快看到有什么可用的，按奖励类型排序，这样你就不会浪费无数的时间去寻找那天你必须玩的那个小时左右的完美任务。这是冰冠*暗影之地*补丁前活动的必备装备。

### 邮政

如果你打算在游戏中发邮件——给你的其他角色、公会会员或者你的朋友——那就发邮件吧。它让在 WoW 中向其他人发送物品变得如此容易，以至于我真的很惊讶暴雪到现在还没有把它的功能直接整合到游戏中。设置你经常发送邮件的联系人，或者使用一个方便的鼠标悬停列表来发送你的其他角色、朋友和公会会员。
