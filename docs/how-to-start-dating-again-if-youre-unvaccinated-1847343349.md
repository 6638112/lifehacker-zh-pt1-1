# how to start dating again if youre unvaccinated 1847343349

> 原文：<https://lifehacker.com/how-to-start-dating-again-if-youre-unvaccinated-1847343349>

vax 后的 [放荡夏天](https://lifehacker.com/how-to-prepare-for-your-post-vax-slutty-summer-1846740170) 正在我们身边发生，但有些参与者实际上不是 vax 后的*。这种疫苗已经免费广泛供应了几个月，但仍有大约一半的人口没有接种新冠肺炎疫苗。一部分未接种疫苗的人仍然对约会感兴趣，如果你是其中的一部分，我们有一些关于如何在道德上做到这一点的提示。*

*(当然，我们建议接种疫苗，但你现在知道了。)*

**

## ***了解疫苗接种状态将成为某些人的障碍***

*上个月，约会应用 Bumble 在接受调查的用户中发布了关于 COVID 时代趋势的最新研究。大约 30%的美国受访者表示，他们不会与未接种疫苗的人约会或发生性关系。YouGov 在英国进行的一项民意调查发现，28%的受访约会者有同样的感受。*

*这大大减少了你的选择余地。*

*你可能会想，你应该试着哄骗一个匹配者和你一起出去，不管他们对你的疫苗状况持什么立场。首先，不意味着不。其次，Bumble 的研究团队还发现，55%的全球用户表示，他们现在不太愿意在他们想要和需要的潜在关系上妥协。因此，30%不愿意与接种疫苗的人一起外出的人比以往任何时候都更有可能坚持己见。*

*“我接种了疫苗，不，我不会和任何没有接种疫苗的人约会，”一位在纽约约会的 29 岁名叫达连的女子说。自从这个世界重新开放以来，她已经有过八次左右的初次约会，她说，“这是她经历过的最糟糕的约会”。她将此归因于所有良好的约会前景“在隔离期间受到阻碍”以及其他人的生疏。如果你去现场时没有接种疫苗，你就给约会者在那里遇到的已经很长的问题列表增加了一个问题。*

**

*每个人都有交易破坏者。应用程序上的一些人可能与你不匹配，因为你与他们的前任或父母同名。有些人可能会因为你的职位、发色、气质、爱好或其他谁知道的因素而不和你约会。作为交易破坏者的疫苗接种状态并不新奇，但你必须为此做好准备。*

*如果你在免费接种几个月后仍未接种疫苗，那么毫不夸张地说，你很可能会主动拒绝接种疫苗。无论如何，那些认为你缺乏刺拳是交易破坏者的人可能不是你的伴侣。*

## ***诚实告知您疫苗接种状态的潜在日期***

*如此多的调查受访者表示他们不会与未接种疫苗的人约会，很明显这是一个很多人都认真对待的立场。这里唯一合乎道德的选择是你要完全诚实地面对你的未纳税身份，即使这看起来有点奇怪，这取决于你有多激进。*

*幸运的是，今年春天，从 Tinder 到 OKCupid 再到 Hinge，许多约会应用程序与白宫合作，鼓励年轻人接种疫苗，这使得信息披露变得更加容易。例如，应用 Bumble 和 Tinder 为用户提供了一个机会，一旦他们受到攻击，就可以在他们的个人资料中添加徽章。Bumble 甚至有一个“COVID 偏好中心”，让用户陈述他们对现实生活或虚拟约会、社交距离和潜在伴侣疫苗状态的偏好，并为使用徽章的人推出 Spotlight 和 SuperSwipe 等高级功能的免费积分。*

**

*“对于那些试图约会却没有接种疫苗的人来说，也许应该坦诚一些。达连说:“对于大多数影响他人的事情，最好是诚实的，让人们选择是否要和你约会，”尽管她指出，她不会在约会档案中透露自己的疫苗接种情况。*

*“我认为通过一个人的个人资料很容易判断他们是否接种过疫苗，所以我不认为有必要在约会资料中添加这个徽章，但如果你这样做了，这很酷，”她说。"如果我不确定，我也会在见面之前的交谈中询问."*

*因此，对于阅读这篇文章的任何人来说，谁*接种了疫苗，信息是明确的:如果这对你很重要，问问别人你是否不确定他们的疫苗接种状态或个人安全协议。Tinder 聊天和第一次约会都是为了交流，对吧？你问他们以什么为生，在哪里长大，喜欢吃什么。这个也问一下。**

## ***也许能找到一些志同道合的人***

*当你在 Twitter 上搜索“未接种疫苗的约会”时，会出现大量的点击。未接种疫苗者谴责应用程序上的徽章，并公开希望有一个“未接种疫苗的约会应用程序”。*

**

*我们甚至发现了一个这样的应用，Unjected，它标榜自己是“一个支持医疗自主的志同道合的人的平台。”它拥有超过 23000 名 Instagram 粉丝 和一个商品系列，所以肯定有一个社区充满了和你想法一样的人。*

*(但我们仍然建议接种疫苗，以防我们还没有说清楚。)*
