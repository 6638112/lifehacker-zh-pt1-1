# 如果你用那个号码发短信，就安装谷歌语音应用

> 原文：<https://lifehacker.com/install-the-google-voice-app-if-you-use-that-number-for-1846453445>

我喜欢谷歌语音。有一天，谷歌认为它和谷歌照片一样值得付费，我可能真的会扣动扳机；有一个备用电话号码，你可以把它转移到你的真实电话号码上，从而隐藏真实电话号码，这非常有用。如果有一天谷歌语音消失了，我会非常难过。



谷歌语音很容易使用，你只需进入它的设置，选择选项将来电和/或短信转移到你的真实号码，谷歌正在做出改变，这意味着如果你想充分利用这项服务的功能，你现在必须在你的设备上下载应用程序。据最近 [Android Police](https://www.androidpolice.com/2021/03/09/google-voice-wont-forward-text-messages-to-outside-numbers-much-longer/) 报道，谷歌将很快取消从你的谷歌语音号码向你的真实电话号码转发短信的功能。

迈克尔·克莱德写道:

> 对于我们这些一直将谷歌语音作为唯一有效电话号码的人来说，这是一个特别坏的消息，当我们收到新的 sim 卡和设备时，会将其转发给它们。手机倾向于依赖其默认的短信应用程序来集成到许多其他服务中。谷歌语音应用程序通常会涵盖这些内容，但除了通知基本功能之外，它与 Android 的其他部分并不特别友好。

然而，在你指责谷歌做出这一改变之前，要知道这一次实际上不是谷歌的错。显然，许多运营商 [开始屏蔽这些转发的短信](https://support.google.com/voice/answer/9182115)——也许错误地认为它们是垃圾短信。我的假设是，谷歌宁愿有一个可管理的体验(在这种情况下，根本没有功能)，而不是不一致的体验。

因此，如果你想查看和回复发送到你的谷歌语音号码的短信，你可能需要安装谷歌语音应用程序( [、iOS](https://apps.apple.com/us/app/google-voice/id318698524) 、 [、Android](https://play.google.com/store/apps/details?id=com.google.android.apps.googlevoice&hl=en_US&gl=US) )。如果你出于某种原因感觉不到这个选项，你也可以进入谷歌语音的设置，把你收到的每条信息转发到你的电子邮件地址。然后你可以回复邮件来回复短信——这是一个更麻烦的方法，尤其是如果你的朋友很健谈，但也是一个选择。
