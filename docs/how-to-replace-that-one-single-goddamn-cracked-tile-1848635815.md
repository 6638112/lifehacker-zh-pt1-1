# 怎么换掉那块破瓷砖

> 原文：<https://lifehacker.com/how-to-replace-that-one-single-goddamn-cracked-tile-1848635815>

瓷砖是一种奇妙和多用途的地板和墙壁覆盖物。防水，手感凉爽，相对容易清洁和维护，如果安装得当，使用寿命长。然而，这并不意味着它不会被损坏，而且什么也没有——什么也没有！——比一块完美的瓷砖上的一块破裂的瓷砖更令人愤怒。如果裂纹是由愚蠢的事故造成的，比如工具掉落或不明智的室内高尔夫练习，愤怒会更强烈。即使你愿意忽略裂缝，你真的不应该——有裂缝的瓷砖可能会有锋利的边缘，可能会伤害裸露的脚和手，潮湿地区受损的瓷砖可能会让水侵入你的墙壁或底层地板。

但是，除非你强烈怀疑你的瓷砖开裂是由于结构问题或安装不当，你没有必要拆掉整面墙或地板来修复一个危险的罪魁祸首。只要你对工具有一点点的亲切感，你就应该能够换掉那块破坏你审美的该死的有裂缝的瓷砖。

## 慢慢来

忍住抓起锤子把瓷砖砸出墙外的冲动。是的，那会非常令人满意！直到你在兴高采烈中打碎了更多的瓷砖和灌浆线，然后意识到你也破坏了瓷砖下面的墙壁或薄的粘合物。为了创造，你必须毁灭，是的，但不是那样。

第二，确保你有替换瓷砖。你*从里诺*那里保存了一些瓷砖，是吗？如果没有，或者瓷砖工作比你早，你就必须寻找一个替换瓷砖。注意:即使你找到了相同品牌和颜色的瓷砖，不同地段的颜色和表面也会有很大差异。如果你现有的瓷砖已经有一段时间了，它可能会有均匀的磨损，这将使新的*非常*引人注目，这可能像损坏的瓷砖一样令人分心。

在做任何事情之前，请将您更换的瓷砖与现有的瓷砖进行比较。如果有必要的话，你也许可以从碎片中复制出你的瓷砖，但是这并不便宜，而且需要一点时间。但这可能是获得完美匹配的唯一方法，尤其是如果您的瓷砖已经停产。

一旦你手里有了瓷砖，拿一些材料:一把灌浆锯，油漆工用的胶带，一把钻子，一把锤子和凿子，还有一些粘合剂。

1.  **移除水泥浆。**用 [灰浆锯](https://www.homedepot.com/p/QEP-Handheld-Grout-Saw-with-2-Blades-for-Cleaning-Stripping-and-Removing-Grout-10012Q/100001353) 锯掉碎瓷砖边缘的灰浆。(也可以使用带有灌浆锯附件的多功能工具。)小心不要刮伤任何好的瓷砖——慢慢来，稳扎稳打。
2.  用胶带贴瓷砖。用油漆工的胶带覆盖破损瓷砖周围未受损的瓷砖。这将保护他们免受损害。
3.  **钻瓷砖。**使用合适的 1/4 英寸钻头(例如，用于瓷砖的陶瓷钻头、用于石材瓷砖的石材钻头)，在破碎的瓷砖碎片上钻孔，注意不要穿透基底。这样更容易把碎片弹出来。
4.  **凿子。**用锤子和冷凿子， [凿瓦，](https://www.homedepot.com/p/Milwaukee-1-7-8-in-Steel-SDS-Plus-SLEDGE-Steel-Tile-Chisel-48-62-6030/300728295) ，或者老老实实地用任何平刃工具(像螺丝刀)，凿出碎瓦。同样，慢慢来——你不想损坏任何周围的瓷砖。一旦瓷砖本身不见了，回去凿掉留下的稀释剂或胶粘剂，直到你有一个漂亮干净的基底暴露出来。
5.  **设置磁贴。**清理孔洞，涂上 thinset 或其他粘合剂，在替换瓷砖的背面涂上黄油，然后固定。一旦准备好了，涂上水泥浆，就大功告成了。

如果你工作认真，有多余的瓷砖，你的工作应该是看不见的。现在你要做的就是找一个新的地方来练习你的锤子杂耍。
