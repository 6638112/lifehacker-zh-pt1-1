# 分摊账单的最佳应用

> 原文：<https://lifehacker.com/use-these-apps-to-seamlessly-split-restaurant-bills-1849144659>

生活中最简单的乐趣之一就是和朋友出去吃饭。当然，那是在账单到来之前。现在所有的话题都必须转向餐桌上分享的土豆块和油炸泡菜，每个人应该付多少钱。你的心情是欢乐，而不是复杂的数学。在过去十年的许多伟大的技术进步中，有一些应用程序致力于解决这个问题——通过无缝地为你分摊账单。通过使用这些免费服务中的任何一种，你将再也不必烦琐地计算出谁为什么付了多少钱。



Splitwise 是一款致力于公平分配给那些 [“想要公平，但无法决定什么是公平”的人的应用程序](https://www.splitwise.com/about)split wise 的工作方式很简单:首先，你在自己和一起吃饭的人之间创建一个群。在添加所有的账单后，你可以将物品分配给特定的人。您甚至可以将某些项目按百分比分开——这个特性对于共享的开胃菜、甜点等非常有用。永远不要再为你没有吃的洋葱圈买单。之后，只需点击“结算按钮”，Splitwise 将要求您验证向任何将餐费记在他们卡上的人发送您那部分账单的 Venmo。就这么简单！

## [选项卡](https://www.tabapp.co/)

Tab 与标准账单拆分应用程序的不同之处在于，它可以让你上传收据的照片，然后自动将信息转换为应用程序中的单个项目(其他应用程序也提供这项服务，但通常是在高级版本中)。一旦 Tab 上传了您的收据，您只需点击属于您的项目，以及您欠部分捐款的项目(即，共享方)，即可得出个人的账单欠款总额。此外，Tab 将自动计算你的个人贡献税和小费，节省你做任何疲惫的餐巾纸数学。Tab 还与 Venmo 接口，允许您将自己的一份支付给任何用信用卡支付餐费的人。

## [摆平](https://settleup.io/)

setup 是一款账单拆分应用，因其基本界面以及与你的谷歌账户自动同步而受到好评。与其他账单拆分应用程序非常相似，setup 允许您将收据项目分配给个人，同时还可以根据需要将项目划分为百分比。定居在两个方面是独特的。第一，可以离线使用。第二，如果你不用美元支付，它提供实时汇率。出于这两个原因，如果你和朋友一起出国旅行，结算是分摊费用的最佳选择。所有这些都包含在该应用的免费版本中，只要准备好耐心看完一两个广告就可以了。

## [Splid](https://splid.app/english)

Splid 带来了其他账单拆分应用程序所没有的功能，即在使用其服务时，可以在 150 多种不同货币中进行选择。此外，Splid 拥有一个用户友好的界面，非常像 setup，也可以在线或离线使用。Splid 不需要帐户就可以使用，还允许将任何输入信息下载为 Excel 或 PDF 格式的费用摘要。Splid 与 Venmo 接口，一旦输入所有信息，就可以快速转移资金。
