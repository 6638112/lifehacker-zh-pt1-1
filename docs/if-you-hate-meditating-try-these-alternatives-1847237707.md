我的大脑一直在不停地鸣叫——失去双眼和深呼吸不会带来平静，更不用说任何接近幸福的事情了。相反，这是所有坏想法开始浮出水面的时候，引发了我只能想象的与涅槃完全相反的情况。

## 尝试“un 冥想”或主动冥想

今年早些时候，我和贝卡·威廉姆斯 坐在一起，体验她的练习，她称之为“T4考试”威廉姆斯是一名营养学家和营养学家，她根据自己的需求量身定制了自己的项目。她试图通过传统的正念冥想来克服头脑中的快速思维，但发现打破静态太过艰巨。

我没有，她开发了自己的程序，她建议与大麻搭配(尽管我们没有走那条路，因为这毕竟是为了工作)。在与威廉姆斯的一次快速通话中，她指示我闭上眼睛，双手放在膝盖上，快速而沉重地呼吸。这是我经历过的最活跃的冥想形式。坦白地说，这感觉就像是锻炼，这才是重点。

“我的作品提供了一种非常活跃的冥想形式，包括有力的呼吸模式、深沉的共鸣声、以及上身、手臂、手和手指的表情运动，”威廉姆斯说。实际上，这种方法旨在“尊重这样一个事实，即大脑经常疯狂地旋转，并试图放慢速度。”

这是对积极冥想的重复，并不是什么新概念。但是威廉姆斯对她的养生法进行了修改，使其与传统的正念环境相反，允许负面情绪出现在脑海中并被直接面对。

她告诉Lifehacker:

> 通过使用这种非常积极的冥想方法，冥想者能够消耗大量被压抑的能量，这有助于他/她放慢速度，让自己脚踏实地。此外，与安静的静坐冥想相反，他们被邀请实际上提出并专注于敲在他们头上最响的东西——压力和压倒、焦虑、自我怀疑、愤怒、绝望、等等。

最终，威廉姆斯认为她的方法允许从业者“见面，问候，并释放让我们的生活不稳定的精神噪音。”

## 其他可能对你有用的冥想方法

所有这些冥想的选择提供了不同的方法来达到同一个目标，那就是达到平静和满足。美妙之处在于，如果一个人不为你工作，另一个人*可能会为你工作*。

*   写日记:把你的想法写下来可以是一种宣泄，尤其是当你压抑某些情绪的时候。这里有各种不同种类的日志，你可以从中选择，而且 [不仅仅是写日记。](https://lifehacker.com/journaling-is-much-more-than-keeping-a-diary-1845986398)
*   **武术:** [武术训练](https://lifehacker.com/which-martial-art-is-right-for-you-1846189330) 至少在一种意义上，可能与冥想达到同样的效果:我 t让你牢牢地扎根于当下。你将测试你的身体，以及你的思想，也许在这个过程中你会了解到一些关于你自己的事情。
*   **清洁:**对，清洁可以冥想。那些恪守这种智慧的人敦促其他人 [真正探身进入清洁的触觉元素。感受泡沫，倾听吱吱声，也许会注意到身体释放出的一些紧张感。](https://www.wsj.com/articles/sick-of-cleaning-turn-it-into-meditation-11590571801)
*   **瑜伽**:瑜伽的冥想特质是众所周知的，它是一种 [增强力量的好方法。](https://lifehacker.com/can-you-make-yourself-like-yoga-1846609699) 与武术类似，不静下心来，很难掌握某些姿势。将瑜伽放松到你的日常生活中，或者一天两次，可以让你的大脑变得更加安静。
*   音乐:听舒缓的音乐可以让大脑安静下来，考虑到你可以让一首歌带你远离你的想法。洗个澡，点上几根蜡烛，放上一瓶能让气氛变得宁静的混合物。
