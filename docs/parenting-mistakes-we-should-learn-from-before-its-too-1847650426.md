# 据Reddit报道，我们应该从10个育儿错误中吸取教训，以免为时过晚

> 原文:[https://life hacker . com/parenting-missons-we-should-learn-from-that-it-too-1847650426](https://lifehacker.com/parenting-mistakes-we-should-learn-from-before-its-too-1847650426)

做父母已经快九年了，我确信这项工作的一半是从你不可避免会犯的所有错误中学习——并原谅自己。最近，Reddit上的一个帖子提出了这个问题:“父母...你觉得自己犯了哪些错误，别人可以从中吸取教训，以免为时过晚？”

虽然不是所有的育儿建议都值得采纳，但从别人的错误中吸取教训还是有好处的。以下是我们找到的10个最佳答案。

尽管有拼写错误(“grows”应该是“礼服”，我们大多数人称之为“连体衣”)，但这是给新父母的可靠建议。因为当你的宝宝半夜醒来，看到纯沙在背上跑，你最不想做的事情就是把连体衣拉起来盖在他们的头上。它只会加剧你面前的恶心的混乱。相反，家长们，不要把衣服铺到全身，把脏衣服卷下来。把*滚下来*。

任何坚决说“不”并忍受了几分钟纠缠的父母都知道，在孩子发脾气的时候改变主意并让步是在邀请魔鬼。当然，这会让他们在那一刻停止胡说八道*，*但正如这位Reddit用户指出的，你的孩子刚刚学到的教训是，如果他们跺脚和抱怨的时间足够长，他们会让你筋疲力尽。然后，这种行为可能会令人作呕地重复，直到你*重新教导*你的孩子“不就是不”，如果他们变得歇斯底里，你也不会屈服。

为你的孩子腾出时间可能是老生常谈，但值得重复。(特别是给*每个孩子*单独腾出时间。造成了多么大的行为差异啊*。)据说你只有在孩子12岁或13岁时才真正“拥有”他们，在这个时候，他们的世界从关注父母的投入转向关注朋友和家庭以外的社会世界。虽然我们拥有这些青春期前的天赋，在此期间他们向我们寻求安慰、友谊和建议，但我们需要明智地使用它们。在我们知道之前，他们会乞求去商场见他们的朋友，而不是在晚饭后玩疯狂的8。*

作为房子里默认的纪律约束者，这一个打击很大。除了让一方家长承担不公平的“坏警察”责任(他有时也想开心，该死！)，它可以使孩子在父母身边表现得更好，父母会直言不讳地表达他们的期望，并坚持不懈地承担不想要的行为的后果。这给孩子们留下了很大的空间来扮演“随和”的父母，而不是强硬的父母，从而导致紧张、怨恨和潜在的更糟糕的行为。

当你怀孕时，有些事情他们不会告诉你:很大一部分父母只是管理你自己的情绪和对所有刺激的反应。吵闹、依赖、情感不成熟的生物会引发许多下意识的坏习惯和低劣的应对机制，这些都被我们埋藏在光鲜的成人外表下。正如Reddit评论者所说，学会识别和有效管理我们自己的愤怒和不耐烦，而不是将它们发泄在我们的孩子身上，将为你的孩子节省多年的治疗时间。我们希望。

作为一个固执地自给自足、独立的人，我可以肯定地说，主要靠自己做婴儿/幼儿的事情是不合适的。作为一个全天候的喂食者、治疗者、情绪抚慰者、玩伴和房屋清洁工，睡眠剥夺、隔离和情绪调整是强烈的和不可抗拒的。抓住、接受、询问、寻找和支付你有幸获得的任何帮助。抛弃你“应该”能够做所有事情的错误观念。养育人类是艰难的，这并不意味着要在真空中完成。

新父母实际上不需要的花哨的婴儿用品清单很长。虽然婴儿用品行业(以及一些朋友和家人)会让你相信你的宝宝需要最新的符合人体工程学设计的、有节奏的、带有透气网眼的“防摇摆”蹦床，但你不相信。你可以从宜家这样的折扣店、Craigslist和脸书这样的在线二手市场、寄售店以及朋友和家人送的旧衣服中找到几乎所有你需要的东西。没必要在装备上花太多钱，反正他们几个月内就可以用光了。

我们努力计划并提供无尽的膳食；小孩子需要营养。但他们也很挑剔，拒不服从，有时，只是不饿。这些因素的结合会导致许多用餐时间的争斗。但是强迫孩子吃东西会导致他们更加抗拒，和父母产生消极的关系——随着年龄的增长，和食物也会产生消极的关系。试着放松下来，相信孩子们饿了就会吃东西，如果这是一个持续的担忧，在网上查找资源，看看如何让小孩子吃得更健康，减少挣扎。

毫无疑问:孩子是金钱的吸血虫。这条建议在要孩子之前，与我们的伴侣进行坦率、详细且可能不舒服的谈话，是一条极好的建议。弄清楚你是否同意孩子的花费——从尿布和钢琴课到返校服装和家庭度假——将如何进行预算和优先排序。

“关键在于把他们当成人，而不是你拥有的物品。”真实的话——我们还要加上一句，“而不是你能控制的东西。”把小孩子当作情感值得尊重的人来对待是前进的方向。当我们作为父母陷入困境时，重要的是向他们表明我们会犯错，并对我们造成的任何伤害感到抱歉。毕竟，我们正在培养最终的成年人，我们希望他们的生活中包含我们，对吗？

所有孩子都有情绪崩溃的时候。(我是说，我们不都是吗？)作为父母，我们的第一本能可能是对这种干扰感到尴尬，或者担心我们无法管理自己的孩子意味着什么——事实是，无论我们做什么，我们都无法控制人们对我们或我们的孩子的看法。所以保持冷静，挺过去，给你的孩子他们需要的东西，不要去理会消极的自我对话。我们都经历过。
