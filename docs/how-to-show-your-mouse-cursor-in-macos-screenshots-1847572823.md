# 如何在 macOS 屏幕截图中显示鼠标光标

> 原文：<https://lifehacker.com/how-to-show-your-mouse-cursor-in-macos-screenshots-1847572823>

有很多方法可以在你的 Mac 上拍摄美丽的截图，但是有一个陷阱——默认情况下，鼠标光标在你的所有截图中都是缺失的，这可能是出于某些目的的交易破坏者。有时你可能不想麻烦注释截图，例如，这样你就可以快速显示鼠标指针，告诉你爸爸点击哪里来修复他的计算机问题。

## 如何在 macOS 屏幕截图中包含鼠标指针

你需要做一点点定制来将鼠标光标添加到你的 Mac 屏幕截图上。为此，你需要找到你的截图选项，你可以通过使用**命令+ Shift + 5** 键盘快捷键来访问。

完成后，点击浮动屏幕截图选项窗格右侧的**选项**菜单。这将打开另一个菜单，其中最后一个子标题标有**选项**。这里选择**显示鼠标指针**。

就这样。从的时刻开始，你所有的 Mac 屏幕截图也会包含鼠标指针。继续前进，并尝试使用键盘快捷键来截图。要捕获整个屏幕(包括友好的鼠标光标)，请按 **Command + Shift + 3** 。要捕获屏幕的一部分，尝试使用 **Command + Shift + 4** ，然后点击并拖动手势。

当你终于开始在你的 Mac 屏幕截图中看到鼠标指针时，兴奋是很自然的，但你不想最终在你的桌面上到处都是随机的截图，所以当你完成截图时，一定要检查这些图像是否最终出现在你的 [专用截图文件夹](https://lifehacker.com/preview/how-to-change-the-default-save-location-for-screenshots-1847572508) 中。

最后，如果您选择隐藏屏幕截图的鼠标光标，您可以通过再次点击 **命令+ Shift + 5** ，选择 **选项**，并在**选项**子标题下，取消选中 **显示鼠标指针**来返回和反转。
