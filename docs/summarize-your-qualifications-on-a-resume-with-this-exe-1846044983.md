# 通过这个练习在简历上总结你的资历

> 原文：<https://lifehacker.com/summarize-your-qualifications-on-a-resume-with-this-exe-1846044983>

申请一份工作可能是一个艰难的过程。你必须使你的简历和求职信适合每个潜在的职位，以最符合所提供工作的方式描述你的能力。 [按照模板](https://www.resumehelp.com/resources/articles/preparation/use-resume-templates) 工作没问题，适当地交换细节，但确保雇主明白你能满足他们独特的需求，这对于获得面试机会至关重要。试图描述你的技能和经验很容易陷入混乱。这里有一个快速的提示来描述你的资历，证明你是这份工作的合适人选。



### 如何描述你以前的角色和职责

当你开始起草你的简历时，从 c ross 开始——用你自己的经验检查该领域专家的职责，以确定你的成就。首先，确保你清楚 T2、责任和成就之间的区别。责任是一个角色所要求的职责，而成就则详细描述了你履行职责的情况。你的简历应该不仅仅是列出你以前的职位；它必须展现你的成就，并突出你的资历。

[职业生涯](https://www.livecareer.com/resources/resumes/basics/accomplishments-rich-resume) 有一个有用的工具，可以从一般的角色和职责中推断出你的独特技能。使用表格格式，在一栏中列出角色和职责，在另一栏中填写你的成就。成绩应该列出具体的例子。例如，该示例将“计划的活动”显示为责任，将“为 300 人组织了一场满座的慈善活动并筹集了 125，000 美元”显示为成绩。前者是一个模糊的陈述，而后者指定了任务被执行的级别。这个写作练习有助于提升你的简历，最大化你获得梦想工作的机会。

### 清楚地解释你的成就

为了让练习更上一层楼，确保你的成就得到了清晰的解释，回答列出的每项职责的“何人、何事、何地、何时”问题。 [国际希腊大学职业指南](https://ecs.ihu.edu.gr/co/employment-cvcletter/how-write-achievements-your-cv) 建议任何列出的成就说明三个组成部分:

1.  使用特殊技能。
2.  进行一项特殊的活动。
3.  实现一个可测量的 / 可量化的结果或效益。

在回答“四个 w”、时，请记住，你希望阅读你简历的人在阅读完你的简历后已经掌握了你的每项成就的三个组成部分，而不需要你提供额外的信息。你的成就构成了你资历的核心。

### 收紧你的资历总结

一份强有力的简历还需要对资历进行简明扼要的概述，这里的“概述”意为简洁明了。正如 [确实说明了](https://www.indeed.com/career-advice/resumes-cover-letters/summary-of-qualifications) ，“资历总结的标准格式是一个要点列表。”你的资历不需要用完整的句子拼写出来；你应该喜欢快速交流你的技能。评审员可能会收到数百份每个职位的简历，你希望你的简历能让你一眼就看上去不错。

如果你在确定摘要中使用的关键词时有困难，仔细看看工作列表，或者简单地搜索一下有类似头衔的工作。使用你找到的角色和职责来编写你自己独特的要点。不要把别人的话当成自己的话。(那叫抄袭。相反，让它们成为描述你自身经历、技能和品质的路线图。
