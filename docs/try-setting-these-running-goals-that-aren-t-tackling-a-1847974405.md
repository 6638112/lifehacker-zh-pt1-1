# 试着设定这些跑步目标(不是跑马拉松)

> 原文:[https://life hacker . com/try-setting-this-running-goals-that-that-turking-a-1847974405](https://lifehacker.com/try-setting-these-running-goals-that-aren-t-tackling-a-1847974405)

你做到了。你跑过5公里，或者半程马拉松，或者甚至是臭名昭著的全程马拉松。恭喜你。因此...现在怎么办？

Watch

如果你是一个跑步者，经常感觉自然(或者唯一)要做的事情就是一步又一步地跑。但是有很多不同的前进方式。不同的比赛需要不同种类的训练；例如， [这是我们对5公里比赛和](https://lifehacker.com/running-goals-showdown-5k-races-vs-marathons-1783400936) 马拉松比赛的指导(阅读:摊牌)。此外，没有规定说进步只能通过一英里又一英里的前进来衡量，直到你穿过波士顿马拉松的终点线。

作为一名马拉松运动员，我可以证明很多理由**不**报名参加。这是一个了不起的时间吮吸。受伤的风险增加了(我自我诊断的足底筋膜炎现在是我的一部分)。你会让你的朋友和家人厌烦，因为你无法停止谈论你的马拉松，这是一次多么糟糕的T2，以及你是如何(自我诊断)患有足底筋膜炎的。

假设你知道你不想跑马拉松。这里有其他的比赛和目标来避免 [跑步者的车辙](https://lifehacker.com/stop-running-the-same-route-all-the-dang-time-1829060474) ，并让你的里程有更多的意义。

## **跑完你的最快一英里**

一英里比赛仍然被认为是一项耐力赛，但它将让你有机会专注于 [结合速度跑](https://lifehacker.com/how-to-add-speedwork-to-your-running-to-get-stronger-a-1695337461) 变得更强更快。你能想象告诉你的中学时代的自己，你不仅喜欢跑一英里，而且你会为此付钱吗？

对于这个距离和这里列出的所有其他距离，尝试使用 [比赛搜索网站，像这样的](https://race-find.com/us) 在你附近注册一个。

## **征服5公里(3.1英里)**

从你的新年决心到 [起床](https://lifehacker.com/all-the-questions-youll-have-when-you-start-couch-to-5k-1830857969) ，再到9个月后你当地的火鸡小跑，5公里是所有级别跑步者最受欢迎的距离之一。就像一英里赛跑一样，降低你的距离目标，转而提高速度有很多好处。如果你习惯于记录更长的距离，5公里可以为你提供带宽 [专注于速度训练](https://lifehacker.com/how-to-add-speedwork-to-your-running-to-get-stronger-a-1695337461) ，如果/当你转换回更长的距离时，这可能会转化为更快的时间。

## **巡游10公里(6.2英里)**

除了个性，我是一个令人印象深刻的普通跑步者。对我来说，10公里是最理想的比赛，尽可能保持舒适的状态。对我来说，6.2英里是跑得更远、更快的完美距离。

当然，你*可以*推动自己完成一英里或5公里这样的比赛，但如果你像我一样，这是一个从更长距离挖掘你的心理弹性的机会，而不会感觉到全程冲刺的压力。我会把**的十英里**也归入这一心理范畴(当然，只是多了几英里的成就)。

## **找一场奇数距离比赛**

听说过8k、15k或25k吗？非同寻常的长跑比赛的最终好处是，如果你以前从未参加过，这将是一个有保证的个人记录。报名参加一场更随机的比赛也可以让你和像你一样的跑步者并肩作战，这些古怪的人正在寻找新的东西，并且知道马拉松不是最终的目的。

## **完成障碍或主题比赛**

跑步对你来说还不够有挑战性？你在寻找传奇的东西，强烈的东西，一些东西...史诗？幸运的是， [Runner's World](https://www.runnersworld.com/training/g22144071/best-obstacle-course-races/) 最近更新了他们的障碍赛列表。如果你没有听说过像斯巴达赛跑或Tough Mudder这样的著名赛事，障碍赛跑就像它们听起来的那样——有单杠、泥坑和各种创造性路障的跑道。除了你会发现的新的力量和精神挑战，这些活动是一个让一群朋友聚在一起的好机会，无论你们都打算竞争或者只是尽力完成。

## **加入跑步俱乐部**

快讯:比赛不是动力的唯一来源。找一个志同道合的社区，结识新朋友，和一群跑步伙伴一起用新的眼光探索你的社区。你可以通过像Strava这样的跑步应用程序或通过脸书找到当地的跑步团体(如果 [现在人们还这么称呼它](https://www.cnn.com/2021/10/28/tech/facebook-mark-zuckerberg-keynote-announcements/index.html) )。

## **找到适合自己的跑步目标**

不要觉得有压力沿着一条直线跑道跑，在这条跑道上你会觉得你需要变得更快，走得更远，直到你成为破纪录的超级马拉松运动员。你根本不需要参加任何比赛就可以称自己为跑步者。我目前的目标是学习如何在不听音乐的情况下跑步，这被证明是一种意志坚强的行为，比我以前完成的任何距离都更具挑战性。

让您的里数对您有意义的方式是无限的。你是一名跑步者，因为你喜欢跑步——就这么简单。
