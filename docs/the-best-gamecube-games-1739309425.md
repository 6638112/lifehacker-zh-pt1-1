# 最佳GameCube游戏

> 原文:[https://kotaku.com/the-best-gamecube-games-1739309425](https://kotaku.com/the-best-gamecube-games-1739309425)

啊，游戏魔方。这可能是我有史以来最喜欢的游戏机。小小的圆盘，可爱的小把手，Wavebird...是的，我们一起度过了一些美好的时光 。但是我们最美好的时光是和*一起度过的，尤其是这些*游戏。

基于几个考虑，我列出了这个清单——这里有一些其他人的意见。首先，当然是我个人最喜欢的。但是，鉴于我们现在已经对GameCube有了一些了解，哪些是最*的GameCubey* 游戏。这类标题定义了(或代表了)一款游戏机，任天堂和它的朋友们用一些古怪的想法冒了很大的风险。

不过，在我们开始之前，请不要给我讲你的悲伤故事。如果你最喜欢的不在这里，请振作起来。仅仅因为我没有挖掘一个游戏——或者认为它不够好或者不够怪异，以至于不能列出一个名为最好的列表(抱歉，*混战*)——并不能让你自己对它的感觉在某种程度上无效！

 *[https://kotaku.com/embed/inset/iframe?id=youtube-list-SHBKAZNOh7E%2FPL5B8C42543FE53380&start=0](https://kotaku.com/embed/inset/iframe?id=youtube-list-SHBKAZNOh7E%2FPL5B8C42543FE53380&start=0)*

#### *大金刚:丛林节拍*

 *[https://kotaku.com/embed/inset/iframe?id=youtube-list-gfde1J3mVsc%2FPLDD3822599AE9B005&start=0](https://kotaku.com/embed/inset/iframe?id=youtube-list-gfde1J3mVsc%2FPLDD3822599AE9B005&start=0)* 

*所以，有一个GameCube音乐游戏叫做*大金刚*，它实际上带有这些巨大的塑料邦戈鼓，你可以啪的一声去玩。像*吉他英雄*，只...带着鼓。这很有趣，但另一个使用手鼓的游戏是GameCube。*丛林节拍*让玩家用邦戈鼓玩一个平台游戏，动作和攻击由...敲鼓鼓掌。*

*这真是太棒了。*

#### *超级猴子球2*

 *[https://kotaku.com/embed/inset/iframe?id=youtube-video-NChrKJZNP0c&start=0](https://kotaku.com/embed/inset/iframe?id=youtube-video-NChrKJZNP0c&start=0)* 

*为什么这个游戏在这里？我的意思是，这很好，但坐在这样一个臭名昭著的公司旁边？它有什么特别之处？*

*控制器。GameCube的控制器在他们的塑料外壳上切割了这个奇怪的八角形“指南”，这意味着不是允许拇指操纵杆不间断的360度移动，你最终会遇到8个方向的“点”，将操纵杆锁定到位。*

*在一个要求极其精确的游戏中，这些导游将一个令人沮丧的练习变成了一种禅宗般的视频游戏体验。*
