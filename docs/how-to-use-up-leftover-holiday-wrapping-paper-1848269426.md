# 如何用完剩余的节日包装纸

> 原文：<https://lifehacker.com/how-to-use-up-leftover-holiday-wrapping-paper-1848269426>

袜子挂好了，礼物包好了，然后再拆开，节日的送礼环节已经接近尾声。但是当你为这一年收拾东西的时候，你注意到你有一些剩余的包装纸。

Watch

也许你今年买得太多了，远远超出了你的需求。或者也许你有一箱你不想扔掉的零碎东西，但也不认为它值得留到明年。不管怎样，这里有一些 [用于剩余的包装纸](https://lifestyle.howstuffworks.com/crafts/holiday-crafts/uses-for-leftover-wrapping-paper.htm) 。

## 用它作为背景来装饰假日照片

你的一些假日照片(可能)很好，可以装框。因此，与其花钱买带垫子的昂贵相框，不如买一个稍大(但更便宜)的不带垫子的相框，并在照片后面使用剩余的包装纸。

## 制作五彩纸屑

不管出于什么原因，如果你经常需要纸屑——比如在给你讨厌的人运送物品时用纸屑——剩下的包装纸就可以派上用场了。简单的把它切成细条，然后把细条叠起来，切成小方块。

## 用作包装材料

不要用皱巴巴的报纸或牛皮纸，你也可以用剩余的包装纸作为包装材料，把东西寄给你不讨厌的人。为了更适合这个季节，用零碎的包装纸安全地包装好精致的饰品和其他装饰品，直到明年再次使用它们的时候。

## 制作一个有框的假日报价

类似于用剩余的包装纸作为背景来裱照片，你也可以用它来制作明年的 节日礼物或装饰品。这最适合浅色包装纸。只需将一张纸切成合适的大小，用你最喜爱/最清晰的笔迹，在纸上写下你最喜欢的节日名言，然后将它放入相框。它不必是任何广泛的东西；简单的“安慰&快乐”就足够了。
