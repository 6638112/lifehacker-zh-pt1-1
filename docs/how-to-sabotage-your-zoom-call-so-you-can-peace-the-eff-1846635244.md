# 如何破坏你的变焦电话，让你可以和平的影响了

> 原文：<https://lifehacker.com/how-to-sabotage-your-zoom-call-so-you-can-peace-the-eff-1846635244>

在一年的远程工作后，花几个小时在 Zoom 上参加无数本来可以是电子邮件的会议已经成为一种生活方式。这就是为什么你需要 [Zoom Escaper](https://zoomescaper.com/) ，一个插件，它可以让你“自我破坏”你的音频流，给你一个很好的借口来中断浪费时间的视频聊天，而不会让你对它们的不屑变得明显。

Zoom Escaper 的创造者承诺，该工具将“让其他人无法忍受你的存在(在会议中)”，在听到它提供的各种干扰选项后，我们相信了他。

以下是使用方法。

### 如何安装变焦擒纵器

首先，转到 [缩放逃逸器页面](https://zoomescaper.com/) ，点击启用麦克风按钮。然后 [下载并为你的操作系统安装 VB-CABLE](https://vb-audio.com/Cable/) 。完成后，刷新 Zoom Escaper 网站。

现在，您可以选择切换麦克风和输出。从输出下拉列表中选择 VB-Cable (Virtual ),点击开始，并在启动缩放会议时保持页面打开。

### 如何使用变焦擒纵器

为了让这个工具在缩放模式下工作，你需要调整一些设置。打开您的**缩放设置>音频**，选择 **VB-Cable** 作为您的麦克风。

选择下的**【低】**【抑制背景噪音】，打开“显示会议选项，打开麦克风原声”此处嵌套的所有其他选项都应取消选中。( [这段 YouTube 视频](https://www.youtube.com/watch?v=r_LhwsMAaAc) 带你了解整个过程。)

加入会议后，您可以从缩放逃生页面添加声音效果。打开你选择的效果(或者同时打开几个，如果你真的想激怒你的同伴)并调整他们的音量。注意*你*将听不到你那边的效果。

虽然一些背景音频选项——如“Bad Cconnection”和“ECHO”——是意料之中的，但其他的可能会向你的同事透露你的游戏。“M an W eeping" 和“U rination" 的噪音确实令人恼火，但如果你的目标是找到一个可信的理由体面地退出会议，那就没那么有用了。但是你也可以上传自己的效果——当 Zoom Escaper 运行时，该选项会出现在页面底部。

要关闭 Zoom Escaper，请按 Stop 按钮，在 Zoom 模式下，将您的麦克风切换回您通常使用的状态。
