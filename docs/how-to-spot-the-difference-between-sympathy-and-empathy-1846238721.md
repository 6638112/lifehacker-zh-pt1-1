# 如何辨别同情和移情的区别

> 原文：<https://lifehacker.com/how-to-spot-the-difference-between-sympathy-and-empathy-1846238721>

“同情”和“感同身受”这两个词经常互换使用，但是虽然两者都指一个人如何对另一个人的痛苦做出反应，但它们并不意味着同一件事，或者为你*或接受它们的人提供相同的经历。*



首先，下面是 [对](https://www.grammarly.com/blog/empathy-sympathy/) 关于移情和同情的历史、以及它们之间的联系的语法描述:

> 在这两个词中，*同理心*是英语中较新的词条。*同情*在十九世纪*感同身受*的第一份书面记录出现之前已经使用了近 300 年。你可能会注意到，这两个词都包含 *-pathy* ，这使得它们有点相似——它们共享同一个希腊语词根 *pathos* ，意思是“感觉”或“情感”，但也表示“痛苦”或“灾难”。虽然这两个词都涉及情感，但它们离同义词还差得很远。

### 什么是共情？

同理心是一种能够想象自己处于他人处境的能力，要么是因为你经历过类似的事情，要么是因为你能够深刻理解他们的感受，就好像你自己也有一样。感同身受就是创造一个与他人分享的经历。

同理心往往看起来像是一个真正倾听的人，听起来像是，“那一定很难，”和“你感觉如何？” [*一本关于移情*](https://akidsbookabout.com/products/a-kids-book-about-empathy) 的小人书(作者 Daron K. Roberts)用大人和小孩都能理解的方式解释道:

> 当你和某人一起经历艰难、悲伤或可怕的事情时。同理心意味着你倾听，不做判断，感同身受，提出问题。

换句话说，*感同身受*就是*体验*另一个人的情感*。*是和某人在一起的感觉*。*

### 什么是同情？

同情不是体验另一个人的情感，而是感受和表达你对他们的痛苦或不幸的关心、同情或悲伤。因此,“同情卡”被恰当地命名，因为它们提供了一种方式来表达你对他们经历的悲伤。

当有人同情时，他们会对其他人正在经历的事情感到难过，而没有完全与他们一起走完情感之旅。同情听起来像是，“对不起，”或者“那真的很糟糕。”

换句话说，*同情*就是*同情*另一个正在经历*斗争的*。*是对*某个人的感觉*。*
