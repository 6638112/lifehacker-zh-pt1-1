# 这些第三方插件让《赛博朋克 2077》变得可以忍受

> 原文：<https://lifehacker.com/these-third-party-mods-make-cyberpunk-2077-tolerable-1846052884>

假设你还没有举起双手， [要求退还](https://lifehacker.com/how-to-get-a-refund-for-cyberpunk-2077-1845881035) [*公爵核弹*](https://kotaku.com/the-problem-with-releasing-duke-nukem-forever-after-all-5812816) 的——比如游戏体验，也就是*【赛博朋克 2077】*，有很多 mod 你可以抓取，解决一些问题，我敢说，让它玩起来更有趣。至少，他们会帮你渡过难关，直到开发商 CD Projeckt Red 在接下来的几周内发布前两个 [*认错*](https://www.cyberpunk.net/en/news/37298/our-commitment) 更新——尽管即使这样也不会让游戏*像*一样酷，不会让游戏的视觉风格变成 [*边境*](#blands) *。*

### [赛博引擎调整](https://www.nexusmods.com/cyberpunk2077/mods/107)

正如其作者所写，“这个插件修复了一些*赛博朋克 2077* 的问题，并增加了一些功能。”这包括允许你的 AMD CPU 在游戏中使用它所有的内核，这对你的整体性能来说很重要；将行人完全排除在游戏之外；并让您可以选择禁用游戏的异步计算或抗锯齿功能，以提高旧系统的性能。

一如既往，我会给国防部一个尝试，看看它是否有助于你的设置，并删除它，如果它没有提高你的帧速率一个可测量的程度。

厌倦了在赛博朋克 2077 驾驶汽车的感觉？这个小调整以。ini 配置文件，可能对您有所帮助。我没怎么玩过这个游戏——或者说还没玩到驾驶部分——但如果这个版本的作者是正确的，那么任何喜欢摩托车的人可能都会想要这个:“例如，当你以 150 公里的时速行驶并使用手刹转弯时，草薙不会再滑入下一个境界。”

还有 [**驾驶它就像你偷了它——一个车辆处理和交通模块**](https://www.nexusmods.com/cyberpunk2077/mods/348) ，如果你想要另一种选择。显然，不要同时使用这两种方法，而是尝试看看哪一种更适合你。

### [边疆 2077](https://www.nexusmods.com/cyberpunk2077/mods/845?tab=description)

哦，见鬼，是的。这个重沙德模式——和上面的 HDR 模式有着相同的视觉效果——给你的游戏元素添加了一个可爱的边界，这给它一种非常明显的 T2 边界的感觉。如果我经常玩*赛博朋克 2077* (仍然在等待官方补丁！)，我会立刻使用它。

### [更好的控制菜单](https://www.nexusmods.com/cyberpunk2077/mods/102)

如果一个游戏允许你重新映射*一些*键盘命令，但不是全部，那是令人沮丧的。有很多很好的理由，从个人偏好到可访问性，为什么你可能需要完全定制你的控制来适应你的游戏体验。使用这个模块，你可以重新分配游戏中通常不允许使用的键盘命令。
