# 哪里有免费的 N95 口罩

> 原文：<https://lifehacker.com/where-to-get-free-n95-masks-1849347887>

与普遍看法相反，新冠肺炎疫情并没有结束。最重要的是， [BA.5，目前正在传播的变种](https://lifehacker.com/how-bad-is-the-ba-5-covid-variant-really-1849170170) 具有高度传染性，并且在(再)感染时往往会逃避我们对病毒的一些保护。(话虽如此，疫苗继续为严重疾病提供保护。)



如果你正在寻找一种方法来减少感染和不知不觉传播病毒的机会——并且你已经接种了疫苗——当你在公共场合时戴上一个合适的 N95 口罩是最好的方法。以下是一些免费的方法。

## 如何获得免费的 N95 口罩

您可能还记得，在新冠肺炎疫情的早期，公众被告知不要购买 N95 口罩，以便提供给医疗从业人员和急救人员。但现在情况不同了，N95 口罩随处可见。

事实上，现在可以通过联邦零售药店计划免费获得限量供应的 N95 口罩。尽管每人限带三个免费口罩，但在大多数东西的价格持续攀升的时候，这总比没有好。

目前，免费口罩在一些全国性零售商处有售，包括 CVS、 [、沃尔格林](https://www.walgreens.com/images/adaptive/pdf/covid/mask_stores.pdf) 、克罗格、Rite Aid、Publix、沃尔玛和山姆会员店，以及一些较小的独立药店。领取免费口罩时，无需购买其他产品。

在你附近找到免费口罩最简单的方法就是使用疾病控制和预防中心(CDC)网站上的 [这个工具](https://www.cdc.gov/coronavirus/2019-ncov/your-health/free-masks.html) 。只需输入你的邮政编码，你就会得到分发这些免费 N95 口罩的药店和其他零售商的名单。

请记住，在疾控中心的名单上并不保证零售商目前有免费口罩库存(但至少让你知道去哪里检查)。你可以提前打电话给商店，询问他们是否有货。如果你在当地找不到免费口罩，可以拨打 1-800-232-0233。
