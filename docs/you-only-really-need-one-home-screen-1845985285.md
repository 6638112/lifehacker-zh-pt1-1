# 你只需要一个主屏幕

> 原文：<https://lifehacker.com/you-only-really-need-one-home-screen-1845985285>

当我去年得到一部新的 iPhone 时，我期待着有一个清新、干净的主屏幕来重新开始——但是我有一大堆应用程序不想丢下，更不用说整理了。但后来我发现了 iOS 14 中的 [功能，可以让你编辑你的主屏幕](https://lifehacker.com/how-to-customize-your-iphone-home-screen-in-ios-14-1845450222) ，隐藏尽可能多的杂乱。

Watch

这几乎太容易了。当你拿起手机时，你会把你的一个主屏幕献给你真正想看的几个应用程序。然后，删除其他屏幕。

这些应用程序都还在，所以你并没有真正失去任何东西。他们只是眼不见心不烦。当然，安卓用户从一开始就可以这么做，但至少苹果最终允许自己的用户享受这种奢侈。

以下是如何在 iPhone 上减少你的主屏幕杂乱的方法:

1.  确保你的 iPhone 是 [运行最新版本的 iOS 14](https://lifehacker.com/how-to-update-your-iphone-and-all-your-apps-1843480748) 。
2.  长按任何应用程序图标，并选择“编辑主屏幕”
3.  将所有你最喜欢的应用程序移到主屏幕的一页上。
4.  点击屏幕底部的点，你会看到你所有的主屏幕。每一个下面都有一个勾号；取消选中除您的收藏夹页面之外的所有页面。

你仍然可以通过其他方式访问那些现在已经消失的应用。向上滑动(或双击旧款手机上的 home 键)以在您最近的应用程序之间切换。或者在主屏幕上向下滑动进入搜索栏，和输入你想要的应用名称的前几个字母。如果你一直向右滑动，你会看到一个屏幕，显示你可能想使用的应用，按类别分组，包括“最近添加的”

我把我唯一的主屏幕献给 [我实际上想要我的手机提醒我关于](https://lifehacker.com/hack-your-notification-badges-to-encourage-good-habits-1819139456) 的应用，或者我想要最容易访问的应用。这些包括我的相机、短信应用，以及一些健康和习惯相关的应用。如果我想浏览旧照片或查看我的工作邮件，我必须多花两秒钟找到合适的应用程序。因此，我的主屏幕代表了我想让*用我的手机来做的事情，而不是一个大杂烩，包含了所有在那里运行的应用程序。这样我更开心。*
