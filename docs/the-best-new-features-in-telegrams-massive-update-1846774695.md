# Telegram 大规模更新中的最佳新功能

> 原文：<https://lifehacker.com/the-best-new-features-in-telegrams-massive-update-1846774695>

Telegram 的一个重要新版本正在所有平台上推出。此次更新包括买家和卖家的新购物功能，更新的群组语音聊天工具，移动设备上的新手势控制等等。



有很多内容需要介绍，但在我们开始大的功能更新之前，我们应该先谈谈 Telegram 的新网络客户端。

### 新电报网络应用

是的，你没看错:网络客户。事实上有两个——Telegram Web K 和 Web Z。这两个版本都是独立的 Telegram Web 应用程序，可在移动和桌面上运行，在您的设备上占用的空间不到 400MB。与其他聊天应用程序的网络客户端不同，Telegram Web K 和 Z 允许您保持登录状态，而无需在附近保持手机同步。

在我们的快速实践中，这两个版本的 web 应用程序只是在视觉效果上有所不同:K 版的聊天屏幕略宽，菜单略紧凑，图标更具 3D“按钮式”设计；而相比之下，Z 版本的图标更扁平，调色板更不饱和。时间戳格式在某些情况下也不同:版本 Z 显示旧消息为“2020 年 11 月 30 日”，而版本 K 遵循“2020 年 11 月 30 日”的格式。

除此之外，这些应用程序都是一样的。它们的 UI/导航是相同的，并且支持相同的功能。这两个版本都可以让你玩 Telegram 的“动画贴纸、黑暗模式、聊天文件夹”和新的应用内支付选项(一分钟后会有更多介绍)。

Telegram 的新 web 应用程序现已推出: [Web K](https://webk.telegram.org/) 或 [Web Z](https://webz.telegram.org/) ，挑你的毒。

### Telegram 的“支付 2.0”有什么新功能

首先，Telegram 的应用内转账工具已经过彻底检查，对于通过 Telegram 销售商品的公司来说更加有用。Telegram 将这一更新称为“支付 2.0”。

这款应用现在支持另外八个第三方支付选项:Stripe、Payme、Click、Yandex。Money、LiqPay、Ecommpay、Sberbank 和 Tranzzo。(该应用还支持 Google Pay 和 Apple Pay。)

新的支付工具可以在任何聊天中使用，在结账时还有一个新的小费选项，因此买家可以在订单中添加额外的小费。

Telegram 不收取应用内交易的佣金，也不自行处理支付。相反，它依靠全球不同的支付提供商来处理和存储你的敏感信息。Telegram 和机器人的开发者都不能访问支付信息。

新的支付 2.0 选项在手机版 Telegram、 [桌面版](https://telegram.org/apps)2 . 7 . 2 和新推出的网络客户端中可用。该公司已经推出了一个 [演示频道](https://t.me/TestStore/2) ，在这个频道里你可以在想象中的产品上试用新功能，而你并没有真正付钱。想要在 Telegram 上出售商品的商家应访问 [机器人支付支持页面](https://core.telegram.org/bots/payments#supported-payment-providers) 开始。

### 在 Telegram 中安排语音聊天

Telegram 最近也推出了自己的俱乐部式音频聊天版本。该功能允许 Telegram 群组管理员主持实时聊天，所有群组成员都可以收听。

最初，语音聊天只能手动开始，但从 iOS 和 Android 的 7.7 版本开始，管理员现在可以设置语音聊天在特定的日期和时间开始，这应该会使规划和宣传这些实时对话更加容易。

以下是如何在 Telegram 中安排语音聊天:

#### **在 iOS 上**

1.  打开您的群或频道页面。
2.  轻触**语音聊天**
3.  选择**“安排语音聊天”**
4.  使用拨号盘设置语音聊天的播放时间，然后按“安排语音聊天”保存您的选择。

#### **在安卓系统上**

1.  转到您的或频道或群的页面。
2.  点击三点菜单按钮。
3.  选择“**开始语音聊天”**，然后选择**“安排语音聊天”**
4.  使用菜单选择聊天开始的日期和时间。
5.  点击“安排语音聊天”进行保存。

任何即将到来的语音聊天的大倒计时定时器显示在群组页面或频道的顶部。群组成员可以点击计时器，然后切换**“设置提醒”**，以便在直播开始时得到通知。

### Telegram 的应用有新的手势控制和外观

Telegram 版本还包括一些针对 Android 和 iOS 用户的较小升级，包括:

*   **语音聊天个人资料控制:**移动用户现在可以在语音聊天过程中点击另一个用户来查看他们的个人资料图片和简历。您还可以点击您自己的个人资料来编辑您的信息，而无需离开语音聊天。
*   **捏放图像:**现在，您可以在聊天视图中用手指“捏放”来放大照片，而不用在媒体视图中打开图像。
*   **视频播放器时移:**观看视频时，双击屏幕左侧倒退 10 秒，或者双击右侧快进 10 秒(也可以长按进行时间跳跃)。
*   Android 上的新动画: Android 用户在使用应用程序时会注意到新的、更流畅的界面动画。

Telegram 7.7 版本现已在 iOS 的 [苹果应用商店发布，而 Android 版本将在未来几天内在 Google Play](https://apps.apple.com/app/telegram-messenger/id686449807) 上发布【Telegram[称](https://telegram.org/blog/payments-2-0-scheduled-voice-chats) 由于谷歌方面的问题，Android 更新已略有延迟。然而，如果你不想等待，Android 用户可以直接从 Telegram 的网站 下载 7.7 版本，并在上下载到他们的设备上。
