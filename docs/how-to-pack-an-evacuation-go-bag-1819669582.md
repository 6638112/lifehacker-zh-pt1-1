# 如何为紧急疏散准备一个“旅行包”

> 原文:[https://life hacker . com/how-to-pack-a-evacuation-go-bag-1819669582](https://lifehacker.com/how-to-pack-an-evacuation-go-bag-1819669582)

如果你不是生活在一个容易发生自然灾害的地区，你可能会认为它永远不会发生在你身上——但不幸的是，事实并非如此。几年前，加州阿纳海姆圣安娜山山麓的一条小路上引发了一场大火。它移动得很快，通过新闻报道，我们看到它朝着我们沉睡的郊区邻居蜿蜒前进。到了晚上，离我们大约半英里的地方有强制疏散。

因为道路封闭，我和我丈夫花了两个小时才下班回家，一到那里，我们也需要疏散。(幸运的是，我的两个孩子和我妈妈在一起，远离大火。)我四处奔波，抢夺贵重物品和衣物。但后来我愣住了。我不知道我应该带什么。感伤的照片？艺术品？我环顾我的家，想知道我能承受失去什么。一个邻居提醒我收拾我们的重要文件，我记得戴上祖母留给我的古董首饰。但我是如此的恐慌，以至于我最后得到的只有装满不合身的衣服、护照和画的行李箱。(值得称赞的是，我没有忘记给猫打包。)

在过去几年里，加利福尼亚遭受火灾重创的许多人当中，我们非常幸运。这场特别的大火蔓延到了大约5000英亩，但是在大火开始后的半天，风力减弱了，消防队员能够将它控制在远离我们家的地方。

根据FEMA 的说法，疏散在美国相当普遍。除了火灾、地震和飓风等自然灾害之外，人们经常被告知由于交通和工业事故而离开家园。我们的峡谷大火促使我做好更多准备，以防灾难再次来袭。

以下是如何准备一个疏散“旅行包”,你可以在需要疏散时快速拿到。

### **选择一款包包**

每个家庭成员都应该有自己的。从一个背包或者一个有拉绳的尼龙露营者洗衣袋开始。

### **给你所有的房间和贵重物品拍照**

将这些照片存储在云服务器上，并备份到闪存驱动器上。

### **扫描并保存您所有的重要文件**

如果你有你信任的加密服务，你也可以把它们保存在云服务器上，或者保存在闪存盘上。

将装有照片和重要文件的闪存盘放在包里。这些文件应包括:

*   驾照

*   你的房契

*   您的意愿和/或信任

*   保险证明

*   医疗记录

*   护照

*   社会保障卡

*   出生证明

*   带有地址和电话号码的个人联系人列表

*   您孩子的免疫记录

*   你的宠物的疫苗接种和病史文件

### **添加必需品，帮助你度过几天**

这包括水(每人每天一加仑)、不易腐烂的食物、急救箱、手电筒、电池、衣服、尿布和宠物用品。Ready.gov有完整的 [应急包清单](https://www.ready.gov/build-a-kit) 可以下载。

### **创建疏散待办事项列表**

记下你在紧急情况下需要打包的物品:你的“旅行包”、宠物，以及一份你生活中不可或缺的贵重物品清单(珠宝、绘画、照片)。也列出你的行动项目，比如关掉公用设施，锁好你的房子。当你需要疏散时，列出疏散任务清单，核对每一项，然后离开。

除了“行囊”，这里还有其他重要的方法来为紧急灾难做准备

*   参加急救和心肺复苏课程。查阅当地的美国红十字会分会的信息。
*   购买并学会使用灭火器。
*   买一个防火防水的保险箱。一场大火后我查看了它们，发现它们并不像你想的那样。它们的价格从25美元到1000美元不等。
*   现在学习如何安全地关闭家中所有的公用设施——电、水、煤气。(FEMA有 [关闭](https://www.ready.gov/safety-skills) 公用设施的提示。)灾后天然气爆炸加剧了很多火灾。与你家中的所有人分享你的知识。
*   确保你的宠物植入了微芯片。

在像自然灾害这样的情况下，当一切都超出你的控制时，至少有几个包打包好准备出发会有所帮助。
