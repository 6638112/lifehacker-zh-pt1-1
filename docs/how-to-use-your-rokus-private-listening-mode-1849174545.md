# 如何使用Roku的“私人监听”模式

> 原文:[https://life hacker . com/how-to-use-your-rokus-private-listening-mode-1849174545](https://lifehacker.com/how-to-use-your-rokus-private-listening-mode-1849174545)

像我一样住在公寓里，我喜欢戴着耳机看电视。我设计一个环绕立体声震撼墙壁、地板和天花板的家庭影院并不容易(也不友好)，所以h 耳机是一个完美的妥协，它允许我把音量调到我想要的(或能忍受的)程度，同时在听力范围内不打扰任何人。

当然，这是另一个好处:如果你在你的Roku上和任何人在同一个空间看一个节目或电影，而那些人不想和你一起欣赏它——不管是因为他们在睡觉、工作，还是因为对你的内容来说太酷了——一副好的耳机只能让你听到娱乐。

根据您的设置，将声音输出到耳机可能简单或复杂。例如， n [Apple TV](https://lifehacker.com/12-clever-apple-tv-4k-settings-everyone-should-know-abo-1847766109) 可以让你快速将苹果耳机如 [AirPods](https://lifehacker.com/21-clever-airpods-pro-settings-everyone-should-be-using-1847590845) 或Beats与流媒体设备配对，获得无缝无线收听体验。然而，有线耳机可能没有这么好听(或者根本没有)。这就是Roku在这里称王的原因:该公司集成了一个功能，可以在流媒体播放时快速轻松地使用任何类型的耳机。

这一功能被称为私人监听，这要归功于Roku应用程序，它允许你从手机或平板电脑上的数字遥控器控制你的Roku。你所需要做的就是将一副耳机连接到你的手机上——有线或无线——，然后点击应用程序上的一个按钮，就可以私下观看你的故事。

## 如何在Roku上启用私人监听

如果你还没有，你需要将智能手机上的Roku应用程序与Roku流媒体设备配对。为此，下载用于 [iOS](https://apps.apple.com/us/app/roku-official-remote-control/id482066631) 或 [Android](https://play.google.com/store/apps/details?id=com.roku.remote&gl=US) 的应用程序，然后前往设备部分，该部分显示为电视图标。只要你的手机或平板电脑和你的 Roku使用相同的wifi连接，你的Roku应该会自动出现在这里。

C 选择遥控器图标，然后用耳机轻触该图标——t这是私人聆听按钮。一旦你这样做了，你的Roku将通过任何连接到你设备的耳机发送音频。

即使你使用的是有线耳机，声音仍然会通过无线方式从Roku 传到你的设备上。因此，如果事情没有完全同步，可能会有音频延迟。如果出现这种情况，轻按远程应用程序中的设置图标，然后选取“调整音频延迟”断开私人聆听，然后选取“开始”从这里开始，你需要使用你手机的摄像头来捕捉你的电视画面中的。点击“继续”，Roku将运行一个大约10秒钟的同步过程，这应该会让事情再次平稳运行。
