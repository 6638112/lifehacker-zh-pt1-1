# 戳我，我胖了

> 原文:[https://jezebel.com/jab-me-im-fat-1846346370](https://jezebel.com/jab-me-im-fat-1846346370)

疫苗在美国的推广一直不太理想，但最终，看起来像是在这个漫长黑暗的隧道的尽头有一盏昏暗的灯。随着更多疫苗的实施，全国各州正在开放资格要求，允许越来越多的人接种疫苗，以便最终公众能够再次社交和交往——不像我们曾经做过的那样，但也差不多了。在我居住的纽约州，我现在有资格接种疫苗，因为我的身体质量指数——一种用你的体重和身高来衡量身体大小的方法——它是**[过时的、种族主义的](https://www.huffpost.com/entry/bmi-scale-racist-health_l_5f15a8a8c5b6d14c336a43b0) ，也是令人尴尬的指标，这一次对我有利。**

**这个消息让我有一种短暂的解脱感，因为我知道每个接种疫苗的人都打破了一个可能的传播链，而且我们最终都将不得不这样做。现在刚好轮到我了。但是，围绕着为什么我有资格接种疫苗的那种挥之不去的负罪感被其他一些情况加剧了，这些情况会使我表面上不太值得或不值得现在接种疫苗。这种感觉，正如艾玛·斯佩克特在《时尚》杂志*中所探讨的[一样，是一种既内疚又羞耻的感觉，一种欠某人一个道歉的感觉，尽管规则明确规定现在是我的时间。我在家工作，一个人生活。我通常是低风险的，比餐馆员工、送货员和其他高风险群体更是如此，据我所知，所有这些人都很难获得预约。“我并不是说你接种疫苗不好，”有人对我说。“但如果我有机会的话，我不一定会做出同样的决定。”](https://www.vogue.com/article/covid-19-vaccine-bmi-qualifier)***

***反对这种想法感觉毫无意义——在疫情的这个节骨眼上，很难改变任何人的想法——但向公共卫生官员和他们的指导打手势却不是。正如我在电视上看到的公益广告告诉我的那样，每个有资格接种疫苗的人都应该在力所能及的时候接种疫苗。我可以接种疫苗，因为我知道这是一种狗屎的共病——在我生命的大部分时间里，这种共病让我对自己的身体感到大大小小的不舒服——但即使只有一秒钟，这种狗屎最终对我有利。那不是我的错。我不制定规则。然而，关于决定现在就做，而不是“等待”更多比我风险更高的人接种疫苗的谈话主旨，让我相信，我不知何故错过了一个道德制高点。***

***任何伪装成道德优越感的判断、愤怒或其他情绪实际上都应该指向我们政府的巨大失败，以及其忽视不成比例地影响黑人和拉丁裔社区的方式。身体质量指数，正如已经说过很多次的，是一派胡言。但它在名单上，我正在听当局的，如果我现在能得到疫苗，我会的。***

***根除与等待或不接种疫苗这一假定的道德游戏相关的负罪感并不容易，特别是当我有资格接种疫苗的原因是因为我的身体质量指数时——这种医学状况实质上是向世界宣布，我已经胖到足以让人们担心。尽管身体质量指数并不是衡量个人健康的一个好的或有效的指标，但许多人仍然认为如此，包括但不限于我的母亲，当我告诉她我正在接种疫苗时，她做出了我所期望的反应。*** 

***“你肥胖？”她关切地问道。“你在健身吗？你吃得健康吗？你注意体重了吗？”***

***她的关心听起来像是判断，但我意识到这是关心。尽管如此，我还是对她很简短，通过向她保证我很好，她不需要担心来减轻她的疑问。为自己辩护一个过时的医学术语，这是一个不准确的健康衡量标准，是一场我不想再忍受的战斗，尤其是来自我自己家庭成员的战斗，但我明白“爱”对每个人来说看起来都不一样。***

***这次谈话并没有让我生气，也没有让我失望，只是因为医学肥胖的耻辱让这个问题的责任完全落在了我的肩上。担心的医生对我的体重皱眉头，并尖锐地询问我是否做任何锻炼，这是我避免去看医生的部分原因，因为向医学专业人士解释我只是“大骨架”过一段时间后会变得很丢脸。身体积极性运动唯一可疑的名声是，我可以找到一种比基尼，看起来像适合我身材的瘦子的比基尼，有时，一个大码模特出现在《Vogue》的封面上。虽然我承认这是某种进步，但这只是表面现象。不管一个穿着舒适的14码衣服的女人出现在时尚杂志的封面上或大型广告活动中多少次，承认我的身体不是社会认可的标准的不适不会消失。***

***《纽约时报》*上一篇有用的专栏文章简明扼要地阐述了当你有资格接种疫苗时无需内疚的论点。简而言之:剂量需要进入武器，因为它们过期了。没有理由假设一个低风险、身体健全、健康的人得到的剂量会给其他风险更高的人，因为这不是事情发生的方式。****

****尽管我在过去的一年半时间里减掉了20多磅，但相对于我的身高来说，我的体重仍然是“肥胖”根据医学界的说法，对于我的身高来说，可接受的体重是一个我从未见过的数字。事实上，我的体重会在目前的基础上上下波动，每波动20磅。我的身体是我所有兄弟姐妹和我母亲中最大的，我母亲是一个跑10公里的小女人，只能勉强跳过5英尺。多年来，我一直在想，为什么我看起来像自己，而我的其他姐妹却不像。在一生中的大部分时间里，我都在关注这个问题，并说服自己，我可以改变我的身体自然想要的方式，我只是试图停止。我的身体从来没有对我有利过——除了现在。****

****一个标准化的疫苗推广，在全国范围内有资格和明确的分发路径，是一个美好的幻想，但现实是每个州都将自己决定谁有资格以及何时有资格**，**，尽管福奇博士和疾病预防控制中心的努力，各州之间的疫苗接种一致性并不存在。如果疫苗是通过符合先存健康状况的方式提供给你的，那就接种疫苗吧。群体免疫是目标，尽快让每个人接种疫苗是实现这一目标的必要条件。看起来，使用像身体质量指数这样可疑的资格所带来的部分虚假的担忧和羞愧来自于无数混蛋插队而不排队的故事。这不是越线；疫苗是为那些合格的人准备的，如果那恰巧是你，尽你的一份力量。****
