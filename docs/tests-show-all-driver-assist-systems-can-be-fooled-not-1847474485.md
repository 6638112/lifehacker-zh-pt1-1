# 测试显示，所有的驾驶辅助系统都可能被愚弄，不仅仅是特斯拉

> 原文:[https://jalopnik . com/tests-show-all-driver-assist-systems-can-but-not-1847474485](https://jalopnik.com/tests-show-all-driver-assist-systems-can-be-fooled-not-1847474485)

在我们在汽车和驾驶员 的朋友进行的一系列 [测试中，证明了几乎所有主要汽车制造商使用的](https://www.caranddriver.com/news/a37260363/driver-assist-systems-tested/?utm_medium=social-media&utm_campaign=socialflowTWCD&src=socialflowTW&utm_source=twitter) [Level 2](https://jalopnik.com/the-sae-autonomy-levels-are-confusing-but-i-think-i-hav-1846440294) 驾驶员辅助半自动系统能够被愚弄，允许驾驶员完全不注意，甚至离开驾驶座。当然，这是一个坏消息，如果不是非常令人惊讶的话，这确实是另一个提醒:所有2级驾驶员辅助系统都有固有的缺陷。

之前，我们主要报道了 [特斯拉的自动驾驶驾驶员监控系统(主要是方向盘上的扭矩传感器)如何击败](https://jalopnik.com/consumer-reports-shows-how-easy-it-is-to-use-tesla-auto-1846739330) ，以及各种 [的例子指出白痴](https://jalopnik.com/idiot-caught-not-driving-tesla-from-backseat-claims-to-1846891411) [利用](https://www.tesplus.com/model-3-y-x-s-steering-wheel-booster-fsd-counterweight-ring-autopilot.html?gclid=CjwKCAjwjdOIBhA_EiwAHz8xmziBNYr-CqLZQSauJRnADMT91Rqg0dqnoBvp9VSOlW6cg8y4NNxztBoCPnUQAvD_BwE) [的](https://jalopnik.com/the-autopilot-buddy-for-your-tesla-is-insidiously-dange-1826048861) 这些系统欺骗 在公共道路上高速行驶时保持沉默。到目前为止，特斯拉是这个问题最著名和最明显的例子，但它绝不是唯一的例子。

我们在 之前的文章中引用了机器人和人机自动化研究员 [米西·卡明斯的话，这不仅适用于特斯拉，也适用于任何使用方向盘扭矩传感器作为判断驾驶员是否注意的手段的汽车制造商:](https://jalopnik.com/listen-to-a-bunch-of-people-much-smarter-than-me-agree-1844719092)

> “要点是，驱动轮上的扭矩监控非常糟糕，根本不应该发生，应该被取缔...我们永远不应该用它来代替司机监控。”

任何依靠方向盘扭矩监控的系统都可能被方向盘上的重量击败。汽车和车手测试证明了这一点。更有趣的是，他们的测试还显示了有多少汽车懒得检查司机座位上是否有人:测试中的17辆汽车都允许这样做。

诚然，这比方向盘上的重量要多一点工作，但这并不是说*比*多得多的工作。虽然很滑稽，但仍然令人担忧。

认为这是特斯拉的某种免责是错误的，因为事实并非如此。这只是证明，整个行业在司机监控方面还有很长的路要走，而且，我认为更大的外卖是我多年来一直在反复强调的一点:[2级是垃圾](https://jalopnik.com/listen-to-a-bunch-of-people-much-smarter-than-me-agree-1844719092) 。

任何半自动系统，在不同的时间，完成大约80%或更多的驾驶任务，但仍然需要驾驶员保持警惕，监控情况，并准备在最少或没有通知的情况下接管，注定会失败和滥用。

当涉及到警戒任务时，人类就不是这样工作的， [和我们几十年前就知道的](https://journals.sagepub.com/doi/10.1080/17470214808416738) 。此外，各种汽车制造商用来处理无反应司机的解决方案是可悲的，甚至是危险的不足。

目前的系统要么切断所有的自动化功能，这可能会带来更多的问题， [，要么在](https://jalopnik.com/mercedes-benz-claims-to-have-level-3-automated-driving-1847363146) 的任何车道上减速停车，这也可能是极其危险的。

是的，司机监控系统并不全面，但事实是，花费在试图开发一个防呆和防篡改系统上的资源真的会更好地用于推动至少3级系统，该系统至少可以在检测到疏忽的司机时，让汽车离开活跃的车道，并尽快停在某个地方。

这是一个很难解决的问题，但除非我们都认为这些容易被击败的驾驶员监控系统在某种程度上是好的(同样，它们不是)，否则这真的是唯一合理的途径。我们必须停止期望人类既不开车又同时开车。那是行不通的。

所以，特斯拉-斯坦斯，如果当白痴打败了不完善的司机监控系统时，你会对所有的负面关注感觉更好，那么享受你没有——真的，从来没有——一个人吸的事实。

所有这些2级系统都很烂。享受这种团结，也许我们可以把它作为一种开始谈论建设更好的东西的方式。
