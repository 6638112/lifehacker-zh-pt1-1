# 热浪来袭时，随身携带这个，给自己(和他人)留下深刻印象

> 原文：<https://lifehacker.com/impress-yourself-and-others-by-always-having-this-wit-1849314408>

我上过一次植物学课，其中包括一次实地考察:在炎热的夏天，绕着一片野花草地走上几个小时。没有树荫，没有微风，只有闷热的痛苦。但是当我和我的拼车伙伴回到车上时，我有一个惊喜:我已经用冰袋打包了一个保温饭盒，还有几瓶水。即使在炎热的车里呆了一下午，水还是冰冷的。这可能是我一生中喝过的最提神的饮料，我的伴侣也同样高兴。我觉得自己非常非常聪明。



食谱很简单:你把水瓶(或者你选择的冷饮)放进午餐盒，用冰袋包住它们。一个小冰袋适合短途旅行，但是如果你要去很长时间，不要害怕双倍或三倍。

我发现，当谈到让自己和他人愉悦和提神时，这种技巧远远超出了它的重量。我在独自外出时为自己打包了冷饮，完全忘记了它们，因为热量使我的大脑嘶嘶作响，然后在我回来时重新发现了冰镇的茶点。

我确信，因为在今年夏天向我的儿子们介绍了这个想法，我获得了年度妈妈奖。他们一直在上击剑课，随着天气变得越来越暖和，他们每次从课上出来都汗流浃背，口渴难忍。在第一个大热天，他们喝完自己的水后，感激地分享了几口我自己水瓶里的温水。我知道我必须做什么。一年中第二个*闷热的上课日，我偷偷装了冰水。当他们爬进车里时，我把冷饮递了回去，这让他们非常高兴。*
