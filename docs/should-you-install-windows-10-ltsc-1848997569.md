# 你应该安装 Windows 10 LTSC 吗？

> 原文：<https://lifehacker.com/should-you-install-windows-10-ltsc-1848997569>

对于大多数人来说，Windows 是事实上的 PC 操作系统，但这并不意味着它是完美的。Windows 10(和 Windows 11)预装了大量你可能不想要的微软应用程序，如 Edge 和 Windows Media Player，并且默认情况下会打开某些数据跟踪设置。它也因错误和损坏的更新而臭名昭著，这些更新会使你的电脑崩溃。



大多数人认为唯一的选择是 Linux，或者会跳到苹果的 Mac 电脑上，但实际上有一个名为 Windows 10 Enterprise LTSC 的 Windows 10 版本，它没有该操作系统的许多最大缺点，但完全稳定，支持所有正常的 Windows 应用程序和进程。

LTSC 代表“长期服务渠道”，是基于 Windows 10 特定版本的精简企业版操作系统。没有像预装应用程序、Cortana 助手这样的额外功能，也没有像微软去年添加的“新闻和兴趣”任务栏标签这样的无关功能(不过，它削减的功能取决于你安装的 LTSC 版本)。缺少额外的功能和应用程序会给你留下更多的硬盘空间和系统资源，因此你的电脑(理论上)会运行得更好。它还使您免受 Windows 功能更新的破坏。

## Windows 10 LTSC 有什么问题？

到目前为止，LTSC 听起来相当不错，但也有一些警告。首先，除非你已经有一个有效的 Windows 10 企业 LTSC 产品密钥，否则你需要从微软购买一个来验证你的安装。

反对使用 LTSC 的另一个主要理由是它不会收到 Windows 的重大更新；您安装的版本就是您坚持使用的版本。但是，正如这个 Reddit 帖子指出的 ，这就是全部要点:你跳过了新功能，而选择了完全稳定的操作系统。

然而，如果你*确实*想要升级到新的 LTSC 版本(或者换回 Windows 10 或 11 的家庭版)，你必须执行完全重新安装——并且每次想要升级时都要购买新的密钥。幸运的是，LTSC 版本仍然可以获得最新的 Windows 10 安全更新和漏洞修复，所以安全不会成为问题。也就是说，2021 年后发布的 LTSC 版本只能获得五年的发布后安全更新支持，而其他版本则可以获得 10 年的支持。

虽然在技术上不是为普通用户设计的，但只要你不介意没有最新的功能，LTSC 版本在你每天使用的电脑上也能很好地工作。你仍然可以连接到互联网，安装第三方应用程序，玩电脑游戏，以及你可能使用电脑进行的几乎所有其他事情——尽管要让某些任务和应用程序正常工作需要一些工作。

你需要安装一些应用程序，如媒体播放器(如 VLC 播放器)、文字处理器和互联网浏览器，以弥补 Windows Media Player、Microsoft Office、Edge 和其他应用程序的不足(注意，Windows 10 LTSC 版确实包括 Internet Explorer，但微软不再支持它，所以你需要安装一个不同的浏览器)。LTSC 也不包括 Windows Store 应用程序，但如果你需要，可以下载并安装。在某些情况下，您还必须手动更新设备驱动程序。

## 如何安装 Windows 10 企业版 LTSC

如果你想亲自试用 Windows 10 企业 LTSC，你可以很容易地免费获得 90 天的评估版，但不要忘记，一旦评估期结束，你需要从微软购买有效的产品密钥来激活你的操作系统。

下面是设置它的方法:

1.  进入 [微软评估中心页面](https://www.microsoft.com/en-us/evalcenter/download-windows-10-enterprise) 。
2.  选择您所在地区/语言的 64 位 LTSC 版本开始下载。
3.  下载完成后，使用 ISO 文件创建一个可启动的 USB 安装程序，并使用它来安装 Windows 10 LTSC。如果你不知道怎么做，我们有一个指南 带你完成整个过程——确保你使用 LTSC ISO。
