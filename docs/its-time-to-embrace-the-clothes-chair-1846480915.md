# 是时候拥抱“衣服椅”了

> 原文：<https://lifehacker.com/its-time-to-embrace-the-clothes-chair-1846480915>

每所房子都至少有一把没人能坐的椅子，因为上面堆满了衣服。眼中钉 撇开令人畏惧的“衣服椅”不谈，它完全无害——然而，对许多人来说，它是一个巨大的耻辱来源。

这必须改变。

晾衣椅一点也不丢人，如果运用得当，它能让你的生活变得更轻松。

正如每个人在过去的一年中所了解到的，说得好听一点，在家工作有一种简化你日常时尚的方式 。衣服椅就是为这一刻而造的。当你的衣柜仅限于商务运动服和夜间运动服时，让两套衣服都在伸手可及的范围内是非常方便的。不用翻抽屉或衣柜，你可以从晾衣椅上拿些东西就走。就我个人而言，这是我的救命稻草，因为我的 110 岁的壁橱门不锁；如果我想从中得到任何东西，我必须移动并更换把门关着的壶铃。这不是一个不可能完成的任务，但这比从椅子背上拿东西要烦人得多。

你不必在家工作也能欣赏到合适的晾衣椅。对于不够干净的衣服来说，这是一个完美的中间位置，可以和刚洗过的朋友一起放回抽屉，但 [也不够脏，不能放入](https://lifehacker.com/where-to-put-clothes-that-are-not-quite-dirty-and-not-q-1820842469) 的篮子。你在两次洗涤之间穿过多次的任何东西——比如牛仔裤、运动衫、 [工装裤](https://lifehacker.com/i-bought-a-pair-of-overalls-and-now-i-will-never-wear-a-1795306121) 、睡裤和长袍——都应该放在椅子上。当你第二天早上穿衣服的时候，你的衣服已经有一半准备好了。我通常穿牛仔裤，搭配 t 恤、球衣和夹克，所以这把椅子非常适合我的需求。如果你的服装遵循相似的模式，它也会适合你。

如同任何一个包罗万象的降落区，衣服椅会很快陷入混乱。为了避免混乱的灾难，注意你实际穿的衣服，把其他的东西都放好。这样做的目的是准备好必备的装备，而不是为脏衣服创造一个墓地。如果你注意到它滑入了墓地，那就是你该编辑的信号了。
