# 你应该喝一杯老式的熏茶

> 原文:[https://life hacker . com/you-worth-a-smoked-tea-老式-1845327205](https://lifehacker.com/you-deserve-a-smoked-tea-old-fashioned-1845327205)

秋天是一个主题鲜明的季节。当然，有些节日也会有同样多的主题，但是食物和饮料的秋天化会持续几个月，秋天主题的味道往往会很甜。

我不是一个在推特上抱怨南瓜香料咖啡饮料的人(因为我不想让人们认为我老了)，但南瓜鸡尾酒对我来说有点多。我可以进入苹果，肉桂也不是不受欢迎，但它们都有点太依赖了。但是抽烟？这种味道仅仅暗示了这个季节。

谈论吸烟是一件好事感觉有点笨拙，但直到最近，我真的很喜欢“烟味”作为秋天的味道。它让我想起(可控的、安全的)火、落叶和舒适的毛衣。很好看！如果你像我一样，仍然有点害怕火焰，你可以让你的鸡尾酒在没有任何火焰的情况下有一个很好的烟熏特征——你只需要一些 [烟熏茶糖浆](https://lifehacker.com/how-to-make-a-smoked-cocktail-without-any-fire-1845135004) 。

这种糖浆是由lapsang sou chong——“熏茶”——一种传统上在松木上烟熏干燥的中国红茶制成的。烟真的透了出来，当与大量的糖结合时，你会得到一种美妙的、闷烧的鸡尾酒糖浆。

你可以把糖浆加到各种各样的鸡尾酒里，但是我真的，真的喜欢老式的。你不需要很多——只要一茶匙。它给酒带来恰到好处的甜味，而不会影响你正在享用的波旁威士忌。要制作传统的熏茶，你需要:

*   1 1/2 盎司波旁威士忌
*   1茶匙熏茶糖浆(配方 [此处](https://lifehacker.com/how-to-make-a-smoked-cocktail-without-any-fire-1845135004) )
*   两个破折号安古斯图拉苦酒

将所有三种成分放入一个低球中搅拌。加入一个大冰块，搅拌几次，如果你喜欢的话，用一条柠檬皮装饰，确保在把柠檬皮放入玻璃杯之前，从柠檬皮中榨出油来。

*更新于2021年2月17日，更正应使用的波旁酒的量。*
