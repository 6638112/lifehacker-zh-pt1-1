# 如何在Disney+上设置新的家长控制

> 原文:[https://life hacker . com/how-to-set-new-parental-controls-on-Disney-1846386999](https://lifehacker.com/how-to-set-new-parental-controls-on-disney-1846386999)

为了防止年幼的孩子无意中接触到成熟的内容——无论是他们自己的个人资料，还是父母或哥哥姐姐的个人资料——迪士尼+宣布 [增加了家长控制](https://help.disneyplus.com/csp?id=csp_article_content&sys_kb_id=9582efcbdb1aac50055ceadc13961917#profilecreation) 。下面是设置它们的方法。

Watch

要调整家长控制，首先在Disney+网站或移动应用程序上点击您想要编辑的档案，然后滚动到“家长控制”

从那里，你可以选择将他们的帐户指定为“ [孩子档案](https://help.disneyplus.com/csp?id=csp_article_content&sys_kb_id=a6e305dfdbabc49860f3eacb139619e3) ”，这是一个以前可用的功能，主要管理来自分级为 [TV-Y7-FV](https://rating-system.fandom.com/wiki/TV-Y7#:~:text=Edit%20page,programs%20contain%20some%20fantasy%20violence.) 的电视节目和美国分级为G的电影(或美国以外的同等分级)的内容。

或者，您现在可以从七个“内容分级”中选择一个，这将从最受限制的TV-Y分级到TV-14分级中过滤内容。但是，您不能将该档案指定为“儿童档案”*和*设置特定的内容分级——您必须选择一种过滤方法。

在家长控制部分，您现在还可以添加一个四位数PIN码来限制孩子访问您的个人资料或可能能够观看更成熟内容的哥哥或姐姐的个人资料。
