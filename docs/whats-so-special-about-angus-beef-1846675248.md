# 安格斯牛肉有什么特别之处？

> 原文:[https://life hacker . com/whats-so-special-about-Angus-beef-1846675248](https://lifehacker.com/whats-so-special-about-angus-beef-1846675248)

“安格斯”这个词让人联想到优质美国牛肉的精神，餐馆和杂货店都用它来推销他们的产品。当我还是个孩子的时候，我最喜欢的食物是牛肉，我最喜欢的“成人”餐馆是黑安格斯，一家中等规模的牛排连锁店，提供美味的牛排汤和甜“黑糖蜜面包”，这两种食物我都非常喜欢。黑安格斯菜单教会了我“三分熟”是什么意思，我将永远把它与鲜红的牛肉联系在一起。

我对安格斯的另一个记忆集中在每次我们碰巧经过一群牛时，我父亲在车里进行的牛突击测验上。考虑到它是美国最常见的肉牛，“安格斯”通常是正确的答案。这也是最容易发现的:这些相当小的黑色肉牛是独一无二的，如果不是视觉上的非凡。(想看卓越的牛？看看 [婆罗门](https://www.google.com/search?q=brahman&tbm=isch&ved=2ahUKEwiS-OCY4_vvAhXTh54KHaHTDWQQ2-cCegQIABAA&oq=brahman&gs_lcp=CgNpbWcQAzIHCAAQsQMQQzIHCAAQsQMQQzIECAAQQzIFCAAQsQMyBAgAEEMyBwgAELEDEEMyAggAMgUIABCxAzICCAAyBQgAELEDUJ6eJFiboSRgsKokaABwAHgAgAFBiAH6AZIBATSYAQCgAQGqAQtnd3Mtd2l6LWltZ8ABAQ&sclient=img&ei=yNh1YJL0GdOP-gShp7egBg&bih=702&biw=1418) ，或者 [布兰格斯](https://www.google.com/search?q=brangus&source=lnms&tbm=isch&sa=X&ved=2ahUKEwjZhumP4_vvAhVF6Z4KHXMHBdIQ_AUoAXoECAEQAw&biw=1418&bih=702) ，这是你把安格斯和婆罗门杂交得到的。)还有一只红色的安格斯牛，但它不太常见，与它的黑色皮毛表亲分开登记。

我要说的是，安格斯牛只是一种牛，但这并不能阻止食品销售商——比如麦当劳——在汉堡上打上“安格斯”这个词，让它听起来更强壮、更美味、更“优质”虽然我很想知道我吃的是哪种动物，但“安格斯”这个词与肉的质量没有关系。为此，你需要看看美国农业部的质量等级。(美国安格斯协会也有自己的认证体系，包括“ [10项质量规格](https://www.certifiedangusbeef.com/brand/specs.php) ”，但这并不妨碍杂货店和餐馆在牛排和汉堡上打上“安格斯”字样。)

这并不是说牛本身不特别。虽然该品种起源于苏格兰，但黑安格斯是美国最受欢迎的肉牛，这是有充分理由的:如果美国安格斯协会 相信的话，“安格斯品种在大理石花纹方面优于所有其他主流牛肉品种。”这正是致力于某一特定品种牛的协会*会*对他们喜爱的牛说的话，但即使是 [百科全书布列塔尼卡](https://www.britannica.com/animal/Angus-breed-of-cattle) 也指出，这种牛拥有“紧凑而低矮的身体，肉质优良，屠宰率高”，而且“安格斯是最高等级的牛肉品种，多年来，纯种或杂交安格斯牛在英国和美国的领先脂肪库存展会上占据了很高的荣誉。”小黑牛也很耐寒，因为它们是为了抵御苏格兰寒冷的冬天而被饲养的。

但是，仅仅因为某一特定品种的牛能够产出嫩滑的大理石花纹的肉，并不意味着每个饲养黑安格斯牛的生产者或农民都能做到这一点。年龄、饮食和生活条件都会影响牛排的嫩度和肌内脂肪含量，不同的安格斯牛在饲养、放牧和加工过程中受到不同程度的关注。所以，如果你下次在杂货店看到“经理特餐”吹嘘其“安格斯”地位，更不用说一家廉价汉堡连锁店宣传其“安格斯”汉堡，不要被愚弄了——肉可能*确实*来自安格斯牛，但美国农业部的等级在预测那头牛的味道时会更有帮助。
