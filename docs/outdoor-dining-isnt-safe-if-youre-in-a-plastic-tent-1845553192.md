# 如果你在一个塑料帐篷里，户外用餐是不安全的

> 原文：<https://lifehacker.com/outdoor-dining-isnt-safe-if-youre-in-a-plastic-tent-1845553192>

为了在疫情生存，餐馆不得不变得有创意。以外卖为中心的经营和户外用餐让我们在这漫长的几个月里一直吃得很好。但随着天气变冷，一些餐馆现在在封闭的塑料帐篷、温室或泡泡屋内为顾客提供“户外”服务。我不想告诉大家，但那不是在户外。

### 为什么室外用餐首先更好？

新冠肺炎主要通过呼吸道飞沫传播。这些液滴大多在离开你的嘴或鼻子后不久就落到地上；这就是六英尺法则的原因。但是漂浮在空气中的小水滴——气溶胶——也可以传播病毒，并可能在超级传播事件中发挥作用。

户外用餐在几个方面扰乱了传播:

*   这里通风良好:微风将空气中的飞沫带走。
*   液滴彼此扩散很远并被稀释。
*   阳光可能有助于杀死冠状病毒，尤其是在表面。
*   户外环境通常使距离更容易实现。

将一家餐馆的室内座位与它的人行道上的桌子相比较。你不是被困在室内，与许多其他人的呼吸飞沫在一起，而是在一个飞沫随风飘散、阳光照射剩余飞沫的地方，理想情况下，你会离邻桌足足六英尺或更远。

### 帐篷几乎解决不了这些问题

如果你在帐篷里呢？嗯，你和你旁边的人之间有一个障碍。这是唯一的优势。不过，不利的一面也很明显:没有微风，也没有空气稀释。如果桌子比没有帐篷时靠得更近，那么另一个因素就是当你进出时，你可能会遇到更多的人。

虽然疾病控制和预防中心没有特别强调帐篷，但他们列出了一些餐馆 的指导原则。其中包括:“确保通风系统正常运行，尽可能增加室外空气的流通，例如打开窗户和门，优先考虑室外座位。”还有:“不鼓励分享难以清洗、消毒或杀菌的物品。”

封闭式帐篷通风不好。也不容易清洗。客户和服务员多久碰一次可爱小温室的门？比起挤在餐厅的室内餐厅里，住在这些小帐篷里可能更好，但这并不能说明什么。

### 你并不孤单

我听过一种为用餐帐篷辩护的说法，相当于:嘿，如果你和家人或密友在帐篷里，这些人就是你已经决定愿意冒险与之接触的人。

但你在帐篷里从来都不是真正的孤独，对吗？你的服务器必须进出。他们也在为所有其他帐篷里的人们服务。即使这种情况对你来说是安全的，但对那些不得不呼吸多个家庭闷热、充满水滴的空气，而他们只是在努力谋生的人来说就不安全了。

你也不是不可能吸入前一批用餐者留下的飞沫。如果帐篷或温室是封闭的，你认为那些水滴去了哪里？有些可能掉在地板、桌子或其他表面上；许多可能仍然漂浮在空中。(如果帐篷有两面或两面以上的开口，在顾客之间展开，那会让它更安全一点，但我还是不会在其中一面吃东西。)

### 就叫外卖吧

听着，我知道适应新环境很难，我们都希望有一点点正常的味道，比如和朋友出去吃饭。但你仍然可以和别人一起吃饭，支持你的当地餐馆，而不会冻得半死:只需叫外卖，给小费，然后在家吃。
