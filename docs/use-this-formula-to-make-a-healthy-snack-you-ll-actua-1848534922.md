# 使用这个公式来制作你真正想吃的“健康”零食

> 原文：<https://lifehacker.com/use-this-formula-to-make-a-healthy-snack-you-ll-actua-1848534922>

二月已经过了一半，在这个月里，决心被削弱，决心被放弃。即使你以最好的饮食习惯开始了新的一年，恶劣的天气、以巧克力为主的假期和*生活*总有办法打乱这些最好的饮食计划。



如果你需要帮助回到正轨——或者只是需要健康零食的灵感——玛莎·斯图尔特的善良的人们有一个制作完美小零食的便捷公式“将水果或蔬菜与蛋白质搭配，然后添加一点令人兴奋的味道。”

如果你想一想，你会意识到这就是为什么像原木上的蚂蚁和鹰嘴豆泥和胡萝卜这样的经典小吃如此成功——蛋白质帮助你感觉饱，植物部分帮助你感觉好。然而，我想强调脂肪的重要性。

添加一种(健康的)脂肪(显然)会使零食更美味，但也会使它尝起来更完整。脂肪将蛋白质和植物结合在一起，不仅能让你感到饱，还能让你感到满足。

玛莎和她的朋友们对这样的组合有几个建议(你应该上 [到她的网站](https://www.marthastewart.com/8212145/healthy-snack-combinations) 去阅读它们)，但是我已经冒昧地提出了我自己的一些建议(因为那是我的工作):

*   奶酪+切成两半的樱桃番茄+少许橄榄油+所有百吉饼调料
*   一个白煮蛋+一份黄油+腌菜花
*   B 切好的芦笋裹在火鸡+香蒜酱里蘸着吃
*   P 耳朵片+ 马斯卡彭奶酪+松脆的格兰诺拉麦片
*   肉片+鹰嘴豆泥+羊乳酪+扎塔尔奶酪
*   酸奶+烤蔬菜+辣椒酥+五香烤坚果
*   苹果片+乳清干酪+蜂蜜+蜜饯山核桃

...诸如此类。

除了蛋白质+植物+脂肪+风味的形式之外，还要考虑你的零食的质地。一点点嘎吱嘎吱的声音可以持续很长时间，让你的嘴保持兴趣，即使你正在吃的零食是“为了你的健康”的“T2”
