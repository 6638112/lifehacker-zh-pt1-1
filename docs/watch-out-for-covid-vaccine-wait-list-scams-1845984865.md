# 警惕COVID疫苗“等候名单”骗局

> 原文:[https://life hacker . com/watch-out-for-covid-vaccine-wait-list-scams-1845984865](https://lifehacker.com/watch-out-for-covid-vaccine-wait-list-scams-1845984865)

虽然我们大多数人都在等待COVID疫苗，但冒充卫生官员的骗子正在利用它作为诱饵，通过可疑的电子邮件、电话、短信甚至上门拜访来窃取我们的金钱和数据。[联邦调查局](https://www.fbi.gov/news/pressrel/press-releases/federal-agencies-warn-of-emerging-fraud-schemes-related-to-covid-19-vaccines) 和 [卫生部](https://oig.hhs.gov/coronavirus/fraud-alert-covid19.asp) 都发布了关于这些骗局的警告——如果有人打电话给你询问测试或让你上疫苗等候名单，那就是骗局。

### 骗局是如何运作的

虽然美国卫生与公众服务部表示，这种欺诈正在“迅速演变”，但它总是以您的金钱和个人信息为目标(这些信息被用于欺诈性地为联邦医疗保健计划开单和实施身份盗窃)。

最新的COVID相关骗局利用了疫苗推出的混乱。冒充卫生官员的摄像师会要求你自掏腰包购买疫苗或将你登记在虚假的疫苗“等候名单”上短信或电子邮件可能会把你带到一个带有“申请福利金”页面的虚假申请，意在欺骗你提供你的个人信息。如果有人通过机器人电话、电子邮件、或提及疫苗的短信联系你，不要以任何方式与他们互动。你也不应该点击文本或电子邮件中的链接，它们可能包含恶意软件。

“骗子真的开始利用人们希望尽快获得疫苗的愿望，这对于大多数人来说现在可能不太现实，”肖恩·赫德里克说，他是南亚利桑那州商业改善局的通讯主任，在当地ABC采访 中。

没有办法确保疫苗的【早期获取】(而且大多数州在这个时候都忙于给前线工作人员接种疫苗)。此外，由于与 COVID疫苗相关的原因，*无需提供您的社会安全号码或银行账户信息。据CDC称，疫苗将免费提供给美国人民，尽管一些疫苗供应商可能会收取管理费。* 

### **谨防主动联系**

R 记得通过你当地的卫生部门(因为疫苗项目因州而异)、你的医生和疾病控制中心寻找关于疫苗推广 *和*的信息，疾病控制中心有一些关于 [的信息在这里](https://www.cdc.gov/coronavirus/2019-ncov/vaccines/expect.html) 。如果你对自己在疫苗行业中的地位感到好奇(至少，现在是这样，看看这篇美国广播公司新闻报道 中的 [工具。](https://abc7.com/vaccine-covid-when-can-i-get-the/8821951/) 

诈骗案的激增不仅限于疫苗，还包括其他与COVID相关的项目，如救济检查和接触者追踪。骗子们利用了这一事实，即这些程序是新的，需要*与政府机构进行一些*互动。无论哪种方式，警惕不请自来的联系，并保护您的个人信息。

如果你发现了一个骗局(不管你有没有赔钱)，向举报。如果你认为你是COVID相关欺诈的受害者，请立即向美国联邦调查局(，或1-800-CALL-FBI)或tips.hhs.gov(或1-800-HHS-TIPS)报告。
