# 把一个充气泳池变成室内球坑

> 原文:[https://life hacker . com/turn-a-inflated-pool-into-a-indoor-ball-pit-1845506504](https://lifehacker.com/turn-an-inflatable-pool-into-an-indoor-ball-pit-1845506504)

这可能会让我们觉得，在疫情，我们已经到了这样的地步，我们已经想不出如何改变现状，如何在我们的四面墙内变出一些新鲜的欢乐。但是如果你还没有建造你自己的室内球坑，你至少还有一个锦囊妙计。

Watch

我从 [的Shari Medini那里得到这个想法，Adore Them Parenting](https://adorethemparenting.com) (也是即将出版的书[*Parenting While work From Home*](https://adorethemparenting.com/book))的合著者)。她写信给我，给有小孩的父母提出了这个想法:“用一个充气的儿童游泳池和你放在房子周围的任何球来创造一个室内球坑。初学走路的孩子特别喜欢把球扔进扔出，然后在里面爬行。”

这个想法真的打动了我，因为我是那些在晚春去寻找我的山丘庭院能容纳的最大充气池的父母之一。我把它拔出来，清理干净，在我的客厅里给它充气。

是的，它在室内比在室外看起来更大。但是我在这一点上很投入，虽然我的儿子已经10岁了，而且显然不是蹒跚学步的孩子，但是他非常乐意暂时放下他的数学作业，去寻找我们家里的每一个软球。为了填满它，让你能一头扎进去，我们还添加了一些填充动物玩具、一把豆袋椅和一个20世纪90年代的巨大费城老鹰抱枕。(一半的乐趣是在房子里搜寻柔软的东西。)

然后，他跳了下去。很多次。他假装被吞噬，被活埋。他朝我扔毛绒玩具。然后，他放松了一下。

如果我有更小的孩子，并且我打算完全投入到球馆的生活方式中，我可能会挥霍掉 [一套真正的球馆球](https://www.amazon.com/Click-Play-Phthalate-Crush-Plastic/dp/B00PYLU3GG/ref=sr_1_6?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/turn-an-inflatable-pool-into-an-indoor-ball-pit-1845506504&asc_source=&dchild=1&keywords=ball pit&qid=1603897538&sr=8-6&tag=kinjalifehackerlink-20) 。然后，如果我有足够的空间，我可能还会添加一个小滑梯，这样他们就可以*嗖*的一声钻进去。

此外，我还没有这样做过，但尝试用一堆吹大的气球填充它感觉很诱人——当然，不是为了跳上去，而是为了一般的闲逛和击球。就像世界上最怪异，最干燥的泡泡浴。

如果你的充气泳池有这么大，你也可以在里面铺上毯子和枕头，并用床单盖住顶部，营造堡垒般的感觉。老实说，这里的可能性是无穷的。
