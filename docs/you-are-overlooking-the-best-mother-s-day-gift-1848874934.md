# 你忽略了最好的母亲节礼物

> 原文：<https://lifehacker.com/you-are-overlooking-the-best-mother-s-day-gift-1848874934>

我先提出一个警告。首先也是最重要的:无论如何，请得到你生命中的母亲明确要求(或暗示)的任何东西。无论是独处时间、按摩、音乐会门票，还是一个新手镯，都要满足。给她一些她喜欢的，珍惜的，或者渴望的东西。



*然而*。如果她没有指定，或者最近过了生日，装了一些她最喜欢的东西，而现在你不知所措:C 考虑送她打扫卫生的礼物。因为很有可能，她想休息一下。

现在，丈夫也做家务吗？是的。但是在一个有着凌乱的孩子、满满的玩具、背包、运动装备和每个孩子每天“带回家”的文件夹里有 67 张纸的房子里，经常会有两个人无法合理处理的杂物。(还有，这不是关于他们的。)

据报道，“根据美国劳工统计局进行的时间使用调查， [职业妈妈平均每周花 21 小时多一点的时间做家务](https://www.bls.gov/charts/american-time-use/activity-by-sex.htm#) ”

当然，没有人想在妈妈的特殊日子里侮辱她，如果她是一个为她所爱的人提供一个干净整洁的家而感到无比自豪的女人，这可能不是她想要的方式。

*然而*。对其他人来说:对 T2 来说，我们中的一些人在睡觉前不总是把厨房台面收拾得干干净净，因为她每天只有一个小时的时间，该死的，她想完成布里杰顿的工作。为了那个以喊时间(或者 4000)出名的人】*为什么我是这个家里唯一一个捡鞋的人？”那个*女人？她会喜欢的。

另外，干净的房子不会伤害一段感情。据 [【赫芬顿邮报】](https://www.huffpost.com/entry/tidy-home-happy-relationship_n_5c3e1c01e4b01c93e00e1df7) 报道，2016 年 [对刚离婚的人](https://www.huffpost.com/entry/tidy-home-happy-relationship_n_5c3e1c01e4b01c93e00e1df7) 进行的一项调查中，30%的受访者将“家务分歧”列为他们分手的首要原因(在不忠和渐行渐远之后)。

你生活中的母亲可能没有足够的时间让所有的东西都保持整洁。她可能不擅长，或者不太关心家务。或者，她可能会优先考虑工作、家庭后勤、晚餐准备、家庭作业帮助和自我照顾，而不是一尘不染的家。

所以这个母亲节，除了早午餐，在床上为她端上的熟食板，或者让她 [潜逃到酒店度过一个荣耀寂静的夜晚](https://lifehacker.com/this-mothers-day-get-away-from-your-kids-1834269870) ，为她履行服务和清洁的终极行为。要么花一天时间把房子从上到下擦洗一遍，以表示对她日常工作的感谢，整理那些塞满东西的柜子和架子，这些柜子和架子会引起很多小声的咒骂，捡起地板上的每一件衣服，或者完成你已经推迟了几个月的庞大的地下室或车库清理工作。

或者，送她一份清洁服务。无论是一次性的、专业的对你家每个角落的深度清洁，还是每两个月一次的服务，或者是每周一次的管家来整理、洗衣服、更换所有的床单，它都会给她的生活带来比其他手镯更多的安宁。

(除非她更想要手镯——在这种情况下，送她手镯。)
