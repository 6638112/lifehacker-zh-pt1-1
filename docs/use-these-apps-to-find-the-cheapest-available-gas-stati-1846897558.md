# 使用这些应用程序找到你附近最便宜的加油站

*短缺导致油价上涨，并导致东海岸部分地区燃料供应不足。对于那些给油箱加油有困难的人来说，这里有几个应用程序可以告诉你附近的加油站在哪里，哪些加油站真的有汽油卖。*

### *GasBuddy ( [安卓](https://play.google.com/store/apps/details?id=gbis.gbandroid) 、T5】iOS)*

*在当前的汽油短缺期间，GasBuddy的iOS和Android下载量激增。这个免费的应用程序显示附近所有的加油站，让你按公司搜索，还包括实时价格地图和断电跟踪器。这些功能将帮助您查看哪些位置仍有燃油库存，而不必提前打电话或开车到镇上寻找最实惠的选项。*

*除了搜索和供应跟踪功能，GasBuddy还可以帮助你找到省油的路线，跟踪你的行程，你甚至可以在应用程序中支付你的汽油费(在参与的加油站)。虽然这些功能是 [智能手机应用](https://www.gasbuddy.com/app) 独有的，但你可以在 [GasBuddy网站](https://www.gasbuddy.com/) 上搜索各州最便宜的汽油。还有一个 [燃料需求跟踪器](https://www.gasbuddy.com/go/fueldemand) 预测价格和供应估计。*

### *谷歌地图( [安卓](https://play.google.com/store/apps/details?id=com.google.android.apps.maps) ， [iOS，](https://apps.apple.com/us/app/google-maps/id585027354) [桌面](https://www.google.com/map) )*

*谷歌地图是另一个选择——大多数人已经在智能手机上使用了。 [网页版](https://www.google.com/map) 也可以从任何浏览器轻松访问。*

*打开谷歌地图，在搜索栏输入**“加油站”**，或者去**“查看附近的地方”>更多>加油站。一些加油站会给你显示当前的价格。使用搜索结果顶部的过滤器仅显示现在开放的电台，或24小时开放的电台。如果你有一辆电动汽车，谷歌地图还会显示附近的充电地点。***

*谷歌地图还允许用户提交评论和其他企业的当前信息，这有助于识别当前库存的加油站。然而，谷歌地图没有专门的选项将加油站标记为空，所以你需要查看最新的评论和评论。*
