# these are the boundaries every dog owner should teach 1846785580

> 原文：<https://lifehacker.com/these-are-the-boundaries-every-dog-owner-should-teach-1846785580>

作为一只聪明、古怪的牧羊犬，我不得不学习一些关于界限重要性的惨痛教训。例如，如果我的小狗没有工作，它会自己找一份。如果有人(尤其是陌生人)侵犯了他的私人空间，他会变得很冷漠。我们不得不建立很多结构来防止他陷入困境，无论是在国内还是在世界上。

狗需要界限。界限让他们安全，帮助他们感到平静。犹他州 [Wasatch K9 健身和训练](http://wasatchk9fitness.com/) 的合伙人兼训练师特里萨·福斯特(Theresa Foster)说，没有组织结构的狗很容易对自己在中的角色感到困惑，这会导致狗和人类的行为问题和许多挫折。



人类也一样，必须意识到并尊重某些界限——无论是与自己的狗还是与其他人的狗——因为狗无法用语言交流什么让它们焦虑或不舒服。

福斯特说，界限——什么是可接受的，什么是不可接受的——对每个人和他们的狗来说都是独特的。另外，每个教练都会以不同的方式构建和维持这些界限。但是有几个基本的界限对大多数狗狗和主人都有帮助。

### 教你的狗的界限

首先，帮助你的狗理解个人空间——他们的和你的。拉塞尔·哈特斯坦是一名 [认证行为主义者和训练师](https://www.funpawcare.com/) ，也是洛杉矶 Fun Paw Care 的创始人，他建议在你的家里给你的狗一个自己的安全空间，你可以送它们去休息——例如，当它们太多地跟着你的时候。

福斯特说，这是教狗在家里放松的一部分，也是在工作(或玩耍)和休息之间划定界限的一部分。这对于精力充沛的运动犬和工作犬来说尤为重要。你可以教你的狗一些特定的指令，来指示是时候玩耍了，或者把它们送到它们的“地方”



给你的狗自己的空间吃饭——不要在餐桌上喂它们。这样可以尽量减少乞讨。

当你出去散步时，教你的狗不要向其他狗或人扑去或吠叫。福斯特说，这种行为往往是由于缺乏指导和良好的牵绳方式。给你的狗明确的行走方向，建立他们的信任，可以帮助缓解这个问题。

最后，Hartstein 说，你应该期待你的狗会和你打招呼，询问它们想要什么。例如，如果他们想跳到床上，他们应该坐在你旁边，进行眼神交流，而不是直接跳过去。

### 尊重你的狗的界限

你和你的狗的关系不是单方面的。如果期望他们尊重你的界限，你也应该尊重他们的界限。一旦你建立了他们的个人空间——一张床，一个箱子，或者一个垫子——不要侵犯它。也不要让孩子骚扰你的狗，尤其是当他们在他们的安全空间的时候。



Hartstein 警告说不要带你的狗去那些过度刺激和令人害怕的地方，比如烟火表演、喧闹的聚会和滑板公园。知道是什么让你的狗紧张，不要把它们推到舒适区之外。

最后，不要强迫你的狗参加他们不喜欢或不喜欢的活动。不是每只狗都注定是服务动物、敏捷冠军或码头潜水员。

### 尊重别人的狗的界限

这不应该说，但请不要在没有询问的情况下接近或抚摸你不认识的狗。就像大多数人不希望被随机的人触摸一样，狗也不希望被人触摸。另外，并不是所有的狗都对陌生人友好——这没关系。

如果你想爱抚一只狗或与它互动，一定要先问，并接受“不”作为回答。如果你得到了许可，不要马上拍它的头或靠近它的脸。询问主人，狗喜欢怎样被爱抚，如果狗的肢体语言表明它不喜欢被抚摸，就走开。



同样，不要允许你自己的狗未经允许接近另一只狗。仅仅因为你的狗“友好”并不意味着其他狗也会友好。如果你的狗没有拴上皮带，而是跑向拴着皮带的狗，这可能会特别成问题。福斯特说，允许这种情况发生会取消训练或“对狗和主人来说都变得非常危险”。

你也应该避免喂不属于你的狗，给不请自来的训练建议，或者教它们新的命令或技巧。你不知道狗在进行什么训练，即使是好的意图也会让主人感觉不好或阻碍狗的进步。

### 如何保护你的狗的界限

“作为主人，保护你的狗是你的职责，”福斯特说。如果有陌生人靠近，不要害怕站在你的狗前面。你可以说你的狗不友好，或者你在专心训练。如果情况对你的狗很危险，你可以(也应该)离开。

“这很难，感觉很粗鲁，但我们必须学会为我们的狗辩护，并在每次带它们出去的时候帮助它们成功，”她说。
