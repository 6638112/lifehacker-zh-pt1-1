# 如果不接受WhatsApp的新隐私政策会怎么样？

> 原文:[https://life hacker . com/whatsapp-new-privacy-p-1846880181](https://lifehacker.com/what-happens-if-you-dont-accept-whatsapps-new-privacy-p-1846880181)

WhatsApp备受争议的新隐私政策将于2021年5月15日生效。更新后的协议 [允许脸书收集你的WhatsApp数据](https://lifehacker.com/how-to-ditch-whatsapp-for-good-1846204849?utm_campaign=Lifehacker&utm_content=1612541515&utm_medium=SocialMarketing&utm_source=facebook) ，很多人对此很不高兴。也就是说，用户不会被迫自动接受新条款——但是那些不同意这些条款的用户*将会体验到他们服务的一些变化。最终，你将无法使用该应用程序，但这将是一个逐渐的过渡，因为在几周的时间里，你会失去越来越多的访问权限。*

### 为什么这些WhatsApp的政策协议不会消失？

WhatsApp已经向其用户发送了关于隐私变化的定期提醒，但现在，你可以忽略这些通知，继续使用该应用程序，而无需同意这些变化。

然而，根据WhatsApp的说法，“经过几周的时间，人们收到的提醒最终会变得持久。”在接下来的几周里，这将在不同的时间对所有用户发生。

当这种情况发生时，应用内通知将阻止访问您的聊天列表或其他功能，直到您同意新的隐私政策。在此期间，你仍然可以接受音频和视频来电，以及通过WhatsApp推送通知阅读或回复消息，但你将无法使用应用内功能。

### 我什么时候会完全无法访问WhatsApp？

在应用功能被截断几周后，用户将被完全锁定在WhatsApp之外——包括通知和来电——直到他们接受隐私协议。WhatsApp没有说明这需要多长时间。

### 那我的账户会怎么样？

虽然WhatsApp *不会因为你不接受协议而立即删除你的账户，但一旦你失去对该应用的访问，该账户在技术上就会被“停用”。120天不活动后，您的帐户将被标记为删除。已删除的帐户和邮件数据无法恢复。*

### 对于WhatsApp的新隐私政策，你该怎么办？

WhatsApp用户有两个选择:第一，你当然可以接受新的隐私协议，继续使用WhatsApp。只要您的帐户仍然有效，您随时都可以这样做，但是尽快接受意味着避免服务中断。

第二，如果你完全拒绝接受新的隐私政策，你可以从WhatsApp的 [迁移到新的消息服务](https://lifehacker.com/the-best-whatsapp-alternatives-1832064581) 。在这样做之后，你还应该删除你的WhatsApp账户，前提是你真的想让脸书远离你的业务——但在此之前，导出你的WhatsApp数据和消息历史记录。您不能将聊天或个人数据导入到其他应用程序，但您会有一份您的信息的备份，供以后参考。

#### **保存WhatsApp消息(Android或iOS)**

1.  打开一个聊天线程。
2.  轻按屏幕顶部的聊天名称以打开菜单。
3.  向下滚动到**“导出聊天”**您可以选择在备份中包含媒体附件，或仅备份文本数据。
4.  对您想要储存的任何其他信息重复上述步骤。

#### **请求您的帐户数据(Android或iOS)**

1.  前往**设置>账户>请求账户信息。**
2.  WhatsApp将在三天内带着您可下载的数据报告与您联系。

请确保在持续通知阻止您使用应用程序之前备份您的数据。也就是说，你可以随时同意新的隐私政策，备份你的账户，然后*然后*删除它，或者在WhatsApp已经闲置120天后，让WhatsApp去做脏活。
