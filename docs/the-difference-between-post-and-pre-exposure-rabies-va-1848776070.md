# 暴露后和暴露前狂犬病疫苗的区别(以及何时需要)

我高中的时候，被猫咬了一口就要打暴露后狂犬病系列。我停下来帮助一只被车撞了的小猫，结果这只小猫咬了我一口就跑了，这意味着我无法知道她是否携带狂犬病病毒。

尽管接受曝光后的注射感觉有点矫枉过正——特别是因为这意味着在周五晚上去当地的急诊室，在那里我接受了两次注射(每个屁股脸颊一次)。在接下来的两个星期里，这些最初的注射随后又进行了三次，一次注射一次。

尽管这些注射很不方便，但暴露后狂犬病系列是一种有效的治疗方法，如果立即实施，几乎100%有效。以下是关于狂犬病风险和为什么你可能需要接种疫苗的知识。

## **你需要了解的狂犬病知识**

如果不治疗， [狂犬病有99.9%的死亡率。](https://www.webmd.com/a-to-z-guides/what-is-rabies) 这不是你想惹的病毒，因为它会攻击你的中枢神经系统，并在这个过程中摧毁它。据估计 [每年有59，000人死于狂犬病](https://www.nytimes.com/2022/01/07/health/rabies-deaths.html) ，大部分是在没有为狗广泛接种狂犬病疫苗的国家，接触后狂犬病疫苗也不容易获得。

人们通常因被受感染的动物咬伤或抓伤而感染狂犬病，蝙蝠或未接种疫苗的狗咬伤是最常见的感染途径。在美国，狂犬病的死亡率非常低，去年共有五人死亡，是十年来最高的记录。

活动性狂犬病感染的幸存者非常少，只有 [治疗方法可以诱导昏迷，在此期间患者接受抗病毒](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC7670764/) 治疗。甚至这种几十年前引进的治疗方法也只是部分成功，死亡仍然是最有可能的结果。

但是，如果一个接触过狂犬病的人马上接受暴露后狂犬病疫苗接种系列，几乎是 [100% 有效](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC3272373/#:~:) 。

## **暴露后狂犬病系列是什么样的**

在过去，暴露后狂犬病系列需要在胃部打一针。令人高兴的是，现在情况不再是这样了，前两针注射在臀部，剩下的注射在手臂。

[前两针](https://www.cdc.gov/rabies/medical_care/index.html) ，要么在暴露当天注射，要么尽可能接近暴露当天注射，是一剂狂犬疫苗，以及一剂人用狂犬病免疫球蛋白。免疫球蛋白提供了短期的帮助来抵御你系统中任何可能的病毒，而疫苗则促使你的免疫系统开始产生抗狂犬病病毒的抗体。

前两次注射后，在第3、7和14天再注射三次疫苗。

## **为什么您可能需要接种*预*暴露狂犬病疫苗**

狂犬病疫苗很贵，这就是为什么大多数人在潜在暴露后才会接种疫苗。然而，在高危环境工作的人可能需要一个 [暴露前疫苗系列](https://www.cdc.gov/rabies/specific_groups/travelers/pre-exposure_vaccinations.html) 。潜在的高风险职业包括兽医或野生动物控制专家。

疾病预防控制中心还建议，如果你要去一个很难获得医疗护理的偏远地区旅行，或者如果你将在一个犬类狂犬病高发区呆上一个月以上，可以考虑接种暴露前疫苗系列。

暴露前疫苗系列在一个月内总共注射了三针。这些初始注射后可进行加强剂量，加强剂量的时间和频率取决于其暴露风险。
