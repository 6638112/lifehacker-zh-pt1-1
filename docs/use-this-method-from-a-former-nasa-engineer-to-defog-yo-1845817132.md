# 使用这种来自前美国国家航空航天局工程师的方法，立刻给你的挡风玻璃除雾

> 原文：<https://lifehacker.com/use-this-method-from-a-former-nasa-engineer-to-defog-yo-1845817132>

更冷的天气更令人讨厌的一点是对那些开车的人来说意味着额外的工作。在装上雪地轮胎、刮掉冰雪和等待你的车预热之间，每件事都要花一点时间。 尽管没有多少人每天通勤去上班，但这并不意味着每个人都会逃避开车时发现挡风玻璃上全是雾气的可怕感觉。但是如果你发现自己处于这个位置，它并不像看起来那么糟糕:一位前美国宇航局工程师分享了他基于科学的除雾技术，设计是为了更快地完成工作。以下是需要了解的内容。



### 如何快速给你的挡风玻璃除雾

这个提示来自于前美国宇航局工程师、自称“科学之友”的马克·罗伯 在一段五分钟的 YouTube 视频中，他用一半的时间向我们展示了他的挡风玻璃除雾四步法。(注意，下面剧照上的标题说他将谈论*给*窗户除霜，但在实际视频中，他只提到*除雾*。)

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-qCmgWiEEZwA&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-qCmgWiEEZwA&start=0) 

在视频中，Rober 用科学的方法测试了各种技术，然后选择了一种他认为对挡风玻璃和车窗除雾最有效的技术。以下是相关的步骤:

1.  把你汽车的加热器开到最大。热空气可以保持更多的水分。
2.  打开你汽车的空调。根据 Rober 的说法，当空气经过空调的冷盘管时，这会将空气中的水分带走。
3.  **确保车内空气循环关闭。“冬天的空气很冷，我们知道它不含太多水分，”罗伯解释道。“所以，如果你把它带进你的汽车，加热它，它有很大的吸收能力。”**
4.  打开尽可能多的窗户——哪怕只有几秒钟——让潮湿的车内空气与外面干燥的冷空气进行交换。

当然，有很多变量，像温度、降雨量和你的车的各种设置，但如果你经常遇到雾，这些都值得一试。
