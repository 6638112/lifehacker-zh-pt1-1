# 你实际上可以做一些关于脚趾骨折的事情

> 原文：<https://lifehacker.com/you-can-actually-do-something-about-broken-toes-1849366052>

传统观点认为，对于脚趾骨折，你真的无能为力。好吧，传统智慧是愚蠢的。事实证明，当你的脚趾骨折时，可以也应该采取一些干预措施。

## 是的，当你脚趾骨折时，你应该去看医生

如果你怀疑你的脚趾被踢伤了，或者有东西掉在上面，不要理会你所听到的关于它们不可修复的说法，去看医生。根据克利夫兰诊所 的说法，以下是你脚趾可能骨折的迹象:

*   一个脚趾指向与其他脚趾不同的方向
*   一只脚趾扭曲并向上翻
*   脚趾肿了
*   脚趾擦伤了
*   你不能把重量放在你的脚上
*   疼痛不止持续一两天

克利夫兰诊所也建议不要冰敷脚趾，因为你需要血液流向伤口以促进愈合，但最好咨询一下你的医生，看他们是否希望你冰敷脚趾。与此同时，抬高它并服用一些止痛药——但要尽快去找专业人士。

## 医生能做什么？

去看医生是值得的，因为即使*他们告诉你简单地用胶带包扎脚趾，他们也有可能为你提供其他治疗方法——跳过这些是不好的。例如，如果你的骨头被拉得错位了，它就需要重新对齐。如果愈合不良，即使是简单的脚趾骨折也会在未来引起关节炎等问题。最好去检查一下。*

[根据梅奥诊所](https://www.mayoclinic.org/diseases-conditions/broken-toe/symptoms-causes/syc-20370463#:~:text=Usually%2C%20you%20can%20treat%20a,surgery%20to%20ensure%20proper%20healing.) 的说法，你通常可以通过将脚趾绑在相邻的脚趾上来治疗脚趾骨折，但严重的骨折可能需要石膏或手术。你的医生可以确定骨折是否严重——特别是如果骨折发生在你的大脚趾上，大脚趾在平衡和行走中起着重要作用。
