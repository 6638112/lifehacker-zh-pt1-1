# 修理自来水马桶的最简单方法

> 原文：<https://lifehacker.com/the-easiest-ways-to-fix-a-running-toilet-1848853435>

啊，厕所。一项如此完美的发明，在问世后的五个世纪左右的时间里，我们没有对它进行任何实质性的更新。你今天使用的抽水马桶基本上与 16 世纪约翰·哈灵顿爵士引入的设计相同——它依靠供水和重力，并使用一种令人惊讶的简单机制让你的废物神奇地消失在你的化粪池或下水道系统中。然而，就像其他任何事情一样，它也可能对我们不利。 



在灾难等级的一端，它会堵塞和溢出，把平常的一天变成一场醒来的噩梦。在天平的另一端，它可以开始持续运行。你的马桶不停地装满水看起来是个小问题，但是一个不停运转的马桶要花你的钱——泵入你家的水不是免费的。

幸运的是，修理一个自来水马桶实际上非常简单，通常不需要叫水管工——这里有一些简单的方法让你自己停止使用自来水马桶。

## 油是如何工作的

首先，让我们上一堂厕所工作原理的速成课。看看马桶后面的墙——你应该会看到一个阀门和一条通向马桶水箱的软管。那是你的水源。如果你认为你需要更换什么东西，你可以关掉马桶的水，但是当你处于诊断模式时，最好让水一直开着，因为你可能需要重新注满水箱几次。

有很多不同的马桶设计，但是一个功能性马桶的基本成分是相同的:进水阀和管道、浮子和挡板。进水阀让水进入水箱，直到浮子达到一定高度，然后关闭，当你使用马桶上的手柄或按钮时，挡板就会抬起，将水释放到马桶中进行冲洗，并重新开始整个过程。

现在我们对我们正在看的东西有了一个基本的概念，有四个常见的原因导致你的马桶不停地跑，而且每一个都很容易解决。

## 检查挡板和链条

马桶不停跑的最常见原因可能是挡板。这是一个橡胶或塑料片，安装在冲洗阀上，阻止水流入马桶。如果挡板没有正确安装在阀门上，少量的水将持续泄漏到马桶中，导致水箱进入无限的补水循环。如果经典的“摇动手柄”技术有时真的对你有用，这可能是你的问题，因为这种摇动本质上是让挡板复位。

拿一根棍子轻轻按下挡板。马桶是不是突然不跑了？那你就有你的罪犯了。有两种方法可以解决这个问题:

*   **更换挡板。**检查挡板本身。是否翘曲变色？如果它翘曲了，它就再也不能正确地安装冲洗阀了，所以你必须更换它。它们很便宜，你可以买到通用的挡板，几乎可以在任何厕所使用。
*   **固定链条长度。**当你使用马桶上的把手时，挡板会被链条提起，然后随着水箱注满水，挡板会慢慢回到原位。如果链条太短，它会阻止挡板完全下降，从而在冲洗阀上形成密封。如果太长，它可能会卡在挡板的唇部下面，结果也一样。用钳子把一条长链条切掉，或者用一条稍微长一点的链条来代替，通常会解决你的问题。

## 检查浮子

如果挡板看起来状态良好，不是你的问题，接下来检查你的浮子。浮子是一个大的塑料圆柱体或球体，字面意思是漂浮在水箱的水面上。随着水位下降，浮子也随之下降，打开进水阀，这样更多的水就可以冲进水箱。一旦它浮回到原来的位置，阀门关闭，水停止。

如果浮子设置得太高，水箱就会溢出，水就会溢出到溢流管中(这是为了防止水箱溢出)。因为水不断溢出到溢流管中，浮子永远不会高到足以切断流入，可怕的无休止的水流循环开始了。

浮子是可调的——通常你可以转动一个简单的螺丝来改变浮子的高度。看看你的水箱内部——可能有一个标记蚀刻在那里显示填充水平，这是你希望你的水位在填充周期后的位置。然后看看水箱中的进水阀——上面会标有“临界水位”。不管水箱上刻的是什么，水位都应该比这个标记低大约一英寸*(因为不同的卫生间工具以不同的方式安装)。调整你的浮子，直到水位重新回到那个点，看看这是否能解决你的问题。*

## 检查加注管

最后，看看进水阀，它是从供水连接处向上延伸的管子，与浮子相连。将有一个短塑料管从进水阀延伸到溢流管的正上方。当你冲马桶的时候，水通过这根管子流入溢流口，重新注满马桶。如果位置不正确，没有足够的水进入马桶，这可能导致马桶持续运行。

如果它完全脱落了，重新装上它，并确保它牢牢固定。还要确保管子的末端位于溢流管上方(或者有时直接进入溢流管)。冲洗马桶，确保进水管中的水全部流入溢流管。这应该能解决这项神奇的厕所技术带来的任何问题。
