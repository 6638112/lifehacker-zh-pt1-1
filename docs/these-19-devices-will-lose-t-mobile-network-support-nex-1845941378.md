# 这19款设备将于下月失去T-Mobile网络支持

> 原文:[https://life hacker . com/these-19-devices-will-lose-t-mobile-network-support-NEX-1845941378](https://lifehacker.com/these-19-devices-will-lose-t-mobile-network-support-nex-1845941378)

根据 [Android Police](https://www.androidpolice.com/2020/12/22/these-are-all-the-devices-t-mobile-says-will-stop-working-on-its-network-next-month/#1) 获得并核实的T-Mobile内部文件，该公司计划于2021年1月29日停止对19款设备的移动网络支持。受影响的设备包括谷歌、三星、索尼等公司的智能手机，以及老式家用安全摄像头和无线路由器。

这一变化会影响T-Mobile、MetroPCS、和Sprint的客户，但影响将取决于你的设备最初使用的运营商网络。T-Mobile和MetroPCS设备将从1月29日开始失去所有网络支持，而Sprint设备将失去T-Mobile漫游连接，但仍可以使用Sprint的传统3G网络，直到2023年关闭。

可能还有更多不受支持的设备，但T-Mobile截至发稿时尚未发布公告，因此我们只能依靠Android Police的信息。也就是说，据报道，T-Mobile将从12月28日开始通过短信提醒客户，这给了那些仍在使用这些旧产品的人几周的时间来升级。

这么短的通知很不方便，但是我们建议用户从上面列出的任何智能手机升级，即使T-Mobile没有停止支持。新手机不仅速度更快、功能更强、摄像头更多，还能定期获得旧手机没有的安全补丁。

如果价格是一个问题，那么看看像Pixel 4a和5a这样的中档设备，或者三星的Galaxy A51。只要手机仍然得到制造商的支持，它的性能和安全性将会有巨大的飞跃，而且你可以用比新的旗舰手机低得多的价格购买一部手机。
