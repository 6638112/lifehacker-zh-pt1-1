# 如何恢复忘记保存的Microsoft Office文件

你正在努力工作在Excel电子表格上，确保所有的数字都是准确的，一切看起来*都很棒*。你是一台机器:没有什么能阻止你完成这项任务，给你的老板留下深刻印象。然而，停电时会在你有机会保存之前关闭你的电脑。现在你要么重新开始，要么辞职永不回头。

别急:虽然不保存你的工作有点危险，但丢失这样一份文件并不意味着死刑。如果您出于任何原因不小心关闭了Office文档而没有保存，您不需要担心从头开始。使用您正在使用的Office应用程序中内置的恢复工具，有一种简单的方法可以恢复文件。

当你在Word、Excel或其他Office应用程序中处理文档时，当你点击“保存”按钮时，文件不仅仅会被备份。有问题的应用程序将该文件的数据保存到用户帐户的AppData文件夹中的漫游文件夹中。这个文件夹通常在大多数用户看不到的地方，为你的用户帐户保存各种Windows应用程序的数据。

在大多数情况下，可以在这里找到最近关闭的未保存的Office文档。主要有两种方法可以找到它。第一种是最简单和最直接的，因为它会直接带你到包含丢失文档的漫游文件夹。首先，打开你原来关闭的应用程序，点击左上角的“文件”标签，然后选择“信息”现在，单击“管理文档”，然后单击“恢复未保存的文档”在这里，你应该看到你没有保存就关闭的文件:选择它，然后点击“另存为”以确保你不会再次丢失它。

如果你想感觉更“科技”，有一个稍微更先进的方法可以达到同样的效果。按Windows+R打开“运行”窗口。然后，键入%appdata%并运行它以打开漫游文件夹。从这里，按照文件路径找到应用程序的文件夹(例如，打开Microsoft文件夹，然后打开Word文件夹)。这是第一种方法直接带您到的文件夹，所以您应该看到您丢失的文件。再次打开它，然后保存它。
