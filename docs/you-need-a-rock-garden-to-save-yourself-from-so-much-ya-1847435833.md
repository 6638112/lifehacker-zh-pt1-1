# 你需要一个岩石花园来把自己从这么多的庭院工作中解救出来

> 原文：<https://lifehacker.com/you-need-a-rock-garden-to-save-yourself-from-so-much-ya-1847435833>

虽然将岩石融入景观设计很常见，但大多数时候是以植物或树木周围的边界的形式，或者是在院子或花园中开辟一条道路。但是还有另一种可能性:你可以计划并创建一个完整的岩石花园。

## 什么是岩石花园？

即使你不熟悉“岩石花园”这个术语，你也可能见过。根据 [GreenMyLife](https://www.greenmylife.in/what-is-rock-garden/) 的说法，基本上，一个岩石花园就像它听起来的那样:一个围绕岩石规划的花园，这些岩石要么是自然形成的一部分，要么是被引进来并安排得看起来好像它们一直就在那里。

根据 [《园林设计](https://www.gardendesign.com/landscape-design/rock-gardens.html) 中的一篇文章，岩石园林可以有不同的形状和大小:

> 岩石花园可以是复杂的大型项目，有许多方面和层次，也可以是简单的用砾石和河石装饰的小角落。甚至一个容器也可以变成一个微缩的岩石景观。

岩石花园除了好看之外，还可以有多种用途，包括填充干涸的河床，界定斜坡，以及作为传统草坪的替代品。他们也是一个很好的选择，在土地不平坦或以某种方式损坏的地方。

## 岩石花园如何节省你的时间和精力

岩石花园的主要吸引力之一是，它们减少了你除草和照料一块土地的时间。想想看:杂草不能在被岩石覆盖的土壤上生长。当然，杂草(理想情况下，你真正想要的植物)可以在岩石之间生长，但这是关键。而且，与传统的草坪不同，岩石花园不是你可以(或者应该)修剪的。

在一个区域种植更少的植物也意味着不需要花太多时间浇水和做其他类型的庭院维护。此外，它还节约用水——这是值得考虑的，因为环境保护署估计 景观灌溉占美国所有住宅用水的近三分之一，相当于每天 70 亿加仑的水。

## 如何打造一个岩石花园

虽然岩石花园维护成本低，而且一年四季都可以看到(除非你住在下雪的地区)，但你不能只是扑通几声扔下几块石头就收工——这个过程需要一些规划。

根据 [花园设计](https://www.gardendesign.com/landscape-design/rock-gardens.html) ，这涉及到从选择合适的地点和勾画出你想要创造的设计，到选择一个*风格*的岩石花园，比如日式、阿尔卑斯式或西南式。你还需要做一些研究，找出什么类型的植物最适合你的气候和地形，并为这个项目确定合适的土壤。

无论是 [花园设计](https://www.gardendesign.com/landscape-design/rock-gardens.html) 还是 [老房子在线](https://www.oldhouseonline.com/repairs-and-how-to/diy-rock-garden/) 都有关于如何建造并维护岩石花园的详细说明，包括植物建议。
