## 如何以第一人称玩*异刃编年史 3*

要以第一人称玩游戏，按住你的开关控制器上的 ZL 按钮，向上推右边的拇指操纵杆，将相机一直缩放到第一人称视角，然后向左或向右推操纵杆来调整高度。您还可以在设置菜单中调整的“相机移动”和“相机变焦”速度，以适应您的偏好。确保“相机位置”设置为中间选项，以你的角色的头部为中心，否则你的视线会笨拙地向左或向右偏移。

为了增强沉浸感，我还禁用了“迷你地图”、“自动定位”和“附加信息”设置，并确保尽可能隐藏或禁用所有任务标记和通知。

这些都是可选的改变，但是它们值得一试，即使你是在默认的相机视角下玩游戏。这样做可以消除很多 UI 混乱，这样你就可以在探索的时候看到更多的世界。它还迫使你更密切地关注你的周围环境，我发现这有助于我更多地与环境联系起来(如果你需要仔细检查你的方向，你可以很容易地从快捷菜单中打开地图)。

## *异刃编年史 3* 真的可以第一人称玩吗？

仅仅因为你*可以*做一些事情并不意味着你*应该，* 但是到目前为止，*异刃编年史 3 的*第一人称视角对我来说感觉很棒，尽管这需要一些时间来适应。

以第一人称玩游戏能让你更近距离地观察你与之互动的角色、敌人和地点，并真正强调世界的巨大规模。与其他大型第一人称 RPG 如*《上古卷轴》*的明显对比是显而易见的，但在这些点缀着高科技军事基地和居住着外星动植物的广阔景观中奔跑让我想起了一个 n 动漫色彩的 [*无人区*](https://lifehacker.com/how-i-found-peace-exploring-the-vastness-of-space-in-no-1785319630) 。

移动、跳跃和捡起可收集的物品仍然很容易，以及你将执行的大多数其他动作——比如在队伍成员之间交换、与 NPC 交谈、攀爬梯子以及与重要物体互动——都是直观的并且按照预期工作。

战斗是游戏中唯一需要一些真正的调整才能运行的部分，但这是可行的。

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1554331229347934208&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1554331229347934208&autosize=1) 

总的来说，类似 MMO 的战斗系统在第一人称中运行良好。由于游戏清楚地标记了你是站在目标的前面、后面还是侧面，所以很容易判断你是否在攻击的正确位置，即使你正盯着一只巨大的外星大猩猩，你的队友的玩笑也会让你知道他们在使用什么动作。在第一人称模式中，战斗感觉更像是《CRPG》中的战术战斗，像*魔法* 8 或者*力量&魔法 X* 。

也就是说，如果你打算长期在第一人称中玩*异种刃编年史 3* ，你需要调整一些设置，因为战斗 UI 和其他效果的持续视觉弹幕在那个模式中可能会完全混乱。按照上面的 BoomStickGaming 视频的建议做，在设置菜单中禁用除了你的角色的伤害弹出窗口之外的所有战斗编号。这将使第一人称的战斗更加舒适。我还选择禁用“战斗摄像机”选项来停止某些攻击的电影过场动画切换。

到目前为止，我发现在实施了那些设置调整后，游戏完全可以在正常难度模式下玩，但是如果你仍然发现战斗太难管理，请随意将游戏难度设置为“容易”。你总是可以在战斗开始前将镜头拉远，以利用第三人称视角，然后在探索时放大。

不，第一人称模式不是玩*异刃编年史 3* 的“预期”方式，但它让我更容易进入年度最佳 JRPGs 之一。如果你更喜欢第一人称游戏，或者只是想尝试用一种新颖的方式玩*异刃编年史 3* ，我绝对推荐给它一试。
