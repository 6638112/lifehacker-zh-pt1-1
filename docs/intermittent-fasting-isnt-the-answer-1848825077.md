# 间歇性禁食不是答案

> 原文:[https://life hacker . com/intermittent-fasting-not-the-answer-1848825077](https://lifehacker.com/intermittent-fasting-isnt-the-answer-1848825077)

间歇性禁食曾被誉为减肥的有力策略，但最新的科学表明，它毕竟不是一个奇迹般的解决方案。

首先:间歇禁食是在一个狭窄的时间窗口内安排你一天的所有饮食的过程，通常是早上8点到下午4点。其他类型的间歇禁食包括“5:2”方法，即一个人一周正常饮食五天，其他两天禁食。这个想法是，通过限制饮食在指定的时间，它将减少一个人的热量摄入，最终导致体重减轻。

可以理解为什么这么多人相信间歇性禁食是减肥的关键。毕竟，美国心脏协会2017年的一份 [声明证实了这种做法，并解释说，“有意进食并注意进食时机和频率可以带来更健康的生活方式和心脏代谢风险因素管理。”休·杰克曼、吉赛尔·邦辰和格里·克鲁斯等名人都对这种做法大加赞赏。但事实证明，科学根本无法支持这一观点。](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC8532518/)

《新英格兰医学杂志》一项新的为期一年的研究 发现，当患者被分成两组(一组进行热量限制和时间限制，另一组只进行热量限制)时，结果显示在狭窄的时间范围内进食没有好处。与对照组相比，那些坚持间歇禁食的人在腰围、身体质量指数、体脂、血压或代谢危险因素方面没有看到任何显著的进步。

值得一提的是:在这项研究中，对照组和变异组*都减轻了体重。这是因为减肥成功的关键是，并且继续是，热量摄入。T2，你什么时候吃都没关系，重要的是你在吃什么。所以，午夜零食爱好者们，请松一口气吧——继续享用午夜零食吧。就考虑用胡萝卜和鹰嘴豆泥代替薯条和饼干吧。*
