# 为什么你应该对你的孩子说“是”

> 原文：<https://lifehacker.com/why-you-should-aim-for-yes-with-your-kids-1845214563>

孩子们在一天中听到很多“不”。不，你不能想熬夜多久就熬夜多久；不行，不戴头盔不能骑车；不，你不能推你妹妹。“不”很容易成为对任何新的或奇怪的请求的反射性回答，通常在你真正处理它之前(同样的方式，孩子们似乎也倾向于对 *us* 说“不”)。“不”通常需要我们花费更少的时间和精力。



但是乔安娜·戈达德为《一杯乔》 写道，作为一名年轻的父母，她听到了一些让她难以忘怀的育儿建议:挑战自动拒绝的倾向，并“争取肯定”因为有些事情总是会被拒绝——这是理所当然的。但并非一切都必须如此，戈达德解释道:

> 比如，这里有几个最近在家的场景:
> 
> *   托比想在洗澡时穿上他的泳衣
> *   安东想从一张沙发跳到另一张沙发
> *   托比想把牛奶和水混合起来
> *   安东想穿睡衣去杂货店
> *   安东想躺在滑板上
> *   托比想在大冷天穿短裤
> *   安东想倒着睡(脚放在枕头上)
> 
> 我的下意识反应有时是说不——我的意思是，跳进浴缸，喝你的牛奶——但后来我想:为什么不呢？如果它没有伤害任何人，并且出于某种原因他们觉得它令人兴奋或诱人，谁会在乎呢？“争取是”贯穿我的脑海，我说去吧。

说*是*鼓励他们的创造力。这给了他们更多的机会去探索，满足他们的好奇心，并建立他们的信心。

如果他们想穿去年的万圣节服装去商店？当然，我想，为什么不呢？如果他们想把你所有的锅碗瓢盆都拉到厨房地板中间，制造出世界上最烦人的架子鼓？还有更糟糕的事情；开始吧！每当你有多余的时间(或耐心)时，你可以试着以“是”为目标。
