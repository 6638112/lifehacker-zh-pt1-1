# 在下次虚拟会议中使用这些复古的 MS Paint 背景

> 原文：<https://lifehacker.com/use-these-retro-ms-paint-inspired-backgrounds-for-your-1845803488>

微软最近在 Xbox Gear store 上推出了一系列新的 MS Paint 主题虚拟背景 [，你可以在微软团队、Skype 或任何其他视频会议应用程序中使用。它们是今年你计划在(虚拟)假日聚会上穿的任何花哨服装的完美补充。](https://gear.xbox.com/pages/ms-paint-accessories) 



节日收藏包含在 Skype 的默认背景库中(你可以从上面的 Xbox Gear 页面下载最新版本)，但如果你想在团队或其他电话会议应用程序中使用它们，你需要将图像下载到你的计算机上，然后手动导入。

为此，请打开 Xbox Gear Store 链接，然后向下滚动并下载“MS Paint Teams 背景”将文件解压缩到电脑上易于访问的位置。启动您选择的视频会议应用程序；如果它有任何好处，你应该能够上传自定义图像，以用作虚拟背景。例如，下面是如何在微软团队中使用它们:

1.  加入通话前，点击背景效果图标，然后点击背景设置菜单顶部的**“添加新项”**。
2.  浏览到您想要使用的文件，然后点击**“打开”**将其添加到您的虚拟背景列表中。

如果你喜欢这些应用，我们也有在缩放 和 [谷歌会议](https://lifehacker.com/how-to-use-virtual-backgrounds-in-google-meet-1845585735) 中添加 [自定义背景的指南。](https://lifehacker.com/how-to-create-a-looping-video-of-yourself-that-attends-1842843207?rev=1588710796319) 

如果所有这些 MS Paint 怀旧情绪战胜了你，Xbox Gear Store 页面上的第三个链接是一个免费的 Windows 95 应用程序[在你的现代 Windows、macOS 或 Linux 机器上模拟老派操作系统](https://lifehacker.com/you-can-now-run-windows-95-as-a-windows-or-mac-app-1828559902) 。它包括大量经典软件，包括你可以启动和玩的 MS Paint 原始版本(它也有一个可用的网景浏览器和 PC 游戏经典，如 *Doom* 和 *Wolfenstein 3D* )。

从 GitHub 页面为您的操作系统下载并安装正确版本的 Windows 95 应用程序。安装完成后，一个虚拟的 Windows 95 桌面会在一个新的应用程序窗口中弹出。进入**开始>程序>附件>画图**打开 ms 画图，获得 a-doodlin。
