# 平衡训练器到底是干什么用的(什么时候你应该使用它)？

平衡板和Bosu球很难使用——它们是不稳定的板，你有时会看到私人教练要求某人站在上面做练习。但是它们是一种好的训练方式吗？如果是的话，你自己使用它们的最好方式是什么？

### 它们肯定不是必需的，但在某些情况下可能有用

许多人在没有使用任何类似平衡板的东西的情况下，已经在力量和运动能力方面取得了进步。所以，如果你从未使用过，并且你想知道你是否错过了，不要担心。你不是。

那么它们是干什么用的呢？这些设备挑战你的平衡和本体感受，这意味着你的大脑可以练习控制你的肌肉来保持你的稳定，你的肌肉可以更好地向大脑发送信号，告诉你身体在空间的位置。

如果你正在恢复受伤，你的训练计划可能包括某种不稳定训练(就像它被称为的那样)，以使你的脚和腿的所有小肌肉再次一起工作。在我做了前交叉韧带手术后，我记得为了这个目的，我被分配用一只脚站立(有时闭着眼睛，有时站在湿软的表面上)。

这种类型的训练也有助于老年人或有平衡问题的人，这些问题可能会影响他们的日常活动。平衡训练可以让一个人更好地稳定他们的身体，但出于安全原因，它也倾向于在物理治疗师或私人教练的监督下进行。在健身房的bosu上跳可能很有趣，但是如果你能安全地在云雀上跳，这可能不是你需要的东西。

## 他们不会让一个强壮的人变得更强壮

国际运动科学协会 在谈到不稳定性训练时说:“这是一种来自物理疗法的练习，对恢复受伤恢复正常非常有用。”。“但就变得更强而言，几乎没有证据表明收益超出了稳定性和机动性的改善。”

平衡训练器使练习更加困难，但这弊大于利。例如，如果你蹲着，你主要想锻炼你的腿和，其次是你的臀部和核心。为了逐步超负荷达到这些目标，你需要用更大的阻力蹲下。换句话说，不要加平衡训练器，只要 [在](https://lifehacker.com/put-some-fucking-weight-on-the-bar-1847176087) 杠上放点他妈的重量就行了。

蹲在平衡训练器上会让你的下蹲感觉更难，但不是通过挑战主要的移动者。相反，不稳定意味着需要更多的稳定力量来参加派对。你会非常努力地让那些稳定器工作，以至于你不能像蹲在平地上一样负重深蹲。

所以，如果你想锻炼稳定肌，试试bosus和平衡板，把力量训练分开。
