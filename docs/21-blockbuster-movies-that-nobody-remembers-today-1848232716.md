# 21 部如今无人记得的大片

> 原文：<https://lifehacker.com/21-blockbuster-movies-that-nobody-remembers-today-1848232716>

有些电影会永远流传下去。但是有些电影，就像旧玩具一样，受到热烈的喜爱，然后被彻底抛弃。有时这是有意的；为抓住一个文化时刻而制作的电影不太可能持久。今天很红的明星或风格不一定明天就能赚钱。有时候，我们被一部令人失望的电影说服了，一次就足够了。有时候，只是我们的口味变了。

被遗忘的大片不一定不好(也不一定好)。每部电影都是某些人的最爱，他们有时会被不公正地遗忘。当然，正如经常发生的那样，很难不去想为什么他们一开始就成功了。

这里有一些你可能已经忘记存在过的最卖座的电影。
