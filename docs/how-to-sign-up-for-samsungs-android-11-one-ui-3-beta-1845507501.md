# 如何报名参加三星的 Android 11 / One UI 3 测试版

> 原文：<https://lifehacker.com/how-to-sign-up-for-samsungs-android-11-one-ui-3-beta-1845507501>

我从来都不是三星如何对待测试版操作系统更新的粉丝。不可否认，我这么说是出于嫉妒，因为我似乎永远没有“资格”在我的任何设备上安装它们。(另一方面，我的 Pixels 总是在某种 Android beta 中注册，它的 [是一个更容易和更通用的过程](https://lifehacker.com/four-ways-to-install-the-new-android-11-public-beta-1843983079) 。)



尽管如此，三星现在正在以一种针对 One UI 3 的公共测试版的形式将 Android 11 引入其旗舰手机。我使用这种措辞是因为三星表示，目前谁可以注册使用它是有限制的。对于 [Galaxy Note 20 系列手机](https://us.community.samsung.com/t5/Note20/Note-20-20-Ultra-OneUI-3-Android-11-Mega-Thread/td-p/1495817) ，以下用户符合条件:

*   使用解锁固件的 Galaxy Note 20 和 Galaxy Note 20 Ultra 用户
*   解锁用户(直接从三星购买的 U1 设备；解锁的固件支持任何载体。)

而对于 [Galaxy S20 系列手机](https://us.community.samsung.com/t5/Galaxy-S20/S20-OneUI-3-Android-11-Mega-Thread/td-p/1455586) ，它的:

*   Sprint & T-Mobile 上的 Galaxy S20、Galaxy S20+和 Galaxy S20 Ultra 用户
*   解锁用户(直接从三星购买的 U1 设备；解锁的固件支持任何载体。)

如果你有资格，我建议你首先通过**设置>软件更新**在你的设备上安装任何未完成的操作系统更新。接下来，调出 Play Store 应用，确保你所有的应用都更新了(**三线“汉堡”图标** > **我的应用&游戏**)。启动 Galaxy Store 应用，做同样的事情(**汉堡图标>更新**)。最后，确保你已经使用 Galaxy Store 应用程序安装了 **Samsung Members** 应用程序——如果没有它，你将没有机会进入测试版。

现在，发动三星会员，交叉手指。如果您在应用程序的顶部看到类似以下内容的横幅，您就成功了:

点击它，你就可以注册测试版了。你需要等一会儿——最多 10 分钟左右——更新才会出现在你手机的**设置>软件更新**部分。考虑利用这段时间 [对你的手机](http://smartswitch) 做一个完全备份，因为安装测试版可能会导致问题。毕竟这是个测试。

此外，要知道 Android 11/One UI 3 可能会破坏您设备上的一些应用程序。这是可能发生的，这只是使用测试版操作系统的一个恼人的副产品。你可以等待应用程序的开发者修复潜在的问题，或者，如果这是一个严重的问题，你可以通过三星会员应用程序的设置屏幕取消注册来跳出测试版。之后，使用三星的 [智能开关应用](https://www.samsung.com/us/support/owners/app/smart-switch) 来降级你的设备。
