# 你的“依恋风格”导致了你们的关系问题吗？

我们被抚养的方式对我们维持健康关系的能力有影响。这个特殊的难题在各种各样的自我帮助和心理健康实践中得到解决，但是最近，你可能听到的最多的是一种叫做“依恋理论”的东西。

依恋理论并不新鲜，但在过去的几年里，它作为一种分析和定义关系的方法受到了关注。在*《华盛顿邮报》*的单人系列 [专栏](https://www.washingtonpost.com/news/soloish/wp/2018/08/16/knowing-your-attachment-style-could-make-you-a-smarter-dater/?noredirect=on&utm_term=.ed9569d70f52) 中，作家詹娜·伯奇说，她最近深入研究了《附件:成人依恋的新科学，以及它如何帮助你在一段失败的关系后找到并保持爱情 ，这对于她如何看待约会产生了奇迹。

作者阿米尔·莱文(Amir Levine)和雷切尔·海勒(Rachel Heller)的书基于这样一个想法，即所有婴儿生来都有一种与生俱来的依恋他人的愿望，这种愿望如何被我们的父母支持或阻挠，有助于决定我们成年后如何尝试(或避免)依恋他人。很清楚为什么这些理论受欢迎:因为你可以 [参加一个](http://www.web-research-design.net/cgi-bin/crq/crq.pl) 的测试，它会告诉你关于你自己的情况。

*依恋*假设了四种主要的依恋结果，其想法是你陷入的那一种可能有助于解释你如何处理亲密关系。附件类型细分如下:

*   安全:谢天谢地，据说安全的人约占总人口的50%。如果他们不这样做，人类可能会灭亡。如果你是安全的，这通常表明你有反应灵敏的照顾者，他们使分离看起来不那么可怕。这些人不会回避亲密关系，他们对关系也不那么焦虑，可能是因为他们没有和他们有过那么多不好的经历。幸运。
*   **焦虑:** [据](https://books.google.com/books?id=vCh9fXUiUjwC&pg=PT11&lpg=PT11&dq=attached+book+50+percent+secure+20+percent+anxious&source=bl&ots=HzwoZaRdC9&sig=WHMTnjrfS09feaHE6qon9pDMGw4&hl=en&sa=X&ved=2ahUKEwjNuIuNkO_cAhXE5YMKHdByBpAQ6AEwEXoECAAQAQ#v=onepage&q=attached%20book%2050%20percent%20secure%2020%20percent%20anxious&f=false) *附*，焦虑的人约占人口的百分之二十。焦虑的人对亲密关系感到非常舒服——事实上，如此舒服，以至于他们基本上会坐在你的腿上，怀疑如果你走开去拿遥控器，你是否已经不再爱他们了。他们需要大量的安慰，因为他们可能有照顾者无法满足他们的需求。他们也非常敏感，对任何可能出现的问题都高度敏感。
*   回避型:回避型的人被认为是对“分离的照顾者”做出反应，变得难以置信的独立，通常对亲密感到不舒服。*附*说他们约占人口的25%，而且你在大学里都和他们约会过。
*   **【杂乱无章】:**有时被称为“恐惧型”或“焦虑型和回避型”，据说大约百分之五的人有一种令人兴奋的依恋类型组合。真正的爱情过山车。

### 依恋理论的局限性

有很多对依恋理论的批评，因为四个类别似乎不足以涵盖人类的所有弱点。2016年，心理学家兼性治疗师迈克尔伦 [写了](https://www.psychologytoday.com/us/blog/standard-deviations/201608/why-attachment-theory-is-all-sizzle-and-no-steak) 对于《今日心理学》来说，依恋理论过于简单化了:

> ...依恋理论似乎假设依恋是某种适用于全球的整体关系思维图，但最近的研究表明，个人可以以不同的方式依恋不同的人。事实上，孩子可能对母亲有一种安全的依恋，但对父亲有一种回避型依恋，对姑姑有一种焦虑型依恋，等等。

他还指出，这一理论被用来推动人们遵循“正常”关系的特定理念，称其对关系和性欲强加了“武断的、道德主义的社会标准。”

这是一个有趣的观点:例如，唯一健康的关系是一夫一妻制吗？你不想以“正常”的方式安定下来，是不是你有问题？依恋理论似乎暗示了我们都应该尝试走一条路，如果我们没有，那是因为我们的成长过程中有一些错误，而不仅仅是对爱情和关系有一个更开放的态度。

### **依恋理论如何帮助**

尽管如此，对你的倾向有一个基线想法可能是一个潜在的有用的指导，即使你不喜欢你在依恋轴上的位置。首先，大多数人都是不同行为的混合体，你应该试着不要认为任何一种行为本质上都是负面的。例如，一个焦虑的人可能对早期的问题更敏感，因此能够解决它们。一个回避型的人可能善于找到解决难题的方法，并且不会太苛求。归根结底，你的特质和什么样的人最合得来。

对伯奇来说，意识到自己是一个焦虑的人让她意识到她需要和一个安全的人在一起，这个人不会对她对爱的需求做出更多的疏远或鄙视。虽然两个缺乏安全感的人可以约会，但有时和一个安全的人在一起可能会让你更安全，因为你在练习和一个更可靠的人在一起。即使没有成功，这些也是你下一段感情的经验教训。

记者和《依恋效应:探索我们最早的纽带塑造我们的关系和生活的强大方式》一书的作者彼得·洛文海姆也告诉伯奇，搞清楚这一点可能会弄清楚为什么某些关系没有成功，而其他关系却成功了:

> 学习你的依恋类型可以增强你的能力。如果你在焦虑中度过一生，却不自知，那是很难的；比如，你不会理解你们关系中的冲突和挫折。当你学习依恋时，当你被某件事触发时，你可以想，'哦，那是我的依恋风格说话'。你甚至可以想，‘我不需要那样回应’，然后改变你的行为。

基本上，洛文海姆和依恋理论运动似乎仍然鼓励人们反思他们的行为，以及他们可以改变什么，不管过去发生了什么。
