# 阻止 Windows PC 进入睡眠状态的最简单方法

> 原文：<https://lifehacker.com/the-easiest-way-to-stop-your-windows-pc-from-going-to-s-1848626466>

Windows PCs 和笔记本电脑被设计为在您不使用时自动进入睡眠状态。这是一件好事:它有助于节省电池寿命，延长电脑的使用寿命。但是有时候，你不想让它进入睡眠状态，m 可能是因为你想整夜运行后台扫描，或者你想下载一个巨大的文件。

当然，你可以进入设置来完全禁用睡眠设置。但是使用微软自己的 PowerToys 工具有一个更好的方法。这个工具包括一个名为 Awake 的特性，它做了两件非常好的事情。 I t 让你完全禁用睡眠模式，直接从系统托盘中，它让你暂时中止睡眠模式*——*而无需触摸设置应用程序。它是这样工作的。

要开始使用，请到 [PowerToys GitHub 页面](https://github.com/microsoft/PowerToys) 下载该软件的最新版本。这款应用目前为 Windows 10 和 Windows 11 用户提供预览版，对于日常使用来说足够稳定。安装完成后，从系统托盘中打开它(启动应用程序本身没有任何作用)。从侧边栏中，选择“**唤醒**”选项，并点击“**启用唤醒**”功能旁边的切换按钮。

Y 你可以从 PowerToys 窗口自定义这个功能的行为，但是有一个更好更快的方法——u 在添加到任务栏的系统托盘中唱出专用的**唤醒**图标。右键点击任务栏，选择你想要使用的模式。要禁用睡眠模式，使用“**无限期保持清醒**”选项。如果您只想暂时关闭，请使用“**暂时保持清醒**”选项，然后在 30 分钟、一小时和两小时时间段之间进行选择。

虽然配置模式实际上并没有启用这个特性。为此，您需要点击**“保持屏幕在**上”按钮。无论哪个后台进程正在运行，你的 Windows 电脑都将保持清醒。

如果你不想再用 PowerToys(或者它对你来说不够可靠)，你随时可以回到老办法。要阻止你的 Windows 电脑无限期进入睡眠状态，在 Windows 10 电脑上进入**设置** > **系统** > **电源&睡眠** > **睡眠**。如果你运行的是 Windows 11，进入**设置** > **系统** > **电源&电池** > **屏幕和睡眠** > **睡眠**。
