# 病理性说谎和强迫性说谎的区别(以及为什么它很重要)

> 原文：<https://lifehacker.com/the-difference-between-pathological-and-compulsive-lyin-1847787451>

一般来说，说谎不是一种被视为积极的品质。但不是所有的谎言都是平等的。谎言的范围相当广泛，从不会造成任何伤害的小谎言到会造成严重伤害的大谎言。



当一个人倾向于频繁撒谎时，“病态”和“强迫性”这两个词就会被频繁使用。但它们不是一回事。在 [对井+好](https://www.wellandgood.com/difference-between-compulsive-pathological-liar/) 的采访中，临床心理学家 Ramani Durvasula 博士分解了强迫性说谎和病理性说谎的区别。以下是需要了解的内容。

## 病理性**和强迫性**说谎的区别

病理性和强迫性说谎都是出于习惯，这使得很难确定一个人说的任何话是否真正准确，或者涉及某种程度的谎言(或隐瞒信息)。此外， [Durvasula 说](https://www.wellandgood.com/difference-between-compulsive-pathological-liar/) 这两种谎言通常是一个人在童年早期作为对创伤的反应而形成的防御或应对机制。

但除此之外，病理性说谎和强迫性说谎还有一些关键的区别。下面是杜尔瓦苏拉如何分解 :

### 病理性说谎

*   说谎涉及到 m 操纵，目的是为了逃避指责、摆脱困境、保全面子或在某种情况下为所欲为。
*   Durvasula 说，它经常涉及不止一个谎言——特别是“为了获得一些个人利益而精心编织的谎言网”,通常“直接以他人为代价”。
*   说谎者可能缺乏同理心或意识(或两者都有),不知道他们的行为如何影响他人。
*   Durvasula 说，“它最终与自恋倾向或自恋型人格障碍共存”，尽管情况并不总是如此。

### 强迫性说谎

*   这纯粹是出于习惯，通常很少或没有明显的原因。
*   这通常没有(如果有的话)后果。
*   对一个人来说感觉很舒服/熟悉。
*   “强迫性说谎者通常会在低风险或无风险的情况下说谎，可能会说他们去了某个度假胜地，而实际上他们并没有去，或者对他们前一天晚上在电视上看到的东西撒谎，”Durvasula 解释道。

## 为什么它很重要

根据 Durvasula 的说法，当涉及到病态和强迫性说谎时，不仅频率很重要，谎言的类型和这个人的意图也很重要。“当这些小谎言成为常态，并夹杂着更邪恶的资本谎言时，问题就更加明显地转向欺骗和不信任，”她说。

一旦你对某人的信任开始减弱，你可能需要重新考虑你和他们的关系——不管他们是同事、恋人、家庭成员、柏拉图式的朋友还是其他什么。“如果你正在处理任何你需要确保准确的高风险信息，确保从其他人那里获得该情报的确认，” [杜尔瓦苏拉建议](https://www.wellandgood.com/difference-between-compulsive-pathological-liar/) 。

最后，如果你发现一个人在说谎，Durvasula 说应该避免和他们当面对质——尤其是当他们的说谎有病态倾向的时候。这是因为很有可能把他们叫出来会导致对方产生防卫心理和/或否认一切，正如她指出的，你的时间和精力最好花在别的地方获取事实。
