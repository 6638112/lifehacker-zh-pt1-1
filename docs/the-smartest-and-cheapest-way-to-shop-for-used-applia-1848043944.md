# 购买二手电器最聪明(也最便宜)的方法

> 原文：<https://lifehacker.com/the-smartest-and-cheapest-way-to-shop-for-used-applia-1848043944>

在新冠肺炎疫情把全球供应链完全抛入混乱之前，购买家用电器已经够紧张的了。如果你足够幸运，找到了你喜欢的东西，而且确实买得起，那么它很可能要么已经停产，要么已经缺货，要么被搁置在物流中心。



由于这个原因和其他许多原因，人们越来越多地选择二手电器而不是新电器。但是寻找完美的二手冰箱、洗衣机、烘干机或炉子并不比购买新的容易多少——这完全是另一种压倒性的事情。以下是如何在不失去理智的情况下找到你需要的东西。

## 为什么二手电器值得一看

价格是购买二手家电最明显的优势。根据《消费者报告》2020 年 11 月的一篇博文——这是一篇关于整个过程的极好的、全面的指南——你经常可以以低于标价 50-75%的价格买到很少使用的电器。如果你愿意买稍微旧一点的东西，你可以得到更好的交易。

旧电器(想想物联网之前)比它们的价格标签更有价值。与现代版本相比，它们的组件通常更少，这使得它们很少有机会发生故障。此外，这些部件往往既坚固又易于更换，因此当它们出现故障时，它们不会带走整个机器。旧电器仍然会增加维修费用，淹没你的厨房——它们只是不会像电脑化的电器那样以极其愚蠢、不必要、经常是永久性的方式坏掉。(此外，250 美元的二手冰箱比 3000 美元的新冰箱更容易收到意外的维修账单。)

## 重新校准你的期望

在你潜入之前，你应该知道只有一些二手电器容易得到。如果你正在寻找一个炉灶，冰箱，冰柜，洗衣机，或烘干机，你可能会找到一个坚实的二手模型没有太大的麻烦。二手洗碗机肯定是存在的，但可能有点难以追查；像微波炉这样的小东西很难找到完好的，所以你最好买新的。

就风格和审美而言，你的选择将取决于你所在地区流行什么，或者更准确地说，直到最近你所在地区流行什么。大多数二手家电经销商从翻新产品中进货，所以预计会看到很多非常接近全新的东西，很多几年前就过时的东西，而不是其他很多东西。

## 了解您的设备的平均寿命

意想不到的故障总是有可能发生的，尤其是那些曾经风光一时的旧电器。在你买任何东西之前，研究一下它的平均寿命，这样你就知道你会买什么了。一般来说，电器的活动部件越多，接触的水越多，磨损就越快。这就是为什么洗衣机和洗碗机通常比干衣机、冰箱、烤箱和炉灶早几年淘汰。

不幸的是，这已经是最具体的了。像 [消费者报告](https://www.consumerreports.org/appliances/used-appliances-should-you-buy-one-a8356373304/) 和 [这种老房子](https://www.thisoldhouse.com/21018361/how-long-things-last) 这样的网站列出了不同类别电器的平均寿命，但这些都是指导方针，而不是硬性规定。一个电器的寿命取决于它的品牌、型号、年龄，最重要的是它的状况:它使用了多长时间？它的清洁和维护情况如何？它是否暴露在硬水、极端温度或洗涤剂堆积等恶劣条件下？这些只是影响其寿命的一些因素；辨别的唯一方法是仔细观察，近距离亲自观察。

## 总是亲自去购买二手电器

和所有二手商品一样，你应该只从知道自己在做什么的人那里购买二手电器。这意味着坚持选择当地信誉良好的经销商和维修店，避开 Facebook Marketplace 和 Craigslist。(除非你知道如何自己修理电器，在这种情况下，去吧。)好的商店只出售经过适当维修和/或翻新的电器，通常有某种类型的保修——这两样你都不会从 Craigslist rando 那里得到。

如今，很多地方都把库存放在网上，所以浏览一下，感受一下价格、选择和保修。(不要忘记查找提货、送货和安装。)但是到了购物的时候，还是要亲自去。尽可能彻底地测试所有东西是很重要的:要求插上电源，确保每个旋钮、刻度盘和设置都按预期工作。你还可以问，为了准备销售，它进行了什么样的修理。一定要检查外观损坏、奇怪的气味和令人担忧的声音，以及任何使电器更难使用的东西。

在背景调查和实际购物之间，找到一件好的二手家电可能需要一些时间。但是，只要有一点耐心和运气，你就能以很低的价格买到像新的一样好用的东西。
