# 如何发现老年宠物的认知障碍(以及你能做些什么来帮助他们)

> 原文：<https://lifehacker.com/how-to-spot-cognitive-dysfunction-in-aging-pets-and-wh-1847497243>

随着年龄的增长，人类并不是唯一遭受记忆丧失和大脑功能受损的人——这也可能发生在我们的宠物身上。对于狗来说，这被称为认知功能障碍综合征，或 CDS，不幸的是，它的许多症状都被视为“正常”衰老。

当然，精神敏锐度的显著丧失绝不是正常的。为了更好地了解 CDS 的表现和治疗方法，我采访了 Elizabeth Stelow 博士，DVM，DACVB，她是加州大学戴维斯分校的兽医行为学家，负责管理他们的 [兽医教学医院的行为服务项目](https://www.vetmed.ucdavis.edu/hospital/small-animal/behavior-service) 。这几乎是你需要知道的保持你的宠物今天、明天和未来健康的一切。

## 了解认知功能障碍的迹象

如果你不知道要找什么，你就不能帮助你的宠物，所以第一步是知道 CD 是什么样子的。它通常表现出与人类年龄相关的认知衰退相似的症状； [经典标志用首字母缩略词 DISHAAL](https://www.researchgate.net/publication/227855691_Cognitive_Dysfunction_Syndrome_A_Disease_of_Canine_and_Feline_Brain_Aging) 来概括，代表:

*   **定向障碍**
*   与主人、其他宠物和环境互动的变化
*   **睡眠-觉醒**周期紊乱，有时伴有起搏或喘息
*   **房屋脏乱**
*   **活动**的变化，增加或减少
*   增加的**焦虑**
*   学习和记忆会发生变化，比如记不住新的技巧和/或忘记他们曾经知道的技巧

Stelow 博士解释说，诊断标准因情况而异，但一般来说，一只中年狗表现出两种或更多这些症状，表明“高度怀疑”。一些症状是显而易见的:如果你的狗似乎不再认识你，白天突然开始睡觉(晚上四处游荡哀嚎)，似乎忘记了训练过的命令，他们可能正在经历认知功能障碍。

但是 Stelow 博士告诉 Lifehacker，有时候，征兆并不总是很明显。“这[听起来]很随机，但事实并非如此:如果你看到一只狗走到一扇关着的门旁边，站在铰链边等着门打开，假设这是狗的认知功能障碍，”她说。“他们只是走错了门，他们坚持认为门会在那里打开 … 这种情况不会发生在所有有认知障碍的狗身上，但有这种情况的狗总是会有这种情况。”

猫的认知功能障碍看起来有点不同；也就是说，症状列表要短得多。失去家庭训练和增加发声——正如 Stelow 博士所说，“所以，一只猫在家里走来走去，对任何人和任何事都不哭”——通常是猫主人会得到的唯一迹象。这些症状中的任何一个都足以成为检查的理由，但是如果你的猫两者都有，询问潜在的认知原因是很重要的。

## 监控*所有*症状

你不能阻止或逆转 CD，但作为一个宠物主人，你可以做一些事情来帮助。首先也是最重要的是了解所有的症状看起来像什么，而不仅仅是那些会让你的生活变得更困难的症状。“我们告诉兽医学生，让主人对你大喊大叫的[症状]是失去了室内训练……以及睡眠/醒来周期的变化………那些其他的事情，有时他们只是耸耸肩，说，‘好吧，你老了，’”Stelow 博士说。

不要太快将怪异的行为归结为“只是年纪大了”如果你有一只中年或老年宠物，不要等它们开始在地毯上拉屎或让你整夜睡不着，再带它们去看兽医。这些可能是他们仅有的症状，但这个想法是为了确保你不会错过其他症状。

## 尝试益智喂食器、训练和运动来锻炼智力

体育锻炼对宠物健康很重要，但当谈到 CDS 时，Stelow 博士说，精神锻炼也很重要。食物拼图和玩具是我们保持宠物思维活跃的最佳工具之一:“几乎每个人都应该用食物玩具 和拼图来喂宠物——无论如何，这有助于避免大脑停滞，”她说。

同样，Stelow 博士建议进行互动训练，甚至(尤其是)对猫进行互动训练——但是是为了好玩，而不是服从。学习新技能可以让你的宠物的大脑保持忙碌，并且对你们双方都有吸引力。如果你不知道从哪里开始，试着想想你的宠物已经做的可爱或其他令人满意的事情:“捕捉行为很容易，”Stelow 博士解释道。“你不会考虑教猫坐下，但这是它们从事的一种行为。”击掌、握手和叫唤时到来都是 [对猫](https://lifehacker.com/how-to-teach-your-cat-tricks-1818772391) 来说是很好的开始技巧。

当然，狗也可以学习新的技巧，但如果你真的想挑战它们，Stelow 博士建议参与一项运动。你的选择几乎是无限的:敏捷性，气味跟踪，音乐犬自由泳(又名狗跳舞)，码头跳跃，狗冲浪，圆盘狗，犬越野赛(带狗)，雪橇比赛，搜索和救援，仅举几例。即使是比较低调的狗狗运动，也需要巨大的脑力劳动，这正是你想要的。

训练和运动不仅仅是一种精神锻炼——它们还能加强你和宠物的关系，这对它们的精神和情感健康非常重要。他们无法从其他任何人那里获得这种联系:你是他们最好的朋友。

## 改变(或补充)他们的饮食

将你的宠物的饮食改为含有中链甘油三酯(MCTs)的饮食也可能有助于缓解 CDS 症状。MCTs 通常是从椰子油中提取的，多年来一直被用于大脑健康食谱。理论是 MCT 分解成酮，大脑可以吸收并作为能量代替或补充葡萄糖。基本上，MCT(和酮)给大脑提供了另一种能量来源，为各种与大脑相关的活动提供燃料， [这可能对大脑功能因某种原因受损的人有所帮助](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC7699472/) 。

对狗来说也是如此，至少理论上是这样。有限的研究表明富含 MCTs 的狗粮有助于缓解 CDS 症状 。Stelow 博士推荐 Purina Bright Mind，因为它可以在柜台上买到，并且“适度富含”MCT。(更高的水平适用于癫痫治疗，对 CDS 可能是过度杀伤。)不过，市场上有几种富含 MCT 的宠物食谱，所以一定要和你的兽医讨论一下。

补充剂也可以帮助你的宠物，尤其是那些含有抗氧化剂的。Stelow 博士说，Denamarin 和 Senalife 是最常用的大脑支持药物。他们不会让时光倒流，但他们坚定地属于“不会伤害，可能会有帮助”这一类。

## 不要等待

从益智喂食器到补品，所有这些干预的酷之处在于，开始永远不会太晚。“一旦狗或猫到了中年，主人完全可以说，‘我要把你的饮食改成含有 MCT 的东西，我要给你补充抗氧化剂，’”斯特洛博士说。"但是如果你开始看到临床症状，你就要加快速度."她还指出，当宠物主人的同伴步入中年时，他们通常会减少训练和高强度的游戏，而这两者对认知健康都变得格外重要。所以继续和你的老宠物玩:用益智喂食器、互动玩具、训练甚至运动来挑战它们。

尽管特殊的食物和训练很有帮助，但最重要的干预是在宠物的一生中关注它们的行为。这是区分正常和不正常的唯一方法。斯特洛博士说得好:“老年不是一种疾病，”她说。"如果你看到了某种疾病的临床症状，不要称之为衰老。"
