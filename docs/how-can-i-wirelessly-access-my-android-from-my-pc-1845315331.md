# 如何从我的电脑无线访问我的 Android？

> 原文：<https://lifehacker.com/how-can-i-wirelessly-access-my-android-from-my-pc-1845315331>

当你不需要的时候，为什么还要用电缆呢？现在是 2020 年。我们没有理由不能够无线连接到我们所有的设备，无论您是想要传输文件、流媒体，还是使用一个设备调整另一个设备的设置。



如果真的如此简单。本周 Lifehacker 的 [**Tech 911**](https://lifehacker.com/c/tech-911) 建议专栏的问题直击极客体验的核心。今天不要胡闹。正如**莫哈末**所写:

> “没有任何线缆，我如何将我的 android 与 PC 连接起来？”

### 为什么让 Android 和 Windows 一起玩这么难？

相信我，没有什么比当我想访问电脑上的东西时，或者，真的，当我想摆弄我可能不应该安装的 *adb* 命令和测试版时，不得不将我的 Android 与我的电脑物理连接起来更烦人的了。

在一个完美的世界里，Android 会有一个很小的选项，允许你创建一个到笔记本电脑或台式机的直接无线连接。然后，你可以远程控制你的手机，就像你可以使用 [(一个远程桌面应用](https://lifehacker.com/five-best-remote-desktop-tools-1508597379) )在远处的电脑上移动鼠标和输入命令，并且在你的电脑和手机之间传输文件，根本不需要任何电缆。

默认情况下，我们没有这种设置，但是我们可以通过一些帮助来实现。

将 Android 连接到 PC 的一个选择是使用 Windows 的内置设置来连接两者。在最新版本的 Windows 10 中，只需调出**设置>你的手机**，点击**添加手机**即可上手。系统会提示你在安卓系统上安装 [**手机**](http://aka.ms/yourpc) 应用。打开它，用你的微软账户登录(或者用你的 Windows 电脑上的二维码连接两者)并接受所有权限。将两者链接后，您就可以直接从 Windows PC 访问手机的通知、消息和照片。你还可以在 Windows PC 上通过手机拨打和接听电话。

你还应该能够使用你的手机和电脑的“附近共享”功能，通过蓝牙在两者之间交换文件。(设置起来很简单，只需在每台设备上调出你的设置应用，搜索“附近的共享”，然后启用它。)然后，你可以使用 Android 的内置共享功能或 Windows 的文件浏览器中的**发送到>蓝牙设备**菜单来回传输数据。

不过，老实说，我觉得“附近的共享”最令人困惑，而且大多是拼凑的。其他应用程序，如 [**Wifi 文件传输**](https://play.google.com/store/apps/details?id=com.smarterdroid.wififiletransfer&referrer=utm_source%3Dsearchfreeapp.com%26utm_medium%3Dweb%26utm_campaign%3Dsearchfreeapp%2520web) 或[**super beam**](https://superbe.am/for-pc-old)使您的 Android 和 PC 之间发送和接收文件*更加容易。还有像[**Snapdrop**](https://snapdrop.net)这样的网站，它们不需要额外的应用程序来传输文件，或者是久经考验的应用程序[**air droid**](https://play.google.com/store/apps/details?id=com.sand.airdroid&referrer=utm_source%3Dwww_index)和[**Portal**](https://play.google.com/store/apps/details?id=com.pushbullet.android.portal&referrer=utm_source%3Dreddit-playstorelinks__bot)。*

如果你用的是三星的安卓系统，我绝对推荐你去看看该公司的 [**Dex**](https://www.samsung.com/us/explore/dex) 应用，它基本上可以直接在你的桌面上弹出你的安卓系统。它需要一根电缆，这并不能完全满足你的无线需求，但这是一种从 PC 访问 Android 的非常好的方式，我不得不提到它。

[**戴尔移动连接**](https://www.dell.com/en-us/shop/dell-mobile-connect/ab/dell-mobile-connect) 是另一个值得尝试的选择。不幸的是，你需要一台戴尔电脑来安装它...除非你有创意。我的意思是，在你的手机上调出该应用程序的 Windows Store 链接，登录你的 Microsoft 帐户，并选择在你的 Windows 10 PC 上安装该应用程序。(如果你没有戴尔，Windows 10 电脑将无法工作。)

我已经安装好了所有的东西，但是只是在让我的基于以太网的 PC 和无线 Android 互相看到对方时遇到了麻烦。我怀疑，如果你试图将无线笔记本电脑连接到你的 Android，这将更适合你。或许戴尔真的有一些创造性的方法来阻止非戴尔用户尝试该应用程序。(我看过许多非戴尔人士对这款应用的评论，所以这可能只是我这边的一个问题。)

如果你幸运的话，一切都会连接得很好，你可以在 Windows 10 PC 上直接访问你的消息、手机应用程序、文件、*和*共享你的 Android 屏幕。然而，如果你遇到了任何问题，放弃飞船，尝试一些其他远程查看你的机器人的替代方法，如经典的[**AirMirror**](https://play.google.com/store/apps/details?id=com.sand.airmirror&referrer=utm_source%3Dwww_get)或[**Scrcpy**](https://github.com/Genymobile/scrcpy)。

我想这就够了吧？抱歉，这是解决方案的大杂烩，但这就是适合你的 Android。除非你有一部三星手机——这又一次通过 Dex 应用程序使这一过程变得异常简单(尽管是有线的)——否则你将不得不依赖几种不同的解决方案，具体取决于你试图在 Android 和 PC 之间无线地做什么。
