# 你年轻时会花光所有零花钱买一个游戏机

> 原文：<https://lifehacker.com/mod-a-game-boy-your-younger-self-would-ve-spent-their-w-1848672691>

Game Boy 的每一次迭代都会带来新的东西。原版会让你觉得自己拥有了一个真正的 feelGame Boy Color 为移动任天堂游戏引入了 32，000 种颜色；Game Boy Advance 在这两者的基础上进行改进，将优秀的(当时的)图形与新的宽高比和肩部按钮相结合。

Watch

然而，没有一种设备是完美的。许多游戏玩家面临的主要问题是缺乏背光显示屏。游戏男孩、游戏男孩颜色和游戏男孩前进都省略了这个选项，这意味着游戏必须在外部光源下观看才能享受。即使在那时，它也不是理想的，当然也不是我们大多数人想玩的方式。唯一喜欢它的似乎是第三方公司，因为他们在 Game Boy light 配件上赚了一笔。

直到 Game Boy Advance SP 问世，移动任天堂游戏才最终拥有了自己的光源，但即便如此，也伴随着一些复杂问题。任天堂最初发布的 SP 采用前置照明显示屏，比其他游戏机的屏幕要好，但看起来远不如他们后来为 AGS-101 型号提供的背光选项。

但是我跑题了。背光可能是 Game Boys 最大的问题，但在 2022 年仍有很大的改进空间。你知道那些内置扬声器不支持，缺少可充电电池意味着当红灯亮起时，你将上下搜索你的房子。

游戏纯粹主义者可能不感兴趣，但有一种方法可以充分发挥这些复古手持设备的潜力。进入游戏男孩改造的世界。只要你有一个原装的 Game Boy 主板，你就可以把你的主机改装成你喜欢的样子。购买与任天堂提供的颜色完全不同的外壳；加入亮灯按钮，在黑暗中轻松玩转；升级声音芯片，以播放一些很棒的 8- 、16- 和 32 位音轨。

最重要的是，在我看来，用背光、高质量的选项取代旧的、糟糕的屏幕。改装者喜欢在所有型号的 Game Boys 上工作，甚至是带亮灯显示屏的 Advance SP。可定制性的结合，加上这些 mod 带来的生活质量的提高，是一件很酷的事情。

复古的未来是我第一次涉足游戏男孩的世界。颜色组合不是我喜欢的，但我更喜欢选择你自己的游戏颜色。 [这个视频](https://youtu.be/NpxPqkLnE-U) 把我送进了一个 Game Boy mods 的兔子洞，也许对你也一样。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-NpxPqkLnE-U&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-NpxPqkLnE-U&start=0) 

正如你从复古未来的视频中看到的，改装在技术上非常复杂，需要烙铁(以及如何焊接的知识)。虽然有很多 mod 需要焊接，但也有一些不需要。例如，安装新的背光 IPS 面板，无需任何焊接工作。但是，跳过这个过程意味着您将无法控制显示器的亮度。

为了获得完整的改装体验，你需要知道如何焊接。屏幕控制、充电电池、声音升级等。我不知道如何焊接，所以我不能提供这方面的建议。然而，一个显示器的替代品可能是你所需要的，它可以极大地提高你和你的游戏机在一起的乐趣。虽然背光显示器是 OG Game Boy、Game Boy Color 和 Game Boy Advances 的明显升级，但如果你没有 101 型号，你可能也想升级你的 SP。

[这不算](https://youtu.be/mDwLAV4Ny74) 有一个很棒的视频展示了如何修改原始 SP 以适应奇妙的新显示。此外，因为他们的 SP 被划伤，他们用一个可怕的半透明选项替换了外壳。我知道我可以自己组装这个模块，但是，说实话，我愿意花大价钱买它。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-mDwLAV4Ny74&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-mDwLAV4Ny74&start=0) 

你应该注意，如果你不打算更换你的游戏机外壳，你需要修剪显示器外壳边缘的一些塑料，以适应你添加的略大的显示器。如果有必要的话，你参考的指南会带你完成这个过程，但是这可能是一个棘手的过程。因此，你会看到修改者推荐购买预制的外壳。除非你热衷于复古风格，否则你可能会考虑这条路。

现在，除了优秀的 YouTube 视频，还有其他地方可以开始寻找。 [复古改装](https://www.retromodding.com/) 当然是一个值得一试的顶级网站，因为他们拥有你开始这些改装项目所需的所有部件。他们甚至有 [一个教程页面](https://www.retromodding.com/blogs/tutorials) ，带你通过不同的步骤完成特定的改装。 [iFixit 也有工具](https://www.ifixit.com/Store/Tools/Toolkits?utm_source=youtube&utm_medium=sponsorship&utm_campaign=retrofuture&sponsor=retrofuture) 供你用于改装， [包括一个焊接工作站](https://www.ifixit.com/Store/Tools/Soldering-Workstation/IF145-238?o=2) 如果你对更复杂的改装感兴趣。

当然，你不一定要进入 modding 才能找到更好的 Game Boy 体验。花 220 美元，你就可以买到一个模拟口袋，拥有超清晰的背光显示屏、出色的声音、插入电视的能力，所有这些都在一个*的令人敬畏的*设计中。

尽管如此，Game Boy modding 拥有这样的社区是有原因的:摆弄技术和制作自己的东西很有趣。任天堂可能从来没有提供过带亮灯按钮和背光显示屏的透明 Game Boy Advance，但你可以自己组装一个。
