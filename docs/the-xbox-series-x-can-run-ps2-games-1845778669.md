# Xbox系列X可以运行...PS2游戏？

> 原文:[https://kotaku . com/the-Xbox-series-x-can-run-PS2-games-1845778669](https://kotaku.com/the-xbox-series-x-can-run-ps2-games-1845778669)

当微软的下一代游戏机拥有比PlayStation 5更好的PlayStation向后兼容性时，我们生活在一个多么美好的世界啊。

Watch

Xbox和X系列都有一个用户可以激活的开发者模式，可以让你运行UWP(通用Windows协议)应用程序。这款应用程序不仅可以在Windows 10上运行，还可以在Xbox和Windows 10 Mobile上运行。

这包括RetroArch，一个各种模拟器的著名前端，这意味着任何人都可以轻松地使用微软的新游戏机，比如说，PlayStation 2游戏，或者GameCube库中的一些游戏。

 [https://kotaku.com/embed/inset/iframe?id=youtube-video-UGAjhjp8dnI&start=0](https://kotaku.com/embed/inset/iframe?id=youtube-video-UGAjhjp8dnI&start=0) 

[现代复古游戏玩家](https://www.youtube.com/channel/UCjFaPUcJU1vwk193mnW_w1w) 在过去的一周内一直在测试这种仿真变通方法，尽管存在一些问题——在开发模式下，游戏机无法使用它们的全部功率，一些平台上的一些游戏的大小受到限制——但对于这样便宜且可访问的机器来说，这仍然是*难以置信的*仿真性能。
