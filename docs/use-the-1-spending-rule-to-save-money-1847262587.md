# 使用1%支出规则来省钱

> 原文:[https://life hacker . com/use-the-1-spending-rule-to-save money-1847262587](https://lifehacker.com/use-the-1-spending-rule-to-save-money-1847262587)

作为一种帮助抑制冲动消费的方法，许多人在大额购买上有一个“ 24小时等待”规则，但是一个问题是它可能不切实际，因为“大额购买”是不明确的。这就是1%支出法则的作用所在:你仍然会等上一天，但仅限于超过你工资1%的支出。下面是它的工作原理，以及为什么它可以帮你省钱。

## **1%消费法则是什么？**

[最早由美国消费者新闻与商业频道](https://www.cnbc.com/2021/06/24/use-the-1-percent-rule-to-control-spending-and-save-money-says-finance-expert.html) 报道，该法则来自澳大利亚财经播客主持人格伦·詹姆斯 [我的千年金钱](https://www.sortyourmoneyout.com/mymillennialmoney) 。这个想法是，如果你想花钱买一个非必需品，比如一块新手表或一副太阳镜，如果它超过了你年总收入的1%(例如，如果你赚了30，000美元，就要花300美元)，你就必须等一天再买。

24小时的缓冲期起到了冷静期的作用，在这段时间里，你可以问自己最初冲动购物的多巴胺热潮是否影响了你的判断。但是这条规则只有在你真正遵守的情况下才是可行的，这对于小额购买来说可能是个问题。举个例子，如果朋友建议去看电影，你会因为是可自由支配的购买而回家趴在钱包上睡一天吗？

1%规则是一个更合理的方法，同时也提供了清晰的护栏，可以保护你不超支。此外，这很容易记住——如果你税后收入为50，000美元，你就会知道500美元是你1%的门槛。即使你只是改变了一次购买的想法(你可以挑战自己这样做)，这条规则也会帮你省钱。

有些警告是适用的:如果你的收入低于20万美元，你的债务偿还已经是可控的，并且你已经维持了一个跟踪每月非必需品支出的预算(因为一系列较小的购买仍然可能导致超支)，I t效果最好。当然，没有什么是一成不变的，你应该调整规则来满足你的需要。正如格伦·詹姆斯所说:“你可以把它改成0.5%规则。无论比例是多少，都应该基于你的财务状况、需求、目标和优先事项。”
