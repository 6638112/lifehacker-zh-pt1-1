## 什么是 Wandermap？

我们在 [第一次从聪明人那里了解到了 Wandermap](https://www.recomendo.com/)—[马克·弗劳恩菲尔德](https://twitter.com/Frauenfelder?utm_campaign=Recomendo&utm_medium=email&utm_source=Revue%20newsletter) ，他把这个网站作为他的精选之一收录在最近的 [简讯](https://www.getrevue.co/profile/Recomendo/issues/shape-catcher-deep-hyperlinks-wandermap-945026)中。

简而言之，这是一张全球“官方”和用户生成路径的地图，在这里你可以搜索各种新的徒步旅行，或者计划自己的路线。而事实上，路线- 规划工具是 Frauenfelder 最喜欢的网站功能，因为它让用户不仅能看到距离，还能看到沿途的海拔变化。

## 如何使用 Wandermap

免费注册该网站，并保存和存储您的个人路线收集和培训日志。(如果你愿意，你也可以不用注册就可以使用这个网站。)如果你想在一个特定的地方徒步旅行，首先要确保你能在地图上看到它所在的大陆。

当你访问网站时，欧洲是地图上默认的大洲。要搜索北美的徒步旅行，请使用光标移动地图，使北美可见。然后，在搜索栏中弹出你想要的徒步地点，或者继续放大，直到在你选择的地区出现单独的徒步地点。

Frauenfelder 最喜欢的另一个功能是，桌面网站会生成一个你徒步路线的二维码，很容易将其传输到你的手机上，然后开始徒步旅行。您还可以选择导出远足的 GPS 数据。
