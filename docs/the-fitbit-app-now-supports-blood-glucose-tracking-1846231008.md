# Fitbit应用程序现在支持血糖跟踪

> 原文:[https://life hacker . com/the-fitbit-app-now-supports-血糖追踪-1846231008](https://lifehacker.com/the-fitbit-app-now-supports-blood-glucose-tracking-1846231008)

Fitbit now [通过其最新的应用程序更新(在整个二月份推出)，支持美国用户在应用程序](https://blog.fitbit.com/fitbit-blood-glucose-tracking-feature/) 中跟踪您的血糖读数。你的fitbit设备还是不能自己测血糖；只是应用程序中的跟踪是新的。

Watch

Fitbit在他们的 [博客](https://blog.fitbit.com/fitbit-blood-glucose-tracking-feature/) 中宣布了这一新功能，并表示:“除了你的日常健康行为(如活动、营养、睡眠或压力)和其他生物特征(如体重)之外，有一个地方可以跟踪血糖，这对学习新策略和帮助改善你的健康管理非常有价值。”

血糖跟踪对患有糖尿病或某些其他疾病的人很有帮助。新功能不会取代医疗建议；这个想法是为了帮助你认识到你的血糖水平与你的膳食、活动水平和睡眠之间的关系。

如果您已经将OneTouch Reveal应用程序与血糖监测仪配合使用，Fitbit现在可以从该应用程序导入读数。否则，您必须手动记录您的血糖读数。你可以设置应用程序来提醒你这样做，包括直接向你的Fitb it 发送提醒。

如果您订阅了Fitbit Premium(每月9.99美元)，您将能够看到您的水平在过去30天内如何落在目标范围内。您还可以将您的血糖水平纳入健康报告，与您的医疗保健提供者分享。

其他一些更新也在同时推出。用户现在将能够在Fitbit Sense 上跟踪心电图读数，并在Charge 4上直接在手腕上获得SpO2(血氧)读数。

健康指标仪表板以前仅适用于Sense和Versa 3，现在可用于Versa 2、Inspire 2和Charge 4。免费用户可以看到7天的数据，而只有高级用户可以看到30天的视图。
