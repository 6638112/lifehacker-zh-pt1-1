# 这种简单的德州辣椒可以做出令人难以置信的墨西哥馅饼

> 原文：<https://lifehacker.com/this-simple-texas-chili-makes-an-incredible-frito-pie-1845452760>

朋友们，你们好，欢迎来到第一期的 [煎锅辣椒烹饪大赛](https://lifehacker.com/welcome-to-the-skillet-chili-cook-off-1845370924) 。让我先说这件事可能会比我最初打算的持续更长时间。我知道我*说过*我会挑选八(8)份辣椒食谱来制作，边吃边琢磨、但是你只是给我发了太多诱人的参赛作品。



另外，我是网络这个角落的老大，你的美味辣椒食谱我想做多少就做多少！我没有欲望把自己(或者你)限制在八个，我也不会这么做。总之。让我们来看看我们的第一碗美味佳肴:简单、正宗的德州辣椒。

我只知道德州两件事:他们不允许辣椒里有豆子，他们的蜘蛛可不是好惹的。(我出生后，我的家人在德克萨斯州短暂居住，当时我父亲正在检查牛群或其他东西。我被一只蜘蛛咬了一口，直到我十多岁的时候，伤疤才消失。)

“没有豆子”的规则对我来说从来都没什么意义。作为一个喜欢豆类的极端主义者，我不明白为什么添加一种豆类会让这么多人如此不安。但是臭名昭著的 H.A.M .用他们慎重、温和的语气和简单、朴实的食谱改变了我的想法。我是不是受了这个辣椒的“农村”和“工人阶级”的描述的影响？当然——我是来自密西西比州的共产主义者。“必要限量成分”这句话让我对 T2 感到兴奋吗？你打赌它是。我非常喜欢任何人——不管他们的收入如何，也不管他们住在哪里——都能制作和享用的食物。我还喜欢 H.A.M .说我们可以把 pintos 作为一种款待，如果只是作为一种配菜。这种津贴展示了一种灵活但仍有原则的饮食和烹饪方法。

我没必要重新输入 H.A.M .详细清晰的指示。我会把它们放在这里，然后在另一边和你碰面。
