# 这款番茄焦糖布丁是秋季的完美享受

“小荡妇”将是我传记的标题。事情是这样的，即使不是- 所以- 令人敬畏的焦糖布丁仍然相当美味。当你把奶油和糖扔在一起时，很难搞砸。我外出时会点这道菜，因为我一直生活在一种误解之下，认为在家很难做这道菜，但这也完全有可能是我的大脑创造了这个故事，这样我就不会呆在家里，沉迷于焦糖布丁和老电视节目《T4》。但是上个月，当我的厨房柜台无法使用时，除非我把它们的西红柿清理掉，我有了一个主意:西红柿奶油布丁。

焦糖布丁是一种奶油蛋羹，上面放糖，然后融化(燃烧)糖，形成一层硬壳。剩下的是你用勺子打碎的玻璃般的糖面，下面呢？柔滑的蛋羹。

有几种制作蛋羹的方法，我尝试了三种受欢迎的食谱，结果都很相似。最简单的是来自 [莎莉的](https://sallysbakingaddiction.com/creme-brulee/) ，所以这就是我接下来要用的，但是你可以随便找一个配方，然后修改成西红柿。

让西红柿发挥作用的关键是要把它过滤掉。你需要它尽可能的平滑。从一磅成熟的西红柿开始，切碎并混合。我们正在寻找维他麦混合水平。用过滤器过滤，越细越好。现在用小火将番茄缩小，直到体积减半。

根据上面链接的食谱制作食谱，但在开始制作之前，先将西红柿和奶油混合在一起。一旦你把它们混合在一起，在你开始煮奶油之前再过滤一次。

最终的焦糖布丁将呈现出令人愉悦的粉红色，带有奶油的质地，但带有淡淡的番茄酸味和淡淡的香草味。它可以在没有糖皮的冰箱里保存几天，这样你就可以在一周的时间里分享布雷的喜悦。香草、奶油、番茄、和糖的组合对夏天来说太丰富了，但正是你需要的投入到秋天。
