# 在当地哪里捐赠来帮助人们堕胎

> 原文：<https://lifehacker.com/where-to-donate-locally-to-help-people-access-abortions-1848873651>

*Roe 诉 Wade* 案正处于被推翻的边缘，根据一份 [泄露的草案，一份即将到来的最高法院意见](https://www.politico.com/news/2022/05/02/supreme-court-abortion-draft-opinion-00029473) 。之后堕胎权的变化取决于你生活在哪个州。正如 [我们先前解释过的](https://lifehacker.com/what-happens-in-your-state-if-roe-v-wade-is-overturned-1827297706) ( 早在阿米科尼·巴雷特被确诊的时候，还记得吗？)，一些州有“触发法”，这意味着一旦 Roe 下降，堕胎就自动被宣布为非法。其他州仍然保留着前罗伊案的堕胎法，这些法律随后会重新生效。



虽然支持女性身体自主的国家努力可能会在未来几周内收到大量捐款，如果你现在正在寻找帮助的方法，当地堕胎基金和独立诊所是捐款的最佳场所。在已经触发了反对堕胎法律的州，人们需要在堕胎仍然合法的州获得医疗服务，下面列出的组织要么经营诊所，邮寄药物，安排旅行，要么帮助人们获得他们需要的医疗服务。

*   巴尔的摩堕胎基金 帮助住在马里兰州或去马里兰州旅行的人接受护理。
*   [布里奇德联盟](https://brigidalliance.org/) 安排并资助全国范围内的堕胎旅行。
*   [卡罗莱纳州堕胎基金](https://www.carolinaabortionfund.org/) ，面向在北卡罗来纳州或南卡罗来纳州寻求堕胎的人。

*   霍勒健康正义，在西弗吉尼亚州提供堕胎基金、免费紧急避孕以及其他医疗保健和减少伤害。
*   [土著妇女崛起](https://www.iwrising.org/abortion-fund) ，为美加土著人经营堕胎基金(是给任何能怀孕的人的，你不一定是女人)。
*   [简基金](https://fund.nnaf.org/campaign/the-jane-fund-of-central-massachusetts-22/c387834) ，用于马萨诸塞州的堕胎准入。
*   [Just the Pill](https://www.justthepill.com/appointment/) ，为身在蒙大拿州、明尼苏达州和怀俄明州或可以前往这些州的人提供药物流产和其他服务。
*   中西部堕胎联盟 帮助前往、来自中西部或在中西部旅行的人堕胎。
*   西弗吉尼亚州的 [妇女健康中心](https://www.womenshealthwv.org/) ，该州仅存的 [最后一家堕胎诊所](https://jezebel.com/the-last-abortion-clinic-in-west-virginia-1838886688) 。
*   [妇女生殖权利援助项目](https://wrrap.org/) 在全国范围内为堕胎和紧急避孕提供资金。
*   黄锤基金会(yellow hammer Fund)，为阿拉巴马州、密西西比州和周边地区的人们提供堕胎基金、免费紧急避孕和其他帮助。

如果你不能决定，全国堕胎基金网络会在 80 个成员组织之间分配捐款。你也可以在他们的网站上找到一份 [各州的地方基金清单](https://abortionfunds.org/need-abortion/) 。
