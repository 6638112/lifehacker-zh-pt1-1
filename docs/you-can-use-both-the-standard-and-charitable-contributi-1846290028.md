# 使用今年新的慈善捐款减免标准扣除额

> 原文：<https://lifehacker.com/you-can-use-both-the-standard-and-charitable-contributi-1846290028>

当你报税时，不要忽视一项新的 [慈善捐款税收减免](https://www.irs.gov/newsroom/special-300-tax-deduction-helps-most-people-give-to-charity-this-year-even-if-they-dont-itemize) ，它允许你从你的应税收入中扣除 300 美元*，甚至*，如果你已经采取了标准扣除。因为这是一个新的在线扣除项目，所以很容易被忽视——特别是如果你是一个典型的 W-2 员工，倾向于自动报税。



### **什么是慈善捐款扣除？**

作为对 COVID 的回应，CARES Act 在 2020 年纳税季为个人或联合申报人的慈善捐赠设立了 300 美元的新扣除额(可在 1040 表格的 [第 10b 行找到)。年底的立法方案还将这一税收优惠延长至 2021 年，允许已婚并共同申报的人每人获得 300 美元的扣除，总计 600 美元。](https://www.efile.com/tax-deduction/income-deduction/charitable-contributions/#:~:text=You%20can%20enter%20your%20%24300,10b%20of%20your%20Form%201040.&text=Cash%20donations%20to%20donor%2Dadvised,to%2060%25%20of%20your%20AGI.) 

这里关键的是，这是一个 [上线推演](https://www.investopedia.com/articles/tax/11/above-the-line-deductions.asp) 。通常，慈善捐款税收优惠要求你逐项列出你的扣除额，但 [87%的申报人采用标准扣除](https://www.irs.gov/newsroom/special-300-tax-deduction-helps-most-people-give-to-charity-this-year-even-if-they-dont-itemize) ，因此错过了慈善捐赠税收优惠。原因是，统一标准扣除通常是降低你总体应纳税收入的更好选择，如果不是比分项扣除更方便的话(单身申报人的扣除额为 12，400 美元，已婚申报人的扣除额为 24，800 美元)。然而，在这种情况下，你可以收回一些捐给慈善机构的税款，即使是标准扣除。

### **有些限制适用**

该扣除仅适用于现金捐赠(货币、信用卡或借记卡、支票、电子转账)，这意味着你不能申请你可能捐给慈善机构的衣物或食品储藏室的非易腐物品。你也不能扣除你贡献的时间或服务，比如志愿者工作。你也不能要求任何涉及你个人可能受益的物品交换的贡献，如从慈善烘烤销售中购买或购买彩票。

最后，该慈善机构必须是国税局眼中的合格组织。在您申请捐赠之前，使用此 [IRS 查询工具](https://www.irs.gov/charities-non-profits/tax-exempt-organization-search) 查看该慈善机构是否有资格获得免税的慈善捐赠。
