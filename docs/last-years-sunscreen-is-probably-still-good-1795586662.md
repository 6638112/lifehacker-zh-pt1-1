# 去年的防晒霜大概还是好的

> 原文：<https://lifehacker.com/last-years-sunscreen-is-probably-still-good-1795586662>

你的浴室壁橱里还有一瓶去年的防晒霜吗？虽然你可能认为你需要一个新的，但防晒霜比你想象的要持久。如果有有效期的话，检查一下，如果没有，从购买之日起你有三年的时间来使用它。



这个建议直接来自美国食品和药物管理局[，它要求的防晒霜制造商在他们的产品上标注一个截止日期，表明何时将停止提供标签上显示的保护水平。如果没有日期，这意味着测试表明它应该在购买后的整整三年内有效。](https://www.fda.gov/drugs/understanding-over-counter-medicines/sunscreen-how-help-protect-your-skin-sun)

比那个时间更久的防晒霜可能会失去效力，但也可能不会。因为这有点冒险，FDA 建议扔掉超过三年或过期的防晒霜。如果没有失效日期，但你能记得是去年，甚至是两年前买的——比如说，在去疫情度假之前——它仍然可能像新的一样。

当我们谈论储存的时候，美国食品和药物管理局建议我们把防晒霜瓶子放在阳光下，讽刺的是。极热可能会导致产品过早变质，所以他们建议你在海滩时将防晒霜放在阴凉处或用毛巾包裹起来。为了更加安全，你也可以把它放在冰箱里。
