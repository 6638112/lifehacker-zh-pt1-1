# 你应该吃香菜根

芫荽叶是烹饪界最具争议的口味之一。你要么在“这是肥皂”派对，要么在“这是绝对的魔法”派对。(又或许我们只是喜欢肥皂的味道？谁知道呢。有一件事是肯定的——这是一种非常有效的草药。茎和叶有强烈的草药功效，但这种草类植物还有一个更微妙且经常被忽视的部分:根。

## 你如何使用香菜根？

一开始可能看起来很奇怪，但是根已经是我们的东西了。胡萝卜、土豆、甜菜和小萝卜只是我们喜欢用来做菜的一些根茎。芫荽根提供了一种更加柔和的著名风味。如果你想在一道精致的菜肴中加入新鲜的芫荽叶，使用根是一种很好的方式，可以与其他香草一起构建一个互补的交响乐。如果你正在做调味汁或蘸酱，并且你想避免大量绿叶斑点粘在你的牙齿上，芫荽根非常适合这里。特别是如果你对芫荽叶的味道有所保留，用根代替粗而黄铜色的茎可以帮助你感受到它的香味，而不会让你的味觉受不了。

你可以创造性地使用芫荽根，但我最喜欢的一些应用是在咖喱酱，烤卤汁和肉摩擦。虽然你当然可以在这些应用中使用茎和叶，但我发现如果把叶子放在卤汁中烧烤，它们会变干并燃烧。根中有更多的水分(大自然奇特的灌溉系统，又来了)，所以它们可以和其他成分很好地研磨，无缝地融合成糊状。试着用根部给蒸菜，尤其是鱼，增加一种醇厚的香味。准备鱼 [en papillote](https://lifehacker.com/cook-en-papillote-for-efficient-healthy-meals-5489013#!) 和几片切碎的香菜根。这种制剂含有大量的水分，给蛋白质一种保湿的香菜根水疗疗法。制作蘸酱、沙拉、酸奶油或酸辣酱时，往食品加工机里放几根。柔和的香菜味道与各种蘸酱和浇头的配料搭配得很好，所以你和你对肥皂敏感的朋友可以一勺接一勺地继续享用。

## 你如何准备和储存它们？

芫荽根在大型超市里并不容易找到。取决于你住在哪里，你可能更容易在绿色市场、街角商店或各种国际杂货店找到它们。在我住的地方，当地商店只卖切好的芫荽叶，但是街角的乌兹别克杂货店总是有带根的芫荽叶。通常，它们的根部仍然相当泥泞，所以当你准备使用它们时，在茎分裂之前切下根部，好好冲洗一下——非常好的冲洗。用一碗冷水，将浸入根部，擦掉所有污垢，直到根部基本变白，或者感觉不到残留的砂砾。现在可以用研钵和杵研磨，放入食品加工机中，或切碎放入蘸酱中。

一旦你开始采摘芫荽根，你会发现它们温和的香味有更多的用途，所以不要让自己无根可寻。如果你所在的地区这些根很稀少，当你看到它们时就买下来。从茎上切下根部，像我之前提到的那样清洗干净。让根部完全风干，然后储存在容器或拉链- 袋中，然后放入冰箱。它们可以保存两个月。当你准备好使用它们时，打开容器，根据需要取出尽可能多的东西。不需要解冻它们；它们可以立即使用。
