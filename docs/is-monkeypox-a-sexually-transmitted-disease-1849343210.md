# 猴痘是一种性传播疾病吗？

> 原文：<https://lifehacker.com/is-monkeypox-a-sexually-transmitted-disease-1849343210>

在宣布猴痘为国际关注的公共卫生紧急事件时，世界卫生组织周一承认，目前这种爆发集中在男男性行为者中。随后， [世卫组织总干事的声明](https://www.who.int/news-room/speeches/item/who-director-general-s-statement-on-the-press-conference-following-IHR-emergency-committee-regarding-the-multi--country-outbreak-of-monkeypox--23-july-2022) 称“耻辱和歧视可以像任何病毒一样危险。”



许多人对猴痘是如何传播的有误解，某些新闻报道无助于澄清困惑。医学专业人士 [一直在网上](https://twitter.com/Arrianna_Planey/status/1550866549673807877) 打鼓，对抗将该病毒与其他性传播感染混为一谈的错误信息的传播。然而，说猴痘不是性传播感染的努力有矫枉过正的危险。

## 任何人都可能得猴痘

首先:猴痘不像衣原体、淋病或艾滋病毒/艾滋病那样是严格意义上的性传播感染/性病。将猴痘归类为性传播感染将是虚假信息，正如世卫组织的声明所指出的那样，这种框架可能会导致对病毒目前所在人群的危险的污名化和歧视。而此刻，病毒肯定是集中的: [世卫组织称](https://www.who.int/news/item/23-07-2022-second-meeting-of-the-international-health-regulations-(2005)-(ihr)-emergency-committee-regarding-the-multi-country-outbreak-of-monkeypox) 美国 99%的猴痘病例都与男性之间的性接触有关。

将猴痘称为性传播感染可能会导致人们产生一种危险的自满情绪，他们认为只要不与同性恋发生性关系，就不会感染。实际上，猴痘 [可以通过任何形式的密切接触](https://lifehacker.com/we-are-not-ready-for-monkeypox-1849035126) 传播，尤其包括与感染者病灶的接触。越来越普遍的观点认为猴痘只影响 LGBTQ+人群，这是不准确的，也是危险的。

## 猴痘可能不是一种性传播疾病，但它正在像性传播疾病一样传播——就目前而言

所以，让我说清楚:猴痘不是性传播疾病。现在，请允许我说得更细致一些:猴痘目前正在像性病一样传播。一种病毒目前在人与人之间传播的方式应该能够说明如何开展治疗，以及疫情是否得到成功控制。

虽然与“只有同性恋男性”才会得猴痘的污名作斗争很重要，但说它不是性传播感染的努力掩盖了一个事实，即现在需要治疗的人 99%是与其他男性发生性行为的男性。语义很重要——T2 正在确保正确的人群获得疫苗的主要途径。

重要的是要记住，病毒通常会在传播到普通人群之前袭击特定的社区。还有 [几乎没有证据表明猴痘会无限期地局限于](https://www.cnbc.com/2022/07/25/who-monkeypox-could-spread-well-beyond-men-gay-bisexual-communities.html) 同性恋、双性恋和酷儿男。目前受打击最大的人群之外的人不应该感到自满；不能保证病毒不会在更广泛的人群中爆发。
