# 长 COVID 的人什么时候能恢复锻炼？

> 原文：<https://lifehacker.com/when-can-people-with-long-covid-return-to-exercising-1848399068>

在任何身体挫折后恢复锻炼是困难的。你相信自己会听从自己的身体，但在你完全康复之前回到那里也很有诱惑力。然而，过早回到你的健身常规会导致弊大于利——如果你正在处理新冠肺炎的长期影响，这尤其如此。



据《T4》中的 [迈克尔·弗雷德里克森(Michael Fredericson)](https://profiles.stanford.edu/michael-fredericson)医学博士说，“在拥有 [【新冠肺炎】](https://www.everydayhealth.com/infectious-diseases/covid-19/) 后恢复体力活动有一层额外的复杂性，因为潜在的并发症，如 [心肌炎](https://www.everydayhealth.com/myocarditis/guide/) ，“这是心肌的炎症。考虑到围绕“长 COVID”的所有不确定性和复杂性，需要格外小心。

在一个肯定的案例之后，可能无法区分长 COVID 和“常规”(yikes) COVID 的案例，后者需要很长时间才能解决。除了对你身体的伤害，还有精神上的挫败感，感觉你正在远离你的健身目标。下面，我们来了解一下我们对长乔维德的了解和不了解，对后乔维德运动员的推荐现状，以及如何在心理上应对健身挫折。

## 关于长 COVID，我们有很多要了解的

无论你是一个认真的运动员还是一个偶尔去健身房的人，没有人想成为一个经历长期病毒症状的 COVID“长途运输者”。这些挥之不去的并发症包括动脉血栓和心脏炎症。一些运动员也报告了疲劳和呼吸问题，这些问题在他们从新冠肺炎康复后仍然存在。

不幸的是，long COVID 仍然很难定义，需要更多的时间来适当地研究和理解它。长 COVID 的症状与其他疾病的症状重叠，包括 [过去被称为慢性疲劳综合征](https://me-pedia.org/wiki/Chronic_fatigue_syndrome) 。就目前而言，我们有行使后 COVID 时保持谨慎的一般指南。

## **COVID**后恢复锻炼的指南

以下 [回归运动指南](https://link.springer.com/article/10.1007/s11420-020-09777-1) 是由纽约特殊外科医院的医生团队创造的。(作者之一乔丹·梅兹尔也在《纽约时报》上 [写了这些指导方针背后的推理。)在这些建议中:](https://www.nytimes.com/2020/11/17/well/move/exercise-covid-recovery-complications-clots-heart-lungs.html)

*   如果你之前身体健康，并且有轻微的病毒感染，你可以考虑在无症状七天后恢复锻炼。
*   预计需要一个月的时间才能回到你的完整训练计划中。
*   如果你有心脏或肺部疾病，请在恢复锻炼前咨询医生。
*   如果你的症状再次出现，特别是胸痛、发烧、心悸或呼吸困难，停止锻炼并去看医生。

在我们之前的 [报道中，Lifehacker 的高级健康编辑](https://lifehacker.com/how-to-safely-return-to-exercise-after-youve-had-covid-1845709729) [贝丝·斯克沃莱茨基](https://lifehacker.com/author/bethskw) 解释道

> 逐渐增加运动量的建议是从你正常运动量的一半开始(所以，如果你是跑步者，这可能是你正常运动量的一半)。这些指导方针并不具体针对举重，但作者指出了一套非针对 COVID 的指导方针 ，同样建议当你回到举重房时，从轻松开始，然后逐渐增加。

纽约市 HSS 的初级运动医学医生、论文的合著者詹姆斯·n·罗宾逊(James N. Robinson)医学博士 说，这些指南是在 delta 变异出现之前编写的，但它们仍然适用于各种变异引起的病例。

## **对运动员的其他建议**

在 [这篇综合了当前对运动员](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC8567919/) 在新冠肺炎之后安全恢复体力活动的建议的文章中，研究人员发现，专家的共识是，在症状出现之前至少休息 10 天，包括最少 7 天无症状。到那时，锻炼的“分级回归”就可以开始了。

如果你正在寻找一些具体的基准来缓解你的日常生活，这里有一些来自美国运动医学院 :
的提示

*   确保您可以轻松地进行日常生活活动，并在平坦的表面上行走 500 米，而不会感到过度疲劳或呼吸急促。
*   最初的身体活动应该包括 15 分钟的轻度锻炼。
*   如果达到 COVID 后的能量水平，可以增加活动持续时间，然后恢复体重锻炼，如瑜伽或充分休息的阻力训练。
*   随之而来的是更大的阻力和运动专项训练。

## **应对长期症状**

每个人从新冠肺炎病中恢复的速度都不一样，目前还没有公式可以准确地确定一个人应该如何以及何时恢复活动。

长途运输者经历了与无法消除的症状斗争的独特挫败感。你听过“倾听你的身体”和“相信你的直觉”的建议，但是当你的身体拒绝回到最佳状态时怎么办？

当我们等待更多关于如何处理长 COVID 的研究时，它可以帮助我们处理你的无形症状，如疲劳，就像它们是有形的伤害一样——比如说，脚踝骨折。你不会像往常一样通过“推它”和敲打路面来治疗脚踝骨折。相反，你会明白留出完全恢复的时间对于你的长期目标是必要的。

同样，它也不会帮助你“度过”新冠肺炎的症状。目前的医学共识是等到所有症状消退，但我知道当你的症状看不到尽头时，这听起来不太现实。因此，如果你仍然遭受疲劳和头痛的长期问题，试着练习你对脚踝骨折时同样的温柔和强迫的耐心。你现在需要恢复，这样从长远来看，你就能回到你原来的样子。。
