# 对于小孩子来说，这些是最好的非糖果复活节篮子填充物

> 原文：<https://lifehacker.com/these-are-the-best-non-candy-easter-basket-fillers-for-1846454833>

我不信教，但我一直喜欢庆祝世俗版的复活节。感觉就像它标志着我们的正式从冬天走出来，所有的 小鸡，彩色鸡蛋，花餐巾，和咧着嘴笑的兔子。我来这里是为了这一切——今年，意味着为我蹒跚学步的侄女和侄子制作篮子。



他们一个三岁，一个一岁，这意味着我正试图避开任何与巧克力或软糖豆有关的东西，相反，我在寻找非糖果的食物。我对我所取得的成功感到相当高兴，把所有这些想法都藏在心里对我来说是不合适的，所以今天我在这里与你分享它们。

#### **篮子本身**

**像我们大多数人一样，你可能每年都会为你的孩子准备一个专用的篮子，这很正常，也很好。但是如果你想给两岁以上的孩子换个环境(或者像我一样，这是给别人的孩子)，我有个建议。 [这个 Bilibo 座椅](https://www.moluk.com/bilibo.php) 是我儿子拥有过的最神奇的东西之一，它也可以做一个漂亮的篮子来装他们所有的零食。**

**在 之前，我已经写过关于 Bilibo 的文章，因为它不仅仅是一把看起来时髦的椅子——它是孩子们能够融入任何数量的想象场景的东西。大约 30 美元，这绝对是一个挥霍；但它非常适合需要摇摆和旋转的烦躁的孩子，他们也会以各种非传统的方式玩它，让你大吃一惊。**
