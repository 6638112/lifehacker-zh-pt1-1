# 今年每个人都应该参加的最佳免费在线课程

你大学毕业，开始职业生涯，一往无前的日子，早已远去。终身学习不仅仅是一个朗朗上口的短语——它是生存的一个重要方面。即使粗略地看一下过去几十年，也会发现事情变化得有多快，不仅仅是在你的实际工作方面，甚至是在我们工作、合作和衡量成功的方式方面。

在人生的某个时刻，你很有可能会换工作——根据美国劳工统计局的数据，24 岁以下的人平均会换 5.7 次工作，45 岁以上的人平均会换两次工作。随着我们经受住“伟大的辞职”，这种步伐可能会加快，这意味着如果你想在未来取得成功，你需要保持你的硬技能和软技能都非常敏锐。

幸运的是，有大量的资源可以帮助你，包括各种各样的免费在线课程，可以帮助你跟上时代。像 [Coursera](https://www.coursera.org/courses?query=free&page=1) 、 [Udemy](https://www.udemy.com/courses/free/) 和 [edX](https://www.edx.org/) 这样的网站都提供了大量的免费课程，其中许多来自顶级学院和大学(如果你认为这很重要的话，还包括常春藤盟校)。这些课程完全是在线的，通常可以按照你自己的进度来学习，它们包括了几乎每个人在未来几年都需要的技能，不管你从事什么职业。

## 参加这个免费的物流和项目管理课程

任何在过去几年中从事专业工作的人都知道，几乎每个人都被期望成为一名项目经理。掌握物流和项目管理技能不仅仅会让你在职业上受益——我们的个人生活也变得越来越混乱，远程工作意味着我们要按照相同的时间表处理从上午的会议到下午的洗衣等所有事情。

要掌握一切，阿德莱德大学的 [【项目管理简介】](https://www.edx.org/course/introduction-to-project-management?index=product&amp;queryID=676552a43a53a5b2bf070fa4f82a7330&amp;position=1) 这样的课程是理想的。这是一门百分之百免费的课程，它将向你传授项目管理的基础知识，并教你如何在工作和个人生活中运用这些知识。这包括如何确定优先顺序，如何确定任何项目的范围，以及如何制定可管理的时间表和现实的成本。

## 参加这个免费的解决问题和决策的课程

经理喜欢能够独立解决问题的人。成为一名问题解决者需要自信和果断的能力——但这还需要经验和理解。另一方面，觉得自己不够资格、不断做出糟糕决定的感觉是普遍存在的。

一个可以帮助你的免费在线课程是 [“有效解决问题和决策”](https://www.coursera.org/learn/problem-solving) 本课程将帮助你培养批判性思维能力，这是解决问题的基础。它还将训练你识别可能导致决策瘫痪的精神障碍、风险分析和其他在评估问题和制定解决方案时有用的技能。

## 参加这门编程和计算机科学的免费课程

你不一定需要成为一个专家计算机程序员，可以在你的午餐时间快速开发一个应用程序，但是你越来越需要理解这些工具是如何概念化、开发和部署的基础——如果仅仅是因为你越来越有可能在某个时候与开发团队互动。

对计算机科学的扎实介绍将为我们生活的未来打下坚实的基础。哈佛大学的 [《计算机科学导论》](https://www.edx.org/course/introduction-computer-science-harvardx-cs50x?source=aw&awc=6798_1642083819_ab6f9cba0c5c3c5b1d0fcde61f327b51) 是很棒的选择。它概述了所涉及的概念技能(包括算法思维、抽象、数据结构、软件工程和 web 开发)，以及编程语言和开发平台等实用技能。本课程结束后，当人们开始讨论开发软件工具所需的资源时，你将能够跟上。

## 参加这个关于正念和快乐的免费课程

你永远不要低估软技能的重要性，其中最关键的是你自己的心理健康和幸福。随着越来越多的人发现自己一直在同一个受限的空间工作、睡觉和生活，这变得更加重要。当你从不离开你的公寓(或穿上裤子)时，很容易承包空间疯狂。

幸福和正念不是偶然的幸运——它们背后有一门科学。加州大学柏克莱分校提供 [【幸福的科学】](https://www.edx.org/course/the-science-of-happiness-3?source=aw&awc=6798_1642085169_67a602dbb3fadc907592b798b3af1ba8) 积极心理学教育，这是一个新兴的概念，教你培养心理和情感健康的实用策略。在一个越来越遥远的世界里，确保你的精神和情感健康将比以往任何时候都更加重要。
