# 现在不是去美国土著民族旅游的时候。

> 原文：<https://lifehacker.com/now-is-not-the-time-to-travel-to-indigenous-nations-in-1845607152>

随着新冠肺炎疫情迎来公路旅行的回归，像公园和其他自然奇观这样的地方已经有了很多交通。这包括美国本土许多受欢迎的景点，如羚羊峡谷、哈瓦苏瀑布和冰川国家公园。

到目前为止，你可能(希望)知道在计划旅行时要查看州和当地的旅行限制。但是，在美国的某些地区，检查并遵守当地土著民族 设定的 [限制也很重要。现在，许多人要求我们推迟访问他们的领土，因为 COVID 的关注。以下是需要了解的内容。](https://www.fodors.com/news/coronavirus/these-nations-within-america-have-a-reminder-for-travelers)

### 什么是部落主权？

如果你的高中政府课忽略了(或完全跳过)部落主权这一课，这里有一个快速复习。据 [美国印第安人全国代表大会](https://www.ncai.org/about-tribes) 统计，美国有 574 个联邦承认的印第安部落。

其中大约 229 个在阿拉斯加，其余的分布在其他 35 个州。这些 [土著民族中的每一个都是主权](https://www.ncai.org/about-tribes) ，这意味着他们是自治的，他们的法律独立于他们领土所属的国家。

### 土著民族和新冠肺炎

像美国其他有色人种社区一样，土著民族也受到了 COVID 的严重影响。但早在疫情之前，系统性健康差异就意味着 [土著民族的公民在许多类别中的死亡率](https://www.ihs.gov/newsroom/factsheets/disparities/) 高于其他美国人，包括慢性肝病和肝硬化、糖尿病、意外伤害、攻击/杀人、故意自残/自杀和慢性下呼吸道疾病。

新冠肺炎也不例外，根据疾病控制中心的数据，土著民族的感染率是白人的 3.5 倍，包括越来越多的年轻人感染了这种病毒。为了帮助减缓其社区内的传播，土著民族正在行使他们的主权——以旅行限制的形式——来保护他们自己的健康。

### 了解旅行限制

如果你计划访问土著民族的任何目的地，请先查看他们的 COVID 旅行规则。例如， [纳瓦霍国家公园和娱乐](https://navajonationparks.org/) 目前已经关闭。公园 [的媒体代表 Louise Tsinijinnie 告诉 Fodor 的](https://www.fodors.com/news/coronavirus/these-nations-within-america-have-a-reminder-for-travelers) “呆在家里，注意安全”。“这是为了让人们将来能来，这样我们就能为每个人保护人们和我们的空间。”

在某些情况下，我可以在他们的网站上找到土著民族的限制条款，例如 T2、黑脚族、黑脚族、哈瓦苏派、霍皮族、霍皮族、黑脚族和黑脚族。一些州的旅游网站，像[新墨西哥](https://www.newmexico.org/places-to-visit/native-culture/pueblos-tribes-nations/) 和 [南达科他州](https://sdvisit.com/covid-19-travel-restrictions-tribal-lands-south-dakota) 也是有用的资源，提供了他们土著民族的旅游限制的链接。

不管是不是疫情，花点时间了解一下你打算去的任何土著国家的法律都是一个好主意，而且在那里的时候要有礼貌。
