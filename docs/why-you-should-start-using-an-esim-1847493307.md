# 为什么您应该开始使用eSIM

> 原文:[https://gizmodo . com/why-you-should-start-using-an-esim-1847493307](https://gizmodo.com/why-you-should-start-using-an-esim-1847493307)

许多较新的智能手机支持esim以及我们多年来一直在设备中插入的物理SIM卡。如果您想知道eSIM到底是什么，以及是否应该使用eSIM，本指南就是为您准备的。

众所周知，SIM卡给你一个号码和一个身份来连接到蜂窝网络。SIM代表用户身份(或识别)模块，这张卡本质上是给你一张进入运营商网络的通行证。sim卡可以在手机之间切换，只要手机解锁或锁定到同一运营商。

这些年来，模拟市民除了变得越来越小之外，并没有太大的变化，但是现在我们也有了模拟市民。“e”代表嵌入式，这给了你一个关于它如何工作的线索。这是一个内置在手机主板上的SIM卡，所以不用担心物理卡。一切都通过软件处理，从理论上来说，这将使一切都变得更加容易——从设置到转换运营商。

越来越多的手机现在提供eSIM支持，通常还有传统的SIM卡插槽。[iPhone 12](https://gizmodo.com/the-iphone-12-feels-like-a-1-000-phone-1845478737)[Pixel 5](https://gizmodo.com/google-is-bucking-trends-with-the-pixel-5-and-its-awes-1845386771)[Galaxy S21](https://gizmodo.com/samsungs-galaxy-s21-is-the-high-end-android-phone-to-bu-1846154811)都启用eSIMs。设置eSIM通常包括扫描二维码，然后引导您在设备上注册手机号码和个人资料。以下是开始的方法。

在iPhones上，从设置中，点击**蜂窝数据**，然后**添加蜂窝计划**即可开始。在stock Android上，打开设置并选择**网络和互联网**，然后点击**移动网络**旁边的 **+** 图标——你会在底部看到一个**下载SIM卡**的选项。

如果您的手机同时具有eSIM和传统的物理SIM卡插槽，在大多数情况下，您可以在同一部手机上操作两个号码，因为您在同一个设备中同时具有标准和嵌入式SIM卡。这是您可能希望建立eSIM的主要原因之一。

eSIM在旅行时也有很多好处。由于无需担心物理SIM卡，无论您身在哪个国家，您都可以快速轻松地注册本地eSIM套餐。这也不仅仅适用于手机——一些便携式wifi热点设备带有eSIM功能，因此在国家之间切换只是安装新配置文件的问题。没有必要去商店或者等待包裹的到来。

虽然你在手机上一次只能使用一个eSIM(至少目前是这样)，但是一旦你设置好了，就可以很容易地在不同的情景模式之间切换——同样，这对于在不同的国家或语音和数据计划之间切换也很有用。您可以使用最适合您当前情况的eSIM。

当谈到转换计划或转换运营商时，使用eSIM一切都很简单。移动只需几秒钟，嵌入式硬件就会被重新编程，然后您就可以再次准备好——无需用弹出工具戳您的手机，无需打开手机上的插槽，也无需混淆哪个是旧SIM卡，哪个是新SIM卡。

这对制造商也有好处，因为即使是最小的物理sim也只占一小部分空间。SIM卡插槽的额外空间可以用于其他用途，或者被切掉以使整个设备更小。

然而，并不是每个人都喜欢eSIM方法。有一种观点认为eSIM更容易匿名使用或使用假凭证，这就是为什么在中国iPhone 12配备了两个SIM端口，而不是一个标准的SIM端口和一个eSIM。大多数手机仍然在eSIM选项旁边包括一个普通的SIM卡插槽，所以行业还没有完全投入虚拟选项。

至少有一种情况下eSIM可能更难使用:在设备之间切换。有了传统的SIM卡，你可以把它从一部手机里取出来，放到另一部手机里，然后继续使用；使用eSIM，当您激活新手机并停用旧手机时，需要花费更多的时间和精力。这究竟有多困难，取决于提供eSIM的是谁。

如果你想开始使用eSIM，你需要找到一部支持该技术的手机(如上所述)和一家向你出售eSIM的运营商或公司。在美国，你现在可以从所有的大运营商那里获得eSIMs，也可以从类似于和 [Ubigi](https://cellulardata.ubigi.com/mobile-devices/get-esim/) 的公司那里获得eSIMs(这些卖家也会给你很多关于兼容设备的建议)。

预付费eSIM计划比按月付费的计划更难找到，但如果你花足够的时间寻找(取决于你的国家)，这两种类型都可用。大多数中档和经济型手机都不支持eSIM，但有迹象表明该技术在未来将会得到更广泛的应用。

现在，如果eSIM是您的一个选项，没有理由不接受它——特别是如果您希望能够在不同的电话计划之间快速切换，或者在手机上一前一后使用两个号码，或者花费大量时间在其他国家旅行，并且需要尽可能灵活。如果出于某种原因，你更喜欢在使用相同SIM卡的设备之间切换，那么尽一切办法坚持使用物理卡一段时间。
