# 运动前吃止痛药真的不好吗？

我在一个支持布洛芬的家庭中长大。直到今天，我父亲坚持认为“一点止痛药不会伤害任何人。”但这并不完全正确，这也绝对不是Lifehacker对服用止痛药的新闻立场。

虽然标题为“ [为什么你不应该在健身前服用止痛药](https://www.menshealth.com/fitness/a19525580/why-you-should-never-take-painkillers-before-working-out/) ”的文章让我觉得是在散布恐惧，但我发现这些标题已经渗透到我的潜意识中。当我在进行马拉松训练时，我经常忍不住服用某种NSAID(非甾体抗炎药，如布洛芬)来抵御超长距离跑步带来的潜在疼痛。但在我伸手去拿止痛药瓶之前，那些“千万不要吃止痛药！”标题在我眼前闪现。

锻炼前吃止痛药好不好？如果不好，到底有多糟？

## **运动前服用止痛药的所谓风险**

你可能听说过服用过量的布洛芬会导致肾脏损伤。谈到运动员的健康，纽约时报2017年 [的一篇文章声称，定期服用流行的非甾体抗炎药可能“在长时间的运动中使肾脏负担过重，并降低运动后肌肉的恢复能力”。NYT的结论基于两项不同的研究:一项是对超级马拉松运动员的研究，另一项是对老鼠的研究。](https://www.nytimes.com/2017/07/05/well/move/bring-on-the-exercise-hold-the-painkillers.html)

我们已经 [解释过](https://lifehacker.com/how-to-work-out-when-you-re-still-sore-from-the-last-wo-1790803145) 虽然像布洛芬这样的止痛药在运动员中很受欢迎，但是所谓的好处可能不值得潜在的成本。事实上，另一项发表在美国国家科学院 [学报](http://www.pnas.org/content/114/26/6675.short) 上的2017年研究发现，非甾体抗炎药阻碍肌肉再生并导致肌肉衰弱——至少，这发生在进行研究的小鼠身上。NYT指出了这个老鼠的事实，称“研究结果表明，对于人类来说，消炎止痛药也可能会轻微损害肌肉在艰苦训练后的再生和强化能力。“当你服用足够的非甾体抗炎药来缓解疼痛时，你也在服用足够的非甾体抗炎药来干扰你的康复。

## **为什么风险低、回报低**

根据由 [SELF](https://www.self.com/story/ibuprofen-before-workouts) 撰写的那篇NYT文章的分类，有几个原因可以让你不必对潜在的肾脏和肌肉损伤过于惊慌。我鼓励你阅读 [SELF的完整分析](https://www.self.com/story/ibuprofen-before-workouts) ，但简而言之:NYT引用的研究可能不适用于你，一个非超级马拉松运动员，非老鼠运动员，可能非常普通(但人很好)。

[美国运动医学医学学会](https://www.sportsmedtoday.com/pain-control-va-232.htm) 的官方立场是，非甾体抗炎药和对乙酰氨基酚不应用于预防损伤，但可以在活动前谨慎使用，以减轻轻微的运动相关疼痛。

不过，请记住，无论你通过锻炼前止痛来寻求什么好处，不管怎样，都可能是安慰剂。阿巴拉契亚州立大学人类表现实验室 [主任大卫·c·尼曼](https://www.self.com/story/ibuprofen-before-workouts) 博士告诉SELF 布洛芬的使用对他的实验室中的运动员如何评定他们的感知消耗、比赛时间、肌肉损伤和疼痛没有影响。因此，虽然你可能没有将肾脏置于迫在眉睫的危险中，但你也没有给自己带来太多好处。

## **运动前对非甾体抗炎药的底线**

任何药物都有风险；拿的越多，风险越大。在你运动前服用NSAID之前，请记住，科学表明，它没有显著的生理益处，但有潜在的长期缺点(如 [、胃溃疡和肝功能衰竭](https://www.healthline.com/health/pain-relief/ibuprofen-advil-side-effects#internal-bleeding) ) 如果你持续服用太多而导致太多。

如果持续的疼痛是锻炼过程中的一个问题，考虑向专业人士寻求帮助，而不是定期自我治疗。如果你需要止痛药来度过一天，它是一种短期的治疗方法，但是不要服用超过你需要的量。
