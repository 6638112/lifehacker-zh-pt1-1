# 如何停止为某事焦虑不安

> 原文：<https://lifehacker.com/how-to-stop-stewing-about-something-1849662660>

每个人都有自己的众所周知的按钮，当按下时，真的会引爆它们:所谓的“痛处”,会引发特别强烈的反应。我们身边的人通常都知道这些按钮，但往往是陌生人或其他不了解我们的人在不知不觉中按下了它们。

Watch

不管是谁或什么让我们生气，如果我们把某件事看做是针对个人的——即使它不是有意冒犯我们的性格或行为——除了让我们在那一刻心烦意乱之外，愤怒和烦恼还有可能挥之不去。

但是当我们放任某事时，它会消耗我们大量的时间和情绪能量，影响我们的情绪，并潜在地提高我们的压力水平。这里有一些方法来停止焦虑，继续你的一天(和生活)。

## 如何停止炖肉

我们可能无法控制别人的行为，但我们可以减轻影响，确保即使某件事让我们感到不安，我们也不会浪费时间和精力去纠结它。这里有几种方法可以做到这一点:

*   **花一分钟确定问题**:一旦你意识到你开始为某事烦恼， [暂停一会儿](https://www.npr.org/2022/09/23/1124806233/when-something-feels-personal-heres-how-to-cope) 确定是什么让你生气，为什么。
*   **编一个背景故事**:如果犯罪者是一个陌生人， [编一个背景故事](https://www.spiritualityhealth.com/articles/2017/01/16/how-not-stew) 来解释他们的行为。也许那个人在高速公路上拦住了你，因为他们正赶去医院，等等。
*   **以第三人称**思考事件:在自己和事件之间制造这种的心理距离，可以帮助你获得清晰和透视。

最后，请记住，冒犯你的人可能没有想到这件事。事实上，他们甚至可能没有意识到他们做了让你难过的事情。这可能会让整件事变得不那么个人化，并且有助于停止焦虑不安的循环。
