# 如果你在室内烹饪，为什么你需要低脂肪的汉堡肉

> 原文：<https://lifehacker.com/why-you-need-a-lower-fat-burger-meat-if-youre-cooking-i-1845172059>

我不会说烧烤季节已经结束了。实际上，我认为在有点冷的时候在外面做饭比在酷热的时候在外面做饭更令人愉快。但大多数人更喜欢在阳光下烧烤汉堡和热狗，许多人(尤其是那些生活在容易下雨地区的人)将在未来几周内转向室内烹饪。



这显然并不意味着一个人应该放弃汉堡和热狗。两者都可以在里面做得很好——事实上，比起烤汉堡，我更喜欢用平底锅做的碎汉堡 。然而，这确实意味着，你可能要考虑使用一种更瘦的汉堡混合料。

当你在烤架上做汉堡时，脂肪会从汉堡上滴下来，穿过烤架，进入火焰。从更高的脂肪含量开始(*至少* 80/20)在这里是有意义的，因为在烹饪过程中你会损失一部分。但是当你在室内，在炉子上的锅里做汉堡时，油脂会留在锅里。汉堡用自己的脂肪烹饪，这有助于形成美味的外壳(在我看来，这导致了更美味的汉堡)。

当你把汉堡带进去时，如果没有调整脂肪含量，会导致两件事:油腻的汉堡(这不一定是坏的，但有些人不喜欢它)，和大量的吸烟油脂(气味难闻，会触发你的烟雾警报)。仅仅减少 5%的脂肪就可以帮助减少油脂，同时仍然提供足够的脂肪来制作美味多汁的汉堡。

如果你的杂货店没有 85/15 的混合物，买一半 80/20 和一半 90/10，然后混合在一起，或者问屠夫他们是否可以为你创造一种混合物。如果你有一台绞肉机，现在是试验向你的卡盘添加更多美味但更瘦的肉的最佳时机。盘子、侧腹和小腿的瘦肉/脂肪比率都在 90/10 或以上，还有一些真正有趣的、餐馆质量的品尝记录。
