# 这就是如何用漂白剂安全地保存南瓜

> 原文：<https://lifehacker.com/this-is-how-to-safely-preserve-pumpkins-with-bleach-1849616721>

最近社交媒体上有很多关于让南瓜更持久的方法的争议——用稀释在水中的漂白剂擦拭南瓜。虽然使用纯漂白剂是不明智的，因为它会对南瓜产生干燥作用，并对周围的土壤或表面产生潜在影响，但一茶匙漂白剂混合一加仑水一旦干燥，就不会像许多批评者所说的那样对动物有毒。



## **漂白剂湿的时候有毒，干的时候安全**

用稀释的漂白剂擦拭南瓜很有效，因为它杀死了南瓜表面的细菌，减缓了它的分解过程。这也有助于阻止霉菌生长，延长收获南瓜的保质期。如果你的南瓜表面有盐残留，你可以用干净的湿布擦去盐结晶。

漂白剂的化学名称为 NaClO，在液体状态下是有毒的，但一旦干燥，剩余的残渣就会变成无害的盐(NaCl)。如果漂白剂残留像一些人担心的那样有毒，想想用它来清洁你的工作台面或浴室水槽会有多危险。虽然漂白剂如果使用不当会很危险，但它的残留物不会像食盐一样杀死野生动物。

## 与漂白剂有关的愤怒

如果大量的漂白剂被吸收到土壤中，会导致更高的盐度，导致许多植物物种的枯萎，所以在地面上倾倒或混合漂白剂不是一个好主意。有些人甚至使用漂白剂作为除草剂，但它是非特异性的，这意味着它会杀死任何接触到的植物。如果你不小心洒了一些，用大量的淡水冲洗该区域，以避免给当地的植物带来问题。

此外，不稀释漂白剂或在封闭环境中使用漂白剂会产生有毒气体，对人类和宠物都有危险。不用说，你不应该喝漂白剂，即使是稀释的形式，因为它作为液体仍然是有毒的。然而，用一茶匙比一加仑的漂白剂和淡水混合在一起，用在你的南瓜上应该相对安全，因为只要你不把它弄到眼睛或嘴里。(使用漂白剂时戴上手套也是一个好主意，这样可以防止手变干。)如果你的皮肤沾上了漂白剂，用清水彻底冲洗。混合漂白剂时，如果不确定使用的比例，请遵循美国疾病控制与预防中心的安全消毒指南。

## 千万不要将漂白剂和氨水混合

使用漂白剂的一个主要危险是可能会意外地与氨混合，这可能会产生致命的烟雾和酸。永远不要把漂白剂和其他清洁产品混在一起，如果你不小心把两者混在一起，立即离开发生事故的地方。在你吸入漂白剂和氨水混合产生的烟雾后寻求医疗救助因为来自这样的事故产生的烟雾会杀死你或伤害你的眼睛和粘膜。

如果你不小心误食漂白剂或让它进入眼睛，请联系当地的毒物控制中心，或拨打 911 了解任何严重症状。
