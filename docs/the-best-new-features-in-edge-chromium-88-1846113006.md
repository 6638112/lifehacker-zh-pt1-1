# Edge Chromium 88 的最佳新功能

> 原文：<https://lifehacker.com/the-best-new-features-in-edge-chromium-88-1846113006>

Edge Chromium 的最新更新版本 88 终于推出了。该版本为 Windows 和 macOS 版本的浏览器添加了许多新功能，如新的密码安全工具、更新的外观和几项生活质量增强。这是很多，所以我们将检查 Edge 88 中的所有新功能，并向您展示如何使用它。



### 密码生成器和安全监视器

头条 Edge 88 的更新笔记是两大密码功能。首先是一个随机密码生成器，每当你创建一个新帐户或更新现有的密码时，它会自动创建密码。Edge 会保存密码并自动填写，因此您不必记住它。

另一个新的密码工具是自动数据泄露监控。Edge 会将任何保存的登录信息与最新的数据泄露进行交叉引用，如果它认为您的任何帐户受到威胁，它会向您发出警告。

这两项功能都要求你使用微软账户登录 Edge，并在**设置>个人资料>同步>密码下开启密码同步。**启用密码生成器:

1.  进入**设置>配置文件>密码。**
2.  打开**“提供保存密码”**和**“建议强密码”**

现在，无论何时输入新密码， Edge 都会为您创建新密码。密码监控是自动的，适用于保存在 Edge 中的任何登录。你可以在**设置>配置文件>密码>保存的密码下找到这些。**

### 睡眠标签

用户现在可以将非活动标签设置为在设定时间后“进入睡眠状态”。Edge 会在后台停止检查睡眠标签，从而释放您电脑的内存和 CPU 以用于其他地方。根据 [哔哔声计算机](https://www.bleepingcomputer.com/news/microsoft/microsoft-edge-gets-a-performance-boost-with-sleeping-tabs/#:~:text=Microsoft%20is%20rolling%20out%20a%20sleeping%20tabs%20feature,unveil,%20in%20September,%20as%20a%20Microsoft%20Edge%20experiment.) 的说法，这平均减少了 32%的非活动标签的内存使用，以及大约 37%的 CPU 使用。

用户会看到一个弹出通知，建议他们打开睡眠标签以减少 Edge 的资源使用，但你可以在**设置>系统>保存资源**中配置设置。您可以选取标签在进入睡眠状态前需要保持不活动的时间，或者将特定网站设定为永不进入睡眠状态。

### 侧栏搜索

现在，您无需打开新标签页或离开当前页面即可搜索网页。简单高亮一个词，然后**右键>搜索**。一个侧边栏将打开搜索结果。

### 新气象

Edge 88 还改变了浏览器的外观，包括与微软其他产品相匹配的新“流畅设计”界面图标，以及超过 24 种新的浏览器主题，您可以使用这些主题来个性化 Edge 的外观。

每个主题都有自己的配色方案和壁纸。壁纸的选择包括来自微软的第一方视频游戏的艺术作品，如光环，Forza，贼海，等等；*神奇女侠 1984；*还有几个原创设计。你可以从新的 Edge 附加组件页面 预览和 [安装它们。](https://microsoftedge.microsoft.com/addons/microsoft-edge-themes) 

### 其他新功能

最后，还有一些较小的调整，其中一些仅在某些平台上可用，但现在是所有桌面版本 Edge 的标准配置:

*   新的标签页现在显示从你的 Outlook 账户收到的电子邮件。
*   [跨设备历史记录和标签同步](https://lifehacker.com/how-to-sync-your-microsoft-edge-tabs-across-your-device-1846001872) 现已对所有用户开放。
*   [自动情景模式切换](https://lifehacker.com/how-to-keep-your-work-and-personal-browsing-separate-on-1843571354) 现已在 Mac 上实现。

要下载 88 版更新，打开 Edge，进入**设置>关于微软 Edge** 。如果可以下载的话，下载会自动开始，尽管微软说一些用户可能需要两个星期才能收到。
