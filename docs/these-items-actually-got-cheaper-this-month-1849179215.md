# 这些商品这个月实际上变得更便宜了

美国劳工统计局 本周发布了新的消费者价格指数，结果并不乐观。根据他们的报告，6月份CPI-U上涨了1.3%，超过了上个月5月份1%的糟糕表现。尽管通胀继续影响普通消费者的钱包，但并非所有指数都感受到了这种压力。有趣的是，一些商品和服务甚至逆势而上，看到它们的价格在上个月下降。所以，下次当你在为油箱加油时感到沮丧的时候，这些物品的价格已经开始下降，这一事实让你感到安慰

电视。随着许多新的流媒体服务如[HBO Max](https://lifehacker.com/whats-new-on-hbo-max-in-july-2022-1849105041)和 [派拉蒙+](https://lifehacker.com/whats-new-on-paramount-in-july-2022-1849125036) 的出现，你可能会想是时候买一台新电视了。考虑到电视价格比上个月下降了2.3%，大盒子零售商可能会同意你的观点。然而，如果你能够 [推迟到11月](https://lifehacker.com/the-best-time-to-buy-anything-during-the-year-5973864) 左右，你可能会得到更有竞争力的价格，因为黑色星期五和网络星期一的电视价格通常会在那时下降。

自行车。 [对于那些正在寻找天文数字油价的替代品的](https://lifehacker.com/how-to-sound-less-stupid-when-you-talk-about-gas-prices-1849171128) ，我很高兴地报告在自行车领域有好消息。与上月相比，两轮车的价格下降了0.6%。你可能不得不 [学习一两件关于自行车保养的事情](https://lifehacker.com/how-to-do-your-own-basic-bike-maintenance-1849097471) ，但是为了能够便宜高效的出行这是值得的。

**智能手机。** [如果你讨厌升级你的智能手机](https://lifehacker.com/the-best-phones-for-people-who-hate-upgrading-1848872793) 但知道这是不可避免的，现在是扣动扳机的大好时机。与上个月相比，智能手机的价格不仅下降了0.5%，而且实际上比去年同期下降了20%。如果你希望保持在一定的预算内，有 [不缺乏智能手机选项](https://lifehacker.com/5-of-the-best-budget-smartphones-of-2022-1848680437)；只是 [确保你在尽你所能削减这个月的总开销](https://lifehacker.com/the-easiest-ways-to-cut-down-your-cell-phone-bill-1849014104) 。

旅馆和汽车旅馆。对于那些希望在今年夏天旅行的人来说，有些人迫切需要好消息；与上个月相比，旅馆和汽车旅馆的价格下降了3.3%。在经历了11.5%的同比增长后，消费者终于看到了这个行业的一些缓解，在就Airbnb 的实用性和质量举行公众公投后，这个行业成为了激烈讨论的中心 [。不仅在酒店里你会更快乐，而且使用正确的工具，你甚至可以在酒店里省下一大笔钱。](https://lifehacker.com/the-best-vacation-rental-alternatives-to-airbnb-1849088885)

汽车和卡车租赁。 [在看到如此高的需求导致汽车租赁短缺一年后，](https://jalopnik.com/great-now-theres-a-rental-car-shortage-1846693354) 从租赁机构购买一套车轮的价格明显降温。与上个月相比，汽车和卡车租赁价格下降了2.2%，同比下降了7.5%。尽管租车的价格已经降低，但是在离开停车场之前，你仍然需要做一些事情，以确保你不会被不合理的费用所困扰。

机票价格。经历了一年令人发指的价格上涨后，机票价格终于开始下降。与上个月相比，购买机票的价格下降了1.8%，给这个价格同比上涨34.1%的行业带来了一些急需的缓解。不仅价格在下降，西南航空公司 [甚至给那些看到自己航班](https://lifehacker.com/how-to-get-money-back-from-southwest-when-their-fares-d-1849141847) 价格下降的顾客退款。
