# 不要再用播放列表来装酷，开始用它们来分享你的感受

> 原文：<https://lifehacker.com/stop-using-playlists-to-look-cool-and-start-using-them-1847855875>

你努力用语言表达自己吗？考虑转向我们共有的一种通用语言:emo 播放列表。我认为，作为一个社会*，我们浪费时间来炫耀“审美”音乐品味。相反，我们应该花更多的时间来创建充满情感的、超特定的播放列表。更重要的是，我们需要克服对看起来老套的恐惧，开始与我们的朋友、家人、爱人、敌人以及任何其他知道如何使用 Spotify 协作功能的特殊人士一起制作这些播放列表。* 

*Watch*

## ***什么是“好”的播放列表***

*我厌倦了假装播放列表需要做任何事情，除了把我带到童年记忆、悲惨的分手和史诗般的团聚(真实的或想象的)。对我来说，一个好的播放列表可以简单到泰勒·斯威夫特的《太好了》的二十个不同的原声封面。这一段花了 15 分钟来写，因为我不得不一次又一次地停下来闭上眼睛，感受那座桥的情感冲击。*

*虽然 Spotify 的“[【blend】功能](https://lifehacker.com/how-to-use-spotifys-blend-to-mix-your-playlists-with-yo-1847614571) 很适合寻找你们共同的音乐品味，但不要沉迷于手动合作播放列表的特殊意图。通过歌曲释放共同的记忆有很大的力量。此外，我不在乎我的 Spotify 包装看起来是否“酷”那年我输了，我是林的前 0.1%的听众。*

*我并不孤单，因为情感协作播放列表:科学站在我这边。音乐是一种传播“情绪传染”的工具， [今日心理学](https://www.psychologytoday.com/gb/blog/science-choice/201807/why-are-we-moved-music) 称之为简单地通过音乐感知一种情绪可以在听者身上引发同样情绪的方式之一。有了这些知识，你要做的合乎逻辑的事情就是创建一个共享的播放列表，用强烈的情感打你所爱的人的脸。*

## ***音乐将人们聚集在一起***

*音乐有一种独特的能力将我们联系在一起，这不会因为有人不在你身边而停止。在为《连线》 撰写的一篇文章中，记者艾米·帕图雷尔讲述了一个协作播放列表将她的家人和她已故的父亲联系起来的力量。为了开始播放，帕图雷尔问了她的家庭成员两个问题:“哪些歌曲让你想起了爸爸？”以及“你对清单上的每首歌都有特定的记忆吗？”*

*作为研究的一部分，帕图雷尔采访了人类学家、波士顿大学教授卢克·格洛瓦契基 ，他解释了“音乐如何进入大脑的情感中心，并且与记忆紧密相关。如果你听到一段音乐，你可以唤起情绪，这种情绪是你在生活中听到它时产生的效果。这是一种即时的感官反应，而口头语言则不然。”帕图雷尔还引用了像 [这样的研究，这一研究发表在*美国心理学*](https://pubmed.ncbi.nlm.nih.gov/34110844/) 上，表明“音乐是加强社会联系的有力工具，即使人们相距遥远。”简单来说，音乐把人们聚集在一起。*

## ***提升播放列表的技巧***

*让我们去旅行吧。具体来说，让我们乘飞机吧，根据 NPR 小办公桌制作人 鲍比·卡特关于如何像乘飞机一样组织播放列表的提示:*

> *一般来说，飞机起飞，在空中停留一会儿，然后轻轻地降落。这就是你想要建立播放列表的弧线。想想你清单上的每首歌都扮演了什么角色，然后就简单的安排和连接这些点了。记住这种结构也可以帮助激发更多的歌曲选择，因为你的音乐大脑会开始将不同的歌曲放入不同的位置。*

*我想把重点放在关于你清单上每首歌的作用的建议上。下一次，当你准备将一个播放列表命名为“情绪”或“氛围”时，我向你发出挑战，要求你真正表达出这种情绪或氛围。怀旧？向往？单恋？(不，我没经历什么，你问这个干嘛)？*

*我的想法是为你的播放列表集思广益一个主题或引导性问题(比如 Paturel 的“哪首歌让你想起爸爸？”).这里有一些非常明显的启发性想法:*

*   *坐进时光机，拥抱童年的自己*
*   *发现你一生的挚爱一直就在你面前*
*   *坐在偷来的车里，把你的手伸出窗外，和一群看似不可能的英雄一起进行一次越野公路旅行，他们都是你最亲密的朋友*
*   *如果你不在我的葬礼上演奏这些歌曲，我会回来缠着你*
*   *那一刻，你意识到你的父母是有缺点的人，也是有缺点的人，T2，你不能期望别人完美*
*   *他的心碎是虚构的，但我们的感觉是真实的*
*   *重新游戏堵塞*

*再说一遍，这些只是我的一些想法。最终，共享播放列表是为了与你关心的人联系，并释放你特有的记忆。所以，开始发短信吧，准备好迎接情绪的过山车吧。因为这是明确的，客观的音乐的全部。*

## ***如何在您的播放列表上开始协作***

*不同于算法上的 [【融合】功能上的](https://lifehacker.com/how-to-use-spotifys-blend-to-mix-your-playlists-with-yo-1847614571) ，制作一个协作播放列表需要你手动选择歌曲。协作功能适用于一系列流媒体服务。 [以下是 Spotify 制作协作播放列表的说明](https://newsroom.spotify.com/2020-09-29/how-to-make-a-collaborative-playlist/) :*

1.  *选择您想要协作的播放列表(请记住，您只能对您是所有者的播放列表执行此操作)。*
2.  *点击标题中的**添加用户**按钮，使播放列表成为协作播放列表。*
3.  *开始邀请其他人在社交媒体、消息应用程序上添加歌曲和播客剧集，或者只是通过复制和粘贴链接。*

*最后，对于那些仍然关心看起来酷的人来说，最后一招是:坚持小写播放列表标题。不管出于什么原因，这种高度策划的漠不关心是青少年正在做的事情，我们必须相信我们的青少年，因为如果我们不相信我们的青少年，我们怎么知道我们在退缩？*

*快乐的聆听，更重要的是快乐的感觉。*
