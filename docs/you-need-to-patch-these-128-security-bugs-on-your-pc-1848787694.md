# 你需要修补你电脑上的这 128 个安全漏洞

> 原文：<https://lifehacker.com/you-need-to-patch-these-128-security-bugs-on-your-pc-1848787694>

网络安全经常出现在新闻中，这是有充分理由的。坏人总是在寻找软件中的漏洞，以利用这些漏洞谋取私利:你运行 Windows 和各种应用程序的个人电脑处于十字路口，所以当微软像本月一样发布修补安全缺陷和漏洞的更新时，你应该尽快安装它们。



这些来自微软 [的最新安全更新修补了惊人的 128 个安全漏洞](https://thehackernews.com/2022/04/microsoft-issues-patches-for-2-windows.html) ，不仅仅是 Windows，还有其他程序，包括 Office、Skype for Business、Edge、Defender、Exchange Server 等等。虽然漏洞的数量令人担忧，但更令人*担忧的是本次更新修补的两个零日漏洞，包括一个被主动利用的漏洞。*

被识别为 CVE-2022-24521，被积极利用的漏洞是 Windows 通用日志文件系统中的特权提升漏洞。 [根据微软](https://msrc.microsoft.com/update-guide/releaseNote/2022-Apr) 的说法，这个漏洞已经被广泛利用，这对任何还没有安装这个补丁的电脑来说都是一个坏消息:任何知道如何利用这个漏洞的坏人理论上都可以利用它来攻击你的系统，这就是为什么尽快更新是很重要的。

另一个令人担忧的主要缺陷是 CVE-2022-26904，这是另一个特权- 升级漏洞，这次与 Windows 用户配置文件服务有关。与之前的漏洞不同，这个漏洞似乎还没有被利用，但它*是*众所周知的，这意味着利用可能在任何时候发生。

虽然这两个零日漏洞是这些新更新的亮点，但其他 126 个缺陷也不应被忽视。微软总共将 10 个缺陷评为“严重”，115 个为“重要”，3 个为“中等”。为了保护你的电脑免受所有 128 个缺陷，请务必尽快安装安全更新。

## 如何更新您的电脑以安装最新的安全补丁

Windows 可能会提前通知您这些新的更新，然后开始自动安装它们。但是，如果没有，您需要从设置中手动安装它们。

进入**开始>设置>更新&安全> Windows 更新** (Windows 10)或者**开始>设置> Windows 更新** (Windows 11)。允许 Windows 检查任何可用的更新。如果补丁可用，你会在这里看到它。然后，您只需按照屏幕上的说明将更新下载并安装到您的计算机上。
