# 如何在任何地方观看帝国大厦的节日灯光秀

> 原文:[https://life hacker . com/how-to-watch-the-empire-state-buildings-holiday-light-s-1845919477](https://lifehacker.com/how-to-watch-the-empire-state-buildings-holiday-light-s-1845919477)

纽约市的假日季节是人们——许多(或大多数)不住在这里的人——似乎真的很喜欢的。这确实很有意义:由于到处都挂着节日灯，这个城市通常的光芒被带到了一个新的高度。甚至我们的建筑也为这一时刻而装扮，包括帝国大厦。

Watch

但是，简单地按动开关打开帝国大厦的节日灯——就像你打开挂在你父母车库上的节日灯一样——并不适合这个标志性的建筑。如果每年只停留几周的洛克菲勒中心圣诞树有自己的仪式，由《今日秀》的演员演唱世俗颂歌，那么毫无疑问，这个装饰艺术杰作——自1931年以来一直是纽约市天际线的一部分——也应该得到某种形式的宣传。

事实证明，的确如此，仪式就在今晚。以下是在世界任何地方观看的方法。

### 如何观看帝国大厦的节日灯光秀

今晚(12月19日星期六)是帝国大厦一年一度的音乐灯光秀(是的，这是官方名称)，美国东部时间晚上8点开始。除了仪式性的开关开灯(是的，有 [一个真正的巨型开关](https://images.app.goo.gl/xPiFKkkKcwc1PDBL9) )今年的表演还包括凯莉·安德伍德和约翰·传奇的二重唱。他们将演唱《哈利路亚》(不， [不是那首](https://www.youtube.com/watch?v=ttEMYvpoR-k) )，这首歌是传奇(和托比·加德一起)写的，出现在安德伍德的第一张圣诞专辑中。

无论你住在哪里，你都可以通过帝国大厦的 [Earthcam](http://esbnyc.com/about/live-cam) 观看今晚的直播。在那之后，视频将会在帝国大厦的YouTube频道 上播放。但是，如果你错过了今晚的节目，而*真的*想看现场直播，它将在12月25日圣诞节每天晚上重播。

### 其他灯光显示

作为一些背景，帝国大厦经常以不同的颜色和展示方式点亮，以纪念某些节日、组织、事业和事件。建筑网站的这一部分让你知道全年的灯光和色彩意味着什么。这些灯光显示自1976年以来一直存在，并于2012年改用LED灯。
