# 工具库是在 DIY 项目上省钱的最聪明的方法

> 原文：<https://lifehacker.com/tool-libraries-are-the-smartest-way-to-save-money-on-di-1848256776>

如果你想自己动手修理房屋或建筑工程，工具的成本可能会高得令人望而却步。这时工具库就可以派上用场了，尤其是如果您希望在特定项目中使用一次某个工具，或者如果您想在投资开发自己的工具之前先试用一种工具。你可以借用工具，而不是购买或租赁，有时需要付费，这比直接购买工具要节省很多钱。



## 在哪里可以找到工具库或 koolshare

工具库可以是社区组织、社区团体或实际的本地图书馆，它们可以借出工具。有些工具共享收取固定的会员费，有些按工具收费，有些完全免费。快速的谷歌搜索应该能为你提供你所在地区的选择，只要确保检查价格，并与你需要的工具的现行价格进行比较，以确保它是值得的。

有时社交媒体网站也会有社区工具共享组，所以你也可以查看。如果你还没有本地的工具共享资源，如果你有几个喜欢 DIY 项目的邻居，你可以自己创建一个。建立一个共享工具和设备的社交媒体小组很简单，这将为你节省一些钱。

## 工具共享通常要花多少钱？

与购买新工具的成本相比，工具库收取的费用很低。会员资格的范围从完全免费到每年 200 美元，但是如果你长期、反复地需要某个特定的工具，可能是时候投资你自己的工具了。

虽然这些工具通常很便宜或者可以免费使用，但是使用钉枪或电动打磨机等消耗品的工具会产生一些成本。通常租借服务只借给你工具，所以你需要提供自己的砂纸和钉子。尽管有一些费用和材料成本，图书馆会为你节省很多钱，特别是在电动工具和设备上。

## 工具库更具可持续性

工具库允许您访问高质量的工具，这些工具在被丢弃之前会被多次使用，而不是购买便宜版本的工具，您只打算使用一次，然后丢弃它们。如果你在寻找让你的 DIY 项目更可持续的方法，可以考虑使用工具库来防止更多的垃圾被填埋。如果你有一些不用的工具，或者你正在清理车库，你也可以考虑捐赠一个工具库，以帮助其他人获得他们需要的设备，并防止工具最终被填埋。

## 工具借用礼仪

借工具的一般规则是归还时要和你收到时一样好或者更好。因为其他人也需要使用该工具，所以请确保妥善保管和妥善存放您使用的任何东西。请确保您遵循任何有关润滑、清洁和维护工具的说明，这样您就不会无意中造成任何损坏。如果工具没有手册，就在网上找一本。尽管如此，大多数工具库不会对你的正常损耗收费。

就当是帮你的社区一个忙，让我确保按时归还工具。一些工具库有一个政策，如果你总是迟到，就不允许你再从他们那里借；另外，这很不礼貌。
