# 如何在不连接 Mac 或 PC 的情况下还原和抹掉锁定的 iPhone

> 原文：<https://lifehacker.com/how-to-reset-and-erase-a-locked-iphone-without-connecti-1848220057>

[iOS 15.2 充斥着大大小小的功能](https://lifehacker.com/16-of-the-best-ios-15-2-feature-updates-worth-knowing-a-1848199396) 。除了数字遗产和应用隐私报告等功能，还有一个我们期待多年的功能；无需先将锁定的 iPhone 连接到 Mac 或 PC，即可完全抹掉并重置它。

如果您或您认识的人忘记了密码，这项新功能将会派上用场。如果你已经尝试输入密码几次，你的 iPhone 会锁定你一段时间，你会看到一个安全锁定屏幕。

以前，在这个位置重置或擦除 iPhone 是非常复杂的。如果你的 iPhone 被锁定，你必须将 iPhone 置于 DFU 恢复模式，然后通过 Windows 上的 iTunes 或 Mac 上的 Finder 来恢复它。然而，现在你可以通过点击右下角的“抹掉 iPhone”按钮来重置你的 iPhone。这同样适用于 iPad 和 iPod Touch。

## 抹掉锁定的 iPhone 仍有限制

然而，要做到这一点，你的设备需要连接到 wifi 或蜂窝网络。另外，这不是解除激活锁的免费通行证。你仍然需要用来设置设备的苹果 ID 和密码——因此，谢天谢地， 不是重置被盗 iPhones 的免费通行证。

## 如何重设或抹掉锁定的 iPhone

这个过程很简单:

1.  唤醒 iPhone 并输入任何密码。一旦苹果注册了你失败的请求，一个“安全锁定”屏幕就会出现。如果你没有马上看到这个屏幕，重复这个过程几次。
2.  T ap 右下角的“**擦除 iPhone** 按钮；再次点击确认。
3.  输入 Apple ID 密码，在 iPhone(链接到此设备的那个)上注销您的 Apple ID。

4.  轻按“**抹掉 iPhone** ”按钮以完全抹掉设备上的所有数据和设置。
5.  给它一些时间。当你的 iPhone 重启时，它将是全新的。

现在，您可以按照屏幕上的说明来设置您的 iPhone。我们希望你有一个 iCloud 备份，或者离线备份——这样，你就可以恢复你的数据，并且很快就可以正常运行。

此外，这一次，请确保您使用一个您真正记得的密码。

正如我们在上面提到的，要让这项功能发挥作用，你需要连接到 wifi 或手机网络。如果做不到这一点，您需要在 PC 或 Mac 上使用经典恢复模式来恢复 iPhone。
