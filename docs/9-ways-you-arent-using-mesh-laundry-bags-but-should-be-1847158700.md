# 9 种不使用网布洗衣袋但应该使用的方法

> 原文：<https://lifehacker.com/9-ways-you-arent-using-mesh-laundry-bags-but-should-be-1847158700>

有些东西进了洗衣机就再也出不来了，而且当你发誓说你把偶数只袜子放进洗衣机，结果只剩下奇数只袜子时，T2 会感觉到像 T4 家养小精灵在偷你的东西。事实是，你的洗衣机真的*有时候*会吃掉你的衣服。 [洗衣机外壳](https://vm.tiktok.com/ZMdyNaxHE/) 可能是装着你的袜子，你的 [烘干机棉绒收集器](https://www.reviewed.com/laundry/features/youre-not-crazy-your-socks-really-can-disappear-in-the-wash) 之间的空间可能也装着你的内衣。所以我如果你还没有听说过网状洗衣袋的魔力，现在是你学习的时候了，因为这些薄、透气的袋子甚至可以做不仅仅是洗衣的事情。以下是在洗衣房内外使用网袋的九种方法。

当你像过去一样外出露营时，也就是说，没有露营车和自来水的 T2，你必须找到创造性的方法来清洗你的盘子，其中一个最好的方法就是用一个浸泡袋。你把脏盘子放在袋子里，把它们浸泡在装满水和肥皂或漂白剂的盆里，然后再把它们浸泡在一桶清水里。完成后，拿起拉绳，把袋子挂在树上或晾衣绳上晾干。这个包也可以用来储物。

用手清洗细小的物品可能是一件痛苦的事情，尤其是当你有一台可以使用的洗碗机，而你更愿意用它的时候。 [像奶嘴](https://lifehacker.com/wash-pacifiers-and-breast-pump-parts-in-the-dishwasher-1833618112) 和小茶杯这样的小件物品不容易放入洗衣机的凹槽和尖头中，但我 n 代替你可以将这些 [物品放入网袋](https://www.onecrazyhouse.com/15-ways-mesh-laundry-bags-can-make-life-easier/) 中，并将其放在顶部或底部的架子。请务必使用带拉链的网袋或按- 和- 关闭功能，因为你只是不想拉绳被你的洗碗机叶片卡住。

防止洗衣机和烘干机偷走你的袜子的最常见方法是一起洗。与其使用 [活页夹](https://lifehacker.com/how-to-avoid-losing-your-socks-in-the-laundry-5858722) 或其他不太安全的方法，不如用把你的 [袜子扔进网状洗衣袋](https://cleanlaundry.com/time-saving-tricks-with-mesh-laundry-bags/) ，然后把整袋直接扔进洗衣机。洗衣机洗完后，袜子袋也可以直接放入烘干机。在那里...你再也不会把袜子丢在洗衣机里了。

许多锄头不一定很精致，但是把它们扔进洗衣机和烘干机会让它们变得畸形和无用。为了让你不用手洗鞋子，把它们和几条旧毛巾放在一个 [拉链网眼洗衣袋里](https://www.whirlpool.com/blog/washers-and-dryers/how-to-wash-shoes-in-washing-machine.html) 。毛巾将有助于防止鞋子在洗衣机中碰撞，和鞋子不会变形。只是确保先把鞋带和鞋垫拿掉——他们可以和鞋子一起放在包里，或者放在一个单独的网袋里，这样他们就不会被缠在一起或者弄坏了。对于烘干机，准备一个网袋，将 [连接到烘干机门](https://www.amazon.com/Smart-Design-Sneaker-Elastic-Straps/dp/B006OYWRSY/ref=asc_df_B006OYWRSY/?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/9-ways-you-arent-using-mesh-laundry-bags-but-should-be-1847158700&asc_source=&hvadid=198070680908&hvdev=c&hvdvcmdl=&hvlocint=&hvlocphy=9004173&hvnetw=g&hvpone=&hvpos=&hvptwo=&hvqmt=&hvrand=13548016487012438481&hvtargid=pla-318223493016&linkCode=df0&psc=1&tag=kinjalifehackerlink-20) 上，这样的鞋子就会在适当的位置烘干。

采摘完自家种植的草药后，你可以选择立即使用或晒干以备后用。你可以把它们放在桌子上，放在烤箱里，或者用微波炉加热，但是干燥草药的最好方法之一是风干它们。只需将和 [网兜挂在外面的](https://thegrownetwork.com/drying-herbs-the-easy-way/) 挂钩上或封闭的门廊上晾干即可。

让你的衣物在洗涤过程中保持良好状态的经典方法(除了在精致的餐具上满载)是将它们扔进 [网眼内衣袋](https://lifehacker.com/how-to-wash-your-clothes-with-lingerie-bags-1846690854) 。一些物品——像精致的毛衣，例如——经常需要手洗或干洗以保持其柔软度和形状，所以如果你的毛衣需要干洗或其他什么，最好遵循那些说明；o 然而，将你的[放入网袋](https://classiccleaners.net/why-mesh-laundry-bags-are-a-must/) 会给它们一个适当的清洁，而不会造成机器的正常损坏。

此外，带有 velcro 或 T2 挂钩的衣物——如胸罩——经常会被洗衣机和烘干机中的其他衣物挂住。为了防止衣服 [撕破、缠结和变形](https://www.housebeautiful.com/lifestyle/cleaning-tips/a28326770/best-laundry-wash-bags/) ，在把它们扔进洗衣机和烘干机之前，先把它们放在网袋里。

我们已经讲过如何 [清洁浴盆](https://lifehacker.com/12-neglected-household-items-you-should-clean-more-ofte-1846588570/slides/13) 里的玩具，但是你也可以用网袋把它们扔进洗衣机里就行了。你不希望一桶乐高玩具在你的洗衣机里叮当乱响，也不希望你孩子最喜欢的毛绒玩具失去一只眼睛，所以给乐高玩具和其他硬玩具准备一个大网袋，和一些旧毛巾一起扔进去，就像你处理鞋子一样。你可以拿一个小的或中等大小的网袋(或根据填充玩具的大小而定的大袋子)，像洗一个精致的玩具一样清洗填充动物。之后，将玩具挂在网袋里或摊开晾干。

如果你把土豆或洋葱放在杂货店的塑料袋里，很可能你不得不在烹饪之前扔掉它们，因为你如何储存你的水果和蔬菜有助于决定它们能保存多长时间。正如 [简单生态](https://www.simpleecology.com/produce) 解释的那样，“当农产品放置和老化时，它开始释放乙烯气体，这就是导致水果和蔬菜腐烂如此之快的原因。”像网眼一样的透气织物可以让这些气体逸出，让你的产品更持久。他们建议把大蒜和洋葱放在挂在厨房的网袋里。你也可以将水果和蔬菜放在网袋里，放在冰箱里(或柜台里，视物品而定)。
