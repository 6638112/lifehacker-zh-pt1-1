# 使用 Mac 新的“快速笔记”功能，再也不用搜索笔记应用程序

> 原文：<https://lifehacker.com/use-macs-new-quick-notes-feature-and-never-search-for-t-1847961948>

macOS Monterey 为 Mac 带来了 iPadOS 15 最有趣的功能之一:快速笔记。苹果的 Notes 应用程序已经是一个强大的笔记解决方案，但它现在更多地集成到了操作系统中。使用新的快速笔记功能，您可以在 Mac 上的任何地方开始记笔记。只需要一个手势或一个键盘快捷键。它是这样工作的。



## 如何在任何地方创建快速笔记

想匆匆记下些什么吗？不用搜索 Notes 应用程序，只需将光标移到屏幕的右下角。当你这样做的时候，一个白色的小盒子会从角落里探出头来。点击它，新的快速笔记功能就会启动。

从这里开始，一个新的便笺会在它自己的窗口中打开，它浮在所有其他窗口之上(如果你切换应用程序，你仍然可以看到你的快速便笺)。你现在可以开始记笔记了。一旦你完成了，就关上窗户。

下一次你这样做的时候，笔记应用程序会打开上次使用的笔记，这样你就可以从你离开的地方继续。如果你想创建一个新的便笺，使用**命令** + **N** 键盘快捷键。

## 如何向快速笔记添加链接

只有 Safari 具有与快速笔记功能的原生集成(稍后将详细介绍)，但 Apple 已经创建了一个功能，允许您将其他打开的应用程序的链接直接添加到快速笔记。

启动快速笔记后，点击顶部工具栏中的**添加应用链接**按钮。您将看到来自不同浏览器或应用程序的所有可用网站或应用程序链接的列表。点击链接旁边的“**添加链接**按钮，将其添加到注释中。

快速笔记在 Notes 应用程序中仍然是一个完全成熟的笔记，因此您可以访问您习惯的所有格式和媒体选项。

## 如何在快速笔记中每次都开始一个新的笔记

如果您不想每次使用快速笔记功能时都看到相同的笔记，该怎么办？有一个设置可以在你每次使用快速笔记手势时创建一个新的笔记。

要启用此功能，请打开 Notes 应用程序，从菜单栏中单击“ **Notes** ”选项，然后转到“**首选项**”部分。在这里，取消选中“**简历最后一条快速备注**”选项。

## 启动快速笔记的更多方法

如果你不喜欢快速记笔记时那种火辣的 C orner 手势，有两种方法可以开始快速记笔记。就像在 iPad 上一样，快速笔记功能被集成到 Safari 应用程序中，在这里，它也可以帮助你添加文本和链接到笔记。选择一些文本或链接，点击右键，选择“**添加到快速注释**选项。

这将打开一个新的快速便笺，其中附加的文本和网页已经链接。

您也可以使用键盘快捷键开始快速笔记。按下 **Fn** + **Q** 组合键。如果你愿意，你可以从**系统偏好** > **键盘** > **快捷键** > **任务控制** > **快速笔记**中更改这个键盘快捷键。

## 如何在 macOS Monterey 中停用或更改快速笔记手势

如果您觉得右下角的手势很烦人，或者您已经将该手势用于其他用途，您可以停用或更改该角。点击菜单栏中的**苹果**图标，选择**系统偏好**G o 到**任务控制** > **热点弯道**。默认情况下，您会看到右下角被“**快速笔记**占据，但是您可以为任何角落启用快速笔记。如果要禁用快捷方式，从菜单中选择“ **-** ”选项，点击“**确定**”按钮保存设置。
