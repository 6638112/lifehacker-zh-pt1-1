# 什么是脚跟脚趾下垂，这对跑步者有影响吗？

> 原文：<https://lifehacker.com/what-is-heel-toe-drop-and-does-it-matter-for-runners-1849596301>

我们之前写过为什么人们 [不应该过度考虑他们的跑鞋](https://lifehacker.com/why-youre-overthinking-your-running-shoes-according-to-1849458897) 。与此同时，了解你的身体并找到合适的鞋子会让系好鞋带去跑步和待在家里变得不同。



在所有使跑鞋购物成为压倒性体验的被夸大的特征中，**鞋跟-鞋尖落差**是一个非常有用的指标。它可能不会创造或打破你的英里时间，但理解这个术语可以帮助你找到最舒适的骑行方式来进行你的步行和跑步。下面是关于如何找到适合你的鞋跟和鞋头的信息。

## 什么是脚跟脚趾下降？

脚跟-脚趾落差(有时简称为“落差”)量化了脚跟下方的缓冲量和脚趾下方的缓冲量之间的差异。最直观的形象就是一只鞋的鞋跟有多大。高跟趾落差意味着更陡的坡度和更多的脚跟缓冲。因此，赤脚跑步者会宣扬零脚跟脚趾落差的力量，因为那实际上是你赤裸的脚。

下面是一些的具体例子来自 [跑仓](https://www.runningwarehouse.com/learningcenter/gear_guides/footwear/heel-toe-drop.html)；stat 也可以在大多数其他跑鞋资源中找到:

*   高跟趾落差(超过 7 毫米)最适合先着地的跑步者、跟腱有问题的跑步者或经常穿高跟鞋的跑步者。
*   脚跟-脚趾的低落差(从 0 到 6 毫米)最适合脚中间或前部着地的跑步者。

如果没有列出，你可以通过查看鞋子描述中的堆叠高度数字和从鞋跟数字中减去前脚掌数字来自己计算鞋跟-脚趾落差。

## 脚跟脚趾落差重要吗？

高低跟趾落差都不会让一双鞋天生比另一双“更好”。如果任何一个跑步者试图告诉你一种下降方式有改变比赛的好处，要持怀疑态度。

落差什么时候很重要:如果你正在换鞋，你可能会注意到从高跟到低跟的落差(反之亦然)。切换到更低的落差尤其可能会导致你的小腿或跟腱疼痛。这可能会导致不适感，所以当你逐渐在不同鞋型之间转换时，记住这个数字是很有帮助的。

在一天结束时，个人偏好优先于具体的鞋跟脚趾下降数字。了解脚跟脚趾下降可以帮助喜欢迎合脚触地的有经验的跑步者，但对我们大多数人来说，我们早期的建议仍然适用: [D 不要想太多](https://lifehacker.com/why-youre-overthinking-your-running-shoes-according-to-1849458897) 。
