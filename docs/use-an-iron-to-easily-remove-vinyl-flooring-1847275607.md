# 使用熨斗轻松去除乙烯基地板

当需要改造你的地板时，移除现有的材料通常是一件痛苦的事情。移除瓷砖或木地板可能需要专业人士，但乙烯基地板不值得这么多的努力或金钱。然而，拆除乙烯基地板的问题是用于将其固定到下面地板上的粘合剂。根据它所附着的地板类型，移除过程意味着 [用温水](https://www.theflooringlady.com/how-to-remove-vinyl-flooring/) 或地板剥离器刮胶数小时。然而，有一种更简单的方法来移除瓷砖和胶水——你只需要一个家用电器和一个移除工具。

## 你应该什么时候移除乙烯基瓷砖？

首先要考虑的是，在拆除乙烯基地板之前，它是何时安装的。1980年 之前生产的乙烯基 [地板可能含有石棉——一种危险的材料，在被发现是导致癌症](https://www.asbestos.com/products/vinyl-products/) 的 [主要原因之前，它曾被用作几种产品的防火剂。不幸的是，20世纪60年代或70年代生产的乙烯树脂地板和油毡很可能含有石棉，所以最好在干扰它们之前联系专业人员，以便安全地清除它们。](https://www.asbestos.com/asbestos/)

如果你的地板很难看(但不会对健康构成威胁)，你*可以* 把它们盖住，这是一个廉价的选择——但有其自身的风险。当已经有几层地板时，它不仅可能阻止门关闭，还可能损害地板的强度。正如ccontractor site[Phillips ' s Floors](https://www.phillipsfloors.com/aboutus/blog/installing-flooring-over-existing-floor.aspx)指出的那样:“N将新地板粘合到现有地板上只会让你的新地板和旧地板一样坚固。如果旧的失效了，你的新的也会随之而去。”

所以，如果你有一个混凝土地板，移除乙烯基材料很容易，省钱，并且让你可以从一个干净的石板开始，而不是在上面盖房子。
