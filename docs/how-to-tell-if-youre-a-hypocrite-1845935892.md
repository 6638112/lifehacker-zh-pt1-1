# 如何辨别你是不是伪君子

> 原文:[https://life hacker . com/how-to-tell-if-you-a-pseudo-1845935892](https://lifehacker.com/how-to-tell-if-youre-a-hypocrite-1845935892)

在某件事上公然犯错已经够糟糕的了。当你坚持努力让每个人相信你实际上是对的时，情况会更糟。面对强烈的反对意见，T4这样做看起来不太好，最糟糕的是，这会给人留下这样的印象:你故意表现得像个无耻的伪君子。

尽管你周围的每个人都在批评你的方法，你可能仍然相信，在内心深处，你对被认可的渴望是合理的。以为例，通常低调的爱荷华州政坛目前正在上演的一幕:民主党国会众议员丽塔·哈特 [最近以总共六票](https://apnews.com/article/election-2020-iowa-mariannette-miller-meeks-elections-iowa-city-b0b8ddb4e878a54fce078ac116e3eb76) 之差输掉了她的连任竞选。鉴于共和党挑战者玛丽安·米勒-米克斯以微弱优势获胜，哈特希望重新计票。实际上，她想要第二次重新计票。具体来说，她认为美联社[在初步选举的重新计票过程中，有](https://apnews.com/article/election-2020-iowa-mariannette-miller-meeks-elections-iowa-city-b0b8ddb4e878a54fce078ac116e3eb76)[张选票被错误地排除，其他的没有被检查。](https://apnews.com/article/election-2020-iowa-mariannette-miller-meeks-elections-iowa-city-b0b8ddb4e878a54fce078ac116e3eb76)

鉴于唐纳德·川普挑战2020年总统大选结果的慢动作马戏正在进行中，一些在线评论家称哈特为伪君子。通过不承认失败，我她不是在公然对抗她的政党一贯努力给川普的选举舞弊指控贴上错误的标签吗？(别介意他们是错误的，因为 [无数失败的诉讼都显示](https://www.buzzfeednews.com/article/zoetillman/trump-election-court-losses-electoral-college) 。批评家们无动于衷，哈特可能会继续要求重新计票，而不会考虑键盘战士是否正确。(当然，值得注意的是，一位候选人要求计算每张选票，而一位 [要求只计算*某些*选票](https://www.thenation.com/article/politics/trump-electoral-college/) 。)

但是这里有一些方法可以辨别你是否是一个伪君子——幸运的是，对我们大多数人来说，你不需要让在公共场合遭遇尴尬就能弄明白。

### 你有优越感

正如迪安·伯内特在2016年《卫报》的中所写的那样，大多数人对自己都有一个非常谄媚的看法，但这并不适用于其他人。

这是一个心理问题，与大脑对自我的概念有关，他解释道:

> 大脑充斥着认知和记忆偏见，这些偏见旨在让我们觉得自己很好，很体面，很有能力 ，不管现实如何。问题是，我们对他人的 [判断远比](http://study.com/academy/lesson/attributional-bias-in-the-workplace-self-serving-bias-and-fundamental-attribution-error.html) 更“现实”。

这是一种过于学术化的说法，如果你认为有些事情不适用于你，但仍然适用于其他人，那么你可能是一个伪君子。不管是因为你认为人们不如你聪明，还是他们缺乏你拥有的资历或经验(T2)，不管是哪种情况，你都可能显得虚伪。减少这种情况发生的一个方法是在 [工作，淡化你的自我](https://lifehacker.com/to-stop-worrying-so-much-deflate-your-own-ego-1832941435) ，这对许多人来说通常是有益的。

### 比起利他主义，你更在乎外表

如果你做好事主要是为了满足自己的个人需求，让自己在别人面前显得善良和无私，那么恭喜你——你可能是个伪君子。正如心理学家苏珊·克劳斯·惠特伯恩四年前在《今日心理学》中写道的那样，用利他主义行为来维护个人形象的需要一直是伪君子武器库中的一部分。

根据惠特伯恩:

> 比起满足个人目标的内在欲望，伪君子更有可能从对外表的渴望中获得的动力。

### 你在生活中通常不稳定

你可能宣扬某些价值观，但总是 无法实践它们。从广义的、压倒一切的意义上来说，不一致是伪善的一个T4定义标志。众所周知，你为各种左翼社会事业奔走呼号，但每次报税时，你都喜欢抱怨政府偷了你的钱？

正如欧洲心理学杂志 的研究人员在2015年写的那样，不一致性的概念是这样定义的:

> 一种对他人的道德要求，而自己却不能遵守；一个人在公共场合说或暗示某事而在私下里表现不同的例子；一个人声明的或隐含的信念与其行为不符的例子。

当然，每个人有时都是不一致的，但是当它成为你性格的定义规则时，而不是在罕见的例外中，你可能就是一个伪君子。
