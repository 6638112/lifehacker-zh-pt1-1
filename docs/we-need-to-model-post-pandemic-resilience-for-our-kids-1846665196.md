# 我们需要为我们的孩子树立后疫情时代韧性的榜样

> 原文:[https://life hacker . com/we-need-to-model-post-疫情-resilience-for-our-kids-1846665196](https://lifehacker.com/we-need-to-model-post-pandemic-resilience-for-our-kids-1846665196)

我们可以合理地假设，在过去的一年里，大多数父母都对我们的孩子错过了什么感到绝望。小孩子错过了很多基本的社交活动，或者“真正的”学校经历。十几岁的孩子已经在拼命地寻找自己在世界上的位置(这里的“世界”，我指的是他们学校的社会等级)，他们可能会发现这几个月特别孤独和孤立。青少年放弃了各种传统的庆祝活动。

Watch

所有这些都是真实有效的。他们已经错过了很多。从各方面来看，今年*都是糟糕的一年，我们可能会担心随着未来几个月事情的继续发展，他们会如何调整。他们会感受到隔离带来的长期负面影响吗？答案是:嗯，也许，对一些孩子来说，这要视情况而定。*

然而，父母不能取消疫情。尽管我们对失去的一切感到悲伤，但这种失去已经超出了我们的控制。然而，我们能控制的是我们如何选择摆脱它。正如朱迪思·华纳为《纽约时报》写的:

> [Phyllis] Fagell女士是一名治疗师、学校辅导员，也是2019年出版的《中学的重要性》一书的作者，与本文采访的十几位青少年发展专家一样，她坚持认为家长不应该恐慌，此外,“失去的一年”的说法需要停止传播。他们一致认为，要全面了解中学生的情况，并准备好帮助他们，需要同时记住两个看似矛盾的想法:过去的一年太糟糕了。大多数中学生会没事的。

人们常说孩子适应力强，从某种程度上说，这是真的。孩子们对创伤也不能免疫——恰恰相反——但是他们适应能力很强，尤其是当他们处在一个充满爱和支持的环境中时。我们无法填补他们在过去一年中所失去的一切；所有那些与朋友面对面的时间，课外活动，以及他们可能有的过夜时间都消失了。但是我们*能*做的是模拟如何走出这段时间，向他们展示如何像我们一样展示弹性肌肉。

早在去年4月，那时还不到四周，我们还在以两周为增量进行操作，无法完全掌握事情已经有多糟糕以及它们会变得多糟糕，我写了关于 [我们的孩子在此期间发展的软技能](https://offspring.lifehacker.com/your-kids-are-learning-tons-of-soft-skills-right-now-1842730189) 。我当时是这么说的:

> 现在，我们的孩子正在上耐心、韧性、沟通、解决冲突、妥协、创造性思维、同理心和正念的速成课。这些都是我们每天都在一定程度上练习的软技能，但现在一切都被提高了。他们必须学会对那些不能从电话会议上跳下来给他们拿零食的父母有耐心(或者他们必须想办法给自己拿零食)。他们解决与兄弟姐妹的冲突，哦，每四分钟一次。他们正在想办法让一个朋友开心起来，他的生日正以最安静的方式过去。

我们不需要对最火热的地狱般的几年进行盲目乐观的描述，但如果我们还没有，我们应该开始关注我们的孩子成长、适应、经历一次又一次打击的所有方式——即使是在艰难的时候，甚至是在不那么优雅的时候。

这不是“失落的一年”是的，这是失去的一年，但如果我们足够仔细地看，我们也可以看到它给我们和我们的孩子带来的好处:对自然的重新热爱，对能够与大家庭共度时光的感激，甚至我们对技术和屏幕时间的看法也发生了变化，因为它成为我们维持重要关系的主要方式。也许他们在数学课上学到的比平时少，但是他们学到了很多关于为保护他人而牺牲的东西。

我们的孩子几乎总是从我们这里得到如何处理一些事情的线索，这是他们必须处理的最大、最困难的事情之一。有许多黑暗的日子，许多崩溃源于我们都被期望做的事情的不可能性——在彼此之间生活、工作和学习，实际上是在彼此之上。但是我们做到了，我们做到了，我们试着给彼此以优雅，我们甚至可能进入了一个小圈子。

提醒他们——还有你自己——穿过这个疫情，从另一边走出来，无论我们感觉多么伤痕累累，都证明我们可以做艰难的事情。这是值得承认甚至庆祝的。因此，当你谈论回归面对面学习是什么样子，或者你在夏天重返社会的一小步时，一定要指出过去几个月里你不仅失去了什么，还收获了什么。
