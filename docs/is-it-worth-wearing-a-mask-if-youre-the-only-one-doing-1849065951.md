# 如果只有你一个人戴面具，值得吗？

> 原文：<https://lifehacker.com/is-it-worth-wearing-a-mask-if-youre-the-only-one-doing-1849065951>

出门能看到口罩的越来越少了。如果你生活在一个人们曾经擅长化妆的地方，那么现在只有少数人还会化妆。如果你所在的地区一开始很少戴面具，我敢打赌你已经有一段时间没见过面具脸了。但是疫情并没有结束，许多躲避病毒两年多的人现在开始咳嗽。所以，即使没有其他人是，戴上面具值得吗？是的，我会这么说。



## 你感染 COVID 的风险仍然很高

大约一年前, ,疾病预防控制中心根据每个县的传播情况提出了一系列建议。他们说，如果每天每 10 万人中有超过 50 个新病例，那么传播是“实质性的”，我们都应该掩盖起来。如果病例上升到每 100，000 人中有 100 人，那么传播是“高的”但是后来他们改变了定义。今天，我所在的县每 10 万人中有 237 例，根据 CDC 的现有地图，我们的风险仅为“中等”。这是一个移动球门柱的经典案例。

今年早些时候采用的新标准是基于入院人数的。一夜之间，我们主要是红色(“高”)的国家变成了主要是绿色(“低”)的地图。的确，病例数据变得越来越没用了，因为更多的人在家里检测，而且那些病例通常很轻微，以至于永远不会把 提交给任何正式的报告系统。但总体效果是让*看起来*我们的风险比以前低了。不是的。

当市政当局或联邦机构停止执行掩蔽准则时，也会发生类似的情况。一项联邦法规要求在公共交通工具上戴口罩，包括在机场和飞机上，直到今年早些时候一项法庭挑战阻止了它。我最近坐过飞机，我敢说 90%以上的乘客都没有戴口罩。(个别航空公司可以选择继续执行他们自己的口罩规则，但该规则一直受到一些乘客的争议， [到了对](https://www.nbcnews.com/news/us-news/passenger-broke-flight-attendants-teeth-mask-fight-sentenced-15-months-rcna30954) 工作人员造成危险的程度，因此似乎更容易放弃这个问题。)

据报道，就在联邦口罩规则被取消后，美国运输安全管理局官员 的 COVID 病例 [激增。COVID 仍然存在。](https://www.chicagotribune.com/opinion/commentary/ct-opinion-covid-19-mask-mandate-airlines-tsa-20220512-cjfgclzthfbjpeyhxnepqnokee-story.html)

## 面具仍然有效

但是戴口罩的事实并没有改变。如果一个房间里的每个人都戴着面具，那么对于碰巧在那里的一个没有戴面具的人来说，这个房间会更安全。但是即使你是唯一一个戴着面具的人——比方说在飞机上——你的面具仍然会保护你。

布口罩提供了少量的保护。宽松的外科口罩可能更好一点。但最好的保护是佩戴合身的 N95 或类似的口罩。我仍然在购物或旅行时戴着它。(我最近确实得到了 COVID，但我是从一个对戴口罩不那么尽职尽责的家人那里得到的。)

正如我们许多人发现的那样，你可以接种疫苗和加强免疫，但仍然会感染 COVID。你的疾病可能比未接种疫苗的人要轻得多，但生病并不好玩，而且当涉及到像 long COVID 这样的并发症时，这仍然是一场赌博。

我们不能再依赖从他人那里获得太多保护，这很糟糕。但是高质量的口罩仍然可以买到，而且仍然有效。如果你担心感染 COVID——你可能应该担心——戴上口罩也不失为一个好主意。
