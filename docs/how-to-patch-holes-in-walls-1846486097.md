# 如何修补墙上的洞

> 原文：<https://lifehacker.com/how-to-patch-holes-in-walls-1846486097>

想必你也经历过和每个租房者一样的思考过程:“我现在就挂一幅画，等我搬出去的时候再担心吧！谁在乎！”

现在是时候拿回你的保证金了，你的墙看起来就像一块瑞士奶酪。幸运的是，我们有几种简单的方法来修补这些漏洞。

首先，小心地从墙上取下钉子或图片挂钩，确保在这个过程中不要增加损坏。为了更加安全，在锤子周围缠上橡皮筋，或者在锤子和墙壁之间滑动纸巾以获得额外的缓冲。继续操作前，确保墙壁干燥且无杂物。

修补漏洞的真正方法是从五金店买一些灰泥和油灰刀。搅拌一下铲子，然后舀一粒豌豆大小的水滴(一个专业术语)到刀刃上。你需要的比你想象的要少一点。刮几下使其平滑，然后用湿海绵或布完成。

如果你处于紧要关头，有一些选择可以作为短期解决方案(足够你走一遍)。可以用旧信用卡刮，不用油灰刀。你可以用白色牙膏填补这个洞，或者用白色肥皂擦拭——这些方法看起来都很好，直到你离开那里，再也不回头。
