# 醋油沙司到底是什么？

> 原文：<https://lifehacker.com/what-are-vinaigrettes-exactly-1847548353>

有一段时间，在 2000 年代早期和中期，我点的每一份沙拉都有“油和香醋”我会一点一点地撒上叶子，然后用叉子轻轻搅动叶子，让 T2 均匀分布。这并不好，但它让我觉得可以控制自己的饮食(和卡路里)，这对我在 2000 年代早期和中期非常重要。



简而言之，这些沙拉毫无乐趣可言，主要是因为油和醋一起使用比单独使用效果好得多。当我说“在一起”时，我不是指“在同一个盘子里”，我是指“乳化成调味汁”

这就是醋油沙司的全部——醋(一种酸性水溶液，有时还有调味品)和油(一种脂肪)的一种非常稳定的乳液。水溶液和脂肪都不喜欢混在一起，但是让它们分开(或者分开倒，就像我年轻时的一样)会严重破坏你的沙拉。

这不仅仅是审美的问题。根据 [严重吃](https://www.seriouseats.com/how-to-make-simple-salad-worth-eating-vinaigrette) 的说法，非乳化油可以毫不夸张地破坏你的绿色:

> 是这样的:在下雨的地方，生菜在室外生长。为了保护自己，莴苣叶的表面有一层薄薄的蜡状防水涂层。从烹饪的角度来说，这意味着醋或柠檬汁等水基液体会直接从叶子上流出，而油类则会附着在叶子上，导致叶子变软枯萎。

然而，将油和醋混合在一起，酸会悬浮在油中，附着在叶子上，给叶子增添味道。

获得适当的乳化是有挑战性的，特别是如果你没有加入其他成分或者仅仅依靠搅拌器。幸运的是，你只需要两样东西就能做出完美乳化的调味汁:芥末和梅森罐。

芥末在化学层面上有帮助。存在于种子外壳中的粘液是一种天然乳化剂[。它将亲水性和疏水性结合在一起，迫使它们和睦相处。粘液越高，你的敷料保持乳化的时间就越长。(黄芥末效果不太好，但第戎和石磨效果很好。)一份](https://lifehacker.com/how-to-choose-the-right-mustard-for-your-vinaigrette-1832912147) [好的配比](https://lifehacker.com/use-this-template-to-make-a-perfect-vinaigrette-every-t-1844718967) 是一份芥末对四份油和四份醋。

梅森罐(或任何有盖的罐子)是另一个关键角色。罐子是我最喜欢的调味汁制作工具，有几个原因，我已经在前面的 中 [解释过了，但是我会再解释一次，只是为了重新加盖:](https://lifehacker.com/a-mason-jar-is-the-only-tool-you-need-for-perfectly-emu-1796819745)

> 不仅你可能已经有一个了，而且把调料搅拌在一起可以让它们乳化，即使是最剧烈的搅拌也不能。你只需将你最喜欢的调味汁配料倒入、舀入或舀入罐中，密封，然后摇动，直到你得到看起来均匀的混合物。然后你把它倒在沙拉上。然后你吃你的沙拉。哦不，你做了太多调料？好在它已经在一个可密封的容器里了。把它放在冰箱里，直到你做好另一份沙拉，那会很快，因为你非常擅长照顾自己。

老实说，忘了你的搅拌器吧。摇动你的调味汁，你就再也不会回去搅拌或者吃用非乳化油和醋做的沙拉了。(说真的，就是这么凄凉的做法。让我们把它留在属于它的 2000 年代早期和中期。)
