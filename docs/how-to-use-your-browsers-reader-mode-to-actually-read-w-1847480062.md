# 如何使用浏览器的“阅读器模式”来阅读你点击的内容

> 原文:[https://life hacker . com/how-to-use-your-browsers-reader-mode-to-actual-read-w-1847480062](https://lifehacker.com/how-to-use-your-browsers-reader-mode-to-actually-read-w-1847480062)

大多数时候，你可能是在浏览网页，而不是真正阅读。这对于快速新闻更新和浏览社交媒体来说没问题。但是当你打开一篇你真正想读的长文章时，事情就开始变得困难了。

Watch

你发现你的思维在走神，并且越来越难注意到同一个页面。是的，你可以把文章保存在 [的](https://lifehacker.com/read-it-later-showdown-pocket-vs-instapaper-1756021479) 口袋里，以便以后阅读，但我们都知道那是你的阅读清单的归宿。互联网无休止的刺激已经 [扼杀了你专注的能力](https://wexnermedical.osu.edu/blog/how-internet-affects-your-brain)——那么该怎么办？

别担心，浏览器中的“阅读器”模式会给你一个战斗的机会。这些特殊模式旨在使网页更易于阅读。他们删除给定网站的格式、空白空间、广告和所有其他额外的元素，以帮助 你专注于文本和一些图像。

更好的是，你可以自定义文本，让你的体验尽可能地友好，这样你就不太可能点击离开。以下是如何在你最喜欢的浏览器中使用阅读器模式(除了 Chrome— 但我们会谈到这一点)。

## 如何使用 Safari阅读器模式

Safari是阅读器模式的王者。不仅内置的阅读器模式很漂亮，而且还有一个默认情况下自动加载“阅读器视图”中所有文章的选项(尽管这对大多数用户来说可能有点太多)。

当你在浏览网页时，你遇到了一篇你想阅读的长文章，只需在Mac、iPhone或iPad的地址栏中按下阅读器查看按钮(这个看起来像一个页面图标)。很快，这个页面就会变成它的不那么忙的表亲。P 按下URL右侧的 **Aa** 按钮自定义格式。

## **如何使用火狐阅读器视图**

Firefox在阅读器视图方面也做得很好(不，你不需要用Pocket来做这个)。

只需点击桌面或移动应用程序内的地址栏中的阅读器视图图标(看起来像一张纸，上面有几行文本)(是的，它也适用于 iPhone、iPad和Android应用程序)，就可以快速将整个页面转换为阅读器友好的版本。再次使用，您可以使用左侧菜单中的 **Aa** 按钮自定义字体和背景。

## 如何在微软Edge中使用沉浸式阅读器

虽然微软Edge的阅读器模式仅限于桌面应用，但它确实做得很好。当你遇到一篇你想不受干扰地阅读的文章时，点击地址栏中的沉浸式阅读器图标(它看起来像一本打开的书)。

要自定义文本，请单击“**文本首选项**”选项。Edge在这里有一个非常巧妙的技巧——y你可以使用“**大声朗读**功能，用自然发声的声音向你朗读整篇文章(如果你想的话，你甚至可以选择不同的声音)。

## 或者试试Chrome上的阅读器模式扩展

Chrome是唯一没有专用阅读器模式的主流浏览器。不过不用担心，使用 [阅读器模式扩展](https://chrome.google.com/webstore/detail/reader-mode/llimhhconnjiflfimocjggfjdlmlhblm) 也能得到同样的体验。

阅读器模式扩展将为你提供一个简单的、可定制的、不受干扰的阅读环境。你可以为 [Pro版](https://chrome.google.com/webstore/detail/reader-mode-pro/koddbhkhginnhnfnhgldkampappgmmje?hl=en) 选择($ 15一次性许可费用 ) 来添加亮点、注释、自定义CSS、Pro字体等等。甚至还有一个高级选项(每月4.99美元)，增加了云存储，以后再看的功能和其他好东西。
