# 如何设置你的电报帐户自毁

> 原文：<https://lifehacker.com/how-to-set-your-telegram-account-to-self-destruct-1846206409>

Kerboom。那是你的电报账号自动删除自己的声音。老实说，我希望这个特性能融入更多的服务中。当你连续两年没有登录的时候，一个你从来没有用过的旧邮箱账户消失不是很棒吗？



电报的自毁程序通常在六个月不活动时启动。为电报 [描述](https://telegram.org/faq#q-how-does-account-self-destruction-work) :

> Telegram 不是一个商业机构，我们非常重视我们的磁盘空间。如果您停止使用 Telegram，并且至少六个月没有上线，您的帐户将与您存储在 Telegram cloud 中的所有消息、媒体、联系人和所有其他数据一起被删除。

但是，六个月只是默认。如果你想要一点额外的安全性，你可以编辑该设置为更短或更长的时间段——比如，你希望你的帐户在一个月后消失*或*你不信任自己，希望在 Telegram 代表你删除所有内容之前有一点额外的回旋余地。

要开始使用，启动**Telegram**([iOS](https://apps.apple.com/us/app/telegram-messenger/id686449807)、 [Android](https://play.google.com/store/apps/details?id=org.telegram.messenger&hl=en_US&gl=US) )，点击右下角调出其设置。接下来，点击隐私和安全，这将启动一个如下所示的屏幕:

点击**下的“如果离开”，自动删除我的账户**部分并将其更改为您喜欢的任何值:

*   1 个月
*   3 个月
*   6 个月
*   12 个月

在这里，为您的帐户设置两步验证。即使你可能没有*一个*账户，如果你让它停留太久的话，至少你的不活动账户在它存在的短(或长)时间内是安全的。作为 Telegram 的 2FA 设置的一部分，除了通过 SMS 接收代码之外，您还可以创建一个二级密码，以便在登录新设备时使用。如果你问我的话，我觉得那很安全。
