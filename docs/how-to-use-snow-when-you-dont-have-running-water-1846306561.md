# 没有自来水怎么用雪

> 原文：<https://lifehacker.com/how-to-use-snow-when-you-dont-have-running-water-1846306561>

现在，德克萨斯州的许多人有电但没有水，他们转向融化的雪来喝水或洗澡。这种方法确实有效——冬季露营者对此心知肚明——但是有一些小贴士可以帮助你有效地完成这项工作。

### 选择宽锅

虽然你可以在任何锅里融化雪(即使是瓦罐)，但锅越宽，你就可以让更多的雪直接接触锅里最热的部分。

### 找到干净的雪

你融化的雪中的任何东西都会成为你的水的一部分，所以如果你要喝雪或者用雪做饭，一定要找到干净的雪。煮沸可以杀死微生物，但不会去除污垢或化学物质。

登山家安迪·乐维基写道 他会避开树枝下的任何积雪，而且他喜欢避开最表面的积雪。寻找积雪被污染的任何可见迹象，如灰尘或臭名昭著的“黄雪”(尿液)。

请注意，你将需要大量的雪。当你加热的时候，你的堆会缩小到几乎没有，所以收集比你认为你需要的更多。

### 开始时只融化少量

你通过在水中加入雪来融化雪，所以你需要从水开始。如果你有一杯左右的白开水，把它放进你的壶里。如果没有，从几把雪开始，不要再加了，直到雪融化。

锅底的水可以导热，防止锅烧坏。如果你无视这个建议，把罐子装满雪，可能会花很长时间，浪费很多燃料来融化它。

一旦你在锅底加热了一些水，每次加入一点点雪。

### 使用盖子

一旦水开始升温，它将开始变成温暖的水蒸气。盖上盖子，你就可以蒸雪，让雪融化得更快。

### 如果你不喝，就不要煮它

你的目标是什么？如果你想确保它可以安全饮用，那就把它加热到沸腾，让它沸腾一分钟。(融化干净的雪一般被露营者认为是安全的，但 [疾控中心建议将它煮沸](https://www.cdc.gov/disasters/winter/duringstorm/indoorsafety.html) 。安全总比后悔好。)

但是如果你只是需要水来清洗，或者如果你有另外一种方法来处理水以确保它可以安全饮用，那就为自己节省一些时间和燃料。要融化雪，水只需要比雪更热；不冷不热的就行。

### 考虑过滤成品水

如果喝从你的院子里收集的雪使你不适，你可以像野营时一样过滤或处理水。(REI 这里有一个 [关于过滤器和净化器的指南](https://www.rei.com/learn/expert-advice/water-treatment-backcountry.html) 。)

如果你融化完雪，发现里面有灰尘或植物，你可以让水通过咖啡过滤器来清除它们。
