# 如何在不显而易见的情况下使用求职信模板

> 原文：<https://lifehacker.com/how-to-use-a-cover-letter-template-without-making-it-ob-1846220876>

对于那些不习惯自我表扬的人来说，写一封求职信可能会让人感到畏惧，尤其是如果你在短时间内申请了一堆工作。但是这种努力是值得的——一开始是一个空白的 word 文档，它将成为一个重要的工具，你可以用它来推销你最好的才能，让陌生人兴奋地给你发工资。

如果为每一份申请制作一份新的求职信令人望而生畏，你应该考虑为你的求职信使用一个模板，但是如果你这样做了，你就必须小心谨慎。使用一个明显的填空模板不会给你在招聘经理那里赢得任何分数——你仍然希望你的个人风格能够展现出来。

### 涵盖基础知识

有一个简单的公式可以涵盖书信写作，就是说每封信不需要*需要*完全原创。幸运的是，你可以修改你的求职信，让它遵循一个总体流程，用段落来讨论你资质的不同方面。

下面是 [简历天才](https://resumegenius.com/blog/cover-letter-help/cover-letter-format) 的建议:

*   介绍性段落:这是引起你注意的部分。这也是信件被全文阅读和被扔进雪泥堆的区别。用这一部分用热情洋溢的词语描述你的经历，并说明为什么它让你非常适合你申请的工作。(我一直认为说出你申请的工作是一个很好的做法，例如，“我认为我的经验使我成为[插入职位名称]的优秀候选人。”")
*   正文段落:描述你的经历将如何在你在公司的角色中发挥作用。请随意详述你在简历中列出的信息，详细说明这些信息如何在职场成功中发挥作用。
*   **行动号召:**在最后一部分——还应该包括一句谢谢——告诉招聘经理你的兴奋之情(利用这个机会最后一次回到你的专业领域),并说明你期待与他们交谈。

### 为不同的工作修补相同的粗略轮廓

虽然一封求职信的总体感觉应该是一致的——重点总是通过赞美你自己来让别人注意到你——但你的信的内容应该根据你想要得到的工作而有所不同。

因此，考虑像拼图游戏一样起草一份求职信，然后剪切/添加你需要的内容，使其更适合各种工作，确保通过 [分享你职业生涯中的例子](https://lifehacker.com/bring-three-key-stories-with-you-to-your-next-job-inter-761573855) 来展示你处理尽可能多的工作职责的能力。如果你一周申请一份社交媒体营销的工作，下一周又申请一份公关的工作，显然你会在求职信中更多地强调不同的技能，这取决于求职清单。但是很多关于你的经历、你的职业抱负以及你作为一个慷慨而勤奋的同事所带来的东西的一般内容将保持不变。

你可以把你的求职信想象成你的职业故事。这样，这封信就像一部小说，从一个基本前提发展到一个预定的终点——只有一些章节根据你申请的工作而变化。

### 认识到什么是全面有效的

如果你能想出让人事经理感到无力的绝妙文案，那就在你所有的求职信中使用它。不要提交完全相同的信件。但是如果你想出了一个有效的方式来表达各种事情，比如你的职业荣誉，或者你的经理给你的赞美，那么不要羞于把这些东西塞进你所有的信中。保存你发出的每封求职信的草稿，这样你就可以在早期版本中找到与其他潜在职位相关的信息。

有一个简单的规则可以证明减少为每个职位起草定制求职信的时间是合理的:如果没有坏，就不要去修补。尤其是关于你作为一个工人和同事的一般信息，无论你在找什么工作，这听起来都是真的。
