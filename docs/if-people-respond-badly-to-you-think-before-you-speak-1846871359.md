# 如果人们对你的反应不好，在你说话(或在网上发帖)之前先“想想”

> 原文：<https://lifehacker.com/if-people-respond-badly-to-you-think-before-you-speak-1846871359>

我们都曾说过后悔的话。也许你有开坏玩笑的倾向(看不懂房间)。也许你喜欢表达你的热情接受最新的社交媒体争议 ( 和发现当愤怒转向你时，你有点困惑。也许你只是有点多嘴。



如果你发现自己比你想象的更频繁地遇到这些情况，考虑记住一个缩写，它可以帮助你更好地测量你说什么，你怎么说，以及*什么时候*。恰当地说，是思考。

### 想

如果你准备告诉我，你不需要一个愚蠢的缩写来控制你的松散的通 ue，由认为传授的教训绝对应该让你暂停，因为它会让冲动的发言者后退一步。

下面是它是如何抖出来的，每个字母拼出了意图的种类，这些意图对于任何真诚的人际对话都是至关重要的:

**T–真**

**H–有用的**

**我——鼓舞人心的**

**N–必要的**

**K-种类**

基本上，这个缩写意在将作为某种停止标志，让你在开始你可能渴望阐述的任何话题之前放慢。衡量你的观点或故事是否包含虚假是一个很好的起点(真的)，但随后的每一封信也值得记住。

问问自己你将要说的话是否有帮助，是否有利于更多的对话(鼓舞人心)总是值得的。衡量你的评论是否真的有必要也是一个有用的练习。(插入你对同事们正在热烈讨论的事情的负面看法有多重要？你说的话是否有意义，尤其是当你在进行批评，甚至是开玩笑的时候？

### 但是认为 *在*说话之前

我并不是说你需要在每一次平凡的谈话之前，都要经历一次五点缩略词；这样就导致了社会瘫痪。但是当讨论逐渐进入辩论领域的时候，或者当你和那些经常惹你生气的人交流的时候，在你发言之前快速而安静地问自己几个问题是值得的。社交媒体也是如此，你可能应该打开手机的锁屏，思考你想说的话是否有用、真实、友好、必要。

此外，抑制你的冲动会帮助你成为一个更好的倾听者。正如公共演讲应用[Orai](https://www.orai.com/blog/think-before-you-speak/)所指出的，“如果不积极倾听，几乎不可能为讨论增加价值或推进对话”，因为“你可能最终会说一些你不想说的话。”

无论你是想避免无意中激怒你身边的人，还是想激怒网上的人，说话之前先思考一下是很有用的。
