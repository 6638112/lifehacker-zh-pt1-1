# kindle对孩子们来说是一个安全的搜索引擎

> 原文:[https://life hacker . com/kiddle-is-a-most-safe-search-engine-for-kids-1845644753](https://lifehacker.com/kiddle-is-a-mostly-safe-search-engine-for-kids-1845644753)

作为一名家长，我不喜欢缓慢但肯定地将我的儿子发布到互联网上。但是这个孩子已经四年级了，他不仅需要偶尔为学校项目研究课题，还需要学习如何在网上搜索信息——这可能是他成年后每天都会做的事情。也就是说，我还不愿意交出整个YouTube或者允许他使用谷歌的免费服务。所以当我通过 [Recomendo](https://www.recomendo.com/) 简讯得知[【Kiddle.co】](https://www.kiddle.co/)的时候，我就被激起了兴趣。

Watch

Recomendo是这样说的:

> Kiddle.co是一个图文并茂的大字体搜索引擎，专为儿童设计。它由谷歌安全搜索提供支持，因此只返回对家庭友好的结果。如果输入任何“不好的”单词，你会得到一个“哎呀，再试一次！”我试图通过搜索我能想到的任何“成人”词汇来打破它。“死亡”告诉我再试一次，“死亡”把我带到了儿童死亡知识的页面，这很有趣。我没有孩子，但如果我有，这将是他们的主页。

不过，我太多疑了，不会轻易相信，所以我继续自己的探索，以确定Kiddle是否是我一直在寻找的解决方案，或者只是互联网上又一个黑暗、不安全的角落。我发现的确如此...在中间的某个地方，但是倾向于一个好的解决方案。请允许我解释。

乍一看，Kiddle似乎是由谷歌自己创建的，或者是谷歌的一部分。Kiddle标志的风格，甚至每个字母的颜色，都让人想起了谷歌。“谷歌定制搜索”的标志就在搜索栏的下方醒目地显示出来——但这是为了表明，任何非Kiddle编辑亲手挑选的结果都已经通过谷歌的 [安全搜索](https://support.google.com/websearch/answer/510?co=GENIE.Platform%3DAndroid&hl=en) 功能过滤掉了，这有助于剔除露骨的内容。不过，这并不意味着Kiddle是由谷歌创造的；事实上，英国广播公司 [曾报道过](https://www.bbc.com/news/technology-35694883) 基德和这个科技巨头之间没有已知的联系。

以下是Kiddle所说的搜索结果是如何被审核的:

*   前1-3个结果:专为儿童编写的安全网站和页面；由Kiddle编辑精心挑选和检查。
*   结果4-7:安全、可信的网站，这些网站不是专门为儿童编写的，但内容以简单的方式编写，易于儿童理解；由Kiddle编辑精心挑选和检查。
*   结果8+:为成年人编写的安全、著名的网站，提供专家内容，但儿童难以理解；通过谷歌安全搜索过滤。

### 这不是傻瓜才知道的

Kiddle为搜索结果的第一页筛选安全的教育网站，做得相当不错。但是一旦你得到了非策划的、过滤的结果，更有可能出现一些不适当的东西——如真实犯罪电视节目中的图形图像，正如 [常识媒体](https://www.commonsensemedia.org/website-reviews/kiddle?utm_expid=.RCp3TD4oT6qWuBfx5635Eg.0&utm_referrer=https%3A%2F%2Fwww.commonsensemedia.org%2Fwebsite-reviews%2Fkiddle%2Fuser-reviews%2Fadult) 所指出的那样——溜走了。以下是常识媒体认为父母应该了解的关于Kiddle的其他信息:

> 该网站旨在阻止搜索结果中的脏话和其他有问题的词。但根据搜索结果，仍然可以找到一些暴力内容，例如电视节目或猎人中的内容。酒精或药物相关搜索提供的事实结果并不认可或促进使用或滥用。一些搜索结果可能指向基于销售的网站，也可能在结果页面上显示基于搜索的广告。父母也应该知道，如果他们遇到任何有问题的网站，他们可以提交请求，让Kiddle屏蔽它们。用户也可以要求禁止某些关键字。

常识媒体认为该网站适合八岁以上的儿童，但在整体质量和学习潜力方面只给了它不冷不热的三星。

### 但是它也可能过于严格

这不是你经常从父母那里听到的抱怨，但Kiddle过去一直面临着批评，因为他们在 [搜索词中屏蔽了](https://www.cbc.ca/news/trending/kiddle-filter-search-words-gay-puberty-1.3472362) ，包括女同性恋、男同性恋、双性恋、跨性别者、青春期、月经、自杀和虐待儿童(这些都是现在可以搜索的词)。

对于大一点的孩子——比如说，大一点的青少年——这只会比他们想要的或者可能已经习惯的更具限制性(我的意思是，这是一种意义)。他们将无法从Kiddle跳转到YouTube或访问各种流行的在线多人游戏。

### 我的结论

在我自己研究和使用它之后，我的主要收获是Kiddle是一个8-12岁儿童(如果你密切监督的话，年龄更小，T1)出于教育目的使用它的好工具。一个11岁的小孩正在写一份关于鲨鱼的报告，他会发现这很有帮助，例如，可以很容易地找到专门为小孩写的适合年龄的信息*，以及可以下载并用于个人或教育目的的照片()。*

孩子们可以在这样的搜索引擎上学习一些很酷的、有教育意义的东西，而且它在很大程度上是安全的(有一些例外)，但这不会成为他们寻求乐趣或娱乐的东西，他们可能会发现它对他们的需求来说有点太限制性了。
