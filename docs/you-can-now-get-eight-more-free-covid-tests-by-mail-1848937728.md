# 您现在可以通过邮件获得八次免费的COVID测试

> 原文:[https://life hacker . com/you-can-now-get-eight-more-free-covid-tests-by-mail-1848937728](https://lifehacker.com/you-can-now-get-eight-more-free-covid-tests-by-mail-1848937728)

首先你可以向政府 申请四次免费快速检测。然后你可以 [请求四更](https://lifehacker.com/how-to-get-a-second-batch-of-at-home-covid-tests-for-fr-1848619121) 。现在，有第三轮产品，这一次你将得到8次测试，每户总共16次。现在我们有进展了。

要获得您的新一批测试，请前往美国邮政署​的测试套件订购页面(或前往【covidtests.gov】[，这会将您重定向到那里)，并填写您的姓名和地址。工具包没有成本，也没有运费。F 把它填出来，你会得到两个四个测试包的跟踪号码(我想对他们来说，处理四个一组还是最容易的。)](https://www.covidtests.gov)

你仍然可以得到前几轮的测试结果。因此，如果你还没有从美国政府那里得到任何免费测试，请填三次表格。(第一次下单后，页面会邀请您下单。)

这些是快速抗原测试，就是那种你用棉签在鼻孔里摩擦，然后在测试卡上寻找两条线的测试。他们可以来自各种品牌，因为政府已经从几家制造商那里购买了测试来满足需求。

与此同时，更多测试和疫苗的资金在国会停滞不前，拜登总统敦促地方政府将他们的COVID救济金花在预防犯罪和T3上。COVID测试过去在许多现场是免费的，但现在政府 [敦促](https://www.hhs.gov/coronavirus/community-based-testing-sites/index.html) 人们联系健康中心和药店了解“低成本或免费测试的可用性”安东尼·福奇，白宫的首席医疗顾问，早在4月份就说我们已经“走出了疫情阶段”，但是COVID病例数量又开始增加了，尽管诊所检测更难获得，家庭检测也没有反映在官方数据中。但是，嘿，至少你不必为一些你无疑会需要的测试付钱。
