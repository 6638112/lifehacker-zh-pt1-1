# 到底什么是功能性健身？

> 原文：<https://lifehacker.com/what-is-functional-fitness-anyway-1847812334>

在过去的几十年里，“功能性”健身已经被视为从小众实践到趋势再到笑话的一切。自称为“功能性”的训练方式也各不相同，从俯卧撑和弓步到壶铃和杠铃训练。那么功能性健身*到底是什么*？



## 功能性健身与其说是一种训练方式，不如说是一个时髦词

如果你问指导功能性健身的人，他们可能会告诉你，这是关于在日常生活中帮助你的锻炼。也许这意味着带着沉重的哑铃走农民的路，这样你就有足够的力气一次拎起所有的杂货。也许它正在做数百个空中深蹲，这样你就可以弯腰抱起你的孩子。也许是 [在 Bosu](https://lifehacker.com/whats-a-balance-trainer-even-for-and-when-should-you-u-1847761841) 上保持平衡，这样你就不太可能滑倒在结冰的人行道上。

历史学家康纳·赫弗南 [将功能性健康](https://barbend.com/functional-fitness-history/) 的根源追溯到为一般健康而不是专门为力量或运动而规定的锻炼。有时，这些会使用不寻常的设备，如滑轮和加重球，或者今天的战斗绳或悬挂训练器。

今天的教练经常定义功能性健身与他们认为的“常规”健身相反。对于一些人来说，常规训练意味着大量的单关节练习，如二头肌弯曲，所以他们会设计涉及全身的复合运动。对于其他人来说，常规训练意味着你在使用大重量，所以他们认为功能性训练是使用小重量或仅使用体重的锻炼。对另外一些人来说，常规训练意味着做成套动作并在其间休息，而功能训练则让你一直运动。

换句话说，“功能性”可以指你的教练喜欢的任何类型的锻炼。

## 有时这是一个暗语

就在功能性健身热潮似乎正在消退的时候，似乎越来越多的健身房和教练又开始使用这个术语了。但这一次，我认为一些具体的事情正在发生:“功能性”是“CrossFit 型锻炼，但不是 CrossFit 品牌”的代码

CrossFit 混合了杠铃训练、体操和健美操动作以及有氧运动。锻炼可能包括技能练习、力量训练和最著名的定时“WODs”(一天的锻炼)，需要有氧健身来完成。

但是 CrossFit 这个名字是有商标的，它和一个特定的公司联系在一起，而那个公司在它的历史上有一些不愉快的事情。如果你喜欢这种锻炼方式，但不想做 *CrossFit* CrossFit，你会怎么做？你叫它别的东西。

## 没有运动是*非*功能性的

训练自己在日常生活中变得更好的想法并不是一个坏主意。我们都需要 [力量](https://lifehacker.com/why-cardio-and-strength-training-are-both-important-1845206358) 和 [机动性](https://lifehacker.com/whats-the-difference-between-flexibility-and-mobility-1847275695) 才能作为一个人存在，而不会一直抱怨我们的膝盖和背部，随着年龄的增长，这一点会加倍。

但是你需要一种特定类型的运动来做到这一点吗？不完全是。在一些人眼里，简单乏味的杠铃深蹲可能没有“功能”，但它们仍然可以增强腿部力量，帮助你抱起孩子。任何改善你健康状况的东西都会对你的日常生活有所帮助。

如果你想从功能性健身的世界中吸取教训，就让它成为你不局限于任何健身的刻板印象。平衡训练既有趣又有帮助； [握力训练](https://lifehacker.com/februarys-fitness-challenge-is-grip-1841383983) 也是如此，还有核心训练，间歇有氧训练，以及各种你平时可能想不到在健身房做的事情。学习新技能对你的大脑和身体都是一种锻炼，而且也是值得的——即使你永远不会发现像倒立俯卧撑这样的东西有什么“功能性”用途。
