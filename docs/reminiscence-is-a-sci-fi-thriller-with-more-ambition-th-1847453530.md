# 《追忆似水年华》是一部野心大于凝聚力的科幻惊悚片

> 原文：<https://gizmodo.com/reminiscence-is-a-sci-fi-thriller-with-more-ambition-th-1847453530>

[](https://gizmodo.com/hugh-jackman-follows-his-heart-with-his-fists-in-the-1847018456?_ga=2.154583034.395639388.1629119564-1794021061.1604501433)*，剧情片处女作 [*西部世界*联合创作人丽莎乔伊](https://gizmodo.com/how-westworld-prepared-lisa-joy-for-her-ambitious-featu-1847451971) ，企图以大制小。这是一部科幻黑色电影，充满了发人深省的酷想法，通常不会融合在一起的流派，令人难以置信的演员和华丽的视觉效果。然而，这些最终很少以有意义或令人满意的方式结合在一起。潜力很大，但最终，脱节的天性会阻碍它的发展。*

*在由乔伊编剧并执导的《T2 回忆》《T3》《T4》《T5》中，气候变化已经将 [世界的大部分淹没在水下](https://gizmodo.com/five-big-takeaways-from-the-new-un-climate-report-1847449687?_ga=2.26371261.1805403967.1628513247-1794021061.1604501433) ，彻底颠覆了生活。白天在外面太热了，所以人们在阳光下睡觉，晚上去工作，并在重温他们的旧记忆中找到安慰和安慰——通常以逝去很久的亲人或干燥土地上的美好时光为特色。尼克·班尼斯特(休·杰克曼饰)专门研究这种技术，为了做到这一点，他和他的商业伙伴瓦特([Thandiwe Newton](https://gizmodo.com/westworlds-thandie-newton-has-mixed-feelings-on-maeves-1844292080?_ga=2.88986585.395639388.1629119564-1794021061.1604501433))给人们戴上头套，给他们注射一种特殊的针剂，然后将他们浸没在一个水箱中。一天，一个叫梅(丽贝卡·弗格森饰)的女人走进来提供服务，尼克很快爱上了她。他们的关系导致所有的人物都纠缠在一个扭曲的、黑色电影启发的网络中。*

*一开始，你会情不自禁地被乔伊创造的世界迷住。作为作家和导演，她构思了一个真实可信的科幻前提。在复杂的世界构建和分层的主题中，她揭开了一个谜——首先将我们带入尼克和梅之间看似感人的、不断发展的浪漫故事，但随着梅消失得无影无踪，故事变成了一部悬疑小说。很引人注目。然而，一旦尼克决定他只专注于寻找她，所有关于*回忆*的事情都被推到一边。曾经独一无二的宇宙让位于男人对寻找这个女人的痴迷。科技奇迹和 [气候反乌托邦](https://gizmodo.com/the-8-best-movies-about-apocalyptic-climate-change-1747018787) 的惊人细节开始让人感到普通和期待，为一个相当普通的追求装点门面。*

*即使当精心设计的故事前提意味着被一些大的曲折打断时，那些也是如此可预见地显露出来，它们失去了所有力量。让我们面对现实吧，如果你引入一个看似不重要的角色，而这个角色恰好是由一个知名演员扮演的(比如说， *【恐惧行尸走肉】*的克利夫·柯蒂斯，*小丑*的布莱特·卡伦，或者*西部世界*的安吉拉·萨拉弗安)，那么这个角色很有可能最终会比背景中最初的简短场景所显示的对剧情更为关键。*

*视觉上，*《追忆似水年华》*遵循类似的模式。当电影展示它自己的 [水世界](https://gizmodo.com/a-waterworld-tv-show-is-in-the-works-1847387840) 或者潜入某人的记忆时，有很多期待和阴谋。保罗·卡麦隆的电影摄影结合霍华德·卡明斯的制作设计，通过在世界的衰落中巧妙地洒下美丽，增加了这一点。然而，在电影中间的大块远离，发生在酒吧或公寓，似乎在我们的世界和电影一样可能。随着乔伊在新兴的神秘事物上磨砺，巨大、杂乱的拍摄场景和华丽实用的水下镜头似乎逐渐消失。*

*杰克曼和弗格森在电影中表现出色，但除了他们是明星这一事实外，他们并不突出，主要是凭借他们的天赋和魅力。牛顿与瓦茨的合作最多，这要归功于一个深入到她家庭历史和与酒精斗争的分层背景故事。因此，她给了这部电影最好的表现，同时掩盖了主要人物，同时增加了一些削弱，似乎无关的全片切线。*

*实际上，令人衰弱的、看似无关的话题是《追忆似水年华》的其他关键特征*。*几个场景或次要情节——比如一些复杂的肉搏战或对话讲述了当水位开始上升时发生的一场战争——偏离了中心，让你头晕目眩。电影中有很多关于梅的过去和一个强大的当地家庭的故事，所有这些都符合故事的内容，但最终几乎有太多的东西需要考虑和思考。当电影加入一些新的东西时，它几乎总是从重要的事情上转移注意力:这个世界以及尼克、梅和瓦茨之间的关系。*

*虽然《追忆似水年华》有很多让人喜欢的地方，但可能也有同样多让人不喜欢的地方。因此，这部电影不像是一个单一的故事，而更像是一堆错过的机会。这有点太密集和压倒性的所有工作在一起。然而，这个野心，虽然有点被误导，却让丽莎·乔伊成为了一个值得关注的电影制作人。她有足够的勇气去尝试一部如此大的原创电影，尽管它并不成功，但你不会因为看了它而生气。你只是不会把它推荐给很多朋友。*

*《追忆似水年华》将于 8 月 20 日在影院上映。*
