# 你用了太多感叹号！！！

> 原文：<https://lifehacker.com/youre-using-too-many-exclamation-points-1847338122>

众所周知，当你以书面形式交流时，很难判断语气。正因为如此，人们经常使用感叹号给他们的文章注入轻浮的气息，但这很容易让人忘乎所以:使用太多感叹号会让人觉得你过于热情，但如果你一个都没用，你会让人觉得你是被动攻击的发短信者 ，受到一致鄙视。



我们可以对语法政治表示同情——像感叹号这样微不足道的东西怎么会引起如此大的争议——但是让我们来探讨一下如何恰当地使用这个语法工具。

## 在不同的情况下使用不同的感叹号

当你写电子邮件或发短信时，承认感叹号的好处是有好处的。用平淡的“当然”或平静的“好的”来回复文本， [在文本中相当于中指](https://lifehacker.com/dont-text-ok-1840037229) 。相反，发送一个“当然！”或者一句“好吧！”向接受者保证你不恨他们，不想让他们死。这是音调的功能，有意识地设置一个好的音调是一件积极的事情！(你看到我刚才做的了吗？你有吗？！？！)

但是你也不想越雷池一步。在办公室环境中，用感叹号筛选你的信息是最奇怪的，因为你必须在真正的热情和肤浅之间划清界限！激动！关于！一切！

[语法上认为](https://www.grammarly.com/blog/multiple-exclamation-points/) ，如果你在给办公室同事写信，把感叹词留给真正积极的情绪吧。在讨论更严肃的问题时，没有必要用这个词，因为你的同事可能会认为“你在对他们大喊大叫。”

他们建议:

> 当写一封更长的邮件时，想象一下如果你大声朗读，你写的东西听起来会怎么样也是值得的。即使你真的对即将到来的报告感到兴奋，连续三个带感叹号的句子听起来也可能很棒！也是！激动！试着坚持每段一个感叹号。

这可能看起来有点严格，所以一个不太严格的经验法则是将感叹号限制在每三句话一个。至少我是这么看的。你应该试一试！(请在实践中观察我的方法)。

## 约会就不一样了

对于你在办公室邮件中发出的每一个感叹号，当你试图在网上追求某人或发送一条随意的短信时，这种影响只会被放大——尤其是发给一个陌生人的时候。 [梅雷迪思·戈尔登](https://spoonmeetspoon.com/) ，一位代写约会简介的约会专家，告诉 Lifehacker 导航地形的细节。

她说“对一个陌生人来说，任何过分的事情都是过分的。重要的是要记住，与闺蜜交流不同于在约会应用上发信息。”当你和刚配对的人交谈时，要记住的一件事是知道一个感叹号“在交流中是可以的，但他们不应该说完每一句话。”

基本上，一切都是有语境的。这是约会时要记住的事情，也是日常生活中更广泛的交流。

戈尔登说:

> 例如，如果有人从一栋着火的大楼里逃了出来，并在此过程中救了四个人的命，这是一个应用程序交流中的共享轶事，使用[感叹号]以上是合适的，但有人说他们已经访问了二十个国家就不合适了。

当谈到对某事感到沮丧时，最好是完全避开感叹号。毕竟，在一篇文章中，一句话末尾的句号是一个很好的信号，表明你可能会生气。但当涉及到所有其他情绪时，当涉及到全能的感叹词时，请保持适度。
