# 什么是“Web3 ”,为什么要关注它？

> 原文：<https://lifehacker.com/what-is-web3-and-why-should-you-care-1848133134>

技术喜欢谈论未来，如果你一直关注最近的行业头条，你可能已经看到了“Web3”这个短语。这不是一个新名词，但随着围绕加密货币、NFTs 和“元宇宙”的大肆宣传，Web3 越来越受到关注。虽然 Web3 *与那些时髦的流行语有着千丝万缕的联系，但拟议中的新网络标准不仅仅是猴子 jpg 和加密挖掘——它还可能使互联网更加开放、更加匿名、更加可靠。* 



至少，这是希望。有很多关于 Web3 和相关技术的宣传，这些技术表面上使这个更好版本的万维网能够运行。毫无疑问，有些议论是合理的。其中一些也完全属于“好得令人难以置信”的范畴。

### 什么是 Web3，我们为什么需要它？

正如 Web2.0 是一系列技术和监管变革的集合，将 21 世纪初的互联网改造成了我们今天所知的样子，Web3 试图对我们在网络上联系和交流的方式做出重要改变。

互联网的最初版本，我们现在称之为“Web 1.0”，非常简单。单一的社交媒体网站和搜索引擎并不存在，几乎没有人能指望仅仅通过在网上发帖来赚钱或成名。简单地连接到互联网有点困难，更不用说设计，托管和运行一个网站。在很大程度上，互联网是一堆小网站和博客的集合，用户聚集在不同的留言板和基于特定兴趣的聊天室。

然后 Web 2.0 出现了，使得上网、与他人联系以及创建能够接触到大量受众的内容变得更加容易。这些内容大多集中在脸书、YouTube 或 Patreon 等特定平台上，尽管由于 WordPress 和 GoDaddy 等公司的存在，托管自己的网站变得更加便宜。这种开放性的缺点是它对大公司及其服务器的依赖:如果这些平台宕机，你就无法访问内容或与其他用户联系。这也意味着这些公司可以收集用户数据来销售广告，并在他们认为合适的时候“控制”他们平台上的内容。这超出了“审查”或“去平台化”的问题；Meta(脸书)一直受到美国政府的抨击，因为他将广告收入置于安全和真相之上，甚至在这个平台被证明是广泛的选举阴谋和新冠肺炎错误信息的绝佳孵化器，并且是人口贩子和国内恐怖分子的宿主之后。

Web3 旨在通过创建一个每个人都可以轻松访问的去中心化互联网，同时尊重他们的隐私和匿名性，来解决 Web 1.0 和 Web 1.0 中固有的问题。总的来说，去中心化和在线匿名是大多数人想要的好东西。这也是为什么有些人已经尽其所能地将沟 [谷歌、](https://lifehacker.com/the-comprehensive-guide-to-quitting-google-1830001964) [脸书](https://lifehacker.com/how-to-delete-your-facebook-account-a-checklist-1829436824) 、以及 [苹果](https://lifehacker.com/how-to-ditch-apple-completely-1821082984) 和 [开源](https://lifehacker.com/find-ethical-open-source-alternatives-to-almost-every-a-1833746899) [替代品](https://lifehacker.com/top-10-free-alternatives-to-expensive-software-1791300629) 产品这些巨头公司自己拥有的原因。但是，Web3 将如何把我们带到这个更加开放、更加平等的互联网呢？

## Web3 技术如何工作

像当今大多数计算趋势一样，Web3 的大部分愿景是基于的区块链技术。区块链基本上是一个建立在点对点连接上的分散式网络。网络上的每台设备处理网络上发生的一小部分计算和通信，从而创建了一个无服务器的在线网络。

像区块链这样的点对点网络允许用户之间的无服务器通信。我们已经通过所谓的 mesh 应用程序看到了这一点，这些应用程序用于基于点对点的消息传递，在香港抗议活动期间，这些应用程序使[的活动人士保持联系，而不会危及他们的匿名性，因为没有中央服务器来促进他们彼此之间的联系。](https://gizmodo.com/how-mesh-messaging-apps-work-and-why-you-might-need-on-1837859615) 

您还可以使用点对点连接来分散在线内容。每个用户的电脑不是连接到托管网站的服务器，而是在他们的设备上存储一些数据。这类似于[BitTorrent 下载如何工作](https://lifehacker.com/a-beginners-guide-to-bittorrent-285489) ，一些浏览器 [已经在试验该技术](https://lifehacker.com/what-is-ipfs-and-why-does-it-matter-in-braves-web-brows-1846103479) 。如果在全球范围内实施，理论上可以完全消除对集中式服务器的需求。

虽然点对点通信的可能性令人兴奋，但区块链迄今为止最出名的是为和 [NFTs](https://lifehacker.com/how-to-create-nfts-for-fun-and-profit-maybe-1846562554) 提供动力，这两项技术本身都是重要的 Web3 技术。许多人(理所当然地)把 NFTs 和 crypto 看作是暂时的 [【技术兄弟】的干扰，往好里说是](https://kotaku.com/someone-right-clicked-every-nft-in-the-heist-of-the-cen-1848084379) ，往坏里说是彻头彻尾的骗局。然而，也有 NFT 和加密货币以有益于艺术家和普通用户的方式使用的例子。

例如，NFTs 可以允许你出售或交易你购买的数字商品，因为你可以证明这是一次真实的购买——就像向你的朋友出售你在 Steam 上购买的电脑游戏的数字副本。它还可以消除艺术家对集中平台销售其作品的依赖:例如，音乐家可以使用 NFTs 来销售他们的音乐，而不是上传到 Spotify。NFTs 的其他拟议用途包括更安全、更匿名的登录认证，甚至使招聘过程近乎即时。

最雄心勃勃的 Web3 项目是“元宇宙”概念。元宇宙被设想为一个虚拟空间，用户将在这里与在线内容进行表面上的 [互动，并在实时](https://medium.com/building-the-metaverse/web3-interoperability-and-the-metaverse-5b252dc39da) 中相互互动，通常是在 VR 和/或 AR 环境中。这个想法已经在科幻小说中出现了几十年，现在得到了马克·扎克伯格的大力推动(因此 [公司更名](https://about.fb.com/news/2021/10/facebook-company-is-now-meta/))，但还有许多公司试图将元宇宙变成现实。

## 炒作是真的吗？

人们很容易对 Web3 技术在纸面上的潜力感到兴奋，但重要的是要记住，这些计划到目前为止还只是空想，很难得到保证。许多 Web3 的传播者在经济上投资于它的成功，而不是无偏见的观察者见证一个革命性的新标准。

问题是，围绕 Web3 的炒作依赖于期票——区块链计算“可能”实现什么，NFTs“可能”如何使用。许多承诺的技术根本还不存在——或者说，即使存在，也还没有被正确实施。

这并不意味着 Web3 是蛇油，但最终的现实可能会与目前的设想大相径庭。我们已经看到 Web3 的承诺落空，并转向更可实现的目标:区块链技术最初被设想为创建一个去中心化的“世界计算机”，将每个人连接到一个巨大的无服务器网络，但现在已经转向主要支持加密货币和 NFT 交易。这些交易非常昂贵，并且需要大量的能源来处理，这对环境有明显的破坏性影响。

NFT/ 密码热潮是也是 [可以说在当前困扰消费硬件产业的芯片短缺中扮演了主要角色](https://www.singlelunch.com/2021/11/12/how-badly-is-cryptocurrency-worsening-the-chip-shortage/) 。支持者反驳说，一旦规模适当扩大，区块链将不再是资源掠夺者，但这是又一个蓝天保证。

目前，公众对 NFT 和加密货币仍然持怀疑态度或者 [无知。像谷歌和苹果这样的平台持有者已经对密码挖掘应用进行了打击，这个行业已经引起了美国和世界各地政府机构的怀疑。强烈的反对使得许多公司暂停使用 NFTs 和相关技术。](https://news.yahoo.com/only-20-of-americans-familiar-with-nf-ts-4-million-have-used-survey-165016231.html)

然而，尽管有这些挫折，Web3 仍有发展势头。

有可能在某个时候，区块链、NFT、crypto，甚至元宇宙- 类似的功能会慢慢进入我们每天使用的浏览器、应用和设备。当这种情况发生时，我们将牢牢地处于“网络 3”时代，并希望 T2 在我们的网上生活中比我们今天有更多的自由、匿名和代理权——但具体有多少还有待观察。
