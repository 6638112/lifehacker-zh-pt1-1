幸运的是，这些功能在默认情况下是不启用的，所以如果你使用浏览器，你应该是安全的。但是，如果你正在使用 Chrome 的增强拼写检查功能，或者 Edge 中的微软编辑器，你需要立即禁用它们。

在 Chrome 中，在地址栏中输入以下文本，以在正确的位置结束: chrome://settings/？搜索=强化+法术+检查

接下来，d 启用“**增强拼写检查**功能。

在微软 Edge 上，微软编辑器作为 [的扩展](https://microsoftedge.microsoft.com/addons/detail/microsoft-editor-spellin/hokifickgkhplphjiodbggjmoafhignh) 。点击工具栏上的扩展图标，进入“**管理扩展**部分。找到微软编辑器扩展，点击**移除。**从弹出菜单中，再次点击**移除**。

在微软和谷歌用基本级别的文本加密技术更新他们的产品之前，我们建议你远离这些高级拼写检查或语法工具。如果你真的想在写作方面得到帮助，我们建议你使用 [的网页编辑器](https://www.grammarly.com/grammar-check) 或 [海明威编辑器](https://hemingwayapp.com/) 。
