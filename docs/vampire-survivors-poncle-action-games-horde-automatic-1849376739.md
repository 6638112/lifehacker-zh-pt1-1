# 吸血鬼幸存者后的7个游戏

> 原文:[https://kotaku . com/vampire-survivors-poncle-action-games-horde-automatic-1849376739](https://kotaku.com/vampire-survivors-poncle-action-games-horde-automatic-1849376739)

我们需要提出我们的条件。吸血鬼幸存者是最近才出现的现象，以至于它和它之后出现的许多游戏的流派名称还没有达成一致。事实上，许多最近的克隆版本试图用模糊的术语来装扮事物，以假装远离比较，比如说“反向子弹地狱”和“带有流氓元素的生存游戏”，尽管这些游戏显然与双棍射击游戏有更多的共同点。

但是这些也不同于那些，因为发射的特殊性质是完全自动化的。这是这里的关键区别:在这些游戏中，新的能力被叠加，每个都有自己的计时器，自动启动。当然，这是入选的标准。(是的，也许是有争议的，这意味着我不会包括[*【还有20分钟就到黎明了】*](https://kotaku.com/vampire-survivors-bullet-hell-roguelike-steam-hit-1849367325) 。)我们必须具体一点，否则在某一点上这就变成了一件关于 *Robotron* 喜欢的事情。

所以，让我们把事情定下来。很多人都提到“部落”游戏，这是一个好的开始，既然“自动”是如此重要，为什么我们不同意“自动部落射手”？这是足够聪明的，因为它为人们留下了空间，让他们可以用我还没有看到的想法更有趣地创新表单，比如说第一人称自动部落射击游戏。或者回合制自动部落射手。

这种类型的另一个特点是价格。它们都是不可思议的，不必要的便宜。VS 设定了一个奇怪的标准，将自己的价格定在3美元，想必很多其他人也觉得有必要效仿。所以你可以用一个普通游戏的一半价格买到这个列表中的所有东西。

因此，承认*吸血鬼幸存者* [并不是第一款提供这些成分的游戏](https://play.google.com/store/apps/details?id=com.vkslrzm.Zombie&hl=en_GB&gl=US) ，事实上你可以合理地追溯它的起源到1982年的*机器人:2084* ，让我们来看看当你最终厌倦了*对*时可以选择的游戏

目前处于非常早期的形式，*魔法书演示者:序幕*已经显示出良好的形式。这与《T2》吸血鬼幸存者的格式没有太大的差异，但它所做的调整很有趣。

首当其冲的当然是敌人的规模。这是一个更大的游戏，有更大胆、更漂亮的精灵。哦，事实上你扮演的所有角色都是字面上的魔法书，神秘地漂浮在地面上。接下来是出现的塔尖，要求你在塔尖周围杀死一定数量的敌人。这样做，你解锁一个临时的奖金，这是一个非常好的功能。

在这里，它没有(到目前为止)令人讨厌的敌人圈，而是有更大的坏人也被势力圈包围。杀死他们，你会得到一个宝箱，它会给你一个先前解锁的法术的随机增益。

到目前为止，只有一个阶段可用，尽管有一堆书要解锁，还有大量的法术和技能。遗憾的是，这个舞台乏味得惊人，没有什么特色可发掘。然而，鉴于这是免费的，并且菜单显示了将在完整游戏中添加的大量方面，有很好的理由将它放在你的愿望清单上，并作为一个演示版本来享受它。

[PC](https://store.steampowered.com/app/2007530/Spellbook_Demonslayers/)

就在几个小时前上映的*《追骨者的爪牙》*肯定会获得后*吸血鬼幸存者*应有的关注。这是一个超低分辨率的NES风格的演示，在这里你扮演一个小生物，躲避成群的敌人，从墓地收集骨头，召唤你自己的动物支持团队为你战斗。

这是一种规模小得多的感觉，这种新兴流派的传统开阔空间，但也更快，需要更多地强调反射，从一开始就躲开敌人。除此之外，它提供了非常熟悉的常数进展，但令人愉快的是，第一次解锁时，一切都是未知的。这是一个发现你能创造的各种生物的好方法，把它们从？？?"默默无闻，然后让他们从此在你的花名册上被永久认可。

然后是所有的解锁支付运行之间，让你开始阐述核心游戏。例如，有一个冲刺能力可以在早期添加，还有施法的选项，以及从你收集的骨头开始建造更危险的爪牙。

它的特点是欢快的chiptunes，令人愉快的紫色哥特式配色方案，我不得不强迫自己停止播放它来继续这篇文章。我认为，这是一场成功游戏的最可靠标志。

[PC](https://store.steampowered.com/app/1944570/Boneraiser_Minions/)

虽然《拉撒路计划》可能是对《T2》吸血鬼幸存者《T3》核心元素最明目张胆的提升，几乎没有自己的新想法，但它以其完全不同的图形方式脱颖而出。在3D中，你控制着大量不同类型的机器人中的一个，它在屏幕上充满了数量不可思议的3D敌人，丝毫没有减慢速度。

对我来说， *Project Lazarus* 是后 *VS* 游戏中吸引公众注意力的较弱游戏之一。这不仅仅是因为这款游戏唯一真正的原创之处在于你扮演的是巨型机甲，每个机甲的控制都略有不同，但尽管它在图形上取得了成就，但它给人的感觉却非常简单。设定平淡无奇，敌人种类太少(虽然数量不多)，而且进程感太慢。然而，尽管如此，它仍然实现了“再来一次！”这个因素对这种类型非常重要。

就受欢迎程度而言，这当然没有阻碍拉扎勒斯项目的发展。早期接入游戏在Steam上有一个“非常积极”的评级，而且它的开发还有一年时间。好消息是有计划引入一系列其他游戏模式，这应该会给它更多的变化。

如果你今天拿起 *Bardbarian* ，你可能会忍不住认为它是*吸血鬼幸存者*和*植物大战僵尸*的巧妙结合。你有你的主角，他通过花费大量的音符，可以让他周围有三个其他的角色——首先是:弓箭手、格斗家和法师——他们在计时器上释放能力。然后你有敌人从竞技场的最右边攻击，试图摧毁最左边的城镇水晶。

因此，当你不在车道上时，你必须防止敌人到达屏幕的左侧，同时还要躲避他们的攻击，并一直花费资源增加你的帮派的攻击，并在其他人倒下时招募新的队友。然后，在每次尝试中收集的黄金可以用来提高你的单位，城镇的防御和你的英雄的核心能力。

只不过， *Bardbarian* 是2014年出的。就在*吸血鬼幸存者*的七年前。这是非同寻常的，这样一个辉煌，坚实的游戏应该淡出人们的视线，尤其是当它为2022年的格式现象提供了几乎所有的成分。

有趣的是，如果有什么不同的话， *Bardbarian* 有一点点*太多*正在进行。我怀疑是《T4》的吸血鬼幸存者的简约风格对它的巨大成功至关重要。(尽管在这一点上不提及Android的*魔法生存*是一种疏忽，因为它之前已经问世，而且 *VS* 有*所以*与它有很多共同点。)但如果你对一款真正感觉像是概念进化的游戏感兴趣，尽管它已经发布了这么多年，这款游戏还是很值得一试的。

现在这里有一个更复杂的关于吸血鬼幸存者的版本。在自上而下的竞技场中使用自动开火攻击的核心概念，与成群的敌人一起，这在较短的级别上抓住了一大把*杀死尖塔*，在分叉的道路上前进，然后使它成为一个有一群*巴德巴里安*的团队游戏！

你有一个国王，他必须受到保护，所以你雇佣其他角色从四面八方包围他，每个角色都有自己的定时攻击。每一关，你都可以赚取硬币，这些硬币可以用来雇佣新的船员或购买奖励物品来增强他们的能力。抛弃现有成员来购买新成员会带来其他奖励，而玩特定的构建会解锁挑战，从而增加新的元素。

当你通过关卡时，会有一些小场景，你的选择决定了你可能获得的额外能力或工具，以及打开装备和硬币的宝箱。然后，你尝试以*杀死塔尖*的方式到达最后一个boss，每次失败都有希望为下一次尝试打开新的东西。

这里面有很多东西，但是它们都非常直观地组合在一起，对于一个早期的游戏来说，它已经令人难以置信地完整和坚固。

这是另一个领先吸血鬼幸存者很长时间的游戏，在这个例子中是六年。*死亡状态*适合人群，提供了一个色彩明亮、厚实的像素概念再现，在这里你可以通过大型、极具挑战性的关卡自动开火，拥有不断增长的角色和能力。

在某些方面，这感觉更像是一款传统的子弹地狱街机游戏，具有艾萨克的*捆绑的味道，尤其是充满障碍和物品的详细位置。但是一切都准备好了，这是一个*对*的选择:当敌人出现时，你会自动发射你手里的任何武器——它甚至会瞄准你——同时收集掉落的金条来购买改进，收集特定的成就可以解锁新角色和新物品供下次使用。唯一显著的区别是水滴的稀少。*

当我开始写这个的时候，我并没有期望找到一个我更喜欢*而不是*吸血鬼幸存者*的游戏。从某种意义上来说，这些游戏中的许多都是衍生的，鉴于这个列表中大多数游戏的持续发展， *VS* 只是有了一个开端。当其他人都在追赶时，它的迭代版本给了它优势。嗯，除了——在我看来——*灵魂猎人。** 

首先，它只是图形华丽。非常清晰的卡通艺术，在复杂的菜单和超清晰的战场中完美呈现。但更重要的是，玩起来感觉要复杂得多，好像这是我们自动部落射手的一个进化步骤，向更RPG的东西推进了一步。

核心游戏大家都很熟悉。你选择一个解锁的英雄，然后他们自动发射你选择的攻击范围(最多四次，从一个不断扩大的池中)，你不断从收集的掉落中升级。仅仅是找到一个最适合你的攻击组合就足够了，但是除此之外，你还有一堆战斗中的改进和战斗外的解锁可以收获。

有金币，可以在产卵帐篷商店使用，以获得一些额外的生命值，其他形式的货币，甚至宠物(一旦解锁)在战场上使用。然后是紫色碎片和灰色符文，每一个都用来解锁新的能力，能力提升，玩家角色，地图，奖励等等，在一个难以置信的复杂的技能网络中。它增加了一个研磨的元素，但在好的方面，因为你可能开始一次跑步，而不是专注于完成15分钟的水平(从而与最后的老板战斗)，而是专注于收集碎片。也许你正在关注一项新技能，或者是一把开启其他机会的钥匙。

这绝对让我着迷，甚至超过了*对*。它感觉更完整，更复杂，有更多有趣的地图可以探索，也有更大的最小化兴趣感。而且还没完成！尽管现在我觉得它已经准备好发布了，但是至少还有六个月的开发时间。
