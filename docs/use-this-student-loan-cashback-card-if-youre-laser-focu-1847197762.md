# 这张新信用卡有助于偿还学生贷款——值得吗？

> 原文：<https://lifehacker.com/use-this-student-loan-cashback-card-if-youre-laser-focu-1847197762>

我们最近谈到了帮你付房租的信用卡，但是帮你付学生贷款的信用卡呢？有一种 [新卡](https://www.laurelroad.com/press/laurel-road-releases-credit-card-with-the-reward-millennials-really-want-cashback-to-pay-student-loans/) 就可以做到这一点 **—** 提供 2%的购物返现，这种卡可以应用于一系列学生贷款服务机构。下面就来看看是不是适合你的好牌。

## 如果你非常关注学生贷款，这张卡是个不错的选择

[月桂路学生贷款返现卡](https://www.laurelroad.com/press/laurel-road-releases-credit-card-with-the-reward-millennials-really-want-cashback-to-pay-student-loans/) (有点拗口)从其他返现卡和旅行奖励卡中脱颖而出，帮助你专注于偿还学生贷款。这不是第一张这样做的卡(Sally Mae 的卡系列是 [一个例子](https://www.nerdwallet.com/article/credit-cards/sallie-mae-mastercard-review) )，但它肯定是最灵活的，因为 Laurel Road 声称你可以为 95%的学生贷款贷方还清贷款(列表可以在他们的一个常见问题的答案中找到， [这里](https://www.laurelroad.com/healthcare-banking/cash-back-credit-card/) )。以下是该卡的主要特性和优势:

*   合格服务机构 2%的学生贷款返现或 1%的返现声明信用
*   在开新卡 90 天内消费 5000 美元后，可获得 500 美元的学生贷款返现欢迎奖金或 250 美元的对账单信用
*   0 美元年费

*   0%国外交易费

## **助学贷款返现卡值不值？**

可以，但前提是你打算用这张卡来偿还学生贷款。针对你的学生贷款的 2%是一个有竞争力的利率，但是 1%的现金返还作为一个声明信贷并不都很好。对于返现卡来说，欢迎奖金相当不错，尽管与其他返现卡 相比，5000 美元的消费目标是相对较高的 [。](https://www.creditcards.com/credit-card-news/500-credit-card-bonus/) 

此外，如果你想简化你的信用卡习惯并需要一个新的开始，12 个月的零利息优惠(如果你从另一张卡转移你的余额)是一个完美的选择。之后，变量 APR 跳到 13.99%到 22.99%，取决于你的信用度。

## **底线**

有更好的返现卡，但如果你致力于尽快偿还学生贷款，这张卡会很好用。额外的好处是，你不会因为奖励而陷入困境，这些奖励可能会无意中导致更多的支出。例如，你可能会从旅行奖励卡中获得免费的酒店或航班，但你可能最终会在度假时在食物、购物或活动上花费更多。另一方面，这张卡相对来说比较简单，有它的奖励 **—** 但这是重点，它应该很简单。
