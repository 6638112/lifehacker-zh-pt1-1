# 用老式托盘中的碎冰制作更好的鸡尾酒

> 原文：<https://lifehacker.com/make-better-cocktails-with-cracked-ice-from-a-vintage-t-1846126120>

鸡尾酒人喜欢谈论透明冰的重要性，但对有裂缝的冰却没有这么多谈论，这太糟糕了，因为有裂缝的冰是冷藏你的饮料的最好的冰。

Watch

根据受人尊敬的鸡尾酒作家戴夫·旺里奇的说法，“用碎冰搅拌”是他能给你的最好建议——这已经说了很多，因为旺里奇有很多建议要给你。

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1270096801828388864&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1270096801828388864&autosize=1) 

这一技巧背后的逻辑很简单:裂解冰会增加其表面积，从而导致“ [”更快的冷却和更多的稀释，但不会太多。](https://twitter.com/DavidWondrich/status/1270132461402574849)

你可以买几袋碎冰，或者自己用酒吧勺子敲碎它。或者，你可以买一个可爱的老式冰块托盘，它可以一次打碎一大堆冰水。(你可以在 Etsy 上找到它们 [)。](https://www.etsy.com/market/1950s_ice_cube_trays)

我不知道这种特殊类型托盘的发明者是否打算创造这样一种优雅的碎冰机。我怀疑这是最简单的方法，可以把冰块打碎，放入玻璃杯中，因为当时材料随处可得。意图是什么并不重要——这东西完全粉碎了它所在的大量冷冻水，产生了各种不同形状和大小的碎片和粗糙的立方体，这意味着你一次得到一大堆碎冰*。如果你搅拌了很多鸡尾酒，你可能应该得到一个。*

*你可能认为这很轻浮，但好的鸡尾酒需要一点轻浮。事实上，我认为轻浮也是乐趣的一部分。你知道什么不好玩吗？每次你想喝马丁尼、曼哈顿或任何其他有利于冷冻和稀释的饮料时，都必须用酒吧勺子手动敲碎冰块。另外，拉动破冰杆是非常令人满意的。*
