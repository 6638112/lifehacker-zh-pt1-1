# Windows 365 没多大意义

> 原文：<https://lifehacker.com/windows-365-doesnt-make-much-sense-1847420584>

继 Office 和 [Xbox](https://lifehacker.com/how-to-play-on-xbox-cloud-streaming-and-why-its-such-a-1847197043) 之后，微软又将 Windows 放入云端。有了 Windows 365 服务，企业将能够直接从微软的服务器上下载 Windows 设备——就像网飞一样，但只适用于你的工作电脑。

与服务器群不同，这些将是 Windows 机器的单独实例，运行着一组独特的可定制的应用程序和用户数据，可以在任何设备(包括手机和平板电脑)上流式传输。如果您的互联网连接可以传输电影，它也可以传输窗口。

## Windows 365 云 PC 自带什么？

使用 [Windows 365](https://www.microsoft.com/en-us/windows-365) ，企业可以为多达 300 个用户创建账户，每个用户都有自己的 Windows PC 实例(目前是 Windows 10，发布后是 Windows 11)。单核虚拟 CPU、2GB RAM 和 64GB 存储的计划起价为每个用户每月 24 美元(如果您拥有 Windows 10 许可证，则为 20 美元)。配备 4GB 存储和 64GB 存储的双核虚拟 CPU 计划起价为每个用户每月 32 美元。

价格从那里开始快速上涨。配备 32GB 内存和 512GB 存储空间的顶级八核虚拟 CPU 将为您支付每个用户每月 162 美元的费用。

也就是说，Windows 365 是一项非常独特的服务。虽然有第三方服务可以让你在他们的数据中心运行 Windows，但没有人能像微软一样提供集成的体验。在这里，帐户管理员不需要做太多事情来设置 Windows 365 用户，他们所要做的就是登录到管理界面，创建一个实例，将其分配给用户，根据业务需求进行定制，这样就完成了。然后，Windows 365 将负责安装和创建实例、更新软件、安装操作系统更新以及修补安全问题。

虽然每月 32 美元的要价似乎很高，但以下是 Windows 365 订阅可以为您带来的好处:

*   **软件更新**:自动、安全地安装软件更新，包括安全补丁。
*   **无需设置**:您不需要整个 IT 团队来设置、监控和更新设备。Microsoft 将在后台为每个实例执行此操作。
*   **极速互联网连接**:微软数据中心的互联网连接速度非常快， [达到 10 Gbps 速度](https://youtu.be/1dmUggAKzEI?t=223) 。下载文件、更新之类的事情会在瞬间发生。
*   **一种状态无处不在** : Windows 365 将在任何设备上的任何浏览器中运行。此外，还有专用于 iPhone、iPad、Android、Mac 和 Windows 的微软远程桌面应用程序。而且不管从哪个设备访问 Windows 365，电脑都会是同样的状态，同样的数据。
*   **易于扩展** : Windows 365 可以根据您的业务需求进行扩展。您可以从基本配置开始，随着需求的变化，您可以增加 RAM、内核数量或存储。

## 该不该为 **Windows 365 云 PC** 买单？

这就给我们带来了一个问题:这是为了谁？不是针对消费者，或者独立职业的。微软只为商业和企业客户提供这项服务——即使你谈论的是商业用户，也只有非常有限的一部分使用案例适合它。

首先，服务只支持 300 个用户，所以大机构出局。如果一家企业想要托管/控制自己的 Windows 实例，情况也是如此；这在 Windows 365 中是不可能的。归根结底，Windows 365 是为那些重视无缝体验、开销小、胜过一切的企业设计的。

如果您在一个分散的团队中经营着一家拥有几十名员工的企业，但您仍然希望自己控制所有用户的 Windows 体验(没有大型 IT 团队)，这对您来说可能是一个好主意。你可以通过发行便宜的笔记本电脑来逃避，因为整个 Windows 体验将在云中运行。也就是说，如果你不介意每个用户每月 30-40 美元的费用。

## Windows 365 的更便宜的替代品

我们上面描述的场景是一个非常狭窄的场景——即使这样也不太划算。如果以三年为时间表，基本双核计划将为每个用户总共花费 1152 美元。比方说，你可以花 500 美元从戴尔买到一台不错的中档笔记本电脑(如果你选择翻新模式 l，价格会更便宜)。

如果你真的需要为你的员工提供远程桌面功能，那么每个用户每月 9.99 美元就可以订购一个 AnyDesk 。需要控制整个体验？微软自己的 [Azure 虚拟桌面](https://azure.microsoft.com/en-us/services/virtual-desktop/) 将让你创建和托管定制的 Windows 10 云电脑。像所有的商业决策一样，这个决策可能会归结到成本上。在你做决定之前，做一下数学计算，看看这是不是最适合你的选择。
