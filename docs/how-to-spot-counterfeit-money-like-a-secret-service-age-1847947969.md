# 如何像特工一样识别假币

> 原文:[https://life hacker . com/how-to-spot-fake-money-like-a-secret-service-age-1847947969](https://lifehacker.com/how-to-spot-counterfeit-money-like-a-secret-service-age-1847947969)

虽然现金不像过去那样普遍使用，但假钞仍然是一个问题。根据财政部 的数据，估计有7000万美元的假钞在流通，或者说每10000张真币中就有一张假钞——尽管实际数字可能远不止这些。这就是为什么知道如何识别假钞不会有坏处，尤其是如果你在服务行业工作的话。以下是一些鉴别伪钞的方法:

*   **纸张的手感:**识别假钞最简单的方法就是摸摸纸张，纸张有颗粒，很脆。假钞通常印在由树木纤维素制成的普通纸上，但真钞是由碎布纸制成的——由亚麻和棉纤维混合而成。大多数人都能直观地感受到其中的区别，尤其是在对比假钞和真钞的时候。

*   凸起的墨水:这种效果很微妙，但对于真的钞票，你可以用指甲在表面上拖动，感觉到凸起的墨水有一点锯齿或振动，特别是在有很多蚀刻的地方。大多数造假者努力复制这种效果。
*   **变色墨水:**在新版钞票上，10美元及以上面值的钞票右下角的数字会从铜色变成绿色。只要把钞票前后倾斜，你就能看到了。有时伪造者会使用闪光剂，这样当你倾斜它时，数字会从闪亮变成哑光，但如果颜色不变，你仍然可以看出它是假的。
*   当面对光线时，新的合法钞票上会有一个与钞票匹配的数字水印，或者是一个描绘在钞票上的幽灵般的总统图像。钞票的两面都可以看到水印。
*   **安全线:**较新的钞票应该有一个嵌入的条纹，在钞票上垂直延伸，可以在肖像的左边或右边(这取决于面额)。钞票的面额也会刻在细线上。
*   **模糊印刷:**伪钞的一个标志是模糊的线条或文字，因为真钞使用极其精确且难以复制的模切印刷。模糊印刷最好通过仔细观察钞票上印刷的非常小的文字是否清晰可辨来识别。
*   红色和蓝色的线:这非常细微，但真正的钞票有细小的、像头发一样的线织在钞票上，看起来是蓝色或粉红色的(我个人认为蓝色的线更容易被发现)。线条非常细，这使得假冒打印机更难复制。
*   紫外线发光:我自己还没有证实这一点，但显然如果一张合法的钞票被举到紫外线灯下，5美元的钞票会发出蓝色的光，10美元的钞票会发出橙色的光，20美元的钞票会发出绿色的光，50美元的钞票会发出黄色的光，100美元的钞票会发出红色的光。如果你有紫外线灯，试试看，分享你的发现。
*   **序列号:**序列号的第一个字母应该对应于票据设计的年份，称为“系列”号，也印在票据上。(供参考:E = 2004，G = 2004A，I = 2006，J = 2009，L = 2009A，M = 2013，N = 2017)
