# 疾控中心的 COVID 疫苗接种指南对家庭意味着什么

> 原文：<https://lifehacker.com/what-the-cdcs-covid-vaccination-guidelines-mean-for-fam-1846446541>

任何祖父母都会告诉你，偶尔的视频聊天或在后门廊的社交距离，蒙面聚会都不能算作与他们的孙子孙女足够的优质时间。祖父母想要的——他们渴望的——所有这些漫长的月份都是他们在现实生活中可以看到的拥抱和微笑。最后，疾病预防控制中心说他们可以拥有它——当然有一些限制。

本周，疾病控制和预防中心 [发布了对已经完全接种疫苗的人的建议](https://www.cdc.gov/coronavirus/2019-ncov/vaccines/fully-vaccinated.html) (并等待两周以使完全保护生效)。以下是已经接种疫苗的祖父母(或阿姨或叔叔)最大的收获:

> 你可以在室内与来自另一个家庭的未接种疫苗的人聚集在一起(例如，拜访住在一起的亲戚),不戴口罩，除非这些人或与他们同住的人患新冠肺炎严重疾病的风险增加。

所以现在还不是和所有的第二代表亲举行家庭烧烤的时候，但是如果你已经接种了疫苗，并且你想和来自另一个家庭的未接种疫苗的人聚在一起(他们不是高风险)，你可以这样做。里面！不带面具！就我个人而言，我希望我的父母会在他们第二次开枪后的 336 个小时敲响我的前门。

如果你*决定邀请第三个家庭的一些未接种疫苗的家人或朋友， [疾病预防控制中心说](https://www.cdc.gov/coronavirus/2019-ncov/vaccines/fully-vaccinated-guidance.html) 你应该把聚会带到外面，每个人都应该戴上口罩，保持距离，由于群体中未接种疫苗的成员对彼此构成的风险。*

*不幸的是，这并不意味着孩子们在生活的其他方面会有很大的变化——他们仍然应该戴着面具，在外面，和 [与朋友一起玩](https://offspring.lifehacker.com/how-to-host-a-socially-distanced-playdate-1843968368) ，如果他们亲自去学校，所有相同的限制将仍然存在。但是他们被允许再次拥抱他们的祖父母，这是值得庆祝的进步。*
