# 这个世界需要更多的烹饪比赛(以及如何举办一场精彩的烹饪比赛)

> 原文：<https://lifehacker.com/the-world-needs-more-cook-offs-and-how-to-host-a-good-1848917757>

在烹饪中，就像在生活中一样，一点健康的竞争可以激励人们出现、展示和闪耀。烹饪比赛，烘焙比赛，任何一种与食物相关的"off "——将一场以食物为主题的聚会设计成一场比赛将会让客人们对参与派对感到兴奋。

烹饪大赛是一个简单的概念:选择一种食物，并要求人们制作并带来他们最好的食物。每个人都吃别人的食物。获胜者被选出，失败者被轻轻嘲笑，但这一切都很有趣，因为在烹饪比赛中没有真正的失败者(因为每个人都吃得很多)。但是举办一场有趣、轻松的烹饪比赛(或者烘焙比赛，或者其他什么)需要一点计划。

## 选择可重复的食物

这是第一步，也是最明显的一步。允许有很多变化的食物是最好的选择。croquembouche-off 不会有太多的乐趣或意义，因为你必须有一个非常先进的糕点厨师即兴重复这样一个标准化的食谱(一个例外是 [我们的微波方法](https://lifehacker.com/make-every-component-of-this-croquembouche-in-the-micro-1831153734) )。

另一方面，巧克力曲奇允许大量重复。辣椒是另一种烹饪方式，鸡翅或任何种类的烧烤也是。千层面、肉卷、肉丸、通心粉和奶酪，还有蘸酱也是不错的烹饪题材。如果你的朋友不是烹饪型的，你随时可以喝杯马提尼。

## 提供无聊的东西

不要指望你的客人带来任何东西，除了他们将被评判的食物。他们不会带碗来盛辣椒，他们不会带餐巾来盛鸡翅，他们很可能不会带任何餐具。这些都是由你作为东道主来提供的。如果你的餐具抽屉里只有两到四口之家的存货，你可能需要补充一下你的收藏，比如——我最喜欢的餐具、食具以及非常便宜的碗、杯和盘子，如果它们在工作中坏掉了，你也不会难过。

如果你的客人打算带慢炖锅、电暖器或其他任何需要插电的东西，确保你有足够的电源板和延长线，这样每个人都可以把他们的菜保持在合适的温度。

哦，别忘了冰块，尤其是如果你要举办马提尼比赛的话。

## 保持小份量

烹饪比赛的目的就是尝试同一种食物的不同版本。如果你给你的客人大碗或大盘子，他们会把它们装满。这简直是人之常情。用茶碟代替餐盘，用杯子代替碗，或者甚至是你在所有美食节上看到的那些 [小纸船](https://amzn.to/3Plca4j) 。

## 也提供有趣的东西

除非配菜是你的客人对他们的菜肴的特定看法的一部分，否则提供配菜和各种装饰的任务很可能会落到你这个主人身上。对于辣椒，这意味着奶酪、酸奶油和墨西哥辣椒。在烘焙比赛中，这可能意味着一罐冷牛奶，或者一些鲜奶油。对于马提尼酒，你应该准备 [橄榄和柠檬皮](https://lifehacker.com/you-can-have-all-the-martini-garnishes-at-once-1843982095) ，甚至可能还要一些 [腌洋葱](https://lifehacker.com/make-it-personal-with-the-gibson-cocktail-1848359019) 。

## 保守秘密

即使是情感上最成熟的朋友群体也充满偏见，所以在获胜者揭晓之前，最好尽可能保持匿名。取而代之的是给每一道菜分配一个数字。一旦肚子饱了，每个人都会觉得更冷。

## 为人们提供多种“赢”的方式

“最佳”是这样一个主观的指标，所以我喜欢提供额外的指标。除了分发简单明了的记分卡来“评价”每一道参赛作品，还提供空间让人们根据手边的菜肴写出最高级的词，比如“最有创意”、“最辣”、“最能利用巧克力”或“最能喝酒”。(这将有助于确保竞争保持健康，这是我希望有一天能达到的状态。)
