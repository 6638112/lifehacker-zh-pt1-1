# 如何通过偷插枝在植物上省钱

> 原文：<https://lifehacker.com/steal-cuttings-to-save-money-on-plants-1839470340>

如果你想在家里种满漂亮的植物，你可以像买吸管一样买。或者你可以从你在某个地方看到的植物上摘下一两片叶子——一片不会被错过的叶子——然后哄它长成你自己的完整植物。

Watch

我不是说你应该在商店里拔掉植物的叶子。我只是让你知道，如果你这样做了，它可能会工作。你也可以从你朋友的室内植物上剪下一些，或者从邻居的花园或者公园里剪下一些。道德和伦理的含义留给读者作为练习。

总的来说，如果有一种植物让你垂涎，在谷歌上搜索它的名字和“扦插繁殖”，你会发现大量的信息。有些植物这样长不好，但很多植物长得很好。多肉植物特别容易从插条中生长出来； [这里有一个向导](https://www.shopterrain.com/blog_entries/propagating-succulents) 来鼓励他们扎根。 [像薄荷、罗勒、鼠尾草和百里香](https://lifehacker.com/why-you-need-an-herb-garden-1843999388) 这样的香草，如果你把它们的小枝放入水中，它们就会长出根来；这里有一个让这一切发生的指南。有时你需要将被偷走的茎浸入粉末状的生根激素中来鼓励它生根，这种东西你可以在任何花园商店买到。

如果你迷上了用偷来的插枝繁殖植物，我要警告你，这可能会导致盗墓。作者 Tom Christopher 和 Marty Asher [在*的 20 分钟园丁*中写道](https://www.thesmartergardener.com/20-minute-gardeners-are-grave-robbers-too/)最好的玫瑰生长在墓地里。曾几何时，在你母亲的坟墓上种一丛玫瑰是一种传统，所以你可能会有好几种可供选择。只要在非常古老、看起来没有得到很好维护的墓碑附近寻找一朵你喜欢的玫瑰就行了——这样你就会知道玫瑰丛是一种即使被忽视也能很好生长的品种，而且你会减少被来访的爱人当场抓住的机会。
