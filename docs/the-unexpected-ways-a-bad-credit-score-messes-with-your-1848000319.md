# 意想不到的方式一个坏的信用评分搅乱你的生活

> 原文：<https://lifehacker.com/the-unexpected-ways-a-bad-credit-score-messes-with-your-1848000319>

虽然你可能认为信用评分是获得抵押贷款或汽车贷款的成败标准，但它也影响着你生活的其他领域。以下是一些不太为人所知的方式，不良信用评分或不良信用记录会让你损失更多的钱。

Watch

*   租房申请:许多房东和业主会询问你的财务信息，以证明你是一个可靠的租户，可以按时支付租金。他们通常会做信用调查，尽管有些人满足于收入证明或以前与另一个房东的租金支付。虽然没有公认的标准来拒绝你的公寓申请，但一项研究表明，分数低于 600 分的会让你的租房申请面临风险。
*   **汽车保险费率:**在几乎所有的州，汽车保险公司都可以用你的信用评分来决定你的保险费率。 [据 Value Penguin](https://www.valuepenguin.com/how-does-your-credit-score-affect-auto-insurance-rates)报道，对于同一类型的保险来说，信用评分差的保险费用几乎是信用评分好的保险费用的两倍，平均总额为 2940 美元。
*   公用事业:煤气、电力和自来水公司在让你注册他们的服务之前，会检查你的信用。在极少数情况下，他们可能会因为你的信用评分低而拒绝为你提供服务，但他们更有可能坚持要你交押金，或者要你写一封同意为你买单的信。如果你受到服务关闭的威胁，你确实可以通过你的 [州](https://www.usa.gov/state-consumer) 或 [地方](https://www.usa.gov/local-governments) 消费者保护办公室做出一些选择。
*   无线电话计划:和公用事业一样，如果电话公司不喜欢你的信用记录，他们可以要求你交押金，或者让你参加预付费计划。
*   有时(但不总是)雇主会做背景调查，包括查看你的信用记录摘要(尽管他们看不到你的实际信用评分)。如果你的工作中需要处理金钱，这种情况会更常见，但也有可能他们会因为你的信用记录而拒绝你。如果他们这样做，他们 [必须通知申请人](https://www.topresume.com/career-advice/can-employers-reject-job-candidates-because-of-a-credit-check#:~:text=Employers%20must%20notify%20candidates%20before,process%20and%20time%20to%20respond.&text=The%20law%20also%20gives%20applicants,copy%20of%20their%20credit%20report.) 并允许他们对该决定提出上诉(因为你的信用记录可以包括可修复的错误)。

一个糟糕的信用评分可能会很昂贵，但好消息是，你信用记录中的大多数污点会在几年后从你的记录中消失。有了良好的信用行为，你的信用评分会提高。关于提高分数的更多信息，请查看 Lifehacker 的帖子 。
