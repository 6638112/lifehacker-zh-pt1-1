# 如何在谷歌地图中使用分屏街景

> 原文：<https://lifehacker.com/how-to-use-split-screen-street-view-in-google-maps-1846173746>

谷歌地图应用程序现在在 Android 上有一个分屏街景选项。打开它可以让你同时使用街景和地图，这使得寻找和导航到一个目的地——或 [虚拟探索周围地区](https://lifehacker.com/explore-neighborhoods-with-google-maps-new-community-fe-1845791688)——比单独依靠地图或街景容易得多。

谷歌地图 10.59.1 版或更高版本的所有用户都可以使用分屏街景。您可以下载应用程序，并通过查看新的更新。

### 如何在谷歌地图中激活分屏街景

1.  在您的 Android 设备上打开谷歌地图，然后搜索您的目的地。
2.  轻触查看模式菜单(它是带有堆叠方块的圆形按钮)，然后选择**“街景”**
3.  轻按地图上的任意位置以缩小至街道视图。在纵向模式下，它将在屏幕的上部打开，在横向模式下，它将在屏幕的左侧打开。

在分割视图中，您可以点击并拖动街道视图窗口或地图上的来移动该区域。任何动作都会影响两个视图。您可以点击街景窗口右下角的**“展开”**按钮，将其拉伸至全屏并隐藏地图。再次轻按该按钮可返回分屏模式。

当你找到目的地并准备好出发时，还有其他一些最新功能可以帮助你安全到达目的地。对于那些蹄子 it 的人来说，app 可以用它的 [AR 步行导航](https://lifehacker.com/how-to-access-google-maps-new-ar-walking-directions-in-1837132289) 告诉你要去哪里。你也可以尝试一下 [详细的语音指导选项](https://lifehacker.com/how-to-use-google-maps-new-detailed-walking-directions-1838976407) ，这样你走路的时候就不用盯着手机了——这对有视觉障碍的用户也很有帮助。如果你在开车，确保你把你最喜欢的音乐应用 [连接到谷歌地图](https://lifehacker.com/how-to-connect-your-favorite-streaming-services-to-goog-1844053932) ，这样你就可以用你的声音用控制你的音乐，而不用切换应用。
