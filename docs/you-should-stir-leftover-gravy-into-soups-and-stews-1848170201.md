# 你应该把剩下的肉汤搅拌到汤和炖菜中

> 原文：<https://lifehacker.com/you-should-stir-leftover-gravy-into-soups-and-stews-1848170201>

现在，你所有的火鸡肉汁都已经被吃完、冷冻或扔掉了，但在我家，把肉汁浇在烤肉、土豆和烤土豆上是一整个冬天的事情。我在感恩节前、感恩节时和感恩节后做肉汁，也就是说一个装满肉汁的冰箱。



这不是一件坏事，因为这意味着我总是离肉汁薯条很近，但我总是在寻找新的和令人兴奋的方法来使用肉汁。我特别强调支持和鼓励大量的创新、大量的创造和大量的聪明才智。就在几天前，当我最喜欢的一位读者给我发来以下内容时，这样的创新出现在了我的 Twitter DMs 中:

> 如果你做了太多肉汁，在你自制的汤里加入一勺凝固的肉汁。

我知道你们中的很多人想要评论说你“从来没有听说过太多肉汁”，但是请不要开那个无聊的玩笑。如果有帮助的话，把它更名为“额外收入”,但是试着*关注*,欣赏上面收入的辉煌。

正如你可能知道的，肉汁是由肉汤和面糊组成的，这使得它美味而浓厚——对于需要一点鲜味和浓郁的汤或炖菜来说是完美的补充。它在冰箱外凝结得越多，在你的汤里就越好。(这种果冻般的稠度表明胶原蛋白的存在，胶原蛋白是肉汤、原汤和汤的令人满意的口感的来源。)

这个小技巧显然非常容易执行。取一大勺剩余的或“额外的”肉汁，搅拌到沸腾的汤锅或炖锅中。品尝它，如果需要的话重复，直到它在你的嘴里尝起来感觉很好，或者直到你没有肉汁了。那就多做点肉汁。
