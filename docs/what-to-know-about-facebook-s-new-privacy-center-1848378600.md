# 关于脸书的新“隐私中心”你知道些什么

脸书 [引入了新的隐私中心](https://about.fb.com/news/2022/01/introducing-privacy-center/) ，让用户对平台如何收集和使用他们的数据有更多的控制权。Meta的前同名公司长期以来一直是投诉和关注其隐私和数据政策的对象。现在，用户对脸书可以访问哪些数据有了更多的发言权。

## 为什么脸书提供新的隐私功能

脸书 [收集你的大量数据](https://www.nytimes.com/2018/04/11/technology/facebook-privacy-hearings.html) ，即使你不在脸书。数百个网站、应用程序和服务已经在你使用它们时收集你的信息，而这些网站中的任何一个都与脸书合作，如果你的账户与你在脸书的个人资料相关联，就会向该平台发送某些类型的信息。脸书将这些信息用于一些不同的目的，包括广告和平台研究。

在许多情况下，广告商使用脸书收集的数据来个性化平台上和网络上的广告，根据你的偏好和浏览历史定制广告内容。新的隐私中心包括非脸书数据源的设置，旨在为用户提供更多关于其账户安全以及他们的信息如何被脸书及其广告合作伙伴使用的信息。

在全球范围内，业务安全、适度和数据隐私政策正受到越来越多的审查。例如，在德国，立法者已经考虑对未能删除非法内容的公司进行罚款，并帮助起草了欧盟的一般数据保护要求(GDPR)和新的德国隐私法案(BDSG-new)，该法案加强了对企业如何获取和使用消费者数据的限制。

像许多其他社交媒体平台一样，脸书正在努力遵守这些法规，并采取积极措施防止用户数据被不当使用。随着用户要求对他们的数据进行更多的控制，以及企业努力遵守新的法规，像这样的工具在网络上可能会变得更加普遍。

## 如何使用脸书隐私中心

脸书目前正在推出隐私中心到它的桌面网站；如果你有这个功能，你可以通过**设置&隐私**菜单找到它。从该菜单中，您可以选择隐私中心五个模块中的一个，以查看所有可用的工具和资源。这些模块包括:

*   **安全:**主要部分，您可以在其中配置大多数可用的安全选项。该模块包括双因素身份验证(2FA)设置。
*   **分享:**你可以控制哪些人可以看到你的内容，并限制某些人的访问。
*   **采集:**可以管理现场和非现场数据采集的部分。
*   **使用:**包含脸书母公司Meta如何以及为什么使用所收集数据的信息。您可以在这里控制和限制Meta对数据的使用。
*   **Ads:** 包含与广告相关的数据收集信息的部分。在此模块中，您可以管理广告个性化并控制脸书向其第三方广告合作伙伴提供的数据。

其中每个部分都提供了有关脸书如何收集和使用您的数据的宝贵信息，以及有关控制、限制或更改您授予脸书的权限的相关选项。
