# 我永远迷恋的是公共游泳池

> 原文：<https://jezebel.com/my-forever-crush-will-always-be-the-public-swimming-poo-1847380847>

我喜欢游泳池，但由于拥有一个带游泳池的房子对我来说只是一个幻想，所以公共游泳池就是答案。我在成长过程中没有游泳池，所以作为一个孩子，我经常去镇上的游泳池，当我把身体抬出水面进行成人游泳时，我会和其他孩子一起抱怨，在漫长的一小时结束后，我们被允许回来时，我会高兴地尖叫。镇游泳池的小吃店，工作人员都是棕褐色的青少年，让我尝到了用微波炉加热，用纸碗盛出的蘸酱(请只吃棍子)和卡夫通心粉和奶酪。我住的镇上的游泳池象征性地收取了一些费用，它是从喷水器或滑梯升级而来的，非常受欢迎。就像童年的大多数事情一样，我并没有完全欣赏游泳池，但它坚定了我对游泳池的贪得无厌的渴望，并激发了我的夏季追求，即尽可能利用一切机会以尽可能低的价格进入游泳池。

当我搬到布鲁克林时，麦卡伦公园的游泳池是空的，作为我没听过的乐队的户外音乐会的场地。但在 2012 年，游泳池再次向公众开放。从我的公寓步行就能到达一个大游泳池，多好啊。作为纽约所有公共游泳池的标准，你必须把手机留在更衣室，只能带一条沙滩巾、一本书、防晒霜和一瓶水，从而迫使你在那里的这段时间几乎什么都不做。

沉入浅水区，靠墙站好，我通常一次浏览一本书，偶尔停下来做一个缓慢而笨拙的蛙泳，然后像世界上最笨拙的 [美人鱼](https://jezebel.com/i-became-a-mermaid-to-prove-i-know-how-to-swim-1796104096) 一样在湖底游泳。聚集在泳池边的青少年和儿童。公共游泳池有一种令人愉快的社区感，因为它真的让人感到包容。进入门槛如此之低:你所需要的只是一件泳衣、一把密码锁，以及在盛夏正午烈日下排队所需的勇气。在这个城市所有值得等待的事物中，游泳池排在首位。
