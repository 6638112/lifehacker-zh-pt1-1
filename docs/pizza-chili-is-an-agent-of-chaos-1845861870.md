# 比萨饼辣椒是混乱的代理人

> 原文:[https://life hacker . com/pizza-Chili-is-a-agent-of-chaos-1845861870](https://lifehacker.com/pizza-chili-is-an-agent-of-chaos-1845861870)

你们中的一些人可能记得，我开始了 [这个辣椒之旅](https://lifehacker.com/c/skillet-chili-cook-off) 来寻找一个适合我的辣椒食谱——或者至少是一种风格。我在成长过程中并没有强烈的辣椒特征，所以品尝那些有辣椒特征的人的食谱似乎是形成对这道高度地方菜的看法的最佳方式。我吃过德州风味的全肉辣椒、豆角辣椒(和鱼露)和华丽的大块胸肉辣椒，但没有什么能让我对披萨辣椒有所准备。

Watch

乍一看，有人可能会认为这是一个食谱巨魔——一种旨在激怒意大利人和得克萨斯人的烹饪狗屎帖子，为那些对菠萝大喊大叫的人提供了愤怒的材料。但这种辣椒是童子军辣椒烹饪比赛的一部分，这意味着它来自婴儿的大脑，因此是纯净的，没有恶意。

然而，这并不意味着它不充满混乱。首先，我们有牛肉和意大利香肠的组合，这不是那么野生，但我们也有豆类(一种已知的德州触发器)，其中一种豆类是黄油豆，又名*利马豆*。我深深地爱着利马豆，但即使是我也从未想过要把它们放在辣椒里，这让我深深地敬佩。(我怀疑名字中的“黄油”是指我们的“比萨饼”的黄油“外壳”，但也许我对它的理解太多了。)

这份食谱还需要整片的意大利香肠片，这感觉很别扭，但长时间的慢炖会使它们变软、卷曲，并折叠成适合勺子的大小和形状。

最后，还有菠萝，如果我对自己和大家都诚实的话，这就是我选择这个食谱的全部原因。我喜欢披萨上的菠萝，我喜欢被人吼，但主要是好奇想看看味道如何。我们一会儿会谈到这一点，但我要先放下菜谱(另一边见)。
