# 破旧:购买可持续服装的挑战

> 原文：<https://gizmodo.com/worn-out-the-challenge-of-shopping-for-sustainable-clo-1846717253>

地球日很快就要到了，这意味着我的收件箱里充满了来自服装品牌的促销邮件，宣称他们正在变得“可持续”或制造“环保”产品。但事实证明，这些标签并没有多大意义。

时尚营销中常见的其他环保术语也是如此，比如“绿色”、“自然”、“清洁”或“道德”。这是因为这些术语缺乏具体的法律定义，这意味着品牌不必满足特定的标准才能使用它们。

随着公众对气候危机、污染和其他生态问题的关注 [与日俱增](https://www.pewresearch.org/science/2020/06/23/two-thirds-of-americans-think-government-should-do-more-on-climate/)——尤其是千禧一代 对品牌垂涎三尺的——我们更多的人在寻找不会让这些问题变得更糟的购物方式。一份 [报告](https://www.forbes.com/sites/andriacheng/2019/10/17/more-consumers-want-sustainable-fashion-but-are-brands-delivering-it/?sh=182bf55934a5) 发现，2016 年至 2019 年间，“可持续时尚”的网络搜索量增长了两倍。公司想要客户；顾客想要更多环保的选择。难怪如此多的品牌在标签上贴上这些模糊的术语，试图吸引人们。

一些公司正在努力做得更好。但对许多其他人来说，只有黏糊糊的营销词。许多品牌在包装上印上“可持续”一词，却没有做出任何真正的改变。对此有一个词来形容:洗绿。

令人恼火的是，绿洗是另一个没有法律定义的术语。加拿大西部大学艾维商学院的可持续发展助理教授雷恩·蒙格玛利说，这种事情你一看就知道。在一份 2020 年关于这一现象的报告中，她和一位同事还试图明确这一术语的工作定义，并得出了以下结论:“误导人们对一个组织的环境实践或产品形成过于积极的信念的沟通。”

绿色洗涤是一种普遍的 T2，它会让那些试图购买不破坏地球的衣服的购物者感到非常困难。

“我们已经看到数据[显示]这些术语真的让人们困惑，”她说。

不要误解我的意思:有些绿色术语经常出现在包装和标签上，它们实际上有明确的含义。一个例子是“有机”，指的是农业部向符合特定标准的品牌发放的认证。你可以争论这些联邦标准是否足够强大，但是要声称某样东西是“有机的”，品牌必须证明这一点。

也有组织致力于评估品牌的环境影响。非营利的立场。例如，地球创造了 [肮脏时尚记分卡](https://www.stand.earth/publication/markets-vs-climate/fashions-coal-pollution/filthy-fashion-climate-scorecard) 来评定服装公司的气候承诺。令人失望的是——它发现几乎没有一个符合巴黎气候协议的标准——但对于那些难以选择去哪里买衣服的人来说，这是一个很好的资源。

然而，最终，我们个人的选择不会清理时尚产业，时尚产业导致了气候危机，制造了成堆的垃圾，破坏了自然生态系统。我们必须为整个时装业争取更严格的标准，并推动我们当选的官员迫使公司清理他们的行为。然后，我们就不需要寻找绿色标签，因为所有的衣服都是真正可持续的。
