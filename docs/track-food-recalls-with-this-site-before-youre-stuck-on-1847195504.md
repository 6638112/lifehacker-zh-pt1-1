# 在你再次被困在马桶上之前，通过这个网站追踪食品召回

> 原文：<https://lifehacker.com/track-food-recalls-with-this-site-before-youre-stuck-on-1847195504>

当你或你的宠物喜欢的食物(或最近消费的)被召回时，可能会有点不安。当然，并不是每一次召回都有可能让你跑去厕所——任何 召回的严重程度因产品和原因而异。有时候只是一种 [膳食补充剂](https://www.fda.gov/safety/recalls-market-withdrawals-safety-alerts/bea-lydeckers-naturals-inc-issues-allergy-alert-undeclared-soy-lecithin-living-free-dietary)没有声明含有大豆卵磷脂——但其他时候就是 [被沙门氏菌污染的冻虾](https://www.fda.gov/safety/recalls-market-withdrawals-safety-alerts/avanti-frozen-foods-recalls-frozen-cooked-shrimp-because-possible-health-risk) 。

Watch

食品和产品召回发生得比你可能意识到的还要频繁，你第一次听说可能会影响你的事情不需要出现在晚间新闻上——或者当你已经对你感到不安恶心的时候。相反，食品和药物管理局( FDA) 有一份详细的召回产品清单，这些产品由监管 — 并带有方便的搜索功能——你可以在自己的时间和仔细阅读，让自己安心。

以下是如何利用 [FDA 的产品召回网网站](https://www.fda.gov/safety/recalls-market-withdrawals-safety-alerts) 为你带来优势。

## 食品被召回意味着什么？

根据 [州食品安全](https://www.statefoodsafety.com/Resources/Resources/food-recalls-what-you-should-know-and-do)食品召回是“供应商或制造商主动采取的措施，移除贴错标签或有潜在或明显危害的产品。”这些由 FDA 监管，以加强公司的责任和消费者的安全。公司和仓库要进行定期检查，以发现任何不当行为或有害环境和成分。这并不意味着每一次召回都是生死攸关的，但是知道什么可能会对你造成潜在的伤害还是有好处的。

## 如何使用 FDA 的食品召回网站

美国食品和药物管理局的食品召回网站 目前收录了 162 页的食品召回，如果你只是想点击其中的亮点，这看起来会让人不知所措。当然，你可以向下滚动列表并点击浏览 *所有* 页面，但是如果你知道如何缩小结果范围的话，还有搜索功能可用。召回名单按召回日期、食品类型、品牌和公司名称分类。每件商品都有产品说明和详细的召回说明。网站顶部的搜索工具栏也可以帮助你个性化你的 T21 搜索。如果您想搜索某些产品类型，您可以使用下拉菜单搜索食品和饮料、动物和兽医、膳食补充剂等。

还有一个下拉选项用于*终止*召回。当 FDA 在其网站上确定“已根据召回策略采取所有合理措施移除或纠正违规产品，并根据危害程度进行了适当处理，”时，就会发布召回。最后一栏将详细说明您正在查看的召回是否已经终止或仍然有效。

## 当食品被召回时该怎么做

仅仅因为一种食品被召回并不一定意味着它会对你造成伤害。例如，一款洗手液产品因为看起来像水瓶而被召回，并被要求更换包装；它不会伤害你，除非你喝了它。

要了解有关个别召回的更多信息，请单击品牌名称展开详细信息。该描述解释了召回的具体情况，以及该公司正在采取哪些措施来纠正这一问题。一次食品召回可能与一种特定的过敏原有关美国食品和药物管理局警告对该产品有特殊过敏反应的人。确保您彻底阅读了详细信息，并遵循任何关于丢弃该物品或采取必要预防措施的说明。
