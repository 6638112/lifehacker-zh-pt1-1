# 硬景观和软景观的区别，以及如何在你的花园中使用两者

> 原文：<https://lifehacker.com/the-difference-between-hardscape-and-softscape-and-how-1847134604>

虽然我们可能会想到植物，树木，花卉和灌木方面的景观，这只是故事的一部分。即使你的花园被设计成野花丛生的草地，你也有可能有一些无生命的组成部分，比如岩石或栅栏。这是“硬景观”，植物是“软景观”以下是关于这两类景观美化需要了解的内容。

### 硬景观与软景观

简而言之，这两种类型的景观就是你所认为的那样。正如丽莎·哈雷特·泰勒在 [为云杉](https://www.thespruce.com/learn-difference-between-hardscape-and-softscape-2736691) 写的一篇文章中解释的那样:

> Hardscape 是你院子里的硬东西:混凝土、砖块和石头。软景是柔软的、生长的东西，像宿根花卉、灌木、多肉植物和树木。软景是生活；hardscape 不是。理想情况下，一个设计良好的景观融合了这两种元素之间的平衡。

### 硬景观如何影响软景观

你花园里的植物可能是你的主要关注点，因为它们需要定期维护，但是硬景可以在它们成功生长中发挥重要作用。事实上，正如凯特·瑞格夫在《T4》杂志的一篇文章中对 [所解释的那样，首先是艰难的前景:](https://www.thespruce.com/learn-difference-between-hardscape-and-softscape-2736691)

> 通常，硬景观元素必须在软景观元素可以放入之前安装、构建或完成。这是因为硬质景观材料通常与地面融为一体，并经常从物理上改变景观——无论是为游泳池清除污垢，还是为了安装挡土墙或梯田而切割和填充土壤。

此外，考虑硬景观中使用的材料也很重要，因为它们对环境有不同的影响。 [每泰勒](https://www.thespruce.com/learn-difference-between-hardscape-and-softscape-2736691) :

> 硬质景观材料对环境有不同的影响。硬质路面可以防止水渗入土壤，从而增加径流，径流会将污染物带入河流。多孔材料允许水渗入土壤。硬景是花园的一个好处，因为它减少了潜在的侵蚀，保持地面完整。

最终，这个想法是让你的硬景观和软景观都融入当地环境，模仿一些可能在自然界中发现的东西， [Reggev 解释说](https://www.dwell.com/article/difference-between-hardscape-and-softscape-f1085fe9) 。这意味着选择当地的植物，在特定的气候下生长良好。
