# 我们把碳水化合物都算错了

> 原文：<https://lifehacker.com/were-counting-carbs-all-wrong-1847652381>

对于我们中的许多人来说，我们的营养方法包括计算碳水化合物，无论是遵循酮或低碳水化合物饮食，还是瞄准每天特定比例的宏。我们也倾向于根据标准将食物分为“好的碳水化合物”和“坏的碳水化合物”，比如它们是否是快速消化的碳水化合物或者它们是否有更多的纤维。

然而，正如两位研究人员最近在*营养进展*发表的 [观点](https://academic.oup.com/advances/article-abstract/12/4/1108/6267226?redirectedFrom=fulltext) 中指出的那样，我们对“好碳水化合物”和“坏碳水化合物”的强调可能会导致我们放弃一些对健康有益的食物。明尼苏达大学食品科学研究生丽贝卡·舒尔茨(Rebekah Schulz)告诉《非常健康》( verywell fit )( T14 )( T15)说:“虽然这些方面可能是拼图中的个别部分，但它们并不能反映碳水化合物质量的全貌。”

## 给食物贴上“好”和“坏”的标签会漏掉很多食物

将食物分为“好”和“坏”碳水化合物是一种简单的区分，缺乏建立健康饮食模式的细微差别。虽然这一区别听起来微不足道，但关键的区别是将“好”和“坏”碳水化合物分类导致人们避免某些水果、蔬菜和全谷物，这些被视为含有“太多”碳水化合物，而事实上这些食物提供了很多好处。

这不仅仅是一个抽象的概念。根据疾病预防控制中心发布的一份报告，只有十分之一的美国人 摄入了每日推荐量的水果和蔬菜。众所周知，富含水果和蔬菜的饮食可以降低心脏病、肥胖、二型糖尿病和某些癌症等慢性疾病的风险，这表明将一些水果和蔬菜视为富含“坏碳水化合物”可能不是考虑营养的最佳方式。(这包括西瓜，太多的人认为西瓜含糖量太高而不予考虑。没有什么比没有一碗切碎的西瓜，撒上一点酸橙和芝麻酱更让人难过的了。)

例如，如果你正在遵循一种" ["如果它符合你的宏](https://lifehacker.com/how-to-use-iifym-to-calculate-your-macros-instead-of-ob-1818524072) " "类型的饮食，并且你只剩下有限数量的碳水化合物，你可能最终会吃一种具有正确宏的高度加工的能量棒，而相反，你可能会更好地吃一片水果或一片全麦吐司，即使宏并不完全是你想要的比例。(是的，这个*是*给低碳水化合物饮食的人的建议，加入一些全谷物和水果可能是个好主意，即使这会打乱你的宏指令。)

## 对于碳水化合物，质量优先于数量

相反，舒尔茨和她的合著者，明尼苏达大学营养和食品科学教授乔安妮·斯拉文建议关注碳水化合物的质量而不是数量。当谈到碳水化合物的质量时，他们建议看看一些因素，如全谷物的百分比，是否有添加糖，纤维的总量，以及总碳水化合物与纤维和添加糖的量的比例。

如果这些因素可以用一句话来概括，那就是:“多吃未经加工的食物。”这是我们都知道在理论上对我们有好处的东西，但是当我们每天做很多很多食物选择时，很容易只关注碳水化合物的数量，而不是质量，让水果和蔬菜半途而废。
