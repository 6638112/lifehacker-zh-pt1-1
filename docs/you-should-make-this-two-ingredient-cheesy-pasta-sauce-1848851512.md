# 你应该做这种两种成分的奶酪意大利面酱

> 原文：<https://lifehacker.com/you-should-make-this-two-ingredient-cheesy-pasta-sauce-1848851512>

我已经老了，还记得晒干的西红柿第一次过时的时候。它们无处不在:在美味的糕点中，烤鸡肉意大利面菜肴中，捣碎成奶酪酱，然后— 突然— 它们不见了，笨拙，不受欢迎。



但是我一直很喜欢他们。它们香甜扑鼻，充满鲜味，有种 90 年代中期的优雅，就像肮脏的马提尼。它们很好吃，我厌倦了人们假装不好吃。

至少还有一个人似乎有这种感觉。艾玛·拉佩鲁克不仅将干苹果作为她的奶油干酪意大利面食谱 的一部分，她还将它们放在了前面和中间。她的意大利面酱正好由两种成分组成——奶油奶酪和晒干的西红柿——老实说？ [它操](http://logy-of-this-guy-fucks-1848059211) 。

这似乎太简单了，就像做一盒卡夫奶酪通心粉一样简单。切碎一些晒干的西红柿，把它们捣成奶油奶酪，然后用加盐的意大利面水稀释，再把混合物和你选择的意大利面形状一起搅拌。Laperruque 还在番茄酱中油炸了一些刺山柑，做成松脆的浇头，如果我没有刺山柑，我会这样做的(悲哀！).

酱是奶油和美味的，感谢那些风味包装的晒干的西红柿。它本身很好，但它是你选择的蛋白质的绝佳基础(搭配烤鲑鱼或鸡肉，营造真正的 90 年代氛围)，或任何烤蔬菜。我做了两次，第二次我加了更多的西红柿干，因为我太喜欢它们了。90 年代回来了，宝贝，这个意大利面酱证明了这一点。

## 晒干的番茄意大利面酱(稍微改编自 Food52)

**配料:**

*   你可以选择 8 盎司意大利面
*   4 盎司全脂奶油奶酪
*   3 盎司晒干的西红柿
*   盐

将一壶浓盐水煮沸。把晒干的西红柿切碎，然后把它们捣成糊状，放在一边。将意大利面加入沸水中，煮至有嚼劲，保留一杯含淀粉的盐水。

向奶油奶酪混合物中加入 1/4 的水，搅拌至稀。将意大利面沥干，与调味汁一起搅拌，如果需要的话，加入更多的水使它变稀，记住它冷却后会变稠一点。立即与大量新鲜罗勒、原始食谱中的脆皮马槟榔或你最喜欢的蛋白质和/或蔬菜一起食用。
