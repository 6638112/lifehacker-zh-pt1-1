# what is deadnaming and how to avoid it 1845802589

> 原文：<https://lifehacker.com/what-is-deadnaming-and-how-to-avoid-it-1845802589>

本周早些时候， [艾利奥特·佩吉在推特上发布了一份声明，介绍自己是变性人](https://twitter.com/TheElliotPage/status/1333820783655837701) 。在同一句话里，他告诉我们他的名字和代词(他/他们)。然而，几分钟之内，社交媒体和传统新闻媒体上的许多人使用他的死名和前代词爆出并讨论了这条新闻。

尽管在过去的几年里，跨性别者在媒体和娱乐中变得越来越明显，但对佩奇声明的反应提醒我们，作为一个整体，我们还有很多工作要做。虽然有些反应是彻头彻尾的仇恨(不值得在这里讨论)，但其他一些反应似乎是善意的，但在关于跨性别身份和社区的对话中被误导了。



此外，本周许多人第一次了解到“死亡命名”这个术语和概念。虽然这只是这篇演讲中许多微妙方面中的一个，但它是需要更广泛理解的一个方面。以下是死亡命名的含义，为什么它是有害的，以及如何在与跨性别者讨论和说话时保持敏感、支持和尊重。

### 什么是 deadnaming？

在我们进入定义之前，这是一个提及跨性别记者协会(TJA)的工作的好时机，该协会不仅在工作场所和职业生涯中支持跨性别记者，还就如何确保跨性别群体的报道准确和敏感向媒体提供指导。但即使你不是记者，他们的资源——特别是 [他们的风格指南](https://transjournalists.org/style-guide/)——提供了清晰、直白的语言和术语信息。词汇表和文件的其余部分都是跨性别记者写的，这一事实恰好与第一课相吻合:倾听跨性别者。

好吧，回到死亡命名。下面是部分[TJA 的定义](https://transjournalists.org/style-guide/) :



> 跨性别者不再使用的名或前名，通常也称为“名”或“法定名”…虽然 deadname 通常是一个名词，但它也用作动词来指代对跨性别者使用错误名字的行为。

TJA 通知记者和其他媒体成员，“永远没有理由在一篇报道中公布某人的姓名”[强调我们的]——这是本周重复发生的与佩奇有关的事情。

### 为什么死名有害？

死亡命名不仅仅是简单地叫错一个人的名字。它也会对跨性别者的幸福产生重大影响。“我们选择适合自己的名字。通常，我们的法定名字，deadname，以前的名字对跨性别者来说并不合适，记者奥利弗-阿什克莱 ，，[，](https://transjournalists.org/) 告诉 Lifehacker。“它剥夺了我们的自主权。它剥夺了我们的自决权，并经常损害我们的性别和身份。真的很不确定，而且会相当苦恼。”



毫不奇怪，直呼姓名会对跨性别者的心理健康产生重大影响——而使用他们合适的名字会产生相反的效果。例如，一项被广泛引用的 [2018 年研究](https://www.jahonline.org/article/S1054-139X(18)30085-5/pdf) 在《青少年健康杂志》上发表发现，使用跨性别者选择的名字与抑郁和自杀意念的风险较低有关。

这是斯坦福大学医学院儿童和青少年精神病学研究员杰克·图尔班(Jack Turban，医学博士 )在他自己的研究中看到的，该研究专注于跨性别青年的心理健康。他告诉 Lifehacker:“对于许多跨性别和性别多样化的人来说，尤其是我接触过的孩子，使用过去的这个名字会带来痛苦的情绪和需要隐藏真实自我的记忆。”

此外，Turban 说，当有人在与跨性别者交谈时使用错误的名字或代词时，可能会让他们想起过去不接受他们是跨性别者的人——比如在学校欺负他们的孩子，甚至可能是把他们赶出家门的父母。“他们可能会认为这个用错名字或代词的人会同样拒绝或攻击他们，”他解释道。“使用一个人的(选择的)名字和代词是一种简单的尊重标志，也是一种表达你接受一个人的方式。”抛开所有这些数据和专家意见，我们也没有必要去解释为什么你不应该指名道姓。你应该用一个人选择的名字，因为这是他们选择的名字——这是你最起码可以对他们表示尊重的方式。如果有帮助的话，可以这样想:如果一个朋友结婚了，改变了他们的姓，你不会固执地坚持用你更熟悉的那个。



### 如何在与跨性别者交谈或谈论他们时给予支持和尊重

在这一点上，不言而喻，你不应该直呼某人的名字。但事实远不止如此。这里有一些方法可以在你所有的谈话中给予支持和尊重。

#### 称呼某人的名字

当和跨性别者交谈或谈论他们时，用他们现在的名字称呼他们。如果你不知道他们合适的名字是什么，问问他们。

#### 进行对话(在适当的时候)

如果你身边的人以变性人的身份出现，而你不确定他们的名字和/或代词，克莱恩建议除了询问你如何支持他们之外，还应该和他们谈谈。他们解释道:“这是一次你可以和你的朋友进行的对话，以确保你使用了他们觉得最合适的代词。”。

如果有人告诉你他们使用了多个代词，例如“他/他们”，而你不明白这实际上是什么意思，也不知道如何正确使用，克莱恩再次表示，找出答案的最佳方法是询问这个人。“如果是你关心和亲近的人，很多时候我们会很乐意谈论这个话题，因为我们真的很感谢你考虑到这一点，并希望尊重我们，”他们解释道。此外，请记住，这可能是一个正在进行的对话，因为正如克莱恩指出的那样，某人的代词可能会随着时间的推移而变化。



#### 如果你搞砸了，纠正自己，继续前进

通常在这种情况下，我们的本能反应是极度抱歉，但克莱恩建议不要这样。“不要老是说，‘哦，真对不起。’尽量不要让你对此的情绪成为跨性别者的责任。“纠正自己，继续前进，”他们解释道。这真的是你能做的最好的事情，只要致力于把它做好。"

#### 练习(用你自己的时间)

如果你发现自己很难记住使用跨性别者的名字或代词，克莱恩建议花时间和精力去改变这一点。“有时这意味着在他们不在的时候练习，只是熟悉新的代词，以确保当你在你关心的人身边时，你能正确使用它，”他们说。这会是什么样子？也许只是在想着那个人的脸时重复这个名字，并在他们的代词旁边大声说出来。
