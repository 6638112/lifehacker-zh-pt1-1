## 如何在变焦中启用聚焦模式

像许多其他变焦功能一样，对焦模式首先需要通过变焦网站在主机帐户上启用— 但根据帐户的类型，步骤会有所不同。对于i 个人帐户，o 打开 [Zoom门户网站](http://www.zoom.us) 、并使用您的帐户登录。然后，在“个人资料”页面点击“**设置**按钮，在“**会议**”选项卡中，查找“**聚焦模式**”功能。单击它旁边的开关以启用该功能。

然而，如果你是一个大型Zoom账户的经理，在这个账户中有多个用户(例如，如果你是一名学校管理员)，请遵循以下步骤。

1.  登录进入[缩放账号](http://www.zoom.us) 。
2.  从“**管理**部分，点击“**账户管理**，进入“**账户设置**选项。
3.  进入“**会议**”选项卡，在“会议中的【T2(高级)】”部分下，启用“**聚焦模式**”功能。
4.  如果您看到一个验证弹出窗口，点击**启用**按钮进行确认。如果您看不到该选项，这意味着您没有启用该功能的帐户权限，您必须与帐户管理员联系。

## 如何在变焦中使用对焦模式

既然焦点模式功能已启用，请检查您是否正在运行版本5.7.3或更高版本的，并且作为主持人或共同主持人开始您的会议。C 点击会议工具栏中的三点“**更多**按钮，选择“**开始聚焦模式**”选项。

点击“**开始**按钮确认(您可以使用“**不要再问我**选项跳过此确认步骤)，聚焦模式现在处于会话中。您会在相机视图的左上角看到一个小图标(一个带点的圆圈),表示对焦模式已启用。

完成对焦模式后，点击三点“ **More** ”按钮，选择“**停止对焦模式**选项。
