# 如何识别网上出售的假冒名牌工具

> 原文：<https://lifehacker.com/how-to-spot-fake-brand-name-tools-sold-online-1848122667>

无论你是在市场上为自己购买工具，还是作为礼物送给别人，你都可能会留意销售和特别优惠——尤其是在接下来的一周左右。但不幸的是，并非所有网上出售的工具都是他们所说的那样。



根据汤姆·斯卡利西在《BobVila.com 的一篇文章中所说，你应该知道一种正在发生的假冒工具行骗——不仅仅是在假期，而是在一年中的其他时间。以下是如何识别假货的方法。

## 假工具球拍是如何工作的

它通常从脸书或谷歌上的定向广告开始，提供像 DeWalt 和 Milwaukee 这样的大牌工具的令人难以置信的交易。我们说的是非常大的折扣， [斯卡利西说的是](https://www.bobvila.com/articles/fake-tools/)——就像广告上宣传的商品零售价的四分之一甚至八分之一。

但在几乎所有类似的情况下，这些工具都来自冒充 DeWalt 和 Milwaukee 商店的假冒在线商店。虽然他们出售的工具可能看起来和原版一样，但它们的质量标准不同，使用起来可能不安全。斯卡利西说，或者在某些情况下，整个假代销店就是一个骗局，顾客根本收不到任何工具(因为根本就没有)。

## 如何识别假名牌工具

下面是 [斯卡利西说的网购工具要注意](https://www.bobvila.com/articles/fake-tools/) :

*   零售商的名称。如果你不是直接从工具品牌自己的网站上购买工具，确保网上商店是该品牌的授权零售商——即使他们有像“密尔沃基工具经销店”或“德瓦尔特工具”这样听起来合法的名字
*   查看工具品牌的网站获取信息。这些假冒的工具零售商在变得如此普遍，以至于一些公司 [像 DeWalt](https://www.dewalt.com/support/shop-safe) 在他们的网站上开设了帮助消费者确定他们购买的工具是否是真货的栏目。
*   熟悉工具的零售价格。在你开始购物之前，先看看这些工具的实际价格，以及打折时通常的折扣。因此，如果一个品牌在一年中最好的销售期间提供工具 20%的折扣，而你看到它们在一个所谓的“商店”以 70%的价格出售，这很可能是一个骗局。
*   查看评论。当你浏览可能是假冒的 tool outlet 网站时，寻找产品评论。显然，如果每个人都留下评论说这是一个假网站和一个骗局，请避免它。但是一个没有评论的网站或产品也是一个危险信号。
