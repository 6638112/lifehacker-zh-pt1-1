# 你应该讨好的事情(但很可能不是)

> 原文:[https://life hacker . com/things-you-should-be-but-possible-parent-1846674471](https://lifehacker.com/things-you-should-be-buttering-but-probably-arent-1846674471)

我家总是备有 [至少两种不同的](https://skillet.lifehacker.com/you-need-two-types-of-butter-in-your-kitchen-1837837122) 黄油:我有炒菜和烘焙用的烹饪黄油，其中大多是廉价商店品牌，还有一种单独食用的黄油，通常是Kerrygold，用于涂抹。

而两种黄油都是 [总是加盐](https://skillet.lifehacker.com/salted-butter-has-always-been-the-secret-to-better-cook-1822776571) ，t 他吃的黄油是放在室温下以便更容易涂上，哦，我怎么涂。我把它涂在面包上(很明显)，但最近我扩展了我的曲目，包括三种有点不寻常，但美味的食物，我认为你应该开始涂类似的东西。

### 土豆c 臀部

将黄油涂在土豆上是一件很平常的事情，就像将薯片蘸到牛奶里一样。那么，为什么我们不把薯片蘸上黄油呢？

如果要我猜的话，我会说这与清教徒的价值观有关，我会鼓励你避开这些。事实是涂了黄油的薯片味道很好。奶油脂肪总是与咸脆的东西搭配得很好，这种特殊的组合是彻头彻尾的颓废，但不会丰富到你吃不下一整串。(如果你之前觉得不能吃“就一个”，那就等着加黄油吧。)

你不需要很多黄油——一点点就够了——也不要对薯片的种类过于挑剔。黄油在任何东西上都很美味，从原味奶酪到味道浓郁的锅煮薯片。

### 格雷厄姆c 架

我不知道这是否是我古怪味觉特有的现象，但是全麦饼干和黄油的组合尝起来有淡淡的草莓味道。也许是因为这种饼干上浓郁的奶油让我想起了奶酪蛋糕，但它绝对读起来像“甜点”(一种简单、两种成分、不需要烘烤、甚至一个蹒跚学步的孩子都能准备的甜点)。它香甜、松脆、浓郁、滑腻，你还想再吃一口吗？

### 煮熟的鸡蛋

是的，我们以前写过关于 [涂了黄油的硬蛋](https://skillet.lifehacker.com/i-am-begging-you-to-butter-your-hard-boiled-eggs-1841156203) 的文章，但我打算再提一次，以防你们中的一些人没有利用它的魔力。就像之前的两个推荐，一个涂了黄油的煮鸡蛋尝起来比它任何权利都要复杂和完整。牛顿将其归功于黄油能够改变最普通的食物:

> 经过一番思考(讨厌的人会说太多)，我认为神奇之处在于butter升级方便早餐的能力。煮鸡蛋过了一段时间后开始觉得很难吃，尤其是如果你在大多数早上吃的话——但是即使是最小的黄油块也会完全改变。突然，我喜欢的富含蛋白质的早餐变成了一种享受；一些我非常期待的事情。

基本上，黄油让一切都变得更好，但以上三种食物是证明它有多么强大的绝佳例子。(它在saltines和matzoh上也非常好，但你可能已经知道了。)
