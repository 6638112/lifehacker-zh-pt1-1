# 如何设置 Instagram 新的防骚扰工具

> 原文：<https://lifehacker.com/how-to-set-up-instagrams-new-anti-harassment-tools-1846733461>

Instagram 增加了两个新的反骚扰工具，旨在减少你可能在平台上收到的辱骂信息。这里有一个快速查看如何设置他们一旦到达。



### 屏蔽一个人在 Instagram 上使用的所有账户

第一个功能很重要，因为它让你屏蔽所有属于一个人*的账户和创建的新账户。怪人从多个一次性账户发送有害信息，在一个账户被屏蔽时创建新账户，或者使用虚假档案跟踪他人，这些都很常见。这一功能极大地限制了一个人继续拖拽、拖拽、拖拽的能力，但你必须等待一段时间才能解锁，因为它将在未来几周内在全球推出。*

当它是实时的，只需打开 Instagram 应用程序，前往你想要阻止的人的个人资料。点击右上角的三点图标，选择**阻止、**，然后选择**阻止此人及其可能创建的任何新账户**

### 阻止滥用的直接消息请求

Instagram 还在该应用的消息请求收件箱中添加了一个新的过滤器，可以自动删除带有攻击性词语或表情符号的消息。过滤后的邮件被移动到一个新的隐藏请求文件夹中，如果您不想阅读来自随机人的讨厌和/或恶心的邮件，可以安全地忽略它们。发件人也不会知道他们已经被过滤了。

Instagram 表示，新过滤器将在未来几周内在“几个国家”推出，并在接下来的几个月里扩展到更多地方。

一旦它可用，你可以在**设置>隐私&安全>隐藏文字**下的新“隐藏文字”菜单中打开它。点击**“隐藏信息请求”**启用收件箱过滤器。该菜单还包括今年早些时候发布的评论过滤器 Instagram 的开关。

默认情况下，消息请求和评论过滤器使用 Instagram 和反欺凌组织创建的被阻止的单词列表，但您可以点击**“添加到列表”**下的“管理自定义单词列表”来自定义过滤器。
