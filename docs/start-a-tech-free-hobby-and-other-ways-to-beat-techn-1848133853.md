# 开始一个“无科技”的爱好，以及其他战胜科技疲劳的方法

你可能已经注意到我们都被科技包围着。我们用它来工作，娱乐，和我们的朋友和家人保持联系。它能感觉到，从你醒来的那一刻到你上床的那一刻，你被拴在你的许多小玩意上。如果你对技术的不断使用感到不知所措，你并不孤单。我们甚至给它起了个名字:技术疲劳。

什么是技术疲劳？生活在一个永远沉浸在新技术中的世界里，我们该如何应对？

## 什么是技术疲劳？

当你发现自己不断被技术包围，以至于变得不堪重负和筋疲力尽时，技术疲劳就发生了。大多数人都需要一定的技术——和享受拥有它——，但是在不断使用之后，屏幕有一种方式让我们精神疲惫不断要求我们的注意力。挑战在于学会如何最大限度地利用我们所有的技术，而不让它压倒我们。

## 休息一下

很容易陷入不断连接的陷阱。互联网使得在全球范围内交朋友变得很容易，这意味着通知和消息经常在一天中的任何时候大量涌入。与其让自己不知所措，不如经常脱离技术。一天一到两次，把你的手机静音，关上笔记本，做些别的事情，任何别的事情。你甚至可以设置几个每日提醒(是的，在你的手机上)来提醒你走开，即使每次只有10或15分钟。

## 锻炼

除非你戴着VR头戴设备锻炼，或者与其他Peloton用户一起骑自行车参加虚拟比赛，否则沉浸在技术中意味着我们要花很多时间坐在椅子上或桌子后面。这种久坐不动的生活方式会导致许多不同的伤害或健康问题，从像腕管一样的重复性压力伤害，到不良姿势导致的背部问题。有时候，我们能做的防止技术疲劳的最好的事情就是在附近散步、跑步或骑自行车。这对你的整体健康有好处*而且*它迫使你休一次科技假。

## 尝试正念

互联网总是在那里，随时准备分享另一个悲伤、紧张或负面的新闻故事。如果你发现你的技术正在增加你的焦虑和压力， [练习正念和冥想可以帮助](https://lifehacker.com/how-to-get-the-most-out-of-meditation-1843713254) 。

如果你不确定从哪里开始，像Calm、Headspace、 [和一系列其他选项](https://lifehacker.com/7-meditation-apps-that-are-cheaper-and-better-than-he-1847455111) 这样的应用程序提供了指导冥想，可以带你完成基本步骤。像Spotify这样的音乐服务也有各种不同主题的冥想指南。选择一条道路，然后开始。

## 开始一个没有科技含量的爱好

有时候，我们需要一些让我们娱乐和参与的东西，而不需要我们启动电脑或花时间盯着屏幕。培养一个与科技无关的爱好，比如编织、 [做拼图](https://lifehacker.com/jigsaw-puzzles-can-improve-your-quality-of-life-more-th-1847667513) 、或 [学习如何弹吉他](https://lifehacker.com/how-to-play-guitar-without-learning-how-to-play-guitar-1848128835) 。选择一些让你的大脑活动起来，让你的手忙碌起来的事情——和让你远离手机或电脑。
