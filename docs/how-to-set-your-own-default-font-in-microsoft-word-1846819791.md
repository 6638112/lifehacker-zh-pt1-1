现在是时候选择您希望Microsoft Word在您打开新文档时使用的默认字体了。虽然你可能很想选择像 Wingdings这样滑稽的东西，但它会很快过时。(不过，把这个想法收藏起来，以后再用，因为这对戏弄不知情的朋友或同事来说很棒。)

一旦你通过点击你选择的字体名称做出选择，确保你点击左下角的按钮“设为默认”。然后会询问您是要将您的选择设置为当前文档的默认选择(否)还是“基于Normal.dotm模板”的每个文档的默认选择(是)。

如果你在一台工作电脑上，这个选择不适用，很可能你的公司禁止你编辑这个模板。正如微软 [指出的](https://support.microsoft.com/en-us/office/change-the-default-font-in-word-2f50603b-9886-47e4-a483-48ac5ab80163) ，你可以通过开始菜单快速搜索“Normal.dotm”来检查是否是这种情况。右键点击结果并打开文件位置，然后右键点击文件并选择**属性**。确保在文件的**常规**标签中**“只读”**没有被选中，如果是则取消选中。

之后，单击安全选项卡，确保您(或您的帐户所在的组)拥有该文件的写权限。如果没有，你可能会陷入困境，除非控制你办公室电脑的人愿意并且有能力帮助你。
