# 止汗剂和除臭剂的区别(以及如何知道使用哪一种)

> 原文：<https://lifehacker.com/the-difference-between-antiperspirant-and-deodorant-an-1847397101>

每个人都有使用(或不使用)除臭剂或止汗剂的理由——无论是为了避免出汗、气味，还是屈服于社会规范。但是如果你是选择滑行或喷洒这些产品的人，你知道它们之间的区别吗？



当然，它们的名字多少透露了一点:除臭剂处理气味，止汗剂处理汗液。但是故事远不止于此。这里有一些关于除臭剂和止汗剂之间区别的知识，以及如何知道使用哪一种(如果你选择使用一种的话)。

## 止汗剂和除臭剂有什么区别？

止汗剂和除臭剂的主要区别在于成分。止汗剂含有铝基化合物，可以堵塞你的汗腺，并保持腋下干燥， [【陈亚颦】](https://www.riverchasedermatology.com/providers/lucy-l-chen-md-faad/) 医生，Riverchase 皮肤科 [的皮肤科医生告诉我们良好的家务管理](https://www.goodhousekeeping.com/beauty/anti-aging/a36608350/deodorant-vs-antiperspirant/) 。在某些情况下，止汗剂也可以减少制造汗味的细菌种类，但这比该产品的目的更有好处。

陈解释说，另一方面，传统除臭剂的配方是为了增加皮肤的酸性并杀死皮肤上的细菌——更具体地说，是汗液中的细菌赋予了它刺鼻的香味。而且，正如 [好管家研究所](https://www.goodhousekeeping.com/institute/about-the-institute/a19748212/good-housekeeping-institute-product-reviews/) 美容实验室主任 [Birnur Aral](https://www.goodhousekeeping.com/author/12432/birnur-aral-ph-d/) 博士所说，也有一些除臭剂含有香味，可以掩盖气味，但不会杀死细菌。无论哪种方式，除臭剂本身不会阻止你出汗。

## 应该用止汗剂还是除臭剂？

根据陈的说法， [和](https://www.goodhousekeeping.com/beauty/anti-aging/a36608350/deodorant-vs-antiperspirant/)都可以安全使用，所以很大程度上要看你想让它们做什么。如果你不是一个多汗的人，或者不介意流汗，你应该用除臭剂。如果你的汗水没有强烈的气味，但你想避免污渍，止汗剂是一个选择。当然，有很多产品是止汗剂和除臭剂的组合，两全其美。
