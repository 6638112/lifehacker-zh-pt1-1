# 为你的下一个视频使用这个免费的网络提词器

> 原文：<https://lifehacker.com/use-this-free-web-based-teleprompter-for-your-next-vide-1846624023>

如果你想制作一个视频，不喜欢即兴演讲，有一个免费的 [提词工具](https://telepromptermirror.com/telepromptersoftware.htm) 可以帮助你——你所要做的就是在你的浏览器中打开他们的网站，复制并粘贴你想在屏幕上滚动的任何脚本(或笔记)。您可以使用滑块来调整滚动速度和文本大小——您肯定希望同时使用这两种方式，以便获得最适合您说话节奏的完美组合。

Watch

当你的文字滚动时，你不能调整任何东西，但是你可以使用一些有用的快捷键来调整你的预提示设置。例如，通过敲击键盘上的左右箭头来调整滚动速度，或者使用“向上”和“向下”来放大或缩小字体。当您的文字滚动时，您可以通过按住空格键来停止和开始，或者通过按下 Escape 来倒回开头(并停止)。

免费的提词器可以让你以纯文本或富文本格式复制和粘贴你的单词，你可以调整你的虚拟提词器的完美外观的边距和对齐方式。如果您需要镜像文本以适应您正在使用的任何设置，您甚至可以水平或垂直翻转整个屏幕。

最后，也是最重要的一点，您可以在桌面上保存和打开脚本。这不会保存您的设置，只是保存文字，但是如果您不想再一次复制和粘贴，这是保存或导入脚本的一种便捷方式。

虽然提词器网站也提供了语音激活功能，这大概可以让文本随时滚动，但我无法让它在 Edge Chromium 上工作，但它在标准 Chrome 上工作得很好。你必须准确地阅读屏幕上的内容才能触发滚动，但如果你需要在你的演讲中加入更多的表演而不是依赖于固定的滚动速度，这是很有用的。
