# 如何试驾谷歌Chrome新的“回忆”页面

> 原文:[https://life hacker . com/how-to-test-drive-Google-chromes-new-memories-page-1846716365](https://lifehacker.com/how-to-test-drive-google-chromes-new-memories-page-1846716365)

谷歌正在为谷歌浏览器测试一项新的“记忆”功能，该功能可以将你的浏览历史、书签和其他网络活动收集到一个可搜索的页面中，这将帮助你更容易地挖掘你的浏览器活动。

Watch

与Chrome当前的浏览器历史页面不同，它按时间顺序列出你的活动，Chrome的Memories将你的活动分成一个基于卡片的菜单。每张卡片都会根据以前的搜索、经常访问的网站、最新的书签、当前打开的标签等提供建议。如果你想查找特定的东西，还有一个搜索栏，以及你最近活动的快照。使用记忆搜索栏将调整各种建议卡，以更好地匹配您正在查找的内容。

谷歌尚未表示Memories是否会取代Chrome当前的历史菜单(甚至是书签管理器、新标签快捷方式、下载列表等其他部分)，但与无休止地滚动浏览你的浏览器活动相比，它已经形成了一种更直观、更个性化的体验。

Chrome的Memories页面还没有对普通用户开放，但你可以在最新桌面版的Chrome Canary 中先睹为快。它隐藏在一个实验性的旗帜后面，但它似乎可以在Mac，Windows，Linux和Chrome OS上工作。Android也将提供内存，但试图在Chrome移动应用的金丝雀版本中打开它会导致错误。

同样值得注意的是，从上面的截图可以看出，如果你使用Chrome的黑暗主题，Memories页面实际上是不可读的。由于该功能仍在测试中，可能会有其他错误和不一致之处，但这些问题应该会在7月份Chrome 92正式推出之前得到解决。

### 如何提前开启桌面Chrome Canary的记忆:

1.  [下载](https://www.google.com/chrome/canary/) ，或更新至，Chrome Canary版本92.0.4479.0(或更高)。注意:不要忘记将Chrome Canary与你的谷歌账户同步——否则记忆页面将会是空白的！
2.  在新的标签页中，进入***chrome://flags/# memories***
3.  从下拉框中选择**“启用”**。
4.  出现提示时重启Chrome。
5.  再次打开Chrome Canary，然后进入 **chrome://memories** 打开memories页面。
