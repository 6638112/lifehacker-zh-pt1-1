# 立即更新Chrome以修补这些安全漏洞

> 原文:[https://life hacker . com/update-chrome-immediately-to-patch-these-security-vulne-1848853708](https://lifehacker.com/update-chrome-immediately-to-patch-these-security-vulne-1848853708)

[不到两周](https://lifehacker.com/why-you-need-to-update-google-chrome-as-soon-as-you-can-1848798753) ，我们已经在谈论另一个关键的Chrome更新了。不过，说实话，谷歌在这些事情上保持领先还是有好处的:有时，我们发现一家公司在关键的安全漏洞上坐了*几个月*，却没有发布新的更新。至少在这里，我们有一个很好的机会来保护我们的Chrome免受潜在黑客的攻击。

Watch

谷歌在4月26日星期二的 [博客文章](https://chromereleases.googleblog.com/2022/04/stable-channel-update-for-desktop_26.html) 中宣布了这一更新。这个新的版本的Chrome是101.0.4951.41，可用于Windows、Mac和Linux。此次更新修补了30个安全漏洞，其中7个被评为“高”其中五个漏洞(加上另外六个评级为“中等”)是“释放后使用”漏洞，由于内存重新分配问题，黑客可以将任意代码传递给程序。

以下是七个“高”漏洞。你可以在 [谷歌的Chrome发布博客](https://chromereleases.googleblog.com/2022/04/stable-channel-update-for-desktop_26.html) 上找到其他23个的完整列表:

*   [CVE-2022-1477](https://crbug.com/1313905) :在Vulkan免费后使用。*2022-04-06由城湾公园报道*
*   [CVE-2022-1478](https://crbug.com/1299261) :在SwiftShader中自由后使用。*2022-02-20由城湾公园报道*
*   [【CVE-2022-1479】](https://crbug.com/1305190):角度自由后使用。*2022-03-10报道*
*   [CVE-2022-1480](https://crbug.com/1307223) :在设备API中免费后使用。*2022-03-17 @ uwu 7586报道*
*   [【CVE-2022-1481】](https://crbug.com/1302949):共享中免费后使用。【T4报道】2022-03-04
*   [CVE-2022-1482](https://crbug.com/1304987) :在WebGL中执行不当。*微软克里斯托夫·迪尔报道2022-03-10*
*   [CVE-2022-1483](https://crbug.com/1314754):web GPU中堆缓冲区溢出。*2022-04-08*Google Project Zero的Mark Brand报道

这些缺陷影响了Chrome *和*基于Chrome的浏览器。值得注意的是，这包括微软的Edge、Brave和Opera。如果您使用Chromium浏览器，您需要尽快更新浏览器，以保护您的活动免受这些漏洞的影响。

## 如何更新谷歌浏览器

你的浏览器可能会自动更新，但由于谷歌仍在推出这一更新，你会想要手动触发它以获得最快的结果。为此，点击浏览器窗口右上角的三个点，然后导航到**帮助>关于谷歌浏览器**。允许Chrome寻找新的更新，然后，当它可用时，点击“重新启动”

当Chrome再次打开时，你将运行最新版本。
