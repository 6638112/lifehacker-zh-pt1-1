# 星空谷后爱上的9款农业游戏

> 原文:[https://kotaku . com/stardew-valley-games-like-farming-sim-cozy-harvestella-1849487513](https://kotaku.com/stardew-valley-games-like-farming-sim-cozy-harvestella-1849487513)

我不想成为一个农民——我是一个城市女孩，或者，在我最脆弱的日子里，成为一个正在成长的郊区妻子——但是关于农业游戏的一些事情实在是太让人放松了。

虽然休闲和舒适的游戏在历史上被抹黑为 [【非真实游戏】](https://kotaku.com/casual-game-players-hardcore-fans-genshin-impact-arknig-1847895923) ，但我觉得像*【动物穿越】*这样的标题可以为人们提供一个低风险的途径，让他们进入他们以前可能没有考虑过的爱好。就我个人而言，我喜欢玩像Nioh 2和Elden Ring这样的游戏，这些游戏让我压力很大，有一天我可能会吐在嘴里，但通常情况下，当一个自称非游戏玩家的人向我询问现实生活中的游戏推荐时，他们会寻找像Stardew Valley一样充满清洁、甜蜜空气的东西。

我认为这真的很好，一些游戏普遍令人愉快。因此，我整理了一份清单，列出了更多的农业游戏 *Stardew Valley* 粉丝们会喜欢的游戏，如果他们想更深地融入游戏，但不知道下一步该去哪里。即使你认为自己像我一样是一个“铁杆”游戏玩家——一个城市女孩，如果你愿意的话——你可能会在这个列表中找到一些东西，激励你坐下来，闭嘴，只是收获。

这款 [仙侠](https://en.wikipedia.org/wiki/Xianxia)-脚踏实地的农场管理sim *永生*是一款大气的游戏，也是一款 *Kotaku* 的最爱——作家Sisi Jiang [今年早些时候写道](https://kotaku.com/immortal-life-harvest-moon-stardew-valley-rune-factory-1848972094) 这款游戏是他们自2009年的*符文工厂3* 以来玩得最好的农场管理sim。

游戏有一些角色扮演的游戏元素，奖励你探索流畅舒缓的2.5D地图，帮助你的社区。姜写道，它的力量在于“它的世界建筑的魅力……我的意思是，看看这些美丽的庄稼。每天早上从床上滚下来，摇晃你的植物，直到细小的蔬菜掉落，这是一种令人难以置信的冥想。”这款游戏仍处于早期访问阶段，这意味着它有“一些粗糙的边缘”，但它也允许你用番茄煎鸡蛋，有时这更重要。

广受欢迎的*符文工厂*系列游戏的2021部分——角色扮演模拟人生于2006年推出，作为大型农场游戏*秋月*的衍生游戏——充满了各种选择。你可以结婚，耕种，享受你的模拟农场和它的节日，或者用剑穿过可怕的地牢。正如许多Steam评论者指出的，你可以享受农业模拟*的宁静和*不同沉浸式RPG活动的刺激。

*特辑*于2021年在Steam 上发布，但它实际上是2012年为任天堂3DS推出的*符文工厂4* 的延伸。正因为如此，你会注意到一些非常2012年的纹理，但如果你正在寻找一个伟大的农业游戏，游戏元素将愉快地让你惊讶(剑！)，你找到了。 

被宣传为“舒适的农耕时代”的 *Moondrop* 是另一款早期游戏，它目前正在Steam 上流行 [。它让你扮演一个蒙着手帕的农场女孩，她必须徒步爬上程序生成的迷雾山脉，以完成谜题并向你破败的村庄运送货物，你希望它恢复昔日的辉煌。](https://store.steampowered.com/app/1417750/Moondrop/)

早期评论表明，与典型的农业模拟自负形成有趣对比的是， *Moondrop* 更倾向于对roguelites的解释，并依赖于元级数。

像*不朽的生命*一样，这款游戏需要一点照顾——根据它的开发者在Steam笔记中的说法， *Moondrop* 将在2023年之前提前进入，并且“目前需要许多玩家(带着他们的想法、错误报告和热情)来使[它]变得神奇。”

这款游戏非常可爱——农场模拟人看起来像是由涂色书上的人物栩栩如生地制作而成，它存在于日本漫画和动画系列*【哆啦a梦* 的梦幻 [静音世界中。在这个*哆啦a梦*的迭代中，玩家享受一个平淡无奇的水彩世界及其一系列的庄稼、鱼和马，并与善良的村民聊天。也有一些胖乎乎的奶牛可以挤奶，但我最喜欢看它们。升级你的农场，在秋雨中跺着脚四处走，在完成一小时左右的入门教程时，不要屏住呼吸。](https://www.youtube.com/watch?v=agg3C3fzFwY)

作为2020年 的早期游戏， [首次推出的湿软的生命模拟游戏——拥有你所期望的所有农业东西。你可以种植弯曲的植物，出售你制作的零食，并参与高贵但经典的农场游戏目标，将你的破败小镇变成迷人的天堂。但它也令人惊讶和愉快地强调了舞蹈。](https://kotaku.com/ooblets-is-all-about-dance-battles-cute-critters-and-1844424564)

很大一部分 *ooblets* 涉及到与Ooblets一起生长和闲逛，Ooblets是一种小生物，有着下颚破碎器大小的眼球，看起来像萝卜，或鱼，或任何其他微小而柔软的东西，你可以像在*神奇宝贝*中一样与它们战斗。除了！你没有参加实战，而是把你的蛋蛋送进了*式的白色小鸡*舞蹈。这是一款不错的休闲游戏，并于9月1日 在Switch、Xbox和PC上发布了最终版本(通过Epic Games store) [。](https://ooblets.com/)

第一人称沙盒游戏 *Slime Rancher* 不是一个传统的农业模拟游戏，但它吸着粘液、遨游太空的农民主角，它的无边缘世界，那里一切看起来都像麻糬，它对随意种植的强调在佳能中开辟了一个舒适的利基。这是一个糖浆般的小游戏，适合那些喜欢游戏体验更多的是共鸣而不是压力的人，以及那些喜欢农业元素，但不想过多考虑管理的人。

正如哈珀·杰伊在他们的2016年 *Kotaku* 评论中所写的那样，“*粘液牧场主*对我来说没有太多需要管理的，我不怕在我的小农场上犯错误。即使我搞砸了，我的小狗们也总是面带微笑。”这是低赌注，这是放松，这是*煤泥牧场主*！

这是对太空农业的一种更具讽刺意味的解释，*没有一个地方像家一样*想象着当大多数人类决定放弃它，收拾行李，前往火星时，我们被毁坏的地球会是什么样子。在这个以生态为中心的模拟游戏中，你扮演一个女孩，她试图在被毁坏的地球上找到她失踪的爷爷，在路上，她必须整理、回收和管理一个农场。你可以在这个游戏里做蛋黄酱，这是我喜欢的——不管人类多么濒临崩溃，我总会有时间做蛋黄酱。

积木角色扮演游戏*龙之任务建造者2* 非常棒，杰森·施莱尔在2019年 *Kotaku* 帖子 的 [中写道。它有一点可笑的自负——可悲的是，它在Steam上的故事描述说，“哈贡的邪恶儿童决心根除所有创造者，并宣布所有事物的建造、烹饪和创造都是非法的。”](https://kotaku.com/dragon-quest-builders-2-is-even-better-than-the-first-1836278922) 

幸运的是，“所有的希望似乎都失去了，直到你，一个年轻的建筑学徒，设法逃离邪恶的魔掌。”对你有好处。

一旦远离邪恶，你可以收获和烹饪美味的自产食物，打击坏人，并通过不同的土地努力恢复希望和创造力。还有*《我的世界》*的职业玩家们，听好了——你们在*龙之旅*中的主岛——觉醒之岛，提供了一个沙盒模式，你可以用最多四个玩家来制作“任何你能想象的东西”“或者，毫无疑问是不可避免的，”施莱尔写道，“一个大阴茎。”

《农场模拟》的粉丝们可能想把11月4日发售的启示农场和角色扮演游戏 *Harvestella* ，但现在可以在Steam 上预售 [，放在他们心中的预期日历上。](https://store.steampowered.com/app/1816300/HARVESTELLA/)

我从来没有真正想在玩游戏之前对游戏抱太大希望，但对于这款看似故事重重、画面精美、拥有神奇生物的农业游戏，我很难不感到兴奋。该游戏的Steam描述指出，其世界受到Quietus的困扰(不要与我们世界的[*Quietus*](https://thequietus.com/)，一个非常好的音乐网站)，一个“随着季节变化而到来的死亡季节”。植物枯萎。由于空气中弥漫着死亡的灰尘，人们甚至不能在外面走动，“但土地足够肥沃，可以生产作家伊桑·加奇所说的“角质产品”，所以情况不会太糟。
