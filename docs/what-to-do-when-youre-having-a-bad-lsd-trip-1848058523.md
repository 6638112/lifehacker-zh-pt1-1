# 当你经历了一次糟糕的迷幻之旅时该怎么办

> 原文：<https://lifehacker.com/what-to-do-when-youre-having-a-bad-lsd-trip-1848058523>

假设一下，我知道安然度过被简单称为“糟糕之旅”的偏执、恐惧和改变的现实是多么令人难以承受。当我们谈论假设时，假设我不建议在雨天在大城市“滴”(又名摄入)酸，尤其是当你倾向于说服自己你室友的仓鼠正密谋谋杀你的时候。(你现在可以笑了，但我保证你会觉得那只仓鼠好像要吸血了。 ) (假设地。 )



当谈到杂草时，我们有一个标志性的视频指南 [如果你太高了该怎么办](https://www.youtube.com/watch?v=_mUvG6x53VM) 。被迷幻药绊倒是另一个完全不同的事情，虽然，一个有着广泛的不可预测的影响的事情。正如 [康复村](https://www.therecoveryvillage.com/teen-addiction/faq/what-is-a-bad-trip/) 所解释的，糟糕的旅行是高度个体化的，并且可能与同一个人先前糟糕的旅行症状不同。一些常见的不良行程症状包括:

*   一种时间静止不动的感觉
*   对你周围事件的消极想法或解释
*   极度偏执
*   极端和突然的情绪波动
*   可能包括感觉、声音或景象的消极幻觉

服用迷幻药的任何人都可能经历一次糟糕的旅行，虽然不可能停止已经开始的旅行，但有一些提示和技巧可以帮助你应对，直到你平安到达另一边。

受 Reddit 帖子 的启发，我们编制了一份在糟糕的旅行中使用的虚拟资源列表，在该帖子中，一名用户有效地充当了互联网上一名心情不好的陌生人的“旅行保姆”(又名，确保其他吸毒人员的安全和健康)。有时屏幕和电子设备在糟糕的旅行中会产生可怕的影响，但如果你能够与它们互动，以下资源可以帮助你度过艰难的吸毒经历。

## 当你服用迷幻药过量时，最好的方法是

### r/LSD

[LSD subreddit](https://www.reddit.com/r/LSD/) 自诩为“致力于麦角酰二乙胺-25 的善良、开明的社区”或者，就像 u/ [Barziboy](https://www.reddit.com/user/Barziboy/) 描述的那样，“Reddit 自己的随机天花板目录&地板图片，配有真正发人深省的标题，比如:‘就是这样！’或者“现在一切都有意义了。”“随意发帖讲述自己的旅行，寻求帮助，就像 [这个例子](https://www.reddit.com/r/HumansBeingBros/comments/qrda8m/this_guy_helping_someone_having_a_bad_trip_in_rlsd/) 中的用户。互联网上的陌生人总是有成为怪人的风险，但在大多数情况下，这个 subreddit 充满了放松和积极的能量。

### r/tripsit

[trips it](https://www.reddit.com/r/tripsit/)subred dit 甚至比 r/LSD 更适合帮助任何旅途不愉快的人。这些帖子被描述为一个减少伤害的社区，充满了平静的建议和普遍的“良好氛围”,有助于引导你经历一次势不可挡的毒品体验。

## Tripsit.me

[Tripsit.me](https://tripsit.me/) 也是一个减少伤害的资源，它提供易于消化的信息和事实说明，让你与现实保持联系。该网站是专门设计来缓解由致幻剂引起的焦虑发作。虽然我不能证明那些在线的人是有益的，但看起来这个网站上的实时聊天确实是全天候活跃的。

## 不调和

像 tripsit.me、 [这种 D iscord 频道](https://discord.gg/9ckj5Cxp) 是一个活跃的服务器，在这里你可以与志同道合的人就你的吸毒经历进行聊天。和任何在线聊天一样，我不能保证屏幕另一边的陌生人的有效性或善意。不过，参与(或只是观察)网上聊天还是有帮助的，尤其是如果你需要的是有人知道你在经历什么的证据。

## **呼叫 62-炉边**

有时候，在你的旅途中，屏幕会让你不知所措。拨打 62-FIRESIDE (623-473-7433)联系训练有素的志愿者，他们可以帮助你度过与裸盖菇素、 [、迷幻药](https://www.rollingstone.com/t/lsd/) 和其他的挑战性遭遇。根据《滚石》 中的一个特写，炉边“旨在促进一种全面的方法来准备、享受和整合迷幻体验。”该热线也支持清醒的人谁是旅行坐他们的朋友，或清醒的人谁想要处理他们过去的经验。

## **度过糟糕旅行的一般技巧**

不管你是否能从网上他人的帮助中找到安慰，记住这些应对你的旅途的一般提示:

*   去一个安全、熟悉的环境(远离过度刺激)
*   播放平静积极的音乐
*   试着冥想
*   喝水
*   不要试图用其他药物进行自我治疗
*   记住它不是真实的，也不会永远持续下去
*   屈服于这种体验(而不是试图控制它)
*   尽快入睡，并且知道你会在一个更好的地方醒来

要知道你很快就会回到现实。它被称为“旅行”是有原因的，它不会永远持续下去。你能行的。
