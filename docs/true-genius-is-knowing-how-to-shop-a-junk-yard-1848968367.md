# 真正的天才是知道如何逛旧货市场

> 原文：<https://lifehacker.com/true-genius-is-knowing-how-to-shop-a-junk-yard-1848968367>

在我大学毕业后的几年里，一次悲惨的 T-boning 事件把我送到了一个垃圾场，为我花一美元从一个朋友那里买的名为 Laverne 的 1978 年天蓝色雪佛兰 Nova 寻找一个替换门。通过我对 Laverne 的爱，一辆甚至在碰撞前就有严重缺陷的车，我了解到一辆 Nova 或类似的车(例如别克云雀)的门可能适合她，但垃圾场除了还有更多教训。



太吓人了。垃圾场很大，由知道曼斯菲尔德酒吧是什么的人经营和居住。就这样，我开始了关于垃圾场的教育，垃圾场不仅仅是成堆的汽车垃圾。垃圾场做了大量工作来回收和处理有毒液体，如防冻剂和旧机油，它们为需要保持旧车运行的人(或有严重汽车爱好的人)提供了负担得起的替换零件来源——在购买二手车需要大量现金的时候，这是至关重要的。但是在你跑到你当地的废品收购站去买零件之前，你需要有一个进攻计划，因为废品收购站的经历是竞争激烈的，复杂的，而且容易搞砸。以下是如何最大限度地利用你的垃圾场之旅。

## 在参观垃圾场之前提前做好计划

如果你正在寻找一个特定的部分(而不是像一个人那样简单地花一个下午四处游荡寻找有趣的东西)，第一步是启动电脑，查看垃圾场的网站。如今，大多数垃圾场都有相当先进的互联网存在——你可以(也应该)进行搜索。你对这个零件和它将要进入的汽车了解得越多越好。如果您在数据库中找不到您的零件，您可以设置一个提醒，以便在零件到货时与您联系。

接下来，找出你需要什么工具。虽然废品收购站有时会为你做脏活，但通常你需要自己拆零件。一路开车到垃圾场，找到生锈的废铁，其中包含你只需要意识到你没有带正确的扳手的部分，这将严重阻碍你的计划和你的骄傲。此外，如果你要找一个特定的零件，了解所需的工具会减少你需要随身携带的重型工具的数量。

一些你在垃圾场不需要的东西:

*   杰克。废车场几乎总是将车辆停放在架子上，如果你需要拆卸发动机或变速器，通常会提供起重机。
*   **手推车。**拖运一台发动机或几个非常重的零件需要运输工具，但大多数废品收购站会提供手推车或类似工具。
*   **火把或研磨机。**垃圾场完全浸泡在易燃物品中，因此任何产生火焰或火花的东西都是不允许的。如果你需要剪掉什么，院子会帮你剪的。

最后，研究你的车。许多汽车和卡车共用传动系统部件和其他零件——这都是汽车制造商利用的规模效率的一部分。许多垃圾场网站将提供这一功能；如果你当地的垃圾场没有，尝试一个不同地区的垃圾场——例如，www.pull-a-part.com 有一个非常强大的零件互换数据库，可以给你提供很多非常棒的信息。或者， [AutoZone](https://www.autozone.com/) 允许您搜索特定的零件以寻找您的品牌和型号，并向您显示一系列合适的零件，然后您可以将这些零件插入到您的垃圾场的数据库中。

## 请求帮助

正如我所说，垃圾场可能令人生畏，因为那里的每个人都生活和呼吸着汽车和卡车零件。但这是一项资产。垃圾场的经营者通过出售人体器官赚钱，所以他们通常非常乐意为你提供专业知识和建议。因此，如果你似乎找不到你要找的东西，或者无论你付出多少努力都无法将某个零件移除，不要害怕寻求帮助。如果您在他们的在线数据库中找不到您想要的零件，打电话几乎总是值得的——许多零件，尤其是老型号的零件，销售速度非常快，打电话有时会带来幸运的发现。

一些垃圾场比其他垃圾场提供更多的服务。有些人会帮你拆卸零件，有些人甚至会在你的车上帮你安装(T2)零件，如果你能把车开到那里的话。如果那是你感兴趣的事情，提前发现是值得的。

对于任何对汽车感兴趣或急需廉价维修的人来说，垃圾场都是难以置信的机会。只需要一点点计划和努力，就能从你的游览中获得最大的收获——但是回报可能是巨大的。
