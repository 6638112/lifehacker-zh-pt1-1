# 立即更新Safari

> 原文:[https://lifehacker.com/update-safari-immediately-1848426551](https://lifehacker.com/update-safari-immediately-1848426551)

如果你使用iPhone、iPad或Mac，你应该尽快更新你的Safari浏览器。今天，苹果发布了三款最新系统——iOS 15.3、iPadOS 15.3和macOS Monterey 12.2的最新软件版本——如果你通常会花时间安装新的更新，你可能会想优先考虑这个版本，因为它修复了一个主要的安全漏洞。

## Safari怎么了？

一段时间以来，Safari 一直存在严重的安全漏洞，这要归功于WebKit的IndexedDB实现错误。这个漏洞允许网站从其他打开的窗口读取你的数据，除了你的谷歌ID和其他谷歌信息。通常，网站不应该访问与自己的域名不匹配的数据库，然而，这个bug打破了安全墙。

这里有一个例子来说明为什么这很糟糕: I 如果你的银行使用这个IndexedDB JavaScript API，一个恶意的网站将能够利用正确的技术查看这些信息。这同样适用于任何使用该API的网站。这只是隐私和安全的定时炸弹。但是，尽管苹果花了很长时间来修复这个问题，补丁终于来了。

## 如何更新以修复此Safari安全漏洞

若要确保Safari已更新并打了补丁，您需要在iPhone、iPad和Mac上安装最新版本的iOS、iPadOS和macOS。只需进入设置>通用>软件更新(iOS和iPadOS)，或系统偏好设置>软件更新(macOS)，然后按照屏幕上的说明下载并安装最新的更新。

## Safari并不是这里唯一的安全更新

虽然Safari是目前为止尽快更新的最大原因，但它不是iOS 15.3、iPadOS 15.3和macOS 12.2给你的设备带来的唯一修复。Apple为ColorSync、Crash Reporter、iCloud等提供了安全更新。你可以在这里 阅读苹果iOS 15.3和iPadOS 15.3 [的完整安全说明(该公司目前还没有macOS 12.2的安全说明)。](https://support.apple.com/en-us/HT213053)
