# 用分期付款的方法来更新你的家

> 原文：<https://lifehacker.com/refresh-your-home-by-using-staging-techniques-1846117591>

如果你已经经历了出售你的房子的过程，当潜在的买家来看房子的时候，很有可能会进行内部装修。staging 背后的想法是消除你的个性、风格或身份的任何痕迹，让感兴趣的人更容易想象他们生活在那里。所以有很多白色、米色和浅灰色。



有些人如果决定出售，他们从来不需要展示任何东西，因为他们喜欢简朴、极简的装饰风格。但是对于我们这些住在很容易被误认为是维多利亚古玩店 的 [室内的人来说——或者只是喜欢颜色和图案(这可能有意也可能无意地冲突)——上演是一个相当大的转变。](https://imgc.artprintimages.com/img/print/interior-charles-dickens-old-curiosity-shop-london_u-l-ppy0hg0.jpg?h=900&p=0&w=900)

但是，如果我们告诉你，这两类人(以及那些处于中间位置的人)可以利用一些共同的分期技术来快速更新家居，同时仍然保持他们的个人风格，会怎么样呢？嗯，这基本上就是文章的主旨了。下面是几个如何做到这一点的例子，来自莎拉·杨和凯蒂·霍尔德菲尔为《真正简单的》写的一篇文章。

### 深度清洁

在向潜在买家展示你的房子之前，你肯定想先对它进行深度清洁(如果你负担得起，你可以自己动手，或者雇别人来做)。你应该为自己住在一个干净的家里，而不是因为你想把它卖给别人。

### 买新窗帘

在某些时候，厚重的窗帘非常流行，但它可以让这个地方感觉像是 20 世纪 90 年代中期假日酒店的一个房间。如果你在布置你的家，你会想要装上新的、有微风的(可能是白色的)窗帘，让房间看起来更明亮、更大。你也可以为自己这么做。

### 重组书架

利用书架的每一寸空间来塞尽可能多的书是很诱人的。虽然这种方式可以用于存储，但可能不是最美观的。因此，如果这是你的目标，你可能要重组。下面是克里斯蒂娜·萨拉威的 [十一二十一设计](http://www.11211design.com/) 告诉 [真实简单](https://www.realsimple.com/home-organizing/home-improvement/home-staging-tips) :

> “把 25%的书全部拿走，然后重新排列剩下的，这样一些书是垂直的，一些书是水平堆放的。在书柜内放置一些小饰品，作为点缀和书挡。这会给房间增添个性和特色，同时也能点亮你的书柜。”

### 更换地毯

扔地毯是伟大的，但他们可以看看一段时间后磨损有点糟糕。它们将是舞台场景中最先出现的东西，所以升级到一张新地毯来犒劳自己吧。

### 挂一面镜子

当布置一个自然光很少的房间时，像萨拉威 这样的专家建议挂一面镜子来照亮一点。此外，如果你不喜欢墙壁装饰，但仍然想要一些 T4 风格的东西，镜子是一个不错的选择。
