# ExpressVPN使用安全吗？

> 原文:[https://lifehacker.com/is-expressvpn-safe-to-use-1848477227](https://lifehacker.com/is-expressvpn-safe-to-use-1848477227)

流行的VPN服务ExpressVPN最近卖给了Kape Technologies，一家有着不光彩过去的公司，一些用户担心这会破坏VPN的隐私。

Kape将自己定位为一家网络安全和数字隐私公司，但该公司以前被称为Crossrider，并因分发数据跟踪广告软件而臭名昭著。Kape的创始人之一是一名被判犯有内幕交易罪的亿万富翁和一名前以色列监视特工。

除了ExpressVPN的新母公司引发的问题，最近还发现ExpressVPN的首席信息官Daniel Gericke因帮助阿联酋对其他政府进行网络攻击而面临非法黑客指控。这引发了人们对ExpressVPN产品安全性的担忧——但ExpressVPN并不是唯一受到审查的VPN。

除了ExpressVPN，Kape还拥有其他三家VPN公司——cyber ghost、Private Internet Access (PIA)和ZenMate。所有四家Kape拥有的VPN服务都是作为独立的公司运营的，虽然没有证据表明这些公司在错误地处理你的数据或从事可疑的活动，但如果你认真对待隐私，也有理由避免它们。

## 隐私和恶意软件问题

首先说一下隐私。在被收购之前，ExpressVPN是一家总部位于英属维尔京群岛的私有公司，与其他国家相比，英属维尔京群岛更有利于隐私保护，因此是VPN公司更理想的管辖区。

虽然该公司仍然主要在英属维尔京群岛经营，但其母公司Kape的总部设在英国。一方面，英国有利于隐私的GDPR法律，但它也是美国、英国、加拿大、澳大利亚和新西兰之间的五只眼睛情报共享联盟的一部分，该联盟通常被认为是“不安全的”，用于严重的匿名和数据隐私。

这意味着Kape理论上可以收集和分享这些VPN保存的任何用户数据——特别是姓名、电子邮件地址和支付信息等账户数据——无论这些公司在哪里正式运营。

## 你应该担心吗？

因此，总结到目前为止:ExpressVPN现在由一家总部位于英国的公司所有，该公司以前以不同的名称制作广告软件，理论上可以与五只眼国家共享用户帐户数据。最重要的是，我们现在知道它的一位高管有着不光彩的过去，这让人们对ExpressVPN的隐私目标产生了质疑。听起来很糟糕。但是有任何切实的证据表明ExpressVPN实际上是在监视其用户，推送广告软件，或者破坏他们的隐私吗？

在这一点上，没有。

一些用户声称ExpressVPN降低了他们电脑的速度，但是这些传闻还没有得到证实，也没有人提出明显变慢的潜在原因。请注意，这并不意味着这些说法是完全错误的，只是它们不应该被视为具体的确凿证据。

事实是，所有四个Kape拥有的VPN都是“无日志”服务——这意味着它们不跟踪浏览器活动或连接日志——所以它们没有任何浏览器数据可以使用。特别是express VPN[例行将其代码提交给第三方审计](https://www.techadvisor.com/news/vpn/should-you-keep-using-expressvpn-3808592/) 以确认其无日志声明，并测试应用程序的恶意软件注入和其他漏洞。到目前为止，所有的审计都没有问题。他们*的确*可以访问你的账户信息，这可能会让一些用户觉得不可接受，但这在几乎所有在线公司中是一种常见的做法。普通用户不应该担心这个，就像你不应该担心把你的信用卡信息交给网飞或亚马逊一样。

然而，尽管所有证据表明ExpressVPN *不是*一个充斥着广告软件、窃取你的数据并将其出售给第三方企业和外国政府的骗局，我们仍然不确定你应该使用express VPN——也不是Cyberghost、PIA或Zen mate——原因很简单:垄断。

虽然Kape拥有的四家VPN公司表面上看起来是“竞争对手”,但所有这些公司最终都为Kape赚钱，并且Kape指导他们的商业策略和隐私政策。Kape拥有的公司越多，这些产品参与竞争的动力就越小——或者制定积极的隐私保护政策。

当然，人们会认为VPN审查人员会指出任何不可信或明显可疑的活动，对吗？嗯，除了收购VPN公司，Kape还收购了VPN评论网站。我们并不是说有证据表明Kape正在干预这些出版物，或者他们受到压力要给Kape的产品一个更好的分数，但任何人都可以看到这里明显的利益冲突。

需要明确的是:ExpressVPN、Cyberghost、PIA和ZenMate对于一般用户来说都是不错的。但是，如果你对数据隐私、在线匿名和整个VPN市场的健康状况很认真，我们建议寻找独立的VPN，这些VPN优先考虑隐私和透明度，而不是支持那些意图瓜分整个市场的大公司，因此抑制了竞争、创新和其拥有的产品中对消费者友好的政策。
