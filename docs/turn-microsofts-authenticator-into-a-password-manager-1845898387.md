# 将微软的验证器变成密码管理器

> 原文:[https://life hacker . com/turn-Microsoft-authenticator-into-a-password-manager-1845898387](https://lifehacker.com/turn-microsofts-authenticator-into-a-password-manager-1845898387)

微软正在其Authenticator应用程序中测试一种新的密码管理功能，这使得前者作为所有帐户相关信息的一站式商店更加有用。它非常容易设置，而且你可以在iOS和Android上同步你的密码，也可以在你的桌面上同步Chrome和Edge。

要开始，抓取(或更新) [**微软认证器**](https://www.microsoft.com/en-us/account/authenticator) app。使用你的Microsoft帐户登录，你现在可以将应用本身用作基于密钥的验证器(或点击允许验证器)。我已经是粉丝了。

从那里，点击左上角的三线“汉堡”图标，并点击**设置**。在**下【Beta】，**启用**自动填充**。

您将被要求再次使用您的Microsoft帐户登录。做到这一点，在提示时启用密码同步，现在你会在验证器应用程序中看到一个全新的密码部分:

然后，您将在此部分看到您之前保存到Microsoft帐户的任何密码，例如，您在使用Edge浏览器时保存的任何密码。此外，你可以使用微软的应用程序设置你正在使用的任何设备来自动填充密码提示。

然而，如果你想扩展这个测试版密码管理器的功能，你需要做更多的工作。显然，你可以在你使用的任何iOS或Android设备上安装这个应用程序(也可以是iPad)。如果你使用的是微软的Edge浏览器(已登录),你保存或修改的任何密码都会同步到云端，并出现在上述设备上。你可以*也可以*下载一个Chrome扩展， [**【自动填充】**](https://chrome.google.com/webstore/detail/microsoft-autofill/fiedbfgcleddlbcmgdigjgdfcggjcion) ，给竞争对手的浏览器带来同样的功能。

虽然自动填充不像那样优雅或安全，但它是免费使用的。您可以查看、删除和编辑存储的密码。如果您从另一个密码管理器进行切换，您也可以从. CSV文件导入保存的密码。仅此而已——基本的功能，但如果你已经融入微软的生态系统，这是一个不错的密码管理器。现在，您可以将它与验证器结合起来，这是一个非常方便的安全包。
