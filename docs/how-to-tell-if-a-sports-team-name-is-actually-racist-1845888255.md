# 如何辨别一个运动队的名字实际上是不是种族歧视

> 原文：<https://lifehacker.com/how-to-tell-if-a-sports-team-name-is-actually-racist-1845888255>

华盛顿足球队过去被称为华盛顿红人队。从 1933 年到今年早些时候，这个现已被丢弃的吉祥物存在了近一个世纪。红皮人的绰号终于寿终正寝了，但这是经过了几十年的批评和土著活动家的运动才实现的。他们的努力最终改变了对话；在改变之前，最近的媒体报道一直呼吁该团队培养轻率的种族主义文化。然而在相当长的一段时间里， [该组织的法人所有权遇到了表面上漠不关心的强烈抗议](https://bleacherreport.com/articles/1634935-dan-snyder-says-he-will-never-change-washington-redskins-controversial-name) 。

战斗继续进行，直到最后发生了变化，也许引发了某种多米诺骨牌效应:2018 年，又是在 [几十年的行动主义](https://www.smithsonianmag.com/smart-news/smithsonian-curator-weighs-cleveland-indians-decision-retire-racist-logo-180968001/) 之后，美国职业棒球大联盟的克利夫兰印第安人 [宣布吉祥物首领瓦胡](https://www.latimes.com/sports/mlb/la-sp-indians-wahoo-20180129-story.html) ， [从其制服和商品中移除了 1948 年首次出现的一幅美国土著](https://news.sportslogos.net/2018/11/19/cleveland-indians-unveil-new-uniform-cap-for-2019/baseball/) 的漫画。两年后，这个团队已经 [宣布打算放弃它的名字，](https://www.nbcnews.com/news/us-news/cleveland-reportedly-change-name-its-baseball-team-n1251070) 也是；2021 赛季后，他们就不再是印第安人了。

最近的公告提出了一个问题:其他团队会是下一个吗？到底是什么使得一个团队需要改名？堪萨斯城酋长队、亚特兰大勇士队，甚至明尼苏达维京人队和达拉斯牛仔队会遭遇同样的命运吗？体育迷们有时认为这些名字没什么大不了的，甚至认为它们实际上是 [对启发他们的](https://globalsportmatters.com/culture/2018/04/24/native-american-mascots-honoring-culture-symbol-disrespect/) 的文化遗产和历史的尊重，但这场辩论实际上是关于更广泛的问题不公正，这种不公正折磨社会的时间比职业体育存在的时间长得多。

### 一些球队延续了种族主义的刻板印象

看过堪萨斯城酋长队的比赛吗？在正常情况下，你可能会发现 76，000 人齐声高喊，一首“ [【战斧劈】](https://www.youtube.com/watch?v=N4P6z_DTHf8)战歌回荡在墙壁上。有些人可能戴着头饰，喝着啤酒，但你很难在看台上找到一个土著人，并愉快地将这种狂热视为对美洲土著文化的大规模庆祝。

活动人士长期以来一直认为，你所看到的不是对美国土著传统的崇拜，而是对一种文化的盗用，以及对试图摧毁这种文化的种族灭绝运动的抹杀。众所周知，酋长们每场比赛的开场都是小跑一匹名为“Warpaint”的马，马背上有一名啦啦队员。然后，仪式鼓在开始前敲响。这些来自真实本土文化的空洞提升令一些土著血统的人感到不快。美国土著记者文森特·席林在二月份接受美国有线电视新闻网 采访时也说了同样的话:“我的祖母甚至不能分享她真实的样子，但他们可以对它做一个刻板印象，并告诉我以此为荣。”

今年早些时候，倡导组织“照明”的创始人兼首席执行官克里斯托·埃克霍克在接受《告诉 NPR》采访时，直言不讳地谴责了华盛顿红人队:

> 这个名字的起源源于谋杀、暴力、种族灭绝和仇恨...这是字典里定义的种族歧视，句号。

是的，红皮人是一个特别突出的例子。正如[Vox](https://www.vox.com/the-highlight/21308236/cleveland-indians-washington-redskins-name-racism-aunt-jemima-dixie-chicks-fair-and-lovely)所指出的，美国印第安人全国代表大会在 2013 年的一份报告中说，“到 20 世纪之交，[红皮人]已经演变成一个在美国文化中贬损和表示劣等和野蛮的术语。”

其他职业体育组织，如国家曲棍球联盟的芝加哥黑鹰队，一直在雷达下飞行，也许是因为他们在描绘土著美国人的刻板印象方面不太令人震惊。然而，这种情况可能开始改变。埃默里大学的市场学教授迈克·刘易斯告诉 NPR:“(亚特兰大)勇士队和黑鹰队不管出于什么原因都没有受到太多的批评。但在这一点上，我认为现在各种各样的球队名称都在发挥作用。”

### 如何知道你的团队名称是否应该更改

一般来说，如果你的团队名字仅仅因为存在而激起传统边缘群体的愤怒，那么是时候改变它了。一些 [与最近从南部](https://www.dailynews.com/2015/07/15/name-changing-symbolism-could-be-a-slippery-slope-dan-walters/) 的学校和城市移除联盟名称和符号的行动相提并论，这场辩论在 2020 年夏天的反种族主义抗议活动中变得尤为明显，但考虑到走到这一步所经历的斗争， [建议其他不太明显的种族主义球队名称可能很快效仿](https://thelandryhat.com/2020/07/09/dallas-cowboys-what-if-change-name/) 似乎有些言过其实。

达拉斯橄榄球队牛仔队的名字 [确实来自历史上一个有问题的时期](https://judithbenhamouhuet.com/where-does-the-myth-of-the-cow-boy-fit-in-a-racist-america-cowboy-the-book/) ，但并不完全代表一个边缘化群体的明显刻板印象；相反，它暗示了一种虚张声势的男子气概和美国英雄主义的形象。是的，明尼苏达维京人因一群掠夺成性的勇士而得名，但是维京人的真实历史并没有融入到美国故事中，因此并不那么令人反感。

也就是说，如果你的团队*因为依赖于边缘化群体的刻板印象而落入前一个阵营，主张改名也没什么错。说到底，你爱的是这项运动，而不是品牌和商品——尤其是如果一支球队的吉祥物深深植根于他人的痛苦和苦难之中。*
