# 这是在母亲节为妈妈提供床上早餐的唯一正确方式

> 原文：<https://lifehacker.com/this-is-the-only-right-way-to-serve-a-mom-breakfast-in-1848855194>

“床上早餐”是一个迷人的概念——传统上被认为是在母亲节为母亲提供早餐的正确方式— ,但是松软的蛋黄、一摊糖浆和任何种类的餐具给本应放松的事情增加了一定程度的压力和尴尬。而母亲节的放松，比一年中的其他任何一天都要多，完全是的重点。

把妈妈从床上解放出来会让任何早餐都更容易吃，但是如果你的孩子坚持在她的卧室里为他们的妈妈服务，请礼貌地为她提供早餐板。

早餐板应该很容易吃，不需要切片或分叉(尽管稍微蘸一点和涂一点也没问题),妈妈应该能够在一两口就吃完每一种成分。(有疑问的时候，把东西切碎，放在牙签上。)这让她在床上吃早餐的压力小了很多，如果她必须这样做的话，但是早餐板可以在房子(或院子)的任何地方提供——这就是早餐板的好处；它很适合旅行。

你的早餐板也要赏心悦目，因为你生命中的妈妈值得你吃得有艺术感。具体写什么由你决定，但我有一些建议。请随意混合搭配以下任何一种，或者提出你自己的概念。

*   **百吉饼和熏鲑鱼板:**迷你百吉饼(或切成四分之一的大百吉饼)、奶油奶酪、熏鲑鱼、马槟榔和一些切成薄片的红洋葱是必备的。如果你真的想哇，加一份白鱼沙拉。(这是我将为我妈妈做的一个，她很兴奋。)
*   **煎饼板:**迷你煎饼配 [枫糖黄油](https://lifehacker.com/your-next-batch-of-pancakes-deserves-this-ultimate-mapl-1848577214) (蘸酱用)、鲜奶油(打蛋用)、新鲜水果(为了健康)。迷你巧克力片和蜜饯山核桃(洒在鲜奶油上)也不是个坏主意。
*   **华夫饼干板:**迷你华夫饼干(自己做或者买小蛋卷)，里面有煎饼板的所有材料。也许是一些培根或香肠，预先切片，插在牙签上。
*   **果酱头板:**这是送给爱吃水果摊的妈妈的。买一堆碳水化合物(吐司、烤饼、迷你羊角面包)和一大堆果酱、果冻和蜜饯。你甚至可以从 Bonne Maman 那里得到一个小样品包。除了水果之外，你还可以用锋利易碎的奶酪、凝结的奶油、腌肉或馅饼，以及非常好的花式黄油来填满这个棋盘。
*   **培根鸡蛋板:**这基本上就是早餐版的 [未加调料的鸡蛋板](https://lifehacker.com/un-devil-your-eggs-this-easter-1848778518) 。煮一些鸡蛋，直到蛋黄变得坚硬，有金黄色的液体中心，然后将它们对半切开，用黄油、切片萝卜和焯过的芦笋尖、脆培根片、鱼子酱、真正的蛋黄酱，也许还有一些腌火腿(肯塔基风味的火腿和 jamob iberico 都可以)。用吐司点和甜椒奶酪酱使它变得更圆。
*   **只是一个普通的该死的奶酪和熟食板:**我不是一个母亲，但我知道一些，每一个都是咸肉和 T3】奶酪 T5】的忠实粉丝。确保你提供了两者的多样性。对于奶酪，我通常选择硬的、软的和有趣的；至于肉，我会做一些腌制的、可以涂抹的东西(比如馅饼或沙拉酱)，以及一种非常别致的意大利腊肠(确保先切片)。加入一些芥末酱、干果、泡菜和非常好的饼干，你就有了一个非常迷人的棋盘。
*   黄油板:一些妈妈不需要肉类、奶酪和鸡蛋的丰盛早餐；有些妈妈只想要碳水化合物和黄油，她们应该得到她们想要的。如果这听起来像你的妈妈，做一个 [黄油板](https://lifehacker.com/impress-your-guests-with-a-butter-board-1848138546) ，配以非常好的面包(和 [薯片](https://lifehacker.com/things-you-should-be-buttering-but-probably-arent-1846674471) )和尽可能多的复合黄油。

在床上、院子里或沙发上把黑板递给妈妈，这样她就可以看她的故事了。饮料搭配由她决定，但搭配很难出错。
