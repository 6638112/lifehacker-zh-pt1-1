# 你真的应该只买蓝狗玩具吗？

> 原文：<https://lifehacker.com/should-you-really-only-buy-blue-dog-toys-1846685170>

狗主人经常带着犬享受到令人眩晕的极致，比如在家里给它们喂做好的饭菜，或者让它们参加 [奇特的露宿营地。](https://www.glencadia.com/?gclid=Cj0KCQjwpdqDBhCSARIsAEUJ0hMkaz8iLDQ5O6Mcb8uzrxhOz3PvaRpVgnwLX--slSpI8_fySfqa2coaAgl3EALw_wcB) 在购买狗狗玩具时，人们普遍会考虑玩具的颜色，力求买到最好的玩具。



由于狗是红绿色盲，并且不像它们的两足动物、人类同伴 [那样在视觉上(或身体上)导航，一些外行专家](https://www.reddit.com/r/LifeProTips/comments/mq3xsi/lpt_have_a_dog_get_them_blue_tennis_balls/) 坚持认为狗玩具应该是蓝色的——这种颜色应该让在绿草如茵、绿色田野或秋天树叶中嬉戏的狗非常醒目。

但是狗玩具的颜色真的有那么重要吗？你不需要成为一名兽医就能知道，狗非常依赖它们的嗅觉来感知周围的环境，并且其他刺激比视觉更能让它们了解物理世界。那么，你是应该听从 Reddit 的建议，还是应该从科学文献《蓝色玩具的重要性》中得到提示呢？

### 颜色没有其他因素重要

狗对玩具的偏好与其说是由颜色决定的，不如说是由许多不同的刺激决定的。纽约城市大学思考狗中心主任萨拉-伊丽莎白·拜奥斯尔说，就视觉感知而言，亮度问题和总体颜色一样重要。她解释说，狗是双色的，这意味着它们只有两个视锥感光细胞，可以检测蓝色和黄色，而人类有三个，可以分辨红色、蓝色、和绿色的组合。

但是当涉及到狗对相似颜色的探测时，亮度是解析细微差异的关键。她告诉 Lifehacker，在详细阐述狗的亮度和颜色之间的关系之前，她说:“当相似颜色的两种色调仅在亮度上有所不同时，它们仍然是可以辨别和识别的。”。

结论是，当一个绿色的网球在草地或人工草皮上滚动时，狗不一定更难看到它，只要微小的颜色差异反映了某种刺激，例如阳光。她解释道:

> 色觉可以根据给定刺激的亮度而改变。当光被这些刺激反射时，就像一个红色的球被扔在绿色的草地上，颜色反射光的方式不同。狗对亮度的变化很敏感，所以虽然球和草的颜色对狗来说可能不像红色和绿色，但它们的亮度可能仍然不同。

还有质地的问题。例如，与人类相似，狗通常依靠周围环境的纹理和形状来区分玩具和植物或土堆。Byosiere 详细描述了另一种情况，这种情况说明了狗是如何不依靠颜色来导航周围环境的:

> 如果你能想象看到一个棕色的球扔在棕色的草地上，你可能仍然能识别出一个物体存在，因为球的结构/轮廓与草地的结构不同。一个瘦长，一个圆润，棱角不一。这可能需要一点时间，但你可能能够找到它使用其他线索(除了颜色)。

### 你如何挑选合适的玩具？

这个问题没有你想象的那么复杂。兽医 Valerie Patton 告诉 Lifehacker，狗对玩具的偏好与孩子被某些玩具吸引的倾向非常相似。

她在邮件中写道:

> 我发现，玩具的质地、大小和毁坏它的能力，或者它是否有吱吱声，都决定了它是否是一个好主人。像孩子一样，狗有喜欢某些玩具的理由，有时我们能理解，有时我们不能。

虽然关注玩具的颜色表明主人考虑到他们的狗的喜好是值得称赞的，但让你的狗决定他们喜欢什么玩具可能更好。正如 Byosiere 所说，主要根据颜色挑选玩具“只是过分简化和低估了狗的感知能力。”相反，主人应该给他们的狗买各种各样的玩具，让他们的宠物自己决定最喜欢什么玩具。

Byosiere 详细介绍了实现这一目标的最佳方式:

> 让他们选择他们玩的物品，旋转玩具，这样他们就不会感到无聊(不要把它们放在家里，总是可以拿到的)，找到创造性的方法，用新奇的和适应他们的玩具来丰富你的小狗的智力。

考虑到这一点，你的狗会立刻把它们最喜欢的玩具或网球咬成碎片，不管它是什么颜色。
