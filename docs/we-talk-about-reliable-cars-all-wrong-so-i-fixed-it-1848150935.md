# 我们谈论可靠的汽车都错了，所以我把它修好了

> 原文:[https://jalopnik . com/we-talk-about-reliable-cars-all-wrong-so-I-fixed-it-1848150935](https://jalopnik.com/we-talk-about-reliable-cars-all-wrong-so-i-fixed-it-1848150935)

我们都听说过一遍又一遍地谈论特定汽车的可靠性，我们都*大体上*理解那是什么意思。从本质上说，我们的意思是汽车会不会在你身上抛锚，让你陷入困境，并留给你一大笔昂贵的修理费，或者你永远不必担心它？问题是我们对“可靠性”这个词要求太高了事实上，可靠性更多的是一个范围，有多种因素在起作用，所以也许是时候扩展我们谈论它的方式了。我有一些想法。

我认为关于可靠性，我们正在处理一种光谱；这不是一个二元问题，因为我们必须考虑汽车如何被处理和维护，以及无时不在的*不可知*因素:汽车可能已经理解并预计到的问题是什么？我们能解释一下吗？

考虑到这一点，我将可靠性的概念分为六个主要类别。将这一数字降低到6是很棘手的，但我认为大多数汽车都可以找到自己的位置。

此外，假设一辆车可以在不同类别之间移动，在某种程度上，这取决于车主能够投入多少精力来保养这辆车。

当你知道我说的是什么类别时，这就更有意义了，所以在这里，继续读下去:

合作:一辆车，如果你按照要求去做，小心而有规律地保养，它永远不会让你失望，除了一些真正灾难性和不可预防的事情。这样一辆车必须没有任何设计上的致命弱点，再多的预防性维护也无济于事。

理想情况下，任何汽车*都应该*能够进入这一类别，但残酷的现实往往不会让这种情况发生，而且要求很高:这一类别只存在于车主直接和一致的行动中。

宽容:宽容的汽车是一件美好的事情。这些车肯定会出故障，但它们似乎有公差，比你想象的要多一点。例如，如果错过了换油，这并不是世界末日，只要你在*太久*之前到达。

如果你缺乏足够的纪律性来成为合作类的一员，你可以尽力而为，希望你的车能够宽容。宽恕并不意味着魔法，这类车可能会在你身上抛锚，但至少它们在缓冲方面更慷慨。

耐用的:耐用的汽车是指能够承受大量的使用而不会损坏的汽车。脆弱的反义词。这并不意味着汽车永远不会抛锚，而是暗示了某种坚固性既可以来自设计的简单，也可以来自实际上的过度建造。

这并不意味着它不会因为某些机械原因而困住你，只是意味着它很难通过你的使用而被打破。

**可滥用:**这一类别适用于很少甚至没有维护或保养的汽车。这并不意味着这些车在*跑得特别好*，很可能这些车可能有很多问题，甚至有些很严重，但不知何故它们还是继续跑着。

有句老话是这样说的:某个品牌的汽车比大多数汽车跑的时间都长，或者你高中朋友的旧卡罗拉从来没有换过油，它的安全带像一个高潮女妖一样发出刺耳的声音，这些都属于这一类。想想那些似乎永远出于怨恨、反抗而不断离开的狗屎盒子。

这些汽车有时会发生故障，可能有一定的规律性，但当它们发生故障时，几乎总是因为一些简单、明显、甚至愚蠢的故障，这些故障很容易修理，也很便宜，这非常适合于你能做的任何即兴的干扰来保持它的运动。

我的Yugo就属于这一类，如t[he two-hose-clamp-and-rock](https://jalopnik.com/i-fixed-my-yugo-with-two-hose-clamps-and-a-rock-1848131984)fix所示。

这可能是可靠性谱中最糟糕的一类。一辆傲慢无礼的汽车会在不可预知的时间，出于不可预知或可预知的原因，幸灾乐祸地抛锚，不管你对汽车的保养和维护水平如何，也不管你在车上投入了多少钱。

傲慢的汽车会有令人困惑和难以捉摸的小精灵，也许是电气的，也许不是，但总是令人讨厌。他们会在某一刻看起来很棒，然后把你扔在一个寒冷、潮湿、遥远的地方。它们往往很复杂，经常有一些众所周知的固有设计缺陷，但更糟糕的是，可能没有。
