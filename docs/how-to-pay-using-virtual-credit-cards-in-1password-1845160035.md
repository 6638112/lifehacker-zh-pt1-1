# 如何使用1Password中的虚拟信用卡支付

> 原文:[https://life hacker . com/how-to-pay-using-virtual-credit-cards-in-1 password-1845160035](https://lifehacker.com/how-to-pay-using-virtual-credit-cards-in-1password-1845160035)

如果你担心网上购物时的隐私和安全，有一种新的方式来保护自己: P opular加密密码管理器 1Password 现在支持一种新的“ [隐私卡](https://blog.1password.com/privacy-virtual-cards) ”功能，使网上购物更加安全。

下面是它的工作原理:当你使用隐私卡时，你仍然用你的借记卡或支票账户支付，但隐私卡是一个代理。你所支付的公司或服务不会分享你的真实财务信息。

你制作的每张卡都与一个商家相关联——比如网飞、苹果、谷歌或亚马逊——在其他任何地方都不起作用。你可以在每张卡上设置消费和频率限制，以防止未经授权的购买或帮助你坚持预算。您还可以随时暂停或删除隐私卡，而不会影响相关的借记卡或支票账户。

通过使用隐私卡，你不需要*购买*a 1密码，但如果你有额外的特权，你会得到一些额外的特权。 1Password的浏览器插件将保存隐私卡的名称、号码、有效期和CVV， [，就像保存登录信息一样](https://lifehacker.com/the-five-best-password-managers-5529133) 。该信息会在结账时自动填写，因此您无需记忆任何内容。您还可以使用[1 password X浏览器扩展](https://support.1password.com/getting-started-1password-x) 随时创建和管理新卡，而无需通过Privacy.com。

### 如何创建和使用带有1密码的隐私卡

1密码用户需要首先在其Privacy.com个人资料的设置中链接其1密码帐户。

1.  登录您的Privacy.com帐户。
2.  单击右上角的个人资料名称。
3.  转到**账户>集成**。
4.  点击1密码旁边的**“连接”**。
5.  遵循屏幕上的说明。

帐户关联后，您现在可以创建新卡。

您可以在Privacy.com上完成此操作，也可以在结账时根据提示点击1密码图标。按照说明创建新的支付方式，或者选择一个已保存的卡来使用该方式。

您还可以将之前创建的Privacy.com卡 添加到您的1密码帐户中:

1.  登录到1Password.com
2.  单击箭头图标。
3.  点击**“+”**
4.  选择**“信用卡”**
5.  输入卡片详细信息，点击**“保存”**
