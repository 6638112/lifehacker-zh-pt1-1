# 打包一个移动箱子的最好方法

> 原文:[https://life hacker . com/the-best-way-to-pack-a-moving-box-1838011535](https://lifehacker.com/the-best-way-to-pack-a-moving-box-1838011535)

搬家总是有压力的。但更糟糕的是，当你到了新家，打开一个盒子，发现你的打印机打碎了所有装着它的酒杯。

因此，为了让搬家变得轻松，我们收集了一些我们最喜欢的技巧，来包装一个高效、安全且(大部分情况下)不会破碎的搬家盒子。

首先，保持条理。把每个盒子放在房子的一个房间里(不要再用酒杯装打印机了！)这也将使*打开*更加容易。另外，一定要把最重的物体放在盒子的底部，这样就不会有东西被压碎。

为了处理你的易碎物品，自己动手做一些包装花生，把t恤塞进任何宽松的空间，把水杯或灯泡放在袜子里以防万一。(如果你还没有收拾好你的厚冬袜，那就更安全了。)
