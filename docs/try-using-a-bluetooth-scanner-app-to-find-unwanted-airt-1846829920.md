# 尝试使用蓝牙扫描仪应用程序来查找不需要的安全气囊

> 原文：<https://lifehacker.com/try-using-a-bluetooth-scanner-app-to-find-unwanted-airt-1846829920>

有一种 [的合唱](https://mashable.com/review/apple-airtags-review/) [的声音](https://gizmodo.com/the-best-thing-about-apples-airtags-is-also-the-scaries-1846786338) 说苹果应该做更多的**来保护每个人——安卓和 iOS 用户——免受不必要的跟踪和追踪。我同意他们的观点，而 AirTag 的整个概念让我很好奇。你能追踪追踪器吗？**

**Watch**

**我说的不是苹果的内置警告，当未识别的空中标签发现自己沿着与你相同的路径移动时。你可以在苹果公司的网站 上阅读关于如何运作的所有内容。剧透:如果你是 Android 用户，或者运行的是早于 14.5 的版本的 iOS，你需要等待一个 AirTag 发出一种声音，这种声音在它不在相关设备附近几天后发出[](https://mashable.com/review/apple-airtags-review/)*。(如果你被定期联系你的人跟踪，那就没什么用了。)***

***我的想法是，一定有某种方法可以通过它发出的信号来追踪一个航空标签。它一定是以某种方式伸出手来，作为它与加入苹果“找到我的”网络的设备握手的一部分。***

***事实证明，你*可以*手动追踪一个 AirTag。排序。你只需要一个蓝牙扫描仪应用程序(大概是类似于或 [蓝牙扫描仪](https://play.google.com/store/apps/details?id=com.pzolee.bluetoothscanner&hl=en_US&gl=US) )。所述设备在被识别时不会显示为空中标签，但你至少能够提示附近有一些奇怪的、未知的蓝牙设备。而且，理想情况下，移动你的手机有可能帮助你确定它的位置。***

 **[https://lifehacker.com/embed/inset/iframe?id=twitter-1387999187954925571&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1387999187954925571&autosize=1)** 

**O 如果你对空中标签进行蓝牙扫描，要记住的一件事是它们的 ID——看起来像 12:c9:34:f8:a1:...会变成不同的随机值。那是故意的。您的目标是简单地找到附近正在传输的蓝牙设备的位置。您将无法具体识别 AirTag(除非它与 iPhone 不成对，这将禁用上述随机化)。**

**我还没有我的航空标签，所以我还不能更详细地查看如何扫描——包括普通人成功做到这一点的可能性有多大。也就是说，使用蓝牙扫描应用程序扫描不想要的 AirTags 不应该是你确保隐私的首选方法。首先，你可能会得到一些转移注意力的东西，而且要找到一个无线标签并不像找到一个不停鸣叫的小设备那么容易*就像*一样。**

**相反，如果你是 iPhone 用户，你绝对应该 [安装 iOS 14.5](https://lifehacker.com/how-to-set-up-every-ios-14-5-feature-worth-knowing-abou-1846820998) ，如果这个问题出现，它会让你被动地知道。如果你使用的是安卓系统，并且你感到怀疑，那么在等待 AirTag 发出啁啾声时，转向扫描仪(假设控制它的人不在附近)。希望苹果公司能为那些希望在一天之内，而不是一周之后，就知道有一个航空标签在跟踪他们的人，提供一个更强大的解决方案。**
