# 在你的薄烤饼上加点爱尔兰奶油

> 原文：<https://lifehacker.com/spike-your-pancakes-with-irish-cream-1848656004>

我从未调查过 23andMe(或任何类似的基因测试),因为我对结果并不特别感兴趣。我是白人。我来自密西西比。没有好的或高贵的水果挂在那棵树上。



我确实知道我有几分爱尔兰血统。结婚的时候，我戴了一些爱尔兰耳环我的爱尔兰高曾祖母在她移居美国之前在爱尔兰举行的爱尔兰婚礼上戴的耳环。我也有一点威尔士血统， b 但是我们在美国没有一天威尔士式的放纵和饮酒来庆祝，所以一些白人倾向于夸大他们与翡翠岛的联系，这样他们就能在圣帕特里克节感受一些东西。这就是我对这些爱尔兰奶油煎饼所做的——在一天中假装热情，我没有任何情感联系，以便感受一些东西。幸运的是，煎饼很好吃，而且非常容易做:拿一盒煎饼粉，用相同体积的爱尔兰 c 令代替说明书中所说的水或牛奶。我用了一个杂货店品牌,它的酒精含量没有 true Bailey 的高，但却保留了明显的酒精成分。

爱尔兰奶油煎饼有点甜，有点醉，和让人欲罢不能。虽然他们用商店里买的爱尔兰奶油做出来很好吃，但是用 [自制的东西](https://lifehacker.com/make-your-own-irish-cream-immediately-1845788014) 会更好，自制的东西也很容易在你自己舒适的厨房里搅拌。(如果你想吃得更甜/更美味，在翻动煎饼之前，在面糊中加入一些巧克力片。)

一旦你得到或生产出你选择的奶油(我听说他们现在甚至生产杏仁牛奶版本)，根据盒子上的说明混合你的面糊，用全部(或一些)牛奶或水代替爱尔兰奶油。 [不要过度混合](https://lifehacker.com/leave-your-pancake-batter-lumpy-1831547219)——块状面糊可以做成松软的煎饼——用黄油煎效果最好。

如果你担心酒精的刺激在早上第一件事就是有点过于强烈，尝试用爱尔兰奶油代替你食谱中一半的液体，然后根据需要进行调整。增加爱尔兰奶油的量也会增加糖的含量，所以把温度调得比平时低一点，以防止你的煎饼在内部完全煮熟之前烧焦。搭配枫糖黄油、爱尔兰奶油口味的鲜奶油，或者 [朗姆酒火腿](https://lifehacker.com/sous-vide-an-always-sunny-rum-ham-that-would-make-frank-1848157451) 。
