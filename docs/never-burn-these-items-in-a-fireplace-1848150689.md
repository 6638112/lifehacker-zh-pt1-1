# 切勿在壁炉中燃烧这些物品

> 原文：<https://lifehacker.com/never-burn-these-items-in-a-fireplace-1848150689>

现在在许多地区是壁炉季节，这(希望)意味着蜷缩在温暖的壁炉前度过舒适的夜晚。或者，也许你正在打扫卫生，无意中发现了一堆前任的来信和礼物，出于其他不那么舒适的原因，你感到那股暖流在向你招手。



如果你有一个真正的烧木头的壁炉，你确实需要给它添柴——b 但事实证明，你不能随便放入任何木头(或其他材料)做的东西就收工。事实上，根据 Carol J. Alexander 在《农民年鉴》中的一篇文章 [，有一个完整的清单列出了木材和其他物品永远不应该进入壁炉。以下是远离这些火焰的方法。](https://www.farmersalmanac.com/fireplace-tips-65919)

## 潮湿或未风干的木材

在某些情况下，即使是看起来像木柴的东西也不应该放在壁炉里。正如亚历山大解释的那样，这是因为木柴需要适当的调味——这意味着它含有很少甚至没有水分。

而且这不是一夜之间就能带来的:调味过程至少需要一年。“你可以通过它的外观来识别干燥的木材——适当干燥的木材是灰色的，有裂缝或裂纹，有松散或脱落的树皮，” [她写道](https://www.farmersalmanac.com/fireplace-tips-65919) 。

即使柴火经过适当的调味，当它潮湿时也不能放在壁炉里，所以一定要把你的柴火存放在远离自然环境的地方。

## 经过处理、涂漆或加工的木材

当你燃烧一块木头时，你也燃烧了上面的一切——包括用于处理或油漆木头的化学物质，或者将人造木材如刨花板粘合在一起的粘合剂。正如 [亚历山大](https://www.farmersalmanac.com/fireplace-tips-65919) 指出的，你不会想吸入化学物质燃烧的副产品。出于这个原因，你也应该避免焚烧木质托盘，因为她说它们经常用化学品处理。

## 圣诞树(及其他含树脂的木材)

知道活圣诞树上的粘性物质吗？它被称为树脂(或树液)，含有这种物质的树木——如松树、云杉和雪松——也不应该进入壁炉。如果你燃烧太多的树脂木材，它会增加你烟囱里杂酚油的数量(意味着你需要更频繁地清理它)。此外，树脂会使火势蔓延，这会导致余火蔓延到壁炉外,烧伤附近的人。

## 印刷、彩色或涂层纸制品

用普通的旧黑白报纸生火是可以的， [亚历山大说](https://www.farmersalmanac.com/fireplace-tips-65919) ，但是上面有任何印刷、着色或涂层的纸制品不应该放在壁炉里。即使它们是用大豆墨水打印的，纸张也可能含有颜料和其他添加剂，。据 Alexander 称，用于加厚礼品包装和直邮传单的涂层也不能完全燃烧。

## 塑料

没人会告诉你不要在壁炉里烧任何塑料制品，但我们会把这个留在这里，以防万一。
