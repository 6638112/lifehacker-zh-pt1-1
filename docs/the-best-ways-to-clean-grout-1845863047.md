# 清洁水泥浆的最佳方法

> 原文：<https://lifehacker.com/the-best-ways-to-clean-grout-1845863047>

瓷砖很容易清洗，但祝你好运，让周围的水泥浆恢复到原来的珍珠白。

因为它们又窄又粗，所以清理起来会很麻烦(除非你喜欢花几个小时弯腰拿着牙刷)。幸运的是，我们有三种方法可以让清理水泥浆变得轻而易举。

## 蒸汽清洁器

正如我们之前测试过的，一个带刷子附件的手持蒸汽清洁器 [可以神奇地用](https://lifehacker.com/is-a-steam-cleaner-as-useful-as-the-internet-claims-1845824477) 将灰浆上的灰尘吹掉。由于蒸锅会把脏水喷到下面的瓷砖上，所以一定要用从上到下的方法进去，一定要边打扫边把脏水全部擦掉。

## 魔法橡皮擦

[一种神奇的橡皮擦](https://lifehacker.com/use-magic-erasers-to-clean-sneakers-grout-and-keyboar-1835154205) (通用名:三聚氰胺泡沫)是增亮水泥浆的绝佳材料。把橡皮擦弄湿，挤出多余的水就行了。沿着水泥浆擦洗，看着它快速去除任何粘糊糊的东西和污垢。

## 漂白凝胶

在你的淋浴间沿着灌浆线喷一些厕所清洁漂白剂凝胶。将一段卫生纸卷成一条长蛇，小心压在漂白剂上。静置一小时，然后取出卫生纸，冲洗，擦干。
