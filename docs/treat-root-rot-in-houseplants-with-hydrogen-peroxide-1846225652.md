# 用过氧化氢处理室内植物根腐病

> 原文：<https://lifehacker.com/treat-root-rot-in-houseplants-with-hydrogen-peroxide-1846225652>

植物人知道，不仅给植物浇水很重要，你还必须知道什么时候该适可而止。给一个计划浇水太容易了，尤其是一周只需要浇一次水的计划。你可能认为你偷偷给植物多浇了一点水，确保它保持水分，这是在帮植物的忙。



但实际上，你可能会给它一种叫做“根腐病”的东西这是看起来的样子，也是一种尝试用可能就在你水槽下面的东西来恢复你衰弱的叶子的方法。

### 什么是根腐病？

根腐病就像它听起来的那样。它可能由两种情况引起，都涉及水，根据 [园艺知识](https://www.gardeningknowhow.com/plant-problems/disease/treating-root-rot-gardening-tips-for-housplants.htm) :

1.  长时间过度浇水，会切断植物根部的氧气供应，导致它们腐烂——有可能蔓延(并杀死)健康的根部。
2.  真菌可能存在于土壤中，并处于休眠状态，直到被过度浇水。

当一株植物开始枯萎并变黄时，你就可以知道它患了根腐病。如果你把植物从花盆里拿出来 [检查它的根](https://www.gardeningknowhow.com/plant-problems/disease/treating-root-rot-gardening-tips-for-housplants.htm) ，你可能注意到的第一件事就是一股沼泽和/或臭鸡蛋的味道。腐烂的根会呈现黑色，触摸时可能会散架。

### 如何用双氧水处理根腐病

不是所有烂根的植物都能救。但在某些情况下，它们是可以的，这要归功于过氧化氢——过氧化氢杀死了导致腐烂的细菌，并向土壤中释放出额外的氧气，这将有助于植物长出新的健康的根，公寓疗法的莫莉·威廉姆斯说。T3】

不要直接往根部倒双氧水。取而代之的是，将一份 3%的过氧化氢和两份水混合，放入喷壶或喷雾瓶中。你既可以在植物还在盆里的时候使用它，也可以在植物从盆里拿出来之后使用它。

下面是如何给花盆里的植物浇水，根据威廉姆斯的说法:

> 让土壤完全干燥。然后，用过氧化氢溶液彻底浇灌植物。重复这个过程，直到你看到植物的健康状况有所改善。如果你不确定你的植物的根是否再次健康，把它从花盆里拿出来检查一下。如果健康，根将是坚定和发白的颜色。

下面是从花盆中取出植物的方法，同样，威廉姆斯的:

> 轻轻地将植物根部周围的盆栽介质全部移除。一旦完全暴露，将过氧化氢溶液喷洒或倾倒在整个根球上。将植物移植到新的潮湿的盆栽混合物中。再次给植物浇水之前，让土壤变干。

虽然这并不是每次都管用，但是如果你打算扔掉一棵烂根的植物，因为它看起来已经无法修复了，试试过氧化氢溶液吧。
