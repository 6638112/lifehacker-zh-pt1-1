# 这是我们能期待的最早的儿童COVID疫苗

> 原文:[https://life hacker . com/this-is-the-early-the-we-can-expect-a-covid-vaccine-for-1848245324](https://lifehacker.com/this-is-the-earliest-we-can-expect-a-covid-vaccine-for-1848245324)

儿童对新冠肺炎疫苗的等待是极其痛苦的。5至11岁年龄组的[终于在上个月](https://lifehacker.com/covid-vaccines-are-finally-being-recommended-for-kids-a-1847985067) 拿到了他们的疫苗，这是在疫苗被批准用于成年人近一年之后。但是五岁以下的孩子仍在等待，不幸的是，他们等待的时间比我们预期的还要长。

Watch

那是因为 [辉瑞公司宣布他们针对小孩的COVID疫苗试验没有成功](https://www.pfizer.com/news/press-release/press-release-detail/pfizer-and-biontech-provide-update-ongoing-studies-covid-19) 。他们给6个月到4岁的儿童服用了两剂疫苗，只有其中一些儿童最终产生了足够的免疫反应，可以宣布疫苗有效。

这里有两条小好消息:首先，疫苗被认为在这个年龄组，是安全的，副作用很小。所以从安全和T4的角度来看，这种疫苗效果很好。另一个好消息是，年龄在6个月到24个月之间的最小的孩子的免疫反应与年轻人从他们的疫苗中获得的免疫反应相当。

坏消息是，2岁、3岁和4岁的儿童没有从正在测试的两剂系列疫苗中获得足够好的免疫反应。如果他们这样做了，辉瑞可能会马上申请授权。但既然他们没有，我们就只能等了。

## 下一步是什么？

该公司的计划是在第二次注射后两个月，为这个年龄段的儿童增加第三次注射。在实践中，这意味着你将有一剂，等待三周，然后第二次，然后等待两个月，然后得到第三次。这几乎就像一个早期助推器。

如果这一战略奏效，**该公司预计在2022年第二季度** — **意为4月、5月或6月期间，为6个月至5岁的儿童** [**申请疫苗授权。**](https://www.washingtonpost.com/health/2021/12/17/pfizer-vaccine-young-children-third-shot/)

与此同时，他们已经开始对年龄较大的儿童进行加强剂量的试验。事实可能证明，即使是成人疫苗系列也需要三剂，因为三剂似乎比两剂更好地对抗Omicron，三剂方案是 [已经推荐](https://lifehacker.com/how-many-covid-vaccine-shots-do-you-need-if-youre-immun-1847947776) 作为免疫缺陷者的标准。

Moderna也在对12岁以下的儿童进行疫苗试验，但他们还没有宣布结果。
