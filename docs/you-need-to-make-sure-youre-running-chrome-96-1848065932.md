# 你需要确保你运行的是Chrome 96

> 原文:[https://life hacker . com/you-need-to-assure-you-running-chrome-96-1848065932](https://lifehacker.com/you-need-to-make-sure-youre-running-chrome-96-1848065932)

互联网请注意:谷歌Chrome浏览器的新更新发布了。 Chrome 96(全名96 . 0 . 4664 . 45——朗朗上口)将于本周在我们所有的谷歌浏览器上推出。有了它，我们获得了一些很酷的新功能和，更重要的是，获得了重要的安全补丁。这意味着如果你不更新，你可能会把你的数字安全置于危险之中。

## 关键安全补丁

你需要更新到Chrome 96的唯一原因是这个一个:新版本包括25个已确认安全漏洞的补丁。虽然的风险状况大多在“低”或“中”，但谷歌突出显示了的七个“高”风险，包括:

*   **高**CVE-2021-38007:V8中的类型混淆。【北极星冯和SGFvamll在奇异安全实验室报道】2021-09-29
*   **高** CVE-2021-38008:在媒体上免费后使用。*思科Talos的Marcin Towalski 2021-10-26报道*
*   **高** CVE-2021-38009:缓存中的不适当实现。*2021-10-16栾Herrera (@lbherrera_)报道*
*   **高** CVE-2021-38006:免费存放在粉底后使用。*2021-08-17*谷歌Zero项目的Sergei Glazunov报道
*   **高** CVE-2021-38005:在装载机中空闲后使用。*2021-08-18*谷歌Zero项目的Sergei Glazunov报道
*   **高** CVE-2021-38010:在服务人员中执行不当。*2021-10-2*谷歌零计划的谢尔盖·格拉祖诺夫报道
*   **高** CVE-2021-38011:免费存放在粉底后使用。*2021-11-09*谷歌零计划的Sergei Glazunov报道

不管你能否破解所有这些，保护自己免受这些安全漏洞的唯一方法就是尽快更新你的浏览器。

## 每个站点的黑暗模式设置

即使你是一个狂热的黑暗模式用户，有时你也不希望所有的网站都默认为黑暗主题。对于Chrome 96，谷歌正在测试一项新功能，让你在逐站点的基础上选择黑暗模式设置。

因为这个特性仍然在测试阶段中，你需要启用它的标志来让它工作。首先，将以下内容复制并粘贴到您的Chrome URL菜单栏中，然后点击go查看您的终端是否有该功能:

## `chrome://flags/#darken-websites-checkbox-in-themes-setting`

启用该功能后，您可以将站点添加到黑暗模式例外列表中；当你以后访问那些网站时，Chrome不会触发黑暗模式。

## 更快的导航

Chrome 96带来的一个很酷的新变化是让浏览你的网页历史变得更快。Chrome现在会保存您最近访问过的网站的缓存，当您使用导航箭头(或键盘快捷键)返回您的历史记录时，就会转到该缓存。 [谷歌在这里声称](https://www.chromestatus.com/features/6279906713403392) 这一改变将导致“即时导航到以前访问过的网页”

## 剪贴板现在保留png元数据

在Chrome的早期版本中，系统会在读取PNG文件之前去除它们的元数据。谷歌已经决定停止这样做，走更传统的路线保存这些元数据。这应该会加快PNG文件的读取速度——根据谷歌 的说法， [快了14倍。](https://www.chromestatus.com/feature/5629962485760000)

Chrome 96也有很多开发者专用的功能，你可以在 [谷歌的Chrome开发者日志](https://developer.chrome.com/blog/new-in-devtools-96/) 中了解更多。

## **如何更新谷歌Chrome**

如果你不更新你的浏览器，以上的功能和补丁对你来说将没有任何用处。幸运的是，谷歌有一个可靠的自动更新系统，我可能你的浏览器已经在你没注意到的情况下更新了。

要进行检查，点击浏览器右上角的三个点，选择“设置”，然后选择“关于谷歌浏览器”(或者，点击三个点，选择“帮助”，然后选择“关于谷歌浏览器”)如果你的浏览器完全更新，你会在这里看到。如果没有，您可以从该页面进行更新。如果有可用的更新，你可能已经在浏览器任务栏看到一个大的“更新”按钮。点击这个，你就可以开始了。
