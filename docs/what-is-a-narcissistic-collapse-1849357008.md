# 什么是自恋型崩溃？

> 原文：<https://lifehacker.com/what-is-a-narcissistic-collapse-1849357008>

“自恋者”这个词经常被用来描述任何以自我为中心的人，但自恋型人格障碍是一种真正的诊断，可能相当严重。当我们谈论自恋者时，我们倾向于将他们框定为恶棍，但重要的是要记住，患有 NPD 的人有精神疾病，因此他们可能会面临挣扎。例如，一个 NPD 患者可能会经历“自恋型崩溃”

## **什么是自恋崩溃？**

正如埃莉诺·格林伯格博士在今年早些时候的[一篇《今日心理学》文章](https://www.psychologytoday.com/us/blog/understanding-narcissism/202202/what-is-narcissistic-collapse) 中解释的那样，关于自恋者的一个主要误解是，他们有很强的自尊心，从不怀疑自己。事实上，一个患有 NPD 症的人有明显不稳定的自尊——这就是为什么他们依赖他人的认可。

当预见到某种公共失败时，NPD 患者会经历所谓的自恋型崩溃，这是由他们的焦虑造成的。被发现是一个有缺陷的人(我们都是)，对患有这种疾病的人来说是毁灭性的，会导致他们陷入基于羞耻的抑郁症。他们对未来暴露的恐惧影响了他们现在的能力。

NPD 的某些人用夸张的外表作为一种辩护。纽约市快乐苹果心理治疗中心玛吉·沃恩博士说，当防御机制停止工作时，一个人会被迫“面对导致他们这种状况的深深的羞耻和无价值感”这是“低”的结果，它粉碎了他们保持非凡的伪装的能力。作为可能引发崩溃的例子，她指出失业或社会拒绝，但补充说这些可能会连续发生，因为一次打击“很少足以穿透自恋者的防御。”

## **自恋崩溃的迹象有哪些？**

根据心理中心 T5、的专家团队 [的说法，一个经历自恋型崩溃的人会表现出一些不同的迹象:](https://psychcentral.com/disorders/narcissistic-personality-disorder/narcissistic-collapse)

*   报复行为
*   对他人的回避
*   抑郁

沃恩解释说，由于自恋型崩溃动摇了一个人的身份，并影响了他们的驱动力，他们通常会感到“空虚和没有方向”，因为他们看不到从他人那里获得钦佩的可能性:“患有 NPD 症的人在经历自恋型崩溃时，会感到非常沮丧、孤独，甚至有自杀倾向，这并不罕见。”

这些症状虽然强烈且有潜在危险，但通常是暂时的。如果你有 NPD，知道某人有，或者 [怀疑你生命中的某人有](https://lifehacker.com/how-to-figure-out-if-youre-dating-a-narcissist-1844390450) ，密切注意上面的迹象，预测自恋的崩溃。格林伯格还在她的文章中指出了自杀的可能性，指出经历自恋崩溃的人可能会将死亡视为比公开“暴露”更好的选择。

除了害怕被“暴露”为一个有缺陷的人，NPD 患者还可能进入自恋型愤怒状态，因为他们没有得到他们认为应该得到的外部认可。对于患有这种疾病的人来说，这是一种压力，会导致自恋的愤怒，甚至可能导致崩溃。

有可用的帮助，但对于 NPD 患者来说，寻求帮助可能很困难，因为正如沃恩指出的那样，NPD 患者可能会抵制治疗和诚实地承认随之而来的缺陷。她补充说，“最重要的是，他们与治疗师建立了信任关系，治疗师可以正常化自恋者的恐惧和需求。”她说，对于 NPD 患者来说，治疗中最具治愈性的因素实际上是与他们的提供者的关系，因为这给了他们被接受的空间——即使他们有缺点。这也可以帮助他们与他人建立更好的关系。
