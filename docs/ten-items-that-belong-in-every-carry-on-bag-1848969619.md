# 每个随身包里都应该有十样东西

> 原文：<https://lifehacker.com/ten-items-that-belong-in-every-carry-on-bag-1848969619>

有很多方法可以打包你的随身包，但有时“什么”比“如何”更重要。坦率地说，坐飞机旅行既恶心又无聊。当我在友好的天空飞行时，我有一份个人物品清单，上面列有我随身行李中的“必备”物品，我试图让自己感觉更清新一点和更愉快一点。这些小件物品中的许多都值得 放在你的随身包里。

Watch

一件干净的 t 恤。我过去从来不会在随身行李中带一件干净的 t 恤—也就是直到上了飞机，我把一整罐姜汁汽水洒在自己身上，一路穿着又湿又粘的 t 恤去阳光明媚的奥兰多。现在我总是带一个备用的。

**一支笔。如果你像我一样，在你的眼睛变得呆滞无神之前，你只能花这么多时间盯着你的设备。如果你想做飞行杂志的填字游戏，填写海关表格，或者只是涂鸦，一支笔就派上用场了。**

一包口香糖。 [据听力健康的](https://www.healthyhearing.com/report/52447-Airplanes-and-ear-pain-why-it-happens-and-what-you-can-do#:~:text=When%20you%20fly%2C%20the%20trick,which%20helps%20equalize%20air%20pressure.) 说，嚼口香糖或(吸吮硬糖)会刺激频繁吞咽，这有助于均衡气压。

**便携式手机充电器。**当你收到手机电量低的通知时，你知道你对此无能为力，这是一个令人羞愧的时刻。不要措手不及；买一个 [便携式充电器](https://gizmodo.com/this-is-the-best-iphone-12-magsafe-battery-1847468501) 给自己一个战斗的机会，在你的跨国飞行中打发无聊。

**太阳镜/睡眠面罩。** [谁有权在](https://lifehacker.com/who-has-the-right-to-the-window-shade-on-a-flight-1838482583) 航班上遮阳还没有定论，但你可以让这成为一个问题。戴上面具比与陌生人进行一场外星人礼仪辩论要容易得多。

一根除臭剂。当你从户外经过机场和飞机时，你会经历不同的温度。考虑到你将要呆的地方很近，如果需要的话可以重新申请(当然是在浴室里小心翼翼地)。

**组织。**纸巾是你想在之前手头有*你需要它们的东西之一。*

**小吃。**随着航空公司不断寻找削减开支的领域，你不能再指望它们在飞行期间为你提供足够的零食。 [只是考虑从乔氏购买而不是哈德逊新闻](https://lifehacker.com/why-i-buy-all-of-my-plane-snacks-from-trader-joes-1793461509) 。

一本杂志。我选择杂志的原因是因为我可以把杂志塞进我的包里，而不用像保存一本书那样关心它的保存状况。另外，是一个很好的借口通过阅读一些没用的东西来放纵自己。

**耳机。**也许是你能拥有的最重要的东西，当你正在寻找可以淹没飞机引擎声或婴儿哭声的东西时，耳机就会出现。起飞前确保你的充电箱充满电。
