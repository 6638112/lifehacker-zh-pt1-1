# 关于新的家庭COVID测试，您需要了解什么

最近几个月，FDA已经授权了几项非处方药COVID测试，这样你就不必去检测中心检查自己是否患有冠状病毒。从本周开始，你可以 [在沃尔格林、沃尔玛，或者CVS](https://www.usatoday.com/story/news/health/2021/04/19/covid-home-tests-walmart-cvs-walgreens-binaxnow-ellume/7286667002/) 进行抗原自测。

### 什么是抗原测试？

这不是PCR检测，PCR检测被认为是检测当前感染最准确的方法。它不是一种抗体测试，可以告诉你是否在过去的某个时候患有COVID。

相反，它测试冠状病毒中蛋白质的存在。BinaxNOW品牌将于本周上架，一包两个测试的价格为23.99美元；你应该两个都用，间隔几天。

其他几个品牌应该很快就会上市，包括 [Ellume新冠肺炎家庭测试](https://www.ellumehealth.com/products/consumer-products/covid-home-test) 和 [Quidel QuickVue](https://www.quidel.com/immunoassays/quickvue-sars-antigen-test) 。所有承诺在15分钟内产生结果。CVS称Ellume售价38.99美元，只需要一次测试。

### 这能代替旅行前的测试吗？

不，因为你需要结果的文档，但是有家庭测试*可以*提供文档。例如，BinaxNOW还制作了 [测试，你可以在家里进行](https://www.emed.com/products/covid-at-home-testkit-six-pack) ，但在视频通话过程中，提供商会指导你完成整个过程。六个装的价格是150美元，或者每个测试是25美元。 [Labcorp进行PCR测试](https://www.pixel.labcorp.com/at-home-test-kits/covid-19-test-home-collection-kit#additional-information) 你可以在家里自己取样本然后把拭子邮寄到他们的实验室，得到结果。这一个是119美元，但如果你满足某些要求，公司可以为你的保险收费或使用联邦资金为你免费。
