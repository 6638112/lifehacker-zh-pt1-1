# 关于新的家庭 COVID 测试，您需要了解什么

> 原文：<https://lifehacker.com/what-you-need-to-know-about-the-new-at-home-covid-tests-1846732303>

最近几个月，FDA 已经授权了几项非处方药 COVID 测试，这样你就不必去检测中心检查自己是否患有冠状病毒。从本周开始，你可以 [在沃尔格林、沃尔玛，或者 CVS](https://www.usatoday.com/story/news/health/2021/04/19/covid-home-tests-walmart-cvs-walgreens-binaxnow-ellume/7286667002/) 进行抗原自测。

### 什么是抗原测试？

这不是 PCR 检测，PCR 检测被认为是检测当前感染最准确的方法。它不是一种抗体测试，可以告诉你是否在过去的某个时候患有 COVID。

相反，它测试冠状病毒中蛋白质的存在。BinaxNOW 品牌将于本周上架，一包两个测试的价格为 23.99 美元；你应该两个都用，间隔几天。

其他几个品牌应该很快就会上市，包括 [Ellume 新冠肺炎家庭测试](https://www.ellumehealth.com/products/consumer-products/covid-home-test) 和 [Quidel QuickVue](https://www.quidel.com/immunoassays/quickvue-sars-antigen-test) 。所有承诺在 15 分钟内产生结果。CVS 称 Ellume 售价 38.99 美元，只需要一次测试。

### 这能代替旅行前的测试吗？

不，因为你需要结果的文档，但是有家庭测试*可以*提供文档。例如，BinaxNOW 还制作了 [测试，你可以在家里进行](https://www.emed.com/products/covid-at-home-testkit-six-pack) ，但在视频通话过程中，提供商会指导你完成整个过程。六个装的价格是 150 美元，或者每个测试是 25 美元。 [Labcorp 进行 PCR 测试](https://www.pixel.labcorp.com/at-home-test-kits/covid-19-test-home-collection-kit#additional-information) 你可以在家里自己取样本然后把拭子邮寄到他们的实验室，得到结果。这一个是 119 美元，但如果你满足某些要求，公司可以为你的保险收费或使用联邦资金为你免费。
