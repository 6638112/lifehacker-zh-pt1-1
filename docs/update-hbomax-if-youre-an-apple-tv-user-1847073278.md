# 如果您是Apple TV用户，请更新HBOMax

> 原文:[https://life hacker . com/update-HBO max-if-you-an-apple-TV-user-1847073278](https://lifehacker.com/update-hbomax-if-youre-an-apple-tv-user-1847073278)

Apple TV用户应该立即更新HBOMax应用程序，以修复上周版本更新中引入的一些(但不是全部)错误。

Watch

HBOMax版本5.30.1于6月5日发布，用定制的播放器取代了原生的tvOS视频播放器，这给许多Apple TV用户带来了问题。根据网上众多帖子 的说法，新的定制播放器速度慢，反应迟钝，似乎无法与最新的Siri遥控器配合使用。

幸运的是，HBOMax开发者周二发布了一个紧急补丁，将该应用恢复为原生tvOS视频播放器，应该可以修复播放问题并恢复Siri远程支持——至少对一些用户来说。

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1402484711423975426&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1402484711423975426&autosize=1) 

## 如何在Apple TV上更新HBOMax

对于在Apple TV上启用了自动更新的用户，该补丁应该在后台下载和安装。您也可以前往**设置>应用程序、**手动下载更新，然后选择HBOMax更新。

如果您在添加自定义视频播放器后删除了HBOMax，您可以 [从app store](https://apps.apple.com/us/app/hbo-max-stream-tv-movies/id971265422) 重新下载，使用原生tvOS播放器获得最新版本。

## 还有更多的错误需要修复

安装更新似乎恢复了正确的播放控制，但它不能修复一切。一些用户仍然无法使用Siri遥控器控制HBOMax，其他用户报告在更新后出现了新问题，包括:

*   内容缩略图未出现
*   加载时间长
*   字幕不正确

有报道称iOS和iPad用户也遇到了类似的问题。对于这些挥之不去的错误，我们找不到任何解决方法，但HBOMax团队已经意识到了这些问题，并正在开发更多的补丁。

HBOMax Reddit官方账号要求用户通过HBOMax帮助页面上的 [这种联系形式](https://help.hbomax.com/ContactUs?linkId=120944166) 或者Twitter上的tweet to[@ HBOMaxHelp](https://twitter.com/HBOMaxHelp)提交bug报告。如果支持代表可以帮助解决问题，他们会做出回应，您的报告可以帮助开发人员确定在未来更新中要解决的问题。
