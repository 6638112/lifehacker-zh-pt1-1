# 如何让谷歌助手正确读出你的名字

> 原文：<https://lifehacker.com/how-to-make-google-assistant-pronounce-your-name-correc-1846791120>

谷歌正在更新谷歌助手，使其具有更好的上下文学习功能，这应该会使数字助手更加准确。其中一个变化是能够教谷歌助手如何正确发音——在一点点自我开始的语音训练之后。



在更新之前，Google Assistant 只会根据拼写来读出你喜欢的昵称。如果它不能正确说出你的名字，你可以试着在助手的设置中通过语音拼写名字来帮助它。然而，你现在可以*通过直接和它说话来教它*如何说出你的名字。方法如下:

1.  说出**“嘿谷歌”**或点击设备上的谷歌助手图标来调用谷歌助手，然后点击底部菜单中的设置按钮。(你也可以从安卓的**设置> App &通知>助手>下的【设置】进入这个菜单查看所有助手设置。**
2.  向下滚动并点击**“基本信息”**，然后选择**“昵称”**
3.  轻按**“播放”**即可听到谷歌助手当前的发音。
4.  要使用您的声音进行更改，请选择**“录制您自己的声音”**
5.  轻按**“录音”**，然后大声说出你的名字。Google assistant 会在你说完话后停止录音并保存你说的话。然后它会大声对你说你的名字。
6.  如果你对发音满意，你现在可以退出设置。如果不正确，点击**“记录”**再试一次。

新的语音训练功能正在推出，但可能不会对所有用户开放。与此同时，你仍然可以教谷歌助手用老式的方法念你的名字——或者至少试着这样做。按发音拼出你的名字并不总是能得到准确的 T2 发音，但是如果新的 T5 方法还不可用，或者如果谷歌助手 T6 仍然很难理解你的声音，那么尝试一下 T4 是值得的。

1.  在谷歌助手的设置中，进入**基本信息>昵称。**
2.  选择**“拼出来。”**
3.  按发音打出你的名字。
4.  按下**“播放”**来听听谷歌助手如何发音新的拼写。

新的语音训练还可以教谷歌助手读出谷歌助手的**“你的人”**列表中保存的联系人姓名。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-6Hw6sywPYY0&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-6Hw6sywPYY0&start=0) 

1.  打开谷歌助手的设置。
2.  向下滚动并选择**“你的人民”**
3.  轻按联系人以打开其信息，或选择**“添加联系人”**从您保存的联系人列表中添加某人，或创建新联系人。
4.  编辑联系人信息时，向下滚动到**姓名发音**。
5.  轻按**“录制您自己的”**根据您的声音更改发音，或轻按**“拼读出来”**按发音键入发音。
6.  完成后点击**“保存”**。

除了新的语音训练，谷歌还宣布了一些变化，以帮助 Assistant 识别上下文措辞，并更准确地跟进你最近的问题或命令。你可以在谷歌官方博客 的 [中了解更多关于这些变化和新的语音训练。](https://blog.google/products/assistant/loud-and-clear-ai-improving-assistant-conversations/)
