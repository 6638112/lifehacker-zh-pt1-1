# 为什么不应该在PlayStation Network上给陌生人发信息

> 原文:[https://life hacker . com/why-you-should-never-messages-innovators-on-PlayStation-n-1849423132](https://lifehacker.com/why-you-should-never-message-strangers-on-playstation-n-1849423132)

听到在网上玩游戏的陌生人不一定“友好”，这并不奇怪。任何经历过现代战争游说的人都有战争伤疤可以证明这一点。但是，即使在2022年，巨魔仍然存在，并试图让你的在线游戏体验成为一个活生生的地狱。如果你的本能是回应骚扰者，告诉他们什么是什么，这里有一些建议:**不要**。

Watch

在r/PS4T3上的一个现已删除的帖子中，Reddit用户findingclarityhere讲述了他们在PSN给一群陌生人发消息的故事。根据Redditor的说法，这群人在*红色死亡救赎2* 的游戏中不断骚扰他们，使得这种体验基本上不可玩。就像你在现实生活中可能做的那样，findingclarityhere向这个群体发出了一个信息，实际上是告诉他们走开。

虽然Redditor只是把陌生人称为“失败者”，但这种冒犯显然是不可原谅的:该组织报告称，他们发现clarityhere骚扰了他们，并用这条消息作为证据。几乎是同时，PSN禁止这位Redditor在线玩游戏七天。正义。

虽然故事本身被r/PS4 mods删除了，但下面的讨论非常活跃，读者分享了类似的对PSN巨魔感到沮丧的故事。一位用户在网上讲述了他们的孩子在 *RDR2* 中的经历，以及他每次在游戏中产卵时是如何成为目标并被杀死的。孩子退出了游戏，这可能是每个网络巨魔的目标。其他人谈论得更广泛，说在网络游戏中给陌生人发信息从来都不是一个好主意，其他人发誓永远不玩网络游戏。

这里有几样东西需要解开。首先，我倾向于同意这样一种观点，那就是给试图找你麻烦的陌生人发信息从来都不是一个好主意，无论是在PSN还是在互联网上的其他地方。“不要喂巨魔”是一句有原因的格言:匿名的混蛋会引起任何注意，即使你试图保持冷静。当然，他们的最终目的是让你生气，无论是嘲笑你的愤怒，还是打一张UNO反牌，让你为他们的恶劣行为承担后果。

不幸的是，似乎PSN的政策尤其不宽容:如果他们收到多份针对你的报告，并看到你与这些记者有过任何负面的互动，那就是禁令。因为一个付费用户称另一个玩家为失败者而禁止他一周，这是不是有点极端？大概吧。但是，鉴于在线游戏的绝对毒性，知道索尼如此严肃地采取行动打击骚扰是件好事。

所以，这个故事的寓意是:不要给PSN的陌生人发信息。我的意思是，永远不要给陌生人发信息，尤其是不要在一个会迅速惩罚你而不是巨魔的平台上。毫无疑问:巨魔们敏锐地意识到了PSN在这里的立场，并将利用这一点来对付你。更好的方法是简单地禁止任何骚扰你的人，试着平静地玩你的在线游戏。
