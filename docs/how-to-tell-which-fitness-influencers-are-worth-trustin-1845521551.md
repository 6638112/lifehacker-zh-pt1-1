# 如何辨别哪些健身影响者值得信任

> 原文：<https://lifehacker.com/how-to-tell-which-fitness-influencers-are-worth-trustin-1845521551>

健身 Instagram，像地球上的其他地方或人口一样，包含众多。有努力工作的运动员，有推销他们生意的教练，有谋生的模特。并非所有这些人都是合法的健身专业人士，所以如果你在 Insta 或 YouTube 上寻找锻炼建议，你需要问自己几个问题。



### 这个人在暗示什么但没有说出来？

许多健身影响者的头号谎言是他们没有说出来的:这是一种暗示，无论我卖什么都能让你拥有你在这里看到的身材。

事实是，以健身为生的人在开始销售锻炼计划或补充品之前就已经变得健康了。一旦你有了强健的体魄，你可以在罐子里的锯末上签名，人们会把他们脑海中的点与联系起来，假设你在向他们出售肌肉强化补充剂。如果你有一个瘦的身体，你可以推销一个饮食计划，人们会认为如果他们遵循这个计划，他们就会像你一样被撕碎。

坦白地说:类固醇的使用在那些参加没有经过药物测试的比赛或者根本不参加比赛的运动员中相当普遍。但是因为在美国娱乐使用是非法的，许多人不想谈论它。(当然也有例外: [这里有一篇运动科学家兼健美运动员迈克·伊斯雷尔](https://www.instagram.com/p/CGaaYuXhQ-w) 的前后对比文章，他拐弯抹角地将自己的“正常”年龄与训练生涯中“不正常”的部分进行了比较)

不要忘记，有影响力的人经常使用照片编辑——通过 PPhotoshop、FaceTune、和其他软件——根据需要让自己看起来更苗条(或更有曲线美)。记者 [沈佳凝·默瑟在她的几个颠覆性的小教程中指出](https://www.instagram.com/p/CFADv9QJbfS) ，摆出突出身体某些部位的姿势绝对是一种可以发挥到极致的艺术。哦，还有:如果你正在寻找出售臀部锻炼计划的有影响力的人，请注意 [臀部植入](https://www.hughesplasticsurgery.com/los-angeles-before-after-photos/butt-implants) 是非常重要的一件事。

### 这个人真正的专业领域是什么？

如果有人在销售饮食计划、锻炼计划、或教练服务，他们应该有正式或非正式的证书来支持他们。例如，从艾斯、ACSM、NSCA、或另一个由 NCAA 认可的 [这样的合法机构获得的个人培训认证是一件要找的事情。一些教练或指导员没有正式的证书，但可以指出他们训练过的顶级运动员或可以为他们担保的同行。这些更主观的联系最有效的方法是，你首先根据它们找到教练或指导者:例如，如果你最喜欢的几个力量举重运动员都是由同一个人指导的，那么这个人可能是你很好的建议来源。](https://www.credentialingexcellence.org/nccadirectory)

但是要记住，运动和营养不是一回事。如果有人提供营养方面的建议，检查他们是否是注册营养师或持有类似资格。在许多地方，“营养师”并不是一个法律定义的术语，但注册营养师是受过大量培训并拥有丰富经验的专业人士。

类似于关于身体外貌的不言而喻的谎言，还有另一个类似的谎言:*科学说我是对的*。这一个更难发现，因为基于证据的信息是一个很好的东西，有很多专业人士公开讨论他们的信息来自哪里，以及它如何形成他们的建议。但是也有人决定他们想说什么，然后挑选一些研究让那些信息看起来更合理。

### 这个人是否有一个一致的，有点无聊的信息？

老实说，关于健身的大部分事情都有点乏味。如果你想减肥，你需要 [想办法少吃卡路里](https://lifehacker.com/there-is-no-best-diet-1845385562) 。如果你想锻炼肌肉，你需要逐渐增加重量。如果你想在一项运动上做得更好，你必须 [坚持多年](https://lifehacker.com/consistency-is-the-solution-to-most-of-your-fitness-pro-1845417358) 。

真正的专家——在 Instagram 或其他地方——不会对此保密。他们将讨论保持一致性的挑战，或者就哪些练习可能有助于解决你可能会发现的小问题提供建议。他们可能会分享成功:自己或客户的前后照片，或者在比赛中的巅峰表现。

他们*不会*突然宣布某项锻炼或某类锻炼是终极目标。他们也没有将练习或锻炼作为独立的解决方案。如果你每天选择一种不同的看起来很酷的锻炼，你可能会玩得开心并移动你的身体——这是一个基本的最低目标。但这并不等同于拥有一个能让你更接近目标的计划。任何人都可以张贴一些练习，并将其称为战利品训练或二头肌 gainz 的隐藏秘密，但长期结果来自于你的训练方式。
