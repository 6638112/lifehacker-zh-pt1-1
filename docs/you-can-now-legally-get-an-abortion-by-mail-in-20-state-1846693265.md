# 你现在可以在20个州通过邮件合法堕胎

> 原文:[https://life hacker . com/you-can-now-legal-get-a-that-on-mail-20-state-1846693265](https://lifehacker.com/you-can-now-legally-get-an-abortion-by-mail-in-20-state-1846693265)

几年前， [我们争论过](https://vitals.lifehacker.com/it-s-not-legal-to-get-an-abortion-online-but-maybe-it-1819365519) 药物流产——怀孕早期服用药物导致流产的那种——非常安全，足不出户就可以合法堕胎，也许可以通过像网上医生出诊和邮寄药物包这样的方式在堕胎。现在，这实际上是20个州和华盛顿特区的现实

以前，美国食品和药物管理局要求米非司酮，一种必要的药物，在个人诊所提供，即使你后来会在家里流产。这导致对寻求堕胎的人提出不合理的要求，比如 [不得不驱车数百英里](https://vitals.lifehacker.com/this-tool-shows-how-far-you-would-have-to-drive-to-reac-1802728838) — 可能在两天内(取决于州法律— )拿到药丸。

为了回应疫情对必须亲自去诊所的担忧，美国妇产科医生大会去年向联邦政府请愿，要求暂停亲自去诊所的规定。一系列的法庭案件来来回回，但问题现在解决了:本周，美国食品和药物管理局 [回复](https://twitter.com/ACOGAction/status/1381781110980501512/photo/2) 给ACOG说，“总体调查结果(来自他们审查的研究)似乎没有显示，由于修改了新冠肺炎疫情期间的当面配药要求，药物流产发生的严重安全问题(如出血、异位妊娠或手术干预)有所增加。”

因此，美国食品和药物管理局将“行使执法自由裁量权”，这意味着即使他们在技术上有要求亲自开处方的规定，他们的官方立场是没有人会因为违反它而惹上麻烦。也就是说:米非司酮可以开处方，邮寄给你。

而作为*那*的结果，一家自称 [按需堕胎](https://abortionondemand.org) 的公司推出了第一个大规模的远程健康堕胎服务 [嘉人报道](https://www.marieclaire.com/health-fitness/a36028641/abortion-on-demand-telemedicine-service-launch/) 。该网站还没有接受订单，但它解释说这个过程包括与医生的视频访问，然后一包药物被连夜运送。这项服务之后“会通过短信登记”，你可以联系他们的医生询问后续问题。费用是239美元。

提供该服务的州包括俄勒冈州、华盛顿州、加利福尼亚州、内华达州、蒙大拿州、科罗拉多州、新墨西哥州、明尼苏达州、伊利诺伊州、佐治亚州、马里兰州、弗吉尼亚州、特拉华州、新泽西州、纽约州、佛蒙特州、马萨诸塞州、康涅狄格州、罗德岛州、缅因州和哥伦比亚特区。该公司表示，阿拉斯加州、爱达荷州、怀俄明州、密歇根州、宾夕法尼亚州和新罕布什尔州“即将到来”

一些州管理药物流产的法律比食品和药物管理局执行的法律更加严格，因此按需流产目前仅在T2东南20个州(【T3加特区】)、提供服务，并且仅限于18岁以上和怀孕不超过8周的人。(如果您不符合这些标准，您仍然可以从其他提供者处获得远程医疗堕胎服务；向您当地的诊所咨询，如 [计划生育](https://www.plannedparenthood.org)了解更多信息。)
