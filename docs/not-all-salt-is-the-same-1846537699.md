# 并非所有的盐都是一样的

> 原文:[https://lifehacker.com/not-all-salt-is-the-same-1846537699](https://lifehacker.com/not-all-salt-is-the-same-1846537699)

如果你上过基础化学课，你就会知道我们随便称之为“盐”的物质只是盐的一种。盐只是一种由带正电的离子(阳离子)和带负电的离子(阴离子)组成的化合物。我们通常撒在食物上的盐是氯化钠——如果你是dweeby的话，是NaCl它让食物味道很好。

粉红色的盐、黑色的盐、红色的盐等等都是从微量的、几乎可以忽略不计的矿物质和其他杂质中获得颜色的，或者至少是如果你试图结晶纯氯化钠时会被认为是杂质的东西。是的，很多食盐含有碘，但每克只有45微克(0.000045克)，这对我们来说是不足为虑的。(有些人因为医学原因不吃钠，他们用氯化钾给食物调味，但我从来没试过。)但是，即使在无色的氯化钠领域，盐之间也有一个很大的差异，可以影响你的食物和烹饪:晶体大小。

例如，食盐具有紧密的晶体结构；这些颗粒是微小的、高度有序的立方体。另一方面，像马尔登这样的东西的薄片是薄而杂乱的金字塔。前者比后者更容易放入茶匙，所以如果你用体积来衡量，一对一的互换没有任何意义。这就是为什么注意食谱中指定的盐的种类非常重要。如果食谱需要一汤匙犹太盐，使用一汤匙精制海盐或食盐会导致菜肴调味过度，因为犹太盐的晶体结构比其他两种都大得多。

如果你没有食谱中要求的盐，或者只是想用不同的盐，用质量来衡量。你可以很容易地在网上找到大多数盐，所以只需输入“一茶匙X牌犹太盐有多重？”进入你的浏览器(不同的品牌可能有不同的质量)，记下质量，然后称出你正在使用的盐的量。粉红色、红色和黑色盐中的杂质可能会让你的计算有些混乱，但无论如何这些都更适合完成——可以说，用它们烹饪隐藏了它们的光芒。
