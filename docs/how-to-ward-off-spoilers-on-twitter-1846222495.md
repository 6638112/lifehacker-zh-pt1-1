# 如何避开推特上的剧透

> 原文：<https://lifehacker.com/how-to-ward-off-spoilers-on-twitter-1846222495>

听说旺达·幻视发生了令人大开眼界的变化，你可能会对登录任何社交媒体感到焦虑。我们已经谈论了许多你可以用来在社交媒体 上 [避免剧透的技巧，但是这里有一个新的技巧要加入进来:](https://lifehacker.com/block-spoilers-for-avengers-game-of-thrones-and-star-1834056733) [**隐藏 Twitter 趋势**](https://chrome.google.com/webstore/detail/hide-twitter-trends/lapmncfnibdclongbkleadoicnkhknia/) 。

这是一次相当大的信息清洗——基本上是 Twitter 用户界面右侧的所有内容。然而，这将有助于确保你的 Twitter 用户界面尽可能不被破坏，尽管它不会对你的实际订阅做任何事情。为此，请确保您继续使用我们的建议 [屏蔽与您不想看到的内容相关的关键词](https://twitter.com/settings/muted_keywords) 。你可以永久设置这些屏蔽，或者从现在起 30 天内设置，希望互联网上关于你所关心的任何剧透性话题的讨论到那时会平息。

虽然 Twitter 的趋势通常都很沉闷，但我不会认为你只要不看就可以免受剧透的影响。不过，也有媒体的关键时刻被 Twitter 的“有益”功能毁掉的例子:

所以，不幸的是，你会想把关键字*和*静音，并安装一个浏览器扩展——即使这样，也希望有人不会笨手笨脚地指出“达斯·维德”是卢克的父亲。当然，如果你的媒体对你那么重要，也许在关键时刻远离 Twitter 是最好的。
