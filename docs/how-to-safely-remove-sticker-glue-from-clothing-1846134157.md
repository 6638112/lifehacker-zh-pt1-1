# 如何安全地去除衣服上的不干胶

> 原文：<https://lifehacker.com/how-to-safely-remove-sticker-glue-from-clothing-1846134157>

这是从购买一件可爱的新上衣中吸取快乐的最快方法:你撕掉标有衣服尺寸的长标签，它会留下一个无情的粘性残留物，吸引棉绒和灰尘。接下来你知道的是，在你崭新的衬衫中间有一个变色的长方形。最有效的清除残留物的方法——包括洗甲水——有毁坏衣物的危险。在你让问题变得更糟之前，试试用这些自然的方法来去除那可笑的不干胶。



### 使用温热的白醋

你的第一次尝试应该是用 [温热的、未稀释的蒸馏白醋](https://www.thespruce.com/remove-sticker-residue-from-clothes-upholstery-2147137) 来清洁粘性区域。你不需要加水——未稀释的醋中的醋酸成分更有效，有助于去除污渍。不用担心；酸是醋自然蒸馏过程的副产品，正确使用时不会伤害你的衣服。

将一块清洁布浸在加热的醋中，然后用不干胶粘贴在该区域。当你浸泡该区域时，粘性应该开始消失。一旦该区域完全浸湿，用金属抹刀或黄油刀轻轻刮去残留的贴纸或胶水颗粒。冲洗该区域，并用污渍处理预洗清洁剂覆盖。用去污剂擦洗污渍，然后冲洗。继续擦洗，直到所有残留的胶水都消失。(这一过程的工作量可能会因粘性水平而异。)之后，像往常一样清洗衣物。

### 使用花生酱和洗碗皂

拿一把黄油刀和一勺花生酱(可能不是大块的，以免花生粘在衣服上)。用黄油刀将 [花生酱抹在](https://www.raisingmemories.com/2013/01/tutorial-tuesday-remove-sticker-residue.html)患处。涂上薄薄的一层，刚好能盖住粘点的边缘，并且足够厚，以至于你透过花生酱涂片看不到布料。等待五分钟，让花生酱浸透。然后用旧牙刷或小清洁刷擦洗花生酱区域。擦洗动作会刮掉残留的胶水，这些胶水会被花生酱中的油脂弄松。

一旦不干胶片和胶水消失，尽可能多的擦掉花生酱。在该区域添加洗碗精，然后再次擦洗。 [洗碗皂有助于去除](https://lifehacker.com/use-dish-soap-to-get-grease-stains-out-of-your-clothes-1714088718) 任何残留的粘性，以及花生酱中的油脂。彻底冲洗该区域，并像往常一样清洗衣物。

### 把衣服放进冰箱

要去除合成纤维上的不干胶残留，将物品放入 [冰箱中 45 至 90 分钟](https://www.bhg.com/homekeeping/house-cleaning/tips/how-to-remove-sticker-residue-281474979530492/) 。当顽固的不干胶残留在布料上时，冷冻衣物尤其有用；冷冻后，它们会变硬，你可以把它们摘下来，或者用金属铲把它们刮掉。弄湿粘粘的地方，在剩下的残渣上涂上洗碗皂。用软刷擦洗，然后像平常一样清洗。
