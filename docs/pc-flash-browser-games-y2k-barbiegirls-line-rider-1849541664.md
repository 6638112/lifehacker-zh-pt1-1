# 8款被遗忘的电脑游戏将带你回到互联网最怪异的地方

> 原文:[https://kotaku . com/PC-flash-browser-games-y2k-barbiegirls-line-rider-1849541664](https://kotaku.com/pc-flash-browser-games-y2k-barbiegirls-line-rider-1849541664)

我不喜欢回忆过去。但是我玩的新视频游戏都没有让我感受到我以前在父母电脑上启动的PC和浏览器游戏的效果。这些游戏通常是免费的，有着危险、诱人的腐烂图形和故事，让我的大脑加班加点地思考“游戏”的可能性——2004年，当 *Kotaku* 刚出生时，我才6岁，刚刚掌握跳房子的规则。

最近，不知为什么，两个不同的人向我提到了1996年的教育电脑益智游戏 *Zoombinis* 。通过Instagram、抖音和Twitter，算法让我想起了我玩了一整个晚上的 [布拉茨娃娃改造游戏](https://www.youtube.com/watch?v=Qb7Xw2T6VyE) ，我加入的 [虚拟世界](https://www.tiktok.com/@kayxmulani/video/7135230802501061930?) 那时马克·扎克伯格还在恳求哈佛女孩和他约会。

我想相信这个世界在告诉我一些事情。该是我长大的时候了，该是我回忆起以前情绪太大，坐在椅子上就意味着不沾地的时候做的事情了。让我们一起回到在线视频游戏的花园小径上——这里有八款游戏，你应该试着回忆它们所有美好的、令人毛骨悚然的荣耀。

对我来说，1994年的sidscrolling run-and-gun*蚯蚓吉姆*是典型的 [*高寿命*](https://kotaku.com/rick-morty-high-on-life-gamescom-2022-justin-roiland-1849448674) (它的大部分时间都花在在线 [浏览器模拟器](https://classicreload.com/genesis-earthworm-jim.html) 上，这就是为什么我在这里计算它)。我只能通过想象作为一个聪明的破解蠕虫的某些方面与近二十年来美国对男性气概的解释产生共鸣来解释这一点。我们需要弄清楚。

但是蚯蚓吉姆这个角色没有时间反省。他更喜欢在他空空如也的粉红色头皮周围翻牌并获胜。在一件高科技连体衣赋予他肌肉和知觉之前，他是一只普通的虫子，游戏包括射杀人物 [，如多克十二指肠](https://earthwormjim.fandom.com/wiki/Doc_Duodenum) ，一片黏糊糊的肠道，试图追求可怜的 [公主，她叫什么名字](https://earthwormjim.fandom.com/wiki/Princess_What%27s-Her-Name) ，她是一个太大的女孩以至于不能获得一个真实的名字，以及导航级别 [，如“巴特维尔”](https://earthwormjim.fandom.com/wiki/Buttville) 是的，这是*里克和莫蒂* 1.0被 [其变性、*布莱巴特*——特约创作者](https://www.avclub.com/earthworm-jim-creator-pops-up-to-remind-twitter-of-his-1798264741) 弄得更糟。

不过，它那愚蠢的创造力还是值得记住和讨论的。这就是我们如何找出如何在新媒体中保持蚯蚓吉姆最初的乐趣和吸引力，同时去除它的不良态度，这是文化中不幸存在的东西。Interplay也在努力把 [做成电视剧](https://www.avclub.com/earthworm-jim-cartoon-to-try-to-resurface-all-your-late-1848086911) 。瑞克和莫蒂3.0。

益智游戏*暗割*在我2006年发布后发现的网站上还在运行， [沉迷游戏](https://www.addictinggames.com/funny/dark-cut) 。我对*黑暗切割*有一种病态的迷恋，在那里我扮演一个中世纪的医生进行实验医学，由于我才八岁，这变得更加具有实验性。你必须完成一系列的程序，比如切开疖子，取出箭头，但不能完全杀死你的无脸病人，他在GarageBand合成器庄严的颤音中静静地躺着。

这个游戏将永远是我和我的朋友在沉迷游戏中玩的最后一个游戏。我想我们都被它实际但充满暴力的展示所震撼和排斥，所以我们喜欢建造它。

我现在又玩了一次——我现在并不比2006年更有资格执行截肢手术，但我注意到游戏的加载屏幕提供了这条建议:“在真正的紧急情况下，不要执行任何这些操作，也不要将此游戏作为指南。立即寻求专业帮助。”我很高兴我看到了这个。我正准备把这些水蛭都释放到我脸上。

BarbieGirls 是一个在线虚拟世界，美泰于2007年启动，2011年关闭。你可以免费注册，或者买一个芭比形状的MP3播放器，搭配混搭的衣服，自动成为VIP会员(我想我有 [这个](https://images-na.ssl-images-amazon.com/images/I/41psYTwcwtL.jpg) )。否则，VIP会员费每月5.99美元[2008年](https://www.escapistmagazine.com/im-a-barbie-girl-in-a-barbiegirls-world/) 。

但是VIP给了你游戏的主要功能，购物。虽然这个游戏被宣传为一个创新，但对女孩来说，膨胀的虚拟世界，它的主要吸引力，至少对我来说，是购买可爱的衣服并把它们放到我的瘦瘦的白人女孩头像 上。有趣的是，在2007年， [*Engadget* 称之为](https://www.engadget.com/2007-04-26-barbie-girls-mmo-and-barbie-girl-mp3-player-for-little-nerds-in.html) 该网站获得了“训练中的小书呆子”的MMO，这是一条通往“更‘成熟’的社区，如[ *魔兽世界* ]、*第二人生*”的道路但我对这个游戏的大部分记忆是它是粉红色的。我，可能，曾经，收养了一只钱包大小的狗。

以我成年人的大脑，我会称*芭比女孩*为“训练中的进食障碍”而不是“小书呆子”但是，一个非营利团队目前正在致力于将一个现已死亡11年的网站的版本带回来，在一个他们称为 [*的项目中，该网站包含更多身体、种族和性别。我很兴奋地看到，如何(如果有的话)将一个女孩的虚拟世界从2011年的女性愿景转变为现代愿景会改变它的样子。*](https://www.youtube.com/watch?v=xzG5woJfSlo)

在视频游戏网站的评论区之外，我在网上遇到的最糟糕的事情发生在2004年成立的 [*Stardoll*](https://www.stardoll.com/en/) (或其他装扮游戏网站，如[【GirlsGoGames.com】](https://www.girlsgogames.com/)和[【DressUpGames.com】](https://www.dressupgames.com/))。

据我所知，星尘仍然是它以前的样子。它的首页大部分是2011年左右上传的纸娃娃，大约是我最热衷使用它的时候。当时是一个以名人换装游戏为中心的网络社区。实际上，*芭比女孩*网站T5】与 *Stardoll* 合作，建议用户在网站关闭后加入。

上传的纸娃娃数量给我留下了深刻的印象，我是一个不知道为什么和如何打开电脑的孩子。你可以设计任何人的风格，从*伊凡塞斯*主唱 [艾米·李](https://www.stardoll.com/en/dolls/731/Amy_Lee3.html) 到 [环保主义者戴安·福西](https://www.stardoll.com/en/dolls/1357/Dian_Fossey.html) ，除了装扮游戏，还有化妆游戏，购物广场上挤满了真正的、开创性的2000年的时尚品牌，如谭燕玉、意大利奢侈品牌Miss Sixty和DKNY。也有限量版的“致敬”商店，向已有成就和当时崭露头角的时尚大师致敬，如Giambattista Valli、Roberto Cavalli和Lanvin，所有这些都传达了我对当今时尚的热爱、知识和批评。

但是就像芭比女孩一样，*斯塔尔多尔*显然延续了一种非常独特的女性观念，以及一个伟大的、值得纸娃娃崇拜的女人。瘦瘦的，白人，对购物非常感兴趣。它的社区，我主要通过你可以加入的基于兴趣的论坛来参与，充满了戏剧性，至少对我来说，充满了谎言。有一次我告诉人们，我有一个长得和《吸血鬼日记》中的 [斯特凡一模一样的男朋友](https://the-vampire-diaries-originals-tv-series.fandom.com/wiki/Stefan_Salvatore) ，除了他坐在轮椅上。我还写了《暮光之城》的同人小说，灵感来自2008年的安娜·法瑞丝电影《兔女郎》。一切都搞砸了，我做了些糟糕的事情。13岁很难。

好吧，我收回我说过的话，所有最糟糕的事情都发生在我的Weeworld上，这个2002年的虚拟世界主要由孩子组成，从个人经验来看，可能是恋童癖。 *WeeWorld* 于2017年正式关闭，但在其2010年初的heydey，它与俱乐部国歌生成器 [合作，如杰森·德鲁罗](https://www.youtube.com/watch?v=QGGjI1jNhmo)[贾斯汀比伯](https://www.bsckids.com/2010/03/justin-bieber-launches-weeworld-page/) 。头像是丑陋的2D南方公园拇指。它最吸引人的是它的聊天功能，但是在我试图给自己找一个WeeWorld男朋友的过程中，我最终还是被这个网站吓跑了。

你现在进入了幻灯片的中性怀旧部分。我们从教育益智系列 *Zoombinis* 开始，它开始于1996年，从2002年到2015年处于休眠状态。然后，非营利组织TERC将它的第一部分带到了 [平板电脑、MAC电脑和Windows](https://www.terc.edu/zoombinis/) 。和*蚯蚓吉姆*一样，我最了解这个游戏 [是通过网络浏览器模拟器](https://playclassic.games/games/educational-windows-games-online/logical-journey-of-the-zoombinis/play/) (虽然我也按照PC gods的意图把它放在了CD上)。

Zoombinis对婴儿来说是一个很好的游戏。“Zoombinis”这种生物是多嘴的蓝精灵蓝莓，有头发，婴儿玩家必须面对越来越难的谜题，才能带领他们回到他们的家乡Zoombiniville。即使你不是一个婴儿，也仍然有时间勇敢面对艰难的林间小路和蜿蜒的道路——售价9.99美元 的 *Zoombinis* 可以在Steam】上买到。

1994年的点击式电脑游戏系列，也被称为 [一款网络仿真游戏](https://www.emulatorgames.net/roms/nintendo-ds/freddi-fish-abc-under-the-sea/) 、*弗雷迪鱼*以快乐的金鱼弗雷迪为中心，他也是一名侦探。鱼类侦探。一年级的时候就说得通了。

为了解开谜团，玩家必须探索动画场景，回头看，这些场景显然是从迪士尼1989年的电影《小美人鱼》中剽窃了他们的艺术和风格，但我当时对盗版并不担心。

更有趣的是，正如美联社 [在1999年](https://apnews.com/article/4895f6a328a2f936ccdf3cd5d69ad12e) 指出的，弗雷迪是少数几个电子游戏女主角之一。她的角色设计也是不伦不类、雌雄同体的，这是电子游戏中的女性不需要为了游戏的成功而看起来或行动起来像天使的早期证明，这个想法在2000年代后期被践踏了。为了纪念弗雷迪的无害，你可以在Steam上花34.99美元 购买一套从1994年到2001年 [的*弗雷迪鱼*游戏。](https://store.steampowered.com/app/283940/Freddi_Fish_and_the_Case_of_the_Missing_Kelp_Seeds/)

如果你在2006年在线，你会知道 *Line Rider* ，这是一个可延展的平台，你可以在上面画出线条、斜坡和地点，让平坦的主角滑行。[2008年](https://kotaku.com/line-rider-coming-to-iphone-itouch-5044383) ，当 *Line Rider* 登陆iPhone应用商店时， *Kotaku* 创始人Brian Crecente幸灾乐祸地宣布自己“非常激动”你可以在你的浏览器 中免费玩*线骑士*。
