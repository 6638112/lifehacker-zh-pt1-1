# 这个网站帮助你找到你居住的地方的书籍

> 原文：<https://lifehacker.com/this-site-helps-you-find-books-set-where-you-live-1848873520>

我读了很多科幻小说和幻想小说，因为我喜欢书籍带我去新的世界或想象新的现实，但对于将你与你已经知道的地方联系起来的写作来说，还是有一些值得一提的。作为一个读者，没有什么比拿起一本书，发现故事发生在现实生活中我去过的一个城市更让我高兴的了——或者更好的是，发生在我生活的城市。不幸的是，我目前住在纽约，那里没有书籍。:(

我当然是 JK；大约 75%的现代文学小说发生在布鲁克林或曼哈顿；找到一本以我所在的城市为背景的书，就像闭上眼睛，从巴诺书店的“买一送一”的桌子上随意挑选一个书名一样容易。但是对于那些不幸生活在这个垃圾岛上的人呢？你怎么能找到一本书设置在这些美国的角落？

输入 [全美图书](https://crossword-solver.io/books-set-around-america) ，这是一个由 Crossword-Solver.com 创建的互动工具，允许你输入邮政编码或城市来搜索发生在你附近(或多或少)的书籍。我通过搜索我现在居住的南布鲁克林区的书籍进行了测试，结果相当准确:

在这三个最佳结果中，我已经读过一个，并且可以确认《卡瓦利尔的惊人冒险》的大部分情节都发生在弗拉特布什附近。虽然珍妮·奥菲尔的*投机部*(我还没读过)的地点更一般地说是“布鲁克林城”(没人这么叫它)，但我读过这位作家后来的小说*天气*，并且能够认出她在小说中描述的一所小学——一所离我曾经住过的地方一个街区的学校。

但是这个网站如何处理不发生在 [最受欢迎的文学场景](https://crossword-solver.io/pictures/0eb40540-c171-48dd-be5f-a71fb672a922_03_Books-Around-America_Top-10_Locations_Overall.png?auto=compress,format&rect=0,0,1200,1161&w=800&h=774) 中的书籍呢？我搜索了一些我以前的邮政编码，结果还不错，虽然有点不太准确。例如，在大学期间，我住在芝加哥的罗杰斯公园附近，与这个邮政编码最匹配的是我读过的其他东西——亚当·兰格的《穿越加利福尼亚》非常令人愉快，这是一本非常有地方特色的小说，它的第一页包括我以前每天走过的街道的地图。更广泛的搜索也返回了合理的匹配——我输入了我家乡的邮政编码(在伊利诺伊州的一个半农村地区)，有人向我推荐了以最近的大城市为背景的小说，大约 20 分钟的路程。与任何类似的工具一样，结果只会和生成它们的数据一样好。该网站指出，一切都来自 Goodreads 以书籍为中心的数据库，并不是每本书都在该网站上被如此彻底地分类——但如果你正在寻找一本能让你有家的味道的小说，这是一个开始。
