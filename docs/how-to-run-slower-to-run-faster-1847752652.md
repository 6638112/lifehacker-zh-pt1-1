# 如何跑得更慢才能跑得更快

> 原文:[https://life hacker . com/how-to-run-slow-to-run-fast-1847752652](https://lifehacker.com/how-to-run-slower-to-run-faster-1847752652)

惊人的消息:你可能跑得太快了。我们之前已经讨论过为什么 [你需要跑得更慢](https://lifehacker.com/you-need-to-run-slower-1844984642) (我们知道它听起来违反直觉)，但是要点是你通过跑得*更快*，你通过跑得*更慢*跑得更快。

Watch

但是即使你知道你跑得太快了——正如我一直在 [r/running](https://www.reddit.com/r/running/) 中看到的，在那里许多跑步者努力达到舒适轻松的配速——你可能不确定如何减速到合适的速度。这里有一些让你现在跑得更慢的技巧，这样你以后就可以跑得更快。

## **放弃正在播放的播放列表**

你有没有一个杀手级的播放列表，叫做“动机组合”或者“在马路上狂奔”或者“跑步是痛苦，爱痛苦，为痛苦而生活？”好吧，把那些宣传曲目留到比赛日吧。

在你轻松跑步的时候选择更冷的配乐，这样你就不会被心跳加速的节奏所诱惑。你也可以借此机会赶上你最喜欢的播客或迷失在有声读物中。就我个人而言，不知羞耻的是，我听了很多音乐剧，让我放松的跑步飞逝而过。看:这对我来说很有效，因为讲故事让我全神贯注，而音乐让我的肌肉充满动力。我只要求你不要看我今年包装的Spotify。

## **寻找跑步伙伴**

判断你是否处于最佳“轻松跑”状态的经典经验法则是，你是否能在交谈中不喘不气。确保每个人都同意慢慢来。否则，如果你和我一样，你就有可能把一次友好的慢跑变成一场冷战式的无声竞争。

所以找个朋友，系好鞋带，聊聊天。或者发泄你的孩子。或者推心置腹。尽量避免任何形式的暗恋，因为这样做的目的是降低你的心率。

## **关注自己，远离其他跑步者**

如果你是一个忍不住要加快其他跑步者速度的人，试着在你的跑步路线上找一个步行较少的时间。这些时间会因地区和的不同而不同，但一个安全的赌注是瞄准最早的晨跑者或晚班成员。如果你要在黑暗中运行，一定要做好安全措施。

这个建议也是关于当你把跑步重新定位为一个向内转的时间，而不仅仅是一种锻炼的方式时，精神上的好处。通过关注你自己，你可能会发现更容易将你的心态从艰难的节奏转移到一些高质量的“自我时间”开始慢慢跑，看看你会有什么样的顿悟。

## 忽略这些数字

改变你的跑步心态并不总是容易的，尤其是如果你习惯于记录距离、时间、海拔、消耗的卡路里等指标。考虑 [直觉跑](https://lifehacker.com/try-intuitive-running-to-exercise-without-overthinking-1845749324)，在跑步过程中关闭任何一种配速提醒。如果你打算使用应用程序或小工具来测量轻松跑步中的任何东西， [关注心率](https://www.lifehacker.com.au/2021/04/what-should-my-heart-rate-be-during-exercise/) 。

## **观赏风景**

虽然听起来很老套，但我对跑步的热爱与其说是关于跑步的行为，不如说是跑步带我去的地方(身体上、精神上、精神上，随便你怎么说)。当你不集中精力加快步伐时，你会发现自己能够更充分地欣赏周围的环境。结合散步休息时间，拍几张跑步的照片——我知道我有太多几乎相同的同一棵树的照片。(平心而论，这是一棵非常漂亮的树)。

对于精神和身体上的好处，慢跑是持续给予的礼物。做对了，你会在更长更快的道路上看到回报。
