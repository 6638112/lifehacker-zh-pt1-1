# 把小蒜瓣留作存货

> 原文：<https://lifehacker.com/save-tiny-garlic-cloves-for-stock-1846173496>

我讨厌小蒜瓣。我非常讨厌他们。最让我失望的事情莫过于剥开大蒜头的纸质外皮，却发现它是由许多微小的蒜瓣组成的，而不是数量更少的美丽、肥胖的蒜瓣。

小丁香剥皮很烦人。与较大的丁香不同，用我的刀狠狠地砸一下后，它们的皮不会滑落。它极其顽固地粘在蒜瓣上——你必须采摘、剥皮并祈祷——而且摘下它所需的时间与你得到的大蒜数量不成比例。小小的丁香去死吧。

但是...尽管它们令人难以置信地讨厌，小丁香仍然是大蒜，你不应该扔掉它们。我绝不会告诉你扔掉大蒜，但我也绝不会告诉你浪费宝贵的时间去剥可怜的、令人愤怒的小蒜瓣。地狱里的东西有一个好用处，那就是股票。

用小蒜瓣调味意味着每个人都赢了。不需要剥它们——[皮可以和剩下的丁香](https://lifehacker.com/use-papery-garlic-skins-to-give-broth-big-flavor-1797726954) 一起直接放入高汤锅中——只要把它们捣碎，扔进锅里。如果你当时没有做高汤，把它们扔进你的高汤袋里，冷冻，然后在你做高汤的时候打碎并扔进锅里。我猜你也可以用它们来给油调味——只需将它们炒几分钟，直到它们变香，然后取出，再加入其余的配料——但从热油中挑出微小的蒜瓣对我来说听起来并不有趣。不。我认为那些小东西最好的用途是股票。或者至少，这是最懒惰的用法，对我来说是一样的。
