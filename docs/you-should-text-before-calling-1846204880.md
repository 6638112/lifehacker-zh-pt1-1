# 打电话前你应该发短信

> 原文:[https://life hacker . com/you-should-text-before-calling-1846204880](https://lifehacker.com/you-should-text-before-calling-1846204880)

大多数时候，打电话是不方便的。如果事情不紧急，有更少打扰的沟通方式，拿起电话会觉得像是一种义务，把时间花在不需要你马上注意的谈话上。

这就是为什么——除了一些例外——你应该试着在给某人打电话前给T2发短信。即使这看起来像是让某人注意你的间接途径，但从长远来看，这可能会有助于你和他们的关系。

### 突如其来的电话会让人觉得咄咄逼人

没有什么比一个即兴的电话更让人尖叫“我要求你为我奉献时间”了。即使在这种与世隔绝的时候，人们仍然很忙，不断有数字通知和缩放会议提醒争夺他们的注意力。一个电话暗示了对另一个人时间的直接需求；一条短信把球向前推了一点点，使用一个礼貌的建议，当你得到机会的时候，多说一点。

诚然，人们被束缚在他们的移动设备上，但这并不意味着他们主要用于实际的电话通话。当你给宝宝换尿布或者试图带着你的狗去散步的时候，口袋里会有意想不到的嗡嗡声，这只会增加日常生活的压力。你试图联系的忙人可能会希望把你的电话直接转到语音信箱。

### 人们会忘记回电话

短信也是如此，但打电话感觉更短暂，因此很可能会忘记。至少，一条短信或一封电子邮件的寿命要比T1长。如果它最初被忽略了，它会像一个轻松的提醒一样留在你的收件箱里，提醒你要做的事情。此外，短信明显不如电话重要——它需要收件人的注意力，尽管不是一瞬间的注意。

这可能不是一件好事，但个人技术的便利性让每个人都变得更加懒惰。时间在生活的各个方面都至关重要，所以读一条短信并打一个回复——即使有点晚——听起来比打电话简单得多。

### 你希望有人真的有时间

尽管你的声音通过扬声器过滤，但电话可以促进一些更深入的 互动，这需要你打电话的人更多的体贴。打电话可能比面对面交谈需要更多的注意力——没有面部暗示，也没有共同的视觉体验。从打电话人的角度来看，你在没有适当通知的情况下给别人打电话是在给自己造成伤害。

如果你需要某人的全部注意力，给他们发短信是让他们知道的一种较少打扰的方式。尤其是当你需要讨论一些私人的或重要的事情时，你不能指望有人有时间放下一切专注于你的需求。虽然那很好，但我们不能说我这么幸运。

### 也有例外

显然，有时由于不可预见的极端情况，需要打电话。例如，如果你不得不传达一些不幸的消息，如果你想在谈话中加入人情味，那就应该打个电话。(需要立即响应的紧急情况也属于这一类，显然是)。

对于一些人来说，他们知道谁属于允许在任何时间打电话的小名单。这些是你最亲近的人，你很难被他们打扰。更重要的是，你明白，如果你不能立即开口，这些人会理解的，因为你们表面上已经有了如此牢固的联系。

但是这些人很少，而且在任何情况下，他们都可能会发短信给你，安排以后的谈话。
