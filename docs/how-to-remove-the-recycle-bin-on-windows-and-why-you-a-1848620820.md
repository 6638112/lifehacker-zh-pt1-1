# 如何删除 Windows 上的回收站(以及为什么你应该这么做)

> 原文：<https://lifehacker.com/how-to-remove-the-recycle-bin-on-windows-and-why-you-a-1848620820>

就像 Mac 电脑上的垃圾文件夹一样，回收站似乎是 Windows 中非常重要的一部分。不小心删了东西，总能找回(除非你是 Shift + Delete 的人)。但是是时候质疑在你的 Windows 机器上是否需要一个回收站了，因为如果太频繁，它就会变成另一个文件夹，在那里毫无理由地堆积杂物。我们之前已经向你展示了如何自动清理你的回收站，但是如果你想更进一步，还有一个方法可以移除回收站。

## 为什么你应该扔掉回收站

抛弃回收站并不是每个人都需要的，但是如果你没有在本地存储太多数据，你可能不再需要它了。如今，大多数人都在播放音乐和电影，在线备份照片，并使用谷歌文档或微软 365(后者默认情况下会将 Office 文档保存到 OneDrive)等服务。云存储服务有自己的垃圾文件夹，你可以用它来恢复你不小心删除的任何东西。当你意识到你电脑上的大部分数据已经在网上备份了，回收站的必要性开始变得模糊。

## 删除回收站之前要做什么

当然，做一些改变以确保你已经准备好过渡到一个没有回收站的世界是很重要的。你可能希望将浏览器的下载文件位置更改为 Google Drive/OneDrive 文件夹，而你可以对其他用于下载或保存文件的应用程序进行相同的更改，如 WhatsApp、Telegram、Slack、Photoshop 等等，以确保你在云上有一份数据的副本，以防你意外删除某些内容。

你也应该投资一个计算机备份解决方案。对于大多数人来说，像 [Backblaze](https://www.backblaze.com/) 这样的在线备份工具是最容易上手的。或者，你可以购买一个连接互联网的外部驱动器，如 WD My Cloud Home，并设置自动备份，以最大限度地减少数据丢失的可能性。

## 如何从 Windows 中删除回收站

现在你已经为后回收站世界做好了准备，你可以在你的 Windows 机器的桌面上右击回收站。选择**属性**、**和**，这将列出您计算机上的每个驱动器。如果您想绕过 c 盘上的回收站，请选择它。

在所选位置的**设置下，您应该选择**不要将文件移动到回收站。删除文件后立即删除。** C 舔 **OK** 。**

您可以通过选择计算机上的所有其他驱动器来重复此过程。或者，您可以绕过某些驱动器的回收站，让其他驱动器继续运行。

最后一步是通过右键单击桌面上的空白区域并选择 **个性化**来隐藏回收站。这将在 Windows 上打开一个设置页面。在此窗口中，选择左侧窗格中的**主题**，然后点击**桌面图标设置**。该选项位于页面右侧，在**相关设置**下。在桌面图标设置窗口中，取消选中**回收站**并点击**确定**。这将从您的计算机中删除回收站。自由生活，繁荣发展。
