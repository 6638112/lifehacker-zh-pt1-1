# 最好的沙滩三明治只有三种成分

我讨厌把它交给法国人，但是最好的海滩/远足/野餐三明治是jambon-beurre(火腿黄油)。只有法棍面包、黄油和火腿，非常完美。

在沙滩和大海(或树木和松鼠)中吃东西绝对是一种夏日用餐方式。三明治就是为这样的正餐制作的——毕竟，它们是被设计成在用餐者做其他事情的时候吃的。其他事情最初是赌博，但户外运动的人可以利用三明治的便携性，在森林里徒步旅行，坐在草地上，或躺在沙滩上。所有的三明治都可以这样享用，但是jambon-beurre是目前为止你最好的选择。

## jambon-beurre是完美的沙滩三明治

除了难以置信的味道——我们一会儿就会谈到这一点——这种三明治是一个结构上的奇迹，可以很好地旅行，保持它的形状，永远不会变湿。两种馅料(火腿和黄油)都是疏水性的(拒水)，所以你的面包根本没有水可以吸收。这种缺水也转化为极度的可包装性；由于高脂肪含量和低水分含量，这种三明治将保持可食用——不，令人愉快！—一整天都放在温暖的沙滩包或背包里。

它也非常容易制作。我喜欢muffaletta(或类似的压制三明治)，同样不太可能sog，但那些有太多成分，必须压制过夜。jambon-beurre更容易，更轻松，并且可以在你出门前匆匆拼凑起来。

与muffaletta和它的同类不同，jambon-buerre不含任何小块——没有切碎的生菜，没有橄榄酱，没有滑滑的泡菜。没有东西会滑出，掉出来，或者溅到你的防晒腿上。

这个三明治也是既好吃又实用。咸猪肉和奶油黄油的组合非常法式颓废——法国人非常擅长颓废。使用两种难以置信的美味，丰富的成分意味着你只需要这两种——它们一起工作，创造出咸，肉，奶油般的口感，奇迹般地感觉不太重。

由于原料如此之少，购买高质量的面包、黄油和火腿是有意义的:买一根新鲜的法棍，有漂亮的焦糖色，挤压时面包皮会发出轻微的噼啪声，买一份有*味道*的黄油。我一直在用Kerrygold，但是你上面看到的三明治是用 [鸭糖霜](https://lifehacker.com/this-compound-butter-is-basically-duck-frosting-1838492514) 做的，因为这是我在室温下吃的。老实说，Kerrygold更好，所以选择纯黄油而不是复合黄油。

至于火腿，我喜欢干腌的东西(熏火腿、斯派克火腿、伊比利亚火腿和乡村火腿都很好吃)，尽管巴黎风格，湿腌火腿是传统的。为了达到最佳效果，保持口味简单，避免蜂蜜火腿之类的。

将法式长棍面包横向切片(保持完整，如果需要也可以切成小块)，然后在每一面涂上厚厚的一层黄油，再放上切成薄片的火腿。合上，包好，拿出来，吃完。不要加奶酪(你已经有奶油了)，也不要加蔬菜。这种三明治的美在于它的简单。拥抱它，你会得到一顿完美的户外大餐。
