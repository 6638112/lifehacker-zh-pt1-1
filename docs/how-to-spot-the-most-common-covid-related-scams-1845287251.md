# 如何识别最常见的与 COVID 相关的骗局

> 原文：<https://lifehacker.com/how-to-spot-the-most-common-covid-related-scams-1845287251>

疫情“新常态”的一部分是与 COVID 相关的诈骗案的上升。自今年年初以来，超过 20 万美国人总共损失了大约 1.45 亿美元，根据美国联邦贸易委员会的 T1】。以下是一些让人们措手不及的常见诈骗。 



### **假接触追踪**

如果有人就可能接触 COVID 的问题联系您，请确保联系您的人是合法的接触追踪人员。为州卫生部门工作的接触追踪人员将通过电话、短信或邮件联系，并能够提供他们的姓名、机构和电话号码。他们会问:

*   你的姓名和地址
*   你的出生日期
*   你在特定日期的行踪
*   关于你是否有任何症状的问题

接触示踪剂**永远不会** :

*   要求付款或财务信息。
*   询问您的医疗保险、医疗补助或保险单号码。
*   询问你的社会安全号码。
*   询问你的移民身份。
*   发短信或电子邮件给你链接。
*   威胁你。

### **机器人呼叫**

根据最近的 [调查](https://www.provisionliving.com/node/189) 显示，1/5 的人收到过关于 COVID 的自动呼叫。这些诈骗电话是预先录制的消息，来自声称是来自 IRS 或州卫生部门等政府机构的联系人跟踪者或代理人的人。通常，打电话的人会谎称提供 COVID 治疗、检测或经济援助。这些消息会要求您提供个人信息，或指导您按下手机上的“1”，从而将您转接到一个活生生的骗子。

如果你接到这样的电话，不要按任何键，挂断电话。然后，**在【donotcall.gov】T3 向 FTC 报告该呼叫**。

### **文本中嵌入的链接，或“微笑”**

全国各地的警察部门最近发布了关于短信中恶意链接的 [警告](https://ktla.com/news/local-news/feds-ocsd-warn-of-package-pending-text-message-phishing-scam)——一种被称为“微笑”的网络钓鱼形式最近，这些短信采取了一连串伪造的“ [包裹待运](https://www.consumer.ftc.gov/blog/2020/09/heard-about-waiting-package-phishing-scam) ”递送通知的形式，但骗子也会发送声称是政府工作人员、技术支持、金融机构或联系人跟踪者的短信。

点击一个虚假链接会导致试图获取您的个人信息或登录详细信息，通常是通过伪装成骗子声称代表的机构的虚假登录屏幕。在某些情况下，点击这些链接将导致骗子上传恶意软件到您的设备。

遵循以下提示来避免这种骗局:

*   不要在没有确认来源的情况下点击链接或下载文本或电子邮件中的附件。当心来自未知发件人的邀请。
*   避免电子邮件中提供的登录屏幕。而是打开你的浏览器，直接去网站。
*   虚假登录页面通常包含不寻常的 URL、不起作用的链接或按钮，以及说明中的拼写错误。
*   请务必三思，询问您的个人信息是否合适。
*   忽略并删除那些带有糟糕语法、混乱不一致和不寻常格式的链接的邮件。

FTC 有 [更多提示](https://www.ftc.gov/coronavirus/scams-consumer-advice) 关于如何举报和防止与 COVID 相关的诈骗，但是作为一个通用规则，如果你不确定谁在联系你**不要点击链接或附件**！
