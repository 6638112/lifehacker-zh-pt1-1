# 根研究所 2021:解开对批判种族理论的攻击

> 原文：<https://www.theroot.com/the-root-institute-2021-unpacking-the-attacks-on-criti-1847711634>

围绕 [批判种族理论](https://www.theroot.com/why-white-people-hate-critical-race-theory-explained-1846578811) 的争议似乎已经达到白热化。正如我们在 *The Root* 报道的那样，愤怒的白人父母在学校董事会会议上猛烈抨击所谓的禁忌教学，政客们将这个话题作为党派避雷针，所有这些热情往往没有真正理解什么是——什么不是。

问题的关键是:作为一个国家，我们能否诚实地承认我们的历史，并允许奴隶制的蹂躏今天仍然影响着我们？这只是 CRT 先驱、法律学者金伯利·克伦肖(kimberéCrenshaw)提出的问题之一，她被认为创造了“CRT”一词。

在今年的**根研究所**上，我们感到有必要消除所有的错误信息，将对话集中在健康的现实中。

为了帮助我们发现我们所知道的真相，我们寻求了著名学者布兰妮·库珀教授的帮助。库珀教授是美国作家、活动家和文化评论家，她的研究和工作领域因其娴熟和毫不掩饰地向权力讲述真理的能力而闻名。

在深入探究 CRT 到底代表了什么的过程中，库珀指出，与质疑历史的白人至上主义者让我们相信的相反，儿童完全有能力理解该理论背后的概念:

迈克尔·哈里奥特(Michael Harriot)为《The Root》撰写了大量关于 CRT 的文章，他这样解释了这些教义的重要性:“不理解宪法，就无法理解美国的政治、经济和社会结构。如果不承认这部宪法是由 39 个白人设计的，其中 25 个是奴隶主，就不可能理解这部宪法。因此，对美国的任何合理理解都始于对种族和奴隶制对这个国家的政治、经济和社会结构的影响的批判性考察。”
