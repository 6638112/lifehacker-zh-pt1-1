# 作为新婚夫妇如何谈债务

> 原文:[https://life hacker . com/how-to-talk-about-debt-as-新婚夫妇-1845139396](https://lifehacker.com/how-to-talk-about-debt-as-newlyweds-1845139396)

百分之七十的美国人带着债务结婚，典型的是学生贷款和信用卡，T2，美国消费者新闻与商业频道 。最近的一份 [NerdWallet报告](https://www.nerdwallet.com/blog/loans/student-loans/student-loan-debt) 显示，2018年大学毕业生平均欠下29200美元的学生贷款余额，而 [Experian报告](https://www.experian.com/blogs/ask-experian/consumer-credit-review)2019年平均信用卡余额为6194美元。

问题是，夫妻可能很难谈论这些债务。根据最近CreditCards.com的一项调查，12%的承认对他们的伴侣隐瞒债务，原因显而易见。“关于债务有很多羞耻和沉默，”凯莉·英格布里森说，她是一位理财教练，在 [Chillax Finance](https://www.chillaxfinance.com) 与LGBTQIA+的客户一起工作。有些人可能会避免债务，因为他们担心判断或开始战斗。

但是作为新婚夫妇，讨论这些话题是至关重要的——即使这会让人不舒服——因为这可能会影响你们夫妻的财务状况。这里有一些方法来解决这个问题，一起制定一个回报计划。

### 谈谈你的价值观

英格布里森说，如果你害怕负债，那就从探索你的价值观、你在成长过程中学到的理财经验以及它对你财务状况的影响开始。例如，你的家庭可能有不同的社会经济背景，这可能会影响你的学生贷款水平。“人们经常会有‘啊哈’的时刻，这是解除他们当前处境的一种方式，”她说。

### 对数字要诚实

这一部分可能是最难的:对债务数字保持诚实。“没有什么比一个好的旧电子表格更好的了，”英格布里森说。你可以用电子表格列出每笔债务的金额，包括利率。把数字放在那里可能会更容易讨论接下来的步骤。

不管债务从何而来，英格布里森说要记住债务并不能定义你、你的伴侣或者你们的未来。“你们在一起，这只是你生活的一部分，”她补充道。

### 制定一个实现目标的计划

一旦你公布了数字，你就可以开始谈论回报策略了。根据你的情况，你可以选择独自解决债务问题，也可以选择两个人一起解决。英格布里森说，没有正确或错误的决定。

说到 [债务清偿策略](https://twocents.lifehacker.com/how-to-get-organized-to-pay-off-your-debt-1844083637) ，有几个流行的战术。你可以使用 [债务雪球](https://twocents.lifehacker.com/more-research-shows-that-the-snowball-method-is-the-bes-1790533681) ，先关注你 [最小的债务](https://lifehacker.com/this-spreadsheet-calculates-when-youll-pay-off-debt-wit-1790746417) 。或者你可能更喜欢先还清债务雪崩的最高利率。 [债务暴雪](https://lifehacker.com/conquer-your-debt-with-the-blizzard-payment-method-1686861400) 是另一个结合了两种方法的选项。

不管你有多少债务，英格布里森说，随着时间的推移，有可能取得进展，并达到一个更好的财务状况。

### 依靠自动化的力量

实现目标的最佳方式之一——不管你使用哪种回报策略——是依靠自动化的力量。英格布里森建议为账单和债务、储蓄和零花钱开设不同的账户。

“我最擅长的一个技巧是检查你的雇主是否提供分割直接存款，因为如今大多数雇主确实提供将你的薪水分割到多个账户的能力，”她建议道。

### 重新构建你的债务方法

当你在与债务作斗争时，很容易迷失在羞愧的循环中，但英格布里森说，这可能是建立关系并使之更加牢固的一种方式。通过共同完成目标，你们可能会形成新的纽带，从而更容易应对未来的挑战。“最终，和你的伴侣一起理财是一种亲密的形式，”她说。
