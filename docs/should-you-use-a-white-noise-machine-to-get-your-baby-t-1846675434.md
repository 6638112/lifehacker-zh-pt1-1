# 该不该用白噪音机哄宝宝睡觉？

> 原文：<https://lifehacker.com/should-you-use-a-white-noise-machine-to-get-your-baby-t-1846675434>

对于大多数初为人父母的人来说，睡眠是最重要的。当婴儿睡觉的时候，他们并不总是能够睡着，因为所有的瓶子需要清洗，衣服需要洗，但是如果他们想要休息，婴儿也需要休息一会儿。一些婴儿很早就进入了良好的常规睡眠状态，并一直保持下去。而其他人，嗯，不要。



你拉下让房间变暗的窗帘，你把房间的温度调得恰到好处，你走来走去，蹦蹦跳跳，在地毯上磨出一条路，但婴儿还是睡不着。所以，你可能会想，是时候来点白噪音了。

市场上有许多白噪音和其他声音机器 (和应用程序)让婴儿和孩子入睡，而*保持*睡眠。但是在你听到舒缓的声音之前，有一些事情需要考虑。

### 它们有用吗？

某些声音是自然舒缓、放松的，有助于促进睡眠(例如，雨林的声音或海浪)。但是白噪音，甚至像轻柔的摇篮曲一样的音乐，也可以阻挡可能干扰孩子睡眠的噪音，从外面的城市噪音到在另一个房间玩耍的哥哥姐姐的声音。

很多宝宝在白噪音播放的背景下会更快入睡，并且睡得更久。特别是，习惯于周围有很多噪音或在噪音中睡觉的婴儿可能会发现夜晚的完全寂静是一个更具挑战性的入睡环境。话说回来，其他宝宝根本就不会喜欢白噪音。如果你决定(安全地)尝试一个声音机器，你可以先从一个免费的应用程序开始，以确定它们在白噪音方面是“是”还是“不是”。

### 职业选手

使用白噪音的“好处”是非常明显的，也是非常独特的:如果你的宝宝喜欢白噪音，你将会有更好的睡眠。无论他们需要噪音只是因为他们喜欢它，还是因为它阻挡了警笛声或其他孩子的尖叫声，支持白噪音的婴儿可能会得到更多更好的休息(所以你也可以这样做)。

然而....

### 坏处

即使你正陶醉在更好的睡眠中，如果你不小心，白噪音的使用可能会产生其他更长期的问题。正如 [健康热线报道的](https://www.healthline.com/health/parenting/white-noise-for-babies#TOC_TITLE_HDR_1) :

> 2014 年， [美国儿科学会(AAP)](https://pediatrics.aappublications.org/content/early/2014/02/25/peds.2013-3617) 测试了 14 台专为婴幼儿设计的白噪音机器。他们发现所有的车都超过了 50 分贝的建议噪音限制。
> 
> 除了增加听力问题，研究发现使用白噪音增加了语言和言语发展问题的风险。

另一个要考虑的可能的“坏处”是，如果白噪音效果特别好，他们实际上可能会依赖它入睡。如果你知道他们不会总是有机会接触到声音机器(因为他们必须在日托中心小睡一会儿，去祖父母家过夜，或者你知道你会经常旅行)，你可能要考虑首先不要冒险养成这个习惯。作为一个多年来经常因为这个原因带着有声机器为孩子旅行的人，我会说这是一个轻微的不便(必要时我会在平板电脑上使用一个应用程序)，但我确实担心我们已经对它产生了太多的依赖。

如果你的宝宝在没有额外噪音的情况下睡得很好，真的没有理由在等式中引入声音机器。但是，如果你们都在努力获得充足的睡眠，并且想试一试，请遵循 AAP 研究 中研究人员的建议:将音量保持在机器的最大水平以下，并将机器放置在距离孩子睡觉的地方至少 200 厘米(约 7 英尺)的地方，以保护他们的听力。
