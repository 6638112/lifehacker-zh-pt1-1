## **尼奎尔鸡的凯旋归来**

本周，食品药品监督管理局 [向美国](https://www.fda.gov/consumers/consumer-updates/recipe-danger-social-media-challenges-involving-medicines) 的父母发布了一条紧急信息，警告“最近的社交媒体视频挑战”，“鼓励人们在尼基尔烹饪鸡肉”

FDA说你应该告诉你的孩子不要用非处方药做饭是多么重要。但是我建议家长们完全不用担心这个。相反，要确保你的孩子跟上了他们的作业进度。原因如下:

*   不存在涉及尼奎尔鸡的“社交媒体挑战”。有几个抖音账户对一个相当古老的4Chan视频做出了反应，认为有人在尼奎尔煮鸡很有趣/奇怪。所有的反应都是，基本上，“恶心！”而不是“你应该试试这个”但即使是这些视频现在也从抖音消失了(你可以在推特上找到它们，不过是T8和T9)。)
*   这种非潮流趋势并不新鲜。 [我是在一月份](https://lifehacker.com/the-out-of-touch-adults-guide-to-kid-culture-did-logan-1848361658) 写的，那时候至少五岁了。
*   即使有“挑战”，也很少有人会在尼奎尔费心烹饪一只鸡，我无法想象有人会吃鸡。它尝起来和闻起来都很臭。
*   为了警告一种没有发生的趋势，FDA *创造了*一种趋势。现在数百家，也许数千家 [新闻机构正在报道](https://news.google.com/search?q=Nyquil%20chicken&hl=en-US&gl=US&ceid=US%3Aen) ，大多数接受FDA的虚假框架，认为这是一个“挑战”，人们正在积极参与，无数人正在搜索这个术语。查看 [谷歌趋势](https://trends.google.com/trends/explore?q=Nyquil%20chicken&geo=US) 。右边的大峰值是美国食品和药物管理局发布警告的时间。左边的大尖峰是这个骗局的最后一次趋势。
*   难道食品和药物管理局不应该对肉类或其他东西进行分级，而不是让父母对想象的事情感到紧张不安吗？FDA的人难道不应该聪明到不为这种事情上当吗？

## **说唱奶奶恐吓国家**

琳达·保尔森是一位80岁的女士，她正在竞选犹他州的州参议员。她爱传统家庭，爱上帝，爱枪，爱网上吐槽吧。为了吸引她选区的年轻选民， [鲍尔森本周发布了一段说唱视频](https://www.youtube.com/watch?v=OXyRPEscXCQ&t=3s) 。这远不是孩子们所说的“火”在从MJ的“光滑的罪犯”中翻出的一段缓慢的重复片段中，保尔森丢掉了尴尬的押韵(和非押韵)，这比你过去认为你听过的最糟糕的事情还要糟糕。

当然，它像病毒一样传播开来，自由主义者从每个平台上蜂拥而至，大肆辱骂，一些保守派实际上假装他们认为这很好，或者很有趣，或者至少说没有人应该取笑一个老太太。

我的问题是:由此产生的虚拟T2意味着广告成功了还是失败了？保尔森试图用85岁以下的人都不喜欢的东西来吸引年轻人的选票，她是一个白痴吗？还是她是一个天才，可以吸引她所在地区每个年轻人的注意，即使只是为了拖累她？我们将在11月看到。Paulson的胜利显然被认为是T4极长的机会，但是也许她的“说唱”会颠覆犹他州第12选区的权力结构。

## **中国青少年使用宜家再造一所美国高中**

据Vice的汉学家介绍，中国潮人青少年中最热的潮流，英文称为*【美高峰】*或“美国高中时尚”。此举是张贴你在宜家蓝色储物柜前摆姿势的照片，因为它们看起来有点像高中储物柜。着装要求是改良的私立学校校服——如果你看起来像《绯闻女孩》的临时演员，那你就穿对了。据报道，这些装腔作势者人数众多，已经被中国的宜家禁止销售。中国的年轻人聚集在瑞典的一家商店里，重现以美国为背景的电影中青少年的样子，这是一件非常有趣的事情。有这么多层次。

中国有影响力的人也喜欢穿着美国服装在上海 [好市多](https://www.globaltimes.cn/page/202111/1238767.shtml) 前拍照，在好车旁摆姿势。他们试图唤起洛杉矶的生活方式，但我去过这里的好市多；我保证，来自中国的照片更迷人。

## 本周热门视频:VideogameDunkey的新公司

超过200万人观看了 [Videogamedunkey的](https://www.youtube.com/user/videogamedunkey/videos) 最新YouTube视频，而且它才上传了一天。邓凯不是审查一些炒作新游戏。相反，他宣布成立一家新的游戏发行公司。 [在视频](https://www.youtube.com/watch?v=PEt27Jgp8gs) 中，邓凯吹嘘他作为游戏评论者的记录，并承诺BigMode games将成为独立游戏开发者的“认可印章”，并将“发布一些最好的游戏”好吧——但这不正是每个视频游戏发行商努力做的吗？他们中有谁打算只出版最差的游戏吗？( [此处插入方块艾尼克斯笑话。](https://kotaku.com/babylons-fall-square-enix-gamestop-rpg-pc-1849563429) )

在过去的11年里，VGDunkey通过诚实、深刻和有趣的游戏评论在YouTube上建立了超过700万订户的粉丝群。他为人正直，有玩家影响力，所以也许他*是*建立一家高质量独立游戏发行公司的完美人选。另一方面，历史的垃圾桶里塞满了电影评论家，他们认为，“导演能有多难？”并不是所有的人都是T2·弗朗西斯·特吕弗。

但也许更重要的是:以我的经验来看，对玩视频游戏有最强烈看法的人几乎总是对视频游戏的*业务*一无所知。因此，请在接下来的几个月里继续关注——邓凯即将经历一场非常公开的火之审判。
