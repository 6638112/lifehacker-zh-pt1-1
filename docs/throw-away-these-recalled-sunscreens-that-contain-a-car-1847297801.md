# 扔掉这些召回的含有致癌物质的防晒霜

> 原文：<https://lifehacker.com/throw-away-these-recalled-sunscreens-that-contain-a-car-1847297801>

全国各地的气温持续攀升，人们在户外的时间越来越多——因此，制药巨头强生公司自然会召回防晒产品。具体来说，这是该公司旗下露得清(Neutrogena)和 Aveeno 品牌的五种气溶胶防晒霜， [食品药品监督管理局(FDA)报道](https://www.fda.gov/safety/recalls-market-withdrawals-safety-alerts/johnson-johnson-consumer-inc-issues-voluntary-recall-specific-neutrogenar-and-aveenor-aerosol) 。原因？在强生公司&的内部测试中发现了低水平的致癌物质。以下是需要了解的内容。



## 哪些强生&强生防晒霜正在被召回？

虽然强生&旗下有许多品牌出售防晒霜，但此次自愿召回涉及两个品牌——露得清(Neutrogena)和 Aveeno——的总共五种产品，所有这些产品都是喷雾状的,并分发给全国的零售商。

根据 [约翰逊&约翰逊](https://www.jnj.com/johnson-johnson-consumer-inc-issues-voluntary-recall-of-specific-neutrogena-and-aveeno-aerosol-sunscreen-products-due-to-the-presence-of-benzene)T3】的说法，他们是:

*   露得清沙滩防御气雾剂防晒霜
*   露得清酷干运动喷雾防晒霜
*   露得清隐形日用防御气雾剂防晒霜
*   露得清超轻薄气雾防晒霜
*   AVEENO 保护+刷新气雾防晒霜

## 为什么防晒霜被召回

在进行内部产品测试时，J&J 发现这五种防晒霜中的致癌物苯含量很低。该公司在一份声明中说:“虽然苯不是我们任何防晒产品的成分，但在受影响的气溶胶防晒成品的一些样品中检测到了苯。”“我们正在调查这个问题的原因，仅限于某些气溶胶防晒产品。”

## 如果你已经购买了被召回的防晒霜，该怎么办

如果你家里有任何召回的产品，停止使用防晒霜并扔掉。您可以通过拨打 1-800-458-1673 联系 JJCI 消费者护理中心申请退款。如果对使用召回的防晒霜有任何问题或疑虑，请联系您的医疗保健提供者。

如果您购买并使用了任何防晒霜，并出现了不良反应或质量问题，您可以通过 [在线](http://www.fda.gov/medwatch/report.htm) 、通过 [普通邮件](http://www.fda.gov/MedWatch/getforms.htm) 或传真(1-800-FDA-0178)向 FDA 报告。
