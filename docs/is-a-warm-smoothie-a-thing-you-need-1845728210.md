# 你需要一杯温暖的奶昔吗？

> 原文:[https://life hacker . com/is-a-warm-smoothie-a-this-a-this-you-need-1845728210](https://lifehacker.com/is-a-warm-smoothie-a-thing-you-need-1845728210)

我从来不喜欢喝奶昔。这不是我的风格。我不喜欢喝我的水果(除非是勾兑成鸡尾酒)；我喜欢吃我的水果，我喜欢喝健怡可乐(和鸡尾酒)。这就是说，我可能不是Tasty的热冰沙 的目标观众，尽管我真的不确定喝冰沙的人也是观众。

我不仅不明白什么样的人会喜欢热冰沙，我也不明白它们是什么。这些温暖粘稠混合物的创造者们难道没有听说过汤吗？也许他们生活中缺少的只是一道美味的浓汤。

Tasty有四种不同的热思慕雪配方，乔尔不想尝试其中任何一种。然而，我不会让他放弃味觉测试，因为内容必须流动。但我让他选择了口味。

乔尔选择了 [甘薯姜饼冰沙](https://tasty.co/recipe/sweet-potato-gingerbread-winter-smoothie) ，它(剧透警告)尝起来不像姜饼。它尝起来像是未调味的、未煮过的馅饼馅。我不喜欢它。它需要更多的糖，需要更多的盐；它需要与鸡蛋混合，倒入一个馅饼壳中，烘烤直到变硬。

它还需要一个完整的小红薯——如果我把它和黄油、盐一起捣碎了吃，我会喜欢得多。没有必要这样对待一个红薯。

你尝过热冰沙吗？你知道这样的事情存在吗？你知道吗，实际上有很多温冰沙的配方。我觉得太多了。(因为他们很烂。他们真的很烂。)
