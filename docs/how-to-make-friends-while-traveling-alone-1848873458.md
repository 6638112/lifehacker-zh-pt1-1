# 独自旅行如何交友

> 原文：<https://lifehacker.com/how-to-make-friends-while-traveling-alone-1848873458>

独自旅行可能是一次非常有益的、恢复性的经历。我非常支持独自旅行。我曾在柬埔寨、德国、蒙大拿独自呆过，谁知道中间有多少次中途停留。我不是说我已经在所有这些新的地方“找到了自我”，但我确实已经学会了如何享受自己的陪伴。我也学到了一些认识新朋友的技巧和诀窍。即使在以独处为目标的旅行中，也总会有我想把自己放在那里并结交一些旅行者朋友的时候。

Watch

独自旅行时交朋友听起来并不像是一件难事，但你必须有策略。除了外向和平易近人，你还必须知道如何在合适的场合与陌生人交朋友。即使你不打算建立一生的友谊，在你独自旅行的很多时候，人多安全会有所帮助。如果你想在独自旅行中结识其他旅行者，这里有一些让你快速交到朋友的小贴士。

## 参加团体旅游活动

有组织的团体活动是迄今为止与同路人搭讪的最有效的方式。当然，这听起来是显而易见的，但是有一个聪明的方法可以实现它。在预订任何旧的旅游之前，确保你阅读了评论，看是否有迹象表明该活动有时间与你团队中的其他人进行交谈。在以旅行为中心的网站(如 Expedia 或 Viator)之上，有一个针对旅行的过滤器，可以浏览各种各样的活动。如果你想做一些与人接触的事情，比如有组织的酒吧爬行，这尤其有用。

## 呆在社交环境中

住宿可以成就或毁掉一个假期。如今，酒店比 Airbnb 更受欢迎的原因有很多(例如，可比的成本、有保证的清洁度和更方便的位置)。但是如果你知道你想结识新朋友，你可能会考虑放弃酒店的隐私。

如果你想交朋友——或者至少是酒友——没有什么能比得上旅社的经历。虽然青年旅社总是和年轻的、喝醉的“留学”学生联系在一起，但我也遇到过住在青年旅社的各种年龄和背景的旅行者。

如果你不想过艰苦的公共生活，那么我的建议是预订 Airbnb 或 [Couchsurfing](https://www.couchsurfing.com/) 主机，这些主机标榜自己是社交的和事必躬亲的。如果你选择酒店体验，试着花时间在公共空间，如大堂或用餐区。

## **拿出你的空气囊**

听着，我喜欢好的旅行列表。但是如果你一直戴着耳机，你可能会错过和旅伴搭讪的机会。这个建议不是关于空气舱，而是关于如何让自己看起来平易近人。

## **利用社交媒体**

甚至在你到达目的地之前，就先去见人。脸书群、Reddit 论坛和像 Bumble BFF 这样的友谊应用都是在旅途中找到其他有兴趣交朋友的人的有效途径。如果你觉得这样做安全，你甚至可以在你现有的社交网络上发帖说你"*想从[insert times]* 在[insert place]与朋友(以及朋友的朋友)见面。"

快速提示:特别是对于脸书和 Reddit，除了“旅行者”之外，试着浏览不同的关键词，比如“外国人”或“旅行伙伴”

## 最后的想法

独自旅行的人可能会过于信任他人。记住基本的安全提示，比如要时刻注意你的饮料，永远不要向一个完全陌生的人透露你住在哪里。

同样:尊重陌生人的界限。每当你想和某人交朋友时，确保他们同样热情。你不想成为“*那个阻碍别人寻求孤独的人。*

在一天结束的时候，提醒你自己你没有去任何地方都可以交到朋友。但对你来说值得一试。
