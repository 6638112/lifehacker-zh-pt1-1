# 这个网站可以告诉你是否有人用你的相机拍过照片

> 原文：<https://lifehacker.com/this-site-can-tell-you-if-anyone-else-has-taken-picture-1847724703>

丢失相机(或者，上帝保佑，被盗)的沮丧和绝望是传奇，即使在智能手机相机的时代也是如此。想想看:失去一件昂贵的装备*和*去扬克斯旅行的无价数码纪念品。但是在你对着天空挥舞你的拳头之后，你还能做什么呢？也许吧。 [本网站](https://www.stolencamerafinder.com/) 提供了一条调查的途径，又提供了一线希望。虽然希望渺茫，但总比没有希望好。



它的工作原理是这样的:你将丢失相机拍摄的照片上传到[【stolencamerafinder.com】](http://stolencamerafinder.com)，然后使用相机的序列号(保存在照片的 EXIF 数据中)在互联网上搜索用同一台相机拍摄的其他照片。如果它找到了匹配，你可能会有线索知道你的相机最后去了哪里。

从那里，你可以试着追踪并通过电子邮件联系“新主人”要求归还你的相机，向当局报案，或者自己献身于追捕小偷，疾速追杀风格。

这些选项中没有一个可能导致你的尼康相机被退回，但是 [它在过去](https://www.stolencamerafinder.com/successes.jsp) 中发挥了作用，也许它会帮助你找到终结。也许知道你的相机到底发生了什么是你最大的希望？而 t 他的网站还提供了一个 [世界各地丢失相机的数据库](http://stolencamerafinder.com)，这样你至少会知道你并不孤单。
