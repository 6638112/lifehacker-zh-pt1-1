# 旅行安全吗？

> 原文:[https://lifehacker.com/is-it-safe-to-travel-yet-1846488590](https://lifehacker.com/is-it-safe-to-travel-yet-1846488590)

疫苗越来越容易获得，冠状病毒病例在全国范围内下降。如果你整个冬天都呆在家里，你可能会想，去某个地方度假是否终于可以了。简短的回答是:不尽然。冗长的回答？让我们开始吧。

### 疾控中心仍然建议不要拆散

旅行可以增加你感染新冠肺炎的机会，也可以增加你将它传播给他人的机会。这仍然是事实，只要COVID在社区中传播，情况就会如此。

疾病预防控制中心有一个清单，上面列有你在旅行前要问自己的事情 。如果你正在考虑计划一个，一定要仔细阅读这个列表，并诚实地回答自己的问题。(是的，即使你或者你生活中的其他人接种了疫苗。)其中一些问题包括:

*   你、你的家庭成员或任何你可能拜访的人是高危人群吗？
*   你的旅行计划会让你和其他人亲密接触吗——例如，坐飞机或者和不同居的人共乘一辆车？
*   在旅行前的14天里，你会和你家外面的人有亲密接触吗？这包括在人群中，在餐馆用餐，使用公共交通工具。

如果你对这些问题中的任何一个或者列表上的其他问题回答“是”，疾病预防控制中心说你应该避免不必要的旅行。随着社区传播的减少，他们的建议可能会发生变化，但目前来看，最好还是保持谨慎。

### 如果你接种了疫苗，对你个人来说风险很低

已接种人群可以走访其他已接种人群，可以仔细走访1户未接种人群， [根据近期疾控中心指导](https://lifehacker.com/what-you-can-do-after-youre-vaccinated-according-to-th-1846430407) 。那么乘飞机穿越整个国家去拜访一年没见的家人可以吗？不一定。

首先，请记住，您可能会将病毒传染给他人，这种可能性很小。因此，虽然你不需要在暴露后隔离，但在周围时，你仍应戴上口罩并遵循其他预防措施。

第二，想一想你旅行后会与*和*联系的所有人。如果你去拜访一位未接种疫苗的朋友并出去吃饭，这对*你*、来说可能是安全的，但对你的朋友或餐馆员工来说就不安全了。如果你带着孩子或其他未接种疫苗的家庭成员一起度假，他们很容易感染病毒，并有可能将其传播给他人。

### 旅行不是做不安全事情的借口

你知道这是怎么回事，对吧？什么时候戴口罩，什么情况下应该避免，这些规则在家里和旅行时是一样的。如果你不想在家乡和一群陌生人一起在一个小房间里呆上几个小时，你就不应该在飞机上做同样的事情。

旅行会让我们面临在国内无法面对的风险。两位专家 [最近告诉NPR](https://www.npr.org/sections/health-shots/2021/03/14/975891969/planning-a-spring-break-these-5-tips-can-help-minimize-risk)他们不会去像德克萨斯和佛罗里达这样已经完全放松限制的州旅行；风险太高了。

随着时间的推移，旅行的风险可能会发生变化；当更多的人接种疫苗时，一切都会安全一点。如果你渴望一个“真正的”假期，最好开始谨慎地为今年晚些时候或明年的某件事计划——密切关注取消政策，以防万一。

### 去户外目的地的公路旅行仍然是最安全的

那么这是否意味着你根本不能旅行呢？嗯，看情况。飞到某个地方去和陌生人聚会是完全不明智的。但是，如果你能设法旅行并呆在目的地，而不与家人以外的人近距离接触，你至少有几个选择。

例如，你家的人(或者你家加上另一个人， [，如果你们中有足够多的人接种了](https://lifehacker.com/what-you-can-do-after-youre-vaccinated-according-to-th-1846430407) )可以挤进家里的车，一起去公路旅行。这消除了机场、飞机和拼车的风险。一路上打包零食或叫外卖，这样你就不用在餐馆里浪费时间了。

去哪里？酒店也可以，但更好的是租一个度假屋或住在帐篷里或的小木屋里，在那里你唯一能联系到的人就是那些和你一起旅行的人。外出时，不要去餐馆和博物馆，而是去海滩和公园等户外景点。
