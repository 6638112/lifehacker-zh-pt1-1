# 如果你有一只生病或年老的猫，你应该下载这个免费的工具包

> 原文：<https://lifehacker.com/you-should-download-this-free-toolkit-if-you-have-a-sic-1846956216>

作为人类，我们大多数人都不愿意谈论(甚至思考)死亡，无论是我们自己的死亡，还是失去亲人——包括宠物。虽然人们从来不想想到他们的宠物会出什么事，但这是拥有一个毛茸茸的伴侣不可避免的一个方面。即使你可能*认为*你知道在你的宠物生命终结的时候你要做什么，但让它真的发生完全是另外一回事。

Watch

最近，美国猫科从业者协会(AAFP)发布了一个免费的数字工具包，以帮助兽医更有效地解决涉及猫的临终决定。但是你不一定要成为兽医才能使用它:它包含的许多信息对任何养猫的人都会有帮助。以下是可用的内容和下载方法。

### 卡特彼勒报废工具箱里有什么？

虽然猫科动物临终工具包的某些部分是专门针对兽医的——比如 [施用某些药物](https://catvets.com/end-of-life-toolkit/euthanasia-experience) 让猫更舒服——但它对养猫的人来说可能很有用，他们希望了解更多关于这个过程的信息。(这绝不包括我们所有人。)

然而，总的来说，关于评估猫的生活质量和做出艰难决定 的 [部分可能对养宠物的人有价值。这包括访问一个可下载的问题表](https://catvets.com/end-of-life-toolkit/decision-making) ，帮助评估您的宠物的生活质量，并帮助通知您接下来的步骤。

AAFP 认为 [想到这些场景很可怕，“如果没有这种准备，情感纽带可能会如此强大，眼前的情况如此混乱，以至于护理者可能无法清楚地思考什么对他们心爱的猫最好。”](https://catvets.com/end-of-life-toolkit/decision-making)

### 如何下载工具包

用户可以选择 [在这里](https://catvets.com/public/PDFs/Toolkit/EOL/AAFP-End-of-Life-Toolkit-Full.pdf) 下载整个工具包，或者选择与他们最相关的主题，包括 [决策](https://catvets.com/end-of-life-toolkit/decision-making)[准备程序](https://catvets.com/end-of-life-toolkit/euthanasia-experience) 和做出 [最终安排](https://catvets.com/end-of-life-toolkit/final-arrangements) 。还有一个部分提供了 [额外的阅读材料和资源](https://catvets.com/end-of-life-toolkit/further-reading) 来帮助你完成这个过程。
