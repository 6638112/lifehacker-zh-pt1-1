# 最简单的方法，使任何单锅食谱变得更好

> 原文:[https://life hacker . com/the-easy-way-to-make-any-sheet-pan-recipe-so-much-be-1848543097](https://lifehacker.com/the-easiest-way-to-make-any-sheet-pan-recipe-so-much-be-1848543097)

理论上，对于晚餐做什么这个永恒的问题来说，单锅餐是一个完美的解决方案。它们就像慢炖锅或速食锅食谱一样简单方便，但是需要等待的时间更少(甚至要洗的盘子更少)。但通常情况下，一个在照片上看起来华丽可口的食谱很容易变成一个潮湿、乏味、令人沮丧的烂摊子。以下是如何使任何单锅食谱实际工作。

Watch

当单锅食谱失败时，通常是因为一个显而易见的原因:将蛋白质和蔬菜并列在一起意味着两者都没有机会正确烹饪。蛋白质，尤其是肉类，比大多数蔬菜烹饪时间更长；肉做好的时候，蔬菜已经糊了。(如果你不把肉完全煮熟，蔬菜只会在未煮熟的肉汁里炖。不太好！)另一个问题:这两种食材在烹饪时都会释放出大量的水分。没有足够的空间呼吸，所有多余的水都挂在锅里，确保一切都变得柔软和潮湿——而不是松脆和棕色。

解决方案和问题本身一样简单明了。只用两个平底锅:一个放蛋白质，一个放其他东西。我通常使用两个半尺寸的平底锅，但这不像你需要出去买一套特殊的配套；只要你有两个平底锅，你就有一切你需要的东西来改善任何单锅食物。将各个部分分开，这样你就可以在烹饪时监控每一口锅，并在烹饪完成后将其取出。你可以将各种成分分开食用，也可以将它们混合在一起，然后在食用前将所有成分放回烤箱中烤几分钟——无论哪种都可以。

除了使简单、容易的食谱更可靠之外，这种方法也非常适合做饭。当你用两个(或更多)平底锅时，你可以很容易地把食谱翻两三倍，而不用担心溢出来——或者，更糟的是，得到一大堆不合格的东西。最近，我真的很喜欢在一个锅里用橄榄油、盐、红糖和大蒜粉烤豆腐丁，在另一个锅里放上我所有的蔬菜。当所有的东西都做好了，我把两个锅里的东西放在一个锅里，加入一点美味的液体——比如红烧酱、醋油沙司、蔬菜汤或豆汁——然后放回烤箱再烤5到10分钟。我的小混合物从烤箱里出来，焦糖化，美味可口，烹饪完美。当然，第二个锅意味着又多了一个盘子要洗——但是一旦你掌握了这种方法，你就会忙着吃饭而无暇顾及。
