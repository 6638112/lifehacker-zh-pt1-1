# 如何用卫生纸清理浴缸、瓷砖和灌浆角落的霉菌

在最近的一篇文章中，我们提供了一些从难以清洁的表面 上去除霉菌的选择，比如木头和室内装潢，但是房子里最潮湿的房间怎么办呢？清洁浴室的浴缸和瓷砖可能看起来很简单，但是像灌浆和填缝这样的地方就有点棘手了。别担心，有很多方法可以去除顽固的浴室霉菌，而不需要用力擦洗或重新填缝。

## 霉菌真的很危险吗？

浴室里的霉菌实际上并不像听起来那么危险。在湿气频繁接触表面的空间，霉菌必然会形成，虽然它 [可能对过敏或呼吸系统疾病患者有害](https://www.webmd.com/lung/mold-mildew) ，但小剂量不会立即造成伤害。但是如果你家里有细菌或真菌，你只想在它变成更大的问题之前处理好它。

## 从灌浆和填缝中移除模具

只要有水、擦洗海绵(或旧牙刷)、水和一些漂白剂，清除水泥浆或填缝材料中的少量霉菌是很容易的。如果你愿意，确保你戴上手套和口罩，如果你有浴室窗户，你应该让它开着，以保持房间通风。

首先，你要清理该地区的任何碎片。莫莉女仆 建议在处理模具之前，用扫帚或吸尘器清除任何污垢，并用肥皂和水清洗尽可能多的污垢。

接下来，将1份漂白剂和10份水混合到喷雾瓶中，浸泡成型区域。让溶液静置至少五分钟，然后用海绵或旧牙刷擦洗浴缸的角落和缝隙。

最后，用干净的布冲洗和擦拭，露出你的瓷砖薄浆和嵌缝。如果漂白剂太多，你可以使用 [替代品](https://www.zumper.com/blog/6-ways-to-get-rid-of-moldy-bathroom-walls/) 如醋或双氧水。

首先，你想要一张卫生纸。一条长到足以覆盖受影响区域的带子。旋转长条，使其呈长螺旋状。将卫生纸放在角落，浸泡在漂白剂或以漂白剂为基础的浴室清洁剂中。你需要让溶液静置12个小时来去除霉菌(所以要确保浴室通风良好)。

将溶液放置12个小时似乎是一段很长的时间，但是如果你一大早就开始打扫房间，它会在夜幕降临前完成，或者就让它静置一夜。然后，拿掉弄脏的卫生纸(确保你戴着手套),并在拿纸的同时擦拭该区域。模子会和纸一起脱落。好好冲洗一下，就大功告成了。
