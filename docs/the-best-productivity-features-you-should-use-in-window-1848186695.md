## 捕捉辅助(捕捉布局和捕捉组)

Snap assist是一个功能，可以很容易地将桌面上的多个窗口对齐在一起。Windows 11通过两个新功能改进了这一选项，即快照布局和快照组，这两个功能可以帮助您快速在屏幕上排列窗口，使多任务处理或跨不同应用程序工作变得更容易。

### 对齐布局

使用快照布局，您可以将鼠标悬停在应用程序的最大化按钮上(或按下Win+Z ),以查看不同的潜在窗口位置。例如，您会看到一个双分屏选项，它会将窗口对齐到显示器的一半，一个四分屏选项，它会将窗口对齐到显示器的四分之一，以及其他可用的选项。

如果你将悬停在其中一个面板上，你会看到它变成蓝色，表明Windows已经准备好将你的窗口移动到那个位置。只需点击，窗口就会自动就位，调整大小。您可以使用此功能快速重组屏幕，并将窗口按组对齐。说到这里:

### 快照组

快照组允许您在快照布局中将应用程序分组在一起。创建布局后，您可以将鼠标悬停在任务栏上以查看新创建的快照组，其中包括您用布局组织的应用程序。

最小化或恢复群组也将最小化或恢复群组的所有应用程序，而无需重置您的布局，允许您根据需要在多个布局之间切换。这是一个很好的方法，当你不需要分组的窗口时，可以把它们移开，但是当你需要的时候，可以很快地把它们放回去。

如果你想调整snap的任何设置，你可以在**设置>系统>多任务**中找到所有可用的选项。

## 任务栏中的微软团队集成

如果你使用团队来与同事、员工或客户保持联系，Windows 11的新集成(称为“聊天”)将允许你直接从任务栏打电话和发送消息，使应用程序感觉更像一个内置的通信工具。

若要使用团队聊天或开始视频通话，只需从任务栏上的图标访问该应用程序，或使用Win+C。在这里，您将看到最近的团队活动。要发起视频通话，请点击**与**会面；点击**聊天**，开始文字对话。您可以随时通过点按窗口底部的“打开Microsoft团队”来打开完整的团队应用程序。

## 焦点会议

Windows 11中的时钟应用程序为看似简单的时钟应用程序提供了一系列令人惊讶的功能。

例如，集中会议是一个 [强大的生产力工具](https://blogs.windows.com/windows-insider/2021/08/19/new-clock-app-for-windows-11-with-focus-sessions-rolling-out-to-windows-insiders/) ，它允许你划出一天中的部分来提高生产力。你选择工作的时间，,应用程序会为你提供一个计时器，帮助你记录你想要高效工作的时间。

如果你选择工作45分钟或更长时间，该应用程序还会在你的时间表中自动安排五分钟的休息时间。

该应用程序可以与Spotify连接，这将让你在工作时播放音乐，包括Spotify的Deep Focus播放列表。焦点会话工具还与Microsoft To Do连接，允许您将待办事项列表中的任务链接到焦点会话。您可以直接从时钟应用程序中将这些任务标记为完成。

焦点会议工具甚至可以跟踪你的进度，记录你完成焦点会议的频率和你当前的会议记录。

## 触摸手势

如果你在带触摸屏的电脑上使用Windows 11，比如微软Surface笔记本电脑，你可以利用新的触摸屏手势。

这些手势 [可以让你使用某些动作](https://support.microsoft.com/en-us/windows/touch-gestures-for-windows-a9d28305-4818-a5df-4e2b-e5590f850741#WindowsVersion=Windows_11) 更快地执行任务。事实上，它们模仿了Windows的触控板手势。如果你习惯了这些手势，你可以在触摸屏上复制它们。例如，你可以通过将两个手指放在屏幕上并水平或垂直滑动来快速滚动，或者通过三个手指向下滑动来跳转到桌面。以下是完整列表:

*   **选择一个项目:**点击屏幕
*   **滚动:**将两个手指放在屏幕上，水平或垂直滑动
*   **放大或缩小:**将两个手指放在屏幕上，向内收缩或向外伸展
*   **显示更多命令(如右击):**按住项目
*   **显示所有打开的窗口:**用三个手指在屏幕上向上滑动
*   **显示桌面:**用三个手指在屏幕上向下滑动
*   **切换到最后打开的应用:**用三个手指在屏幕上向左或向右滑动
*   **打开通知中心:**用一个手指从屏幕右边缘向内滑动
*   **查看widgets:** 用一个手指从屏幕左边缘向内滑动
*   **切换桌面:**用四个手指在屏幕上向左或向右滑动
