# 为什么我打喷嚏那么大声？

> 原文：<https://lifehacker.com/why-do-i-sneeze-so-loudly-1845665216>

当一个漂浮的灰尘颗粒或任性的花粉颗粒飘进我的鼻窦时，一种可预见的生理反应就会发生:我通过鼻子发出一声大而刺耳的喷嚏，尖叫着血腥的谋杀。

灯光闪烁，我的猫在一阵听觉混乱中随着鼻潮爆发而奔跑。我老婆觉得我矫情；我经常对她坚持说，我的音爆喷嚏是无意识的，完全超出了身体控制的范围。

但是谁是对的呢？大声打喷嚏的人与更谨慎的人共存，他们的小可爱们有着芭蕾舞演员的优雅和沉着。我的挥鞭式鼻爆裂是我个性的产物，还是我基因构成的残酷后果？

幸运的是，这些问题都很容易回答。让我们揭开一些关于大声打喷嚏的传统智慧的神秘面纱。

### 首先，什么是喷嚏？

这是一个奇妙的问题，也是一个在人类应对更重要的生活挑战时可能被忽视的问题。网站 Live Science 生动地描述了 2010 年的现象:

> 从鼻子发痒开始。一些东西可能是一片灰尘或一点花粉刺激上呼吸道的粘膜层，并使神经末梢发出刺耳的声音。神经向大脑最原始的部分脑干发出信号，脑干立即行动，命令肺部深呼吸。声带突然关闭， [眼睛关闭](https://www.livescience.com/32108-why-do-my-eyes-close-when-i-sneeze.html) 空气从嘴巴和鼻子里爆出来:*阿嚏！*

### 每个喷嚏都不一样

将喷嚏添加到完全特殊的身体功能的万神殿中，因为没有两个喷嚏是完全相同的。它们是个人的怪癖，就像笑、放屁和打嗝一样。

正如神经病学家、精神病学家、芝加哥嗅觉味觉治疗和研究基金会创始人艾伦·赫希在 2013 年告诉 NBC 新闻的:

> 打喷嚏就像笑。有些声音很大，有些声音很轻。和打喷嚏也差不多。从青年时期开始，它听起来往往是一样的。

### 你打喷嚏的方式至少部分是身体上的

在我的生活中，许多人把我的迫击炮弹喷嚏归因于令人讨厌的对关注的需求，但事实证明他们只是部分正确。

正如耳鼻喉科专家、NYU 朗格内医疗中心耳鼻喉科临床副教授 Erich Voigt[在 2017 年](https://www.refinery29.com/en-us/scream-sneezing-loud-reasons) 告诉 Refinery 29:“打喷嚏是清理上呼吸道的非常强烈的反射，通常是自发和无意识的，因此我们无法控制力量。”

Voigt 说，你打喷嚏的大小和次数在某种程度上取决于他们身体和肌肉的大小，但最重要的是，这是压力从鼻子或嘴里释放出来的产物。

医生解释:

> 有些人让喷嚏的力量通过鼻子，而有些人会让力量通过嘴，而有些人是从鼻子和嘴出来的。每种类型都会产生不同的噪音。

打喷嚏、身体和喷嚏量之间的关系还有更多。澳大利亚圣文森和麦考瑞大学医院的理查德·哈维教授去年向澳大利亚广播公司 解释说，“一个人打喷嚏的响度取决于他们的肺活量、大小以及他们屏住呼吸的时间。”

### 打喷嚏也是个性的反映

如果你大声打喷嚏，这是你的诽谤者会抓住。如果你不事先打一个致命的喷嚏，把他们打得灰飞烟灭，他们就会死在这座山上。

正如赫希博士向 NBC 解释的那样:打喷嚏的音量“更多的是一种心理因素，代表了打喷嚏的人潜在的个性或性格结构”。

毕竟，你可能无法控制打喷嚏前吸入的确切空气量，或者憋气多长时间，但你最终可以做出选择，减轻打喷嚏的音量。澳大利亚的哈维博士解释说，嘴最终允许更大的喷嚏，而鼻子往往会抑制噪音:“如果你通过嘴打喷嚏，它会更大，但如果你通过鼻子打喷嚏，它会更湿，更脏。”

也就是说，重要的是不要破坏你的喷嚏。打喷嚏时捂住嘴和鼻子会导致严重的医疗状况，如脑动脉瘤和鼓膜破裂。T3】

所以，如果有人因为打喷嚏太大声而让你不爽，告诉他们去处理。忍受吵闹似乎比要求某人忍住喷嚏要合理得多，因为那样会让他们进医院。
