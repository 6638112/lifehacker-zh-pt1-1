# 我们终于有了儿童COVID疫苗的时间表

> 原文:[https://life hacker . com/we-finally-have-a-timeline-for-kids-covid-vaccines-1847901772](https://lifehacker.com/we-finally-have-a-timeline-for-kids-covid-vaccines-1847901772)

经过近一年的等待和思考，我们终于有了一个暂定的时间表，5至11岁儿童的新冠肺炎疫苗可能会获得批准。白宫今天宣布了一项计划 ，该计划取决于FDA和CDC的决定，如果一切顺利，将于11月3日最终确定。目前还没有为5岁以下儿童接种疫苗的时间表。

Watch

即将做出的决定将涵盖辉瑞公司的镜头，特别是；辉瑞公司已要求美国食品和药物管理局扩大其紧急使用授权(EUA ),以包括5至11岁的儿童。实现这一目标需要几个政府机构的参与，他们按顺序开会，就像这样:

1.  FDA的疫苗顾问小组VRBPAC将于10月26日召开会议，讨论辉瑞的申请，并投票决定该疫苗是否应该授权给5至11岁的儿童。这个会议是公开的，你可以在这里收听。
2.  此后不久，FDA将发布其官方决定。如果顾问团投票通过，食品和药物管理局很可能会批准。
3.  接下来，疾病预防控制中心的疫苗顾问小组，ACIP，开会讨论并投票决定谁应该被推荐接种疫苗。这个会议也是公开的， [网络直播链接就分享到这里](https://www.cdc.gov/vaccines/acip/index.html) 。
4.  不久之后，疾病控制中心提出了他们的官方建议。关注新闻。
5.  根据白宫的说法，“在疾病预防控制中心的最终建议出台后的几天内，我们将准备好开始战斗。”

## 我们什么时候能知道疫苗对儿童是否安全？

这些机构将审查辉瑞提交的安全性和有效性数据，以及其他可能获得的数据。如果对疫苗是否足够安全或足够有效有任何担忧，我们会在这个过程中听到。

需要注意的一个关键时刻将发生在10月26日的会议之前:FDA的工作人员通常会准备一份报告，对提交的数据进行解释，并在VRBPAC会议页面上公开发布。(期待它被上传 [这里](https://www.fda.gov/advisory-committees/advisory-committee-calendar/vaccines-and-related-biological-products-advisory-committee-october-26-2021-meeting-announcement#event-information) ，朝向底部，贴上类似“简报文件- FDA”的标签)主要新闻媒体将对此进行报道，因为FDA对这些数据的处理是关于它是否达标的第一个重要线索。

作为一名家长和健康记者，我会关注这份文件以及对它的反应。如果有危险信号，这可能意味着根据提议的时间表，疫苗不会被批准或推荐。毕竟，11月份的准备只针对*如果*疫苗被判定为安全有效。

万圣节之前，食品和药物管理局可能已经做出决定；决定疫苗是否应该上市的是食品和药物管理局，而疾病预防控制中心就谁应该接种疫苗提出建议。如果FDA认为疫苗看起来不错，疾控中心的阶段应该非常顺利和简单。另一方面，如果FDA的小组担心疫苗在某些人群中的安全性或有效性，CDC可能决定不推荐这些人群使用。如果食品和药物管理局断然拒绝，疾控中心的专家小组可能都懒得开会了。

## 我什么时候可以预约我孩子的疫苗？

根据白宫的 [计划](https://www.whitehouse.gov/briefing-room/statements-releases/2021/10/20/fact-sheet-biden-administration-announces-update-on-operational-planning-for-covid-19-vaccinations-for-kids-ages-5-11/) ，联邦政府已经购买了足够的辉瑞疫苗来供应T4所有2800万5到11岁的儿童

除了成年人已经可以接种疫苗的地方，还会有专门为儿童设立的新疫苗诊所。该计划指出这些将包括:

*   儿科医生办公室
*   儿童医院
*   药房
*   学校和社区诊所
*   社区卫生中心和农村卫生中心

在你的社区留意这些诊所的公告，并考虑询问你孩子的学校或儿科医生他们是否有计划开始接种疫苗。与其他由医疗保健提供商购买的疫苗不同，所有新冠肺炎疫苗目前都由联邦政府支付费用，并分发到授权的疫苗接种点。根据白宫的计划，这些站点将能够在疾病预防控制中心最终建议的“几天内”开始接种疫苗——看起来这可能意味着11月的第一周。
