# 接种疫苗的人应该戴上口罩来抵御 Delta 变异吗？

> 原文：<https://lifehacker.com/should-vaccinated-people-mask-up-against-the-delta-vari-1847212949>

洛杉矶县 [最近要求居民](https://www.latimes.com/california/story/2021-06-28/as-delta-variant-spreads-l-a-county-recommends-everyone-mask-indoors) 再次开始在公共场合戴口罩，原因是导致新冠肺炎病毒的 [Delta 变种](https://lifehacker.com/how-worried-should-we-be-about-the-delta-variant-1847151295) 带来的风险增加。几天后，疾病预防控制中心主任 [说](https://thehill.com/policy/healthcare/560871-cdc-director-vaccinated-people-safe-from-delta-variant-do-not-need-to-wear) 接种疫苗的人仍然是“安全的”，在大多数公共场合仍然不需要戴口罩。谁是对的，为什么是相互矛盾的信息？



## 疫苗对德尔塔病毒仍然非常有效

与旧形式的病毒相比，疫苗对德尔塔病毒的效果可能稍差，但它们仍然有效。早期数据显示，如果您已经完全接种疫苗，辉瑞疫苗对德尔塔 的有效率为 88%(相比之下，早期变种阿尔法的有效率为 93%)。 [Moderna 宣布](https://www.npr.org/sections/coronavirus-live-updates/2021/06/30/1011684609/moderna-says-studies-show-its-vaccine-is-effective-against-the-delta-variant) 实验室测试表明其疫苗可以中和所有变异体，包括 Delta，而 [强生&强生疫苗似乎有 60%的有效性](https://www.nbcboston.com/news/local/pfizer-moderna-jj-vaccines-efficacy-as-delta-variant-concerns-rise/2419162/) 。

疫苗似乎也有可能对严重疾病有效，因此，如果你真的出现突破性病例，很可能比你没有接种疫苗的情况要温和；辉瑞 [在预防住院方面有 96%的效率](https://www.scientificamerican.com/article/how-dangerous-is-the-delta-variant-and-will-it-cause-a-covid-surge-in-the-u-s/) 。

所以，如果你已经接种了疫苗，你基本上还是受到保护的，但是要格外小心是合理的。

## 德尔塔比其他形式的病毒更具传染性

Delta 是一种“令人担忧的变种”，理由很充分:I t 比其他形式的病毒更容易传播，它可能导致更严重的疾病，而且不太可能被疫苗阻止。遗传性是目前最受关注的方面。

在美国，德尔塔病毒几乎占据了所有新感染病例；在美国，这一比例仍然只有 20%。德尔塔可能会成为这里的主要菌株，因为它更容易传播。事实上，我们最近降低了口罩的要求，这可能有助于它的传播，一些社区的低疫苗接种率意味着这些地方可能特别容易受到影响。病毒学家安吉拉·拉斯姆森 [告诉*科学美国人*](https://www.scientificamerican.com/article/how-dangerous-is-the-delta-variant-and-will-it-cause-a-covid-surge-in-the-u-s/) 亚利桑那州和得克萨斯州等地的炎热天气可能会导致更多的人聚集在室内空调空间分享空气，从而导致这些地区的传播增加。

## 更频繁地掩盖可能是一个好主意

根据我们对德尔塔的了解，接种疫苗的人仍然不太可能感染或传播病毒——但这种可能性比香草 COVID 更大。

有些地方，比如医院和机场，要求每个人都戴口罩。(最近我第一次乘飞机旅行，这是我有生以来的第一次；鉴于飞机上挤满了人，我很欣赏面具规则。)如果你接种了疫苗，你可能会想在拥挤的室内区域或其他对你有高风险的地方戴上口罩。我大多数外出都不戴口罩，但我口袋里总会有一个。

这里有一个很有帮助的比喻，来自一则 [*【纽约时报】的*篇](https://www.nytimes.com/2021/06/30/well/live/delta-variant-vaccines-masks.html?smid=tw-share) 上相互矛盾的消息:

> “在这一点上，考虑戴口罩有点像为天气穿衣，”弗吉尼亚理工大学土木和环境工程教授林西·马尔说，他是病毒传播方面的世界领先专家之一。“无论你去哪里，你都需要考虑病例数和疫苗接种率，你将进行什么活动，以及你自己的健康状况。”

在世界上 Delta 是主要变异体的地方，病例很高，许多人没有接种疫苗，口罩可以帮助控制局势。在其他地方，包括美国的部分地区，病例很少，疫苗接种率很高，这意味着如果你接种了疫苗，不带口罩是相对安全的。

如果你没有接种疫苗，继续戴口罩保护自己比以往任何时候都重要。孩子们可能不再需要在学校这样的环境中戴口罩，但如果病例在你所在的地区传播，戴口罩可能是个好主意。口罩还可以帮助免疫系统减弱的人， [有些人可能更容易感染病毒，即使他们已经接种了](https://lifehacker.com/what-the-cdcs-new-mask-rules-mean-for-the-immunocomprom-1847064806) 。
