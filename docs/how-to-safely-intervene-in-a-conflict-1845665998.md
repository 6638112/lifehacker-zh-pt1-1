# 如何安全地介入冲突

> 原文：<https://lifehacker.com/how-to-safely-intervene-in-a-conflict-1845665998>

对于我们这些厌恶冲突的人来说，涉及骚扰或歧视的情况可能更容易避免或忽视，希望威胁会消失或有人会干预。但无论是你朋友群体中偶然的种族主义或微攻击，对陌生人的街头骚扰，还是抗议中的警察暴力，作为旁观者，你都有很多机会站出来说话或介入。



“旁观者有力量，”制止街头骚扰组织 的董事拉尼·肖特洛-林孔说。“这种力量可以用来防止骚扰，在骚扰发生时化解骚扰，并帮助骚扰的受害者最终应对和治愈骚扰经历。”

以下是如何决定何时以及如何介入的方法。

### 观察力敏锐

犹他大学学生健康中心健康教育助理主任詹娜·邓普顿说，决定是否以及如何干预冲突的第一步是注意你周围发生的事情，并补充说，干预取决于我们看到潜在伤害的能力和意愿。

### **优先考虑自己的安全**

在你干预之前，考虑一下对你身体和情绪健康的风险。例如，不建议你直接卷入一场可能会受伤的暴力冲突。

类似地，避免会引发、不安全或只是让人精疲力尽的情况——首先保护好自己。

### 成为一个积极的旁观者

Shotlow-Rincon 说，使用语言和肢体语言的直接干预有助于消除冲突和谴责不可接受的行为。没有对每种情况都“正确”的反应，但是即使你没有经过正式的积极旁观者训练，也有一些基本的工具可以使用。

*   **询问是否需要帮助**。一句简单的“你需要帮助吗？”告诉被骚扰的人你看到了他们面临的情况，并让他们有能力表达他们的需求。
*   **口头回应，不要升级**。侮辱欺负者可能会使情况变得更糟。大声说出具体的行为，而不是攻击个人。
*   转移、分散或扰乱。如果你是旁观者，你可以将注意力从被骚扰的人身上转移开，或者扰乱不良行为的正常化。例如，直接回答骚扰者，比如“你怎么知道我是……”把侮辱转移到自己身上。或者对目标人物发表支持性的评论。
*   **签到**。如果冲突扩散，询问受害者是否需要任何额外的支持。Shotlow-Rincon 说，简单地承认这种经历可以帮助人们感到更安全，不那么孤独。
*   **适当时报告**。并非每种情况都需要向执法部门报告(稍后会有更多相关内容)，但你可以向各种反骚扰组织寻求支持，如 [【瑞恩】](https://www.rainn.org/)[全国家庭暴力热线](https://www.thehotline.org/) 和 [停止街头骚扰](https://hotline.rainn.org/ssh-en) 。如果你正在考虑报警，试试本报告指南 中的 [策略。](http://www.stopstreetharassment.org/strategies/sshlaw-police/)

### 检查您的权限

Shotlow-Rincon 和 Templeton 都表示，种族或性别等特权会影响每一个干预决定。例如，如果你是一个白人，你可以代表一个有色人种利用你的特权。

然而，仅仅因为你从各种特权中受益并不意味着你可以为被骚扰或受害的人决定最佳的行动方案。

“避免扮演救世主的角色，”邓普顿说。"与那些可能正在遭受伤害的人联系，询问你如何提供帮助."

最后，请记住，报警或报告骚扰并不是每种情况下的最佳选择或唯一答案。邓普顿说，要广泛地考虑你能提供什么样的支持——不要假设解决方案包括执法。个人和社区行动也有助于追究骚扰者的责任或阻止骚扰。
