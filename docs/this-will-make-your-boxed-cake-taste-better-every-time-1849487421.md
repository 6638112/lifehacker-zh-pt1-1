# 这会让你的盒装蛋糕每次都更好吃

> 原文：<https://lifehacker.com/this-will-make-your-boxed-cake-taste-better-every-time-1849487421>

像许多厨房起源的故事一样，这个技巧来自需要。我和一个朋友正在为一个生日聚会准备一个 Funfetti 蛋糕，发现我们只有两个鸡蛋，而不是必需的三个。我们想，没有时间去商店，也没有勇气喝一杯酒，最坏的情况会是什么呢？我们省去了鸡蛋，加上我朋友笨拙的油量器，我们烤了一个 bundt 蛋糕。那天，世界上最嫩、碎得最细的 Funfetti 蛋糕出炉了。

盒装蛋糕尝起来有很多人造的味道，但如果你喜欢它的味道——惊人的糖精味道混合，再加上一点仿香草味道——那么你可以把注意力转向真正需要改善的地方:橡胶质地。值得庆幸的是，这个技巧非常简单，不需要任何可疑的苹果酱骗局。盒子背面的说明通常包括添加的成分，如鸡蛋、水和油。(我指的是那些购买邓肯·海因斯(Duncan Hines)、贝蒂·克罗克(Betty Crocker)、皮尔斯伯里(Pillsbury)等品牌的人；如果你使用一些有机的，公平贸易的，无麸质的盒装混合物，我不能保证同样的成功。)只需减去一个全蛋，再加入一杯油，按照盒子背面的说明继续搅拌方法和烘烤时间。

漏掉一个鸡蛋，你就带走了鸡蛋中蛋白质提供的一些结合力——如果鸡蛋太多，最终的蛋糕会变得坚韧而有弹性。不是你的叉子滑过它，就像一个从头开始做的蛋糕，而是蛋糕与你的器皿对抗，反弹回来，露出它坚韧的质地。但是去掉一个，你就带走了足够的弹性蛋白质，使蛋糕变成一个精致、柔软的面包屑。与嚼牛腩相比，它的口感更容易入口即化。这并没有错:盒装蛋糕被设计成对业余面包师来说很容易——有些人可能会过度烘烤蛋糕，没有在还热的时候正确地准备平底锅，提升，分层和霜冻——并且*仍然*而不是分崩离析。鸡蛋成分有助于使蛋糕更耐用。

多加一杯油有两个作用。它从省略的鸡蛋中替换掉面糊中丢失的体积，并返回从省略的蛋黄中丢失的丰富度，外加少量额外的好运。结果是一个令人愉快的潮湿的蛋糕，它刚刚粘在一起，用叉子很容易切开，舌头上一点也不觉得干。你甚至可以一起跳过糖霜。

尽管我不经常做盒装蛋糕，但我绝对会每次都这样做，以获得更自制的质地。结果太好了，不容忽视。然而，我对处理精致的蛋糕很有经验。注意，越嫩意味着越难处理。如果你打算堆叠蛋糕层，或者在上面覆盖一层粘性或坚硬的糖霜，记得一定要先彻底冷却蛋糕层。更好的是，一旦这些层从锅里出来，放在冷却架上，把冷却架放在冰箱或冰柜里一个小时。它们会特别结实，而且破损的风险也小得多。如果你时间紧迫的话，我是邦德蛋糕的超级粉丝。它们很可爱，低调，只要撒上一点糖粉，看起来就很优雅。
