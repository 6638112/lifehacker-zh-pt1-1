*堡垒之夜的*新的“性能”渲染模式简化了游戏中的图形设置，以实现更高的帧率。这个游戏看起来不会很详细，但是如果你的电脑已经很难运行它，帧速率的提高将使*堡垒之夜*玩起来更愉快。

下面是一些 boost 的例子，老款电脑可以在性能模式下体验， [据 Epic:](https://www.epicgames.com/fortnite/en-US/news/fortnite-performance-mode-boosts-pc-fps?sessionInvalidated=true)

*堡垒之夜*会自动推荐使用性能模式，如果它注意到你的 PC 组件在低端，但玩家也可以在游戏的设置菜单中**视频>高级图形>渲染模式**下打开或关闭新模式。

### 在 Epic Games 应用中卸载堡垒之夜的高清纹理

老式个人电脑面临的另一个问题是其庞大的安装尺寸。Epic 最近将 PC 上的磁盘空间要求从 90GBs 降至 31.71GBs，但玩家现在可以选择从 Epics Games 应用程序的安装文件夹中删除*堡垒之夜的*高分辨率纹理。

虽然这将进一步降低视觉质量，但结果是安装尺寸*小得多*。

正如你从截图中看到的，关闭高清纹理将*堡垒之夜*所需的磁盘空间从 31.71GB 降至仅 16.81GB。这几乎是*当前安装大小的一半*，也是该游戏之前在 PC 上要求的 90GB 的一小部分。这给你的硬盘留下了更多的空间，同时也将减少游戏的安装时间，潜在地，减少加载时间。

您可以在 PC 上的 Epic Games 应用程序中卸载*堡垒之夜的*高清纹理:

1.  转到**“库”**选项卡
2.  点击*堡垒之夜*旁边的三点按钮，选择**选项**
3.  在选项屏幕中，取消选中**“高分辨率纹理”旁边的框**
4.  点击**“应用”**，等待 app 更新安装文件夹。

你可以在任何时候通过重新选择“高分辨率纹理”选项来重新安装纹理。

### 让堡垒之夜在旧电脑上玩得更好的其他方法

在性能模式下玩，卸载*堡垒之夜的*高清纹理，会让*堡垒之夜*运行得更好，但提高游戏性能的最好方法是投资一些硬件升级。

然而，这些升级并不一定要以昂贵的新图形处理器或构建全功能游戏装备的形式出现；即使很小的改进也会有所不同。

Epic 建议旧硬件上的用户升级到至少 6GB RAM (8GB 是最好的)，并在内置 SSD 上安装游戏，如果他们的 PC 或笔记本电脑支持的话。这些都是相对较小的升级，可以帮助*堡垒之夜* l 看起来——和跑起来——更好一点。
