# 澄清黄油和酥油的区别

> 原文：<https://lifehacker.com/the-difference-between-clarified-butter-and-ghee-1849143100>

在闷热的夏季，当朝鲜蓟准备被蒸，螃蟹准备被煮的时候，黄油是理想的蘸酱。如果你经常吃这些，你可能会注意到它们都带有一小桶蘸黄油。但是，一个人如何去画黄油，这和做酥油有什么不同吗？



两种黄油的澄清和制作方法非常相似，只有一点小小的区别。脱脂黄油是用小火煨黄油制成的；水起泡沫并沸腾，然后牛奶蛋白开始凝结成白色的小块并沉入锅底。过滤掉这些脂肪，你只剩下纯乳脂(或 99%的乳脂；产量可能会有所不同)。

酥油需要多花几分钟来制作。主要用于印度烹饪(尽管它已被整个 30 人和 Keto 人采纳)，制作酥油的过程几乎与制作脱脂黄油的过程相同，但不是在白色牛奶固体沉入锅底时将其滤出，而是让它们在热黄油中烹饪，就像你在制作棕色黄油一样。然后你过滤出棕色的部分，你就有酥油了。(你也可以用微波炉 做酥油，如果你愿意的话。)

这种额外的烹饪赋予酥油一种“普通”脱脂黄油所没有的坚果味道。由于缺乏水和蛋白质，这两种澄清的黄油在室温下可以保持稳定数月，比未澄清的黄油具有更高的发烟点，并且可以在为难以消化牛奶蛋白质和糖的人烹饪时代替黄油使用。

酥油是印度烹饪中最常用的，而你会看到法国食谱中要求的澄清黄油(尽管，公平地说，酥油是澄清黄油的一种形式)。你可以互换使用——用酥油蘸蟹腿！—但是要注意酥油会给你烘焙/烹饪的任何东西带来令人愉快的坚果味。(这不是坏事。)
