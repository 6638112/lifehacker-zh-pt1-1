不过，尽管Game Pass是一笔好交易，但它也可能会让人不堪重负。在控制台上，你会发现数百个标题唾手可得。鉴于现在的游戏比以往任何时候都大，在任何给定的时间，你的主机上只能有少量的游戏通行证。那么，你应该从哪里开始呢？

在这里，你会发现Xbox Game Pass上目前最好的游戏的概要。一些基本规则:不要指望在这里看到任何微软的tentpoles你的 *Forza* s，你的 *Halo* s，等等——如果你有一台Xbox，你可能很清楚这些游戏的存在，值得一玩。我们进一步保留了这个列表——现在只限于控制台。(Xbox Game Pass for PC的会员可以访问一个类似的列表，其中包括大多数游戏，但也有一些游戏在主机上不可用。)当然，游戏也会定期添加到游戏通行证中，也会定期离开。随着可用性的变化，我们将继续更新此列表。

[*祭袍*](https://kotaku.com/tunic-xbox-game-pass-zelda-dark-souls-fez-indie-impress-1848668747) 最初以一个标准的地牢爬虫的身份出现，但那只是一个掩盖其本质的面具:现代时代最扭曲现实、质疑现实的益智游戏之一。大部分文本是用难以理解的北欧古语言写成的。当你在*外衣*的等距空间中导航时，你慢慢地重新组装了一本 [游戏内指导手册](https://kotaku.com/tunic-the-legend-of-zelda-instruction-manual-pc-xbox-ga-1848685988)——模仿NES游戏的老派印刷手册进行了奇妙的程式化——每一页 [回答一个问题三个以上的](https://kotaku.com/tunic-tips-how-to-get-sword-shield-bombs-xbox-series-x-1848683077) 。你永远不会完全知道发生了什么，但你永远无法摆脱一种持续的感觉，是的，外衣比它所展现的要多，一个等待被发现的可能性源泉。还有:小狐狸。

适合:*菲斯*、*塞尔达*和soulslikes的粉丝。任何想质疑他们所知道的电子游戏工作原理的人。

**不适合:**第一次遇到障碍就跑去谷歌的人。

**大致平均播放时间:** [15小时](https://howlongtobeat.com/game?id=52546#search)

*Skate* 对滑板的作用，[*Shredders*](https://kotaku.com/shredders-xbox-series-x-s-pc-snowboarding-game-pass-foa-1848733449)对单板滑雪的作用。与传统冬季动作运动游戏的专利荒谬性相反，像*骑手共和国*或 *SSX* 、 [*碎纸机*](https://kotaku.com/shredders-review-xbox-series-snowboarding-foam-punch-ea-1848674340) 更接地气一点。一个双拇指控制方案，很容易掌握，但很难掌握，让你控制你的董事会与音高完美的精度。与所谓的“未来旋转”(你旋转的总度数大于当前年份的数字)不同，你更有可能实现平滑的旋转和有计划的翻转，这些小技巧更注重看起来时尚，而不是让数字上升。碎纸机开始实现一个目标——在斜坡上钉上蓝鸟日的寒气——然后踩碎它。

**适合:**任何玩过*滑板*但花时间在大型滑板公园，拼命尝试大于720转的人。

不适合:喜欢高分、长连击和陀螺般旋转的玩家。

**大致平均播放时间:** [5.5小时](https://howlongtobeat.com/game?id=93952) ，还不算你打完故事后乱搞的时间

[*【死亡之门】*](https://kotaku.com/death-s-door-is-as-good-as-everyone-says-1847410512) ，一款关于来世的等距动作游戏，没有一个像素错位。你扮演一只乌鸦，一个机构的普通代理人，负责在死后引导灵魂到他们应该在的地方。战斗是艰难的，当然，但是要调整到你知道每一次失败都是你的错的程度。Boss打架是无情的， [要求你背熟模式](https://kotaku.com/13-things-i-wish-someone-told-me-before-i-started-death-1848117572) ，掸掉身上的灰尘，再来一次(一次又一次)。富有创意的地牢慢慢展开，展现出错综复杂的迷宫。《死亡之门》还附加了一些犀利的文字，对生命、死亡以及两者之间的一切都有着深刻而幽默的描述。

适合:地牢爬虫和动作游戏的粉丝。宿舍里贴着M·C·埃舍尔海报的人。

**不适合:**任何寻找简单出路的人——*死亡之门*提供了相当大的挑战。

**大致平均播放时间:** [11.5小时](https://howlongtobeat.com/game?id=90590)

[](https://kotaku.com/lost-in-random-the-kotaku-review-1847734306)*，2021年令人惊喜的地下宝石之一，以惊人的轻松融合流派。实时动作被回合制战斗打断，这种战斗以甲板建造游戏为特色。当你玩的时候，你会找到卡片——一把剑，一把弓，一个咒语——你可以用它们来定制一副牌。然后在战斗时，你可以冻结战斗并使用一张牌，暂时给予你这张牌的毁灭工具，从而在每场战斗中产生不可预知的变化。这是即时战术和长期战略的完美结合。[](https://kotaku.com/what-lost-in-random-2021-s-most-innovative-game-share-1847917323)*

*适合:任何看过《T2》的人都会想，“这应该是个游戏。”*

***不适合:**没有耐心的玩家，因为*随机迷路*起步有点慢。*

***大致平均播放时间:** [16小时](https://howlongtobeat.com/game?id=71958)*

 *如果你喜欢不断前进，你会喜欢 [*无人拯救世界*](https://kotaku.com/game-pass-drinkbox-action-rpg-zelda-roguelite-guacamele-1848402886) 。自上而下的RPG把你塑造成一个拥有令人印象深刻的变形能力的巫师。起初，你只能接触到基本的形态——游侠，士兵——然后才能解锁更有趣的形态，比如美人鱼，僵尸，鬼魂，甚至是一条畸形的龙。每种形态都有自己的一套战斗通行证式的挑战(“用Z法术杀死X数量的Y个敌人”)，所有这些都会给你XP，这可以解锁新的形态，每个形态都有自己的一套新的挑战。*没人拯救世界*是一个不断看着数字上升的疯狂循环，但没有通常与这种循环相关的微交易包袱。

**一个很好的搭配:**人们渴望与众不同，大约在2012年的独立游戏时代。

**不适合:**任何寻找稳定合作游戏的人；虽然*没人拯救世界*在技术上以合作为特色，但它相当简单。

**大致平均播放时间:** [18.5小时](https://howlongtobeat.com/game?id=90632)

[*远:变幻的潮汐*](https://kotaku.com/far-changing-tides-loan-sails-xbox-game-pass-puzzle-pla-1848660892) 是一款孤独的游戏。侧边滚动的解谜平台打开时是冰冷的，你的角色，沉默而无名，在水下。这个星球似乎已经被一场起源不明的世界末日洪水淹没了。你浮出水面。你穿过淹没在水中的高楼的屋顶。然后你找到一艘蒸汽朋克帆船，沿着海岸驾驶它(大部分)是游戏的主要部分。通过机械但令人愉快的操纵船帆和风的动作，或者搅动引擎的燃料，你致力于向屏幕右侧前进的永恒的视频游戏探索，无论那里有什么。令人惊讶的是，灵巧的环境谜题为你的旅行添加了标点符号，结果是一个隐喻游戏，说了很多，但实际上一个字也没说。

**很适合:**粉丝的*地狱边缘*和里面的*，以及其他诸如此类的游戏。*

**不适合:**寻找精力充沛、心跳加速的侧翻者的人。

**大致平均播放时间:** [5.5小时](https://howlongtobeat.com/game?id=94066)

这款 [南部哥特式的点击冒险](https://kotaku.com/game-pass-norco-review-adventure-southern-gothic-raw-fu-1848726472) 可能只适用于PC，但它在非主机游戏通行证上的存在足以打破我们所有的规则。在你母亲去世后，你回到你的家乡诺科，试图在21世纪南路易斯安那州的魔幻现实主义表演中找到你失踪的哥哥。诺科是新奥尔良郊区的一个真正的城镇，围绕着密西西比河畔的炼油厂而建。在游戏中，这片工业沼泽被赋予了一层黯淡的科幻涂料，一个机器人存在但几乎不被使用的世界，一个一切都被耗尽的世界，一个技术的可能性被抛弃、被毁灭的世界。出色的写作使*诺科*脱颖而出，而其壮观的像素艺术令人回味地呈现了其总体故事中模糊的希望观。

**适合:**寻找故事情节的人。

不适合:讨厌点击游戏的人。

**大致平均播放时间:** [7小时](https://howlongtobeat.com/game?id=105587)

一个关于希腊神的游戏，应该会让你觉得自己是神。 [*哈迪斯*](https://kotaku.com/hades-the-kotaku-review-1845204803)T5】确信无疑。由Supergiant Games开发的等距roguelike将你塑造成扎格列欧斯，哈迪斯之子。你的目标是逃离死亡之地，在希腊来世的各个地区战斗到底。沿着这条路， [你获得了新的力量、武器和战术](https://kotaku.com/tips-for-playing-hades-1845096466) ，并很快到达你像切黄油一样切开地狱阴影的地步。*冥府*有时可能是一种重复的惩罚练习，但它永远不会过时，这要归功于一种非线性叙事，它随着每一次奔跑而前进——无论你是生是死。但是如果*哈迪斯*正在折磨你，有一种模式(字面上叫做“上帝模式”)会让天平向你倾斜。 

**很好的搭配:**喜欢罗格列克斯、动作游戏、令人费解的叙事设计以及构成希腊神话经典的无尽阴谋的人。

**与:**不匹配【未找到信息。]

**大致平均播放时间:** [45.5小时](https://howlongtobeat.com/game?id=62941)

[*《步行者》*](https://kotaku.com/the-pedestrian-is-a-good-platformer-but-great-puzzle-g-1841339177) ，一款益智类平台游戏，在第二名方面并不出色，但绝对是第一名。你扮演路标上的小人。你的目标是从一个标志到下一个标志，连接门、梯子和其他形式的通道，以创建可行的路径。这听起来可能很简单，但是*行人*很快变成了一个看似复杂的益智游戏——它提供最少的说明，完全信任你，每次你解决一个令人抓狂的问题时都会让你觉得自己是个天才。整个事情是由直接来自世纪中叶电影的配乐支撑的，在一个已经可笑的迷人的游戏中增加了魅力。

适合:想被难倒但又不想让*太*难倒的益智游戏迷。街头安全倡导者。

**不适合:**任何寻求*马里奥*级精准平台的人。

**大致平均播放时间:** [4小时](https://howlongtobeat.com/game?id=74727)

地球上没有一款游戏像 [*外太空*](https://kotaku.com/the-brilliant-outer-wilds-breaks-many-of-gamings-unwrit-1839065568) 这样。你最好在完全寒冷的情况下进入，但如果你必须知道一些东西，那么你可以这样做:你 [探索一个不寻常的行星系统](https://kotaku.com/outer-wilds-has-made-exploration-in-other-games-boring-1840148560) ，在那里每个行星都在自己的一套奇怪规则下运行。 [你一颗子弹都不会开](https://kotaku.com/outer-wilds-is-one-of-the-best-games-ive-ever-played-1835801654) 。你不会参与传统意义上的“战斗” [你会死很多次](https://kotaku.com/how-to-get-into-outer-wilds-1835919229) ，但会 [很少会觉得不公平](https://kotaku.com/playing-outer-wilds-in-times-square-was-the-best-way-to-1840775214) 。你不会获得任何经验值，或解锁任何技能、装备或其他有形的进展。如果你没有首先遇到不合时宜的死亡，每场游戏最多持续22分钟。除了你自己的知识，你一无所有——即使是最小的发现也能帮助你更好地理解这个游戏，是的，还有很多等待你去发现。

**非常适合:**好奇心永不满足的游戏玩家。汤姆·克鲁斯的粉丝们希望《T2明日边缘》与其说是一部动作片，不如说是一部对存在意义的深刻思考。

**不适合:**那些喜欢枪支、手榴弹或其他各种暴力小玩意的视频游戏的人。无法应付飞船笨拙控制的人。

**大致平均播放时间:** [21小时](https://howlongtobeat.com/game?id=57527)

在 [*猩红关系*](https://kotaku.com/12-hours-with-scarlet-nexus-bandai-namco-s-new-action-1847156189) 中，你可以 [使用心灵异能](https://kotaku.com/tips-for-playing-scarlet-nexus-1847174868) 将一辆中型轿车扔向一个异次元蘑菇僵尸。仅仅根据它的动作(拜托，你刚刚读了前面那句台词吗？)， *Scarlet Nexus* 值得一试，但RPG的秘制酱是一个引人入胜、一贯惊人的故事。两个主角——一所军事学院的学生训练孩子们去战斗，你知道，那些蘑菇僵尸——标题 [一个双重叙事的故事](https://kotaku.com/scarlet-nexus-should-you-play-as-yuito-or-kasane-1847172196) 在关键的时间间隔交叉路径。曲折来得又快又激烈，即使没有必要把两个故事情节都玩一遍，但如果你想看完整个故事，你就必须这么做。谢天谢地，两个情节都足够吸引人，让你想看。

适合:万代纳木错风格相似的*故事*系列的粉丝。

不适合:很快厌倦重复使用地形的人，因为游戏会让你回到两个故事情节中的特定地点。

**大致平均播放时间:** [34.5小时](https://howlongtobeat.com/game?id=77921)

*拆包*是一个无与伦比的感人故事，以益智游戏的形式包装。你扮演一个无名的主角，在她生活中的不同阶段搬家——宿舍、起始公寓、第一个同居伴侣等等——每个阶段都呈现为等轴立体布景。简单地说，你的目标是把物品从盒子里拿出来，放在它们通常的地方(浴室里有牙刷，厨房里有银器)。但是这些项目，总的来说，告诉你在她的生活中发生了什么:新的人，新的问题，随着生命的每个阶段飞进飞出她的范围。

**适合:**思想保守的玩家。任何人都在寻找一个短暂的，舒缓的游戏。

不适合:不能处理暧昧故事的人。人们寻找令人头疼的难题；*拆包*其实不是那么回事。

**大致平均播放时间:** [4小时](https://howlongtobeat.com/game?id=69666)

[*男朋友地下城*](https://kotaku.com/boyfriend-dungeons-great-story-wouldnt-work-without-its-1847607513) 正如铁皮上写的那样:一个约会sim，扮演双重角色地下城爬虫。你可以和大约六个人约会，所有人都可以变成各种各样的武器——剑、矛、指关节之类的东西。然后你挥舞着这些武器穿过地牢，在那里你的恐惧以恶魔的形式出现。你使用武器越多，你与他们非武器形态的联系就越紧密。男朋友地牢听起来像是那种构思在酒吧餐巾纸背面的游戏，但它尽可能地利用这一点。

**很好的搭配:**约会模拟人生和地牢爬虫的粉丝。

**不适合:**那些寻找残酷惩罚动作游戏的人；*男友地牢*并不十分艰难或复杂。

**大致平均播放时间:** [8小时](https://howlongtobeat.com/game?id=76606)

[*破冰者*](https://kotaku.com/diablo-meets-minecraft-in-this-game-pass-hidden-gem-1848152135) 听起来像是一堆乱七八糟的想法。它结合了*暗黑*自上而下的动作-RPG战斗与基地建造和科幻探索。它甚至加入了技能树、工艺和能量管理。在纸面上，这听起来像是一个大杂烩，但*裂缝破坏者*实际上是一个制作精良的游戏，毫不费力地结合了所有这些功能(甚至更多！)变成一颗难以放下的宝石。另外:如果你在玩播客的时候一直在寻找新的游戏，那就去看看 *The Riftbreaker* 。这也是完美的。

**非常适合:**渴望拥有基础元素的坚固RTS的人。机甲战士。

**不适合:**希望进行回合制机甲策略游戏的玩家，*进入突破口*。

**大致平均播放时间:** [36.5小时](https://howlongtobeat.com/game?id=80159)

[*八道旅行者*](https://kotaku.com/octopath-traveler-the-kotaku-review-1827537100) 既是一种倒退，也是一种启示。这款由Square Enix开发的回合制角色扮演游戏回归了这一类型的经典。但从视觉上看，这是一部现代杰作，展现了现在新诞生的“2D高清”风格，这种风格定义了一些现代Square Enix游戏(*勇敢地默认2* 、*三角形策略*)。这个故事分为八个部分，由八个主要角色组成，很难跟踪，但每个主角 [都有自己的特殊能力](https://kotaku.com/tips-for-playing-octopath-traveler-1827586581) ，这使得RPG从一开始就一直引人入胜。

**适合:**寻找坚固可靠的回合制角色扮演游戏的玩家。

**不适合:**不想开始一个可能需要几十个小时才能完成的游戏的人。
