# 我们仍然不知道学校里有多少COVID病例

> 原文:[https://life hacker . com/we-still-not-know-how-know-how-covid-cases-is-in-schools-1845158873](https://lifehacker.com/we-still-dont-know-how-many-covid-cases-are-in-schools-1845158873)

在疫情期间将儿童送回面对面的学校是一项史无前例的实验。结果可能是一场灾难；如果 [所有的学校再次](https://offspring.lifehacker.com/schools-will-close-again-1844604561) 关闭，我们这里没有人会感到惊讶。但是现在看来，我们甚至可能不知道疫情是否以及何时在学校传播。

Watch

《纽约时报》报道 只有12个州在学校或地区公开了艾滋病病例的信息。一些州只在全州范围内公布这些数据，而另一些州根本不跟踪这些数据。所以我们正在我们的孩子身上进行一项巨大的实验，而我们甚至不知道它是否会失败？

问题是多层次的。首先，在联邦层面上没有一致的疫情回应或数据跟踪。NYT正在尽最大努力收集零碎数据，国家教育协会也是如此，它已经从志愿者提交的新闻故事中 [建立了自己的追踪器](https://vitals.lifehacker.com/how-to-track-covid-19-outbreaks-in-local-schools-1844917259) 。

第二，大多数学校不要求监督测试，这可能有助于他们了解有多少学生或员工患有COVID。 [很多考点不会考孩子](https://lifehacker.com/figure-out-now-where-to-get-your-child-tested-for-covid-1845012797) ，也不是每个家长都花得起小时或者天去找一个会的。我敢打赌，只有极小一部分发烧和咳嗽的孩子会接受检查，他们的父母假设或希望他们可能只是感冒了。

还有一个事实是，孩子患有COVID的父母可能会陷入困境。如果你要去上班，又不想把孩子交给奶奶，你会怎么做？威斯康辛州 [的一名卫生官员告诉当地](https://www.wisn.com/article/health-officer-parents-sending-sick-symptomatic-kids-to-school-coronavirus/34117451#) 新闻站，她“做梦也没想到”父母会故意把他们的艾滋病毒阳性孩子送到学校——但他们确实这么做了。

不收集和公布学校中COVID病例的数据从多个层面来看也是一种失败。我们不总是知道哪些学校爆发了疫情，或者如果我们发现了，可能是在游戏的很晚时候。没有这些可供学习的信息，其他地区和州将无法从他们正在进行的实验中学到东西。例如，混合教学是否减少了传播COVID的机会，或者是否由于父母必须拼凑的多个儿童保育选项而增加了风险？学校中的COVID爆发是否会引发社区中更大规模的爆发？

遗憾的是，我们没有这些信息，也没有机会利用这些信息来做出更好的决定，决定我们应该如何以及何时将孩子送回学校。我们信任的负责我们安全的人 [又一次没有保护我们的安全](https://vitals.lifehacker.com/blame-the-people-in-charge-1845109900) ，也没有提供足够的信息让我们做出明智的决定来保护我们的安全。
