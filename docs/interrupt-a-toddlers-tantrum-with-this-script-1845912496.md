# 用这个脚本打断一个蹒跚学步的孩子的发脾气

> 原文：<https://lifehacker.com/interrupt-a-toddlers-tantrum-with-this-script-1845912496>

如果你有一个蹒跚学步的孩子，而那个蹒跚学步的孩子在发脾气，你知道没有一个万无一失的方法每次都能让他们摆脱出来。蹒跚学步的孩子发脾气是世界上最纯粹的发泄沮丧的方式，逻辑在这些兽性的情感展示中没有位置。然而，偶尔，父母能够设计出一种策略，既能在孩子发脾气的时候安抚孩子*又能让*找到他们到底在烦恼什么的根源。而这就是 [这位老妈](https://www.tiktok.com/@bareyoursoul/video/6906243476166348037?_d=secCgsIARCbDRgBIAIoARI%2BCjxxLgYDqs%2FJY4yoGQQX1Jqjr18NNFiafrRKuz%2Bev6OqnjP%2BzO2QDurZRLkJOOWOnFJ5UZrWBIBLxk8DkSkaAA%3D%3D&language=en&preview_pb=0&sec_user_id=MS4wLjABAAAA-urxM-eTO3o8kR1ItylBD6a_tG6hhnYeYQq7vAjNlWDwmNAEzPWJxl_Su2Vq-eWo&share_item_id=6906243476166348037&share_link_id=8418F7F0-C6A7-4DD9-BFC5-A7A955F40276&timestamp=1608240682&tt_from=copy&u_code=d7mej85ij7dc28&user_id=6727451730230952965&utm_campaign=client_share&utm_medium=ios&utm_source=copy&source=h5_m) ，在抖音身上，已经成就了。

重要的是要认识到，当视频开始时，这个可爱的蹒跚学步的孩子还没有完全失控地尖叫和捶击地板，但这种方法很可能会阻止它到达那一步。当试图阻止一次发脾气时，预防和早期干预总是你最好的选择。

让我们一步一步地分析凯的所作所为:

1.她用她最甜美的妈妈的声音，让他坐下来深呼吸四次，这显然是她让他练习过很多次的事情——正如她所说，从他两岁开始。这是开始实施这种实践的最佳时机。

2.接下来，她让他“摆脱它”，这有助于满足他的身体在那一刻释放情绪的渴望。

3.然后她说，“倾听的耳朵”，他们每个人都指着一只耳朵，表示他们在关注对方并在倾听。这是绝对关键的，因为一个蹒跚学步的孩子的大部分挫折感来自于感觉他们不能交流他们想要什么或需要什么。知道他妈妈全神贯注于找出他不开心的原因是成功的一半。

4.一旦他进行眼神交流并指着他的耳朵，她说，“告诉妈妈怎么了，”他立刻跳起来指出他垂涎的 [米奇金鱼](https://www.target.com/p/pepperidge-farm-goldfish-special-edition-disney-mickey-mouse-cheddar-crackers-6-6oz/-/A-53470412?ref=tgt_adv_XS000000&AFID=google_pla_df&fndsrc=tgtao&DFA=71700000012732781&CPNG=PLA_Grocery%2BShopping_Local&adgroup=SC_Grocery&LID=700000001170770pgs&LNM=PRODUCT_GROUP&network=g&device=c&location=9006934&targetid=pla-652518105343&ds_rl=1246978&ds_rl=1248099&gclid=Cj0KCQiAw_H-BRD-ARIsALQE_2PT5vFzY3pGi-a3_fmkx8YUlVyJgIsDusyDnRhce78wUmFVEe8N9gwaAnxFEALw_wcB&gclsrc=aw.ds) 。(我在想象，在她开始录音的前一刻，她试图给他 [米奇蔬菜薯片](https://www.kroger.com/p/good-health-mickey-shaped-veggie-chips/0075535500575) ，他立即开始融化，她很困惑，因为小男人通常喜欢自己吃一些米奇蔬菜薯片。)

5.当她确认他可以吃点心时，他明显高兴起来。她让他再深呼吸一次，最后一次“抖掉它”，然后他们出发去啃一些金鱼。

我敢打赌，这种方法非常有效，因为她在他很小的时候就开始和他一起练习*和*，因为他已经看到了很多这样的积极结果，所以他对这个过程有一些认同。
