# 你可能在社交媒体上用错了这些心理学术语

> 原文：<https://lifehacker.com/youre-probably-using-these-psychological-terms-wrong-on-1848582251>

当我在 2020 年第一次加入抖音的时候，我一天都没有看到虐待自恋者的视频，无论是以“有毒”前任还是情感缺失的父母的形式。虽然自恋者显然确实存在，但其内容如此普遍，以至于你认为自恋者和叫鲍勃的人一样普遍也情有可原。自恋已经成为新的“强迫症”——也就是说，另一个临床术语，人们经常误用它来表示“我把我的办公室整理得井井有条，我喜欢把我的回形针和图钉搭配起来。”

两年后，在社交媒体上几乎不可能不看到以下情况之一:多动症、煤气灯、创伤、焦虑、神经发散或触发(触发警告的后代)。

心理健康意识很重要，但随着越来越多的数字创作者制作与治疗相关的心理健康内容，临床心理学术语被随意抛来抛去，以至于模糊了它们的实际意义。虽然公开谈论心理健康问题可能是一个明显的积极因素，但正如澳大利亚出版商 [Zee Feed](https://zeefeed.com.au/overuse-clinical-terms-social-media-trauma-adhd-anxiety/) 所写的那样，“在社交媒体上过度使用临床术语可能会使行为病态化，并将生活体验变成内容趋势。”

在 Sci 秀 上，科学传播者汉克·格林(Hank Green)表示同意，“心理健康专家指出，使用诊断术语作为古怪行为、人格特质甚至股票市场变化的错位隐喻，最终会最小化严重的情况和患有这些情况的人。”

下面让我们来看看一些最常被错误使用的心理学术语，以及它们的作用，更重要的是*不意味着什么。*

**ADHD:** 根据 [美国国家心理健康研究所](https://www.nimh.nih.gov/health/topics/attention-deficit-hyperactivity-disorder-adhd) 的说法，ADHD 是“一种干扰功能或发展的注意力不集中和/或多动/冲动的持续模式。”这不是简单的心不在焉，忘记给别人回电话，或者有一辆永远脏乱的车。

点燃煤气灯:点燃煤气灯是一种精神虐待的形式，它让受害者质疑自己的理智。在《煤气灯》中，施虐者让受害者相信他们所说的事件从未发生过，并操纵他们相信自己过度敏感或精神不稳定。当有人不同意你的观点并试图让你相信他们的观点时，当你真的不同意时，认为你很沮丧，或者是一个 [少女时代](https://www.michigandaily.com/statement/gaslight-gatekeep-how-girlboss-went-from-aspirational-to-insulting/) ，煤气灯不是*。*

**自恋型施虐者**:据 [知情人](https://www.insider.com/narcissist-abuse-gaslighting-trend-trisha-paytas-gabbie-hanna-shane-dawson-2021-7) 透露，自恋型人格障碍是一种可诊断的病症，影响 0.5%至 1%的人口。它的特点是控制、侵犯隐私、社交孤立、言语辱骂、 以及给受害者灌输恐惧。自恋者不是伤害你感情或在 Tinder 上跟踪你的人。

**OCD** :强迫症是“ [一种人们反复出现的、不想要的想法、想法或感觉(强迫观念)使他们感到被驱使重复做某事的障碍](https://www.psychiatry.org/patients-families/ocd/what-is-obsessive-compulsive-disorder) ...这会严重干扰一个人的日常活动。”这是*而不是*经常吸尘，移动你卧室的家具，或者因为你无法忍受看到它而心血来潮地重组你的冰箱。

触发因素:触发因素是提醒一个人创伤经历的东西，无论是虐待、饮食失调、药物使用、事故还是其他形式的创伤。说“我被触发了”经常是开玩笑的；比如“一个女人的完美男友在床上给她送早餐的照片触发了我。”但是触发因素是创伤经历的提醒，而不是仅仅让你感到不舒服、焦虑、愤怒或不满足的东西。

**焦虑:**根据 [梅奥诊所](https://www.mayoclinic.org/diseases-conditions/anxiety/symptoms-causes/syc-20350961) 的说法，焦虑症的特征是“对日常情况强烈、过度和持续的担忧和恐惧。”通常他们会突然感到恐惧或害怕(惊恐发作)。我们都有焦虑的时候吗？是的。但是没有必要在演讲、聚会或排队等待“焦虑”之前记录你的感受。你现在经历的是紧张和不耐烦。

创伤:也许在网上没有比“创伤”更被滥用的词了正如心理学教授尼克·哈斯拉姆为 [芝加哥论坛报](https://www.chicagotribune.com/opinion/ct-trauma-microaggressions-trigger-warnings-20160815-story.html) 写的那样，人们被高调的陪审团裁决、“特朗普 2016 口号”，甚至是他们卷曲的卷发“创伤”了。(我曾经上传了一个抖音的视频，是关于在我孩子洗澡的时候把水倒在他头上的，收到了很多人的评论，这些人小时候被水泼到脸上，受到了“精神创伤”。)

根据美国心理协会 的说法，创伤是“对事故、强奸或自然灾害等可怕事件的情绪反应”，其特征是震惊、否认、“不可预测的情绪、闪回、紧张的关系，甚至是头痛或恶心等身体症状。”创伤是*而不是*小时候被强迫吃蔬菜，或者洗澡时眼睛进水。
