# 如何使用过渡对象作为处理悲伤的方式

> 原文：<https://lifehacker.com/how-to-use-transitional-objects-as-a-way-to-process-gri-1847829394>

悲伤没有对错之分。每个人都以自己的方式，用自己的时间处理损失。悲伤也是非常偷偷摸摸的:你可能认为你已经解决了它，却发现某首歌、某个气味或某段记忆让你再次体验到失去的痛苦。

事实证明，有些人已经发现，拥有一个过渡物体可能会帮助他们哀悼一个已经死去的人，同时仍然紧紧抓住他们的一部分。这可能是这样的。

## 什么是过渡对象？

过渡物体最常出现在孩子的环境中——尤其是那些可能面临分离焦虑的孩子。以下是 2018 年 Lifehacker 文章 中对这个概念的描述:

> 如果一个孩子很难离开你，一个 [过渡物体](https://www.psychologytoday.com/blog/the-guest-room/201407/more-just-teddy-bears) 比如一个填充动物或者最喜欢的玩具会很有帮助。对于更小的孩子来说，它让他们保持一种舒适感和一致性。

那么过渡对象对于成年人来说是什么样子的呢？就像其他悲伤的过程一样，这是非常私人的。丽莎·卡纳雷克在为 [Well+Good](https://www.wellandgood.com/transitional-objects/) 撰写的一篇文章中说，虽然有些人可能会在他们生命中重要的人去世的照片或视频中找到安慰，但其他人更喜欢有形的物品，并发现拥有属于他们失去的人的东西会让他们感觉更亲近。

## 过渡对象是如何帮助人们处理损失的？

让我们先说，有些人根本不觉得过渡性物品令人欣慰，事实上，他们更容易完全避开死者的个人物品。但对其他人来说，这是他们悲伤和康复过程中不可或缺的一部分。

“对许多人来说，这是人存在的证据，尤其是如果死亡是意外的，”梅根·迪瓦恩，LPC，心理治疗师和畅销书 [*的作者，在一次采访中告诉*](https://www.amazon.com/Its-That-Youre-Not-Understand/dp/1622039076/ref=sr_1_2?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-to-use-transitional-objects-as-a-way-to-process-gri-1847829394&asc_source=&crid=YJO8RF4Q6EKL&dchild=1&keywords=it's&tag=kinjalifehackerlink-20) *[好+好](https://www.wellandgood.com/transitional-objects/)*没关系。“即使死亡是预料中的，有时也有不真实感，就像他们在这里，而现在他们不在了。”
