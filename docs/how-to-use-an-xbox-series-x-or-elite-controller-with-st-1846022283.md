与一些输入设备不同，你需要配置一些设置，然后才能使用新的Xbox控制器来播放你巨大的Steam库。首先，Xbox Series X和Xbox Elite控制器只有在运行最新的Steam应用测试版时才能工作。全面支持应该很快就会推出稳定版本，但如果您想立即使用它们，您必须选择进入Steam测试版:

1.  打开蒸汽。
2.  进入**蒸汽>设置>账户。**
3.  点击测试版参与部分下的**“更改**”。
4.  从下拉菜单中选择“Steam Beta Update”。
5.  点击**“确定”，**然后重启，更新Steam。

接下来，您需要将控制器连接到您的PC。您可以通过USB或蓝牙 :
将其插入

1.  按Xbox按钮打开控制器。
2.  按住**“Pair”**按钮三秒钟，使控制器进入配对模式。Xbox按钮将开始闪烁。
3.  在你的PC上，进入**设置>设备>蓝牙&其他设备**。
4.  确定蓝牙已打开，然后点按**“添加蓝牙或其他设备”**
5.  从可用设备列表中选择您的Xbox控制器。
6.  等待您的电脑设置控制器并下载任何必要的驱动程序。

随着Steam的更新、控制器的连接和驱动程序的安装，您现在应该可以在Steam的连接设备中看到您的Xbox控制器。但在你开始玩之前，打开Steam的大图模式，然后进入**设置>控制器设置**，并确保**“Xbox配置支持”**已打开，这样你就可以定制你的控制器的按钮布局，并使用像X系列控制器的触发器隆隆声这样的功能。
