# 如何提前解锁谷歌Chrome的实验标签滚动

> 原文:[https://life hacker . com/how-to-unlock-Google-chromes-experimental-tab-scrolling-1845312474](https://lifehacker.com/how-to-unlock-google-chromes-experimental-tab-scrolling-1845312474)

Chrome即将推出一项新的标签管理功能。你将需要 [Chrome金丝雀](https://www.google.com/chrome/canary) 来尝试这个新的“标签滚动”功能，但它值得单独安装。

Watch

启用后，Chrome顶部的开放标签条将会延伸到窗口之外。当你添加更多标签时，新的标签会出现在滚动条的更下方，而不是缩小每个标签并把它们挤成一团。

您可以通过单击现在出现在选项卡条边缘的三点小图标来导航可滚动条。整个方法比Chrome传统上显示开放标签的方式优雅得多，但它仍处于测试阶段，因此在推出稳定版Chrome之前，可能会发生一些变化。

至于Chrome Canary，这是一个单独的浏览器版本，作为谷歌各种实验性和未完成的Chrome功能的测试场。其中一些测试从未进入Chrome的公共版本，但考虑到标签滚动是Chrome的许多竞争对手如Firefox的标准功能，它肯定会成为近期Chrome更新的一部分。

在此之前，以下是如何在Chrome Canary中启用实验性的标签滚动功能:

1.  [下载安装](https://www.google.com/chrome/canary) Chrome金丝雀。
2.  打开Chrome Canary，然后进入 **chrome://flags/**
3.  搜索 **#scrollable-tabstrip** 。
4.  使用下拉菜单启用可滚动的TabStrip。
5.  点击**“重启**”重启Chrome Canary并应用设置更改。

简要说明:我们在两台不同的Windows 10电脑上尝试了这一方法，获得了不同程度的成功。启用Chrome Canary的标签滚动功能在一台电脑上运行良好，但在另一台电脑上经常会导致浏览器崩溃。使用测试版软件时，不一致和bug是很常见的，但仍然很烦人。如果这种情况发生在你身上，禁用 **#scrollable-tabstrip** 或者只是暂时回到使用普通的旧Chrome， [最近增加了几个新功能](https://lifehacker.com/improve-chrome-86-with-these-recommended-tweaks-1845291730) 。
