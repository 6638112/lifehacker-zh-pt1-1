# 如何和你的孩子开始写日记

> 原文：<https://lifehacker.com/how-to-start-a-journaling-practice-with-your-kids-1846017402>

当我们进入新的一年——尽管仍在艰难地穿越毁灭性的疫情——你可能正在寻找新的习惯，帮助你和你的孩子提高你的正念，并帮助处理我们正在经历的时代。如果是这样的话，现在是时候开始和你的孩子一起定期写日记了。



写日记对所有年龄的人都有很多好处。它可以提高你的写作水平，记录你的经历，增强你的创造力，并有助于缓解压力。它还可以帮助我们更好地认识和处理我们的情绪——碰巧我们现在都在经历很多，包括我们的孩子。正如常瑞华为 [《华盛顿邮报》](https://www.washingtonpost.com/lifestyle/2020/12/15/journaling-kids-practice-covid/) 所写的:

> 通过这种表达情感的过程，孩子们可以开始理解这些情感，这建立了自我意识和情绪调节技能。
> 
> 写日记还有助于“孩子们关注当下，反思今天发生的事情，他们现在正在经历的事情，”约书亚·麦基维根说，他是一位行为健康治疗师，在宾夕法尼亚州为高中生工作。在当前危机的背景下，这种意识可以驯服当我们面对不确定性时经常出现的焦虑。

### 获得他们的认同

如果你想和你的孩子一起开始写日记，你不想做的第一件事就是宣布你们都要这么做，而且是时候坐下来开始倾诉他们的心声了，不管他们喜不喜欢。为了让它成为一项有价值的活动，而不是一件苦差事，你需要他们的认同。

想一想你需要什么才能对这个过程感到兴奋。挑选合适的工具是一个好的开始:让他们选择自己的日记、时髦的闪光笔、或笔记本电脑，这样他们可以在沙发或床上更容易地写日记。然后让他们帮助制定基本规则——这是一个你想和他们*一起做的练习，而不是*到*他们。*

### 决定一个结构

一起谈论你希望你的日志实践看起来如何。没有一种方法可以做到这一点——最好的方法是你们都能从中获得最大乐趣和利益的。也许你想在周五晚上坐下来写日记，周六午饭后，或者周一早上第一件事。另一方面，不情愿的日记作者可能更喜欢“微日记”，每天只写一句话。

如果还不会写字的孩子想加入进来，他们可以用图片而不是文字来描述他们的经历，可以自己画画，也可以从杂志上剪切粘贴图片。

制定一些基本规则，比如写日志时不要看屏幕，或者达成一个协议，尽管你不必分享你写的日志(除非你想)，但每个人都可以同时坐下来分享。

### 但是不要过于结构化

既然你已经决定了你希望你的家庭日志是什么样子，那就做好准备改变方向，必要时放弃规则或调整期望值。也许你的孩子有时想使用书写提示(如果是这样， [你可以在网上找到大量的](https://www.waterford.org/resources/journal-prompts-for-kids/))；但也许有时候他们只是想做自己的事。如果有些日子他们想写作，但有些日子他们想画画，那很好。

他们可能想反思他们生活中正在发生的事情，或者他们可能想写一个小故事。养成习惯，为思考和创造提供一个出口远比实际的结构更重要——所以跟随他们的领导。
