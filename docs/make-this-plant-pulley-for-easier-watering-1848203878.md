# 让这个植物滑轮更容易浇水

> 原文：<https://lifehacker.com/make-this-plant-pulley-for-easier-watering-1848203878>

悬挂植物对你的家来说是一个很好的补充，但是给它们浇水可能是一件麻烦的事情——尤其是如果你的天花板很高的话。为了省事，你可以制作一个悬挂系统，让你可以轻松地将植物降低到浇水高度，然后再将它们升起来，不需要任何梯凳，也不会有水溅到地板上。查看 [这款抖音](https://www.tiktok.com/@thrift.more.waste.less/video/6982356456079232262) 看看它的运行，这里有更多关于如何构建自己的细节。



## 悬挂式植物滑轮需要什么

这个项目你需要一些硬件和一根结实的绳子。对于每株植物，你需要两个挂钩(如果你已经在天花板上放了植物挂钩，那么你只需要一个)和一根从天花板到腰部的绳子。为了正确安装钩子，你需要一个鞋钉探测器和一个带导向钻头的钻头。选择带有内置滴水盘的花盆来保持地板干燥是个好主意。

## 如何安装您的工厂滑轮硬件

首先，在离墙不太远的天花板上找一个螺柱，在那里你可以安装第二个挂钩。在螺柱上钻一个孔，然后拧上天花板挂钩。然后，在邻近的墙上找一个螺柱，在地板上钻一个容易够到的孔，然后拧入第二个挂钩。

然后，从你的植物衣架的顶端，安全地系住你的弦的一端。将绳子穿过天花板上的钩子，然后拉动绳子，直到植物到达天花板。打一个环，挂在墙上的挂钩上。这将确保你的植物在天花板的高度。然后，降低你的植物到浇水的高度，系上第二个环，把它挂在墙上。这将确保你的植物在浇水高度。

## 如何标记你的循环

你可以通过解开天花板上的环，把你的植物降低到浇水高度的钩子上来提升和降低你的植物。你可以用记号笔或一点点的电工胶带来标记其中一个环，这样你就可以一眼看出区别。你也可以将绳子的浇水端牢牢系在墙上的挂钩上，而不是留一个开口环，这样可以减少混淆的可能性。

## 一定要安全悬挂你的植物

为了确保您的弦不会从天花板的挂钩中滑出，请确保挂钩的开口背向墙壁。当然，在松开绳子之前，一定要把你的环固定在钩子上，否则你的植物会掉下来。

同样重要的是，让你的天花板挂钩离墙不到两英尺，因为离墙越远，角度就越危险，有可能成为一种危险。这不是一个在房间中央悬挂植物的好项目。
