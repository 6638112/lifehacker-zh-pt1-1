# 用更多松脆的东西让每一道感恩节菜肴更美味

> 原文:[https://life hacker . com/make-every-thanksgivity-dish-better-more-crunchy-s-1845572856](https://lifehacker.com/make-every-thanksgiving-dish-better-with-more-crunchy-s-1845572856)

在家做饭时，即使是感恩节大餐，你也可以做一些事情来让它有“厨师”的感觉。添加比你认为必要的更多的黄油和盐，以赏心悦目的方式摆放食物，使用真正的餐巾都有助于培养餐厅氛围，但我认为提升你的食物的真正关键在于自由地使用松脆的食物。

就像一点点新鲜的切碎的香草或一点点美味的橄榄油，松脆的小块让你的菜感觉像是完成了，但它们也让看起来*有趣*和*完全实现了*。(它们还会分散人们对任何低劣元素的注意力，就像悲伤圣诞树上的金属箔片或低劣艺术作品上的闪光。)

我问你:绿豆砂锅最好的部分是什么？不要撒谎说是豆子是绿色的。你知道这不是魔豆。这是炸洋葱。这是松脆的部分。*这种*让人们对绿豆砂锅感到兴奋，你可以利用这种松脆的力量为你在火鸡日供应的其他菜肴创造类似的宣传。

将炸蒜片和土豆泥一起上桌，在火鸡上撒上炸香草，在蔬菜和沙拉上撒上烤坚果。煎一整串葱和/或洋葱，放在每个砂锅里，而不仅仅是T2绿豆。将多种松脆的东西放在一个菜里——你能想出一个你不能在由浓缩汤和青豆制成的砂锅上放炸葱、*和*蒜片、*和*烤坚果的理由吗？因为我肯定不能！如果你认为食物变得太“油炸”了——我从来没有想到过这个想法——用新鲜的草药来平衡它。

现在，如果你真的想让每个人都大吃一惊，如果你真的想像昔日美食频道的明星们那样“更上一层楼”，可以考虑向你的屠夫要一些额外的火鸡皮。把那层皮带回家，油炸或烤(T4)成酥脆的(T5、T6)，然后把那种松脆的美味撒在每一道配菜上(T7、T8、T9、T10)。(也可以用鸡皮；无论哪种方式，脑残都会被打破。)

哦，别忘了甜点:蜜饯坚果、酥脆饼干碎和任何一种脆片都可以放在馅饼等上面，非常好吃。我最喜欢的是蜜饯山核桃。
