# 如何让你的猫使用怪异的垃圾垫

> 原文：<https://lifehacker.com/how-to-make-your-cats-use-the-freaking-litter-mat-1846359503>

猫很棒；不断踩他们的垃圾不是。如果你买了一个捕捉猫砂的垫子，只是为了看你的猫每次从猫砂箱出来时，戏剧性地跃过垫子，跳到你刚刚打扫过的地板上，你可能会想，你为什么还要费心。在你花钱买一台机器人吸尘器之前，试着把垃圾箱转过来，让它面向墙壁。

Watch

有时候，让猫听话的最简单、最快的方法就是别无选择。这就是这里的策略。没有那么大的开放空间，你的猫每次进出厕所时都不得不穿过垫子。它不会完全消除垃圾颗粒，但它会给垫子一个在它们被释放到野外之前抓住它们的机会。此外，这是一个免费解决问题的方法，而解决这个问题的成本可能高得令人恼火。没有必要去买一个更新更大的垫子——你我都知道你的猫也会*和*拒绝碰它——或者甚至重新放置猫砂盒。将盒子放在原处，将它转过来面对墙壁，并放置垫子，尽可能多地盖住它们之间的地板。如果你能把它楔入一个角落，那就更好了。

这个完美的计划有一个缺陷:你不能把一个打开的垃圾箱转向一面墙。如果这是你的猫使用的，他们将不幸地继续来去自如，而不考虑你精心制定的计划。但是，如果你是一个封闭的猫砂箱家庭，你的猫绝对拒绝踏上垫子，这个技巧将保持跟踪垃圾最少。
