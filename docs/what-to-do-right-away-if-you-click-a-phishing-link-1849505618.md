## **如果您在可疑网站输入了信息**该怎么办

这似乎是理所当然的，但值得一提:理想情况下，你不会在任何文本框中输入任何信息，也不会从一个可疑网站下载任何东西。尽管如此，如果你确实在某个地方输入了你的信息，根据布法罗 [大学信息技术系](https://www.buffalo.edu/ubit/service-guides/safe-computing/managing/recognizing-a-phishing-attempt/what-to-do-if-you-gave-your-personal-information-to-a-phisher.html) 的说法，你需要修改被锁定的账户的密码。如果您在任何其他帐户上使用相同的密码，也要更改它们，并使它们都是唯一的。

如果您输入了与您的财务相关的信息，请拨打您银行的热线电话(您可能会在您的借记卡或信用卡背面找到)并报告该事件。检查你的报表，看是否有账户滥用或违规的迹象。考虑锁定你的信用记录，以防止任何人开设新账户，并查看你的银行网站，了解下一步该怎么做的具体提示。如果你把钱转给了骗子，请向当地警方报案。

锁定你的信用报告(Transunion、Equifax、Experian、*和* Innovis)并通知美国联邦贸易委员会你被钓鱼，特别是如果你输入了你的社会安全号码。

## **如何保护您的账户和设备**

无论您是否输入了任何信息，以下是您在点击网络钓鱼链接后应该做的事情， [根据网络安全公司 an networks](https://www.anetworks.com/what-to-do-if-you-clicked-on-a-phishing-link/):

*   断开您的设备与互联网的连接，以阻止任何恶意软件传播到网络上的其他设备。
*   使用 USB 等物理外部设备备份您的设备，这样您就不必重新连接到互联网，但可以确保您的数据不会被擦除。
*   扫描你的设备是否有恶意软件，但是如果你没有安装防病毒软件，并且你不是技术专家，你可能需要找专业人士来帮助你。
*   更改您所有帐户的凭据，并在可能的情况下启用双因素身份验证。

G/O Media 的 IT 服务总监克里斯托弗·德·盖塔(Christopher De Gaeta)解释说，一旦你清除了恶意软件——或者如果你没有发现任何恶意软件——你就可以安全地重新连接到互联网，这样你就可以更改密码并启用多因素身份验证，尽管“有些人可能会认为这并不完全安全，只有清除电脑才是确保安全的真正方法。”请咨询专业人员或您的 IT 部门，以确定重新连接前是否需要擦除。我曾经点击了一个网络钓鱼链接，德·盖塔说服了我，让我摆脱了厄运螺旋，所以我可以担保他的权威*和*直接去找你的 IT 部门的价值，即使你很尴尬。

德·盖塔还建议设立欺诈警报，尤其是如果你交出了任何敏感信息，比如你的社会安全号码，因为掌握了你的详细信息的人“很少会迫不及待地开始使用它们”

在你遵循了这些预防措施，恐慌消退后，不要忘记采取一些措施来阻止这一切再次发生。花些时间了解不同种类的骗局——如 [【微笑攻击】](https://lifehacker.com/what-are-smishing-attacks-and-how-to-avoid-them-1848495597)——和 [提高你的技术素养](https://lifehacker.com/the-complete-guide-to-avoiding-online-scams-for-your-l-5420356) 这样你就可以更容易地识别下一次网络钓鱼企图，因为还会有下一次。骗子不休息，你也不应该休息。
