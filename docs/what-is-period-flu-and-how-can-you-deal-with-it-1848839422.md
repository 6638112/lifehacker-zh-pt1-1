# 什么是“经期流感”,你如何应对？

> 原文：<https://lifehacker.com/what-is-period-flu-and-how-can-you-deal-with-it-1848839422>

如果你有月经，你就知道月经前综合症。经前综合症，通常被称为，是在月经即将到来之前可能发生的身体和情绪烦恼的集合。但是对于一些人来说，一个月中的这个时候可能会出现一些感觉像是真的生病的症状。



## “经期流感”是真的吗？

从人们经历它并给它命名的意义上来说，经期流感是真实的，但它不是流感，它不具有传染性，也不是一种医学定义的疾病。换句话说，你不可能从医生那里得到“经期流感”的诊断。

## 经期流感的症状是什么？

被称为“经期流感”的症状包括:

*   发冷
*   乏力
*   恶心和/或呕吐
*   腹泻
*   虚弱或颤抖的感觉
*   潮热
*   晚上出汗，或者当你不希望感到热的时候
*   束缚
*   膨胀
*   背痛
*   腹痛

一些人还报告体温升高、流鼻涕或喉咙痛。

这些症状的一个原因是，前列腺素在月经前和月经开始时偏高。这些是帮助子宫颈打开的激素，但它们也会导致你的肠道运动(因此出现了 [腹泻和](https://lifehacker.com/why-you-get-gas-and-diarrhea-when-you-have-your-period-1821308466) )。它们也被身体用来对抗感染，并在发烧等症状中发挥作用。因此，即将到来的时期感觉上类似于疾病也就不足为奇了。

当然，还有其他荷尔蒙在起作用。你的雌激素和黄体酮水平同时下降，并且有各种各样的理论关于到底是什么导致了最严重的“经期流感”症状。

## 期间流感症状与其他疾病和情况重叠

在把你的症状归结为经期流感之前，考虑一下是否有其他的原因。如果你感到恶心、发冷和虚弱，而你在月经前通常不会有这些症状，那么你完全有可能患了真正的流感(或 COVID，或胃病，或其他疾病)。)

[子宫内膜异位症](https://www.womenshealth.gov/a-z-topics/endometriosis) 是子宫内膜生长在子宫以外的地方的一种情况。症状可能包括极度疼痛的时期，性交时的疼痛，以及恶心、腹泻和排便疼痛等胃肠道问题。

[经前焦虑障碍](https://www.womenshealth.gov/menstrual-cycle/premenstrual-syndrome/premenstrual-dysphoric-disorder-pmdd) 是与月经和经前综合症相关的另一种状况，但其主要特征是情绪的变化。人们可能会易怒或抑郁到自杀的程度，或者可能会恐慌发作或疲劳。你可能也有身体症状，包括痉挛、头痛或肌肉疼痛。

围绝经期可能发生在月经停止前几年，它可能包括热潮红、情绪波动以及较重、较轻或不规则的月经。

最后，值得考虑的是，除了经期或传染病之外，是否还有其他因素影响着你的健康。在一次关于经期流感的在线讨论中，一个人说他们的症状是由于对对乙酰氨基酚(泰诺)过敏，这是一种人们经常用来缓解经期症状的常见止痛药。

## 如何应对经期流感

首先，如果你的症状很严重，和专业的 谈谈，比如你的妇科医生。他们可以帮助你弄清楚你是否正在经历上述和中的一种情况，并开出适当的治疗方案。他们还可以帮助排除其他事情发生的可能性，比如真正的传染病。

如果你的症状不太严重，可以考虑服用非处方的消炎药，比如布洛芬。这阻止了前列腺素的形成，从而减少症状，但 [的问题是，你需要在事情变坏](https://lifehacker.com/take-ibuprofen-before-period-cramps-hit-1845157259) 之前服用*。如果你跟踪你的经期，并且倾向于知道它什么时候开始，考虑在前一天服用 Motrin。*

与你的医生讨论的另一件事是改变生育控制是否有帮助。如果你不使用激素方法，开始一个可以帮助驯服你的一些症状。也有一些药物可能有助于缓解某些症状:例如，抗抑郁药可能适用于一些有抑郁情绪问题的人。
