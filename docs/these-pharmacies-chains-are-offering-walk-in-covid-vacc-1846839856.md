# 这些连锁药店提供无需预约的 COVID 疫苗

> 原文：<https://lifehacker.com/these-pharmacies-chains-are-offering-walk-in-covid-vacc-1846839856>

新冠肺炎疫苗越来越容易找到。几周前，美国所有成年人都有资格预约疫苗，不管你住在哪个州。现在，几家连锁药店在大部分地区提供预约和当天预约服务。



拜登总统本周宣布，他正在命令从联邦政府获得疫苗的药店开始提供预约服务。成千上万的药房参与了这一计划，包括许多主要的连锁店。你可以在这里找到；他们包括 CVS，Walgreens，Walmart，Albertsons(包括 Safeway)，Rite Aid，Kroger，Costco，H-E-B，Publix，Topco 等等。

不是每个地方每天都会提供无预约服务，但是当你查看当地药店的供应情况时，你很可能会注意到无预约服务是可用的。一些药店还在你预约的同一天甚至同一个小时内提供预约，所以人们越来越能够在短时间内获得疫苗。

一些连锁店已经发布了他们自己的公告:

*   CVS
*   沃尔格林 现在在其所有药店提供当天预约，并在选定的地点提供无预约服务。
*   所有沃尔玛 [和山姆会员店](https://corporate.walmart.com/newsroom/2021/05/04/walmart-and-sams-club-now-administering-walk-up-covid-19-vaccines-at-5-100-pharmacies-nationwide) 现在都提供无预约服务。(山姆会员店是*而不是*接种疫苗的必要条件。)预约也可以。

要找到你附近的无预约药店，请查看每家药店的信息。联邦计划还包括一些独立药店和小型连锁店。国营诊所也经常提供无预约服务，所以请检查一下你附近的诊所。接种疫苗可能比你想象的要容易。
