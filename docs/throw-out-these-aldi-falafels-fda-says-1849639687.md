# FDA 说，扔掉这些 ALDI 沙拉三明治

> 原文：<https://lifehacker.com/throw-out-these-aldi-falafels-fda-says-1849639687>

美国美国食品药品监督管理局 (FDA)已经召回了已经在美国各地销售的品牌 Earth Grown 的沙拉三明治，因为它们可能含有产志贺毒素的*大肠杆菌*。如果食用这种细菌，会导致疾病，症状包括胃痉挛、血性腹泻和呕吐。老年人、儿童和免疫系统弱的人受到的影响最大。以下是需要了解的内容。

## 哪些沙拉三明治正在被召回？

被召回的来自“地球生长”的沙拉三明治由 ALDI 超市独家销售。具体来说，这两种可能被感染的沙拉三明治被称为“地球生长的纯素食传统沙拉三明治”和“地球生长的纯素食大蒜和香草沙拉三明治”
