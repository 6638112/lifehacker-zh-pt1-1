# 让你的Android运行更快的最简单的方法

> 原文:[https://life hacker . com/the-simple-way-to-make-your-Android-run-faster-1849146167](https://lifehacker.com/the-simplest-way-to-make-your-android-run-faster-1849146167)

无论你在安卓系统上使用什么浏览器，它都会随着时间的推移积累垃圾文件。如果不加检查，这些文件可能会占用数百兆字节到数千兆字节的存储空间。它不仅让你的浏览器变慢，还会影响你手机的速度，解决方法很简单:定期清除你浏览器里的垃圾文件。

## 如何清除Android 浏览器的缓存

每个浏览器在清理缓存和垃圾文件时都有不同的步骤。我们将在下面介绍流行的浏览器。

### 铬

打开浏览器，点击三点**菜单**按钮，进入**历史** > **清除浏览数据**。

从这里开始，确保**时间范围**选项设置为**所有时间**接下来，选择“**cookie和站点数据**”和“**缓存图像和文件**”选项。点击**清除数据**清除所有保存的数据。

### 三星互联网

如果你使用三星智能手机，你可能会使用他们的捆绑浏览器。打开浏览器，点击右下角的三行**菜单**。G o到**设置**>T6】个人资料 > **删除浏览资料**。C 选择“**cookie和站点数据**”和“**缓存图像和文件**”选项。点击**删除数据**和从弹出的中确认。

### 火狐浏览器

点击右上角的三点**菜单**按钮，进入**设置** > **删除浏览数据**。选择“ **Cookies** ”和“**缓存图像和文件**”选项。然后轻点**删除浏览数据**。

## 如何清除任何Android 应用的缓存

如果你用的是Brave或者Opera这样的小众浏览器呢？在**设置**的**历史**部分查看，你会在那里看到清除垃圾文件的选项。

还有一个清除缓存的全局选项，适用于任何应用程序，包括浏览器，但我们不建议你对银行或密码管理器等应用程序使用这个选项，因为你会冒着丢失重要数据的风险:在你的安卓手机上打开**设置**应用程序，进入**应用程序&通知**(或**应用程序** ) > **应用程序信息**，然后选择你想要的应用程序所有的临时文件将被立即删除。
