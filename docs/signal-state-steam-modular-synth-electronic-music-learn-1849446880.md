# 这个益智游戏是一个有趣的，负担得起的学习电子音乐的方式

> 原文:[https://kotaku . com/signal-state-steam-modular-synth-electronic-music-learn-1849446880](https://kotaku.com/signal-state-steam-modular-synth-electronic-music-learn-1849446880)

去年9月发布的[](https://signalstate.io)*信号状态是一种令人惊讶和有趣的方式，将制作模块化synth补丁的过程转化为很酷的逻辑谜题——这就是模块化合成的初衷。在新的更新中，开发商Reckoner Industries为游戏添加了一个有趣的新沙盒模式，让你可以深入到模块化合成更具创造性的一面。现在，你可以不用解决难题，只需玩一玩就能创造出很酷的声音，看到和听到当你把不同的东西插在一起时会发生什么，同时学习这些仪器如何工作，而成本只是现实世界的一小部分。结果很好地转化为现实，如果您正在寻找一个有趣的机会来学习如何在一个友好、负担得起的环境中创建synth补丁，这将非常酷。*

*Watch*

*在Steam和GOG的Windows和Mac上可用的信号状态(Signal State)采用了模块化合成器如何工作的最基本概念，并围绕它包装了一个后启示录情节。该游戏采用基本上是“模块”的离散声音设备，这些设备具有特定的功能，可以通过电压连接到其他声音设备，与计算机中GPU的工作方式没有太大不同，并将它们描述为重建农场各种电气系统所必需的组件。通过它的拼图，您可以学习如何在不同的方向发送电压，同时还可以选择正确的模块来影响电路中的电压流量和数量。这是一个超级冷酷，但具有挑战性的逻辑益智游戏，它密切反映了与真正的 [欧洲轨道合成器](https://en.wikipedia.org/wiki/Eurorack) 一起工作的感觉。*

*以前，玩*信号状态*的模块化设备的唯一方法是通过谜题，这些谜题通常有非常具体的，如果不是有点令人费解的解决方案的话。实际上，你不能只是将模块连接成“补丁”，然后观察会产生什么样的声音。然而，随着新沙盒模式的加入，开发者增加了一些新的模块，让你探索模块化合成和基于音序器的音乐的基础。*

*虽然这种新模式远不是放弃真正合成器甚至复杂软件合成器如[【VCV机架】](https://vcvrack.com) 的理由，但新模式是理解模块化合成器如何工作的一套完美的训练轮。正如老兵们可能意识到的那样，答案在于电压的来源和去向。*

*“训练轮”是描述这款游戏和这种新模式的一种完美方式，因为如果你真的想进入非常棒的(可能非常昂贵的)模块化合成世界，这款游戏将帮助你了解基本知识，但如果你真的被这个世界吸引，如果你想走得更远，你必须考虑专业音频软件或设备。这不是批评的重点。这款游戏将会以很多人不知道的方式涵盖基础知识。最重要的是，*信号状态*在幕后为您做了一点基础工作，对于那些已经知道如何使用这些仪器的真实等效物的人来说，它可能会感到有点限制和困惑。*

*例如，游戏中路由音频信号的大多数方式并不真正符合这些设备在现实世界中的工作方式。作为一名使用模块化和半模块化合成器的艺术家，我实际上很难弄清楚如何让事情正常工作。我最终创作出了感觉比我在真正的合成器上创作的音乐更加僵硬的音乐。此外，它与时钟一起工作的方式对我来说很奇怪，低频振荡器模块(LFO)并不完全符合我在现实世界合成器中习惯的行为方式。当我看到电子音乐家本·乔丹在他的新模式之流中开始时也有点磕磕绊绊时，我对自己早期的挣扎感到不那么害羞了。*

*作为一个非常熟悉乐器和合成器的人，如果我在教别人，有一些事情我会做得不同。例如，游戏处理实际声音生成模块的方式，不是我打破声音设计和电子音乐原则的方式，但这不是一种损失，只是清算者工业公司认为对于一个精简的、初学者友好的环境有意义的不同。对于我发现自己说“是的，我会以不同的方式教学生”的所有小方法，该游戏完全弥补了这一点，它提供了一种更简单的学习方法，有清晰的解释，当你陷入困境时的提示，以及非常方便的电压走向和电压大小的可视化。*

*也就是说，如果你在演奏完*信号状态*后，真的进入了模块化合成的真实世界，那就做好准备，当你升级到更高的音乐制作水平时，会遇到一些成长的烦恼。总的来说，这个游戏会给你必要的词汇和视角来涵盖设计一个“补丁”的最基本的东西，这个“补丁”只是一个术语，指的是模块的特定排列以及它们是如何连接的。*

*我唯一真正的批评是关于游戏在沙盒模式下播放的环境音乐，虽然听起来很美，但可能会很分散注意力，对学习体验来说也不太好，因为你可能不知道声音来自哪里。您可以通过在“系统源”中选择“同步”来缓解它在游戏的设置中调低音乐会杀死所有声音，所以为了学习和探索，有一个专门的选项杀死背景音乐会很好。*

*虽然像VCV Rack这样的真正的音乐制作软件是免费的，*的信号状态*不是，但如果你是初学者，你可能会发现这个游戏更有帮助，因为它提供了提示，更清楚地说明了每个模块的确切功能，这是大多数专业音频软件所不具备的。此外，模块化合成会变得非常非常昂贵。因此，在一个20美元的游戏上学习可能会让你在未来避免犯一些代价高昂的错误，比如花钱买你还不知道如何使用或实际上并不需要的设备。购买一件昂贵的音频设备，然后不使用它对音乐家来说是非常正常的(就像过去几年一直放在我衣柜里的那把3000美元的定制吉他……)，但*信号状态*将帮助你避免许多初出茅庐的音乐家遇到的许多昂贵的颠簸。但预计现实世界中的游戏会很贵。今年我组装了一个非常简单的synth，只有五个模块，价格约为1000美元。如果你是认真的，这个游戏形式的20美元的永久课程可以是一个救命稻草。然后你可以去VCV，从那里开始。*

*如果你完成了游戏的所有主要谜题，并在沙盒模式中花了足够的时间探索和计划补丁，当你进入Eurorack时，你将远远领先于大多数人。所以，尽情享受制造哔哔声的乐趣，设置有趣的序列，在你计划出一个真正的合成器后，尽量不要花太多钱在上。小贴士:买二手的，寻找DIY工具包。你会省下一大笔钱。*
