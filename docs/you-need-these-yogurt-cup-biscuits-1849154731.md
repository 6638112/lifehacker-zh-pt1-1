# 你需要这些酸奶杯饼干

> 原文:[https://life hacker . com/you-need-these-酸奶杯饼干-1849154731](https://lifehacker.com/you-need-these-yogurt-cup-biscuits-1849154731)

当你建议修改传统食谱时，饼干会引起很大的紧张。事情会变得很尴尬。我用底部有樱桃水果的酸奶杯做了饼干，我很喜欢。坦白说，比起其他饼干，我更喜欢它。我完全理解有些家庭食谱很棒，而且对一些人来说，那个食谱将会是那里唯一的饼干。但是这个饼干配方更好。

我偶尔需要吃点热饼干。它们是一种享受——令人舒服，有黄油味，非常适合搭配一顿大餐或简单的果酱。饼干只需要很少的主料——黄油、面粉、盐、发酵粉、，通常还有酪乳——虽然你可以不加酪乳做饼干，但我发现它添加的水分会创造出更美味的快速面包。

不幸的是，我很少把酪乳放在手边，以备半个月一次的饼干需求。但是,我和经常做的东西是酸奶。我储备了一些5.3盎司的单份水果味酸奶杯，以及一大桶纯希腊酸奶，用于各种烹饪应用。酸奶是制作蘸酱、沙拉酱的好材料，也是辛辣食物的清凉调味品，如果你用牛奶稀释它，它是酪乳的合理替代品。如果你正在寻找其他东西来弥补你的酪乳不足，鲍勃的红磨坊有一个很好的细分 [这里](https://www.bobsredmill.com/blog/special-diets/what-can-you-substitute-for-buttermilk/#:~:text=Yogurt%20works%20well%20as%20a,%2C%20texture%2C%20and%20moist%20composition.) 。

我想要一种快速的方法来制作小批量的饼干，并且不想费心去稀释任何酸奶(在我的手套中的枕头饼干之间的步骤越少，越好)。通常我会吃原味酸奶，,但吃水果味饼干的想法改变了一切。

这种饼干配方使用了黄油块和面粉的常见混合方法，但不是扮演化学家，将醋搅拌到牛奶中，并怀疑它是否足够厚，而是倒空一份酸奶杯。令人兴奋的部分是决定你想要什么口味的饼干— 桃子、樱桃、椰子、柠檬、草莓香蕉— 选择的范围和奶制品区一样广。

这些饼干的诀窍是将黄油捏成扁平的小块，并相信酸奶的量。小心点。加入酸奶后，很容易想到，“哦，天啊，这需要牛奶，太干了，”但我还是忍不住想喝。在混合物中加入液体会使饼干变硬。

加入酸奶的过程进行到一半时，你将把蓬松的混合物倒在柜台上，在那里完成揉捏。一旦你把手伸进去，你会发现不需要额外的水分。坚持这个计划，你的回报将是嫩滑的奶油饼干，回味中带着水果味。

我选择了樱桃酸奶，我已经知道桃子是我饼干清单上的下一个。如果你想要更传统的饼干，请随意使用普通酸奶杯。在光谱的另一端，如果你想要更多的水果味，混合一把切碎的樱桃或任何你酸奶杯中的水果。

成分:

*   1杯和2汤匙(5.7盎司)通用面粉
*   1汤匙糖
*   1茶匙发酵粉
*   1茶匙玉米淀粉
*   茶匙小苏打
*   少茶匙盐
*   3盎司(6汤匙)黄油(冷的)
*   1份(5.3盎司)风味或原味酸奶杯(我用的是Chobani cherry希腊酸奶)
*   用于洗蛋的蛋黄(可选)

将烤箱预热到400华氏度。在烤盘上铺上羊皮纸。将烤架移到烤箱的中间位置。

在一个中等大小的碗中，加入干材料，用叉子搅拌混合。将黄油加入干混合物中，每汤匙两三片即可。将面粉撒在黄油上，这样每一片都有单独的涂层。捏住黄油块，让它们变平并分开。在面粉中不停地捏和搅拌黄油，直到你最大的一块有五分镍币那么大。这种混合物看起来会很不稳定。如果你的厨房很热，你可以将碗放入冰箱几分钟，同时搅拌酸奶杯底部的水果。

将酸奶杯倒入黄油和面粉混合物中。搅拌并用橡胶抹刀挤压，直到大部分面粉混合在一起。将混合物倒在台面上，拌入剩余的面粉。动作要快，但要轻柔，拿起较粘的团块，将它们压入任何现有的干团块中。将面团拍成一团。

将面团移到铺有羊皮纸的烤盘上。用手指尖将面团压平成大约1/4英寸高的椭圆形。这个食谱可以做四块2英寸的圆形饼干。对于软边和较高的饼干， [用下脚料](https://lifehacker.com/you-should-bake-biscuit-scraps-with-your-biscuits-1849054918) 烤圆饼。如果你喜欢闪亮的金棕色表面，可以用鸡蛋清洗表面。在400华氏度下烘烤15-20分钟。用咸黄油从烤箱中取出来，趁热享用。
