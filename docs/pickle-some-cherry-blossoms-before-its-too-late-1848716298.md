# 趁早腌制一些樱花

> 原文:[https://life hacker . com/pickle-some-cherry-flowers-before-that-tain-late-1848716298](https://lifehacker.com/pickle-some-cherry-blossoms-before-its-too-late-1848716298)

他们称之为花朵炸弹。

卡车大小的粉红色棉球被安装在人行道上，而上周这里只有树枝。到了四月中旬，最轻微的微风就会使它飘落粉红色的花瓣，很快就会铺满街道。现在是樱花时间，一夜之间出现。

Watch

在这个长达数周的粉红世界末日期间，邻居们在门前的草坪上乘凉，比平时更加友善；虽然这可能是一种错觉，但每个人似乎都更加满足了。由于这些日子缺乏满足感，我一直在思考如何及时将那一刻装瓶。去年，古老的亚洲腌制大师给我送来了一个计划: [腌制一些樱花](https://www.justonecookbook.com/salt-pickled-cherry-blossom/)。

真的不难。找你知道没被喷过的樱桃树。 [确定它们真的是樱花树](https://vcbf.ca/2014/03/27/plum-versus-cherry-trees-how-to-tell-the-difference/) ，因为很多花看起来都一样；快速数一下花瓣会有帮助。这些花可以来自结果树或不结果树，但在我看来，后者更好，因为结果树上的每一朵花都是一颗潜在的樱桃，我不会梦想去抢任何人的樱桃。

收获花朵的最佳时间是在T2雨后阳光明媚的日子，那时花朵干净干燥。否则，在露水干了之后，上午10点左右出发。摘下你的花朵，瞄准更年轻的，半开放的花蕾。

将头放入中，将花朵和海盐放入玻璃 或其他带盖的非反应性容器中，按照1份盐对4份樱花的重量比例(25克盐对100克花朵)。在开始分层之前，我会用伏特加擦拭杯子内部，以确保它真的很干净。

完成分层后，将塑料袋放入容器中。(使用更厚的Ziploc或真空密封袋，而不是杂货店生产的袋子，因为袋子太薄，会渗漏或撕裂。)打开顶部的袋子，加入足够的水，使其充满容器的其余部分，排出所有空气，并给花朵和盐施加压力。将顶部下垂的袋子放在碗的边缘，并扣上盖子。现在你有一个密封。

在接下来的几天里，在冰箱里，盐将与花朵上的水分混合，形成一种醋，多亏了我们的水袋，花朵将继续浸泡在这种液体中。三天后，你有两个选择:晒干花，或者把它们放入沐浴池。要弄干它们，只需将它们冲洗干净，然后放入脱水机，或者将烤箱调到最低温度。

我更喜欢它们在umeshu，所以我给它们一个冰水浴，排干水分，然后重复两次以上，确保我已经让去掉了所有的盐。接下来，我(非常小心地)将花瓣铺在茶巾上，轻轻卷起挤出水分。然后，我把它们放在一个玻璃罐里，并在里面装满豆荚。(梅树是制作梅博的梅子醋，这个 [我们之前已经讲过](https://lifehacker.com/7-perfect-pickles-you-must-keep-on-hand-at-all-times-1848161356) 。可以买。)(平心而论，你也可以只买盐- 樱花脯，但那有什么好玩的？)

虽然你可以在这里使用任何醋，但乌梅树是T2的传统选择，因其辛辣、咸甜的味道而值得追寻。你可以把这瓶花放在冰箱里或外面的架子上。(它们放在醋里，所以只要所有的花都被淹没，pH值就会保持的安全，但是如果这让你紧张，而且你有冰箱空间，冷却它们也无妨。)

确保为一年采摘足够的花朵，因为你只有有限的时间来制作它们。传统用途包括饭团和和食，如，但一旦它们出现在你的厨房，从架子上嘲笑你，你就会找到一些额外的借口来获得鲜味。

**冰淇淋**

我喜欢咸的和甜的，这就是为什么我在鲜奶油中放了一些香草冰淇淋，在顶部放了一些盐樱花。

**寿司**

如果你想改变你的素食寿司游戏，这就是你的选择——用樱花、玉兰花、和黄瓜做的简单寿司给你的味蕾一个有弹性的冲击。仅仅是在寿司饭中加入樱花就已经很壮观了，但是把它们变成 [饭团](https://www.instagram.com/stories/highlights/18020432737372051/) 就像 [Jane Hashimawari](https://www.instagram.com/ippaipdx/) 就更好了。

**烤猪肉**

我喜欢这些花的盐和甜如何减少T2烤猪肉的丰富性，所以我做了一个非常简单的樱桃釉(见我用在鸭子 [这里](https://lifehacker.com/actually-you-should-roast-this-duck-instead-of-a-turke-1848028209) 上的一个类似的)，然后用一些樱花装饰。

**菜花浓汤**

我每年都做花椰菜浓汤。暗示炒一些韭菜和洋葱，然后加入切碎的花椰菜，包括绿叶蔬菜。用高汤盖上盖子，炖30分钟。搅拌足够长的时间，让它如丝般顺滑，在再次搅拌之前，向4份汤中加入1份重奶油。最终的结果是s o豪华，加上盐腌樱花装饰，离天堂又近了一步。

即使你还不知道如何使用它们，也可以出去一个小时，慢慢地从樱桃树上摘下花蕾，从一个巨大的粉红色棉花球下面仰望天空。如果没有别的，这个简单的户外活动会让你感到更加满足。
