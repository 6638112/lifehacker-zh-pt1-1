# 什么是行为镜像，什么时候应该使用它？

> 原文：<https://lifehacker.com/what-is-behavior-mirroring-and-when-should-you-use-it-1848849649>

下一次你和某人深入交谈时，不管是柏拉图式的，浪漫的，还是商业上的互动，注意:你们是以相似的姿势坐着还是站着？你们说话的音量一样吗？你的手臂或手在做同样的事情吗？

如果是这样，你们中的一个很可能(有意识或无意识地)参与了被称为镜像的行为。也被称为“变色龙效应”，这种心理技术被专业网络人员和销售人员在战术上使用，也被其他所有人在不知不觉中使用，以形成强大的联系，并在人与人之间产生更大的信任感。

## 什么是镜像？

科学上称为“边缘同步”，镜像是模仿另一个人的肢体语言、言语模式、面部表情，有时甚至是他们的外表，以努力建立融洽关系，获得信任，培养更深层次的联系。虽然它可以被有意地用于需要销售、谈判和获得他人信任的职业，如治疗或警察调查，但它通常是无意的，是我们归属感需求的进化副产品。正如他们所说，模仿是最真诚的奉承形式(而且它*可以*，只要它不冒险进入令人毛骨悚然、*致命诱惑*的领域，让你模仿的人更喜欢你。)

实际上，镜像可以采取多种形式。在夫妻治疗中，伴侣可能被鼓励面对面坐着，用“我的陈述”来表达他们的感受，并逐字重复他们伴侣所说的话(只改变人称代词)。这种精确的语言镜像方法用来反映他人的感受，直到他们感到被充分倾听和理解。(对初学走路的孩子也很有效。)最重要的是，这也迫使双方真正倾听对方在说什么，而不是在对方说话的整个过程中计划反驳。

在柏拉图式或商业互动中，镜像看起来像是模仿他人的肢体语言、言语和面部表情。如果他们袖手旁观，你也袖手旁观；如果他们使用复杂的词汇，你就利用你的 SAT 词库，如果他们开始大声或小声说话，你就调整你的声音跟着做。在这一点上，你可能会想，“这听起来很奇怪，可能会适得其反。”你是对的。但是首先，好处。

## 镜像有用吗？

根据芝加哥私人诊所临床心理学家玛莎·劳伯(Martha Lauber)的说法，镜像是解决争论的最佳方式。它不仅迫使你倾听，而不是计划你的反驳，“每个人都认为对方是问题所在。镜像有助于你理解你只看到了画面的一半。”

据华尔街日报报道，

> “研究人员以新的方式使用脑成像技术，最近发现这些共有的行为超出了简单的模仿。普林斯顿大学心理学和神经科学副教授尤里·哈森(Uri Hasson)在 2016 年与人合著的一项研究中表示，科学家使用功能性核磁共振成像技术研究听者和说话者，发现他们是“动态耦合的”，说话者和听者的大脑对彼此的信号做出反应和适应。哈森博士将这种联系比作大脑的一种无线连接。

《华尔街日报》接着指出，在任何合作比敌对(大多数情况下)更有帮助的社会环境中，“这种结盟会培养亲密和信任。”

## 模仿该做的和不该做的

当以一种微妙的、不引人注目的方式进行，目的是产生共鸣或培养真正的联系时，镜像是最有效的。虽然许多人不会意识到这种情况的发生，但其他人会意识到，如果这种策略使用得太过肆无忌惮，他们可能会反感。

如果没有最基本的真诚参与，所有这些都不会奏效。因此，不要模仿别人做的每一件事，这既明显又令人讨厌，而是要以建立在最初融洽关系基础上的更微妙的方式为目标。首先，通过传统的非语言暗示建立联系，比如眼神交流、微笑、直接面对对方，避免看手机这种大忌。

一旦这个基础建立起来，注意他们的哪些说话方式、口音、用词、表情、手势和姿势对你来说是自然的，并在你这边的比喻桌上重现。不要用仿英国口音说话，或者以一种让人感觉被迫的尴尬方式坐着。关注小事情，比如:改变你说话的速度或音量，交叉或不交叉你的腿，配合他们快乐或惊讶的表情(如果你是一个好的健谈者，你可能已经这样做了)。

情绪高涨的时候不要镜像。当愤怒自由流淌时，不是模仿他人语气和表情的时候。当涉及到解决问题的时候，等到冷静下来再使用镜像。

有效镜像的关键是，不要在你的方法上太明显和刻意，以至于让人觉得死缠烂打，绝望，或者让别人觉得被操纵。但是，当巧妙运用时，它可以成为培养“ [”脑-脑耦合](https://www.wsj.com/articles/use-mirroring-to-connect-with-others-1474394329#:~:text=Mirroring%20a%20conversation%20partner's%20gestures,meetings%20and%20conversations%20with%20colleagues) ”合作、共鸣和信任的有力方式。
