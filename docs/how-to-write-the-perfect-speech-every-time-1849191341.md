# 如何写出每次都完美的演讲稿

> 原文：<https://lifehacker.com/how-to-write-the-perfect-speech-every-time-1849191341>

[发表伟大演讲的关键](https://lifehacker.com/how-to-give-a-great-speech-1835908227) 其实就是写一篇伟大的演讲。所有与演讲相关的建议都集中在我们的集体(并且完全理性) [对公开演讲的恐惧](https://lifehacker.com/how-to-ace-public-speaking-1839289746) 。如果我告诉你，当你知道文章无懈可击、精彩绝伦时，发表演讲会容易得多，你会怎么想？欢迎来到我的 TED 演讲。



我们都听过一些单调乏味的毕业典礼演讲，或者可能是你公司的最后一次市政厅会议上一些不那么激动人心的独白。尽管所有的演讲者都让我们在座位上打瞌睡，但写一篇好的演讲稿并不像看起来那么难。

我采访了演讲稿撰写人 [钱德勒院长](https://www.chndlr.com/) ，导演 [西楼编剧](http://www.westwingwriters.com/) 。无论你是想激励一些毕业生，获得晋升，还是想统治世界，这里有一些你需要知道的东西，可以满足你所有与说服相关的需求，写出最好的演讲。

## 基础知识:好的演讲都遵循同样的公式

迪安认为绝大多数的说服性演讲都遵循一个基本结构: [门罗的动机序列](https://bluetoad.com/publication/?m=27649&i=412996&view=articleBrowser&article_id=2798149&ver=html5) 。为了精心准备一场有效的演讲，试着遵循这个五步结构:

1.  **注意。**吸引你的观众。
2.  **问题。**提出你要解决的问题。
3.  **解决方案。**提供解决问题的方法。
4.  **远见。**帮助你的听众形象化你的解决方案(详见下文)。
5.  **行动号召。**给听众一些行动项目，结束你的演讲。

门罗动机序列的美妙之处在于，你一直在使用它，甚至没有意识到这一点。例如，迪恩指出，当我给他发信息要求采访时，我自己也用过这个词。尽管这种说服的流程对你来说可能已经很自然了，但这五个步骤是一个有用的清单，可以确保你在演讲中没有遗漏任何东西。

## 向你的听众解释清楚

迪恩解释了一句老话，告诉人们你的主要观点三次:“告诉他们你要说什么，说事情，然后告诉他们你刚刚说了什么。”

迪恩解释说，虽然这可能会让你的演讲在纸上看起来重复，但请记住，这种论点是为了听，而不是为了看。这一点之所以如此重要，是因为在一个活生生的口头媒体中，人们无法回过头去重读最重要的部分。为了帮助你的听众组织和处理你想让他们听到的内容，你需要有点多余。

所以，让我重申:不是一次，不是两次，而是三次击中你的要点。

## 让你的演讲更上一层楼的技巧和诀窍

迪恩提供了这些最后的提醒，以确保你的演讲尽可能有说服力。

解决房间里的大象。如果你演讲的活动有什么不寻常的地方，直接说出来，你的听众会觉得你更有共鸣

按时间顺序写。信息应该按照事情发生的顺序传达给观众，除非你故意隐瞒细节

使用视觉隐喻。找到能唤起某种视觉的语言，而不是用抽象或理论的术语说话。迪安提到了 [柯南·奥布莱恩对哈佛 2000 届](https://www.youtube.com/watch?v=2pnnWp1lNps) 学生的演讲，作为一个有趣的例子，说明了如何利用特殊性来描绘一幅图画。这不仅能抓住听众的注意力，还能确保他们记住你说的话。

为你的听众量身定做你的演讲。记住你的听众是谁，用细节来具体地和他们说话。 [埃里克·施努雷](https://bluetoad.com/publication/?i=412996&article_id=2798149&view=articleBrowser&ver=html5) ， [*的作者政治演讲稿撰写人的伙伴*](https://us.sagepub.com/en-us/nam/the-political-speechwriters-companion/book257507) 称这些时刻为“howdahells”，比如“*他们到底是怎么知道的？如果你能找到一些特定的话题，让你的特定听众产生共鸣，这会让你在人群中更受欢迎。*

**简洁。使用尽可能少的词语来阐述你的观点。完成了。**
