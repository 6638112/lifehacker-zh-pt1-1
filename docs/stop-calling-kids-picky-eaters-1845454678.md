# 不要再叫孩子“挑食”

> 原文：<https://lifehacker.com/stop-calling-kids-picky-eaters-1845454678>

生活就是成长、学习、改变(或试图改变)成我们自己最好的版本。因此，有一件事我要停止做，我希望你们都加入到我的探索中来:我认为是时候停止称某些孩子为“挑食者”了。

Watch

需要澄清的是，我从未当着孩子的面给他们贴上“挑食”的标签——同样，我也不会当着他们的面说孩子“顽固”。有些人可能会说我自己的儿子属于这两种类型；但是*我*不会这么说，因为那样不好。他喜欢吃的东西是不是有点有限？当然可以。他是否坚持在他的意大利面条上放多少酱是可以接受的？为什么是的，他是。不过没关系。他是个孩子，有时候孩子不喜欢大人喜欢的东西。

但是我会和其他不是 T2 孩子的人谈论抚养一个挑食者的困难。其实，正如 Lifehacker 自己的档案显示，我又犯了这个 [超过](https://offspring.lifehacker.com/create-a-menu-for-your-picky-eater-1842058400) 和 [超过](https://offspring.lifehacker.com/let-your-picky-eater-pretend-to-be-a-cooking-show-judge-1845000666) 和 [超过](https://lifehacker.com/yes-you-should-accommodate-your-friends-picky-eating-k-1841204883) 的错误。(看，我并不完美！此外，“挑食者”在标题中往往比“味觉特别敏锐的孩子”更好。)

但是最近来自作家安妮·瑟里奥特 在推特上的一个帖子让我想到我们关于这个的框架有点，有点，完全错误:

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1306767552882802688&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1306767552882802688&autosize=1) 

我建议你点击阅读整个帖子，因为她提出了一些非常好的观点。同时，以下是我对所谓挑食的看法:

### 这不是你造成的

许多人认为挑食是由父母的坏习惯“造成”的。也就是说，除非你的孩子最终变得很挑剔，在这种情况下，你会说，“哦，哼，我错了。”

另一方面，如果你的孩子什么都吃，从扇贝到花椰菜，再到最好的寿司，不要告诉我这是因为他们 18 个月大时你的喂养方式；我也这么做了，你的自命不凡显露出来了。孩子终归是孩子。

### 吃是一种复杂的感官体验

任何说“当他们足够饿的时候，他们会吃提供的东西”的人，从来没有遇到过坚决绝食的孩子。并不是他们不*要*吃。(嗯，有时候是)。更多的时候，你提供的东西对他们来说比饥饿的感觉更糟糕。

你可能会说，*但是如果你喜欢在披萨上加酱，为什么不试试在面条上加酱呢*？或者，*如果你喜欢冷奶酪三明治，为什么我不能给你做个烤奶酪三明治*？因为这些是不同的质感体验，这就是原因。我个人喜欢杏仁。不过，杏仁黄油呢？没有尝试的欲望。我不感兴趣。

吃食物是一种复杂的感官体验，充满了气味、味道和质地，许多孩子对不同的气味或质地很敏感。如果一个孩子不能忍受袜子接缝的感觉，为什么当他们看到三明治上的碎生菜时，我们会感到震惊？

用你已经长大的逻辑推理能力去尝试解释为什么面包*和*吐司都好吃，好吗？是的，当然！但是如果他们完全抵制熟面包，也许现在不用担心。因为，毕竟...

### 这是一个*阶段*

对大多数人来说，味觉受限只是一个阶段。猜猜从小到大谁是“挑食”的？是的，这个女孩。猜猜谁长大后会吃你放在她面前的任何东西？也是我！(除了任何与花生有关的东西，但我仍然希望我的口味会有所改变；你可能已经注意到了，花生在很多东西里。)

但在我开始吃海鲜和蔬菜等非玉米类食物之前，我已经是一个真正的成年人了。当你以某种方式给孩子贴上标签时，孩子往往会认同这个标签，认同你对他们的看法。更不用说这可能是一种不准确的方式来描述他们的一生。他们(很可能)不会想一辈子天天吃鸡块。

即使你没有直截了当地称他们为“挑食者”，他们也可能意识到了这样一个事实:他们有限的喜好是你沮丧的来源。你越少关注他们似乎只喜欢米色食物的事实，越多关注与食物建立健康的关系，从长远来看，他们会过得越好。

哦，当我们谈论孩子不想吃东西的时候，这里有一个你不应该做的事情的快速提醒:不要强迫孩子吃他们不想吃的食物。我的父母在我和利马豆身上犯了一次同样的错误。结局不太好，直到今天，一看到利马豆就让我反胃，尽管成年后我几乎喜欢所有其他的豆子。

处理用餐时间冲突的方法有很多，但我会告诉你我们家的规矩。你必须先尝一口，然后才能说:“不，谢谢。”我这样做是为了鼓励我的儿子，一点一点地扩展。他知道如果他不喜欢，我不会强迫他吃第二口，所以他能忍受一小口。

随着时间的推移，这种方法让他最终喜欢上了偶尔接触的不同食物，以及一些令人惊讶的立竿见影的胜利(我从来不会认为他是一个喜欢吃虾的人，但奇迹仍然是可能的)。然而，如果是我知道他真的不喜欢的东西——古斯古斯，我在看着你——我会给他一些我知道他喜欢的东西。我是想开阔他的视野，而不是让他在饭桌上吐得到处都是。
