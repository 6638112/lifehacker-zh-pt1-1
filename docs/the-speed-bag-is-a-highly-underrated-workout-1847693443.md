# 速度袋是一项被严重低估的锻炼

> 原文：<https://lifehacker.com/the-speed-bag-is-a-highly-underrated-workout-1847693443>

如果你曾经走进一家武术馆，你会对速度袋的快速砰砰声很熟悉。挂在转体上的小充气袋可能看起来无害——尤其是在与更大、更重的袋子相比时——但它恰巧是一种被严重低估的锻炼形式，其好处远远超出了健身房。



## 发展力量、速度和手眼协调能力

即使你从未使用过速度袋，你也可能熟悉它的基本姿势和形式:你用拳头的一侧击打它，以圆周运动的方式移动手臂，并以两次、三次或四次击打的方式交替进行。与此同时，你的脚也在弹跳，所以这对你的肩膀和胸部肌肉来说是一种很好的锻炼*和*它能锻炼你的核心。

斯坦福大学医学院的运动医学医生迈克尔·弗雷德里克森 指出，这些都是你在日常生活中使用的肌肉，比如当你举起或搬运东西时。“这些都是非常有功能的肌肉，”他说。

除了增强力量，速度袋还能帮助你提高手眼协调能力和速度。你必须掌握袋子撞击平台和反弹回来的时间，在恰当的时候撞击以保持平稳的节奏；这并不像看起来那么容易——至少一开始不容易。

仅仅几分钟后，你就会感觉到效果——你的肩部肌肉开始燃烧，快速的步伐会让你疲惫不堪。这是一项艰苦的锻炼，但也是一项令人满意的锻炼。

“这几乎就像一场游戏，”德怀特·普拉切特说，他是大街拳击[&泰拳](https://www.mainboxinggym.com/) 的拳击教练，训练职业拳击手 Raphael‘Trouble’Igbokwe 和 Austin‘Ammo’Williams。“挺好玩的。”

## **帮助预防受伤**

速度袋练习可以增强肩部肌肉，稳定肩关节。我们的手臂通过球窝关节与身体相连，这就是为什么我们可以向多个方向移动手臂——但这也是肩部受伤如此常见的原因。

美国物理治疗协会 的物理治疗师兼发言人 Ben Fung 说，使用速度袋可以帮助防止受伤。此外，他说，因为速度袋是一种低冲击运动，“它非常安全。这是一项低风险、潜在回报丰厚的活动。”

速度袋训练也有助于缓解整天坐在办公桌前的疼痛。直立姿势有助于保持良好的姿势，而力量训练有助于抵消一天过多的变焦会议和缺乏运动带来的背部和颈部疼痛。

## **如何入门**

当谈到学习速度袋，这是有益的，有一些适当的技术指导。正如圣路易斯华盛顿大学医学院的物理治疗师克里斯托夫·戈登 所说，培养良好的控制和力学能力真的很重要。

一个开始的方法是去当地的拳击和/或综合武术健身房。确保找到一个强调学习适当技术的健身房，同时也为你的特定目标而努力。这个 [Youtube 视频](https://www.youtube.com/watch?v=pDy3L3dkLWw&t=71s) ，由 csquaredboxing 频道提供，对于初学者来说也是一个很好的指南。

速度袋确实有一个陡峭的学习曲线，但一旦你弄明白了，这是一个令人满意的锻炼方式。“我从教人们如何做(速度袋)中得到乐趣，”Pratchett 说。“一旦他们把它拿下来，你可以看到他们眼中的光芒，他们正在这样做。”
