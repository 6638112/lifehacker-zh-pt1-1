# 如果你计划感恩节旅行，读读这条推特吧

> 原文：<https://lifehacker.com/if-youre-planning-to-travel-for-thanksgiving-read-this-1845749179>

尽管横跨美国的-19【疫情】 的 [加快了的步伐，本周仍有数以百万计的人为了感恩节而去看望他们的家人。集会将按计划继续，就好像美国疾病控制中心没有发布反对假期旅行的警告一样。随着人群的聚集，八个月的疫情不可避免地会增长，延长美国住院和死亡的下降螺旋，后者现在接近 26 万。](https://www.theguardian.com/world/2020/nov/19/us-hospitals-coronavirus-latest-spike-staffing-shortages-patients)



对于那些没有感染这种疾病、没有陷入经济困境或没有看到病毒影响到他们认识的人来说，很多痛苦是看不见的。对这些人来说，展示病毒不受控制的传播给人类带来的真实代价是有用的。

一名重症护理护士最近写的 Twitter 帖子正是这么做的。对于那些可能还在考虑周四假期前去拜访家人的人来说，这将有助于强调为什么你绝对不应该去。

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1330328485265612804&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1330328485265612804&autosize=1) 

这位女士，是一名重症护理护士，她在推特上讲述了她在整个疫情治疗 COVID 患者的工作，坦率地讨论了她将面临的情感代价假期与女儿隔离——这是由于她在工作中暴露于病毒的要求。她将被迫分离归因于全国范围内的病例激增，这是由许多人不戴口罩或拒绝遵守医疗界更广泛的指导方针造成的。

### 疏忽给别人带来无法形容的伤害

医护人员的负担不完全是治疗大量病人的结果。有一种情感层面悄悄进入他们的生活，尤其是在以家庭为中心的时候。

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1330332123220348938&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1330332123220348938&autosize=1) 

### 医护人员很累，需要休息

今年，让医护人员喘口气的最好方式是呆在家里。我知道这很糟糕——但很有可能的是，无论你的家人在感恩节采取什么样的预防措施都不会奏效。 今天，美国创下 [连续第 13 次因 COVID 住院的记录。八个多月来，重症护理护士、医生和当地重症监护室的所有人都在与疫情做斗争。](https://www.wsj.com/livecoverage/covid-2020-11-23)

如果我——一个从三月起就可以在家安全工作的作家——对你来说没有任何效果，那么就听一个重症护理护士的吧:

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1330333979027890177&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1330333979027890177&autosize=1) 

### 有时做对每件事是不够的

鉴于我们给自己制造的混乱，通过戴口罩、洗手、和避免人群来严格遵守安全准则是必不可少的，但 [有时不足以避免感染病毒](https://vitals.lifehacker.com/you-can-get-covid-even-if-you-do-everything-right-1845700270) 。即使你已经试着 [过着明显不那么社交化的生活，](https://www.nytimes.com/2020/11/20/opinion/sunday/covid-bubble-thanksgiving-family.html) 你也有可能在一起用餐时危及到餐桌对面的人。无症状传播是疾病传播中最常见的，根据 CDC 的说法，所以即使你感觉很好( [或检测阴性](https://www.wzzm13.com/article/news/health/coronavirus/why-a-negative-covid-19-test-doesnt-necessarily-mean-you-arent-contagious/69-d67d92db-aab6-4161-b6a3-c890654c9ee4) ) ，你也有可能把你家里的每个人都置于危险之中。

当你细读火车时刻表，浏览最后一班第二班航班的订票，或者等公共汽车送你回家时，不要只想着你的家人。想想整个行业的护士、医生和医护人员。然后待在家里。
