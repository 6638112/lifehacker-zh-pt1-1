# 编辑你的iMessages信息不会让你的错误消失

> 原文:[https://life hacker . com/you-can-finally-edit-and-undo-sent-messages-on-iphone-1849167883](https://lifehacker.com/you-can-finally-edit-and-undo-sent-messages-on-iphone-1849167883)

像 [Telegram](https://lifehacker.com/why-you-probably-shouldnt-pay-for-telegram-premium-1849104588) 和 [WhatsApp](https://lifehacker.com/how-to-finally-hide-your-whatsapp-last-seen-status-from-1849092145) 这样的消息应用程序会定期更新新功能，但苹果的Messages应用程序每年只有一次重大更新，即新版iOS发布时。幸运的是，今年的更新很大:随着iOS 16 的推出，我们终于可以*编辑和撤销已发送的消息，这两项功能在其他平台上已经存在了一段时间。*

不过，这些特性并不是没有星号，而且有好几个。首先，这些功能仅适用于iMessages。短信(绿色气泡)文本不能编辑或发送，这意味着任何发送到Android设备的文本仍然是永久的。也许有一天， [苹果将采用RCS](https://lifehacker.com/whats-rcs-messaging-and-why-should-you-care-1832235783) ，这将允许信息在iOS和Android上不被发送和编辑。目前，iMessage仍然无法与Android兼容。

另一个警告:收件人仍然可以通过点击编辑历史来查看您的原始消息(错误、错报等)。如果你不发送信息，他们不会看到，但他们会知道你发送了一些东西。

解决了这个问题之后，让我们来关注这些特性是如何工作的，以及它们应该用来做什么。

## **如何编辑已发送的iMessages信息**

当你发送一条iMessage时，你有15分钟的时间编辑它。在这15分钟内，你最多可以编辑一条iMessage信息五次，但一旦窗口关闭，这条信息就永远被锁定了。若要编辑信息，请长按它，然后选择“编辑”想怎么改就怎么改。然后轻按勾号。您也可以选择“X”来恢复原始消息。

您可能会试图依赖这个编辑功能来保存那些让您后悔发送的令人尴尬的消息。如果你有第二种想法，可以快速编辑有争议的文本，那么风险会小很多。然而，如前所述，苹果并没有设计这一功能。每当您编辑邮件时，新邮件都会带有“已编辑”标签。然后，任何人都可以点击此标签来查看邮件的完整编辑历史。如果您编辑邮件五次，邮件的所有五个版本都会出现在此处。

更糟糕的是，任何使用iOS 16的人*而不是*,都会将每个编辑视为一条新消息，前缀为“编辑到”不管怎样，如果你把信息改成完全不同的，每个人都会知道。

知道了这一点，编辑功能最适合用来纠正打字错误或其他小错误——你不介意你的联系人在编辑历史中看到的事情。我希望这个特性能终结这样的消息。像这样。像这样。像*这个*。尽管它们很有趣，但你可能不想让自己尴尬。

然而，有时候你真的会因为这样或那样的原因想要掩盖你的踪迹。在这些情况下，转到撤销发送。

## **如何撤销已发送的iMessages信息**

撤销发送可以让你快速回忆起一条考虑不周的iMessage。它还配有一个很好的动画，当消息气泡在你面前破裂时(非常苹果)。更好的是，它几乎没有留下任何痕迹。虽然收件人*会*看到你未发送信息的提醒，但他们看不到信息是什么。我接受这个交易。

当然，还有更多的警告需要考虑。你只有两分钟的时间来撤销已发送的iMessage，而不是15分钟的编辑时间。这也是完全现实的，因为在你给机会撤销信息或者甚至截屏之前，接收者已经看到了信息。此外，与编辑iMessage一样，撤销已发送的iMessage需要连接到聊天的所有设备都安装iOS 16。如果你用运行iOS 15.7或更早版本的用户撤销了一条iMessage信息，他们可能仍然会在自己的终端上看到这条信息，即使你没有看到。(以及您试图删除它的通知。)

尽管如此，这个特性仍然是有用的。若要使用它，请长按信息，然后选取“撤销发送”该消息将立即消失，并留下一个通知:“你未发送一条消息。”不管你撤销信息的原因是什么，如果对方很想知道它说了什么，不要感到惊讶。
