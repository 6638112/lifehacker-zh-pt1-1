# 如何在 Gmail 中批量删除邮件

> 原文：<https://lifehacker.com/how-to-mass-delete-emails-in-gmail-1847614640>

我最近收到通知，我的 Gmail 帐户已满 90%。我认为那是不可能的，因为 Gmail 必须有无限的空间。然后我想起来我从 2008 年就有了这个账户，从来没有清理过旧的、不必要的电子邮件。如果你也有同样的情况(越来越拥挤)，下面是批量删除 Gmail 中旧邮件的步骤-。

在顶部搜索栏中，输入“标签:所有邮件”和搜索选项过滤器将显示在搜索栏下方。你可以使用这些下拉列表按时间框架(对我来说，是 2014 年之前的任何事情，因为我的生活直到 25 岁左右才开始)、特定发件人(比如某个特定的时事通讯或你不再是其成员的组织)或任何带附件的东西(以清空宝贵的存储空间)来过滤电子邮件。

缩小参数范围后，单击左上角的框选择所有显示的电子邮件。如果你有超过 50 封电子邮件(你会的)，点击“选择所有符合这个搜索的对话”，这样你就不只是删除显示在结果第一页的电子邮件。然后单击垃圾桶图标。

在永久删除之前，Gmail 会将邮件保留在回收站中 30 天。如果你很快就需要这个空间，你可以进入你的垃圾文件夹，马上删除所有的项目。
