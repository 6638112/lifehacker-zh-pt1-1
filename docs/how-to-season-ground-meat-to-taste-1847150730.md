# 如何给绞肉调味

> 原文：<https://lifehacker.com/how-to-season-ground-meat-to-taste-1847150730>

无论你是在做肉丸、烘肉卷，还是其他一些需要对大量生碎肉调味的食物，都很难预测味道的结果，尤其是如果你正在使用一个新的食谱、一个模糊的食谱或一个完全由你自己开发的食谱。



如果你在做酱汁、汤、蘸酱和无肉菜肴，品尝和调整很容易，但当主要成分不仅生吃不安全，而且煮熟后味道截然不同时，调味“品尝”就有点难了。好消息是，在整个肉卷进入烤箱之前，你可以尝一尝你的肉卷——你只需要做一个小小的、珍贵的、小小的试验肉饼。

你的测试肉饼尝起来不会和你的肉卷或肉球完全一样，因为它有更大的表面积和褐变，但是它尝起来足够接近，让你知道你的混合物调味有多好。你所要做的就是取一点生肉、调味肉，做成小肉饼，然后在锅里煎。(为了方便，我一般用不沾。)煮吧。让它稍微冷却。把它放进你的嘴里。咀嚼它。吞下去。仔细想想。得出你的结论，如果需要的话，给你的肉卷或蛋卷加更多的调料。重复这个过程，直到肉尝起来像你想要的味道，但不要太野。很明显，你这样做的次数是有限的。即使很小的测试肉饼也会对你的肉块产生影响，所以测试次数最多保持在三次，一次不要吃超过一茶匙的混合肉。(否则你可能会吃到半个肉卷，我不喜欢你这样。)
