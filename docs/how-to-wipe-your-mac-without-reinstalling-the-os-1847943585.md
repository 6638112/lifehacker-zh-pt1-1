# 如何在不重新安装操作系统的情况下擦除 Mac

> 原文：<https://lifehacker.com/how-to-wipe-your-mac-without-reinstalling-the-os-1847943585>

重置 Mac 一直是一件令人头疼的事情:唯一安全的方法是格式化整个 Mac 并重新安装操作系统。每次你卖掉你的 Mac 电脑或者遇到软件的严重问题时，你都必须这么做。

Watch

对于一些人来说，这种摩擦足以无限期推迟过程，但随着 macOS Monterey update 的推出，装有苹果认证芯片的最新 MAC 电脑获得了新的全新安装功能，就像 iPhone 和 iPad 一样。你可以重置你的整个 Mac，抹掉所有内容和设置，而不需要接触操作系统。

## 首先，检查您的 Mac 是否受支持

该功能仅在部分 MAC 上提供。支持所有运行苹果硅(M1、M1 专业、M1 最大)处理器的 MAC 电脑。第二，对于英特尔 MAC 电脑，所有配备苹果 T2 安全芯片的 MAC 电脑都支持(2017 年至 2019 年)型号。

那就是硬件；软件方面，首先需要升级到 [macOS Monterey](https://lifehacker.com/31-new-macos-monterey-features-youll-actually-want-to-u-1847900965) 才可以做。如果你还没有，点击菜单栏中的**苹果**图标，进入“**系统偏好设置**”这里选择“**软件更新**”选项，升级到最新版本的 macOS Monterey。

当你使用全新安装功能时，苹果会将你从你的 Apple ID 中注销，删除所有数据，包括你的触控 ID 指纹。它还会禁用查找我的和激活锁，所以你不需要担心。

## 如何在 macOS Monterey 中干净安装 Mac 而不重新安装操作系统

现在你已经更新到了 macOS Monterey，是时候找到全新安装功能了，这个功能隐藏在系统偏好设置的菜单里面。在你的 Mac 上，点击菜单栏中的**苹果**图标，选择**系统偏好**。”

点击菜单栏中的**【系统首选项】**后，从菜单中选择**清除所有内容和设置**选项。

输入您的帐户密码，一个新的擦除助手应用程序将打开。选择您要删除的帐户。单击下拉列表可查看有关清洗过程的更多详细信息。点击**继续**按钮。然后，输入您的 Apple ID 密码以注销您的帐户。

在弹出菜单中，点击“**擦除所有内容&设置**选项进行确认。T 他动作删除过程将开始，你可以期待 y 我们的计算机在这个过程中重启几次。一旦完成,你的 Mac 就会寻找连接的键盘和鼠标。如果你用的是 MacBook，就不需要担心这个了。之后，您的 Mac 会要求您选择一种语言。

最后和最重要的步骤是激活。为此，只需在设置过程中连接 wifi 。一两秒钟后，您会看到“您的 Mac 已激活”提示。C 点击**重启**完成到的过程。

完成后，你会看到熟悉的“Hello”屏幕来设置你的 Mac，就像它是全新的一样。仅此而已。从这里，您可以选择将 Mac 设置为新的，或者您可以从 Time Machine 恢复备份。如果你要卖你的 Mac，你不需要做任何其他事情。
