# 魔戒的年龄，解释

> 原文：<https://gizmodo.com/lord-of-the-rings-ages-explained-1847683541>

Arda，物理世界 J.R.R .托尔金的 [*魔戒*](https://gizmodo.com/who-is-the-lord-of-the-rings-exactly-1845131474) 以此为背景，有着很远很远的历史，远溯到佛罗多的冒险毁灭索伦之前。其中一些将在亚马逊即将推出的流媒体系列 中探索，甚至更早的 中的 [故事也将在那里被取笑。但是阿尔达有着非常长的历史——这里有一个简单的指南来说明什么时候会发生。](https://gizmodo.com/why-the-glowing-trees-in-amazons-lord-of-the-rings-are-1847496857) 

精灵语 Quenya 中的 Ainulindal 是一个创世神话，它不仅是 Arda 本身的基础，也是未来几代人塑造它的神话生物。在时间和物质的概念存在之前，第一个存在的是*魔戒*对一种神圣力量的回答:Eru Ilúvatar。是伊鲁瓦塔尔唱着创世之歌来创造他们最重要的代理人，艾努尔——十四个天使般的梵拉和他们自己的仆人——以及被称为 Maiar 的原始灵魂，埃鲁伊鲁瓦塔尔实际上把创造神话的大部分工作留给了他们。

在 *The Silmarillion* 中，这一过程被描述为一场天堂般的即兴表演——ilu vatar 会向他的 Ainur 歌唱他对创作本身的概念和想法，而他们反过来会将这些主题发展成和声，最终以 Arda 本身的创作达到高潮。梵拉人和玛雅人随后带着阿尔达(获得了伊鲁瓦塔对这个创造物未来的预见)去引导它和它的存有们沿着他们神圣的计划前进——b 但是这并不都是愉快的交易。在艾努林达尔期间，埃鲁·伊鲁瓦塔尔和梅尔科尔之间的最初分裂——一个梵拉人，他寻求的不是与他的创造者的和谐而是与之类似的力量——开始于梅尔科尔试图给伊鲁瓦塔尔和其余的艾努尔人带来不和，在这个过程中腐蚀了许多与他的事业有关的玛雅人。

有两种实际的方法来测量阿尔达历史的时间，分为阿伊努尔计算的神圣社会和阿尔达自己的凡人种族。“灯年”是第一个以瓦良年来衡量的年份，但是在我们理解为传统年份的年份中——通过绘制行星围绕太阳的轨迹，这在阿尔达星球上*还不存在*——灯年是一个持续大约 15000 年的时期。它是以两个光源命名的，当梵拉第一次到达阿尔达表面时，他们发现了一片完全平坦、完全对称的陆地。收集阿尔达星球上的光线，梵拉人在巨大的塔楼顶端铸造了南北两盏灯；伊利亚在北方，奥勒玛在南方，而梵拉人自己则在阿尔达的正中心建立了他们的家园，存在于阿尔玛伦岛上两盏灯的最远处。

在建立了自己的堡垒并积聚力量后，梅尔科尔攻击了阿尔玛伦和两盏灯，摧毁了他们的塔，这样他们就可以直落阿尔达。破坏是灾难性的，将曾经单一的大陆分裂成四大洲:阿曼、中土世界(最初被称为或)、太阳之地和黑暗之地。

梅尔科的恶作剧夺走了阿达的光明后，梵拉人扎根于他们的社会并向西迁移，在阿曼建立了瓦利诺地区。由梵拉·亚瓦娜精心制作，并得到她的同胞梵拉·尼安娜的眼泪支持， [瓦利诺](https://gizmodo.com/why-the-glowing-trees-in-amazons-lord-of-the-rings-are-1847496857) ( L 奥雷林和特尔佩里昂)的两棵树为梵拉的新社会提供了光明。一束投射出金色的光，另一束投射出银色的光以及它们的露珠被梵拉·瓦尔达收集起来形成了星光的基础，这将会给阿尔达的其余部分一些随着伊鲁恩和奥勒玛的毁灭而消失的光的表象。《树之年》持续了大约一万四千年，当梅尔科从阿曼的囚禁中被释放出来，与原始蜘蛛 Ungoliant 签订了一个杀死树的契约时达到了高潮。然而，与 Illuin 和 Ormal 的命运不同，Laurelin 和 Telperion 发出的一些光在他们毁灭之前被保存了下来并分别给了 Maiar— Arien 和 Tilion— 中的两个，他们把他们带到了天空，分别形成了太阳(来自 Laurelin 的果实)和月亮(来自 Telperion 的叶子)。

我们知道我们将在亚马逊的*指环王*节目中一瞥这一时期，因为该系列的第一张图像 [给了一瞥](https://gizmodo.com/amazon-s-lord-of-the-rings-release-date-comes-with-a-br-1847408700) 劳瑞林和特尔佩里昂，但同样重要的是注意到这一时期也是第一批精灵和矮人在阿尔达觉醒的时候，开始了他们与梵拉和梅尔科本人们的长期联系。

我们实际上所知道的《指环王》历史上的*时代和之前的时期一样，统称为太阳年——这实际上是这两个天体首次存在的时间框架。第一纪元本身只是第一章，不到六百年，其中大部分时间梅尔科尔(现在被称为摩哥特人！掌控着中土世界。人类在这一时期首次出现在大陆上，其标志是中土世界的诺尔多里安流亡者的出现，这些精灵在树木之年来到阿曼，在他消灭了他们并偷走了含有劳瑞林和特尔佩里昂之光残余的强大珠宝后，追逐着他们。* 

三个人类部落(统称为伊甸人)、矮人、诺多、以及一群梵拉和梅尔与摩哥特人展开了一场被称为“愤怒之战”的战争。这场最终导致了莫哥特的毁灭和对中土世界的灾难性破坏，大部分贝里亚人沉入了海洋，无数的死亡，包括几乎大批死亡的伊甸人。

这一时期持续了将近 3500 年，为我们在《霍比特人》和《指环王》中所知道的大部分内容奠定了基础。这也是亚马逊即将推出的系列的主要背景。尽管摩哥特在愤怒之战中被毁灭，黑暗势力仍在暗中徘徊，等待着继续他对梵拉和埃鲁·伊鲁瓦塔尔的探索。但是在大多数情况下，第二个纪元开始于一个相对和平的时期，为阿尔达星球上的生命重建；精灵开始缓慢地向西迁移，无论是在中土世界的海岸定居还是迁移到阿曼的梵拉土地，而 Maiar Eö则提升了 Edain 的残余势力并建立了 Númenor 岛。然而这并没有持续多久。索伦，摩哥特的主要副手之一，在第二纪元进行到一半的时候，开始策划腐蚀精灵和人类的计划。

索隆·恩瑟尔斯被假扮成大地之精灵安纳塔，他铸造了力量之戒，以此来统治那些使用它的人。索伦骗局的发现引发了索伦和中土精灵之间的战争，最终第二个黑暗君主被赶回了魔多。但是几个世纪后，努美诺国王阿尔-法拉佐恩——一个现在已经延伸到努美诺之外的王国的首领，一个现在不信任精灵的王国——带领一支军队进入魔多，抓住索隆，把他囚禁在努美诺...在那里索隆迅速开始腐蚀努梅诺亚人，让他们反对梵拉，并说服阿尔-法拉宗试图登陆阿曼并进攻梵拉。然而，这是一个非常不寻常的举动:埃鲁·伊鲁·瓦塔尔出现了，他非常愤怒，不仅粉碎了努梅诺人和索隆的肉体，还完全击沉了努梅诺，并切断了阿曼与其余阿尔达的联系。他是怎么做到的？几千年后，在实际上已经造出了第一颗行星之后，他把阿尔达变成了一个球形的行星，而不是简单的平面。

剩余的努梅诺亚人逃到了中土世界，建立了刚铎和阿诺王国，几百年后，索伦重新恢复了他的身体形态。你知道接下来会发生什么，因为这是《魔戒之友》的开场:精灵和人类的最后联盟是为了击败索伦而建立的，最终他的身体再次被埃西铎王子在末日火山的脚步上摧毁。

第三个时代是我们最熟悉的时代，因为它是《霍比特人》和《T2》《指环王》事件发生的时候，尽管它们发生在持续三千年的时期的最后几十年。索隆又一次失去了实体形态，他在魔多和扮演一只大眼睛，在埃西铎返回刚铎的路上遭遇兽人突袭后寻找他丢失已久的戒指。精灵，继续他们在不朽的瓦利诺土地上向梵拉那边的过渡，分离出许多力量给上升的人类统治，然后...嗯，你知道的。《霍比特人》*和*《魔戒》*发生了，由于一个非常好的霍比特人和他的园丁，第三时代以索伦和魔戒的毁灭而结束。* 

在《指环王》的高潮部分，我们只看到了第四纪元的一小部分，从阿拉贡加冕为新统一的刚铎和阿诺王国的国王埃勒萨开始。精灵在中土世界的影响力逐渐减弱，因为他们中的许多人在这个时期向西航行，与梵拉人一起生活在阿曼。在弗罗多、梅里、皮聘和山姆集结了夏尔的霍比特人，将它从萨鲁曼和格里玛·沃姆东格的控制下解放出来，以及国王伊莱沙的军队击退了索伦的残余军队之后，我们所知道的阿尔达进入了一个数千年未见的和平时期。
