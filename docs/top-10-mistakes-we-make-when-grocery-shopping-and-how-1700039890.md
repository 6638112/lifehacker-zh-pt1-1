# 我们在杂货店购物时常犯的10个错误，以及如何改正它们

> 原文:[https://life hacker . com/top-10-errors-we-make-when-that-shopping-how-1700039890](https://lifehacker.com/top-10-mistakes-we-make-when-grocery-shopping-and-how-1700039890)

购买杂货是我们大多数人可能做得更好的普遍家务之一，无论是在食物上省钱的还是减少购物时间的。以下是我们在杂货店容易犯的十个常见错误，以及如何避免它们。

购物时，一周中的每一天都不平等。你可能已经知道不要在比赛日去杂货店买薯条和沙拉，因为人太多，但是如果你想买到最划算的东西，可以考虑在一周中的和一天中的 [(特别是在周三)](http://lifehacker.com/save-on-groceries-by-shopping-late-in-the-evening-espe-5809759#_ga=1.232787229.2004117440.1429967269) 购物，因为面包、肉类和其他保质期有限的食物往往会降价。

你也可以问问当地的商店什么时候降价，什么时候进货，这样你就能买到最新鲜的食物。当食品杂货 [在](http://lifehacker.com/a-month-by-month-guide-to-grocery-sales-5877851#_ga=1.2098483.2004117440.1429967269) 月打折时，囤积它们也是值得的；例如，四月是购买烘焙用品的好时机。

我们大多数人只想带着食品袋进出超市。为了尽快做到这一点，我们需要一个结帐策略。快速车道，尽管它的名字叫 [，可能并不是最快的车道](https://lifehacker.com/why-express-checkout-lanes-are-a-waste-of-time-5972618) 。 [迷信见鬼的13号](http://lifehacker.com/choose-lucky-lane-13-for-speedy-supermarket-checkouts-5822559#_ga=1.4571568.2004117440.1429967269) 或 [中的一条球道最左边的](http://lifehacker.com/choose-the-left-line-instead-of-the-right-for-a-possibl-1649425419#_ga=1.4571568.2004117440.1429967269) 可能是更好的选择。或者， [算一算](http://lifehacker.com/how-to-choose-the-fastest-line-at-the-market-5369876#_ga=1.4571568.2004117440.1429967269) 估计一下已经在线的人的篮子里有多少物品。如果你在自助结账，帮你自己一个忙，只刷商品 [，不要找条形码](http://lifehacker.com/stop-searching-for-the-barcode-to-get-through-self-chec-1697862623#_ga=1.7130545.2004117440.1429967269) 。

和其他零售商一样，杂货店都是 [设](https://lifehacker.com/think-of-supermarkets-like-obstacle-courses-to-avoid-ov-5902266) [来哄骗你](http://lifehacker.com/keep-your-grocery-bill-low-by-avoiding-these-marketing-5231106#_ga=1.7130545.2004117440.1429967269) 购买超过你需要的东西。即使你不相信他们的营销伎俩，也很容易抓住太多，尤其是当东西在打折或有东西吸引你的眼球(或鼻子——比如新鲜烤鸡的香味)时。了解在杂货店最常见的冲动购物在你去购物之前，一定要带着购物清单去购物，如果物品容易腐烂，避免大量购买。还有，把孩子留在家里。

另一方面，我们中的一些人偶尔会买太少的食物，忘记我们需要的东西。然后我们在最后一分钟去购物，或者更糟，只是去买外卖。再次强调，购物时一定要有清单。如果你时间不够， [一个备用杂货清单](https://lifehacker.com/keep-a-standby-grocery-list-for-quick-and-easy-shopping-5896743) 你一直喜欢的食材可以帮上忙。储备 [餐具室必需品](http://lifehacker.com/the-ingredients-you-should-have-in-your-pantry-for-quic-1606602875#_ga=1.17228090.2004117440.1429967269) 并跟踪你的库存，或许可以使用类似 [餐具室](http://lifehacker.com/pantry-keeps-track-of-what-you-have-on-hand-5321379#_ga=1.17228090.2004117440.1429967269) 的应用程序，或者用老式的方法在你的餐具室 中列出一份实际的 [清单。](http://lifehacker.com/keep-your-shopping-list-in-your-pantry-to-remember-to-p-5857006) 

当水果和蔬菜当季时，它们不仅更便宜，而且味道更好，对你来说更健康 。如果你在本地购买，你也支持你的社区。几个资源会告诉你现在什么是当季的。还有 [【偷天换日】](http://lifehacker.com/ripetrack-shows-you-what-produce-is-in-season-on-a-colo-1570927537#_ga=1.35570210.2004117440.1429967269)[吃四季](http://lifehacker.com/eat-the-seasons-tells-you-whats-in-season-now-5590903) ，还有 [美食家‘互动地图](http://www.epicurious.com/archive/seasonalcooking/farmtotable/seasonalingredientmap) 。

总的来说，有些东西在一家商店买比在其他商店买更好，而某些商品 ，像名牌香料，在超市买并不总是很好。下面是在 [好市多和你的杂货店](http://lifehacker.com/what-to-buy-at-costco-versus-your-grocery-store-5883360) 买什么，为什么你应该 [在药店](http://lifehacker.com/buy-milk-at-the-pharmacy-and-other-places-to-save-on-g-5903496) 买牛奶，在 [当地民族市场](http://lifehacker.com/shop-at-local-ethnic-markets-to-lower-your-grocery-bill-5316914) 买什么或者不买什么，在 [全食超市和乔氏超市](http://lifehacker.com/what-to-buy-and-what-to-skip-at-whole-foods-and-trader-5889646) 买什么。 [废弃的杂货店](http://lifehacker.com/shop-at-a-salvage-grocery-store-and-save-a-ton-of-money-5971552) 和 [食品合作社](http://lifehacker.com/how-to-find-a-food-co-op-in-your-area-and-score-some-hi-5912937) 也是超市的替代品，可以帮你在食品上省下一大笔钱。另一方面，在网上购买食品杂货可以节省你的时间、麻烦，还可能省钱。多逛逛商店也能帮你省下一大笔钱，帮你决定在哪里买什么。

只买你一周需要的东西听起来像是一个节俭的策略，但是当我们这样做的时候，我们最终会为我们放在购物车里的大部分东西支付全价。虽然你不想买太多的食物，但是如果你的冰箱或食品柜还有空间，那就储存一些你知道打折时会用到的东西。(使用 [冰柜烹饪](http://lifehacker.com/the-best-way-to-freeze-meals-for-cooking-later-in-the-o-1610251969) 为一周或一个月的晚些时候准备一吨饭菜，并用完你储存的食品杂货。此外，两周的购物周期可以帮助你减少食物浪费。)

即使食品杂货正在打折，你怎么知道它们是好东西，你应该囤积起来？ [一本价格书](https://lifehacker.com/how-to-save-the-most-money-on-your-grocery-budget-with-1518202640) 。记下你经常在不同商店购买的物品的价格，你就再也不会上当受骗了。你会 [在杂货店](http://lifehacker.com/look-for-unit-pricing-to-save-money-at-the-grocery-stor-5987916) 寻找单价来比较商品，但要小心一些 [单价的诡计](http://lifehacker.com/how-the-unit-pricing-labels-in-stores-can-trick-you-int-1641793755)——不一致的单价或令人困惑的度量。

在 之前，你已经听过这句话了，但它值得重复:吃饱了再去杂货店购物。否则，店里的一切可能都太诱人了。你也可以 [在购物的时候嚼薄荷糖](http://twocents.lifehacker.com/chew-mint-gum-while-shopping-to-avoid-retail-temptation-1574769469) 以避开商店的气味营销策略，戴上耳机以防 [商店的音乐会操纵你](http://lifehacker.com/how-stores-manipulate-your-senses-so-you-spend-more-mon-475987594) 花更多的钱。

简而言之，购物时你需要一个计划。也许最重要的计划是你的饮食计划，它解决了几个常见的错误。以下是如何 [开始计划你的每周膳食](https://lifehacker.com/how-to-plan-your-weekly-meals-stress-free-30791921) ，如何在预算紧张的情况下做，以及一些我们最喜欢的膳食计划应用 。
