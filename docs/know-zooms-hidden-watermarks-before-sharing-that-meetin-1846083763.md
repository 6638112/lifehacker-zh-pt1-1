# 在共享该会议之前了解Zoom的隐藏水印

> 原文:[https://life hacker . com/know-zooms-hidden-watermark-before-sharing-that-meetin-1846083763](https://lifehacker.com/know-zooms-hidden-watermarks-before-sharing-that-meetin-1846083763)

在你公开分享Zoom会议的私人内容之前——最好是出于高尚的原因——要知道会议主持人有很多不同的方法可以确定你是泄密者。这可能会带来一大堆法律或专业问题。

我有点震惊地发现,发现哪一个与会者应对泄露会议信息负责是如此容易。虽然我仔细研究了我的Zoom Pro帐户设置的内容，但我从未偶然发现Zoom的水印功能——主要是因为它们是为更好的企业帐户管理员保留的。但是它们非常强大，而且我不希望大多数人知道它们。(向 [脱帽致敬，向](https://theintercept.com/2021/01/18/leak-zoom-meeting/) 拦截，为大家指出这些方法。)

一家公司可以通过两种不同的方式为其Zoom会议添加水印。首先，有一个明显的问题:视频水印。如果一家公司启用了这一功能，那么当用户观看共享屏幕时，他们的部分电子邮件地址将会叠加在视频上；类似地，分享他们的屏幕的人*会将其电子邮件地址的一部分粘贴到视频上，就像这样:*

如果一家公司正在使用这种水印功能，这是显而易见的——我不相信你会看到任何类型的警告图标。你需要寻找的只是在屏幕分享时你脸上难以看到的文字。在一次常规的视频聊天中，这段文字*不会*出现在网络视频上(如果我没看错Zoom的指南的话)。

第二种方法更加阴险，我怀疑紧张的公司会比第一种更频繁地使用这种方法。这是一种音频水印，大多数人完全察觉不到，除非他们知道要寻找什么——缩放会议中绿色挂锁图标右侧的一个小小的绿色图标，表明该功能处于活动状态。

好消息(对泄密者而言)是，录音长度至少需要大约两分钟，以确保音频水印会出现在会议的任何官方录音中。少于此，你就孤注一掷了。您的音频可能有水印；你可能没有一个。

坏消息是，无论Zoom在音频中注入什么来识别某人，你都无法听到。 Zoom的方式它的音频水印是Zoom自己应要求为公司分析的东西——t这里没有一个神奇的软件工具，公司可以用来从泄露的音频中提取水印，你也不能确保你已经混淆了会议的音频，足以弄乱任何隐藏的水印。有太多的潜在途径。

因此，如果你被要求登录官方Zoom帐户参加会议，并且你看到任何一个水印工具都已启用，那么在记录和共享其内容时请三思——即使你这样做是出于正当、道德或法律上适当的原因。你分享的东西可能会被追踪到你，即使你认为你在以一种聪明的方式掩盖你的秘密。

而这些只是Zoom的水印工具。公司还有很多其他方法来追踪Zoom泄密者，包括:查看用于连接会议的设备，并将其与会议截图或视频进行比较；检查共享缩放呼叫中的参与者，以基于他们在屏幕上出现的位置来查看谁可能已经共享；以及无意中分享了你桌面的某个部分或者其他识别特征(比如带有你电子邮件地址的弹出通知 ) 。《拦截》列出了一个相当全面的清单，列出了潜在泄密者可能会把事情搞砸的各种方式。值得一读。
