# 为什么你应该在宽恕之前申请学生贷款退款

> 原文：<https://lifehacker.com/why-you-should-apply-for-a-student-loan-refund-before-f-1849563860>

现在你知道了，乔·拜登总统的政府正在推进一项计划，该计划将使数百万美国人获得高达 10，000 美元的联邦学生贷款减免(如果他们获得佩尔助学金 ，则最高可获得 20，000 美元 [)。但如果你是那些在疫情冻结还款期间继续偿还贷款的人之一，你可能会想，你是否能收回这些钱。虽然](https://lifehacker.com/how-to-find-out-if-you-have-a-pell-grant-1849453465) [一直都有可能获得退款](https://lifehacker.com/how-to-get-a-refund-on-federal-student-loan-payments-yo-1846790933) ，但即将到来的宽恕改变了一些事情。



## **获得联邦学生贷款还款**

如果你从 2020 年 3 月开始付款，当付款变成自愿时，你可以得到退款。联邦学生资助网站 主张你应该直接联系你的贷款服务商来申请。现在，你应该确保在申请贷款豁免之前做这件事*，尤其是如果一旦你拿回你的钱，你仍然低于最低 10，000 美元(或 20，000 美元)。*

退款将以现金形式到达你手中，根据美国消费者新闻与商业频道[，因此如果你自 2020 年 3 月以来支付了 3000 美元，还剩下 3000 美元，你的余额将回到 6000 美元。](https://www.cnbc.com/select/how-to-get-a-refund-for-federal-student-loans-paid-during-the-pandemic/) [据 PBS](https://www.pbs.org/newshour/education/how-to-get-a-student-loan-refund-if-you-paid-during-the-pandemic) 报道，任何人的疫情付款导致其余额低于最大豁免额，退款将自动发生。查看您的余额 [这里](https://studentaid.gov/fsa-id/sign-in/landing)；如果你的余额大大超过了完全原谅的门槛，那就不值得要求退款，因为你将来还得再付一次。

如果您没有获得自动退款，或者您不清楚是否获得了退款，请立即致电您的服务人员，询问您的退款是否会自动进行，或者您是否需要做任何其他事情。

## 如果你在疫情期间还清了所有的学生贷款会怎么样？

根据公共广播公司的说法，虽然总余额低于最大豁免金额的人预计会自动退款，但如果你在 2020 年 3 月到现在的某个时候还清了全部欠款，退款就不是自动的了。你将有资格获得贷款豁免，但首先你需要拿回你的钱，这将暂时恢复你的平衡。在申请退款之前，请确认您是否有资格获得宽恕。请记住，只有在疫情开始后支付的款项才有资格退款。

## 你现在应该要求退款吗？

不过，你可能想推迟做这些事情。虽然我们比宣布减免时更清楚这将如何运作，但更多细节仍可能会出现，说明冻结期间的付款将如何影响总体余额，因为在大多数情况下，利息也没有在暂停期间资本化。

专家提醒不要要求退款，直到关于他们将如何实施该计划的更多细节出台。在此期间，你应该做的是打电话给你特定的贷款服务商，了解他们需要什么来处理你的退款，以及他们建议如何推进。他们可能会想要一份你自 2020 年 3 月以来的还款记录，按贷方、支付金额和付款日期逐项列出。

利用这段时间把所有的事情都准备好，这样当你准备要求退款，并最终申请原谅时，你就有了你需要的一切。
