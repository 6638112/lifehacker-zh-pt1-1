# Lifehacker正在招聘一名个人金融作家

> 原文:[https://life hacker . com/life hacker-is-hiring-a-personal-finance-writer-1848213371](https://lifehacker.com/lifehacker-is-hiring-a-personal-finance-writer-1848213371)

Lifehacker正在寻找一位 [高级财经作家](https://boards.greenhouse.io/gomedia/jobs/3639098) ，这意味着我们正在寻找一种特殊的人:我们的财经作家涵盖专门的话题，但这些话题需要以平易近人、脚踏实地的方式进行解释。换句话说，你不是在为他妈的《金融时报》写稿，你应该能够说“他妈的”之类的话，同时在你关心的主题上保持你的权威。

这是一份全职工作，有一个了不起的作家团队，他们喜欢帮助人们，你在这个团队中的角色显然是与财务决策有关的一切:你的财务建议涵盖从预算和退休等基本问题到加密货币和NFT等更复杂的小众领域。显然，专业知识很重要，但为广大读者提供明智建议的能力也很重要。这意味着坚持己见和自信，但也要深思熟虑和感同身受。

这是一个高级职位，所以需要有经验。在钱的问题上，你基本上应该比我聪明(我知道一个合适的数目)。

推销想法往往是最难的部分；我们已经写了这么长时间，有时候感觉我们已经把所有的事情都写完了。但是如果你很好奇，有主见，并且能够用别人能理解的方式解释复杂的金融概念，你会喜欢这里的。有学生贷款、薪资谈判、航空里程、通货膨胀、税收、交易、诈骗，还有一些我不知道该怎么建议的事情，所以我只想说，你可以给出很多建议。

根据我们的工会合同，这也是一份最低工资为7.2万美元的工会工作。你可以在纽约、芝加哥或洛杉矶。如果你想申请， [这里是链接](https://boards.greenhouse.io/gomedia/jobs/3639098)——只要有思想、有经验、善良、有趣就好。不管怎样，这是我们努力的目标。
