# 疫情期间父母需要了解的儿童口腔卫生知识

> 原文：<https://lifehacker.com/what-parents-need-to-know-about-kids-dental-hygiene-dur-1846280158>

在过去的一年里，我们不得不仔细审查我们做的每一个约会，我们跑的每一个差事，以及我们接受或拒绝的每一个邀请。这是风险管理中的一个持续练习。有可能接触到新冠肺炎值得吗？值得拿着一份稳定的薪水，让我的孩子继续上学，买一些化妆品，理发，见一个朋友？



一个不幸的副作用是，一些父母选择跳过常规的预防性牙科保健预约，而喜欢让孩子呆在家里，远离细菌——或者他们发现现在预约很难，因为积压的访问和社交距离练习。正如 [CNN 报道](https://www.cnn.com/2021/02/15/health/kids-missing-dental-care-pandemic-wellness/index.html) 援引 [C.S .莫特儿童医院公布的一项关于儿童健康的全国民意调查](https://mottpoll.org/about) :

> 自疫情开始以来，40%的父母根本不寻求护理，理由是担心感染、办公室关闭和费用。
> 
> 这可不好，儿科牙医、美国牙科协会发言人乔纳森·申金博士说。“预防保健的延误可能会导致孩子患更多的蛀牙，”他说。“蛀牙的问题是，当它在童年开始时，它真的是成年后最强的风险指标。”

在疫情期间，尽可能多地进行常规体检和预防性牙科预约是非常重要的。还有一些日常卫生习惯你应该优先考虑，以帮助保持牙齿和牙龈的清洁和健康。

### 牙科检查应该什么时候开始？

你可能会认为推迟你孩子的第一次牙科预约，直到他们有几颗真正的牙齿需要检查。但是 [美国儿科学会](https://www.healthychildren.org/English/healthy-living/oral-health/Pages/Brushing-Up-on-Oral-Health-Never-Too-Early-to-Start.aspx) 和 [美国儿科牙科学会](https://www.aapd.org/resources/parent/faq/) 同意，孩子应该在他们的第一个生日前*或前*长出第一颗牙齿时去看牙医——以先到者为准——并且从那以后每六个月去看一次牙医。

这是因为即使是婴儿也会出现蛀牙——根据美国牙科协会 的说法，尽管这些乳牙只是即将长出的恒牙的临时占位符，但乳牙的蛀牙可能意味着恒牙蛀牙的风险更高。在严重的情况下，它会影响他们的整体健康。

### 在家刷牙

在这些定期检查预约之间，家庭口腔卫生是至关重要的，也是具有挑战性的。有一个原因就是有无穷无尽的 [提示](https://lifehacker.com/your-favorite-hacks-for-getting-kids-to-brush-their-tee-1840856622) 、歌曲和 [应用](https://offspring.lifehacker.com/get-kids-to-brush-their-teeth-with-the-pokemon-smile-ap-1844068485) 来帮助父母说服小孩子每天刷两次牙。这个原因是因为当一个小孩和一把牙刷在一起时，两分钟(推荐的刷牙时间)会感觉像是永恒。

然而，如果你从他们的乳牙开始出现的时候开始——或者甚至当他们的微笑只是牙龈的时候——你就可以在早期养成一个持续的习惯。(在牙齿长出之前，一块温水软毛巾就足够了——向你的牙医咨询何时改用软毛儿童牙刷。)

美国儿科学会和 [美国儿科牙科学会](https://www.aapd.org/resources/parent/faq/) 现在都推荐牙齿一长出来就使用含氟牙膏——但是不要过量。 [推荐量可能比你想象的还要少](https://offspring.lifehacker.com/your-kid-is-probably-using-way-too-much-toothpaste-1833130163) :三岁前一小抹(米粒大小)，三到六岁的孩子一粒豌豆大小。直到你可以教他们把牙膏吐出来，让他们的嘴向下倾斜，这样牙膏就可以滴到水槽里或者毛巾上。

一天中的时间也很重要:早餐后和就寝前是帮助孩子刷牙的理想时间，因为这可以让他们在一天的大部分时间里保持清洁。记住孩子睡觉前最不应该接触牙齿的东西是他们的牙刷是很有帮助的(例如，刷牙后的瓶子会抵消你刚刚做的好事)。

饮食也是家庭口腔卫生的一个重要组成部分，正如 CNN 报道的:

> 这要从限制糖的饮食开始，糖会滋生导致蛀牙的有害细菌。申金说:“孩子们白天应该喝的唯一饮料是水，也许还有牛奶。”。
> 
> 他说，最好完全避免含糖饮料，但如果你的孩子真的喝了，时间很重要。“如果你要吃，就在吃饭的时候吃。”

### 用牙线剔牙的两齿接触法则

你可能已经意识到孩子们也必须使用牙线，但是如果你不确定什么时候需要开始使用牙线，这很简单——只要他们至少有两颗牙齿接触。在牙齿接触之前，牙刷足以刷掉大部分牙菌斑。然而，一旦一对牙齿紧挨着，就该拔掉牙线了。

当他们学习的时候(或者当你还在为他们做的时候)，你可能更喜欢使用 [预穿线牙线器](https://www.amazon.com/Firefly-Kids-Flossers-Packs-Fresh/dp/B008IGT806/ref=sr_1_27?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/what-parents-need-to-know-about-kids-dental-hygiene-dur-1846280158&asc_source=&dchild=1&keywords=pre-threaded flosser for kids&qid=1613508723&s=beauty&sr=1-27&tag=kinjalifehackerlink-20#customerReviews) 。它们比普通的牙线更容易抓握和控制。然而，它们会产生更多的废物，所以一旦你可以改用普通的牙线，斯坦福儿童健康中心建议你尝试一下 [【线轴法】或者“线圈法”](https://www.stanfordchildrens.org/en/topic/default?id=flossing-and-children-90-P01852)
