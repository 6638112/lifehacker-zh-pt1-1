# 如何利用卡通作为健康的逃避方式

> 原文：<https://lifehacker.com/how-to-use-cartoons-as-a-healthy-escape-1845885947>

看我最喜欢的节目直到我的眼睛掉出来了有它的好处，尤其是在疫情期间。当情绪高涨时，在整个隔离期间，电视一直是一种健康的逃避和安慰。我已经就为什么我不是《T2》《T3》《不安全》《T4》中的莫莉的粉丝，或者我如何与《小溪》中的*克雷格*有点太过相关，进行了深入的交谈。我可以认同角色，并在他们的挣扎中看到自己，帮助我克服自己的心理障碍。

《媒体马拉松》[2018 年的一项研究证实了我们许多人已经知道的事情，即那些“与抑郁或焦虑作斗争的人能够通过媒体更多地了解人际关系，并将媒体参与体验作为沟通的跳板。”看一个虚构的人物经历与你相似的事情，提供了一个外部视角；而且，对我来说，漫画尤其以一种非威胁性和可联系的方式促进了宣泄。](https://journals.sagepub.com/doi/10.1177/0196859918814826)

[Brian Kong 博士，](https://briankongpsyd.com/)PsyD 专门研究流行文化和治疗，并分享了他关于观看有关抑郁症的卡通片的好处的专业知识。“我认为电视有时能够以一种更丰富、细致入微和更容易理解的方式描绘一种情感体验，”孔说。“媒介在视觉上偏离我们的日常现实，让我们沉浸在另一个想象的世界中，这是有价值的。”

### 使用复杂字符进行自我反思

受欢迎的成人游泳节目*中的疯狂科学家里克·桑切斯、*中的里克和莫蒂以及网飞的*中过气的演员博杰克·骑士*都是复杂的角色，他们将自己破坏性行为的重量内在化。在"休息和放松，"这一集里，瑞克和莫蒂花了一天时间放松自己，最终让有毒的自己从身体里脱离出来，并且必须与最坏的自己重聚，再次成为一个完整的人。在《杰克骑士》这一集里，杰克面临着自己伤害行为的后果，而他看似乐观友好的搭档花生酱先生则成了“抑郁之脸”。

这两集都细致入微地展示了我们认为的“最佳”和“最差”的部分，节目的“观点”与通常过于简单的好人或坏人的分类概念相反，”孔评论道。抑郁症有很多面，而《杰克骑士》中的里克和莫蒂将帮助观众识别和联系的层面分开。孔说:“在电视上看到自己的情感斗争画面，会非常有震撼力和说服力。”。

### 重塑你的个人奋斗

冒险时间是关于小男孩芬恩和他的魔法狗杰克在 Ooo 土地上寻求成为英雄的故事。主要的反派人物，冰王，戴着一顶神奇的王冠永生，反过来，王冠抹去了他的记忆，让他独自一人，无法与他人进行社交互动。与此同时，芬恩用他的勇敢和乐观的性格来避免处理创伤。

两个角色都表现出不同的抑郁因素，当他们在一个幻想的、虚构的土地上时，他们的故事就像一面镜子，反映了现实世界以及人们如何与精神疾病互动。孔解释道:“尤其是像《冒险时光》和《T2 未来》这样更有创意的节目，动画片可以激发我们的想象力——尽管是幻想——帮助我们以不同的方式思考生活和我们自己的生活，这对重新规划一个人的奋斗是必不可少的。”。

### 帮助“去认同”抑郁症

史蒂文宇宙是史蒂文的故事，一个半人半外星人被称为“水晶宝石”他和其他水晶宝石保卫地球免受外星力量，但他们真正战斗的力量是自我怀疑，抑郁和关系。 *Steven Universe* 帮助我走出了严重的崩溃，并给了我承认我的处境和理解我的感受的工具。

节目中有一首歌叫做“ [”在名为“正念教育”*的一集里，史蒂文的一个名叫康妮的朋友有一个抑郁的时刻。这首歌提醒她这只是一个想法——情绪是真实的，但它们不必消耗她。*](https://steven-universe.fandom.com/wiki/Here_Comes_a_Thought)

*另一部卡通片《大嘴巴》因通过虚构的人物将青春期的焦虑拟人化而闻名。一只“ [抑郁小猫](https://bigmouth.fandom.com/wiki/Depression_Kitty#:~:text=First%20appearance&text=Kitty%20Beaumont%20Bouchet%2C%20known%20as,Season%204%20alongside%20Nick%20Starr.) ”被介绍给《青春期的部门》一集中的一个角色，并通过观看*老友记*重播和吃冰淇淋来影响她忽视自己的责任。然后小猫躺在她的胸部，阻止杰西移动，象征着抑郁症的严重影响。*

*“心理学中有一些思想流派，如接受和承诺疗法，它们提供经验支持的干预措施，帮助个人将自己从抑郁或焦虑的想法中分离出来，并保持一种更疏远、更专注的态度。《大嘴》和的《史蒂文·宇宙》以一种观众可以接受的方式挖掘了这种普遍的理解，”孔说。“虽然不要把抑郁症或其他心理健康问题过于简单化很重要，但‘去认同’你的想法和感受对于不感到被消耗或陷入其中，并最终让它们消失是不可或缺的。”*
