# 通过这些网站跟踪 COVID 疫苗接种率

> 原文：<https://lifehacker.com/track-covid-vaccinations-with-these-websites-1846001330>

在美国批准使用第一种抗新冠肺炎疫苗后不到一个月，[CDC](https://covid.cdc.gov/covid-data-tracker/#vaccinations)报告称，我们已经分发了 1700 万剂疫苗，为 480 万人接种了疫苗。(全面保护需要两剂，所以大多数人只注射了第一剂。)如果你想跟踪疫苗在国内外的推广，这里有一些网站可以帮助你。



### 疾控中心疫苗跟踪器

[CDC 的页面在这里](https://covid.cdc.gov/covid-data-tracker/#vaccinations) ，它显示了分发和管理的总剂量，旁边还有一张地图显示每个州有多少人接种了一剂疫苗，如上图中的所示。

你可以在州地图上看到每 10 万人口中接种疫苗的人数，或者仅仅是目前接种疫苗人数的粗略统计。也有美国领土的统计，加上四个机构:监狱局，国防部，印度医疗服务，和退伍军人健康管理局。

该页面还包括分发和管理给长期护理机构(如疗养院)的疫苗数量。到目前为止，已有 320 万剂疫苗被运送到这些设施中使用，其中不到 50 万剂已经分发给居民和工作人员。

### 全局数据

[《我们的世界》在数据中有一页](https://ourworldindata.org/covid-vaccinations) ，上面有世界各国疫苗管理的各种图表。你可以观看显示每 100，000 人剂量随时间变化的动画地图，同样的动画条形图，以及更多的总剂量地图。

还有地图和图表显示每天服用的剂量，包括总量和占人口的比例。另一张地图根据各国的政策用不同的颜色对它们进行编码:它们是给一个还是几个高危人群接种疫苗，或者疫苗的可用性是否打算普及。(美国目前被列为两个弱势群体接种疫苗，尽管并非所有的州都遵循 CDC 的政策。)
