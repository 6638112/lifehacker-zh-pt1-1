# 你的冰箱的理想温度是多少？

> 原文：<https://lifehacker.com/what-temperature-should-you-keep-your-refrigerator-set-533534221>

你注意到你的牛奶变冷了吗？肉很快变坏？很可能你的冰箱设置了错误的温度。 [FDA 建议](https://www.fda.gov/consumers/consumer-updates/are-you-storing-food-safely) 将冰箱温度保持在 40°F(4°C)或以下，冰柜温度保持在 0°F(-18°C)，以最大限度地保持食物新鲜，并防止可能导致食源性疾病的细菌滋生。这里有更多的建议来帮助你确保你的易腐食品不会腐烂。

### 拿一个冰箱温度计

“我从来不相信一个电器的自报温度；在 Stack Exchange 的食品安全论坛上， [的一位用户](https://cooking.stackexchange.com/questions/4529/what-temperatures-should-i-keep-my-refrigerator-and-freezer-set-at/4531#comment4400_4542) 建议道:“我已经受够了有热点的烤箱，要时刻保持警惕。”。“有了我们的冰箱，我们的保鲜盒抽屉会定期结冰并部分冷冻我们的蔬菜，有了温度计，我能够确定为什么:下面比冰箱其他部分设定的 35 华氏度冷得多。至少 5-8 度。”移动温度计来检查冷点和热点，并相应地安排你的食物。

### 让你的冰箱装满

如果你经常打开你的冰箱和冰柜，那么 [让它们装满](https://lifehacker.com/why-your-freezer-needs-water-bottles-1844701670) 可能有助于保持温度一致。《裸体科学家》的冰箱爱好者 [说:“在一个空冰箱里，每次你开关门都会循环里面的大部分空气，用更热的空气取代冷空气，然后再次冷却。”。“如果冰箱满了，不仅需要循环和再次冷却的空气变少了，所有其他已经冷却的物品都会留在冰箱里。冰箱中有许多物品也会减少空气流通，这意味着每次开门和关门时交换的空气会更少。”](https://www.thenakedscientists.com/forum/index.php?topic=23913.msg260797#msg260797)

### 但是不要吃太多

没错，让冰箱/冰柜保持满满的状态更有效率——但是把它包装得太紧会限制循环和空气流动，最终迫使它更努力地将东西保持在合适的温度下，根据 Hunker 的说法 [。另一个风险是:冰箱里塞得太多会堵塞通风口，导致一系列温度和湿度问题，并可能导致一些食物更快变质。](https://www.hunker.com/12592157/can-you-overfill-a-refrigerator)

### 检查通风口

扩展到最后一点:为了保持温度稳定，空气需要在冷冻室和冷藏室之间畅通无阻。如果你看到冰箱里有霜，确保没有一条任性的面包堵住通风口，阻碍流通。你应该可以通过 [在网上为你的特定冰箱型号找到一个指南](https://producthelp.whirlpool.com/Refrigeration/Full-size_Refrigerators/Tips_and_Tricks/How_to_Check_the_Air_Vents_for_Blockage_-_Ensuring_Proper_Air_Flow) 。

### 避开“危险区域”

如果你想确保你的食物没有滋生可能让你生病的有害细菌，那就把它放在寒冷但不够冷的细菌滋生的“危险地带”。 [根据好管家](https://www.goodhousekeeping.com/appliances/refrigerator-reviews/a28496980/best-refrigerator-temperature/) 的说法，理想的冰箱温度在 35 华氏度(1.6 摄氏度)到 38 华氏度(3.3 摄氏度)之间

### 了解你的冰箱自然最冷的地方

“在正常使用的情况下，冰箱内的温度会有很大的变化，”[Stack Exchange 的](https://cooking.stackexchange.com/questions/4529/what-temperatures-should-i-keep-my-refrigerator-and-freezer-set-at/4531#comment4400_4542)hobodave 说。“冰箱最冷的部分是背面和底部。背面是因为那里有冷却元件，底部是因为暖空气上升。如果你不想让它们结霜，那就把它们从冰箱后面移走。”

### 不要把容易损坏的物品放在门里

冰箱门可能是冰箱里最温暖的地方之一，也是温度波动最大的地方——所以不要在那里存放任何可能变质的东西。“冰箱门上的东西很容易达到 59 华氏度(15 摄氏度)，而且经常如此，”hobodave 说。"把牛奶和鸡蛋放在你的门前会大大减少它们的保质期."事实上，对 [和](https://www.wellandgood.com/fridge-food-storage-mistakes/) 来说，营养师丽莎·德法齐奥建议*永远不要在门里存放牛奶、鸡蛋、奶制品或肉类。相反，把这些东西放在冰箱后面，那里温度更低，也更稳定。
