# 为什么1Password现在是Mac的最佳密码管理器

> 原文:[https://life hacker . com/why-1 password-is-now-the-best-password-manager-for-MAC-1848947731](https://lifehacker.com/why-1password-is-now-the-best-password-manager-for-mac-1848947731)

对于Mac爱好者和超级用户来说，1Password十多年来一直是首选的密码管理器。但是通过重新设计和新的杀手级功能，我们认为1Password 8现在是每个使用Mac的人的最佳密码管理器。

这款应用在Mac上如鱼得水，并提供了竞争对手无法提供的独特功能。Mac的内置密码工具非常出色(并且每年都在不断改进)，但1Password现在可以无缝集成到所有Apple产品、Windows和Linux上，提供功能对等和即时同步。

以下是更有说服力的证据，说明为什么1Password应该是您的首选密码管理器。

## 坚如磐石的基础

存在十多年有它的好处。1Password拥有优秀密码管理器的所有功能:您可以轻松地从其他密码管理器导入数据，自动填充功能可以在各种流行的浏览器上运行，并且它与iPhone和Android的密码自动填充功能进行了本机集成。

1Password的双因素认证也非常出色。一旦您启用该功能并将其添加到网站，它将在您尝试登录后自动填写六位数字代码2FA。

## 一个杀手锏:通用自动填充

1Password 8如此令人兴奋的最大原因是其新的快速访问栏和通用自动填充功能，这使1Password遥遥领先于其竞争对手，并实现了连苹果自己的密码应用程序都无法实现的功能。

使用 **Command + \** 键盘快捷键可以调出快速访问功能。按这些键搜索任何登录信息、查看更多详细信息或快速复制用户名或密码，而无需打开1Password应用程序。

通用自动填充，适用于任何支持的应用程序或网站，甚至更酷。假设您正在登录以放大您的Mac。以前，您必须打开1Password应用程序，并复制粘贴密码和2FA代码。现在，你可以使用**命令** + **\** 快捷方式调出快速访问栏，你会看到选择了自动填充功能的缩放登录选项。点击Enter和1Password将自动填充您的登录详细信息和双因素代码。

这一功能已经在许多主要应用程序中激活，但即使一个应用程序尚不支持，你也可以在手动登录一次后为应用程序启用自动填充。

## 界面和工作流程是同类最佳的

当1Password宣布将在1Password 8更新中使用电子设计应用程序的一部分时，技术负责人之间发生了很大的骚动，但我们很高兴地报告说，到目前为止一切都很好。1Password在M1 MacBook Air的后台安静地运行，不会过多消耗电池(如果你担心这将 [变成Dropbox场景](https://www.dropboxforum.com/t5/Dropbox-installs-integrations/Apple-Silicon-Massive-Battery-Drain/td-p/501091) ，不要担心)。

1Password还用更新、更大的按钮更新了它的界面，这些按钮与苹果新的macOS设计语言非常契合。不过，由于新的快速访问功能，大多数情况下您甚至不需要打开1Password应用程序。

## 生物识别解锁使输入密码更加容易

密码保护密码管理器是一把双刃剑:您肯定需要这种安全性，但您不想每次需要访问密码时都输入16位代码。

这就是1Password对触控ID和Apple Watch解锁的支持的真正亮点:只要设置了生物认证，你就可以自动填充密码并打开应用程序，而无需输入主密码。

## 瞭望塔仪表板帮助您评估您的安全健康状况

告诉某人(或你自己)在每个网站上使用唯一的密码很容易。做这件事完全是另一回事。但是有一种方法可以摆脱这种混乱，1Password的了望塔可能会有所帮助。该功能可以留意重复使用的密码和已经在网上泄露的密码。新的了望塔仪表板可以给你一个全面的安全报告，突出哪些登录需要更新。

如果你有很多糟糕的密码，也不要觉得太糟糕:1Password会帮你全部更改，并会建议一个强密码，直接存储，这样你就不用担心记住它了。

## 保管库和类别等专业级功能让您井井有条

虽然1Password 8简化了界面(新用户现在更容易导航)，但它并没有失去其超级用户优势。1Password支持多种数据类型，可以存储从护照、软件许可证到自定义笔记的任何内容。此外，您可以为不同的用户(工作、个人等)创建不同的保管库。

## 朋友和同事之间分享很容易

1Password的团队功能(每月19.99美元)使与同事共享密码变得非常容易。您可以轻松地为选定的团队成员提供密码访问权限，并根据需要共享特定的保管库。还有一个临时成员的客人帐户功能。

1Password的家庭计划以隐私为出发点。对于一个5口之家来说，每月4.99美元，这是保护全家众多密码的有效方法。默认情况下，所有家庭帐户都是私有的，但在需要时，成员之间可以轻松共享密码。

## 这些功能值得每月花费

1Password 8包含许多功能，但它是有代价的。在此次更新中，1Password将转向仅订阅价格，并将收取36美元/年或3.99美元/月的费用。那可不便宜。

但是通过此次更新，1Password已经表明它致力于通过每年的重大更新来改进密码管理器。这种积极的发展需要资源。定价是一个主观问题，因此完全由您来决定1Password的持续开发以及所有新的安全功能是否值得每月3美元。
