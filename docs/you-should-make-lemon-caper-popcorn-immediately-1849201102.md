# 你应该马上做柠檬马槟榔爆米花

> 原文：<https://lifehacker.com/you-should-make-lemon-caper-popcorn-immediately-1849201102>

我喜欢把粉末放在爆米花上，不管是奶酪的，还是简单的精、 [盐粉](https://lifehacker.com/pulverize-seasonings-before-sprinkling-them-on-popcorn-1831437477) 。因此，当我看到这位厨师关于马槟榔粉的插图文章时，我没有想到米饭或蔬菜或鸡肉或鱼——我想到的只是爆米花。但是刺山柑爆米花是不够的。我喜欢盐，但马槟榔有一种辛辣的味道，需要一点柠檬。这样，柠檬马槟榔爆米花就诞生了。

柠檬汁显然不行，因为湿爆米花很恶心。然而，Zest 的效果非常好，它提供了新鲜的、令人陶醉的香味和必要的风味来平衡马槟榔鲜味、咸味。

我按照《厨师画报》的指导做了马槟榔粉，只是我没有先冲洗马槟榔，因为我不相信要冲洗马槟榔。我还用微波炉加热了比规定的 8-10 分钟长一点的时间，因为在这段时间里它们没有完全变干。一旦它们变得又好又脆，我就让它们冷却，在我的香料研磨机里把它们磨碎，和把它们和新鲜爆开的玉米和柠檬皮一起搅拌。结果是神圣的，我强烈建议尽可能快地制作。

## 柠檬马槟榔爆米花

**配料:**

*   爆米花
*   橄榄油
*   一罐马槟榔
*   一个柠檬

将一罐马槟榔沥干，用纸巾轻轻压干。将它们转移到微波炉安全的盘子里，以两分钟的增量用一半的功率加热，直到它们变干变脆，你可以在外面看到白色的盐块。让它们充分冷却，然后在香料粉碎机中粉碎成粉末。

像往常一样爆你的爆米花。橄榄油在这里是一个自然的选择，但最重要的是，你用了比你认为体面的更多的脂肪。(我一般用 [杰西卡科斯罗](https://sqirlla.com/) 的比例 [1/2 杯油换 1/3 杯未爆玉米粒](https://skillet.lifehacker.com/youre-not-using-enough-oil-to-pop-your-popcorn-1827137012) 。)转移到一个大碗中，加入马槟榔粉，品尝直到你达到你想要的盐含量。在上面撒上新鲜的柠檬皮，然后享用。
