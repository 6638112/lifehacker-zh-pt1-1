# 如何去除书上的霉菌(并防止它卷土重来)

考虑到与霉菌 相关的 [健康风险，这可不是你想在家里或家附近看到的东西。不幸的是，洪水、潮湿和/或不当储存会导致霉菌在我们拥有的任何东西上生长，包括我们的书籍。](https://www.cdc.gov/mold/faqs.htm)

撇开健康问题不谈，霉菌会污染书页，导致书页散架，并让书页散发出明显的霉味。一旦霉菌开始在一本书上生长，如果你不清除它，它很可能会蔓延到你收藏的其他书上。以下是如何在你的书上发现、去除和防止霉菌生长的方法。

## 霉菌在书上是什么样子的？

一些最常见的 [书籍上霉菌生长的迹象](https://www.biblio.com/book-collecting/care-preservation/prevent-remove-mold-mildew/) 包括:

*   模糊生长 ( 任何颜色)
*   覆盖在多孔表面的白色细丝
*   过去水渍的证据
*   黑斑或污点

长在一本书上的霉菌 [可能看起来像](https://www.biblio.com/book-collecting/care-preservation/prevent-remove-mold-mildew/) 一片斑点，或一层片状粉末(通常是白色、黑色或灰色)覆盖在书页或其表面。

## 如何去除书上的霉菌

不幸的是，一旦霉菌在书里或书上生长，你在家里无法完全清除它，也无法保证它不会卷土重来。根据美国图书馆协会(ALA)的说法，你能做的最好的事情就是把发霉的书冷冻起来(先把它们放在密封的冷冻袋里)，然后用酒精把发霉的地方磨平。

## 如何防止霉菌在书上生长

防止霉菌在书上生长的关键是 [把它们放在不易受潮的地方](https://www.biblio.com/book-collecting/care-preservation/prevent-remove-mold-mildew/) 。这意味着避开地下室、壁橱，把它们靠着房子的外墙存放。你还需要尽可能控制存放它们的房间的湿度。运行除湿机和风扇(促进空气流通)会有所帮助。
