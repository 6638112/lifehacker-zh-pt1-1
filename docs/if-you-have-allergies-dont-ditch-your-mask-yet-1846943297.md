# 如果你有过敏症，还不要扔掉你的口罩

> 原文:[https://life hacker . com/if-you-hacken-don ' t-丢弃你的面膜-1846943297](https://lifehacker.com/if-you-have-allergies-dont-ditch-your-mask-yet-1846943297)

如果你完全接种了疫苗，你可能会期待越来越少地使用口罩现在 [疾病预防控制中心已经表示，完全接种疫苗的美国人在许多情况下可以不戴口罩](https://vitals.lifehacker.com/what-you-can-do-after-youre-vaccinated-according-to-th-1846430407) 。但是如果你对灰尘或花粉过敏，把它们留在身边可能是值得的。

花粉颗粒大小不一，一些风传播的树和草的花粉小至10至20微米。为了给人一种尺度感，N95口罩得名是因为它们能过滤掉95%大小为2.5微米的颗粒。由于花粉比口罩过滤掉的水滴和微粒大得多，这意味着这些口罩是抵御花粉和类似过敏原的有效屏障。 [外科口罩和多种布口罩](https://smartairfilters.com/en/blog/mask-filter-pollen-purifier-protect-hayfever/) 也是如此。有可能的是，无论你戴的是什么防护新冠肺炎的口罩，都可能对过敏原有效。

在疫情期间，一位医生 [注意到](https://www.healthline.com/health-news/how-covid-19-face-masks-can-reduce-outdoor-allergy-symptoms) 他们的病人在户外(花粉季节)和其他可能存在空气传播过敏原(如灰尘)的地方戴口罩时，更容易控制过敏症状。

如果你想使用口罩来预防过敏症状，选择一种过滤效果好的口罩(如多层编织棉口罩，或外科或N95口罩)，戴上它，让它紧密贴合你的脸。例如，如果面罩和鼻子两侧之间有缝隙，你可以通过缝隙吸入充满花粉的空气。

面膜不会是解决你季节性问题的完美方案，因为一些过敏原可能仍然会刺激你的眼睛和皮肤，但它可能会对你否则会吸入的过敏原有很大帮助。如果你除了过敏还有哮喘，你可能会发现(像我一样！)在凉爽的日子戴上口罩有助于防止因 [运动诱发的哮喘](https://vitals.lifehacker.com/why-you-wheeze-when-you-exercise-in-the-cold-1753746217) 带来的喘息。所以，即使你现在没有过敏问题，也要为今年秋天可能遇到的凉爽天气和豚草花粉保留那些口罩。
