# 算了，我们去喝粉红杜松子酒吧

> 原文：<https://lifehacker.com/screw-it-lets-drink-pink-gin-1845597724>

大家*过得怎么样*？我，我自己？我很累了。这一整周我都有轻微的头痛，因为我一直在喝酒，但也因为这一周非常非常令人讨厌。



我不会列出这周过得很糟糕的所有原因。大多数是众所周知的，尽管有些是个人的。我想说的是，我非常分心，我的注意力不集中，甚至我的工作——一份我非常幸运拥有的可爱的工作——感觉，如果说不是不可能，至少比它应该的要困难得多。

我们目前所处的分心边缘状态正是那种需要粉红色杜松子酒的情况—杜松子酒由于加入了苦味而呈粉红色。(我觉得它看起来更像橙色而不是粉色，不过管它呢。)如果你曾经啜饮过干马提尼酒，并且一直在说，“这款马提尼酒可能更烈，颜色也是珊瑚色的，”这款酒最适合你。

有些人简单地用苦味酒冲洗冰镇过的杯子就能调制出粉红色杜松子酒，但我喜欢我的加了许多苦味酒的杜松子酒——大约六杯。由于这种饮料主要是杜松子酒，稀释非常重要，所以搅拌的时间要比你认为需要的时间长。如果这仍然看起来像太多杜松子酒，你可以添加一个粉红色的杜松子酒&滋补品，这是一个更合理的一点。粉红杜松子酒传统上配有一条柠檬皮(榨出的油)，但如果你能找到粉红色的柠檬(如上图)，你会得到额外的加分。要制作粉红杜松子酒，你需要:

*   2 盎司杜松子酒
*   六破折号安古斯图拉苦酒
*   1 条柠檬皮(最好是粉色柠檬)

将杜松子酒和苦味酒加入到装满冰块的搅拌杯中，搅拌至非常非常冷。滤入一个小酒杯，将柠檬皮中的油挤入杯中。喝吧。那就多喝点。也许如果你喝得够多，这个星期就会结束。
