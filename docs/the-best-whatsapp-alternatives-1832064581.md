# WhatsApp 的最佳替代品

> 原文：<https://lifehacker.com/the-best-whatsapp-alternatives-1832064581>

去年，脸书将 WhatsApp、Facebook Messenger 和 Instagram 的私人消息合并为一个单一的消息平台。虽然每个应用程序在手机的应用程序库中仍是一个单独的图标，但用户现在可以跨平台相互聊天，并享受新功能和端到端加密选项。然而，由于 WhatsApp 的隐私政策 即将发生变化，这也意味着 WhatsApp 的用户数据将从 2 月 8 日开始与脸书及其合作伙伴共享。这适用于所有 WhatsApp 用户——即使他们不使用脸书、Instagram 或 Messenger。

Watch

脸书通过应用内通知提醒 WhatsApp 用户注意这一变化，但没有解释脸书将收集 WhatsApp 用户的数据类型。要找到这一点，你必须通读 WhatsApp 网站 上的 [新隐私政策。根据 2021 年 1 月 4 日发布的最新版本，脸书现在将收集所有 WhatsApp 账户的以下信息:](https://www.whatsapp.com/legal/updates/privacy-policy?eea=0#privacy-policy-updates-how-we-work-with-other-facebook-companies)

*   帐户注册信息，包括您的姓名和电话号码。
*   你的 IP 地址。
*   网络和服务信息。
*   使用 WhatsApp 的设备的技术信息。
*   与其他用户和企业的通信数据。
*   应用内交易数据。

该政策还表示，脸书可能会收集“隐私政策中题为‘我们收集的信息’一节中确定的其他信息，或者在通知您后或在您同意的基础上获得的其他信息。”

这与 WhatsApp 之前的隐私政策相比是一个很大的变化，用户可以选择不与脸书分享他们的数据，但可以继续使用该应用。不幸的是，新政策要求用户同意这些变化，或者在 2 月 8 日政策更新后停止使用该应用。

可以理解的是，许多用户对隐私政策、平台合并，甚至是脸书对 WhatsApp 的收购都感到不满。在用户隐私和数据方面，脸书并没有干净的犯罪记录，我们中的一些人只是不想被束缚在像脸书这样的大公司 T2 T3 的生态系统中。因此，让我们来看看 WhatsApp 和 Facebook Messenger 的替代品，它们将让你与脸书的多平台聊天世界彻底决裂。

### [**信号**](https://signal.org/)T5】

Signal 是一个经常被推荐的消息应用程序，因为它具有高度的隐私性、用户安全性和透明度。Signal 支持文本和群发消息；图片、音频、视频和文档的媒体消息；甚至用户之间的语音/视频通话。Android 版本也可以作为短信应用，取代现有的 Android 或三星消息应用。

Signal 得到了爱德华·斯诺登(Edward Snowden)和许多其他隐私/数据安全倡导者的支持(他们的评论出现在该应用的 [主页](https://signal.org/) 的显著位置)。信号 app 中的所有通信都是端到端加密的，你可以 [为自动删除的消息](https://lifehacker.com/how-to-make-sure-your-disappearing-signal-messages-actu-1825921690) 设置定时器。最近的功能更新还包括内置的照片模糊工具和新的 PIN 安全设置。

Signal 可以下载用于 [iOS、Android、Windows、macOS 以及 Debian-Linux 发行版](https://signal.org/download/) 。

### [**盛博**](https://www.viber.com/)T5】

Viber 的聊天功能包括群聊、即时语音/视频消息以及数以千计的 gif 和动画贴纸。您甚至可以安装聊天扩展来共享 YouTube 视频、Spotify 曲目、位置信息等。Viber 还支持音频和视频通话。(虽然 Viber 用户之间的通话是免费的，但您可以使用 [Viber Out 服务](https://account.viber.com/en/) 向任何座机或手机号码拨打付费国际电话。)

Viber 上的所有消息都是端到端加密的，可以定时自我删除。Viber 还包括一个可信联系人功能，可以通过交换密钥来验证其他聊天参与者的身份。如果您的联系人将来更改了他们的帐户详细信息，该应用程序将会更新您。

Viber 会显示你的状态，就像脸书信使一样，但是如果你需要隐藏你的状态，你可以禁用它。您还可以禁用已读回执、隐藏对话以及使用 PIN 码锁定邮件。

Viber 在 [安卓](https://play.google.com/store/apps/details?id=com.viber.voip&referrer=adjust_reftag%3Dc6rrmLFvkaBNc%26utm_source%3DWebsite%26utm_campaign%3Ddownload%26utm_content%3Den%26utm_term%3D1) 和 [iOS](https://itunes.apple.com/app/id382617920?mt=8) 上都有，可以 [同步到你的 Windows 或者 Mac 桌面](https://www.viber.com/download/) 。

### [**电报信使**](https://telegram.org/)T5】

像这里列出的许多其他应用程序一样，Telegram 为所有通信和自动删除设置提供端到端加密。它允许用户发送文本和媒体消息，并支持群发消息，在一个聊天线程中可以包含多达 20 万人。

Telegram 基于云的聊天运行在全球分布的服务器上，Telegram 声称，这种服务器可以比任何其他可用的消息应用程序更快地传递消息。这也意味着您可以在所有设备之间同步消息，并为您的共享媒体获得无限的存储空间。如果你是一名开发者，Telegram API 完全是 [开源的，可以免费使用](https://github.com/telegramdesktop/tdesktop) 。

Telegram 适用于 [Android](https://telegram.org/dl/android) 、 [iOS](https://telegram.org/dl/ios) 、 [Windows Phone](https://telegram.org/dl/wp) 、 [Windows 桌面](https://desktop.telegram.org/) 、[Linux](https://desktop.telegram.org/)(32 位和 64 位都有)、 [macOS](https://macos.telegram.org/) ，还提供了 [原生 web 版 app](https://telegram.org/dl/webogram)

### [**GroupMe**](https://groupme.com/)T5】

顾名思义，GroupMe 是一种提供跨平台聊天的群组消息服务。它归 Skype 所有(Skype 归微软所有)，所以它不像列表中的其他服务一样是开源的。它也受到与所有微软产品相同的数据收集做法的制约，它没有加密功能，也没有我们包括的其他应用程序的许多安全功能。然而，使 GroupMe 与众不同的是它对群聊的关注，以及其他一些有趣的功能。

GroupMe 允许用户共享视频、图像和文档，可以通过“画廊”模式查看，该模式显示了一个群历史中共享的所有媒体。您甚至可以向群组中的特定用户发送直接消息。GroupMe 还支持位置共享，以及“喜欢”消息和基于应用程序的自定义表情集等次要功能。

有 [安卓](https://play.google.com/store/apps/details?id=com.groupme.android&hl=en_US)[iOS](https://itunes.apple.com/us/app/groupme/id392796698?mt=8)[Windows](https://www.microsoft.com/en-us/p/groupme/9nblggh5z4f2?activetab=pivot:overviewtab)group me 应用，还有一个 [基于网络的聊天客户端](https://web.groupme.com/signin) 。然而，也有一个基于短信的 GroupMe 版本，你不需要应用程序甚至智能手机就可以使用。用户向 GroupMe 电话号码发送 [文本特定命令](https://groupme.com/en-US/sms) 以创建和进行群聊。这个基于文本的系统甚至允许你管理你的群和改变聊天设置。
