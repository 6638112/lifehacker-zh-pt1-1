# 如何制作《我独自生活》Ganache

> 原文:[https://life hacker . com/how-to-make-I-live-alone-ganache-1846887459](https://lifehacker.com/how-to-make-i-live-alone-ganache-1846887459)

一个人生活的好处之一是，你可以随心所欲地解释“正餐”这个短语。即使你的同居者的饮食偏好与你的一致，他们的存在也会抑制你更野性的倾向，比如用微波炉做一碗ganache，然后用勺子吃(这只是一个例子)。

我把这道菜叫做“我独自生活”ganache，这是一道非常自由的甜点。这很容易制作，而且由于缺乏测量，它永远不会完全一样——这很有趣。你所要做的就是将一些半甜的巧克力片放入碗中，用浓稠的搅打奶油覆盖，然后用微波炉加热，直到巧克力完全融化。你可以用它蘸东西(比如水果)或倒在东西上(比如冰淇淋或蛋糕)，或者你可以让它静置冷却，直到它足够坚硬，可以直接舀进嘴里。我把凉了的咖啡放在微波炉里，想吃的时候就吃一勺。

当然，你不必独自生活来享受这种巧克力，你只需要在你的ganache碗周围建立界限，或者致力于分享。幸运的是，这很容易做，所以如果你家里的其他人决定加入，你可以再做一碗。

由于只有两种成分，这两种成分的比例是你唯一需要担心“控制”的事情我总是盯着它。你不会想把巧克力片完全浸没在奶油中——你会想看到许多巧克力片在白色的海洋中露出来。如果你想要一种在室温和常压下保持可蘸的柔软的ganache，你在上面的照片中看到的比例是完美的，但我最终添加了另一小把来巩固它。幸运的是，你不会真的把这东西弄混——这是巧克力和奶油，融在一起的。你能行的！

### 如何制作《我独自生活》Ganache

**配料:**

*   半甜巧克力片
*   重搅打奶油

将适量的巧克力片倒入微波炉专用的碗中。什么是“健康量”？我的朋友，这是你当时需要的任何东西。在薯片上浇上厚厚的奶油，就像它含有维生素D一样。牛奶和薯片都是谷类食品。加入刚好能盖住巧克力的量——奶油加得越多，你的ganache就会越软。将碗放入微波炉，开足马力加热30秒。移开碗，搅拌巧克力(它会变得有点结块和僵硬)，然后把碗放回微波炉。用微波炉加热15秒，然后再搅拌一次。重复一到两次，当你接近“完全融化”阶段时，减少秒数。如果你的ganache看起来像牛奶巧克力，它会很松散，所以再加一小把巧克力片，让它更有质感，直到颜色变深。(你希望它看起来像顶部照片中的ganache。)

现在你已经准备好用它蘸东西，或者倒东西了。你也可以让它静止一段时间，然后变得结实，这正是我喜欢做的。如果你想变得花哨，你可以在你的巧克力创作上撒一点盐。如果你想做得更精致，你可以让它完全变硬几个小时，然后把它卷成小球，撒上可可粉做成松露。你可以随心所欲地做自己喜欢的菜，但我强烈建议保持这道菜的野性，把它放在微波炉里，这样你就可以在每次有心情的时候吃一勺。这就是享受的方式。
