# 这个 Chrome 扩展帮助你减少工作量

> 原文：<https://lifehacker.com/this-chrome-extension-helps-you-work-less-1848918834>

我们经常在 Lifehacker 上写关于生产力和生活黑客的文章；如何 [黑掉你的空间](https://lifehacker.com/13-clever-ways-to-create-more-storage-space-in-your-hom-1848905348) ，你的 [iPhone](https://lifehacker.com/62-yes-62-of-our-favorite-iphone-hacks-and-explainer-1848260080) ，你的 [简历](https://lifehacker.com/stop-saying-you-helped-on-your-resume-and-use-these-ve-1848425106) 和下一次 [工作面试](https://lifehacker.com/the-best-way-to-answer-tell-me-about-a-mistake-you-made-1848770251) ，甚至你的 [洗碗机平板电脑](https://lifehacker.com/the-smartest-ways-to-use-dishwasher-tablets-to-clean-yo-1848712078) 来创造更多的储物空间，更多的功能，更多的机会，更大的整洁度。但是当我们需要减少创造的时候呢？更少的压力，更少的期限压力，更少的事情要做？当我们被对目标和金钱的职业追求(以及对 Pinterest 值得信赖的组织的国内追求)弄得筋疲力尽时，我们真正需要做的是拔掉插头，让*根本不工作？由于脱离工作可能很难，现在有一个 Chrome 扩展可以帮助你做到这一点，它叫做 [生产力拦截器](https://www.productivityblocker.com/blocked) 。*

## 生产力阻碍者是个笑话吗？

算是吧。但不完全是。根据撰稿人兼联合创始人史蒂夫·纳斯(Steve Nass)的说法，这个最初是对“我们社会的‘永远在线’喧嚣文化”的模仿，具有合法的功能来阻止 80 多个纳斯和他的联合创始人科里·布罗西厄斯(Kory Brocious)认为“有成效”的网站，包括:Slack、Fiverr、Duolingo、Linkedin、和 Dropbox。“既然有这么多的 Chrome 扩展可以帮助你专心做事，我们觉得反其道而行之会很有趣，”Nass 说。

事实上，有超过 100，000 个 Chrome 扩展 可用，其中大多数预示着更快地安排会议，定位联系信息，控制失控的浏览器标签，以及“ [简化 Gmail 内的外联](https://www.oberlo.com/blog/google-chrome-extensions) ”(无论这意味着什么)。将我们自己从美国的“为工作而活”的心态中分离出来并不总是容易的，这种心态要求我们必须一直努力、证明和超越——同时在前进的道路上变得越来越有效率。(参见#Riseandgrind 标签的美化。)

## 生产力拦截器是如何工作的？

那么，为什么不在数以千计的 Chrome 扩展程序中加入一个来减少我们的工作时间呢？启用生产力拦截器后，所有试图访问“生产力”网站的行为都将被重定向到他们的 [自定义拦截页面](https://www.productivityblocker.com/blocked) ，上面印有“啊哦。看起来你已经尝试在预定的闲暇时间提高工作效率了。请仔细阅读我们下面的娱乐链接。”(链接中的带有诱人的无意义标题，如:*蹦床购物、* *即时内裤、*和*有人类证件的动物名单*。)

“顺便说一句，我们鼓励人们向我们发送我们错过的任何有价值的网站，” Nass 补充道。"我们很乐意阻止这些，也是."

## **如何获得生产力阻止程序扩展**

生产力拦截器可以从 网站免费下载 [，也可以从](https://www.productivityblocker.com/) [Chrome 网上商店](https://chrome.google.com/webstore/detail/productivity-blocker/cfbhoifjdjphbjcjmkpilkdeplfomjbn) 下载。(点击**添加到 Chrome，添加扩展**。)下载后，通过单击浏览器上的生产力阻止程序图标，可以轻松打开或关闭该扩展。根据 Nass 的说法，“当你安装 Chrome 扩展时，它开始是关闭的，所以你可以随时打开它。”虽然它目前只能在笔记本电脑/台式机上使用，但创作者表示，他们可能会开发一个智能手机版本。

我们都应该少花一点时间沉迷于幻灯片，多花一点时间欣赏愚蠢的动物恶作剧视频，在 Reddit 辩论中选择立场，讨论在杂货店自助结账时使用手持扫描仪是否比使用平板扫描仪更好。
