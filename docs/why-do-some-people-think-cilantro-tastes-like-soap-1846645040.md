# 为什么有人觉得香菜味道像肥皂？

> 原文：<https://lifehacker.com/why-do-some-people-think-cilantro-tastes-like-soap-1846645040>

你可能认为你的舌头理所当然，或者至少没有完全理解它如何解读它遇到的的味道并将那些美味或难吃的信号发送到你的大脑的复杂性。

这一点尤为重要，因为不同的人品尝的东西并不总是一样的。事实上，当谈到香草芫荽叶时，你品尝到的新鲜泥土味道对其他人来说可能实际上像棕榄或香皂。当谈到时，有很多关于食物的常见神话要经历，但是对于一些人来说，香菜尝起来像肥皂的想法不是古怪的谣言。

### 为什么对某些人来说香菜尝起来像肥皂

当你咬一口富含芫荽叶的沙拉时，你是否尝到了满嘴的防腐剂，这取决于基因骰子的 T2 滚动。具体来说，归结于香菜中发现的醛类化合物——一个碳原子与另一个原子或原子团共享一个键的有机化合物，根据《大英百科全书》[——由于其显著的香味，它们经常被用于各种香水和清洁产品中。](https://www.britannica.com/science/aldehyde)

根据 T4 克里夫兰诊所的说法，醛类物质最好通过嗅觉感受器来检测，嗅觉感受器使你能够闻到东西。组织备注:

> 这种差异在分子水平上可能很小，但在嗅觉水平上却有很大的影响。人们怀疑对芫荽叶的厌恶很大程度上是由气味引起的(气味与我们的味觉直接相关。)

你是否品尝美味的香草或其他东西取决于你的遗传基因。在克里夫兰诊所指出的一项研究中，14000 名受访者接受了遗传特性测试，这可能解释了为什么有些人觉得芫荽叶很滑而其他人不觉得。大约 10%的人表现出嗅觉受体基因*或 6A2* 的优势，作者在 2012 年发表在风味杂志上的研究中写道，“这种基因对赋予芫荽叶特有气味的几种醛类具有很高的结合特异性”。

### 你的种族背景是一个因素

2012 年的研究代表了官方对芫荽叶这种两极分化草药的科学共识。尽管这只是一项研究，但它揭示了为什么一些文化和种族背景似乎都不喜欢芫荽叶的有趣见解。

根据这项研究，被调查的不同种族不一定都报告尝起来有肥皂味，但是对香菜的厌恶和 T2 种族有明显的关联。(女性也更有可能报告在食用草药时尝到肥皂的味道)。

正如 [自然写于 2012 年](https://www.nature.com/news/soapy-taste-of-coriander-linked-to-genetic-variants-1.11398) :

> 21%的东亚人、17%的欧洲血统的人和 14%的非洲血统的人说他们不喜欢这些东西...相比之下，3-7%的南亚人、拉丁美洲人和中东人不喜欢这种草药，因为这种草药在他们的本土菜肴中更常见。

有人可能会推断，这些偏好自然与世界各地烹饪中使用芫荽叶和芫荽——最终会长成芫荽叶的种子——有关。使用较多的文化中，品尝到肥皂味的人比例较小，反之亦然。

重点外卖？它其实并不是偏好的问题。对某些人来说，ilantro *确实尝起来像肥皂。与其嘲笑他们，我们可以同情他们的困境:他们永远不会知道正确制作鳄梨色拉酱的真正乐趣。*
