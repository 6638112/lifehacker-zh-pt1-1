# how to stop being so damn self deprecating 1848316538

> 原文：<https://lifehacker.com/how-to-stop-being-so-damn-self-deprecating-1848316538>

没有人喜欢自我膨胀，自嘲是一项重要的技能。另一方面，谦逊也有其局限性。如果你真的不能接受赞美，或者你总是贬低自己，那么你的自我贬低可能弊大于利。

虽然自嘲常常与谦逊(好事)联系在一起，但研究表明，它可能会促进自我破坏(嗯，坏事)。超越自己，太多的自嘲影响到身边的每一个人。无论你在哪个房间，一堆看似即兴的自怜评论都会让你不舒服。



幸运的是，就像 [打破任何坏习惯](https://lifehacker.com/top-10-ways-to-break-bad-habits-1694247761) 一样，你可以采取一些措施来改变你的行为。当然，真正在更深层次上提高你的自尊将是一个漫长的旅程。同时，这里有一些实用的建议来抑制你的自嘲。

## **识别行为**

打破任何坏习惯的第一步是确定你试图改变的是什么。想一想:你是否以自嘲作为幽默的主要风格？你是否无法在赞美别人的同时又不贬低自己？你有没有发现自己经常和周围的人比较？

坐下来想一想，对你来说，自嘲到底是什么样子的。如果你实时注意到了它，花点时间把它记在你的笔记应用程序中(这也将有助于识别你的触发因素——稍后会有更多相关内容)。

## **识别你的触发器**

当你识别出让自己崩溃的方式时，记下引发这些想法和感觉的情况。也许你很难接受老板的表扬，或者你在更大的群体中感到尴尬，或者你总是试图保护别人的感受。无论你的诱因是什么，认识到它们将帮助你预测什么时候你需要练习更自信的态度。



## **查找填充短语**

自我贬低的本能不会马上消失。试着准备一些你可以说的话来填补这个空白，即使是一些非常明显的话，比如，“你知道，我通常会在这里自嘲，但我正努力变得更加积极。”

这个 Reddit 帖子 提供了另一个例子:不要说“你为什么和我做朋友，我很可怜”，你可以把这种情绪重新定义为“我是怎么和你交朋友的，你太棒了。”自信是关键，即使这是在提升别人的自信。

## **告诉别人你的意图**

就像发誓要更多地去健身房，或者尝试戒烟一样，你的武器库中最好的工具之一就是建立责任感。

此外，任何关心你的人都可能会对你停止取笑自己的目标充满热情。如果你周围的每个人都在默默地乞求你不要再如此自嘲，你会给他们一个机会大声说出自己的想法。人们喜欢和其他自信的人在一起。



## **试试骂人罐子法**

一旦你开始认识到什么时候以及如何打败自己，试着建立一个系统来激励自己战胜这个习惯。金钱是一个巨大的动力，所以你可以使用 [“发誓罐子”法](http://lifehacker.com/break-bad-habits-and-save-money-with-the-swear-jar-me-1692013716) 或 [每当你的朋友发现你在做你想停止做的事情时，付给他们一美元](http://lifehacker.com/break-your-bad-habits-by-paying-1-for-each-offense-5946965) 。反过来也一样:奖励自己善待自己。

当你犯错时，不要开始自责——否则，你将不得不在罐子里再放一美元。

## **最后:对自己宽容点**

任何行为改变都是一种挑战。你的自我贬低很可能是源于更深层次的东西——一些你可能想 [和专业](https://lifehacker.com/how-to-select-the-right-therapist-for-you-1833246447) 谈论的东西。

[改变坏习惯](https://lifehacker.com/top-10-ways-to-break-bad-habits-1694247761) 不会一夜之间发生，所以当这个过程需要时间时，尽量不要对自己感到不安或沮丧。对自己实践一些善意。毕竟，一点点自爱不正是你想要做的吗？
