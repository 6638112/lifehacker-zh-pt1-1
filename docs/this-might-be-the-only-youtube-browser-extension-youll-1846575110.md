# 这可能是你唯一需要的YouTube浏览器扩展

> 原文:[https://life hacker . com/this-may-be-the-only-YouTube-browser-extension-youll-1846575110](https://lifehacker.com/this-might-be-the-only-youtube-browser-extension-youll-1846575110)

YouTube真是鱼龙混杂。它是我们在互联网上最喜欢的东西的家园，但使用网站本身的体验并不好——充满了令人分心的推荐、糟糕的个性化和广告、广告、广告——除非你利用桌面浏览器扩展来使它变得更好。

虽然我过去常常依赖一系列较小的插件来删除不想要的推荐，清理YouTube主页，并弥补个性化选项的缺乏，但有一个强大的开源扩展叫做Improve YouTube，它可以自己完成所有这些甚至更多。

在阅读了 [gHacks对该扩展的一系列功能](https://www.ghacks.net/2021/03/26/improved-youtube-enhances-youtube-in-meaningful-ways/) 的赞美之后，我安装了**Improve YouTube**——目前已经有超过80个了，并且会定期添加更多。它可以做任何事情，从为YouTube的网站( [或网络应用](https://lifehacker.com/how-to-install-youtube-as-a-progressive-web-app-on-wind-1846137659) )配备更好的控制和布局选项，到阻止广告、评论和不想要的频道推荐，以及*方式*更多。

改进的YouTube可用于 [、火狐](https://addons.mozilla.org/en-US/firefox/addon/youtube-addon/) 和 [Chrome](https://chrome.google.com/webstore/detail/improve-youtube-open-sour/bnomihfieiccainjcjblhegjgglakjdd/related) (因此其他Chrome浏览器如Edge也是如此)，两个版本都具有相同的功能。若要开始，请下载适用于您的浏览器的改进YouTube，然后单击URL栏旁边的改进管道图标。从这里，您可以选择九个菜单中的一个来访问Improve Tube的许多功能:

*   **【常规】**包括用于更改基本行为的开关，如标记观看的视频、更改缩略图外观、更改默认登录页面等。
*   **【外观】**是一个可视化的定制菜单，可以让你重新排列或隐藏网站的几乎每一个元素。我删除了“相关”视频部分并隐藏了评论——因为说实话，没人需要阅读 YouTube评论。
*   **【主题】**允许你将定制的主题应用到YouTube，改变字体大小和颜色，甚至打开内置的蓝光滤镜。对于那些怀旧(或固执)的人，如果你愿意，你甚至可以恢复“经典”的YouTube布局。
*   通过**“Player”**菜单，您可以定制视频播放器并调整YouTube的默认视频和音频质量。您甚至可以添加额外的视频控件，如播放速度、循环、截图和旋转视频按钮。
*   **“播放列表”**让您更改默认播放列表和自动播放行为。
*   **【频道】**有打开特定频道时看到的选项。
*   **【快捷键】**让您修改或添加用于控制网站和网络播放器的键盘快捷键。
*   **“黑名单”**菜单用于屏蔽整个频道或特定视频。
*   **“分析器”**将收集您的YouTube使用数据。默认情况下是关闭的。

Firefox和Chrome扩展页面上包含了改进YouTube功能的完整列表(以及使用指南)。你也可以在该项目的 [GitHub页面](https://github.com/code4charity/YouTube-Extension) 和 [官方网站](http://www.improvedtube.com/) 上查看开源代码和阅读更多文档。

这只是改进YouTube完整功能集的一个快速亮点，但还有更多内容需要探索，包括微小但有用的调整，如更改视频自动播放设置和添加新的键盘快捷键。其他，更大的功能包括广告拦截，一个带宽限制器，窗口全屏模式，等等。

公平地说，有些事情你*不能*用改进的YouTube做——比如 [把你的订阅组织到文件夹](https://lifehacker.com/streamline-your-youtube-subscriptions-using-folders-1846094297)——但是它更有可能做你当前大部分或所有与YouTube相关的浏览器扩展的工作。
