# 如果你想远程启动你的丰田车，你必须付费订阅

> 原文:[https://jalopnik . com/you-to-have-to-pay-for-a-subscription-if-you-wa-1848191246](https://jalopnik.com/youre-going-to-have-to-pay-for-a-subscription-if-you-wa-1848191246)

不得不 [付出](https://jalopnik.com/a-carmaker-s-23-billion-plan-to-keep-you-paying-long-a-1848172449) 换取 [东西](https://jalopnik.com/oh-good-volkswagen-is-trying-on-demand-feature-subscri-1846511749) 的时代，你的 [车辆](https://jalopnik.com/all-the-cool-features-in-your-car-will-probably-be-subs-1845658200) 已经降临 [配备](https://jalopnik.com/rivians-ipo-filing-leaked-how-the-company-plans-to-make-1847829458) 的时代终于来临了。订阅已经到达丰田。 [The Drive](https://www.thedrive.com/news/43329/toyota-made-its-key-fob-remote-start-into-a-subscription-service) 报道称，汽车制造商将要求配备远程启动功能的车主付费使用。

Watch

这一切都始于11月中旬的一篇Reddit帖子 。一位在 [r/cars](https://www.reddit.com/r/cars/) 上的用户发现了一些丰田的营销，这些营销指向该汽车制造商可能会使远程启动成为一项基于订阅的功能。但是没有人真的相信汽车制造商会完成它。以至于版主认为这是胡说八道，给这个帖子贴上了“潜在误导”的标签。但事实并非如此。一名丰田代表向司机确认，将需要订阅。

> 丰田发言人向*司机*证实，如果2018年或以后的丰田配备了丰田的远程连接功能，车辆必须注册有效的订阅(无论是免费试用期还是其他)，以便钥匙链启动汽车。

所有这一切的一个好处是，业主不会只是简单地订阅远程启动。远程启动是更大的车辆服务互联服务套件的一部分，包括远程启动、安全辅助、车辆服务更新、车载wifi、丰田自己的类似OnStar的服务Destination Assist以及丰田智能手机应用程序。为什么丰田决定把远程启动和这些东西捆绑在一起(大部分一般车主并不是真的需要)，这是我无法理解的。但是不管你喜不喜欢，你必须每月支付8美元或每年支付80美元来启动你的汽车。

好像订阅本身还不够，事情会变得有点混乱。你知道，远程连接服务有三年的试用期。三年后，将需要订阅。服务因车辆而异。从驱动器:

> 例如，配备Audio Plus的车辆可获得长达三年的试用期，而高级音响可获得长达10年的试用期。除此之外，司机还需要付费订阅才能继续使用远程服务。

因此，如果你驾驶2018年或更新的丰田汽车，你将会为远程启动车辆的简单便利而感到不必要的头痛。
