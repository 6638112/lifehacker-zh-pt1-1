# 做一个无聊的决定

> 原文:[https://lifehacker.com/make-a-boring-resolution-1845956345](https://lifehacker.com/make-a-boring-resolution-1845956345)

决心被高估了，我们都知道。即使你喜欢为新的一年制定崇高的目标，简单地宣布一个决心并不等同于制定一个实现目标的计划。但是你知道吗？你也不需要这么做。今年，做一个小而无聊的决定。

Watch

毕竟2020年已经够辛苦了，我们也不知道2021年会带来什么。你不必彻底改变你的生活；只挑一件能构成小改进的事情。如果你坚持这样做，你会比启动并放弃一个宏大的自我提升计划要好得多。

例如:有一年，我们的主编爱丽丝·布拉德利决定每天早上整理床铺。她说，这不仅是她多年来最喜欢的决心，而且从那以后她一直坚持下去。“这些小事实际上对你的生活质量有很大的影响，”她补充道。

我们说的是“多吃水果”你没有改变你的饮食，你只是打算每天吃一个苹果。或者“保持我的桌子整洁”不是你的整个*房子*，只是你的桌子。换句话说，我们在寻找一些不仅容易做，而且完全没有创意的东西。一些不会让你在社交媒体上吹嘘的东西。一些除了你自己之外对任何观众都没有吸引力的东西。

对我来说，是卷发。每个健身房的兄弟都做二头肌弯曲，但我总是忘记。卷发不会改变我的身体或我的生活，但它们会让我变得更强壮，我听说它们对肘部有好处。因此，在2020年，是时候每周一到两次把这种单一、无聊、简单的运动加入我的日常生活中了。
