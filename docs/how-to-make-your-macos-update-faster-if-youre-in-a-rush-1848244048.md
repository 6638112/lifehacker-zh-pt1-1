# 如果你很急，如何让你的macOS更新更快

> 原文:[https://life hacker . com/how-to-make-your-MAC OS-update-fast-if-you-in-a-rush-1848244048](https://lifehacker.com/how-to-make-your-macos-update-faster-if-youre-in-a-rush-1848244048)

如果你发现自己比任何其他软件更新更回避macOS更新，你并不孤单。即使你没有遇到任何稳定性问题或其他错误，更新macOS需要很长时间，忽略更新似乎是更好的选择。毕竟，默认情况下，关键的安全更新是在后台自动安装的。大多数人可以在没有新macOS更新带来的一些新功能的情况下生活，所以这成为我们倾向于避免的事情。

不过，有一种简单的方法可以减少更新macOS所需的时间，它隐藏在一个看起来无害的按钮后面。如果您使用此功能，您将能够减少下载更新所需的时间，并且还能够稍微加快安装过程。

## 如何加快macOS更新

下次你想更新macOS时，点击Mac桌面左上角的**苹果标志**，选择**系统偏好**。现在点击**软件更新**，让你的Mac检查是否有新的更新(亲提示: [重新排列系统偏好设置，让所有选项都容易找到](https://lifehacker.com/how-to-arrange-your-macos-system-preferences-so-you-can-1847583158) )。

如果有可用的软件更新，您的Mac将显示有关它的一些信息，如“有适用于您的Mac的更新”，后面是列出可用更新的项目符号。要减小更新的大小，单击软件更新窗口中的**更多信息**按钮。

这将显示macOS更新中包含的每个组件的附加信息。您会看到每个项目旁边的复选框，您可以快速**取消选择**您不需要立即安装的项目。例如，您可能想要跳过更新到macOS的新版本，但不是Xcode的命令行工具。你可以继续并取消macOS选项，它将允许你避免从你的Mac上删除文件，只是为了给更新腾出空间。

选择完这些项目后，点击**立即安装**开始更新。你的Mac会处理剩下的。请注意，您可能并不总是可以在Mac上取消选中更新，但当您这样做时，它会让您更快地完成工作。
