# 如何获得100%补贴COBRA保费的资格

> 原文:[https://life hacker . com/how-to-qualify-for-100-supported-cobra-premium-1846466275](https://lifehacker.com/how-to-qualify-for-100-subsidized-cobra-premiums-1846466275)

作为扩大的COBRA计划的一部分，拜登总统的COVID救济法案包括一项临时医疗补贴，如果你在疫情期间被解雇或减少了工作时间，该补贴将支付100%的保费。覆盖时间为2021年4月1日至9月30日，已经加入该计划的人也可以参加。以下是你如何获得资格。

### **眼镜蛇是什么？**

COBRA，即综合综合预算调节法案，允许您参加公司的医疗保险计划长达18个月，前提是公司拥有 [20名或更多员工](https://help.zenefits.com/COBRA/Understanding_COBRA/01_Employee_Eligibility_for_COBRA/) (这可以根据某些 [特定资格事件](https://www.dol.gov/sites/dolgov/files/ebsa/about-ebsa/our-activities/resource-center/faqs/cobra-continuation-health-coverage-consumer.pdf) )再延长18个月)。如果你被解雇了，减少了工作时间，或者甚至被解雇了，你就符合资格，除非解雇是因为严重的不当行为(这是 [模糊定义的](https://eastcoastriskmanagement.com/constitutes-gross-misconduct-cobra/)——一些雇主会忽略这一点，提供COBRA保险只是为了避免法庭)。

听起来不错，对吧？不过，有个问题。虽然你可以保留雇主的COBRA保险，但你仍需自己支付每月保费。由于雇主通常会为你支付大约80%的保费[，这是一个很高的价格**——**到个人保险每年7470美元，家庭保险每年21342美元，根据凯泽家庭基金会](https://www.investopedia.com/articles/insurance/11/intro-cobra-health-insurance.asp#citation-6)[。](https://www.kff.org/report-section/ehbs-2020-summary-of-findings/)

COBRA也不会涵盖其他雇主的政策，如人寿保险或残疾保险，这也将增加您的成本。在没有任何补贴的情况下，其他选择，如或 [奥巴马医改](https://www.healthcare.gov/) 可能更好。

### 谁有资格获得增强型COBRA补贴？

[根据美国消费者新闻与商业频道](https://www.cnbc.com/2021/03/11/the-government-will-fully-cover-laid-off-workers-cobra-premiums-.html) ，100%的保费补贴适用于已经符合COBRA保险资格的所有人。它也适用于在最初选择期间没有选择COBRA保险的员工，以及最初选择COBRA但让其保险失效的员工。然而，那些有资格享受其他团体健康保险或医疗保险的人*没有资格享受补贴。*

这意味着几乎所有在疫情开始时失去工作的人都有资格，因为他们18个月的COBRA期包括从4月1日到2021年9月30日的补贴结束期间。

此外，从4月1日开始，雇主可能会为你提供为期90天的特殊注册期，让你加入不同的团体健康计划，尽管保费不能超过你已经支付的金额。对于雇主来说，这是一个可选的政策，所以你需要和你的前雇主核实一下，看看是否有这个政策。

### **如何获得补贴**

你的雇主或健康保险公司应该给你发一份合格通知。如果你期待一份，但还没有收到，直接联系你公司的人力资源部或健康保险公司。但是，一旦您收到COBRA资格通知，您将需要在60天 内注册 [。](https://www.shrm.org/resourcesandtools/hr-topics/benefits/pages/stimulus-bill-provides-cobra-subsidy-through-september.aspx)
