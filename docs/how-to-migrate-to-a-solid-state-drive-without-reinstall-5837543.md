# 如何在不重新安装 Windows 的情况下迁移到固态驱动器

> 原文：<https://lifehacker.com/how-to-migrate-to-a-solid-state-drive-without-reinstall-5837543>

固态硬盘是对台式电脑进行升级的最佳方式之一。而且设置起来也不是很难:物理安装， [接对线缆](https://lifehacker.com/how-to-upgrade-your-computer-with-a-new-drive-1825299803) ， [重新安装 Windows 从头开始](https://lifehacker.com/how-to-set-up-your-windows-laptop-from-scratch-1826146907#_ga=2.219304350.607923588.1534777249-396842925.1520800403) 。



但也许你不想(再次)设置 Windows，从你最喜欢的备份位置复制所有关键文件和文件夹，并重新安装所有应用程序。我们理解你。

当你迁移到新的固态硬盘时，还有另一个选择可以尝试:将你的旧硬盘克隆到新硬盘上。它(相当)快而且容易做到，如果你不想等待和观看，你可以设置运行一整夜。当你醒来，将系统切换到新的固态硬盘时，一切都会和你离开时一模一样。(理论上。)

## 开始之前...确保您的硬盘足够大

当像这样克隆驱动器时，有一个重要的事实你需要记住。**你的新固态硬盘必须足够大，能够容纳你的旧主硬盘中的所有内容。**如果不是这样，克隆就行不通了。

为了释放空间，您可以删除不再需要的文件或重新下载。您还可以在新的固态硬盘上安装全新版本的 Windows，使其成为主要引导驱动器(通过主板的 BIOS)，并将旧硬盘用作次要文件、游戏、电影等的辅助存储。毕竟，你不需要在新的固态硬盘上安装所有的东西。

## 如果你有笔记本电脑，事情就更棘手了

如果你要用新的 SSD 替换笔记本电脑的驱动器，整个过程会变得有点困难，因为你可能只有一个驱动器的空间(除非你的笔记本电脑有一个 M.2 SSD 的备用插槽)。要将你的主硬盘克隆到你的新固态硬盘，你可以选择 USB 转 SATA 适配器、外部坞站或这些花哨小工具 中的 [。](https://www.amazon.com/Kingwin-Adapter-2-5-Inch-3-5-Inch-USI-2535CLU3/dp/B00LSU4SHW?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-to-migrate-to-a-solid-state-drive-without-reinstall-5837543&asc_source=&tag=kinjalifehackerlink-20)

根据您的 USB 连接和笔记本电脑驱动器的大小，克隆过程可能会花费从合理到大量的时间。然而，等待是值得的:用全新的固态硬盘取代旧的机械硬盘是你可以给你的笔记本电脑带来的最佳性能升级之一。

我们将使用应用程序[**Macrium Reflect**](https://www.macrium.com/reflectfree)将您的硬盘克隆到您的新 SSD 中。当你双击安装程序时，你实际上会看到一个看起来像下载工具的屏幕，而不是典型的应用程序安装程序。没错。我不知道为什么 Macrium 软件走这条路，而不是提供整个应用程序作为下载，但你去那里。

## **第三步:激活克隆**

现在，您将看到一个屏幕，上面有一个非常详细的回顾，介绍了一旦您的克隆启动，Macrium Reflect 将要做的所有事情。不，它还没有做任何事情—你只是在设置它。

如果您愿意，您可以检查这些设置，但是您可能只需点击“完成”就很安全了，这将启动该过程:

根据你要去的驱动器的大小 Macrium Reflect 需要移动的数据量——以及它的速度，这个过程可能需要一点时间。我的工作只花了半个多小时，但是在这个例子中，我是将一个 SSD(我的 Windows 分区所在的地方)克隆到一个空的 SSD。换句话说，转移速度相当快。从硬盘迁移到 SSD 可能需要四倍(或更多)的时间。如果你不耐烦，你可以设置你的克隆运行一夜，一旦你醒来，一切都会设置好。

## **第四步:完成你的驱动安装**

现在您有了原始驱动器的克隆，不要在您的主驱动器上做任何事情，将您想要保存的数据放在您的计算机上，因为这些数据不会出现在您的克隆驱动器上。我建议在你的桌面上放一个文本文件，上面写着“这是旧硬盘”，或者更机智的说法，然后关闭你的电脑。

如果您要用新的 SSD 替换旧的硬盘，请从台式机或笔记本电脑上断开旧硬盘(可能是 SATA *和*电源线),并在旧硬盘原来的位置插入新的 SSD。您不需要调整系统 BIOS 中的任何其他内容，它应该直接引导到新 SSD 上的主 Windows 分区。(或者至少，我的是这样。)

如果您要保留旧硬盘，请将其重新连接到台式机系统上的另一个 SATA 端口。通过查看 c:\驱动器的总大小(在其“属性”屏幕中以字节为单位)是否与新 SSD 的容量相匹配，而不是与旧硬盘的容量相匹配，检查以确保您的计算机不会意外启动到它而不是新 SSD。或者在你的 BIOS 中寻找“这是旧硬盘”文本文件，假设你没有跳过这一步。如果你启动的是旧硬盘而不是新硬盘，你必须在 BIOS 的中改变系统的启动顺序。

假设您的计算机正确启动到您的新 SSD，拉起计算机管理(通过开始菜单)，点击磁盘管理，找到您的旧硬盘驱动器，右键单击它的各个分区，并为每个分区选择“删除卷”。如果此选项呈灰色，您可能需要使用第三方应用程序，如 [Paragon 硬盘管理器](https://www.paragon-software.com/home/hdm-windows/) (免费版本)。同样的概念，它只是允许你删除你的旧卷和重新分区驱动器作为一个大块的空白空间。

## **第五步:如何正确维护固态硬盘**

为了确认 Windows 10 能够正确识别您的新 SSD，并定期执行所有 [正确的整理功能](https://lifehacker.com/you-probably-dont-need-to-optimize-your-ssd-anymore-1780293698) ，请单击开始按钮，键入“碎片整理”，并选择第一个选项:“碎片整理并优化驱动器。”

您也可以通过命令提示检查是否启用了修剪。以管理员权限打开命令提示符(通过开始菜单右键单击快捷方式，选择“以管理员身份运行”)，并输入此命令:`fsutil behavior query DisableDeleteNotify`

如果你看到这样的屏幕，你就成功了:

否则，您可以通过输入以下命令强制 Windows 启用 TRIM:`fsutil behavior set DisableDeleteNotify 0`

当你这样做的时候，你的 SSD 制造商可能会提供某种应用程序，你可以使用它来确保 Windows(和你的主板)得到最佳性能的正确配置。此外，这些应用程序通常让你检查和安装新的固件为您的固态硬盘。前往你的固态硬盘制造商的网站，或你的特定固态硬盘的产品页面，看看是否有你可以下载和安装的应用程序。下面是一个例子: [三星的魔术师](https://www.samsung.com/semiconductor/minisite/ssd/product/consumer/magician/) 。
