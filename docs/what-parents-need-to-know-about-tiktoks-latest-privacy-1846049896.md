# 关于抖音最新的隐私控制，父母需要知道什么

> 原文：<https://lifehacker.com/what-parents-need-to-know-about-tiktoks-latest-privacy-1846049896>

[抖音今日](https://newsroom.tiktok.com/en-us/strengthening-privacy-and-safety-for-youth) 宣布，将为 18 岁以下用户增加一些新的隐私保护措施，包括增强的默认隐私设置，对谁可以评论视频以及哪些视频可以下载的更严格限制。

Watch

根据流行的 [平台的网站](https://newsroom.tiktok.com/en-us/strengthening-privacy-and-safety-for-youth) ，这些变化“旨在提高用户隐私和安全的默认标准。”这些变化包括:

*   针对 13-15 岁用户的新默认隐私设置。使用私人帐户时，用户必须先批准关注者，然后才能查看他们的视频。对于这些用户，“向其他人建议您的帐户”的设置也被默认关闭。
*   对谁可以评论 13-15 岁用户制作的视频进行了更严格的限制。他们现在可以选择允许“朋友”或“没有人”发表评论；“每个人”发表评论的选项将被删除。
*   限制用户访问 [二重唱](https://newsroom.tiktok.com/en-us/feature-highlight-new-layouts-for-duet) (一个视频协作功能)和 [缝合](https://newsroom.tiktok.com/en-us/new-on-tiktok-introducing-stitch) (一个视频混音工具)。这些功能现在仅对 16 岁及以上的用户可用。对于 16-17 岁的用户，在这些工具中谁可以使用他们的视频的默认设置将是“朋友”
*   限制可以下载的视频。现在，只有 16 岁以上的用户制作的视频可以下载。对于 16-17 岁的用户，允许他们下载视频的默认设置将被关闭，除非他们选择允许。

这些更改中的大多数只是“默认的”，这意味着它们仍然可以手动更改，以使用户的帐户更容易被公开访问。但是父母可以利用这个机会和他们的孩子谈论保护他们隐私的重要性。

此外，家长仍然可以使用去年发布的“ [家庭配对](https://offspring.lifehacker.com/what-parents-need-to-know-about-tiktoks-new-parental-co-1842921219) ”工具，将自己的账户与孩子的账户进行链接，并远程监控他们的限制。
