# 你在晚年尴尬的学到了什么？

> 原文：<https://lifehacker.com/what-s-something-you-learned-embarrassingly-late-in-lif-1848601501>

不管我们的年龄有多大，我们都不应该停止学习、成长和扩展我们对周围世界的理解。虽然这一事实可以导致美丽的顿悟，但也可能导致彻底的羞辱。当我写这篇文章的时候，某个地方的某人刚刚纠正了“特百惠 ”的 [发音——这让纠正者很高兴，也让冒犯者很尴尬。](https://www.tiktok.com/@bestofpodcasting/video/6925557048058858757) 



我们可以长时间快乐地生活，却有某些错觉或盲点，这并不总是我们的错。这很大程度上与我们的成长方式有关。你永远不会忘记你的第一个室友，她需要知道当他们擦掉柜台上的面包屑时，这些面包屑必须最终到达某个地方。我知道，当我想起我的第一个搭档温和地告诉我，我把他们的好刀例行公事地 [扔进洗碗机](https://lifehacker.com/8-things-you-should-never-put-in-your-dishwasher-1848341293) 时，我仍然感到畏缩。更仁慈的是，我有一个朋友，当他们发现小马实际上并不是小马的时候，他已经成年了；但是在他们的辩护中，这个事实真的出现了多久？

但这并不是所有的小马对小马(可悲的是)。这就是为什么我要求你，亲爱的 Lifehacker 读者，在下面**评论一个你(或你认识的人)在晚年尴尬得知的事实**。我明白我是在让你变得脆弱:*你第一次知道如何疏通马桶的时候是几岁？你有没有认为税收是可有可无的伴侣？难道你从来没有* [*学过如何卷一个关节*](https://lifehacker.com/how-to-roll-a-perfect-joint-1848559512)*然后现在想起来问也晚了？我在这里告诉你，问问题永远不会太迟。)*

在你分享你的回答后，我会把它们整理成一个单独的帖子，这样那些大器晚成的人就可以从你的错误中吸取教训。别的不说，你晚年令人尴尬的发现可能会让网上的陌生人窃笑。为此，我提前感谢你。
