# 费雪-普莱斯控制器，改装成玩婴儿游戏埃尔登环

> 原文:[https://kotaku . com/fisher-price-game-learn-controller-elden-ring-mod-1848868924](https://kotaku.com/fisher-price-game-learn-controller-elden-ring-mod-1848868924)

又到了重设 [创新](https://kotaku.com/elden-ring-let-me-solo-her-spirit-ashes-pc-mod-malenia-1848808548) 和 [怪异](https://kotaku.com/elden-ring-toastedshoes-pc-mods-shrek-homer-simpson-elm-1848693567) 里程碑的时候了，玩家们正在 *Elden Ring* 社区中忙碌着。这一次，一个游戏玩家将一个儿童玩具改装成了一个可玩的Xbox控制器。这不是一篇洋葱文章，你来对地方了。

Watch

[迪伦“Rudeism”贝克](https://www.youtube.com/c/Rudeism/featured) ，一个“玩游戏玩错了”的YouTuber，把一个 [费雪游戏&学习控制器](https://shop.mattel.com/en-ca/products/fisher-price-laugh-learn-game-learn-controller-fnt06-en-ca) 改造成了一个全功能的大男孩控制器。对于那些不了解婴儿“R”美国生活方式的人来说([RIP Geoffrey The Giraffe](https://twitter.com/ginamartinuk/status/1012368876791828480?s=20&t=z70iVpRf0nELWr0-Rail1g)),费雪牌游戏&学习控制器，就像任何其他游戏控制器一样，是一款教育玩具，适合6-36个月大的受训游戏玩家学习形状、颜色、字母和数字，同时微调他们的手眼协调和精细运动技能。但是贝克并没有蹲下来做一些“学习字母的动作”，而是想在《埃尔登环》中用这个玩具来对付老板。虽然这个玩具的形状像一个控制器，但它不一定是为了与现代游戏机一起使用而设置的。

 [https://kotaku.com/embed/inset/iframe?id=twitter-1520874547205271553&autosize=1](https://kotaku.com/embed/inset/iframe?id=twitter-1520874547205271553&autosize=1) 

所有游戏交易的守护神Wario64在推特上发了一个关于费雪价格控制器 的即兴笑话，称它“非常适合*埃尔登戒指*，但“可能不允许用于EVO”，这给了贝克发明的灵感贝克认为这是针对他个人的。在两个星期的时间里，Beck努力工作，根据[*The Verge*](https://www.theverge.com/2022/5/1/23051545/fisher-price-toy-baby-gamepad-real-xbox-controller-rudeism-elden-ring)将这个形成性的儿童玩具改装成一个成熟的控制器。

你可能想知道如何将你孩子早期的学术玩具升级为游戏控制器，你知道，对孩子来说。事实证明，你所需要做的就是给Fisher-Price控制器添加一个Arduino Pro微型克隆控制器板，真正的操纵杆，以及一些微型开关到它的肩按钮上，瞧，宝宝的第一个控制器刚刚移除了训练轮。

 [https://kotaku.com/embed/inset/iframe?id=twitter-1516522052865474566&autosize=1](https://kotaku.com/embed/inset/iframe?id=twitter-1516522052865474566&autosize=1) 

别担心，Mattel faithful， [尽管没有足够的空间放电池](https://twitter.com/rudeism/status/1520504573428781057?ref_src=twsrc%5Etfw%7Ctwcamp%5Etweetembed%7Ctwterm%5E1520504573428781057%7Ctwgr%5E%7Ctwcon%5Es1_&ref_url=https%3A%2F%2Fwww.theverge.com%2F2022%2F5%2F1%2F23051545%2Ffisher-price-toy-baby-gamepad-real-xbox-controller-rudeism-elden-ring) ，Beck的改装控制器仍然有所有OG Learn控制器的音效和歌曲来通知你“1，2，3，4，你的分数上升了”以及 [其Konami码复活节彩蛋](https://www.theverge.com/circuitbreaker/21133169/fisher-price-konami-code-baby-toy-game-and-learn-controller) 。

这并不是贝克第一次在竞技表演中“玩错游戏”以前贝克玩过 [*黑暗之魂*使用摩尔斯码](https://twitter.com/rudeism/status/1452190376505323520)T5[*无题鹅游戏*使用声控鸣笛和动作控制](https://twitter.com/rudeism/status/1179372156691587074) 。
