# 如果你失业了，如何有资格享受免费医疗

> 原文:[https://life hacker . com/how-to-qualify-for-free-health care-if-you ' s失业者-1847215619](https://lifehacker.com/how-to-qualify-for-free-healthcare-if-youre-unemployed-1847215619)

截至7月1日，失业的美国人有资格获得巨额奥巴马医改补贴，其中60%的人每月保费降至0美元。如果你在2021年的任何时候失业，从现在到8月15日，当一个正在进行的特殊登记期结束时，你可以通过访问健康保险市场来获得新计划的资格。以下是你需要知道的。

## **如果您还没有购买新计划，现在就购买吧**

上周，美国卫生与人类服务部宣布，从7月1日开始，任何“从2021年开始接受或被批准接受失业补偿的人都可以通过HealthCare.gov找到更低成本的计划，并节省额外的现金支出，”当然，这是患者保护与平价医疗法案(ACA)建立的联邦补贴医疗保健交易所(也称为奥巴马医改)的所在地。这些补贴由美国救援计划支付，该法案于今年早些时候通过成为法律。

[据美国消费者新闻与商业频道](https://www.cnbc.com/2021/07/01/unemployed-can-now-get-zero-premium-health-insurance-via-marketplace.html) 报道，全额补贴适用于成本较低的银牌和铜牌计划，因此你仍有可能为更全面的金牌和白金保单支付额外费用。也就是说，作为的雅虎T5报告 ，目前在交易所有新降低的0美元保费的白银计划。

然而，还是有一些限制。补贴因州而异，特别是如果提供的服务超出ACA认为的基本健康福利。例如，如果你可以通过你的雇主获得医疗保险，你也不能追溯性地获得补贴。

## **如何报名参加奥巴马医保计划**

如果你在2021年的任何时候失业，你可以在[【healthcare.gov】](https://www.healthcare.gov/)报名参加保险，直到8月15日当前的开放注册期结束。如果你已经有保险，你可以回到市场更新你的信息，看看你是否有资格获得额外的补贴。当您查看提供的计划时，您可能会看到至少几个以前没有的免费银牌计划。关于如何购买ACA计划的更多信息，请查看Lifehacker的帖子 。
