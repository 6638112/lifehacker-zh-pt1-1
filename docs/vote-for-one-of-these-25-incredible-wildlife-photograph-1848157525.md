又到了一年中的这个时候，通过地球上一些最有才华的野生动物摄影师的眼睛来看看自然世界——你可以投票选出你最喜欢的。本周，伦敦自然历史博物馆发布了25张入围年度最佳野生动物摄影师奖的照片。它们是你想从野生动物摄影中得到的一切:令人惊叹的特写镜头，难以置信的故事，可爱的母亲和婴儿照片。

今年，来自95个不同国家的50多万张照片参加了该博物馆第57届年度野生动物摄影师大赛。在线投票的前五名获奖者将在网上展示。博物馆已经选出了整体竞赛 的 [获胜者，其中包括灰熊狼吞虎咽、驯鹿锁定鹿角和巨型蝾螈制造甜蜜爱情的照片。](https://gizmodo.com/here-are-the-2021-wildlife-photographer-of-the-year-win-1847845533)

“人民选择奖提供了对自然以及我们与自然的关系的惊人观察，激发了我们的好奇心，加强了我们与自然世界的联系，”自然历史博物馆的研究员兼评委团成员娜塔莉·库珀在一份声明中说。“只从这些照片中挑选一张是一个不可思议的挑战，所以我们期待着发现哪个狂野的瞬间会成为公众的最爱。”

我们在这里收集了所有25张照片，从这张照片开始，斑马在纳米比亚埃托沙国家公园的Okaukuejo水坑喝水，这让摄影师卢卡斯·布斯塔曼特想起了“活条形码”。您可以在2月2日之前通过 [为您最喜爱的照片](https://www.nhm.ac.uk/wpy/peoples-choice) 投票，获奖者将于2月9日公布。如果你是一名希望明年晋级的野生动物摄影师，你很幸运——第58届年度比赛的报名将于12月9日 结束。

今天是假期，所以让我们从节日开始吧。这张拍摄于中国山西省历山自然保护区的两只雄性雉鸡的照片，是对这些美丽但经常害羞的鸟类的罕见一瞥，它们花了很多时间在森林的地面上寻找食物。

这张照片也很喜庆，在某种程度上，它让我想起了节日期间拥挤的购物中心。摄影师Yung Sen Wu在帕劳与一群梭鱼一起游了五天，试图找到完美的角度，以跟上鱼群的快节奏。第五天，他打了预防针。

嘿，姑娘。南非茨瓦卢卡拉哈里自然保护区的猫鼬非常习惯与人相处，摄影师托马斯·佩查克拍摄的这张充满好奇的脸就证明了这一点。肯定是这个猫鼬约会应用的一个。

摄影师安迪·斯基林在零下22华氏度(零下30摄氏度)的天气里，在加拿大育空地区钓鱼支流的这个地方等待，只有直升机才能到达，拍摄了一只熊穿越一根木头的照片。一只雌性灰熊在寒冷的钢索上行走，她的皮毛上覆盖着在河里捕鱼时留下的冰柱。

“当她走过时，你能听到它们叮当作响，”斯基林说。

在寒冷的几个月里，一只北极狐正在斯瓦尔巴特群岛的斯匹次卑尔根岛上喘息。拍摄这张照片时的温度是零下31华氏度(零下35摄氏度)，这种寒冷会让你窒息。北极狐很难拍照，因为它们非常活跃，但这只小家伙停下来休息了足够长的时间，让马可·盖奥蒂拍下了这张照片。

可爱不止。这张照片拍摄于南非的阿多大象保护区，一只小象正被它的象群抚慰着。“当你观察大象时，有一种神奇而美丽的东西——它触动你的灵魂，拨动你的心弦，”摄影师彼得·德莱尼说。

嗯，这是一个看起来有点令人不安的合作关系。这张照片展示了在中国云南省西双版纳热带植物园里，一群绿树蚁一起工作来捕捉一只螽斯。蚂蚁并没有杀死螽斯，而是以这些昆虫分泌的汁液为食。作为回报，螽斯帮助保护蚂蚁免受捕食者的伤害。

在动物摄影的世界里，没有什么比两个陌生的伙伴一起闲逛更好的了。这张照片展示了一只小黑熊在阿拉斯加阿南的温带雨林中打盹，被一只幼年金雕注视着。

你不必走很远就能拍到野生动物的精彩照片。这张画眉鸟的照片是由摄影师Jan lemann在德国格赖夫斯瓦尔德自家前门拍摄的。

另一幅令人难以置信的人类试图帮助野生动物的图像来自哥伦比亚，一位生物学家怀抱着一只亚马逊河海豚。直接接触能让海豚在紧张的情况下平静下来，而海豚并不缺少这种接触。该团队正在将海豚运送到一个兽医设施，在它的鳍上放置一个GPS标签，这是一个更大的项目的一部分，以了解海豚在哪里生活以及它们的迁徙模式。由于渔具缠绕和污染，这种动物被列入世界自然保护联盟红色名录的濒临灭绝。

一些年轻人不想这么安静地坐着。摄影师Maxime Aliaga抓拍到了一只名叫Marconi的猩猩试图将她名叫马森的小猩猩留在印度尼西亚苏门答腊Pinus Jantho自然保护区的巢中。马可尼曾被非法当作宠物饲养。2011年，她被放回野外，马森在野外出生——这是濒危物种的微弱希望。

这是另一对母子生活在一个被人类永远改变了的栖息地。这只东部灰袋鼠和她的小袋鼠在澳大利亚维多利亚州马拉库塔附近一片被烧毁的森林中被拍摄下来。他们是毁灭性的丛林大火 [的幸存者，这场大火是由气候变化](https://gizmodo.com/we-now-know-how-much-climate-change-contributed-to-aust-1842090462) 引发的，席卷全国，在2019年和2020年造成大约30亿动物死亡或流离失所。
