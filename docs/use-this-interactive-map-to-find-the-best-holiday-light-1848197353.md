在过去，你必须依靠口口相传才能找到你所在地区最像Griswold的房子。但是现在，[next door app](https://blog.nextdoor.com/2021/12/01/find-holiday-decorations-in-your-neighborhood-with-the-nextdoor-cheer-map/)上的一个互动地图，只要轻轻敲几下，就会给你指出正确的方向。以下是需要了解的内容。

## 如何使用交互式地图

从技术上来说，它被称为，为了使用它，你必须有一个Nextdoor账户，这个账户 ch可以免费创建和使用(就像在，没有金钱成本——像任何应用程序一样，你用你的个人数据付费)。一旦你设置好了，你就可以使用地图来定位你所在区域的灯火通明的房屋，或者将你自己的家添加到地图上。

### 去找装修好的房子

要 [定位你家附近的助威地图](https://help.nextdoor.com/s/article/holiday-cheer-map?language=en_US) ，在你的iPhone或Android上打开Nextdoor应用。然后点击屏幕右下角的“更多”，然后点击“邻居”下方的“啦啦队地图”然后，您将被重定向到啦啦队地图，在这里您将看到您的邻居地图以您家的大头针为中心。你的邻居将会用一个节日灯泡图标来标记。

### 将您的家添加到地图上

按照 上面的 [指示。当您进入您的社区啦啦队地图并看到您家的大头针时，轻按“添加您的家”然后为你的灯泡选择一种颜色，点击“保存”要从地图上移除您的家，请点击“编辑您的Pin”，然后点击“移除我的Pin”，您应该就一切就绪了。](https://help.nextdoor.com/s/article/holiday-cheer-map?language=en_US)
