# 应该用 Airbnb 还是 Vrbo？

> 原文：<https://lifehacker.com/should-you-use-airbnb-or-vrbo-1847238697>

在所有的点对点租赁平台中， [Airbnb](https://www.airbnb.com/) 和 [Vrbo](https://www.vrbo.com/) 是两个最大的名字，它们有很多相似之处。但是，虽然他们都有比大多数酒店房价更便宜的交易，但两者之间也有值得了解的差异。下面来看看他们是如何比较的。

## **Airbnb 比 Vrbo 有更好的选择**

Airbnb 和 Vrbo 都是点对点租赁网站，允许私人业主在他们的应用程序上向客户出租私人和商业空间。Vrbo 代表“业主的度假租赁”，实际上是两个平台中较老的一个，尽管它的选择范围较小，部分原因是 Vrbo 只出租整个物业，而不是像 Airbnb 那样出租有共享空间的房间。

出于这个原因，最便宜的交易通常是在 Airbnb 上找到的，尽管权衡的结果是你会看到共享空间或其他古怪的空间(例如，我曾经住过一个狭窄的 Airbnb，在淋浴间有一个厕所——但很便宜)。还值得一提的是，Airbnb 已经努力从其家庭共享的根源中拓展出来，现在也提供酒店客房。

## **Airbnb 和 Vrbo 的价格相当**

尽管 Airbnb 和 Vrbo 以服务费的形式从每笔交易中抽取一定比例，但这两种服务实际上都是直接向房东预订。由于每处房产都是独一无二的，定价也部分取决于业主的要价，因此很难对这两个平台进行比较。但总的来说，Airbnb 和 Vrbo 的基础票价和附加服务费不相上下。

Airbnb 通常会收取 [的额外服务费，相当于](https://www.airbnb.com/help/article/1857/what-are-airbnb-service-fees) 小计(扣除税款)的 5%至 20%，其中可能包括每夜房费、清洁费和任何其他额外的客人费用(Airbnb 表示，他们的提成通常是 [14-16%](https://www.airbnb.com/help/article/1857/what-are-airbnb-service-fees#:~:text=2.-,Host%2Donly%20fee,hotels%20and%20some%20other%20hosts.&text=Airbnb%20Plus%20hosts%2C%20and%20hosts,cancellation%20policies%20may%20pay%20more.) )。有了 Vrbo，价格也在同样的范围内——你将被收取小计(减去税款)的大约 6%[到 12%](https://travelfreak.com/airbnb-vs-vrbo/) 的费用，其中也包括附加费用。因此，由于额外的费用是由 Airbnb 和 Vrbo 主机设置的，所以总成本在给定的酒店上有所不同。另外值得一提的是，Airbnb 提供长期入住折扣，而 Vrbo 没有。

## Airbnb 和 Vrbo 都有一系列取消选择

Airbnb 和 Vrbo 都有取消政策，当你在这两个平台上选择住宿地点时，你可以在单个酒店的网页上找到这些政策。基本上，您可以根据预订日期前不同的取消窗口获得退款，同样，这是由业主决定的。

至少对消费者来说，Vrbo 的一个问题是，用户可以有更多的选择来定制他们的取消政策，这可能会让消费者感到困惑。为了确保给定的政策不会过于繁琐，你必须仔细阅读 Airbnb 和 Vrbo 的业主政策，尤其是 Vrbo。

## Vrbo 的界面比 Airbnb 笨拙，但有更好的搜索选项

Vrbo 的设计不像 Airbnb 那样具有视觉吸引力(它有广告)，但它提供了更好的搜索功能，允许你根据整体排名、受欢迎程度、取消政策或清洁度来过滤结果。这是 Airbnb 非常缺少的东西。

## 结果

如果你不得不选择一个平台，Airbnb 将是最好的选择，因为他们提供了大量可供选择的场地。但是当然，你*不需要*只选择一个平台，这意味着你可以继续使用 Vrbo 作为 Airbnb 的补充。一些房产在两个网站上都有列出，所以两个网站的搜索结果可能会有一些重叠，但每个平台都有独特的列表，所以你更有可能通过搜索 Airbnb 和 Vrbo 找到最好的交易。
