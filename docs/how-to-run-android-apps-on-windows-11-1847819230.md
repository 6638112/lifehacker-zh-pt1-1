# 如何在 Windows 11 上运行 Android 应用

> 原文：<https://lifehacker.com/how-to-run-android-apps-on-windows-11-1847819230>

如果你已经安装了微软个人电脑操作系统的最新版本 Windows 11——，你现在可能想知道如何在你的电脑上运行安卓应用程序；b 因为虽然微软承诺让你通过亚马逊安卓应用商店运行这些应用，但是功能实际上还没有推出。直到微软在未来的更新中为你提供安卓功能，你仍然可以在你的 Windows 电脑上下载和运行安卓应用和游戏。

(加上，值得一提的是亚马逊安卓应用商店没有 Google Play 的所有应用。所以即使你等待更新，一些你最喜欢的应用 可能仍然不可用。)

我们将使用 Android 模拟器，而不是使用亚马逊的 Appstore。这里是我们在找到的几个最好的模拟器——警告说因为 Windows 11 是新的，你可以期待在所有这些模拟器中看到一些错误。然而，随着时间的推移，软件更新应该带来更多的稳定性。

## 蓝色背包

很久以前就有了，这些年来改进了很多。这个模拟器可以让你从 Google Play 下载和安装应用程序，这是一个很大的优势，而且它的任何最好的功能都是专注于游戏的。如果你想用键盘和鼠标玩你最喜欢的 Android 游戏，Bluestacks 是一个不错的选择——它让你可以创建宏和脚本来将你最喜欢的动作映射到键盘快捷键上，这可以节省你玩游戏的时间。

此外，它还可以让您快速更改设备配置文件，以运行特定设备专用的应用程序。有些游戏，比如 PUBG Mobile，最好在开发者自己的模拟器中运行，但大多数其他应用和游戏在 Bluestacks 上运行良好。

## 游戏大观

中国科技巨头腾讯拥有 [GameLoop](https://www.gameloop.com/us/) ，而顾名思义，其专注于游戏。因为腾讯发布了 PUBG Mobile 和一系列其他手机游戏，GameLoop 的重点是在玩*那些*游戏的时候给你最好的体验。它还有一个反作弊系统，如果你想在游戏中避开黑客，这是个好主意。

## 夜神模拟器

[NoxPlayer](https://www.bignox.com) 也主要针对想玩游戏的人，如果你对 Bluestacks 或 GameLoop 有问题，给 NoxPlayer 一个机会。它拥有游戏所需的功能-支持键盘和鼠标，录制宏，并能够同时运行多个游戏。
