# 你应该做小番茄吐司

> 原文：<https://lifehacker.com/you-should-make-tiny-tomato-toast-1846239535>

自从我采访了美食记者艾丽西娅·肯尼迪关于她的日常饮食习惯之后，我就一直渴望着西红柿吐司，这是一种明显的夏季美食，在二月中旬感觉遥不可及。大而多汁的西红柿目前还没有上市，但是小的西红柿(如樱桃或葡萄)多汁、香甜，一年四季都很容易找到，对我来说，幸运的是，它们可以做很棒的西红柿吐司。

像它的大番茄兄弟一样，小番茄吐司是一道明亮、快乐的菜，非常适合给寒冷、灰暗的早晨(T2)或雨天的午餐时间(T3)注入活力。你可以保持简单——用磨碎的西红柿、橄榄油和盐——或者你可以将西红柿磨碎，放在某种奶油基质的上面:奶油干酪、 [labneh](https://skillet.lifehacker.com/how-to-make-your-own-labneh-and-how-to-eat-it-1796785416) 、 [搅打的农家干酪](https://skillet.lifehacker.com/cottage-cheese-is-even-better-whipped-1834726636) 或鳄梨。我喜欢把奶油层保持得很薄；你希望在西红柿和烤面包片之间形成一层足够的疏水层，以防止前者的汁液渗入面包并使其变湿。

就配料而言，盐通常是很多的，但我最近一直在用 [辣椒油](https://skillet.lifehacker.com/how-to-make-your-own-chili-oil-1842478518)[风油精](https://skillet.lifehacker.com/toss-a-hunk-of-ginger-in-your-air-fryer-1846221741) 和 [大葱](https://skillet.lifehacker.com/air-fry-your-scallions-for-the-perfect-garnish-1846184934) 。柠檬皮、za'atar、漆树、 [各种百吉饼调料](https://skillet.lifehacker.com/12-seasonings-i-keep-by-my-stove-at-all-times-1844313629/slides/13) 、你最喜欢的醋和磨碎的奶酪都是可行的选择；记住西红柿是明星。为他们留出发光的空间。

磨碎小番茄的唯一缺点是当你磨碎的时候没有那么多东西可以抓，但是我——一个非常笨拙的人——没有遇到过任何意外。H 将 f 水果 放在它长方形身体的一端(或者抓住茎，如果它有茎的话),然后在烤面包上磨碎,让种子、凝胶和多汁的果肉落下,直到你只剩下一小块被压碎的皮和肉。把它也放在吐司上。(一定要用大孔的刨丝器，不要用微刨。)加入你的调味料和任何你想要的最后润色，然后大口大口地吃。
