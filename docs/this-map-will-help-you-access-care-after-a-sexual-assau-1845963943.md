# 这张地图将帮助你在遭受性侵犯后获得治疗

> 原文:[https://life hacker . com/this-map-will-help-you-access-care-after-a-sex-assau-1845963943](https://lifehacker.com/this-map-will-help-you-access-care-after-a-sexual-assau-1845963943)

如果你被性侵，你可能知道你可以去医院急诊室做法医检查，有时被称为 [强奸试剂盒](https://www.rainn.org/articles/rape-kit) ，收集DNA证据，确保你得到适当的照顾。但是并不是每家医院都有训练有素的人提供这项服务，所以NBC新闻编辑了一张地图，上面有在每个州以SANEs的形式提供帮助的信息。

SANEs帮助性侵犯提供者完成整个过程，让他们在收集证据时了解自己的权利和选择。NBC报道称，许多医院没有全天候可用的SANEs，有些根本没有，并指出培训护士进行这些检查对医院来说不是赚钱的——所以 SANEs往往来自州政府或非盈利组织的项目。NBC的记者打电话给每个州询问人们如何进行性侵犯检查，发现许多人没有跟踪人员配备，或者无法提供在哪里可以找到合格考官的最新列表。

因此，他们建造了 [这张地图](https://www.nbcnews.com/health/sexual-health/map-where-sexual-assault-forensic-exams-are-available-your-state-n1252376) ，这应该给你一个开始的地方。如果有的话，你可以找到一个链接到州名录，以及关于当地医院和强奸危机中心的信息。有时候最好先联系医院；在其他情况下，你最好打电话给危机中心。

还有一个选择，不管你在哪个州，都可以拨打RAINN运营的 [全国性侵电话热线](https://www.rainn.org/about-national-sexual-assault-telephone-hotline) 。该热线将把您连接到当地的附属机构，那里应该有人能够帮助您找到合适的护理。
