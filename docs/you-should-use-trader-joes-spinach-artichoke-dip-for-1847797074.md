# 你应该用乔氏菠菜和朝鲜蓟蘸酱来做一个一流的鸡蛋烘烤

> 原文：<https://lifehacker.com/you-should-use-trader-joes-spinach-artichoke-dip-for-1847797074>

如果有一件事是妈妈们擅长的，那就是找出可以快速、省力地准备好的便餐，这就是为什么我——一个没有孩子的女人——加入了脸书忙碌妈妈群 的 [乔氏膳食创意。](https://www.facebook.com/groups/traderjoesmealideas4busymoms/permalink/1713228095549348)



这些妈妈们非常擅长将乔氏的产品组合成大于其各部分总和的食物。就拿这种两种成分的蛋烘糕来说吧，T2 的厨房和 T4 的厨房最近也有报道。它简单、优雅，只使用了两种食物:乔氏奶油菠菜& 洋蓟酱(在冷冻食品区可以找到)和鸡蛋。最初的食谱要求一个容器的蘸料和八个鸡蛋，但是——正如厨师所指出的——这个比例可能会变得有点干。

我讨厌烤干鸡蛋，不管是乳蛋饼、煎蛋饼，还是介于两者之间的任何东西，所以我把鸡蛋的数量减少到六个，并加入少量的奶酪。这些调整带来了很大的不同；烤得又嫩又美味，看起来比八个鸡蛋的版本更有活力。

额外的奶酪——或者至少额外的某样东西——也很关键。虽然冷冻蘸酱提供了调味料、脂肪和两种蔬菜，但当它与一堆未调味的鸡蛋混合在一起时，会变得有点乏味。然而，它确实非常适合作为的基础，用来建造低强度的蛋糕店。至少，像平常一样用盐和胡椒给鸡蛋调味，但是你可以加入一些培根、一点焦糖洋葱，或者——是的——你周围的任何一点奶酪，让你的烘焙真正发光。

烘焙后，你可以像吃无壳乳蛋饼一样吃它，把它放在三明治上，或者冷冻成单独的一份，作为简单的工作日早餐。要制作这个优雅的小东西，你需要:

*   1 包乔氏奶油菠菜&洋蓟酱
*   6 个鸡蛋
*   盐和胡椒
*   奶酪、培根块、熟香肠、炒蔬菜，或者任何你想在这个蛋烘培中看到的其他配料
*   烹饪喷雾(或黄油)

把你的烤箱加热到 350 华氏度。将冷冻蘸酱从包装中取出，放入微波炉专用的碗中。加热一分钟，用叉子把它弄碎，然后以 30 秒的增量加热，中间不断搅拌，直到蘸酱解冻，但不烫手。用 1/4 茶匙盐和几粒新鲜胡椒搅拌鸡蛋，然后用搅拌器或密封的塑料容器彻底搅拌。

准备一个 8 英寸的圆形或方形砂锅，混合鸡蛋和蘸酱，将混合物倒入砂锅。在顶部撒上更多的奶酪、熏肉或任何你已经沾上的东西。(我用了大约两盎司新鲜的山羊奶酪。)烤 25-30 分钟，直到中间刚刚定型，边缘开始焦黄。静置五分钟，然后切片上菜。
