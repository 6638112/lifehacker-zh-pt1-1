# 神奇宝贝猩红和紫罗兰的预购奖金是如此可怕，他们被嘲笑

> 原文:[https://kotaku . com/pokemon-猩红-紫罗兰-预购-奖金-任天堂-1849479057](https://kotaku.com/pokemon-scarlet-violet-preorder-bonuses-nintendo-1849479057)

玩家对预购奖金的狂热并不是什么新鲜事。但是这一次，*神奇宝贝*社区的盐感觉部分合理。亚马逊 [最近展示了一张信息图](https://twitter.com/amazongames/status/1563211056243437577?s=20&t=o1J1U3yrMzqQAS6BkdsmSw) ，详细说明了粉丝在发布前通过数字订购*猩红*和*紫罗兰*可以得到什么:一套包括10种药剂、10种解毒剂和3种复活剂。奖励感觉如此无关紧要，以至于粉丝们在网上嘲笑奖金。

Watch [https://kotaku.com/embed/inset/iframe?id=twitter-1563211056243437577&autosize=1](https://kotaku.com/embed/inset/iframe?id=twitter-1563211056243437577&autosize=1) 

我们已经在本月早些时候了解了这些奖励，但是在图像中显示微不足道的奖励与简单地在项目列表中阅读它们有很大的不同。

“我最初对这些游戏毫无兴趣，但一旦他们说我可以预订游戏中的10种药剂，一切都变了，”一位用户讽刺地说，指出微不足道的奖金可能不会迫使任何人购买任天堂今年最令人期待的游戏之一。另一名粉丝的批评更加直白:“你可以在第一条路线上击败两三名教练，获得足够的神奇美元来购买所有这些物品。”

为了公平起见， [玩家数字化购买游戏](https://kotaku.com/pokemon-scarlet-violet-preorder-special-pikachu-1849365884)*也将*获得五个满疗、三个乙醚、一个稀有糖果和一个金块。但是这些仍然是你在游戏中经常遇到的普通物品。有粉丝 [指出](https://twitter.com/anzasquiddles/status/1564420270680842240?s=20&t=YWRTvcxREni8sYLh2iil5w) 日本*猩红*和*紫罗兰*买家拿到的是实体画册，这更符合大多数粉丝在预购游戏时的预期。还有的在缓解在发行事件中获得专属神奇宝贝的日子，像*神奇宝贝X* 和 *Y.* 中的特殊 [火稚鸡](https://twitter.com/PancakesSuicide/status/1564455953763794944?s=20&t=YWRTvcxREni8sYLh2iil5w)

然而，这只是针对如果你购买一个游戏的数字。购买两个版本的两个包装的玩家将获得100个神奇球。如果你从官方神奇宝贝中心购买游戏，你将能够获得这个 [印花背包](https://scarletviolet.pokemon.com/en-us/news/pcenter_backpack/) 。如果几年前神奇宝贝中心没有给我一个Scorbunny毛绒玩具用于预订*神奇宝贝盾*，我可能会更兴奋。

商店列表*应该*已经宣传的是玩家在2023年2月28日前购买游戏将获得的 [特殊水晶皮卡丘](https://scarletviolet.pokemon.com/en-us/news/pre_order/) 。这个皮卡丘会用飞式的招飞，这是它天生学不会的。所以提前购买游戏会有特别的奖励，但并不排斥在发售日期前简单地购买*猩红*和*紫罗兰*。

*神奇宝贝猩红*和*紫罗兰*将于11月18日上映。
