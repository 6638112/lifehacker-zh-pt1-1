# 如何获得超级碗门票，如果你绝对必须的话

> 原文：<https://lifehacker.com/how-to-score-super-bowl-tickets-if-you-absolutely-must-1848474218>

超级碗是这个国家最后的真正公共事件之一。从音乐到电视，其他娱乐形式正越来越多地被细分为拥有少量观众的特定领域，而体育仍保持着普遍的吸引力，足球占据着至高无上的地位。有多至高无上？2021 年，超过 9100 万人观看了超级碗 LV 2021 年——这比 2020 年下降了 9%。



每个人都喜欢一场精彩的超级碗派对，包括小吃、赌桌，当然还有广告。但是真正的球迷梦想着亲自去看比赛——这在身体上和经济上都是一个挑战。实际上，最多只有 [10 万个](https://www.hksinc.com/what-we-do/case-studies/sofi-stadium/) 座位，数百万人不介意坐在其中。美国橄榄球联盟也不容易做到这一点。联盟通常 [将 75%的可用门票分配给球队、合作公司和内部人士](https://www.cbssports.com/nfl/news/super-bowl-2020-chiefs-and-49ers-will-get-35-percent-of-tickets-heres-how-the-rest-are-distributed/) ，这严重限制了你获得门票的机会。

NFL 过去每年都提供彩票，但今年仅限于 500 张票——已经在赛季期间分发出去了——他们的美国残疾人法案(ADA)彩票也已经结束。今年纯粹的运气不会帮助你。

那么你能做什么呢？你可以做好准备，把它当作一个项目来攻击，因为它就是这样。哦——把你的财务状况整理好，因为如果你*真的*找到了票，你就要付出高昂的代价了。

## 超级碗门票多少钱？

精确的票价是不可能的，因为超级碗门票市场正是“不稳定”这个词的定义这是因为合法购买只有几种方式:

*   售票员。 [Ticketmaster](https://www.ticketmaster.com/super-bowl-lvi-inglewood-california-02-13-2022/event/0A005B68903A3611) 是 NFL 的官方门票来源。一些贵宾区门票的价格从 6000 多美元(含费用)到 72000 多美元不等。
*   **转卖。**像 [SeatGeek](https://seatgeek.com/super-bowl-lvi-cincinnati-bengals-vs-los-angeles-rams-tickets/2-13-2022-inglewood-california-sofi-stadium/nfl/r/5428684) 和 [StubHub](https://www.stubhub.com/super-bowl-inglewood-tickets-2-13-2022/event/105055051/?priceWithFees=true) 这样的门票转售网站也是声誉良好的门票来源(Ticketmaster 也购买和转售门票)，但你不会看到太多的折扣。
*   **外景。** [外景](https://onlocationexp.com/nfl/super-bowl-ticket-packages) 是 NFL 的官方“招待供应商”，这意味着他们出售带有额外津贴和“体验”的活动套餐信不信由你，这里的价格与 Ticketmaster 的一致，尽管提供一些额外服务，如赛后场地使用权或派对邀请。

## 从黄牛手中购买超级碗门票

无论你如何分配，两张超级碗的门票将会让你花费至少 12，000 美元。这可能会让你找到愿意以低于大多数网站要价的 6000 英镑卖给你门票的人。超级碗门票的面值不是众所周知的，但据 [估计](https://www.cbssports.com/nfl/news/super-bowl-2020-chiefs-and-49ers-will-get-35-percent-of-tickets-heres-how-the-rest-are-distributed/) 约为 1000 美元，所以如果有人与某个球队有关系，得到了一些(相对)便宜的门票，他们可能会以比现行价格至少低一点的价格出售这些门票，这并不完全是疯狂的——即使他们要价 5000 美元，你至少可以节省那些惩罚性的费用。

但是要非常非常小心。首先，看看倒票在你所在的州是否合法。最重要的是，请记住，购买黄牛票是最容易和最快的方式被骗。有 [*只有*超级碗 LVI](https://adage.com/article/special-report-super-bowl/super-bowl-2022-tickets-will-come-nfts-nfl/2397181) 的移动门票(事实上，购买官方门票的一部分可以获得一张纪念这场比赛的免费 NFT)，而你 [甚至不能再在家打印](https://www.nfl.com/news/mobile-ticketing-frequently-asked-questions-0ap3000001012693) ，所以任何向你提供纸质门票的人都很可能在骗你。你最好的选择是使用上面提到的合法的票务经纪人。

## 最终提示

所以，你认为花几个月的房租买几张超级碗门票是值得的，你的孩子不需要去名牌大学，所以你加入了。以下是一些可能会帮你减轻一些痛苦的建议:

*   关注价格。由于几乎所有的超级碗球票都是根据市场力量出售的，它们会随着时间的推移而上下浮动，有时会非常剧烈。事实上，购买超级碗球票的最佳时间是比赛前的三到五天，此时持有球票的人开始为收回投资而紧张。 [SeatGeek 提供了一个价格追踪器](https://seatgeek.com/super-bowl-ticket-tracker) 你可以用它来查看价格在过去和现在的波动情况，这样你就可以策划一个策略。
*   **显示费用。**因为这些天关于门票的一切都非常粗略，所有“合法”的门票经销商都会给你显示价格*而没有默认的*所需费用。因此，如果你认为你已经找到了一张便宜得惊人的机票，请再想想——这些费用可能会在你的最终价格上增加 1-2，000 美元。查找并单击显示包含费用的价格的过滤器。
*   不要忘记相关成本。如果你有足够的钱买一张票，太好了！除了，不太好，因为你还得去看比赛，找个地方睡觉。请记住，如果你耐心等待，票价可能会下降，但随着时间的推移，住宿和机票价格几乎肯定会上涨*，因此达到最佳状态可能是一个挑战。*

*如果你有现金并且愿意付出努力，你可以在超级碗开球时到场。对于我们其余的人来说，是时候开始为聚会 批量购买玉米片了。*
