# 根据科学，在小便池里小便的唯一正确方式

> 原文：<https://lifehacker.com/physicists-reveal-the-best-way-to-pee-into-a-urinal-1462946787>

杨百翰大学的一组物理学家说:“随意使用小便池可能会带来灾难性的后果。”他们发现了往小便池里撒尿的最佳方式，而不会溅到地板上，或者更糟糕的是溅到你的衣服或你自己。



该团队设置了一个水箱和喷嘴，以模拟尿液离开人体的方式，并用高速摄像机拍摄了这一过程。他们还设置了第二个水箱来模拟坐在马桶上，而不是站在便池旁。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-UgOx7zPtsaM&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-UgOx7zPtsaM&start=0) 

在他们的分析中，研究人员确定，如果男人坐在马桶上而不是站在便池旁，他们会有最少的溅回。如果你必须使用小便池，Phys.org 报道:

> 男人应该尽可能站在靠近便池的地方...同样有帮助的是引导水流以向下的角度冲击便池的后部。这就减少了回溅，而那些反弹的水滴会向下进入便池排水口。相反，为了防止弄脏自己的裤子(或激怒邻居)，他们建议男性不要直接喷到便池或便池底部的水池中，这两种方式都会引起大量的溅水。

所以记住:靠近，瞄准。
