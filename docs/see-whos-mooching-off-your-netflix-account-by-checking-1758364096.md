# 通过查看最近的访问权限，了解谁在窃取你的网飞帐户

> 原文：<https://lifehacker.com/see-whos-mooching-off-your-netflix-account-by-checking-1758364096>

取决于 [你支付你的计划](https://www.netflix.com/ChangePlan) 的多少，你的网飞账户一次只能有这么多人使用。前任、旧室友或轻率的兄弟姐妹——任何你不明智地信任其密码的人— 可能会从你花大价钱购买的账户中偷钱，但找出他们是否是并不太困难。

一次简单的网飞账户设置之旅，你就能发现 访问过你账户的的 ip 地址和位置。以下是找到它的方法:

1.  在浏览器中进入网飞主页并登录。

2.  在右上角你会看到你的帐户符号。将鼠标放在上面，然后点击**“账户”**

3.  向下滚动并点击**“最近的设备流活动”**链接。

4.  然后单击“查看最近的帐户访问”链接。

你会看到 IP 地址，位置和类型的设备已经一直在看网飞与你的帐户，以及何时。你大概能从那里推断出谁一直在使用你的账户。一旦你知道谁在偷钱，你可以要求他们获得自己的帐户，不要再用不良电影来填充你的队列，或者你可以返回到帐户页面，选择**“注销所有设备”**，然后在你的帐户设置中更改你的密码，永远将他们拒之门外。

网飞也允许你下载一些内容到你的设备上，但是允许下载的数量会根据你每月计划的细节而有所不同。在您的帐户页面中，您还可以选择**“管理下载设备”**来删除任何未授权的下载设备；这将删除下载到这些设备上的内容，让您可以将内容卸载到自己的手机或平板电脑上。
