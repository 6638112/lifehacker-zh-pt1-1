# 最新的Wordle克隆版非常适合电影迷

> 原文:[https://life hacker . com/the-latest-wordle-clone-is-perfect-for-movie-geeks-1848635062](https://lifehacker.com/the-latest-wordle-clone-is-perfect-for-movie-geeks-1848635062)

一开始，有沃尔多。沃尔多在线，沃尔多很好。万物从中萌发，结果，繁衍: [一次四个字](https://lifehacker.com/you-should-try-quordle-if-youre-too-good-at-wordle-1848540319) 。 [*八个*字同时出现](https://kotaku.com/wordle-dordle-quordle-octordle-word-game-browser-puzzle-1848593721) 。 [数学单词](https://kotaku.com/wordle-clone-nerdle-free-word-game-puzzle-browser-math-1848547825) 。 [地铁二字](https://kotaku.com/games-like-wordle-subwaydle-nyct-mta-best-routes-1848461481) 。 [同性恋词](https://lifehacker.com/6-wordle-esque-games-that-are-just-as-good-or-better-1848426197/slides/5) 。正当我的信念动摇，我开始怀疑沃尔得是否会横行地球时，在最大的考验和最深的厌倦的时候，我自己的救世主来了:一个基于票房琐事的沃尔得版本。

许多人用来储存社会可接受信息的大脑部分，如各种体育比赛的获胜者或有用的历史事实，我已经花了几十年时间塞满了票房统计数据。说出一部在过去30年里上映的电影，我很有可能给你一个粗略的估计，它最终在美国的票房总收入。如果这是一部真正受欢迎的电影，我甚至可以告诉你它在首映周末的票房。与大多数琐事相比，这些信息完全没有意义——可悲的是，知道一部电影的票房总收入并不意味着你有权从中分一杯羹——或者说*是*，直到我发现了 [票房游戏](https://boxofficega.me/) 。

像Worldle一样，这款基于浏览器的游戏每天都给你提供一个新的谜题。但你不是试图去猜一个特定的单词或完成一个数学等式，而是试图找出某个特定周末的票房前五名。你会看到发行这部电影的电影公司的名字，以及那一周的收入和到那时为止的总收入。正确猜中名单上五部电影中的一部将为你赢得200分。

如果你需要更多的帮助——你很可能需要，除非你在业余时间完美地回忆和研究 [票房魔咒](https://www.boxofficemojo.com/) (有一项罪名成立)——你可以将这200分中的一部分“花”在一条线索上，这条线索可以是电影的口号、流派、导演、演员阵容或最终总收入，越多的提示会更显著地降低你的总分。每天的最高分数是1000分，所以你可以简单地把它们都做对，或者尝试竞争电影Twitter吹牛的权利。

我第一次知道票房游戏 [是在Reddit](https://www.reddit.com/r/blankies/comments/t8p32l/wordle_style_box_office_game/) 上，在 [某论坛](https://www.reddit.com/r/blankies/) 专门给我 [喜欢的电影播客](https://lifehacker.com/10-podcasts-for-discovering-new-movies-1844172713)[*空白支票与格里芬*](https://www.blankcheckpod.com/) 。该节目的每一集都专注于特定导演电影史上的一部电影，每一集都以大卫·格里芬对该电影首映周票房最高的电影的提问结束。(此时，你要么在电脑前做鬼脸，要么冲到你最喜欢的播客应用程序前订阅。Redditor u/hebleb(真名:Mark Uvari)创作了这款游戏，灵感来自于对Wordle的狂热和Griffin在他出生两年后的一个随机周末回忆起影院第三部电影的不可思议的能力。乌瓦里在日常生活中是一名软件工程师，拥有八年的网络和移动应用程序开发经验，他通过Reddit聊天告诉我，他是“一个普通的琐事爱好者，尽管我也承认自己的游戏很糟糕。”

驱动游戏的数据来自[the-numbers.com](https://www.the-numbers.com/)；Uvari还使用API从[(IMDb)](https://www.imdb.com/)互联网电影数据库中提取关于某部电影的所有特定信息。直到20世纪80年代，票房数据还没有可靠的汇编，所以谜题总是只涉及过去40年中的某个特定周末——这仍然足以运行大约2000个日常游戏。正如Wordle的克隆体一样，票房游戏无疑是一个更狭窄的领域，但它也很好地说明了互联网团结的力量。因为不管你有什么奇怪的执念，你离一群让你看起来像个业余爱好者的人只有几下点击的距离。还有，我 [终于比我老婆](https://lifehacker.com/how-to-win-at-wordle-according-to-my-wife-1848480285) 更擅长其中一项了。
