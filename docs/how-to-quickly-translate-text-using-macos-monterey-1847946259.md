# 如何使用macOS Monterey快速翻译文本

> 原文:[https://life hacker . com/how-to-quick-translate-text-using-MAC OS-Monterey-1847946259](https://lifehacker.com/how-to-quickly-translate-text-using-macos-monterey-1847946259)

macOS 12中最好的新功能之一[Monterey](https://lifehacker.com/31-new-macos-monterey-features-youll-actually-want-to-u-1847900965)允许你翻译Mac上的任何可选文本——无需让谷歌为你做这件事。虽然以前可以通过Safari进行翻译，但它的用途有限，因为它只能翻译整个网页；其他应用程序中的文本翻译起来就不那么容易了。现在，这个特性已经被普遍实现了，我们将向您展示如何使用它。

## 使用Mac全系统翻译功能需要什么

苹果随macOS Monterey推出了全系统翻译功能，所以你显然需要先把你的Mac更新到最新版本的macOS。除了此更新之外，有问题的应用程序必须添加对该功能的支持，对于更新频率较低的程序，这可能需要一些时间。

苹果自己的应用已经支持这一功能，许多第三方应用也是如此。对于大多数人来说，这个选项应该是无缝的。如果一个应用程序还不支持快速翻译，你可以随时将文本复制到苹果的Notes应用程序，并从那里进行翻译。

## 如何在Mac上快速翻译文本

首先，选择您想要翻译的文本(记住，macOS Monterey也允许您选择照片中的文本)并右键单击它(或使用control + click快捷键)。这将打开一个上下文菜单，您可以在其中选择**翻译**。原文将出现在浮动窗口的上半部分，译文出现在下半部分。

在两种语言中，文本旁边的播放按钮可以让您听到大声朗读的文本，如果您想要检查某些内容的发音，这将很有帮助。

在这个浮动窗口中还有几个其他有用的选项。**复制翻译**让您快速将翻译的文本复制到剪贴板。如果你使用的是文本编辑应用，你还会看到选项**替换为翻译**。这正如它所说的那样，用翻译的版本替换您选择的文本。

## 如何强制你的Mac即使离线也能翻译文本

默认情况下，你的Mac会连接到苹果的服务器来提供你的翻译。如果你离线或担心隐私，你可以切换到离线翻译。这样做会降低翻译的准确性，但总比什么都没有好。

要切换到离线翻译，请前往**系统偏好设置>语言&区域**并点击窗口底部附近的**翻译语言**。此页面将显示支持翻译的所有语言；点击您需要离线使用的语言旁边的**下载**按钮。(如果你想把英语翻译成西班牙语，你需要下载这两种语言，这样你的Mac才能离线翻译文本。)

一旦你下载了你需要的所有语言，在同一窗口的底部启用**设备模式**并点击**完成**。从现在开始，你所有的翻译都将离线处理，不会向苹果的服务器发送任何数据。
