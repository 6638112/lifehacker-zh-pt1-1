# 如何和孩子谈论令人心烦的时事

> 原文:[https://life hacker . com/how-to-talk-to-kids-upsetting-current-events-1846007672](https://lifehacker.com/how-to-talk-to-kids-about-upsetting-current-events-1846007672)

很多时候，我们可以和孩子一起准备“”的大型会谈。我们可以研究和计划如何与他们谈论从性到色情，从LGBTQ问题到种族主义的一切。但是，不知从哪里来的，一些意想不到的——和令人深感不安的——出现在新闻中。毁灭性的飓风。大规模枪击。一群暴徒闯入了我们国家的国会大厦。

即使我们试图处理我们所目睹的事情，即使恐惧、悲伤和焦虑有可能吞噬我们，我们的孩子也需要我们。他们需要我们帮助他们处理他们的所见所闻。他们需要我们引导他们度过任何可怕的事情。这里有一些方法可以做到这一点。

### 向他们报到

当新闻中有不寻常的事情发生时，我们的孩子很有可能会听到一些关于它的消息——无论是在家里还是从同学或朋友那里。即使你在一个更小的孩子面前设法关掉电视，他们也可能会注意到你情绪的变化，注意到你比平时更专注于手机(脸上带着忧郁的表情)，或者他们会听到你在厨房小声告诉你的伴侣最新情况。

请记住，在缺乏真实信息的情况下，如果某件事看起来不对劲，而他们不知道那是什么事，他们可以假设*他们*在某种程度上有错。小孩子非常擅长为与他们完全无关的事情责备自己。没有理由不必要地惊吓学龄前或小学年龄的孩子，但是如果你知道或怀疑他们已经染上了某些东西，与他们联系是很重要的。问他们看到或听到了什么，问他们感觉如何，以平静、理性和 [与年龄相适应的方式](https://www.commonsensemedia.org/blog/explaining-the-news-to-our-kids) 回答他们可能有的任何问题。

对于青少年来说，谈论他们所看到的尤其重要，因为他们更有可能从网上独立于你吸收他们自己的新闻，或者与可能被误导的朋友讨论时事。你可以帮助纠正任何不准确的地方，提供额外的上下文，或者引导他们去寻找其他的资源，以帮助他们处理发生的事情。

请记住，如果你不知道某件事的答案，可以这么说。告诉他们你会找到他们问题的答案，或者通过安全可靠的来源一起研究答案。

### 塑造健康的新闻摄入和应对机制

当自然灾害、暴力犯罪或极端的政治动荡发生时，人们很容易打开电视继续看新闻。特别是当一个故事仍在展开时，如果有可能发生令人不安的事情，最好限制他们接触正在进行的新闻。

只要有可能，试着一起看新闻，这样你们就可以谈论你们看到或读到的东西。决定新闻的哪些方面可能有助于他们看到，例如领导人鼓舞人心的讲话或守夜或纪念活动的图像。然后，关掉它，休息一下，和家人一起共度时光。一起去散步，一起做晚饭，拿出一个桌游，或者一起蜷缩在沙发上看一部搞笑的电影。

你的孩子会看着你来决定他们应该如何对这个消息做出反应，以及如何应对他们可能感到的任何悲伤或愤怒，所以现在是时候保持冷静，并树立自我照顾的榜样。

### 关注积极的一面

当压力或痛苦的消息展现在我们面前时，很难找到积极的一面——但积极的一面就在那里。也许有一些人正在保护我们的民主，也许有一些邻居在自然灾害后互相帮助重建家园，也许有一些第一反应者冒着生命危险去拯救他人。在令人沮丧的消息之后，关注任何你能找到的团结、坚持或支持的主题。

然后谈谈作为一个家庭，你能做些什么来帮助他们。有没有你可以支持的组织，或者可以一起参加的集会？也许你可以写封信寄给你选出的官员。即使是在你自己的社区做出一些善举，也能帮助孩子们感受到成为世界上美好事物的一部分的动力。
