# 如何使用 Windows 10 的新“功能体验包”

> 原文：<https://lifehacker.com/how-to-use-windows-10s-new-feature-experience-pack-1845799918>

准备好接受更多 Windows 10 功能了吗？微软现在正在测试一个新的“功能体验”程序，它将在常规的可能会破坏你的电脑的大更新节奏之外，给 Windows 10 添加一些小的额外功能。换句话说，这是微软给你新玩具玩的一种方式，除了它为 Windows 10 发布的一年两次的更新。

Watch

微软首个 Windows 10“功能体验包”目前面向 Windows 10 内部人员。我在“beta channel”赛道，昨天刚看到在 Windows Update 弹出的包；我怀疑对于那些在 Windows Insider“开发”频道中生活在野外的人来说也是如此，但是那些在“发布预览”频道 [中最小化风险的人还不会有包装](https://blogs.windows.com/windows-insider/2020/11/30/releasing-windows-feature-experience-pack-120-2212-1070-0-to-the-beta-channel/) 。

一旦你安装了功能体验包，你将重新启动 Windows 10，发现什么都没有改变——你不会得到一个弹出窗口或任何其他东西来通知你刚刚安装的所说的功能包中的*是什么。我想微软认为你天生就知道，或者你在安装更新之前已经做了功课。*

所以，你可以这么做。

首先，你现在可以使用 **Snip & Sketch** 截图，并更容易地将它们转储到文件资源管理器中的任何文件夹。使用您的 Windows 键+ Shift + S 快捷键，像平常一样进行“截图”,或者直接打开应用程序，您可以选择——一旦您拍摄了截图(在此过程中将其复制到您的剪贴板),就可以在文件资源管理器中访问您想要的任何文件夹。按 CTRL + V 将你的图片直接粘贴到文件夹中，然后将其重命名为易于管理的名称。

特性包中的第二个更新有点深奥。我让微软来描述一下这个:

> *现在，在 2 合 1 触摸设备上以纵向姿势使用触摸键盘支持分离键盘模式。*

我怀疑这不适用于大多数人，所以没必要担心；复制和粘贴更新是这个包的“大”部分，它实际上是有帮助的，因为它使你不必总是手动保存文件。复制粘贴到文件夹要快得多；现在，如果微软也允许你从我最喜欢的应用程序，即“Snip ing Tool”，来做这件事，Snip & Sketch 将最终取代它。我会想你的。
