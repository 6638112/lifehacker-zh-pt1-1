# 为什么你应该在购买游戏前尝试试玩

> 原文：<https://lifehacker.com/why-you-should-always-try-the-demo-before-buying-a-game-1847988653>

你多久买一次电子游戏？你是经常花 60 美元买最新的任天堂游戏的人，还是只等那些你一直密切关注的游戏？不管怎样，不管你的购买习惯如何，不管 T2 有多确定你想要这款游戏，只要有可能，你应该在购买之前试一试它的试玩版。

Watch

## 游戏演示帮助您做出明智的购买决定

电子游戏很贵。AAA 级图书通常售价 60 美元，最新一代的一些要价 70 美元。也就是说，我总是喜欢指出，在 90 年代，许多视频游戏的价格是 60 美元，相比之下，这无疑使今天的游戏看起来更实惠一些。

但是我跑题了。2021 年六七十美元还是很多钱。如果你是一个狂热的游戏玩家，即使是独立游戏和价格可能低得多的老游戏也能赚到钱；到处买一个“便宜”的游戏，却在月底发现你的信用卡账单已经满了，这太容易了。

演示在这里真的很有帮助。他们满足了对“新”渴望你想玩一些新鲜的，不同于你收藏的东西，但是几个小时后，你不会对一个 60 美元的游戏感到厌倦，演示是免费的。如果你真的喜欢这个演示，并且觉得购买完整的游戏是值得的，那就好！去做，一路支持游戏开发者。然而，如果游戏不是你所期望的，或者觉得不值得，你会免费知道。

但是你可能已经知道玩这个演示的内在好处。我认为有第二个更迫切的理由让这些演示先玩一把；他们人数不够。

## 播放演示可以激发市场上更多的演示

如今演示并不常见。当然，有*有*的演示；例如，你可以在 PlayStation 上免费播放《T2》和《银河战士恐惧》的试玩版，还可以尝试《T4》和《《我的世界》》。但是对于许多游戏来说，在你购买之前根本没有办法去尝试。你唯一的选择是在线观看尽可能多的评论并玩-through，这和你自己玩游戏的效果不一样。

最终，你可能会觉得你别无选择，只能放弃这个游戏。也许这是一个正确的决定，你对你的新冒险感到兴奋。或者，也许游戏并不完全是你想象的那样，现在你有了购买者的懊悔。又来了。

那么，我们该怎么办？嗯，当然是播放演示。如果越来越多的人在购买游戏前玩游戏演示，也许开发者会考虑更频繁地加入它们。毕竟，如果这些工作室希望抓住尽可能多的观众，难道他们不希望观众能够看到一款游戏是否适合他们吗？

我不能代表这些工作室，但我可以想象一个问题是，给玩家演示会劝阻他们购买游戏，如果他们发现他们不喜欢它。虽然这肯定是一种反消费者的理念，但我认为他们会发现更多的人最终会购买他们的游戏——当我真的想要一个游戏时，我不能独自购买游戏。如果我有机会尝试更多的游戏，我可能会找到其他我不知道我想要的游戏。

我发现这是我买 PSVR 时的情况；由于该系统非常新颖，索尼创建了不同的“演示光盘”，其中包括一系列 VR 游戏演示，让您体验不同类型的虚拟现实游戏。这是天才；我很快找到了我 T3 买的游戏。如果同样的精神可以应用到更多的新游戏中，我相信我会更频繁地购买它们。

在那之前，你可以在线浏览*和*的演示，也可以在你的系统的电子商店上浏览。我在这里链接了 [任天堂](https://www.nintendo.com/games/games-with-demos/)[索尼](https://store.playstation.com/en-us/search/demo/1)[微软](https://www.microsoft.com/en-us/store/game-demos/games/xbox) 的试玩列表；浏览一下这些列表，看看是否有你感兴趣的标题。演示可能会帮助您进行新的购买。
