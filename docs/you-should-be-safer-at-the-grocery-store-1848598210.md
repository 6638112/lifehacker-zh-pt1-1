# 你在杂货店应该更安全

> 原文：<https://lifehacker.com/you-should-be-safer-at-the-grocery-store-1848598210>

当我们想到食物中毒时，通常是在频繁的食物召回背景下，从 [沙拉](https://lifehacker.com/don-t-eat-these-recalled-salads-fda-says-1848344484) ，到 [洋葱](https://lifehacker.com/throw-your-onions-in-the-trash-cdc-says-1847907564) ，到 [干香料](https://lifehacker.com/check-your-spices-to-see-if-theyre-part-of-a-recall-ove-1845468692) 。



但是 [也有可能食物中毒](https://www.consumerreports.org/medical-conditions/surprising-ways-you-can-get-food-poisoning-a7726863758/) 如果感染了病毒或细菌的人(可能有也可能没有症状)上完厕所后没有彻底洗手，然后触摸会接触到食物的东西 你最终会吃东西(或食物本身)。事实上，根据美国食品药品监督管理局的说法，沙门氏菌、大肠杆菌和诺如病毒——食源性疾病的主要原因——经常通过这种方式传播。

不仅如此，你购物的方式，以及从商店到你家的食品包装和储存方式，都会导致食品安全问题。以下是如何在你下次去杂货店时避免这些。

## 如何在杂货店更安全

在《消费者报告》 最近的一篇文章中，该杂志的首席科学官詹姆斯·迪克森博士提供了以下建议，让你下次去杂货店更安全:

### 清洗你的可重复使用的帆布包

如果你依靠帆布包把你的杂货从商店运到家里，定期清洗它们是很重要的。迪克森解释说，除了可见的污垢和污渍，这些袋子还会被细菌和霉菌污染，这些细菌和霉菌会随着时间的推移从袋子中携带的食物中积累起来。

### 擦净手推车或篮子

迪克森说，不管有没有疫情，这都是我们应该一直做的事情。没有人期望你做一次彻底的擦拭，但是如果没有别的，确保你有把手(这是每个人都接触的)和座位(那是孩子们尿布的屁股去的地方)，以及你期望经常接触的任何其他部分。

### 根据食物变质的时间顺序购物

从在中间通道买你需要的东西开始你的杂货店之旅，中间通道通常包含不需要冷藏的食物，并且有更长的保质期。

然后，迪克森建议先去冷冻食品区，然后是新鲜食品、农产品和奶制品，最后是你单子上的生食——包括熟食柜台上的任何东西——因为它们对温度最敏感。他还建议购物时至少带一个隔热袋，这样有助于保持从商店到冰箱所有东西的凉爽。

### 关注新鲜农产品

在把你打算买的任何湿农产品放进塑料袋之前，轻轻摇晃一下。当袋装的湿农产品放在冰箱里时，它会比干燥的农产品腐烂得更快。

### 小心漏肉汁

用一个从里到外的袋子抓住任何生肉和鲜肉(包括鱼),然后把它从右到外翻过来。迪克森说，这可以防止你接触任何可能漏出的肉汁，以及无意中将细菌传播到你的食品杂货上，以及你在商店的其他地方接触的任何东西。将肉包放在塑料袋中可以确保任何泄漏的汁液不会扩散到车上的其他物品。
