# 哪种冷冻土豆做的土豆煎饼最好吃？

> 原文：<https://lifehacker.com/which-frozen-potato-makes-the-best-hash-browns-1845313831>

我不知道有哪个人不喜欢脆皮土豆。事实上，我们非常爱它们，我们已经知道如何在早餐、午餐和晚餐时享用它们。早餐土豆有两种主要形式:土豆煎饼和家庭薯条。我不喜欢后者，主要是因为它们太大了。我喜欢把我的早餐土豆切丝或剁碎，以获得最大的口感享受，如果你和煎鸡蛋或荷包蛋一起吃，这一点很重要。

土豆煎饼也是商店里买的比自制的要好的食物之一。即使你从零开始做了一个神奇的土豆煎饼，也不值得额外的努力，尤其是当预先成型的土豆煎饼(一个完美的产品)存在的时候。冷冻土豆丝也很好吃。

最近我一直在搅拌许多预先成型的棕色肉饼，并把它们放在早餐三明治 中 [。华夫饼干比在烤箱里煮要快得多，而且它们会变得更脆。华夫饼干烙铁将油脂从土豆饼中挤压出来，进入华夫饼干烙铁的脊部，在那里烤得咝咝作响，将土豆馅饼煎得完美无缺。](https://skillet.lifehacker.com/put-hash-browns-inside-your-breakfast-sandwich-1843731822)

我听说搅打土豆泥也同样有效，虽然从技术上来说，土豆泥不是早餐土豆，但它们肯定能起到同样的作用，尤其是如果你把它们搅打成土豆泥小馅饼的话。由于我有一颗非常好奇的心，我决定比较华夫饼干和华夫饼干，以及经典的碎薯饼。(所有的土豆都是从冷冻的放入不粘的电动华夫饼干炉中，设定为中高火，不添加油或调料烹饪 4-7 分钟，直到外面变脆。)

### 最糟糕的:土豆丝

如你所见，这些东西不是用来瞎扯的。事实上，它们自身不提供油脂，尽管它们最终会形成一种外壳，但它们永远不会真正变脆，只会变硬。这些贫血的土豆配不上你煮得很熟的鸡蛋。看着它们让我难过。

### 非常好的食物:土豆饼

对于这些，我没有什么不好说的；它们是完美的金黄色、油腻的马铃薯片，同样适合放在炒菜旁边或三明治上。华夫饼机不仅能在几分钟内完成烘烤，还能通过制造草皮来最大限度地增加松脆的表面积。到处都是高分。

### 最好的:土豆泥

华夫饼干的味道和感觉几乎与肉饼一模一样，但它们有一个主要优势:定制。由于它们体积小，形状圆，你可以根据自己的需要在华夫饼机中放入或多或少的土豆饼。需要一大堆土豆来做早餐杂烩吗？将华夫饼机装满到最大容量。需要一排脆皮土豆来填早餐玉米煎饼？像小兵一样把它们排成一排。想创造一个鸡蛋本尼迪克特式的场景吗？用四到五个做一个英式松饼大小的馅饼。它们基本上是土豆乐高积木，最好的一种。(是的，我知道这在技术上是“乐高积木”或其他什么，但我不在乎。)
