# 如何做出最好的大麻布朗尼

> 原文:[https://life hacker . com/how-to-make-the-best-possible-pot-brownies-1819455350](https://lifehacker.com/how-to-make-the-best-possible-pot-brownies-1819455350)

我们可能还不知道谁会是我们的下一任总统，但2020年的选举结果已经明确了一件事:无论是红州还是蓝州，美国人都喜欢大麻。四个州——新泽西、南达科他、蒙大拿和亚利桑那— [在这个选举周期投票将娱乐性大麻](https://www.nytimes.com/2020/11/04/us/ballot-measures-propositions-2020.html) 合法化，使得以娱乐为目的吸食大麻合法的州达到15个。为什么不吃一块真正美味的巧克力饼来庆祝这个好消息呢？

我敢打赌，相当一部分14-64岁的人会问“这里有杂草吗？”每当他们拿到自制的巧克力饼。THC-laded巧克力糖果是如此普遍，以至于这个好友喜剧燃料往往是人们第一次尝试食用大麻。(这可能是因为巧克力能够掩盖几乎所有令人不快的味道。)太浓的核仁巧克力饼有时会带有生大麻的苦涩，但如果比例合适，你可以做出既美味又卑鄙的核仁巧克力饼。

一些面包师选择巧克力ganache，而其他人则喜欢可可粉。两者都可以单独给你又湿又粘的核仁巧克力饼，但是把两者结合在一起会使它们变得令人上瘾，不管是不是大麻，它们都是独立的。核仁巧克力饼烘烤起来就像一个特别致密的蛋糕，平底锅和烘烤时间决定了你最终得到的软糖的程度。一个大而宽的平底锅会给你一个厚而有嚼劲的结果，而 [较小份的平底锅](https://www.amazon.com/Wilton-2105-1829-Mini-Round-1-25-Inch/dp/B007SQ7XXQ?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-to-make-the-best-possible-pot-brownies-1819455350&asc_source=&tag=kinjalifehackerlink-20) 会烤得超级快，而且长高。

为了做出完美的巧克力饼，我们调用了稍微奢华一点的亚瑟王面粉T3的配方。我喜欢这个，因为它经过了专业配方开发者的充分测试，但如果你想做一些调整，也可以原谅——这是我们为了最大限度地消除忽悠因素而推荐的。提前做好你的大麻黄油，放在冰箱里，使用前解冻。(你*能不能*把日子定下来；在你打算开始烘烤之前，一定要给自己三个小时的黄油准备时间。)

### 准备注入大麻的黄油

所有烘焙大麻的新手都可以庆幸，到2020年，你不必在互联网的编年史中寻找兰多从1996年 开始的 [恐怖尝试。“遗产”食谱倾向于昏迷诱导强烈，大多数人真的不觉得这是一个好时机。在像](https://erowid.org/experiences/exp.php?ID=39965) [神奇黄油机](https://magicalbutter.com/) 和 [Levo](https://www.levooil.com/) 这样的设备之间，您可以将(大部分)猜测留给设备，并根据您的确切规格生产出一致的产品。

小批量输液的最佳方法是life hacker classic:。简单地用八分之一的中高档细磨黄油和 [脱羧芽](https://www.leafly.com/news/cannabis-101/what-is-decarboxylation) 密封一条黄油，在170℉或160℉加热一两个小时。较短的烹饪时间会让你获得更多的早期大麻素，如CBD和THC，并保留赋予每种大麻独特气味和味道的萜烯。更长的烹饪时间提供了更多的令人昏昏欲睡的黄油，其中含有更多的CBN，这是众所周知的沙发锁，如果你患有失眠症，就不要忽视它。

### 做布朗尼

**为布朗尼:**

*   1块浸有3.5克大麻的无盐黄油(见上文)

*   杯装蔬菜起酥油

*   一杯可可粉

*   1杯蛋糕粉

*   两杯糖

*   茶匙盐

*   4盎司苦甜或半甜巧克力

*   茶匙发酵粉

*   ⅛茶匙小苏打

*   2个巨型鸡蛋

*   一颗香草豆的内脏

将烤箱预热到350华氏度。如果你已经把黄油放在一个塑料袋里，在搅拌面糊之前，把它倒入一个装有温水的碗里，让它完全变软。烘烤任何核仁巧克力饼，甚至是大麻核仁巧克力饼，都需要你的标准湿碗、干碗面糊技术，所以先把所有的干原料混合在一起。然后，在第二个碗里，混合香草和鸡蛋。

用双蒸锅或微波炉短时间融化巧克力。将起酥油搅拌到巧克力融化。将巧克力混合物慢慢倒入鸡蛋中，用刮刀搅拌，然后加入大麻精。确保所有东西混合均匀，但不要充气过多。

将湿面糊倒入混合的干材料中，再次搅拌这种极其粘稠的糊状物，直到完全混合。这就是我的bff [OXO刮碗器](https://www.oxo.com/products/cooking-baking/baking-tools/2-piece-bowl-scraper-set) 派上用场的地方:它们可以让你把最后一滴软糖挤到一个等待着的涂油并铺有羊皮纸的9x7平底锅中。弄平混合物，在柜台上拍打几次，确保里面没有空气。

烤15分钟，旋转平底锅，然后再烤15分钟。从烤箱中取出，放入锅中冷却至少10分钟。此时，你的布朗尼是温热的，可以食用，但要注意，它们的结构将完全冷却，类似于 [巧克力饼干](https://lifehacker.com/bake-the-best-chocolate-chip-cookies-by-knowing-what-to-1593805654) 。如果你不能等到它们完全冷却，至少用一把又大又锋利的刀切下一块肉，这样你就不会对内脏造成太大的伤害。用纸把它们从锅里拿出来，剩下的放在铁丝架上冷却。

一份，或者不要，但是记住每1平方英寸含有大约29毫克的四氢大麻酚，这是一种有效的剂量，所以要吃小块。你也可以把一半的黄油换成非精神类的，把剂量降到15毫克，这对不习惯吸烟的人来说更合理。遵循由 [莫琳·多德](https://www.nytimes.com/2014/06/04/opinion/dowd-dont-harsh-our-mellow-dude.html) 制定的基本食用协议*而不是*，如果你是一个新的旅行者，尽量不要一次摄入超过5毫克。总是等60分钟再吃，因为吃得太多会让你一次受损几个小时。

这些极其丰富的食物太美味了，让人无法抗拒，所以没有人会因为你不含草药而对你评头论足，但是如果你想给某人留下深刻印象或者只是想怀旧一下，这就是你想要的。
