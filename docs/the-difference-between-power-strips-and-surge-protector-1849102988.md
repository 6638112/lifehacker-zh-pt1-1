# 配电盘和电涌保护器之间的区别

> 原文：<https://lifehacker.com/the-difference-between-power-strips-and-surge-protector-1849102988>

除了你的浴室之外，你可能有多种电器、设备、照明和电子设备在你家的大多数房间里——所有这些都需要插上电源。此外，还有像吸尘器、风扇和其他物品可能没有自己的专用插座，但有时需要一个。

除非你的家是新建或翻新的，否则很可能没有足够数量的电源插座为你日常使用的所有东西供电。为了帮助弥补这一点，您可以使用电源板或电涌保护器来容纳额外的插头。但是哪个是哪个呢？

虽然它们看起来一样，而且这两个术语经常互换使用，但电源板和电涌保护器不是一回事。以下是需要了解的内容。

## 配电盘和电涌保护器之间的区别

电源板和电涌保护器都由一根带有多个插座的延长线(长度不等)组成，因此很容易看出哪里会出现混乱。

不同的是，电涌保护器有另一个元素，确保任何插入其中的电子设备在电涌事件中不会损坏(这是当 [额外的电力](https://www.cnet.com/tech/home-entertainment/10-shocking-facts-about-power-strips/) 进入你的家)。插线板可能有一个开/关开关，但 [并没有提供比墙上插座更多的保护(如果有的话)](https://www.cnet.com/tech/home-entertainment/10-shocking-facts-about-power-strips/) 。

据美国保险商实验室(UL)[的专家称，电涌保护器将“钳制”电涌——这意味着它将电涌从下游带走(即插入其插座)。如果你住在农村地区，住在有很多大型电机的建筑里(比如一个大的炉子或者电梯)，或者住在经常有暴风雨的地区，电涌保护器就特别重要——所有这些都会沿着电力线发送电涌。](https://www.ul.com/insights/guide-power-strips-and-surge-protectors)

底线:购买电源板或电涌保护器时要注意包装，以确保你买的是你想要的产品。
