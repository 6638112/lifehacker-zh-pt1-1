# 购买合适房车的初学者指南

如果你想去偏远的地方住在房车里(或者只是用房车露营或公路旅行)，你会想了解许多选择——从泪珠拖车到全公共汽车大小的房车。

重要的事情先来。有三种机动房车 和一整类拖车:

*   A级车:A级车是最大的房车，类似于商用巴士。它们通常拥有你实际家中的所有便利设施，最多可睡八个人，长21-45英尺。
*   B类:这是你的野营车或改装货车，是最小的房车。它们配有折叠床和基本的便利设施，最多可睡四个人，长17-19英尺。
*   C级:这些中型房车只是a级的缩小版。它们还配备了大多数便利设施，最多可睡八个人，，身长20-31英尺。
*   **拖车:**这一类别包括房车折叠拖车、房车旅行拖车、和第五轮拖车(安装在皮卡车底板上)。显然，你需要一辆能够牵引这些拖车的车。这里的其他选择包括较小的泪珠和弹出式拖车，用于较短的周末旅行。

### 你需要多大的空间？

购买房车的一个重要考虑因素是你需要多少人睡觉。如果只是你和一个伴侣，你可能会享受到A级的舒适，但你可能不需要。如果你打算带上你的孩子、朋友、和*他们的*孩子，额外的空间可能是必不可少的。

空间考虑还包括做饭、工作和闲逛的空间。你会在更大的车辆中得到很多，但在第五轮或弹出式拖车中不会有这么多。整天住在你的房车里是另一个挥霍更多空间的理由。

### 你打算去哪？

不是所有的房车都适合所有的地方。一些国家公园、国道和山路上的车辆长度限制可能会阻止你驾车通过。例如，冰川国家公园的阳光大道不允许长度超过21英尺的车辆通行。在某些地区，您还可能会遇到净空限制或停车问题。这不应该阻止你买你需要的尺寸，但是如果你犹豫不决，较小的房车可能会给你更多的选择。

### 在路上需要做什么？

例如，一些房车可以更好地安装内置办公桌和许多插座，以便远程工作。其他人有豪华厨房供娱乐或严肃的家庭厨师使用。一辆Airstream或Sprinter面包车具有美学吸引力。你也要考虑你是否有水、电、和污水、的外部连接，或者你是否需要自给自足。

### 你是一个自信的司机吗？

如果你不确定20-40英尺。-长途公共汽车或拖车在你的车后晃来晃去，B级货车可能适合你。考虑你是否会开车穿过繁忙的城市，你是否需要在拥挤的地方停车，以及你多久会开着你的房车在镇上逛一次。

### 你有拖车吗？

如果你驾驶一辆轿车，拖车通常是不可用的(除了少数例外:微型 [Polydrop](https://www.polydrops.com) 几乎适用于任何乘用车)。确保你的SUV或卡车能够承受你正在考虑的拖车的尺寸。如果你想要一辆第五轮拖车，你需要一辆皮卡。

### 你想花多少？

这是最重要的问题之一。对于拖车，你看的是一辆新车的价格，尽管有些可能会超过10万美元。B级房车的价格可能在5万至8万美元左右，最豪华的a级房车可能高达20万至30万美元。

### 先租

没有几次试驾就不用做这个决定。 [RVShare](https://rvshare.com) 和 [Outdoorsy](https://www.outdoorsy.com) 提供点对点RV、露营车、和拖车租赁——带轮子的航空站。也有针对房车和面包车的地区性度假租赁公司，像西海岸的 [Trail Mix房车](https://trailmixrv.com) 和西部山区的 [Moterra](https://gomoterra.com) 。
