# 我是如何学会停止嫉妒并最终继续我的生活的

> 原文：<https://lifehacker.com/how-i-learned-to-stop-being-so-jealous-and-finally-get-1529920256>

嫉妒和羡慕是我们许多人最常见的两种情绪，然而却是消极和无用的。很长一段时间，我让这两种破坏性的感觉压倒并毒害着我。这就是我最终控制他们的方法。

### 嫉妒和羡慕:一个案例研究

承认这些缺点对我来说很难(尤其是面对成千上万的陌生人)，但我一直在学习，要真正克服这些缺点，需要认真审视自己的缺点。也许是因为我有“中间孩子综合症””或者也许是我通常隐藏的好胜心，但是嫉妒——有人试图拿走你拥有的东西的感觉——和嫉妒——因为有人拥有你没有的东西而感到愤怒——这两种感觉对我来说总是很自然。

我对这些丑陋情绪的最早记忆是在我大约九岁时的一个圣诞节。我弟弟把他珍藏的变形金刚玩具作为礼物送给了我姐姐。(我相信那是救护车，当它变成一个机器人时，它的两边有红色的十字和枪站。我从他那里得到的只是一张微不足道的卡片——于是我大发雷霆。这是完全合适的。我把玩具扔向墙壁，撕掉卡片，跺着脚走上楼梯，在枕头上放声大哭。(我告诉过你它们是丑陋的情绪。)

在后来的几年里，当男朋友花更多的时间和我们的女性朋友交谈而不是和我交谈时，当同事因为我做得一样好而受到表扬时，或者当别人去做更好更大的事情而我被抛在后面时，类似的感觉就会席卷我。

这就像幸灾乐祸的反义词，但同样琐碎:我没有从别人的不幸中获得快乐，而是对他们的成功感到折磨。在这一切的背后，是我认为自己受到了冷落，这种情况是不公平的，有时，我是不够的。

### 我是如何从嫉妒走向慷慨的

我的突破是偶然和渐进的，而不是一个高潮，为电视制作的时刻。说实话，我甚至不知道这些感觉对我和我的人际关系造成的伤害，甚至没有意识到它们正在发生。

然而，在过去十年左右的时间里，我所做的一些改变帮助我从更健康的角度看待事情:

#### 我开始更加意识到自己的感受和想法

嫉妒和羡慕是本能的感觉，但是当它们露出丑陋的嘴脸时，你可以把它们扼杀在萌芽状态。但首先你必须意识到它正在发生。几年前，我开始学习瑜伽，那时我要去的健身房提供了一个非常好的课程。仅仅是有规律的锻炼可能 [就渗透到了我生活的其他领域](http://lifehacker.com/top-10-reasons-to-exercise-regularly-besides-losing-we-1473616982) :更好的睡眠，增强信心，更好的整体幸福感，但是瑜伽也是 [冥想](http://lifehacker.com/tag/meditation) 或 [正念训练](http://lifehacker.com/what-is-mindfulness-and-why-is-everyone-talking-abo-1502693174) 的运动。我发现自己越来越多地给自己的负面情绪贴上标签，并把自己从负面情绪中分离出来。(不仅要说“我感到一阵嫉妒”，还要说“我感到紧张”和其他所有的话。在某种程度上，我认为经常有其他负面情绪的人，比如愤怒，可以从这些策略中受益。

#### 我知道了竞争和比较的区别

引用“ [比较令人厌恶](http://www.phrases.org.uk/meanings/Comparisons%20are%20odious.html) ”已被认为是几个受人尊敬的作家。基本上它意味着比较(尤其是人的比较)是令人厌恶的。嫉妒和羡慕都是关于比较——计算一个人和你之间的差异，就好像生活是一场会计游戏，以确保你没有亏空。*另一方面，竞争*可能是有益的——只要我们不把它看得太严肃和个人化。我的高中英语老师总是说“攀比是令人厌恶的”，我从来不明白这一点，直到我开始意识到我是在拿自己和别人比较，而不仅仅是和他们竞争(像优秀的女运动员一样)。

#### 我开始练习感恩和快乐

这是哈罗德·科芬的另一句名言:“嫉妒是一种计算别人的幸福而不是自己的幸福的艺术。”当我年轻的时候，我常常数着我的幸福，但不知何故，它们让我感到内疚，而不是幸运。我觉得我不配拥有我出生的这个伟大的世界，因为我没有赢得它。现在，几乎每天早上，我都会在起床前 [练习](https://lifehacker.com/change-your-state-of-mind-with-a-gratitude-session-1278856276) 大约十分钟。当我女儿出生时，我就开始了，因为她是我长久以来的梦想——这一次我觉得我的幸运是应得的，而不是一些需要道歉的幸福事故。我认为，实践感恩让我变得更加慷慨，不仅仅是对我的时间，还有我的情感能量。我已经开始庆祝别人的胜利了。以前，我常常会在脑海中想“这是一篇很棒的文章”，但懒得告诉作者，但现在我意识到，诚实地赞美别人或至少点击“喜欢”按钮不会让我*付出任何代价。(还有，“ [沉默的感激对任何人都没有多大用处](http://lifehacker.com/silent-gratitude-isn-t-much-use-to-anyone-5969631) 。))*

#### **我明白了表扬不是有限的资源**

过去，当我的父母花更多的时间和我的一个兄弟姐妹在一起时，我会很生气(做一个中间的孩子是很难的)，但是我现在意识到这种事情并没有贬低我。这并不像人们在短缺时定量分配他们的爱、欣赏或其他良好的感觉，如汽油(例如，通过说“嘿惠特森，我喜欢你的帖子”，他们在说“嘿梅勒妮，我讨厌你的帖子”)。我在试图向我女儿解释她有兄弟姐妹的概念时学到了这一点，但是——不要就此评判我——我也是很久很久以前在*爆满*的一集里学到的，在那集里，鲍勃·萨吉特解释说他的爱就像无尽的水，他的孩子都是茶杯，这种爱正在溢出。我只是花了一段时间去理解和真正接受这个教训。

以上所有这些都是为了提高自己，但它们也改变了我欣赏他人和与他人互动的方式。我还会时不时地嫉妒或羡慕吗？当然了。但随着我不断练习成为一个更好的人，我意识到我开始变绿，并能控制这些感觉，而不是让它们控制我。
