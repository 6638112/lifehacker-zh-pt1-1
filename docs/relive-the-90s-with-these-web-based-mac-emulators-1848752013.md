# 用这些基于网络的 Mac 模拟器重温 90 年代

> 原文：<https://lifehacker.com/relive-the-90s-with-these-web-based-mac-emulators-1848752013>

回到 90 年代，你需要有强大的硬件来使用操作系统，比如，哦， Mac OS 8 。如今，你可以在一个 Chrome 标签中运行所有的东西。人们很容易忘记计算机在相对较短的时间里取得了多大的进步。如果你想要一个快速提醒，你可以试着在你的浏览器中运行这两个旧版本的 macOS。

Quip(不是牙刷)工程师 Mihai parparparita 为 90 年代的两款传统 Mac 系统和 [Mac OS 8](https://macos8.app/) 开发了模拟器。System 7 最初于 1991 年发布，引入了 QuickTime、个人文件共享和虚拟内存等功能；苹果于 1997 年发布了 Mac OS 8 ，随之而来的是 spring- 加载文件夹、Finder 中的弹出窗口、可定制的系统字体、主题和控制条，也就是我们今天所熟知和喜爱的 dock 的前身。

对于任何关注 Mac 仿真历史的人来说，这肯定不是第一次有人在现代硬件上体验像 Mac OS 8 这样的软件。你可以下载模拟器已经有一段时间了——几年前，我们向你推荐了 [菲利克斯·里斯伯格的选项、](https://lifehacker.com/this-free-macos-8-emulator-lets-you-play-oregon-trail-1844547631) ，它提供了 一种有趣的方式来体验游戏*俄勒冈小径*的本意。

然而，让我们今天谈论的两个模拟器如此酷的是它们在你的浏览器中运行——不需要下载。这意味着无论你运行的是 Mac 还是 PC，M1 还是英特尔，你都可以在浏览器标签中体验系统 7 或 Mac OS 8。

虽然只是点击模拟器本身就很有趣——特别是对于我们这些还清楚记得过去使用这些操作系统的人来说——parparparita 的还包括有用的和仍然可用的老式应用。你可以查看像 Word、Photoshop 和 Adobe Reader 这样的程序，因为它们在时就已经存在了。更好的是，因为这些模拟器支持导入和导出文件，所以您实际上可以使用程序来完成实际工作。

要将任何仿真器文件移动到您的实际 Mac 上，只需将它拖到名为“外部世界”的文件夹中；that 会把文件直接发送到你电脑的下载文件夹。要从你的计算机发送东西到模拟器，把它放到模拟器窗口，在那里它会保存到文件夹“外部世界”

这不全是生意:你也可以在你的 Mac OS 8 模拟器上玩游戏——试试宝石，比如*旅鼠*、*基德皮克斯*、*波斯王子*、*模拟城市*，当然还有*俄勒冈小径*。如果有一个单一的原因来检查这些模拟器，我会说，这是为了狩猎一些水牛和哀悼你倒下的旅客。

也就是说，没有什么是完美的。有些程序可能无法正常运行，但可以通过调整系统设置来解决。其他人可能会在你每次尝试时给你错误，原因不明。B 但是嘿，这是一个在互联网上运行的完整的 90 年代操作系统！撇开 bug 不谈，那个还是挺酷的。

虽然回到更简单的计算时代是有趣而难忘的，但他的经历却是短暂的:当你关闭模拟器标签的那一刻，你所做的一切都将被永远抹去。下次你打开模拟器，即使是同一个浏览器，你也会看到一个完全重置的 System 7 或 Mac OS 8。
