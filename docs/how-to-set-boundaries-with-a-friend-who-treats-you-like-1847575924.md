# 如何和一个把你当成治疗师的朋友划清界限

> 原文：<https://lifehacker.com/how-to-set-boundaries-with-a-friend-who-treats-you-like-1847575924>

友谊的好处之一是有人可以倾诉，分享你的消息(好的和坏的)，并寻求支持。但有时，如果你觉得一个朋友不仅把你当成他们的朋友，还把你当成他们的治疗师——经常求助于你，倾听他们的问题，在某些情况下，提供建议或解决方案。



虽然看起来你做的是对的，但最终，这种安排对你们都不理想。在 [的一篇文章中，来自纽约市的治疗师和心理健康教育家 Minaa B.](https://www.wellandgood.com/my-friend-treats-me-like-therapist/) 讨论了为什么朋友不能代替治疗师，以及如何在必要时设定友谊的界限。以下是需要了解的内容。

## 你不是你朋友的治疗师

如果你的同事或熟人开始经常和你说话，并期望你会倾听、理解他们的话并提供建议，你可能会马上让他们知道这种安排对你不起作用。但是如果是你的朋友在挣扎，就很难说不了。

正如[Minaa b .](https://www.wellandgood.com/my-friend-treats-me-like-therapist/)指出的，我们生活在一个被鼓励公开谈论我们的心理健康，包括我们的挫折的时代。“虽然这很好，但同样重要的是要记住，不是每个人都有心理准备来承担和管理我们自己的个人情感斗争，”T5 她写道 。

完全有可能深深地关心你的朋友，但没有精力或情感带宽来为他们提供他们想要或需要的支持。

## 如何在你们的友谊中设定健康的界限

以下是 Minaa B .提供的三条建议，可以帮助你与像治疗师一样对待你的朋友建立必要的界限:

### 让他们知道你有极限

除非你告诉你的朋友，你在友谊中能承受的情绪劳动是有限度的——而且你已经达到了——否则他们可能不会知道你有这种感觉。“有时候，我们会等着其他人理解我们有权分享的信息，” [米纳·b .](https://www.wellandgood.com/my-friend-treats-me-like-therapist/)写道。"愿意交流你的界限是什么."

### 指引他们找到资源

尽管你不能充当你朋友的治疗师，但这并不意味着你不能给他们指出他们可能会发现有帮助的信息。如果你确实遇到了你认为对你的朋友有用的资源， [Minaa B .建议](https://www.wellandgood.com/my-friend-treats-me-like-therapist/) 使用支持性的语言来加强你在提出建议时已经建立的界限。

“比如，‘我意识到我没有能力提供帮助或建议，我真的很想支持你。她写道:“我知道一个叫 x 的资源，我认为你可以考虑向他们寻求额外的支持。

### 对你的朋友(和你自己)诚实

除非你自己是一名训练有素的治疗师(在这种情况下，你无论如何都不会把朋友当成客户)，尽管你是好意，但总有可能你的建议会把你的朋友引向错误的方向，或者把他们置于没有他们需要的支持的境地。为此，[Minaa b .](https://www.wellandgood.com/my-friend-treats-me-like-therapist/)强调，对你的朋友和你自己坦诚这些局限性是很重要的——这符合每个人的最佳利益。
