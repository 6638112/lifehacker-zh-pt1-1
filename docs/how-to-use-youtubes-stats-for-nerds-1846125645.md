# 如何使用 YouTube 的“书呆子统计”

> 原文：<https://lifehacker.com/how-to-use-youtubes-stats-for-nerds-1846125645>

尽管名字如此，YouTube 的“书呆子统计”功能并不只是为书呆子准备的。默认情况下，该选项在所有 YouTube 视频中都是关闭的，但启用后，视频上方会出现一个小窗口，显示当前视频质量、缓冲信息等信息。



以下是对每个字段含义的简要解释:

**视频 ID/sCPN:**YouTube 数据库中视频的 ID 和您的特定播放实例的代码。

**视口/帧:**视频播放器的分辨率(不是视频的分辨率)和你所在的当前帧。

**当前/最佳分辨率:**视频的当前分辨率与其原始分辨率。

**音量/正常化:**您正在观看的视频的当前音量以及音频的“正常化”程度正常化将声音中较大和较小的部分更紧密地匹配，以保护听者的耳朵和扬声器。括号显示了视频与 YouTube 基线相比的音量。

**编解码器:**媒体的视频和音频压缩编解码器。

**颜色**:显示您正在观看的内容的当前颜色范围。这些信息可以帮助您确认您的 Xbox Series X|S 或 Xbox One 主机可以 [最终在 YouTube](https://www.engadget.com/you-tube-finally-delivers-hdr-on-the-xbox-one-114053690.html) 上播放 HDR 视频。

**连接速度:**您设备的网络连接。

**网络活动:**从 YouTube 的服务器到您的设备的数据传输。视频的分辨率越高，网络活动——也就是带宽——就越高。

**缓冲状况:【YouTube 能够缓冲多少视频，以避免因连接速度慢而导致播放中断和丢失。**

神秘文字:来自人类无法理解的神秘信息。或者至少这是我最好的猜测。没有人真正知道这个数据意味着什么，尽管很多人已经尝试过用来猜测。

有些信息很简洁，但是对于普通用户来说，大部分看起来像是技术上的胡言乱语。然而，这并不意味着它没有帮助。如果您遇到回放问题，并且不能确定是什么原因造成的，网络信息会很有用。

然而，Stats for Nerds 的主要用途是帮助 YouTube 的工程师排除 bug。如果你曾经遇到过 YouTube 的问题，比如突然的应用程序崩溃，缓慢的视频缓冲，或其他问题，抓拍一张“书呆子统计”窗口的截图，然后 [将它发送给 YouTube 支持人员，并附上一份错误报告](https://support.google.com/youtube/answer/7519898?co=GENIE.Platform%3DAndroid&hl=en) 。

### 如何为书呆子打开 YouTube 的统计

对书呆子的统计是有用的，但是根据你使用的设备的不同，查找和打开它会略有不同。

**桌面(Windows、Linux、Mac)**

1.  在您选择的浏览器中打开 YouTube.com。
2.  开始看视频。
3.  右键单击视频一次，打开视频菜单。
4.  选择**“书呆子统计”**切换覆盖。

**安卓**

1.  在 YouTube 应用中，进入**设置>常规>启用书呆子统计**(也在**设置>高级>启用书呆子统计**下)。
2.  接下来，播放一段视频。
3.  点击视频播放器中的三点**“更多”**按钮。
4.  点击**“统计”**按钮。

**iOS/iPad**

1.  打开 YouTube 应用程序
2.  转到设置
3.  为书呆子启用**“统计”**
4.  播放视频。
5.  点击视频播放器中的三点**“更多”**按钮。
6.  点击**“统计”**图标。

**智能电视、游戏机和流媒体设备**

一般来说，在这些设备上切换书呆子的属性应该是一样的。

1.  打开设备上的应用程序。
2.  开始视频。
3.  调出视频播放控件(通常通过按下遥控器/控制器的默认“选择”按钮)。
4.  切换并选择三点**“更多”**按钮。
5.  滚动并选择**“书呆子统计”**
