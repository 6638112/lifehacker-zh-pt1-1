# 负鼠和负鼠的区别

> 原文：<https://lifehacker.com/the-difference-between-a-possum-and-an-opossum-1848583776>

负鼠看起来像衣衫褴褛的动物——或者至少我过去是这么认为的。你会看到它们在夜晚穿过马路，锋利的毛束拖着长长的裸尾。但是有一天，我参观了一个博物馆，那里有几十种不同动物的毛皮在现场展示。有些是柔软的，有些是粗糙的毛发，然后我抚摸负鼠的皮毛，几乎是开玩笑。就像，这种感觉会有多糟糕。那是*天上的*。直到今天，负鼠毛皮仍然是我接触过的最柔软的毛皮。

我说这些并不是为了赞美被误解的负鼠的美德，尽管它们有很多。我这么说是为了指出，作为一个对这种动物非常熟悉的北美居民，我称它为负鼠。其他人也一样。你不会对你的邻居说“啊，我的狗昨天和一条死狗一起滚了进来，那是 T2；“你说，“我的狗发现了一只死负鼠。”

既然我们已经确定了这一点，我们必须谈谈另一种动物:澳大利亚的毛尾负鼠，或普通的刷尾负鼠。两种负鼠都是有袋动物，都生活在城市里和城市附近，通常都被称为负鼠。

## 那么什么是负鼠呢？

在正式场合，这种北美动物被称为“弗吉尼亚负鼠”,在其他地方几乎都被称为“负鼠”。

然后，有些人会变得迂腐，说北美有负鼠，澳大利亚有负鼠，但这只有在你谈论花哨的学术语言时才是真的(在这种情况下，你还不如用学名)。负鼠是*一个*的正确名称，指的是那种看起来邋遢但实际上软绵绵的老鼠尾巴的生物，它们在晚上穿越高速公路，但它不是*唯一的*正确名称。正如 [字典所明确的](https://www.merriam-webster.com/words-at-play/possum-vs-opossum-difference-pronunciation) ，这两个名字对两种动物都是正确的。
