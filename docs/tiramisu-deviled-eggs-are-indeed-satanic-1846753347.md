# 提拉米苏炒鸡蛋确实很邪恶

> 原文:[https://life hacker . com/tiramisu-deviled-eggs-is-dec-satanic-1846753347](https://lifehacker.com/tiramisu-deviled-eggs-are-indeed-satanic-1846753347)

我们爱我们的读者，不是吗，伙计们？我知道我有。在很大程度上，阅读和评论Skillet(以及一般的Lifehacker)的人构成了互联网上相当令人愉快的一部分，我总是对你们中一些人提出的有创意和独特的食物诀窍和食谱印象深刻。事实上，读者提出的今天视频的主题是对经典的deviled egg的一个创意和独特的呈现。

如果你比较了解我，你会知道我是一个魔鬼蛋的忠实粉丝，有些人甚至会称我为“纯粹主义者”。蛋黄酱、芥末(这是“魔鬼”的代理)，也许一点点调味品和熏辣椒粉就是你做一个好鸡蛋所需要的，尽管我有时会被一个花哨的lil' egg迷住，它顶上有火腿片或少量的鲑鱼子。但是这些蘸酱的鸡蛋上面没有其他更小的鸡蛋——哦，不。这些鸡蛋用可可粉和碎手指点缀。我不知道我们可以称他们为“恶魔”，但他们肯定是撒旦。

我要说的是，我对这个鸡蛋的某一方面的喜爱远远超过了我的预期。馅料完全令人愉快——甚至美味，特别是当用作女士手指的蘸酱时——但我遇到了一些白色的问题。也许这是我自己的问题，但我无法停止思考这道小菜是多么的棒，我会多么喜欢颠倒食材的顺序，把馅料堆在一层女士手指上，然后用蛋清做蛋白酥皮。这种馅料很好，但它不是在家里煮白的。
