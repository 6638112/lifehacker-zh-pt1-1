# 到目前为止，我们2021年的10个最佳职业黑客

> 原文:[https://life hacker . com/our-best-career-advice-of-2021-至今-1847235546](https://lifehacker.com/our-best-career-advice-of-2021-so-far-1847235546)

打造一个职业就是接受不断学习的生活，这意味着总是敞开胸怀吸收那些更有洞察力的人的智慧。尽管2021年才过了一半，但今年我们已经遇到并分享了太多可能改变人生的职业建议——在这一年里，持续的疫情让许多人重新审视他们对工作与生活平衡的看法，以及他们如何想象自己的职业生涯——我们在中点总结了其中的精华。

离开一份每周占据你40多个小时的工作后，每个人都需要充电。但是，当你拒绝一份你不喜欢的工作时，又会怎样呢？ 如果你要休假(并让它变得有价值)，你必须以正确的方式去做，考虑到你可能会遇到的任何问题。相应地做好预算，花些时间追求新的学习机会，如果你的假期比预期的要长，不要感到惊讶。( [此处阅读全文。](https://lifehacker.com/how-to-recharge-after-quitting-a-job-you-hate-without-1847212354) )

这一条适用于职业生涯，但也可能适用于抚养孩子，或执教一支运动队，或任何其他引起共鸣的背景:如果你希望人们茁壮成长，你就必须给他们茁壮成长所必需的工具，这 [涉及在他们做工作时唤起一种与生俱来的归属感、自主感和能力](https://lifehacker.com/use-self-determination-theory-to-motivate-others-1847173631)——甚至是在家里完成基本任务。这是自我决定理论的主旨，它可以成为激励你周围人的变革方式。( [此处阅读全文。](https://lifehacker.com/use-self-determination-theory-to-motivate-others-1847173631) )

爬上 [职业等级的人很少会有自知之明认为他们的成功可能是不值得的](https://lifehacker.com/how-to-ease-your-guilt-about-failing-upwards-1847150876) ，但是我们这些有一点自知之明的人可能熟悉这种内心斗争。如果你觉得自己在某种程度上“向上失败了”，你可以通过帮助周围的人从你新获得的权力中受益来减轻随之而来的负罪感——尤其是那些没有得到和你一样机会的年轻员工。( [此处阅读全文。](https://lifehacker.com/how-to-ease-your-guilt-about-failing-upwards-1847150876) )

在你职业生涯的某个阶段，这种情况肯定会发生:尽管你很努力，但你发现自己渴望同事取得的成功或认可，并因此越来越嫉妒他们。然而，你不必让这种感觉恶化——你甚至可以利用这种感觉为自己谋利 ，一旦你接受了这个事实，你很有可能真的*钦佩这个人胜过其他任何东西。( [此处阅读全文。](https://lifehacker.com/how-to-use-jealously-to-your-advantage-at-work-1847127276) )* 

去他妈的细节，更不用说当你真正应该在海滩上或以其他方式放松，远离工作压力的时候，不断地检查你的收件箱的压力。告诉试图联系你的人，在你回来上班之前，你绝对不会查看电子邮件。即使你是，他们可能仍然需要等待对他们信息的回应。( [此处阅读全文。](https://lifehacker.com/your-out-of-office-message-should-be-brutally-honest-1847120647) )

时不时地，你需要顶撞老板的决定和指示，即使你不是在为一个微观管理者的噩梦工作。但是你想以一种不过分对抗性的方式来做，这意味着 [根据你老板的沟通风格](https://lifehacker.com/how-to-argue-with-your-manager-without-risking-your-job-1847104543) 来调整对话。如果你做到了这一点，你可能会得到回报。( [此处阅读全文。](https://lifehacker.com/how-to-argue-with-your-manager-without-risking-your-job-1847104543) )

有一个职业专家和励志演说家的行业在兜售这样一个概念，即你生活中的一切都是通向你梦想工作的道路，只要你坚持寻找。但实际上在工作世界中找到这样的涅槃状态是短暂的——幸运的是， [你不需要接受完美工作的想法来为你*所做的*工作感到快乐。专注于你在职业生涯中不断经历的职业和个人进步——并品味这些胜利——将帮助你避免陷入这种误导的思维模式。(](https://lifehacker.com/your-dream-job-is-a-farce-and-how-to-be-happy-at-work-1847072368) [此处阅读全文。](https://lifehacker.com/your-dream-job-is-a-farce-and-how-to-be-happy-at-work-1847072368) )

企业团结往往听起来很空洞，对骄傲等事业的支持往往只不过是精心策划的公关 [如果你想知道你的公司及其领导人*对社会公正*](https://lifehacker.com/how-to-tell-if-your-companys-pride-campaign-is-bullshit-1847026971) 的真实感受，或者如果他们真的关心的话，看看他们可能会向哪些事业捐款，或者他们如何对待各种LGBTQ或BIPOC员工。(点击此处阅读全文。)

工作面试不是审问——它是一次对话，让你——被面试者——也可以问尽可能多的问题。为此，你应该在下次 面试时问一些有针对性的问题:关于工作日的流程，工作要求，公司文化，甚至你自己面试后获得工作的机会。如果回答这些问题中的任何一个让你的面试官不舒服，那应该会告诉你一些关于你申请的地方的事情。( [此处阅读全文。](https://lifehacker.com/ask-these-questions-at-your-next-job-interview-1846417108) )

如果你只能从这份清单中拿走一条职业建议，那就把它变成这条吧。因为让你的工作日变得更好、更可忍受的最可靠的方式——以及给自己安全感和团结感，这将在你追求职业目标时鼓舞你的勇气——是为一家你知道你的同事会支持你的公司工作。( [此处阅读全文。](https://lifehacker.com/how-to-unionize-your-workplace-1845991328) )
