# 关于Instagram的新“监督”功能，父母需要知道什么

> 原文:[https://life hacker . com/what-parents-need-to-know-on-insta gram-s-new-superv-1848683921](https://lifehacker.com/what-parents-need-to-know-about-instagram-s-new-superv-1848683921)

我们都知道Instagram对任何人来说都不是很好，但对青少年来说尤其不好。脸书自己的数据(去年秋天公布，当时Meta还不存在)显示了Instagram对青少年用户心理健康的负面影响，尽管有人说更广泛的研究 [并不支持这样明确的结论](https://www.npr.org/2021/10/06/1043138622/facebook-instagram-teens-mental-health) 。

不管怎样，如果你是一个青少年的父母或看护者，你可能想对他们的Instagram使用情况有更多的了解。Meta最近推出了一套名为监管的父母控制系统， [，提供有限的监控功能](https://about.fb.com/news/2022/03/parental-supervision-tools-instagram-vr/) ，目前仅在美国可用。

虽然这项新功能允许父母和监护人设置应用程序使用的时间限制，并接收青少年关注的人(以及谁在关注他们)的更新，但它必须由青少年的帐户启动。梅塔表示，未来，看护者将能够设置监管，并设定他们的青少年可以访问Instagram的时间。

下面是 [如何利用到目前为止监理提供的](https://help.instagram.com/309877544512275) 。

## 如何在Instagram上设置监督

目前，13-17岁的青少年必须通过向他们的父母或监护人发送邀请来发起监督请求，他们也必须有一个Instagram帐户(或 [创建一个](https://help.instagram.com/182492381886913) )。

让你的孩子打开他们的Instagram应用程序，进入他们的个人资料页面，点击右上角的三条竖线。轻触**设置>监督>下一个**。选择**设置监督**，并选择您希望将邀请链接发送给家长或监护人的方式。

若要接受邀请，请打开Instagram并登录到您要用来监督的帐户。然后在使用的任何消息应用程序中打开邀请链接，这将打开Instagram家庭中心。点击**下一个>接受邀请**。

## 如何调整Instagram上的监督功能

要更改您孩子的Instagram时间限制，请前往**设置>监督>您监督的账户**并选择您孩子的账户。点击设置限额，选择限额，然后点击保存。你也可以在这里看到他们最近7天在Instagram上的时间。

您还可以查看您孩子的关注者和关注列表。一旦您选择了您孩子的帐户，点击**关注者并跟随>查看更多**。
