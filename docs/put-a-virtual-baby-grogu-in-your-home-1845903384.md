# 在你的家里放一个虚拟的婴儿格罗古

> 原文：<https://lifehacker.com/put-a-virtual-baby-grogu-in-your-home-1845903384>

如果你还没有带尤达宝宝/格罗古/孩子来你家过节——无论是以一个巨大的填充动物玩具、行走玩具的形式，还是仅仅是沉迷于重温*的曼达洛人* — 的形式，还有一个...选项(借用尤达在*绝地归来*中的戏剧性措辞):使用你的安卓手机将一个虚拟的 Grogu 放入你的物理空间。



不，你的虚拟朋友不会像拥有 [一个小小的、有血有肉的外星婴儿朋友和你的](https://www.youtube.com/watch?v=opB_qb7pv9A) 宇宙飞船一起在周围做爱那样提供同样的爱的温暖，但这是值得的。首先，确保你的 Android 已经更新到最新版本的操作系统——你所有的应用程序也更新了，因为为什么不呢——然后启动 Chrome 或谷歌应用程序。快速搜索“尤达宝宝”(或“格罗古”)，你应该会在谷歌的小事实框中看到这个选项:

点击**“3D 视图”**，你会得到这个可爱的孩子的大图。一定要把音量调大，因为格罗古甚至会发出他标志性的咯咯声。(算是吧。他们听起来和你在《T4》中听到的《曼达洛人》有一点点不同。)

在那里，点击**“在你的空间中查看”**将银河系最想要的宝贝带回家。你可能会被要求确认一些权限，然后你需要将你的手机指向地面，以便增强现实工作。但是当你这么做的时候，会有咕噜咕噜的声音。

就是这样！你可以试着捏一捏让他稍微变大变小，但我真的没见过这样做太多。实际上，你的目标应该是把他放在尽可能多的愚蠢场景中，并拍很多照片。另外，我不确定这是否是严格的标准，但是我没有意识到这个孩子只有蓝色雪人麦克风那么高。如果是这样的话，我认为格罗古最神奇的地方在于他设法避免被踩在上。那个昏昏欲睡的小原力拥有伟大的记录。
