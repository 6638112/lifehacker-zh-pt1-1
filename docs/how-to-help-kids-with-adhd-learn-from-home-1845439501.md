# how to help kids with adhd learn from home 1845439501

> 原文：<https://lifehacker.com/how-to-help-kids-with-adhd-learn-from-home-1845439501>

在这段时间里，我们都有自己独特的挑战；没有人能完全摆脱疫情带来的压力。但是当今年春天第一次关闭虚拟学习的时候，患有 ADHD 的孩子的父母们，特别是那些患有 ADHD 的孩子的父母们，正在与虚拟学习作斗争，甚至可能完全放弃了虚拟学习。不幸的是，这种“哦，谁还在乎呢”的态度在四月中旬左右打击了许多家长，这不是我们可以在整个 2020-21 学年都忍受的心态。孩子们*是否*需要学习，疫情与否，焦虑与否，特殊需求与否。

然而， [父母不是教育者](https://offspring.lifehacker.com/remember-you-are-not-your-childs-teacher-1844870755) 。看，这就是为什么老师是伟大的:他们去学校学习如何做这份工作，不仅仅是为了那些勤奋、专注、总是听讲的孩子*和*不干涉自己，也是为了其他 99%的孩子。然后，在他们学会如何做之后，他们会和一个以前做过的老师一起练习！在你意识到之前，幸运的事情发生了——他们自己变得有经验了。在这一过程中，他们发现并制定策略来接触各种各样的学生，这些学生有着无穷无尽的个性和气质特征的组合，以及独特的技能和挑战。



比如，患有注意力缺陷多动症(ADHD)的孩子。如果你的孩子患有多动症，很可能你已经学会了一些很好的育儿策略。然而，作为他们的全职教育项目经理，你可能并没有做好准备。如果你发现自己处于这种情况，你可以尝试一些策略来帮助缓解你和你的孩子(可能还有整个家庭)的压力和挫败感。

### 首先，承认斗争

挣扎中的孩子知道他们在挣扎。承认它不会造成任何伤害。事实上，你应该和你的孩子开诚布公地讨论他们面临的挑战。

“不谈论它是没有帮助的，”特里尼亚·考夫曼说，她曾是一名特殊教育教师，也是[understand](https://www.understood.org)的编辑研究高级经理，该公司为那些学习和思考方式不同的人提供资源。“这次谈话没有任何指责或评判的意思。每个人都有他们正在努力解决的事情，所以你可以说，“我们作为一个家庭或与我们的老师一起要做的是找出我们如何解决这些挑战，并让你更容易。”"

考夫曼说，通过这样做，以实事求是和支持的方式，你让你的孩子知道你站在他们一边，你知道他们正在尽最大努力。这是一个很好的起点。



### 深入了解他们的具体需求

每个孩子，无论是否患有多动症，都有自己特定的挑战，无论是变得有条理、管理时间还是保持专注。他们可能会被大量的工作压垮，他们可能会分心，或者他们可能只是对某个特定的主题不感兴趣。如果你不确定他们的挑战到底是什么——你只知道有*个*挑战——考夫曼建议从一个“”观察追踪器开始观察和跟踪他们的行为和困难可以帮助你开始注意到与他们的斗争有关的迹象或模式。

如果你的孩子在学校有一个 IEP()个性化教育项目，那也是寻找他们的老师在课堂上帮助他们成功的具体策略的好地方。但是，即使他们没有 IEP，你也可以和他们的老师谈谈，听听他们认为有帮助的策略，也许你可以在家里实施。

### 限制分心

现在限制分心显然说起来容易做起来难。在一个完美的疫情世界里，你只有一个(较大的)孩子可以在他们自己安静的空间里专注于他们的功课。事实上，我们中的许多人正在与伴侣和多个孩子共享工作空间，他们同时在家工作和学习。(这就是为什么我丈夫每周一下午在我们的卧室召开团队视频会议，而不是在他喜欢的办公室——餐桌上。缩放太多，空间太小。)



但是考夫曼说，父母可以做几件小事来减少，如果不能消除的话，分心。首先， [在最有意义的时候使用耳机](https://offspring.lifehacker.com/how-to-prevent-hearing-loss-in-kids-from-headphone-use-1844470288) ，特别是在一个空间同时有多个会议的时候。第二，让他们远离周围的人和/或面向墙壁。没有人想把宝宝放在角落里，但是面对一面普通的墙比面对家庭活动的熙熙攘攘更容易让人分心。

### 让他们按计划行事，并听取他们的意见

大多数孩子在有规律的时间表或常规可循时表现最好——这对于患有多动症的孩子来说尤其重要，他们现在正在家里学习。

“但我们不能只是为他们制定时间表，并要求他们这样做，”考夫曼说。"我们必须有一定的灵活性，这样才能得到他们的认可。"



坐下来，列出优先事项，一起制定时间表，尽可能给他们选择的机会，比如决定什么时候做阅读作业，什么时候做数学。

当你制定时间表时，记得要留出足够的休息时间。如果可能的话，以一些体育活动开始一天——这有助于缓解冲动和过度活跃。然后根据他们的年龄或他们个人需要多少体力活动，根据需要在一天中进行更多的休息。

### 寻找小胜利

我们——以及我们的孩子——正在经历一个充满挑战和巨大压力的时代。每天都不会取得巨大的成功；恰恰相反。但是如果你足够仔细地观察，一整天都会有小小的胜利。一项任务的好成绩——或者仅仅是完成了一项任务。比预期多花了五分钟在数学上；一个 n 的下午，所有必要的技术都在工作；一个有趣的新游戏，你们在休息时一起做的。留心那些时刻。

“我们的大脑倾向于关注消极的东西，”考夫曼说。“所以我们实际上必须强迫自己思考，‘我做得好的事情是什么？我的孩子做得好的事情有哪些？然后让他们知道自己在哪些方面做得很好，这样他们就可以在此基础上再接再厉。
