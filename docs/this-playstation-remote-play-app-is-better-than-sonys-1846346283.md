# 这个PlayStation远程播放应用比索尼的好

> 原文:[https://life hacker . com/this-PlayStation-remote-play-app-is-better-than-sonys-1846346283](https://lifehacker.com/this-playstation-remote-play-app-is-better-than-sonys-1846346283)

我最喜欢的现代游戏的创新之一是将你正在玩的游戏流式传输到你的智能手机的能力，无论是来自云的、你的家用游戏机还是PC。流媒体传输f将你的手机变成一个类似任天堂Switch的便携设备，可以在它能连接的任何系统上玩游戏。

Watch

我们已经介绍了如何使用来自 [、索尼](https://lifehacker.com/how-to-stream-your-ps4-to-any-android-phone-1838909366) 、 [、微软](https://lifehacker.com/how-to-stream-xbox-games-on-your-iphone-or-ipad-1845428424) 和 [Steam](https://lifehacker.com/use-steam-to-stream-your-desktop-instead-of-your-games-1818722875) 的官方应用来实现这一点，但是有一个名为 **PSPlay** 的非官方第三方Android应用是PlayStation粉丝应该考虑的，因为它比索尼的官方PS Remote Play应用更好。

PSPlay的工作方式就像索尼的PS Remote Play应用程序一样，但设置更快，更可靠，并且包括PS Remote Play所缺少的重要功能。

有一个警告: [PSPlay在Google Play](https://play.google.com/store/apps/details?id=psplay.grill.com&hl=en_US&gl=US) 上售价4美元，而索尼的PS Remote Play应用是免费的。有些人可能不愿花钱购买索尼免费提供的东西，但PSPlay的功能值得花钱购买。

在发现PS Remote Play不支持第三方控制器后，我买了PSPlay，这主要是我在Android上玩游戏时使用的。你可以使用DualShock 4或DualSense这两个应用程序，但PSPlay也支持第三方选项，如Razer Kishi。

(这两款应用都提供触摸控制，但是，拜托，没人想用劣质的屏幕按钮玩。)

不要误解我， [我*喜欢*dual sense](https://lifehacker.com/how-to-use-a-playstation-5-controller-with-steam-1845742247)，但我发现像Razer Kishi这样的控制器外壳更适合手持游戏，PSPlay是将它们用于PS4和ps5游戏的唯一方式。

PSPlay还可以比官方的PS Remote Play应用程序更快地连接到PS4或PS5。我的PS5通过PS Remote Play识别我的手机最多需要三分钟，但通常通过PSPlay只需要几秒钟。

PSPlay还允许你通过移动连接进行游戏，而远程游戏只能通过wifi进行。这意味着PSPlay可以在旅途中将游戏从家中传输到手机上，而PS Remote Play只能在手机和游戏机处于同一wifi网络时工作。

公平地说，我没有注意到PSPlay和PS Remote Play在wifi上的延迟或流媒体质量有任何差异——我大部分时间都是这么玩的。如果你有一个强大的wifi连接，这两个应用程序都很好，尽管如果连接不是很强，你可能会注意到短暂的输入延迟。不过，在手机上玩PSPlay时，我的输入延迟比T2更糟糕。除非你是少数几个拥有5G覆盖的人之一，否则最好还是坚持使用wifi，但通过移动设备玩游戏的能力仍然让它比PS远程游戏有优势。
