# 所以你忘了预定一些通心粉水

> 原文：<https://lifehacker.com/so-you-forgot-to-reserve-some-pasta-water-1849327454>

尽管这个词可能意味着什么，奶油并不是创造“奶油”面食的关键。这种区别属于面食水，这种淀粉药剂能够使酱汁变稠，并帮助它们粘在面条上，使面条具有韧性。在沥干意大利面之前保留一个杯子是重要的一步，但是意外发生了，有时在你意识到你做了什么之前所有的水都被倒进了下水道。



如果发生这种情况，不要惊慌。有两种方法可以制作面食- 水的替代品。

## 如果你不小心把自己的意面水倒进了下水道，如何制作应急意面水

淀粉是使面食水如此有价值的东西。通常淀粉是在煮面的时候从面条进入水中的，但是你也可以用其他方法来制造含淀粉的水，这些方法都不需要额外购买或烹饪意大利面条。

通过向一锅沸水中添加粗面粉 (最常用于制作意大利面的面粉)，你用淀粉对其进行增压，有效地模仿了“真正的”意大利面水的效果。

事实上，你甚至不用把你的意大利面水倒掉就可以使用这个技巧。我经常用面条往我的意大利面锅里扔一些粗面粉，让水变得格外甘美和滑腻，尤其是在我为我乳糖不耐受的男朋友做不含乳制品的菜时。在这种情况下，我会用 1/2 杯面粉兑 3 夸脱水，但如果需要的话，你可以将比例缩小到每杯水兑 2 茶匙面粉，制作一杯紧急意大利面水。

如果你没有粗面粉，你可以用玉米淀粉代替。 [美国的测试厨房](https://www.americastestkitchen.com/articles/5721-accidentally-drained-pasta-water-replacement) 使用 1/4 茶匙玉米淀粉和 1 杯水的比例。(与面粉不同，玉米淀粉是纯淀粉，所以你不需要那么多。)

不管你用的是面粉还是玉米淀粉，你都要把面粉搅拌到水里，然后用微波炉加热一两分钟，直到它变得又热又香。像平常一样溅到你的酱里，或者用它来恢复僵硬的剩余意大利面 昔日的辉煌。
