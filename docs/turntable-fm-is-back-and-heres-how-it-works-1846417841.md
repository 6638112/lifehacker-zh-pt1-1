# Turntable.fm又回来了，下面是它的工作原理

> 原文:[https://life hacker . com/turntable-FM-is-back-and-heres-how-it-works-1846417841](https://lifehacker.com/turntable-fm-is-back-and-heres-how-it-works-1846417841)

受人喜爱的社交媒体音乐网站Turntable.fm在2013年被关闭近八年后，出人意料地卷土重来。其实有*两个*新版本:最初的 [Turntable.fm](https://turntable.fm/) ，最近由创始人比利·蔡森复活，以及专注于移动的[Turntable.org](https://turntable.org/)，由几位Turntable.fm校友创建。

Watch

虽然这两种服务都允许用户创建个性化的头像，主持或加入在线收听聚会，但它们在商业模式、功能和可用性方面有所不同。

### Turntable.fm

Turntable.fm保留了网站原有的体验。它是免费使用的，你可以从YouTube上的音乐创建播放列表。(一旦整合开始，SoundCloud也是如此。)但有一个关键问题。 [现在](https://turntable.fm/) 已经有了Turntable.fm，但是只对知道密码的人开放。

您可以通过提交您的电子邮件和您最喜爱的歌曲来请求密码。如果歌曲是“好的”，你会在电子邮件中获得密码。我们不知道Turntable.fm认为什么是“好”的歌曲——否则，我们会提供建议来帮助你加入。

我的申请还没有回音。(希望分发代码的人喜欢睡眠的“傻瓜”T3)

### Turntable.org

与免费的转盘调频不同，Turntable.fm需要订阅才能使用。虽然您还不能注册该服务，但也有特别的奖励套餐可供购买，当Turntable.org上线时，您将获得独特的项目和订阅折扣。这笔钱将用于开发成本。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-7C_D7Tm9gw4&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-7C_D7Tm9gw4&start=0) 

Turntable.org的另一个主要区别是，用户可以上传和播放自己的MP3歌曲——他们不会局限于其他网站上的音乐。这将有助于独立艺术家举办他们自己音乐的聆听聚会。

### 加入测试版

Turntable.org测试版将于4月推出。感兴趣的用户可以在官网 报名加入 [。对于想在Turntable.org举办数字秀的艺术家来说，还有一个特殊的注册表格。](https://turntable.org/)
