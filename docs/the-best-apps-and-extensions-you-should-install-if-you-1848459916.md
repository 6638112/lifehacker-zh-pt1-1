# 如果你使用Reddit，你应该安装的最好的应用和扩展

Reddit是少数几个允许你为这项服务开发成熟应用的社交媒体网站之一。这就是为什么你会在App Store和谷歌Play商店上找到很多高质量的Reddit应用。在很多方面，这些第三方应用都比官方的Reddit体验更好。

有很多事情Reddit做得不好。这个视频播放器一团糟，因为它经常需要很长时间才能加载；它隐藏了网址预览，所以你永远不知道你在点击什么(也许当你第5000次被点击时很有趣，但如果你在工作中点击一个NSFW链接，你就不会咯咯笑了)；Reddit很喜欢gif，但你不能在适当的时候浏览gif来暂停；不要让我开始谈论网站上的垃圾广告和无用的搜索框。

最近，Reddit的重点更多的是增加脸书风格的功能，包括无限滚动，向你显示还有谁在线，以及有多少人在实时评论。如果你对这一切感到不舒服，你应该考虑为Reddit使用最好的第三方应用和浏览器扩展。我们已经缩小了范围，以便您可以选择适合您的方式。

## 阿波罗(iPhone)

如果你有一部iPhone，你不会找到比 [阿波罗](https://apps.apple.com/app/id979274575) 更好的Reddit应用了。它把所有的基本功能都做对了——Reddit加载速度很快，发布和评论都非常快，你不会看到任何广告。Apollo擅长提供生活品质的功能，极大地改善你的浏览体验。

其中之一是突出显示新的Reddit帐户的能力，以帮助您快速发现抢劫、钓鱼或低质量评论的企图。它还可以让你禁用无限滚动，这有助于减少你的社交媒体成瘾。Apollo还可以自动折叠对顶级评论的所有回复，隐藏AutoModerator评论，并允许您清理gif。

Apollo经常更新您甚至不知道自己需要的新功能，这就是为什么您应该尝试一下。该应用程序有两个高级层，可以释放其最佳功能——阿波罗专业版(4.99美元)和阿波罗超级版(每月0.99美元)。Apollo Ultra是为那些需要通知的人准备的，但Apollo Pro几乎可以提供所有其他功能。

## 独角鲸(iPhone)

如果你喜欢旧的Reddit设计，并希望它能在你的iPhone上工作， [Narwhal](https://apps.apple.com/app/narwhal-for-reddit/id845422455) 是完美的选择。独角鲸是非常可配置的。它让您决定是否要自动将帖子标记为已读，也可以从该设置中排除NSFW帖子。这个应用程序可以让你保存每个子编辑级别的排序参数，并让你决定是否要折叠粘贴的评论、自动排序器评论和子评论。你对这个应用探索得越多，你就会越喜欢它。

Narwhal是免费的，只在屏幕下方显示一个广告，你可以花3.99美元删除它。唯一的缺点是Narwhal的更新不是很频繁，尽管它很稳定，但它不像一些竞争对手那样经常获得新功能。

## Reddit增强套件(桌面)

在桌面上，如果你不尝试一下Reddit增强套件(RES)，你会给自己帮倒忙。这是Reddit的一个浏览器插件，可以在所有主流浏览器上免费下载。RES为你提供了功能，比如完全可定制的键盘快捷键，用于所有你最喜欢的Reddit操作，能够改变Reddit顶部栏中的子编辑，以及为线程中的重要用户提供可爱的小亮点——比如op、管理员、朋友或版主。

RES消除了Reddit的大量烦恼。你可以默认隐藏子评论，登录Reddit时隐藏用户名，甚至可以点击垃圾邮件按钮快速举报评论。

## Boost(安卓)

[Boost](https://play.google.com/store/apps/details?id=com.rubenmayayo.reddit) 是安卓最好的Reddit应用之一。它的重点是设计。如果你想要一个遵循材料设计原则的应用程序，或者如果你正在为Reddit寻找一个光滑的Android应用程序，Boost是一个很好的选择。它有很棒的后期过滤工具，所以你可以选择显示或隐藏带有照片或视频的帖子。

您还可以为每个子编辑设置不同的查看模式，Boost提供了一些很好的模式。有一个图片库模式，可以很容易地在照片丰富的子编辑中浏览不同的帖子，还有一个幻灯片显示视图，可以让你向任一方向滑动，以移动到下一个或上一个帖子。

## 乔伊(安卓)

[乔伊](https://play.google.com/store/apps/details?id=o.o.joey) 有很多好功能，但最好的一个是从Reddit下载视频的功能——这是对那些通勤时网络质量不佳的人来说是一个很好的功能。它还支持Reddit Live，这是一个让人们在Reddit上直播视频的功能。除了这些独特的功能之外，Joey还允许你以杂志风格的格式查看AMA的帖子，它还有一个很好的可定制的评论折叠功能。

## 荣誉奖

有太多好的移动Reddit应用程序，每个应用程序都有独特之处，值得一试。如果我们的推荐不适合你，试试下面的:

*   [Infinity](https://play.google.com/store/apps/details?id=ml.docilealligator.infinityforreddit) (安卓):这个应用是免费的，没有广告，为Reddit提供了一个漂亮的设计。它最大的特点是一种懒惰模式，即自动在Reddit中滚动。如果你厌倦了不停地滚动，这个应用程序可以帮你完成这项工作。
*   [RIF很有趣](https://play.google.com/store/apps/details?id=com.andrewshu.android.reddit) (Android): RIF是Android上最受欢迎的Reddit应用程序之一，在优化Reddit的旧移动设计方面做得很好。在更名之前，人们曾称Reddit很有趣，但现在… 很难理解。
*   [Relay](https://play.google.com/store/apps/details?id=free.reddit.news)(Android):Relay有一个很好的设计，让你在一个链接和它对应的Reddit讨论之间无缝切换。如果你在Reddit上点击了很多链接，Relay可能是一个很适合你的。
*   Slide ( [iPhone](https://apps.apple.com/app/id1260626828) ， [Android](https://play.google.com/store/apps/details?id=me.ccrama.redditslide) ): Slide在Android和iPhone上都可以使用，它为这两个平台带来了素材设计，i t让你可以下载你的feed中的所有东西供离线观看。
