# 如何使用Apple Notes与他人进行秘密聊天

> 原文:[https://life hacker . com/how-to-use-apple-notes-to-have-secret-chats-with-others-1849112538](https://lifehacker.com/how-to-use-apple-notes-to-have-secret-chats-with-others-1849112538)

当你想对别人隐藏对话时，苹果的Notes应用程序可能不是第一个想到的选择。但通过使用其协作功能邀请其他人加入“对话”——然后在你完成后删除消息——你可以抹去所有聊天的证据。当然，这远不如使用加密的消息应用程序来隐藏消息安全(而且很容易对你的共享笔记进行截图或将你的聊天记录复制到另一个应用程序)，但notes应用程序是紧急情况下秘密消息的快速简单的选择。

## 如何将Notes用作秘密消息应用程序

若要开始，请打开便笺并创建一个新便笺。在笔记中键入一些内容，然后点击页面右上角的**三点**图标，再点击**分享笔记**。你现在可以点击**分享选项**并禁用**任何人都可以添加好友**。在**权限**下，确保你已经选择了**可以进行修改**。

返回上一页，选择您希望与其他人共享笔记的方式。只要他们有Apple ID，就可以访问你的笔记。我们用iMessage分享了笔记。一旦其他人加入，你就可以开始输入信息，他们将能够实时看到。

为了便于区分您的消息和联系人的消息，请点击右上角的**三点**图标，然后选择**管理共享笔记**。选择**突出显示所有更改**并返回到您的笔记。当他们回复您的邮件时，笔记会用不同的颜色突出显示该邮件。

当您完成对话时，请先删除您键入的所有内容。然后轻点右上角的**三点**图标，进入**股票期权**，将文档权限改为**仅查看**。

返回上一页，向左滑动联系人姓名，然后选择**移除**。这将阻止他们进一步访问该便笺。你现在可以从Apple Notes中删除该笔记，有了它，你的所有对话痕迹都将消失。

## 更多私密的聊天方式

虽然苹果笔记(或者就此而言，谷歌文档)允许你进行协作聊天，但这些并不是你真正想要寻求真正隐私的应用程序。理想情况下，你会希望使用 [加密信息应用，如信号](https://lifehacker.com/the-best-whatsapp-alternatives-1832064581) 进行这些对话。

在Signal中，每个聊天都是默认加密的，应用程序不会保存不需要的信息。在Signal中开始聊天后，点击页面顶部的联系人姓名，然后选择**消失消息**。您可以为这些消息设置一个自定义时间，之后对话中的每条消息都会自动消失。

你不需要手动删除任何东西，也不需要为秘密消息使用未加密的平台。当你上传图像到Signal时，它有一个方便的选项 [自动模糊所有的面孔](https://lifehacker.com/how-to-quickly-blur-faces-in-photos-1843908720) 。诸如此类的功能使它更适合私人谈话。如果你不想用信号，我们为你准备了 [和](https://lifehacker.com/for-better-privacy-use-these-apps-instead-of-whatsapp-1845551260) 的备选方案。
