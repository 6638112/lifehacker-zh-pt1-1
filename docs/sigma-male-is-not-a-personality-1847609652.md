# sigma male is not a personality 1847609652

> 原文：<https://lifehacker.com/sigma-male-is-not-a-personality-1847609652>

一个来自 incel 世界的狗屁概念已经渗入主流文化，所以是我们解释的时候了。我说的是“西格玛男性”的概念，他应该存在于与“阿尔法”和“贝塔”男性的关系中。

这些希腊字母的类别被认为描述了等级制度中的位置和个性类型。这应该已经让你明白了一个事实，那就是它们没有任何意义。等级制度中的职位是工作，而不是内在的生物特征。没有人生来就是将军、四分卫或首席执行官(当然，老板宝贝除外)。)



我们常常相信一个人的性格注定了他的伟大。这是个很可爱的想法，但和心理学甚至现实都没有太大关系；这是一个幻想的比喻。灰姑娘可以嫁给王子，因为她善良、甜美、漂亮。这个比喻很吸引人，因为我们可以想象自己是被选中的人。那么，如果这些都说不通，它是从哪里来的呢？

## “大男子主义”是动物行为研究中被揭穿的概念

在我们谈论西格玛男性之前，我们必须谈论阿尔法。1947 年，鲁道夫·申克尔根据在瑞士动物园的观察，写了一篇关于狼的行为的论文。他写道，似乎有一个等级制度，“阿尔法”对在顶端。另一位野生动物生物学家大卫·梅奇(David Mech)在一本书中普及了“阿尔法”一词，但在研究了野狼之后，他改变了主意。真正的狼群根本没有那种多层次的结构。

[这篇关于 Gizmodo](https://gizmodo.com/why-everything-you-know-about-wolf-packs-is-wrong-502754629) 的文章讲述了“阿尔法狼”想法的历史以及为什么它是错的。简而言之:狼群是家庭，有父母和孩子。父母狩猎，喂养幼崽。他们是族群的老大，就像人类的父母掌管他们的家庭一样: [例如，确保最小的孩子有足够的食物](https://sciencenorway.no/ulv/wolf-packs-dont-actually-have-alpha-males-and-alpha-females-the-idea-is-based-on-a-misunderstanding/1850514) 。一些狼群有更复杂的结构，但它们都是以家庭为基础的，不是残酷竞争的结果。

等级结构的想法也渗透到了狗的训练中，在那里有一个长达十年的时尚，训练者试图惩罚狗，让它们相信它们(训练者或主人)在群体等级中是“主导者”。这对于训狗来说也不是一个有用的比喻。



动物行为学家帕特里夏·麦康奈尔 [解释说](https://www.patriciamcconnell.com/sites/default/files/Bark%202001%20Winter%20Alpha%20Schmalpha%3F.pdf) 主人被教导将他们的幼崽翻个身以显示他们的统治地位，但狼通常不会这么做。相反，在狼群中，年轻或“顺从”的动物会把自己*翻过来。狼和狗在它们的关系中确实有等级观念，但这种观念并不是建立在处于顶端的家伙对周围的人都是混蛋的基础上的。*

## “阿尔法”对人类来说更没有意义

现在让我们来谈谈这一切是如何演变成有毒的男子气概的。最初的“阿尔法”狼被认为是雌雄成对的，但当人们争先恐后地用这种所谓的自然法则来描述人类社会时，这种说法很快就被搁置了。

如果你认为人类社会必然被组织成一个“阿尔法”在顶端的等级体系，你会想成为一个阿尔法，对吗？通过误解一堆过时的狼研究，你会得出结论，这意味着你需要对你周围的每个人都是一个占主导地位的混蛋。

像狼一样，人类确实有地位感，但也像狼一样，这很复杂。例如，你可能会适应工作场所社会结构中的一个位置，但同时你在朋友团体或你所在的社区组织中处于完全不同的地位。你也不需要敏锐地意识到社会地位的每一个等级。如果你想和某个级别比你高或低的人约会(例如，他们赚的钱比你多)，那可能会很尴尬。但是你也可以过正常快乐的生活，不用纠结于每个人在等级制度中的确切位置，反正不是每个人都能同意。



## “西格玛”的由来

“西格玛男性”的概念比阿尔法比喻更新。基本上，一群家伙已经说服自己所有的男人都处于一个阿尔法在顶端的等级体系中(已经错了)，然后他们意识到，为了赢得这些冲突并证明自己是阿尔法，不得不不断挑起与其他人的冲突是令人疲惫不堪的。

这甚至不是我们最近的野生亲戚黑猩猩所做的。灵长类动物学家 Frans deWaal 就“阿尔法”黑猩猩做了一场的演讲，强调了它们关心他人和反过来被关心的方式。他说:

> 我觉得大男子主义这个词，如果你在网上查一下，你会发现所有这些商业书籍都告诉你如何成为一个大男子主义者，他们的意思是如何打别人，打他们的头，让他们知道你是老板，不要惹我等等。基本上，阿尔法男性对他们来说就是一个恶霸。我真的不喜欢那种描述...它被用在一种非常肤浅的方式上，与真正的(黑猩猩)雄性首领无关。

但是，阿尔法男性的观念在有毒的男性特征定义的圈子里变得如此根深蒂固，以至于坚持它的人无法放弃它。换句话说，他们崇拜阿尔法，但也用一种让他们成为可怕的人的方式来定义阿尔法。你不会想成为那种阿尔法，也不会想和他们成为朋友。所以，是时候扔掉比喻了，对吧？



你可能会想。如果你已经认同男性等级制度是绝对的自然秩序，那就很难停止这样做，像正常人一样生活。但是这些家伙已经通过他们对男性气概的奇怪信仰定义了他们的亚文化(阿尔法只是其中的一部分——我们现在进入了泡妞艺术家/redpill/incel/MGTOW 的领域)。是时候给阿尔法一个替代品了:一个站在等级体系之外的家伙，但他仍然比等级体系中的每个人都要好。

这就是“西格玛男性”应该有的样子，这甚至比阿尔法/贝塔/欧米茄业务更扯淡，因为它不是基于过时的动物行为理论，而是简单地编造出来的。(贝塔只是等级体系中的第二个；omega 是希腊字母表的最后一个字母，是最下面的一层。适马是另一个希腊字母。)

你可以在各种解说片段中读到这个术语流行起来的简史，比如 [这个在梅尔](https://melmagazine.com/en-us/story/sigma-male-theory-incel-book) 。作者 Miles Klee 引用了下面这段话:

> 从根本上来说，[确定为西格玛男性]与试图通过黄道十二宫、迈尔斯-布里格斯类型、九格图数字或霍格沃茨学院的组织过滤器来了解自己没有什么不同。我们希望更好地了解自己，并且在我们的偏见的引导下，我们通过一个指定(在这种情况下是分级)个性的选择框架来寻求知识。当然，这一过程往往会导致依附于你认为最有利的原型，这解释了为什么围绕西格玛头衔有许多奇怪的守门人:如果它拥有任何权力，它必须是不寻常的，然而每个呆子都想声称他属于这个群体。



至少对于占星术，阅读图表和分享占星术迷因的人*知道*这是胡说八道(好吧，他们大多数人都知道。)当这些性格类型不适合你，或者你根本不在乎再去想它们的时候，这让你更容易把它们放在一边。也许你曾经真的认为自己是一个拉文克劳，但现在你 [对 J. K .罗琳](https://lifehacker.com/what-to-do-when-your-favorite-artist-has-been-canceled-1844731557) 有了更多的了解，你宁愿继续前进，分享关于成为九型人格的迷因。

当你选择的性格类型与对性别和社会地位的强烈信念以及它们在生物学中的假定根源有关时，这种转变就更难实现了。或者简而言之:如果你认为自己是(或者想成为)西格玛，你已经看穿了一些屁话。帮自己一个忙，一路抽身。
