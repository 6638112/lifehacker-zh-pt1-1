# 如何使用 Twitch 的新安全功能来阻止仇恨袭击和骚扰

> 原文：<https://lifehacker.com/how-to-use-twitch-s-new-security-features-to-stop-hate-1847776289>

视频流媒体网站 [本周发布了一套新的工具](https://blog.twitch.tv/en/2021/09/29/securing-your-chat-with-phone-and-email-verification/?utm_referrer=https://www.google.com/) ，旨在防止“仇恨袭击”和其他骚扰，这对任何曾经在 Twitch 上成为目标的流媒体用户来说都是一种解脱。期待已久的功能让创建者和版主能够更好地控制 Twitch 的聊天，允许他们根据用户账户的电话号码和电子邮件验证来排除用户。

Twitch 还试图遏制规避禁令的行为。虽然用户仍然可以用同一个电话号码验证最多五个账户，但如果一个账户在整个网站范围内被禁止，所有其他与该电话号码相关的账户也将被禁止，这使得屏蔽更加困难。同样的规则也适用于频道范围的封禁:Streamers 和他们的 mods 可以一次性封禁所有与一个电话号码或电子邮件地址相关的账户。

## **如何使用 Twitch 的新安全功能**

这些工具为创建者和他们的 mod 提供了一系列灵活的基于验证的选项来控制对聊天的访问。下面是设置它们的方法。

**开启新工具:**新功能的默认设置是“关”以下是如何扳动开关。

*   创建者可以访问 [仪表板下的新功能](https://dashboard.twitch.tv/settings/moderation)) > 设置 > 审核
*   Mods 可以通过聊天 >“管理审核设置”来访问它们

**选择您的限制:**新功能允许您选择访问聊天者的级别。以下情况可能需要电话和电子邮件验证:

*   所有账户
*   初次聊天的人
*   帐户时间少于 1 小时、1 天、2 天、3 天、1 周、2 周、1 个月、3 个月或 6 个月的聊天者
*   持续时间少于 10 分钟、30 分钟、1 小时、6 小时、1 天、3 天、1 周、1 个月或 3 个月

**设置豁免:**如果有任何种类的帐户您想要从新功能中豁免，您也可以这样做。您可以免除:

*   重要人物
*   订阅者
*   调解人

**检查你的其他安全设置:**这些并不是 Twitch 向创作者提供的唯一反骚扰工具，所以请查看 [Twitch 关于打击有针对性的攻击和骚扰的建议](https://safety.twitch.tv/s/article/Combating-Targeted-Attacks?language=en_US) 到确保你正在做你能做的事情将不良行为者排除在你的聊天之外。

## **关于 Twitch 的新安全工具，用户需要知道什么**

如果你发现自己被关在你最喜欢的流媒体聊天的门外，这可能是因为他们开始使用这些新的安全功能。你可以这么做。

**验证您的帐户:**如果您的帐户没有通过电话号码验证，当您尝试使用需要连接手机的聊天时，会要求您连接手机。你只需要验证一次，它将适用于整个网站。

**习惯潜伏**:你不能用座机或网络电话验证 Twitch 账户，所以如果你没有手机，你将被排除在使用新功能的聊天之外(除非你在豁免用户的创建者列表中)。但是你仍然可以观看视频流。
