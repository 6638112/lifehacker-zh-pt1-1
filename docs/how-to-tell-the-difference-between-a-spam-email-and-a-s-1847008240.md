# 如何区分垃圾邮件和骗局

> 原文：<https://lifehacker.com/how-to-tell-the-difference-between-a-spam-email-and-a-s-1847008240>

根据法律，公司应该通过“取消订阅”按钮或链接，给你一个不请自来的电子邮件的退出选项。唯一的问题？骗子做*完全相同的事情*——除了按钮或链接包含恶意软件，将窃取你的个人信息。下面我们来看看如何辨别这两者的区别，以及你是否应该取消订阅。



## **如何识别诈骗邮件**

幸运的是，Gmail 或 Outlook 等大多数电子邮件服务都有良好的垃圾邮件检测过滤器，这意味着发送到您电子邮件地址的绝大多数诈骗邮件很少被发现。然而，一些垃圾邮件仍然存在，要么是肆无忌惮的促销活动仅仅是绕过了反垃圾邮件法，要么是彻头彻尾的诈骗，企图窃取你的个人信息。对于诈骗邮件，它们通常很容易被发现:

*   他们使用像“gmail.com”这样的公共域名，或者域名拼写错误。
*   这家公司的名字并不熟悉，也没有出现在网络搜索结果中。
*   有一种点击链接的需求，带有一种似乎不专业的紧迫感。
*   这封邮件包含大量拼写错误和奇怪的字体(通常带有标点符号不当的主题行，如“我们需要您尽快确认”或“请求，请确认您的退订”)。
*   有一个虚假的，通常是超大的“取消订阅”链接或按钮可以点击，尽管它也可以有不同的标签(“立即参与！”或者“点击确定开始”)。

不幸的是，对于诈骗电子邮件，切换取消订阅按钮或链接只会向骗子确认您的帐户是活跃的，并且您很容易成为更多诈骗电子邮件的目标。更糟糕的是，这些链接可能包含恶意软件(包括 [勒索软件](https://www.zdnet.com/article/ransomware-an-executive-guide-to-one-of-the-biggest-menaces-on-the-web/) )，它们会从你的电脑中窃取个人信息。

## 我应该完全避开取消订阅按钮吗？

一般来说:如果你了解并信任这家公司,并且明白你为什么会收到这封邮件，那么对于那些不是真正垃圾邮件的邮件(例如，你最近停止阅读的时事通讯，或者购买后讨厌的 Warby Parker 跟进邮件)来说，选择“退订”可能是安全的。

如果这封邮件是未知的、不请自来的或者完全奇怪的，就把它标为“未读”(如果可能的话)，然后报告为垃圾邮件。这样做会将该电子邮件标记为垃圾邮件，从而阻止任何其他电子邮件进入您的收件箱。Business Insider 还建议 [屏蔽个人发件人](https://www.businessinsider.com/how-to-stop-spam-email) ，但这可能不是很有效，因为诈骗者一直在改变他们的地址。

此外，为了进一步保护你自己:你可能要考虑一个单独的“一次性”电子邮件，不包含任何个人信息，严格用于购物或通讯。这样，如果垃圾邮件成为一个大问题，你可以简单地创建一个新的电子邮件帐户，重新开始。

最后，如果你是一封诈骗邮件的受害者，考虑在向联邦贸易委员会举报。
