# 温柔的钩子，Expresso，以及其他会让人嘲笑你的误用

> 原文：<https://lifehacker.com/tender-hooks-expresso-and-other-malapropisms-that-wil-1848931843>

用迈克·泰森(Mike Tyson)在输掉一场比赛后的不朽名言来说，“我可能会变成玻利维亚人。”泰森在回答一名记者关于他职业生涯的下一步走向的问题时，他指的是那种幸福的无意识或不知道发生了什么的状态，或“遗忘”哎呀。



英语，以及使用英语的人，像泰森一样充满了误用。(谷歌对误用的定义是，“错误地使用一个单词来代替发音相似的单词，通常会无意中产生有趣的效果，例如，“跳一只*火烈鸟*”(而不是*弗拉明戈*)

我们很多人都已经知道 [最常见的人们说错的](https://lifehacker.com/10-common-phrases-people-constantly-get-wrong-and-how-1847456997) 的事情，比如:“为了所有密集的目的”，“我可以不在乎”，“could of”(而不是“could have”)和“。”(没错，即使因为这么久以来人们一直说错了，所以无论如何都在字典里，但它仍然是*无论如何*的非标准和非首选版本。引用韦氏词典的话，“记住，定义并不是对一个词的用法的认可。”)

在这里，我们对误用进行了更深入的探讨。为了启迪，因为 [我们以前写过最常见的罪犯](https://lifehacker.com/10-common-phrases-people-constantly-get-wrong-and-how-1847456997) ，也因为当州领导人如 [得克萨斯州州长里克·佩里](https://www.texasmonthly.com/news-politics/is-it-any-wonder-theyre-flush-with-prosperity/) 说“我们需要看看各州，它们是创新和民主的厕所”时，我们应该享受一下看，一些不太为人所知，但同样被误用的英语短语:

**“齐心协力:**”正确的说法是“**齐心协力**”意思是计划好的、协调好的或一起表演的，如音乐会。

一个完整的另一个故事:另一个，虽然已经被说了几百万遍，却不是一个真正的词。正确的说法是“一个完全不同的故事”“一个完全不同的故事”或“另一个故事”

“有人有解药可以分享吗？”除非你刚刚目睹了一次毒蛇咬伤，你正在为救命药拉票，你的意思是“有人有什么**趣闻**吗”或者，简短有趣的故事。

Gun-ho:Gun-ho，Gun-ho，该上班了，我们走吧...等等，不。那不是一首歌。当人们想表示某人对某事非常兴奋或热情时，他们想说的是“ **gung ho**

当你想说某人用他们的四肢留下了一场争吵，或者自尊心没有受到太大伤害时，他们完好无损地离开了那场争吵**。**

**当某人喋喋不休地谈论同一个话题时，他们谈论到了过度或令人作呕的程度(因此提到了恶心)。这就是拉丁语中令人作呕的***的意思。*****

******"半危险**:"所以这里没有完全危险，只有一半？好的，明白了。(虽然这个词很酷，但实际上它并不存在。)当你想说某事缺乏条理时，它是**偶然的。******

******“下一步棋:”**一步棋是一步棋，或者是任何有风险的行动或旨在获得优势的言论。(一个 Gambit 也是 x 战警之一。)选项不“运行策略”，它们**运行整个范围**，也就是某事物的全部范围。****

******”用瘸子的话说:“**用任何人都能理解的方式解释某件事的时候，正确的说法是*通俗地说——*的意思，不用任何专业语言。(俗人，传统上是指非牧师。)做出解释过于复杂？太逊了，伙计。****

****前麦当娜:没有前麦当娜这种东西。(我的错。它实际上是一个花花公子麦当娜在 1980 年发布的一组 [的小样](https://en.wikipedia.org/wiki/Pre-Madonna) 。)如果你的意思是说某人表现得像一个喜怒无常的自我主义者，他们表现得像一个首席女演员。这个短语来自意大利歌剧场景。****

******"Expresso:"** 跟我念:****espresso****中没有“x”。浓缩咖啡里没有 x。浓缩咖啡里没有“x”！*那里。你痊愈了。*****

*******"无疑**:"如果某件事是确定无疑的，那么它就是**无疑**如此。*****

*******"大概地:"**不，应该是**。(指的是“假定”或者一般认为或者相信的东西。)*******

*******保险箱:之所以叫保险箱，是因为你可以在那里进行安全的存款。*******

*********“倒在垃圾旁**:“垃圾”可以，也可能应该是一个词，但它不是。这个短语的意思是“半途而废”——路旁就是路边。*******

*******“紧张地等待某事发生就是**提心吊胆**，这是一种古老的方法，通过在一个大木架或“帐篷”上拉紧来干燥布料*******

*******"By in large:"** 虽然这个词在某些人的说话中很难察觉，但当试图说“总体上”时，正确的短语是 **by *和* large** 。*****

*****厌倦某人:这个人太棒了。虽然你*可能*对某人或某事感到厌倦(变得筋疲力尽)，但如果你由于感知到的危险而持怀疑或谨慎态度，那么你*就是警惕的*。(要警惕就是要*提防*。)例句:“小心杰克。他是一名球员。”(附言:你也可以对某人保持警惕，意思和警惕一样，但不幸的是和疲倦押韵。)英语很有趣。*****
