# 如何在 iOS 16 和 macOS Ventura 上使用“锁定模式”

> 原文：<https://lifehacker.com/how-to-use-lockdown-mode-on-ios-16-and-macos-ventura-1849179969>

苹果经常实施一些功能来保护普通用户的个人数据安全，但防止来自国家支持的黑客组织的复杂网络攻击要困难得多。为了应对这些高级别的威胁，苹果公司正在为 iOS/iPadOS 16 和 macOS Ventura 推出一种新的“锁定模式”。但是这对你来说有什么意义吗？



在官方发布的中，苹果表示，锁定模式是为“极少数用户设计的，这些用户由于他们的身份或行为，可能会成为一些最复杂的数字威胁的个人目标，例如来自 NSO 集团和其他开发国家支持的雇佣军间谍软件的私营公司的用户。”顾名思义，该功能彻底屏蔽了苹果产品上许多最易受攻击的功能，以阻止黑客获取用户数据。

在锁定模式下，任何传入的邮件附件(图像除外)都会被阻止，包括链接预览。来自未知联系人的 FaceTime 通话和通过其他 Apple 服务发出的邀请也会被自动阻止。这些都是黑客利用 [将恶意软件或间谍软件植入易受攻击设备](https://lifehacker.com/update-your-apple-devices-right-now-to-patch-this-serio-1847668729) 的常见媒介。

Safari 的 JavaScript 实时(JIT)编译器和其他网络功能也被禁用，因为恶意广告和网站是另一个主要威胁，特别是如果有 [未打补丁的 Safari 漏洞](https://lifehacker.com/you-already-need-to-update-safari-again-1848522230)[黑客可以利用](https://lifehacker.com/stop-using-safari-immediately-at-least-for-now-1848384461) 。这适用于其他也使用 Safari 的 WebKit 引擎的 iOS 和 Mac 浏览器。

锁定模式也可以防止有人物理访问您的手机上的某些文件。例如，使所有共享相册不可见，并阻止共享相册邀请。同样，当屏幕锁定时，与外部设备或附件(如电脑)的有线连接将被禁用，从而阻止他人访问您的数据。它还会阻止功能和设置更改，如安装配置描述文件或注册移动设备管理。

默认情况下，锁定模式是关闭的，因为其增强的安全设置对普通用户来说可能有些过分，但该功能将在所有 iOS 16 和 macOS Ventura 兼容的设备上可用，所以任何人*都可以使用它*。如果你想早点尝试，现在可以在最新的苹果操作系统测试版 中找到它，以及 iOS 16 中提供的许多其他新功能 [。](https://lifehacker.com/the-best-new-features-in-ios-16-and-ipados-16-1849023814)

## 如何在 iOS/iPadOS 16 或 macOS Ventura 上启用锁定模式

1.  **在 iOS/iPad 上:**进入**设置>隐私&安全。**
2.  **在 Mac 上:**进入**系统偏好设置>隐私&安全。**
3.  在隐私与安全屏幕中，向下滚动并选择**“锁定模式”**
4.  将出现一个弹出窗口，显示锁定模式的详细信息。如需更多信息，点击**“了解更多”**，否则选择**“开启锁定模式**继续。
5.  选择**“打开&重启”**确认。
6.  您的设备将重新启动。使用您设备的 pin/密码解锁。所有锁定模式设置现在都已打开。您也可以使用上述步骤禁用锁定模式。

很难想象您需要使用锁定模式的原因，但知道它的存在令人欣慰。如果你希望以不那么激进的方式来调整你的苹果设备的安全性，那么普通的 iOS、iPad 和 Mac 用户可以使用大量的其他隐私设置，包括 [让你的 iPhone 数据远离广告商的功能](https://lifehacker.com/how-to-hide-from-advertisers-with-ios-14-5s-new-app-tra-1846766232) ，以及 [阻止网站跟踪你的 Safari 设置](https://lifehacker.com/13-safari-settings-you-should-change-to-make-a-better-b-1847236951) 。
