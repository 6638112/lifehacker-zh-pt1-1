# 如何阻止 OneDrive 向您发送垃圾邮件和通知

> 原文：<https://lifehacker.com/how-to-stop-onedrive-from-spamming-you-with-emails-and-1848633234>

OneDrive 是围绕 T1 最好的云存储服务之一，但是它有一个 T2 问题——它倾向于用 T4 通知和垃圾邮件轰炸你。U 将照片上传到 OneDrive，然后该服务开始在常规上给你发送“今日记忆”通知。你将每天收到 OneDrive 的电子邮件，要求你重温记忆，并且在你登录到服务的每一台设备上都会收到类似的通知。



如果你厌倦了所有的闲聊、，这里有如何管理你所有的设备。

## 如何阻止 OneDrive 的垃圾邮件

用你的微软账号登录 OneDrive.com 和打开 [通知选项](https://onedrive.live.com/options/NotificationOptions) 页面。un 检查**在这一天内存可用**并且**人们对我们正在共享的文件进行更改**，停止所有 OneDrive 电子邮件。

## 如何在 Windows 上禁用 OneDrive 通知

Windows 上的 OneDrive 应用程序发送了很多通知，但其中一些确实有用，所以你可能想在禁用它们之前检查一下你所有的选项。要开始，请单击任务栏中的 OneDrive 图标，并点击右上角的齿轮图标。选择**设置**和点击顶部的选项卡。在**通知**下，检查每个选项，取消勾选不需要的。D 如果你的目标是停止每天的 ping ，当 OneDrive 建议我收集新的照片和视频时，会立即禁用**。**

## 如何在 iPhone 或 iPad 上禁用 OneDrive 通知

在苹果设备上打开 OneDrive 应用程序，然后点击左上角的个人资料图标。选择选择**设置**，然后选择**通知。** D 是可以不要的，比如 **OneDrive 为我创建相册**， **OneDrive 为我增强照片**，**我从今天起有了回忆**。

## 如何在 Android 上禁用 OneDrive 通知

如果 OneDrive 的提醒让你在 Android 上感到厌烦，请打开手机上的应用程序，然后点击屏幕右下角的 **Me** 选项卡。N 导航至**设置>【你的电子邮件地址】**，在**下【T6 时获得通知】，禁用任何你不想要的通知。**
