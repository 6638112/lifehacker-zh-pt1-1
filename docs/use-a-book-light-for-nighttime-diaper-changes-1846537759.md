# 夜间换尿布时使用台灯

> 原文：<https://lifehacker.com/use-a-book-light-for-nighttime-diaper-changes-1846537759>

关于婴儿的一件事是，他们不只是在白天弄脏他们的尿布，当每个人都醒着、警觉着并准备好处理这项工作的时候。你也要做大量的夜间改变，你需要照亮至少足够的情况，以确保你在做一份体面的工作——但不要太多，以免对每个人的系统造成冲击。



我们的 [后代脸书集团](https://www.facebook.com/groups/2018785615043946) 的一名成员想出了一个你可以在自己家里复制的解决方案:通过使用一个小的、可充电的书灯来获得你需要的柔和光线。詹妮弗告诉我:“我三个月大的孩子睡在我们卧室的摇篮里，我懒得在半夜把他抱到托儿所。所以我开始在床上换尿布，但是因为看不清楚自己在做什么而感到沮丧。书光解决了我的问题。”

这是她的场景，她把书灯夹在一个淋浴盒上，用来存放尿布、湿巾和其他必需品:

她把她的灯夹在球车上，因为摇篮的边太宽了，夹不下——但你可以在摇篮或换衣区附近找个地方把灯夹起来。

如果你决定走这条路，并想全力以赴，你可以考虑买一个备用的台灯，放在你的床头柜上，以备第一个台灯的电池没电的时候。它们不会被浪费:最终，你不会筋疲力尽，像以前一样阅读真正的书籍将再次成为一件事，你和你的伴侣(或最终你要改变的那个孩子)可以将这些灯用于他们想要的用途。
