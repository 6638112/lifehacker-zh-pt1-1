# 用这个恐慌决策矩阵做最后一分钟的选举捐赠

> 原文:[https://life hacker . com/make-last-minute-election-contributions-with this-panic-dec-1845482109](https://lifehacker.com/make-last-minute-election-donations-with-this-panic-dec-1845482109)

离明天还有一周就是选举日，这意味着我们正处于全国、全州、和地方政府竞选活动的最后冲刺阶段 。无论民意调查看起来多么有希望，我们从2016年的中学到了一点，即在民意调查中看到你喜欢的候选人*领先*[会增加你的焦虑](https://lifehacker.com/how-to-deal-with-anxiety-related-to-the-election-and-po-1845414578?rev=1603135867630) 。也许你感到无助，好像已经到了无计可施的地步。

如果是这样的话，你希望有什么事情——任何事情——你现在就可以做，和只要你有经济能力这样做，做一些有意义的捐赠还不算太晚。但是并不是所有的政治捐款都是平等的，你不想把你的血汗钱浪费在已经做得很好的竞选活动上，或者浪费在几乎总是以某种方式投票的地区。这里有一个 [恐慌决策矩阵](https://idlewords.com/panic_decision_matrix.htm) 可能会有所帮助。

### 如何决定在选举前的最后一刻捐款

如果你想在下周进行政治捐款，但又不想做任何研究，经营书签网站 [Pinboard](http://pinboard.in) 和一个名为[Tech Solidarity](https://techsolidarity.org)的政治组织 [马切伊·切格夫斯基](https://idlewords.com/about.htm) 想要帮忙。他建立了一个 [恐慌决策矩阵](https://idlewords.com/panic_decision_matrix.htm) 来帮助人们决定他们最后一刻的政治捐款将在哪里产生最大的效益。

这很简单:只需回答几个问题，关于你希望用你的捐款实现什么(例如，帮助 [最需要的候选人](https://secure.actblue.com/donate/state_slate_supplemental) ，或者关注某个特定的州)，它会指引你去捐款最有效的地方。是的，这包括许多州级职位的竞争。

如果必须是联邦政府的话，有这些 [五个农村众议院竞选](https://secure.actblue.com/donate/great_slate) 可以利用一些帮助。如果你坚持捐款给拜登/哈里斯竞选团队，切夫斯基说，捐款给 [【宾夕法尼亚】](https://secure.actblue.com/donate/state_slate_pennsylvania)[佛罗里达州](https://secure.actblue.com/donate/state_slate_florida)[北卡罗来纳州](https://secure.actblue.com/donate/state_slate_north_carolina) 的候选人。你可以在这篇 [更详细的博客文章](https://idlewords.com/2020/09/effective_political_giving.htm) 中读到更多关于他对政治捐赠的想法。

最后，切格夫斯基鼓励人们不要花太多时间担心向哪里捐款，而是把钱拿出来:“记住避免分析瘫痪的格言，”他 [在博客文章](https://idlewords.com/2020/09/effective_political_giving.htm) 中写道。“随便挑一个就走！”
