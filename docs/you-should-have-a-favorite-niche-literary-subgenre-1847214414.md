# 你应该有一个最喜欢的文学分类

> 原文：<https://lifehacker.com/you-should-have-a-favorite-niche-literary-subgenre-1847214414>

有时你想比传统的神秘、浪漫、惊悚、科幻和奇幻书籍更深入地探索。如果你喜欢这五种主要文学类型中的任何一种，但是已经厌倦了——或者你不知道所有这些文学类型之间的区别——是时候用一些小众类型分支了...而且有很多可以选择。这里有一些我们最喜欢的。



## 如果你想探索言情小说的子流派

也许老套的“浪漫”不再管用了。许多爱情片都植根于特定的时间或背景，因此故事将包含更多的小众主题和意象。

*   色情浪漫:这将会比你的平均价格更性感。你想要更多的描述，或者——我敢说——图形语言贯穿整个页面吗？那么这可能会满足你想象两个虚构人物撞上丑八怪的渴望。下面列举一些 [最喜欢的情色小说](https://bookriot.com/best-erotic-fiction/) 。
*   西部浪漫:对粗犷的牛仔和落难的少女感兴趣？《西部罗曼史》以狂野的西部为背景(或者可能是当代西部，如果那更合你的口味的话)，可能会偶尔涉足竞技表演或枪战。这里有一份 [西方言情小说的清单，让你入门](https://bookriot.com/diverse-western-romance-novels/) 。
*   **灵异浪漫:**说一个小众。寻找一个关于在闹鬼的地方找到爱情的故事？这应该符合要求。可能会有鬼魂、恶魔、灵魂之类的东西，但也会有，呃，浪漫。这里有一个 [列表，可以从](https://www.goodreads.com/list/tag/paranormal-romance) 中选择。

## 如果你想探索推理小说的子流派

推理小说总是关于解决某种谜题或犯罪，但是不同的分支类型涉及不同的调查方法，而且通常是不同类型的律师和侦探。

*   历史之谜:尝试一个发生在很久很久以前的谜团。你可能会发现追踪开膛手杰克的足迹比现代的犯罪调查更有意思。这里有一个的例子列表。
*   **惬意的神秘:**如果你喜欢神秘，但不喜欢血腥、胆量、血腥和心理变态，那么这就是你渴望的 PG 内容。舒适的神秘有点像看一集她写的 T2 谋杀案，而不是更可怕的惊悚片。这里有一个 [跳离点](https://bookriot.com/best-cozy-mystery-series/) 。
*   私家侦探疑案:如果你喜欢有破案癖好的流氓侦探，那么你可能会选择私家侦探疑案。*夏洛克·福尔摩斯*是一个经典的例子，也符合历史分类，因为它最初出版于 1893 年。下面是从 中选择的 [一些人物。](https://bookriot.com/best-private-detectives-in-mystery-series/)

## 如果你想探索惊悚小说的子流派

大多数惊险小说都围绕着一场与即将到来的厄运的赛跑。但是，是所说的厄运的背景决定了你正在阅读的惊悚小说的类型。

*   **法律惊悚:**你喜欢高风险的法庭剧吗？你不介意法律术语吗？法律惊悚片会让你陷入悬疑之中，这通常是叙事中的一个常量，直到做出判决。这里有 [十种可供选择](https://celadonbooks.com/courtroom-drama-best-legal-thrillers/) 。
*   恐怖惊悚片:比大多数惊悚片更恐怖，但比大多数恐怖小说更惊险。一个很好的例子， [编剧 Relief 指出](https://writersrelief.com/2008/12/18/a-thrilling-genre-thriller-novels-and-subgenres/) ，是*沉默的羔羊*，它以一些相当恐怖的背景为基础，但涉及一名侦探的搜索，以在为时已晚之前抓住一名杀手。还有一个 [吨的选项](https://bookriot.com/psychological-horror-books/) 。
*   政治惊悚片:自由世界最强大的领导人之间的高风险戏剧是这一类别的惯用手法。如果你想找一个容易回忆的例子，那就想想*《纸牌屋》。*下面是 [一些比较流行的](https://www.barnesandnoble.com/blog/10-of-the-best-political-thrillers-ever/) 。

## 如果你想探索科幻小说的亚流派

科幻小说种类繁多，所以让我们忽略一些更受欢迎的种类。

*   蒸汽朋克科幻:这种类型主要是关于美学的，并且从 19 世纪的蒸汽动力技术中汲取了很多灵感。在这个亚流派中呈现的技术与当代社会一样现代——如果不是更现代的话——但是都是以蒸汽为动力的。正如蒸汽朋克大道所言:“蒸汽朋克技术呈现出一种复古的外观，让人想起工业革命时代。”下面是 [亚属](https://reedsy.com/discovery/blog/steampunk-books) 的一些定义标题。
*   **交替历史科幻:**如果美国没有在二战中占上风会怎样？正如我们所知，这种对世界的剧烈重新定位是交替历史的一般前提。如果你正在寻找一个使用二战主题的介绍，那么试试菲利普·K·迪克的*人在高高的城堡里。*
*   军事科幻:太空中所有史诗般的战斗接踵而至。它不像《T2》中的《星球大战》——从技术上来说，这是一部被狂热爱好者称为“太空歌剧”的传奇——而更像是《T4》中的《星舰战队》，它更加独特地专注于星际战争以及随之而来的一切。这里有一些让你开始 的 [选项。](https://best-sci-fi-books.com/23-best-military-science-fiction-books/)

## 如果你想探索奇幻小说的子流派

幻想总是关于魔法和魔法场景，但并不是所有的魔法都以同样的方式描绘。

*   这些小说将神秘与平凡和正常融合在一起，就像一个正常的人类角色与一个巫师或喷火的龙建立了友谊。 [如《暴动笔记:](https://bookriot.com/what-is-low-fantasy/) 【魔法元素侵入已知世界，而非指示另一个完整的想象世界】Reddit 提供了 [这份推荐名单](https://www.reddit.com/r/Fantasy/comments/3ygxgr/low_fantasy_recommendations/) 。
*   史诗般的幻想:大而宏大的故事浮现在脑海中，比如*的《指环王》*或者*的《权力的游戏》。这些故事发生在完全不同的世界，通常涉及的角色的行为决定了他们周围环境的命运，通常也决定了他们认识的每个人的命运。*
