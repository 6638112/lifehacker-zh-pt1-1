# 何时选择储蓄而非投资

> 原文：<https://lifehacker.com/when-to-choose-saving-over-investing-1849671926>

即使你的预算有限，留出一些钱给你的未来(T4)总是一个好主意。但是即使当你建立你的 [应急基金](https://lifehacker.com/what-you-should-do-now-to-prepare-for-a-recession-1849129353) 的时候，你可能会想如果你投资的话，你的钱是否会更好地为你服务。无论你是计划明年度假还是 30 年后退休，准确知道你的钱该花在哪里总是一个挑战。根据你的财务目标，这里有一些关于选择储蓄而不是投资的知识。

## 何时确定储蓄的优先顺序

选择投资而不是储蓄的诱惑有点像金融 FOMO——尤其是当股票市场的平均回报率似乎远比所谓的“高收益”储蓄账户更有利可图的时候。( [《商业内幕》](https://www.businessinsider.com/personal-finance/mistake-investing-money-instead-of-saving-it-2022-10) )很好地突出了这个确切的难题。然而，在你选择股票而不是储蓄之前，仔细考虑一下你动用资金的时间表。

如果你预见到未来五年内任何时候都需要钱，那就优先存钱。也许你已经把目光放在了度假或房子的首付上，或者也许你需要增加你的应急基金。(供参考:一个“启动”应急基金大约是一个月的租金加上你的保险免赔额；从这里开始，假设你已经还清了任何高利息的债务，你应该继续建立一个应急基金，可以覆盖你六个月或更长时间。)

归根结底，要记住储蓄账户比投资账户更容易获得，风险也更低。当然，较低的风险可能听起来像“较低的回报，”但更重要的是，这意味着你不会在储蓄中损失金钱(除非你考虑到通货膨胀可能带来的损失)。 [下面是我们选择高收益储蓄账户的指南](https://lifehacker.com/how-to-choose-a-high-yield-savings-account-1835461088) 。

## 何时优先投资

如果你已经有一个舒适的应急基金，梦想是你的投资计划与你的储蓄计划相吻合。虽然储蓄账户是短期工具的选择，但你可以转向投资来实现你的长期目标，比如为退休后的或支付孩子的大学费用存钱。

关键是不要仅仅因为短期的金融 FOMO 而选择投资而不是储蓄。例如，如果你因为计划几个月后度一次长假而想将你的储蓄重新投入股票，而直觉认为当前的准衰退市场已经触底，那么将你的钱投入到至少持有五年的成长型股票将是不明智的。

## 底线

即使你储蓄账户的利率很低(而且 [现在利率正在上升](https://time.com/nextadvisor/banking/savings/experts-predict-how-high-savings-account-rates-will-go/) ) ，如果你打算在未来五年的任何时候利用这些钱，将这些钱重新投入股市仍然是不明智的。回顾你的财务目标，不要犯把短期资金投入长期储蓄工具的错误。
