# 这就是你应对路怒症的方式

> 原文：<https://lifehacker.com/this-is-how-you-should-respond-to-road-rage-1848832353>

当有人有路怒症时，所有人都是输家——包括他们的司机同伴。看到(也可能听到)某人变得如此愤怒，不知道他们会愤怒到什么程度，这可能会令人沮丧。



那么，当附近的司机加速行驶时，你能做些什么来保持安全和冷静呢？以下是需要了解的内容。

## 如果可能的话，请让开

被困在缓慢移动或停滞的交通中可能是这个人道路愤怒的原因，在这种情况下，你可能无法与他们保持距离。但是，如果你在一个有路怒症的人的附近(或目标)，并且有 [的机会拐下另一条街](https://driving.ca/auto-news/news/heres-how-to-deal-with-someone-experiencing-road-rage) 并远离他们，那么就去做吧。

## 抛开你的骄傲

当有人以一种挑衅和/或不尊重的方式与你交往时，很容易堕落到他们的水平并与他们的行为相匹配。但是即使他们是错误的一方， [现在也不是](https://www.azcentral.com/story/news/local/phoenix/2019/04/05/road-rage-tips-how-cope-and-respond-when-someone-else-isnt/3369955002/) 靠近他们，通过超速向他们挑战，回应他们的任何手势，或者做任何其他可能进一步激怒他们的事情的时候。

## 不要眼神交流

避免 [与愤怒的司机](https://www.azcentral.com/story/news/local/phoenix/2019/04/05/road-rage-tips-how-cope-and-respond-when-someone-else-isnt/3369955002/) 目光接触。这可能会加剧已经紧张的局势，可能导致对抗。把手放在方向盘上，眼睛看着路。

## 同情司机

除了路上发生的事情(或他们认为发生的事情)，经历过路怒症的人通常还会经历生活中的其他事情。所以临床心理学家 [拉克莎·沙利文博士](http://www.intowntherapy.com/) 说，最好对另一个司机感同身受。

“假设这个人的愤怒是情境性的，而不是性格缺陷的象征，” [沙利文告诉《驾驶](https://driving.ca/auto-news/news/heres-how-to-deal-with-someone-experiencing-road-rage) 。“愤怒的背后通常是悲伤。所以，想象一下，他们刚刚收到了人生中最糟糕的消息，他们不知道如何应对。祝他们一切顺利，继续前进。”
