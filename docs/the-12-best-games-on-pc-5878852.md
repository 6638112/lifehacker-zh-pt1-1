# 2022年Steam及以后下载的18款最佳电脑游戏

> 原文:[https://kotaku.com/the-12-best-games-on-pc-5878852](https://kotaku.com/the-12-best-games-on-pc-5878852)

PC游戏总是领先一两步。即使是在新一代游戏机的早期，大型呼哧呼哧的游戏塔在技术和游戏性能方面也取得了飞速的进步。再加上更广泛的可能类型，如复杂战略和MMO以及令人难以置信的充满活力的独立场景，加上PC如何倾向于回避主机战争的“排他性”，这是玩视频游戏最广泛的方式。

现在，就连PlayStation的第一方游戏也以增强版姗姗来迟地出现在PC上，再加上Xbox通过Game Pass实现的跨平台无处不在，简单塑料盒的便利性的丧失可以更容易地被忽视。然而，在这个列表中，我们关注的是最以PC为中心的游戏，而不是从其他平台移植过来的优秀游戏。

但是应该从哪里开始呢？你可能错过了什么？以下是你现在可以安装的18款最好的电脑游戏。

随着1.0版的即将到来，2022年的爆发将被认为是完整的。改造*机器人*-像子弹地狱类型，庞塞的*吸血鬼幸存者*是一个可笑的更有趣的游戏，收集越来越多的自动射击能力，以对付数百名屏幕上的敌人，因为他们向你涌来。

衍生出一整套自己的风格 ，这是今年最流行的游戏，在去年圣诞节前悄悄推出。这也不足为奇，因为玩一次，失败一次，再玩一次是一种令人难以置信的满足感。随着流氓建兴元素意味着你在尝试之间进行增量改进，以及越来越多的位置和英雄可以玩，停止玩是非常困难的。相信我们，几天来每次闭上眼睛，你都会看到那些蓝色的宝石。这似乎有可能成为有史以来的PC经典。

适合:任何想忘记凌晨3点睡觉，然后醒来想着再玩一会儿的人。

不适合:那些没有耐心输掉很多东西的人。
