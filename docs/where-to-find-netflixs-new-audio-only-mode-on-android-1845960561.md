# 在哪里可以找到网飞在 Android 上的新的纯音频模式

> 原文：<https://lifehacker.com/where-to-find-netflixs-new-audio-only-mode-on-android-1845960561>

网飞目前正在为其移动应用测试一种特殊的纯音频模式——特别是 Android。虽然不能保证这个非常有用的功能会在稳定的版本中出现，但它是有希望的，如果你幸运的话，你现在就可以测试这个功能。

Watch

我完全理解你对“听”电影而不是看电影的想法嗤之以鼻，相信我。我也会有同样的反应，但是在我做技术编辑的早期，我们经常这样做。一群*最大 PC* 的员工会花一整天在实验室里运行各种基准测试，构建(和破坏)各种组件，以及包装和打开无限的装运箱。我们会放电影来打发时间，但这并不意味着我们可以蹲下来看电影。我们有屁事要做。

纯音频体验有什么损失吗？显然，是的，但是如果这是一部你已经相当熟悉的电影，失去视觉效果——或者只是时不时地查看一下，或者浏览一下最喜欢的场景——并不是什么大不了的事情。如果你在一次公路旅行中，并且想要听一些比收音机更令人兴奋的东西(或者 [你最喜欢的播客](https://lifehacker.com/the-podcast-episodes-that-sustained-us-in-2020-1845956170) )，我可以看到同样的事情是真实的。为什么不放上你最喜欢的电影？由于你不传输视频，你应该少用很多宝贵的蜂窝数据流量(理论上)。

要查看你是否有资格参加网飞的纯音频试用，你需要确保你至少运行 7.84.1 版本(build 28 35243)的 Android 应用程序。这在谷歌 Play 商店上很容易检查；只需点击**“关于此应用”**并向下滚动，直到您在应用信息下看到版本号。

如果可以的话，试试在应用列表底部注册网飞测试版。我检查的时候已经满了，但是如果你能进去，你只会增加你被选中测试这些限制发布特性的几率。

有一句话要说:由于网飞的纯音频模式是服务器端解锁，即使你运行的是你能得到的最新版本，你也可能看不到它。如果你有资格，一旦你在网飞找到任何可以看的东西，你就会很快知道。在视图的顶部，暂停按钮 [的正上方，你会看到一个打开和关闭视频的选项](https://www.androidpolice.com/2020/12/16/netflix-is-rolling-out-an-audio-only-mode/?scrolla=5eb6d68b7fedc32c19ef33b4) 。

我没有这个选项，所以我得到了常规屏幕，如上所示。(此外，指责网飞不允许其内容截图。)

如果你能访问这个功能——你真幸运——你还会在网飞的设置屏幕中找到一个全新的选项。在应用程序设置中修改后的**视频&音频播放**类别下，你可以选择网飞何时应该默认为纯音频模式:总是、从不或每当它检测到你正在使用耳机或外部扬声器时。

我们不知道网飞何时会向所有人推出这一功能，如果会的话，但如果情况有变，我们会及时报告。不过，我迫不及待地想看看这个功能，因为我迫不及待地想享受电影，而不是在我不能盯着屏幕几个小时的时候浪费宝贵的数据。
