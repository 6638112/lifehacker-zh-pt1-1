# 试用Microsoft Outlook的未来外观

> 原文:[https://life hacker . com/test-drive-Microsoft-outlook-future-look-1846021518](https://lifehacker.com/test-drive-microsoft-outlooks-future-look-1846021518)

作为该公司“一个Outlook”计划的一部分，微软在Windows、Mac和网络上的Outlook和电子邮件应用程序将在未来几个月内发生一些重大变化。为什么要等呢？您现在就可以预览您的Outlook体验将会变成什么样。

Watch

今年晚些时候，你可以期待微软在Mac、Windows和web上的Outlook应用程序以及Windows 10上的邮件和日历应用程序中实现用户体验的标准化。然后，在2022年，这些应用程序都将过渡到一个新的“君主”客户端，这意味着你将在所有平台上使用相同的应用程序。

据 [Windows Central](https://www.windowscentral.com/project-monarch-outlook-web-universal-email-client-microsoft) 报道，微软表示，新的Monarch客户端将为Windows和Mac添加更多的操作系统级集成，包括“离线存储、共享目标、通知”，因此新的Outlook将感觉像你使用的每个桌面操作系统或网络浏览器上的原生应用程序。

这些变化的推出将是缓慢的，但您可以通过今天安装新的邮件渐进式Web应用程序(PWA)来预览新的跨平台界面。

**如何在Edge Chromium上安装Outlook PWA:**

1.  去outlook.com。
2.  点击**“…”**打开菜单
3.  进入应用程序，点击**“将此网站安装为应用程序”**
4.  安装完成后，PWA将会打开。

**在谷歌浏览器上:**

1.  在新标签页中打开outlook.com。
2.  点击地址栏最右侧的 **"+"** 按钮(这可能需要一段时间才会出现。如果没有显示，请尝试单击地址栏)。
3.  出现提示时，点击**“安装”**。

与Windows 10上的邮件和日历应用相比，Outlook PWA的外观要简单得多。它没有你可能用来随意调整Outlook的可视化定制选项，但它更整洁，包括相同的电子邮件功能，以及日历和待办事项视图。这种相似性应该会使向PWA的过渡——或者最终在今年晚些时候为各种Outlook和邮件应用程序提供新的用户界面——变得更加容易。

Outlook PWA还包括其他MS应用程序的快捷方式，如Word、OneDrive、One Note、Meet和Skype。如果Xbox Live和Skype帐户使用相同的电子邮件地址，您甚至可以从个人资料设置中管理它们。

这已经是对MS生态系统的一个更强大的表面级集成，我很好奇一旦Monarch在2022年推出，它将如何扩展。
