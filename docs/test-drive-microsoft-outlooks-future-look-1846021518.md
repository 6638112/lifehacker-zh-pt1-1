# 试用 Microsoft Outlook 的未来外观

> 原文：<https://lifehacker.com/test-drive-microsoft-outlooks-future-look-1846021518>

作为该公司“一个 Outlook”计划的一部分，微软在 Windows、Mac 和网络上的 Outlook 和电子邮件应用程序将在未来几个月内发生一些重大变化。为什么要等呢？您现在就可以预览您的 Outlook 体验将会变成什么样。

Watch

今年晚些时候，你可以期待微软在 Mac、Windows 和 web 上的 Outlook 应用程序以及 Windows 10 上的邮件和日历应用程序中实现用户体验的标准化。然后，在 2022 年，这些应用程序都将过渡到一个新的“君主”客户端，这意味着你将在所有平台上使用相同的应用程序。

据 [Windows Central](https://www.windowscentral.com/project-monarch-outlook-web-universal-email-client-microsoft) 报道，微软表示，新的 Monarch 客户端将为 Windows 和 Mac 添加更多的操作系统级集成，包括“离线存储、共享目标、通知”，因此新的 Outlook 将感觉像你使用的每个桌面操作系统或网络浏览器上的原生应用程序。

这些变化的推出将是缓慢的，但您可以通过今天安装新的邮件渐进式 Web 应用程序(PWA)来预览新的跨平台界面。

**如何在 Edge Chromium 上安装 Outlook PWA:**

1.  去 outlook.com。
2.  点击**“…”**打开菜单
3.  进入应用程序，点击**“将此网站安装为应用程序”**
4.  安装完成后，PWA 将会打开。

**在谷歌浏览器上:**

1.  在新标签页中打开 outlook.com。
2.  点击地址栏最右侧的 **"+"** 按钮(这可能需要一段时间才会出现。如果没有显示，请尝试单击地址栏)。
3.  出现提示时，点击**“安装”**。

与 Windows 10 上的邮件和日历应用相比，Outlook PWA 的外观要简单得多。它没有你可能用来随意调整 Outlook 的可视化定制选项，但它更整洁，包括相同的电子邮件功能，以及日历和待办事项视图。这种相似性应该会使向 PWA 的过渡——或者最终在今年晚些时候为各种 Outlook 和邮件应用程序提供新的用户界面——变得更加容易。

Outlook PWA 还包括其他 MS 应用程序的快捷方式，如 Word、OneDrive、One Note、Meet 和 Skype。如果 Xbox Live 和 Skype 帐户使用相同的电子邮件地址，您甚至可以从个人资料设置中管理它们。

这已经是对 MS 生态系统的一个更强大的表面级集成，我很好奇一旦 Monarch 在 2022 年推出，它将如何扩展。
