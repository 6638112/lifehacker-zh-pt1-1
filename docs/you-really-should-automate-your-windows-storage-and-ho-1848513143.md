# 您确实应该自动化您的 Windows 存储(以及如何做)

> 原文：<https://lifehacker.com/you-really-should-automate-your-windows-storage-and-ho-1848513143>

如果你经常耗尽电脑上的存储空间，现在是时候采取一些措施来解决你的问题了。我们中的许多人倾向于在我们的计算机上积累大量的垃圾，虽然你可能想购买更多的存储空间，但你也应该考虑采取一些措施来教你的计算机如何免费清理你的垃圾。

## 使用内置的 Windows 功能删除垃圾邮件

Windows 附带了一个名为 Storage Sense 的功能，可以清除不需要的文件。它可以很容易地被配置为按计划运行，并且它也可以在您的计算机存储空间不足时自行运行。Storage Sense 会自动删除回收站中的旧文件，清除临时文件，并帮助您识别哪些类型的文件占用了您每个驱动器上的大量空间。要在你的 Windows 机器上设置它，进入**设置>系统>存储**。您将在窗口的右窗格中看到有关大文件或文件夹的更多信息。

## 如何在 Windows 中配置存储感知

您可以通过单击标记为**配置存储感知或立即运行**的选项来配置存储感知。您可以单击存储感知下方的“关闭”按钮来启用它(启用后，该按钮会变为“打开”)。在**下运行存储感知**，选择你想什么时候自动清理你的电脑。您还可以在可用磁盘空间不足时选择**，在电脑存储空间不足时运行存储感知。**

**这个页面还有一些其他漂亮的选项，比如**删除我的应用程序不使用的临时文件**，删除不需要的文件。你也可以点击**下面的下拉菜单，删除我回收站里超过的文件。**您可以在 1 到 60 天之间选择一个选项，它会在这段时间内自动清空回收站。**

**另一个有用的选项是**删除我的下载文件夹中超过**未打开的文件。这可让您选择是否要从下载文件夹中自动删除未使用的文件，并且您可以选择 1 到 60 天的时间间隔。**

## **如何自动清理磁盘**

**您也可以使用 Windows 的磁盘清理工具来自动释放存储空间。这种方法要求您在命令提示符下键入一行简单的代码，但不要因此而却步，因为它与前一种方法一样简单。你只需要在你的账户上有管理员权限就可以了。**

**按下 **Windows 键+ X** ，从菜单中选择 **Windows Powershell (Admin)** 。这将打开命令提示符，您可以在其中键入这个命令: **cleanmgr /sageset:9000** 。如果愿意，您可以将该数字更改为不同的值，只要它是从 0 到 9999 的任何整数。**

**这将打开一个对话框，您可以在其中选择要自动清理的文件。当你选择了你需要的文件后，点击 **OK** 并关闭 Powershell 窗口。使用 Windows 中的搜索框打开电脑上的**任务调度器**，进入**动作>创建基本任务**。给这个任务一个名称和描述，并按照屏幕上的提示为它设置一个时间表(每天、每周、每月、计算机启动时等等)。).**

**当时间表确定后，您将到达 Action 选项卡。在这里你可以选择**启动一个程序**，在下一页输入**C:\ Windows \ system32 \ clean mgr . exe**，点击**下一步**，检查时间表是否看起来不错，点击**完成**。现在，磁盘清理将根据您设置的时间表自动删除指定的文件。**

## **如何从特定的 Windows 文件夹中删除文件**

**如果您的存储空间问题源于特定的文件夹，如下载或文档，那么您可以使用另一种方法来消除它们。打开 PC 上的记事本，将此命令粘贴到新便笺中:**for files-p " C:\ Users \ your username \ Downloads "-s-m*。* /c "cmd /c Del @path" /d -60** 。** 

**这已被预先配置为从您计算机上的下载文件夹中删除所有超过 60 天的文件。您可以将您的用户名替换为您帐户的用户名，,这样就可以了。**

## **如何定制你的命令提示符**

**要更改自动清理的文件夹，您可以更改路径。那是 C:\ Users \ your username \ Downloads 的部分。您可以在 Windows 资源管理器中打开任何文件夹，并从地址栏中复制其路径。命令中的 **-s** 位要求您的自动化在删除文件时也考虑下载文件夹中的子文件夹。如果您不想删除子文件夹中的任何内容，可以删除它。**

**最有趣的部分是 **-m *。*** 参数，允许您从文件夹中选择特定的文件类型。你可以把这个改成 **-m *。jpg*** 只删除 jpg 文件，或者将其改为。pdf，。png 或您想要的任何其他文件类型。注意，离开*。*在命令中要求它从文件夹中删除所有文件。**

**最后，术语 **/D** 让您决定文件应该有多长时间才会被考虑删除。 **/D -30** 表示它将删除超过 30 天的文件，并且您可以根据自己的喜好更改数字。**

**这将允许您根据自己的需要定制命令；完成后，转到**文件>另存为**并将文件另存为 DeleteAutomatically.bat。文件名本身并不重要，但。bat 扩展名可以，所以一定要避免将其保存为常规的 txt 文件。**

**现在，您可以再次使用任务计划程序以固定的时间间隔运行程序，如前一节所述。当选择程序时，您可以使用**浏览**选项来定位名为 DeleteAutomatically.bat 或任何您命名的文件。**
