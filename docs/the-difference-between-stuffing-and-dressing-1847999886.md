# “馅料”和“调料”的区别

> 原文：<https://lifehacker.com/the-difference-between-stuffing-and-dressing-1847999886>

来自密西西比州，但在洛杉矶长大，这造就了一个混乱的童年，尤其是在烹饪方面。不仅食物在各方面都不同——连一罐培根油都找不到——东西的名称也不同。“可乐”变成了“苏打水”，我妈妈在任何地方都找不到煮花生，调料被填料取代——这是我在搬到西部之前从未听说过的感恩节菜肴。



馅料和玉米面包的分类应该是简单明了的。如果你把它塞在鸟里面，那就是填料；如果你用另外的锅烹饪，那就是调料。但是在南方腹地有几个州甚至拒绝承认“馅料”是一个词或一个概念，更不用说是一道菜了，而且——根据《美食》2013 年的这篇文章——密西西比州是所有州中最顽固的。(我还听说爱荷华州优秀的非南方人称之为“穿衣打扮”，这让我很高兴。)

两者都是面包菜肴，尽管大多数南方调料是用玉米面包做的。(“玉米面包馅”不是个东西。)馅料和调料都是用汤汁浸湿，用鸡蛋粘在一起，通常都含有洋葱和芹菜，以及鼠尾草和所有其他常见的香草。两者都很好地适应了调整和重复，这是我喜欢的。我不知道梅森-狄克逊线以上的人是否加香肠，但这在我的族人中很常见。我喜欢这道菜的几乎每一个版本，不管它叫什么，但我更喜欢它在鸟的之外烹饪。

## 为什么感恩节时调料比馅料更好

抛开地区忠诚度不谈，使用“馅料”这个词已经没有多大意义了，因为把面包塞在鸟肚子里没有任何意义。在火鸡的空腔内烹饪面包对这两种菜肴都不好。馅料被充满生鸟细菌的鸟汁浸透，这意味着你现在不得不担心浸有肉汁的面包达到安全温度。(根据美国农业部的说法，这是 165 华氏度)。当馅料放在热烤箱里的死动物体内时，很难对馅料进行温度检查，而且那只动物的内部是潮湿的，这意味着你不会看到任何褐变、任何脆片或除了“潮湿面包”之外的任何质地

这对火鸡也没有帮助。将馅料加热到 165 华氏度(或者根据食品实验室[的说法，技术上安全的温度是 150 华氏度)会导致胸肉烤过头、变干。它还会阻碍空气流入和流出鸟，这可能导致烹饪不均匀，或者至少使事情花费的时间远远超过它们应该花费的时间。而且，如果你在炒火鸡——如果你重视味道胜过审美，这是你](https://www.seriouseats.com/how-to-take-the-temperature-of-your-turkey-video) [应该做的](https://lifehacker.com/choose-taste-over-looks-when-cooking-your-turkey-1820444769)——填料在物理上是不可能的。

## 如果我在感恩节用错了词，我会被吼吗？

老实说，你叫它什么并不重要，因为无论你说“馅料”还是“调料”，几乎每个美国人都会知道你的意思我交替使用这两个词，既有专业上的原因(出于搜索引擎优化的原因)，也有个人原因，因为每当我说“调料”，我出生在康涅狄格州的男朋友的大脑就会想到“沙拉”。(另外，基于我把它塞在嘴里的事实，可以把它称为“填料”。)

我——一个来自把所有碳酸软饮料都称为“可乐”的州的人——绝不会从语义上争论，但在美国各地的家庭中制造和供应的大多数东西都是调料，或者至少应该是调料。密西西比州并不以在许多事情上“正确”而闻名(如内战或 COVID 限制)，但他们拒绝称之为除了着装之外的任何事情，这在技术上是正确的，是最好的正确。所以我让我们有这一个。足球赛季是如此艰难。
