# 学习如何像复仇者那样移动

> 原文：<https://lifehacker.com/learn-how-to-move-like-the-avengers-1845182897>

你的孩子对超级英雄的热爱和他们站起来身体动起来的需要以最聪明和可爱的方式碰撞在一起。漫威与 [【莱斯米尔斯】](https://www.lesmills.com/avengers) 和一群健身教练一起加入了部队，带领孩子们进行五分钟的训练，在训练中他们学习出拳、踢腿、格挡、飞行和扔锤子，复仇者联盟的风格。

Watch

[这种短暂的身临其境的锻炼](https://www.youtube.com/watch?v=uYi1kyMeFHQ&t=2s) 现在正被各地的体育教师分享，作为一种让孩子们在疫情期间在自己家里活动几分钟的方式:

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-uYi1kyMeFHQ&start=2](https://lifehacker.com/embed/inset/iframe?id=youtube-video-uYi1kyMeFHQ&start=2) 

公平的警告:处于或接近“青少年”状态的大孩子可能会觉得这很幼稚和/或俗气——问问我 10 岁的孩子就知道了。它确实最适合 4-6 岁( *maaaaybe* 到 7 岁)的年龄段。

一旦你和你的孩子学会了如何像复仇者那样移动，如果你愿意，英国索伦特大学 的研究人员正在研究沉浸式身体活动，并有一个 [5-10 分钟的调查，你可以填写关于体验的](https://solent.onlinesurveys.ac.uk/move-like-the-avengers-final-version?fbclid=IwAR0sL8rVd1ZDVP1AFI0jqPOfYKFrIPAtiRS0FYUaIP47htrRveozLtFTtLY) 。
