# 你应该为2021年的假日旅行准备的最低限度的备用计划

> 原文:[https://life hacker . com/the-bare-minimum-backup-plan-you-should-haulida-1848005465](https://lifehacker.com/the-bare-minimum-backup-plan-you-should-have-for-holida-1848005465)

2021年假日旅游季看起来不太好。即使你已经提前几个月预订了机票、租车或酒店房间，旅游业变化的速度意味着有票或预订并不一定保证你的旅行会按计划进行。

潜在的天气问题、随时可能改变的COVID法规、员工短缺，以及去年灵活的预订和取消政策在很大程度上已经成为过去的事实，最好在进入这个季节时至少有一些备用计划。在 [美联社](https://apnews.com/article/coronavirus-pandemic-lifestyle-business-holidays-travel-1423fd4abfd3da3076101c1ea9ef2361) 的一篇文章中，莎莉·弗伦奇分享了今年要准备的几种不同的方式。以下是需要了解的内容。

## 如果你要坐飞机，假设你的计划会改变

还没到感恩节，航空旅行已经(还？)不靠谱。出于这个原因， [法语建议](https://apnews.com/article/coronavirus-pandemic-lifestyle-business-holidays-travel-1423fd4abfd3da3076101c1ea9ef2361) 进去为你的旅行稍微延长做好准备，并且对你的计划大体上灵活一些。

一个稍微不同的方法(也就是我的方法)是假设你的计划会改变，在航班延误或取消之间，租赁公司会把你预订的车给你前面的人——诸如此类。然后，万一事情*按照计划进行*，你会格外感激，这会看起来像一个节日奇迹。

## 限制自己随身携带

同样，如果你将乘飞机旅行， [法国人建议](https://apnews.com/article/coronavirus-pandemic-lifestyle-business-holidays-travel-1423fd4abfd3da3076101c1ea9ef2361) 尽可能把你需要的所有东西都放在你的随身行李中。这样，如果你的航班最终改变了，你就不会因为没有行李而被困住了。

## 投资旅游保险

即使你在预订航班时，在获得旅行保险时总是点击“不，谢谢”按钮， [法语说](https://apnews.com/article/coronavirus-pandemic-lifestyle-business-holidays-travel-1423fd4abfd3da3076101c1ea9ef2361) 这个假期是时候采取不同的方式(并实际获得保险)。

首先，检查一下你的信用卡中是否有旅行保险作为额外津贴——如果有，确保这个计划不是垃圾，并且确实为你提供了保险。如果你没有这个选择，是时候点击“是”购买旅游保险了(或者至少研究一下你的选择)。

## 至少有一条可供选择的路线和运输方式

不像Moira Rose独自在家，大多数人不会选择跳上一辆有波尔卡乐队的货车回家过节。这意味着，如果你的计划包括飞往某个地方，你应该在到达机场时准备好一个充分考虑的备用计划。现在不是为你的手机寻找出路和做初步研究的时候:那是你需要在家里做的事情。

如果开车去你的目的地是一种选择，弄清楚在那个地区有哪些租车公司，并且知道(大概)开车要花多长时间。你可以尝试给不同的公司定价，但如果你的航班被取消，其他人也会面临同样的情况，成本可能会大幅增加。

你还会想看看哪些其他航空公司飞往(或接近)你的最终目的地，以及他们的航班时间，这样你就能准确地知道在哪里预定另一张机票，如果有必要的话。
