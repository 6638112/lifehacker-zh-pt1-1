# 如果你打算买一个Meta Quest 2，最好快点

> 原文:[https://life hacker . com/if-you-planning-to-buy-a-meta-quest-2-better-do-it-f-1849338662](https://lifehacker.com/if-youre-planning-to-buy-a-meta-quest-2-better-do-it-f-1849338662)

如果你一直在犹豫是否购买Meta Quest 2虚拟现实系统，是时候做出决定了 : Meta昨天宣布Quest 2的价格将于8月1日上涨100美元。这意味着你只有五天的时间来刷出信用卡，否则你将面临给马克·扎克伯格多余的一百美元~~扔进焚化炉~~ [打造元宇宙](https://fortune.com/2022/04/28/meta-facebook-mark-zuckerberg-metaverse-business-losing-billions-part-plan/#:~:text=But%20the%20company's%20metaverse%20business,Mark%20Zuckerberg%20isn't%20worried.) 的未来。

128GB的Quest 2将从299美元涨到399美元，而256GB的价格将从399美元涨到499美元。该公司称涨价是必要的，，“为了继续投资推动虚拟现实产业的长期发展。”

一个两年前的科技产品价格上涨超过30%起初令人惊讶，但当考虑到Meta最近的困境和该公司即将发布的下一代VR设备时，这就(有点)更有意义了。Meta/脸书最近遇到了困难——自2021年以来，该公司的股票已经跌至其以前价值的一半，并且 it [很可能因为苹果的一项政策变化而损失了许多数十亿](https://gizmodo.com/apples-privacy-policy-cost-snap-facebook-twitter-and-1847971994)——所以看起来该公司已经决定其通过生产令人震惊的平价耳机来补贴VR行业的日子已经结束了。

## 关于Meta的下一个虚拟现实系统“项目Cambria”的所有信息

虽然Meta尚未宣布其下一款VR系统(代号为“Project Cambria”)的价格、发布日期或全部细节，但在过去几个月中，它一直在不断透露信息。

在4月27日的投资者电话会议上， 马克·扎克伯格表示，该公司的下一代虚拟现实耳机即将推出，并有一些严肃的雄心。扎克伯格说:“今年晚些时候，我们将发布一款更高端的耳机，代号为Project Cambria，它将更专注于工作用例，并最终取代你的笔记本电脑或工作设置。”。“这款高端设备将具有改进的人体工程学和全色穿透混合现实，将虚拟现实与物理世界无缝融合。”干净利落。

五月份，官方Meta账户发布了一段视频，展示了一款漂亮的新耳机；马克·扎克伯格也分享了一段脸书视频，展示了他自己戴着一款(经过严格审查的)Project Cambria耳机。

是否有人真的希望他们的笔记本电脑被基于工作的虚拟现实取代仍有待观察——对我来说，这项技术似乎更适合于用剑砍人头的T2和看色情片的T4。但是不管怎样，这对消费者来说都是双赢的。我们都希望虚拟现实设备能够完全实现技术承诺，但如果它对Meta不起作用，我们至少可以欣赏马克·扎克伯格的失败。
