# 如何开始速度训练而不受伤

> 原文：<https://lifehacker.com/how-to-start-speed-training-without-getting-injured-1848193655>

当这种情况发生时，有许多 [的跑步成绩让你关注](https://lifehacker.com/try-setting-these-running-goals-that-aren-t-tackling-a-1847974405) ，最好的选择之一就是在你的日常训练中加入一些速度训练。将速度训练融入你的常规锻炼中有很多好处，从增强你的心血管健康，到帮助你跑得更快。

然而，如果你刚刚开始，速度训练可能会令人生畏。“当你进入以最大强度做事的境界时，你给身体带来的压力是极端的，”安东尼·沃尔说，他是一名跑步教练，也是 [美国锻炼委员会](https://www.acefitness.org/) 的主任。“当我们看预处理，当我们看运动的灵活性部分，当我们看不同的训练时，理解这个概念是我们想要努力的地方。”

## **起步慢，持之以恒**

因为短跑比慢跑给你的身体带来更多的压力，这意味着受伤的可能性更大。这就是为什么慢慢开始并在提高速度时保持保守是如此重要。这意味着从更少的重复次数、更短的冲刺以及比你可能尝试的更保守的配速开始。Y 你需要加强你的身体，准备以更高的速度前进。

“我们想给身体时间来调整，”沃尔说。“这意味着让身体有机会加强韧带和结缔组织，以适应你将要施加在它们身上的力。”

就像健康的许多其他方面一样，冲刺需要时间、努力，最重要的是，坚持不懈。“跑步者应该对自己和自己的身体有耐心，”Josh Honore 说，他是一名私人教练，也是的首席教练。

## **关注移动性和稳定性**

“当我们在基于短跑的计划开始时让人们适应时，我们所做的一部分是增加他们身体的灵活性和稳定性，以便他们可以随着时间的推移处理增加的速度，”沃尔说。这包括结合灵活性训练，这样你的关节就有了正确冲刺所需的全范围运动，包括像踢屁股、高膝和高膝跨步等。为了了解要做哪些练习，Wall 推荐观看 [这段视频](https://www.youtube.com/watch?v=lGxt-sxfFvg) 。

“这些训练和演习是双重的，”沃尔说。“一个是确保你的身体能够满足要求，另一个是让跑步更有效率。短跑是高效的，能够以一种不会很快疲劳的方式运动。”如果你的状态不好，那么你将会消耗比你需要的更多的能量，并且不能快速移动。

## **知道什么时候开始**速度训练

如果你已经能够坚持慢跑 20 到 30 分钟，那么可能是开始增加短跑的好时机。沃尔对新跑步者的建议是从一些机动性训练开始，之后你将以 50%的努力重复跑 30 米，中间有一些恢复时间。

然后，一旦你完成了，你就可以开始你的长距离慢跑。虽然这听起来像是一个适度的目标，但在几周内每周做几次将有助于你慢慢建立身体的速度能力，在这一点上你可以开始提高强度。

“你会看到你感觉更好，你对自己的身体如何运作有了更多的了解，你可能在某些方面更加有效和高效地移动，因为你一直专注于其中的一些训练，”沃尔说。“速度会来的，只是需要的时间稍微长一点。”

一旦你开始达到 70%或 80%的努力，沃尔建议减少重复次数，至少直到你的身体调整过来。一旦你的身体更适应短跑，你会觉得更加舒适结合速度的训练。
