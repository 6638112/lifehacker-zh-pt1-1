# parents dont need self care they need help 1845239057

> 原文：<https://lifehacker.com/parents-dont-need-self-care-they-need-help-1845239057>

甚至在疫情袭击之前，优先考虑“自我保健”就已经很流行了像 [【一个坏妈妈】](https://maximumfun.org/podcasts/one-bad-mother) 这样的育儿播客长期以来一直在讨论父母(尤其是母亲)在寻找平衡时面临的挑战:他们的一个标志性短语是“ [我是一个自我](https://topatoco.com/collections/maximum-fun/products/maxf-obm-aself) ”，这是一个事实，当我们觉得自己不是*而是*一个自我时，我们应该提醒自己。如果说以前为人父母还不够艰难的话，冠状病毒的出现破坏了我们努力为自己创造的工作和生活平衡的表象。

突然之间，和你最好的朋友共度一点快乐时光不再是一种选择。甚至和你的闺蜜去 T2 散步也变得很危险。你不能去按摩或修脚，也不能和你的朋友去打高尔夫。在你自己的工作和孩子的学校作业之间，谁有时间在浴缸里泡一个舒服的长时间的澡，然后是更多的你自己的工作，因为你什么都落后了。



然后我们开始说——半是讽刺，半不是——在附近快速散步(独自)是自我保健。或者(独自)在城市里兜风是自我照顾。或者五分钟被锁在浴室里(一个人)。单纯的生存成了我们所能获得的全部自我关怀。戴安娜·斯伯丁为写下的这段对话变得再熟悉不过了:

> 我还想到，我最近坐在牙医的椅子上开了一个相关的玩笑。“你还好吗？”卫生员问。
> 
> “哦，我很好，这感觉像是在度假，”我回答道。
> 
> 没有孩子的陪伴，跑腿、打扫浴室或去看牙医会更容易吗？你打赌。这些活动构成休息吗？绝对不行。



我们降低了自我保健的标准——我们把标准一直降到了地面上——因为我们不得不这样做，但六个多月过去了，一切都没有改变。父母不需要每周额外的自我照顾；他们需要实际的帮助。不空提醒要“ [为自己腾出时间](https://www.zerotothree.org/resources/3262-young-children-at-home-during-the-covid-19-outbreak-the-importance-of-self-care)——跳过关于氧气面罩、*请*的陈词滥调，我们都知道。不是父母不想照顾自己，而是现在没有帮助会觉得不可能。

对于有小孩的父母、有特殊需求的孩子以及远程全日制学习的孩子来说，尤其是如果这些父母也在努力保住自己的工作的话。

如果你不是家长，或者如果你是家长，现在因为你的孩子长大了，或者他们至少有一部分时间回到了学校(我也属于这一类)，所以不要提供自我保健建议，而是提供实际的帮助。



[假装他们是全新的父母](https://offspring.lifehacker.com/send-a-new-mom-gifts-for-her-not-the-baby-1830135088) ，从医院回家，从孩子出生的身体伤害中恢复，并适应无尽的白天和不眠之夜的精神和情感伤害。 [你会怎么做](https://offspring.lifehacker.com/the-best-gifts-for-new-parents-are-services-not-things-1798702598) ？送饭。花钱请专业人士打扫他们的房子。拿起他们的脏衣服，第二天还回来，干净叠好。你现在可以做所有这些事情，即使(尤其是如果！)他们的孩子分别是五岁、三岁和一岁。他们现在可能会为一个新生儿的挣扎付出一切；与他们现在所做的相比，这似乎算不了什么。

以任何安全的、保持社会距离的、每个人都感到舒适的方式提供帮助。一周带孩子去几次公园，或者在他们骑着自行车在街上走来走去的时候监督他们。当妈妈或爸爸在开电话会议时，和他们一起玩棋盘游戏，玩变焦游戏 。放下晚餐或杂货。问他们需要什么。如果他们说“没什么”，不要相信他们。他们*确实*需要一些东西，他们只是不知道要什么，或者怎么问。

只是因为我们已经习惯了 2020 年是一个大狗屎秀的想法，并不意味着父母突然没事了。是的，他们需要自我照顾，希望 2021 年能带给他们一些。但与此同时，他们最需要的是帮助。
