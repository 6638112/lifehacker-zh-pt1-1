# 如何(远程)帮助你的家人使用他们的新技术

> 原文：<https://lifehacker.com/how-to-remotely-help-your-family-members-with-their-n-1848133025>

节日礼物对所有人都意味着新技术，新技术对所有人都意味着新鲜"技术支持"让您头疼！你知道你的朋友和家人会对他们的 设备和小玩意有意见；能够亲自帮助他们是一回事，但如果你住得很远，远程解决这些问题可能会很困难。这里有一些小贴士，可以让你和你想要帮助的人的事情变得更简单。



## 通过视频电话向他们演示

如果可能的话，请不要依靠普通的电话来帮助你的朋友和家人解决技术问题。有人打电话给你询问他们的设备可能是很自然的事情，但是在解决问题时，只有他们的声音才能继续，这很快就会变成一个问题。

不要误会我；一个电话就能解决许多技术问题。但是通常，如果你能看到你正在处理的东西，你会更快地发现发生了什么。当然可以，你可以问你爸，“后面有没有写 HDMI 1 的端口？我不知道，它应该在所有其他港口旁边...”*或*你可以让他把 iPhone 指向电视机后面，告诉他哪个端口是哪个端口。

如果你的家人对他们的新智能电视、扬声器设置、智能手表或类似的东西有疑问，最简单的方法可能是通过视频通话来帮助他们。通常，FaceTime 或其他智能手机视频通话将是最简单的方式，因为他们可以轻松地移动摄像头，向你展示他们在看什么。

## 让他们分享他们的屏幕

视频通话是技术支持的重要资源。但是，由于你通过智能手机、平板电脑和电脑进行视频通话，如果技术问题与这三种设备中的一种有关，这种方法可能不会很好。你可以让你的朋友分享他们的屏幕，而不是让他们把智能手机指向他们的电脑(反之亦然)。

屏幕共享过去比现在更复杂，但由于视频会议应用的普及，现在几乎所有现代设备都可以轻松实现这一功能。您的朋友可以将他们笔记本电脑的屏幕共享到您的手机，将他们手机的屏幕共享到您的平板电脑，并将他们平板电脑的屏幕共享到您的笔记本电脑。根据所讨论的设备，您也可以使用多个选项。

对于苹果用户来说，FaceTime 的新 SharePlay 可能是最好的途径。新功能允许你与其他 FaceTime 参与者分享你的屏幕，只要你运行的是 iOS 15、iPadOS 15 或 macOS Monterey。无论您的朋友在使用他们的 iPhone、iPad 或 Mac 时遇到问题，他们都可以加入 FaceTime 通话，共享他们的屏幕，您可以帮助他们解决问题。你可以从我们的完整指南 中了解更多关于如何使用这个功能 [。](https://lifehacker.com/how-to-finally-use-shareplay-to-watch-videos-together-1847943309)

无论你的朋友使用苹果、安卓还是 Windows，还有其他的屏幕共享解决方案。像 Zoom、Teams 和 Skype 这样的热门应用程序都内置了这一功能；你所要做的就是确保他们下载并设置了有问题的应用程序(你这边也一样)，然后启动屏幕共享。

Zoom 在其视频通话窗口有一个非常明显的“分享屏幕”选项；让他们点击它，然后点击“桌面”，再点击“完成”团队和 Skype 有一个类似的功能；让他们点击带箭头的方框，然后点击“桌面”、“T1”、“共享”、“T2”或“开始共享”从这里开始，给 T4 指路就容易多了。

## 使用应用程序进行远程访问

有时候，最简单的解决方法是你自己做。是的，是的，教会一个人钓鱼等等，但是如果技术支持会议陷入沮丧，没有人会去学习钓鱼。

如果你准备好迎接最初的挑战，远程访问你朋友的设备可以大大简化你这边的技术帮助。当然，这涉及到远程设置远程访问*。您需要使用上述方法之一来设置和安装该程序，您的家人或朋友需要按照您的指示进行操作。祝你好运！*

不幸的是，远程访问大多只能在 Mac 或 PC 上进行。对于 iPhones 和 Androids，你会发现大多数“远程访问”解决方案实际上是美化的屏幕共享工具。即使是我们将要推荐的应用程序，虽然可以在移动设备上使用，但也只提供直接在这些设备上使用的屏幕镜像。

有许多方法可以远程访问 Mac 或 PC，但最简单的是通过 TeamViewer 这样的应用程序。TeamViewer 将远程访问归结为一个唯一的 ID 和密码；你的朋友会给你这些信息，几秒钟后，你就可以从你自己的电脑、智能手机或平板电脑上访问他们的整个电脑。

在你朋友的电脑上设置这个应用程序时，你应该*肯定*进行一次屏幕- 分享会话，因为在电话上解释起来可能会很复杂。

## 如何使用 TeamViewer 远程访问计算机

让你的朋友去点击“免费下载”你也应该这样做，或者，如果在移动设备上安装，下载 TeamViewer 遥控器，用于 iOS 系统或安卓系统。让他们像安装其他程序一样安装程序。在 Windows 上，安装非常简单；只需选择“默认安装”，“接受-下一步”，然后允许 TeamViewer 对您的设备进行更改。

如果他们在 Mac 上，将有三个不同的安全设置启用。幸运的是，TeamViewer 显示了一个弹出窗口，解释您需要启用哪些设置；**屏幕录制**、**访问**、**全磁盘访问**。如果你没有看到弹出窗口，你会在**系统首选项>安全性&隐私**下找到所有三个选项。点击每个设置后，启用 TeamViewer。如果您没有看到该选项，请单击 **(+)** ，然后从您的应用程序中选择 TeamViewer。

此时，您的朋友将拥有其 TeamViewer 应用程序的唯一 ID 和密码。在 TeamViewer 应用程序的“控制远程计算机”字段中输入该 ID，然后在出现提示时输入密码。一旦你这样做了，你应该在你自己的设备上的一个窗口中看到你朋友的计算机屏幕！这里有很多设置你可以乱动，但本质上，你现在已经控制了他们的电脑。

完成会话后，请务必单击 TeamViewer 窗口中的(X)。您还需要确保没有人能够访问您的朋友和您的 TeamViewer 应用程序的 ID 和密码。远程访问非常有用，但如果落入坏人之手，可能会非常危险。努力把那一点讲清楚；您不希望您方便的远程会话演变成网络安全灾难。
