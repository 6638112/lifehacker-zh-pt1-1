# 有人制造了一个4D·《我的世界》的克隆体

> 原文:[https://life hacker . com/someone-build-a-4d-minecraft-clone-1849677485](https://lifehacker.com/someone-built-a-4d-minecraft-clone-1849677485)

视频游戏通常是二维的:你有你的2D侧滚动条，或者，更常见的，你典型的3D冒险。你很少看到超越这些维度的游戏，因为，你知道，我们生活在第三维度。那么，为什么4D会有一个《我的世界》的克隆体呢？

Watch

乍一看， [*4D矿工*](https://store.steampowered.com/app/1941640/4D_Miner/) 好像就跟*《我的世界》*一样。你被丢进了一个无限的、由程序生成的世界，任务是尽可能长时间地开采土地以生存下去。迄今为止，如此*《我的世界》*。也就是说，直到你移动鼠标上的滚轮，狗屎开始凭空出现。

每当你滚动到 *4D矿工*，你的世界就会发生巨大的变化。就在你面前的元素消失了，而其他元素显露出来。你以为是固体的土地变得开阔，露出了山丘和洞穴，而洞穴的墙壁则变成了深深的裂缝。问题是，这些事情都没有真正改变。从4D的角度来看，你滚动时看到的一切其实一直都在。我们只是看不到它，因为我们受限于我们的3D世界观。

想象一个二维的存在，比如原版*超级马里奥兄弟*中的马里奥，他们可以看到左、右、上、下，但仅限于2D世界观。如果你把他们放在三维空间，他们将看不到额外的空间维度。相反，他们会看到我们三维空间的2D横截面，这只会揭示我们所看到的世界的一部分。移动横截面，他们会看到空间的不同视图，而一些元素从其原始视角“消失”一直以来，我们看到他们所看到的一切，*和*世界本来的样子。

如果那个解释对你没有意义，我完全理解。这个东西比较复杂，直观解释就容易多了。看一看 [*4D矿工的*演示视频](https://youtu.be/u8LMyWcKL_c) ，有效地炫耀了这个概念:

一款4D游戏，如 *4D矿工*，采用了这一概念，并将其应用于我们身上。现在，*我们是*2D存在，除了我们正在体验4D世界的3D横截面。对一个4D观察者来说，这个世界看起来完全正常，所有的树木、洞穴、熔岩流和水体随时可见。然而，我们只能看到我们当前的横截面所允许的:当你滚动时，你移动了横截面，展现了4D世界的不同部分。

这种机制为我们熟知的*《我的世界》*探险增加了另一个维度。你需要穿越第四维度的不同层面，而不是简单地寻找补给。你可能认为当你开采一棵树的时候没有物品掉落，但是滚动4D空间，你可能会发现一个先前隐藏在视野之外的树枝。

同样，你会发现洞穴潜水比在《我的世界》T2要复杂得多。为了找到洞穴入口，你可能需要在第四维空间滚动，并在里面滚动一次。否则，你会错过元素，路径，熔岩和敌人。

哦，是的:敌人。在这个游戏中，你的敌人是4D人，这意味着如果你没有站在他们正确的角度，你甚至可能看不到他们。你也永远看不到他们的完整形态:你可以在飞机上滚动来看到敌人的不同侧面，但是他们的完整形象将永远隐藏在你看不到的4D空间中。你似乎遇到的第一个敌人是一种叫做超体的4D蜘蛛。如果藏在另一个空间的蜘蛛的概念引发了你的蜘蛛恐惧症，我不会责怪你。

当你玩的时候，你将能够建造工具来帮助你在这个4D世界中导航。例如，4D眼镜可以显示物品和敌人的轮廓，这是你从现在的视角看不到的。突然间，那些蜘蛛就不那么可怕了。您还可以构建一个4D罗盘来帮助您跟踪当前的位置，并更好地可视化4D世界的3D横截面。

你将需要那些工具，因为在4D矿工中做任何事情都比在3D T2《我的世界》中复杂得多。即使像建造庇护所这样简单的事情也成为一个挑战，因为你很快就会意识到你的3D制作技能并不能解决你需要建造的所有4D墙。如果你不小心的话，你的第一个避难所将会充满裂开的4D洞。突然，蜘蛛又变得可怕了。

*4D矿工*是 [目前在Steam](https://store.steampowered.com/app/1941640/4D_Miner/) 上免费试玩，正式发布计划在11月。它不会是第一款上市的4D游戏，但是， [根据这个维基百科列表](https://en.wikipedia.org/wiki/List_of_four-dimensional_games) ，它*是*2022年的第一款4D游戏，也是自2020年以来第三款发布的游戏。尽管这些类型的游戏很有限，但玩起来仍然很有趣。另一个 [有趣的概念是 *4D玩具*](https://youtu.be/0t4aKJuKP0Q) ，可以让你在4D空间玩不同的形状。像 *4D矿工*一样，你受限于你的3D横截面，所以你会看着这些玩具随着你的移动而改变形状和消失。
