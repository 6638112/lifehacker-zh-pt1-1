# 无伴奏合唱海洋小屋播放列表

> 原文:[https://life hacker . com/the-a-cappella-sea-shanty-playlist-1829039503](https://lifehacker.com/the-a-cappella-sea-shanty-playlist-1829039503)

如果你在抖音或社交媒体上很活跃，你可能会注意到最近你的订阅中有节奏的吟唱明显增多了。似乎年轻的孩子们已经接受了一种新的狂热，它是旧时代海洋棚屋的复杂混搭。

*据KnowYourMeme报道，自夏季* *以来，这一趋势一直在升温，几周前当* [*歌曲《Wellerman》*](https://twitter.com/EidolonFox/status/1348867807367344129)*的各种越来越复杂的版本的片段开始在推特上疯传时，这一趋势也开始蔓延。(如果你还需要更多的证据来证明这不仅仅是一次性的，这里有* [*一个有趣的线索，是关于迷因的其他最近的例子*](https://twitter.com/jennyrae/status/1347717112912097282) *。)然而，我们Lifehacker远远领先于抖音潮流:我们在2018年首次发布了下面的sea shanty explainer/playlist。视频制作人贾米森·赫尔曼在 [神秘海港](https://www.mysticseaport.org/) 海事博物馆的船上工作时演唱了这首歌。“海事史学界的说法是，一个好的船工抵得上一条线上的十个水手”——因为他一下子帮大家拉绳子。赫尔曼将60多首海洋歌曲——纯粹的无伴奏合唱——放入一个合作的Spotify播放列表， [海港音乐](https://open.spotify.com/user/127843054/playlist/1dqZocnsrClzzRUuIrEX3T?si=AtvJtnbmTxSv4jJPHrp3dg) 。这是经典的生产力音乐，适合体力劳动。对于我们这些伏案工作的人来说，这是一个极好的提神饮料。*

赫尔曼从他的叔叔Craig Edwards那里继承了对划船和海洋音乐的热爱，Craig Edwards是一名民间音乐家、教师，也是神秘海港海洋音乐节 的前总监。写在*投石车*、 [布莱尔·索恩伯格解释](https://catapult.co/stories/how-sea-shanties-helped-me-weather-the-storm-of-depression) :

> 棚屋是实用的歌曲，从甲板上工人们的狂野的“歌唱”演变成旋律和叠句，经过打磨和磨练，适合举起或拖运——从不轻浮，但不可或缺。

shanty(有时拼写为chantey或chanty，都源自法语 *chantez* 意为“歌唱”)是一种原始、粗糙的音乐，是圣歌的一个升级。但如果你寻找录制的棚屋，你会发现许多现代民谣歌手精心制作的曲目，堆满了乐器和录音室工程。甚至斯坦·罗杰斯，这位拥有 [的民谣歌手，也超越了无伴奏版的《巴瑞特的私掠者》，](https://youtu.be/B6Nl3PaTimA?t=41) 用曼陀林、吉他、贝斯、提琴、架子鼓，甚至钢琴来填充他的大部分录音。当整个乐队都加入进来时，很难感觉到你在拉绳子和扬帆。

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1040269572258054146&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1040269572258054146&autosize=1) 

幸运的是，大多数民谣歌手至少会录制一些歌曲，有时是整张专辑，无伴奏合唱。赫尔曼只为他的播放列表选择了声乐曲目，从几张专辑中四处抓取——包括里克·斯潘塞的《海洋歌曲和圣歌》，《T3》，其中以克雷格叔叔为背景声乐。我添加了几首曲目，包括《兰迪·丹迪·欧》，来自《刺客信条4:黑旗》 的 [原声带。听完后，加入你自己的无伴奏合唱。你也可以抵得上一条线上的十个水手。](https://open.spotify.com/album/2lMgz2BBwwzkc9ZfNB9zYi?si=yLytajdKRDe0UY62ME2cXw)
