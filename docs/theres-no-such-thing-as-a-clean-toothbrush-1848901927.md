# 没有“干净”的牙刷

> 原文：<https://lifehacker.com/theres-no-such-thing-as-a-clean-toothbrush-1848901927>

世界上有两种人。那种不假思索地把牙刷放在户外，放在一个厕所里的人(恶心，对吗？)，还有那种在不用牙刷的时候会煞费苦心地把它们盖起来，甚至会定期在洗碗机里或者用花哨的紫外线杀菌装置给它们消毒的类型。在这两个群体中，猜猜谁在遵循 CDC 和 ADA 的指导方针来保养你的牙刷？惊讶吧，是那些懒惰的家伙。



但是他们的牙刷上不是布满了细菌和其他恶心的微生物吗？他们确实是。*所有的*牙刷都充满了病菌，而且大部分都来自我们的口腔。 [这篇关于牙刷污染研究的综述](https://www.hindawi.com/journals/nrp/2012/420630/) 报道称，所有这些研究都“在使用后的牙刷上发现了明显的细菌滞留和存活”

牙刷一开始就带着细菌(毕竟，它们在出厂时是干净的，但不是无菌的)，并且很快就会带走你嘴里的任何细菌。如果你的口腔中有任何致病微生物(牙龈疾病、唇疱疹),它们会进入你的牙刷，你无法完全清除它们。这里有一个额外的有趣事实:随着牙刷越来越旧，它们的表面会磨损，它们甚至能藏更多的细菌*。*

## 你不会死于牙刷细菌

在我们进一步讨论之前，让我们先弄清楚一件事:虽然牙刷上的细菌可能会让你感到恶心，但对大多数人来说，这实际上并不会危害健康。 [疾病预防控制中心报道](https://www.cdc.gov/oralhealth/infectioncontrol/faqs/toothbrush-handling.html) 患有出血性疾病或免疫抑制的人可能需要寻找其他的口腔护理方式，如果是你，请告诉你的医生。对于我们其他人来说:“疾病控制和预防中心没有意识到任何与使用牙刷直接相关的不良健康影响。”

## 让你的牙刷变干

对付这些细菌的最好方法是什么？根据美国疾病控制中心和美国牙科协会的[和](https://www.ada.org/resources/research/science-and-research-institute/oral-health-topics/toothbrushes)，风干是关键。

刷牙后，用自来水冲洗牙刷。这并不能消除细菌，但它确实可以去除任何可能干扰干燥过程的牙膏或其他粘性物质。将你的牙刷放在一个支架上，让它直立放置并风干。让细菌变干是防止它们繁殖的原因。把用过的牙刷放在容器里或者盖上盖子很可能会助长细菌的生长。你挡在外面的少数细菌将会被在里面狂欢的细菌数量超过。

有关正确护理牙刷的更多信息:

*   不要共用牙刷；把你的体液残留留给自己。
*   出于同样的原因，不要让你的牙刷接触到别人的牙刷。
*   当你的牙刷磨损时，或者每三到四个月更换一次。当牙刷处于良好状态时，它能最好地清洁你的牙齿(并且含有较少的细菌)。

如果细菌的想法让你如此厌恶，以至于你觉得*必须*做点什么，美国糖尿病协会说你可以把你的牙刷浸泡在李斯特菌或 3%的过氧化氢(你可以在药店的棕色瓶子里买到的东西)中。这不会杀死所有的细菌，但它可能会减少 85%的水平。(现在，85%的一百兆仍然是一百兆，但也许这让你感觉更好。)

不要把牙刷放进洗碗机或微波炉；这可能会杀死细菌，但也会损坏你的牙刷。专家们似乎对紫外线消毒器意见不一。疾控中心说它们“可能会损坏”牙刷；美国糖尿病协会承认它们的存在，但没有提出支持或反对它们的建议。

## 我的牙刷上到底有没有便便？

那么，那些厕所的烟柱是怎么回事呢？诚然，厕所会将水滴散发到空气中 ，其中一些会蒸发成悬浮微粒，漂浮在房间里，最终可能会落在你的牙刷或其他物体上。

但是没有研究将厕所产生的气溶胶与让人生病的牙刷直接联系起来。相反，我们有像 [这篇论文](https://www.sciencedaily.com/releases/2015/06/150602130650.htm) 这样的研究，在公共浴室(平均 9 个人共用一个浴室)存放的 60%的牙刷上发现了潜在的粪便衍生细菌的痕迹。但是在这项研究中没有对照组；这不是存放在浴室的牙刷和存放在其他地方的牙刷的比较，这只是牙刷的调查。(研究还发现，细菌在用漱口水清洗过的牙刷上更常见，这表明漱口水浸泡可能不是减轻你细菌恐惧症的最佳方式。)

另一方面，一个微生物组项目对全国各地志愿者的牙刷上发现的微生物进行了测序，但未能找到任何确凿的证据表明牙刷上有粪便细菌。(包括这篇论文在内的早期研究使用的方法无法确定细菌是否来自粪便，只是它们似乎与常见的粪便细菌属于同一类。作者之一 [告诉 Gizmodo](https://gizmodo.com/toothbrushes-might-not-be-covered-in-poo-after-all-1846152736) “在你的牙刷上从我们的粪便中找到细菌是相当不可能的。

不管你的牙刷上是否有粪便细菌，我们已经确定让细菌变干比在一个封闭的容器里培养它们更好。如果你愿意，你可以把你的牙刷放在不同的房间里，但是要小心那个房间里隐藏的细菌来源。
