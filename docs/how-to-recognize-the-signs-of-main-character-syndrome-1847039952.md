# 如何识别“主要性格综合征”的症状

> 原文：<https://lifehacker.com/how-to-recognize-the-signs-of-main-character-syndrome-1847039952>

人们很容易陷入自己的生活——尤其是在一年的有限社交机会之后。生活在全球化的疫情中感觉很奇怪:每个人都知道世界上发生了很多事情，但长期的隔离会让我们觉得好像我们是周围唯一的人。

Watch

有些人走得更远，就好像他们的生活是一部电影，他们是主角。虽然这不是一个官方的心理诊断或心理障碍，但它已经被称为“主要角色综合症”这里有一些关于这种行为的知识，包括如何识别你或你认识的人可能陷入这种模式的迹象。

### 什么是‘主角综合症’？

正如斯旺西大学心理学教授菲尔·里德博士 [在《今日心理学》](https://www.psychologytoday.com/us/blog/digital-world-real-world/202106/what-is-main-character-syndrome) 中写道，“主角综合症”是社交媒体导致的特定症状的最新例子。下面是的[他拿](https://www.psychologytoday.com/us/blog/digital-world-real-world/202106/what-is-main-character-syndrome)的对着它:

> 目前，主角综合症是一个模糊的术语，其媒体和社交媒体的使用多于科学。这个术语指的是广泛的行为和思想，但是，从根本上来说，它是指当某人将自己作为他们生活的虚构版本(通常是他们自己的，尽管有时令人不安的是，其他人的)的主角，并通过社交媒体呈现这种“生活”。

当然，这种以自我为中心的行为并不新鲜。我们都认识一些人，他们似乎真的相信这个世界是围绕着他们和他们的需求转的，其他人都是为了以某种方式让他们受益。社交媒体为他们提供了一个向世界传播他们的故事的媒介——也就是他们的观众。

临床心理学家 Michael G. Wetter 博士在接受《新闻周刊》 采访时表示，主角综合征是“人类渴望被认可和被认可的自然欲望与快速发展的技术相结合的必然结果，这种技术允许即时和广泛的自我宣传”。

### **如何发现主角综合症的征兆**

那么，你如何判断你或你认识的人是否已经陷入了与主角综合症相关的行为模式呢？以下是里德和维特提供的一些迹象，他们都是《新闻周刊》文章 采访的 [:](https://www.newsweek.com/do-you-have-main-character-syndrome-1589245)

*   创造一个依靠观众来验证你的故事和生活的故事
*   创造并生活在现实的另一个版本中
*   在社交媒体视频上观看视频会引发一轮比较，让你想知道为什么你看起来不像视频中的人，为什么你不像他们那样快乐(即使他们的内容是高度策划的)

虽然有些人把主要性格综合症比作正念，里德说，虽然有趣，但这种理论是错误的。

“正念是关于意识到你现在的现实，注意你的环境，把你从过去的影响中解放出来，”他 [告诉《新闻周刊》](https://www.newsweek.com/do-you-have-main-character-syndrome-1589245) 。"在主角综合症的情况下，你正在把自己从现实中抽离出来."
