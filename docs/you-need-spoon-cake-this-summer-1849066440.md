# 今年夏天你需要勺子蛋糕

> 原文:[https://life hacker . com/you-need-spoon-cake-this-summer-1849066440](https://lifehacker.com/you-need-spoon-cake-this-summer-1849066440)

最好的蛋糕配料是鲜奶油。它清淡、细腻、略带甜味，可以搭配从天使蛋糕到双层巧克力慕斯的任何食物。釉料、糖霜和奶油奶酪都很棒，但鲜奶油是最棒的。它突出了明星成分，同时增加了丰富度和水分，而没有与压倒性的，令人作呕的甜味减损。

可悲的是，重力使得这种松软的蛋糕配料不能像我希望的那样被大量使用。如果不添加稳定剂，它的结构太弱，无法支撑厚饼层，而且它有一个坏习惯，即变回原始状态——液体。但与其承认失败，不如让我们把这些蛋糕挑战换成蛋糕机遇。我们将建造“向下”而不是“向上”与其为哭泣的鲜奶油哭泣，不如指望它！今年是勺子蛋糕的夏天。

勺子蛋糕就是你用勺子端上来的蛋糕。不只是因为你能，而是因为你必须。它采用了[三支莱克斯蛋糕](https://www.mexicoinmykitchen.com/tres-leches-cake/) 和 [经典英式松糕](https://www.thespruceeats.com/traditional-english-trifle-recipe-435147) 的精华部分，制作出令人印象深刻的奶油甜点，而不用担心它会崩溃。它本质上是两层蛋糕，中间夹着厚厚的一层生奶油。然后用更多的鲜奶油和你想要的任何东西装饰——夏天的水果、巧克力片、可可粉、里斯巧克力片、橄榄油(我知道你在外面)、可食用的花——所有东西用鲜奶油味道都很好。另一个好处是，它是在一个13 x 9英寸的蛋糕盘中烘焙的，然后在同一个盘中组装。这减少了清理，便于运输，让你可以放心地使用所有的鲜奶油。

## 如何制作勺子蛋糕

首先，在一个13×9英寸的蛋糕盘中做一个片状蛋糕，蛋糕盘至少有两英寸深，但是越深越好。你可以使用几乎任何类型的蛋糕食谱；一个标准的黄色蛋糕或巧克力蛋糕是非常好的，但它应该足够通风，以吸收多余的鲜奶油液体(像布朗尼这样密度大的蛋糕就不好了)。应该有足够的面糊填满盘子的一半左右。这个高度可以让你将蛋糕分成两层，同时也为以后的鲜奶油留有空间。

蛋糕冷却后，把它倒放在铁丝架上。把它水平分成两片均匀的蛋糕。如果你不习惯用锯齿面包刀，你可以用一根 [的牙线](https://lifehacker.com/you-should-be-cutting-these-foods-with-dental-floss-1848910295) 轻轻切开蛋糕。将底层(黄油)放回烤盘，切边朝上。如果完全合适，你就知道你做对了。

开始搅拌大量的鲜奶油，加入糖调味。一夸脱淡奶油就够了；你需要足够的奶油在中间形成半英寸厚的固体层，在顶部多加一些。一旦达到 [中峰](https://www.finecooking.com/article/whipping-to-soft-medium-and-firm-peaks) 阶段就停下来。中等峰值将缩小到足以湿润蛋糕过夜，而不会完全消失。将一半加糖的鲜奶油均匀地涂在蛋糕底层。在这个阶段，你可以将一些水果切片压入奶油中。将顶层翻转到鲜奶油上，切下一面，轻轻按压以确保接触良好。把剩下的鲜奶油涂在蛋糕上。添加一些装饰图案或保持平滑。盖上盖子，在冰箱里放置六个小时或一夜。

休息时间会让鲜奶油部分润湿蛋糕层，蛋糕层的作用就像海绵一样，让鲜奶油变得有点干燥，用勺子舀的时候保持形状。作为一种可选的装饰，就在上菜之前，在上面放更多的切片水果、面包屑或其他辅助装饰物。拿起一把大勺子，想想你已经准备好迎接夏天了。
