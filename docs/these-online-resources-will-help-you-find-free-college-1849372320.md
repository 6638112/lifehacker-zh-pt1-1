# 这些在线资源将帮助你找到免费的大学课本

> 原文：<https://lifehacker.com/these-online-resources-will-help-you-find-free-college-1849372320>

在 2021-2022 学年期间，普通大学生在课本和用品上的花费在 628 美元到 1471 美元之间。由于教科书 的价格通常会随着每一个新版本平均上涨 12%，这一成本只会继续上涨。这里有一些学生可以采取的措施来减少这些成本:租借而不是购买教科书，在二手书店购物，或者购买以前的版本(并祈祷该学科的基本原则从那时到现在没有实质性的改变)。但是也有一些网站提供数据库来搜索学生需要的教科书是否可以免费下载。



通过搜索你需要的教科书的 ISBNs(国际标准书号)你可以找到你要找的确切的教科书，而不必支付出版商荒谬的标价。在花钱买这学期的课本之前，这里有几个你应该去看看的网站。

[**库创世纪**](http://libgen.rs/) **。**在线被认为是免费教科书下载的“圣地”,图书馆 Genesis 是任何学生寻找所需教科书 PDF 文件的第一站。其简单的界面允许用户通过作者、书名或 ISPN 进行搜索。Library Genesis 声称拥有超过 240 万本非虚构书籍，8000 万篇科学杂志文章，以及数百万个漫画文件，小说书籍和数十万期杂志。

[**免费预定现货**](http://www.freebookspot.club/default) 。Free Book Spot 是一个网站，它实际上并不托管教科书的 PDF 文件，而是将用户与托管他们感兴趣的教科书的网站联系起来。尽管如此，Free Book Spot 有几十个类别的几十万本书。如果你需要的书没有列出来，有一个队列可以上传你的请求，让系统管理员去尝试和查找。

[**PDF 驱动**](http://www.pdfdrive.com/)T5**。** PDF Drive 是一个在线 PDF 搜索引擎，拥有超过 7500 万本电子书、杂志和文章。凭借直观的界面，PDF Drive 允许其用户在下载前预览文件(这很重要，因为该网站只允许 100MB 的免费下载)。它还允许您将链接标记为书签，以便日后参考，如果您需要方便地找到您没有下载的 PDF 。

**[**项目**](https://www.gutenberg.org/) **。以印刷机的著名发明者命名的古腾堡计划是一个拥有超过 60，000 本免费电子书的图书馆。作家迈克尔·j·哈特(Michael J. Hart)创立的一个志愿者项目表示，他的使命是“以尽可能多的格式为全世界提供尽可能多的电子书，以尽可能多的语言阅读，打破无知和文盲的藩篱。”他们的收藏有纯文本格式，也有 HTML、PDF、EPUB、MOBI 和 Plucker 格式。****

**[](http://theassayer.org/)****。分析师是一个在线图书目录，其作者选择免费提供这些图书。成立于 2000 年，分析师不同于其他资源，因为它的重点是与数学，科学和计算机相关的书籍。这个平台还允许你对你读过的书发表评论，如果你喜欢的话，还可以和其他读者交流。******

****[**开库**](https://openlibrary.org/) **。**打开听起来很像的库函数；用户可以“借出”一本书，并在你读完之后还给你。虽然它可能没有大量的教科书，但它确实有一些经常出现在教学大纲中的经典文学作品。开放图书馆的突出之处在于它的收藏可以翻译成几种语言，包括西班牙语、法语、德语、捷克语等等。****
