# 削番茄皮的三种极其简单的方法

> 原文：<https://lifehacker.com/thee-incredibly-easy-ways-to-peel-a-tomato-1849525275>

即使你喜欢的吃西红柿的方法是“像咬苹果一样咬”，在你的生活中也会有你不得不削西红柿皮的时候。也许你会处理你夏天的收获，或者简单地做一点点 [腌番茄沙拉](https://lifehacker.com/this-naked-tomato-salad-may-replace-your-caprese-1798634063) 。但是你*会*在那些皮需要脱落的时刻到来。

你应该如何剥西红柿取决于你对裸西红柿的需求有多迫切。幸运的是，有三种简单的方法可以去掉这些皮(然后，保存好皮和 [做番茄酱](https://lifehacker.com/eating-trash-with-claire-tomato-skin-salt-and-peach-sk-1794819973) 作为额外的奖励)。

## 如果你想保留味道和质地，使用经典的热烫和电击

[我们最早在 2012 年就报道过](https://lifehacker.com/quickly-peel-peaches-and-tomatoes-with-a-hot-water-soak-5928869) 这个经典的“黑客”了。这是最广泛使用的去除西红柿(和核果)皮的方法，而且非常简单——尽管有点潮湿。正如我们之前解释过的，你需要的只是一把锋利的刀、一壶开水和一个冰浴:

> 把水烧开，在你把水果放进去之前，在桃子或西红柿的底部画一个小 X。然后将水果轻轻放入沸水中，让它漂大约一分钟(更熟的水果漂短一点，不熟的漂长一点，只要能让果皮变松就行！)把水果拿出来，立即放进一个装满冰水的碗里。冰水停止了烹饪过程，当水果凉到可以触摸的时候，你可以直接把皮剥掉。

这是我用在我最喜欢的番茄沙拉 或者任何其他以生的、裸露的番茄为特色的食谱上的方法。西红柿不会在热水中停留足够长的时间，从而对它们的味道或质地产生不利影响，投入冰浴使它们无法烹饪。

## 如果你觉得懒，就把它们放在冰箱里

这就是 [如果你发现自己吃了太多的西红柿，可以使用](https://lifehacker.com/why-you-should-freeze-tomatoes-without-peeling-them-fir-1844714372) 的方法，因为它只需要你做很少的工作。不需要一壶开水，也不需要冰浴，甚至不用在水果顶端打一个小小的“x”。

将整个西红柿直接放入冰箱，可以保存一整年。当你准备好使用它们的时候，把它们放在热水中，轻轻地摩擦它们，直到表皮脱落。将它们拌入调味汁或任何其他需要去皮番茄的食谱中。然而，我不会生吃它们——冷冻会导致水果中的水分膨胀，破坏细胞壁，使它们变得有点软。(你不会在酱汁中注意到它，但你会在沙拉中注意到它。)

## 烧了他们，如果你只有几个

用一壶水烧开来剥两三个西红柿，对一些人来说，可能会觉得有点过头，但是你可以用 [驾驭火的力量](https://lifehacker.com/quickly-peel-tomatoes-by-torching-them-on-a-gas-stoveto-1787462339) 来得到非常相似的结果。除了用水换火之外，程序基本相同:在水果底部画一个小小的“x ”,然后用钳子夹住它，在明火上慢慢旋转(炉子和厨房用的火把效果一样好)。一旦果皮开始从水果上剥落，就把它放入冰水中或放在冷水龙头下。剥去外皮，照常进行。
