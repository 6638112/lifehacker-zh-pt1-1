# 用昨晚的炒饭做早餐玉米煎饼

> 原文：<https://lifehacker.com/make-a-breakfast-burrito-with-last-nights-fried-rice-1849525922>

最好的炒饭是用“剩饭”做的。S 具体来说，已经煮好的米饭，然后晒干，通常放在冰箱里。反过来，最好的早餐玉米煎饼是用剩下的炒饭做的——或者至少，那是我有幸吃过的最好的早餐玉米煎饼。



它始于 phat kaphrao，这不是一道炒饭，而是一道泰国菜，用泰国圣罗勒炒肉，配以米饭。这是一份非常丰盛(非常辣)的食物，我可以在吃好几顿饭。第一个被消费了——就像现在这样(很明显)。第二餐是各种各样的 DIY 炒饭——用黄油和少许酱油调味的剩肉和米饭。味道很好。即便如此，我还有第二次迭代的剩余部分，对此我并不生气。

第三道也是最后一道餐采用了早餐玉米煎饼的形式——香辣猪肉、罗勒猪肉、米饭、鸡蛋、一点火腿和一点奶酪一起放在一张温热的玉米饼里。这道泰国菜将一份不错的早餐玉米煎饼变成了一份美味，带来了热量、鲜味(多亏了怪味鱼酱)和香草味(通过神圣的罗勒)——正是我需要的来平衡鸡蛋和奶酪的丰富性。(如果必须的话，你可以称之为“融合”，但这实际上只是一个女人在尽其所能地利用她的剩菜。)

你不需要自己做炒饭来利用这个策略。任何炒饭，无论是外卖还是自制，都会工作的很漂亮。首先用中高火重新加热不粘锅里的米饭。一旦它变热并发出噼啪声，在米饭中间挖一个洞，在里面打一两个鸡蛋，然后在混入米饭之前快速搅拌。加入你想要的任何其他(熟的)早餐肉，然后在上放一些奶酪，移开锅，让它融化。当奶酪融化时，在炉子上加热你的玉米粉圆饼，让它变成一些漂亮的棕色斑点，然后将馅料放在上面，卷起，用嘴包住它。
