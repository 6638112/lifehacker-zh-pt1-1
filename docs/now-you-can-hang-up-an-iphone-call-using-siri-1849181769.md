# 现在你可以使用 Siri 挂断 iPhone 电话了

> 原文：<https://lifehacker.com/now-you-can-hang-up-an-iphone-call-using-siri-1849181769>

随着 iOS 15 中最近的更新，设备上的 Siri 功能在执行设备上的任务(如设置定时器、闹钟、提醒和给人打电话)时惊人地一致。事实上，拨打电话是 Siri 一直擅长的事情，它在蜂窝电话或 FaceTime 电话方面做得同样出色。现在，在 iOS 16 中有了一个新功能，Siri 也可以为你结束通话，完全不用手。如果你经常用 iPhone 扬声器说话，或者使用 AirPods 或 Beats 之类的无线耳机，这真的很有用。



(**注**:这是 iOS 16 的一个功能，目前只提供 [公测](https://lifehacker.com/how-to-install-the-ios-16-and-ipados-16-betas-1849023051) 。随着 iOS 16 公开发布定于秋季，它将适用于全球所有兼容的 iPhone。)

一旦你安装了 iOS 16，进入**设置** > **Siri &搜索** > **Siri 呼叫挂断**，启用“ **Siri 呼叫挂断**功能。现在，说“**嘿 Siri，挂掉**”来结束呼叫。当然，打电话的人*也会*听到这些话，这对他们来说可能有点奇怪，但是你可以自己决定为了免提的方便和自由这种小小的尴尬是否值得它。

如果您在“设置”中看不到此功能，可能是因为您的设备不受支持。据苹果公司称，仅支持 iPhone 11 及以上版本。但老款 iPhones 可以将该功能与支持 Hey Siri 功能的 AirPods 和 Beats 耳机一起使用。最初，这个特性也仅限于这七种语言:德语、英语、西班牙语、法语、日语、汉语普通话和粤语。

当你在 Siri 设置中时，看看我们关于 [恼人的 Siri 设置](https://lifehacker.com/8-ways-you-can-make-siri-less-annoying-1848343339) 的指南，你应该为了一些和平和安静而改变它们。
