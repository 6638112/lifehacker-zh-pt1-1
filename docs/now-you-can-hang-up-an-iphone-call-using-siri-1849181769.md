# 现在你可以使用Siri挂断iPhone电话了

> 原文:[https://life hacker . com/now-you-can-hang-up-an-iphone-call-using-Siri-1849181769](https://lifehacker.com/now-you-can-hang-up-an-iphone-call-using-siri-1849181769)

随着iOS 15中最近的更新，设备上的Siri功能在执行设备上的任务(如设置定时器、闹钟、提醒和给人打电话)时惊人地一致。事实上，拨打电话是Siri一直擅长的事情，它在蜂窝电话或FaceTime电话方面做得同样出色。现在，在iOS 16中有了一个新功能，Siri也可以为你结束通话，完全不用手。如果你经常用iPhone扬声器说话，或者使用AirPods或Beats之类的无线耳机，这真的很有用。

Watch

(**注**:这是iOS 16的一个功能，目前只提供 [公测](https://lifehacker.com/how-to-install-the-ios-16-and-ipados-16-betas-1849023051) 。随着iOS 16公开发布定于秋季，它将适用于全球所有兼容的iPhone。)

一旦你安装了iOS 16，进入**设置** > **Siri &搜索** > **Siri呼叫挂断**，启用“ **Siri呼叫挂断**功能。现在，说“**嘿Siri，挂掉**”来结束呼叫。当然，打电话的人*也会*听到这些话，这对他们来说可能有点奇怪，但是你可以自己决定为了免提的方便和自由这种小小的尴尬是否值得它。

如果您在“设置”中看不到此功能，可能是因为您的设备不受支持。据苹果公司称，仅支持iPhone 11及以上版本。但老款iPhones可以将该功能与支持Hey Siri功能的AirPods和Beats耳机一起使用。最初，这个特性也仅限于这七种语言:德语、英语、西班牙语、法语、日语、汉语普通话和粤语。

当你在Siri设置中时，看看我们关于 [恼人的Siri设置](https://lifehacker.com/8-ways-you-can-make-siri-less-annoying-1848343339) 的指南，你应该为了一些和平和安静而改变它们。
