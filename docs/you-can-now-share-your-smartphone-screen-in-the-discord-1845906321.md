# 你现在可以在 Discord 应用程序中分享你的智能手机屏幕

> 原文：<https://lifehacker.com/you-can-now-share-your-smartphone-screen-in-the-discord-1845906321>

Discord 终于在其 Android 和 iOS 应用中加入了屏幕共享功能。现在，您可以在语音频道或视频通话中与多达 50 人共享您的移动设备屏幕。

Watch

Discord 主要是作为一个专注于游戏的应用程序进行营销的，因此与他人分享你的设备的屏幕的能力对其用户来说具有明显的吸引力；例如，你可以从你的移动设备上玩游戏，或者向你的 D 组展示他们所在的地牢的地图。但是屏幕共享也有很多非游戏用途。

许多人使用 Discord 作为远程办公应用程序的免费替代品，如 Zoom 和 Slack，这意味着你现在可以在语音或视频会议中分享来自 iOS 或 Android 设备的演示文稿。如果您的爱人也使用该应用程序，这也是一种与他们轻松共享照片、视频或其他媒体的方式。下面是怎么做的。

### **如何在 Android 和 iOS 上的 Discord 应用中筛选分享**

要开始使用，请更新您的 Discord 应用程序或从[Google Play](https://play.google.com/store/apps/details?id=com.discord&hl=en_US&gl=US)(Android)或 [苹果应用商店](https://apps.apple.com/us/app/discord-talk-chat-hang-out/id985746746) (iOS)下载最新版本。Discord 正在推出屏幕共享功能，所以你可能无法立即使用它。如果是这样，不要绝望；你不应该等很久才能得到它。

从那里开始:

1.  打开 Discord 应用程序，然后加入语音频道或视频通话。
2.  点击呼叫控制栏中的**“屏幕共享”**图标。
3.  一个对话框将解释屏幕共享是如何工作的。轻点**“试试看！”**开始。
4.  该应用程序将再次警告您，您即将与其他人共享您的屏幕。如果您准备好分享您屏幕上的内容，请点击**“立即开始”**
5.  该应用程序将 sap 到一个新的屏幕共享用户界面，但你可以切换到其他应用程序，并像往常一样使用你的手机，而不和谐的声音在后台传输给你的观众。
6.  当您准备停止时，选择**“停止共享”**

请记住，屏幕共享可以让你的观众实时看到手机或平板电脑屏幕上显示的所有内容(除非应用程序不允许屏幕共享访问)。虽然这将使与你的不和群一起观看 YouTube 更加容易，但这也意味着每个人都将看到你的通知或其他潜在的敏感材料，如果它们在屏幕上可见的话。请确定您关闭了不想意外切换到的任何应用程序，并在共享前打开“请勿打扰”模式来静音通知。
