# 如何打开“嘿，Spotify”语音控制

> 原文:[https://life hacker . com/how-to-turn-on-hey-Spotify-voice-controls-1846666076](https://lifehacker.com/how-to-turn-on-hey-spotify-voice-controls-1846666076)

Spotify在其Android和iOS应用程序中有一个全新的语音助手。对着你的设备大喊“嘿，Spotify ”,你就可以不用手来控制Spotify应用程序的各个方面——这在你开车或打扫房间时非常有用。美国的第一批用户现在应该可以使用这项功能，该公司表示，未来几周将在更多设备上推出这项功能。

虽然你可以通过Android Audio和Android上的谷歌助手用语音控制Spotify，也可以通过苹果设备上的Siri和CarPlay以有限的方式控制Spotify，但“嘿Spotify”可以直接与Spotify应用程序进行交互。这意味着你可以使用语音控制，而不必训练你的设备的数字助理识别Spotify特定的命令。

一旦“嘿Spotify”在你的设备上可用，你需要在应用程序中打开语音控制来使用它们:

### 如何启用“嘿，Spotify”语音控制

1.  打开Spotify应用程序。
2.  转到**“搜索”**选项卡。
3.  轻按搜索栏旁边的麦克风图标。
4.  选择**“打开‘嘿，Spotify’"**
5.  授予Spotify录制音频的权限。

你现在可以使用Spotify的 [内置语音控制](https://newsroom.spotify.com/2021-04-09/5-things-to-know-about-the-new-hey-spotify-feature/) 。只需说“嘿，Spotify”，然后跟着一个命令。请它播放您制作的特定艺术家或播放列表，暂停或跳过曲目，或者加载播客的最新一集。您还可以执行语音搜索，而无需先切换特殊的语音模式。

如果你想关闭该功能，请前往**设置>语音交互**并禁用**“嘿，Spotify”**这将防止意外激活，如果您发现自己在启用该功能后从未使用过，这对于个人隐私来说也是明智的。

像其他语音驱动的应用程序一样，Spotify会在后台听你说“嘿，Spotify”Spotify表示，该应用程序只收听短时间的突发内容，这些内容会被立即删除。如果它识别出一个命令，应用程序内的通知和音频会播放，让你知道应用程序正在记录你的那一刻。应用程序在处理你的命令时需要录音，但如果你想了解更多关于如何处理你的语音音频的信息，你可以阅读Spotify的语音数据隐私政策 。
