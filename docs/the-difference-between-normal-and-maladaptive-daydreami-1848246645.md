# 正常和不良白日梦的区别

> 原文:[https://life hacker . com/the-difference-between-normal-and-dis-adaptive-day dreami-1848246645](https://lifehacker.com/the-difference-between-normal-and-maladaptive-daydreami-1848246645)

我们所有人偶尔都会走神。事实上，据估计，从 [到](https://behavioralscientist.org/daydreaming-might-make-you-more-creative-but-it-depends-on-what-you-daydream-about/#:~:text=Daydreaming%2C%20also%20known%20as%20mind,even%20notice%20when%20it%20happens.) 我们醒着的时间有四分之一到一半是在做白日梦，因为我们的意识从我们目前正在做的或正在经历的事情中游离出来，完全专注于另一个话题。

Watch

[研究显示](https://www.science.org/doi/10.1126/science.1192439) 相比不愉快或中性的事情，我们更容易做关于愉快事情的白日梦，并表明这种习惯 [可能会积极影响我们的创造力、](https://www.psychologytoday.com/us/blog/fulfillment-any-age/201901/why-daydreaming-can-improve-your-mental-health) 情绪和生产力。但是有些人经历了比“正常”更强烈的走神，频率更高，持续时间更长——这种模式被称为适应性不良白日梦。

### 什么是适应不良白日梦？

适应不良白日梦，顾名思义，是一种强烈的、分散注意力的、对我们的功能 具有破坏性的 [走神形式。在那些经历适应不良白日梦的人身上有一系列常见的](https://psycnet.apa.org/record/2002-11146-005) [症状](https://www.sleepfoundation.org/mental-health/maladaptive-daydreaming):

*   生动、详细的角色和故事情节体验
*   由现实生活中的经历和刺激引发
*   作为白日梦动作一部分的动作或言语
*   持续时间长(几分钟到几小时)
*   想要继续体验的愿望
*   日常工作困难，包括注意力不集中和睡眠困难

即使有与这种经历相关的特定症状，适应不良白日梦也不是官方诊断，也不包括在当前的精神障碍诊断和统计手册(DSM-5)中。事实上，大多数发表的相关研究都来自以色列一个名为 [的组织的研究人员，该组织是国际适应不良白日做梦研究联盟](https://daydreamresearch.wixsite.com/md-research) (ICMDR)。

### 什么原因导致不适应白日梦？

不适应的白日梦的动力和机制还不清楚，但在某些情况下，它可能与人们的创伤经历和相关的应对机制的发展有关。

“适应性不良白日梦通常是经历过孤独、创伤和虐待的人的一种逃避形式——他们创造了一个复杂的内心世界，每当他们陷入困境时，他们都可以去那里，”萨姆·纳比尔说，他是 [纳亚诊所](https://www.nayaclinics.com/) 的持牌专业顾问。"因此，他们过度地做白日梦，因为他们更喜欢幻想而不是现实生活."

这些强烈的白日梦通常 [有暴力、](https://psycnet.apa.org/record/2002-11146-005) 的主题、权力和控制、囚禁、性、营救和逃脱。一项 [研究显示](https://pubmed.ncbi.nlm.nih.gov/29867613/) 患有适应不良白日梦的人还会经历分离(与自我断开)、强迫症状以及焦虑和抑郁等负面情绪。

### 那么，为什么这是一个问题呢？

所谓的适应不良白日梦在某些情况下可能实际上令人愉快并具有治疗作用，但当它妨碍学校、工作、人际关系和其他日常活动时，就会变得令人烦恼。任何官方诊断都是如此——功能障碍是一个关键因素。也就是说，如果你觉得这是一个问题，它可能值得寻求支持。

如果你认为你正在经历适应不良的白日梦，考虑 [去找一个治疗师](https://lifehacker.com/how-to-select-the-right-therapist-for-you-1833246447) ，他可以帮你导航你的症状(我们知道，说起来容易做起来难)。由于适应不良白日梦是一个相对较新的概念，没有“官方”的治疗方法，你可能不容易找到专门研究它的人。但是，有认知行为疗法和/或正念经验的临床医生，或者处理行为或强迫症状的临床医生可能是一个很好的人选。
