# 你应该给你的爆米花加酱，是的，你应该

> 原文:[https://life hacker . com/you-should-sauce-your-popcorn-yes-you-should-1846811392](https://lifehacker.com/you-should-sauce-your-popcorn-yes-you-should-1846811392)

用干调味料给爆米花调味很容易——你只需摇晃它们。但是有很多味道生活在那个领域之外。这些口味以酱料的形式出现，其中一些也非常适合爆米花。

但是将液体调味料直接倒在一大碗新鲜蓬松的爆米花上，会让它变成一大碗味道不均匀、湿漉漉的爆米花，这可不是你想要的。为了均匀有效地分散你选择的调味料，你必须先搅拌黄油。

通过用黄油(或素食黄油替代品 )乳化你的酱料，你将能够将它分配到整个碗中，而不会无意中产生浸湿的斑点。我们以前曾用这种技术制作过 [酱油黄油](https://lifehacker.com/turn-popcorn-into-a-super-savory-treat-with-some-soy-sa-1784845132) 和 [鱼酱黄油](https://skillet.lifehacker.com/your-melted-butter-needs-fish-sauce-1846328405) ，但它适用于几乎任何*酱油或液体调味料。辣酱是很好的一种——有很多辣酱可以尝试！—但我的新宠是枫糖浆，它读起来像一种甜咸的低能量玉米(显然没有玉米的质感)。*

为了给你的爆米花调味，在一个小锅中用中低火融化两汤匙黄油或黄油替代品，或者把它切碎，放入微波炉中快速加热5秒钟，其间不断搅拌，直到它完全融化。向黄油中加入一茶匙你选择的调味汁，用叉子搅拌。(如果需要，你可以扩大规模，这取决于你做了多少爆米花。将一片爆米花浸入你的混合物中品尝，如果需要的话，加入更多的酱料，确保每次加入时都搅拌均匀，与黄油形成均匀的混合物。淋在爆米花上，搅拌均匀，然后决定下一步你想尝试哪种酱和黄油的组合。我强烈推荐前面提到的鱼露、酱油、枫糖浆和辣酱，但是要搜刮你的调味品，发挥创造力。你也可以*混合*酱料——以我的浅见，枫叶辣酱黄油听起来相当精致。
