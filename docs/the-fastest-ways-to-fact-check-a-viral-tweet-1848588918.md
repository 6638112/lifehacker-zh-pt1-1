# 核实病毒式推特的最快方法

> 原文：<https://lifehacker.com/the-fastest-ways-to-fact-check-a-viral-tweet-1848588918>

人类知识的总和(T1)和数百万人的观点都可以在你的手机上找到。不幸的是，并不是所有分享信息的人都是可靠的。一些人或消息来源可能想要故意误导你，而其他人可能只是出于无知才这么做的。

下一次，当你看到一条充斥着耸人听闻的消息的推特时，你不能相信 T2 是真的，你应该暂停一下。它可能*不*是真的，即使你可能被迫把它发给你的妈妈、朋友、同学，或者——通过简单的转发——你的所有粉丝，请记住，虚假信息广泛传播的后果可能是可怕的。以下是如何快速核实一条病毒性推文的方法。

## **首先，关于媒介素养的说明**

这部分并不是我们承诺的快速解决方案，但是现在做一些工作来加强你的媒体素养会让你在未来更快地识别 suss 内容。坦率地说，媒体素养应该在学校教授，尤其是在我们周围这个科技发达的世界每天都在加速发展的时候。可惜不是。(美国的学校系统有一个 [习惯](https://lifehacker.com/10-lies-you-learned-in-sex-ed-1848517072) 掩盖或者干脆 [忽略](https://www.theroot.com/why-everyone-is-wrong-about-critical-race-theory-in-sch-1847081734) 困难的话题，以防你没注意到。)

就像你成年后不得不重新学习[*真实*历史](https://lifehacker.com/the-real-history-of-labor-day-1847614981) 或 [性教育](https://lifehacker.com/10-lies-you-learned-in-sex-ed-1848517072) 的基础知识一样，你需要自己解决媒体素养问题。幸运的是，学者和其他有头脑的人也注意到普通成年人对他们每天遇到的虚假新闻和与上下文无关的推文准备不足。例如，教堂山的北卡罗来纳大学有一个专门帮助你提高技能的网页。简单来说，你需要做的就是:

*   **了解媒体格局**。一个标题或推文产生强烈的情绪，如愤怒，就是为了做到这一点。如果有一个链接，你最好相信有人想让你点击它，花时间在那个网站上，并帮助他们赚广告费。如果这是一条独立的推文，要明白创作者正在寻找转发，这将为他们带来更多的转发，关注者，参与度，以及这样或那样的金钱或影响力。当你开始考虑你的情绪被操纵来填满别人的口袋或提高他们的地位时，你会觉得不太愿意参与。除了高血压和有点容易受骗的名声之外，玩这个游戏你还能得到什么？
*   **注意语言**。有问题的推文是否有情绪化的词语或明确的观点？来自可靠来源的消息会很直接。我们很快会继续讨论如何谷歌更多的上下文，但现在，练习用一种更平静、不那么愤怒的语气重新构建你正在阅读的内容。没有了哗众取宠，病毒式的推文还能告诉你什么特别值得注意的事情吗？
*   **建立可信新闻来源的个人数据库**。这将需要一些时间，但是花一点时间在，比如说，美联社的主页上。然后对小报做同样的事情 s. 花足够的时间这样做，你会开始看到过度报道新闻的迹象，而没有报道任何值得真正反应的事情。把这种新的视角应用到所有事情上，包括推特，你很快就会成为媒体素养的大师。寻找少一些煽情，多一些语境。
*   **了解你的来源**。作家或专栏作家不同于记者。观点和事实不一样。总是检查谁写了你正在看的推文，并试图找出他们从中获得了什么。如果他们在政界工作，或者在为一个有特殊倾向的组织工作，或者以某种方式受到激励去广泛传播他们自己的观点，当然*和*他们想让他们的内容尽可能地令人发指。在思想的战争中，双方由情感反应决定。

## **想想推特回复中的万事通**

你可能是新闻的普通消费者，一个容易受到令人震惊的信息影响的冷静的搜寻者。然而，有些人是这种东西的狂热爱好者，他们会争先恐后地指出，一个据称显示当今暴力的嵌入视频是多年前的，或者一段引文被错误地归因于某个历史人物。让他们成为面向公众的万事通的热情帮助你，但是记住他们也有某种偏见。

## 阅读 Twitter 是否将其标记为虚假或误导性声明

Twitter 还会对虚假或误导性的声明附加一些通知。我们也可以来回讨论 Twitter 可能存在的偏见，但现在，只需点击通知。我会带你进入一个页面，里面有关于相关推文内容的更多背景信息，这将帮助你形成一个更明智的观点。如果一个声明足够病毒化，Twitter 将在其趋势专栏中也有关于该声明的事实核查资源。

## **谷歌不带偏见语言的报价**

剔除任何带有偏见或情绪化的短语，比如“灌输”或“唤醒暴徒”将其归结为最基本、最公正的形式，然后谷歌一下。

举个例子，看看围绕右翼影响者和媒体所称的“免费吸食大麻”计划的狂热。众议员麦迪森·考索恩和参议员马尔科·卢比奥几周来一直坚持政府希望在低收入社区分发“免费快克烟斗”。

“这种疯狂看不到尽头，”卢比奥在一条 [推文](https://twitter.com/marcorubio/status/1491060747119771650) 中说，这条推文已经获得了数千次参与。当然，这些类型的推文有很多转发和点赞，但几乎没有上下文。这是当你前往谷歌搜索精简的条款。“精神错乱”已经过时了；但是当你搜索“拜登政府破解管道”时，你会发现来自知名来源的事实核查，这些来源调查了那些与上下文无关的推文，并试图解释你应该知道的事情。(这就是你的固体出口个人数据库派上用场的地方。)

如果在主流媒体有机会发布深入的事实核查之前，你碰巧看到了一条病毒式的、与上下文无关的推文，那就尽可能地阅读所有内容。在这个例子中，你会发现来自物质滥用和精神健康服务管理局的 3000 万美元的拨款不是快克管道分配基金，而是为旨在减少药物过量和疾病在吸毒者中传播的减少危害服务分配资金。你还会发现，格兰特成立的消息已经是几个月前的事了，而且在“免费烟斗”的愤怒循环之前，它很少受到关注。

一旦你有了背景，关于“免费裂缝管道”的推文就显得荒谬而短视，即使它们来自高调的民选官员。请记住 每个发布耸人听闻的推文的人都会从操纵你的情绪中有所收获。政客们从激起愤怒和恐惧中获得了什么？投票，一旦他们承诺对他们刚刚让你相信存在的问题做些什么。

然而，得到更好的信息后，你可以自由地成为回复中的万事通之一。玩得开心。
