# 买了这些东西后要洗干净，因为你不知道它们去过哪里

> 原文：<https://lifehacker.com/wash-these-items-after-buying-them-because-you-dont-kn-1846775183>

如果你只是在第一次使用新的家居用品后才清洗它们，你可能会错过关键的第一步。通常，包装或标签一被移除，清洁过程就应该开始。即使它们看起来很干净，在你掏钱用信用卡购买之前，从货架上买下的衣服已经被摸过、试穿过，并被运到工厂各处。锅碗瓢盆可能已经在架子上放了不知道多长时间，在你把它们带进厨房之前会被成百上千的手指触摸过。

### 总是 w 灰新衣服在穿它们之前

尽管马上穿上一套新衣服很有趣，但你应该克制住这种冲动，直到洗衣日之后。除了 [之外，还有几十个人](https://www.huffpost.com/entry/wash-clothes-before-wearing_n_5ac64e0fe4b07a3485e207fc) 可能会在你购买衣服之前试穿——即使你在网上购买，试穿的可能性也不太大——衣服必须从工厂到仓库再到商店、 [可能会在途中沾染细菌](https://www.goodhousekeeping.com/home/cleaning/tips/a26576/germs-in-new-clothes/) 。

但是即使你不担心潜在的病原体，也可能有未知的皮肤刺激物残留在布料纤维中。正如服装制造专家拉娜·霍格告诉《Elle》杂志的那样，“大多数用于织物染色和纱线整理的化学物质都是已知的皮肤刺激物。 " 这就是为什么预洗内衣等贴身衣物尤为重要的原因——将那些化学物质带入与皮肤非常接近——尤其是粘膜——可能会导致潜在的问题。有很多方法可以给你的衣服消毒——比如，你可以在下次洗衣服的时候扔一些白醋和漱口水——以保证第一次穿的时候没有细菌，但是一次标准的洗涤就足够了。所有这些也适用于床单——如果你整晚都要睡在上面，你会想确保它们没有来自工厂的化学物质。

### **清洗毛巾，确保它们真的能让你保持干爽**

出于以上所有原因，你也会想洗你的新毛巾，但你也应该这样做，以确保它们确实能让你在淋浴后变干。 [正如我们之前提到的](https://lifehacker.com/how-to-make-your-old-sheets-feel-like-theyre-from-a-5-s-1846592604) ，w 使用前的灰化毛巾会去除制造过程中的任何残留物——包括特殊涂层，这些涂层是为了让它们在货架上看起来更有吸引力，可能会妨碍它们工作和实际吸水。

### 烹饪前清洗新的锅碗瓢盆和银器

为什么银器、锅、盘和其他厨房用品在第一次使用前应该清洗，这是不言自明的,但重申一下也无妨。T 在把搅拌碗放回架子之前，我们无法知道有多少只手接触过它，所以在用炊具吃东西或喝水之前，你要确保陌生人手里的细菌都被清除了。正如北卡罗来纳州立大学食品安全推广专家本·查普曼、博士在接受 Kitchn 采访时指出的那样，不这样做是“真的，真的风险很低【但是】 电器中可能会有一些塑料或其他材料，如塑料布或仓库中的灰尘”，如果没有其他原因，它们可能会影响你食物的味道。

此外，Silverware 总部的餐具网站解释说，制造商在餐具上涂了一层薄薄的油或保护涂层，这样它们就不会在运输过程中被划伤或损坏为了防止任何人摄入这种涂层(或粘在上面的东西)，在使用前，用洗碗精和水手洗你的炊具和器皿，或把它们扔进洗碗机。同样的道理也适用于可重复使用的 [水瓶](https://delightjar.com/how-to-wash-a-new-water-bottle-and-get-it-ready-to-use/) ，水壶和水罐。

### 预洗婴儿玩具

如果你刚为人父母，你可能一直都是这样做的，但是定期清洗孩子的玩具是最好的做法。当婴儿的免疫系统还在发育的时候，你想尽你所能把有害细菌从他们的嘴里赶出去。无需用刺激性化学物质清洗；用 [温水和少量肥皂](https://howtoadult.com/should-prewash-newborn-toys-27581.html) 快速擦拭会大大杀死塑料上的任何细菌。
