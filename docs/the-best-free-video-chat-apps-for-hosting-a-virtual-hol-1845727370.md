# 举办虚拟假日派对的最佳免费视频聊天应用

几个视频聊天应用程序正在做出重大改变，以便到 [帮助家庭今年举行虚拟假日聚会](https://lifehacker.com/how-to-host-a-virtual-thanksgiving-1845702404) 。例如，Zoom将在感恩节 取消免费用户的 [通话时间限制。免费的Zoom会议通常限制在40分钟内，但您可以在11月26日上午12点至11月27日上午6点(美国东部时间)之间随意拨打电话。](https://lifehacker.com/use-zoom-for-free-on-thanksgiving-all-you-want-1845688838)

Zoom并不是唯一一款为假期做改变的应用。回到9月， [谷歌将](https://blog.google/products/meet/stay-connected-meet/) 的通话时长延长至整整24小时，免费与用户见面。2021年3月31日之前，您可以享受一吨通话时间；在那之后，谁知道我们会在疫情的哪里。

最近， [微软团队推出了变革](https://www.theverge.com/2020/11/20/21579076/microsoft-teams-free-video-calling-web-zoom-competition) ，这样它的免费用户可以举行多达300人的24小时会议。即使你没有应用程序或微软账户，你也可以参加团队会议，因此这种新的无限通话时间可能会使团队成为通过视频电话举行感恩节晚餐的最佳选择——以及任何其他12月份的假期，你也可以庆祝。

也就是说，组队、缩放、会合都不是没有的*只有*免费的 [视频聊天选项](https://lifehacker.com/the-best-alternatives-to-zoom-for-remote-meetings-1842431075) 。许多应用程序允许与大型群组进行无限通话——其中许多你可能已经使用过了。但是，参与者的确切人数会有所不同，有些参与者需要下载软件或创建帐户才能加入通话。

以下是我们最喜欢的完整列表:

*   [**微软团队**](https://www.microsoft.com/en-us/microsoft-365/microsoft-teams/group-chat-software) **:** 24小时通话，最多300人参加。只有主持人需要一个团队帐户；来宾可以从任何设备打开会议链接，而无需安装Teams应用程序或需要Microsoft帐户加入。
*   [**变焦**](https://zoom.us/)**:**通常每次会议 40分钟，但un限定在11月26日。无需应用程序或缩放帐户，就可以从网络加入会议。
*   [**Google Meet**](https://meet.google.com/)**:**24小时会议时长(至2021年3月31日，之后降至60分钟)。免费通话支持100名参与者，但需要一个谷歌账户才能加入。
*   [**Facebook Messenger**](https://www.messenger.com/)**:**通话时长不限，最多50个用户，但需要Messenger app或桌面客户端以及一个Messenger/脸书账号才能加入。
*   [**face time**](https://apps.apple.com/us/app/facetime/id1110145091)**:**不限通话时长，最多可有32名参与者，但该应用仅在iOS、Mac或iPad设备上可用。
*   [**WhatsApp**](https://www.whatsapp.com/?lang=en)**:**最多8个用户的群组不限通话时间，但需要WhatsApp app。
*   [**house party**](https://houseparty.com/):最多8人无限视频聊天时间。你需要一个Houseparty应用程序，但这可能是今年让 [*堡垒之夜*](https://lifehacker.com/how-to-host-video-chats-inside-fortnite-1845726829)痴迷青少年在你家参加虚拟家庭聚会的唯一方法。
