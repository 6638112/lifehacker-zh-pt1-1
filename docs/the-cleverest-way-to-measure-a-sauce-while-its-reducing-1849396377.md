# 最聪明的方法是在调味汁减少的时候测量它

> 原文：<https://lifehacker.com/the-cleverest-way-to-measure-a-sauce-while-its-reducing-1849396377>

如果你有制作美味食谱的习惯，你可能会遇到这样的指示，告诉你“将液体减少一半”或“煨到酱汁减少三分之一。”但是除非你定期把你的调味汁倒入量杯，否则你的调味汁减少的速度和减少的量很难测量。倾斜平底锅可以帮助你观察它，但是那仍然是一个猜测估计。为了更准确地测量正在减少的调味汁，你需要一根木制筷子。

拿一根筷子——外卖里经常带的那种木头筷子——直接蘸到未稀释的酱汁里。你想让棍子完全垂直，你想让它碰到锅底。用手指将指尖放在调味汁的表面来标记深度。把筷子拔出来，然后用铅笔标出你手指尖的位置(这就是为什么你想要一根木制筷子)。

继续煨你的调味汁，偶尔插入筷子，看看调味汁碰到你原来标记的什么地方。如果你的食谱告诉你减少一半，继续煮，直到液体线达到筷子尖和铅笔标记之间的一半。如果上面写着“减少三分之一”，那么煮到筷子下 1/3 处。你明白了。

这一招用适量的调味汁效果最好。如果你在一个大的煎锅里减少少量的调味汁，你的棍子不会有太多的深度，但是如果你在一个小的调味汁锅里处理超过半杯的任何东西，它就像一个梦一样。如果你想变得真正技术化，你可以在开始煨之前预先标记一半(或三分之一)的点，只是为了完全摆脱猜测工作。这看起来似乎有些矫枉过正，但精度是准确、可重复结果的最好朋友。
