# 你会被抓到偷跑，这可不太妙

> 原文：<https://lifehacker.com/youre-going-to-get-caught-skiplagging-and-it-s-not-goi-1849041104>

随着燃料和劳动力成本的上升，再加上前所未有的通货膨胀，今年夏天的机票价格简直成了天文数字。这就是为什么计划现在度假的人们正在寻找任何节省航空旅行费用的方法。在过去的几年中，一种节省机票费用的策略变得流行起来，那就是“跳过”——当你预订一个行程，其中*中途停留*实际上是旅行者真正想要的目的地。这个想法是，与直接飞往那个城市相比，你可以在你的真实目的地*的中间点找到更便宜的转接航班。这是一个聪明的想法；但不幸的是，这不是一个好的。* 



要理解 skiplagging，你需要了解这种做法最流行的供应商，网站[skiplaging](https://skiplagged.com/)。该网站成立于 2013 年，号称有机会利用他们的做法在 [航班上节省超过 100 美元。它也发现自己立即陷入了困境，面临联合航空公司和 Orbitz 的诉讼，这两家公司都对这种做法感到不满——但是诉讼被驳回，网站被允许继续运营。](https://skiplagged.com/about)

虽然航空公司没有赢得法律战，但是他们仍然有能力惩罚那些玩忽职守的人。 [美联航将对屡次参与练习的乘客](https://simpleflying.com/united-trying-to-punish-skiplagging-passengers/) 进行罚款。 [据报道，美国航空公司给一名乘客送去了一张 2500 美元的账单](https://simpleflying.com/american-airlines-skiplagging-warning/) 涉及 52 起逃票案件的鉴定。前者甚至到了 [威胁使用外债催收机构](https://simpleflying.com/united-trying-to-punish-skiplagging-passengers/) 的地步，往好了说会让你大伤脑筋，往坏了说会对你的信用产生负面影响。此外，如果乘客认为这种做法违反了他们的运输合同，他们可能会面临航空公司的禁令。

我坚信走非常规路线来省钱，但前提是这条路线是为了节省一些钱，同时也不会让你的生活变得更困难。暂时不玩滑雪了。相反，通过乘坐 [航空公司的航班来省钱，因为会让你免费使用无线网络](https://lifehacker.com/which-airlines-offer-free-wifi-or-free-messaging-and-w-1848899990) 或者让你 [改变你的航班而无需额外付费](https://lifehacker.com/these-airlines-still-let-you-change-your-flight-for-fre-1848987886) 。
