# 如何去除玻璃窗上的划痕

> 原文：<https://lifehacker.com/how-to-remove-scratches-from-glass-windows-1848084784>

随着时间的推移，你家的窗户可能会发生很多事情——无论是室内还是室外——包括划痕。如果你碰巧注意到玻璃上的一些划痕，你可能甚至不知道它们是怎么来的。(除非你刚刚看到一个蹒跚学步的孩子伸手去拿一把钥匙，然后在一扇窗户上乱涂乱画。如果有，大概就是这样。)



但幸运的是，有一些简单的方法可以去除玻璃窗上的划痕，前提是划痕不要太深。(那样的话，是窗口专业人士的工作。)在 [一篇关于 ThisOldHouse.com](https://www.thisoldhouse.com/windows/21323179/how-to-remove-scratches-from-glass)的文章中，Anne Holub 分享了一些消除抓痕的策略。以下是需要了解的内容。

## 使用透明指甲油

即使你从来没有修过指甲，在家里准备一瓶透明指甲油也很方便，比如去除窗户上的划痕。

在这种情况下，在窗户被刮伤的部分涂上一层薄薄的透明指甲油，然后让它干燥大约一个小时。“指甲油干了之后，用沾有洗甲水的布轻轻擦拭表面，去除划痕以外的多余指甲油，” [Holub 写道](https://www.thisoldhouse.com/windows/21323179/how-to-remove-scratches-from-glass) 。

## 转向牙膏或浮石肥皂

这种方法既可以使用普通的白色牙膏，也可以使用轻度磨损的肥皂，比如液体浮石(或者其他你用来去除手上严重污渍的东西)。首先，清洁有刮痕的窗户部分，然后让它干燥。

Holub 说，然后在柔软的微湿抹布上放少量牙膏或温和的肥皂，用小的圆周运动和轻的压力摩擦抓痕区域约 30 秒。然后擦掉任何剩余的牙膏或肥皂，并检查划痕。如果它们还在，重复这个过程。

## 使用最柔软的钢丝绒抛光

根据 Holub 的说法，虽然钢丝绒可能听起来像是窗户划痕的原因，而不是消除它们的方法，但使用最优质的钢丝绒可以帮助抛光你的划痕。

关键是确保你得到了正确的那种，(那将是 **#0000 钢丝绒**)并从一个新的垫子开始，“因为旧的钢丝绒上的铁锈或灰尘堆积可能会给窗户表面带来新的划痕，” [Holub 解释说](https://www.thisoldhouse.com/windows/21323179/how-to-remove-scratches-from-glass) 。

在抓痕处用细钢丝棉沿圆周方向轻轻摩擦几分钟。然后擦掉任何残留物，检查你的进度。这一次，Holub 不建议在留下划痕的情况下重复这一过程——所以如果是这样，你可能想尝试其他方法。
