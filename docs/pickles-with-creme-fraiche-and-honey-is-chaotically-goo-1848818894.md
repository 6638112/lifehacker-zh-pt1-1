# 泡菜配鲜奶油和蜂蜜非常好吃

> 原文：<https://lifehacker.com/pickles-with-creme-fraiche-and-honey-is-chaotically-goo-1848818894>

昨天，我的亲姐姐悉尼试图公开羞辱我，因为十多年前在伯班克好市多停车场发生的一件事。我年轻，饥饿，还没有被各种各样的样品完全满足，所以我撕开了一块刚买的 Colby jack，咬了一大口，砰的一声把一颗草莓放进了嘴里。对西德妮来说，这既无礼又粗鲁，她一有机会就会提起这件事。

Watch [https://lifehacker.com/embed/inset/iframe?id=twitter-1516464451465060358&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1516464451465060358&autosize=1) 

我后来了解到《料理鼠王》中的老鼠不仅喜欢这种精确的味道搭配，而且它还是电影 中的关键场景 [(我没有看过这部电影)。这证实了我已经知道的事情:我有一种本能的、天生的口味搭配能力，但是它来自一个野性的、混乱的地方。](https://www.youtube.com/watch?v=Yyah49_Oz78)

这可能就是为什么这种泡菜小吃，我在 Instagram 账户(eatsmalfood)上看到的 [，在本能层面上吸引了我。超级酸泡菜、鲜奶油和蜂蜜的组合是来自*朋友*的蛋糕，我是它的乔伊。泡菜？很好。培养乳制品？很好。亲爱的。太好了。(不像《老友记》里的蛋糕，我会把这种点心提供给人们。)](https://www.instagram.com/eatsmallfood/)

 [https://lifehacker.com/embed/inset/iframe?id=instagram-CcgzIyQPAns&autosize=1](https://lifehacker.com/embed/inset/iframe?id=instagram-CcgzIyQPAns&autosize=1) 

根据 eatsmallfood 的说法，这种组合是基于一种“受欢迎的小黄瓜、奶油和蜂蜜小吃”稍微搜索一下就发现还有一种俄罗斯版本的乳酸发酵黄瓜泡菜和 smetana(类似于鲜奶油的俄罗斯酸奶油)。这两个版本似乎都符合我混乱的能量，满足了我对看似不和谐的配对的渴望。我是正确的。

至于我的泡菜点心，我选择了 Trader Joe 品牌的 corni chons——一种非常有攻击性的泡菜——还有一些奶酪和蜂蜜，装在一只熊里。我把牛奶推到一个盘子上，用勺子挖一个小坑，然后把蜂蜜淋到坑里。我擦掉泡菜上多余的盐水，然后将它们在蜂蜜和鲜奶油中旋转，确保每一种都在小小的腌黄瓜上得到足够的量。

像大多数美味、简单的小吃一样(像 [涂黄油的煮鸡蛋](https://lifehacker.com/i-am-begging-you-to-butter-your-hard-boiled-eggs-1841156203) ，或者 [ho ney 烤花生配玉米糖](https://lifehacker.com/how-to-stop-hating-candy-corn-1838855471) )，这种搭配比它应有的任何权利都要好。蜂蜜和泡菜可能看起来有冲突，但是有一整个类型的甜黄瓜(和西葫芦)泡菜，它们很好。这种小吃几乎涵盖了所有的味道基础——甜、咸、酸和脂肪在你的味觉上发挥——和没有一种味道或质地逗留太久。甜味能缓和咸味，脂肪能减轻酸味。它只是工作。

对于我的下一个泡菜点心，我将尝试得到一些思美达和乳酸泡菜，只是为了比较和对比。我甚至会换掉泡菜本身，尝试腌绿番茄、黄瓜或花椰菜。我有一个专用的泡菜冰箱，里面装满了泡菜，正等着蘸蜂蜜和培养奶油。
