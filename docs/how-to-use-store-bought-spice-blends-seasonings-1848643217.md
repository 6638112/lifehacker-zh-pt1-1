# 使用商店购买的混合香料的8种聪明方法

跟我一起说:使用商店里买来的混合香料并不可耻。事实上，唯一可耻的事情是过于势利，不愿接受现成的混合物。混合香料能让任何美味的食物快速入味，加深和完善任何菜肴。当然，你可以自己调配——但是当你赶时间的时候，这些预制的混合物是救命稻草。就拿 [Cavender的希腊调料](https://thetakeout.com/use-cavenders-greek-seasoning-blend-on-everything-1845994558) 来说吧，我的伪希腊最爱。或者香料屋的新款 [黑蒜拉面辣椒](https://www.thespicehouse.com/products/black-garlic-ramen-pepper) ，对十字花科的所有东西都很棒。当你有15分钟时间做一顿饭时，这两者都是真正的天赐之物。

不确定如何处理商店购买的混合物？我们有一些想法。

混合香料是用来烧烤的，不管你是在做肉、豆腐还是蔬菜。如果你想吃蔬菜，就把它们放入橄榄油中，在烤之前撒上大量的调料。如果你正在做牛排或鸡肉，继续摩擦肉的各个面，让它休息一会儿。如果你感觉很有冒险精神，你可以将混合物与玉米淀粉混合，用混合物包裹一些豆腐，然后马上煎炸这些吸盘。

土豆泥很好吃。调味土豆泥是一道神级配菜。来吧，把你选择的混合香料倒入做好的土豆中，搅拌一下。其他土豆菜肴也是如此，包括土豆煎饼和薯条。*你*弄点味道来！还有*你*弄点味道！

这是我个人最喜欢的使用我最喜欢的混合调料的方法。无论你是在抹油、煎炸还是搅拌鸡蛋，都可以在成品上浇上一层从商店买来的混合料，以增加风味。只是在鸡蛋煮熟之前不要加入调料——你不想烧焦那种甜甜的糖果。

一个小小的Dash夫人就能把一个 [沙拉](https://thetakeout.com/the-perfect-pasta-salad-has-one-secret-ingredient-1844666491) 从 *meh* 变成催眠。不要害怕在这里变得怪异——如果你觉得狂野，尝试一点粉状牧场或甚至墨西哥玉米卷调料。如果你还没有在沙拉中加入盐和胡椒，当你准备好和大狗一起奔跑的时候再回来。

盐是旧闻了，宝贝！把一袋 [爆米花](https://thetakeout.com/tips-tricks-toppings-that-will-make-you-a-popcorn-nin-1822383430) 扔进微波炉，用调料混合撒在成品上。这是粉状牧场能够大放异彩的另一个领域，但也有各种其他的可能性。因为大多数美味的调料都含有相当数量的盐，当你准备一碗完美的碗时，你甚至不需要伸手去拿两个不同的调酒器。

复合黄油 非常容易制作:只要将任何有很多味道的东西混合到软化的无盐黄油中。(有没有在高级牛排餐厅见过你的眼唇上涂着美丽的香草黄油？那是复合黄油，任何人都可以在家准备。)只需在一块(或八汤匙)黄油中加入一茶匙调料，混合，然后根据你的喜好调整口味。

这里的可能性真的是无穷无尽的，但一个很好的开始是一些非常美味的东西，如马萨拉咖喱或烧烤香料摩擦。你可以把这些黄油用在烘焙食品上，或者用在肉、家禽、鱼和蔬菜上。它们可以在冰箱里保存长达一周，也可以冷冻几个月。

烟熏奶油奶酪是那些户外吸烟者的下一个大趋势。这太简单了，听起来几乎像个笑话:拿一个8盎司的。一块奶油奶酪，外面覆盖一层混合调料(除了百吉饼调料和麦考密克烧烤调料外，其他东西都很好用)，然后将它们放在铝箔中，在225度的烟熏器上烘烤两到三个小时。它看起来会很奇怪，有裂缝，而且很干——但是一旦你在里兹饼干上尝了一口，你就会明白这有什么大惊小怪的。
