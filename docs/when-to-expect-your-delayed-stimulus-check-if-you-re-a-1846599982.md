# 如果你是一个社会保障的接受者，什么时候期待你的延迟刺激检查

> 原文:[https://life hacker . com/when-to-expect-you-delayed-stimulus-check-if-you-re-a-1846599982](https://lifehacker.com/when-to-expect-your-delayed-stimulus-check-if-you-re-a-1846599982)

最后，经过数周的拖延，估计有3000万有资格获得第三轮救济支票的社会保障领取者将在4月份收到他们的付款。新的时间表影响了那些接受SSI，SSDI，铁路退休，或退伍军人事务福利的人。如果你有资格的话，看看时间表和如何追踪你的付款。

### **大多数项目预计在4月7日前付款**

美国国税局 [周二宣布](https://home.treasury.gov/news/press-releases/jy0087) 他们将于本周末开始寄送支票，并且大部分付款将于4月7日以电子方式发送，要么通过直接存款或现有的 [Direct Express](https://www.fiscal.treasury.gov/directexpress/) 借记卡。一个例外是退伍军人事务受益人，因为国税局预计他们的付款将在4月中旬支付。

然而，一些社会保障受助人已经收到了他们的付款，包括那些去年使用 [非报税工具](https://www.irs.gov/coronavirus/non-filers-enter-payment-info-here) 的人(该工具适用于通常不需要报税的非常低收入者)，或者已经提交了2019年或2020年的报税单。

### 为什么付款会延迟？

第三轮刺激检查的资格最常见的是基于2019年或2020年的纳税申报。然而，根据基普林格T3[的说法，由于许多社会保障和SSI领取者不报税，国税局不得不从监督这些权利项目的社会保障管理局获得资格信息。然而，移交被推迟了将近两周，这导致四名民主党国会议员给SSA 写了一封信，要求他们加快移交付款文件，他们第二天就这么做了。](https://www.kiplinger.com/retirement/social-security/602504/social-security-recipients-to-get-delayed-third-stimulus-checks-after-ssa-sends-data-to-irs) 

### 如何追踪我的付款？

你可以使用国税局的 [获取我的支付工具](https://www.irs.gov/coronavirus/get-my-payment) 来跟踪你的支付，尽管它要到本周末才会为联邦社会保障受益人更新(有关使用该工具的更多信息，请查看这篇 [Lifehacker帖子](https://twocents.lifehacker.com/check-the-status-of-your-stimulus-with-this-tool-1845993388) )。此外，请注意，由于救济立法中的一个怪癖，最新一轮的[剩余支票可以由私人债务催收员](https://twocents.lifehacker.com/can-your-stimulus-be-taken-away-by-debt-collectors-1846487402) 扣押，但不能从过期的 [子女抚养费](https://www.cnet.com/personal-finance/child-support-and-the-third-stimulus-check-let-us-clear-up-the-confusion/) 中扣押(之前的支票就是这种情况)。
