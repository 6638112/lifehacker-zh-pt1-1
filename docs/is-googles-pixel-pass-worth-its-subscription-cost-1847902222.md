# 谷歌的 Pixel Pass 值它的订阅费用吗？

> 原文：<https://lifehacker.com/is-googles-pixel-pass-worth-its-subscription-cost-1847902222>

本周，在宣布新的 Pixel 6 和 Pixel 6 Pro 的同时， [谷歌还推出了 Pixel Pass](https://fi.google.com/about/pixel-pass)——一个将订阅服务捆绑包和上述 Pixel 6(或 Pixel 6 Pro)结合在一起的程序。如果您正试图决定该捆绑包是否适合您，这里有一些事情需要考虑。

## 什么是 Pixel Pass，费用是多少？

Pixel Pass 包括谷歌的内部订阅服务:Google One、Play Pass 和 YouTube Music Premium。它还配有延长保修期和 Google Fi 手机服务折扣。(不过，用户可以选择退出 Google Fi 服务，使用自己的运营商。)Pixel 6 的套餐起价为每月 45 美元，6 Pro 的套餐起价为每月 55 美元。用户可以通过谷歌 Fi 或谷歌商店(如果他们选择退出 Fi 网络覆盖)注册该捆绑包。有一个以旧设备折价的选项，但用户只能用 Pixel 手机注册 Pixel Pass。

Pixel Pass 本质上是谷歌对两个类似的苹果程序的混搭:Apple One 和 iPhone 升级程序。苹果和谷歌的产品价格可能大致相同，或者，根据具体情况，苹果的产品可能会贵得多。 [Apple One 套餐](https://www.apple.com/apple-one/) 每月从 15 美元到 30 美元不等，[iPhone 升级计划](https://www.apple.com/shop/iphone/iphone-upgrade-program) 每月从 35 美元(128 GB iPhone 13 mini)到 74.91 美元(1 TB iPhone 13 Pro Max)不等，这意味着“捆绑包”每月从 50 美元到 104.91 美元不等。

## 像素通道的缺点

取决于用户的优先级，像素通道可能会有一些缺点。这些主要归结为人们希望如何花钱，以及他们对某款手机的承诺程度。

如上所述，Pixel Pass 仅适用于购买两款新 Pixel 手机 之一的 [用户。它确实包括定期升级，很像苹果的 iPhone 升级计划。然而，如果你对使用 Pixel 手机不感兴趣，这个要求就是一个障碍。例如，三星或一加的流行安卓手机就不符合这个项目的条件。](https://store.google.com/us/category/phones?hl=en-US)

此外，客户应该注意这些定期升级需要什么。与类似的项目一样，你实际上是在租用一部 Pixel 手机，当时间流逝时，你可以很容易地以旧换新。一旦手机付清，它就像其他手机一样是你的了。然而，你需要决定是继续订阅该计划并获得一部新的 Pixel 手机，还是取消该计划。

最后一个缺点是偏好问题。一些用户可能不想要或不需要谷歌的所有订阅服务。事实上，Pixel Pass 最终可能会让一些人每年花费比他们购买 Pixel 6 本身或一两个个人订阅更多的钱。例如，虽然他们可能喜欢使用 Play Pass，但他们也更喜欢 Spotify 而不是 YouTube 音乐。

## Pixel Pass 的优点

就个人而言，Pixel Pass 包含的所有服务每月总计 39 美元以上，加上一部新的 Pixel 手机的费用，基准 Pixel 6 的起价为 599 美元。这是细目分类:

*   **Play Pass** (谷歌的优质游戏服务):每月 4.99 美元。

*   **Google One:** 起价每月 1.99 美元，这是谷歌的云存储和技术支持计划。然而，Pixel Pass 附带的 200 GB 计划通常是每月 2.99 美元。
*   **YouTube Premium:** $11.99 每月，包含 YouTube 音乐。这是 YouTube 的一个高级版本，它删除了所有广告，并允许从 YouTube 音乐下载视频和流媒体，这与 Spotify 或 Apple Music 类似。
*   Google Fi: 个人计划每月 20 美元起，由谷歌提供网络覆盖，并有多种覆盖范围和数据选项。Pixel Pass 仅涵盖 Google Fi 计划的 5 美元月折扣，而不是全部价格。

如果你经常使用谷歌的套餐，Pixel Pass 是在购买新手机的同时省钱的好方法。这也让事情变得更加方便——你可以在一笔款项下支付所有费用，包括手机上的每月付款，而不是几笔较小的经常性费用。Pixel 6 本身的价格使得该计划成为一个非常实惠的选择，因为对于许多人来说，支付更少的月费可能比预付 600 美元或更多的新手机更容易。

对一些用户来说，另一个因素是延长保修期:手机的折价价值总是取决于它们的状况，这使得那些容易发生事故的人很难得到一笔好交易。Pixel Pass 可以轻松修复您的 Pixel，所有用户都可以参加新 Pixel 型号的定期升级，即使他们的手机有点粗糙。

对于那些寻找升级的人来说，定期升级也是一件大事。谷歌的 Pixel 手机系列越来越好，一直与苹果最新的 iphone 并驾齐驱。对于 Android 用户来说，除了他们个人运营商可能有的升级程序之外，真的没有其他类似的程序。

## Pixel Pass 是给谁的？

Pixel Pass 主要面向那些享受完整的谷歌生态系统、想要一部新的 Pixel 手机、想要一个类似于 Apple One 和 iPhone 升级计划的便捷订阅系统的人。如果你已经喜欢使用谷歌的订阅服务套件，那么转而使用 Pixel Pass 将很容易获得所有服务的折扣。对于其他所有人来说，这可能是一个*通行证。*
