# 如何停止醒来尿尿

> 原文：<https://lifehacker.com/how-to-stop-waking-up-to-pee-1846242731>

我们都有过这样的经历:昏昏沉沉，半夜意识到，啊，又到了撒尿的时候了。虽然有时这是医疗条件的结果，但通常这只是一个不方便的例行公事。在这种情况下，有一些简单的方法可以让夜间上厕所变得不那么常见。



但是首先，关于医疗问题的警告。如果你担心你的睡眠或膀胱，一定要去看医疗保健专家。有时候，醒来尿尿(他们称之为夜尿症)是因为你总是尿尿太多。这可能有多种原因，从常见的尿路感染到一些严重的糖尿病或心力衰竭。

另一方面，你可能会因为其他影响你睡眠的问题而醒来，然后一旦你醒来，你的大脑会说，好吧，不妨利用这段时间去趟洗手间。在这种情况下，撒尿不是问题，所以如果有什么不对劲，请寻求医疗帮助。否则，这些建议可能会有所帮助:

### 当天早些时候补充水分

通常我们在早上并不太渴，也许我们白天太忙了，没时间喝太多。如果你发现自己在晚上喝了大部分的液体，那就是让你带着满满的膀胱去睡觉。这里的解决方法很简单:在一天的早些时候喝水，这样你就不会发现自己在睡觉前就饱了。

(如果你在一天中追求一个补水目标——比如说八杯水——重新考虑一下这个量的水是否真的对你有帮助。我们中的许多人 [并不像我们想象的](https://vitals.lifehacker.com/do-you-really-need-to-drink-more-water-1734665155) 需要那么多额外的水，所以尝试减少这个数字，看看是否有帮助。)

请注意，我们只是改变了饮酒量的定时，而不是停止饮水和脱水。由于口干或头痛等干扰，脱水也可能导致晚上难以舒适，。

### 不要在睡觉前喝酒

如果你一整天都在补充水分，在睡觉前一两个小时停止喝水应该没问题。这包括咖啡因，原因很明显:你不想让它让你保持清醒。虽然咖啡因因其利尿作用(让你尿尿)而臭名昭著，但如果你习惯了它， [这种作用是非常小的](https://lifehacker.com/four-popular-coffee-myths-debunked-by-science-1780764499) 。

如果可以的话，你不应该在睡觉前喝酒。酒精绝对是一种利尿剂，也是众所周知的扰乱睡眠的。

### 抬高你的腿

睡眠基金会还建议 睡前几个小时你可以试着抬高你的腿，因为当你躺下时，你的身体可以从你的腿上重新吸收一些水。这可以仅仅意味着翘着腿坐在沙发上；你不必做倒立。在你的就寝时间安排中加入一些高质量的休息。

### 让重新入睡变得容易

尽管我们尽了最大努力，偶尔上厕所还是会发生。有时候，问题不在于你必须起床，而在于之后很难再入睡。

所以，尽你所能让你的卧室 [成为一个容易入睡的地方](https://lifehacker.com/how-to-transform-your-bright-and-noisy-bedroom-into-a-s-1172988690) 。使用遮光窗帘来保持卧室的黑暗，或者使用睡眠面罩，这只是对眼球的遮光窗帘。如果噪音困扰你，耳塞或白噪音机器会有所帮助。无论你做什么，不要仅仅因为你醒着就查看你的手机*。*

考虑在晚上调节你房间的温度，因为我们大多数人发现在稍微凉爽的温度下睡觉更容易。在你的床边放一盏小夜灯或者一个小而暗的手电筒，这样你就不用打开任何灯了。(甚至有带 [内置小夜灯](https://www.us.kohler.com/us/Nightlight-–-Lighted-toilet-seats-by-Kohler/article/CNT120400001.htm) 的马桶座圈也正是出于这个原因。)
