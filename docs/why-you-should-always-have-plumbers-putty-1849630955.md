# 为什么你应该总是有水管工油灰

> 原文：<https://lifehacker.com/why-you-should-always-have-plumbers-putty-1849630955>

如果你真的花时间去过当地的五金店，你可能会在货架上看到一种叫做管道工油灰的东西。或者也许你雇了一个水管工来做一些工作，你在他们的工具和材料中发现了它:I 它通常是一种以粘土或面团为基础的化合物，结合一些其他材料(通常是油)来赋予它弹性，被用来创造水密密封，在很长一段时间内保持柔软和柔韧。这实际上是非常有用的东西:它很便宜，如果储存得当，可以持续很长时间，并且在很多情况下都可以派上用场。

## 密封夹具

首先，水管工油灰的主要用途是将[像排水管](https://www.thespruce.com/plumbers-putty-2718686) 或水龙头这样的管道设备密封到表面(工作台面或水槽)。如果有一天你醒来，发现你的厨房水槽开始漏水，用一桶 3 美元的管道工油灰和一把扳手修理它实际上是非常容易的，而不是支付昂贵的管道工账单。方法如下:

1.  通过拧下下面的螺母拆卸排水管
2.  彻底清洁该区域
3.  取一个高尔夫球大小的管道油灰，把它卷成一卷
4.  在排水管周围涂抹油灰
5.  将排水管推回原位，固定，并清除多余的油灰

就是这样！水管工油灰的工作原理是形成一道水无法渗透的屏障。它不能处理高水压，但像稳定泄漏这样的低压问题就在它的手推车里。你可以用水管工油灰封住大多数水管装置周围的水，比如水龙头、厨房和浴室的水槽排水管和淋浴排水管。

与填缝相比，最大的优点是你可以很容易地移除夹具。填缝剂变硬并粘附，使过程变得复杂。此外，当使用管道工油灰时，你可以立即使用你的排水管或水龙头，无需等待它凝固或干燥。

请注意，当你购买新的排水管或水龙头套件时，制造商通常会提供一个垫圈，这样就不需要管道工的油灰了。这些通常工作得很好，但是在手边准备一些油灰以防垫圈在某个时候失效仍然是一个好主意。

## 连接管道

如果你要连接两根管子，你可以用管道工的油灰代替铁氟龙胶带。只需将油灰涂在内螺纹上，然后连接管道，擦掉多余的部分。

同样，这将适用于低压配件，但如果你的管道将有高水压，你会做得更好。但是 i 如果你需要紧急修理而又没有胶带，水管工的油灰很可能足够用来临时修理，直到你可以去五金店或者找专业人员来修理。

## 密封厕所

你的厕所应该密封在地板上——与外面的许多错误信息相反，你的厕所 [应该密封在](https://www.finehomebuilding.com/project-guides/plumbing/should-you-seal-the-joint-where-a-toilet-meets-the-floor) 周围。任何人告诉你留一个缝隙，这样你就能知道是否有漏水，这显然是错误的。

马桶也应该用填缝剂密封，因为将马桶密封在地板上的作用之一是减少移动的机会。由于填缝剂具有粘性，它有助于稳定您的马桶，同时还能阻挡气味和渗漏。然而，如果你正在安装一个马桶，你可以用水管工的油灰在马桶和地板之间形成一个初始的防水密封，然后在上面填缝。你也可以在蜡环周围使用管道油灰，蜡环将马桶密封在地板的凸缘上。这将防止蜡粘在马桶上，这将使清除它变得轻而易举。未来的你或你房子的下一任主人会感谢你的。

水管工油灰的应用范围很窄，但在这个范围内它非常有用。当你的厨房或浴室有漏水的设备时，放一小盆这种东西在身边会很方便。另外，玩起来也挺有趣的。
