# 这是存放吸管杯的最佳方式

> 原文：<https://lifehacker.com/this-is-the-best-way-to-store-sippy-cups-1845287010>

我开玩笑的。这张照片是一个笑话，我不是在这里建议你把孩子的吸管杯塞在两棵树干之间。这似乎是一个只适用于这一个吸管杯的解决方案，当然你有不止一个吸管杯。事实上，你可能有多个不同形状和大小的和塑料水瓶。你有在家里使用的杯子和在车里随身携带的杯子；你已经买了你挥霍的昂贵的杯子和孩子们喜欢的便宜的杯子。这就是为什么*这个*实际上是存放所有杯子和盖子的最佳方式:



这是我们的 [后代脸书集团](https://www.facebook.com/groups/2018785615043946) 成员之一 Julia 的建议，她意识到 [挂鞋组织者](https://www.amazon.com/24-Pockets-SimpleHouseware-Crystal-Organizer/dp/B01D58DRVC/ref=sr_1_1_sspa?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/this-is-the-best-way-to-store-sippy-cups-1845287010&asc_source=&crid=2WDFKKI2WX0C5&dchild=1&keywords=shoe organizer door hanger&psc=1&qid=1601996207&spLa=ZW5jcnlwdGVkUXVhbGlmaWVyPUEySzFXTTNBSjQ1WUlZJmVuY3J5cHRlZElkPUEwNDY1NTkzVFJTWE80VU9ZM0JLJmVuY3J5cHRlZEFkSWQ9QTA3MTE2OTExT0xDRjdaS01FWjdKJndpZGdldE5hbWU9c3BfYXRmJmFjdGlvbj1jbGlja1JlZGlyZWN0JmRvTm90TG9nQ2xpY2s9dHJ1ZQ==&sprefix=shoe organizer door,aps,149&sr=8-1-spons&tag=kinjalifehackerlink-20) 不一定要用*只是用*做鞋。它们也可以用来驯服那些可笑的塑料杯子、水瓶、点心杯、和盖子，这些都是小孩子的父母多年来积攒的。

作为额外的奖励，底部插槽中的杯子伸手可及，因此孩子们可以根据需要拿起自己的杯子。

当然，你也可以使用这样的鞋子整理器来整理真正的鞋子——事实上，它们特别适合小孩子的鞋子，因为你可以在每个插槽中放入一双完整的鞋子(而不仅仅是一只成人尺码的鞋子)。朱莉娅也这么做:

但是，如果你的橱柜里堆满了一堆塑料杯子和碗，那就把其中一个整理袋扔到你的储藏室门后面，享受新的空间吧。
