# 去他妈的那些孩子，自己吃掉所有的糖果

> 原文:[https://life hacker . com/screw-thes-kids-eat-all-the-candy-yourself-1845497824](https://lifehacker.com/screw-those-kids-eat-all-the-candy-yourself-1845497824)

这个月——万圣节之月——已经被塞得满满的绞手和焦虑，其中一些围绕着 [恶作剧- 或- 对待](https://offspring.lifehacker.com/this-map-can-help-you-decide-whether-to-go-trick-or-tre-1845128409) 和*孩子*。人们(理所当然地)关心如何以最安全的方式给孩子发糖果，但他们也担心缺乏恶作剧或款待可能会造成持久的创伤。(我想他们会熬过去的。)而且，尽管这个网站已经想出了一些 [巧妙的变通办法](https://lifehacker.com/make-a-candy-chute-for-safer-pandemic-trick-or-treating-1845229874) 和 [活动](https://offspring.lifehacker.com/have-a-glow-in-the-dark-candy-hunt-on-halloween-1845383100) 让今年的万圣节更安全，我还是会采取一个简单得多的方法，自己吃掉所有的糖果。

Watch

这可能看起来很冷，甚至很残忍，但从来没有一个更好的借口来囤积所有的糖果(甚至是全尺寸的巧克力棒！)，而我，作为一个人，将会利用这一点。我的意思是，作为一名共产主义者，我喜欢施舍，但这真的关系到健康和安全(以及吃掉所有糖果的欲望)。孩子们在他们最美好的日子里都很恶心，我只是不想在疫情时靠近他们，即使我们之间隔着一条 [糖果滑道](https://lifehacker.com/make-a-candy-chute-for-safer-pandemic-trick-or-treating-1845229874) 。(还有，我想吃掉他们所有的糖果。)

我必须承认，我也住在公寓里，所以在囤积巧克力的同时躲避孩子们对我来说非常容易。通常我会去我男朋友家，帮他发传单，但今年我真的会说“去他妈的那些孩子”，然后躲在我的小工作室里。如果你像我的男朋友一样，也住在一个吸引一群索要免费糖果的孩子(你宁愿自己吃掉)的房子里，你总是可以用去年的一个邪恶周hacks来为自己掩饰， [端出一个空碗](https://lifehacker.com/put-out-an-empty-candy-bowl-on-halloween-1839502911) 。它会感觉有点邪恶，但得到COVID 会感觉更糟。我只是尽我所能降低风险(同时吃掉所有的糖果)。
