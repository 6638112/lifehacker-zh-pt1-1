# 如何在不牺牲隐私的情况下使用翻译应用和扩展

> 原文：<https://lifehacker.com/how-to-use-translation-apps-and-extensions-without-sacr-1849087236>

当你想翻译一些东西时，最简单的选择往往是像 Google Translate 这样的免费翻译服务。但是你应该避免将敏感信息粘贴到免费翻译服务中，因为你的数据可能会被公开或用于广告。(一般来说，如果你有高度敏感的信息需要翻译，你应该起草一份保密协议，并聘请一名你可以信任的翻译。)但是如果你想翻译一些不太敏感的东西，同时又能保护你的隐私，可以考虑下面这些以隐私为导向的翻译服务。

## **火狐翻译**

Mozilla 开发了一个名为 [Firefox 翻译](https://addons.mozilla.org/en-US/firefox/addon/firefox-translations/) 的 Firefox 插件，可以自动为你翻译网页。该服务在客户端本地工作，这意味着你的数据不会离开你的设备，它目前支持保加利亚语、捷克语、英语、爱沙尼亚语、德语、意大利语、葡萄牙语和西班牙语，因此它缺少几种语言，但它与它支持的语言配合良好。波斯语、冰岛语、挪威语和俄语等其他语言正在开发中。

## **苹果翻译**

Apple 为您的 iPhone、iPad 和 Mac 提供翻译服务，它可以在阿拉伯语、中文(普通话)、英语、法语、德语、意大利语、日语、韩语、葡萄牙语、俄语和西班牙语之间进行翻译。默认情况下，它使用苹果的服务器，但为了更好的隐私，你可以强制它 [离线翻译一切](https://lifehacker.com/how-to-quickly-translate-text-using-macos-monterey-1847946259) 。

## **维瓦尔第翻译**

[维瓦尔第翻译](https://vivaldi.com/features/translate/) 是在浏览器中烘焙的，支持 108 种语言——比火狐或苹果多得多——并且是为隐私而设计的。您可以决定是要自动翻译每个网页，还是每次都让浏览器询问您是否需要翻译网页。

## **离线翻译应用并不一定更好**

对于关心隐私的人来说，离线使用翻译应用程序并不理想，但总比没有好:你最喜欢的翻译服务可能有离线模式，这意味着你可以断开你的设备与互联网的连接，但仍然使用他们的服务，你的数据不会立即被发送到某个远程服务器。缺点是，一旦你重新上线，没有什么可以阻止这些应用程序将数据发送到它们的服务器。诸如 [谷歌翻译](https://lifehacker.com/google-translates-offline-mode-just-got-a-lot-better-1826890599) 和 [微软翻译](https://lifehacker.com/how-to-use-microsoft-translators-enhanced-offline-langu-1825393990) 等应用提供了很好的离线翻译选择。

## **或者你可以使用专用的翻译设备**

如果你真的需要私人翻译，可以考虑购买专用的翻译设备。大约 100 到 250 美元，你就能找到提供离线翻译支持的好的翻译设备。你可以在亚马逊这样的网站上寻找一些选项，但一定要选择一个支持你需要的语言、支持离线翻译、不需要连接到智能手机的型号。
