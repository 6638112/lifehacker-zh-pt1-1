# 这些是你应该在任何新的MacBook上做的第一个改变

> 原文:[https://life hacker . com/these-is-the-first-changes-you-should-make-on-any-new-1849428528](https://lifehacker.com/these-are-the-first-changes-you-should-make-on-any-new-1849428528)

如果你最近买了一台新的Mac电脑，你不必花几天时间来根据自己的喜好进行设置。无论是去工作、上学，还是其他任何地方，遵循这八条建议会让你的新笔记本电脑准备好迎接你扔过来的任何东西。

## 使用迁移助手从备份中恢复

如果您要从旧Mac迁移到新Mac,“迁移助理”是您最好的朋友。它出现在初始设置过程中，可让您从iCloud或Time Machine备份中恢复所有数据，这比手动将文件从一台Mac移到另一台Mac要容易得多。或者，你可以使用优秀的第三方备份软件 [对你的旧Mac做一个拷贝，恢复到新电脑](https://lifehacker.com/the-five-easiest-ways-to-take-a-complete-backup-of-your-1848778306) 。

## 跳过苹果应用商店

很多人犯了一个错误，在安装了一台新的苹果电脑后就去了苹果应用商店，而没有考虑到在苹果的主要应用市场上有很棒的应用软件。在苹果的围墙花园之外，有*吨*有用的第三方应用，但是你不需要一个一个下载它们。为了帮助您开始，请查看我们的指南，了解在新Mac 上批量安装应用程序的最简单方法。

## 拥抱聚光灯搜索(或更好的东西)

如果你想在Mac上找到任何东西，Spotlight Search是一个不错的选择:只需按下 p 按下 **Command + Space** ，然后寻找应用程序、文件或文件夹，甚至尝试计算或货币兑换。为了更好的替代， [阿尔弗雷德和瑞卡斯特都是优秀的](https://lifehacker.com/how-to-disable-spotlight-search-on-mac-and-replace-it-1847227585) 。

## 更改您的 Safari设置

从Chrome切换到Safari将延长Mac的电池寿命，这可能是在MacBook上使用它的足够理由。( Safari总体来说还是不错的，但是 [改几个设置就能让 it 更好](https://lifehacker.com/13-safari-settings-you-should-change-to-make-a-better-b-1847236951) 。)然而，它没有其他浏览器一样的扩展级别。如果这是一个障碍，试试这个浏览器，它可以让你安装Chrome和Firefox插件。

## 安装音量控制应用程序

你的Mac不允许你为每个应用程序设置自定义音量控制。这意味着每个应用程序都将被设置为相同的系统音量，除非应用程序有内置的音量设置。但是第三方应用可以让你降低某些应用的提醒音量(比如说 Messages或者Slack)而不用调整其他来源的音量。要试用， [下载免费应用BackgroundMusic或尝试付费音源](https://lifehacker.com/how-to-individually-control-audio-volume-for-every-app-1847455401) 。

## 定制您的右键菜单

如果你不想让和其他人一样使用默认的右键选项，你可以更改选项来删除不必要的动作，并优先处理你经常使用的动作。 [下面是你如何做到](https://lifehacker.com/personalize-your-macs-right-click-menu-1821787923) 。

## 自动清空垃圾桶

如果你一直在删除文件，并且发现自己每隔几天就要清空垃圾箱，你可以让它自动化。macOS让你 [自动删除超过30天](https://lifehacker.com/how-to-automatically-empty-the-trash-on-your-mac-and-w-1848244468) 的文件，如果你不想每次都手动释放空间，这是很棒的。打开Finder，导航到 **Finder >首选项>高级**，勾选**30天后从垃圾桶中移除物品**，将您的 macOS设置为自动清理垃圾桶。在启用功能之前，只需确保为您的Mac使用良好的文件备份服务，以避免意外丢失数据。我们也有几个 [优秀的Finder建议](https://lifehacker.com/customize-your-macs-finder-so-it-shows-you-the-things-y-1848108213) 让Mac的文件浏览器变得更好。

## 充分利用您的触控板

手势在macOS上运行得如此之好，以至于你可能觉得没有必要买鼠标。如果你是MacBook触控板或的新手，想要一些指点，请查看我们列出的 [最佳触控板设置](https://lifehacker.com/10-macbook-trackpad-gestures-that-save-you-time-1788778916) 。手势将为你节省大量时间，允许你在桌面之间滑动，显示所有打开的窗口，以及其他有用的快捷方式。如果你*确实*在Mac上使用鼠标，你仍然可以使用 [这个免费实用程序](https://lifehacker.com/this-app-makes-your-cheap-mouse-work-better-than-trackp-1848416099) 。
