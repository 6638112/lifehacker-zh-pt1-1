# 玉米粉、粗燕麦粉和玉米粥的区别

> 原文：<https://lifehacker.com/the-difference-between-cornmeal-grits-and-polenta-1848379579>

我吃很多玉米和玉米制品。我喜欢玉米面包、玉米砂锅、玉米面脆皮鲶鱼、玉米粉和玉米粥，并认为后两种是独特的菜肴。就像色情文学和情色文学一样，很难区分玉米粥和粗燕麦粉，但是我可以一眼就分辨出哪一种是(或者味道)。

## 玉米面是什么？

玉米糁和玉米粥从技术上来说都是玉米粉——玉米被干燥和研磨后制成一顿饭——但是我不会从烘焙通道里拿起一袋商业玉米糁并试着把它做成一碗玉米糁。玉米粉可以是粗的、中等的、细磨的，可以由黄色、白色、蓝色或红色的玉米制成。它主要用于烘焙(玉米面包)、搅打(鲶鱼)和油炸(hushpuppies)，常见于披萨和 [酸面团](https://lifehacker.com/what-to-use-if-you-run-out-of-cornmeal-when-you-re-baki-1848373728) 的底部，可以防止面团粘在锅上，并提供一点质感。

像大多数谷物一样，你可以买到两种形式的玉米粉:普通的和石磨的。根据 [食者](https://www.eater.com/2019/9/4/20847254/whats-the-difference-grits-polenta-cornmeal) 的说法，这一切都归结于病菌:

> 传统玉米面——杂货店货架上的大多数东西——是“脱胚的”，这意味着外壳和胚芽已经从玉米粒中去除；这产生了具有某种均匀质地的耐储存产品。另一方面，石磨玉米粉是全谷物；它仍然有外壳和富含油的胚芽，这使得它比普通的东西更容易腐烂(和手工制作)。

粗糙的石磨玉米粉在技术上可以用来做一碗粗燕麦粉，尽管这不是传统的用法，对我来说也不像粗燕麦粉(稍后会详细介绍)。精细和中等的玉米面最适合烘焙玉米面包(或玉米面蛋糕)，而粗糙和中等的传统玉米面完全可以用来制作披萨饼皮。

## 登特还是弗林特？

玉米粉和玉米粥都是由玉米制成的，比我们从玉米棒上啃下来的东西甜。(我的曾祖父更喜欢玉米地，抱怨杂货店的东西“太甜了”)

虽然被认为是南方的一道菜，但是早在任何殖民者到达这里之前，美洲土著人就已经在吃玉米粉了。粗玉米粉是由粗磨的凹痕玉米制成的，它的名字来源于玉米仁上的小凹痕，低糖高淀粉。这种淀粉在粗磨的玉米烹饪时释放出来，给这道菜一种不添加乳制品的奶油般的口感(尽管你当然可以添加黄油和奶酪)。粗燕麦粉可以用黄色或白色的玉米制成。

另一方面，玉米粥起源于意大利，传统上由黄色燧石玉米制成。燧石玉米因其质地而得名(“硬如燧石”)，比凹痕玉米含更少的软淀粉。这给了玉米粥一种更坚韧、更有嚼劲的质地，比传统的粗燕麦粉煮出来的奶油少一点。

## 你真的不知道吗，兄弟？

在我看来，真正的玉米粉是由浸泡在碱性溶液中的玉米制成的(这一过程被称为蒸煮)。我们之前已经讨论过玉米浸泡，但是现在来回顾一下:将玉米浸泡在石灰、碱液或其他碱性液体中，会给玉米带来一种坚果的、令人兴奋的香味，同时使它更有营养。

在他的书*美丽的玉米*的摘录中，Anthony Boutard 解释了 nixtamalization 如何使玉米中的某些营养成分更易获得:

> 将谷物浸泡在碱性溶液中使其更易消化，最重要的是，更有营养。在未经处理的玉米中，它含有的烟酸(维生素 B3)与一种大分子结合在一起，这种大分子在我们的肠道中不会分解。碱性处理分裂出这种分子，使烟酸在人类消化道中可用。此外，浸泡在石灰中的玉米比生玉米含钙量更高，这在哥伦布发现美洲大陆之前的文化中尤为重要，当时没有任何产奶动物以牛奶和奶酪的形式提供这种必需的矿物质。

这种蓬松的超玉米(在美国通常被称为“玉米粥”)通常被作为一种配菜食用，用来制作玉米糊，磨成玉米粉圆饼或玉米粉蒸肉，或者磨碎煮熟制成粗玉米粉。

虽然从技术上讲，你可以用未经蒸煮的玉米制作玉米粉，但我不明白这有什么意义。玉米糁有一种更甜、更坚果、更有层次的味道，我坚信非玉米糁是玉米糁“T2”名声的原因。(想从头开始做自己的玉米粥吗？ [查看我们的向导](https://lifehacker.com/how-to-make-grits-from-fresh-hominy-1845408529) 。)

## 认为是“菜”，而不是“配料”

准备工作也在一碗糊状煮玉米片应该被认为是粗燕麦粉还是玉米粥中扮演着重要角色。除了 Nixtamalization 和 corn type 之外，将每一种都视为一种实现的菜肴，而不是一种单一的配料(即使从技术上讲它们都是)。

由于软淀粉含量高，粗磨粉通常在水中烹制，并用黄油、盐和奶酪调味。它们也可以用作虾和玉米粉的基料，烤成砂锅，或者 [在培根油中炸](https://lifehacker.com/the-best-thing-to-do-with-leftover-grits-1844377998) ，但它们也可以作为令人满意的美味配菜而不需要太多的折腾，*只要它们是用玉米制作的*。

玉米粥通常在高汤或牛奶中烹制，后者使菜肴具有奶油般的口感，弥补了燧石玉米中缺乏软淀粉的不足。它也可以与香草和奶酪一起煨，然后盛在碗里或让它凝固。一旦凝固，玉米粥可以切片和油炸。(你也可以买一管结实的、预煮好的玉米粥，准备油炸。)

你有很多选择，这就是我要说的，你可以随意处理这些信息。没有人会阻止你煮玉米汤或牛奶，也没有人会指责你把玉米汤和虾一起食用。意大利人和南方人都有一定程度的 T2 学究倾向，但是你怎么煮玉米粥是你的事。(只是不要告诉我你的粗燕麦粉加糖了。或者用即时类。这些决定是你和你的目标之间的 T4。)
