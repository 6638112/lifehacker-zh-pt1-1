# 不安装任何东西如何试驾 Windows 11

> 原文：<https://lifehacker.com/how-to-test-drive-windows-11-without-installing-anythin-1847527997>

你现在可以在你的网络浏览器上演示 Windows 11，这要感谢由一位足智多谋的开发者“蓝色边缘”创建的新网页

Watch

网页让任何人都可以直接看到 Windows 11，即使你的 PC 不符合 [操作系统的严格硬件要求](https://lifehacker.com/what-is-a-tpm-and-why-do-you-need-one-to-run-windows-11-1847186073) ，也不必 [在你的 PC 上安装一个未完成的 Windows 11](https://lifehacker.com/how-to-go-back-to-windows-10-if-you-regret-installing-w-1847283558) 测试版。我在 Chrome 和 Edge 上试用过，效果很好，所以它应该可以在任何浏览器上使用。

现在，要明确的是，这不是 Windows 11 的全功能再造，也不是运行该操作系统的远程桌面。网页实际上只是真实事物的(非常令人信服的)模拟——尽管互动有限。例如，你可以打开开始菜单、搜索工具、Edge 浏览器和 Windows Store，但这些仅仅是令人信服的模型，带有一些互动部分来销售效果。

右下角的任务栏图标也能准确地响应系统的时间、日期、电池和互联网连接状态，但将鼠标悬停在图标或快捷方式上不会显示通常在 Windows 11 中出现的工具提示文本。

Windows 11 经过彻底改革的文件浏览器也无法在 Blue Edge sim 卡上运行——至少目前是这样。打开文件资源管理器将打开一个新的文件夹窗口，但它只说“即将推出”尽管如此，它还是表明了 Windows 11 更时尚、更圆润的应用程序和文件夹窗口的实际效果。

然而，除了这几个模型之外，页面的大部分都是默认 Windows 11 桌面的纯视觉表示。

通知中心和新闻窗口小部件是静态图像，而不是交互式元素，当您单击回收站或设置菜单图标时，或者当您尝试显示隐藏的任务栏图标时，什么也不会发生。“开始”菜单中的大多数“应用程序”也是如此。

然而，一些人会打开 Blue Edge 社交媒体档案的新页面，Github 图标会打开 React 项目页面 中的 [Win 11，供那些想了解更多信息的人使用。](https://github.com/blueedgetechno/windows11) 

尽管如此，React 页面中的 Win 11 是对 Windows 11 外观和行为的最准确的描述，任何对下一版本 Windows 感兴趣的人都应该尝试一下。这不是全面的 Windows 11 体验，但这比升级你的电脑来安装一个未完成的测试版要容易得多。
