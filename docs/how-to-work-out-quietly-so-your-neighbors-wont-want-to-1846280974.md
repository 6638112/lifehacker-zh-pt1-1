# 如何安静地工作，这样你的邻居就不会想杀你

> 原文：<https://lifehacker.com/how-to-work-out-quietly-so-your-neighbors-wont-want-to-1846280974>

在家锻炼很方便，但肯定会很吵。在健身舞蹈课上，你可以尽情地跳跃和跺脚，如果你在合适的健身房，甚至可以 [放下杠铃](https://vitals.lifehacker.com/sometimes-its-okay-to-drop-the-weights-1837481047) 。但是在家里，你楼下的邻居或者熟睡的小孩可能不会同意。相反，根据你的环境来调整你的锻炼。



### 寻找“低影响”的锻炼

许多锻炼视频都标有“低冲击”的标签，以更好地适应那些跑步或跳跃时没有疼痛的人的膝盖。但由于它们主要是消除有弹性的动作，当你想原地踏步而不是跳上跳下时，它们也很有用。

一些舞蹈和 HIIT 视频会提供低冲击选项作为锻炼的一部分(例如，跨进和走出弓箭步，而不是跳跃弓箭步)。一旦你看到了一些典型的修改，你可以把它们带到你看的任何视频中。

### 在铺有地毯的房间里锻炼

如果你的房间里铺满了地毯，那么通常会垫上额外的填充物。这些层有助于消除你的脚步或哑铃在地板上发出的声音。

不管有没有地毯，也可以加地垫。在地毯上，结实的橡胶垫可以给你一个坚实的立足点。如果你必须在硬木地板上锻炼，柔软的泡沫垫可以吸收一些冲击和噪音。

### 戴耳机

我们通常认为耳机是在健身房佩戴的东西，但你也可以在家里使用它们进行锻炼。蓝牙耳机特别适合跟随有音乐或大声的教练为你加油的视频。

如果你有一个正在睡觉的婴儿或伙伴在隔壁房间，也要研究他们的听觉环境。设置一个白噪音机器，让他们屏蔽掉你仍然发出的任何轻柔的声音。

### 穿袜子

即使你试图保持安静，当你做运动时，也很容易忘记自己，在地板上跺脚。不过，穿袜子时，你会有一些触觉反馈。没有运动鞋柔软的鞋底，你自然会走得更安静。

### 找到一个柔软的地方放下重物

举重相当安静；放下它们会很吵。所以，如果你从地上拿起一样东西，集中精力尽可能轻轻地放下它。

然而，当你放下重量开始下一项运动时，很容易忘记保持安静。所以在两次练习之间，指定一个地方放下你的重量。这可能是一个折叠起来的瑜伽垫，甚至是一个装着枕头的牛奶箱。养成这个习惯，你将消除意外锻炼噪音的最大来源之一。
