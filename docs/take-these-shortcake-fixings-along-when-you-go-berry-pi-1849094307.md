# 当你去摘浆果时，带上这些酥饼配料

> 原文：<https://lifehacker.com/take-these-shortcake-fixings-along-when-you-go-berry-pi-1849094307>

一年一次，我和我的朋友阿曼达去摘草莓。(你可能认识阿曼达；她写的是 [酱菜](https://lifehacker.com/7-perfect-pickles-you-must-keep-on-hand-at-all-times-1848161356) 和 [聚会](https://lifehacker.com/you-can-make-ultra-impressive-party-food-without-spendi-1848250408) 。弯腰在树叶中翻找，全身发痒地寻找完美、鲜艳的红色浆果，这听起来可能不好玩，但它有一种冥想的味道，我确实喜欢和朋友聊天的那部分。不过，最重要的是，我享受奖励。不是我们装进汽车的草莓，尽管那是一种奖励，而是我们辛苦工作后吃的代用草莓酥饼。



酥饼很简单，而且不是真正的酥饼。这是从商店买来的天使蛋糕，上面放着刚采摘的浆果和罐装鲜奶油。我不能把这种纯粹的夏季天才的想法归功于我，因为这都是阿曼达的主意，但我确实做出了贡献。我提供蛋糕和奶油，阿曼达带来小纸船和餐具。我们尽最大努力掸掉浆果上的灰尘，然后做好甜点，在太阳开始落到树后的时候大吃起来。这是最完美的方式来强调采摘浆果的体验，而且花费不到 10 美元。

当然，你可以对其他浆果使用这种策略。你可以用它采摘树莓，或者黑莓。(尽管采摘黑莓通常不那么田园，因为黑莓喜欢生长在空地和路边。)对桃子也很管用(带把小刀就行)。

你也可以从蛋糕和奶油中扩展开来，根据任何种类的产品定制你的服装。你可以带一个小奶酪盘，和刚摘的苹果或梨一起享用，或者——如果你是一个喜欢摘番茄的番茄狂——抓一些小的，用小奶酪和一点香醋吃，或者也许带一个切菜板，一瓶蛋黄酱，和一袋白色三明治面包做一个户外番茄三明治(对我来说，这就是行动)。
