# 如何在Android上安排短信

> 原文:[https://life hacker . com/how-to-schedule-text-messages-on-Android-1831323365](https://lifehacker.com/how-to-schedule-text-messages-on-android-1831323365)

无论你的朋友或家人的工作和生活时间表与你不同，还是你发现自己发送的“迟来的生日快乐”消息比你希望的多，提前安排短信是一个非常有用的功能。

Watch

在Android中安排短信曾经是一个罕见的功能，但现在有多个应用程序可以让你起草消息以便稍后发送，甚至是在现有的Android短信应用程序上。

### **在三星智能手机上安排文本**

三星Galaxy和Note智能手机用户可以使用这些智能手机上预装的默认三星短信应用程序来安排短信:

1.  打开三星短信应用程序
2.  起草你的短信
3.  轻按文本栏附近的“**+”**按钮，或屏幕右上角的三个点来打开日历
4.  选择日期和时间
5.  点击“**发送“**至日程安排。

### **在Android信息中安排文本**

大多数非三星Android手机可以在默认的Android Messages应用程序中安排文本。我们 [在去年年底安卓](https://lifehacker.com/how-to-schedule-texts-on-android-1845576310) 进入测试时，报道了如何安排和编辑安卓信息中的文本，但该功能现在可以在所有运行安卓7.0或更高版本的手机上使用。谷歌制作了一个方便的动画，展示了Android消息中文本调度的工作原理，但你可能需要更新你的系统和应用程序才能使用它。

### **其他让你安排短信的应用**

如果你的手机不能使用三星短信，你也不能使用Android Message的新调度工具，Telegram和 [脉冲短信](https://play.google.com/store/apps/details?id=xyz.klinker.messenger&hl=en_US&gl=US) 是两个可以替代的选择。

#### **如何在电报中安排文本**

1.  打开Telegram，与您正在为其安排信息的人开始聊天。
2.  像平常一样输入信息。
3.  要安排日程，长按**“发送”**打开发送选项。选择**的“日程消息”**
4.  选择您想要发送消息的日期和时间。
5.  点击**“发送”**确认。一旦时间到了，你的短信会自动发送。

如果您想要编辑或删除预定的电报文本，请打开对话并轻按日历图标以查看所有未发送的消息。轻按一个按钮来编辑信息、调整日期或删除信息。

#### **如何在脉冲短信中安排文本**

1.  在Pulse SMS中，轻触橙色的 **"+"** 按钮，然后选择您希望发送短信的联系人。
2.  在对话窗口中，轻按右上角的三个垂直圆点图标，然后轻按“**安排信息。**
3.  使用日历界面设置日期和时间
4.  起草你的信息。完成后，轻按“**将**添加到日程安排中。

要查看、编辑或删除Pulse SMS中的预定消息，请在侧边栏菜单中选择“预定消息”。选择对话，然后长按未发送的信息。选择“**编辑**编辑文本，或“**删除**删除预定信息。

### **使用IFTTT安排文本**

这种安排短信的方法比通过Pulse SMS直接安排短信稍微复杂一点，但如果你不想换成一个全新的SMS应用程序，它就能完成工作。

[这个IFTTT小程序可以用来在谷歌日历事件被触发时安排一条文本消息](https://ifttt.com/applets/251413p-schedule-sms) 。唯一的要求是一个IFTTT帐户链接到必要的应用程序(如果你还没有链接到正确的帐户/应用程序，IFTTT将请求许可)。只需使用上面的链接，并按照屏幕上的指示来设置消息。

### **使用闹钟或日历提醒**

这最后一个建议只是一个折中的办法，因为它要求你坚持到底，自己发送信息，但是有一个预先写好的信息总比没有好。

虽然您不能通过Google日历直接安排和发送短信，但您仍然可以使用它来设置事件提醒，以便在特殊场合发送短信。在设置提醒时，你甚至可以在“笔记”部分写下你想发送的文本草稿——这样，当最终发送时，你可以复制并粘贴文本。
