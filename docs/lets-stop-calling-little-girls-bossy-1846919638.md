# 我们不要再说小女孩“专横”了

> 原文：<https://lifehacker.com/lets-stop-calling-little-girls-bossy-1846919638>

我们本能地给我们的孩子贴了很多标签，但它们并不都是好的。我们看到不良行为，我们会以同样的方式指出，这些行为在我们小时候经常被指出: [*不要那么固执*](https://offspring.lifehacker.com/lets-stop-calling-kids-stubborn-even-if-they-are-1846885929) ，我们说。 [*他这么挑食*](https://offspring.lifehacker.com/stop-calling-kids-picky-eaters-1845454678) ，我们呻吟。 [*哦耶，她真的很害羞*](https://offspring.lifehacker.com/lets-stop-calling-kids-shy-1834778014) ，我们借故。也许是因为我们不想让他们成长为固执的成年人，不吃任何蔬菜，在群体环境中不能坚持自己的立场，但给他们贴上标签并不是避免这种情况的方法。另一个我们需要从语言中剔除的是*专横——尤其是当我们谈论小女孩的时候。*

我喜欢人们说小女孩“专横”，就像我喜欢他们说职场女性“情绪化”一样根据我的经验，如果一个男性在指挥别人(不管是在操场上还是在会议室里)，他会被认为是一个强有力的领导者，一个大男人。当一个女性想要通过直接或表现自信来领导时，她就是专横——这个词很快会被女孩们内化为一种带有负面含义的特征。

不，我们不希望一个小女孩告诉所有其他孩子他们能玩什么和怎么玩，但我们绝对*不希望*发出这样的信息，即我们更希望他们保留自己的意见或偏好。我们不希望他们在成长过程中认为，为了讨人喜欢，他们需要变得更安静，更少主张。

幸运的是，你不需要叫你的孩子“专横”来让他们停止专横。有更好的方法来纠正这种行为，阿什利·奥尔蒂斯为 [PopSugar 家庭](https://www.popsugar.com/family/what-to-say-to-kids-instead-bossy-48233607) 写道:

> [玛茜]贝格尔博士鼓励看护者大声说出孩子在做什么，而不是用“专横”这个词来描述一种行为。例如，如果孩子在没有其他人参与的情况下主导游戏，你可以说，“我听到你给出了很多方向。问问你妹妹对游戏的看法，”或者“我看你只想玩你脑子里的想法。把你朋友的想法也包括进来也很重要。贝格尔博士举的另一个例子是这样说的，“我听说你想坐在那张椅子上，但是你哥哥已经坐在那张椅子上了。”当你的孩子坚持让事情按他们的方式发展时，你为什么不在餐桌上另选一个位置。

把标签留给好的东西——强壮、有创造力、有爱心——因为孩子们倾向于相信他们就是我们所说的那个人。
