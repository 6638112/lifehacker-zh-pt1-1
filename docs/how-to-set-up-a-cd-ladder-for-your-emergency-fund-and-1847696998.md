# 如何为你的应急基金建立一个 CD 阶梯(以及你为什么要这么做)

> 原文：<https://lifehacker.com/how-to-set-up-a-cd-ladder-for-your-emergency-fund-and-1847696998>

这种金融举措需要一些工作，但它可能是在高通胀时期最大化应急基金利息的最佳方式:它被称为[CD ladder](https://www.investopedia.com/terms/c/cd-ladder.asp)——一系列在不同时间到期的重叠存款单，因此你有稳定的现金流，仍然可以利用长期利率。它是这样工作的。



## 什么是存款单？

存款证实际上很简单:你把一笔钱交给银行，交换条件是在 3 个月到 5 年的固定期限内不提取存款。作为交换，你可以锁定该期限的年利率(期限越长，利率越好)。

定期存款的优势在于，它们被认为是一种安全的存钱方式，而且它们通常比支票、储蓄和货币市场账户获得更多的利息(目前，APY 的利率接近 1%，而 APY 为 0.50%)。CDs 的不利之处在于，提前取款会产生费用，这基本上会抹去你累积的所有利息。

## 好吧，但是什么是 CD 阶梯呢？

定期存单是一种储蓄策略，通过这种策略，你可以将钱投资于多个到期日交错的长期定期存单，这样你手头就有了稳定的现金流，而无需提前取钱。例如，如果有 10，000 美元，CD 阶梯将如下所示:

*   2000 美元一年期的定期存款
*   两年期存款 2000 美元
*   2000 美元的三年期定期存款
*   2000 美元的四年期定期存款
*   2000 美元的五年期定期存款

然后，当定期存款到期时，你将这笔钱存入五年期定期存款，如下所示:

*   2000 美元(外加一年利息)的五年期定期存款
*   2000 美元(外加两年利息)的五年期定期存款
*   2000 美元(外加三年利息)的五年期定期存款
*   2000 美元(外加四年利息)的五年期定期存款
*   2000 美元(外加五年利息)的五年期定期存款

通过这样做，你可以最大化长期利率，同时给自己一些灵活性，在需要时使用现金，因为 CD 每年都会到期。(9 月最佳存款利率， [查看此 Bankrate 帖子](https://www.bankrate.com/banking/cds/cd-rates/) )。

这有一些风险，因为你不会有所有的现金可用，除非你想提前套现，失去本金的利息。出于这个原因，你可能只把紧急基金的*部分*放入 CD 梯。

## 我为什么不把那笔钱投资股票呢？

几个原因:风险和流动性。投资股票总有赔钱的风险，当你遇到紧急情况时，以有利的价格卖出股票并不总是容易的。另外， [除了你的投资，你总想手头有点现金](https://lifehacker.com/how-big-your-emergency-fund-should-be-1825319181) 。

## **底线**

虽然定期存款的流动性不如储蓄账户(储蓄账户允许无罚款提款)，但它会赚取更多的利息 。当然，现在利率非常低，所以它可能只会给你的现金储备增加几百美元，但总比什么都没有强。你可以用这个 [CD 阶梯计算器](https://www.ally.com/bank/cd-ladder/quick-calculator.html#local=the-basics) 到处玩玩，看看对你来说值不值。
