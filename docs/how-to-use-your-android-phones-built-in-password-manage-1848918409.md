# 如何使用 Android 手机内置的密码管理器

> 原文：<https://lifehacker.com/how-to-use-your-android-phones-built-in-password-manage-1848918409>

Android 设备有很多功能可以让你的数字生活更容易导航，其中最有帮助的就是密码管理器。



密码管理器不仅可以 [存储你的登录信息，以便顺利登录](https://lifehacker.com/how-to-get-started-with-a-password-manager-1846890484) ，它们还可以让 [更容易地为你使用的每个应用程序或网站创建强大、唯一的密码](https://lifehacker.com/your-passwords-are-worse-than-you-think-and-the-easies-1848874581) 。这一点很重要，因为黑客有很多绕过弱密码的策略，如果有人窃取你的登录信息，重复使用密码会危及多个帐户。

虽然我们通常建议使用高评级的第三方密码管理服务 ，但在 Android 上使用该服务的好处是它是免费的，内置在你的设备中——你不必购买或下载额外的应用程序来使用它。它还可以方便地将您的加密密码信息与您的 Google 帐户关联起来，因此无论您在哪里使用 Google 帐户登录，都可以访问您保存的登录信息。谷歌还会监控你的密码，如果有泄露的话会提醒你。

## 如何在 Android 上启用谷歌密码管理器

1.  在你的安卓设备上，进入**设置>隐私>谷歌**的自动填充服务，或者使用设置菜单中的搜索栏找到“自动填充”
2.  启用**“使用谷歌自动填充”**
3.  确保您登录了正确的 Google 帐户。
4.  从自动填充菜单列表中点击**“密码”**打开密码管理器菜单。
5.  如果你想让你的设备保存你在 Chrome 中使用的应用程序和访问的网站的登录信息，点击右上角的齿轮形设置图标，然后打开**“提供保存密码”**。请注意，谷歌的密码管理器不会保存在 Firefox 或 Edge 等其他浏览应用程序中使用的密码，但这些应用程序也提供了自己的密码管理功能。

启用这些设置后，每当您在应用程序或网站中创建新帐户时，您的手机都会建议您使用新密码，并将其安全存储。但是，您可以启用一些其他可选设置。

*   如果你想让登录变得更容易，打开**“自动登录”**，你的手机会在你被提示时自动输入保存的用户名和密码。
*   为了增加安全性，启用**“密码提醒”**以便在您的任何密码在线被发现时得到通知。这就像使用 [来检查泄露的密码，但谷歌会不断自动检查，并在登录受到威胁时提醒你。](https://haveibeenpwned.com/)

## 如何使用谷歌密码管理器管理存储的密码

随着谷歌密码管理器现在在您的设备上启用，您可以在谷歌密码管理器菜单中搜索，查看和编辑保存的密码。所有存储的登录信息都显示在屏幕底部的列表中。

轻按列表中的一个网站或应用程序来查看帐户信息和当前存储的密码(您需要提供锁定屏幕的 pin、密码或指纹来访问此菜单)。登录后，您可以编辑或删除存储的密码信息。

## 安全检查

如果您收到密码安全警报，或者您想检查每个保存的密码，请从谷歌>密码进入**设置>隐私>自动填充服务，并选择**“检查密码”**打开密码检查工具。点击下一屏幕上的**“检查密码”**，然后按照说明进行操作。**

## 导出密码数据

您可以在**设置>隐私>谷歌自动填充服务>密码>设置>导出密码**下从谷歌密码管理器导出密码数据。这将创建一个包含您的密码数据的文件，然后您可以将其导入到另一个服务(如 1Password)中，或者用作备份以便离线保管。

## 自动填充个人信息、支付信息等

虽然没有严格绑定到谷歌密码管理器，但 Android 手机可以存储和自动填充许多其他有用的个人信息，如你的姓名、生日和性别；你家、工作地点或经常去的地方的地址；以及信用卡号码或 Google Pay 等支付方式。你可以启用这些功能，并通过谷歌的**设置>隐私>自动填充服务编辑任何保存的信息。**
