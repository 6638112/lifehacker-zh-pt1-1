# 欢迎回到“沙锅炖吗？”

> 原文：<https://lifehacker.com/welcome-back-to-will-it-casserole-1849378059>

大家好，欢迎回到 [*会砂锅*](https://lifehacker.com/tag/will-it-casserole) [*？*我已经有一段时间——将近两年——没有采纳读者提供的概念，把它变成一盘砂锅菜，我怀念被人指手画脚的日子。](https://lifehacker.com/tag/will-it-casserole)



我做的最后一个砂锅是这个 [塔特托普牧羊人馅饼](https://lifehacker.com/crown-your-shepherds-pie-with-tater-tots-1845154781) ，它很好，但我需要一个挑战，一个深奥的概念，一个美食狂热梦想来转变成一个多层次的热菜。类似于 [加思的早餐碗](https://lifehacker.com/i-turned-garth-brooks-breakfast-bowl-into-a-casserole-a-1840541163)[一个奶酪盘](https://lifehacker.com/this-cheese-plate-casserole-is-a-gooey-hot-masterpiece-1828255068) ，或者 [一整个野炊](https://lifehacker.com/theres-an-entire-cookout-in-this-casserole-1827321119) 。

这就是你进来的地方。如果你已经在 Skillet 呆了几年，你可能还记得本专栏 的 [读者来源部分。很简单:我建议一些砂锅点子，你们可以在下面评论自己的点子，或者给我的点子添加注释和建议。然后我看了评论，挑了我最喜欢的，做了一个砂锅，并分享了食谱。](https://lifehacker.com/what-should-we-casserole-next-1831769527)

和往常一样，我有一些想法让你开始，其中大部分我很久以前就想到了，但从未实现过:

*   **烤奶酪&番茄汤砂锅:**我们可以用几种不同的方法来做这道菜，但我想象有很多硬皮面包，很多不同的奶酪，还有一个番茄汤。
*   **法式洋葱汤砂锅:**和上面的相似，但是用了大量的焦糖洋葱，并且用牛肉代替了番茄(很明显)。
*   **乔氏砂锅菜:**我想看看只用乔氏产品能做出什么样的砂锅菜——如果它们都来自冷冻区，那就更好了。
*   **塔可钟砂锅:**所有塔可钟的项目都是 [基本上是相同的五种配料](https://www.theonion.com/taco-bells-five-ingredients-combined-in-totally-new-way-1819564909) 的重排，所以我看不出有什么理由砂锅不行。我们可以用实际的塔可钟产品、自制的复制品或者两者的混合来建造它。

你明白了。请在下方落下自己的砂锅点子，喜欢的就点。我不能保证我会做一个最喜欢的，但这是一个相当安全的赌注。
