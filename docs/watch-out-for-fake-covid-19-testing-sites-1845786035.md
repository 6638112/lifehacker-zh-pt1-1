# 警惕假冒的新冠肺炎考试网站

> 原文：<https://lifehacker.com/watch-out-for-fake-covid-19-testing-sites-1845786035>

不要听信最近几周出现的虚假冠状病毒检测站点。是的，没错——根据 [【美国联邦贸易委员会】](https://www.consumer.ftc.gov/blog/2020/04/not-every-covid-19-testing-site-legit) 的说法，骗子装扮成医疗专业人员，进行虚假、不卫生的金钱和身份盗窃测试，同时可能传播病毒。

Watch

根据联邦贸易委员会:

> 假网站可以看起来像真的，有看起来合法的标志、帐篷、防护服和看起来真实的测试...除了带走无辜者的信息之外，这些目的地还引发了与健康相关的问题。他们没有遵守卫生协议，所以他们可以传播病毒。最糟糕的是，他们没有给人们保持健康所需的帮助。

阿拉巴马州、亚利桑那州、佛罗里达州、佐治亚州、肯塔基州、纽约州和华盛顿州都出现了关于此类网站的报道，AARP 称。通常，他们对这种虚假的测试收取现金(在一个实例中为 240 美元)，他们还会试图窃取你的社会保障、医疗保险、医疗补助和信用卡信息，以达到身份盗窃的目的。

### **如何识别假考点**

乍一看，诈骗者可能看起来很合法，但再看一看:服装会不一致，因为诈骗者可能穿着油漆工工作服或实验室外套，而不是合法的个人防护设备。也看看姓名标签——在纽约时报 报道的 e 案 [中，骗子戴着一个假 ID 为“HIPPA”的挂绳，这是一个拼写错误的联邦法律的缩写，该法律限制如何共享健康数据。](https://www.nytimes.com/2020/04/05/us/politics/coronavirus-scams-fraud-price-gouging.html)

更糟糕的是，这些是肆无忌惮的欺诈者，所以像在测试之间更换手套这样的健康协议可能不会被遵守。由于这些都是靠不住的行动，他们往往很小——也许是几辆卡车和帐篷，可以在警察发现骗局之前迅速打包转移。

如果你担心附近的检测点是否合法，先去查一下你的或 [当地](https://www.naccho.org/membership/lhd-directory) 卫生部门的网站。如果您认为某个网站是假冒的，请在[ftc.gov/complaint](https://www.ftccomplaintassistant.gov/)**举报。**
