# 在几乎任何 Android 设备上安装这些未发布的 OxygenOS 11 壁纸

> 原文：<https://lifehacker.com/install-these-unreleased-oxygenos-11-wallpapers-on-almo-1845349770>

一加的“OxygenOS”是一加智能手机专用的安卓操作系统的修改版本，为提供了自己独特的怪癖和功能，与三星运行传统安卓操作系统的设备或手机截然不同。例如，即将到来的 OxygenOS 11 更新和严重泄露的一加 8T 具有独特的动画壁纸，你在其他任何地方都找不到。



或者说这就是我们的意图。幸运的是，对于我们这些没有一加设备的人来说，XDA 开发者论坛上一个名为“Linuxct”的用户已经提取了新的、动态壁纸，这应该是 OxygenOS 11 和一加 8T 独有的。T 他们现在可以安装在一系列非一加安卓设备上——oxygen OS 是可选的。

Linuxct 的应用程序中有 3 种壁纸。每个都有独特的配色方案，当你解锁设备或在不同窗口之间滑动时，分层的墨水状斑点会变形。每种模式都有亮暗两种模式。

你可以在下面链接的 Linuxct 的 Twitter 线程中查看它们。

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1314969775093800960&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1314969775093800960&autosize=1) 

可惜三星和联发科的设备都用不了直播壁纸。类似地，一些旧设备可能无法像预期的那样流畅地显示动画，或者根本无法显示。但是那些有支持设备的人可以通过以下步骤尝试 OxygenOS 11 的未发布壁纸

### 如何在无氧 OS 设备上导入和使用一加 8T 的 live 壁纸

1.  从 [Linuxct 的原 XDA 论坛帖子](https://forum.xda-developers.com/android/themes/8-0-oyxgenos-11-live-wallpaper-ported-t4164033) 下载氧气 OS11 LWP APK 文件。
2.  安装 APK(如果询问，允许从未知来源安装)。
3.  接下来，进入你的设备的壁纸选择菜单。如果你的安卓设备不允许你手动选择壁纸，你也可以安装 [谷歌壁纸](https://play.google.com/store/apps/details?id=com.google.android.apps.wallpaper) 。
4.  选择您想要使用的壁纸。就是这样！四处滑动以查看动态壁纸的效果。
