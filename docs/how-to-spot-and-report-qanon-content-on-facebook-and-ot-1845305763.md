# 如何发现和报告脸书和其他平台上的 QAnon 内容

我们都有别人可能会觉得奇怪的观点和信念，但有时这些想法会膨胀成危险的运动，就像卡农一样。

QAnon 于 2017 年开始发布一张名为“Q”的神秘海报，在某个匿名留言板上写下深入的阴谋帖子，这往往是所有在线荒谬行为的中心。这个“Q”声称唐纳德·特朗普正在发动一场针对美国政府“深层国家”的秘密圣战，该政府由一群崇拜撒旦的民主党人、媒体大亨、好莱坞精英和硅谷高管领导。这个邪教显然监管着一个庞大的儿童性交易团伙，从事食人、基因操纵和人类献祭。

哦，小约翰·肯尼迪显然还活着，预计随时会被确认为川普总统的竞选伙伴(取代副总统迈克·彭斯)。" Q "显然知道这一切，因为他们是拥有最高级别政府安全许可的时间旅行者。

尽管这听起来很疯狂，但这一理论很快就在主流社交媒体平台上流传开来。该运动在世界各地都有成员，成群结队的 QAnon 支持者现在经常出现在支持特朗普的集会上。许多政客声称信仰 QAnon，特朗普总统对这场运动给予了积极评价，甚至转发了 QAnon 相关内容。甚至还有一个 QAnon PAC。这不再仅仅是一场边缘运动，而且比一般的阴谋论更古怪(也更危险)。

### 如何在线识别 QAnon

QAnon 积极发展，隐藏在普通的视线和传播其信息，但仍然有可识别的标志和口号与运动。

追随者称自己为“anons”、“数字士兵”和“Q 军”，他们的座右铭是，“我们走到哪里，我们就走到哪里”，通常缩写为“WWG1WGA”。他们还使用“风暴”或“伟大的觉醒”等短语来描述他们的思想的传播以及特朗普最终战胜“深层国家”。

当然，还有“Q”标志，它通常带有美国国旗图案。这表现在旗帜、迷因、个人资料照片、保险杠贴纸、传单、纹身上——任何地方，真的。

这些都是运动的明显标志，但还有更微妙的线索需要寻找。这个组织经常躲在与虐待儿童和性交易相关的标签和短语后面。最近 [保守团体对网飞电影](https://www.wired.com/story/cuties-netflix-internet-clash)*的喧嚣，部分是受卡农思想的影响。*

*他们也喜欢传播新冠肺炎的错误信息——比如喝漂白剂或者使用危险药物进行治疗。*

*还有极右翼的出版物，如《信息战》和《布莱巴特》,他们经常发布不相关的理论和狗哨，即使他们没有直接引用这个阴谋。*

### *是否应该在社交媒体上举报 QAnon 相关内容？*

*没有人喜欢告密者，但 QAnon 不只是一些古怪的阴谋集团；这是一个危险的运动。成员们经常鼓吹极端主义行动，几起 [真实世界的枪击和绑架](https://www.cnet.com/news/qanon-what-you-need-to-know-about-this-crazy-conspiracy-theory/#ftag=CAD590a51e) 都是由 QAnon 和类似 Pizzagate 的相关阴谋推动的。美国联邦调查局表示，卡侬是一个公共威胁，甚至有一个两党的努力，以平息卡侬的文化影响。*

*脸书、Instagram、Reddit、Patreon、Twitter、抖音、LinkedIn 和 YouTube 主动从其平台上删除与 QAnon 相关的内容。 [脸书和 Instagram 有最严格的禁令](https://www.nbcnews.com/tech/tech-news/facebook-bans-qanon-across-its-platforms-n1242339) ，但 QAnon 的追随者仍有可能隐藏在众目睽睽之下，传播错误信息和煽动暴力。*
