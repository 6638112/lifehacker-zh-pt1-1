# 如何用Chrome和Edge Chromium同步iCloud密码

> 原文:[https://life hacker . com/how-to-sync-I cloud-passwords-with-chrome-and-edge-chrom-1846173552](https://lifehacker.com/how-to-sync-icloud-passwords-with-chrome-and-edge-chrom-1846173552)

过着macOS和T2 Windows的双平台生活，或者仅仅是“苹果和所有不是苹果的东西”，可能会很烦人但是如果你把你的密码保存到钥匙链上，然后上传到iCloud来同步你的苹果设备，你现在就可以把这个功能扩展到Windows系统了。(算是吧。)

Watch

苹果公司推出了一个新的浏览器扩展，允许你将你的钥匙链密码拉入Chrome(稍加调整，Edge Chromium)。您还可以将您在这些浏览器上创建的任何新密码同步回iCloud Keychain，然后就可以在您连接的苹果设备上使用它们了。

是的，这意味着火狐用户在这个问题上陷入困境，这也意味着你仍然只能通过你的浏览器访问你的密码；Keychain仍然没有内置到好的ol' icloud-dot-com中，也没有可以在Windows上使用的苹果应用程序，你可能会有 [1Password或LastPass](https://lifehacker.com/the-five-best-password-managers-5529133) 。

要开始使用Chrome，启动浏览器并导航到 [Chrome网络商店](https://chrome.google.com/webstore/detail/icloud-passwords/pejdijmoenmkgeppbflobdenhhabjlaj) 以获取 **iCloud密码**扩展。安装它，然后去安装[**I cloud for Windows**](https://www.microsoft.com/store/apps/9PKTQ5699M62)如果你还没有的话。你需要应用程序的版本12，扩展才能做任何事情。苹果目前似乎已经从微软商店下载了更新，但有可能你上周已经更新了(或将很快更新)。

我没有密码，但我知道你需要勾选那个小密码框，以便你的浏览器扩展可以与iCloud来回同步密码。因此，如果你已经有了12.0版本的Windows I cloud——或者当它出现时——确保你勾选了那个选项，否则你的扩展将无法做任何事情。除此之外，使用iCloud密码扩展和其他密码管理器一样简单；你不应该有问题。

然而，当使用Edge Chromium时，如果您试图将扩展与Windows的iCloud同步，可能会遇到问题。该扩展安装得很好，但你需要表明你正在同步到Internet Explorer，而不是Chrome，这可能需要你调整“书签”选项的设置。我一拿到应用程序的更新版本就会测试这个，但这是我看过的对其他人有效的快速解决方法。假装Edge Chromium是Internet Explorer，而不是Chrome，即使它与后者有更多的共同点。
