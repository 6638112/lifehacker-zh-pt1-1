# 你需要的是一种减少灰尘的室内植物

> 原文:[https://life hacker . com/what-you-need is a-dust-a-降尘室内植物-1847983143](https://lifehacker.com/what-you-need-is-a-dust-reducing-houseplant-1847983143)

虽然 [已经有很多关于植物能够净化空气中有害化学物质(如甲醛、苯和挥发性有机化合物)的文章](https://www.designboom.com/technology/nasa-clean-air-study-plants-natural-filter-06-13-2021/)(基于一项经常被引用的1989年美国宇航局的研究)，但那项研究 [也被揭穿与你家的空气质量没有显著相关性](https://www.theatlantic.com/science/archive/2019/03/indoor-plants-clean-air-best-none-them/584509/) (例如，它是在两英尺的密封室中进行的 )。 

Watch

虽然植物可能无法清除你家中的挥发性有机化合物，但它们会增加氧气，可能会减少压力，并通过将颗粒吸附到表面来减少空气中的灰尘。根据博主兼园艺家托尼·科尔格罗夫的说法，“室内植物确实通过将灰尘颗粒吸附在叶子表面来去除空气中的灰尘颗粒。有褶皱和/或多毛叶子的植物比光滑或有带子的叶子能去除更多的灰尘。一些影响是因为树叶减缓了空气流动...一些...是由于起皱或多毛的叶子在空气穿过叶子表面时从空气中物理过滤灰尘。”

## 甚至更好？买一个自我清洁的植物

你知道实际上有一些植物可以有效地为自己除尘吗？这是由于德国植物学家威廉·巴特洛特首先在 [莲花](https://www.google.com/search?q=lotus+plant+indoors&tbm=isch&ved=2ahUKEwjputjcs_rzAhU7hXIEHcNFAJ0Q2-cCegQIABAA&oq=lotus+plant+indoors&gs_lcp=CgNpbWcQAzIGCAAQCBAeOgcIABCxAxBDOgQIABBDOgUIABCABFCpBFijDGCmDWgAcAB4AIABVogBvQSSAQE5mAEAoAEBqgELZ3dzLXdpei1pbWfAAQE&sclient=img&ei=RJCBYemIEruKytMPw4uB6Ak&bih=681&biw=1200) 的叶子上观察到一种被称为“莲花效应”或“超疏水性”的现象。正如 [维基百科(相当)琥珀try所说的](https://en.wikipedia.org/wiki/Lotus_effect) ，这是当“由于表面上的微米和纳米结构，灰尘颗粒被水滴拾取，从而最大限度地减少了水滴对该表面的粘附。”

更简单地说，某些叶子有微小的穗状突起，形成凹凸不平的粗糙表面，排斥水。当水接触到叶子时，不是浸入到微小的凹槽中，而是从叶子表面滚下，带走污垢和灰尘。(这一过程也是对致病病原体的一层保护。)其他具有此属性的植物还有: [紫罗兰](https://www.google.com/search?q=violet+leaves&tbm=isch&ved=2ahUKEwjBgtD3s_rzAhXRgnIEHRiOAI4Q2-cCegQIABAA&oq=violet+leaves&gs_lcp=CgNpbWcQAzIFCAAQgAQyBQgAEIAEMgUIABCABDIFCAAQgAQyBQgAEIAEMgUIABCABDIFCAAQgAQyBQgAEIAEMgUIABCABDIGCAAQBRAeOgQIABBDOgYIABAIEB46CAgAELEDEIMBOggIABCABBCxAzoLCAAQgAQQsQMQgwFQ3wVY4RRgzhVoAHAAeACAAY8BiAGpB5IBBDEzLjGYAQCgAQGqAQtnd3Mtd2l6LWltZ8ABAQ&sclient=img&ei=fJCBYcHUL9GFytMPmJyC8Ag&bih=681&biw=1200)[芦荟](https://www.google.com/search?q=aloe&tbm=isch&ved=2ahUKEwjUyriftPrzAhU9sHIEHSM8CDAQ2-cCegQIABAA&oq=aloe&gs_lcp=CgNpbWcQAzIHCAAQsQMQQzIHCAAQsQMQQzIICAAQgAQQsQMyCAgAEIAEELEDMggIABCABBCxAzIFCAAQgAQyCAgAEIAEELEDMggIABCABBCxAzIICAAQgAQQsQMyBQgAEIAEOgQIABBDOggIABCxAxCDAVDlBVixCWD9CWgAcAB4AIABWogBpwKSAQE0mAEAoAEBqgELZ3dzLXdpei1pbWfAAQE&sclient=img&ei=0JCBYdSiEb3gytMPo_iggAM&bih=681&biw=1200)[马缨丹](https://www.google.com/search?q=lantana&tbm=isch&ved=2ahUKEwiFtJmVtPrzAhVjZTUKHSP-CTQQ2-cCegQIABAA&oq=lantana&gs_lcp=CgNpbWcQAzIHCAAQsQMQQzIHCAAQsQMQQzIFCAAQgAQyBQgAEIAEMgQIABBDMgUIABCABDIECAAQQzIFCAAQgAQyBQgAEIAEMgUIABCABDoICAAQgAQQsQNQwgVY7wpgtAxoAHAAeAGAAf4BiAHCBpIBBTYuMS4xmAEAoAEBqgELZ3dzLXdpei1pbWfAAQE&sclient=img&ei=upCBYYXvMOPK1QGj_KegAw&bih=681&biw=1200)[芋头](https://www.google.com/search?q=taro+plant&tbm=isch&ved=2ahUKEwjwu4iTtPrzAhXnqHIEHX_VBt8Q2-cCegQIABAA&oq=taro+plant&gs_lcp=CgNpbWcQAzIICAAQgAQQsQMyBQgAEIAEMgQIABBDMgUIABCABDIFCAAQgAQyBQgAEIAEMgUIABCABDIFCAAQgAQyBQgAEIAEMgUIABCABDoHCAAQsQMQQ1DTA1iOCWDlCWgAcAB4AIABWIgB6AOSAQE3mAEAoAEBqgELZ3dzLXdpei1pbWfAAQE&sclient=img&ei=tpCBYfCIFOfRytMP_6qb-A0&bih=681&biw=1200)[印度美人蕉](https://www.google.com/search?q=indian+canna&tbm=isch&ved=2ahUKEwj0zObhs_rzAhV9lHIEHWJvDLwQ2-cCegQIABAA&oq=indian+canna&gs_lcp=CgNpbWcQAzIFCAAQgAQyBQgAEIAEMgYIABAIEB4yBggAEAgQHjIGCAAQCBAeMgYIABAIEB46BAgAEEM6CAgAEIAEELEDOgcIABCxAxBDOgsIABCABBCxAxCDAVCfBljeD2C7EGgAcAB4AIABVYgB1gaSAQIxM5gBAKABAaoBC2d3cy13aXotaW1nwAEB&sclient=img&ei=T5CBYbRp_ajK0w_i3rHgCw&bih=681&biw=1200)
