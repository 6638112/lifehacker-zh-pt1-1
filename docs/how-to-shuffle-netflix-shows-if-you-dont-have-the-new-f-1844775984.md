# 如何洗牌网飞秀

> 原文：<https://lifehacker.com/how-to-shuffle-netflix-shows-if-you-dont-have-the-new-f-1844775984>

网飞将在今年上半年推出一项新的“随机播放”功能，这正是我们在疫情时代需要的东西——一种让我们的狂欢会议持续更长时间的方式。在你等待的时候，有几个聪明的变通办法可以让你现在就在网飞的 [庞大的内容库](https://lifehacker.com/tag/netflix) 中穿梭。



### **随机化网飞的浏览器扩展**

如果你是 Chrome 的用户，我建议你看看以下内容:

*   [**shuffle flix**](https://chrome.google.com/webstore/detail/shuffleflix-netflix-rando/copfkimibpnfpdggfjencnehgfgmpphf?hl=en)**:**让你随意跳转到你正在观看的剧集。你不会只是得到一些你没有兴趣观看的随机网飞秀；相反，这种体验类似于所有那些你打开*法律&命令*并发誓你从未看过你碰巧看到的那一集的时候，尽管在你看重播的这些年里，你可能实际上已经看过六次了。
*   [**rand flix**](https://chrome.google.com/webstore/detail/randflix-random-episodes/enjakkkpkgpcnjbmagjgkccljfimgbhh?hl=en)**:**同一笔交易，不同分机。如果你有 ShuffleFlix 的问题，Randflix 是一个很好的选择。

**火狐**用户可以尝试:

*   [**网飞 Shuffle**](https://addons.mozilla.org/en-US/firefox/addon/netflix-shuffle/) **:** 它是火狐版本的那些其他“随机播放剧集”的扩展。不过，它只适用于特定的节目列表，这使得这个选项对火狐粉丝来说不那么令人印象深刻。如果你只关心一个节目——比如说，[](https://addons.mozilla.org/en-US/firefox/addon/random-office-episode/)**——*你可能也能找到特定的随机函数扩展。*
*   *[**更好的网飞**](https://addons.mozilla.org/en-US/firefox/addon/better-netflix/) **:** 这个插件在网飞基于网络的用户界面中加入了一系列有用的功能，包括支持超宽显示屏，使用鼠标滚轮改变音量的能力，以及一个可以用来跳过随机网飞节目的按钮。*

*任何人都可以尝试:*

*   *[**【网飞轮盘**](https://reelgood.com/roulette/netflix) :虽然不是浏览器扩展或者任何直接捆绑到网飞本身的东西，这个网站至少可以随机挑选一个节目或者电影给你看(根据你给它的任何参数)。如果你感到犹豫不决，就像我每次盯着网飞庞大的内容列表时一样，网飞轮盘赌会让你尽快离开菜单，进入电影。*

### ***网飞洗牌的 iOS 快捷方式***

*如果你喜欢在 iOS 上看《网飞》,有几个选项可以帮你调出网飞的随机内容(或者某个特定剧集的剧集)。这里有两部 [一般的](https://routinehub.co/shortcut/5146) [一般的](https://www.reddit.com/r/shortcuts/comments/dtv4cu/netflix_shuffle_v2_play_a_random_episode_from_any/) 和 [这里的](https://routinehub.co/shortcut/2649/) 是让你从网飞最佳电影名单中挑选的一部。从你按下一个按钮在节目和电影中随意切换的意义上来说，这并不完全是“随机”的，但这是事实。*

### ***至于安卓...***

*我还没有为 **Android** 找到一个好的选项，真扫兴。 [**Upflix**](https://play.google.com/store/apps/details?id=com.douglas.upflix) 应用程序可以帮助你随机选择观看的节目，但这不如专用的随机播放按钮。[**just watch**](https://play.google.com/store/apps/details?id=com.justwatch.justwatch)是另一个你至少可以用来搜索网飞上有什么的工具，但是，同样，我们需要一个随机数发生器。还有 [**在网飞我到底该看什么？**](http://whatthehellshouldiwatchonnetflix.com/) ，我欣赏它的直白手法，还有[**flix boss**](https://flixomat.com/)。*

*老实说，你可能最好使用 [**网飞轮盘**](https://reelgood.com/roulette/netflix) 来识别要观看的东西，然后像往常一样在你的 Android 上运行。*
