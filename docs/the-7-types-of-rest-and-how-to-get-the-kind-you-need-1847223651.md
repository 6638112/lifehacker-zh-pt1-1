# 7种类型的休息，以及如何得到你需要的那种

长期以来，我们对“你好吗？”这个问题的默认回答是已经“很好”，或者它的一些变体。但在过去几年里，尤其是在疫情期间，另一种回应变得越来越普遍:告诉问你的人你“累了”，甚至“筋疲力尽了”

有很多东西需要解开，但可以肯定的是，由于各种各样的原因，我们很多人真的在所有时间或大部分时间都感到疲惫不堪。你还记得上一次完全休息是什么时候吗？是啊，我也不能。

医生、研究员、《神圣的休息:恢复你的生活、恢复你的能量、恢复你的理智》一书的作者桑德拉·多尔顿·史密斯 博士说，这可能是因为我们没有得到我们真正需要的特定类型的休息。以下是关于七种休息方式的知识，以及为什么选择正确的休息方式可以改变游戏规则。

## 休息不仅仅是睡觉

很有可能，如果你告诉某人你一直很累，他们可能会对你如何获得更多的睡眠或改善睡眠卫生发表一些意见。当然，这没什么坏处，但正如道尔顿·史密斯在接受造型师 采访时解释的那样，休息不仅仅是睡觉。

“我们喜欢这种简单的想法——如果我们睡觉，应该可以解决我们所有的恢复需求——但不幸的是，有一些类型的休息是睡眠无法解决的，”她告诉该杂志。

## 休息的7种类型是什么？

在她自己的研究中，Dalton Smith已经确定了人体和大脑需要的七种不同类型的休息 。因此，即使你睡了一夜好觉，你醒来时可能仍然会感到筋疲力尽，因为你缺乏不同类型的休息。这些类型是:

1.  精神休息:让你的大脑休息一下
2.  **感官休息**:花时间拔掉插头，减少我们的外部刺激
3.  创造性休息:做一些你觉得鼓舞人心的事情，从产生新想法中休息一下
4.  情绪休息:处理你的情绪，这样你就不必压抑它们，你就能做真实的自己
5.  社交休息:当别人耗尽你的精力时，花时间独处
6.  精神上的休息:与超越身体和精神的东西联系起来(即使它不是宗教的)
7.  **身体休息**:睡觉，让你的身体放松下来

## 如何获得你需要的休息

根据道尔顿·史密斯的说法，获得你所需要的那种休息的关键是首先确定你所缺乏的类型，然后从那里开始。她说，这个过程从一些自我反思开始，因为你通常在一天中使用七个区域中的每一个，你必须找出你在哪里花费了最多的能量。

“你的日常生活是什么样的？，”道尔顿·史密斯问道。“你一天都在头脑风暴和想出新主意吗？还是你每天都在和像咨询师或治疗师这样的人面对面讨论艰难的情感问题？”

她说，一旦你确定了你在哪里倾注了最多的精力，“这将对你最有可能出现赤字的领域产生巨大影响，尤其是如果你还没有考虑清楚那个职位或工作的复苏情况。”
