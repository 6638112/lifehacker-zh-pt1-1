你可能已经熟悉经典的水果/风味搭配，比如苹果和肉桂，或者小红莓和橙子。但是葡萄柚和龙蒿呢？还是桃子和枫糖浆？

## 如何使用水果配对表

《面包师年鉴》 背后受过专业训练的糕点师莱斯利·全(Leslie Jeon)整理了 [这张方便的图表](https://thebakersalmanac.com/fruit-flavor-pairing-chart/) 清楚地展示了哪些口味在搭配各种水果时味道最好。

如果你喜欢烘焙，烹饪，或者用水果制作手工鸡尾酒，这是一个你想要加入书签的页面。(如果你订阅了网站的邮件列表，你也可以下载一个看起来更整洁的PDF版本。)

水果按字母顺序排列在最左边的一栏中，然后是其他水果、香草和香料、坚果、烈酒和其他各种食物(如糖浆、咖啡、巧克力、蜂蜜等)，它们可以做出最佳的风味搭配。您也可以使用图表右上角的搜索栏来搜索特定的水果或风味。

如果你在图表上找不到你想要的水果或其他配料，只需在下面给Jeon留言，她会尽量在下次更新中加入。(她经常添加新的水果和口味。)

当你最终拥有大量特定类型的水果时，无论是自己种植的水果，还是在当地农贸市场或CSA购买的水果，这个图表都非常有用。

例如，假设你家后院的树结出了大量的梨子，而你已经没有办法把它们加入沙拉中，并且厌倦了单独吃它们。虽然你可能已经把梨和姜配在一起了，但是迷迭香或罗勒怎么样呢？还是马斯卡彭和香草？也许开心果和豆蔻？准备好做(吃)很多甜点。
