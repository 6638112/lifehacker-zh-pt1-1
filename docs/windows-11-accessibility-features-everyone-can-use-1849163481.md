# 每个人都可以使用的Windows 11辅助功能

> 原文:[https://life hacker . com/windows-11-accessibility-features-every one-can-use-1849163481](https://lifehacker.com/windows-11-accessibility-features-everyone-can-use-1849163481)

在过去的几年里，微软在让更多的用户更容易使用Windows方面取得了长足的进步。 Windows 11包括一个专用的辅助功能部分，旨在帮助用户根据他们的需求调整操作系统的显示和行为。这一部分对有视觉障碍、听力损失或身体残疾的用户非常有帮助— 但是任何人都可以使用这些辅助功能来改善他们的Windows 11体验。以下是一些值得强调的。

Watch

## 让一切变大

如果你戴着眼镜，或者如果你不喜欢整天盯着一个小小的笔记本电脑屏幕，试着把Windows 11 中的所有东西都变大，从文本到用户界面。

进入**设置** > **辅助功能** > **显示** > **比例**并在100%和200%之间选择一个设置 。特别是如果你使用的是4K显示器， 200%缩放模式将让一切变得漂亮和大，同时保持可用。

如果您只想增加文本大小，可以从**设置** > **辅助功能** > **文本大小**中进行。

## 使用放大镜快速放大任何内容

放大镜是一个容易使用的缩放工具，你可以通过同时按下**窗口**和**加**键来激活它。启用放大镜功能后，再次按下**加**键放大屏幕，鼠标在屏幕上来回移动。您可以使用**加**和**减**键继续放大或缩小(同时按下Windows键)。完成后，按下**窗口** + **退出**键禁用该功能。

## 使用夜灯减少蓝光发射

与普遍的看法相反， [蓝光并不一定让你](https://lifehacker.com/does-blue-light-actually-affect-your-health-1839035806)晚上睡不着觉，阻止光线影响你睡眠的最佳选择是晚上停止使用屏幕。然而，一些人发现蓝光限制器从舒适的角度来看是有帮助的。如果你有兴趣，试试Windows 11的**夜灯**功能。它可以从任务栏中的wifi菜单访问，并且它会立即为整个屏幕添加一个温暖的滤镜。

## 使鼠标更大或更有色彩

你不需要忍受默认的鼠标光标。 G o到**设置** > **辅助功能** > **鼠标指针**改变其大小和样式。要更改其颜色，请单击最后一个选项。

您可以对文本光标做同样的操作，从**设置** > **辅助功能** > **文本光标**。

## 使用高对比度主题

Windows 11有许多透明和半透明的元素。如果你发现很难看出什么是按钮，什么是链接，尝试使用Windows 11的高对比度主题之一。你可以尝试一个简单的甜点主题，用蓝色的棕褐色调突出，或者夜空主题，这种主题完全是黑色的，突出了亮黄色的元素。或者点击**编辑**按钮自定义自己的主题。

## 用你的声音打字

语音转文本已经走过了漫长的道路，Windows 11的语音打字工具就证明了这一点。Windows 11使用Azure语音服务为任何语音生成文本，包括标点符号，所以你说的话应该(希望)一字不差地出现在页面上。

按下 **Windows** + **H** 键盘快捷键，或触摸键盘上空格键旁边的麦克风图标即可开始。语音打字功能理解标点符号和特殊字符，所以你可以说类似“**开单引号**”这样的话，Windows就会为你启动一个带引号的句子。阅读此 [文档](https://support.microsoft.com/en-us/windows/use-voice-typing-to-talk-instead-of-type-on-your-pc-fec94565-c4bd-329d-e59a-af033fa5689f#PickTab=Windows_11&WindowsVersion=Windows_11) 中关于短语和支持语言的更多详细信息。

## 为任何内容获取自动字幕

Windows 11有自动字幕，适用于各种视频、电视节目和电影。您可以使用**窗口** + **控件** + **L** 键盘快捷键随时启用它，或者您可以进入**设置** > **辅助功能** > **字幕** > **实时字幕**。在这里，您也可以编辑标题样式。

在你按下键盘快捷键后，Windows 11会自动开始将口语文字转换成屏幕上的实时文本。

## 让Windows成为你的讲述者

Windows 11的讲述人功能非常酷，允许你在不需要鼠标的情况下完成基本任务。它是为盲人或弱视用户设计的，作为一个屏幕阅读器，它将阅读屏幕上的所有内容，包括文本、按钮和链接。你可以使用键盘来导航和选择东西。

使用**窗口** + **控制** + **回车**键开始。要进入讲述人设置，使用**窗口** + **控制** + **N** 键盘快捷键。
