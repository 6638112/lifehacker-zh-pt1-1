# 微软有一个保护密码安全的新技巧

> 原文：<https://lifehacker.com/microsoft-has-a-new-trick-for-keeping-your-password-saf-1849580498>

你的电脑密码是不能轻易透露的。这是你的王国的钥匙:从工作，到社交媒体，到银行，你的整个世界都可以从你的 Windows 机器上访问。因此，微软正在*恳求*我们不要做傻瓜，不要让把我们的密码交给任何询问的人。但是他们知道我们中的一些人会，这就是为什么他们最近推出了一些强大的功能来保护我们免受伤害。

## 坏演员如何窃取你的电脑密码

这个问题归结为网络钓鱼，或者说诱骗某人交出个人数字信息，比如他们个人电脑的密码。骗子们变得*非常*擅长从不知情的受害者那里获取这一信息，绕过如今显得做作而明显的传统策略。一个策略是创建*看起来*就像你试图登录的网站，但完全是假的。即使你及时意识到骗局并且*没有*点击登录按钮，有时甚至在这些网站中键入你的密码就足以让黑客窃取。虽然这不是一件好事，但如果你在脸书使用的密码和你在电脑上使用的密码是一样的，那就更糟了。现在，坏演员都知道怎么登录你的机器了。

另一个问题涉及不安全的密码存储。例如，如果您将所有密码保存在一个标准的 Word 或 Excel 文件中，您很容易遭受数据窃取。你应该只把你的密码保存在有密码保护和加密的来源中，而绝不是任何人都能阅读的标准文件。

虽然个人风险很大，但公司和网络的风险也很大。黑客攻破庞大的机构得益于的[弱项或者泄露的密码](https://www.bbc.com/news/technology-62937678) 。有时候 访问一个用户的机器就足以触发一次彻底的安全崩溃。

## Microsoft 如何帮助保护您的 Windows 密码

随着上周的大 [Windows 11 2022 更新](https://lifehacker.com/why-you-should-upgrade-to-windows-11-2022-1849562445) ，微软又发布了新武器来保卫你对抗 PC 密码被盗。第一个是当你在另一个网站上使用你的 Windows 登录密码时弹出的警告。希望是阻止你在任何其他服务中使用你的电脑密码。如果你“登录”的网站实际上是一个钓鱼网站，黑客现在会知道你电脑的密码，但即使是一个合法的网站也会经历密码泄露。第二个选项是当你在一个不安全的程序中输入密码时发出警告。微软正试图阻止你在 Word 或 Notepad 之类的应用程序中保存密码，因为它们不会提供与真正的密码管理器相同的保护。

## 如何启用微软的新密码保护协议

出于某种原因，这些密码保护选项在默认情况下是不启用的，它们被隐藏在设置中。为了让它们工作，你需要用你的 Windows 密码而不是 Windows Hello 登录你的电脑。如果你使用 PIN 码登录你的电脑，它们将不起作用，所以你需要先关闭那个选项。( G o 到**开始** > **设置** > **账号** > **登录选项**，选择想要禁用的 Windows Hello 方式，然后选择“移除”)

接下来，打开**开始菜单** > **设置** > **隐私&安全** > **Windows 安全** > **App &浏览器控制** > **基于信誉的保护设置**。检查**网络钓鱼保护**，它应该已经设置为“开”。关注的其他选项是“警告我密码重复使用”，和“警告我密码存储不安全”。“打开这两个选项，每当你在不安全的应用程序中输入密码时，你都会收到提醒。
