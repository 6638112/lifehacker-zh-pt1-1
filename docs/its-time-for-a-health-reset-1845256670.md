# 是时候进行健康重置了

> 原文:[https://life hacker . com/its-time-for-a-health-reset-1845256670](https://lifehacker.com/its-time-for-a-health-reset-1845256670)

自从疫情关闭和封锁开始以来，已经过去了漫长的六、七个月。还记得头几周待在家里是件新鲜事，社交媒体上充斥着自制面包的情景吗？嗯，开始变老了。

在家呆的时间越长，你的健康可能会受到潜移默化的影响。我们中的许多人锻炼得越来越少，吃饭和睡觉的时间都不规律，筋疲力尽，失去了社交联系。即使你在这些方面做得不错，和我们一起去看看我们是否能解决一些问题，因为我们要为冬天和可能更多的室内时间做准备。

每周五，我会发布一份周末作业，你可以在你健康的某个方面做一些调整(或者彻底改变你的整个方法)。我们还会有一个日历，让你每天都有一个小小的收获——下周继续关注。

顺便说一下:虽然我们称之为重置，但我们并不意味着节食。我不会告诉你吃什么或者穿多大的。我只是提醒你注意你的健康。

### 本周的任务:安排一些约会

我们将从你健康的医学方面开始。自年初以来，你可能已经错过或推迟了一些约会。也许其中一些无关紧要，但现在是时候盘点那些有关系的了。

首先，确保你 [接种了流感疫苗](https://vitals.lifehacker.com/its-even-more-important-to-get-your-flu-shot-this-year-1844481707) ，以及任何 [其他可能到期的疫苗](https://vitals.lifehacker.com/vaccines-you-need-as-an-adult-1720902072) 。你可以在药店几分钟内买到大部分这些药物，避免你在疫情期间因其他疾病而生病的可能性。

如果你的孩子错过了健康访视，确保他们没有错过疫苗 或必要的筛查。同样，如果您错过了体检或其他常规护理，如洗牙，请致电您的医疗服务提供者，询问您是否应该接受检查。他们可能会说不，没什么大不了的，明年见；但是如果某些事情对你很重要，考虑到你的病史，他们可以让你知道。

疫情的一个亮点是，医疗服务提供商正在发现他们可以通过远程医疗(如视频通话)做多少事情。你也许可以这样做一些你通常会亲自做的访问:例如，治疗。

说到治疗，想想你是否有以前没有的与健康相关的新需求。焦虑和抑郁是人们在压力下突然出现的情况，对我们许多人来说，今年是前所未有的压力来源。 [需要的话找治疗师](https://lifehacker.com/how-to-select-the-right-therapist-for-you-1833246447) ，或者去 [看看这些免费又便宜的心理健康资源](https://vitals.lifehacker.com/if-you-cant-afford-a-therapist-try-these-mental-health-1818506208) 。

因此，列出你需要的疫苗、筛查或预约，并确定哪些仍然是优先事项。然后提出那些预约请求，或者给自己做个记录，周一早上打几个电话。你会很高兴你做了。
