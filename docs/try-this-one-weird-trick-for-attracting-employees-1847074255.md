# 试试这个吸引员工的奇怪招数

> 原文:[https://Jezebel . com/try-this-one-verged-trick-for-attracting-employees-1847074255](https://jezebel.com/try-this-one-weird-trick-for-attracting-employees-1847074255)

当全国各地的餐馆老板对他们根本找不到愿意被无面具反吸血鬼者唾弃的人来换取分发食物的特权这一事实感到困惑时，一些大胆进取的企业主正在测试一种吸引工人的奇怪的新方法:P 阿英生活工资！

Watch

正如《华盛顿邮报》 [报道](https://www.washingtonpost.com/business/2021/06/10/worker-shortage-raising-wages/?utm_source=twitter&utm_medium=social&utm_campaign=wp_main) 所言，一些美国餐馆，如Klavon's冰淇淋店，正在试验“付给员工更多的钱”来换取他们的劳动，而过去的方法——每小时付给员工7.25美元，并让他们向越来越不满的陌生人寻求帮助——似乎实际上让未来的员工对一次站8个小时，休息时间很少，没有健康

> 因此，老板雅各布·汉查尔决定将起始工资提高一倍多，达到每小时15美元，外加小费，“看看会发生什么”。“商店突然被申请淹没了。超过1000人在一周内蜂拥而至。”

狂野！如果存在某种制度，让大多数美国人除了可以支付房租和食物的薪水之外，还可以享受托儿服务，甚至看医生，那会怎么样？可能还会有更多的人既有时间又足够健康，去做有益于每个人的工作。

原来不只是我有这种离谱的理论。真正聪明的人，比如加州大学伯克利分校的劳动经济学家Enrique Lopezlira，认为对待人类就像他们有权获得公平的补偿和尊严，这可能是打击资本主义的最热门趋势，因为支付人类微薄的报酬，并要求他们对这种不公平的待遇表示极大的感谢:

> “雇主越是提高工作质量，越是将员工视为需要最大化的资产，他们就越能找到并长期留住员工，”他告诉《邮报》记者。

如果是真的，那就太大了。
