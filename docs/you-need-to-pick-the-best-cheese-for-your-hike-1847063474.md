# 你需要为你的远足挑选最好的奶酪

> 原文：<https://lifehacker.com/you-need-to-pick-the-best-cheese-for-your-hike-1847063474>

奶酪容易腐烂，但大多数人对它都非常珍惜。是的，它含有乳制品——或者更确切地说，它是乳制品——如果放在冰箱外面太久，很容易滋生细菌并腐败。但是，根据奶酪的不同，这扇窗户足够大，可以让你享受一次以奶酪为燃料的短途旅行、长途跋涉或徒步进入荒野。(我不知道你是怎么想的，但我宁愿享受大自然，吃一块有水晶斑点的陈年切达干酪和一个苹果，而不是一块干燥、没有灵魂的蛋白质棒。)



当然，并不是所有的奶酪都非常适合作为徒步旅行的零食。例如，把一轮令人流泪的卡门贝干酪带进森林是不明智的。软而粘的奶酪不仅不实用，而且不安全。

正如威斯康星州 奶农 [食品安全、质量和监管合规总监 Adam Brock 向](http://www.wisconsindairy.org/Who-We-Are/dairy-farmers-of-wisconsin) [【好胃口】](https://www.bonappetit.com/story/how-long-can-cheese-sit-out) 解释的那样，奶酪在从冰箱中取出后的四个小时内食用是最安全的，但并非所有的奶酪都具有相同的水分含量，水分在细菌生长和腐败中都起着重要作用。

如果你去过杂货店，你可能会注意到一块块的帕尔马干酪摆在外面，不受温度控制。布洛克解释说:“帕尔马干酪、罗马诺干酪或更硬的干酪在整个聚会期间可能不会有微生物生长，或者数量非常少。”。(布洛克在和 Bon App 谈论派对奶酪，这种奶酪与徒步奶酪相邻，如果不是完全一样的话。)“那些你会经常看到挂在意大利市场上或在杂货店切成块陈列的奶酪，因为它们不需要持续冷藏，”他补充道。

美国农业部 也对硬奶酪和软奶酪进行了安全区分，称“软奶酪，如奶油奶酪、农家奶酪、碎奶酪和山羊奶酪，为了安全起见，必须冷藏。一般来说，硬奶酪如切达干酪、加工奶酪(美国)以及块状和磨碎的帕尔马干酪出于安全考虑不需要冷藏，但如果冷藏，它们会保存更长时间。”

我想说的是，带奶酪去远足是安全的，只要你选择了正确的奶酪。低水分、加工过的午餐盒奶酪，如蜡包裹的 Babybel 和 string cheese 可能是你的最佳选择——这些是我的徒步旅行- 和露营爱好者朋友，维多利亚，经常为一天的旅行打包——但是更硬、更老的切达干酪和 goudas 含有更少的水分，在温暖的温度下它们尝起来棒极了。冷冻的、冰箱冷藏的奶酪比室温的(或背包温度的)奶酪更难感觉和品尝。脂肪需要温暖来移动和开槽——冷脂肪在质地上并不令人愉快，寒冷的温度使你的舌头更难接触到奶酪提供的更微妙的味道。

如果你不想带刀和切菜板，就把奶酪切成方块，扔进有拉链的袋子或其他可密封的容器里，旁边放一根或三根牙签(以防你的手脏了，不想碰奶酪)。在你的包里扔一个苹果或一些干果，还有一小袋坚果，你就有了一份美味营养的徒步旅行零食，它会比任何味道的 Clif Bar 带来更多的欢乐。
