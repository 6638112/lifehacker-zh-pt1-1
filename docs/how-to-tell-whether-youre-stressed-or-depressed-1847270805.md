# 如何辨别你是压力大还是抑郁

> 原文：<https://lifehacker.com/how-to-tell-whether-youre-stressed-or-depressed-1847270805>

压力和抑郁看起来和感觉起来非常相似。两者的一些共同症状包括睡眠、饮食、注意力和情绪问题，以及执行日常任务的困难。在生理上，压力和抑郁之间有很多重叠，因为两者都会影响免疫系统，导致某些炎症标记物的增加。



F 或抑郁症患者，他们大脑中的变化类似于在慢性压力下观察到的情况。慢性压力，如果不治疗，会导致抑郁。例如，在童年时期经历了高度逆境，导致毒性压力的成年人有更高的抑郁率。

哥伦比亚大学精神病学教授菲利普·马斯金 说:“它们的生物学并不相同，但它们有很多相似之处。”。

以下是如何区分压力和抑郁的方法。

## **压力是阶段性的**

当谈到压力与抑郁时，有明显的差异，特别是在谈到有效的治疗方案时。压力和抑郁的一个主要区别是压力可以来来去去。

“对大多数人来说，压力是阶段性的。“你会有一段紧张时期，然后你会走出这段时期，”马斯金说。“抑郁症不是这样的。有些人的抑郁症会持续数年。它可以在一些人身上自然消退，但不是所有人。”

例如，如果一件快乐的事情发生了，比如朋友或爱人来拜访，一个有压力的人会在那一刻感到快乐，尽管一旦他们离开，压力可能会回来。对于一个抑郁的人来说，他们将无法在那一刻感受到幸福，即使他们知道他们应该这样做。

“如果你可以下班回家，还在充电，那不是主要的临床抑郁症，”心理学家谢丽尔·齐格勒 说，她是《妈咪倦怠:如何重新开始你的生活并在这个过程中养育更健康的孩子 *一书的作者。*“临床抑郁症不会来来去去。”

那么，压力的治疗方法是什么？马斯金建议，通过锻炼、冥想和正念等措施来减少压力，同时减少压力的*来源*。

## **抑郁症是一种疾病**

对于一个抑郁的人来说，尽管减压措施——比如锻炼或者到大自然中去——可以有所帮助，但并不能治愈他们。

“在某种程度的抑郁症中，除了药物治疗，没有什么能有所帮助，”马斯金说。

抑郁症本质上是一种大脑疾病。就像我们用抗生素治疗感染一样，抑郁症通常需要药物治疗。对于患有严重抑郁症的人来说，再多的“精神战胜物质”或“希望它消失”都不起作用。相反，抑郁症患者需要药物治疗。

“抑郁症和其他疾病没什么不同，”马斯金说。“是医学上的病。”

如果你正经历压力或抑郁，最重要的是要知道帮助是可以得到的，而且会变得更好。对于压力，这包括减少压力的来源和寻找应对的方法。对于抑郁症，这种涉及到治疗，比如疗法和药物治疗。

无论正确的解决方案是什么，要知道总有一个是正确的，迈出第一步去获得你需要的帮助可能是其中最重要的一个。
