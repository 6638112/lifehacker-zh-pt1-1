# 这是吃鸡翅最简单的方法吗？

> 原文：<https://lifehacker.com/is-this-the-easiest-way-to-eat-chicken-wings-1845145029>

很少有东西像不起眼的鸡翅一样具有美国特色，这就是为什么当我们在最新的全美应用程序抖音上看到这个俏皮的软件时，我们不得不尝试一下。

在抖音，用户 [@josh__kirk](https://www.tiktok.com/@josh__kirk?_d=secCgsIARCbDRgBIAIoARI%2BCjzeHd5JLmsaAA62jolQjChCTfjEkFo12FQNBebUgt2hdFpkJYG0FV1StzSoqgZpeSFMU1NWmgblNM1%2F%2BecaAA%3D%3D&language=en&preview_pb=0&sec_user_id=MS4wLjABAAAA-urxM-eTO3o8kR1ItylBD6a_tG6hhnYeYQq7vAjNlWDwmNAEzPWJxl_Su2Vq-eWo&share_item_id=6861361717432044806&share_link_id=72560819-C50E-48E5-86F7-632114AC0AC2&timestamp=1599096010&tt_from=copy&u_code=d7mej85ij7dc28&user_id=6727451730230952965&utm_campaign=client_share&utm_medium=ios&utm_source=copy&source=h5_m) 按下鸡翅的肉，使其与骨头分离，并浓缩成美味的小棒棒糖。柯克先生然后把他的纸风车肉蘸上大概是蓝色奶酪调料，一口就吃完了。

与排骨、蟹腿、 [Cap'n Crunch Smartfood 爆米花混合](https://skillet.lifehacker.com/is-capn-crunch-smartfood-popcorn-worthy-of-your-mouth-1844945916) 一样，鸡翅肯定是更难(也更乱)正确食用的食物之一。虽然鸡腿可以产出大量的肉，但双骨鸡翅部分往往会让你感到饥饿，剩余的额外肉似乎不值得额外努力。

这就是为什么我们采购了正好两个(两个！)鸡翅来测试这个 hack。现在，我手头没有传统的布法罗鸡翅。相反，我从当地杂货店的烤鸡上取下了鸡翅。在副主编(坚定的素食主义者)乔丹惊恐地看着的时候，我能复制出抖音的结果吗？看看上面的视频就知道了。
