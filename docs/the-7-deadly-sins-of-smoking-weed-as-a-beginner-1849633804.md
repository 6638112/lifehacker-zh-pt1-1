# 初学者吸食大麻的7大罪

我知道你已经等了很久，但是现在拜登总统已经向联邦政府合法化大麻 迈出了一大步，也许你终于该试试抽大麻了。这真的没什么大不了的——说真的，任何白痴都会做——但还是有一些错误你应该避免，这样你就不会过得很糟糕，或者看起来像来自特拉华州的广场。

如果这是你第一次嗑药，或者你已经有一段时间不玩这个游戏了，放轻松。宽容是真实的，你可能没有多少，所以不要试图匹配你的瘾君子朋友加里的肺。

尽管许多瘾君子很随意地服用这种药物，但大麻可能非常强烈。记住:你可以一直抽更多的烟，但永远不能少抽。这尤其适用于食物。它们需要很长时间才能开始起作用，而且兴奋持续的时间更长，所以从小剂量开始——也许5毫克——不要在45分钟后认为“这不起作用，我要多吃点”。给它时间。

大麻会让一些人变得偏执，并引发恐慌。如果你开始感到恐惧或末日即将来临的感觉，记住它只是一种药物。你安全了。你吓坏了，但只是因为你高，它不会持续超过几个小时。不要认为“杂草一定被掺了进去！”大麻没有掺进去。不要因为你太高 而 [报警。他们将无法帮助你，而且可能会嘲笑你。](https://people.com/crime/man-calls-cops-on-self-for-being-too-high/)

注意设置和设置是60年代的老规则，瘾君子用来避免迷幻药引起的恐慌，但它也适用于吸食大麻。设定是你个人正在做的事情；环境是你的物质环境。你希望两者都是积极的，特别是如果你是一个杂草noob。确保你处于一个总体良好的心情和舒适的环境中。对大多数人来说，在内华达山脉露营是抽大麻的好地方；在纽瓦克市中心的瓢泼大雨中。

对布景和布景来说最重要的是和你认识和信任的人一起抽大麻。事实上，试着和你认识和信任的人一起做每件事——这是生活的全部意义。

吸食大麻会导致口干舌燥、眼睛发痒、饥饿感以及对琐碎事物的过度迷恋。为所有这些副作用做好准备。准备好水，眼药水，零食，还有一个喜欢谈论星球大战的好朋友。

当我兴奋的时候，我能做的就是开愚蠢的玩笑和欣赏波萨诺瓦音乐。除此之外，我一无是处。我不能订披萨，不能跟随电影情节，也不能驾驶公共交通工具。有些人则相反，他们可以在工作或孩子的洗礼时吸烟，而且完全正常。直到你知道你是哪种人，只为你的大麻吸烟制定最轻的计划。行程应该是这样的:1)抽大麻，2)玩拼字游戏。可能不应该是1)抽大麻，2)见婚礼策划人。

老人们会受到沉重的打击，并尽可能长久地保持下去。这可能是毫无意义的；大多数让你兴奋的化学物质一进入你的肺部就会被吸收。你因吸烟而获得的任何额外快感都可能来自头晕。所以，把它吸进你的肺里，像星期天早晨一样轻松地释放出来。

但是你需要真正吸气，否则效果不会很好。对于一些从来没有吸过烟的人来说，吸入会很困难。如果是这样，试试这个:当你把烟吸进嘴里的时候，想想“哦，我的上帝，我的孩子刚刚走进来了！”伴随着一个小小的惊吓而来的剧烈的“哼”的吸气就是你应该如何吸气。
