# 如何早期尝试 Chrome 的标签组保存功能

> 原文：<https://lifehacker.com/how-to-try-chromes-tab-group-saving-feature-early-1847541441>

Chrome 的新 tab ggroups 在用户中是一个两极分化的话题，但谷歌继续用新功能来支持这一功能，使其值得一试。选项卡群组让你将打开的页面组织到不同的群组中——这使得处理所有这些选项卡变得更加容易——现在，谷歌正在测试保存选项卡群组的能力，以便你以后可以重新打开它们。

这基本上是为页面添加书签的另一种方式，但与普通书签或阅读列表菜单不同，标签存储允许您同时打开存储组中的所有标签，如果您经常需要同时打开同一组网站，这将更加方便。

就我个人而言，保存标签很有帮助，因为我需要在工作日打开浏览器时打开 Google Drive、多个电子邮件账户、工作日历等等——我相信你们很多人都有同感。能够一次性加载我需要的所有页面节省了时间。

选项卡组保存目前只在桌面版 Chrome Canary 中可用，但它在最新测试版本中的出现意味着它应该很快就会推出到稳定版本中。您可以通过下载 Chrome 金丝雀版本 95 来提前试用，该版本可以在 Chrome 下载页面上找到。已经安装了 Chrome Canary 的用户可以在**设置>下更新 Chrome** 。

不过，请记住，Chrome Canary 是目前最具实验性的 Chrome 版本。这是谷歌未完成功能的试验场，因此容易出现错误、崩溃和其他问题。幸运的是，Chrome Canary 作为一个独立于稳定版和测试版的浏览器安装，所以如果你遇到麻烦，你可以很容易地切换到一个更安全的版本。

此外，像许多未完成的 Chrome 功能一样，打开 tab ggroup saving 需要更改浏览器的实验标志。如果你不小心的话，弄乱这些会导致问题；只要确保只更改与该特性相关联的标志，就可以了。然而，一旦该特性推出稳定版本，就没有必要再切换实验标志了。

1.  打开 Chrome Canary，进入 **chrome://flags。**
2.  使用搜索栏找到**“选项卡-组-保存”**
3.  点击**下的下拉菜单“Tab Groups Save”**选项，将其设置为**“Enabled”**
4.  当提示保存更改并重启浏览器时，单击**“重新启动”**。

你现在可以在 Chrome Canary 中保存和调用群组。它是这样工作的:

1.  在 Chrome Canary 中打开一个新标签，然后**在标签菜单中右键单击**并选择**“将标签添加到新组”**
2.  设定标签的名称和颜色(如果需要)。您还可以向该组添加更多选项卡。
3.  要保存选项卡组，请右键单击选项卡组的彩色圆圈。
4.  切换**“保存组”**如果您想在以后调用，这些标签现在会显示在您的历史记录中。

要恢复选项卡组:

*   点击浏览器右上角的三点菜单图标，进入**历史>最近关闭>选项卡>(选项卡组名)。**
*   您也可以通过按下 **Ctrl + Shift + T.** 来立即调用已关闭的选项卡组
