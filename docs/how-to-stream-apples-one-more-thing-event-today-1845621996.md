# 如何在今天直播苹果的“还有一件事”活动

> 原文:[https://life hacker . com/how-to-stream-apple-one-more-thing-event-today-1845621996](https://lifehacker.com/how-to-stream-apples-one-more-thing-event-today-1845621996)

苹果公司在一次新闻发布会上使用这个神奇的、乔布斯时代的词语已经有多久了？自2017年以来，随着不太的iPhone X发布会，苹果发布会也没有什么不同:这是一个关于新笔记本电脑的“秘密”发布会，自苹果6月份宣布将移动其基于ARM的MacBooks定制芯片以来，人们就一直在期待。

这就是我们今天从 美国太平洋时间上午10点开始看到的——因为除此之外，苹果今天还能预览什么呢？剧透:苹果不会炫耀专家们仍喜欢不时提起的“空中力量垫”。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-5AwdkGKmZ0I&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-5AwdkGKmZ0I&start=0) 

我们不知道苹果今天将首次亮相的笔记本电脑的完整阵容，但我们可以做出一个非常有信心的猜测。期待看到ARM MacBook Air和ARM MacBook Pro，因为不再有真正的准系统MacBook。苹果肯定会在其所有不同的MacBook外形中加入ARM芯片，这意味着你会看到苹果13英寸和16英寸MacBook Pro都有ARM替代产品。

是否所有的三款笔记本电脑都将在今天预展，甚至宣布正式发布日期(相对于“很快”)，还有待观察。这一切都取决于苹果公司能够做出什么样的东西来击败它目前的产品，或者它今天计划发布的任何其他基于英特尔的MacBooks，这感觉像是一个漫长的拍摄。

我不认为苹果展示的基于ARM的笔记本电脑会比它目前销售的(或即将发布的)最好的基于Intel的MacBooks还要慢。吹嘘银牌不是苹果的本性，尤其是当它是第一家试图推动ARM笔记本电脑性能*和*便携性的公司时。

我们不知道的一件事是，苹果的ARM MacBooks的性价比将如何与其英特尔macbook相抗衡；还有，当苹果ARM首次亮相时*他们*会发生什么。如果苹果的中央处理器降低了其当前一代macbook、*和*的性能，苹果可以在价格上削弱它们，我认为苹果只会对保留其基于英特尔的macbook感兴趣，以便在未来两年内为客户提供更温和的过渡。(当然，也是为了履行苹果与英特尔的任何合同。)

至于当苹果更新其英特尔macbook的芯片与基于ARM的macbook竞争(但可能不会击败)时，这将如何影响事情，我们将拭目以待。毕竟，MacBook就是MacBook。条条大路(和美元)通向库比蒂诺，但我不认为未来的新闻发布会上苹果会强调任何英特尔芯片优于自己的芯片，如果英特尔芯片 [甚至能像](https://www.techradar.com/news/apple-arm-cpu-benchmark-leak-vs-intel-core-i9-16-inch-macbook-pro) 那样应付自如的话。

所以，不要指望苹果从明天开始在苹果商店埋葬其基于英特尔的MacBooks。你会看到基于ARM和基于Intel的MacBook直接竞争一个更长的时间，这提出了一个奇怪的购买困境:你应该花钱买一个可能更快的基于ARM的MacBook，它(可能)有更好的电池寿命和更低的价格，但潜在的应用程序兼容性问题就在门外？还是应该选择速度较慢的基于英特尔的MacBook，在笔记本电脑过时之前，它可能会失去苹果的青睐？

我们明天会带着一些购买建议回来。不过，根据传言，如果我想尽快升级的话，我肯定会关注苹果的ARM MacBooks。我不认为应用程序兼容性是一个大问题，除非你真的需要在你的ARM MacBook上获得完整的Windows 10体验。否则，我无法想象苹果的新macbook会是你能买到的最好的macbook——而且很快，会是你能买到的唯一的macbook。
