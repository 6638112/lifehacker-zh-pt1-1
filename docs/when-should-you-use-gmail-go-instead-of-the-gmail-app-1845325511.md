# 什么时候应该使用Gmail Go而不是Gmail应用程序？

> 原文:[https://life hacker . com/when-should-you-use-Gmail-go-inst-of-the-Gmail-app-1845325511](https://lifehacker.com/when-should-you-use-gmail-go-instead-of-the-gmail-app-1845325511)

谷歌刚刚在Play Store 上取消了轻量级的 [Gmail Go应用，为安卓用户提供了两种在设备上使用谷歌电子邮件服务的选择。但是这两个Gmail应用有什么区别，你应该使用哪一个呢？](https://play.google.com/store/apps/details?id=com.google.android.gm.lite&hl=en_US&gl=US)

谷歌的“Go”应用程序是其常规Android应用程序的更轻、更精简的版本。Gmail Go拥有Gmail应用程序的所有基本电子邮件功能，以及可定制的手势控制、收件箱过滤、多帐户交换和黑暗模式主题。然而，有一些小的变化使Gmail在较慢的硬件上不那么费力。

长期使用Gmail的用户会注意到，Gmail Go没有集成Google Meet，所以应用程序中没有Google Meet横幅或视频通话快捷方式——至少目前没有。

去掉Google Meet的功能可以减少应用的安装规模，并有助于将重点放在核心的电子邮件功能上。也就是说，你不需要从Gmail直接进入nix Meet [如果你想要](https://lifehacker.com/how-to-get-rid-of-google-meet-in-your-gmail-1844481331) ，你可以在主安卓应用和桌面上隐藏Google Meet。

Gmail Go还放弃了Gmail应用程序的“分层”用户界面元素，而是使用微妙的轮廓来区分应用程序的各种按钮和功能。它还按比例缩小动画并限制帧速率。这些变化保留了内存，并使应用程序更容易在旧的或较弱的设备上运行。

Gmail Go是主要Gmail应用程序的替代产品，如果你的手机在运行标准Gmail应用程序时遇到问题，它可以发挥很大作用。 [看看](https://play.google.com/store/apps/details?id=com.google.android.gm.lite&hl=en_US&gl=US) 如果你还在使用旧的安卓手机，但如果你的手机已经能很好地处理Gmail，那就没有理由换了——它甚至不能安装在一些新设备上。(此外，如果你正在寻找一种 [替代电子邮件服务](https://lifehacker.com/ditch-gmail-with-these-alternatives-1829337583) ，还有很多其他选择。)
