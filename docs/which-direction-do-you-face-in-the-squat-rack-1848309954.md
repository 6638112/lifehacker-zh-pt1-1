# 在深蹲架里你面向哪个方向？

> 原文：<https://lifehacker.com/which-direction-do-you-face-in-the-squat-rack-1848309954>

当你蹲着的时候，要面对一个正确的方向和一个错误的方向。

我知道，我知道——众所周知，我能容忍各种不同类型和风格的举重。我告诉过你 [屁股对着草地蹲下或者在某些情况下甚至保持在平行上方](https://lifehacker.com/how-deep-should-you-squat-1841473363) 是可以的。但是当谈到使用深蹲架时，有向前和向后之分，是时候让你知道哪个是哪个了。



所以这里有一个大秘密:*当你蹲下的时候，你面向蹲架的挂钩*。以下是一步一步的步骤:

1.  你从杠铃所在的一侧接近深蹲架。你把你的手放在杠铃上，把你的头低下，然后站起来，把杠铃放在你的肩膀上。
2.  你后退几步*。*
3.  *你做你的深蹲。*
4.  *然后你向前*走*直到你的杠铃接触深蹲架的立柱，并向下滑动到挂钩上。*
5.  *重量现在回到架子上，你将你的头和肩膀从深蹲的架子上移开，只需在另一个 [三到五分钟](https://lifehacker.com/how-long-you-should-rest-between-sets-for-the-biggest-t-1782785683) 中重复这个过程，当你休息的时候。*

## *为什么你下蹲姿势的方向很重要*

*为什么重要？嗯，想想当你在一系列深蹲后筋疲力尽时，你将如何安全地重新爬上杠铃。如果你按照我的指示做，行李架就在你面前。你可以看到它在哪里，钩子在哪里，把横杆放在它需要去的地方。*

*如果你反其道而行之，从深蹲架后面靠近吧台，你将完成你的动作，然后需要*回到*深蹲架上。你不知道它在你身后到底有多远，你也不能放心地背着 200 磅(或其他什么)的东西扭着脖子朝身后看。你更有可能错过钩子，撞到架子上，或者冒险经历从令人讨厌到轻微伤害的事情。*

*还有另一个好处，如果你在没有安全装置的架子上训练。在举重馆和健身房，通常使用缓冲板，不考虑安全问题(尽管有这个名字，但从安全角度来看完全没问题)。在这种情况下，如果你下蹲并且不能把杠铃举起来，你只能把杠铃放在你身后的地板上。没什么大不了的。(这里有一个我做这个的视频。)*

*如果深蹲架在你面前，这不是一个问题；希望你确定你身后的地板上没有人或障碍物。但是如果深蹲架在你身后...你知道*离*到底有多远吗？有足够的空间保释吗？不要把自己放在一个问自己这个问题的位置上。首先正确地接近深蹲架。*
