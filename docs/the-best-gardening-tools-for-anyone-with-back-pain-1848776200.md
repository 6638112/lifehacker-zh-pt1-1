# 背痛者的最佳园艺工具

> 原文：<https://lifehacker.com/the-best-gardening-tools-for-anyone-with-back-pain-1848776200>

园艺季节充满了希望:新鲜的蔬菜、美丽的花朵和成就感。B 但是它也可以是一种— 字面意思— 颈部(和背部)的疼痛。跪、蹲、伸、抬、拉都是花园养护的一部分，有时这些事情甚至会让最热情的园丁也感到头疼。防止受伤会让你在院子里的时光更加愉快。这里有几个工具可以让你的拇指保持绿色，脊椎保持挺直。



## 把园艺工作交给你

避免背部受伤的最好方法是不要长时间弯腰。相反，试试把园艺工作*带给你*。

抬高的花园床是一种提升土壤并使园艺变得不那么—密集的方法。对于一些人来说，一两英尺是不够的，所以 [高出地面的滚动种植机](https://vegogarden.com/products/elevated-rolling-garden-bed?currency=USD&variant=39573841379363&utm_medium=cpc&utm_source=google&utm_campaign=Google%20Shopping) 是一个很好的选择。这些是可以购买的，但是它们自己构建起来相对简单，也是如此。它们可以升至桌子高度或站立高度，便于拿取，无需下跪或弯腰。你甚至可以拉一把凳子或椅子来除草、种植或浇水，这是一种友好的园艺方式。

如果你没有空间放置滚动式高架花盆，[窗盒](https://www.lowes.com/pd/27-95-in-W-x-9-84-in-H-Carbonize-Wood-Window-Box/999979132) 也提供高架园艺平台。窗盒不一定要放在窗户下面——它们也可以沿着栏杆或栅栏摆放。您也可以调整这些按钮的高度，和以获得最大的可及性。

## 选择蹲下和跪下的辅助姿势

对于更大的花园和庭院，你不能总是依赖于将作品搬到离你更近的地方，你必须找到一种安全的方式来接近地面。在这种情况下，让你的背部不紧张就能轻松够到东西是关键。一个带衬垫的跪凳让你更靠近地面，并在你行走时保护你的膝盖，而不是一直向下。

还有 [滚动凳](https://www.hayneedle.com/product/miraclegrorollnkneel4in1gardenaccessory.cfm?source=pla&kwid=GardeningAccessories%20level05&tid=DFOM005-1&srsltid=AWLEVJxidBqano0D3y1ybn_fk9WFmirXWEFLUYCIyS91EfdgLqdOdUSAmLA) 兼作储物和护膝，让你背上的装备更轻松，也是。对于那些已经有背部问题，但仍然迫不及待地想涉足污垢的人来说，一辆 [花园滑板车](https://www.plowhearth.com/https://www.plowhearth.com/p/55703?aff=6443) 是一个很好的选择。这些凳子比一般的花园凳子更结实，让你在工作时可以坐着。

要想少弯腰少蹲地浇水，试试 aT2】浇水棒 T4。在院子里工作时，站得更直，保持下背部不紧张，将有助于防止受伤。

## 立式除草机

给花园除草是背痛的一个常见原因，因为这通常需要蹲下或跪下用力。工具你可以站着使用，而利用你的身体重量意味着在征服杂草时，你将花费更少的时间弯腰和蹲下。

一些 [除草机](https://www.acehardware.com/departments/lawn-and-garden/gardening-tools/pruning-tools/7166549?store=16144&gclid=Cj0KCQjwgMqSBhDCARIsAIIVN1VA8lmfPMpvM5logbIXodEms4lEidSk4Ec0znMyZ5yGdVpzDsM1PxwaAsNaEALw_wcB&gclsrc=aw.ds) 的操作类似于花园耙子，将杂草连根拔起，而另一些则意在 [切断顽固的根](https://www.lowes.com/pd/CRAFTSMAN-48-in-Wood-Handle-Action-Hoe/1000752686) 。将这些与其他人体工程学改进结合使用，可以减少背部疼痛，减少除草时间。
