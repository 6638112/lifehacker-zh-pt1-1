# 这是美国

> 原文:[https://life hacker . com/stop-saying-this-not-America-1846017707](https://lifehacker.com/stop-saying-this-isnt-america-1846017707)

自从周三一群精神错乱的国内恐怖分子袭击了美国国会大厦大楼以来，在总统努力阻挠行政权力自由交换的刺激下，一些媒体和政治人物一直在努力克制，试图净化和划分在世界各地播出的徒劳政变企图的混乱图像。“这里不是美国，”他们吟诵道。“这不是我们。我们比这更好。”

我们虽然是吗？

延续美国作为与民主原则和平等主义理想相结合的正义的清晰典范的概念，是传播伟大的美国神话。然而，美国最重要的权力掮客仍然坚定不移地支持美国的大无畏正义:在周三的一次演讲中，当选总统乔·拜登表示，这一可耻的事件并不代表“真正的美国” 显然，这是一群来自*其他*地方的外来者——不要介意，在一个政治极端两极化的时代， [根据NPR/PBS新闻小时/马里斯特的调查，超过70%的共和党选民表示，他们不信任2020年选举的结果。](https://www.washingtonpost.com/politics/2020/12/22/do-republicans-really-believe-election-was-stolen-or-are-they-just-saying-that/)

充满希望的“这不是我们”的保证有效地粉饰了美国历史的污点——事实上，奴隶制、种族隔离、帝国主义、猖獗的不平等、枪支暴力、系统性的种族主义和其他一系列社会弊病是我们现代困境的前身。除此之外，这是一个很多人都觉得令人不快的老生常谈，因为它有效地抹去了那些生活在一个非常不同于想象中的国家的人的斗争，在这个国家里，正义占上风，每个人都有平等的机会实现美国梦。

我们不应该说“这不是美国”或“我们比现在更好”，而是应该以更清晰的眼光、更多的细微差别和敏感性来讨论下一次社会剧变。

### 为什么你应该停止说它

在大多数情况下，如果你说“我们比这更好”，那是出于善意和乐观。但这是一种简化的说法，尽管具有讽刺意味，但它强调了导致周三恐怖事件的情况。

如果你想谈论这个，你至少应该试着听起来像你没有掩盖这一集令人不舒服但明显的美国男高音。首先，你必须不仅在更广泛的历史背景下审视周三的事件，这些历史背景几个世纪以来一直在塑造美国的不平等，而且要特别指出过去四年是加速燃烧的煤油:唐纳德·特朗普(Donald Trump)的总统任期将阴谋论推向了主流，令人鼓舞地向白人至上主义运动眨眼，并催生了一个错误信息和“替代事实”的生态系统，这种生态系统在本周达到了顶点，其形式是由2020年大选被操纵的虚构故事所推动的起义。

你还需要理解笼罩在周三事件上的种族主义乌云:暴徒几乎清一色是白人，这一事实至少可以部分解释华盛顿特区和国会警察部队对试图破坏政府所在地的反应迟缓。相比之下，去年夏天黑人的命也是命特区的抗议活动吸引了军事化的警察，国民警卫队成员全副武装站在国会大厦的台阶上放哨。T3】

来到国会大厦的这批人是真正的美国人，他们当中有 [卡农阴谋论者、白人民族主义者、大屠杀否认者](https://www.nytimes.com/2021/01/07/us/names-of-rioters-capitol.html) ，以及至少一位来自芝加哥的 [公司首席执行官](https://www.businessinsider.com/chicago-tech-ceo-arrested-capitol-riots-2021-1) 。如果你想解释为什么我们不会比现在更好，你必须解释为什么我们不会比现在更好——这要从理解赋予周三起义明确无误的美国风味的力量开始。

### 请说这些事情

我们可以用听起来不那么冷酷无情或错误的方式来表达意见和哀叹事情的糟糕状态。美国的故事在很多方面是一部永恒奋斗的故事。值得注意的是，我们还没有真正实现全面的伟大，但将这个国家塑造成真正平等的灯塔的斗争必须继续。

不要同意“我们比这更好”，试着从更有抱负的角度思考。考虑一个更微妙的角度，说:“我认为我们可以做得更好，但是还有工作要做。”与其说:“这不是美国”，不如想想说:“这绝对是美国，但不一定是美国。”

忽视当前的美国现实就是否认许多人的声音，他们早就知道这个版本的美国存在 ，因为他们根本无法逃避。你当然不必仅仅因为时局艰难就抹去它们，或者向一个错误的美国理想致敬。
