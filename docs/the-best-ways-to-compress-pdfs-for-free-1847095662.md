# 免费压缩 pdf 的最佳方式

> 原文：<https://lifehacker.com/the-best-ways-to-compress-pdfs-for-free-1847095662>

你有没有注意到*网站对 PDF 的尺寸有多严格？无论是上传简历，提交学校项目，还是向同事发送重要文件，我们大多数人可能都发现自己需要压缩 PDF 文件，以满足网站秘密的 PDF 大小限制。*



虽然 [Adobe Acrobat DC](https://acrobat.adobe.com/us/en/acrobat/free-trial-download.html) 是压缩 PDF 文件的最佳方式，但压缩功能仅在该软件的付费版本中可用。在 T4，如果你以前没有使用过这款应用，你可以免费试用 7 天，但是一旦免费期结束，你就必须付费。幸运的是，有几种免费的方法可以在 PC 和 Mac 上立刻将 PDF 压缩到更易于管理的大小。

## 使用 Adobe online (Mac、Windows)缩小您的 PDF

Adobe 有一个免费的在线 PDF 压缩程序，任何人都可以使用。虽然它只支持高达 2gb 或更小的文件，但这对大多数用户来说应该足够了，并且有三种压缩级别—高、中和低—可供选择。

1.  打开[Adobe Acrobat Online PDF compressor。](https://www.adobe.com/acrobat/online/compress-pdf.html)T3】
2.  单击**“选择”**上传 PDF，或者将文件从您的计算机拖放到页面上。
3.  选择您想要的压缩级别，然后点击**“压缩”**
4.  点击**“下载”**将文件保存至您的设备。

## 使用 MacOS 预览版(Mac)缩小您的 PDF 文件

Mac 用户在预览应用中内置了 PDF 压缩工具。

1.  在预览中打开 PDF 文件。
2.  G o 到**文件>导出**或**文件>导出为…** 如果你想以新的名称导出压缩后的副本。
3.  将格式设置为**“PDF”**
4.  从石英过滤器下拉菜单中选择**“减小文件大小”**。
5.  点击**“保存”**

### 使用 Wondershare HiPDF 压缩您的 PDF (Mac、Windows)

Wondershare 提供了一个免费的 PDF 压缩工具，可从 [HiPDF 网站](https://www.hipdf.com/compress-pdf) 下载，作为浏览器扩展或离线桌面应用。

像 Adobe Online Compressor 一样，HiPDF 允许您在三个压缩级别之间进行选择，但只支持 10MBs 或 50 页长的文件，免费用户每天只能转换两个文件。对 [Pro 版](https://www.hipdf.com/pricing) 订阅用户解除了这些限制(web 6 美元/月起，web 和桌面 8 美元/月起)，Pro 版甚至可以一次压缩多个文件。

不过现在，我们将介绍免费版本的工作原理:

1.  打开 [HiDPF 压缩页面](https://www.hipdf.com/compress-pdf) 。
2.  点击**“上传”**选择保存在您电脑上的 PDF，或点击向下箭头按钮从您的 OneDrive、Dropbox 或 Box cloud drive 上传。
3.  点击**“添加”**上传额外的 pdf 进行压缩。
4.  使用滑块选择所需的按压级别。
5.  点击**“压缩”**开始压缩过程。
6.  点击**“下载”**保存压缩文件。
