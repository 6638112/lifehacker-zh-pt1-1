# 如何使用 Snapchat 扫描条形码获取葡萄酒和营养信息

> 原文：<https://lifehacker.com/how-to-scan-a-barcode-with-snapchat-for-wine-and-nutrit-1845455235>

Snapchat 最出名的是短期照片和视频——以及所有怪异的面部过滤器——但该应用程序可以做的远不止这些。一旦最新的 Snapchat 更新安装到你的设备上，你就可以通过“扫描”食品包装来查看其营养信息。更重要的是，你还可以通过给你的葡萄酒拍照来查找品尝细节和食物搭配建议。谁知道 Snapchat 可以升级你的侍酒师游戏？

Snapchat 的扫描仪只对包装食品有效。因此，虽然它可以帮助你了解你的早餐燕麦片包中有多少卡路里，但你无法通过扫描一块冷冻肉来追踪鸡胸肉的宏指令。

然而，你得到的不仅仅是基本的营养信息。该应用程序列出了食物的味道，以及食用它的营养“优点”和“缺点”——例如，如果它的糖分太高，或者与其他食物相比它富含蛋白质。搜索结果由优香营养追踪应用程序提供。另一方面，葡萄酒扫描仪从 Vivino 的应用程序中提取了超过 1200 万种葡萄酒的数据。

### 使用 Snapchat 的视觉搜索可以找到的其他内容

Snapchat 新的食物和葡萄酒扫描功能将面向所有人，但它们现在仍在推出。如果你还没有看到它们，也许可以同时试试 Snapchat 的其他视觉搜索过滤器。

有解数学方程的过滤器，在亚马逊上查找产品，甚至识别狗的品种或植物品种。还有一个内置的 Shazam 功能，当你试图找出电台播放的歌曲名称时。

要访问它们:

1.  打开 Snapchat。
2.  在相机选项卡中，点击滤镜图标。
3.  点击底部菜单中的**“扫描”**。
4.  滑动到左侧的一个可视搜索图标。
5.  在取景器中标出您要查找的项目，然后点击快门按钮下的**“扫描”**。

6.  一旦应用程序识别出扫描的商品，搜索结果就会在屏幕上弹出。
