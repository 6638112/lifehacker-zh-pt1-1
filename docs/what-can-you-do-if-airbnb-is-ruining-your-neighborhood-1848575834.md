# 如果 Airbnb 正在破坏你的邻居，你能做什么？

> 原文：<https://lifehacker.com/what-can-you-do-if-airbnb-is-ruining-your-neighborhood-1848575834>

罗马被蛮族攻陷，不是一朝一夕的事。同样的事情也可以说是因为像 Airbnbs 这样的短期租赁而被毁掉的社区。当你买了房子或签了公寓租约时，你可能认为你买了一个避难所，一个你可以放松、充电和过上最好生活的地方。你成为一个社区的一部分，因为你们都粘在一起，每个人通常都试图保持尊重。



然后，野蛮人以 Airbnb、Vrbo 和其他短期租赁平台的形式出现，这些平台允许人们短期出租他们的房屋或部分房屋。一方面，这一创新让数百万美国人获得收入或自由旅行，这是好事。另一方面，它也将许多住宅和公寓变成了基本上不受监管的迷你酒店。在你家附近短期租房会导致一长串的不幸: [喧闹的聚会，堵塞的停车场，以及财产损失](https://www.airbnbhell.com/tag/airbnb-neighbors/) 。如果你没有听过很多噩梦般的故事，那可能是因为 Airbnb [花了很多钱让那些看不见的](https://www.bloomberg.com/news/features/2021-06-15/airbnb-spends-millions-making-nightmares-at-live-anywhere-rentals-go-away) 。

## 短期租赁的问题是什么？

问题是这些客人不是你的邻居，所以他们可能觉得不受礼貌共处的常规约束。但短期租赁也缺乏传统酒店提供的那种员工和管理。所有者通常不在现场，甚至基本匿名。虽然大多数短期租赁进展顺利，但也有很多 Airbnbs 毁掉社区的例子。问题变得如此严重，以至于 Airbnb 和 Vrbo 不得不合作创建了 [一项倡议，关闭他们平台上的所谓“派对屋”，以避免一场公关灾难](https://news.airbnb.com/airbnb-and-vrbo-announce-partnership-to-address-party-houses/) 。即使是最尽责的主人也会发现，一旦他们把钥匙交给镇上的一群酒鬼，就很难控制会发生什么。

隔壁的 Airbnb 偶尔会有一群吵闹的工作人员是一回事，但当源源不断的陌生人破坏你的邻居时，就是另一回事了。那么你能做些什么来反击呢？实际上，很多。

## 和你的邻居谈谈你的担忧

套用乔治·克斯坦萨的话，我们生活在一个社会里。自从私有财产出现以来，邻居就一直在噪音、停车和路边吸引力问题上争吵不休，所以你的第一步应该永远是伸出手去和房主或 Airbnb 房东谈谈。很有可能他们没有意识到他们的客人正在引起的问题，他们在避免与邻居发生战争方面有经济利益。

这可能是街上的一次闲聊，或者是一个电话，如果你认识主人的话。通常，Airbnb 的主人会出现监督装修或清洁，或者交出钥匙，所以你也许能抓住一点时间说出你的担忧。如果你不知道隔壁的房产到底是谁的，你可以试试 [通过 Airbnb](https://www.airbnb.com/help/article/1428/contacting-a-host-without-booking-a-reservation#:~:text=Go%20to%20the%20listing%20and,trip%20before%20hitting%20Contact%20Host.) 联系房东，如果你有账户，也可以试试联系 [Airbnb 的客服](https://www.airbnbhell.com/contact-airbnb/) 。

Airbnb 还维护着一个 [【邻里支持】资源](https://www.airbnb.com/neighbors) ，如果其他方法都失败了，你可以试试。有时候，仅仅和主人接触就可以改变这种情况的基调；主人可以采取措施更有效地筛选他们的客人，而仅仅是开放沟通渠道就可以改善一切。

最后，考虑一下当你看到新租户时问候他们，并就噪音和其他问题进行一次文明的讨论。大多数人住在短期租赁的房子里，并不是想找人打架。只要提醒他们，附近还有其他人在关注着他们，通常就可以压制那些捣乱者。

## 查看有关短期租赁的规则

如果 Airbnb 的房东对你的担忧不感兴趣，公开表示敌意，或者给了你很多空头承诺(或者彻头彻尾的谎言)让事情变得更好，你的下一步就是做一些研究。全国范围内的 [立法](https://benkallos.com/press-release/37000-nyc-homes-airbnb-will-need-register-under-new-law-stop-illegal-short-term) 浪潮愈演愈烈，旨在限制 Airbnb 房东可以逍遥法外的行为。其中一些法律限制了主人一年中可以出租房屋的天数，而另一些法律禁止他们短期出租房屋，以防止讨厌的陌生人的旋转门影响你的生活。此外，任何城市都要求 Airbnb 的房东获得许可或注册他们的房产。在大多数情况下，只要打个电话就能知道你的邻居是否这么做了。

如果没有具体的法律可以参考，请联系当地的规划部门或分区委员会，了解您所在社区的分区是否对短期租赁有任何影响。如果你是业主协会(HOA)的成员，或者你的邻居有管理噪音或物业使用的私人契约、条件和限制(CC&Rs)，也要调查这些。在土地使用律师身上花点钱通常是有用的。他们可以帮助筛选法律和条例，告诉你你有哪些权利。

关键是用事实武装自己，这样你就知道你能做什么，不能做什么。例如，如果你知道你可以提交一份长长的官方投诉清单，你可以回到房东(或 Airbnb 本身)那里，再次尝试推理——但这一次，你拿着一根大棒。

## 就你的短期租赁问题与每个人联系

如果你掌握了真实的信息，而你与房东寻求和平解决的真诚努力失败了，那么是时候打几个电话或发几封电子邮件了。

当然， [向 Airbnb](https://www.airbnb.com/neighbors) 投诉。但是不要止步于此。如果你知道 Airbnb 违反了当地法规，请联系当地的分区或规划部门并提出投诉。如果你的市政当局要求执照或注册，而你的邻居没有遵守，那也扔掉那一角硬币。

如果主人没有违反任何规则，你的下一步可能是打电话给当地警察，投诉噪音。大多数社区都有“”法律条文，旨在防止任何“干扰财产使用”的行为，包括噪音或其他令人不快的行为。你也可以 [向商业改善局](https://www.peopleclerk.com/post/how-to-file-a-complaint-with-the-better-business-bureau) 投诉 Airbnb。

## Airbnb 威胁的核选项

如果其他方法都失败了，考虑提起诉讼。你可能有理由声称噪音和其他干扰构成了“私人滋扰”，你可以 [向小额索赔法庭](https://www.peopleclerk.com/post/how-to-sue-airbnb) 起诉，只要你只是寻求损害赔偿。J 必须检查你当地管辖区的金钱赔偿限额。如果你想寻求禁令来阻止你的邻居出租他们的财产，事情显然会变得复杂一点——但这可能是值得的，取决于你有多痛苦。

大多数 Airbnb 的主人都是只想赚点外快的普通人，大多数关于短期租赁的纠纷都可以通过一个电话解决。但是当他们不能的时候，至少现在你知道你的选择。
