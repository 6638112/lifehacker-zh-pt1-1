# 使用“透明笔记”应用程序完成下一次虚拟会议或面试

> 原文：<https://lifehacker.com/use-the-transparent-note-app-to-get-through-your-next-1848660749>

我们都不是超人:尽管我们试图记住每个话题或每个问题，但如果没有偶尔的参考，我们就无法以最佳状态工作。尽管如此，如果我们在采访或演讲时永远不需要将目光从摄像机移开，这似乎更令人印象深刻，尤其是在视频通话中。那么，解决办法就是想办法同时看你的笔记*和*视频通话。W 我们找到了一款可以帮助解决这个问题的应用:它叫做透明笔记，它的名字叫不要乱来。

Watch

透明笔记是一个简单的笔记应用，除了一个关键的不同:你的笔记可以是透明的，允许你在 Mac 或 PC 上看到笔记背后的内容。苹果用户可以免费下载这款应用程序(付费墙后面有高级功能)，但 Windows 用户需要支付 4.99 美元。

## 透明笔记如何改善您的视频通话

这种类型的应用有很多使用案例，但视频通话是我使用最多的地方。如果你正在面试一家公司，如果你被卡住了，你不再需要尴尬地看一眼你的简历或你准备好的问题清单。相反，你可以在视频通话上覆盖所有相关的谈话要点:你仍然像预期的那样看着屏幕，但你可以提前阅读你做的任何和所有笔记。

同样，如果你正在为一份的工作面试另一个人,这个方法也适用。你可以参考你对候选人的问题，以及他们简历中任何你认为相关的注释。做演讲怎么样？你可以边说边直接照着稿子念，不会让 Zoom 上的人知道。

## 如何使用透明注释

设置透明笔记很简单:当你启动它时，你可以像任何笔记应用程序一样开始写新笔记，但现在你的笔记将是透明的。如果您想看到更多或更少的背景，您可以通过点按左下角的设置齿轮来更改透明度级别。

在这里，你还可以更改便笺的颜色，从白色、黄色、蓝色或绿色，并选择便笺是否会停留在所有窗口的顶部，或者当你切换应用程序时它是否会消失。

由于 Mac 版本可以免费下载，它缺少了两个高级功能:一次只能做一个笔记，并且会遇到广告(尽管在我的体验中到目前为止还没有看到)。如果你想做多个笔记，和/或你想消除这些广告，你可以每月支付 1.99 美元，每年 9.99 美元，或一次性支付 99.99 美元的终身使用费用。

然而，对我来说，透明笔记不会是我主要的笔记应用程序。在视频通话之前，我可能会将我的谈话要点复制粘贴到我的一个免费笔记中，并在 doc 上草草记下通话笔记。如果这听起来也像你，你可能会很好地使用免费版本。
