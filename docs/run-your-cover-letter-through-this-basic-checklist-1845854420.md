# 将你的求职信按照这个基本的清单运行

> 原文：<https://lifehacker.com/run-your-cover-letter-through-this-basic-checklist-1845854420>

可怕的求职信可能看起来像是一种过时的形式，但是根据 ResumeGo 的调查，87%的招聘专业人士说他们*确实*阅读求职信。为了增加你被录用的机会，你的求职信应该包括以下内容。



### **确保你有正确的格式**

一封求职信应该大约有 400 个单词长，或者三到四个段落，字体大小在 10-12 之间。保持简洁——求职信不应该超过一页。ResumeGenius 有一些很好的例子来说明你的求职信应该是什么样的。

一般来说，第一段会解释你为什么申请这份工作，第二段强调你的成就，接下来的段落会传达你的经历如何与未来雇主的目标或要求相一致。结尾部分应该感谢雇主的时间和考虑。

### **引人注目的第一段**

你确实有几秒钟的时间来引起招聘经理的注意。你的写作必须简洁明了，重点突出雇主在潜在的新雇员身上寻找什么——通过传达对工作的热情，强调推荐人(如果你有推荐人的话)，或者列出与你申请的工作相关的令人印象深刻的成就。天平在这里 有一些很好的开句 [的例子。](https://www.thebalancecareers.com/cover-letter-opening-sentences-examples-2061030) 

然而，需要注意的是:不要在你的介绍中浪费太多描述对工作热情的词语。你对这份工作的兴趣是显而易见的(否则你为什么要申请？)而且不会把你和其他候选人区分开来。记住，雇主最感兴趣的是你的工作经验和技能，所以你会想尽快了解这一点。

### **包括相关工作成就的例子**

这里的关键词是*相关*。仔细阅读(或重读)招聘启事，准确理解*雇主在寻找什么，并以此作为你描述工作经历时的写作指南。你会想要模仿招聘启事的语言，尤其是当它列出技能的时候。例如，如果帖子中多次提到 Salesforce，而你有该平台的经验，你会想强调这是你技能的一部分。* 

你还需要突出任何成就，最好是一个展示职业成长的小故事，有开头、中间和结尾。举个例子，假设你在客户服务部工作:展示你是如何达到所有的绩效指标，如何解决问题，以及如何被提升为团队领导的。

### **证明你了解你申请的公司**

在你的求职信中讨论公司的价值观可能是让你从其他候选人中脱颖而出的最简单的方法，但是很少有人这么做。为什么？因为不容易知道写什么，而且当你一天申请多份工作时，工作量很大(不能保证会有面试)。招聘经理不想看样板求职信——他们想雇佣一个“明白”并热衷于为公司工作的人。谷歌一下这家公司，在他们的网站上查找任何描述“使命宣言”或“公司价值观”的内容了解这些价值观对你也有好处，因为你不会想加入一家文化不适应的公司。

### **设定正确的基调，不要使用**陈词滥调

求职信的正确基调是专业、友好、热情，不浪费任何语言——你不想呆得太久而不受欢迎。如果你在语气上有困难，试着大声朗读你的信:它听起来像你想与之交谈的正常人吗？如果没有，你的写作可能会缺乏诚意，这可能是依赖太多陈词滥调的结果。(当然，陈词滥调是让人昏昏欲睡的陈词滥调。)

挑战自己，避免无用的短语，比如“我是个优秀的挣钱者”或者“我跳出框框思考”，因为招聘经理已经读过它们成千上万遍了。TopResume 网站上还有一些你想要剔除的其他 [陈词滥调的好例子](https://www.topresume.com/career-advice/cliches-cut-from-cover-letter) 。

### 确保你已经发现了错别字

总是，总是，*总是*校对你的求职信。记住，招聘经理是在找理由驳回你的申请，你求职信上的一个小小的打字错误都可能让你失去机会。
