# 这个建议将使寻找一辆新车不那么令人沮丧

> 原文:[https://jalopnik . com/this-tip-will-make-searching-for-a-new-car-less-frustra-1849686217](https://jalopnik.com/this-tip-will-make-searching-for-a-new-car-less-frustra-1849686217)

试图在目前的市场上找到一款新车是一件令人沮丧的事情。购买者发现 [很大一部分广告中的汽车都卖不出去，](https://jalopnik.com/why-are-dealers-running-ads-for-cars-they-dont-have-1849460994) 以及那些在售的汽车经常会受到市场溢价的影响。我发现了一个搜索策略来增加你在这个市场的成功机会。

作为一个专业的汽车购买者，我一次又一次地从消费者那里得到的问题是:“为什么所有这些做广告的汽车实际上并不出售？” [我在之前的一篇文章](https://jalopnik.com/why-do-websites-show-so-many-cars-that-arent-available-1847678738) 中更详细地介绍了这一点，但是，概括一下，这与经销商用于库存的自动列表系统有关。实质上，一旦汽车被“运送”到经销商处，库存列表就会自动填充，然后该列表会被踢出到各种第三方网站，即使该汽车已经“售出”

我经常每周在不同的城市运行几十个新车和二手车搜索，当涉及到新车上市时，我已经看到了一个模式，它与这些广告的显示方式有关。经销商广告使用汽车的“库存照片”而不是实际销售的汽车，这种情况出现了大幅上升。它们看起来会像这样-

正如你所看到的汽车图片不是真正的RAV4，而是该车的一般形象。如果列表使用库存照片，很有可能汽车不在停车场上，而是在“运送中”到经销商处。我注意到的畅销车型的模式是，根据市场区域的不同，80-90%带有库存照片的列表是“预售”给客户的汽车。我给各种各样的经销商打了数百个电话，但每次得到的回答都是一样的:“对不起，那辆已经卖了。”在罕见的情况下，股票照片上市可用，经销商通常有一个很大的加价…因此，为什么汽车仍然可用。

[在这个市场上，我不断向买家重复的最大秘诀是灵活性](https://jalopnik.com/writer-s-choice-2021-here-are-some-tips-to-navigate-wh-1846991534) ，如果你在时间表上灵活，或者在购买品牌/型号上灵活，你就增加了成功的机会。许多人只是没有奢侈的等待几个月的新车，所以他们需要专注于什么是现在可用的。如果你发现自己在那个位置，寻找有最多照片列表的汽车。

例如，如果我在休斯顿地铁运行RAV4搜索，第一页都是库存照片，这可能意味着这些汽车实际可用和/或以合理价格提供的机会低得多。其中一些广告甚至不会宣传一个数字，而是 [“喊价”，这在历史上不是一个好兆头。【T2](https://jalopnik.com/heres-what-call-for-price-on-a-dealers-website-really-m-1820001720)

如果我对大众途观进行同样的搜索，大约一半的广告清单是现场照片，其中一些甚至广告价格低于建议零售价。

如果需要跨界车的买家现在可以在几个月内选择RAV4和/或以高于MSRP的价格购买或以潜在折扣购买途观，大众可能不是寻找丰田的首选，但大众似乎不是一个坏主意。

如果你想买一辆新车，请记住，仅仅因为一辆车做了出售广告，并不意味着它真的可以买。如果你没有能力为你的理想模型等上几个月，在当前可用的产品中磨练你的搜索将会减少你在这个疯狂市场中的挫败感。同样，现在买些东西只是为了“拖住你”，直到市场变得对你更有利，这也没什么错。
