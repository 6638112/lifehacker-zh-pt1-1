# 我想是时候谈谈睾丸晒黑了

我不时会遇到人们在健身或健康或 [生物黑客论坛](https://lifehacker.com/what-is-biohacking-and-is-any-of-it-not-stupid-1848469897) 上谈论的事情，这些事情似乎是揭穿真相的时机成熟了，只是这个想法还没有主流到足以让 Lifehacker 报道。晒晒你的蛋蛋——或者更好的是，把它们暴露在 [一个价值 1649 美元的红灯面板](https://joovv.com/products/joovv-solo-3-0)——是逃避我的“至少我不用写这个”心理清单的最新方法。

那么，像卡尔森视频中提到的那样，让你的球袋暴露在红光下有“大量的好处”吗？嗯，人们声称从实践中有很多好处，但是实际的科学还不能支持它们。出售这种灯的公司和赞美这种灯的生物黑客往往会引用一些在动物或试管中进行的研究，或者是在人类身上进行的研究，但这些研究并不针对睾丸激素、睾丸或阴囊。

那么睾丸晒黑对你有什么好处呢？一种假设是，红光有助于线粒体产生更多的 ATP，这有助于睾丸中的间质细胞产生更多的睾丸激素。另一个假设围绕维生素 D，一些研究表明，睾酮水平低的男性体内维生素 D 含量低。

但是这些想法存在问题。举一个明显的例子:维生素 D 不是专门在阴囊皮肤中产生的。你可以通过晒晒你选择的任何身体部位来增加你的维生素 D，或者简单地通过多吃一些含有维生素 D 的食物，比如富含脂肪的鱼。

当谈到红光对线粒体的影响时，这可能是真的——在*皮肤细胞*中。睾丸是内部器官，光线最多只能穿透皮肤几毫米。有对皮肤起作用的光疗法，但是并没有真正合理的方法让你的睾丸增加睾丸激素的产生，仅仅因为有光照射在阴囊的皮肤上。

那么，为什么有些人认为它一定有效呢？视频剪辑中的光疗法支持者安德鲁·麦戈文(Andrew McGovern)说，当人们“不相信主流信息”时，他们就会接受他所说的“溴病”，这可能是正确的。这听起来对我来说是准确的，这是一种助长各种伪科学实践的现象，当我在 Goop 峰会 上寻求健康建议时，我亲身体会到了这一点。又是同样的事情。用光照射你的蛋蛋有一种医生不想让你知道的秘密健康秘诀的感觉，然而你可以自己用太阳免费做这件事。或者你可以像专家一样研究光疗设备，然后 [穿着裤子站在一台光疗设备前查看你的电子邮件并给你妈妈打电话](https://www.menshealth.com/health/a19539973/i-put-a-giant-red-light-on-my-balls-to-triple-my-testosterone-levels/) ，生物黑客本·格林菲尔德写下了他的经历。

这也可能感觉有点特别和尖锐。还记得几年前在 instagram 上摆姿势的那位女士“ [晒[她]屁股和 yoni](https://www.instagram.com/p/B34mk7shhrH/) ”？她 [曾说](https://metro.co.uk/2019/11/27/meet-influencer-says-sunning-naked-bum-boosts-energy-sex-drive-11228517/) “作为一个女人，它把我和我的子宫空间联系得更紧密，”这对我来说听起来很像格林菲尔德热衷于感受“我胯下独特的神圣、温暖、刺痛的光芒”，然后和他的伴侣度过一个美好的约会之夜。

所以，你看:睾丸晒黑的秘密被揭开了。当下一个趋势走向相反的方向时，不要感到惊讶，就像坐在阴囊上冰袋一样，因为 [也应该增加你的睾丸激素](https://www.ergogenic.health/blog/icing-testicles) 。
