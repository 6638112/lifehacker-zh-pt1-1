# 简历什么时候可以超过一页？

> 原文:[https://life hacker . com/when-a-resume-be-than-one-page-1846773753](https://lifehacker.com/when-can-a-resume-be-more-than-one-page-1846773753)

当谈到商务礼仪时，似乎存在一些坚定不移的标准，没有任何明确的规则解释*为什么*——考虑一下你总是会在面试后发一封 [感谢信](https://lifehacker.com/do-you-really-need-to-send-a-thank-you-note-after-a-job-1846667904) 的期望。虽然选择 [中的文件格式来发送你的简历](https://lifehacker.com/should-i-send-my-resume-as-a-pdf-or-word-document-1846128229) 可能取决于你为什么要发送它，但另一个常见的问题困扰着许多求职者:那份简历能有多少页(或应该有多少页)？

传统智慧通常认为一页纸的简历是最好的选择，因为这将为潜在雇主提供对你的才能和专业知识最简洁、一目了然的总结。但是如果你有大量的经验和值得注意的荣誉，为什么要把自己局限在这样一份简略的简历中呢？正如大多数与找工作有关的事情一样，发现他对这些问题的回答并不确定，而是需要考虑更多的指导原则——但是如果你记住一些基本原则，它们就不会成为太大的问题。

### 初级员工应该有更短的简历

自然，很容易理解为什么一个刚刚起步的人的简历会更短。他们的职业生涯才刚刚开始，他们没有足够的经验来扩展一个经验丰富的专业人士的简历。所以我如果你刚刚开始，不要觉得有必要用高中的荣誉和大学俱乐部来填充你的简历——坚持与你申请的职位相关的内容。

但如果你是一个更有经验的员工，正试图转行，也可能是因为你没有很多与你正在进入的新领域完全相关的经验，因此你的简历可能需要精简。人们常说，招聘人员会花六秒钟 浏览一下的简历，然后再决定是否要进一步考察一名候选人，所以你必须简洁，同时尽可能展现自己最好的一面。正如Moira Lawler为招聘平台怪物所写的，你应该强调那些既能解决你所拥有的经验又能表明你理解你所申请的工作的具体要求的关键词。她指出，要记住的最重要的事情是:

> 使用关键词，量化你的成就，最重要的是，保持简洁。这样，你的简历会给招聘人员留下良好的第一印象，因为它通常是你申请公司时的第一个接触点。

换句话说，如果你的简历真的很短，重要的是简历中列出的内容说明你了解工作的要求，并向你展示你有能力满足这些要求。

### 简历什么时候可以(也应该)更长

当我刚开始工作的时候，一位年长的同事告诉我，我的简历必须是一页纸，没有例外。从那以后，我并没有完全遵守这条规则，这个决定也没有对我的职业生涯产生任何不利影响(至少，我现在有工作)。

如果你打算写一份很长的简历，你必须充分利用其中的每一寸空间。一份简洁经济的一页简历远比一份散漫不连贯的两页简历好。但是，如果你有很多相关的经验和成就可以炫耀，把你的资历拉长到两页甚至三页也没什么不好。

职业资源网站Ladders的创始人兼首席执行官马克·塞内代拉(Marc Cenedella)， 提出了一条通用规则:对于已经工作了10到25年的人来说，简历必须有两页纸。对于职业生涯不到十年的人来说，一页的简历可能就足够了。

谈到走得更远，他建议:

> 两页足够详述你的背景、成功和过去的任务。这个长度可以让你展示你几十年来的职业发展，同时提供足够多的过去五年的细节，让你的简历给人一种你的能力和成就的强烈感觉。

如果你碰巧属于那些工作经验不到10年，但仍有特别的奖励或成就值得注意的员工，那么协议应该尽可能简洁，即使你不得不翻到第二页。简而言之，你不希望它看起来像是你在*试图*将你的经历延伸到一页以上，这样，如果你这样做了，你可以自信地说，这是因为你的成就值得额外的喘息空间。
