# 服务犬和治疗犬的区别

> 原文：<https://lifehacker.com/the-difference-between-a-service-dog-and-a-therapy-dog-1848627962>

我们可能都认为我们的狗对我们的日常生活是必不可少的，并希望我们可以带他们去任何地方。许多狗实际上有真正的工作 ，每天和它们的人类一起工作，不管是放牧牲畜、协助搜救队、还是安慰医院的病人。但并不是所有的工作犬都是服务犬，在公共场所有特定的法律权利。



下面是服务犬、治疗犬和其他最优秀的工作犬的区别。

## 什么是服务犬？

服务犬 [提供美国残疾人法案(ADA)定义的一种特定类型的援助](https://www.akc.org/expert-advice/training/service-dog-training-101/#:~:text=A%20service%20dog%20is%20trained,impaired%20individuals%20navigate%20their%20environments.) ，这意味着它们经过正式训练，可以执行任务，为“身体、感官、精神、智力或其他精神残疾”的人提供支持 [例如](https://adata.org/guide/service-animals-and-emotional-support-animals) :

*   盲人或视力受损者的导盲犬
*   向对声音不敏感的人发出警报的信号犬
*   在癫痫发作或低血糖发作时提供保护或寻求帮助的医疗支持犬
*   [精神支持犬](https://www.instagram.com/wafflenugget/?hl=en) 协助患有创伤后应激障碍等精神健康状况的训导员
*   帮助人类使用轮椅或助行器的助行器
*   帮助自闭症患者的感觉信号狗

服务犬的任务必须与其人类训练员的残疾直接相关。他们被认为是工作犬，而不是宠物——根据 ADA，他们有权完全进入其他动物不允许进入的公共场所。这就是为什么你可能会在餐馆、飞机上、不方便养宠物的房子和旅馆里看到免费的服务犬。

请注意，ADA 并不要求服务动物穿特殊的背心或标签来识别它们，所以(或没有)不一定意味着狗的工作。

## 什么是治疗犬？

治疗犬在特定的环境中为人们提供安慰，比如医院、诊所、学校、治疗办公室、和疗养院。他们也可能在创伤事件或自然灾害后被带来支持人们。在一些地方，治疗犬也被允许出现在法庭上，以帮助那些正在经历法律或审判程序的人。

治疗犬和类似的辅助动物没有被训练来为特定的残疾人完成特定的任务，所以它们没有受到 ADA 同样的保护。然而，他们被期望处理他们环境的需求，例如不寻常的声音和被陌生人触摸。

虽然有训练和认证治疗犬的组织，但过程并不规范。公共访问可能是有限的，他们可能需要支付标准的宠物费用。注意 [情感支持动物](https://www.akc.org/expert-advice/news/everything-about-emotional-support-animals/) 虽然安慰主人，但是是宠物，不是有公共访问权的服务犬。

## 什么是工作犬？

如上所述，许多狗还有其他本质上不具有治疗作用的工作。搜救犬、爆炸物探测犬和牧羊犬就是几个例子。虽然这些狗在工作时确实有非常专注的任务要完成，但根据美国残疾人法案，它们也不被视为服务动物。
