# 停止使用这些召回的 fitbit，可能会导致烧伤

> 原文：<https://lifehacker.com/stop-using-these-recalled-fitbits-that-can-cause-burns-1848617138>

根据美国消费者产品安全委员会 (CPSC)的一份公告，健身技术制造商 Fitbit 已经宣布自愿召回在美国销售的大约 100 万只 Ionic 智能手表，此前有报道称该产品的电池过热，以至于灼伤了佩戴者的皮肤。召回还包括在国际上销售的 693，000 块手表。

## 为什么 Fitbit Ionic 智能手表会被召回？

在收到世界各地购买了 Ionic 智能手表的客户受伤的报告后， [Fitbit 进行了内部调查](https://help.fitbit.com/en_US/ionic.htm) 以查明原因。事实证明，手表中的锂离子电池可能会过热，使佩戴者面临皮肤灼伤的风险。根据 Fitbit 的说法，有问题的电池只出现在“非常有限的情况下”，召回是“出于充分的谨慎”

截至 3 月 2 日，Fitbit 已经收到至少 175 起手表电池过热的报告，其中 118 起导致烧伤，包括二度和三度烧伤， [CPSC 在其召回公告中表示](https://www.cpsc.gov/Recalls/2022/Fitbit-Recalls-Ionic-Smartwatches-Due%20to-Burn-Hazard-One-Million-Sold-in-the-U-S) 。

## 如何判断您的 Fitbit 是否属于召回范围

Fitbit 在 2017 年至 2020 年期间生产了他们的 Ionic 智能手表，它们在 2017 年 9 月至 2021 年 12 月期间可以从包括百思买、美国柯尔百货公司、塔吉特和亚马逊(以及 Fitbit.com)在内的零售商处购买。这些手表售价从 200 美元到 330 美元不等。

所有 Fitbit Ionic 手表都被召回，无论颜色如何，也无论个别设备是否过热。 [爱奥尼亚的型号为 FB503](https://www.cpsc.gov/Recalls/2022/Fitbit-Recalls-Ionic-Smartwatches-Due%20to-Burn-Hazard-One-Million-Sold-in-the-U-S) 。您可以在设备背面靠近表带的地方找到它，在“ce”标志下方和表扣上方。

Ionic 的其他一些特性包括:

*   1.4 英寸彩色液晶显示屏，带三个按钮
*   配有聚氨酯带
*   跟踪活动、心率和睡眠
*   手表正面印有“Fitbit”
*   颜色:石板蓝/焦橙色、炭灰色/烟灰色、蓝灰色/银灰色，以及与阿迪达斯联合推出的墨蓝色/银灰色特别版

## 如何获得 Fitbit Ionic 智能手表的退款

首先，停止佩戴手表(如果你还没有戴的话)。然后， [联系 Fitbit](https://www.fitbitionic.expertinquiry.com/) 开始退款流程。最简单的方法是 [访问他们专门的召回网站](https://www.fitbitionic.expertinquiry.com/) ，但你也可以拨打他们的召回热线，在美国是 1-888-925-1764。美国以外的人可以在 Fitbit 的召回公告上找到他们国家的号码。

一旦您提交了初始信息，该公司将向您发送预付费包装，以便您能够免费寄回手表。当该公司收到你被召回的手表时，你将获得 299 美元的退款，以及根据 CPSC 的规定，选择 Fitbit 设备 [的 40%的折扣代码。](https://www.cpsc.gov/Recalls/2022/Fitbit-Recalls-Ionic-Smartwatches-Due%20to-Burn-Hazard-One-Million-Sold-in-the-U-S)
