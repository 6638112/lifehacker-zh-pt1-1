# 如何培养移情能力

> 原文:[https://lifehacker.com/how-to-parent-an-empath-1848876177](https://lifehacker.com/how-to-parent-an-empath-1848876177)

虽然大多数人都有移情的能力，设身处地为他人着想，感受他们的感受，但移情者感受他人的感受或身体感觉，就好像是自己的一样。当一个孩子是移情者时，他们会感到困惑，常常会失去控制。作为父母，很难知道对一个似乎对一切都有感觉的孩子来说什么是最好的。

您的孩子可能是移情者的一些迹象包括:

*   对声音、人群或光线等外部刺激过敏
*   容易伤害感情
*   容易被其他人伤害的感情所困扰，比如看到一个同学被欺负
*   比起玩伴或团队运动，更喜欢独自玩耍
*   放学或活动后感到特别累或精疲力尽

虽然这不是一个详尽的列表，而且移情别恋不像精神疾病一样被诊断为移情别恋——1%到2%的人可能是移情别恋者，所以你的孩子可能是其中之一。

移情者报告说，当他们不知道如何管理每天袭击他们的情绪波动时，他们会感到疲惫、沮丧和恐惧。我们希望保护我们的孩子免受这种负面经历的影响，如果你怀疑你的孩子可能有移情特质，你可以帮助他们以这样一种方式驾驭他们的世界，让他们甚至可以把他们的移情特质视为一种天赋。

## 培养他们最好的环境

移情者往往是 [高度敏感的人](https://lifehacker.com/how-to-challenge-your-highly-sensitive-child-1840270441) 。他们周围的世界很容易压倒他们，因为他们比一般人获得更多的刺激。辛辛那提大学医学院的精神病学家兼助理教授 [莎娜·费贝尔](https://www.uchealth.com/physician/shana-feibel/)说，对于父母来说，“减少环境混乱以减少分心是至关重要的。她说“杂乱”可以包括视觉、听觉和纹理刺激。

你的孩子可能更喜欢中性色而不是亮色。Feibel建议，你的孩子可能想带着耳机、耳塞或白噪声旅行，当谈到质地时，可能更喜欢柔软、宽松的衣服。

因为移情者有时会从陌生人那里获得情感反馈，他们通常不喜欢拥挤的地方，如杂货店或室内游戏场所。如果你的孩子在团队运动中不知所措，他们可能更喜欢其他活动或放学后的休息时间。

## 帮助他们设定限制

移情者通常比其他人需要更多的独处时间，可能比较内向，需要独处来充电。对于那些经常感到压力的父母来说，过度安排和过度社交他们的孩子，这可能是一个很难的调整。费贝尔说，“制定并维持一个时间表，限制孩子一天必须完成的责任量，这样他们就不会变得不堪重负，这会很有帮助。”

作为他们的父母，你可以帮助你的孩子变得更加自我主张。

“因为移情者可能没有意识到他们何时变得过度疲劳或过度饥饿，他们很容易过度伸展自己，” Feibel说。“教会他们倾听自己的身体暗示非常重要，这样他们就可以在很小的时候开始调节自己。”

教孩子对事情说不并设定界限将有助于他们长大，他们需要设定与朋友和浪漫伴侣的情感界限。

## 管理自己的需求

你是你移情的重要人物，因此，你目前的状态会影响他们。为了你自己，是的，也为了你的孩子，照顾好自己的身心健康。

“花你需要的时间来支持自己，并在必要时让别人支持你，”Feibel说。"这是一段漫长的旅程，为了你自己和你感同身受的孩子，保持健康至关重要

让你的周围围绕着一群爱你和支持你孩子的人。你的孩子有一颗宽广的心和丰富的感情，他将在一个家庭和朋友的圈子里茁壮成长，这些家庭和朋友接受他们，给他们所需要的东西，让他们成为最好的、最可爱的自己。

## 爱他们本来的样子

“移情者需要比一般孩子更多的情感支持，”Feibel说。"不要担心你会因为格外关心培养和支持他们而宠坏他们。"

虽然可以理解你可能会把你的孩子与其他能更好地“应对”这个世界的人进行比较，但费贝尔说，“理解你的孩子对于养育他们是必不可少的”，费贝尔说。
