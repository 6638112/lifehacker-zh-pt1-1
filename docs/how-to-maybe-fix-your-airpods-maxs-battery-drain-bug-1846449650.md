# 如何(或许)修复你的AirPods Max的电池耗尽Bug

> 原文:[https://life hacker . com/how-to-maybe-fix-your-ai rpods-maxs-battery-drain-bug-1846449650](https://lifehacker.com/how-to-maybe-fix-your-airpods-maxs-battery-drain-bug-1846449650)

一些AirPods Max用户遇到了一个奇怪的电池耗尽问题，即使在不使用时也会影响耳机，但可能有一个简单的修复方法。

Watch

根据众多社交媒体和论坛帖子，AirPods Max可以迅速失去电池，甚至在坐在智能外壳中时也可以在短短几个小时内完全耗尽(应该会将它们置于超低功耗模式)。一些用户说，他们的耳机在一夜之间从充满电降到只有1%。

苹果公司尚未对该问题发表评论，也没有任何官方解决方案，但有一个可能的解决方案:苹果公司刚刚发布了一个新的AirPods max固件，版本3C39。苹果不像iOS和macOS一样提供AirPods更新的补丁说明，所以我们不知道这个新固件有什么变化。希望，电池耗尽问题的修复包括在内。一旦更多的用户安装了固件3C39，我们就可以确定了。

不幸的是，它正在慢慢推出，所以我们可能需要一段时间才能知道。也就是说，即使我们不能确认3C39版本修复了AirPods Max的电池寿命，但尽快安装更新仍然是明智的。

### 如何更新你的AirPods Max

当耳机通过蓝牙和充电连接到iPhone时，AirPods固件更新会自动安装。整个过程是间接发生的；如果有可用的补丁，它将在后台下载并安装。这使得系统维护变得容易，也意味着你可能已经安装了补丁，但你需要通过iPhone的设置来确认你的AirPods Max固件版本。

以下是检查方法:

1.  打开iPhone上的设置应用程序*。*
2.  轻触**“蓝牙”**
3.  确定您的AirPods Max耳机已打开，并且列在“连接的设备”栏中。如果没有，请在继续之前将它们与您的iPhone配对。
4.  轻按AirPods Max旁边的信息图标。
5.  检查**“固件版本”下列出的数字**

如果您的AirPods Max list版本为3C39(或更高)，那么您拥有最新的固件。之前的版本号是3C16，所以如果这是你看到的，那么你需要更新你的固件——尽管我们之前说过，你不能手动执行更新。

请注意，安装固件更新可能需要30分钟。让你的AirPods Max在充电器上停留半个小时，然后再次检查。如果固件仍然是旧版本，您可以尝试强制更新:

1.  在你的iPhone上，进入**设置>蓝牙。**
2.  在“连接的设备”列表中，轻按AirPods Max旁边的简介图标。
3.  轻按**“忘记这个设备。”**
4.  将您的Airpods Max插入充电器。
5.  在你的iPhone上，回到**设置>蓝牙。**
6.  从“其他设备”列表中寻找您的AirPods。轻按以将耳机与iPhone配对。
7.  等待30分钟，然后再次检查固件版本。

这不是一个有保证的解决方法，但是值得一试。如果你尝试强制更新，但*仍然*没有看到它被安装，有可能新的固件还没有为你的设备准备好。等一会儿，然后再试一次。它应该在接下来的几天里出现。
