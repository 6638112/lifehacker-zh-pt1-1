# 你绝对应该做些圣诞树糖浆

> 原文：<https://lifehacker.com/you-should-definitely-make-some-christmas-tree-syrup-1848133356>

在疫情中部的某个地方，我掉进了一个兔子洞，学会了所有关于采树液制作 T2 糖浆的知识。是的——在我们这些普通人当中，在美国各地的社区里，都有这样的人，他们从自己的树上采集枫糖浆。

我知道。

我立即开始怀疑地打量我家附近的所有树木。去你的，PNW，你雄伟的天际线和没有枫树的森林。

我们*所做的*绝对是的松树和杉树，用它们*制作糖浆*不需要枫树所做的任何劳动。(钻研一下如何制作枫糖浆，你就再也不会抱怨价格了)。更好的是，取决于你住在哪里，现在可能是做这件事的最佳时机。

第一步是找到一棵树——一棵仍在地下茁壮成长的活树，所以你的圣诞树是不可行的，抱歉。你想要一棵冷杉或云杉，你想确定它没有被喷洒过。说真的，如果你不喜欢 [树、](https://www.growforagecookferment.com/conifer-tree-identification/) 树，打电话给朋友吧，因为不是所有的松树都可以吃。

一旦你找到这个宏伟的标本，你会想要采摘顶端较浅的绿色生长。这些是云杉(或冷杉)尖。有些树在春天形成这种生长，有些在春天形成，有时这取决于年份和天气。观察你的树的这种生长，记住，这是树的新生长，所以从一棵树上拿走*所有的*技巧来阻止那棵树。分散一点，你知道吗？你想装满半个杂货袋。

把小贴士扔到一个大碗里。用红糖盖住它们，搅拌使它们混合在一起。你希望糖的重量是云杉梢的两倍(对于 8 盎司的梢，使用 16 盎司的糖)。现在把它们都装进一个罐子里，我的意思是把它们装进去。在那里捣碎它。

把它放在阳光充足的窗户里，在室温下，接下来的几个星期里，每当你想到它的时候，就把罐子翻过来。随着时间的推移，糖会带走糖尖中的水分，产生糖浆。如果你幸运的话，它会开始发酵，如果是的话，你应该每隔几天就打一次嗝。(你；如果你看到 bbbbb，我就知道你需要这么做。)你可以用白糖，但是不要。你也可以加入一些东西，比如杜松子或者年轻的雄性松果。不要添加南瓜香料。不要侮辱云杉树梢。

两个月后，你会看到一种糖浆。我把它从罐子里拿出来，让它像活酵素一样挂在外面，但是如果你想让 [一直到](https://foragerchef.com/classic-spruce-tip-syrup/) ，把它倒入炖锅里，用针和所有的东西，让它慢炖，然后过滤。把它趁热倒进罐子里，放在冰箱里可以保存三个月。

现在:用它来做什么呢？配制鸡尾酒，*显然是*。考虑到云杉树梢中含有多少维生素 C，你可以用这种材料做一杯热棕榈酒。我个人在里面加入碳酸水，创造了一种汽水。你也可以把它扔进你一直在想的那个 [火苹果酒](https://www.storey.com/article/warm-up-this-winter-with-fire-cider/) 。(你一直在想吗？).

当有疑问时:就像亲爱的那样使用它。当我读到用云杉糖浆作为五花肉的釉料时，我摔倒了。(唉，我的心。)我怀疑它会做成一种 [超赞的复合黄油](https://lifehacker.com/impress-your-guests-with-a-butter-board-1848138546) ，非常适合做早餐饼干。想象一下你的假日蔓越莓酱里有一些！

所以，如果在这个季节，砍伐和杀死树木来安装在你的客厅里还不能让你从*捐赠树*中获得足够的音符，那就给自己找一棵幸存的云杉，并利用它的技巧。圣诞树糖浆不会有死树圣礼的视觉冲击，但味道会好很多。
