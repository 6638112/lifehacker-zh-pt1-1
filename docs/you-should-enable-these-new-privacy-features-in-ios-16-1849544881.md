# 你应该在 iOS 16 中启用这些新的隐私功能

> 原文：<https://lifehacker.com/you-should-enable-these-new-privacy-features-in-ios-16-1849544881>

苹果的 [iOS 16 为你的 iPhone 带来了一系列新功能和变化](https://lifehacker.com/the-best-new-features-in-ios-16-and-ipados-16-1849023814) ，但这并不都是为了展示。该公司使命的核心原则是隐私，因此，iOS 16 引入了四个新功能，应该可以扩展你的 iPhone 保护个人信息隐私的能力。



## 从共享联系人中移除某些字段

如果你对 iPhone 上的联系人数据很挑剔，你可能已经保存了人们的电话号码、电子邮件地址、生日、昵称和其他个人信息。当你与他人分享联系方式时，所有这些信息都会在不经意间传递到。

为了避免这种情况，打开联系人应用程序，点击你的任何一个联系人。向下滚动并选择**分享联系人**，然后选择分享表格顶部的 **过滤字段**。在这里，您会看到每个联系人详细信息字段旁边的复选标记。取消选中不想分享的内容，然后点击**完成**。当您共享此联系人时，它会自动删除您未选中的字段。遗憾的是，每次从 iPhone 上共享联系人时，你都必须记住这样做。

当你在看的时候，不要忘记在分享照片之前 [删除照片中的位置数据](https://lifehacker.com/how-to-remove-location-data-from-your-iphone-photos-bef-1847661277) 。

## 增强的剪贴板保护

应用程序将不再能够自动检查你的剪贴板上的内容。当您从一个应用程序拷贝一些内容并粘贴到另一个应用程序时，您总是会看到一个弹出窗口，询问您是否要允许该应用程序访问您的剪贴板。

如果您手动粘贴内容，您应该授予此权限。然而，在过去，邪恶的应用程序试图自动获取你剪贴板中的所有内容。你现在可以拒绝这些应用程序访问你的剪贴板— ，并且，为了更好的措施，卸载它们。

## 隐藏照片的更好方法

iOS 16 中的照片应用做得更好 [隐藏你的照片](https://lifehacker.com/how-to-find-and-hide-your-nsfw-photos-in-ios-1833889371) 。你现在必须输入密码或使用 Face ID 或触控 ID 来打开应用程序中隐藏的和最近删除的相册。这个特性是默认启用的，所以你不需要做任何事情就可以开始使用它。

你可以更进一步，完全隐藏你的隐藏相册。G o 到**设置>照片**，并取消勾选**显示隐藏相册**。您的隐藏照片仍然存在，但相册将不会在照片应用程序中显示。

## 完成定期安全检查

iOS 16 引入了一个名为安全检查的功能，可以让你快速重新访问授予各种应用的权限，撤销对共享数据的访问，以及更改你的 Apple ID 密码。你可以将安全检查视为一种快速保护 iPhone 上数据的方法。
