# 什么是“氟罗娜”，我们应该担心它吗？

> 原文：<https://lifehacker.com/what-is-flurona-and-should-we-be-worried-about-it-1848320894>

新冠肺炎和流感都是呼吸道病毒，可能从令人讨厌到危及生命，公共卫生专家一直担心如果两者同时传播会发生什么(可怕的")。除了他们个人在医疗保健系统上的费用，还有一个担心是人们可能会同时患上两种疾病。



这就是发生在头条新闻中的一些被称为“氟罗娜”的人身上的事情尽管潜在的恐慌语气，有一件事你需要知道:这是一种新的疾病。

## 氟罗娜是什么？

氟罗娜不是东西，好吗？尽管只有一个名字，但它是两种病毒的混合体*而不是*。简单来说就是一个人同时感染了这两种病毒。(也不是科学或医学术语，只是一个编造出来听起来吓人的名字，就像“ [【杀人黄蜂】](https://lifehacker.com/murder-hornets-are-unlikely-to-murder-you-personally-1843264533) ”)

这就像你踩到了狗屎，然后同一天一只鸟在你身上拉屎。这可能很糟糕，也可能是一个奇怪的巧合，但这并不是什么特殊的新变种便便悲剧。

流感和新型冠状病毒混合感染的案例并不是一个新现象；甚至在 2020 年初就有案例。我们还知道，最近在加利福尼亚州、得克萨斯州和佛罗里达州有病例，在以色列有两名孕妇最近合并感染，巴西至少有六例最近的病例，据华盛顿邮报报道[。](https://www.washingtonpost.com/health/2022/01/05/flurona-coronavirus-flu-symptoms/)

大多数地方没有同时检测 COVID 和 flu，也没有跟踪混合感染，所以我们还不知道它们是否变得更加普遍。要记住的一件事是，由于与前几年相比增加了掩蔽和隔离，去年的流感病例非常少。流感病例已经比去年 高很多，看起来更像一个典型的季节，所以预计今年比去年有更多的混合感染是合理的。

## 这对我意味着什么？

最重要的一点是接种疫苗预防流感是很重要的。一些混合感染发生在注射了 COVID 疫苗但最近没有注射流感疫苗的人群中。注射流感疫苗是一个简单的步骤，可以帮助你防止双重感染。如果你要注射疫苗或加强剂，你甚至可以在同一天注射流感疫苗。

没有一种疫苗是 100%有效的，但是每一种都提供了针对其病毒的有效保护。如果你真的被感染了，这两种方法都会降低你出现严重并发症的可能性。

洛杉矶县最近发生了一起上了头条的混合感染病例， [在一些检测点提供了 COVID 和流感联合检测](https://covid19.lacounty.gov/dual-covid-19-flue-testing-faq/) 。其他一些测试实验室也提供组合测试；如果你感觉特别不舒服，并且这些测试对你来说是可行的，考虑进行这些测试。(也有可能对两者分别进行测试；如果你找不到，问问你的医生。)这两种情况的症状都很相似，你不能只从症状上区分。
