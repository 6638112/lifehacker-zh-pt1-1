# 我想我学会了真正喜欢跑步，所以也许你也可以

> 原文:[https://life hacker . com/I-think-I-learned-to-actually-like-running-so-maybe-yo-1846935850](https://lifehacker.com/i-think-i-learned-to-actually-like-running-so-maybe-yo-1846935850)

在为期一个月的“ [Lifehacker健身挑战赛](https://lifehacker.com/c/lifehacker-fitness-challenge) ”中，当我结束第三周努力学习*喜欢*跑步——这是我一生中最讨厌的事情时，我不得不报告几个打嗝和一个难以置信的、几乎难以置信的成功。

Watch

首先，打嗝，有几个。尽管上周给自己定了三次跑的 [目标](https://vitals.lifehacker.com/how-zombies-made-me-hate-running-slightly-less-than-i-d-1846887981) ，但上周我还是准确地跑了一次*:*

> *我下周的目标包括跑三次，其中至少一次是在风景优美的小道上。我本周最长的跑步/散步是20分钟多一点，所以如果我能达到25分钟，那就感觉是进步了。我希望至少有一次跑步会在情绪表上达到或接近“2”。*

*这是有正当理由的，那就是我的父母和祖父上周末来镇上拜访。这是我一年来第一次亲眼见到他们，所以和他们一起出去玩就排在了“以一种我都不喜欢的方式锻炼”的前面。然后，因为我告诉自己，我也可以用我喜欢的方式锻炼，所以我选择了周一的椭圆机。*

*不过，在周二，Lifehacker的主编乔丹·卡尔霍恩和我为Lifehacker的播客采访了力量跑的杰森·菲茨杰拉德[【升级版](https://lifehacker.com/c/the-upgrade) 。(那一集定于周一上映，届时 [我会回来，这里掉链接](https://lifehacker.com/how-to-learn-to-like-running-with-strength-runnings-ja-1846965032) 。)我们正在谈论这个话题——一个成年人如何开始跑步？——杰森对我非常友好，鼓励我，甚至在乔丹告诉他我的脚踝如何不能正确弯曲之后。*

*从那次谈话中，我获得了一些重要的个人收获。首先，为了开始喜欢跑步，你必须给它30天的时间。(我偶然发现了准确的时间量！)第二，你可能会在头两个星期——也许三个星期——讨厌它。(我花了那么多时间，但我仍然不是一个粉丝！)和*那么*就会有一个转折点；我的身体会开始感觉不同。我的腿会感觉更强壮，整个过程会变得更自然，我会开始享受它。我可以说他真的相信这一点，我可以看到他希望我这样做，我允许自己认为这实际上是可能的。毕竟，他是专家。*

*有了新的动力，我离开了Zoom会议，系好运动鞋的鞋带，出去跑步。我需要换个环境，所以我去了我家隔壁的社区——那里有美丽的巨大梧桐树，有点多山，但不像我家那样多山。我走了几分钟，然后开始慢跑。我的脚踝又开始僵硬了，我想，“哦，该死的，【T1，】T2，我们又来了，【T3，”但是他们松开了，我走了。*

*几分钟后，我考虑是否应该放慢速度去散步，因为这是我的身体通常坚持几分钟后我要做的。然而，我感觉很好。我没有特别上气不接下气或不舒服，所以我只是继续跑。我没有自己计时，但我最好的估计是我连续跑了12-13分钟，没有停下来。如果当时我没有转向上坡，我想我可以跑得更久。开始时走了几分钟，结束时走了20分钟，我回到家，对自己感到相当满意。*

*杰森是对的。我的身体感觉不同了。我的腿感觉更强壮了。我觉得自己不像是在假装跑，而更像是T2在跑。我有点...喜欢...那种感觉。*

*在我的跑步情绪量表1-5分中，我给自己打了“2分”。(如果你不记得的话:第五个是，“他妈的直接下地狱”，第一个是，“耶！跑步是最好的！”)两分是我本周的另一个目标，也是我认为自己不会完成的目标。*

*现在，回到打嗝(比喻的那种)。正如我所说的，这是我本周唯一一次跑步——尽管公平地说，我写这篇文章的时候才周四。我昨天和今天没有跑步的主要原因是因为我昨天醒来的时候，我的右脚踝有点疼。不像完全受伤类型的疼痛，但足够紧，我很紧张运行它。是我自己的错；我没有做足够的热身，尽管我知道我的脚踝不灵活。杰森还强调，人们试图做得太多，太快，可能会加重受伤，所以我打算在尝试跑步前再等一两天。*

*不管是不是愚蠢的脚踝，我都认为这一周是一场胜利。*
