# 您可以在不受支持的硬件上安装 Windows 11，但您可能不想这样做

> 原文：<https://lifehacker.com/you-can-install-windows-11-on-unsupported-hardware-but-1847630897>

微软对即将到来的 Windows 11 的严格硬件要求让许多用户望而却步，但事实证明，即使你的个人电脑的处理器或安全功能不符合该公司的标准，也有办法安装新的操作系统——尽管它不能保证正常运行。

Watch

[据 The Verge](https://www.theverge.com/2021/8/28/22646035/microsoft-windows-11-iso-workaround-no-update-guarantee) 报道，微软确认用户可以用一个 ISO 手动安装 Windows 11。这绕过了正常的 Windows 更新程序，意味着你不必通过 Windows PC 健康检查来安装 Windows 11。但是，您的电脑仍必须满足最低硬件要求:

*   至少有两个内核的 64 位 1 GHz 处理器
*   4 GB 内存
*   至少 64 GB 的可用存储空间

一旦 Windows 11 在未来几周推出，我们将了解更多关于手动安装过程的信息，但这个过程可能类似于用 ISO 安装 [Windows 10，这意味着你还需要以下设置之一:](https://www.microsoft.com/en-us/software-download/windows10)

*   可以刻录光盘的 DVD 驱动器
*   或者安装在您电脑上的虚拟驱动器

只要满足以上所有硬件要求，即使不满足通过正常渠道升级所需的 TPM 或 SecureBoot 要求，也可以安装 Windows 11。

当然，这不是预期的升级路线。微软更希望用户购买新电脑或升级到推荐的硬件规格，但它提供手动 ISO 安装，作为希望预览 Windows 11 的企业以及不想升级硬件的顽固用户的后备措施。当然，在不受支持的硬件上手动安装 Windows 11 并非没有风险。

最大的缺点是，旧电脑可能没有资格在未来更新到 Windows 11。错过新功能和升级是一回事，但更大的担忧是安全性和驱动程序更新。这些较小的更新更频繁地推出，独立于大规模的版本升级。虽然微软还没有证实它将阻止旧硬件上的更新，但该公司肯定正在考虑这一点。如果发生这种情况，旧电脑可能会面临未打补丁的安全问题，例如最近的[print nightman 漏洞](https://lifehacker.com/how-to-avoid-windows-printnightmare-security-threat-1847221653) 。

然而，即使旧电脑不能自动安装更新，微软也有可能以 iso 的形式发布 Windows 11 的后续版本，你可以用它来手动 [更新到最新版本的](https://lifehacker.com/how-do-i-install-windows-10-without-all-those-extra-upd-1846790600) (只要它们实际可用)。

同样，旧电脑可能会因为过时的驱动程序而遇到兼容性问题。不匹配或过时的驱动程序会阻止您使用 USB 设备，破坏基本功能，甚至阻止应用程序运行。类似地，如果你的显示器驱动不是最新的，一些游戏将运行不佳。虽然您可以手动更新驱动程序，但这是一个繁琐的过程，并且很难找到并安装正确的驱动程序。

再说一次，还没有完全确定微软会阻止旧硬件的更新，但这随时都有可能发生。如果你不想处理这些问题，在旧电脑上坚持使用 Windows 10 可能会更好。除此之外，还有办法 [不用安装任何东西就可以试用 Windows 11。](https://lifehacker.com/how-to-test-drive-windows-11-without-installing-anythin-1847527997)T3】
