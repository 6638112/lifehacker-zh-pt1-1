# 使用“57 英寸法则”来决定在哪里挂墙壁装饰

> 原文：<https://lifehacker.com/use-the-57-rule-to-figure-out-where-to-hang-wall-decor-1847874720>

在你十几岁和 20 岁出头的时候，用胶带或大头钉把海报、照片或其他任何你想挂在墙上的东西贴在墙上是完全可以接受的，你可以称之为装饰。但是随着年龄的增长，你的品味可能会改变——包括艺术。

也许你想让你的客厅更上一层楼，在你的墙上挂些带框的东西。好主意，但是你到底应该把它挂在哪里？这就是“ 57 英寸规则”的由来。这里有一些关于这个设计技巧的知识，由云杉 的 [Ashley Chalmers 提供。](https://www.thespruce.com/57-inch-rule-for-hanging-artwork-5198388)

## 57 寸规则是什么？

事实证明，设计师和画廊老板经常使用 57 英寸作为眼睛水平的标准测量。曾经有人做过某种数学计算，但现在——不管它是否仍然是(或曾经是)人眼的平均水平——它只是一直存在的行业标准之一。

如果在这一点上还不清楚，57 英寸规则规定，墙壁艺术和装饰应该悬挂在离地面 57 英寸的地方。以下是艺术家、哈比神艺术 T3 的创始人克里斯蒂·科胡特对云杉的解释:

> “在悬挂艺术品时，57 英寸规则是一个很好的标准。这意味着艺术品的中心应该距离地面 57 英寸，有助于将艺术品与眼睛水平对齐。在家中各处使用时，它可以在艺术收藏品之间创造平衡与和谐。”

不仅室内设计师在装饰客户的家时使用它，美术馆和博物馆也经常使用 57 英寸的规则。

## 如何使用 57 英寸尺

虽然这种装修策略很简单，但根据 [查默斯](https://www.thespruce.com/57-inch-rule-for-hanging-artwork-5198388) 的说法，使用时还是有一些事情要知道的:

*   如果你是第一次在墙上挂东西，57 英寸的规则会给你一个开始的地方。
*   “规则”实际上非常灵活，也可以用作基线。此外，不可能在每种类型的墙上都挂壁画，所以有时你必须利用你现有的东西——即使这意味着打破 57 英寸的规则。
*   创建画廊墙？使用 57 英寸的规则来放置你的锚件，然后围绕它建立其余的系列。
*   根据房间的大小来考虑艺术品的大小。较小的零件通常可以挂在低于 57 英寸的地方，正好适合。
