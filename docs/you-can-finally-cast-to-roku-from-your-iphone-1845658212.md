# 你终于可以从你的 iPhone 上投射到 Roku 了

> 原文：<https://lifehacker.com/you-can-finally-cast-to-roku-from-your-iphone-1845658212>

不要跳过最新的 Roku OS 更新，尤其是如果你在同一间房子里混合搭配 Roku streamers 和苹果设备。Roku OS 9.4 将 HomeKit 和 Airplay 2 添加到了 [该公司几款最强大的流媒体设备](https://blog.roku.com/apple-airplay-homekit) ，包括:

*   一年超
*   Roku Streambar
*   Smart Soundbar 一年
*   Roku 流媒体棒+
*   首相一年
*   选择 4K Roku 电视

通过 HomeKit，你可以将 Roku streamer 或电视添加到苹果智能家居设置中，允许你通过 iPhone、iPad、Mac、HomePod 或 Apple Watch 使用 Siri 语音命令来控制你的 Roku 设备。



另一个新功能 Airplay 2 可以让你通过 Roku 设备将 iPhone、iPad 或 Mac 上的媒体播放到电视上——除了其他功能外， [最终将为 Roku 用户提供一种从他们的 iPhone 或 macbook](https://lifehacker.com/what-hbo-max-lacks-1843710742)上访问 HBO Max 的简单方法。Airplay 还可以让你控制 Roku 电视的音量，改变输入，从苹果设备上打开或关闭电视，而不是用遥控器。

### 如何为 HomeKit 和 AirPlay 2 更新 Roku 设备

专用的 Roku 流媒体盒和棒可以通过系统更新立即安装 Roku OS 版本 9.4，而受支持的 4K Roku 电视将在未来几周内收到补丁。操作系统补丁通常会自动安装，但如果您的设备上还没有 HomeKit 或 AirPlay 2，您可以手动检查并下载最新的更新。所有 Roku 设备的更新过程都是一样的:

1.  按下 Roku 遥控器的**“Home”**按钮。
2.  滚动并选择**“设置”**
3.  转到**系统>系统更新。**
4.  在系统更新页面上，选择**“立即检查”**
5.  您的设备将在线检查任何未决更新(包括任何“渠道”更新)，然后下载并安装它们。
6.  等待安装过程完成。Roku 设备将在更新完成后重启。
