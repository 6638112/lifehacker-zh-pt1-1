# 如何判断预订航班服务是否物有所值

> 原文:[https://life hacker . com/how-to-tell-if-a-subscription-flight-service-is-worth-y-1848577760](https://lifehacker.com/how-to-tell-if-a-subscription-flight-service-is-worth-y-1848577760)

订阅服务对企业和消费者来说都是双赢的。有时候，这些模式对消费者来说甚至比商家([RIP movie pass](https://lifehacker.com/rip-moviepass-1838106387))更好。现在，阿拉斯加航空公司正在通过Flight Pass进入订阅经济，这是一项年度计划，允许订户以每月固定费用的一部分预订参与航班，据称 [的费用低于标准机票价格](https://www.alaskaair.com/content/gifts-and-products/flight-pass) 。

阿拉斯加航空公司 [最初宣称](https://www.forbes.com/sites/michaelgoldstein/2022/02/17/will-alaska-airlines-new-flight-pass-subscription-service-take-off/?sh=6e8f6e338b13) 的Flight Pass是有史以来第一个主要的航班订阅服务，这并不完全准确。尽管如此，Flight Pass似乎比Jet Blue误导性的 [All You Can Jet](https://thepointsguy.com/news/jetblue-all-you-can-jet-2019/) 计划等前辈更有前途——只要你一直在参与目的地旅行。

我们已经做了计算，看看飞行通行证是否值得你花钱。但首先，这里有一个阿拉斯加航空公司的订阅模式提供什么概述。

## **飞行通行证概述**

阿拉斯加的航班通票订阅选项包括每年从加利福尼亚州、内华达州和亚利桑那州的15个目的地出发的6、12或24次往返航班。所以，如果你不是每两个月至少去一次西南部，那么飞行通行证对你来说就不是什么好东西了。如果你是喜欢去迪斯尼乐园、拉斯维加斯或棕榈泉的常旅客，请继续阅读。

[Flight Plass](http://www.flightpass.alaskaair.com/) 有两个版本:T he超值计划( Flight Pass) 每月49美元起，t he更灵活版本(Flight Pass Pro)199美元起。这两个计划的主要区别是预订的时间。标准航班套票要求至少提前14天预订，而航班套票Pro允许当天预订。这两项计划都允许用户提前90天预订。

[阿拉斯加声称](https://www.alaskaair.com/content/gifts-and-products/flight-pass)12个月旅行的锁定价格通常低于标准机票价格。在每月费用之后，航班本身(据称)只需0.01美元，外加14.60美元的税费。要预订您的航班，您需要累积并兑换积分(积分不可滚动；“缺点”一节中有更多相关内容)。

那么，阿拉斯加的固定价格有多划算呢？有一些警告和缺点，但首先，让我们运行的数字，看看它是否值得在飞行成本的基础上。

## **最佳价值:航班通行证**

最基本的飞行通行证计划是每月49美元(税前和费用)。这包括每两个月一次往返航班，或每年六次往返预订。从阿拉斯加的 [网站](https://www.alaskaair.com/content/gifts-and-products/flight-pass) 还不清楚单美分的航班加14.60美元的税是否适用于总预订，但 [根据积分盖伊](https://thepointsguy.com/news/alaska-airlines-flight-pass/) 的说法，该金额适用于每个单程航段。

每月49美元加起来就是每年588美元，但如果算上6次往返预订中每趟航班的14.60美元，每年的总额实际上是763.32美元。如果您每年兑换六次往返航班，那么763.32美元相当于每月63.61美元，或者每次预订127.22美元。

我花了几个小时预订了穿越加州、内华达州和亚利桑那州的虚拟航班，看看每次预订127.22美元是否比没有飞行通行证的机票更划算。这些假设的预订都在主舱(包括机票)，从今天起至少两周。我加入了工作日的航班，以避免周五和周日的价格飙升。

在很大程度上，机票价格胜出。我很难找到一个低于127.22美元的预订。如果我想要洛杉矶和内华达州里诺之间的往返航班，最实惠的选择加起来是207.20美元；我可以用飞行通行证省下79.98美元。我从凤凰城到圣地亚哥的旅行总共花费了435.18美元，比飞行通行证的统一价格多了307.96美元。我能想到的最便宜的预订之一，拉斯维加斯到洛杉矶，仍然比157.20美元的机票贵30美元。

然而，所有这些节省都是假设你更喜欢坐在主舱里。如果我愿意放弃舒适的主舱、有保证的头顶置物箱空间和任何改变我行程的机会？然后我可以以97.20美元的“优惠价”预订从拉斯维加斯到洛杉矶的旅程。这可以归结为个人偏好，即30美元的储蓄是否值得你在旅行中获得有限的灵活性和舒适性。

## **对于最后一分钟的乘客:飞行通行证Pro**

如果你是一个经常旅行的人，需要在最后一分钟预订和改变计划的能力，更贵的Flight Pass Pro可能会有回报。

使用与上述相同的数学方法，但每月6次预订的起价为199美元，Flight Pass Pro细分为每次往返行程的费用为398美元。一旦你看到该地区的当天航班通常要花费499美元以上，Flight Pass Pro的价值就变得显而易见了。然而，如果你知道你通常可以通过提前至少14天预订找到更实惠的航班，那么每月199美元(实际上是213.61美元加上税费)可能不值得。

## **小字提醒**

如果你要考虑两个飞行通行证的警告，那就这样吧:你被锁定一整年，你的积分不能展期。这意味着你需要每两个月至少去一次这个地区，这样订阅才值得。在您的12个月结束之前，您不能取消您的年度订阅。另外，你需要亲自打电话给代表取消它，否则它会自动续订。呀。

## **底线**

如果你忠诚于阿拉斯加航空公司，并且一直在西南地区飞行，那么飞行通行证可以带来一些合理的节省。看看你的日历，算算你是否每年至少乘坐六次往返航班。在此基础上，思考您可以在出发前至少14天预订航班的频率，以确定基本航班套票或更灵活的航班套票Pro对您来说是否值得。
