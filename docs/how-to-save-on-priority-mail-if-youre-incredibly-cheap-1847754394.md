# 如果你非常便宜，如何节省优先邮件

> 原文：<https://lifehacker.com/how-to-save-on-priority-mail-if-youre-incredibly-cheap-1847754394>

嘿，孩子。你想每次从邮局寄东西都节省五毛钱吗？真的吗？然后围过来，仔细听着。

Watch

您可能已经知道，您当地的美国邮局为客户提供不同尺寸的免费信封和盒子，用于通过优先邮件发送邮件。对于固定的费用，你可以邮寄任何你能塞进这些容器的东西，只要你能密封它。下图是一个 USPS 优先邮件 [小平邮箱](https://www.usps.com/ship/priority-mail.htm) 。你可以花 8.45 美元从当地邮局把它寄到美国的任何一个地址。

相比之下，7.95 美元(整整便宜了 50 美分)，美国邮政将把这个小的统一费率信封寄到美国的任何地址:

但是你可能没有意识到:盒子可以装进信封里，而且密封得很紧，成本保持在同样低的水平。

*因此，*如果你需要用一个结实的纸箱来保护你的优先邮件，你可以把你珍贵的货物封在箱子里，把箱子塞进信封，然后带着口袋里额外的 0.50 美元叮当声离开你当地的邮局。

虽然邮局没有宣传这个“功能”，但据我当地的邮件管理员说，这样做没有什么不合法或不道德的。但是，如果你想在伦理上吹毛求疵，它确实使用了比单独发送盒子更多的无用包装，这意味着更多的树木必须死亡。但是如果你同意，那就去做吧。

你根本不能邮寄安全气囊、弹药、爆炸物、汽油、大麻、香烟和其他形式的烟草，而且有数千种物品有运输限制，包括火化遗体、水蛭、鸸鹋和放射性物质。( [*完整列表此处*](https://pe.usps.com/text/pub52/index.htm?_gl=1*5k0vny*_ga*MTI5NDE5MzczNC4xNjMyNzUzNDkz*_ga_3NXP3C8S9V*MTYzMjc2NTk1OS4yLjEuMTYzMjc2NzE0My4w) *)。)*
