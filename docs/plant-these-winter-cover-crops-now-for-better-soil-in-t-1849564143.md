# 现在种植这些冬季覆盖作物，为春天提供更好的土壤

> 原文：<https://lifehacker.com/plant-these-winter-cover-crops-now-for-better-soil-in-t-1849564143>

很难夸大覆盖作物对你的花园的好处。除了帮助补充土壤中的氮，覆盖作物还可以作为健康的花园生态系统的栖息地，并帮助防止土壤在不种植蔬菜时受到侵蚀。使用覆盖作物，如配套种植，是一种侵入性较小的促进土壤健康的方法，同时还能让菜园茁壮成长。种植覆盖作物将使明年的蔬菜更加健康，抑制杂草将使春季种植变得轻而易举。

覆盖作物也很有用，被称为“绿肥”——本质上是一种就地堆肥系统，不费吹灰之力就能给土壤增加养分。一旦覆盖作物死亡，你可以让它在原地腐烂，或者在它们结籽前翻土来加速这个过程。

## 覆盖作物对菜园土壤有什么影响？

两种主要的覆盖作物是豆类和非豆类。除了我们熟悉的花园豆类、豌豆和蚕豆，还有大量的豆类品种，从三叶草到野豌豆，都可以用作覆盖作物，将氮返回到土壤中。非豆类品种包括黑麦草和羽衣甘蓝，与豆类相反，利用土壤中的氮。

如果你的菜园主要种植西红柿、辣椒或绿色蔬菜，豆类“固氮”作物将有助于恢复土壤。如果你主要种植豆类、豌豆或南瓜，一种非豆类的、吸收氮的覆盖作物将会起到平衡作用。根据你想种植什么和你的气候，这两种主要的覆盖作物将有助于补充土壤和防止侵蚀。

## 你应该种植哪些覆盖作物？

几乎可以在任何气候带种植的最通用的覆盖作物是豆科品种的草木樨、苜蓿和毛苕子，以及非豆科品种的黑麦、羽衣甘蓝和黑麦草。这些植物对土壤类型不挑剔，在大多数气候下都很耐寒。如果你正在寻找最简单的覆盖作物，这些是你最好的选择。但是，如果你想要一种作物在你的特定地区茁壮成长，你需要知道以下几点:

### 覆盖作物在南方种植

在南方，表现良好的豆类品种是白色羽扇豆、紫色野豌豆、普通野豌豆、黄色草木樨和豌豆，以及刺三叶草和深红色三叶草的非豆类品种。这些植物会在温暖干燥的环境中茁壮成长。

### 覆盖作物种植在北方

随着你向北走，你的选择变得更加有限，但有一种非豆类覆盖作物非常适合美国北部的气候。

### 区域覆盖作物

另一个影响覆盖作物表现的因素是它是否靠近海岸。在美国东北部地区，豆科品种红三叶、白三叶和两年生草木樨可能会熬过冬天。在东南部和墨西哥湾沿岸地区，蓝色羽扇豆和紫色野豌豆是不错的选择。在中西部，毛苕子、冬黑麦和大麦是很好的土壤稳定选择。在西南部，豇豆、固着三叶草和紫色草原三叶草都会长得很好。在太平洋西北部，蚕豆、各种各样的三叶草和荞麦都可以作为覆盖作物。

如果你不确定你想要的覆盖作物在你的特定地区是否耐寒，你可以根据美国农业部的耐寒性 [区域地图](https://www.loc.gov/item/2013585127/) 检查你的种子包。更多的信息和额外的资源通常可以通过咨询当地的大学分校找到。
