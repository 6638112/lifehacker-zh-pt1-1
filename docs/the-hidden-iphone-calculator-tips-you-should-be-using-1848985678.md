## 轻松地把它变成一个科学计算器

只要你禁用了你的iPhone旋转锁(你可以从控制中心做到这一点)，在计算器应用程序中侧向旋转手机就可以访问一个完全成熟的科学计算器应用程序。

## **刷卡删除号码**

好吧，好吧，你可能不经常需要科学计算器，但是你*需要*可能偶尔需要删除单个的数字:只需在屏幕顶部附近向左或向右滑动即可快速删除一个数字，而无需重新开始全部计算。
