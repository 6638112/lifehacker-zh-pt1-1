# 你需要马上更新你的苹果设备

> 原文：<https://lifehacker.com/you-need-to-update-your-apple-devices-right-now-1847371879>

确保您立即安装最新的 iOS、iPadOS 和 macOS Big Sur 系统更新，以修补黑客正在积极利用的危险安全漏洞。



iOS 和 iPad 的更新还修复了一个令人讨厌的错误，即阻止一些 iPhone 用户通过手机的触控 ID 解锁 Apple Watch。然而，关键的一点是，所有 iPhone、iPad 和 Mac 用户都需要立即安装更新，以保护他们的设备和数据免受最新安全漏洞的影响。

[iOS/iPadOS 14.7.1](https://support.apple.com/en-us/HT212623) 和 [macOS 11.5.1](https://support.apple.com/en-us/HT212622) 更新的补丁说明将该漏洞列为 CVE-2021-30807。这是在所有三个受影响的操作系统上发现的 IOMobileFrameBuffer 组件中的“内存损坏问题”，该问题允许安装在设备上的恶意应用程序获得提升的权限并执行任意代码攻击。

不幸的是，两份错误报告都证实苹果收到了一份匿名的“报告，称这个问题可能已被积极利用”，而 [ZDNet 的报告](https://www.zdnet.com/article/time-to-update-your-iphone-as-apple-fixes-actively-exploited-zero-day-flaw/) 表明一个概念验证攻击已在网上出现。换句话说:U 苹果的专利设备是公开的目标。

对于任何对这个问题的技术分析感兴趣的人，微软安全响应中心(MSRC)的萨尔·阿马尔就这个问题写了一篇详细的博文。Amar 实际上在近四个月前就发现了这个 bug，并且在苹果公司披露这个问题之前，他一直在做一份全面的 bug 报告，他的解释让我们对这个 bug 有了比我们从苹果公司得到的更深入的了解。

## 如何在 iPhone、iPad 和 Mac 上安装更新

这些更新适用于所有运行 iOS 14 的 iPhones、运行 iPadOS 14 的 iPads 以及运行 macOS Big Sur 的 Mac 台式机和 Macbooks。以下是检查设备操作系统版本的方法:

*   **iOS/iPadOS:** 进入**设置>通用>关于，**确保你的 iPhones 是 iOS 14.7.1 或更高版本，iPads 是 iPadOS 14.7.1 或更高版本。
*   **macOS:** 在苹果菜单中点击**“关于这台 Mac”**。确保它运行的是 macOS 11.5.1 或更高版本。

以下是如何更新您的设备:

*   **iOS/iPadOS:** 进入**设置>常规>软件更新>下载并安装**然后按照屏幕上的指示进行操作。
*   **macOS:** 从**苹果菜单**，进入**系统偏好>软件更新>立即更新**，按照指示操作。
