# 给自己烤些小土豆

> 原文：<https://lifehacker.com/waffle-yourself-some-tiny-potatoes-1846782265>

几天前的一个晚上，我看到了一段视频，视频中的 [一个女人对着一个土豆](https://twitter.com/cuntanadien/status/1386566668961255424) 大喊大叫。我仍然不知道那个女人是谁，但是她在视频中提出的观点是绝对正确的，最主要的一点是土豆永远是很好很正常的食物。从那以后，我一直很想吃土豆，昨晚我吃了一些。



他们的准备很简单:我买了一袋紫色的小土豆和切成两半，尽可能多的烹饪到我的空气油炸锅的篮子里。这给我留下了七个小土豆，我决定今天早上当早餐吃。

把它们做成炸薯条是个不错的选择，但是我不想切丁或者油炸。不。我感觉像是*核爆*和*粉碎。我用微波炉加热小土豆，直到它们变软，在它们上面涂上一层薄薄的培根油，然后把它们放在我的华夫饼干烙铁的热盘子里。就像任何好吃的华夫饼干食物一样，小华夫饼干土豆出来时又脆又脆。额外的纹理使它们变得又紧又大——非常适合蘸酸奶油(或鲜奶油)和抹酱(或蛋黄)。正如我前面提到的，这里有一个额外的预微波步骤，但这只是一点微波。

要做华夫饼干小土豆，你需要:*

*   一些小土豆
*   某种烹饪油脂(我用了熏肉油脂)

将土豆放入微波炉安全的碗或容器中，盖上盖子。你可以使用保鲜膜(并用削皮刀开一个小孔)，或者你可以把容器附带的任何盖子放在上面(但不要突然关上)。用微波炉加热直到土豆完全变软——通常需要 4-7 分钟，这取决于它们的大小。从微波炉中取出，揭开盖子，让多余的水分蒸发。如果你看到一些积水，简单地用纸巾擦干净。勺上少许培根油脂(或其他一些烹饪油脂)，每次 1/2 茶匙，直到土豆被涂上。

尽可能加热华夫饼干烙铁，加入土豆。用洗碗巾或锅架保护你的手不被热金属烫伤，砸在华夫饼机的顶部，把小家伙压扁。煮到外面松脆— 大约 2 分钟— 然后从华夫饼铁上取下，用盐调味(如果你喜欢，也可以用胡椒)。配上稀鸡蛋或某种蘸酱。
