# 谣言是真的，你真的可以在蛋黄酱中煎鸡蛋

> 原文：<https://lifehacker.com/the-rumors-are-true-you-really-can-fry-an-egg-in-mayon-1847949171>

蛋黄酱是由鸡蛋制成的，经常被用在以鸡蛋为特色的菜肴中，尤其是煮熟的鸡蛋。从 [到](https://www.youtube.com/watch?v=2oo6iEqkFnI) 的蛋黄酱，这种组合受到很多人的喜爱(和厌恶)。

Watch

这两种食物都倾向于产生大量的话语，这使它们成为名副其实的永无止境的内容来源(对我来说)。鸡蛋含量产生更多的鸡蛋含量。蛋黄酱内容产生更多蛋黄酱内容。有时，鸡蛋含量会产生蛋黄酱含量。这条推特就是如此，它是为了回应 [这篇博客](https://lifehacker.com/serve-your-fried-eggs-upside-down-1847939280) 而发出的:

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1453397766919503878&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1453397766919503878&autosize=1) 

这条推文写道:

> 我发誓这不是一个笑话，考虑到你的历史，但尝试用蛋黄酱而不是油或黄油来煎炸鸡蛋-这是我的最爱。

得到保证我不会被跟踪，我决定试试这个蛋黄酱煎蛋。我在不粘锅里融化了几茶匙蛋黄酱，同时喃喃自语:[，然而一丝真我存在于假我](https://knowyourmeme.com/memes/and-yet-a-trace-of-the-true-self-exists-in-the-false-self-circle-of-life) 【一遍又一遍】就像一句咒语。

一旦蛋黄酱完全融化并开始咝咝作响，我就在锅里打一个鸡蛋，然后像煮其他煎蛋一样煮它。正如你在上面的照片中看到的，蛋黄酱微妙地将鸡蛋的边缘和底部变成棕色，在蛋白周围形成一个薄而脆的边缘。

通过在油、醋、糖和(是的)更多鸡蛋的乳液中烹饪鸡蛋，你会得到一点点棕色，一点点甜味，和一点点酸。这是一个令人愉快的鸡蛋——精致、内敛、美味。它不油腻，也不黏腻，只是一个浅棕色、带花边的煎蛋。与用黄油煎鸡蛋不同，蛋黄酱煎鸡蛋后退一步，说“在这里，鸡蛋，你占据了中心位置。”

如果没有别的，很高兴知道你可以用蛋黄酱来做早餐。我可能会用完黄油，我可能会用完蛋黄酱，但我不会同时用完两者。
