# 你终于可以在 Windows 上访问你的 iPhone 照片了

> 原文：<https://lifehacker.com/you-can-finally-access-your-iphone-photos-on-windows-1849649496>

尽管一度不可思议，但科技公司开始友好相处。生态系统不再如此封闭，以至于不可能拥有一个 Android 和一个 Mac，或者一个 iPhone 和一台 PC。事实上，微软现在允许你将 iCloud 照片从 iPhone 同步到 Windows 11——你只需要跳到队伍的前面。



微软在周三的大型 Surface 发布会上宣布了令人兴奋的更新 。在 Surface 9 和下一代 Windows 11 的所有新功能中，该公司偷偷提到了与苹果的合作。通过 iCloud for Windows 应用程序，您现在可以在 Windows Photos 应用程序中看到 iCloud 照片；如果你想的话。iCloud 也是这里的关键词。为了在电脑上查看您的 iPhone 照片，这些照片需要同步到 iCloud。如果不使用 iCloud 照片图库，照片不会显示在 Windows 中。

这项功能还没有正式推出，所以你还不会看到这个选项出现在你的电脑上。然而， [据 Tech Crunch](https://techcrunch.com/2022/10/12/apple-brings-more-of-its-services-including-icloud-and-apple-music-to-microsoft-platforms/?tpcc=tcplustwitter) ，报道，如果你在 Windows Insider 计划中注册了你的电脑，你就可以提前查看。不过，在你注册之前，要知道你将会在你的电脑上安装尚未完成的软件。如果你宁愿在你的机器上保持稳定的版本，这个功能应该会在 Windows 11 的下一次大更新中出现，所以你可以等到那个时候。

不过，要想尽早试用，请点击 [进入官方 Windows Insider 计划网站，点击](https://insider.windows.com/en-us/about-windows-insider-program) ，然后点击“注册”使用与您的电脑关联的 Microsoft 帐户登录，接受 Microsoft 计划协议和隐私声明的条款，然后提交。

现在，进入**开始>设置> Windows 更新**。点按“Windows Insider 程序”，然后选取“开始”出现提示时，链接您在该计划中注册的同一个 Microsoft 帐户，然后从“开发频道”、“测试频道”或“发布预览”中选择 Windows 向大多数用户推荐测试版，我们也是，因为你会收到更稳定的软件，实际上是与正式版本捆绑在一起的。另一方面，开发频道在微软准备好测试软件的时候就提供软件测试，这有时会出问题。在你的电脑上看你的 iPhone 照片可能不值得。

无论您选择哪个选项，当提示安装最新的 Windows 版本时，请重新启动您的设备。接下来，确保你的电脑上安装了最新版本的 [iCloud for Windows](https://apps.microsoft.com/store/detail/icloud/9PKTQ5699M62?hl=en-us&gl=us) 。因为您已将您的设备注册到 Insider 计划中，所以您将能够获得比大多数设备更新的版本。在应用程序中登录您的 iCloud 帐户，然后确定照片已启用同步。

现在，当你在电脑上打开照片应用程序时，你应该会在窗口的一侧看到一个新的 iCloud 照片标签。点按它，您与 iCloud 照片图库同步的任何照片都会出现在您的电脑上，以满足您的查看和共享需求。请记住，虽然这个功能仍在测试中，所以你的结果可能会有所不同。
