# 这些机场现在提供现场 COVID 测试

> 原文：<https://lifehacker.com/these-airports-now-offer-on-site-covid-testing-1845602134>

假期期间的航空旅行可能会很紧张，所以如果你可以选择离开或经过几个不同的机场，你可能需要在预订航班前考虑一下它们的便利设施。在其他任何一年，这可能会涉及到调查休息室的可用性，或者检查你的航站楼里的餐厅。但现在是 2020 年，这不是舒适的问题，而是生存的问题。



当你计划假期旅行时，新冠肺炎疫情可能是你最想去的地方。虽然某些航空公司已经实施了测试政策，但这还不是行业标准。现在，美国和世界其他地方的一些机场正在提供现场新冠肺炎测试。以下是拥有这种非常 2020 舒适环境的机场，由《旅行+休闲 中的一篇 [文章提供。](https://www.travelandleisure.com/airlines-airports/airport-covid-testing-sites)
