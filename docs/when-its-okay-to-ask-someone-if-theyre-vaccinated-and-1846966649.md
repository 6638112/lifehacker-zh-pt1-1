# 何时可以询问某人是否接种了疫苗(以及如何接种)

> 原文：<https://lifehacker.com/when-its-okay-to-ask-someone-if-theyre-vaccinated-and-1846966649>

尽管新冠肺炎感染率和住院率在全国范围内都在下降，但一些灰色地带和潜在危险仍然存在: [百分之五十的美国成年人已经完全接种了](https://www.cnn.com/2021/05/24/health/us-coronavirus-monday/index.html) ，这是一个了不起的成就，也是常态即将到来的迹象，但很大一部分 [成年人仍然犹豫不决](https://www.newyorker.com/science/medical-dispatch/the-beginning-of-the-end-of-the-american-pandemic) 或 [完全反对接种疫苗](https://vitals.lifehacker.com/heres-your-covid-vaccine-rumor-roundup-1845935618) 。对于那些没有接受注射的人来说，感染的可能性仍然很大。

不管你怎么看，关于何时戴口罩都有相当多的灰色地带，无论是根据疾病预防控制中心的指导方针还是由独立企业制定的。例如，我继续在我进入的任何杂货店或加油站戴口罩，尽管根据疾病预防控制中心和我当地杂货店的规定，我在技术上不需要戴口罩。对我来说，这是为了让其他人感到安全，很大程度上是因为没有人知道我是否免疫(除了政府签发的一些假设的疫苗护照)。

随着疫情的持续减弱，你可能会发现自己和一个不戴面具的人在一起，好奇他们是否已经接种疫苗，或者他们是否比你更不关心安全问题。以下是你如何认为询问某人的疫苗接种情况是合适的，以及何时深入他人的个人领域肯定不是一个好主意。

## 如何询问某人是否接种过疫苗

### 如果这是你的家，那当然可以

如果有人在世界被 COVID 撞离轴心后第一次来到你的住处，那么你应该放心地询问他们的疫苗状况。毕竟，这是你的私人空间，如果你在室内做一些事情，不戴面具，你询问它才有意义。

### 如果你在照顾一个免疫系统受损的人

如果你在寻找一个尽管接种了疫苗，但在技术上仍有风险的人，你完全可以询问另一个人的接种情况。例如，如果您在保龄球馆招待客人，并且您的祖母将会在那里，您可能想要询问客人是否已经免疫。同样，如果你有未接种疫苗的孩子。

### **以您自己的疫苗接种情况开始**

人们是否已经免疫是许多人心中的疑问——不仅仅是你。从某人那里得到明确答案的一个好方法是首先指出你已经完全接种了疫苗，或者在两次接种之间。首先分享你自己的状态表明你明白保护自己是你义不容辞的责任，这也比脱口而出“你接种疫苗了吗？”对一个你并不十分亲近的人。

假设你在户外聚会，想了解人们对室内聚会的感受。如果你向一群人提到你已经完全接种了疫苗，很可能其他人也会跟着做。

## 绝对不能问为什么有人还戴着面具

尽管我们正慢慢走出疫情，对于完全接种疫苗的人来说，口罩基本上是可选的，但请试着理解为什么有些人继续佩戴口罩。首先，在可预见的未来，免疫功能低下的人可能会继续戴口罩，这主要是因为美国不太可能实现群体免疫，高传染性变异体继续驱动大多数感染。

在此基础上，有很多变量在起作用，如果不知道某人的情况，你就无法真正破译。有人可能仍然坚持掩蔽，因为他们家里有未接种疫苗的儿童。他们可能在心理上厌恶摘下面具；一些人享受着面部覆盖 赋予他们的 [隐姓埋名，其中许多人是喜欢不被告知要经常微笑的女性。](https://www.theguardian.com/us-news/2021/may/10/the-people-who-want-to-keep-masking-its-like-an-invisibility-cloak)

我们已经从伪装成保护他人的手段，发展成保护自己的手段，这并没有错。这仅仅表明，尽管接种了疫苗，但由于个人情况或偏好，一些人将继续戴口罩。
