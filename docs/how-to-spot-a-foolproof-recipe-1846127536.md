# 如何识别简单的食谱

从任何定义来看，隐谷农场蘸酱都是绝对安全的配方。如果你给一百万人一个调料包、一桶16盎司的酸奶油和一把勺子，他们会做出一百万份完美的蘸酱。现在*这就是*一致性。

可悲的是，大多数食谱并不像那样低投入高回报——但包装背后的可靠性并不像你想象的那样罕见。一旦你知道要寻找什么，更重要的是，要避免什么，在野外发现一件确定无疑的事情其实很简单。

## **像菜谱开发者一样思考**

当你看到一个万无一失的食谱时，你不必成为一个专业的食谱专家，这是一个可以学习的技能。秘密在于知道食谱通常会走向何方，这样你就可以不惜一切代价避免它们。

所有万无一失的食谱都有这些特点:

*   **无烘焙**
*   **清楚、明显的指示**

*   **至少一种批量生产的配料**

如果这不是你所期望的清单，相信我，当我说每一点都是有原因的。当我需要一个配方来为每个人工作时，这些是我用来开发它的核心指南。我很乐意告诉你原因。

### 烤箱是混蛋

食谱开发商知道，烤箱比所有其他普通厨房设备加起来造成的麻烦还多。几乎每个人都有一个， [没有两个工作完全一样的](https://lifehacker.com/calibrate-your-oven-with-sugar-keep-its-temperature-st-30810543) 。热点、不可靠的温度读数和不一致的热循环以独特的、不可预测的组合折磨着各种烤箱。加上烤盘的复杂性，情况只会变得更糟:玻璃、陶瓷和金属保持和传导热量的方式不同，这影响了食谱的结果——这些至关重要的保温和传导特性也因品牌而异。好玩！

将所有这些个人的怪癖结合在一起，给一个给定的食谱带来了大量令人难以置信的变化。这并不一定意味着灾难——砂锅菜 和 [烤蘸酱](https://skillet.lifehacker.com/3-cheesy-dips-that-are-way-better-than-whatever-chipotl-1818528845) 是出了名的好吃——但通常来说，新的烤箱食谱与万无一失相反。如果你需要一个有保证的灌篮，寻找炉灶、微波炉、 [速溶锅](https://skillet.lifehacker.com/7-unexpected-things-to-make-in-your-new-instant-pot-1845958274) 或 [空气油炸锅食谱](https://skillet.lifehacker.com/what-to-make-with-your-new-air-fryer-1845937133) 来代替。

### 不给误解留下余地

墨菲的配方发展定律表明，任何指令，如果*被*误解为*，就会被*。即使我没有制定一个万无一失的10分钟食谱，我也会试着写下说明，就好像我是一样。

这并不意味着万无一失的食谱都是三行或更短。它们可以复杂到你能接受的程度。更重要的是，从第一眼看上去，这些指令就能产生完美、明确的意义*到* *和*。如果你已经通读了三遍，仍然不确定作者希望你在第四步做什么，那就继续吧——有更好的选择。

### 转移你的配料重点

食谱消费者有一种痴迷于成分表的倾向，除此之外别无他物。这通常采取替代问题的形式，既有效，又——请不要对我大喊大叫——被误导了。有些替代品比其他替代品效果更好，但严重的并发症来自设备和技术，而不是成分。每个人的厨房都摆放着不同的电器，每个人的大脑工作方式也略有不同；如果有的话，成分是唯一的共同点。为了证明这一点，看看任何一个食谱开发者在Instagram上的故事:每一道转发的菜都是用同样的原料做的，然而没有两个版本是完全一样的。

加工成分是最终的共同点，因为它们的数量是已知的。你确切地知道每次你会得到什么，这就是万无一失的食谱。

这并不是说把牧场或者 [洋葱汤混合](https://lifehacker.com/just-add-store-bought-french-onion-dip-to-mashed-potato-1839621708) 放在所有东西里。(但是如果你做了我会支持你的。相反，找到你自己版本的牧场混合物:一种美味的、大规模生产的配料，让一切对你来说都更好吃*。橄榄、凤尾鱼、 [比肉汤](https://skillet.lifehacker.com/make-better-pantry-pasta-with-bouillon-1842615372) 、调味酱、Sriracha、速食布丁、甚至凉奶油都符合这种描述，还有无数其他的东西。如果你寻找包含这些成分的简单食谱，你就在通往美味、万无一失的成功路上。*
