# 尝试解决Android 11的“最近应用”错误的方法

> 原文:[https://life hacker . com/try-these-变通办法-for-Android-11s-recent-apps-bug-1845179726](https://lifehacker.com/try-these-workarounds-for-android-11s-recent-apps-bug-1845179726)

一个古怪的Android 11漏洞是影响未知数量的用户和设备的和使多任务几乎不可能的。如果 s 在你的Pixel设备 上擦除数据以召唤其“最近”屏幕，或者你打开的应用程序列表，却什么也没有完成，你就会知道你受到了折磨。

Watch

好消息是什么？谷歌 [知道了](https://issuetracker.google.com/issues/163731217) 的问题，并正在解决。如果你在一个安装了窃听器的安卓系统上切换应用程序时遇到困难，这些信息现在对你没什么帮助。幸运的是，你可以尝试一些变通方法来缓解这个问题，如果T8不能消除这个问题的话。

首先，尝试锁定和解锁您的设备。这可能就是你要让你的最近应用列表再次工作所要做的一切。每次你想切换应用程序时，你都必须让*做*，，但是这是你工具箱里最简单的修复。如果这不起作用，尝试重启你的Android，看看是否能暂时缓解这种烦躁，或者甚至可能在 [安全模式](https://support.google.com/android/answer/7665064?hl=en) 和下重启，然后恢复正常。

否则，尝试切换到一个不同的发射器，看看这是否会有助于缓解问题。 [**智能发射器5**](https://play.google.com/store/apps/details?id=ginlemon.flowerfree) 是实实在在的选择，同样是 [**新星发射器**](https://play.google.com/store/apps/details?id=com.teslacoilsw.launcher)；我很确定我在我的第一台Android设备上使用了后者。(我们已经走了多远。)

不能保证这些修复程序会解决你最近的应用程序问题，但是我不会建议做出更激烈的故障排除努力。你将不得不从安卓的应用抽屉中启动你想要使用的应用——你可能想切换到 [三按钮导航](https://support.google.com/android/answer/9079644?hl=en) 而不是基于手势的导航，这样做更容易。但是这不是一个“工厂重置你的手机，看看是否有帮助”级别的问题。谷歌应该很快就会部署其修复，尽管令人惊讶的是错误竟然出现在最终的Android 11版本中，鉴于用户已经在各种预发布测试版中报告了它。
