# 教孩子“公平”和“平等”的区别

> 原文:[https://life hacker . com/teach-kids-the-difference-better-equal-1846458537](https://lifehacker.com/teach-kids-the-difference-between-fair-and-equal-1846458537)

这是世界各地都能听到的童年抱怨:“这不*公平。*“这是用来抗议兄弟姐妹被允许和朋友呆在一起或者有额外的屏幕时间，或者回应同学被允许有额外的时间完成任务。你可能会忍不住反驳说“*生活是不公平的，*”，虽然这是真的，但从长远来看，教会他们公平和公正的区别会更有帮助。因为平等的不总是公平的，公平的也不总是平等的。

Watch

成年人可以理解为什么有时一个人的需求或环境会比确保每个人得到完全相同的帮助或遵守完全相同的规则更重要。青少年应该被允许比幼儿园小朋友晚睡；这可能不公平，，但它是公平的。然而，公平和公正对孩子来说是抽象的概念，所以在他们完全理解之前，需要父母的一些指导(甚至可能是很多指导)。

这里有一些练习，你可以在家尝试，帮助他们更好地理解这种差异。

### 任何人都不能戴眼镜

有些人接受更多的帮助是因为他们需要——仅仅因为他们需要，并不意味着每个人都应该接受。芭芭拉·格鲁纳在 [自由精神出版社](https://freespiritpublishingblog.com/2016/02/01/thats-not-fair-teaching-kids-the-difference-between-fair-and-equal/) 的博客上为教师提供了这个练习，父母可以在家里进行类似的对话来说明这一点:

> 从佩戴处方眼镜这个令人大开眼界的例子开始。在问学生公平是否意味着平等(他们的典型反应是响亮的，“是的！”)，恭敬地要求每个戴眼镜的人摘下眼镜，因为如果班上有些人戴眼镜而其他人没有，这是不“公平”的。这将挑战他们关于公平的想法，意思是我们都是一样的。

### 每人一个创可贴

另一个经典策略是 [创可贴课程](https://lifehacker.com/teach-kids-about-fairness-vs-equity-with-the-band-aid-l-1844954001) ，挑战家里的每个人想出一个假装生病或受伤的故事。一旦每个人都选择了一个，给每个人发一个创可贴来治疗受伤——不管是被纸割伤，腿断了，还是重感冒。

哼，*那是*不公平。

### 谁能达到？

萨曼莎·宋(Samantha Song)老师为 [Better Kids](https://betterkids.education/blog/thats-not-fair-teaching-the-meaning-of-fairness) 写道，她有自己最喜欢的练习，每年都在课堂上使用，以充分说明公平和公正之间的区别。她把几样东西放在一个高架子上，然后叫一个高一点的学生和一个矮一点的学生去拿这些东西。当个子较矮的学生够不着的时候，她让全班集体讨论帮助他们的方法(拉一把椅子，从一个个子较高的同学那里寻求帮助，等等)。).宋写道:

> 这很好地引出了公平与平等的讨论。这两个志愿者不能得到平等的待遇，因为这真的不公平，甚至没有必要。那个高个子孩子已经占了上风。*Ba dum pum*很明显，学生们不需要凳子或其他人来帮他们拿东西。然而，另一个孩子需要一些适应，以便在成功时有一个*等于* *的机会*。
> 
> 平等与相同有关，就像数学一样。等号告诉我们等式两边完全一样(3 + 2 = 5)。公平是不同的。公平意味着基于可见的机会差距，每个人都得到他们需要的东西。有时人们会有不同的需求，因为我们是独特的个体。

你可以在家里这样做，指导一个青少年或成年家庭成员，以及一个更小、更矮的孩子去够厨房橱柜顶层的碗。你给了他们同样的任务，但你不公平地期望孩子能够达到和成年人一样的高度。

### 谈论公平与平等

现在你已经完成了一些基本练习，你可以开始谈论公平和平等的区别了。尽管表面上看起来它们似乎是相同的基本概念，但理解它们的区别是关键。 [社会变革英国解释得很好](https://social-change.co.uk/blog/2019-03-29-equality-and-equity) :

> 虽然两者都促进公平，但平等是通过平等对待每个人来实现的，而公平是通过根据需要区别对待不同的人来实现的。然而，这种不同的待遇可能是达到平等的关键。
> 
> 回到学生的例子，通过平等实现公平意味着给予所有学生同等水平的支持。然而，那些需要超过这一初始水平的更多支持才能成功的人将因此与那些不需要的人没有平等的机会。

平等就是没有人戴眼镜；衡平法给那些需要眼镜的人提供眼镜，这样他们就能和其他人一样看得清楚。平等是每个人的创可贴，是高高货架上的物品；公平是治疗个人的具体伤害和按需使用的阶梯凳。对小孩子来说，更早的就寝时间也是公平的——他们的身体需要更多的休息。一旦孩子们开始理解公平和公正之间的区别，他们就能够 [进一步发展他们对他人的移情](https://lifehacker.com/want-to-raise-empathetic-kids-show-em-how-its-done-1831050481) 。
