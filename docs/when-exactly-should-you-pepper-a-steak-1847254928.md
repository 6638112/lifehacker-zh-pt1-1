# 你应该什么时候在牛排上撒胡椒粉？

> 原文:[https://life hacker . com/when-just-should-you pepper-a-steak-1847254928](https://lifehacker.com/when-exactly-should-you-pepper-a-steak-1847254928)

除了盐，胡椒是美国烹饪中最常见、最普遍的调味品。浆果本身的颜色和味道千差万别，但是你何时以及如何研磨和烹饪它们也对它们的味道有很大影响。(在我的成长过程中，我姐姐称预先磨碎的黑胡椒为“热盐”，尽管它足够温和，可以大量撒在哈密瓜上。)

预包装午餐和外卖中的小包黑“胡椒”与现磨的东西相去甚远，很难相信它们来自同一种浆果。研磨胡椒会释放其芳香化合物，这些化合物会很快蒸发，这就是为什么人们在白色餐桌布餐馆对“新鲜研磨的胡椒”如此重视。它越新鲜，你能品尝到的化合物就越多。

热量对胡椒的味道也有很大的影响。温和的烘烤可能会赶走一些更不稳定的味道，但它也可以增强和丰富香气；烧焦的胡椒尝起来很苦，甚至是辛辣的，这就是为什么我很少在烹饪牛排之前用胡椒调味。

这个决定最终归结为偏好——我不想让任何苦味与牛排的纯肉味相竞争——但这也与方法有关。我喜欢在灼热的锅里烤焦我的牛排，以最大限度地增加美拉德褐变和结壳的形成，而“灼热”不是一个对胡椒及其化合物友好的温度。胡椒粒的粗屑还会阻止肉与锅接触，导致不均匀的烤焦。

但是只要你考虑胡椒和你想从中获得的味道，在烹饪牛排之前给牛排加点胡椒是没有问题的。有时候一点苦味是好事，别忘了还有一道经典的T4法式菜肴，庆祝牛肉和烤胡椒的完美结合——波夫牛排。

不同之处在于，几乎所有的牛排烹饪法都要求较低、较温和的温度——通常是中火或中火——而不是许多牛排食客(包括我自己)似乎痴迷的那种撕开、打开所有窗户并打开风扇的锅温度。胡椒脆皮牛排是美味、浓郁、辛辣和——是的——一点苦味的令人愉快的混合物。

那么，你应该什么时候给牛排加胡椒呢？这取决于你和你的口味。如果你想要稍微苦一点、辛辣一点、对比强烈一点的面包皮，那就提前调味，也许还可以把温度调低一点。如果你想品尝胡椒更浓郁的花香、柑橘味和木质味，可以在牛排烹饪完毕后磨碎。你也可以两样都做，看你有多喜欢辣椒。
