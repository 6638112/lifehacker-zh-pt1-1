# 美国农业部召回65吨冷冻鸡肉，所以检查一下你的鸡肉是否有问题

> 原文:[https://life hacker . com/USDA-revokes-65吨冷冻鸡肉-so-check-if-you-1846978641](https://lifehacker.com/usda-recalls-65-tons-of-frozen-chicken-so-check-if-you-1846978641)

对于家禽来说，这是艰难的一周。首先，疾病控制和预防中心(CDC)不得不提醒我们不要亲吻或拥抱活的鸡、鸭和其他家禽，因为人们这样做与43个州报道的T4沙门氏菌爆发有关。

现在，美国农业部(USDA)食品安全和检验局(FSIS)已经 [发布了公共卫生警报](http:// https://www.fsis.usda.gov/recalls-alerts/fsis-issues-public-health-alert-frozen-cooked-diced-poultry-products-due-possible) ，针对大约65吨(130，860磅)冷冻全熟鸡肉丁产品，因为它们可能被*单核细胞增生李斯特菌*污染。以下是受影响产品的相关信息。

## 为什么会有冻鸡召回？

一名FSIS检查员在一次例行检查中发现了可能的李斯特菌污染——特别是，需要重新烹饪的产品在没有重新烹饪的情况下被重新包装。不幸的是，同样受到影响的家禽产品已经进入世界，并可能进入我们的冰柜。好的一面是，此时 [美国农业部说](https://www.fsis.usda.gov/recalls-alerts/fsis-issues-public-health-alert-frozen-cooked-diced-poultry-products-due-possible) 还没有因为食用召回的冷冻鸡肉而出现不良反应的确认报告。

## 美国农业部的冷冻鸡肉召回包括哪些内容？

考虑到召回的家禽数量，美国农业部建议*所有*消费者将即食鸡肉加热至热腾腾。召回的冷冻、完全煮熟的鸡丁产品包装于2021年1月25日、2021年1月26日、2021年3月23日和2021年3月24日，包括:

*   4磅。塑料袋装“全熟鸡肉丁”，代码13530，Est。编号P-18237，包装日期为“2021年1月25日”和“2021年1月26日”
*   4磅。塑料袋装着代码为16598的“全熟黑/白鸡丁”。编号为P-45638，包装日期为“2021年3月24日”和“2021年3月23日”

你可以在美国农业部的检验标记中找到鸡肉的生产编号——“P-18237”或“P-45638”。

受影响的家禽由德克萨斯州休斯顿的Big Daddy Foods，Inc .分发，并于2021年2月24日至2021年3月1日和2021年3月29日至2021年4月8日分发到佛罗里达州各地的食品银行，包装在单独的食品箱中。美国农业部公共卫生警报 [有更多关于召回鸡肉具体分布地点的信息](https://www.fsis.usda.gov/recalls-alerts/fsis-issues-public-health-alert-frozen-cooked-diced-poultry-products-due-possible) 。

## 如何处理召回的冷冻鸡肉

首先，如果你家里有任何召回的鸡肉，不要吃它——美国农业部说你应该扔掉它。任何经历李斯特菌病症状的人——包括发烧、肌肉疼痛、头痛、颈部僵硬、意识模糊、失去平衡和抽搐(有时之前会有腹泻或其他胃肠道症状)——都应该立即就医。

如果您有任何食品安全问题或担忧，您可以拨打美国农业部肉类和家禽热线1-888-MPHotline (1-888-674-6854)，通过 [实时聊天询问美国农业部](https://ask.usda.gov/) (美国东部时间)周一至周五上午10点至下午6点，或通过电子邮件发送问题至MPHotline@usda.gov。

您可以通过 [在线电子消费者投诉监控系统](https://foodcomplaint.fsis.usda.gov/eCCF/) 举报任何有关肉类、家禽或蛋制品的问题，该系统可24小时访问。
