# 这款辛纳酸葡萄酒是完美的餐前酒

> 原文:[https://life hacker . com/this-cynar-sour-is-the-perfect-pre-dinner-libation-1846069403](https://lifehacker.com/this-cynar-sour-is-the-perfect-pre-dinner-libation-1846069403)

尽管如此，我还是喜欢啜饮由乙醇、糖和(有时)果汁混合而成的冰冷饮料的体验，尤其是在我做饭的时候。这就是低证明阿马里-如西纳尔-派上用场。像几乎所有其他类型的酒一样，这种木质朝鲜蓟基酒在我们的基本酸味配方 中表现出色，尽管我会建议你将普通糖浆从通常的3/4盎司减少到一半，因为朝鲜蓟本身带来了一点甜味。

对我来说，这是一种完美的开胃酒。这是一种令人愉快的味觉清洁剂——明亮、香甜、苦涩——酒味足以让你保持好心情，而不会喝醉(即使你的耐受力很低，并且你的胃是空的)。我喜欢它装在别致的小玻璃杯里，但只有在用力摇晃之后。稀释在这里真的很重要——没有它，我们会觉得饮料太粘稠，像糖浆一样。要做到这一点，你需要:

*   2盎司辛纳
*   3/4盎司柠檬汁
*   1/2盎司简单糖浆

将所有东西加入到装满碎冰(或一块大石头)的鸡尾酒摇壶中，用力摇晃，直到摇壶变得冷得无法忍受。过滤成小轿车或类似的东西，在做晚餐(或午餐，甚至早餐，取决于当天)时啜饮。
