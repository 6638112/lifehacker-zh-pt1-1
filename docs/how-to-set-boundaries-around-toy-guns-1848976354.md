# 如何设定玩具枪的界限

> 原文:[https://life hacker . com/how-to-set-boundaries-around-toy-guns-1848976354](https://lifehacker.com/how-to-set-boundaries-around-toy-guns-1848976354)

许多父母不喜欢枪——真枪*或玩具*T2。因此，即使我们可能知道 [玩假装的枪在发育上是正常的](https://lifehacker.com/what-to-know-about-kids-and-pretend-guns-1823249214) ，但每当我们的孩子开始用乐高积木造枪或用棍子互相假装射击时，我们的本能是关闭它。但是枪是我们现实的一部分也是孩子们玩耍的正常部分，这意味着与他们定期谈论真枪、玩具枪以及如何在玩耍中设定界限是很重要的。

## 与孩子和其他家长谈论真枪

你可能认为你可以猜到你的哪些朋友和邻居家里有枪，但是除非我们去问，否则我们永远无法确定。诚然，与另一位家长进行这样的对话有些尴尬——但是 [我们有完整的指南来帮助你开始](https://lifehacker.com/how-to-have-the-gun-talk-with-other-parents-1846874799) 。这一点尤其重要，因为正如艾米·莫兰为VeryWell Family 所写的那样，“多项研究发现，即使孩子们被告知永远不要碰枪，当机会出现时，他们也可能会去碰枪支。而后果可能是致命的。”

当然，这并不意味着你不应该和你的孩子谈论枪支。这只是意味着随着年龄的增长，你需要继续进行这些对话。对于小学生来说，你可以从给他们看真枪和玩具人的照片开始，这样他们就可以开始想象其中的区别。强调仍然很难区分，所以如果他们遇到任何类型的枪，他们应该1) 不要碰它，2) 立即找成年人帮忙。

随着孩子年龄的增长，他们可能会更加熟悉枪支暴力的现实，他们甚至可能有一个朋友认为他们可以“安全地”使用家庭枪支，并想炫耀一下。在这种情况下，莫林建议:

> 勇敢地面对朋友是一个微妙的话题，所以集思广益，想办法让你的孩子在不引起任何骚动的情况下摆脱困境。

> 建议他们说“这很无聊。我们去做点别的吧。”如果你的孩子有一个朋友主动向他们展示一把枪，训练他们说这样的话，“也许以后。我们去外面做点什么吧。”
> 
> 你的孩子不需要向朋友们宣讲枪支安全。他们只需要把自己从这种情况中解脱出来。

对美国父母来说，不幸的现实是，我们也必须和孩子谈论校园枪击事件——[这是我们对](https://lifehacker.com/how-to-talk-to-your-kids-about-school-shootings-1848975621) 的指导。当然，如果你有枪，你应该总是把它们安全地锁起来，即使你总是向你自己的孩子强调不要碰它们。

## 如何围绕假枪划定界限

一些父母通过选择大型霓虹色或泡沫种类来确保他们允许在家里使用的玩具枪和水枪看起来明显是假的。但是一些父母甚至不希望家里有最显眼的玩具枪，所以在给别人的孩子买礼物时记住这一点很重要——并且总是先征得同意。

儿童早期教育专家克里斯·德鲁 说:“在夏天，孩子们被赠予玩具水枪是很常见的。”。“我一般不建议给孩子送玩具枪(或在玩耍时拿出来)，除非所有孩子的父母都同意。”

但是，如果所有的父母和孩子都同意他们，那么在涉及任何形式接触的游戏中，你可以把他们的出现作为一个关于同意的教育时刻。当孩子们在玩耍时，他们会发射Nerf飞镖或水枪，确保每个玩耍的人都同意被“射杀”你也可以同意一些基本规则，比如“不准打脸”和“如果有人说他们不玩了，他们就不能*被枪击”*

另一个好的基本原则是:如果你的孩子在公共场所玩耍，比如在公园，一个旁观者经过，不要向他们“开枪”，即使只是用棍子指着他们说“砰！”他们也不玩了。你也可以对语言进行一些限制——而不是“我要杀了你！”你可以鼓励创造性的解决方法，比如，“我会打败你！”或者更温和的，“明白了！”德鲁还建议重新构思剧本。也许你可以惊叹重力作用于水面的方式和水在皮肤上的感觉，而不是用玩具枪来正常化对抗和战斗。”

如果你担心你的小孩过于沉迷于枪支或暴力，可以和你的儿科医生谈谈。他们可能会推荐一位心理健康专家进行评估，或会引导你寻找其他资源。另外一个很好的资源就是那些活着的和[男生！](https://www.boysalive.com/) 组织，该组织帮助父母应对棘手的育儿情况——它有关于孩子和枪支的一个 [很棒的博客系列](https://www.boysalive.com/gun-play) 。
