# 这是切洋葱的最好方法吗？

> 原文:[https://life hacker . com/is-this-the-best-way-to-dice-an-onion-1846607647](https://lifehacker.com/is-this-the-best-way-to-dice-an-onion-1846607647)

在Lifehacker工作意味着质疑一切，并接受这样一个事实，即“我们一直采用的方式”并不一定是继续做任何事情的最好方式。但是“最好”是一个相对的术语，特别是当我们谈论食物的时候，因为通常有多种方法可以达到相同的终点；更不用说文化、历史背景和个人偏好都影响着我们所认为的“最好”

然而，对于像洋葱丁这样的东西，标准非常简单:方法应该尽可能安全，并且产生均匀一致的碎片，同时最大限度地减少浪费。

虽然我很喜欢来自英国的林恩德西T4【发现了一种适合她需要的切洋葱的方法，但我不确定我会把它推荐给每个家庭厨师。

 [https://lifehacker.com/embed/inset/iframe?id=tiktok-6935404464434154757&autosize=1](https://lifehacker.com/embed/inset/iframe?id=tiktok-6935404464434154757&autosize=1) 

林赛的技术并不像T2表现出来的那样与传统的技术有所不同。然而，她确实跳过了几个步骤，基本上是先做一个开花的洋葱，然后再把它切成小块。如果你曾经吃过开花的洋葱，你会知道“花瓣”的两端是细长的，中间是较厚的，所以最终的洋葱片非常不均匀。

但并不是每一道菜都需要完全统一的骰子，所以我对这种方法的更大问题是安全问题:洋葱花不稳定，不灵活，并且不断试图开花。没有一个扁平的部分放在砧板上，你不得不在切片和切块时努力将洋葱握在一起，这是手指撕裂的一个原因。

但是林赛的方法确实有它的优点。我喜欢她原封不动地留下根——或者说“有毛的部分”——我也喜欢她在“像切橘子一样切洋葱”时顺着洋葱的脊这两种方法都是切洋葱的好方法，但尝试切一个摇摇晃晃开花的洋葱却不是。看看这段视频，看看我们对她切洋葱技术的测试是如何进行的。
