# 如何在微软关闭之前解锁Halo Infinite隐藏的多人游戏模式

> 原文:[https://life hacker . com/how-to-unlock-halo-infinites-hidden-multiplayer-modes-b-1848165514](https://lifehacker.com/how-to-unlock-halo-infinites-hidden-multiplayer-modes-b-1848165514)

在*光环无限*的故事模式公开之前，微软邀请用户在公开测试版中进入多人模式，该模式自11月15日以来一直在Steam上运行。默认情况下，Halo Infinite开始时有17种多人模式，但如果你在离线模式下启动游戏，你会有14种隐藏模式。

Watch

这些模式是由[Redditor wicked soldier 991](https://www.reddit.com/r/halo/comments/r7q6pz/fun_fact_if_you_launch_infinite_offline_you_can/?context=3)发现的。虽然它们在离线模式下可用，但你可以将它们保存到你的游戏文件中，并在在线模式下启动*光环无限*时访问它们。

更有意思的是:这些不仅仅是当前地图的变体。有一个全新的游戏模式叫做消耗，每个团队都有一个有限的回复池，你可以通过消耗他们来赢得比赛。淘汰是损耗模式的一个变种，有CS风格的多回合无重生。有战术杀戮者，这是光环无限版的特警，没有盾牌或雷达。

该公司还没有正式宣布这些模式，所以我们不知道他们将使用这个隐藏的技巧多久。我们建议您在Microsoft决定删除它们之前，尽快将它们保存到本地存储。

这是所有隐藏模式的列表。

*   竞技场:消耗
*   竞技场:摩擦躲避球
*   竞技场:淘汰
*   嘉年华:损耗
*   嘉年华:CTF
*   嘉年华:一面CTF国旗
*   嘉年华:据点
*   排名:淘汰
*   排名:一面旗帜
*   战术:杀戮者
*   战术:杀戮者突击队
*   战术:杀戮机器
*   战术:杀戮助手
*   战术:杀戮者跟踪者步枪

## 如何保存Halo Infinite的隐藏多人模式

将这些模式保存到你的Halo Infinite游戏中是很容易的:

1.  在Steam设置为离线模式的情况下打开Halo Infinite游戏。
2.  打开自定义游戏菜单。
3.  选择其中一种模式，点击“**模式编辑器**按钮。
4.  按下 **R** 按钮保存模式的副本。
