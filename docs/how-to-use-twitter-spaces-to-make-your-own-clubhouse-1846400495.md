# 如何使用Twitter“空间”打造自己的俱乐部会所[更新]

> 原文:[https://life hacker . com/how-to-use-Twitter-spaces-to-make-your-own-club house-1846400495](https://lifehacker.com/how-to-use-twitter-spaces-to-make-your-own-clubhouse-1846400495)

Twitter喜欢通过“借用”竞争对手的功能来创新。 [去年它是“舰队”，](https://lifehacker.com/how-to-send-a-disappearing-fleet-on-twitter-1845699408) 它自己版本的Instagram Stories，现在Twitter正通过其新的“空间”与蓬勃发展的社交媒体应用Clubhouse竞争。

Watch

Twitter空间是公共聊天室，任何人都可以创建和加入。主持人控制谁有发言的特权——目前最多可以有10个发言人参与对话——但是任何人都可以收听，而且听众的数量没有限制。Twitter会自动记录这些对话，你可以下载长达30天的记录。

所以，是的，Twitter的空间在功能上几乎和Clubhouse的聊天室一样——但是，你知道，是通过Twitter。

[Clubhouse作为今年早些时候的下一件大事](https://lifehacker.com/how-to-set-up-clubhouse-1846208087) (尽管它的快速增长 [似乎已经停滞](https://www.cultofmac.com/741575/social-audio-app-clubhouse-growth-appears-to-be-stalling/) )抢占了头条，所以Twitter想要进入新的格式是有道理的。我不确定Twitter是不是我想要主持公开对话的地方，但Spaces仍处于测试阶段，从现在到最终公开推出之间可能会有大量修改。

目前，功能有限；付费“售票”空间、日程安排和提醒工具以及联合托管选项等附加功能正在计划中，但目前你最多只能在移动应用程序中托管和加入空间。

截至5月3日，拥有至少600名粉丝的账户可以拥有Twitter空间，但只要使用Android或iOS应用，任何人都可以收听。当该功能完全实现时，所有用户都可以主持和加入共享空间。(有一点需要注意:受保护的账户只能*加入*个空间，不能拥有自己的空间。)

### **开始你自己的推特空间**

1.  长按**“撰写”**，然后轻触Spaces图标。(或者，进入你的Twitter个人资料，点击**“车队”**，然后滚动选择**“空间”**)
2.  选择获得发言权限的人。选项有:**“所有人”、“你关注的人”、“T1”或**“只有你邀请的人才能发言。”****
3.  如果您选择**“只有您邀请的人才能发言”，**轻按您想要邀请的人。他们将收到作为Twitter DM的邀请链接。
4.  点击**“开始你的空间”**开始，然后允许Twitter麦克风访问。

当呼叫处于活动状态时，主持人可以选择是否共享对空间副本的访问，并可以阻止、静音、删除和报告其他参与者。

### **加入其他人的推特空间**

1.  在你的时间线顶部寻找一个人的“空间”，或者打开他们的Twitter 个人资料。
2.  在他们的舰队部分，滚动直到你看到生活空间。
3.  点击以加入。

在太空中，用户可以看到其他听众，并用表情符号对对话做出反应。还有 [必要时向主持人](https://help.twitter.com/en/using-twitter/spaces) 举报他人的工具。您可以共享共享空间的链接，并且可以在会议结束后30天内下载会议记录。

Twitter计划在未来扩展您加入共享空间的方式，例如加入您直接从Twitter主订阅源关注的人主持的活动共享空间。
