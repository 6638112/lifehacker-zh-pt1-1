# 野兽先生没有抓住重点

> 原文:[https://gizmodo.com/mrbeast-misses-the-point-1848148459](https://gizmodo.com/mrbeast-misses-the-point-1848148459)

本周，YouTuber MrBeast发布了一个视频，其中有一个 [精心制作的，无死亡重建的](https://kotaku.com/mrbeasts-irl-squid-game-sees-contestants-just-running-t-1848134062) 挑战从热门韩剧 [*乌贼游戏*](https://gizmodo.com/seeing-how-squid-game-got-made-is-both-relieving-and-in-1848055193) 到他的，截至本文写作时，8290万追随者。这个视频受到了许多人的批评，他们认为这是一个音盲的噱头，完全没有抓住节目关于收入不平等的要点。作为回应，MrBeast的粉丝指出，在整个视频中，他如何赠送大量现金，甚至向那些被淘汰的人赠送，并且通常被视为人道主义者。

Watch

“野兽先生真的捐了很多钱，给无家可归的人送饭，拯救海洋等等，所以让这个人开心吧，”他的一个捍卫者在 [推特](https://twitter.com/NicoGestalt/status/1463968952418283522) 上写道。

“拯救海洋”指的是MrBeast和他的同事YouTuber Mark Rober正在开展的一项名为 [Team Seas](https://teamseas.org/) 的活动。目标是筹集3000万美元来清理海洋中3000万磅(1400万公斤)的垃圾。“让我们向世界展示我们想要一个干净的海洋，”MrBeast在 [官方竞选视频](https://www.youtube.com/watch?v=cV2gBU6hKfY) 中说，他招募了几十名志愿者在海滩上捡垃圾；截至12月初，该视频的浏览量超过4800万次，该活动在短短几周内已经筹集了1700万美元。

 [https://gizmodo.com/embed/inset/iframe?id=youtube-video-cV2gBU6hKfY&start=0](https://gizmodo.com/embed/inset/iframe?id=youtube-video-cV2gBU6hKfY&start=0) 

但是团队海洋感觉更接近于MrBeast的*乌贼游戏*和他的其他特技，而不是真正令人向往的东西。看起来很有趣，让你支持参与者，让你对结果感觉良好。但是这种景象对解决眼前更深层次的问题没有任何作用，甚至为一些污染者提供了掩盖其名誉的借口。最终，这是一种积极的娱乐形式。然而，该运动从海洋中取出这数百万英镑的努力将在令人沮丧的短时间内付诸东流。为了真正有效，这场(或任何)运动必须着眼于从源头上阻止塑料污染的更广泛的图景。

* * *

因其惊人的视频点击率，野兽先生 对数百万年轻人来说是一个家喻户晓的名字。他花了50个小时被活埋的视频是2021年YouTube的热门视频；它目前有令人瞠目结舌的1.48亿次浏览。

野兽先生的吸引力部分来自于他的行善倾向。他的许多精心设计的挑战都涉及到在最后送出大量金钱的。(他曾经花80万美元买了一个文字岛， [把它送给了一系列挑战](https://www.youtube.com/watch?v=NkE0AMGzpJY) 的获胜者。)业内人士的投票显示，他是YouTube上最受欢迎的创作者。团队海洋活动遵循了去年MrBeast发起的类似倡议，名为团队树木，筹集了2000万美元种植2000万棵树。两者都让人想起了其他MrBeast风格的特技，除了这些是为了让他的观众参与到一个好的事业中。

但在团队海洋活动开始之前，它就已经成为YouTuber社区中一些科学家的争议来源。这是因为它是与海洋清洁组织合作完成的，该组织因其从公海中收集塑料的技术而受到科学家们的广泛批评 。团队海洋活动被用来资助该组织的河流清理活动，该活动主要依靠模仿先前存在的垃圾轮的机器人，这些机器人在像巴尔的摩这样的城市工作，被动地清理河口的垃圾。科学家们说，清理海洋的最好方法之一是首先防止垃圾进入，这是像海洋清理这样的机器非常擅长的。

“我们希望团队海洋伙伴关系能够提供资金，从世界上污染最严重的河流中收集1500万磅的垃圾，”海洋清洁组织的通讯主任约斯特·杜布瓦(Joost Dubois)在一封电子邮件中说。

尽管如此，拥有生态学博士学位的科学传播者弗吉尼亚·舒特(Virginia Schutte)经营着一个YouTube频道和抖音，她记得在10月下旬推出之前，她打开了一个未上市的视频，向创作者推销如何参与Team Seas。甚至在她点击播放之前，她就能猜到清理垃圾的伙伴会是谁——还有深深的恐惧。

“我去看了视频，感觉就像‘海洋。我喜欢海洋。请不要做海洋清洁工，”她说。“他们说，‘海洋清理’，我对着手机大喊，‘不！’我知道[野兽先生]的目的是做好事。我担心他可能不知道关于海洋清理的有争议的事情。"

10月，在正式启动的前几天，舒特和一群其他气候和科学传播者在一个私人的Discord服务器上讨论他们对这场运动的担忧。其中一个邀请了团队海洋和团队树木的活动总监马特·菲茨杰拉德参与对话。(Earther多次试图联系Fitzgerald以及Team Seas和Team Trees新闻团队，询问有关活动的问题，但他们没有回复我们的任何信息。)

“我们充分意识到围绕海洋清理的海洋工作的争议，”菲茨杰拉德在与Earther分享的Discord chats中写道，尽管他本人对该项目持“极度怀疑”态度，但海洋清理是唯一一家以每磅垃圾1美元的团队海洋活动价格从事河流清理工作的非营利组织。当被问及该活动可能如何无意中促进海洋清洁工作时，菲茨杰拉德指出，团队海洋形象是该活动的“主导——通常是唯一”品牌，这意味着人们会在心理上将他们给团队海洋工作的钱联系起来，而不是海洋清洁。

但实际的竞选材料是另一回事。罗伯特的团队海洋运动发布的视频 在12月初有近2100万的浏览量，几乎完全集中在海洋清理的机器人上。这些机器人突出显示了海洋清洁组织的标志，视频强调了非营利组织的“清除海洋中所有垃圾和塑料的大胆目标”，罗伯采访了海洋清洁组织的创始人柏杨·史莱特，他十几岁时受到启发清理海洋的故事一直是该品牌大部分筹款的名片。(Slat在视频中强调，随着人们想出如何停止生产这么多塑料，他希望他的非营利组织最终“倒闭”。)

"很高兴看到Slat的创作得到了应有的认可！"视频上的一条评论写道。

舒特说:“从海洋中捡起3000万磅的垃圾是很棒的，但我担心一群人的长期遗产，他们只是走进他们倡导的自我，认为海洋清理是污染的首要解决方案。”

除了参与海洋清理，筹款还带来了一些有趣的合作伙伴。从各方面来看，Team Seas活动有望在1月份之前超过其筹集3000万美元的目标，这要归功于YouTube等公司及其首席执行官的六位数捐款，以及Shopify首席执行官的巨额捐款和一系列NFT和加密令牌的捐款。

然而， [的顶级捐赠者名单中，](https://teamseas.org/all-donors/) 也有皇家加勒比。邮轮公司 [做了一个关于禁止塑料吸管和削减60%一次性塑料的大动作](https://www.royalcaribbean.com/blog/protecting-the-environment-is-everyones-business-2/) 。但它也因其船只的环保标准而获得了地球之友的D+。这包括在水质达标方面得到F。

另一个顶级捐助者是比科夫基金会，根据团队海洋排行榜，该基金会捐助了100万美元。该基金会由吉尔·比科夫和她的丈夫j·大流士·比科夫经营，他们在将自己的公司 出售给可口可乐后赚了数亿美元。瓶装饮料行业是海洋塑料污染的巨大来源，可口可乐是世界上最大的塑料垃圾生产商。尽管对回收做出了重大承诺，但该公司拒绝设定任何停止使用一次性塑料的目标，并积极游说幕后反对回收倡议。(可口可乐也是海洋清洁组织河流项目的赞助商。)

* * *

很难不为团队海洋活动*可能发生的事情感到失望。科学家说，虽然有多种有效的方法来清理海洋垃圾，但完全清除海洋垃圾基本上是不可能的，任何忽视塑料污染的*原因*的活动都没有抓住要点。3000万磅垃圾的团队海洋目标听起来可能很高，但这只是沧海一粟。据估计，每年大约有176亿磅(80亿公斤)的垃圾被倒入海洋，相当于每小时200万磅(90万公斤)多一点。这意味着团队海洋的清理工作将在大约15个小时内完成——比MrBeast最早的病毒视频之一 中的 [少了大约8个小时，该视频中他数到10万。如果污染者得不到制止，塑料流入海洋的速度将在未来几十年内增加两倍。](https://www.youtube.com/watch?v=xWcldHxHFpo)* 

Team Seas [网站](https://teamseas.org/) 包括一个非常全面的常见问题解答，解释了塑料污染的一些核心原因，记录了石油和天然气公司是如何造成这一问题的。它还将该运动选择的清理方法联系起来，并解释说，它最终希望激发一场更大的运动，要求更持久的解决方案。但是，竞选中许多最受欢迎的视频——本质上是《大众脸》——却没有这种信息。无论是MrBeast还是Rober的官方团队Seas视频都没有提到任何塑料污染的原因，他们没有公开要求 [想要参与的创作者做任何关于问题根源或更有效参与方式的教育。](https://docs.google.com/presentation/d/1whZJ30DLCBimh2gZ0Er_qJpwwl0VY_GlEeXFB1R6BJY/edit)

然而，在视频格式中，从源头上切断塑料并不是一件难事。在一个批评团队海洋项目 的 [视频中，YouTuber Simon Clark在一个盖着塑料瓶的浴缸里拍摄自己；一只手伸进去从他手里拿走了一个瓶子。“谢谢你，”他对屏幕外的人说。"顺便问一下，你能对那些刚刚加入塑料的人做点什么吗？"作为回应，这个人扔给他更多的垃圾。该视频继续讨论解决方案以及目前正在努力解决这些问题的慈善机构，克拉克说，如果像海洋团队这样的活动将焦点转向他们，“将产生比仅仅从海洋中取出一些塑料更大的影响”。](https://youtu.be/ZSG8BtZn9-8?t=258)

 [https://gizmodo.com/embed/inset/iframe?id=youtube-video-ZSG8BtZn9-8&start=259](https://gizmodo.com/embed/inset/iframe?id=youtube-video-ZSG8BtZn9-8&start=259) 

像MrBeast这样有才华的YouTuber，拥有数百万想要帮助解决问题的订阅者的关注，应该能够找到一种有趣的方式来传达关于减少个人塑料垃圾、致力于当地回收倡议或让继续制造塑料的公司负责的信息。这样做可以极大地改变他的观众对塑料使用的看法，实现持久的变化，而不是依赖于浮华的清理，虽然重要，但掩盖了更大的问题。

也许这一切看起来太过关键。毕竟，团队海洋活动中展示的方法甚至被那些怀疑大规模海洋清洁工作的人称赞为保持社区和海滩清洁的关键。MrBeast的正面品牌并不完全是攻击像可口可乐这样的大污染者的最佳形式，可口可乐本身经营着YouTube上最受欢迎的品牌频道 之一。尽管清理3000万磅的垃圾对海洋几乎没有影响，但如果看着野兽先生在海滩上捡塑料瓶可以激励那些不关心污染的人，那就太棒了。

但MrBeast品牌的正面公关也不应被忽视。他的团队树木活动赢得了广泛的赞誉，该活动获得了来自威瑞森等公司和埃隆·马斯克等名人的捐款，并且是一部 [纪录片](https://www.businessinsider.com/discovery-channel-feature-on-youtube-movement-teamtrees-details-release-date-2019-11) 的主题；他已经 [与Rober一起在吉米·基梅尔](https://www.youtube.com/watch?v=G1YSfHWHM44) 上亮相，宣传组队出海。YouTube部分赞助了这项活动，它也受益于对本土明星的积极关注。

MrBeast有一个 [整个频道](https://www.youtube.com/channel/UCAiLfjNXkNv24uhpzUgPa6A) 致力于慈善事业，“慈善”特技是他品牌的核心部分。考虑到世界面临的无数挑战，这很好，但希望他会想办法传达如何在前进的源头解决这些问题，而不是仅仅贴上创可贴。全世界都在关注他的一举一动，如果他真的这么做了，他将有难得的机会*真正做出改变。*
