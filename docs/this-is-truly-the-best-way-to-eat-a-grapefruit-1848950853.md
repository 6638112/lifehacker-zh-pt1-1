# 这确实是吃柚子的最好方法

> 原文：<https://lifehacker.com/this-is-truly-the-best-way-to-eat-a-grapefruit-1848950853>

在我生命的大部分时间里，我以为我不喜欢葡萄柚。我喜欢那种味道，葡萄柚汽水是最好的，还有那种香味，但是真正的水果呢？太苦了，即使糖倒在上面(我爸爸过去总是这样吃)，而且太脏了(我试图舀一口时，柑橘果汁喷到自己眼睛里的次数)。但事实证明，我只是做错了一切。

葡萄柚确实比橙子或柠檬更有苦味，你可以归咎于一种叫做柚皮苷的化学物质，这是一种在许多柑橘类水果果皮中发现的类黄酮，但在 [葡萄柚](https://academic.oup.com/jn/article/131/2/235/4686993) 中含量更高。所有的柚皮苷使得这种水果耐嚼的纤维部分——即“白色物质”或髓心——尝起来相当苦。你认为它尝起来有多苦似乎取决于你的基因。这可能意味着葡萄柚不适合你，但也可能意味着你吃了太多白色的东西。



直到几年前，我通常试图用普通的旧勺子吃葡萄柚，这基本上不可能不造成巨大的滴水的混乱。这也意味着我每咬一口都会刮出大部分的髓，这可能解释了为什么我在所有的苦味下品尝果汁的甜味有困难。(是的，我知道 [葡萄柚勺](https://en.wikipedia.org/wiki/Grapefruit_spoon) 的存在，但我从未拥有过，因为我没有购买一种为食用我不喜欢的水果而设计的器具的欲望。)

然后有一天，面对办公室厨房里一碗免费的柚子，我决定再试一次。因为我不想把果汁弄得满桌子都是，所以我在谷歌上搜索了“吃柚子的最简单方法”,找到了一个简单、整洁、几乎万无一失的方法。这是下面的视频中概述的第一个(虽然我最初发现它的来源已被时间遗忘):

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-Ptnv3cFM78Y&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-Ptnv3cFM78Y&start=0) 

简而言之，正如埃丽卡·丽亚(Erica Lea)所解释的，在你用勺子从对半切开的葡萄柚中挖出水果之前，你需要用一把锋利的锯齿边缘的刀在每一部分的周围和中间切开，以便将好的东西从苦味的薄膜中分离出来。埃丽卡·丽亚也写了关于 [的食物的文章。一旦你这样做了足够多的次数，你会变得非常擅长——我可以在大约一分钟内准备好两半——如果你做对了，你会吃掉几乎所有多汁的部分和很少的髓。因为我思想开放，我也尝试了 Erica 在视频中描述的其他方法。很明显，她在准备方面比我熟练得多，但我发现，如果不把事情搞得一团糟，所有这些基本上都是不可能的。](https://www.butteredsideupblog.com/) 

*   方法 2(像剥橘子一样去皮并切片)既棘手又耗时(对我来说——她称之为快速，嗯),而且你最终会吃掉更多的髓。
*   方法 3(把它切成段，然后削去果皮)看起来很漂亮，但是，同样，给你留下了很多髓。
*   方法 4(也是方法 2，但是你要在分开后剥下每一个单独的部分)基本上是不可能的，因为如果不弄破里面的水果，很难去掉每个部分周围的薄膜。
*   方法 5 (Erica 的首选)需要很高的用刀技巧和大量繁琐的去髓工作，基本上和方法 1 一样(我要提醒你，这是吃柚子的最好方法)，但首先你得把柚子皮剥掉。我会坚持我所知道的，谢谢。不过话说回来，她确实称之为“亲”法，我绝对不是亲。
