# 第一次约会时不应该问的问题(以及应该问什么)

> 原文：<https://lifehacker.com/questions-you-should-never-ask-on-a-first-date-and-wha-1848680416>

我不会把第一次约会称为“面试”同时，一些有价值的面试技巧也能增加你第一次约会成功的几率。你要展现出最好的一面，让谈话感觉自然，并收集坐在你对面的人的重要信息。

想想你上一次的工作面试。对你来说，回答一个具体的问题可能要容易得多，比如:“c 你能告诉我你在工作中克服挑战困难处境的一次经历吗？在含糊不清的、开放式的、令人畏惧的“t 告诉我关于你自己的事情”

无论在哪种情况下，后一个问题都太宽泛，无法开始有意义的对话 n .每当我在第一次约会时感到有联系，那是因为我们都自然而然地发现我们都喜欢同一场演出的同一集，或者认为某个音乐家被高估了，或者想成为同一家餐馆的常客。在有人问我是“热情”还是“在乎诚实”之后，我从来没有感觉到一丝一毫。很多善意的第一次约会问题最终都变得过于模糊或者无聊或者两者兼而有之。

那么，一个单身的人该怎么办呢？幸运的是，与你在 [在应用](https://lifehacker.com/how-to-actually-get-good-responses-on-your-dating-app-1847740520) 上获得高质量响应的机会相比，一旦你让一个真实的、活生生的人出现在你面前，你就有了更多的回旋余地来展开对话。但就像应用程序一样，你能问的所有最好的问题都应该是具体的、原创的和个性化的。考虑到经典的第一次约会的压力会填满每一刻的沉默，你不会后悔准备好一些问题。所以有一些可怕的问题你应该在第一次约会时避免问，而应该问些什么。

## 避免这些初次约会的问题

我先假定你是无辜的，假设你甚至没有考虑过像“你多重？”这样的问题。你认为你能跑得过我吗？这个周末你能帮我照看一下我的猫吗？我会假设你是好意的，但是在遇到新的人时，努力产生对话的火花。下面的问题是善意的，但是太无聊或者太令人不快，以至于你的约会对象可能会开始策划他们的逃脱。

*   “那么，你有什么故事？”这是浪漫追求的“说说你自己”。它不仅没有人情味，它是如此的开放，以至于你没有给你的约会对象一个有意义的途径来回答它。一个好的第一次约会会给你一些后续工作，但在第一个问题上搞砸后，你最好用的特异性和独创性来表达。
*   **“你怎么还是单身？”**有些人试图把这变成一句搭讪的话，或者赞美之类的话。 ( “你这么棒……怎么还是单身？”)即使你想调情，也要理智地想想这会如何发展。另一个人可能会忸怩作态地回答“哦，谁能说……”但是内心深处，他们在想你的“赞美”是多么的讽刺。现在，他们争先恐后地将话题从为什么他们实际上是单身转移开，比如他们根深蒂固的承诺问题。或者更糟的是，你已经为他们打开了大门，将谈话引导到他们根深蒂固的承诺问题上。
*   **“我哪里吸引你了？”再次，一次错误的调情尝试。特别是这个问题，听起来要么太傲慢，要么太不自信。不管怎样，不要去寻找赞美。**
*   **“你的爱好是什么？”**太泛。理想的情况是，他们会在谈话中不经意地暗示他们的一个爱好，或者你会在谈话中有所了解。在此基础上，询问他们如何、何时以及为什么做他们正在做的事情的具体问题。
*   你的家庭是什么样的？我认为在第一次约会时，谈论家庭是公平的(也是令人着迷的)话题。然而，你必须个性化这一个。如果你感到自信，你可以试着猜猜他们是不是中间的孩子，或者你可以问他们是否和父母关系亲密。避免“d 如果你导致了你父母的离婚？”或者“你为什么不和你父亲说话？”保持清淡。
*   “红酒还是白酒？”红色。你呢。白色？酷毙了。很棒的谈话。

## **试试这些初次约会的问题吧**

如果你想真正了解你的约会对象，这里有一些启发。关键是要让对方觉得舒服，觉得有趣。在支持他们的同时，你也会让自己与众不同，成为一个有创意、自信、体贴的人。

*   “毫无疑问，哪个假期是最棒的？”坚定的立场很重要，只要你保持低风险和好玩的态度。有了这个最好的节日问题，你就可以敞开大门，倾听他们的家庭、传统、一年中最喜欢的时间、食物偏好——更不用说他们的整体创造力了。像这样的问题非常适合开玩笑。
*   “你小时候是什么样的？”这给了对方一个谈论自己和自己性格的机会，但比你现在直接要求他们描述自己的距离要远一些。
*   **“你愿意……”一个“你愿意”的提示是展示个性的绝佳方式，不会有太快泄露太多个人信息的压力。这类问题可能是天真的、轻浮的、深刻的，或者是你的任何想法。例如:“你愿意和你最亲密的朋友还是最大的敌人睡觉？”或“你愿意拥有读心术(但你永远无法关闭它)还是能够每年有一天隐身？”**
*   “你今天做了什么？”是的，我知道这看起来很基本。我知道这似乎与我自己的建议相矛盾，因为我总是说得很具体。重要的是，你仍然，嗯，*正常*。如果你遇到约会对象的第一个问题是一个 " 你会更愿意...你可能会给人留下古怪的印象。Y 你仍然需要这种类型的问题，以便了解对方是什么样的人，他们喜欢谈论什么，并确保你们两人都乐于参与进来。另外，“你今天做了什么？”比开放式的“你今天过得怎么样？”更具体、更有成效这里的要点是:不要太强势(“牧场”还是蓝纹奶酪？)不涉及常见的客套话先。
*   “如果你有 24 小时可以做任何你想做的事，你会做什么？”这个很傻，很有想象力，给提供了很多来来回回的机会，它将揭示很多关于人们的优先考虑和想象力以及所有那些废话。
*   “你投了谁的票？”有争议，但节省时间。

## 秘诀:要有趣，但要做你自己

不幸的是，第一次约会最重要的部分是弄清楚如何做你自己。我的建议是选择具体而新颖的问题，这是值得的，T2，但这不能代替让谈话尽可能自然地进行。考虑周全的问题是一个好的开始，但是约会也应该不会像工作面试或审问。
