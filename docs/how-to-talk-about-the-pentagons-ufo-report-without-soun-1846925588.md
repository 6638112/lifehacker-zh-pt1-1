# 如何谈论五角大楼的不明飞行物报告而不听起来像阴谋论者

> 原文：<https://lifehacker.com/how-to-talk-about-the-pentagons-ufo-report-without-soun-1846925588>

外星生命的想法一直以来都像是一个幻想，在流行文化中永垂不朽，在主流中被视为一个古怪的阴谋，但来自佛罗里达州的某个共和党参议员希望你知道，嗯，真相可能就在那里。



在最近的一次 60 分钟节目中，马尔科·卢比奥谈到了“不明空中现象”的存在，根据海军情报办公室和联邦调查局即将发布的报告，这些现象多年来一直在美国上空未经授权的空域飞行，令飞行员和五角大楼的高级官员感到困惑。

卢比奥在接受《坦帕湾时报》 采访时说:“我们委以保卫国家重任的男女士兵正在报告遭遇具有超强能力的不明飞行器。”。“我们不能让不明飞行物的耻辱阻止我们认真调查这件事。即将发表的报告是这一进程中的一步，但不会是最后一步。”

这些是一个人产生幻觉的传言，但是一部分越来越多的担忧是真的有不明飞行物在地球的天空中飞行，这得到了海军飞行员和高级情报官员的证实。在 *60 分钟*节目中，先进航天威胁识别项目的前主管路易斯·埃利宗多描述了具有超凡能力的技术:

> 想象一种技术，它可以承受 6 到 700 g 的重力，可以以每小时 13000 英里的速度飞行，可以躲避雷达，可以在空气和水中飞行，甚至可能在太空中飞行。哦，顺便说一下，没有明显的推进迹象，没有翅膀，没有控制表面，但仍然可以对抗地球引力的自然效应。这正是我们所看到的。

当谈到外星生命的可能性时，很容易就下结论，这是一个经常在两个阵营之间坚定划分的辩论——一个坚持认为有一个未知的真相值得研究，而另一个则认为这是科幻小说。但是，随着美国政府真的认真对待这些目击事件，我们该如何看待这个问题，你如何谈论外星生命的可能性，而不显得你会抓住任何机会宣扬一个毫无根据的阴谋？

## 了解什么关于五角飞碟的报道

近年来，国会一直在公开探索对各种所谓的 UFO 目击事件的研究，前内华达州参议员哈里·里德(Harry Reid)在 2017 年敦促五角大楼研究这一现象( [机构在当年发布了一份关于该主题的报告](https://www.cnn.com/2017/12/19/us/pilot-david-fravor-ufo-jim-sciutto-outfront-cnntv) )。

目前的报告计划在大约六周内部分公开，这是去年 2.3 万亿美元 COVID-relief 一揽子计划中巧妙地加入的一个条款的一部分。具体来说，这是适用于 2021 财年情报授权法案的条款的一部分，特别是在审查“高级空中威胁”的章节下， [华盛顿邮报在 3 月](https://www.washingtonpost.com/national-security/2021/03/23/ufo-report-covid-bill/) 报道。这项工作由卢比奥和参议院情报委员会的一个两党派系共同牵头。

海军情报办公室、 [不明空中现象特遣部队](https://www.cnn.com/2020/08/13/politics/pentagon-ufo-task-force/index.html) 和联邦调查局编制了详细资料。如果大肆宣传得到证实，6 月份的报告将成为一枚重磅炸弹，提供“对不明空中现象数据和情报的详细分析”，这可能是公众对政府先前关于不明飞行物的机密情报的最坦率的看法。

当然，这份报告并非没有先例。去年 4 月，国防部解密了海军飞行员在 2004 年和 2015 年拍摄的两段视频。这些视频随后分别在 2007 年和 2017 年被泄露，显示了与飞机的离奇相遇，这让军方官员感到困惑。尽管这些年来这些视频引发了人们的深思，但五角大楼并没有得出任何结论或给出明确的答案， [在一份声明](https://www.cnbc.com/2020/04/28/pentagon-declassifies-ufo-videos-taken-by-navy-pilots.html) 中指出，“视频中观察到的空中现象仍被定性为‘不明’。”"

现在， *60 分钟*片段在全国播出，这一阴谋被进一步点燃，很可能预示着一个重大的新闻事件。前情报官员埃利宗多在接受哥伦比亚广播公司采访时，用了一种不祥的语气:“我不是说这听起来不古怪...我告诉你的是这是真的。问题是，它是什么？其意图何在？它的能力是什么？”

## 我们现在如何谈论不明飞行物？

虽然所有这些证据都可以构成非常引人注目的戏剧，但事实仍然是你真的什么都不知道。就平民而言，我们只能附和目睹这些奇怪事件的情报官员和海军飞行员的公开声明，以免你听起来像个阴谋论者。

当然，从五角大楼到参议院和联邦调查局的官员都在闪烁其词，敦促保持谨慎，还有一个事实——很可能是——政府仍然在保护这些未知目击事件的敏感细节。

我们所能做的，就像那些无法接触到事实或机密政府文件的人所做的，就是推测，这很好，可能也很有趣，但要明白我们 *所能*做的就是推测。仅仅基于几个简短的新闻视频来假设你自己的理论会让某人看起来像一个让他们的想象力天马行空的孩子。

我的意思是，以削弱人类第一次(可能)与外星生命擦肩而过的所有乐趣为代价，让我们帮自己一个忙，等到我们知道更多之后再大喊“外星人！”从我们卧室的窗户出去。毕竟，该报告将于下月发布。然后我们可以尖叫更多。
