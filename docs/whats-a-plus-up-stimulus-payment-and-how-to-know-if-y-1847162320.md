# 什么是追加刺激付款？(以及如何知道你是否会得到一个)

> 原文：<https://lifehacker.com/whats-a-plus-up-stimulus-payment-and-how-to-know-if-y-1847162320>

你可能会从国税局收到更多的钱，但这不是第四张刺激支票——这是“追加付款”如果你的第三笔刺激付款是使用 2019 年报税表中的收入计算的，并且如果你 2020 年的收入低于 2019 年，你将收到这张支票。下面是追加付款的工作方式。



## **追加支付涵盖了你第三次刺激计划的剩余部分**

由于你的第三份刺激计划支票是在 2020 年纳税季中期发出的，美国国税局不得不使用手头的收入信息来计算你的付款，无论这是 2019 年还是 2020 年的纳税申报单。因此，如果你的第三次刺激检查是基于你 2019 年的回报，而你的收入从那以后*减少了*，那么国税局仍然欠你差额。

请注意，您的支付金额将根据您的收入而有所不同。如果你在 2020 年的收入低于 75，000 美元(或联合申报人的收入低于 150，000 美元)，那么你将有权获得 1，400 美元。但是，如果您的收入超过这些阈值，那么如果您的收入为 80，000 美元(或 160，000 美元的联合申报人)，该金额将迅速减少到 0 美元。另一种方式你会被欠更多的钱，如果你*从 2019 年起在你的家庭中增加了*受抚养人，因为他们也有资格获得刺激付款。

## **如何申请追加刺激付款**

如果你欠了一个加号，国税局会根据你 2020 年的纳税申报表自动计算你欠了多少钱，所以你除了确保纳税申报表存档之外，什么都不用做。然而，如果你要求延期报税，请注意，你的 2020 年申报表*必须在 8 月 16 日之前*提交并处理，才能收到加付款， [per Kiplinger](https://www.kiplinger.com/taxes/tax-deadline/601054/tax-extension-how-to-get-extra-time-to-file-your-taxes) 。

如果你正在期待一张支票，但还没有收到，很可能国税局还没有处理它，因为他们仍在处理大量积压的工作，其中仍包括 2019 年的纳税申报单。

## 如果前两张支票欠我钱怎么办？

在这一点上，你可以得到那笔钱的唯一方法是在你的 2020 年纳税申报表中申请 [回收退税信贷](https://www.irs.gov/newsroom/recovery-rebate-credit) 。
