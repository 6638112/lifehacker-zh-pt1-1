# 如何观看英伟达今天的“GeForce RTX:游戏开始”活动

> 原文:[https://life hacker . com/how-to-watch-nvidias-geforce-RTX-game-on-event-today-1846037405](https://lifehacker.com/how-to-watch-nvidias-geforce-rtx-game-on-event-today-1846037405)

今天晚些时候，Nvidia的“GeForce RTX: Game On”现场活动将让我们首次看到该公司最新的游戏硬件。(这里希望任何新的公告 [降低](https://lifehacker.com/why-graphics-cards-are-even-more-expensive-in-2021-1846036657) 的价格，提高顶级显卡的可用性。)

Watch

直播将于1月12日星期二美国东部时间中午/太平洋时间上午9点开始。可以在 [Nvidia官方直播活动页面](https://www.nvidia.com/en-us/geforce/special-event/) ，以及其官方 [YouTube](https://www.youtube.com/NVIDIA) 和 [Twitch](https://www.twitch.tv/nvidia) 频道观看。官方页面也有链接，可以给你的Google或者iCal日历添加提醒。

Nvidia GeForce部门高级副总裁杰夫·费舍尔(Jeff Fisher)将主持此次活动。这意味着我们可能不会再次看到英伟达首席执行官黄仁勋的 [莫名其妙的大刮刀系列](https://www.youtube.com/watch?v=So7TNRhIYJ8) ，但至少我们会看到英伟达RTX 3000系列的下一组显卡——希望如此。

Nvidia没有给出任何关于它计划推出什么的暗示，但我们可以根据泄露和传言假设一个或两个以上的RTX 3000系列GPU将被宣布。这包括RTX 3050，理论上这将是Nvidia当前一代卡中最实惠的。也有传言称，Nvidia正在开发RTX 3080 Ti，将填补RTX 3080和3090之间的规格和价格差距。

同样，3050和3080 Ti都只是传言，但从商业角度来看，它们是有意义的。供应短缺和 [进口关税正在推高所有](https://lifehacker.com/why-graphics-cards-are-even-more-expensive-in-2021-1846036657)GPU的价格。为预算有限的客户提供一个实惠的选择是一个明智的举动，而一个新的高端型号可能有助于增加你在售罄之前找到任何*类型的*高端3000系列卡的几率。

预计英伟达还将展示其用于笔记本电脑和便携式游戏硬件的移动RTX芯片。泄露的华硕即将推出的笔记本电脑规格包括RTX 3080，它通常太大，不适合放在笔记本电脑中。我们也很好奇这里会发生什么——假设所说的泄露是准确的。
