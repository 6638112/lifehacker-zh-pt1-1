# 修订版类似于 wordle，但是是针对整个维基百科页面的

> 原文：<https://lifehacker.com/redactle-is-like-wordle-but-for-a-whole-wikipedia-page-1849648942>

沃尔多很小；Redactle 很大。当 Wordle 要求你解决一个五个字母的单词时，Redactle 会呈现给你一个涂黑的维基百科页面，而不是一个存根。你一次猜一个单词，你有无限的猜测机会。你甚至可以得到一堆免费赠品，比如“曾经”、“曾经”、“为了”。那么这有什么难的呢？

嗯，整个想法是你需要弄清楚这篇文章是关于哪个特定的人或事或地方。在 Wordle 中，知道几个常见的字母通常就足以让你走完大部分路程；如果我找到了一些像 E，T，R 这样的字母，那就没什么可做的了。但是在 Redactle 中，知道“他”出现了 159 次真的对你有帮助吗？也许吧。算是吧。

## 玩编校游戏是什么感觉

我将带你玩一个我昨天玩过的游戏(今天没有剧透)。这个页面很长，这是有道理的，因为它取自维基百科的 10，000 篇重要文章——所以它不太可能是一个存根或任何太不为人知或晦涩难懂的东西。

一篇维基百科文章的结构很明显，标题用大号字体，尽管标题中的大部分单词都被涂黑了。有一个项目符号列表，上面有很多“的”和“关于的”，这表明这可能是关于一个写了很多书的人。因为页面是以等宽字体布局的，所以你可以猜测单词的长度，即使你不知道单词是什么。

但是*是关于一个人吗？标题只有一个词，没有名字和姓氏的空间。前几个单词之间有一个连字符，这让我想起了那些法语连字符的名字。*

*我从“出生”开始，获得了三次成功。所以可能是个人。我心血来潮尝试了“法国”，得到了 19 次点击。“Book”找到了 20 个，但“author”只有 3 个，而且在第一句话中没有一个能指出一个人最出名的地方。我发现自己在特定的、不寻常的词(这将揭示更多细节)和传记中出现的常用词(这更有可能成为热门词)之间寻找一个平衡点。“国王”有几个热门词，所以我想到了美国/英国/法国革命时代。不过，对“英国”或“美国”来说并不多。我猜是“女人”，我以为一个著名的女性会被描述成很多“第一个做这样那样事情的女人”一击命中。所以我试了试“他”，得到了 159 分，包括引言中的几个。好的，是个纨绔子弟。找不到"剧作家"、"西班牙"、"医学"、" 1700 "或" 1800 "的匹配。最后，我决定从法国的角度出发，猜“法国”(34 次点击)、“革命”(2 次点击)和“启蒙运动”(8 次点击)。启蒙就在第一句话里。*

*得了吧，得了吧，能登上维基百科顶级文章的名人就那么几个。让我们试试某个著名的人；也许一次失误会让你提到合作者或灵感。伏尔泰？没错。 [全篇](https://en.wikipedia.org/wiki/Voltaire) 揭晓。我是 46 次猜对得到的，猜对准确率 89.13%。*

## *编校有多难？*

*我发现这很难，很令人沮丧，但我昨天的分数接近中间值(根据当天的统计，43 分)。像沃尔多一样，每个人每天都会遇到同样的难题，有些日子会比其他日子更难。在 Reddit 上一个关于游戏 的帖子上，有人提到要猜 250 次才能解决一个难题。*

## *我们什么时候能拿到新的版本？*

## *编辑的最佳策略是什么？*

*嗯，我正在想办法。将来，我可能会尝试更具体的词；“启蒙运动”和“革命”将是“法国”之后的良好后续，而不是在游戏的后期播放万福玛利亚。*

*我还将开始更多地关注维基百科文章的标题，以及开头段落中描述人和事物的方式。例如，事实证明伏尔泰被描述为“作家”而不是“作者”。*

*a[Reddit thread on strategy](https://www.reddit.com/r/Redactle/comments/u90k25/what_are_some_common_strategies_with_this/)暗示好的线索词可能包括:*

*   *“W ar”*
*   *“城市”或“国家”*
*   *“Ffirst”或“last”*
*   *“她”或其他代词*
*   *描述地区的词，如“欧洲”或“美国”*
*   *“C entury”(然后，如果这是一个命中，猜测类似“16 日”的数字)*

*在那篇文章中还有一些关于识别特定短语格式的提示，比如某人的出生和死亡日期，维基百科在翻译外国短语时使用的分号。所以试一试吧，留心观察，尽情享受吧！*
